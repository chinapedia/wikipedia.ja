**ベータ分布**（ベータぶんぷ、）は、[連続確率分布](../Page/連続確率分布.md "wikilink")であり、第1種ベータ分布および[第2種ベータ分布](https://ja.wikipedia.org/wiki/第2種ベータ分布 "wikilink")がある。単にベータ分布と呼んだ場合、第1種ベータ分布を指す。

## 第1種ベータ分布

第1種ベータ分布（）の[確率密度関数](../Page/確率密度関数.md "wikilink")は以下で定義される。

\[f(x; \alpha ,\beta )=\frac{x^{\alpha -1}(1-x)^{\beta-1}}{B(\alpha ,\beta)}\] ここで  は[ベータ関数](https://ja.wikipedia.org/wiki/ベータ関数 "wikilink")であり、確率変数の取る値は 、パラメータ  はともに正の実数である。期待値は 、分散は \(\frac{\alpha\beta}{(\alpha+\beta)^2(\alpha+\beta+1)}\) である。自然パラメータを  (*α* − 1, *β* − 1)}} として以下のように書き換えられるので、ベータ分布は指数型分布族である。

\[f(x;\eta )=h(\eta )\exp (\eta \cdot u(x))\] ただし \(h(\eta )=\frac{1}{B(\alpha ,\beta )} ,u(x)=(\log x,\log (1-x))\) である。

### 累積分布関数

[累積分布関数](../Page/累積分布関数.md "wikilink")は、以下の式で与えられる。

\[F(x;\alpha,\beta) = \frac{\Beta{}(x;\alpha,\beta)}{\Beta{}(\alpha,\beta)} = I_x(\alpha,\beta)\]

ここで、 \(\Beta(x;\alpha,\beta)\) は、[ベータ関数](https://ja.wikipedia.org/wiki/ベータ関数 "wikilink")であり、 \(I_x(\alpha,\beta)\) は、正則化[不完全ベータ関数](../Page/不完全ベータ関数.md "wikilink")である。

## 一般化ベータ分布

a, b, c, p, q が実数パラメータで、0 ≦ c ≦ 1 で、b, p, q が正の時、下記の確率密度関数を一般化ベータ分布（）という。

\[GB(x;a,b,c,p,q) = \frac{|a|x^{ap-1}(1-(1-c)(x/b)^{a})^{q-1}}{b^{ap}B(p,q)(1+c(x/b)^{a})^{p+q}} \quad \quad \text{  for } 0<x^{a}< \frac{b^a}{1-c} ,\]

### 一般化第1種ベータ分布

c = 0 の時、一般化第1種ベータ分布（）という。

\[GB1(x;a,b,p,q) = GB(x;a,b,c=0,p,q).\]

\[GB1(x;a,b,p,q) = \frac{|a|x^{ap-1}(1-(x/b)^{a})^{q-1}}{b^{ap}B(p,q)}\]

### 一般化第2種ベータ分布

c = 1 の時、一般化第2種ベータ分布（）という。台は \(x \in (0,\infty)\!\) 。

\[GB2(x;a,b,p,q) = GB(x;a,b,c=1,p,q).\]

\[GB2(x;a,b,p,q) = \frac{|a|x^{ap-1}}{b^{ap}B(p,q)(1+(x/b)^a)^{p+q}}\]
