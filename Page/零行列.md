[数学](../Page/数学.md "wikilink")において、**零行列**（ぜろぎょうれつ、れいぎょうれつ、zero matrix, null matrix）とは、その成分（[要素](../Page/行列要素.md "wikilink")）が全て 0 の[行列](https://ja.wikipedia.org/wiki/行列 "wikilink")。*O* あるいは 0 と記述されることが多い。

\[0 = \begin{pmatrix}
  0      & 0      & 0      & \cdots & 0      \\
  0      & 0      & 0      & \cdots & 0      \\
  \vdots & \vdots & \vdots & \ddots & \vdots \\
  0      & 0      & 0      & \cdots & 0
\end{pmatrix}\] また、下付き添字によって行列の型を明記することもある。

\[O_2 = O_{2,2} = \begin{pmatrix} 0 & 0 \\ 0 & 0 \end{pmatrix},
O_{2,3} = \begin{pmatrix} 0 & 0 & 0 \\ 0 & 0 & 0 \end{pmatrix}\] 自明な[線形変換](https://ja.wikipedia.org/wiki/線形変換 "wikilink")である零作用素を表す行列であり、正方行列の場合には行列環の零元を与えている。

## 性質

以下、*l*, *m*, *n* は任意の自然数とする。

  - *m* 行 *n* 列の零行列 *O* と *m* 行 *n* 列の任意の行列 *A* の和は *A* + *O* = *O* + *A* = *A* となり、差は *A* − *O* = *A*, *O* − *A* = −*A* となる。
  - *l* 行 *m* 列の零行列 *O* と *m* 行 *n* 列の任意の行列 *A* の積 *OA* は、*l* 行 *n* 列の零行列となる。
  - *l* 行 *m* 列の任意の行列 *B* と *m* 行 *n* 列の零行列 *O* の積 *BO* は、*l* 行 *n* 列の零行列となる。

これらのことから、*n* 次の[正方行列](../Page/正方行列.md "wikilink")全体のなす[環を考えているとき](../Page/環論.md "wikilink")、零行列はその[零元](https://ja.wikipedia.org/wiki/零元 "wikilink")になる。
