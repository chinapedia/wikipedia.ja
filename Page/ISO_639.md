**ISO 639**は、[言語](../Page/言語.md "wikilink")の名称の略号を規定した国際規格である。[国際標準化機構](https://ja.wikipedia.org/wiki/国際標準化機構 "wikilink")が発行。複数の巻で構成され、現在は5巻が公表されている。日本工業規格では「JIS X 0412」がほぼ相当する。

ISO 639として正式に標準化されているのは以下の5つである。

  - [ISO 639-1](https://ja.wikipedia.org/wiki/ISO_639-1 "wikilink"): 2002 <span lang="en">*Codes for the representation of names of languages -- Part 1: Alpha-2 code*</span>（言語名コード-第一部:2文字コード）2002年制定

  - [ISO 639-2](https://ja.wikipedia.org/wiki/ISO_639-2 "wikilink"): 1998 <span lang="en">*Codes for the representation of names of languages -- Part 2: Alpha-3 code*</span>（言語名コード-第二部:3文字コード）1998年制定

  - [ISO 639-3](https://ja.wikipedia.org/wiki/ISO_639-3 "wikilink"): 2007 <span lang="en">*Codes for the representation of names of languages -- Part 3: Alpha-3 code for comprehensive coverage of languages*</span>（包括的な3文字言語コード）2007年制定

  - : 2010 <span lang="en">*Codes for the representation of names of languages -- Part 4: Implementation guidelines and general principles for language coding*</span>（実装のためのガイドライン）2010年制定

  - : 2008 <span lang="en">*Codes for the representation of names of languages -- Part 5: Alpha-3 code for language families and groups*</span>（言語のグループを表すための3文字コード）2008年制定

以下は廃止された。

  - : 2009 <span lang="en">*Codes for the representation of names of languages -- Part 6: Alpha-4 representation for comprehensive coverage of language variation*</span>（言語変種のための4文字コード）2009年制定、2014年廃止\[1\]。

第一部で規定されているのがアルファベット2文字による表記であり、第二部では同3文字の組み合わせによる識別子群を規定している。3文字コードには著書目録など書誌用の「Bコード」と用語学用の「Tコード」の2種類があり、22の言語でそれぞれ異なる識別子を持つが、それ以外は同一である。
