**名岐鉄道デボ600形電車**（めいぎてつどうデボ600がたでんしゃ）は、[現・名古屋鉄道](https://ja.wikipedia.org/wiki/名古屋鉄道 "wikilink")（名鉄）の前身事業者の一つである旧・名古屋鉄道が導入した**[1500形電車](https://ja.wikipedia.org/wiki/名古屋電気鉄道1500形電車 "wikilink")**のうち1519 - 1525の7両について、旧・名古屋鉄道の後身である[名岐鉄道当時の](https://ja.wikipedia.org/wiki/名古屋鉄道#名古屋電気鉄道 "wikilink")[1935年](../Page/1935年.md "wikilink")（[昭和](../Page/昭和.md "wikilink")5年）に実施された形式称号改訂に際して付与された形式区分である。

デボ600形に区分された7両は、[1925年](../Page/1925年.md "wikilink")（[大正](../Page/大正.md "wikilink")14年）に製造された[2軸ボギー構造を採用した木造車体の](../Page/ボギー台車.md "wikilink")[電車](../Page/電車.md "wikilink")（[制御電動車](../Page/動力車.md "wikilink")）で、先行して落成した1500形1501 - 1518（後の[名岐デボ300形・デボ350形および](https://ja.wikipedia.org/wiki/名岐鉄道デボ300形電車 "wikilink")[デボ400形・デボ450形](https://ja.wikipedia.org/wiki/名岐鉄道デボ400形電車 "wikilink")）と同様に「郡部線」と通称される鉄道線区間に導入された。

## 沿革

1500形1511 - 1518に次ぐ増備車として、1925年（大正14年）8月から同年11月にかけて1519 - 1525の7両が名古屋電車製作所において新製された\[1\]。木造二重屋根（ダブルルーフ）構造の車体は1511 - 1518と同様であるが、[側面窓配置が変更されるなど](https://ja.wikipedia.org/wiki/構体_\(鉄道車両\)#側面窓配置 "wikilink")、一部設計変更が加えられた\[2\]。主要機器については1511 - 1518と同じく[イングリッシュ・エレクトリック](https://ja.wikipedia.org/wiki/イングリッシュ・エレクトリック "wikilink") (EE) 製の機種を採用するが、主電動機を従来の1両あたり2基搭載から4基搭載に増強して出力向上および動力性能の向上を図った\[3\]。1519 - 1525にて確立された主要機器の仕様は、1500形に次いで導入された[デボ650形のほか](https://ja.wikipedia.org/wiki/名古屋鉄道デボ650形電車 "wikilink")、旧・名古屋鉄道初の鋼製車である[デセホ700形・デセホ750形にも継承された](../Page/名古屋鉄道デセホ700形電車.md "wikilink")\[4\]。また、1521以降の5両は落成当初より集電装置として[パンタグラフを旧](https://ja.wikipedia.org/wiki/集電装置#パンタグラフ "wikilink")・名古屋鉄道の在籍車両において初めて採用した点も特筆される\[5\]。

旧・名古屋鉄道が[1925年](../Page/1925年.md "wikilink")（大正14年）に[尾西鉄道](https://ja.wikipedia.org/wiki/尾西鉄道 "wikilink")の鉄道事業譲受を機に実施した車番改訂に際しては\[6\]、1519 - 1525は**デボ600形**601 - 607と形式称号および記号番号を改めた\[7\]。さらに現・名古屋鉄道（名鉄）成立後には**モ600形**（初代）と車両番号はそのままに形式称号のみを改め、[1966年](../Page/1966年.md "wikilink")（昭和41年）まで運用された\[8\]。なお、モ600形（初代）が搭載した台車・主要機器は[廃車後に](../Page/廃車_\(鉄道\).md "wikilink")[制御車](../Page/制御車.md "wikilink")ク2330形（元[モ910形](../Page/知多鉄道デハ910形電車.md "wikilink")）へ転用され、同形式の再電動車化に際して用いられた\[9\]。

## 車体・主要機器

全長14 m級の、屋根部をダブルルーフ構造とした木造車体を備える\[10\]。運転台を前後妻面に設けた両運転台構造とし、妻面は大きな円弧を描く丸妻形状で、3枚の前面窓を均等配置し、妻面裾部には連結器取付座を兼ねた台枠端梁が露出する\[11\]。側面に3箇所設けられた客用扉は、先行して導入された1511 - 1518と同様に両端部の扉が両開構造の引扉で、中央部の扉のみ片開構造の引扉である\[12\]。ただし、1519 - 1525においては両端扉の戸袋部に戸袋窓が設置され、車体中央側戸袋窓については楕円形の丸窓とした点が異なる\[13\]。各客用扉間には丸窓のほか5枚の側面窓を配し、側面窓配置は1 D e 5 D 5 e D 1（D：客用扉、e：丸窓）とである\[14\]。その他、二重屋根部の両脇に設置される通風器（ベンチレーター）が従来の水雷形からガーランド形に改良されている\[15\]。

主要機器は前述の通りイングリッシュ・エレクトリック (EE) の製品を採用、制御装置は同社の前身事業者の一つである[ディック・カー・アンド・カンパニーが開発した](https://ja.wikipedia.org/wiki/:en:Dick,_Kerr_&_Co. "wikilink")、「デッカーシステム」と通称される電動カム軸式自動加速制御器を採用した\[16\]。主電動機は同じくEE製の定格出力70 [PSの](https://ja.wikipedia.org/wiki/馬力#英馬力 "wikilink")[直流直巻電動機を](../Page/直巻整流子電動機.md "wikilink")1両あたり4基搭載した\[17\]。制動装置については先行導入された1501 - 1518において採用されたゼネラル・エレクトリック (GE) 製のGE非常直通ブレーキではなく\[18\]、[ウェスティングハウス・エア・ブレーキ](https://ja.wikipedia.org/wiki/ウェスティングハウス・エア・ブレーキ "wikilink") (WABCO) 製のSME[非常直通ブレーキを採用した](https://ja.wikipedia.org/wiki/直通ブレーキ#SME "wikilink")\[19\]。台車は1511 - 1518において採用された[ボールドウィン・ロコモティブ・ワークス](../Page/ボールドウィン・ロコモティブ・ワークス.md "wikilink") (BLW) 製の[ボールドウィンA形台車](../Page/ボールドウィンA形台車.md "wikilink")を設計の基本として住友製鋼所（のちの[住友金属工業](../Page/住友金属工業.md "wikilink")）が模倣製造したST-2形鋼組立形[釣り合い梁式台車を装着する](https://ja.wikipedia.org/wiki/鉄道車両の台車#イコライザー式 "wikilink")\[20\]。

集電装置は、1519・1520の2両は[トロリーポールを屋根上に前後各](https://ja.wikipedia.org/wiki/集電装置#トロリーポール "wikilink")1基搭載して落成したが\[21\]、1521 - 1525の5両は屋根上前後のトロリーポールのほか屋根上中央部に菱形パンタグラフを1基搭載して落成した\[22\]。1519・1520についても落成後間もなくパンタグラフが追加搭載された\[23\]。

## 運用

[Meitetsu_mo_605_iwakura.jpg](https://ja.wikipedia.org/wiki/File:Meitetsu_mo_605_iwakura.jpg "fig:Meitetsu_mo_605_iwakura.jpg")で運用されるモ600形605\]\] 導入後は郡部線と総称される各路線において運用され、尾西鉄道譲受後に実施された車番改訂\[24\]に際しては、1519 - 1525は**デボ600形**601 - 607と、新たな形式称号および記号番号が付与された\[25\]。後年、両開構造であった両端部の客用扉を中央部の客用扉と同じく片開構造に改め、さらに時期は不明ながら後年丸窓の埋込撤去が実施されて側面窓配置は1 D 5 D 5 D 1となり\[26\]、デボ400形（旧1500形1511 - 1518）との車体外観上の差異は屋根上ベンチレーターの相違程度となった\[27\]。

名岐鉄道と[愛知電気鉄道の対等合併による現](https://ja.wikipedia.org/wiki/名古屋鉄道#愛知電気鉄道 "wikilink")・名古屋鉄道（名鉄）成立後の[1941年](https://ja.wikipedia.org/wiki/1941年 "wikilink")（昭和16年）2月に実施された形式称号改訂\[28\]に際しては、デボ600形は**モ600形**（初代）601 - 607と車両番号はそのままに形式称号のみを改めた\[29\]。

[太平洋戦争](https://ja.wikipedia.org/wiki/太平洋戦争 "wikilink")末期から終戦直後にかけて、モ602が戦災によって、モ603・モ604は事故によってそれぞれ車体を焼失し\[30\]、いずれも原形に近い木造車体を新製して復旧されたが、同3両は屋根部構造が原形のダブルルーフからシングルルーフ（丸屋根）へ改められた\[31\]。

[1948年](../Page/1948年.md "wikilink")（昭和23年）5月に、架線電圧が600 [V規格であった旧名古屋鉄道](../Page/ボルト_\(単位\).md "wikilink")・名岐鉄道に由来する西部線に属する各路線のうち、現在の犬山線・津島線および名古屋本線の一部に相当する区間の架線電圧1,500 V昇圧が実施された\[32\]。架線電圧600 V仕様の制御電動車であったモ600形（初代）は架線電圧600 V規格のまま存置された支線区用へ転用され、[小牧線](https://ja.wikipedia.org/wiki/名鉄小牧線 "wikilink")・[広見線など主に犬山地区の支線区において運用された](https://ja.wikipedia.org/wiki/名鉄広見線 "wikilink")\[33\]。

その後、支線区における架線電圧1,500 V昇圧工事の完成や新型車両の導入に伴って、モ600形（初代）は順次[瀬戸線へ転属し](https://ja.wikipedia.org/wiki/名鉄瀬戸線 "wikilink")\[34\]、広見線の架線電圧1,500 V昇圧に際して余剰となったモ607が[1965年](../Page/1965年.md "wikilink")（昭和40年）4月に転属したのを最後に全車とも瀬戸線へ集約され\[35\]、同路線の主力車両として運用された\[36\]。またその間、最後に転属したモ607を除くモ601 - モ606の6両を対象として、外板に鋼板を貼り付けて補強する簡易鋼体化改造が[1961年](https://ja.wikipedia.org/wiki/1961年 "wikilink")（昭和36年）\[37\]から[1963年](../Page/1963年.md "wikilink")（昭和38年）6月\[38\]にかけて順次施工された。

同時期には本線系統において余剰となった半鋼製車体を備える各形式の瀬戸線への転属による、瀬戸線運用車両の体質改善が進められたが\[39\]、転属車両の中には[3730系ク](../Page/名鉄3700系電車_\(2代\).md "wikilink")2730形の新製に際して台車を含む主要機器を供出するため、車体のみが転属対象となったク2330形が含まれていた\[40\]。このク2330形の車体と、瀬戸線所属の木造電動車としては最晩年まで残存したモ600形（初代）の台車・主要機器を組み合わせて**モ900形**として導入することとし\[41\]、瀬戸線在籍車両の全面鋼体化の方針が策定された\[42\]。

| 車体    | 機器供出元 | モ900形             |
| ----- | ----- | ----------------- |
| ク2331 | モ601  | モ901 I （→モ907 II） |
| ク2332 | モ604  | モ902 I （→モ906 II） |
| ク2333 | モ602  | モ903 I （→モ905 II） |
| ク2334 | モ606  | モ904              |
| ク2335 | モ603  | モ905 I （→モ903 II） |
| ク2336 | モ605  | モ906 I （→モ902 II） |
| ク2337 | モ607  | モ901 II           |
|       |       |                   |

この一連の改造に際して、唯一簡易鋼体化改造を施工されなかったモ607が1965年（昭和40年）8月24日付\[43\]で除籍となったことを皮切りに、以降ク2330形の本線系統からの転属・整備入場に際して台車・主要機器を供出する目的でモ600形（初代）の廃車が順次進行した\[44\]。そして1966年（昭和41年）3月22日付\[45\]のモ605の除籍をもって、名古屋鉄道1519 - 1525として導入された車両群は全廃となった\[46\]。

なお、ク2330形2337への機器転用が予定されたモ607の主要機器のうち、ST-2台車のみは一旦モ750形751へ転用され、モ751が従来装着した台車は3730系ク2730形の新製に際して用いられた\[47\]。その後ク2337がモ900形901（2代）として導入された際、モ751から再びST-2台車を転用して装着し\[48\]、モ751は[揖斐線系統への転属に際して台車を含む主要機器換装が実施されたモ](https://ja.wikipedia.org/wiki/名鉄揖斐線 "wikilink")757\[49\]より発生した台車を装着するという\[50\]、複雑な台車振替が実施されている。
