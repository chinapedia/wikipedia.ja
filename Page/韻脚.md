verse（[韻文](https://ja.wikipedia.org/wiki/韻文 "wikilink")、[詩](https://ja.wikipedia.org/wiki/詩 "wikilink")）における脚（**pes; foot**）は、**韻脚**（いんきゃく）、**詩脚**（しきゃく）あるいは**音歩**と訳され、詩のrhythm（[リズム](../Page/リズム.md "wikilink")、律動）の基本単位として、多くのmeter（[韻律](../Page/韻律_\(韻文\).md "wikilink")、格調）に用いられる。

例えば「Shall I compare thee to a summer's day?」（[ウィリアム・シェイクスピア](../Page/ウィリアム・シェイクスピア.md "wikilink")『Sonnet 18』）は、5つの韻脚から成っている。（「//」は韻脚の区切りで、太字は強いアクセント）。

  - Shall **I** // com-**pare** // thee **to** // a **sum**- // mer's **day**?

韻脚は特定の数の音節から成り、それは複数の言葉から成る場合（Shall I）も1語だけで成る場合（com-pare）もある。また、語が韻脚をまたいでもよい（sum- ・ mer's）。

多くの[英語](../Page/英語.md "wikilink")詩や[ドイツ語](../Page/ドイツ語.md "wikilink")詩ではアクセントの強・弱（揚・抑）で韻律をつけるが、[ギリシャ語](https://ja.wikipedia.org/wiki/ギリシャ語 "wikilink")や[ラテン語](../Page/ラテン語.md "wikilink")で書かれた古典詩は音量（quantity）つまり母音の長・短でつける。

## 韻脚の種類

以下のリストは音節の数ごとに分類された韻脚の種類で、名称は古典詩の韻律学によって付けられた名前である。

英語詩ならびにドイツ語詩で最も一般的なものは、[アイアンブ](../Page/アイアンブ.md "wikilink")、[トロキー](https://ja.wikipedia.org/wiki/トロキー "wikilink")、[ダクティル](https://ja.wikipedia.org/wiki/ダクティル_\(詩\) "wikilink")、[アナペスト](https://ja.wikipedia.org/wiki/アナペスト "wikilink")である。

### 2音節（Disyllables）

*`u` = 短・ `-` = 長/古典詩/英語詩*

|       |                                                                  |                                                                    |
| ----- | ---------------------------------------------------------------- | ------------------------------------------------------------------ |
| `u u` | ピュリキオス（短短格）                                                      | pyrrhic（[ピリック](https://ja.wikipedia.org/wiki/ピリック "wikilink")、弱弱格） |
| `u -` | [イアンボス](https://ja.wikipedia.org/wiki/イアンボス "wikilink")（短長格）     | iamb（アイアンブ、弱強格）                                                    |
| `- u` | [トロカイオス](https://ja.wikipedia.org/wiki/トロカイオス "wikilink")（長短格）   | trochee（[トロキー](https://ja.wikipedia.org/wiki/トロキー "wikilink")、強弱格） |
| `- -` | [スポンデイオス](https://ja.wikipedia.org/wiki/スポンデイオス "wikilink")（長長格） | spondee（スポンデー, 強強格）                                                |
|       |                                                                  |                                                                    |

  - 英語詩のアイアンブ（弱強格）の例 - To **strive**, to **seek**, to **find**, and **not** to **yield**.（[アルフレッド・テニスン](https://ja.wikipedia.org/wiki/アルフレッド・テニスン "wikilink")『ユリシーズ』）
  - 英語詩のトロキー（強弱格）の例 - **Should** you **ask** me, **whence** these **stor**-ies?（[ヘンリー・ワーズワース・ロングフェロー](../Page/ヘンリー・ワーズワース・ロングフェロー.md "wikilink")『[ハイアワサの歌](../Page/ハイアワサの歌.md "wikilink")』）

*`u` = 短・ `-` = 長/古典詩/英語詩*

| `u u u` | トリブラキュス（短短短格）                                                   | tribrach（弱弱弱格）      |
| `- u u` | [ダクテュロス](https://ja.wikipedia.org/wiki/ダクテュロス "wikilink")（長短短格） | dactyl（ダクティル、強弱弱格）  |
| `u - u` | アンピブラキュス（短長短格）                                                  | amphibrach（弱強弱格）    |
| `u u -` | アナパイストス（短短長格）                                                   | anapest（アナペスト、弱弱強格） |
| `u - -` | バッケイオス（短長長格）                                                    |                     |
| `- - u` | 逆バッケイオス（長長短格）                                                   |                     |
| `- u -` | クレティコス（長短長格）                                                    |                     |
| `- - -` | モロッソス（長長長格）                                                     | molossus（強強強格。稀に）   |

  - 英語詩のダクティル（強弱弱格）の例 - **This** is the **for**-est prim-**ev**-al. The **mur**-muring **pines** and the **hem**locks（ロングフェロー『ユヴァンジェリン』）
  - 英語詩のアナペスト（弱弱強格）の例 - The As**syr**ian came **down** like a **wolf** on the **fold**（[ジョージ・ゴードン・バイロン](https://ja.wikipedia.org/wiki/ジョージ・ゴードン・バイロン "wikilink")『ゼンナケリブの破滅』）
