**ヘスの法則**（へすのほうそく、）は、[スイス](../Page/スイス.md "wikilink")生まれの[ロシア](../Page/ロシア.md "wikilink")の[化学者](https://ja.wikipedia.org/wiki/化学者 "wikilink")[ジェルマン・アンリ・ヘス](../Page/ジェルマン・アンリ・ヘス.md "wikilink")が1840年に発表した[熱化学](../Page/熱化学.md "wikilink")の法則。総熱量不変の法則（）ともいう。ヘスの法則は[熱力学第一法則](https://ja.wikipedia.org/wiki/熱力学第一法則 "wikilink")の化学的言い換えであるが、熱力学第一法則の提唱以前に発見されたことは特筆すべき点である。

## 解説

ヘスは[硫酸](https://ja.wikipedia.org/wiki/硫酸 "wikilink")と[水](../Page/水.md "wikilink")を様々な割合で混合し、各々の組み合わせに対して[反応熱](../Page/反応熱.md "wikilink")を測定した。これより、[化学反応](https://ja.wikipedia.org/wiki/化学反応 "wikilink")の反応熱は反応前後の状態のみで決まり反応経路によらず一定であることを実験的に確認した。

現代の化学では、化学反応の生成熱（より厳密には[エンタルピー](../Page/エンタルピー.md "wikilink")変化）は反応経路にかかわらず一定と言い表すことができる。もともとエンタルピーは[状態量](../Page/状態量.md "wikilink")であるから、熱力学第一法則からヘスの法則は容易に誘導されるともいえる。（[エンタルピー](../Page/エンタルピー.md "wikilink")は後世の概念である事に留意。）

ヘスの法則の示すところは、化合物AがBに変化する反応熱は、化合物Aの生成熱と化合物Bの生成熱から決定できることを意味しており、同様にして少数の既知の反応エンタルピー変化を用いて、未知の反応のエンタルピー変化を導くことも可能である。

## 例

以下では炭素からメタンが生成する反応熱を既知の生成熱および燃焼熱から決定する。

|                                                                                                                            |                                            |
| -------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------ |
| \(\mbox{C(s)}+2\mbox{H}_2 \mbox{(g)} \longrightarrow \mbox{CH}_4 \mbox{(g)}\)                                              | \(\Delta H^0 \,\)                          |
| この反応の別経路の反応として、以下の反応を考える\[1\]。                                                                                             |                                            |
| \(\mbox{C(s)}+\mbox{O}_2\mbox{(g)} \longrightarrow\mbox{CO}_2\mbox{(g)}\)                                                  | \(\Delta H^0_1 = -393.51 ~ \mbox{kJ/mol}\) |
| \(\mbox{H}_2 \mbox{(g)} +{1 \over 2} \mbox{O}_2 \mbox{(g)} \longrightarrow\mbox{H}_2\mbox{O} \mbox{(l)}\)                  | \(\Delta H^0_2 = -285.83 ~ \mbox{kJ/mol}\) |
| \(\mbox{CH}_4 \mbox{(g)} +2\mbox{O}_2 \mbox{(g)} \longrightarrow\mbox{CO}_2 \mbox{(g)}  + 2\mbox{H}_2\mbox{O} \mbox{(l)}\) | \(\Delta H^0_3 = -890.36 ~ \mbox{kJ/mol}\) |

上記エンタルピーの関係は下記の式で表される。

\[\begin{align}
\Delta H^0 &= \Delta H^0_1 + 2 \Delta H^0_2 - \Delta H^0_3 \\
&= -393.51 ~ \mbox{kJ/mol} + 2 \times (-285.83 ~ \mbox{kJ/mol}) - (-890.36 ~ \mbox{kJ/mol}) \\
&= -74.81 ~ \mbox{kJ/mol}
\end{align}\]
