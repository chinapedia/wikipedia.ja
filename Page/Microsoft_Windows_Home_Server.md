**Microsoft Windows Home Server**（マイクロソフト ウィンドウズ ホーム サーバ、WHSとも）は、[マイクロソフト](../Page/マイクロソフト.md "wikilink")が開発した家庭向けメディア[サーバ](../Page/サーバ.md "wikilink")または[ホームサーバ](https://ja.wikipedia.org/wiki/ホームサーバ "wikilink")用[OSである](../Page/オペレーティングシステム.md "wikilink")。サーバ版Windowsとしては初の家庭向けバージョンでもある。

2007年のCES (Consumer Electronics Show) で[ビル・ゲイツ](../Page/ビル・ゲイツ.md "wikilink")により発表された。[Windows Server 2003 R2をベースにしている](https://ja.wikipedia.org/wiki/Microsoft_Windows_Server_2003#Windows_Server_2003_R2 "wikilink")。OS単体での販売はなく、[プリインストール](https://ja.wikipedia.org/wiki/プリインストール "wikilink")または[DSP](https://ja.wikipedia.org/wiki/DSP "wikilink")でのみ提供される。

英語版のみだったが、2008年8月30日に日本版が発売開始となった。

2013年1月8日にメインストリームサポートが終了し、延長サポートの対象外であるため、以降のサポートは提供されていない。

## 特徴

中核部分はWindows Server 2003 R2（厳密には同Small Business Server）相当であり、これを家庭向けに最適化したものである。WHSには[遠隔操作](https://ja.wikipedia.org/wiki/遠隔操作 "wikilink")（リモート）で管理するための専用[ツール](https://ja.wikipedia.org/wiki/ツール "wikilink")が付属しており、基本的にこれを利用して管理を行う。ただし、実際にWHSをリモート管理するためにはその専用ツールを[Windows XPまたは](../Page/Microsoft_Windows_XP.md "wikilink")[Windows Vistaが稼働している](https://ja.wikipedia.org/wiki/Microsoft_Windows_Vista "wikilink")[コンピュータ](../Page/コンピュータ.md "wikilink")へ[インストール](../Page/インストール.md "wikilink")しなければならないが、従来の[リモートデスクトップ](https://ja.wikipedia.org/wiki/リモートデスクトップ "wikilink")接続を利用した遠隔操作も可能である。

このような設計から、WHSがプリインストールされたPCでは[キーボードや](https://ja.wikipedia.org/wiki/キーボード_\(コンピュータ\) "wikilink")[マウス](../Page/マウス_\(コンピュータ\).md "wikilink")、[ディスプレイは基本的に不要であり](../Page/ディスプレイ_\(コンピュータ\).md "wikilink")、用意するものが[LANケーブル](https://ja.wikipedia.org/wiki/イーサネット#ケーブル "wikilink")1本のみで済むため、設置場所を選ばないのも大きなメリットである。

なお、家庭向けに最適化されていることもあり、サーバ版Windowsには必ずと言っていいほど搭載されている[Active Directoryといったビジネス用途向けの機能は省かれている](https://ja.wikipedia.org/wiki/Active_Directory "wikilink")。

## Power Pack

Windows Home ServerではService Packが提供されない代わりにPower Pack（パワーパック）が提供されている。このPower Packを通じて各種機能の追加・拡張が行えるようになっており、役割的にはService Packに近いものとなっている。

### Power Pack 1

2008年7月20日公開。[64ビット](https://ja.wikipedia.org/wiki/64ビット "wikilink")版[Windows Vistaが稼働しているコンピュータへのサポートが盛り込まれたほか](https://ja.wikipedia.org/wiki/Microsoft_Windows_Vista "wikilink")、消費電力の効率化やリモート アクセスの改善などが含まれている。また、初期版で見つかった不具合修正も行われている。

### Power Pack 2

2009年3月24日公開。Power Pack 1で見つかった不具合の修正に加え、Windows Media Center Connect機能の追加や[MP4](https://ja.wikipedia.org/wiki/MP4 "wikilink")形式の[ストリーミング](../Page/ストリーミング.md "wikilink")対応が行われるなど[マルチメディア](../Page/マルチメディア.md "wikilink")関係の強化が中心となっている。

### Power Pack 3

2009年7月17日より[ベータ版](../Page/ベータ版.md "wikilink")を公開\[1\]。当初は[Windows 7が発売される前に正式版が公開される予定だったが](../Page/Microsoft_Windows_7.md "wikilink")、不具合が見つかったとして延期となり\[2\]、2009年11月25日よりWindows Update経由で提供が開始された。

Windows 7への対応、日本語環境における[ユーザインタフェース](../Page/ユーザインタフェース.md "wikilink")の改善が行われた。

また、Home Server Connector Softwareの強化が行われ、Windows VistaおよびWindows 7に搭載されているWindows Media CenterへのHome Server項目と2つの機能の追加が行われた、一つは、Media centerで録画したテレビ番組の自動転送、保護されていない録画した[テレビ番組](../Page/テレビ番組.md "wikilink")の自動変換転送機能（国内ではアナログ放送、アナログ[キャプチャ](https://ja.wikipedia.org/wiki/キャプチャ "wikilink")のみ）を持った「TV アーカイブ」、今ひとつは、Home Serverのコンソールの状況などを表示する「コンソール ビュー」といった新機能が盛り込まれている。 なお、2009年12月20日時点でHome Server [コンソール](https://ja.wikipedia.org/wiki/コンソール "wikilink")において一部の[エラー](https://ja.wikipedia.org/wiki/エラー "wikilink")を伝える[ダイアログの未訳が見受けられる](../Page/ダイアログボックス.md "wikilink")。

## システム要件

WHSの[システム](../Page/システム.md "wikilink")[要件](https://ja.wikipedia.org/wiki/要件 "wikilink")は以下の通りとなっている。

  - 1GHz以上のプロセッサ（[Pentium 4または](../Page/Pentium_4.md "wikilink")[x64](https://ja.wikipedia.org/wiki/x64 "wikilink")を推奨）
  - [メモリ](../Page/Random_Access_Memory.md "wikilink") 512MB 以上（[ECC付を推奨](../Page/誤り検出訂正.md "wikilink")）
  - [ハードディスク](https://ja.wikipedia.org/wiki/ハードディスクドライブ "wikilink") 80GB 以上
  - [ディスク装置](../Page/ディスクドライブ.md "wikilink") [DVD](../Page/DVD.md "wikilink")-ROMドライブ
  - ディスプレイ [VGA](https://ja.wikipedia.org/wiki/VGA "wikilink")以上（[SVGA](https://ja.wikipedia.org/wiki/SVGA "wikilink")以上）

ただし、ハードディスクに関しては搭載容量のうち 20[GB](../Page/ギガバイト.md "wikilink") をシステム用ドライブ（Cドライブ）に、残りの容量をDドライブにそれぞれ割り当てる仕様になっているため、インストール時は[パーティション](https://ja.wikipedia.org/wiki/パーティション "wikilink")を割り当てることができないので注意が必要である。（言い換えれば、**インストール先のハードディスクに保存されている既存のデータはすべて削除される**ことを意味する。）
