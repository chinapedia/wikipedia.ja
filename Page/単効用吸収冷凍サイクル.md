**単効用吸収冷凍サイクル**（たんこうようきゅうしゅうれいとうサイクル）は、[液体](../Page/液体.md "wikilink")の[冷媒](../Page/冷媒.md "wikilink")を低い圧力に保った[蒸発器](https://ja.wikipedia.org/wiki/蒸発器 "wikilink")で低温で気化させ[気化熱](https://ja.wikipedia.org/wiki/気化熱 "wikilink")で[熱](https://ja.wikipedia.org/wiki/熱 "wikilink")を奪い取り、その[蒸気](https://ja.wikipedia.org/wiki/蒸気 "wikilink")を[吸収器](https://ja.wikipedia.org/wiki/吸収器 "wikilink")で吸収液に[吸収させ](https://ja.wikipedia.org/wiki/吸収_\(化学\) "wikilink")、濃度の低下した吸収液を[再生器](https://ja.wikipedia.org/wiki/再生器 "wikilink")で加熱し冷媒を蒸気として分離し、冷媒を[凝縮器](https://ja.wikipedia.org/wiki/凝縮器 "wikilink")で液化し蒸発器に・吸収液を冷却し吸収器に戻す、[冷凍機](../Page/冷凍機.md "wikilink")の[熱サイクルである](../Page/熱機関の理論サイクル.md "wikilink")。

## サイクルの流れ

1 - [蒸発器](https://ja.wikipedia.org/wiki/蒸発器 "wikilink")で液体の冷媒を低温・低圧で気化させ Q<sub>E</sub> 冷却する。 →1'

1',6 - [吸収器](https://ja.wikipedia.org/wiki/吸収器 "wikilink")で吸収液に冷媒蒸気を吸収させる。吸収液濃度 ξ<sub>1</sub> →2

2 - 吸収液ポンプで希吸収液を吸収液[熱交換器](../Page/熱交換器.md "wikilink")へ送り Q<sub>H</sub> 熱交換する →7

7 - [再生器](https://ja.wikipedia.org/wiki/再生器 "wikilink")で Q<sub>G</sub> の熱で希吸収液を加熱する。 →5

5 - 濃縮吸収液と冷媒蒸気に分離する。 →4,4'

4' - [凝縮器](https://ja.wikipedia.org/wiki/凝縮器 "wikilink")で Q<sub>C</sub> 冷却し冷媒を液化する。 →3

3 - 凝縮器から蒸発器へ[圧力](../Page/圧力.md "wikilink")差で液化した冷媒を移送する。 →1

4 - 再生器から吸収液[熱交換器](../Page/熱交換器.md "wikilink")へ[圧力](../Page/圧力.md "wikilink")差で濃吸収液を送り Q<sub>H</sub> 熱交換する。吸収液濃度 ξ<sub>2</sub> →8

8 - [吸収器](https://ja.wikipedia.org/wiki/吸収器 "wikilink")で吸収液を Q<sub>A</sub> 冷却する。 →6

### 理論成績係数・冷凍能力

熱交換器などの損失がない場合を考えると。

  -
    Q<sub>E</sub> + Q<sub>G</sub> = Q<sub>A</sub> + Q<sub>C</sub>
    Q<sub>E</sub> = D ( h<sub>1</sub>' - h<sub>3</sub> )
    Q<sub>C</sub> = D ( h<sub>4</sub>' - h<sub>3</sub> )
    Q<sub>G</sub> = ( a -1 ) D h<sub>4</sub> + D h<sub>4</sub>' - a D h<sub>7</sub>
    Q<sub>A</sub> = ( a -1 ) D h<sub>8</sub> + D h<sub>1</sub>' - a D h<sub>2</sub>
    Q<sub>H</sub> = a D ( h<sub>7</sub> - h<sub>2</sub> )
    (COP)<sub>R</sub> = Q<sub>E</sub> / Q<sub>G</sub> = ( h<sub>1</sub>' - h<sub>3</sub> ) / { h<sub>4</sub>' - h<sub>4</sub> + a ( h<sub>4</sub> - h<sub>7</sub> )}

<!-- end list -->

  -
    (COP)<sub>R</sub> : 冷凍サイクルの理論成績係数
    Q : 熱量
    h : 吸収液の[エンタルピー](../Page/エンタルピー.md "wikilink") 　h' : 吸収液と平衡する冷媒蒸気のエンタルピー
    a : 吸収液循環比、1kgの冷媒蒸気を再生器で発生させるために必要な希吸収液の循環量 a = ξ<sub>2</sub> / ( ξ<sub>2</sub> - ξ<sub>1</sub> )
    D : 再生器の発生冷媒蒸気量

## 単効用第一種吸収ヒートポンプ

このサイクルを利用し、低温排熱を高温熱源で汲み上げて加熱するものを**単効用第一種吸収ヒートポンプ**と呼ぶ。

  -
    (COP)<sub>H</sub> = (COP)<sub>R</sub> + 1

<!-- end list -->

  -
    (COP)<sub>H</sub> : ヒートポンプサイクルの理論成績係数
