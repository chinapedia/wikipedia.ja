**遅延記憶装置**（ちえんきおくそうち）とは、媒体が音波などを伝える際の遅れを利用し、信号を循環させ[記憶装置](../Page/記憶装置.md "wikilink")として使用するものである。種類によっては、遅延線記憶装置（delay line memory）という。技術的には[レーダー](../Page/レーダー.md "wikilink")で使う信号処理技術などから生まれたもので、黎明期の[コンピュータ](../Page/コンピュータ.md "wikilink")でよく使われた。

構成方法にもよるが、たとえば加算器に[直列加算器が使えるなど](https://ja.wikipedia.org/wiki/加算器#直列加算器 "wikilink")、部品数を抑えて多くの情報を扱えるコンピュータを作ることができる、という利点がある。これは安上がりという点だけではなく、初期のコンピュータの多くが苦労した信頼性という点でも重要である。

## 水銀遅延線

黎明期のコンピュータで主記憶装置としてよく使われた遅延記憶装置が、**水銀遅延線**（すいぎんちえんせん）である。アナログ信号処理デバイスとして使われていたものであるが、ディジタル計算機のための記憶装置としては、ENIACに関わった[ジョン・プレスパー・エッカート](../Page/ジョン・プレスパー・エッカート.md "wikilink")による（ENIAC自体では、稼働初期の時点では本格的には使われていない）。[水銀](../Page/水銀.md "wikilink")の中の[超音波](../Page/超音波.md "wikilink")を利用する。英語 delay line memory から（delay memory とはしない）「遅延**線**」の語があるが、線というより管であることから**水銀遅延管**（すいぎんちえんかん）とも呼ばれる。英語でも mercury delay tube としている例が見られる。（真空管と勘違いしないように\[1\]）tank の語を使うことも見られる。

[<File:Delay> line memory fr.svg](https://ja.wikipedia.org/wiki/File:Delay_line_memory_fr.svg "fig:File:Delay line memory fr.svg")

水銀を詰めた管の両端に[圧電素子](../Page/圧電素子.md "wikilink")をとり付けた構造をしている。片方の素子に信号を入力し、[圧電効果](../Page/圧電効果.md "wikilink")（逆圧電効果）による[振動](../Page/振動.md "wikilink")で超音波を発生させる。超音波は水銀を媒体として管の中を伝わり、反対側の圧電素子を振動させる。振動させられた圧電素子は[圧電効果](../Page/圧電効果.md "wikilink")により電圧を誘起するため、ここから入力信号と同様の波形を持つ信号が取り出せる。これを[増幅](../Page/増幅.md "wikilink")して再び入力側に戻すと、信号が循環して、記憶装置として使うことができる。この操作について当時の用語では、英語では regeneration、日本語では訳して「再生」などと表現されている。

水銀遅延線は初期のコンピュータに用いられ、[EDSAC](../Page/EDSAC.md "wikilink")や[EDVAC](../Page/EDVAC.md "wikilink")、[UNIVAC Iで採用された](../Page/UNIVAC_I.md "wikilink")。また、日本初の電子式コンピュータ、[FUJIC](../Page/FUJIC.md "wikilink")にも水銀遅延線が使われている。

（なお、設計としては、片方を開放端か固定端として信号が反射するようにし、入力と取り出しを同じ側でおこなう、というものもありうる）

## その他の材料

水銀以外に、記憶装置として使われた媒体としては、[ETL Mark IIIなどで使われたガラス](../Page/ETL_Mark_III.md "wikilink")\[2\]、磁性体（金属）ワイヤを使った磁歪遅延線\[3\]\[4\]、ETL Mark IVでタイミング調整用に使われ、[HITAC](../Page/HITAC.md "wikilink") 5020で（主記憶ではなく）レジスタに使われた、同軸ケーブル内を伝わる高周波信号の遅れを利用する電磁遅延線がある。固体遅延線や電磁遅延線は記憶以外に、アナログ時代にはフィルタの部品としてよく使われ、[カラーテレビ](../Page/カラーテレビ.md "wikilink")などのために大量生産された電気部品でもある。HITAC 5020のそれには製造元である日立のテレビのものが使われた、という例もある（この信号処理については[NTSC\#ライン相関を利用したクシ形フィルタ](https://ja.wikipedia.org/wiki/NTSC#ライン相関を利用したクシ形フィルタ "wikilink")を参照。ディジタル信号処理では逆に、記憶装置を使って信号の遅延が実現されている）。

また、循環型で大容量のレジスタを作るために、[磁気ドラムのトラックの一部を遅延線のように使うという手法があり](../Page/磁気ドラムメモリ.md "wikilink")、日本では[マルス1で使われたという例がある](https://ja.wikipedia.org/wiki/マルス_\(システム\)#マルス1 "wikilink")\[5\]。

変わったエピソードとしては、[モーリス・ウィルクス](../Page/モーリス・ウィルクス.md "wikilink")によれば、水銀より安い媒体を1947年に検討していた際、[アラン・チューリング](../Page/アラン・チューリング.md "wikilink")が[ジンを提案した](../Page/ジン_\(蒸留酒\).md "wikilink")、というものがある。アルコールと水が、室温において温度係数がゼロになる割合で含まれているから、だという\[6\]。遅延記憶装置において、温度変化により動作速度がズレることは問題であり、恒温槽を必要とした、といった話があるが、[FUJIC](../Page/FUJIC.md "wikilink")では逆転の発想で、水銀遅延線を動作させるクロックの速度を温度に合わて調節する（計算機本体との同期は、記憶装置側をマスタークロックとする）ことにより解決している\[7\]。

波の伝搬といったような物理現象を利用しているわけではないが、使い方としては[シフトレジスタ](../Page/シフトレジスタ.md "wikilink")も少し似ている。また近年の研究としては、NHK放送技術研究所による「微小磁区記録デバイス」\[8\]は、磁性細線中を移動する磁区を利用するもので、遅延記憶装置に似ている。
