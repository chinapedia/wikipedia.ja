『**すばらしい味の世界**』（すばらしいあじのせかい）は、1977年4月1日から1992年3月29日まで[テレビ東京](https://ja.wikipedia.org/wiki/テレビ東京 "wikilink")（1981年9月までは東京12チャンネル）で放送されていた[料理番組](https://ja.wikipedia.org/wiki/料理番組 "wikilink")である。[サントリー](../Page/サントリー.md "wikilink")の一社提供。

## 概要

[タキシード](https://ja.wikipedia.org/wiki/タキシード "wikilink")を着た[柳生博](../Page/柳生博.md "wikilink")が案内役となり、毎回[東京都](https://ja.wikipedia.org/wiki/東京都 "wikilink")内にある高級[レストラン](../Page/レストラン.md "wikilink")・[料理店](https://ja.wikipedia.org/wiki/料理店 "wikilink")のメニューを1つ紹介していた[ミニ番組](https://ja.wikipedia.org/wiki/ミニ番組 "wikilink")。柳生のほか、[江守徹](https://ja.wikipedia.org/wiki/江守徹 "wikilink")、[西田敏行](../Page/西田敏行.md "wikilink")、[武田鉄矢](../Page/武田鉄矢.md "wikilink")、[川津祐介](https://ja.wikipedia.org/wiki/川津祐介 "wikilink")も案内役を務めた。

番組中に流れるVTRは料理完成までの過程を追った内容で、映すのは基本的に料理人の手元のみであった。過剰な[演出](../Page/演出.md "wikilink")も[ナレーション](https://ja.wikipedia.org/wiki/ナレーション "wikilink")も無ければ[BGMも無く](https://ja.wikipedia.org/wiki/バックグラウンドミュージック "wikilink")、調理されていく素材の発する音のみが流された。そして料理の完成後には、その回の案内役が料理に対する感想を述べて終了するという至ってシンプルな構成であった。

[ゲオルク・フィリップ・テレマン](https://ja.wikipedia.org/wiki/ゲオルク・フィリップ・テレマン "wikilink")作曲の[ターフェルムジーク](https://ja.wikipedia.org/wiki/ターフェルムジーク "wikilink")（食卓の音楽）第2集第2曲「[ブロックフレーテ](../Page/リコーダー.md "wikilink")、2本の[フルート](../Page/フルート.md "wikilink")と[通奏低音](../Page/通奏低音.md "wikilink")のための[四重奏曲](../Page/弦楽四重奏曲.md "wikilink") [ニ短調](https://ja.wikipedia.org/wiki/ニ短調 "wikilink")」（[アルヒーフ版](https://ja.wikipedia.org/wiki/ドイツ・グラモフォン#保有レーベル "wikilink")）を番組のオープニングテーマにしていた。

## 放送時間

いずれも[日本標準時](../Page/日本標準時.md "wikilink")。

本番組は平日にも土曜・日曜にも放送されていたが、21時台と22時台に『[木曜洋画劇場](https://ja.wikipedia.org/wiki/木曜洋画劇場 "wikilink")』が編成されていた木曜のみ一貫して放送がなかった。また、放送を続けるにつれて次第に放送曜日が減っていき、最終的には土日のみの放送となった。

| 編成期間       | 放送日時       |
| ---------- | ---------- |
| 開始時期       | 終了時期       |
| 1977.04.01 | 1978.10.01 |
| 1978.10.02 | 1980.08.03 |
| 1980.08.04 | 1980.10.05 |
| 1980.10.07 | 1981.10.04 |
| 1981.10.06 | 1982.04.04 |
| 1982.04.09 | 1985.07.28 |
| 1985.08.03 | 1992.03.29 |
