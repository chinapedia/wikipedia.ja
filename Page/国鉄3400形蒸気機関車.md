**3400形**は、かつて[日本国有鉄道](../Page/日本国有鉄道.md "wikilink")の前身である[鉄道院・鉄道省に在籍した](https://ja.wikipedia.org/wiki/鉄道省 "wikilink")[タンク式](../Page/タンク機関車.md "wikilink")[蒸気機関車](../Page/蒸気機関車.md "wikilink")である。

## 概要

[アメリカ合衆国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")の[ピッツバーグ・ロコモティブ・アンド・カー・ワークス](https://ja.wikipedia.org/wiki/ピッツバーグ・ロコモティブ・アンド・カー・ワークス "wikilink")から複数の[私設鉄道に導入された](../Page/私鉄.md "wikilink")、40tクラスの[車軸配置](https://ja.wikipedia.org/wiki/車軸配置 "wikilink")[2-6-2(1C1)の単式](https://ja.wikipedia.org/wiki/車輪配置_2-6-2 "wikilink")2気筒、[飽和式タンク機関車である](https://ja.wikipedia.org/wiki/水蒸気#飽和蒸気と過熱蒸気 "wikilink")。[ボールドウィン・ロコモティブ・ワークス](../Page/ボールドウィン・ロコモティブ・ワークス.md "wikilink")製の[3300系よりやや大きく](../Page/国鉄3300形蒸気機関車.md "wikilink")、形態もピッツバーグ流に洗練されていた。[車軸配置2-4-2(1B1)の](https://ja.wikipedia.org/wiki/車輪配置_2-4-2 "wikilink")[210形とは系列設計で](../Page/国鉄210形蒸気機関車.md "wikilink")、デザインラインが類似している。

本形式を最初に導入したのは、[1896年](../Page/1896年.md "wikilink")（明治29年）の[南海鉄道と](../Page/南海電気鉄道.md "wikilink")[播但鉄道](https://ja.wikipedia.org/wiki/播但鉄道 "wikilink")で、翌[1897年](../Page/1897年.md "wikilink")（明治30年）には[豊州鉄道](https://ja.wikipedia.org/wiki/豊州鉄道 "wikilink")（初代）、[1907年](../Page/1907年.md "wikilink")（明治40年）には[横浜鉄道](https://ja.wikipedia.org/wiki/横浜鉄道 "wikilink")が導入し、計23両が日本に輸入されている。

これらのうち[鉄道国有法](../Page/鉄道国有法.md "wikilink")により国有鉄道籍となったのは、播但鉄道から[山陽鉄道](https://ja.wikipedia.org/wiki/山陽鉄道 "wikilink")に譲渡された1両、豊州鉄道から[九州鉄道](../Page/九州鉄道.md "wikilink")に譲渡された12両で、その後の買収により横浜鉄道の5両と、[戦時買収によって国有化された](../Page/戦時買収私鉄.md "wikilink")[西日本鉄道](https://ja.wikipedia.org/wiki/西日本鉄道 "wikilink")（[糟屋線](https://ja.wikipedia.org/wiki/香椎線 "wikilink")）の2両の合わせて20両であるが、西日本鉄道買収の際の買収機のうち、本来[3350形に編入されるべき](https://ja.wikipedia.org/wiki/国鉄3350形蒸気機関車 "wikilink")[ブルックス・ロコモティブ・ワークス](https://ja.wikipedia.org/wiki/ブルックス・ロコモティブ・ワークス "wikilink")製の1両が本形式に編入されており、3400形を称した機関車は、都合21両ということになる。

## 南海鉄道

南海鉄道へは1896年に3両（[製造番号](../Page/製造番号.md "wikilink")1688 - 1690）が入線したが、[後述する播但鉄道の2両](https://ja.wikipedia.org/wiki/#播但鉄道 "wikilink")（製造番号1691, 1692）も引き受けて、**1形**と称した。製造番号と南海鉄道での番号の対照は、次のとおりである。

  - 製造番号1688 → 南海鉄道5
  - 製造番号1689 → 南海鉄道1
  - 製造番号1690 → 南海鉄道2
  - 製造番号1691 → 南海鉄道3
  - 製造番号1692 → 南海鉄道4

これらは、1907年の[電化後も](../Page/鉄道の電化.md "wikilink")[貨物列車](https://ja.wikipedia.org/wiki/貨物列車 "wikilink")用として残留したが、[1917年](../Page/1917年.md "wikilink")（大正6年）に3が、翌[1918年](../Page/1918年.md "wikilink")（大正7年）に4が播州鉄道を介して博多湾鉄道（後の[博多湾鉄道汽船](../Page/博多湾鉄道汽船.md "wikilink")）に譲渡され、[1944年](../Page/1944年.md "wikilink")（昭和19年）の戦時買収により国有鉄道籍となったが、南海鉄道に残った3両（1, 2, 5）は、国有鉄道との接点のないまま、[1923年](../Page/1923年.md "wikilink")（大正12年）の[電気機関車](../Page/電気機関車.md "wikilink")全面的採用にともなって用途を失い、[1927年](../Page/1927年.md "wikilink")（昭和2年）に[廃車解体された](../Page/廃車_\(鉄道\).md "wikilink")。

博多湾鉄道から西日本鉄道に編入された2両については、戦時買収により本形式に編入されたが、3が3419、4が廃車による欠番を埋める形で3410（2代目）となった。3410が埋番2代目となったのは、横浜鉄道引継ぎ車の続番に3両を編入すると、次形式（[3420形](https://ja.wikipedia.org/wiki/国鉄3420形蒸気機関車 "wikilink")）と番号が重複してしまうためと推定されている。

3418は、博多湾鉄道汽船の1（2代目）であったが、前述のように[1896年](../Page/1896年.md "wikilink")（明治29年）ブルックス社製の別物である。元は、[南和鉄道](https://ja.wikipedia.org/wiki/南和鉄道 "wikilink")が発注したもの（製造番号2710, 2711 → **4, 5**）であったが、[1899年](../Page/1899年.md "wikilink")（明治32年）、[1898年](../Page/1898年.md "wikilink")（明治31年）に南海鉄道に譲渡され、同社の**3形**（**7, 8**）となり、そのうちの7が[1925年](../Page/1925年.md "wikilink")（大正14年）に博多湾鉄道へ譲渡され、それが国有化されたわけである。南海鉄道に残った8は、1928年の全面電気動力化にともない、廃車解体された。

## 播但鉄道

播但鉄道へは1896年に2両（製造番号1691, 1692）が、1897年に1両（製造番号1693）が入線したが、前述のように2両（製造番号1691, 1692）が南海鉄道に譲渡されたため、製造番号1693のみが播但鉄道に入り、同社の**L3形**（**6**）となった。1903年には、播但鉄道が山陽鉄道に事業を譲渡したため、山陽鉄道の**24形**(**116**)となった。1907年に山陽鉄道が国有化されると、1909年の鉄道院の車両形式称号規程により3400形に類別され、**3412**となった。

## 豊州鉄道

豊州鉄道へは1897年に4両（製造番号1705 - 1708 → **9 - 12**)、1898年に4両（製造番号1787 - 1790 → **13 - 16**）、1899年に4両（製造番号1970 - 1973 → **17 - 20**）の計12両が入線した。これらは、1901年、事業譲渡により九州鉄道に編入されて同社の**174形**（**174 - 185**）とされた後、国有化され、鉄道院の車両形式称号規程では3400形（**3400 - 3411**）となった。

## 横浜鉄道

横浜鉄道へは1907年に5両（製造番号44190 - 44194\[1\] ）が導入され、 **A形**（**1 - 5**）と称した。その後1911年に、鉄道院の横浜鉄道借り上げにより、全車が国有鉄道の番号体系に組み込まれ、**3413 - 3417**に付番された。横浜鉄道の正式買収は1917年である。

## 国有化後の状況と譲渡

本形式は、客貨両用の機関車として重宝され、旧九州鉄道のものは北九州地区で、旧横浜鉄道のものは沼津や敦賀に配置されていた。これらは、鉄道院制式の標準形式増備とともに次第に[入換用に転じていった](https://ja.wikipedia.org/wiki/入換_\(鉄道\) "wikilink")。

廃車は、博多湾鉄道汽船 → 西日本鉄道糟屋線の国有化に先立つ[1934年](../Page/1934年.md "wikilink")（昭和9年）から始まり、[1949年](../Page/1949年.md "wikilink")（昭和24年）までに全車が除籍された。そのうち12両は民間に払い下げられたが、博多湾鉄道汽船に譲渡された4両と[小倉鉄道](https://ja.wikipedia.org/wiki/小倉鉄道 "wikilink")に譲渡された1両が、戦時買収により再び国有鉄道籍となった。その状況は次のとおりである。

  - 3400 → 博多湾鉄道汽船17（1935年） → 西日本鉄道 → 国有鉄道3400（1944年） → 1948年廃車
  - 3401 → 小倉鉄道3401（1939年） → 国有鉄道3401（1943年） → [大分交通](../Page/大分交通.md "wikilink")（[耶馬渓線](https://ja.wikipedia.org/wiki/大分交通耶馬渓線 "wikilink")）3401\[2\]（1945年） →1955年廃車
  - 3402 → 博多湾鉄道汽船15（1934年） → 西日本鉄道 → 国有鉄道3402（1944年） → 1948年廃車
  - 3403 → [三菱鉱業](https://ja.wikipedia.org/wiki/三菱鉱業 "wikilink")鯰田鉱業所（1936年）
  - 3405 → 博多湾鉄道汽船16（1934年） → 西日本鉄道 → 国有鉄道3405（1944年） → [熊延鉄道](https://ja.wikipedia.org/wiki/熊延鉄道 "wikilink")11（1948年）
  - 3406 → [北九州鉄道](../Page/北九州鉄道.md "wikilink")3406（1934年） → 国有鉄道3406（1937年） → 1949年廃車
  - 3410 → [浅野セメント](https://ja.wikipedia.org/wiki/浅野セメント "wikilink")（1939年）
  - 3411 → 博多湾鉄道汽船18（1935年） → 西日本鉄道 → 国有鉄道3411（1944年） → 1948年廃車
  - 3415 → [熊延鉄道](https://ja.wikipedia.org/wiki/熊延鉄道 "wikilink")12（1949年）
  - 3416 → 大分交通（耶馬渓線）3416\[3\]（1945年） →1955年廃車
  - 3417 → 西日本鉄道3417（1942年） → 1950年廃車
  - 3410\[II\] → 熊延鉄道10（1948年）
