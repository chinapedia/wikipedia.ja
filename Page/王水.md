[Aqua_regia_in_Davenport_Laboratories.jpg](https://ja.wikipedia.org/wiki/File:Aqua_regia_in_Davenport_Laboratories.jpg "fig:Aqua_regia_in_Davenport_Laboratories.jpg") [Aqua_regia_in_NMR_tubes.jpg](https://ja.wikipedia.org/wiki/File:Aqua_regia_in_NMR_tubes.jpg "fig:Aqua_regia_in_NMR_tubes.jpg") **王水**（おうすい、aqua regia）は、濃[塩酸](https://ja.wikipedia.org/wiki/塩酸 "wikilink")と濃[硝酸](../Page/硝酸.md "wikilink")を3:1の体積比で混合してできる橙赤色の[液体](../Page/液体.md "wikilink")である。[CAS登録番号](../Page/CAS登録番号.md "wikilink")は8007-56-5。全ての金属ではないが、[金](../Page/金.md "wikilink")や[白金](../Page/白金.md "wikilink")といった[貴金属](../Page/貴金属.md "wikilink")を始めとして多くの金属を溶解できることから、[錬金術師](https://ja.wikipedia.org/wiki/錬金術師 "wikilink")によってこのように命名された。

濃塩酸と濃硝酸を1:3の体積比で混合したものは「逆王水」と呼称され、[分析化学](../Page/分析化学.md "wikilink")において金属の溶解などに用いる。[塩化アンモニウム](../Page/塩化アンモニウム.md "wikilink")と[硝酸アンモニウム](../Page/硝酸アンモニウム.md "wikilink")とを目分量1:3の混合比としたものは「固体王水」と呼称され、粉末試験法においてほとんどの[金属](../Page/金属.md "wikilink")[酸化物](https://ja.wikipedia.org/wiki/酸化物 "wikilink")を混合して加熱することにより、[塩化することができる](../Page/塩化物.md "wikilink")。

## 性質

[酸化](../Page/酸化.md "wikilink")力が非常に強く、王水との反応で生じた金属[化合物](../Page/化合物.md "wikilink")はその金属の最高[酸化数](../Page/酸化数.md "wikilink")を示す。また、通常の[酸](../Page/酸.md "wikilink")には溶けない[金](../Page/金.md "wikilink")や[白金](../Page/白金.md "wikilink")などの[貴金属](../Page/貴金属.md "wikilink")も溶解できる。ただし[タンタル](../Page/タンタル.md "wikilink")、[イリジウム](../Page/イリジウム.md "wikilink")は酸に対しての耐性が極めて高いため、溶解できない（イリジウムは粉末にすればわずかに溶ける）。また、[銀](https://ja.wikipedia.org/wiki/銀 "wikilink")もほとんど溶けない（王水と反応してできる[塩化銀](https://ja.wikipedia.org/wiki/塩化銀 "wikilink") (AgCl) が表面に膜を形成し、反応の進行を妨げる）。[ルテニウム](../Page/ルテニウム.md "wikilink")、[ロジウム](../Page/ロジウム.md "wikilink")、[オスミウム](../Page/オスミウム.md "wikilink")とは反応するが、反応速度は低く、徐々に侵される。

[腐食性](https://ja.wikipedia.org/wiki/腐食性 "wikilink")が非常に強いため、人体にとっては極めて有害である。日本では[毒物及び劇物取締法](https://ja.wikipedia.org/wiki/毒物及び劇物取締法 "wikilink")により、10 %を超える[塩化水素](../Page/塩化水素.md "wikilink")の製剤として[劇物](https://ja.wikipedia.org/wiki/劇物 "wikilink")となる。

## 利用

王水は、主に最高品質（99.999%）の金の精製法であるに使われる[電解質](../Page/電解質.md "wikilink")である[塩化金酸の製造に使用される](https://ja.wikipedia.org/wiki/テトラクロリド金\(III\)酸 "wikilink")。

また、多くの金属を溶解できることから、[分析化学](../Page/分析化学.md "wikilink")での試料調製・貴金属塩の製造・[ガラス器具](../Page/ガラス器具.md "wikilink")の精密洗浄などに用いられる。そのほか、電子部品や装飾品の加工くずなどから貴金属を回収する時にも使われる\[1\]。

成分間の分解を引き起こす反応により、王水はすぐにその効果を失うため、通常は使用直前に調製される。

地域の規制により異なる場合もあるが、王水は十分に中和することで下水に流すことができる。溶存金属による汚染がある場合は、中和された溶液を廃棄するために収集する必要がある\[2\]\[3\]。

## 反応式

### 生成と分解

濃硝酸と濃塩酸を混合すると、以下の反応により[塩化ニトロシル](../Page/塩化ニトロシル.md "wikilink")(NOCl)と[塩素](../Page/塩素.md "wikilink")と[水](../Page/水.md "wikilink")が生成される。

  -
    <ce>{HNO3(aq)} + 3HCl(aq) -\> {NOCl(g)} \\uparrow + {Cl2(g)} \\uparrow + 2H2O(l)</ce>

王水の成分は揮発性であり、調製してすぐの王水は無色だが、数秒で橙色に変わる。揮発性の成分が全て王水から揮発すると、王水としての性質はなくなる。また、塩化ニトロシルは、[一酸化窒素](../Page/一酸化窒素.md "wikilink")と塩素に分解する可能性がある。

  -
    <ce>{2NOCl(g)} -\> {2NO(g)} \\uparrow + {Cl2(g)} \\uparrow</ce>

この解離は平衡制約であり、王水の煙には塩化ニトロシルと塩素のほかに一酸化窒素も含まれている。一酸化窒素は大気中の[酸素](../Page/酸素.md "wikilink")と容易に反応するため、生成される気体には[二酸化窒素](../Page/二酸化窒素.md "wikilink")(NO<sub>2</sub>)も含まれる。

  -
    <ce>{2NO(g)} + {O2(g)} -\> {2NO2(g)} \\uparrow</ce>

### 金の溶解

[Golddust.jpg](https://ja.wikipedia.org/wiki/File:Golddust.jpg "fig:Golddust.jpg") 王水は金を溶解するが、王水を構成するどの酸も単独では金を溶解できない。これは、それぞれの酸が異なる役割を実行するためである。硝酸は強力な酸化剤であり、検出不可能なわずかな量の金を溶解し、金イオン()を形成する。塩酸は塩化物イオン()を供給する。塩化物イオンは金イオンと反応して、溶液中にテトラクロリド金(III)イオンを生成する。塩酸との反応は、塩化金イオン()の生成を促進する平衡反応である。これにより、溶液から金イオンが除去され、金のさらなる酸化が起こる。金は溶解して[塩化金酸になる](https://ja.wikipedia.org/wiki/テトラクロリド金\(III\)酸 "wikilink")。さらに、金は王水に存在する塩素によって溶解する可能性がある。反応式は次の通りである。

  -
    <ce>{Au} + {3HNO3} + 4HCl \<=\> {\[AuCl_4\]^-} + \\ {\[3NO2\]} + {\[H3O\]^+} + 2H2O</ce>

<!-- end list -->

  -
    または
    <ce>{Au} + {HNO3} + 4HCl \<=\> {\[AuCl_4\]^-} + \\ {\[NO\]} + {\[H3O\]^+} + H2O</ce>

王水が金のみを含む場合、残った王水を煮沸し、塩酸で繰り返し加熱して残留硝酸を除去することにより、固体のテトラクロロ金酸を調製できる。ここから金のみを得る場合、[二酸化硫黄](../Page/二酸化硫黄.md "wikilink")、[ヒドラジン](../Page/ヒドラジン.md "wikilink")、[シュウ酸](../Page/シュウ酸.md "wikilink")などで選択的に還元することができる\[4\]。二酸化硫黄による金の還元の式は次の通りである。

  -
    <ce>{2AuCl4^-(aq)} + {3SO2(g)} + 6H2O (l) -\> {2Au (s)} + {12H+ (aq)} + {3SO4^{2-} (aq)} + 8Cl^- (aq)</ce>

### 白金の溶解

[白金](../Page/白金.md "wikilink")についても金と同様の反応式を書くことができる。金と同様に、酸化反応は、窒素酸化物として一酸化窒素または二酸化窒素のいずれかを使用して記述できる。

  -
    <ce>{Pt(s)} + {4NO3^-(aq)} + 8H+ (aq) -\> {Pt^{4+} (aq)} + {4NO2(g)} + 4H2O(l)</ce>
    <ce>{3Pt(s)} + {4NO3^-(aq)} + 16H+ (aq) -\> {3Pt^{4+}(aq)} + {4NO(g)} + 8H2O(l)</ce>

酸化された白金イオンは、塩化物イオンと反応して、塩化白金酸イオンになる。

  -
    <ce>{Pt^4+(aq)} + {6Cl^-(aq)} -\> PtCl6^{2-}(aq)</ce>

実験的証拠により、白金と王水との反応がかなり複雑であることが明らかである。最初の反応では、テトラクロリド白金(II)酸(H<sub>2</sub>PtCl<sub>4</sub>)と塩化ニトロソ白金((NO)<sub>2</sub>PtCl<sub>4</sub>)の混合物が生成される。塩化ニトロソ白金は固体生成物である。白金を完全に溶解したい場合は、濃塩酸で残留固形物を繰り返し抽出する必要がある。

  -
    <ce>2Pt(s) + 2HNO3(aq) + 8HCl(aq) -\> (NO)2PtCl4(s) + H2PtCl4(aq) + 4H2O(l)</ce>

and

  -
    <ce>(NO)2PtCl4(s) + 2HCl(aq) \<=\> H2PtCl4(aq) + 2NOCl(g)</ce>

加熱中に溶液を塩素で飽和させることにより、テトラクロリド白金(II)酸を[ヘキサクロリド白金(IV)酸](https://ja.wikipedia.org/wiki/ヘキサクロリド白金\(IV\)酸 "wikilink")に酸化できる。

  -
    <ce>H2PtCl4(aq) + Cl2(g) -\> H2PtCl6(aq)</ce>

王水での白金固体の溶解は、最も密度の高い金属である[イリジウム](../Page/イリジウム.md "wikilink")と[オスミウム](../Page/オスミウム.md "wikilink")の発見方法だった。どちらも白金鉱石の中から見つかり、酸によって溶解せず、容器の底に集まる。

### 溶解した白金の沈殿

[白金族元素](https://ja.wikipedia.org/wiki/白金族元素 "wikilink")を王水に溶解した後、[塩化鉄(II)](../Page/塩化鉄\(II\).md "wikilink")での処理により白金族元素が沈殿する。ろ液中の白金（ヘキサクロリド白金(IV)酸）に[塩化アンモニウム](../Page/塩化アンモニウム.md "wikilink")を加えることで、[ヘキサクロリド白金(IV)酸アンモニウム](https://ja.wikipedia.org/wiki/ヘキサクロリド白金\(IV\)酸アンモニウム "wikilink")が生成される。このアンモニウム塩は非常に不溶性であるため、ろ過することができ、強力な加熱により金属の白金にすることができる\[5\]。

  -
    <ce>3(NH4)2PtCl6 -\> 3Pt + 2N2 + 2NH4Cl + 16HCl</ce>

未沈殿のヘキサクロリド白金(IV)酸塩は[亜鉛](../Page/亜鉛.md "wikilink")で還元できる。この方法は、実験室残留物からの白金の小規模回収に適している\[6\]。

### スズとの反応

王水は[スズ](../Page/スズ.md "wikilink")と反応して、[塩化スズ(IV)](https://ja.wikipedia.org/wiki/塩化スズ\(IV\) "wikilink")を生成する。

  -
    <ce>4HCl + 2HNO3 + Sn -\> SnCl4 + NO2 + NO + 3H2O</ce>

### その他の物質との反応

王水は[黄鉄鉱](../Page/黄鉄鉱.md "wikilink")と反応して窒素酸化物を生成する。

  -
    <ce>FeS2 + 5HNO3 + 3HCl -\> FeCl3 + 2H2SO4 + 5NO + 2H2O</ce>

## 歴史

[Musaeum_Hermeticum_1678_p_398_III._Clavis_AQ27.tif](https://ja.wikipedia.org/wiki/File:Musaeum_Hermeticum_1678_p_398_III._Clavis_AQ27.tif "fig:Musaeum_Hermeticum_1678_p_398_III._Clavis_AQ27.tif") 王水は、[アル・ラーズィー](https://ja.wikipedia.org/wiki/アル・ラーズィー "wikilink")（854年 - 925年）などのによって最初に言及され\[7\]、その後、1300年ごろのによって言及された\[8\]。

1600年頃のによる『』の第3の鍵には、手前にドラゴンが、後ろに雄鶏を食べる狐が描かれている。雄鶏は金を象徴し（日の出との関係と太陽と金の関係から）、狐は王水を表す。雄鶏を食べる狐に別の雄鶏が食いついているのは、金の精製のために溶解・加熱・再溶解を繰り返すことを表す。その後、金は塩化金の形で結晶化するが、その赤い結晶はドラゴンの血と呼ばれており、これが手前のドラゴンで表されている。この反応が化学文献で再び報告されたのは1890年である\[9\]。

1789年、[アントワーヌ・ラヴォアジェ](https://ja.wikipedia.org/wiki/アントワーヌ・ラヴォアジェ "wikilink")がこの物質を"aqua regia"（王の水）と名付けた\[10\]。日本語の「王水」はこの直訳である。

[第二次世界大戦](https://ja.wikipedia.org/wiki/第二次世界大戦 "wikilink")で[ドイツがデンマークに侵攻したとき](../Page/ヴェーザー演習作戦.md "wikilink")、ハンガリーの化学者[ゲオルク・ド・ヘヴェシー](../Page/ゲオルク・ド・ヘヴェシー.md "wikilink")は、[マックス・フォン・ラウエ](../Page/マックス・フォン・ラウエ.md "wikilink")と[ジェイムス・フランク](../Page/ジェイムス・フランク.md "wikilink")から預かっていた[ノーベル物理学賞](../Page/ノーベル物理学賞.md "wikilink")の金製のメダルを、ドイツ軍に奪われるのを防ぐために王水で溶かした。ドイツ政府は、1935年に投獄された平和活動家[カール・フォン・オシエツキー](../Page/カール・フォン・オシエツキー.md "wikilink")が[ノーベル平和賞](../Page/ノーベル平和賞.md "wikilink")を受賞した後、ドイツ人がノーベル賞を受賞することを禁止していた。ヘヴェシーは、金メダルを溶かした王水を入れた容器を[ニールス・ボーア研究所](https://ja.wikipedia.org/wiki/ニールス・ボーア研究所 "wikilink")の棚にしまったが、ナチスには気づかれなかった。戦後、ヘヴェシーは王水から金を復元した。金は[スウェーデン王立科学アカデミー](../Page/スウェーデン王立科学アカデミー.md "wikilink")と[ノーベル財団](../Page/ノーベル財団.md "wikilink")に返還され、金メダルが作り直されて、ラウエとフランクに再び贈られた\[11\]\[12\]。
