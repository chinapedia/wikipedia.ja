[Gregorian_chant.gif](https://ja.wikipedia.org/wiki/File:Gregorian_chant.gif "fig:Gregorian_chant.gif")のネウマ譜\]\] **ネウマ譜**（ネウマふ、）とは、[楽曲](https://ja.wikipedia.org/wiki/楽曲 "wikilink")の[楽譜](../Page/楽譜.md "wikilink")を「ネウマ」と呼ばれる記号を用いて書き表す[記譜法](../Page/記譜法.md "wikilink")のことである。

## 初期の歴史

[キリスト教](../Page/キリスト教.md "wikilink")の聖歌（[チャント](https://ja.wikipedia.org/wiki/チャント "wikilink")）は、[教会の初期から歌われてきたと見られるが](../Page/教会_\(キリスト教\).md "wikilink")、何世紀もの間[口伝](https://ja.wikipedia.org/wiki/口伝 "wikilink")によって伝えられてきたのみであった。[9世紀](../Page/9世紀.md "wikilink")頃に現れ、音高を明示しないネウマ、音高ネウマ（ダイアステマ記譜法）といった初期の記譜法が発展し、[11世紀](../Page/11世紀.md "wikilink")になると譜線ネウマが見られ、4本の譜線が用いられるようになった。この頃、線譜表には、C音とF音が付けられるようになり、それが[ハ音記号](https://ja.wikipedia.org/wiki/ハ音記号 "wikilink")と[ヘ音記号](https://ja.wikipedia.org/wiki/ヘ音記号 "wikilink")となる（[ト音記号](https://ja.wikipedia.org/wiki/ト音記号 "wikilink")もG音の記号である）。ネウマ（）とは[ギリシャ語](https://ja.wikipedia.org/wiki/ギリシャ語 "wikilink")で「合図、身振り」という意味であり、[合唱](../Page/合唱.md "wikilink")を指揮する際の手の合図である。

ネウマによる記譜を含むもので、2016年7月現在知られているもので最古のものは、[アラム語](../Page/アラム語.md "wikilink")起源で[聖書](../Page/聖書.md "wikilink")の準旋律的な朗読の抑揚を記録するために用いられた。これは[クルアーン](../Page/クルアーン.md "wikilink")（コーラン）の朗読の表記法に、機能的には似ている。この初期の記譜法は、[ギリシャ語](https://ja.wikipedia.org/wiki/ギリシャ語 "wikilink")で「準旋律的な朗読」を意味するἐκφώνησις（ekphonesis）から、エクフォネティック記譜法（ekphonetic notation）と呼ばれた。

9世紀頃、ネウマは聖歌の旋律的朗読のための速記的な記憶補助として出現した。一般的には、ネウマによる記譜法は[東ローマ帝国](../Page/東ローマ帝国.md "wikilink")で開発されたと考えられている。これは、当時の帝国（現在の南[トルコ](../Page/トルコ.md "wikilink")、[シリア](../Page/シリア.md "wikilink")、[レバノン](https://ja.wikipedia.org/wiki/レバノン "wikilink")、[イスラエル](../Page/イスラエル.md "wikilink")）の主要都市で文書化された数多くの作曲や文化的活動から、妥当だと思われる。今日でも[東方教会](../Page/東方教会.md "wikilink")の伝統として、[ギリシャ正教](../Page/ギリシャ正教.md "wikilink")音楽や改良ネウマ記譜法では有効であり、[西ヨーロッパ](../Page/西ヨーロッパ.md "wikilink")では新しい技術である[ポリフォニー](https://ja.wikipedia.org/wiki/ポリフォニー "wikilink")の発展に伴いネウマによる記譜法が使用されなくなっていった事実も手伝い、現存する[ビザンティン聖歌](https://ja.wikipedia.org/wiki/ビザンティン聖歌 "wikilink")の[写本](../Page/写本.md "wikilink")や印刷資料は、[グレゴリオ聖歌](../Page/グレゴリオ聖歌.md "wikilink")のものよりはるかに多い。

[スラヴ](https://ja.wikipedia.org/wiki/スラヴ "wikilink")式ネウマ記譜法（ズナメニー・チャント[Znamenny Chantの記譜法](https://ja.wikipedia.org/wiki/:en:Znamenny_Chant "wikilink")）は、ビザンティン聖歌やグレゴリオ聖歌の記譜法より、解読・転写がさらに困難である。

## ネウマによる記譜法

ネウマによる記譜法は、時代、[地方](../Page/地方.md "wikilink")によって様々なバリエーションがあるが、ここでは譜線ネウマについて記述する。

### 音部記号

譜線ネウマでは4本の譜線が使われ、[近代記譜法と同じように線上または線と線の間に音符が書かれる](../Page/五線譜.md "wikilink")。先頭にはC音（ハ音、ド）またはF音（ヘ音、ファ）の[音部記号](../Page/音部記号.md "wikilink")が書かれる。これが後に[ハ音](https://ja.wikipedia.org/wiki/ハ音 "wikilink")記号、[ヘ音](https://ja.wikipedia.org/wiki/ヘ音 "wikilink")記号となる。

### 1音ネウマ

|                                                                                                            |                     |
| ---------------------------------------------------------------------------------------------------------- | ------------------- |
| [<File:punctum1.gif>](https://ja.wikipedia.org/wiki/File:punctum1.gif "fig:File:punctum1.gif")             | **プンクトゥム** (「点」)    |
| [virga1.gif](https://ja.wikipedia.org/wiki/File:virga1.gif "fig:virga1.gif")                               | **ヴィルガ** （「ロッド」）    |
| [<File:repercussive.gif>](https://ja.wikipedia.org/wiki/File:repercussive.gif "fig:File:repercussive.gif") | **ビプンクトゥム**（「2つの点」） |

ヴィルガとプンクトゥムは同じように歌われる。ビプンクトゥムは、二倍の長さの音を表すのか、同じ音を2回はっきりと分けるのかについて、研究者の間で一致しない。後者の解釈が好ましい場合、ビプンクトゥムは「リパーカッシヴ（反射した）・ネウマ」と呼ばれるかもしれない。

|                                                                                             |                           |       |

### 合成ネウマ

いくつかのネウマは一つの音節のために一列に並置することができるが、以下の用例には特定の名前がある。 これらはごく一部の例だけである。

| ------------------------------------------------------------------------------------------------------------------------------------------ | ------------- | --------------------------------------------------------- |
