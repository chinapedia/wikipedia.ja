**カオスの縁**（カオスのふち、）とは、[クリストファー・ラングトン](../Page/クリストファー・ラングトン.md "wikilink")により発見され、[ノーマン・パッカード](https://ja.wikipedia.org/wiki/ノーマン・パッカード "wikilink")により名付けられた、[セルオートマトン](https://ja.wikipedia.org/wiki/セルオートマトン "wikilink")における概念。振る舞いが秩序からカオスへ移るような[システム](../Page/システム.md "wikilink")において、秩序とカオスの境界に位置する領域。[複雑系](../Page/複雑系.md "wikilink")や[人工生命](../Page/人工生命.md "wikilink")、[生命](../Page/生命.md "wikilink")の[進化](../Page/進化.md "wikilink")などの研究において着目されてきた。[理論生物学](../Page/理論生物学.md "wikilink")においては、[スチュアート・カウフマン](../Page/スチュアート・カウフマン.md "wikilink")による、生命の発生と進化には[自然淘汰](https://ja.wikipedia.org/wiki/自然淘汰 "wikilink")の他に[自己組織化](../Page/自己組織化.md "wikilink")が必要であり、進化の結果、生命は「カオスの縁」で存在するという仮説がよく知られる。

## セル・オートマトン

1980年代初頭から[スティーブン・ウルフラム](../Page/スティーブン・ウルフラム.md "wikilink")は1次元セル・オートマトンのルール（遷移関数）ごとの挙動を調査し、その挙動を以下のように4つにクラス分けした。

  - クラスI：均一な一定状態に漸近する挙動
  - クラスII：周期的な状態に漸近する挙動
  - クラスIII：ランダムな状態を維持する挙動
  - クラスIV：他のクラスほど厳密に定義されないが、上記の3クラスに当てはまらない挙動

ウルフラムはクラスIからIIIまでに対し、[力学系](../Page/力学系.md "wikilink")の挙動とアナロジー的に該当するものを当て嵌めている。

  - クラスI：安定[不動点](../Page/不動点.md "wikilink")
  - クラスII：[リミットサイクル](../Page/リミットサイクル.md "wikilink")
  - クラスIII：[カオス](../Page/カオス理論.md "wikilink")

ウルフラムによればクラスIVについては該当する力学系の挙動が存在しない。クラスIVでは非常に複雑な挙動が起こる。いくつかの局所的な構造が生み出され、それらはセル空間内を移動し、相互作用を起こし合う。また、ある初期値では全て一定状態に漸近したり、別の初期値では周期的状態に漸近したり、ランダム状態を維持したりなどの変化も見せる。以下の図はウルフラムのルール番号によってルール110と呼ばれるルールを採用したときのセル・オートマトンの挙動（時間発展）を示している。初期配置は黒一点のみが存在する場合である。クラスIVに分類される。 [CA_rule110s.png](https://ja.wikipedia.org/wiki/File:CA_rule110s.png "fig:CA_rule110s.png")

クリストファー・ラングトンはクラスIVについてさらに調べるために、次のようなパラメータを導入した。

\[\lambda = \frac{k^\rho-n_q}{k^\rho}\] ここで、*k* は状態数、 *ρ* は近傍数を意味し、*k<sup>ρ</sup>* は可能な近傍の状態数となる。状態数 *k* の内の任意な一つの状態 *q* を「静止状態」と呼ぶとする。*n<sub>q</sub>* は *k<sup>ρ</sup>* の内の次の時刻に静止状態（すなわち *q* ）となる数を示す。*λ* は静止状態とならない割合を示しており、一般には *λ* パラメータなどと呼ばれる。あるいは、ラングトン自身は *λ* パラメータのことを「あるレベルの挙動の複雑さに関連する統計量」と位置づけている。

*n<sub>q</sub>* の最小から最大までの範囲は、0 ≤ *n<sub>q</sub>* ≤ *k<sup>ρ</sup>* なので、*λ* の範囲は 0 ≤ *λ* ≤ 1 となる。ラングトンによれば、*λ* = 0 で最少である[複雑性](https://ja.wikipedia.org/wiki/複雑性 "wikilink")は、*λ* の増加とともにも複雑性も増加し、*λ* がある値となったところで極大となり、その後は複雑性は減少していき、*λ* = 1 でまた最少となる。複雑性が極大となる臨界値は *λ<sub>c</sub>* で表される。ウルフラムのクラスと一緒にまとめると、挙動とクラスと *λ* パラメータは以下のような関係の下に変化する。

|            |       |   |       |   |                 |   |        |
| ---------- | ----- | - | ----- | - | --------------- | - | ------ |
| 挙動：        | 不動点   |   | 周期的   |   | "複雑"            |   | カオス    |
| ウルフラムのクラス： | クラス I | ⇒ | クラスII | ⇒ | クラスIV           | ⇒ | クラスIII |
| *λ* パラメータ： | 0     |   |       |   | *λ<sub>c</sub>* |   |        |

ただし、上記の区分は *k* や *ρ* が大きな値のときは良く機能するが、小さいときはあまりうまく働かない。

このように、クラスIVはカオス的・ランダム的振る舞いと秩序的・静的振る舞いの境界に存在し、この領域を「カオスの縁」と呼ぶ。
