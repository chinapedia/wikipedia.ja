は、かつて存在した長崎県[佐世保市](../Page/佐世保市.md "wikilink")に本店を置いていた[第二地方銀行](../Page/第二地方銀行.md "wikilink")である。

## 概要

1940年8月、平戸無尽、大正無尽、朝日無尽が合併し佐世保無尽を設立。1949年2月に九州無尽と改称。1951年10月に[相互銀行法](https://ja.wikipedia.org/wiki/相互銀行法 "wikilink")の施行によって九州相互銀行として発足し、1989年2月に[第二地銀](https://ja.wikipedia.org/wiki/第二地銀 "wikilink")に転換し九州銀行となる。

九州銀は、佐世保市、[福岡市](https://ja.wikipedia.org/wiki/福岡市 "wikilink")の2本部制を敷き、本籍佐世保市、現住所福岡市[中洲](../Page/中洲.md "wikilink")と呼ばれるなど\[1\]、福岡県内での営業に力を入れ、1986年8月には福岡市[中央区](../Page/中央区_\(福岡市\).md "wikilink")[西中洲に所在した旧福岡支店を取り壊し](https://ja.wikipedia.org/wiki/西中洲_\(福岡市\) "wikilink")、総工費20億円を投じ建設した地下1階、地上11階からなる九銀ビルが竣工\[2\]。同ビルに福岡支店・本部を置き、[取締役会](https://ja.wikipedia.org/wiki/取締役会 "wikilink")や支店長会議なども開催していた\[3\]\[4\]。

地方公共団体との取引では、佐世保市、[松浦市](../Page/松浦市.md "wikilink")からは古くから[指定代理金融機関](../Page/指定代理金融機関.md "wikilink")を受託してきたが、普銀転換後は積極的に自治体に対してPRを展開し、1989年4月7日から[布津町](../Page/布津町.md "wikilink")（現：[南島原市](../Page/南島原市.md "wikilink")）\[5\]、同年10月28日には[鷹島町](../Page/鷹島町.md "wikilink")（現：[松浦市](../Page/松浦市.md "wikilink")）からはそれぞれ[指定金融機関](../Page/指定金融機関.md "wikilink")を受託していた\[6\]。

1990年9月に設立以来参画し、[勘定系システム](https://ja.wikipedia.org/wiki/勘定系システム "wikilink")の開発・運用を委託してきた[システムバンキング九州共同センター](https://ja.wikipedia.org/wiki/システムバンキング九州共同センター "wikilink")から脱退。同年10月1日には[日本ユニシス](https://ja.wikipedia.org/wiki/日本ユニシス "wikilink")開発による自営の第三次オンラインシステムを稼働させた\[7\]。

[バブル経済](../Page/バブル経済.md "wikilink")期に九州銀は、関連会社であった九州ファイナンス\[8\]や同行OBが社長を歴任していた九州流通サービス\[9\]の両[ノンバンク](https://ja.wikipedia.org/wiki/ノンバンク "wikilink")が不動産のほか、中洲の飲食業などに多額の融資を行い、それが[バブル崩壊](../Page/バブル崩壊.md "wikilink")によって[不良債権](../Page/不良債権.md "wikilink")となった\[10\]。また[有価証券](../Page/有価証券.md "wikilink")の含み損も100億円抱えるなどしたため、1993年4月には、利益金処分や昇格などを実施するにあたって[大蔵省](../Page/大蔵省.md "wikilink")の承認が必要となる決算承認銀行に指定された。指定に先立ち同省から招聘した渡部裕資頭取の陣頭指揮のもとに経営改革委員会を設置。人員の削減や資金の調達、運用の見直しを始めた\[11\]。その後[ネットバブル](https://ja.wikipedia.org/wiki/ネットバブル "wikilink")が到来したため、運用益をあげていたが、程なくネットバブルが崩壊。株価下落で50億の損失を出した\[12\]。

経営の瀬戸際に立たされた九州銀の救済策を思案した[金融庁](../Page/金融庁.md "wikilink")は、九州銀・[福岡シティ銀行](../Page/福岡シティ銀行.md "wikilink")・[長崎銀行](../Page/長崎銀行.md "wikilink")を[経営統合](https://ja.wikipedia.org/wiki/経営統合 "wikilink")させる青写真を描き、2000年12月末、3行は2002年4月に統合する方針で検討を始めるが、金融庁が[公的資金](https://ja.wikipedia.org/wiki/公的資金 "wikilink")の投入を経営基盤が盤石な福岡シティ銀に一本化したい意向を示したため、福岡シティ銀が負担が大きすぎるとして拒否。代わりに福岡シティ銀が資本提携による緩やかな統合を提案するも金融庁がそれを拒絶した。これによって3行統合はご破算となった\[13\]\[14\]。

公的資金注入の申請期限である2001年3月末が迫る中、金融庁は次善の策として、不正融資事件で元頭取が逮捕され、その後遺症から1998年3月期から2期連続で巨額の経常赤字を計上し、同庁の統制下にあった[親和銀行](../Page/親和銀行.md "wikilink")に対し九州銀との経営統合を迫った\[15\]。2001年3月13日、九州・親和の両行は経営統合に向け最終調整に入ったと発表\[16\]。2002年3月、九州銀は300億円の公的資金を金融庁から受け入れ、同年4月1日、両行は[金融持株会社](../Page/金融持株会社.md "wikilink")である[九州親和ホールディングス](../Page/九州親和ホールディングス.md "wikilink")を設立し傘下に入り、翌年4月1日に親和銀と九州銀は合併した。

この再編劇に関しては、金融庁からの要求を拒否しなかった当時の親和銀頭取に対し、後々まで同行内で怨嗟の声が上がっていたという\[17\]。

## 沿革

| 時                  | 出来事                                                                                                                    |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------- |
| 1940年（昭和15年）8月     | 長崎県北部の[無尽](../Page/無尽.md "wikilink")会社3社が合併して**佐世保無尽株式会社**が設立される。                                                      |
| 1949年（昭和24年）2月     | **九州無尽株式会社**に改称。                                                                                                       |
| 1951年（昭和26年）10月    | [相互銀行](../Page/相互銀行.md "wikilink")となり、**株式会社九州相互銀行**に改称。                                                               |
| 1984年（昭和59年）12月1日  | [福岡証券取引所](../Page/福岡証券取引所.md "wikilink")に株式上場。                                                                         |
| 1987年（昭和62年）12月16日 | [東京証券取引所](../Page/東京証券取引所.md "wikilink")及び[大阪証券取引所](https://ja.wikipedia.org/wiki/大阪証券取引所 "wikilink")両所第二部に株式上場\[18\]。 |
| 1989年（平成元年）2月      | [第二地方銀行](../Page/第二地方銀行.md "wikilink")に転換し**株式会社九州銀行**となる。                                                             |
| 2001年（平成13年）3月16日  | 親和銀行と持株会社方式による経営統合で合意。                                                                                                 |
| 2002年（平成14年）4月1日   | 持株会社の九州親和ホールディングスの完全子会社化。                                                                                              |
| 2002年（平成14年）3月26日  | 各証券取引所の上場廃止。                                                                                                           |
| 2003年（平成15年）4月1日   | 親和銀行と合併。存続行は親和銀行であり、九州銀行の法人格は消滅した。                                                                                     |
