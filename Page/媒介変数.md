[数学](../Page/数学.md "wikilink")における**媒介変数**（ばいかいへんすう）、 **助変数**（じょへんすう、; **補助変数**）、**パラメータ**（; **母数**、**径数**）は、主たる[変数](https://ja.wikipedia.org/wiki/変数_\(数学\) "wikilink")（[自由変数](https://ja.wikipedia.org/wiki/自由変数 "wikilink")）に対して補助的に用いられる。異なる変数の間の陰伏的な関係を記述する媒介変数は、その媒介変数が変化したときの系の振る舞いを見るという意味で「変数」と見ることができるが、対照的に主変数の変化に伴う系の振る舞いを調べたい場合などでは、しばしば補助変数は（「値を取り換えることができる」という意味で値は任意にとれるけれども）「定数」として扱われる。

広範な各分野においてより特定の意味を持って用いられるけれども、一般に「[パラメータ](https://ja.wikipedia.org/wiki/パラメータ "wikilink")」は特定の系（事象や対象や状況など）を決定したり分類したりする助けとなる任意の特徴量を言う。つまり、パラメータは系の同定（あるいは、状態や振る舞いの評価、条件の特定など）に際して有用あるいは重大な役割を果たす系の要素となるものである。

## 概観

### 補助的な変数を含む函数

[函数](https://ja.wikipedia.org/wiki/函数 "wikilink")を定義することには、[変数を一つまたは多数](https://ja.wikipedia.org/wiki/変数_\(数学\) "wikilink")、として指定することが含まれる。補助変数を含む形で函数を定義することもできるが、ふつう補助変数はその函数のとる引数としてはリストしない。補助変数を含めて考えるとき、実際には一つの函数ではなく函数の[族の全体を定めているのだと考えなければならない](https://ja.wikipedia.org/wiki/族_\(数学\) "wikilink")。例えば、一般の[二次函数](https://ja.wikipedia.org/wiki/二次函数 "wikilink")を \(f(x):=ax^2+bx+c\) と宣言する場合、 はこの函数の引数を表すもので、 は「任意定数」であるものとするが、この  の値を一つ決めるごとに個々の特定の二次函数が決定されると考えることができるという意味で、 はこの二次函数の族のパラメータである。

函数がパラメータに依存して決まることを陽に表すために、パラメータを函数名に含めることができる。例えば、\[\[底を持つ対数函数|底 </math> を考察の最も基本的な対象として考え、[カリー化](https://ja.wikipedia.org/wiki/カリー化 "wikilink")などを用いてより少ない変数を持つ函数を定義することになる。

パラメータを含む函数の全体をひとつの「パラメータ付けられた族」(*parametric family*), すなわち函数の[添字付けられた族](https://ja.wikipedia.org/wiki/添字付けられた族 "wikilink")と見ることはしばしば有用である。

### 解析幾何学

[解析幾何学](https://ja.wikipedia.org/wiki/解析幾何学 "wikilink")において[曲線](https://ja.wikipedia.org/wiki/曲線 "wikilink")は適当な函数の像としてしばしば与えられる。この函数の引数は常に「媒介変数」と呼ばれる。例えば、原点を中心とする半径  の円は複数の表し方がある:

  - [陰伏関係式](https://ja.wikipedia.org/wiki/陰函数 "wikilink"): \(x^2+y^2=1.\)
  - [媒介変数表示](https://ja.wikipedia.org/wiki/媒介変数表示 "wikilink"): \((x,y)=(\cos t,\sin t).\) このときの変数  が媒介変数。

これらは他の分野では函数と呼ぶことはあるかもしれないが、解析幾何学においてはその[独立変数](https://ja.wikipedia.org/wiki/独立変数 "wikilink")を媒介変数とするとして特徴づけられる。

### 解析学

[解析学](../Page/解析学.md "wikilink")において、補助変数に依存する積分をしばしば考える。例えば \(F(t)=\int_{x_0(t)}^{x_1(t)}f(x;t)\,dx\) において  は左辺の函数  の引数であるが、同時に右辺の積分がそれに依存してきまるという意味でパラメータである。右辺の積分の評価に際して  は一貫して「定数」として扱われる（つまり、その意味ではパラメータであると考えるべきである）。しかし  が  の異なる値に対して値をどう変えるかを知りたいならば  は変数として扱われなければならない。なお  は「積分変数」と呼ばれる**見かけの変数** (*dummy variable*) である（これも紛らわしいことに積分のパラメータと呼ぶことがある）。

### 論理学

[論理学](https://ja.wikipedia.org/wiki/論理学 "wikilink")において**開述語** (*open predicate*) に引き渡される（あるいは、開述語が引数にとる）項を「パラメータ」と呼び、その述語の中で局所的に定義されるパラメータを「変項」と呼び分ける場合がある。この余分な区別は代入を定義するときの面倒にたいして効果がある（この区別が無いとき、変数の取り込みを避けるためには特別の注意を要する）。大抵の文献では、単に開述語に引き渡される項という意味で変項と呼んで、代入の定義において[自由変数](https://ja.wikipedia.org/wiki/自由変数 "wikilink")と[束縛変数](https://ja.wikipedia.org/wiki/束縛変数 "wikilink")とを区別するという手段をとる。

## 媒介変数を用いて解かれている常微分方程式の例

媒介変数を二つ用いて、求積法により解かれている常微分方程式と、その一般解を以下に示す \[1\] \[2\] \[3\] \[4\] \[5\]。

陰関数表示の連立常微分方程式\[6\]\[7\]\[8\]\[9\]\[10\] 。

\[\; F\left(z,\; \frac{dy}{dx}\right) = 0,\]

\[\; G\left(y,\; \frac{dz}{dx}\right) = 0.\] 一般解を以下に示す。

\[\; x = \int \frac{1}{\eta(s)} \cdot  \frac{d\phi(t)}{dt}\,dt + C_1,\]

\[\; y = \xi(s),\]

\[\; z = \phi(t),\]

\[\; \int \psi(t)\cdot \frac{d\phi(t)}{dt}\,dt = \int \eta(s) \cdot \frac{d\xi(s)}{ds}\,ds + C_2,\]

\[\; F(\phi(t),\; \psi(t)) \equiv 0,\]

\[\; G(\xi(s),\; \eta(s))  \equiv 0.\]

上式の、*s* と *t* が媒介変数であり、*F* と *G* は既知関数である。また、 *φ*(*t*) と *ψ*(*t*) は、*F*(*φ*(*t*), *ψ*(*t*)) ≡ 0 を満たす *t* の関数であり、 *ξ*(*s*) と *η*(*s*) は、*G*(*ξ*(*s*), *η*(*s*)) ≡ 0 を満たす *s* の関数である。

なお， が独立変数であり、 と  は  を変数とする関数である。 また、*C*<sub>1</sub> と *C*<sub>2</sub> は積分定数である。
