**1999年J1最終節**（1999ねんJ1さいしゅうせつ）は、[1999年](../Page/1999年.md "wikilink")[11月27日](../Page/11月27日.md "wikilink")に行われた[日本プロサッカーリーグ](../Page/日本プロサッカーリーグ.md "wikilink")（Jリーグ）[ディビジョン1](https://ja.wikipedia.org/wiki/J1リーグ "wikilink")（J1）セカンドステージ第15節のことを指す。本項では特に、その中で[ディビジョン2](https://ja.wikipedia.org/wiki/J2リーグ "wikilink")（J2）降格の可能性があった[アビスパ福岡](../Page/アビスパ福岡.md "wikilink")（福岡）、[ジェフユナイテッド市原](https://ja.wikipedia.org/wiki/ジェフユナイテッド市原・千葉 "wikilink")（市原）、[浦和レッドダイヤモンズ](https://ja.wikipedia.org/wiki/浦和レッドダイヤモンズ "wikilink")（浦和）の試合について記す。

## 最終戦までの経緯

[1999年](../Page/1999年.md "wikilink")、Jリーグはこのシーズンから2部制を導入。J1（当時16チーム）年間総合成績の下位2チームがJ2（当時10チーム）の上位2チームと自動入れ替えすることとなった。

この年、まず低迷したのはベルマーレ平塚（平塚、現・[湘南ベルマーレ](https://ja.wikipedia.org/wiki/湘南ベルマーレ "wikilink")）だった。平塚は前年、親会社である[フジタ](../Page/フジタ.md "wikilink")の経営不振によりクラブの規模を縮小し、[呂比須ワグナー](https://ja.wikipedia.org/wiki/呂比須ワグナー "wikilink")や[名塚善寛](https://ja.wikipedia.org/wiki/名塚善寛 "wikilink")、[小島伸幸](https://ja.wikipedia.org/wiki/小島伸幸 "wikilink")、[田坂和昭](https://ja.wikipedia.org/wiki/田坂和昭 "wikilink")、[洪明甫](../Page/洪明甫.md "wikilink")といった代表クラスの選手を大量放出。若手中心で99年シーズンに臨むも1stステージは最下位。2ndステージも浮上のきっかけをつかめず年間わずか4勝にとどまり、2ndステージ第12節に15位以下（降格）が決定した。

一方、浦和は、[前シーズン（1998年）の](https://ja.wikipedia.org/wiki/1998年のJリーグ "wikilink")2ndステージで3位と躍進したことにより優勝戦線に絡むと予想されていたが、[永井雄一郎](https://ja.wikipedia.org/wiki/永井雄一郎 "wikilink")や[小野伸二](../Page/小野伸二.md "wikilink")が[FIFAワールドユース日本代表](../Page/FIFA_U-20ワールドカップ.md "wikilink")、[石井俊也](https://ja.wikipedia.org/wiki/石井俊也 "wikilink")が[シドニー五輪予選の代表としてチームを離れることが多かった上に](../Page/2000年シドニーオリンピックのサッカー競技.md "wikilink")、[吉野智行](https://ja.wikipedia.org/wiki/吉野智行 "wikilink")や[盛田剛平](https://ja.wikipedia.org/wiki/盛田剛平 "wikilink")などの新人選手の成績が振るわずベストメンバーが組めない状況が続き、1stステージは13位と振るわず、浦和は[原博実](https://ja.wikipedia.org/wiki/原博実 "wikilink")監督を解任し、2ndステージからオランダ人の[ア・デモス](https://ja.wikipedia.org/wiki/ア・デモス "wikilink")を監督として招聘した。ところが、シドニー五輪の一次予選で小野伸二がひざを負傷してシーズンのほとんどを離脱するなどのため連敗を重ね、さらには第6節・[ジュビロ磐田](../Page/ジュビロ磐田.md "wikilink")戦から第9節・[名古屋グランパスエイト](https://ja.wikipedia.org/wiki/名古屋グランパスエイト "wikilink")戦まで4試合連続延長Vゴール負けを喫するなどにより、J1残留争いに巻き込まれることになる。

市原はこの年、[前年の天皇杯で](../Page/第78回天皇杯全日本サッカー選手権大会.md "wikilink")[横浜フリューゲルス](../Page/横浜フリューゲルス.md "wikilink")を優勝に導いた[ゲルト・エンゲルス](../Page/ゲルト・エンゲルス.md "wikilink")を監督として招聘するも、前シーズンまで司令塔として機能した[ネナド・マスロバル](https://ja.wikipedia.org/wiki/ネナド・マスロバル "wikilink")の穴を埋めることが出来ず、守備が崩壊したこともあって1stステージで15位と低迷。エンゲルスを解任し監督で臨んだ2ndステージもチーム状況の劇的な改善とはならず、降格圏をなかなか脱出できない。

また、前年の[J1参入決定戦](../Page/J1参入決定戦.md "wikilink")で辛うじてJ1参入を果たした福岡もなかなか波に乗れず、降格圏ぎりぎりの順位をさまようこととなり、シーズン終盤には浦和・市原・福岡の3チームが残留を争う構図となった。

11月23日に行われた2ndステージ第14節、この時点で降格圏の15位は市原（勝ち点22）。14位の浦和とは勝ち点差3、13位の福岡とは勝ち点差が6あり、さらに福岡とは得失点差で10点離され絶体絶命の状況にあったが、福岡のホーム・[博多の森で行われた直接対決で市原が福岡に](https://ja.wikipedia.org/wiki/東平尾公園博多の森球技場 "wikilink")5-0と大勝。残留可能性を残す勝ち点差3以内を確保すると共に、逆に得失点差でのアドバンテージを確保して残留への望みをつなぐ。一方、14位の浦和はヴェルディ川崎（現・[東京ヴェルディ](https://ja.wikipedia.org/wiki/東京ヴェルディ1969 "wikilink")）に試合終盤に追いつかれ引き分けて勝ち点1の上積みにとどまる。これにより市原に勝ち点差1にまで詰め寄られ、浦和にとって後味の悪い形で残留争いは最終節にまでもつれ込むこととなった。その最終節を残し、順位は以下の通りとなった。       |} ※平塚は11月20日の時点で降格決定。

## 最終節

### 概要

3チームが絡む試合は下の3試合（太字が降格の可能性が残っていたチーム）。

  - **[浦和レッドダイヤモンズ](https://ja.wikipedia.org/wiki/浦和レッドダイヤモンズ "wikilink")** - [サンフレッチェ広島](https://ja.wikipedia.org/wiki/サンフレッチェ広島 "wikilink")
      -
        浦和市駒場スタジアム（現・[さいたま市駒場スタジアム](../Page/さいたま市駒場スタジアム.md "wikilink")、浦和駒場スタジアム）
  - [横浜F・マリノス](https://ja.wikipedia.org/wiki/横浜F・マリノス "wikilink") - **[アビスパ福岡](../Page/アビスパ福岡.md "wikilink")**
      -
        [横浜国際総合競技場](../Page/横浜国際総合競技場.md "wikilink")（日産スタジアム）
  - [ガンバ大阪](../Page/ガンバ大阪.md "wikilink") - **[ジェフユナイテッド市原](https://ja.wikipedia.org/wiki/ジェフユナイテッド市原・千葉 "wikilink")**
      -
        [万博記念競技場](../Page/万博記念競技場.md "wikilink")

この年のレギュレーションでは勝ち点の配分が「90分勝利：3、延長戦勝利：2、引き分け：1、敗戦（90分・延長問わず）：0」であったため、福岡は引き分け以上、浦和は90分勝利で無条件に残留が決まり、市原は引き分け以上が絶対条件であった。

一方、福岡と市原の得失点差が並んでおり、浦和は両チームより得失点差が4点下回っていたため、市原は浦和に勝ち点で追いつけば浦和より上位となり、その場合には福岡の4点差以上での敗戦が浦和の事実上の残留条件となる\[1\]ことから、浦和にとっても90分勝利が必須とも言える条件であった。

### 試合展開

浦和のゲームは、浦和が主力である[ピクンと](../Page/フェルナンド・ピクン.md "wikilink")[岡野雅行が欠場](https://ja.wikipedia.org/wiki/岡野雅行_\(サッカー選手\) "wikilink")。対する広島も[ポポヴィッチ](https://ja.wikipedia.org/wiki/トニー・ポポヴィッチ "wikilink")、[藤本主税](../Page/藤本主税.md "wikilink")、[久保竜彦](../Page/久保竜彦.md "wikilink")が欠場し、お互いに主力を欠いてのスタートとなった。

### 前半

浦和では前半から一進一退の緊迫した好ゲームとなった。浦和は何度も広島ゴールへ攻めるも、広島の堅守を崩すことができず苦戦を強いられた。

一方、横浜では横浜FMが前半11分、[城彰二](https://ja.wikipedia.org/wiki/城彰二 "wikilink")のゴールで先制。その後、前半23分に福岡は攻守の要である[フェルナンドがラフプレーにより一発退場となり](https://ja.wikipedia.org/wiki/フェルナンド・エンリケ・マリアーノ "wikilink")、10人での戦いを強いられる。その後は試合は膠着状態となり、そのまま前半終了。

浦和と大阪では得点の動きなし。

  -
    **浦和 0 – 0 広島**
    **横浜 1 – 0 福岡**
    **G大阪 0 – 0 市原**

|}

### 後半

後半に入り、浦和は後半9分に[ベギリスタインから](https://ja.wikipedia.org/wiki/アイトール・ベギリスタイン "wikilink")[大柴健二](https://ja.wikipedia.org/wiki/大柴健二 "wikilink")に交代するも、相変わらず広島の堅守に苦戦する。

後半15分、万博では市原の[武藤真一](https://ja.wikipedia.org/wiki/武藤真一 "wikilink")がゴールを上げ先制。市原は両チームを上回って降格圏を脱出、代わって浦和が降格圏に。その一報が浦和の[ア・デモス](https://ja.wikipedia.org/wiki/ア・デモス "wikilink")監督のもとに入ると、後半18分に[MFの](../Page/ミッドフィールダー.md "wikilink")[中村忠に代えて](../Page/中村忠_\(サッカー選手\).md "wikilink")[盛田剛平](https://ja.wikipedia.org/wiki/盛田剛平 "wikilink")へ交代し、攻撃的な布陣を引いた。

  -
    **浦和 0 – 0 広島**
    **横浜 1 – 0 福岡**
    **G大阪 0 – 1 市原**

|}

浦和は後半34分に[DFの](../Page/ディフェンダー_\(サッカー\).md "wikilink")[城定信次](https://ja.wikipedia.org/wiki/城定信次 "wikilink")に代え、この日ベンチスタートだった[福田正博](../Page/福田正博.md "wikilink")と続けて[FWを投入し](https://ja.wikipedia.org/wiki/フォワード_\(サッカー\) "wikilink")、3-3-4の攻撃的なフォーメーションへ切り替え、何としても点を取るという作戦に出た。その後、小野伸二やDFの[山田暢久](https://ja.wikipedia.org/wiki/山田暢久 "wikilink")もチャンスメークに徹し、盛田、大柴、福田を中心として広島ゴールを何度も攻め、点を狙いに行くも全て阻まれる。

そんな中、横浜では後半35分に横浜FMの城がまたもゴールを決め2点目。福岡はこれ以上点を取られれば降格の危険が出てきたことから、これ以上の失点を防ぐべく守備に徹した。

  -
    **浦和 0 – 0 広島**
    **横浜 2 – 0 福岡**
    **G大阪 0 – 1 市原**

一方、浦和は後半42分、小野のコーナーキックから盛田が飛び込みヘディングするも外してしまい、決定的なチャンスを逃してしまう。そして、ついに90分間が終了した時点で両チームとも得点をあげることができず、延長戦へ突入となった。

この時点で福岡は横浜FMに0-2で敗戦し勝ち点28・得失点差-18、市原はG大阪に1-0で勝利し勝ち点28・得失点差-15で共に試合終了。このためもし浦和が延長勝ちしても、福岡や市原と勝ち点で並ぶが得失点差で下回るため、この時点で浦和のJ2降格が決定した。

  -
    **浦和 0 – 0 広島**
    **横浜 2 – 0 福岡**（試合終了）
    **G大阪 0 – 1 市原**（試合終了）

|}

### その後

残留の可能性が消えた浦和の試合は、そのまま延長後半まで進む。延長後半1分に小野のショートコーナーからペトロビッチがゴール前へのクロスボールを上げ、それを福田が蹴り込み、Vゴールとなって勝負は決まった。

この時、J1残留が決定したと思い抱きついてきた[池田学](https://ja.wikipedia.org/wiki/池田学 "wikilink")（ルーキーの池田だけには試合に集中させるべく、チームの降格を知らせなかった。引退後、福田は「学が喜んでいたということは、90分間の戦いで降格が決まったことを知らなかったということ」「それがわからない選手が11人の中にいたということが、そのときの浦和のレベルってことだよね」、即ち池田を含めた全員で意識を統一する必要があったと当時の首脳陣の行動を批判的に振り返っている\[2\]）とそれを手荒に振り払った福田、小野や岡野の号泣する姿、ピッチに倒れこむペトロビッチのシーンなどの姿が見られた。[Vゴールと言う呼び方はJリーグ特有のものであるが](../Page/ゴールデンゴール.md "wikilink")、その日のスポーツニュースにて**「世界で一番悲しいVゴール」**と表現され、のちにこの試合と決勝ゴールがこう呼ばれるようになったといわれる。

号泣する福田に[ドーハの悲劇](https://ja.wikipedia.org/wiki/ドーハの悲劇 "wikilink")のメンバーでもあった広島の[森保一](https://ja.wikipedia.org/wiki/森保一 "wikilink")が慰めに行ったシーンは名シーンとも言われる。この時、森保は福田に対し**「必ずJ1に戻って来い」**とささやいたといわれている。

  -
    **浦和 1v – 0 広島**（試合終了）
    **横浜 2 – 0 福岡**（試合終了）
    **G大阪 0 – 1 市原**（試合終了）

|}
