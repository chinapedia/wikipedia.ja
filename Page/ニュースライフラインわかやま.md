**ニュースライフラインわかやま**(News Lifeline Wakayama)は、[テレビ和歌山](../Page/テレビ和歌山.md "wikilink")で放送されていた[ニュース](../Page/ニュース.md "wikilink")番組。県内ニュース、[天気予報](../Page/天気予報.md "wikilink")、季節の話題などを伝えている。

## 放送時間

|    | 月-金曜日                                           | 土曜日         | 日曜日                        |
| -- | ----------------------------------------------- | ----------- | -------------------------- |
| 昼  | 14:58-15:05                                     | なし          | なし                         |
| 夕方 | （[@あっと\!テレわか](../Page/@あっと!テレわか.md "wikilink")） | 18:00-18:15 | 18:00-18:20（第1・2週は18:15まで） |
| 夜  | 22:54-23:00                                     | 20:54-21:00 | 21:52-22:00                |

以前は平日夕方にも同名のニュースワイド番組(25分程度)を放送していた。

## 放送内容

|    | 月-金曜日                  | 土曜日             | 日曜日                |
| -- | ---------------------- | --------------- | ------------------ |
| 昼  | ストレートニュース、天気           |                 |                    |
| 夕方 |                        | ニュース、ふるさとの話題、天気 | ニュース、ニュースピックアップ、天気 |
| 夜  | ニュース（天気予報は直前に別番組として放送） |                 |                    |
