**エストニア語**（エストニアご、 ）は、[ウラル語族](../Page/ウラル語族.md "wikilink")・[フィン・ウゴル語派](../Page/フィン・ウゴル語派.md "wikilink")・[バルト・フィン諸語](https://ja.wikipedia.org/wiki/バルト・フィン諸語 "wikilink")に属する[言語](../Page/言語.md "wikilink")。話者は約110万人で、[エストニア](../Page/エストニア.md "wikilink")の主要言語であり、また[公用語](../Page/公用語.md "wikilink")となっている。[フィンランド語](../Page/フィンランド語.md "wikilink")に近く、[ハンガリー語](../Page/ハンガリー語.md "wikilink")とも系統を同じくする。

南エストニア方言と[タリン](../Page/タリン.md "wikilink")周辺で使われる北エストニア方言の2つの方言があり、後者は現在の[標準語](../Page/標準語.md "wikilink")の元となっている。

## 概要

[18世紀](../Page/18世紀.md "wikilink")の初めから[1918年](../Page/1918年.md "wikilink")に至るまでエストニアは[ロシア帝国](../Page/ロシア帝国.md "wikilink")の支配下にあり、約20年後再び[ソ連](https://ja.wikipedia.org/wiki/ソ連 "wikilink")に併合され、[1991年](../Page/1991年.md "wikilink")に再び独立したが、エストニア語にはロシア語の影響があまりない。むしろ、[フィンランド語](../Page/フィンランド語.md "wikilink")や[ドイツ語](../Page/ドイツ語.md "wikilink")のほうが密接に関わりを持っている\[1\]。

フィンランド語とエストニア語は同じ[ウラル語族](../Page/ウラル語族.md "wikilink")に属しており、語彙の面でも文法の面でも共通点が多い。また、ドイツ語との関係は[中国語](https://ja.wikipedia.org/wiki/中国語 "wikilink")と[日本語](../Page/日本語.md "wikilink")の関係に似ており、長期にわたってドイツ語やその一方言[低地ドイツ語](https://ja.wikipedia.org/wiki/低地ドイツ語 "wikilink")から影響を受けてきた。その証拠として、[エストニア共和国](https://ja.wikipedia.org/wiki/エストニア共和国 "wikilink")の首都[タリン](../Page/タリン.md "wikilink")には低地ドイツ語で書かれた歴史文書が大量に保管されている\[2\]。

## 方言

エストニア語には首都・[タリン](../Page/タリン.md "wikilink")を中心にした北方言と、第二の都市・[タルトゥ](../Page/タルトゥ.md "wikilink")を中心にした南方言がある。現在の標準エストニア語は北方言に基づいている。

## 歴史

エストニア語が文字を使って書かれ始めたのは[1520年代](https://ja.wikipedia.org/wiki/1520年代 "wikilink")以降だが、当時はキリスト教の教会に住んでいたドイツ人がエストニア人に布教するために使われたもので、実際に使われていたエストニア語とはかけ離れたものだったという\[3\]。また、1739年には[エストニア語訳聖書](https://ja.wikipedia.org/wiki/エストニア語訳聖書 "wikilink")の全訳が出た。

19世紀に入るころには徐々にエストニア人にも読み書きのできる人々が学校教育の普及によって増え始め、そのころから新聞や小説などが出るようになった\[4\]。独立直後の憲法制定会議の議事録はすでに現代のエストニア語とほぼ変わらないものだという。その後、ソ連時代を経て、現在に至る。

他の[バルト諸国同様](../Page/バルト三国.md "wikilink")、エストニアには[ドイツ系移民が多く](../Page/ドイツ人.md "wikilink")、このためエストニア語は語彙および統語法の両面で[ドイツ語](../Page/ドイツ語.md "wikilink")の影響を強く受けている。言語形態論的には、ウラル語族に多い[膠着語](../Page/膠着語.md "wikilink")からインド・ヨーロッパ語族に多い[屈折語](../Page/屈折語.md "wikilink")（[総合的言語](../Page/総合的言語.md "wikilink")）への移行形態を見せている。informatsioonやkontsertなどドイツ語と発音が似ている単語もしばしば見受けられる。

## 文字

表記には[ラテン文字](../Page/ラテン文字.md "wikilink")を用いる。ただし c, f, q, w, x, y, z, š, žは外国語由来の語にのみ用いられる\[5\]。また、c, q, w, x, yは言語のつづりのまま表記される場合のみ、f, z, š, žは比較的新しい外来語に用いられる\[6\]。

エストニア語独自の文字としては、š, ž, ä, ö, ü, õ がある。ä 、ö、 üはドイツ語と同じでそれぞれのaとe、oとe、uとeの中間の音となり õ は非円唇の o である。 基本的にローマ字読みが可能である。

b, d, gと p, t, kはそれぞれ同じp, t, kの音をあらわす。例)[paar](https://ja.wikipedia.org/wiki/wikt:en:paar "wikilink")-[baar](https://ja.wikipedia.org/wiki/wikt:en:baar "wikilink")「対、組」-「バー(酒場)」\[8\]。但し、母音にはさまれている場合は、p, t, kは「長い」p, t, kとなる。(日本語の「坂(さか)」と「作家(さっか)」に近い。)

語頭のhは読まない話し手も多い。また、šとžはともに(無声後部歯茎摩擦音、英語の*sh*)をあらわし、šは「長く」、žは「短く」発音する(pとbの違いと同じ)\[9\]。

また、jは[ドイツ語](../Page/ドイツ語.md "wikilink")などと同様に//をあらわす(ヤ行の音)\[10\]。

## 文法

[名詞](../Page/名詞.md "wikilink")に[性はないが](../Page/性_\(文法\).md "wikilink")、単数と複数の[数をもち](../Page/数_\(文法\).md "wikilink")、また14の[格変化](../Page/格変化.md "wikilink")を有する。[形容詞](../Page/形容詞.md "wikilink")は名詞同様に数および格変化を有する。[対格](../Page/対格.md "wikilink")を持たず、直接[目的語](../Page/目的語.md "wikilink")は[属格](../Page/属格.md "wikilink")あるいは[分格](https://ja.wikipedia.org/wiki/分格 "wikilink")によって表す。直接目的語は否定文においてはつねに分格となる。肯定文におけるこの区別は[動詞](../Page/動詞.md "wikilink")の[相における](https://ja.wikipedia.org/wiki/相_\(文法\) "wikilink")[完了相](https://ja.wikipedia.org/wiki/完了相 "wikilink")と不完了相の対立におおまかに対応しており一般に可算である個体に対しては属格、非可算な非個体に対しては分格を用いる。また動詞の不定詞は2つあり-ma不定詞と-taまたは-da不定詞が存在する。

### 名詞

#### 格変化

エストニア語の名詞はさまざまな[格変化](../Page/格変化.md "wikilink")をする。また、[属格](../Page/属格.md "wikilink")は必ず終わりが[母音](../Page/母音.md "wikilink")で、[分格](https://ja.wikipedia.org/wiki/分格 "wikilink")は終わりが母音、若しくは[子音](../Page/子音.md "wikilink")のt,dである\[11\]。

単数[属格](../Page/属格.md "wikilink")形に-dをつけることで複数主格形をあらわす\[12\]。

格変化の内訳は[主格](../Page/主格.md "wikilink")、[属格](../Page/属格.md "wikilink")、[分格](https://ja.wikipedia.org/wiki/分格 "wikilink")、[入格](https://ja.wikipedia.org/wiki/入格 "wikilink")、[内格](https://ja.wikipedia.org/wiki/内格 "wikilink")、[出格](https://ja.wikipedia.org/wiki/出格 "wikilink")、[向格](https://ja.wikipedia.org/wiki/向格 "wikilink")、[接格](https://ja.wikipedia.org/wiki/接格 "wikilink")（所格）、[奪格](../Page/奪格.md "wikilink")（離格）、[変格](https://ja.wikipedia.org/wiki/変格 "wikilink")、[様格](https://ja.wikipedia.org/wiki/様格 "wikilink")、、[欠格](https://ja.wikipedia.org/wiki/欠格_\(言語学\) "wikilink")、[共格](https://ja.wikipedia.org/wiki/共格 "wikilink")の14種類\[13\]。

このうち、場所をあらわす格は六種類ある。詳細は以下の表を参照。

|     | 中                                                        | 上                                                       |
| --- | -------------------------------------------------------- | ------------------------------------------------------- |
| ～へ  | \-sse([入格](https://ja.wikipedia.org/wiki/入格 "wikilink")) | \-le([向格](https://ja.wikipedia.org/wiki/向格 "wikilink")) |
| ～で  | \-s([内格](https://ja.wikipedia.org/wiki/内格 "wikilink"))   | \-l([接格](https://ja.wikipedia.org/wiki/接格 "wikilink"))  |
| ～から | \-st([出格](https://ja.wikipedia.org/wiki/出格 "wikilink"))  | \-lt([奪格](../Page/奪格.md "wikilink"))                    |

場所格の変化

ただし、「中」、「上」というのはあくまで目安のようなものである。また、以上の語尾は全て単数属格形につける\[14\]。

#### 人称代名詞

|     | 単数       | 複数         |
| --- | -------- | ---------- |
| 1人称 | mina(ma) | meie(me)   |
| 2人称 | sina(sa) | teie(te)   |
| 3人称 | tema(ta) | nemad(nad) |

人称代名詞\[15\]

カッコ内は強勢のない短形と呼ばれる形。対し、入っていないものを長形という。また、3人称には英語やドイツ語のような男女の区別はない。

### 動詞

動詞の-maで終わる形を不定詞という。-maを取り除いた形を不定詞語幹という\[16\]。

エストニア語は主語の人称によって動詞が変化する。ここでは、動詞[elama](https://ja.wikipedia.org/wiki/wikt:elama "wikilink")(生きている、住んでいる)を例に挙げる。

|     | 単数    | 複数      |
| --- | ----- | ------- |
| 1人称 | ela-n | ela-me  |
| 2人称 | ela-d | ela-te  |
| 3人称 | ela-b | ela-vad |

動詞の活用(直接法現在)\[17\]

不定詞語幹に「-」以下の部分をつけると、文が成立する。また、エストニア語の現在形は非過去をあらわすため、未来のこともあらわす\[18\]。 エストニア語の英語で言うbe動詞に当たるもの([コピュラ](../Page/コピュラ.md "wikilink"))はエストニア語では[olemaという](https://ja.wikipedia.org/wiki/wikt:olema "wikilink")。olemaは不規則動詞で、動詞の規則に当てはまらない\[19\]。

|     | 単数    | 複数     |
| --- | ----- | ------ |
| 1人称 | ole-n | ole-me |
| 2人称 | ole-d | ole-te |
| 3人称 | on    | on     |

olemaの活用(直接法現在)

また、否定形を作る際は否定の小詞「[ei](https://ja.wikipedia.org/wiki/wikt:en:ei#Estonian "wikilink")」を動詞の前に置く。例)Ma olen eestlane. → Ma **ei ole** eestlane.(私はエストニア人です→私はエストニア人ではありません\[20\]。)

但し、ei oleに限っては、特殊な否定形[poleを代わりに使用することもある](https://ja.wikipedia.org/wiki/wikt:en:pole "wikilink")\[21\]。

|     | 単数       | 複数           |
| --- | -------- | ------------ |
| 1人称 | ela-si-n | ela-si-me    |
| 2人称 | ela-si-d | ela-si-te    |
| 3人称 | ela-s    | ela-si-**d** |

動詞の活用(直接法過去)\[22\]

以上のように、elamaという動詞は人称語尾の直前に-si-をつけている。これはエストニア語の動詞の多くが持つ特徴である\[23\]。(但し、olemaはsiの代わりにiを使う。)
