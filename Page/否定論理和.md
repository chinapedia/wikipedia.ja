**否定論理和**（ひていろんりわ）とは、与えられた複数の[命題](../Page/命題.md "wikilink")の全てが偽であることを示す[論理演算](../Page/論理演算.md "wikilink")である。**NOR**と表記される。[矢印](../Page/矢印.md "wikilink")の「↓」を用いて"A ↓ B"とする表記方法もある。

## 性質

[Venn1000.svg](https://ja.wikipedia.org/wiki/File:Venn1000.svg "fig:Venn1000.svg") 否定論理和（NOR）は[否定論理積](../Page/否定論理積.md "wikilink")（NAND）と同様に[完全性](https://ja.wikipedia.org/wiki/否定論理積#完全性 "wikilink")(万能性とも)を持ち、NORのみで任意の論理関数を表現することが出来る。以下にNOT・AND・ORのNORのみによる構成を示す。

  - NOT **A** = **A** NOR **A**
  - **A** AND **B** = ( NOT **A** ) NOR ( NOT **B** ) = ( **A** NOR **A** ) NOR ( **B** NOR **B** )
  - **A** OR **B** = NOT ( **A** NOR **B** ) = ( **A** NOR **B** ) NOR ( **A** NOR **B** )

### 真理値表

| 命題 *P* | 命題 *Q* | *P* NOR *Q* |
| ------ | ------ | ----------- |
| 真      | 真      | **偽**       |
| 真      | 偽      | **偽**       |
| 偽      | 真      | **偽**       |
| 偽      | 偽      | **真**       |
