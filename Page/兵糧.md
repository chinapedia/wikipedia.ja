**兵糧**（ひょうろう、**兵粮**、**軍糧**とも）とは、[戦争](../Page/戦争.md "wikilink")時における[軍隊](../Page/軍隊.md "wikilink")の[食糧](https://ja.wikipedia.org/wiki/食糧 "wikilink")のこと。[日本](https://ja.wikipedia.org/wiki/日本 "wikilink")においては主食である[米](../Page/米.md "wikilink")について論じられる事が多く、**兵糧米**（ひょうろうまい・**兵粮米**）などとも呼ばれている。米の他にも[塩や](https://ja.wikipedia.org/wiki/食塩 "wikilink")[大豆](https://ja.wikipedia.org/wiki/大豆 "wikilink")（[馬の食糧](../Page/ウマ.md "wikilink")（馬糧）としても重要視された）なども含まれている。

## 律令時代

古来から従軍する兵士には兵糧携帯の義務があり、[律令法](../Page/律令法.md "wikilink")においては[糒](https://ja.wikipedia.org/wiki/糒 "wikilink")6斗及び塩2升の自弁が定められていたが、実際には60日分に過ぎず、かつ大量の兵糧携帯は場合によっては行軍の妨げになる可能性もあった。そこで、[蝦夷](../Page/蝦夷.md "wikilink")討伐に際しては[東国](../Page/東国.md "wikilink")からの調達が許され、『[延喜式](../Page/延喜式.md "wikilink")』においては[長門国](../Page/長門国.md "wikilink")の[公出挙稲](../Page/出挙.md "wikilink")4万束が兵粮料として充てることが定められている。また、実際の軍事行動の際には地元有力者からの献納や徴発に頼ることが多かった。

## 平安末期－鎌倉時代

[中世](../Page/中世.md "wikilink")以後は[一国平均役](../Page/一国平均役.md "wikilink")の一環として徴収される例が見られ、特に源平合戦（[治承・寿永の乱](../Page/治承・寿永の乱.md "wikilink")）においては[平家](https://ja.wikipedia.org/wiki/平家 "wikilink")・[源氏双方が兵粮米の賦課を行っている](../Page/河内源氏.md "wikilink")。だが、現地における兵粮米の賦課・徴発は兵士による[濫妨を招く可能性があった](../Page/乱妨取り.md "wikilink")。[文治](../Page/文治.md "wikilink")元年（[1185年](../Page/1185年.md "wikilink")）に[源頼朝](../Page/源頼朝.md "wikilink")が[守護](../Page/守護.md "wikilink")・[地頭](../Page/地頭.md "wikilink")の設置求めて[文治の勅許](../Page/文治の勅許.md "wikilink")を受けると、同時に[荘園](../Page/荘園.md "wikilink")・[国衙領](../Page/国衙領.md "wikilink")の田1段から兵粮米5升を徴収する権利を得た。だが、[国司](../Page/国司.md "wikilink")・[荘園領主](../Page/荘園領主.md "wikilink")達の反発が強く、翌年には撤回された。

## 南北朝－戦国時代

[南北朝時代に入ると](../Page/南北朝時代_\(日本\).md "wikilink")、[北朝](https://ja.wikipedia.org/wiki/北朝_\(日本\) "wikilink")（[室町幕府](../Page/室町幕府.md "wikilink")）は [兵粮料所](https://ja.wikipedia.org/wiki/兵粮料所 "wikilink")（[半済令](https://ja.wikipedia.org/wiki/半済令 "wikilink")参照のこと）を、[南朝は](https://ja.wikipedia.org/wiki/南朝_\(日本\) "wikilink")[朝用分](https://ja.wikipedia.org/wiki/朝用分 "wikilink")を設定して兵糧確保にあたった。室町幕府や[守護大名](../Page/守護大名.md "wikilink")の職制では、[御蔵奉行](https://ja.wikipedia.org/wiki/御蔵奉行 "wikilink")が兵糧確保の任務にあたっていたが、 [戦国時代には](../Page/戦国時代_\(日本\).md "wikilink")、平時より[蔵入地](../Page/蔵入地.md "wikilink")を設置して兵粮確保に力を注ぎ、戦時に際して[小荷駄奉行](https://ja.wikipedia.org/wiki/小荷駄奉行 "wikilink")とその下に[小荷駄隊](https://ja.wikipedia.org/wiki/小荷駄隊 "wikilink")を設けるのが一般的となった。

[上泉信綱](https://ja.wikipedia.org/wiki/上泉信綱 "wikilink")伝の『訓閲集』（大江家兵法書を戦国風に改めた書）巻六「士鑑・軍役」の「小荷駄奉行のこと」の項目には兵糧を3つに分けており、「公儀の糧（腰につけ、帰る時に食べる）」、「主人の糧（着陣1[里](../Page/里.md "wikilink")前に食べる）」、「私の糧（主人より渡される昼飯で何時でも食べる）」と記し、また、上兵には白米、下兵には黒米を渡すことなどが記述されている。

敵方城下の兵糧を買い占めるなどして兵糧を断つ戦術を「兵糧攻め」といい（『広辞苑』）、例として、『[信長公記](../Page/信長公記.md "wikilink")』には[豊臣秀吉](https://ja.wikipedia.org/wiki/豊臣秀吉 "wikilink")が[鳥取城](../Page/鳥取城.md "wikilink")に行ったことが記されているが（鳥取城も参照）、大大名の財源あって可能な戦術であり、逆に[商人](https://ja.wikipedia.org/wiki/商人 "wikilink")との交渉で兵糧米の買い入れに失敗した事例としては、[永禄](../Page/永禄.md "wikilink")7年（[1564年](../Page/1564年.md "wikilink")）に[国府台城](../Page/国府台城.md "wikilink")の[里見義弘](../Page/里見義弘.md "wikilink")・[太田康資](../Page/太田康資.md "wikilink")が商人との交渉で価格が折り合わず、[岩槻城](../Page/岩槻城.md "wikilink")向けの兵糧を調達できなかった話がある\[1\]。

## 近世

[豊臣政権](../Page/豊臣政権.md "wikilink")によって[兵農分離](../Page/兵農分離.md "wikilink")が進められると、[武士](../Page/武士.md "wikilink")が兵士としての役目を行うことが原則となるとともに兵糧携帯の義務が廃されて、代わりに兵糧の調達・運搬は[農民](../Page/農民.md "wikilink")ら領民の義務とされた。また、大名は戦時に備えてあらかじめ米や塩・[味噌](../Page/味噌.md "wikilink")などの調達・輸送計画を立案してこれに基づいた兵糧調達・購入が行われ、円滑な軍隊動員が行われるようになった。これと同時に現地における兵糧調達は原則として禁止されて濫妨や[刈田は](../Page/刈田狼藉.md "wikilink")[軍律](https://ja.wikipedia.org/wiki/軍律 "wikilink")によって厳しく禁じられることになった。

## 大日本帝国の陸軍給与令の兵食

兵営内で炊爨し、在営中の下士官兵およびその他特に定められた者に給される。 陸軍における兵食の給与量は、平時は主食として精米600g 、精麦186g を給し、副食物はその地方の物価その他の状況を顧慮して定められた定額を現在人員に対して部隊に交付し、該部隊において適宜調弁して炊爨調理のうえ給与し、演習あるいは特殊の労務に服する者にはこのほか増賄をなす。

| :---------: |
| **糧食および食料** |
|     日額      |
|     食糧      |
|     精米      |
|    600g     |
|    18銭8厘    |
|    18銭5厘    |
|    18銭2厘    |
|    野外増賄料    |
|     増賄料     |
|     夜食料     |

（上の表について）(1)地方区分は、別に定めがある。(2)各区内の賄料は、土地の状況または兵員の多少によって増減することがあるが、1人1日の平均額は、表の金額を超過しない。(3)賄料は、表の金額の範囲内において別に規定するところにより現品で交付することがある。

また拘禁中、留置懲罰中の者には減給の規定がある。 なお平時は上記糧食のかわりに乾パン、缶詰肉などを用いる場合がある。

|    醤油エキス    |

| **野戦食糧および加給品** |
|       区分       |
|       品種       |
|      野戦食糧      |
|       副食       |
|      野菜類       |

|      紙巻煙草      |

（上の表の野戦食糧について）(1)現地で調弁し得るときは、無骨生肉または卵をそれぞれ260g、骨付生肉を340gまで給することができる。 (2)パンを給する場合は、1食につき砂糖（またはジャム）を35gまで給することができる。 (3)現地調弁の野菜で製造した漬物は、1人1日の定量を100gまでとし、これに要する食塩は適宜使用することができる。 (4)酢、ソースは、醤油と同一割合で換給することができる。 (5)この表のほか所要の香辛料および脂油 を給することができる。 (6)特別の状況によって清水の給与を要するときは、飲料および調理用（洗浄その他雑用を含まない）をあわせ1人1日量4lを標準とする。 (7)給与上特別の必要のある場合にかぎりこの表の品種の一部に対し他の品種で換給することができる。その品種定量は戦地の最高等司令官の定めるところによる。

（上の表の加給品について）他の品種で換給する場合にはこの表の品種の価格を標準とする。

さらに状況に応じて一定の増額を行なうほか滞陣間、定量の一部を金額で支給することがある。 また非常の場合には携帯口糧で一時の飢えを凌ぐことになっている。

|            |
| :--------: |
| **野戦携帯口糧** |
|     品種     |
|     精米     |
|    乾パン     |
|    缶詰肉     |
|     食塩     |

（上の表について）缶詰肉は騎兵および騎兵隊と行動をともにする部隊の乗馬者は20匁とする。現地で調弁することができるときは野戦糧食の定量まで給することができる。

日本陸軍の身体健康な兵が中程度の兵業に従事した場合の1日の体内消費エネルギー量は平均2769カロリーであり、野外演習、戦闘教練などにおいては5000ないし7000カロリーとされた。 そして諸点を考えると、少なくとも兵1人1日の給与量は3100カロリー以上が必要であるとされた。

陸軍の平時定量は約3160カロリー、戦時定量は3643ないし3797カロリー、携帯口糧（乾パンの場合）は2639カロリーであった。

部隊の給与については、衛生部員ならびに経理委員は廉価で滋養豊富な食品を選択し、品質を毎日検査し、食品の配合ならびに調理法を考究し、時々各人の嗜好を調査考慮して食味の単調を避け、食欲を良好にして兵業に堪え得る立派な体力と健康を保持し、きわめて旺盛な士気を発揚させることに努めなければならないとされた。
