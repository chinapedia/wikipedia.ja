[World Wide Web Consortium](../Page/World_Wide_Web_Consortium.md "wikilink")（W3C）の**Web Accessibility Initiative**（**WAI**）は、標準のブラウザだけでなく様々なユーザーエージェントを用いて[World Wide Webにアクセスする人々のためにウェブ](../Page/World_Wide_Web.md "wikilink")[アクセシビリティ](../Page/アクセシビリティ.md "wikilink")を向上することを目的としている団体である。この組織は、補助デバイスを用いてウェブにアクセスしている、身体障害者にとって特に重要といえる。

W3CはWWWを発展することを目的に[1994年](../Page/1994年.md "wikilink")に設立された。W3Cは、ウェブの[相互運用性](https://ja.wikipedia.org/wiki/相互運用性 "wikilink")を確保するため、統一的な[プロトコルの開発に責任をもっている](../Page/通信プロトコル.md "wikilink")。この団体の一部であるWAIは、ウェブサイトを、特に身体に障害をもつ人々の観点から見てよりアクセス可能なものにするためのガイドラインを数多く制定してきた。

## ガイドライン

### Web Content Accessibility Guidelines（WCAG）

この*Web Content Accessibility Guideline 1.0* は WCAGとして知られており、[1999年](../Page/1999年.md "wikilink")に開発された。 WCAGは、特に障害のあるユーザー向けにコンテンツをアクセス可能にすることを目的に書かれたガイドラインであるが、[携帯電話](../Page/携帯電話.md "wikilink")のような制限されたデバイスを含むあらゆるユーザーエージェント向けにも書かれた。

2008年12月11日、WCAG 2.0が[W3C勧告](https://ja.wikipedia.org/wiki/W3C勧告 "wikilink")となった\[1\]。

### Authoring Tool Accessibility Guidelines (ATAG)

Authoring Tool Accessibility Guidelines Working Groupによって開発される。ATAG 1.0は[2000年](../Page/2000年.md "wikilink")2月3日に[W3C勧告](https://ja.wikipedia.org/wiki/W3C勧告 "wikilink")となった。 ATAGは、障害のあるユーザーがアクセス可能なオーサリングツール（[HTMLエディタ](https://ja.wikipedia.org/wiki/HTMLエディタ "wikilink")）の作成と、障害のあるユーザーがアクセス可能なページを作成するツールを作る際の、開発者向けガイドラインである。

### User Agent Accessibility Guidelines (UAAG)

User Agent Accessibility Guidelines Working Groupによって開発される。UAAG 1.0は[2002年](../Page/2002年.md "wikilink")12月17日にW3C勧告となった。UAAGは、[ウェブブラウザ](../Page/ウェブブラウザ.md "wikilink")や[メディアプレーヤー](../Page/メディアプレーヤー.md "wikilink")といったユーザーエージェントの開発者向けのガイドラインである。障害のあるユーザーがユーザーエージェントにアクセス出来るようにすることを目的としている。

### XML Accessibility Guidelines (XAG)

XAGは、アクセシビリティを促進するXMLアプリケーションに機能を含める方法を説明している。
