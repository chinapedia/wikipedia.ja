**式占**（しきせん）は占いの一種である。特徴は占うにあたって計算を行うときに、式盤（しきばん）あるいは**栻**（ちょく）と呼ばれる簡易な器具を使用するところにある。

## 式盤

[300px](https://ja.wikipedia.org/wiki/ファイル:Model_Si_Nan_of_Han_Dynasty.jpg "wikilink")」（漢代）六壬式の式盤との関連が推測される。\]\] [300px](https://ja.wikipedia.org/wiki/ファイル:Chokuban.svg "wikilink") 式盤は天地盤と呼ばれることもあり、天盤と呼ばれる円形の盤と地盤と呼ばれる方形の盤を組み合わせたものが基本形で、円形の天盤が回転する構造となっている。天盤や地盤の形状は、天は円く地は四角いとする中国の天地観に基づいている\[1\]。

天盤や地盤には[十干](../Page/十干.md "wikilink")、[十二支](../Page/十二支.md "wikilink")、といった占うために必要な文字や記号が記入されており、天盤の文字や記号を地盤のそれと合わせることで簡単な計算を行ったのと同じ効果が得られる。

式盤を作成するにあたって、材料は非常に限定されており、天盤では**[楓](../Page/楓.md "wikilink")**（ふう）\[2\]にできるコブである楓人（ふうじん）、地盤では雷に撃たれた**[棗](https://ja.wikipedia.org/wiki/棗 "wikilink")**の木が正しい材料とされている。正しい材料で正しく作成した式盤には呪力があるとされ、[調伏](https://ja.wikipedia.org/wiki/調伏 "wikilink")などの儀式で使用されることがある。

代表的な**式占**には、太乙式(「[太乙神数](https://ja.wikipedia.org/wiki/太乙神数 "wikilink")」)、遁甲式(「[奇門遁甲](https://ja.wikipedia.org/wiki/奇門遁甲 "wikilink")」)、六壬式(「[六壬神課](../Page/六壬神課.md "wikilink")」)があり、これらをまとめて[三式](../Page/三式.md "wikilink")と呼ぶ。なおこれら三式の他に名前だけが伝わっていて具体的な内容が不明の[雷公式](https://ja.wikipedia.org/wiki/雷公式 "wikilink")があり、三式に雷公式を加えて[四式](https://ja.wikipedia.org/wiki/四式 "wikilink")と呼ぶことがある。三式それぞれで異なる形態の「式盤」を使用する。六壬式の式盤では地盤の十二支に天盤の十二神や[十二天将](../Page/十二天将.md "wikilink")を合わせるので12の要素を基本とし、遁甲式の式盤は[八卦](../Page/八卦.md "wikilink")で表される8の要素、太乙式の式盤は、十二支に四隅の門を加えた16の要素を基本としている。

このうち六壬式は、[平安時代](../Page/平安時代.md "wikilink")から[鎌倉時代](../Page/鎌倉時代.md "wikilink")にかけて[陰陽師](../Page/陰陽師.md "wikilink")にとって必須の占術であった。陰陽師として名高い[安倍晴明](https://ja.wikipedia.org/wiki/安倍晴明 "wikilink")は子孫のために『[占事略决](https://ja.wikipedia.org/wiki/占事略决 "wikilink")』を残した。

[六壬式](https://ja.wikipedia.org/wiki/六壬式 "wikilink")の式盤では、地盤に十二支、十干、四隅の門<ref>天門、地戸、鬼門、人門の総称。これは八卦と対応付けられており、そのため方位や季節とも対応付けられている。

| 天門 | 乾  | 北西 | 秋から冬への移行期間 |
| 地戸 | 巽  | 南西 | 春から夏への移行期間 |
| 鬼門 | 艮  | 北東 | 冬から春への移行期間 |
| 人門 | 坤  | 南西 | 夏から秋への移行期間 |
|    |    |    |            |

</ref>、東西南北の四方の門が記入されており、これが[二十四山](https://ja.wikipedia.org/wiki/二十四山 "wikilink")の原型ではないかとする説がある。また六壬式盤の地盤中央に天盤の代わりに匙形の方位磁石を置いたものである『指南』が、後に[風水](../Page/風水.md "wikilink")で使用される[羅盤](https://ja.wikipedia.org/wiki/羅盤 "wikilink")の原型ではないかと推測されている\[3\]が、実際の『指南』をみれば頷ける説である。
