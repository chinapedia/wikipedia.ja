**eX-BOARD**（エクスボード）とは、株式会社[エクサム](https://ja.wikipedia.org/wiki/エクサム "wikilink")が発表したアーケードゲーム基板である。基板の設計及び、生産自体は韓国のINTERPARK GAMES社が行っている\[1\]。

## 概要

エクサムが[2008年](https://ja.wikipedia.org/wiki/2008年 "wikilink")[3月](https://ja.wikipedia.org/wiki/3月 "wikilink")の『[アルカナハート](https://ja.wikipedia.org/wiki/アルカナハート "wikilink")2』稼動と同時に発売した、アーケードゲーム用システム基板。

「オペレーター（操作者）とディベロッパー（製作者）を繋ぐ」というコンセプトのもと、開発自体がアミューズメント活性化を目標としており、扱いやすさを目指した仕様設計が徹底されている。その特徴として、カートリッジ形式を採用しており、アーケードゲーム用の筐体専用機材がなくても、[VGA出力を用いてモニターと繋ぎ](../Page/Video_Graphics_Array.md "wikilink")、[USBゲームパッドを繫げることでプレイ可能とするなど](../Page/ユニバーサル・シリアル・バス.md "wikilink")、高い汎用性を持つ。また、カートリッジ制の導入により、特殊なインストール作業が必要なくオペレーター側のセットアップが容易い仕様になり、ソフトの交換も容易に行うことが可能。プログラムに関しても、[Windows XP Embeddedをベースに基礎設計がなされており](../Page/Microsoft_Windows_XP.md "wikilink")、開発のための初期投資による経費削減が可能となっている。

eX-BOARDは上記の要素をアピールポイントとして発売されたが、発売時期に対して対応スペックの弱さは否めないものであり、特にグラフィック面においてはビデオゲームにおいても既にデジタル[HD映像出力への移行が進んでいた当時のアーケードシーンの流れを読み違えアナログSD映像出力にしか対応させなかったため](../Page/高精細度ビデオ.md "wikilink")、これらに対応していた[Taito Type X2や](https://ja.wikipedia.org/wiki/Taito_Type_X "wikilink")[RINGといった他社の競合プラットフォームの存在もありシェア獲得に苦戦](https://ja.wikipedia.org/wiki/RING_\(アーケードゲーム基板\) "wikilink")、日本国内におけるリリースタイトルも自社開発の3本のみ、他社タイトルは開発中止または2014年現在も稼働未定（実質的な白紙）の状態となっている。

## スペック

  - [CPU](../Page/CPU.md "wikilink")：Fanless Via C7 NanoBGA2
  - [Chipset](../Page/チップセット.md "wikilink")：Via CN700, VT8237R Plus
  - [Memory](../Page/主記憶装置.md "wikilink")：1GByte DDR2 SDRAM（VRAM共用,UMA64MBまで）
  - Video：Integrated S3 UniChrome Pro AGP graphics with MPEG-2 decoding acceleration (AGP 8x : UMAによる64MBまでのメモリサポート)
  - Audio：VIA VT1618Cidec 192KHz/32bit 8ch AC’97
  - Network：RJ-45 10/100Mbps EATHERNET
  - Video端子：[JAMMA出力](https://ja.wikipedia.org/wiki/日本アミューズメントマシン工業協会 "wikilink")・[D-Sub](../Page/D-subminiature.md "wikilink") 15pin×2・コンポジット端子・[S-Video端子](../Page/S端子.md "wikilink")（同時接続可）
  - Audio端子：JAMMA出力・L/R音声出力端子×2
  - Input：JAMMA入力（拡張ボタン1を ボタン4として利用）・追加拡張ボタンコネクタにより追加ボタンサポート
  - USB：USB2.0（2ポート）
  - パワー：12V 5A（JAMMA供給）・AC100 - 240V 50/60Hz 1.7A（専用電源アダプタ）
  - サイズ：W317.80mm×D250.00mm×H104.20mm
  - 重さ：4kg
  - [OS](../Page/オペレーティングシステム.md "wikilink")：Microsoft Windows XP Embedded
  - [Flash Disk](https://ja.wikipedia.org/wiki/ソリッドステートドライブ "wikilink")：512M・1G・2G
