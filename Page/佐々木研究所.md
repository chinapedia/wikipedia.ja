[杏雲堂病院_-_panoramio.jpg](https://ja.wikipedia.org/wiki/File:杏雲堂病院_-_panoramio.jpg "fig:杏雲堂病院_-_panoramio.jpg") **公益財団法人 佐々木研究所**（こうえきざいだんほうじん ささきけんきゅうじょ）は、[がん及び疾患の予防法](../Page/悪性腫瘍.md "wikilink")・診断法・治療法に関する臨床を中心とした調査及び研究開発の事業を実施している[公益法人](https://ja.wikipedia.org/wiki/公益法人 "wikilink")。事業所として[東京都](https://ja.wikipedia.org/wiki/東京都 "wikilink")[千代田区](https://ja.wikipedia.org/wiki/千代田区 "wikilink")[神田駿河台](https://ja.wikipedia.org/wiki/神田駿河台 "wikilink")に佐々木研究所、杏雲堂病院、[神奈川県](https://ja.wikipedia.org/wiki/神奈川県 "wikilink")[平塚市](../Page/平塚市.md "wikilink")に湘南健診センターを有している。

## 佐々木研究所

### 概要

東京都千代田区神田駿河台にある[公益財団法人](https://ja.wikipedia.org/wiki/公益財団法人 "wikilink")佐々木研究所の附属[研究所](../Page/研究所.md "wikilink")で、[基礎医学のうち主に](../Page/医学.md "wikilink")[病理学](../Page/病理学.md "wikilink")、[生化学](../Page/生化学.md "wikilink")の観点からがん分野の研究を行ってきた。現在、がんと共存しながらがんでは死なない社会の実現を目指し、がんを中心とした疾病に関する基礎研究を行っている。初代理事長兼所長の[佐々木隆興](../Page/佐々木隆興.md "wikilink")（[佐々木政吉](https://ja.wikipedia.org/wiki/佐々木政吉 "wikilink")の養嗣子）は、二度の[帝国学士院恩賜賞](https://ja.wikipedia.org/wiki/帝国学士院恩賜賞 "wikilink")及び[文化勲章](../Page/文化勲章.md "wikilink")を受章し、第二代所長の[吉田富三](https://ja.wikipedia.org/wiki/吉田富三 "wikilink")は、帝国学士院恩賜賞及び文化勲章を受章している。

### 沿革

|       |      |                                                     |
| ----- | ---- | --------------------------------------------------- |
| 明治15年 | 1882 | 佐々木東洋、神田駿河台に杏雲堂医院設立                                 |
| 明治27年 | 1894 | 第二代院長の佐々木政吉、自邸敷地に研究室を新築し結核の研究開始、佐々木研究所の母体となる        |
| 明治36年 | 1903 | 第三代院長に佐々木隆興、研究を開始                                   |
| 大正13年 | 1924 | 佐々木隆興、「細菌によるアミノ酸の分解に関する研究」で帝国学士院恩賜賞受賞               |
| 昭和11年 | 1936 | 佐々木隆興、吉田富三と共に「アゾ色素肝癌に関する研究」で、再度帝国学士院恩賜賞             |
| 昭和14年 | 1939 | 文部省所管財団法人「佐々木研究所」設立認可、佐々木隆興初代理事長兼研究所長               |
| 昭和15年 | 1940 | 佐々木隆興、文化勲章受章                                        |
| 昭和25年 | 1950 | 吉田富三、吉田肉腫・腹水型肝癌の研究開始                                |
| 昭和28年 | 1953 | 吉田富三、「吉田肉腫の病理学的研究」で、再度帝国学士院恩賜賞を受賞                   |
| 昭和34年 | 1959 | 吉田富三、文化勲章を受章                                        |
| 昭和41年 | 1966 | 第9回UICC国際癌会議、吉田富三会長、佐々木隆興名誉会長                       |
| 平成2年  | 1990 | 研究所棟新築落成、3研究部門（病理部・生化学部・細胞遺伝部）を設置                   |
| 平成18年 | 2006 | 臨床研究に特化、6臨床研究部門（内科系、外科系、予防医学系、看護学系、診療支援系、病院管理系）を設置  |
| 平成22年 | 2010 | 腫瘍ゲノム学系臨床研究部門を新設、7臨床研究部門(部長9名、主任研究員9名,全て杏雲堂病院兼任）となる |
| 平成29年 | 2017 | 5基礎研究部体制に改変．腫瘍ゲノム、分子代謝制御、腫瘍細胞の3研究部を設置               |
| 平成30年 | 2018 | ペプチドミクス研究部を設置                                       |
|       |      |                                                     |
