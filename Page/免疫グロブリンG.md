[IgG_molecular_surface.jpg](https://ja.wikipedia.org/wiki/File:IgG_molecular_surface.jpg "fig:IgG_molecular_surface.jpg") **免疫グロブリンG**（めんえきグロブリンG、Immunoglobulin G、IgG）は単量体型の[免疫グロブリンで](../Page/抗体.md "wikilink")、2つの重鎖γと2つの軽鎖からなっている。それぞれの複合体は2つずつの[抗原](../Page/抗原.md "wikilink")結合部位を持っている。免疫グロブリンの中では最も数の多いものである。ヒトの[血清](../Page/血清.md "wikilink")の免疫グロブリンの75%を占め\[1\]、体中の[血液](../Page/血液.md "wikilink")、[組織液](https://ja.wikipedia.org/wiki/組織液 "wikilink")に存在する。

[鳥類](../Page/鳥類.md "wikilink")のIgGはしばしばIgYと呼ばれ、血清と[卵黄](../Page/卵黄.md "wikilink")の中に見られる\[2\]。

## 機能

IgGはヒトの[胎盤](../Page/胎盤.md "wikilink")を通過できる唯一のアイソタイプであり、自分の[免疫系](../Page/免疫系.md "wikilink")を確立する生後1週間までの間、[胎児](../Page/胎児.md "wikilink")を守っている。

IgGは[ウイルス](../Page/ウイルス.md "wikilink")、[細菌](../Page/細菌.md "wikilink")、[真菌](https://ja.wikipedia.org/wiki/真菌 "wikilink")など様々な種類の[病原体](https://ja.wikipedia.org/wiki/病原体 "wikilink")と結合し、[補体](../Page/補体.md "wikilink")、[オプソニン](https://ja.wikipedia.org/wiki/オプソニン "wikilink")による[食作用](https://ja.wikipedia.org/wiki/食作用 "wikilink")、毒素の中和などによって生体を守っている。

特異的IgGは[食物アレルギー](../Page/食物アレルギー.md "wikilink")でない人にも存在し、[食物アレルギー](../Page/食物アレルギー.md "wikilink")の診断における有用性は確立されていない

## サブクラス

ヒトの場合

|        |        |           |            |                  |
| ------ | ------ | --------- | ---------- | ---------------- |
| **種類** | **割合** | **胎盤の通過** | **補体の活性化** | **食細胞のFc部位への結合** |
| IgG1   | 66%    | する        | 2番目に高い     | 高い               |
| IgG2   | 23%    | しない       | 3番目に高い     | 極めて低い            |
| IgG3   | 7%     | する        | 最も高い       | 高い               |
| IgG4   | 4%     | する        | しない        | 中程度              |

動物種によってサブクラスの数は異なる。
