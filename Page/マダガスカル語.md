**マダガスカル語**（**Malagasy**、マラガシ）はアフリカ東部のインド洋に浮かぶマダガスカル島で話される[言語](../Page/言語.md "wikilink")。[マダガスカル](../Page/マダガスカル.md "wikilink")共和国の国語であると共に、[フランス語](https://ja.wikipedia.org/wiki/フランス語 "wikilink")と並んで同共和国の[公用語](../Page/公用語.md "wikilink")となっている。

マダガスカル語は[オーストロネシア語族](../Page/オーストロネシア語族.md "wikilink")[マレー・ポリネシア語派](../Page/マレー・ポリネシア語派.md "wikilink")に所属する。紀元5世紀頃に[マレー諸島](../Page/マレー諸島.md "wikilink")から[インド洋](../Page/インド洋.md "wikilink")を越えて移住した結果とされる。この移住について歴史上の記録は一切ないので詳細は不明だが、[言語学](../Page/言語学.md "wikilink")的には、（[ダヤク族](https://ja.wikipedia.org/wiki/ダヤク族 "wikilink")の言語）に含まれ、[ボルネオ島](../Page/ボルネオ島.md "wikilink")のに近いとされる。

| マダガスカル語     | マアンヤン語（バリト諸語） |
| ----------- | ------------- |
| vatana"体"   | watang        |
| elatra"羽"   | elat          |
| vitsika"アリ" | wisik         |
| hena"肉"     | kenah"魚"      |
| afo"火"      | apui          |
| rano"水"     | ranu          |
|             |               |

その後、アフリカ大陸海岸部から来たバンツー族や交易目的で来たアラブ人などと[混血](../Page/混血.md "wikilink")しているため、[オーストロネシア語](https://ja.wikipedia.org/wiki/オーストロネシア語 "wikilink")を基調としつつもバンツー語系の単語、アラビア語系の単語、サンスクリット語系の単語が混じる。

## 音韻

/a/、/e/、/i/、/u/の4つの[母音](../Page/母音.md "wikilink")がある。つづり字のoは通常/u/と読まれる。

## 文法

語順は、[能動態](https://ja.wikipedia.org/wiki/能動態 "wikilink")（行為者焦点）の場合、[VOS](https://ja.wikipedia.org/wiki/VOS "wikilink")（動詞―目的語―主語）である。

文例　Mamaky boky aho. 「読む＋本＋私」

この能動態文例の場合、主格を文頭に置くことも可能である。

文例　Izaho dia mamaky boky.　「私＋は＋読む＋本」　主格の aho　が文頭に置かれた場合には、izaho　となる。

主格を強調する、能動態の場合には、下記の語順になる。

文例　Izaho no mamaky boky.　「私＋が＋読む＋本」

同じ文例を、[受動態](../Page/受動態.md "wikilink")（対象焦点）構文にすると、下記のようになる。

文例　Vakiko ny boky.　「私によって読まれる＋本」

マダガスカル語には、[相（aspect）](../Page/相_\(言語学\).md "wikilink")（完了相・未完了相・指向相：それぞれ過去形・現在形・未来形ということもある）があり、時・場所・手段などを強調する構文（焦点という：行為者焦点・対象焦点・場所焦点・事情焦点）をとる。動詞は3つの相と4つの焦点に応じて形が変わる。

文例　Tsy azo anariana fako eto.　「否定辞＋できる＋捨てられる＋ゴミ＋ここ」→　「ここに、ゴミを捨てることはできません」　最後尾の　「ここ」（eto）と言う場所が強調される構文である。
