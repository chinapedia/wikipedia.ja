**千葉県道268号北柏停車場線**（ちばけんどう268ごう きたかしわていしゃじょうせん）は、[千葉県](../Page/千葉県.md "wikilink")[柏市](https://ja.wikipedia.org/wiki/柏市 "wikilink")の[北柏駅](https://ja.wikipedia.org/wiki/北柏駅 "wikilink")から[国道6号](../Page/国道6号.md "wikilink")に至る[道路](../Page/道路.md "wikilink")（[一般県道](../Page/都道府県道.md "wikilink")）である。実延長はおよそ500m。終点付近の[常磐線](https://ja.wikipedia.org/wiki/常磐線 "wikilink")ガード下は、集中豪雨があると冠水するため注意が必要。 [250px](https://ja.wikipedia.org/wiki/ファイル:Chiba_Route_268_in_Kita-Kashiwa.JPG "wikilink")

## 起点・終点

  - 起点：[千葉県](../Page/千葉県.md "wikilink")[柏市](https://ja.wikipedia.org/wiki/柏市 "wikilink")北柏一丁目 北柏停車場（[北柏駅](https://ja.wikipedia.org/wiki/北柏駅 "wikilink")）
  - 終点：千葉県柏市呼塚新田 [国道6号](../Page/国道6号.md "wikilink")との交差点

## 路線状況

  - [平成22年](https://ja.wikipedia.org/wiki/平成22年 "wikilink")（[2010年](https://ja.wikipedia.org/wiki/2010年 "wikilink")）度時点での北柏停車場線の交通量についてはそれぞれ以下の通りになっている\[1\]。

|                   |                 |
| ----------------- | --------------- |
| **昼間12時間自動車類交通量** | **24時間自動車類交通量** |
| style= |小型車       | style= |大型車     |
| 3861台             | 410台            |

ちなみに[平成17年](https://ja.wikipedia.org/wiki/平成17年 "wikilink")（[2005年](../Page/2005年.md "wikilink")）時の昼夜12時間と24時間の自動車類交通量の合計ではそれぞれ4494台、6157台だった。
幅員構成について道路部幅員25.00m、車道部幅員17.00m、車道幅員15.00m、歩道幅員は上りと下り共に4.00m。車線数は4車線（片側2車線）である。
