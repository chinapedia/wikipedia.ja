『**NNNニュースフラッシュ**』（エヌエヌエヌニュースフラッシュ）は、[1956年](../Page/1956年.md "wikilink")4月から[1974年](../Page/1974年.md "wikilink")3月まで[日本テレビ系列局で放送された夕方の](../Page/日本テレビ放送網.md "wikilink")[ニュース番組である](../Page/報道番組.md "wikilink")。

| 期間      | 放送時間（[日本時間](../Page/日本標準時.md "wikilink")） | 備考                         |
| ------- | ----------------------------------------- | -------------------------- |
| 1956.04 | 1956.09                                   | 月曜 - 土曜 19:05 - 19:15（10分） |
| 1956.10 | 1958.09                                   | 月曜 - 土曜 19:00 - 19:10（10分） |
| 1958.10 | 1973.09                                   | 月曜 - 土曜 18:45 - 18:55（10分） |
| 1973.10 | 1974.03                                   | 月曜 - 土曜 18:45 - 19:00（15分） |

（参考：日本テレビ社史「大衆とともに25年・沿革史」427頁 - 497頁 1978年）
