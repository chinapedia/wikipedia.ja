『**COMPLETE SET Limited Box**』（コンプリート・セット・リミテッド・ボックス）は、[東方神起](https://ja.wikipedia.org/wiki/東方神起 "wikilink")の[CD-BOX](https://ja.wikipedia.org/wiki/CD-BOX "wikilink")。[2010年](https://ja.wikipedia.org/wiki/2010年 "wikilink")[6月30日](../Page/6月30日.md "wikilink")に[rhythm zoneより発売された](https://ja.wikipedia.org/wiki/rhythm_zone "wikilink")。

## 概要

同月に発売された[ベスト・アルバム](../Page/ベスト・アルバム.md "wikilink")『[COMPLETE -SINGLE A-SIDE COLLECTION-](https://ja.wikipedia.org/wiki/COMPLETE_-SINGLE_A-SIDE_COLLECTION- "wikilink")』『[SINGLE B-SIDE COLLECTION](https://ja.wikipedia.org/wiki/SINGLE_B-SIDE_COLLECTION "wikilink")』の2作を同梱したボックス・セット。

プレスリリース時の仮タイトルは「**COMPLETE＋B-Side Collection BOX SET**」\[1\]。

初回生産限定盤には、1000ピースの[ジグソーパズル](../Page/ジグソーパズル.md "wikilink")が特典として封入されている\[2\]。
