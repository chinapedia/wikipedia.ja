**製造原価**（せいぞうげんか、manufacturing cost）とは、[工場](../Page/工場.md "wikilink")で[製品](../Page/製品.md "wikilink")の製造に要した[費用](https://ja.wikipedia.org/wiki/費用 "wikilink")である。

## 概要

製造原価は、当期に完成した製品に対する[原価](https://ja.wikipedia.org/wiki/原価 "wikilink")なので、次式のように算出される。

  -
    **当期の製造原価** ＝ **期首[仕掛品](https://ja.wikipedia.org/wiki/仕掛品 "wikilink")棚卸高** ＋ **当期製造費用** － **期末仕掛品[棚卸高](https://ja.wikipedia.org/wiki/棚卸資産 "wikilink")**

製造費用の内訳は「原価の三大要素」と呼ばれる、以下の3つに区分される。（「形態別分類」）

| 分類                                                  | 説明                                                                | 内訳                                                                                                                                                                                                                                                                                                                     |
| --------------------------------------------------- | ----------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [材料費](https://ja.wikipedia.org/wiki/材料費 "wikilink") | 物品の[消費](../Page/消費.md "wikilink")によって生じる原価要素                      | [原料費](https://ja.wikipedia.org/wiki/原材料 "wikilink")、買入[部品](https://ja.wikipedia.org/wiki/部品 "wikilink")費、[燃料](../Page/燃料.md "wikilink")費、工場[消耗品](https://ja.wikipedia.org/wiki/消耗品 "wikilink")費、消耗[工具](../Page/工具.md "wikilink")・[器具](https://ja.wikipedia.org/wiki/器具 "wikilink")・備品費など                               |
| [労務費](https://ja.wikipedia.org/wiki/労務費 "wikilink") | [労働力](https://ja.wikipedia.org/wiki/労働力 "wikilink")の消費によって生じる原価要素 | [従業員](https://ja.wikipedia.org/wiki/従業員 "wikilink")に対する[賃金](https://ja.wikipedia.org/wiki/賃金 "wikilink")、[給料](https://ja.wikipedia.org/wiki/給与 "wikilink")、雑給、[賞与](https://ja.wikipedia.org/wiki/賞与 "wikilink")、[手当](../Page/手当_\(給与\).md "wikilink")、[退職給付引当金](https://ja.wikipedia.org/wiki/退職給付引当金 "wikilink")繰入額など |
| [経費](https://ja.wikipedia.org/wiki/経費 "wikilink")   | 材料費と労務費以外の原価要素                                                    | [減価償却](../Page/減価償却.md "wikilink")費、[水道光熱費](https://ja.wikipedia.org/wiki/光熱費#事業所における水道光熱費 "wikilink")、[賃貸料](https://ja.wikipedia.org/wiki/賃料 "wikilink")、[外注](https://ja.wikipedia.org/wiki/外注 "wikilink")加工費、[ロイヤルティー](https://ja.wikipedia.org/wiki/ロイヤルティー "wikilink")など                                           |

**原価の三要素**

さらに、各要素は製品との関連により、**直接費**と**間接費**に区分される。直接費は製品に**賦課**（**直課**）されるのに対し、間接費は一定の配賦基準に従って、各製品に**配賦**する必要がある。（[原価計算](https://ja.wikipedia.org/wiki/原価計算 "wikilink")を参考）

|       |          |       |       |
| ----- | -------- | ----- | ----- |
| 総原価   | **製造原価** | 製造直接費 | 直接材料費 |
| 直接労務費 |          |       |       |
| 直接経費  |          |       |       |
| 製造間接費 | 間接材料費    |       |       |
| 間接労務費 |          |       |       |
| 間接経費  |          |       |       |
| 営業費   | 販売費      |       |       |
| 一般管理費 |          |       |       |
