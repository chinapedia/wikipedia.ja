**Consolas**は、によって設計された[等幅フォント](../Page/等幅フォント.md "wikilink")。[Microsoft](https://ja.wikipedia.org/wiki/Microsoft "wikilink")の[アンチエイリアス](../Page/アンチエイリアス.md "wikilink")技術[ClearType](../Page/ClearType.md "wikilink")を活用したフォントのひとつである。[Windows Vista](https://ja.wikipedia.org/wiki/Windows_Vista "wikilink")、[Office 2007](https://ja.wikipedia.org/wiki/Office_2007 "wikilink")、[Visual Studio 2010以降は](https://ja.wikipedia.org/wiki/Visual_Studio_2010 "wikilink")[Windows](https://ja.wikipedia.org/wiki/Windows "wikilink")に含まれており、Microsoftからダウンロードが可能となっている。また、Windows Vistaの標準のフォントで唯一、[斜線付きゼロ](https://ja.wikipedia.org/wiki/斜線付きゼロ "wikilink")を使用したフォントでもある。

## 特徴

Consolasは[OpenType](../Page/OpenType.md "wikilink")のレイアウト機能の、[異体字](https://ja.wikipedia.org/wiki/異体字 "wikilink")、字形の現地化、 伝統的配置の数字、 分数、[上付き文字](../Page/上付き文字.md "wikilink")、[下付き文字](../Page/下付き文字.md "wikilink")など をサポートしている。

の代替フォントとして設計されたが、Courier Newのバージョン2.90がサポートしていた1318[グリフ](https://ja.wikipedia.org/wiki/グリフ "wikilink")に対して、最初から利用可能だったのは713グリフのみだった。それから徐々に拡大し、バージョン5.32になるとサポートされているグリフの総数は2735個に増えている。

## 使用

、、、、とともに、[Microsoft Excel Viewer](https://ja.wikipedia.org/wiki/Microsoft_Excel "wikilink")、[Microsoft PowerPoint Viewer](../Page/Microsoft_PowerPoint.md "wikilink")、Windows用[Microsoft Office](../Page/Microsoft_Office.md "wikilink") Compatibility Pack、Mac用Open XML File Format Converterにおいても配布されている。

また、からのライレンスによっても使用することができる。

[オープンソース](../Page/オープンソース.md "wikilink")の代替フォントは、[Google Fontsで入手することができる](https://ja.wikipedia.org/wiki/Google_Fonts "wikilink")\[1\]。
