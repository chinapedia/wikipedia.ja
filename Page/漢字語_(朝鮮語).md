[朝鮮語](../Page/朝鮮語.md "wikilink")における**漢字語**（かんじご、、）は、語彙部類の一種で、[朝鮮漢字音](../Page/朝鮮漢字音.md "wikilink")で発音される単語を言う。[漢語系語彙](https://ja.wikipedia.org/wiki/漢語系語彙 "wikilink")（）の一種であり、[日本語](../Page/日本語.md "wikilink")[語種](../Page/語種.md "wikilink")の上で単に「[漢語](../Page/漢語.md "wikilink")」と総称される語彙体系がこれに対応する。

## 概要

### 歴史

朝鮮は日本と同じく[漢字文化圏](../Page/漢字文化圏.md "wikilink")に属し、古くから陸続きの中国より[漢字](../Page/漢字.md "wikilink")文化を受容してきた。[漢四郡](../Page/漢四郡.md "wikilink")が置かれた紀元前2世紀頃にはすでに漢字文化が朝鮮半島にもたらされたと考えられている。

漢字語は時代によって盛衰があり、語彙に異なりを見せる。例えば[中期朝鮮語](../Page/中期朝鮮語.md "wikilink")においては現代朝鮮語に見られない以下のような漢字語やその意味が見られる。

  - 常談 「日常会話」
  - 禮數 「お辞儀」
  - 爲頭 「始め」
  - 一定 「定めること」（現代語では「一定」の意）

中期朝鮮語においてはまた、当時の中国語の話し言葉から直接借用されたと見られる語彙も散見される。それらは通常、漢字表記されない。

  - /j@gia\[1\] 「ご自身」 ＜ 自家

  - 「（よい）食事」 ＜ 茶飯

  - 「靴」 ＜ 靴

### 使用

朝鮮語の語彙には[固有語](../Page/固有語.md "wikilink")・漢字語・[外来語](../Page/外来語.md "wikilink")とそれらの混種語があるが、漢字語は最も大きな比重を占めている。[ハングル学会](../Page/ハングル学会.md "wikilink")の編纂した『（大辞典）』の総見出し語164125語のうち漢字語は85527語（52.1%）を占めているという\[2\]。朝鮮語における漢字語は日本語の漢語と同じく教養語彙としての性格が強く、抽象概念や専門用語などに漢字語が多い。従って、日常語彙としては[固有語](../Page/固有語.md "wikilink")を用いつつも、改まった場では漢字語を用いることがある。

  -  「あたま」（固有語） ―   「頭部」（漢字語）

漢字語は日本語の[漢語](../Page/漢語.md "wikilink")と共通する語彙がすこぶる多いが、これらの語彙の「硬さ」（すなわち日常語化の程度）は必ずしも同じではない。例えば、日本語で「徹夜」は「徹夜する」など日常語として用いるが、朝鮮語の「徹夜（）」 は「徹夜作業」など一部の漢字語とともにのみ用いる非日常語である。その反対に日本語の「答弁」は「国会答弁」など公的な場での質問に対する答えという意味で用いる非日常語であるが、朝鮮語の「答辯（）」 は広く質問に対する答えという意味で用いる日常語である。

[大韓民国](../Page/大韓民国.md "wikilink")（以下「韓国」）では近年、既存の漢字語において意味のずれの生じているものがいくつか見受けられる。例えば、「飲料水（）」 は清涼飲料水の意味で用いられる場合が多く、さらにはコーヒーやジュースなどを含めたソフトドリンク一般を指すこともある。また、「水族館（）」 は水生動物の展示施設の他に、魚類を水槽に入れて販売する商店を指すようにもなってきている。

### 表記

漢字語は[漢字でも表記することができ](https://ja.wikipedia.org/wiki/朝鮮漢字 "wikilink")、[ハングル](https://ja.wikipedia.org/wiki/ハングル "wikilink")でも表記することができる。朝鮮語の表記法は日本語のそれと異なり、漢字表記が可能なのは漢字語だけであり、固有語・外来語は漢字表記されることが決してない。しかしながら、[朝鮮民主主義人民共和国](../Page/朝鮮民主主義人民共和国.md "wikilink")（以下「北朝鮮」）では建国早々（1948年）に漢字表記を全廃したので、漢字語であっても実際には全てハングルで表記される。韓国では以前は漢字・ハングル混じりの表記が多く見られたが、現在はハングルのみの表記が主流である。

### 造語

朝鮮語において漢字語は基本的に[名詞](../Page/名詞.md "wikilink")であり、そのままの形では[動詞](../Page/動詞.md "wikilink")や[形容詞](../Page/形容詞.md "wikilink")などの[用言](https://ja.wikipedia.org/wiki/用言 "wikilink")になりえない。漢字語を用言化するには、一般的には「する」に当たる[接尾辞](../Page/接尾辞.md "wikilink")「」  を付ける。「」が動作的な意味の漢字語に付けば動詞となり、状態的な意味の漢字語に付けば形容詞となる。

  - 使用（）  「使用する」
  - 寬大（）  「寛大だ」

一部の漢字語はそのままの形で[副詞](../Page/副詞.md "wikilink")（または[接続詞](../Page/接続詞.md "wikilink")）として機能する。

  - 恒常（）  「いつも」
  - 或是（）  「ひょっとして」

## 漢字語の種類

漢字語はその来歴により、中国起源のもの、朝鮮起源のもの、日本起源のものの大きく3種に分けられる。しかしながら、個々の語彙の起源については不明瞭な場合も多い。

### 中国起源の漢字語

#### 古典中国語に由来するもの

[中国語](https://ja.wikipedia.org/wiki/中国語 "wikilink")に由来する語彙は、漢字語の中で最も中心的な位置を占めている。その多くは古典中国語に由来するものであり、現代の朝鮮語・[中国語](https://ja.wikipedia.org/wiki/中国語 "wikilink")・[日本語](../Page/日本語.md "wikilink")に共通する語彙が多い。

  - 天下（）  「天下」
  - 結婚（）  「結婚」
  - 愉快（）  「愉快」
  - 學問（）  「学問」
  - 四面楚歌（）  「四面楚歌」
  - 漁夫之利（）  「漁夫の利」

故事成語の中には途中に語尾など朝鮮語の[固有語](../Page/固有語.md "wikilink")（日本語の[和語](https://ja.wikipedia.org/wiki/和語 "wikilink")に相当する）の要素が入り込む場合もある（下例のうち下線部が固有語）。

  - 百聞 不如一見（）  「百聞は一見に如かず」（逐語訳：百聞<span style="text-decoration:underline;">が</span>不如一見）

あるものは単語を構成する漢字の順序が異なっている。

  - 賢母良妻（）  「良妻賢母」

漢字1字から成る漢字語の中には、漢字語であるという意識が希薄で、あたかも固有語のような感覚で用いられているものがある。ちょうど日本語の「本（ホン）」などの単語が漢語だという意識が希薄であるのと似ている。

  - 山（）  「山」
  - 江（）  「川」
  - 冊（）  「本」

#### 語義が変化したもの

中国語として存在する語彙であっても、朝鮮語において意味変化を起こしたものがある。このような語彙は中国語とも日本語とも異なった意味で用いられる場合が多い。

  - 去來（）  「取引」
  - 工夫（）  「勉強」
  - 放心（）  「油断」
  - 複道（）  「廊下」
  - 砂糖（）  「飴」
  - 是非（）  「いさかい」
  - 議論（）  「相談」
  - 人事（）  「あいさつ」

「放心」は中国語では「安心する」という肯定的な意味でも用いられるが、朝鮮語では通常否定的な「油断」の意味も用い、肯定的な「安心」は「安心（）」を用いる。「複道」はもともと上下二重の廊下を指すものが、広く廊下一般を指すように変わったと見られる。「砂糖」は[中期朝鮮語](../Page/中期朝鮮語.md "wikilink")では砂糖の意で用いられていた。「是非」は中国語でも「いさかい」の意味に転じている。「人事」は「人の行なうべき事」の意であったものが、朝鮮語において意味がずれたものである。

#### 現代中国語に由来するもの

現代中国語に由来する語彙は少ない。ただし、北朝鮮で用いられる朝鮮語の中には、現代中国語に由来すると見られる語彙がいくつか見られる。

  - 一邊倒（）  「一辺倒」
  - 圓珠筆（）  「ボールペン」（北朝鮮の言語）

なお、現代中国語に由来する語彙は中国語音のまま導入され、外来語として定着したものが多い。

  -  「ジャージャー麺」 ＜ 炸醬麵 zhájiàngmiàn

  -  「ロートル」 ＜ 老頭兒 lǎotóur

### 朝鮮起源の漢字語

朝鮮において作られた漢字語にはいくつかの類型がある。

#### 新造語

漢字を組み合わせて朝鮮で独自に作り出した語彙である。このようなものは中国語や日本語に同一の漢字を用いた単語自体が存在しない。

| 朝鮮語     | 日本語      | 中国語                       |
| ------- | -------- | ------------------------- |
| 矯導所（）   | 刑務所      | 監獄 jiānyù                 |
| 名銜（）    | 名刺       | 名片 míngpiàn               |
| 膳物（）    | 《贈物》     | 礼物 lǐwù                   |
| 手票（）    | 《小切手》    | 支票 zhīpiào                |
| 藥房（）    | 《薬屋》     | 藥店 yàodiàn　または　藥房 yàofáng |
| 洋襪（）    | 《靴下》     | 襪子 wàzi                   |
| 掌匣（）    | 《手袋》     | 手套 shǒutào                |
| 次例（）    | 順番       | 次第 cìdì                   |
| 便紙/片紙（） | 《手紙》     | 信 xìn                     |
| 注油所（）   | ガソリンスタンド | 加油站 jiāyóuzhàn            |
| 便宜店（）   | コンビニ     | 便利店 biànlìdiàn            |
| 外界人（）   | 宇宙人      | 外星人 wàixīngrén            |
| 出市（）    | 発売       | 上市 shàngshì               |

  -
    <span style="font-size:90%;">日本語の《  》表記は漢語でなく和語であることを表す。便宜上、中国語は日本の常用漢字体で表記する。</span>

「房」は「部屋」の意であるが、朝鮮語では「店舗」の意の[接尾辞](../Page/接尾辞.md "wikilink")として用いる。「襪」は「足袋（たび）」の意で「洋襪」は「西洋の足袋」を意味する。「掌甲」は「掌匣」とも表記される（「匣」は「はこ」の意）。「次例」は本来「次第」であったが、「第」の音が本来の「」 から「」 に変化したために、それに合わせて漢字を当て直したものである。

朝鮮で作られた語彙の中には四字熟語もある。「賊反荷杖（）」  は盗人猛々しいことを表すが、これを漢文の読み下し文にすると「賊、反（かえ）って杖を荷う」となる。\[3\]

#### 朝鮮製漢字を含むもの

  - 田（）  「田畑」

  - 地（）  「敷地」

  - 宅（）  「夫の家」

「」は「水田」の意、「」は「場所」の意、「」は「夫の家」の意の朝鮮製漢字（国字）である。\[4\]

### 日本起源の漢字語

近代以降とりわけ日本統治時代に日本語から多くの語彙が漢字語として朝鮮語に流入した。日本語に由来する漢字語にもいくつかの類型がある。

#### 和製漢語

近代西欧概念などの翻訳語彙としての[和製漢語](../Page/和製漢語.md "wikilink")は抽象概念を表す語彙から具体的な物体などを表すものまで広範囲にわたっており、その数は枚挙にいとまがない。

  - 個人（）  「個人」
  - 主權（）  「主権」
  - 義務（）  「義務」
  - 鉛筆（）  「鉛筆」
  - 電話（）  「電話」
  - 博物館（）  「博物館」

これらの語彙のいくらかは中国に逆輸出され中国語とも共通している。しかし、中国語で別途に語彙を作った場合は、朝鮮語と日本語では共通しているが中国語とは共通しない。

| 日本語・朝鮮語 | 中国語         |
| ------- | ----------- |
| 飛行機（ ）  | 飛機 fēijī    |
| 汽車（ ）   | 火車 huǒchē   |
| 野球（ ）   | 棒球 bàngqiú  |
| 会社（ ）   | 公司 gōngsī   |
| 写真（ ）   | 照片 zhàopiàn |

なお、日本において「[同音の漢字による書きかえ](../Page/同音の漢字による書きかえ.md "wikilink")」の規定により、一部の和製漢語の表記は変更されたが、朝鮮語では旧表記の漢字による読みが維持される。

| 朝鮮語    | 日本語 |
| ------ | --- |
| 弘報（ ）  | 広報  |
| 刺戟（ ）  | 刺激  |
| 手帖（ ）  | 手帳  |
| 慰藉料（ ） | 慰謝料 |
| 破毀（ ）  | 破棄  |
| 洗滌（ ）  | 洗浄  |

近代以前の和製漢語で朝鮮語にもたらされたものは多くない。ただし、これらの語彙も日本統治時代にもたらされたものと見られる。

  - 十八番（）  「十八番」

また、日本語の漢語と意味の異なるものもある。「八方美人（）」 は朝鮮語ではあらゆることに才を発揮する人を指し、肯定的な意味で用いられる。

#### 日本語における訓読み熟語

日本語における訓読み熟語とは、[和語](https://ja.wikipedia.org/wiki/和語 "wikilink")に漢字を当てて表記したものである。この語彙が朝鮮語に導入されたとき、朝鮮語において「訓読み」（[固有語](../Page/固有語.md "wikilink")に漢字を当てて表記する方法）がなく「音読み」しかないため、表記された漢字をそのまま[朝鮮漢字音](../Page/朝鮮漢字音.md "wikilink")で発音した。このようにして多くの漢字表記された和語が漢字語として朝鮮語に導入されることとなった。朝鮮語にもたらされた訓読み熟語の中には、[重箱読み](../Page/重箱読み.md "wikilink")・[湯桶読み](https://ja.wikipedia.org/wiki/湯桶読み "wikilink")のものも含まれる。

  - 建物（）  「建物」
  - 賣上（）  「売り上げ」
  - 船着場（）  「船着場」
  - 不渡（）  「不渡り」〔重箱読み〕
  - 待合室（）  「待合室」〔湯桶読み〕

また、動作的な意味の単語は後ろに「する」に当たる接尾辞「」  を付けることで動詞化される。以下の例は後ろに「」が付きうる単語である。

  - （）  「貸し出し」

  - （）  「上回ること」

  - （）  「組み立て」

  - （）  「取扱い」

  - （）  「割引」

## 漢字語に由来する固有語

朝鮮語の[固有語](../Page/固有語.md "wikilink")の中には、漢字語が固有語化したものがある。

  -  「貧乏」 ＜ 艱難（）

  -  「じゃがいも」 ＜ 甘藷（）

  -  「唐辛子」 ＜ 苦椒（）

  -  「キムチ」 ＜ 沈菜（）

  -  「女官」 ＜ 內人（）

  -  「のち」 ＜ 乃終（）

  -  「大凡」 ＜ 大總（）

  -  「宝」 ＜ 宝貝（）

  -  「狩り」 ＜ 山行（）

  -  「マッチ」 ＜ 石硫黄（）

  -  「そり」 ＜ 雪馬（）

  -  「敷布団」 ＜ 褥（）

  -  「才能」 ＜ 才調（）

  -  「ミミズ」 ＜ 地龍이（）

  -  「獣」 ＜ 衆生（）

  -  「雷」 ＜ 天動（）

  -  「蝋燭」 ＜ 燭（）

固有語の中には音形が漢字語的であるため、漢字語に由来するのではないかと疑われるものがある。それらに対しては漢字が当てられることがあるが、歴史的に文献上から確認できるものではない。以下の例において、（  ）内の漢字が当て字である。

  -  「見物」 （求景）

  -  「思い」 （生覺）

  -  「引き出し」 （舌盒 ）

  -  「いたずら」 （作亂 ）

  -  「愛」 （思量, ）

また、北朝鮮においては、頭音法則との関係から、次のような語彙についても、固有語化した漢字語と見なされる。

  - 유리「ガラス」（琉璃）
  - 나사「ネジ」（螺絲）

なお、従前は、남색「青色」（藍色）も固有語化した漢字語として扱われていたが、現在では、通常の漢字語（람색）として扱われている。
