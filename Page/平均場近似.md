**平均場近似**（へいきんばきんじ、）とは[多体問題](../Page/多体問題.md "wikilink")を一体問題に帰着させる近似のことである。

名称に関しては、元々は**分子場近似**（molecular field approximation）と呼ばれていたが、現在では平均場近似と呼ばれることの方が多い。これとほぼ同義な呼び方として、[一体近似](https://ja.wikipedia.org/wiki/一体近似 "wikilink")、有効媒質近似（effective medium approximation）などがある。

## 詳細

平均場近似は多体系を扱う場合、その多体の相互作用をまともに解くことが通常非常に困難であることから、多体の効果をある平均的なもの（**平均場**）とみなす近似である。そして、これを解くための[セルフコンシステント](https://ja.wikipedia.org/wiki/セルフコンシステント "wikilink")（自己無撞着）な方程式を導き出し、これを解くことにより求めるべき解が得られる。

[強磁性](../Page/強磁性.md "wikilink")に関するワイス理論（[1907年](../Page/1907年.md "wikilink")）が、この近似（これは分子場近似と言われた）が使われた最も初期のもの。その後、平均場近似（分子場近似）を用いたものとして、ブラッグ‐ウィリアムス近似（二元合金の規則・不規則問題）、[ハートリー近似](https://ja.wikipedia.org/wiki/ハートリー近似 "wikilink")（ハートリー‐フォック近似）などがある。[バンド計算](../Page/バンド計算.md "wikilink")での[一電子近似](https://ja.wikipedia.org/wiki/一電子近似 "wikilink")も平均場近似の一つである。

当然、ゆらぎ（平均からのずれ）の大きな系には、この近似は成立しなくなる。ゆらぎの大きな場合としては、電子相関の強い[強相関電子系](../Page/強相関電子系.md "wikilink")などが挙げられる。

## 簡単な例

[イジング模型](https://ja.wikipedia.org/wiki/イジング模型 "wikilink")におけるスピン系を考え、相互作用は最近接格子間にのみ働くとして、その[ハミルトニアン](../Page/ハミルトニアン.md "wikilink")は、

\[H = - \sum_{\left\langle i,j \right\rangle} J S_i S_j\]

となる。Jは[交換相互作用](https://ja.wikipedia.org/wiki/交換相互作用 "wikilink")のエネルギー。ここで、\(\left\langle i,j \right\rangle\)は最近接格子間のみ足しあげることを意味する。この時、添え字jに関しての和の部分を平均化すると、

\[- \sum_{\left\langle i,j \right\rangle} J S_i S_j = - \sum_i S_i \sum_j J S_j \simeq - \sum_i S_i J \sum_j \left\langle S_j \right\rangle\]

となる。ここで、添え字i,jに関しての和がそれぞれに分かれた形になっても最近接格子間のみの和であることに変わりない。この平均化により、平均化された有効磁場(\(H_i^{mag} = J \sum_j \left\langle S_j \right\rangle\))の問題に帰着し、ハミルトニアンは、

\[H = - \sum_i S_i J \sum_j \left\langle S_j \right\rangle = - \sum_i H_i^{mag} S_i\]

となる。ここで、平均化された有効磁場は単なる係数と同じであり交換可能な量である。最終的に求めるべき平均場を\(\left\langle S \right\rangle\)として、

\[\left\langle S_i \right\rangle = \left\langle S \right\rangle\]

となるように、セルフコンシステントに解くこととなり、このためのセルフコンシステントな方程式は、

\[\left\langle S \right\rangle = \tanh { { {z J \left\langle S \right\rangle } \over {k_B T} } }\]

となる。zは最近接格子数、k<sub>B</sub>は[ボルツマン定数](https://ja.wikipedia.org/wiki/ボルツマン定数 "wikilink")、Tは温度である。この場合、求めるべき解である磁場の平均\(\left\langle S \right\rangle\)が有限の値となる場合が強磁性状態であり、ゼロの場合が磁性がない（常磁性）状態である。有限の値としての解が存在するためにはtanh内の係数部分が、\(\, { {z J} / {k_B T} } > 1\)である必要があり、\({ {z J} / {k_B T} } \le 1\)ならば解は\(\left\langle S \right\rangle = 0\)となる。従って、平均場近似における強磁性－常磁性の転移温度T<sub>c</sub>は、\(\, { {z J} / {k_B T} } = 1\)より、

\[T_c = { {z J} \over {k_B} }\]

である。
