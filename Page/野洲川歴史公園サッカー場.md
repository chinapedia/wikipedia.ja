**野洲川歴史公園サッカー場**（やすがわれきしこうえんサッカーじょう）は、[滋賀県](../Page/滋賀県.md "wikilink")[守山市](../Page/守山市.md "wikilink")の野洲川歴史公園内にある[サッカー](https://ja.wikipedia.org/wiki/サッカー "wikilink")グラウンド。施設は守山市が所有し、[公益財団法人](https://ja.wikipedia.org/wiki/公益財団法人 "wikilink")守山市文化体育振興事業団が[指定管理者](../Page/指定管理者.md "wikilink")として運営管理を行っている。通称は**ビッグレイク**。

## 概要

[サムネイル](https://ja.wikipedia.org/wiki/ファイル:Big-Lake-Moriyama-2.jpg "wikilink") 「[2002 FIFAワールドカップ記念事業](../Page/2002_FIFAワールドカップ.md "wikilink")」として実施された[日本サッカー協会](../Page/日本サッカー協会.md "wikilink")（JFA）の「[サッカーを中心としたスポーツ環境整備モデル事業](https://ja.wikipedia.org/wiki/サッカーを中心としたスポーツ環境整備モデル事業 "wikilink")」の一環として\[1\]JFAの助成金2億1,400万円を得て地域のスポーツ文化の活性化を目指して整備された。総工費約9億5000万円、[2005年](../Page/2005年.md "wikilink")[10月1日](../Page/10月1日.md "wikilink")開業。

サッカー場以外にクラブハウスも併設されており、施設完成時から滋賀県サッカー協会が入居している。

### 屋外施設

| コート | 芝                                | 照明設備                                   | 面積       | 収容人数           |
| --- | -------------------------------- | -------------------------------------- | -------- | -------------- |
| A   | [人工芝](../Page/人工芝.md "wikilink") | 有り(75[ルクス](../Page/ルクス.md "wikilink")) | 114m×77m | 約400人（法面）      |
| B   | 人工芝                              | 有り(200ルクス)                             | 114m×76m | 約1,200人（スタンド席） |
| C   | [天然芝](../Page/芝.md "wikilink")   | 無し                                     | 119m×74m | 約500人（法面）      |
|     |                                  |                                        |          |                |

人工芝のAコートとBコートには共に白色でサッカー用のラインが、赤色でミニサイズのサッカーのラインが一面ずつペイントされている。更にサッカーラインの四隅に[フットサル](https://ja.wikipedia.org/wiki/フットサル "wikilink")用のラインがペイントされていて4面のフットサル場としても利用可能であり、1/4面単位での貸し出しも行われている。
