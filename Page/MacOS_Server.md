**macOS Server** (マックオーエス サーバ)、**OS X Server** (オーエス テン サーバ)、**Mac OS X Server** (マック オーエス テン サーバ)は、[アップルが開発](../Page/アップル_\(企業\).md "wikilink")・販売する、[Macintosh](../Page/Macintosh.md "wikilink")の[サーバ](../Page/サーバ.md "wikilink")[オペレーティングシステム](../Page/オペレーティングシステム.md "wikilink") (OS) である。

## 概要

macOS Serverは、[macOS](https://ja.wikipedia.org/wiki/macOS "wikilink")と同じ機能を持ち、さらに[DNS](../Page/Domain_Name_System.md "wikilink")、[POP3](../Page/Post_Office_Protocol.md "wikilink")、[IMAP](../Page/Internet_Message_Access_Protocol.md "wikilink")、[SMTP](../Page/Simple_Mail_Transfer_Protocol.md "wikilink")、[Mailing List](../Page/メーリングリスト.md "wikilink")、[World Wide Web](../Page/World_Wide_Web.md "wikilink")（[Wiki](../Page/ウィキ.md "wikilink")、[Blog](../Page/ブログ.md "wikilink")）、[iChat](https://ja.wikipedia.org/wiki/iChat "wikilink")、[iCal](https://ja.wikipedia.org/wiki/iCal "wikilink")、[SMB](../Page/Samba.md "wikilink")、[AFP](../Page/Apple_Filing_Protocol.md "wikilink")、[NFS](../Page/Network_File_System.md "wikilink")、[DHCP](https://ja.wikipedia.org/wiki/Dynamic_Host_Configuration_Protocol "wikilink")、[VPN](../Page/Virtual_Private_Network.md "wikilink")、[RADIUS](../Page/RADIUS.md "wikilink")、[Open Directory](https://ja.wikipedia.org/wiki/Apple_Open_Directory "wikilink")、[NetBoot](../Page/NetBoot.md "wikilink")、[QuickTime Streaming Server](../Page/QuickTime_Streaming_Server.md "wikilink")、[Podcast プロデューサーなど](https://ja.wikipedia.org/wiki/Podcast_プロデューサー "wikilink")、多様かつ高度なサーバとしての機能と[GUIによるリモートサーバ管理ツール](https://ja.wikipedia.org/wiki/グラフィカルユーザインタフェース "wikilink")（[サーバ管理](https://ja.wikipedia.org/wiki/サーバ管理 "wikilink")、[ワークグループマネージャ](../Page/ワークグループマネージャ.md "wikilink")、[サーバアシスタント](https://ja.wikipedia.org/wiki/サーバアシスタント "wikilink")、[Xgrid Adminなど](../Page/Xgrid.md "wikilink")）を合わせ持つ、Snow Leopard Server(Mac OS X Server 10.6)までは別製品のサーバ向けオペレーティングシステムであった。

2018年9月にリリースされたSever.app 5.7.1からは、プロファイルマネージャに特化している\[1\]。

## 沿革

## 特徴

最初に発売されたMac OS X Server 1.x系統と現在のMac OS X v10.xをベースとしたMac OS X Server v10.x系統は、GUIのみならず[カーネル](../Page/カーネル.md "wikilink")そのものや[APIの違いにより基本的に互換性のない別のシステムである](../Page/アプリケーションプログラミングインタフェース.md "wikilink")。

### Mac OS X Server 1.0

  - [1999年](../Page/1999年.md "wikilink")夏 - 最初のバージョン (Mac OS X Server 1.0) が発売された。

### Mac OS X Server 10.0

  - [2001年](../Page/2001年.md "wikilink")3月 - Mac OS X Server v10.0が発売された。

### Mac OS X Server 10.1

  - [2001年](../Page/2001年.md "wikilink")9月 - Mac OS X Server v10.1が発売された。

### Mac OS X Server 10.2

  - [2002年](../Page/2002年.md "wikilink")7月18日発表\[2\]
  - 2002年8月24日 - Mac OS X Server v10.2 (Jaguar Server) が発売された。
      - [Open Directoryが初めて実装された](https://ja.wikipedia.org/wiki/Apple_Open_Directory "wikilink")。
      - サーバ管理、ワークグループマネージャ、サーバモニタ、ネットワークイメージユーティリティ、Macintoshマネージャ等がツールとして付属。
      - Mac OS X Server v10.2系列の最終バージョンは、10.2.8であり、セキュリティアップデートは2005年1月で終了している。

### Mac OS X Server 10.3

  - [2003年](../Page/2003年.md "wikilink")10月 - Mac OS X Server v10.3 (Panther Server) が発売された。
      - Mac OS X Server v10.3系列の最終バージョンは、10.3.9であり、セキュリティアップデートは2007年11月で終了している。

### Mac OS X Server 10.4

  - [2005年](../Page/2005年.md "wikilink")4月 - Mac OS X Server v10.4 (Tiger Server) が発売された。
  - [2006年](../Page/2006年.md "wikilink")8月 - [Universal Binary版](https://ja.wikipedia.org/wiki/Universal_Binary "wikilink") Mac OS X Server v10.4.7が発売された。
      - Mac OS X Server v10.4系列の最終バージョンは、10.4.11（2007年11月14日にリリース）である。
      - 2009年9月のSecurity Update 2009-005にて、セキュリティアップデートは終了した。

### Mac OS X Server 10.5

  - [2007年](../Page/2007年.md "wikilink")10月26日 - [UNIX](../Page/UNIX.md "wikilink")®サーバオペレーティングシステム、Mac OS X Server v10.5 Leopard（Universal Binary版）が発売された\[3\]。
      - [NEXTSTEP](../Page/NEXTSTEP.md "wikilink")から引き継がれてきたディレクトリサービスの[NetInfo](https://ja.wikipedia.org/wiki/NetInfo "wikilink")は完全に排除され、Open DirectoryとdsLocalに置き換わった。
      - Mac OS X Server v10.5系列の最終バージョンは、10.5.8 v1.1(build 9L34)、2009年8月31日にリリースされたものである\[4\]。2009年8月5日にリリースされた10.5.8 (build 9L30) での問題点が修正されている。
      - 2011年6月のセキュリティアップデート 2011-004\[5\]\[6\]にて、セキュリティアップデートは終了している。

### Mac OS X Server 10.6

  - 2009年6月の[WWDCにてSnow](../Page/Worldwide_Developers_Conference.md "wikilink") Leopard Serverが2009年9月にリリースされる予定と発表された。
  - 2009年8月24日、Unilimitedクライアント版が￥53,800にて、同年8月28日より発売されると発表された\[7\]。
      - Mac OS X Server v10.6系列の最新バージョンは、10.6.8 v1.1 (build 10K549)、2011年7月25日にリリースされたものである2011年6月23日にリリースされた10.6.8 (build 10K540) での問題点が修正されている\[8\]\[9\]。

### Lion Server 10.7 (1.0)

  - Lion Server 10.7\[10\]は、2011年6月の[WWDC当日](../Page/Worldwide_Developers_Conference.md "wikilink")、Lion Serverが2011年7月にMac App Storeにてリリースされる予定と発表、同年7月20日に発売された。
  - Mac OS X Server v10.7系列の最新バージョンは、10.7.3 (build 11D50b)、2012年2月3日にリリースされたものである\[11\]\[12\]\[13\]。
  - Server Admin Tools（サーバ管理ツール）はオプションとされ、別途ダウンロードが必要となった。

### OS X Server 2.0

  - [OS X Mountain Lionと同時に](https://ja.wikipedia.org/wiki/OS_X_Mountain_Lion "wikilink")2012年7月にMac App Storeでリリースされた。このバージョンから名称が「OS X Server」になった。[ワークグループマネージャ 10.8](../Page/ワークグループマネージャ.md "wikilink")\[14\]を別途ダウンロードすることで利用できる。

### OS X Server 3.0

  - [OS X Mavericksと同時に](https://ja.wikipedia.org/wiki/OS_X_Mavericks "wikilink")2013年10月にMac App Storeでリリースされた。Server 3.0単体アプリケーションである\[15\]。[Workgroup Manager 10.9](../Page/ワークグループマネージャ.md "wikilink")\[16\]を別途ダウンロードすることで利用できる。

### OS X Server 4.0

  - [OS X Yosemiteと同時に](https://ja.wikipedia.org/wiki/OS_X_Yosemite "wikilink")2014年10月17日にMac App Storeでリリースされた。Workgroup Manager廃止。

### OS X Server 5.0.3

  - 2015年9月17日にリリースされた。[OS X Yosemiteもしくは](https://ja.wikipedia.org/wiki/OS_X_Yosemite "wikilink")[OS X El Capitanで動作する](https://ja.wikipedia.org/wiki/OS_X_El_Capitan "wikilink")。

### OS X Server 5.1

  - 2016年3月22日にリリースされた。

### macOS Server 5.2

  - 2016年9月21日にリリースされた。OS X El Capitanもしくは[macOS Sierraで動作する](https://ja.wikipedia.org/wiki/macOS_Sierra "wikilink")。

### macOS Server 5.3

  - 2017年3月28日にリリースされた。macOS Sierraで動作する。

### macOS Server 5.4

  - 2017年9月26日にリリースされた。macOS Sierraもしくは[macOS High Sierraで動作する](https://ja.wikipedia.org/wiki/macOS_High_Sierra "wikilink")。APFSボリューム、macOS High Sierra、iOS 11、tvOS 11に対応。

### macOS Server 5.5

  - 2018年1月24日にリリースされた。macOS High Sierraで動作する。

### macOS Server 5.6

  - 2018年3月30日にリリースされた。macOS High Sierraで動作する。Open Directory、プロファイルマネージャ、Xsan管理以外のオープンソースサービス管理機能が2018年秋に廃止との予告が表示される。

### macOS Server 5.7.1

  - 2018年9月29日にリリースされた。[macOS Mojaveで動作する](https://ja.wikipedia.org/wiki/macOS_Mojave "wikilink")。予告通り、プロファイルマネージャに特化し、Open DirectoryとXsan管理以外、他の機能やサービスは削除された\[17\]。

### macOS Server 5.8

  - 2019年3月26日にリリースされた。[macOS Mojaveで動作する](https://ja.wikipedia.org/wiki/macOS_Mojave "wikilink")。プロファイルマネージャが、新しい制限、ペイロード、コマンドに対応した。

### macOS Server 5.9

  - 2019年10月8日にリリースされた。[macOS Catalinaで動作する](https://ja.wikipedia.org/wiki/macOS_Catalina "wikilink")。プロファイルマネージャが、追加・変更された、新しい制限、ペイロード、コマンドに対応した\[18\]。
