**積分回路**は、[電気回路](../Page/電気回路.md "wikilink")の一種で[入力](https://ja.wikipedia.org/wiki/入力 "wikilink")[電圧](../Page/電圧.md "wikilink")の[波形](https://ja.wikipedia.org/wiki/波形 "wikilink")の[時間積分](https://ja.wikipedia.org/wiki/時間積分 "wikilink")に等しい波形の電圧を[出力](https://ja.wikipedia.org/wiki/出力 "wikilink")する回路である。

[コンデンサ](../Page/コンデンサ.md "wikilink")両端の電圧は、流れ込んだ[電流](../Page/電流.md "wikilink")の[積分](https://ja.wikipedia.org/wiki/積分 "wikilink")（[電荷](../Page/電荷.md "wikilink")の総量）に[比例](https://ja.wikipedia.org/wiki/比例 "wikilink")するという事実を利用している。

## RC回路

電気回路中を流れる電流というのは、実は[荷電粒子](https://ja.wikipedia.org/wiki/荷電粒子 "wikilink")（[電子](https://ja.wikipedia.org/wiki/電子 "wikilink")など）の移動によって現れる、

電荷の流れである。[導体](https://ja.wikipedia.org/wiki/導体 "wikilink")に電流iがt=0からt秒間に渡って流れたとき、

流れ込んで導体を通過した電荷の総量Qは

  -
    \(q = \int_0^ti dt\)

で与えられる。導体の間に[誘電体](../Page/誘電体.md "wikilink")を挟んだ場合（つまり、コンデンサの場合）、

誘電体中には移動できる[自由電子](../Page/自由電子.md "wikilink")が無いため、

流れ込んだ電流は誘電体の境界面で[帯電](https://ja.wikipedia.org/wiki/帯電 "wikilink")する。

  -
    \(q = \int_0^tidt + Q_0\)

\(Q_0\)はt=0で既に誘電体が帯びていた電荷である。

[静電誘導](https://ja.wikipedia.org/wiki/静電誘導 "wikilink")によって反対側の境界面にも逆の[極性](https://ja.wikipedia.org/wiki/極性 "wikilink")の電荷が帯電するので、

誘電体を挟んで[電位差](https://ja.wikipedia.org/wiki/電位差 "wikilink")vが生じる。複雑な形状をしていなければ、vはqに比例する。

[比例定数](https://ja.wikipedia.org/wiki/比例定数 "wikilink")をCとすると、

  -
    \(v = \frac{q}{C} = \frac{1}{C} \int_0^ti dt\)

である。

[series-RC.svg](https://ja.wikipedia.org/wiki/File:series-RC.svg "fig:series-RC.svg")[RC回路](https://ja.wikipedia.org/wiki/RC回路 "wikilink")\]\]

ここで、図のようなRC直列回路を考えて、

[交流](../Page/交流.md "wikilink")電圧Vを印加する。

[初期状態](https://ja.wikipedia.org/wiki/初期状態 "wikilink")におけるコンデンサCの電荷\(Q_0 = 0\)とすると、

t=0では、[オームの法則](../Page/オームの法則.md "wikilink")に従って\(I_0 = V_{in}/R\)が流れる。

これが、

コンデンサへ流れ込んでコンデンサに電荷qが蓄えられると、

コンデンサが[逆起電力](https://ja.wikipedia.org/wiki/逆起電力 "wikilink")を生じるので、Vの[抵抗](https://ja.wikipedia.org/wiki/抵抗 "wikilink")Rへの分圧が低下し、

回路を流れる電流は小さくなる。

しかし、*印加した交流電気の[周波数](../Page/周波数.md "wikilink")fが十分に大きいならば*、

この交流に対してCは[短絡](https://ja.wikipedia.org/wiki/短絡 "wikilink")とみなせるので、

回路を流れる電流Iは常に\(I = V_{in}/R\)で与えられる。

従って、このときに限り、

  -
    \(V_C = \frac{1}{C} \int_0^tI dt = \frac{1}{RC} \int_0^tV_{in} dt\)

である。つまり、RC回路の両端には入力Vの積分の波形をした電圧が現れる。

## アンプを利用した積分回路

### ミラー積分回路

[Miller_Form.jpg](https://ja.wikipedia.org/wiki/File:Miller_Form.jpg "fig:Miller_Form.jpg") より精度の高い積分波形を得るために、[オペアンプ](https://ja.wikipedia.org/wiki/オペアンプ "wikilink")など増幅器を用いた回路があり「ミラー積分回路」と呼ばれている。 図中で、アンプが[入力インピーダンス](https://ja.wikipedia.org/wiki/入力インピーダンス "wikilink")無限大の理想的なものであれば、[非反転入力端子](https://ja.wikipedia.org/wiki/非反転入力端子 "wikilink")に出入りする電流は0であるので、抵抗Rを流れる電流と等しい大きさの電流IがコンデンサCに入力端子の方向から流れ込んでくる。オペアンプの非反転入力端子は[仮想接地](https://ja.wikipedia.org/wiki/仮想接地 "wikilink")されているので、Rに流れる電流の大きさは\(I_0 = V/R\)である。以上より、

  -
    \(I = \frac{V_{in}}{R} = -C\frac{dV_{out}}{dt}\)

が成り立っていることが分かる。初期状態において、Cに蓄えられている電荷は既に[放電](https://ja.wikipedia.org/wiki/放電 "wikilink")してあるとすると、

  -
    \(V_{out} = -\frac{1}{RC} \int_0^tV_{in}dt\)

であり、入力信号を積分した出力が得られることが分かる。

実際の理想的でないオペアンプでは、入力端子に[バイアス電流が流れ込み](https://ja.wikipedia.org/wiki/バイアス_\(電子工学\) "wikilink")、また増幅度が有限であるためにまだ誤差を生じ、信号を入力しなくても出力電圧が生じる。元々は真空管で構成した回路で、直線性の良さからオシロスコープの時間軸信号発生回路や、[アナログ計算機で](https://ja.wikipedia.org/wiki/アナログコンピュータ "wikilink")、演算増幅器と組んで積分器を構成し主に「微分方程式解析表示器」として製品開発に使われた。 真空管の入力電流はほぼゼロで、コンデンサーの漏洩電流を問題にして選別していたのだが、半導体アンプでは素子が電流制御で、これを防ぐため、電圧制御素子であるFET入力オペアンプを用いたり、入力にバイアス抵抗 を繋いで放電させたりしている。時間軸発生など常に初期化される繰り返し型では問題ない。
