『**ゴゴワイド**』は、 [テレビ朝日](../Page/テレビ朝日.md "wikilink")で放送している[ドラマ](../Page/テレビドラマ.md "wikilink")[再放送](../Page/再放送.md "wikilink")枠である。

## 概要

前身ドラマ再放送枠の『[劇的5ch\!](https://ja.wikipedia.org/wiki/劇的5ch! "wikilink")』と『[劇的空間](https://ja.wikipedia.org/wiki/劇的空間 "wikilink")』を統合して2014年4月1日から放送されている関東ローカルのドラマの再放送枠である。

前番組と同じくテレビ朝日系で放送されていたテレビドラマを再放送しているが、前番組で放送されていた時代劇は早朝の『[おはよう\!時代劇](https://ja.wikipedia.org/wiki/おはよう!時代劇 "wikilink")』で放送されていることから現在はテレビドラマに限って放送されている。

放送する番組によっては1時間ドラマを3部構成（第1部、13:59 - 14:57）（第2部、14:57 - 15:53）（第3部、15:53 - 16:50）で放送されたり、2時間ドラマと1時間ドラマを１つずつ放送する2部構成（第1・2部、13:59 - 15:53）（第3部、15:53 - 16:50）で放送されたりと自由な編成をとっている。

基本的に放送する作品は決まってないが平日15:53 - 16:50は連日、テレビドラマ『[相棒](https://ja.wikipedia.org/wiki/相棒 "wikilink")』が放送されており『劇的空間』のサブ枠名で用いられていた枠名『[相棒セレクション](https://ja.wikipedia.org/wiki/相棒セレクション "wikilink")』が付けられている。

## 放送時間

2018年4月現在、[月曜日](../Page/月曜日.md "wikilink") - [金曜日](../Page/金曜日.md "wikilink")の13:59 - 16:50（[JST](../Page/日本標準時.md "wikilink")）で放送されている。

2015年度、2016年度は金曜日に限り後続番組の『[スーパーJチャンネル](https://ja.wikipedia.org/wiki/スーパーJチャンネル "wikilink")』が15:50から開始することに伴い、1時間短縮となっていたが2017年度からは金曜日の放送時間が、月曜日から木曜日と同じ16:50に統一されたため、放送時間が2時間から3時間に復した。

| 期間        | 月曜日 - 金曜日 |
| --------- | --------- |
| 月曜日 - 木曜日 | 金曜日       |
| 2014.4.1  | 2014.9.26 |
| 2014.9.29 | 2015.3.27 |
| 2015.3.30 | 2017.3.31 |
| 2017.4.3  | 2018.3.30 |
| 2018.4.2  | 現在        |

## 備考

  - ゴゴワイド第1部開始時にはテレビ朝日のマスコットキャラクターの「[ゴーちゃん。](https://ja.wikipedia.org/wiki/ゴーちゃん。 "wikilink")」のアニメを使用したOPと提供クレジットが流れる。
  - ゴゴワイド第3部のエンディング終了後は後続番組『[スーパーJチャンネル](https://ja.wikipedia.org/wiki/スーパーJチャンネル "wikilink")』にステーションブレイク無しで接続する。
  - 午後に重要な中継や地震など緊急を要する事柄が放送時間前に発生した場合は前座番組の『[ワイド\!スクランブル](../Page/ワイド!スクランブル.md "wikilink")』の放送時間拡大、『[スーパーJチャンネル](https://ja.wikipedia.org/wiki/スーパーJチャンネル "wikilink")』の開始時間前倒し、『ANN[報道特別番組](../Page/報道特別番組.md "wikilink")』などの編成で、当番組が放送休止もしくは短縮になることがある。その際は翌日もしくは翌週に放送内容が順延する。
  - [EPGや](../Page/電子番組ガイド.md "wikilink")[新聞](../Page/新聞.md "wikilink")などの[番組表](https://ja.wikipedia.org/wiki/番組表 "wikilink")では、番組名のみが表記されている。局の公式ページでも同様。
