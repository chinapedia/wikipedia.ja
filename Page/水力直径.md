**水力直径**（すいりきちょっけい、）とは、非円形の[管](../Page/管.md "wikilink")および[水路](../Page/水路.md "wikilink")で[流れ](../Page/流れ.md "wikilink")を扱う際、円管の直径に相当する長さを表す量である。

\[D_\text{H} = \frac{4A}{P}\] ここで、が水力直径、は流れの断面積、は断面のうち濡れている部分（[濡れ縁](https://ja.wikipedia.org/wiki/濡れ縁 "wikilink")）の周長である。

また、4倍せず単純に流れの断面積を濡れ縁の長さで割ったものは**水力半径**（すいりきはんけい、）と呼ばれ、次のように定義される\[1\]\[2\]。

\[R_\text{H} = \frac{A}{P}\] ここで、が水力半径である。

水力半径と同じものを[土木工学](../Page/土木工学.md "wikilink")や[水理学](../Page/水理学.md "wikilink")では**径深**（けいしん）と呼ぶ。水力半径は主に[流体力学](../Page/流体力学.md "wikilink")や[水力学](https://ja.wikipedia.org/wiki/水力学 "wikilink")で用いられる。 他には、**水力水深**（すいりきすいしん）、**動水半径**（どうすいはんけい）、**水力平均深さ**（すいりきへいきんふかさ）、**水理学的平均深さ**（すいりがくてきへいきんふかさ）、なども同義語である\[3\]。

この濡れ縁の長さは流体からのせん断応力が作用するすべての表面を含む\[4\]。土木工学や水理学では、この長さのことを**潤辺**（じゅんぺん）も呼ぶ。

## 性質

直径と半径という名前ではあるものの、定義から明らかに、水力直径は水力半径の2倍ではなく4倍である。

\[D_\text{H} = 4R_\text{H}\]

ただし、特に円管の場合は、水力直径は円管半径の2倍である。

\[D_\text{H} =\frac{4 \pi R^{2}}{2 \pi R}=2R\]

## 応用

水力直径は主に[乱流](../Page/乱流.md "wikilink")を含む計算に使用される。 非円形の管においては乱流[せん断応力](https://ja.wikipedia.org/wiki/せん断応力 "wikilink")の結果として[二次流れ](https://ja.wikipedia.org/wiki/二次流れ "wikilink")が観察される。 このような流れ場において水力直径を用いると、円管の場合と同じ方法で[圧力損失](https://ja.wikipedia.org/wiki/圧力損失 "wikilink")や[熱伝達率](../Page/熱伝達率.md "wikilink")等を計算できる。

具体的には、水力直径は[レイノルズ数](../Page/レイノルズ数.md "wikilink")等の[無次元量](../Page/無次元量.md "wikilink")の場合に単一の次元を使用するために必要となる。 例えば、実際に[マニング公式](https://ja.wikipedia.org/wiki/マニング公式 "wikilink")では径深（水力半径）が用いられている。

また、水力直径は[内部流れ](https://ja.wikipedia.org/wiki/内部流れ "wikilink")問題の[熱伝達の計算にも使用される](../Page/熱伝達率.md "wikilink")。

## 例

| 形状                                                      | 水力直径                                                                                                                                           | 備考                            |
| ------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------- |
| 円管                                                      | \(D_\text{H} = \frac{4\cdot\frac{ \pi D^2 }{4}}{\pi D} = D\)                                                                                   | 円管の場合、水力直径は単に円管の直径に等しい。       |
| [アニュラス](https://ja.wikipedia.org/wiki/アニュラス "wikilink") | \(D_\text{H} = \frac{4 \cdot \frac{\pi (D_\text{out}^2 - D_\text{in}^2)}{4}} {\pi (D_\text{out} + D_\text{in})} = D_\text{out} - D_\text{in}\) |                               |
| 正方形管                                                    | \(D_\text{H} = \frac{4 a^2}{4a} = a\)                                                                                                          |                               |
| 長方形管。4辺が閉じており、すべて濡れ縁であるもの。                              | \(D_\text{H} = \frac{4 ab}{2(a+b)} = \frac{2 ab}{a+b}\)                                                                                        | 幅広の場合、幅がもう一方の辺に対しのとき、 2*a*}}。 |
| 水路、または部分的に満たされた長方形管。濡れ縁は管の3つの側面（床面と2つの側面）で構成されている。      | \(D_\text{H} = \frac{4 ab}{2a + b}\)                                                                                                           | 幅広の場合、幅が水深に対しのとき、 4*a*}}。     |

断面が[正多角形](../Page/正多角形.md "wikilink")である完全に満たされた管の場合、水力直径は濡れ縁内の[内接円](https://ja.wikipedia.org/wiki/内接円 "wikilink")の直径に相当する。
