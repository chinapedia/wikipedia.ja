[Northern_Hemisphere_LamAz.png](https://ja.wikipedia.org/wiki/File:Northern_Hemisphere_LamAz.png "fig:Northern_Hemisphere_LamAz.png") [Lambert-azimuthal-equal-area.jpg](https://ja.wikipedia.org/wiki/File:Lambert-azimuthal-equal-area.jpg "fig:Lambert-azimuthal-equal-area.jpg") **ランベルト正積方位図法**（ランベルトせいせきほういずほう）とは、[投影法の一種であり](../Page/投影法_\(地図\).md "wikilink")、[方位図法](../Page/方位図法.md "wikilink")（[地図](../Page/地図.md "wikilink")の中心からの[方位](../Page/方位.md "wikilink")が正しく示される）および正積図法（[面積](../Page/面積.md "wikilink")が正しく示される）の両方の性質を持つ。

[北極点](../Page/北極点.md "wikilink")もしくは[南極点](../Page/南極点.md "wikilink")を基準点（[中心](https://ja.wikipedia.org/wiki/中心 "wikilink")）とした場合、[経線](https://ja.wikipedia.org/wiki/経線 "wikilink")は中心から放射状に、[緯線](https://ja.wikipedia.org/wiki/緯線 "wikilink")は基準点を中心とする[同心円に描かれる](https://ja.wikipedia.org/wiki/円_\(数学\)#2つの円 "wikilink")。面積が正しく表されるよう、緯線の間隔は特に図の外側（基準点に対して[赤道](../Page/赤道.md "wikilink")より遠い側の半球）で狭くなっている。中心付近の歪みは比較的小さいので、大陸図や分布図に用いられる。

緯度が l° である緯圏を投射図上に描くための半径 r は、r = 2 R sin((90－l)/2) （Rは[地球](https://ja.wikipedia.org/wiki/地球 "wikilink")の[半径](../Page/半径.md "wikilink")）で与えられる。

同様に世界全体が円形に描かれる図法には、[正距方位図法](../Page/正距方位図法.md "wikilink")などがある。

## 定義

[Lambertprojection.svg](https://ja.wikipedia.org/wiki/File:Lambertprojection.svg "fig:Lambertprojection.svg")図。対蹠点を除く球面上の全ての点は、球と平面の接点を中心とする円弧に沿って平面に投影される。\]\]

ランベルト正積方位図法を定めるには、球と、その球にある点 *S* で接する平面を考える。*P* を *S* の[対蹠点でない球上の任意の点とする](../Page/対蹠地.md "wikilink")。*d* を *S* と *P* の三次元空間での距離とすれば（球面に沿っての距離ではない）、この投影によって *P* は平面上で *S* から *d* の距離にある点 *P*′ に移される。

より厳密にいえば以下のようになる。*S* を中心とし、*P* を通り、今考えている平面と直交するような円がただひとつ存在する。この円は平面と2点で交わるので、*P*′ を *P* に近い方として定める。これが投影後の点である（図を参照）。*S* の対蹠点はこの円がただひとつに定まらないため投影から省かれる。*S* は半径 0 の円弧に沿って自身に投影される\[1\]。

[コンピュータ](../Page/コンピュータ.md "wikilink")上で投影を行うためには明示的な式を与える必要がある。[単位球面](https://ja.wikipedia.org/wiki/単位球面 "wikilink")（すなわち *x*<sup>2</sup> + *y*<sup>2</sup> + *z*<sup>2</sup> = 1 を満たす三次元空間 \(\mathbb{R}^3\) 上の点 (*x*, *y*, *z*) の集合）上の *S* = (0, 0, -1) を図の中心とする投影を考えてみよう。[デカルト座標](https://ja.wikipedia.org/wiki/デカルト座標 "wikilink")で球面上の点 \((x, y, z)\) と平面上の点 \((X, Y)\) を表すこととすると、この投影とその逆写像は次で表される。

\[(X, Y) = \left(\sqrt{\frac{2}{1 - z}} x, \sqrt{\frac{2}{1 - z}} y\right),\]

\[(x, y, z) = \left(\sqrt{1 - \frac{X^2 + Y^2}{4}} X, \sqrt{1 - \frac{X^2 + Y^2}{4}} Y, -1 + \frac{X^2 + Y^2}{2}\right).\] [球面座標](https://ja.wikipedia.org/wiki/球面座標系 "wikilink") (φ, θ) （φ が[天頂](../Page/天頂.md "wikilink")からの角度、θ が[方位角](../Page/方位角.md "wikilink")）と[極座標](../Page/極座標系.md "wikilink") (*R*, Θ) を用いると次のように表される\[2\]。

\[(R, \Theta) = \left(2 \cos(\phi / 2), \theta\right),\]

\[(\phi, \theta) = \left(2 \arccos(R / 2), \Theta\right).\] [円柱座標系](https://ja.wikipedia.org/wiki/円柱座標系 "wikilink") (*r*, θ, z) と極座標 (*R*, Θ) では

\[(R, \Theta) = \left(\sqrt{2(1 + z)}, \theta\right),\]

\[(r, \theta, z) = \left(R \sqrt{1 - \frac{R^2}{4}}, \Theta, -1 + \frac{R^2}{2}\right).\] となる。

他の点を中心とする場合や半径が 1 以外の球面に対して定義される射影も同様の式で表現される\[3\]。

## 性質

前節で述べた通り、単位球面にたいするランベルト正積方位図法による投射は (0, 0, 1) では定義されない。それ以外の点は、平面上で中心が原点 (0, 0), 半径 2 の[開円板に移される](../Page/円板.md "wikilink")。球面上の点 (0, 0, -1) は (0, 0) へ、赤道 *z* = 0 は半径 \(\sqrt 2\), 中心 (0, 0) の円に、南半球 *z* \< 0 はその円に含まれる開円板に移ることになる。

この射影は球面（から点 (0, 0, 1) を除いたもの）と半径 2 の開円板との[微分同相写像](https://ja.wikipedia.org/wiki/微分同相写像 "wikilink") （両方向について無限回微分可能な[全単射](../Page/全単射.md "wikilink")）である。正積図法であることは球面上の[面素の面積を](../Page/面積分.md "wikilink")、逆写像を用いて変数変換して計算することで示すことができる。デカルト座標では

\[dA = dX \; dY.\] したがって球上で面積を測ることは円板上（地図上）で対応する部分の面積を測ることと同等であることが分かる。

一方、この図法では球面上の曲線間の角度の関係は保存されない。一般に、球面の一部と平面間の写像では角度と面積を両方保存することはできない（もしそれが可能なら、それは局所的[等長写像](https://ja.wikipedia.org/wiki/等長写像 "wikilink")となり、[ガウス曲率](https://ja.wikipedia.org/wiki/ガウス曲率 "wikilink")が保存されるはずであるが、球面と円板はこれが異なるので、これは不可能である）。この、平らな図では球面の一部分を完全に表すことはできないということは、地図学における基本的な問題である。

結果として、球面上の区域は地図上ではかなり形が歪むことがある。この歪みはとくに図法の中心 (0, 0, -1) から離れたところで顕著である。このため、実用上はこの図法で描く地図は半球までに制限するということがよく行われる。もう半球は必要なら別の図で描けばよい。
