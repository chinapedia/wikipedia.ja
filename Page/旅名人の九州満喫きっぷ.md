**旅名人の九州満喫きっぷ**（たびめいじんのきゅうしゅうまんきつきっぷ）は、[日本](https://ja.wikipedia.org/wiki/日本 "wikilink")の[九州](../Page/九州.md "wikilink")16社局の[鉄道](../Page/鉄道.md "wikilink")・[軌道線が](../Page/軌道_\(鉄道\).md "wikilink")3回まで一日乗り降り自由となる[特別企画乗車券](../Page/特別企画乗車券.md "wikilink")である。[九州旅客鉄道](../Page/九州旅客鉄道.md "wikilink")（JR九州）が発売する。 [thumb](https://ja.wikipedia.org/wiki/ファイル:Japan_Kyushu_Map_Chikei.gif "wikilink")

## 概要

利用可能期間中（発売日から3か月間）、参加事業者の鉄道・軌道線の[普通列車](../Page/普通列車.md "wikilink")・[快速列車](../Page/快速列車.md "wikilink")に3回（1回につき1人1日）乗車できる。1回分は乗車当日のみ有効。同じ日に1枚を複数人で使用することができる。九州にはバスにおいても類似した企画乗車券「[SUNQパス](../Page/SUNQパス.md "wikilink")」が存在する。

JR九州路線での基本的な使用方法は[青春18きっぷ](../Page/青春18きっぷ.md "wikilink")に準じ、[特急列車を](https://ja.wikipedia.org/wiki/特別急行列車 "wikilink")[特急料金なしで乗車できる](../Page/特別急行券.md "wikilink")[佐世保](https://ja.wikipedia.org/wiki/佐世保駅 "wikilink") - [早岐及び](https://ja.wikipedia.org/wiki/早岐駅 "wikilink")[宮崎](https://ja.wikipedia.org/wiki/宮崎駅 "wikilink") - [宮崎空港間では特急列車を利用することができる](https://ja.wikipedia.org/wiki/宮崎空港駅 "wikilink")。普通列車・快速列車の指定席は別に指定席券を購入することで利用できる。[新幹線](https://ja.wikipedia.org/wiki/九州新幹線 "wikilink")、[博多南線](../Page/博多南線.md "wikilink")、[有料特急](https://ja.wikipedia.org/wiki/有料特急 "wikilink")への乗車はできない。

私鉄のうち、[西日本鉄道](https://ja.wikipedia.org/wiki/西日本鉄道 "wikilink")（西鉄）の[特急](../Page/西鉄特急.md "wikilink")・急行、[島原鉄道](https://ja.wikipedia.org/wiki/島原鉄道 "wikilink")の急行は特別料金が不要であるため利用可能である。[南阿蘇鉄道](../Page/南阿蘇鉄道.md "wikilink")の[トロッコ列車](../Page/トロッコ列車.md "wikilink")、[くま川鉄道](../Page/くま川鉄道.md "wikilink")の「[田園シンフォニー](https://ja.wikipedia.org/wiki/くま川鉄道KT-500形気動車 "wikilink")」は別途料金券を購入しても利用できない。[平成筑豊鉄道](../Page/平成筑豊鉄道.md "wikilink")[門司港レトロ観光線列車](https://ja.wikipedia.org/wiki/平成筑豊鉄道門司港レトロ観光線 "wikilink")、[肥薩おれんじ鉄道](../Page/肥薩おれんじ鉄道.md "wikilink")「[おれんじ食堂](https://ja.wikipedia.org/wiki/おれんじ食堂 "wikilink")」リビングカーの[指定席は別に](../Page/座席指定席.md "wikilink")[指定席券を購入することで利用できる](../Page/座席指定券.md "wikilink")。

利用可能期間内であれば連続した3日間でなくても使用できる。乗車日が翌日にまたがる場合、JR九州線は0時を過ぎて最初に停車する駅まで、その他の鉄軌道線は終列車まで有効となる。

JR九州の[みどりの窓口](../Page/みどりの窓口.md "wikilink")・旅行センター・[JR九州旅行](https://ja.wikipedia.org/wiki/JR九州旅行 "wikilink")・九州内の主な旅行会社で発売している。なお、[新大牟田駅](../Page/新大牟田駅.md "wikilink")など当該駅からの当パスの利用ができない九州新幹線単独駅のみどりの窓口でも発売を行っている。JR九州が営業する窓口がない[下関駅](../Page/下関駅.md "wikilink")及びJR九州以外の鉄軌道事業者では発売していない。

## 略史

当初は[2007年](../Page/2007年.md "wikilink")[10月1日](../Page/10月1日.md "wikilink")から[2008年](https://ja.wikipedia.org/wiki/2008年 "wikilink")[3月31日](../Page/3月31日.md "wikilink")の間を発売・使用期間として10,000円で発売された。JR九州および同社との[連絡運輸](../Page/連絡運輸.md "wikilink")を行っている各社局の計11社の鉄道線が使えた。

発売期間中に約4,400枚が発売され、利用者から発売継続の要望が寄せられたため、2008年[4月18日](../Page/4月18日.md "wikilink")から対象社局を拡大してリニューアルし、再度発売されることになった。この対象社局拡大で、九州島内は[鋼索鉄道](https://ja.wikipedia.org/wiki/鋼索鉄道 "wikilink")や[索道](../Page/索道.md "wikilink")及び[西日本旅客鉄道](../Page/西日本旅客鉄道.md "wikilink")（JR西日本）、新幹線線を除き全ての鉄道・軌道線に乗車できるようになった。リニューアルされたきっぷの発売期間および使用期間は2008年4月18日から[7月13日](../Page/7月13日.md "wikilink")と同年[9月12日](../Page/9月12日.md "wikilink")から[11月30日](../Page/11月30日.md "wikilink")であり、その他の条件は同じとなっていた。

2009年も設定され、発売期間および使用期間は2009年4月1日から7月31日と同年9月1日から11月30日であった。

2010年度は価格が10,500円に値上げされ、発売・使用期間ともに拡大され、2010年4月1日から2011年3月31日までと、通年利用できるようになった。また前年は利用対象外とされていた[平成筑豊鉄道門司港レトロ観光線](https://ja.wikipedia.org/wiki/平成筑豊鉄道門司港レトロ観光線 "wikilink")が利用対象に加えられた。

2011年度も2010年度と同内容で発売し、2011年4月1日から2012年3月31日まで利用できる。

2012年度からは有効期間を発売日から3か月間とし、発売期間を設けずに発売する\[1\]。

2014年4月1日から[消費税](../Page/消費税.md "wikilink")率が5%から8%となったことに伴い、価格が10,800円に値上げされた。

2019年10月1日から消費税率が8%から10%となったことに伴い、価格が11,000円に値上げされた。

## SUNQパスとの比較

前述したようにSUNQパスを意識した内容になっているが、細かい条件では異なっている部分も多いため以下に比較表を記述する。

|          |                                                                                                   |                                                                          |
| -------- | ------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------ |
| 項目       | 旅名人の九州満喫きっぷ                                                                                       | SUNQパス（全九州+下関版/3日間用）                                                     |
| 価格       | 11,000円\[2\]                                                                                      | 11,000円\[3\]                                                             |
| 使用期間     | 任意の3日間（利用期間内であれば使用日が飛び飛びになっていても使用できる）。                                                            | 使用開始日から起算し連続した3日間。                                                       |
| 複数人での使用  | 1枚を複数人数で利用可能。1日で2 - 3人で利用したり、利用開始時に1 - 2人で利用し別日で別の人が1 - 2人利用することも可能。                              | 不可                                                                       |
| 優等車種への乗車 | 特別料金不要列車のみ可能。                                                                                     | 高速バス及び一部船舶まで\[4\]                                                        |
| 定時性      | 大幅な遅延する確率はバスより少ない。                                                                                | 道路状況により、数分から数十分の遅延が発生しやすい。                                               |
| 着席可能性    | 指定席であっても乗車券の他に特急券が必要な特急列車は、特急券のみを購入しても乗車不可能。指定席が導入されている普通列車に乗車する場合は別に520円または820円の指定席券を購入すれば、乗車可能。 | 予約制高速バスに別料金を支払うことなく予約しての乗車が可能。高速バスでは予約制でなくても着席が原則であるが、満席の際に乗車できない可能性がある。 |
| 夜行便      | 普通夜行列車が運行されていないため夜行列車での利用不可。                                                                      | 夜行高速バス2路線を利用できる。                                                         |
