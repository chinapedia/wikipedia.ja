[サムネイル](https://ja.wikipedia.org/wiki/ファイル:Minkowski_question_mark.svg "wikilink") [数学](../Page/数学.md "wikilink")において、**ミンコフスキー疑問符関数**（）は、によって定義された  と表される[関数であり](../Page/関数_\(数学\).md "wikilink")、さまざまな奇妙な[フラクタル](../Page/フラクタル.md "wikilink")特性を持つ。この関数は、を有理数の[二進展開に](../Page/二進法.md "wikilink")[連分数](../Page/連分数.md "wikilink")展開する関係式を介して、二次無理数を[単位間隔内の](https://ja.wikipedia.org/wiki/単位区間 "wikilink")[有理数](https://ja.wikipedia.org/wiki/有理数 "wikilink")に写す。この関係式は1938年に(Arnaud Denjoy)によって与えられた。 また、(Stern–Brocot tree)に密接に関連する再帰的な定義でわかるように、この関数は有理数をに写す。

## 定義

が[無理数](../Page/無理数.md "wikilink")の場合、 の[連分数](../Page/連分数.md "wikilink")表現を  とすると、疑問符関数は次のように定義される。

  -
    \(\operatorname{?}(x) = a_0 + 2 \sum_{n=1}^\infty \frac{\left(-1\right)^{n+1}}{2^{a_1 + \cdots + a_n}}\)

が[有理数](https://ja.wikipedia.org/wiki/有理数 "wikilink")の場合、 の連分数表現を  とすると、次のようになる。

  -
    \(\operatorname{?}(x) = a_0 + 2 \sum_{n=1}^m \frac{\left(-1\right)^{n+1}}{2^{a_1 + \cdots + a_n}}\)

## 直感的な説明

上記の定義を直感的に理解するために、0で始まる無限ビット列を  内の実数と解釈する、2つの相異なる解釈方法を考える。

まず明白な方法は、最初の0の後に2進小数点を置き、[二進小数として読む方法である](../Page/二進法.md "wikilink")。たとえば、ビット列 001001001001001001001001... は、2進数 0.010010010010... すなわち  を表す。

別の方法は、ビット列を[連分数](../Page/連分数.md "wikilink")  とみなす方法である。ここで整数  は、ビット列を[連長圧縮](../Page/連長圧縮.md "wikilink")したときの連続回数である。この場合、先ほどと同じビット列 001001001001001001001001...は  }} に対応する。もしビット列で同じビットが無限に続いて終わる場合、それを無視して連分数表現を終了する。この操作の妥当性は以下の恒等式に基づく。

  -

    .

に対する疑問符関数は、[カントール関数](https://ja.wikipedia.org/wiki/カントール関数 "wikilink")が[三進法](../Page/三進法.md "wikilink")表現を二進法表現に写すのと同様に、先程のビット列の2つ目の解釈方法を同じ列の1つ目の解釈方法に写すものと理解できる\[1\]\[2\]。先程のビット列を例に挙げると、以下の等式が成り立つ。

  -
    \(\operatorname{?}\left(\frac{\sqrt 3 - 1}{2}\right) = \frac{2}{7}\)

## 有理数引数に対する再帰的定義

単位区間内の有理数の場合、関数を[再帰的](https://ja.wikipedia.org/wiki/再帰的 "wikilink")に定義することもできる。  と  が  1}} を満たす（[ファレイ数列](../Page/ファレイ数列.md "wikilink")の隣接する項である）[既約分数](https://ja.wikipedia.org/wiki/既約分数 "wikilink")である場合\[3\]、次のようになる。

  -
    \(\operatorname{?}\left(\frac{p + r}{q + s}\right) = \frac12 \left[\operatorname{?}\left(\frac{p}{q}\right) + \operatorname{?}\left(\frac{r}{s}\right)\right]\)

初期条件を次のように与えると

  -
    \(\operatorname{?}\left(\frac{0}{1}\right) = 0 \quad \text{ and } \quad \operatorname{?}\left(\frac{1}{1}\right) = 1\)

[ファレイ数列](../Page/ファレイ数列.md "wikilink")を F<sub>2</sub>、F<sub>3</sub>と順に求めることで、任意の有理数  に対して  を計算することが可能になる。

と  が同じ[連分数](../Page/連分数.md "wikilink")をそれぞれ *n* - 1 段、*n* 段で打ち切ったものとすると、行列

  -
    <math>\\begin{pmatrix}

`p_{n-1} & p_n \\`
`q_{n-1} & q_n`

\\end{pmatrix}</math>

の[行列式](../Page/行列式.md "wikilink")は ±1 となる。このような行列は、2 × 2 行列で行列式が ±1 となる群  の元である。 この群は、[モジュラー群](https://ja.wikipedia.org/wiki/モジュラー群 "wikilink")に関係する。

### アルゴリズム

この再帰的な定義は、次の[C言語](../Page/C言語.md "wikilink")関数が示すように、任意の実数に対して任意の精度で関数値を計算する[アルゴリズム](../Page/アルゴリズム.md "wikilink")に適している。 このアルゴリズムは、入力  を探してを下降する途中で、 ?(*x*)}} の二進展開の項を合計していく。 [ループ不変量](https://ja.wikipedia.org/wiki/ループ不変条件 "wikilink")  1}}が満たされる限り、分数  }} を約分する必要はない。 別の不変量は  である。このプログラムの`for`ループは、 `while`ループのように扱われており、最初の3行の条件付きbreak構文が終了条件となる。 ループ内で不変条件に影響を与えるのは最後の2行のみであり、最初の3行がループから抜け出すことなく正常に実行されている限り、両方の不変条件が真であると示すことができる。ループ内部の第三の不変量（浮動小数点精度）は  であるが、どの条件もテストされる前に、ループの初めに  が半分にされるため、結果的にループの終了時でしか  は担保されない。 プログラムのには、ループの反復ごとに合計 `q + s` が少なくとも1ずつ増加し、この合計がC言語のプリミティブ型 **`long`** で表現するには大きすぎる場合にループが終了することに注意すれば十分である。 ただし実際には、 `y + d == y` の条件付き break があるため、短時間でループが終了する。

``` c
/* Minkowski's question-mark function */
double minkowski(double x) {
  long p = x;
  if ((double)p > x) --p; /* p=floor(x) */
  long q = 1, r = p + 1, s = 1, m, n;
  double d = 1, y = p;
  if (x < (double)p || (p < 0) ^ (r <= 0))
    return x; /* out of range ?(x) =~ x */
  for (;;) { /* invariants: q * r - p * s == 1 && (double)p / q <= x && x < (double)r / s */
    d /= 2;
    if (y + d == y)
      break; /* reached max possible precision */
    m = p + r;
    if ((m < 0) ^ (p < 0))
      break; /* sum overflowed */
    n = q + s;
    if (n < 0)
      break; /* sum overflowed */

    if (x < (double)m / n) {
      r = m;
      s = n;
    } else {
      y += d;
      p = m;
      q = n;
    }
  }
  return y + d; /* final round-off */
}
```

## 自己相似性

疑問符関数は明らかに視覚的に自己相似である。 自己相似の[モノイド](https://ja.wikipedia.org/wiki/モノイド "wikilink")は、単位正方形に作用する2つの演算子  と  によって生成され、次のように定義される。

  -
    <math>\\begin{align}

`S(x, y) &= \left( \frac x {x+1}, \frac y 2 \right) \\[5px]`
`R(x, y) &= (1 - x, 1 - y)`

\\end{align}</math>

視覚的には、  は単位正方形を左下4分の1に縮小し、  は中心を通る[点反射を実行する](../Page/点対称.md "wikilink")。

の[グラフ上の点は単位区間内の](../Page/グラフ_\(関数\).md "wikilink")  に対して座標  を持つ。 はすべての  で以下の等式を満たすため、そのような点は  と  によってグラフ上の別の点に変換される。

  -
    <math>\\begin{align}

` \operatorname{?}\left(\frac{x}{x+1}\right) &= \frac{\operatorname{?}(x)}{2} \\[5px]`
` \operatorname{?}(1 - x) &= 1 - \operatorname{?}(x)`
`\end{align}`</math>

これら2つの演算子は繰り返し結合され、モノイドを形成する。 モノイドの一般的な要素は正の整数  に対して

  -
    \(S^{a_1} R S^{a_2} R S^{a_3} \cdots\)

となる。このような各要素によって、疑問符関数の[自己相似](../Page/自己相似.md "wikilink")性が記述される。 このモノイドは**モノイドとも呼ばれ、すべての周期倍加フラクタル曲線は、周期倍加モノイドによって記述される自己対称性を持つ（疑問符関数は(de Rham curve)の特殊なケースであり、ド・ラーム曲線はこのような自己相似曲線のカテゴリーである）。 モノイドの要素は、と連分数  を同一視することによって有理数と対応づけられる 。

  -
    \(S : x \mapsto \frac{x}{x+1}\)
    \(T : x \mapsto 1 - x\)

はいずれも整数係数の[線形分数変換であるため](https://ja.wikipedia.org/wiki/一次分数変換 "wikilink")、モノイドは[モジュラー群](https://ja.wikipedia.org/wiki/モジュラー群 "wikilink") の部分集合と見なすことができる。

##  の特徴

疑問符関数は[狭義単調増加な連続関数である](../Page/単調写像.md "wikilink")\[4\]が、[絶対連続](https://ja.wikipedia.org/wiki/絶対連続 "wikilink")ではない。[有理数](https://ja.wikipedia.org/wiki/有理数 "wikilink")における[微分](../Page/微分.md "wikilink")係数はゼロである。積分されたときに疑問符関数を生成する[測度には](../Page/測度論.md "wikilink")、いくつかの構成法が存在する。そのような構成の1つは、実数直線上の[フェリー数の密度を測定することによって得られます](../Page/ファレイ数列.md "wikilink")。 疑問符測度は、とも呼ばれるものに対する典型的な例である。 疑問符関数は有理数をに写す。二進分数とは、上で述べた再帰的定義から帰納的に証明されるように、 その[二進表現が終了するものを意味する](../Page/二進法.md "wikilink")。このほか、を非二進有理数に写す。また、疑問符関数は[奇関数であり](../Page/偶関数と奇関数.md "wikilink")、関数方程式  ?(*x*) + 1}} を満たすことから、  は、[周期](../Page/周期関数.md "wikilink") 1 の奇[周期関数](../Page/周期関数.md "wikilink")である。  が無理数である場合、  は次数が2より大きい[代数的数](../Page/代数的数.md "wikilink")か、[超越数](../Page/超越数.md "wikilink")である。

疑問符関数は 0, , 1、そして少なくとももう2つの[不動点](../Page/不動点.md "wikilink")を持ち、中点に対して対称である。不動点の一つはは約 0.42037 である\[5\]。

1943年、(Raphaël Salem)は疑問符関数のフーリエ・スティールチェス係数(Fourier–Stieltjes coefficients)が無限大でゼロになるかという問題を提起した\[6\]。つまり、

  -
    \(\lim_{n \to \infty}\int_0^1 e^{2\pi inx} \, \operatorname{d?}(x)=0\)

となるかどうか、ということである。この問題はの結果の特別なケースとして、ジョーダン(Jordan)とザールステン(Sahlsten)\[7\]によって肯定的に解決された。 ミンコフスキー疑問符関数のグラフは、と呼ばれるフラクタル曲線の特殊なケースである。
