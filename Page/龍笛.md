**龍笛**（りゅうてき、竜笛）とは、[雅楽](../Page/雅楽.md "wikilink")で使う[管楽器](../Page/管楽器.md "wikilink")の一つで、横笛（おうてき、おうじょう）とも呼ばれる。吹き物。

## 概要

龍笛は[竹](../Page/竹.md "wikilink")の管で作られ、表側に「[歌口](https://ja.wikipedia.org/wiki/歌口 "wikilink")（うたぐち）」と7つの「[指孔](../Page/音孔.md "wikilink")（ゆびあな）」を持つ[横笛](https://ja.wikipedia.org/wiki/横笛 "wikilink")であり、[能管](../Page/能管.md "wikilink")、[篠笛](https://ja.wikipedia.org/wiki/篠笛 "wikilink")など[和楽器](../Page/和楽器.md "wikilink")の横笛全般の原型・先祖であるとも考えられている。 現在では入門用に[ABS樹脂](https://ja.wikipedia.org/wiki/ABS樹脂 "wikilink")でできた物（通称「プラ管」）も存在する。更に、ABS樹脂製より高価で、竹製より安価な花梨製の物も作られている。

[雅楽](../Page/雅楽.md "wikilink")の楽器の中では広い2[オクターブ](https://ja.wikipedia.org/wiki/オクターブ "wikilink")の音域(E5〜D7)をもち、低い音から高い音の間を縦横無尽に駆け抜けるその音色は「**舞い立ち昇る[龍](https://ja.wikipedia.org/wiki/龍 "wikilink")の鳴き声**」と例えられ、それが名前の由来となっている。同じ運指（指使い）であっても、息の吹き方によって、低音である和（フクラ）と高音である責（セメ）とに吹き分ける事ができる。又、竹製の「本管」は竹の種類や使用年数などで、別の龍笛と音階がズレる事も多い。（元々龍笛自体が絶対的な音階で作られたものではない。）

音量を高めるために、管の中に[鉛](https://ja.wikipedia.org/wiki/鉛 "wikilink")を入れたり、外側を[樺](https://ja.wikipedia.org/wiki/樺 "wikilink")や[籐](https://ja.wikipedia.org/wiki/籐 "wikilink")で巻いたりするなど意匠が凝らされている。

[奈良時代](../Page/奈良時代.md "wikilink")の楽人、[尾張浜主](../Page/尾張浜主.md "wikilink")が龍笛を広めたといわれ、「笛之楽祖」と称えられる\[1\]。

龍笛は古くから貴族や武将に好まれ、[堀河天皇](../Page/堀河天皇.md "wikilink")や[源義経](../Page/源義経.md "wikilink")、[源博雅](https://ja.wikipedia.org/wiki/源博雅 "wikilink")（みなもとのひろまさ）などの、龍笛にまつわるエピソードはいくつも伝えられている。また、[清少納言](https://ja.wikipedia.org/wiki/清少納言 "wikilink")も『[枕草子](../Page/枕草子.md "wikilink")』の中で、「楽器の中では、[笛](../Page/笛.md "wikilink")がとても良い」と書いている。

合奏では、主旋律を[篳篥](../Page/篳篥.md "wikilink")が担当し、龍笛はその音域の広さを活かし、主旋律に絡み合うように演奏する。 また通常、楽曲の最初の部分は龍笛の[ソロ演奏となっている](../Page/ソロ_\(音楽\).md "wikilink")。このソロ演奏は、その楽曲の龍笛パートのリーダー（音頭（おんど）、または主管とも呼ぶ）が担当する。

龍笛は[唐楽](https://ja.wikipedia.org/wiki/唐楽 "wikilink")をはじめ、[催馬楽](../Page/催馬楽.md "wikilink")・[朗詠](../Page/朗詠.md "wikilink")などでも用いられる。雅楽の横笛には龍笛のほか、[神楽](https://ja.wikipedia.org/wiki/神楽 "wikilink")で用いられる[神楽笛](https://ja.wikipedia.org/wiki/神楽笛 "wikilink")と、[高麗楽](https://ja.wikipedia.org/wiki/高麗楽 "wikilink")などで用いられる[高麗笛](https://ja.wikipedia.org/wiki/高麗笛 "wikilink")があるが、前者は龍笛より全音（長2度）低く、後者は龍笛より全音（長2度）高い音域を持つ。

横笛の起源は一説では[中央アジア](../Page/中央アジア.md "wikilink")が発祥といわれており、これが[シルクロード](../Page/シルクロード.md "wikilink")を経て中国に伝わり龍笛となり、日本に伝来されたといわれている。龍笛がシルクロードを経て、[ヨーロッパ](../Page/ヨーロッパ.md "wikilink")で[フルート](../Page/フルート.md "wikilink")になったという説もあるが、フルートの起源は諸説あり、真偽のほどは定かではない。

## 指孔名（譜字）と音程

龍笛の指孔は、吹き口に近い順に「六」「中」「夕」「丄」「五」「〒」「ン」と名付けられている。運指の形もそれぞれの孔名と同じ名称を用いるが、その場合は孔名の指孔を開け、その直前までの指孔を閉じた形を基本とするが、「丅」など孔名にない運指もある。

|          |        |        |    |         |     |     |     |         |    |
| -------- | ------ | ------ | -- | ------- | --- | --- | --- | ------- | -- |
| **名称**   | 口      | ン      | 〒  | 五       | 丄   | 夕   | 中   | 丅       | 六  |
| **読み**   | く      | じ      | かん | ご       | じょう | しゃく | ちゅう | げ       | ろく |
| **和の音程** | (C\#5) | (D\#5) | E5 | F5-F\#5 | G5  | A5  | B5  | C6-C\#6 | D6 |
| **責の音程** | (C\#6) | (D\#6) | E6 | F6-F\#6 | G6  | A6  | B6  | C7-C\#7 | D7 |

このうち全ての指孔を閉じた形である「口」と「ン」は実際の曲（少なくとも現行の古典曲）では用いられない。また、神楽笛と高麗笛には「ン」に相当する指孔がない。
