**かりがねサッカー場**（かりがねさっかーじょう）は、[長野県](../Page/長野県.md "wikilink")[松本市](../Page/松本市.md "wikilink")惣社にあるサッカーグラウンド。前身は**[かりがね自転車競技場](https://ja.wikipedia.org/wiki/かりがね自転車競技場 "wikilink")**及び**かりがね運動広場**。

## 歴史

[1978年](https://ja.wikipedia.org/wiki/1978年 "wikilink")の[やまびこ国体の](../Page/第33回国民体育大会.md "wikilink")[トラックレース](../Page/トラックレース.md "wikilink")会場だったかりがね自転車競技場と隣接するかりがね運動広場を[松本山雅FC](../Page/松本山雅FC.md "wikilink")の練習拠点として再整備され\[1\]2015年に開設。

## 設備

| ------ | --- | ---- | -------- | --- |
| 天然芝コート | 天然芝 | なし   | 105m×68m | あり  |
| 人工芝コート | 人工芝 | あり   | 105m×68m | あり  |
|        |     |      |          |     |

[天然芝](https://ja.wikipedia.org/wiki/天然芝 "wikilink")･[人工芝](../Page/人工芝.md "wikilink")コートがそれぞれ各1面、他にも[フットサル](https://ja.wikipedia.org/wiki/フットサル "wikilink")や[ゲートボール](../Page/ゲートボール.md "wikilink")が可能な屋根付きの多目的広場を備える。また松本山雅FCのクラブハウスが隣接している。

天然芝コートは主に松本山雅FCのトップチームの練習に、人工芝コートはトップチームが天然芝コートの養生で使用できない場合や下部組織の練習を行う他に、各年代の試合会場として使用されている。
