**トレランスファクター**（）は、結晶構造の安定性とゆがみの指標である\[1\]。**許容係数**、**寛容性因子**、**寛容係数**とも呼ばれる\[2\]。

もともとは[ペロブスカイト構造](../Page/ペロブスカイト構造.md "wikilink")(ABO<sub>3</sub>)を説明するものであるが、現在は[イルメナイト](../Page/イルメナイト.md "wikilink")型構造にも用いられる\[3\]。 あるいは、あるイオンに対して結晶構造の適合性を計算することができる\[4\]。

ペロブスカイトのトレランスファクターは、1926年に[ヴィクトール・ゴルトシュミットが述べた](https://ja.wikipedia.org/wiki/ヴィクトール・モーリッツ・ゴルトシュミット "wikilink")\[5\]。

## 定義式

トレランスファクター \(t\) は、 [イオン半径](https://ja.wikipedia.org/wiki/イオン半径 "wikilink")の比から計算される無次元数である\[6\]。

|                                         |
| :-------------------------------------: |
| \(t={r_A+r_O \over \sqrt{2}(r_B+r_O)}\) |
|      *r<sub>A</sub>*はAサイトの陽イオンの半径       |

理想的な立方晶ペロブスカイト構造では、単位格子の格子定数 *a* は、次の式で求められる\[7\]。

|                                    |
| :--------------------------------: |
| \(a=\sqrt{2}(r_A+r_O)=2(r_B+r_O)\) |
|    *r<sub>A</sub>*はAサイトの陽イオンの半径    |
