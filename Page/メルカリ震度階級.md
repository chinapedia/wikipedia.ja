**メルカリ震度階級**（メルカリしんどかいきゅう、）とは、ある地点における[地震](../Page/地震.md "wikilink")の程度（[地震動](../Page/地震動.md "wikilink")）を表現する指標。[地球](https://ja.wikipedia.org/wiki/地球 "wikilink")表面の構造物に与える影響や[人間](../Page/人間.md "wikilink")が感じる揺れの大きさに基づいて、人が判定する。

世界で数種類使用されている[震度階級](https://ja.wikipedia.org/wiki/震度階級 "wikilink")の1つであり、日本における[気象庁震度階級](../Page/気象庁震度階級.md "wikilink")にあたる。1873年考案の（10段階）をもとにして、1884年に[イタリア](../Page/イタリア.md "wikilink")の火山学者[ジュゼッペ・メルカリ](https://ja.wikipedia.org/wiki/ジュゼッペ・メルカリ "wikilink")によって考案された。

## 概要

1902年には12段階に修正したものを[アドルフォ・カンカーニ](https://ja.wikipedia.org/wiki/アドルフォ・カンカーニ "wikilink")が発表し、後の1906年にはメルカリが再修正したものを発表した。このときの震度階級は後に[アウグスト・ハインリッヒ・シーベルグ](https://ja.wikipedia.org/wiki/アウグスト・ハインリッヒ・シーベルグ "wikilink")が書きなおしたので、**メルカリ・カンカーニ・シーベルグ震度階級**（**MCS**\[1\]）と呼ばれる。

更にこの後の1931年、[アメリカの](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")2人の地震学者[ハリー・ウッド](https://ja.wikipedia.org/wiki/ハリー・ウッド "wikilink")と[フランク・ニューマン](https://ja.wikipedia.org/wiki/フランク・ニューマン "wikilink")によって修正されたものが発表された。このときの震度階級は**メルカリ・ウッド・ニューマン震度階級**（**MWN**\[2\]）と呼ばれる。

現在のメルカリ震度階級はMCSかMWNが用いられており、**改正メルカリ震度階級**（**MM**または**MMI**\[3\]）として知られている。アメリカなどで使用されている。

メルカリ震度階級は過去の地震の被害状況をもとに、被害の大きさの違いが明瞭に分かるよう区分されており、日本の[気象庁震度階級](../Page/気象庁震度階級.md "wikilink")のように計器観測に基づく数値により厳密に定義されたものではない。一般に、小さな揺れの場合には人間の感じた揺れの大きさに基づいて判断され、揺れが大きい場合には[地形](../Page/地形.md "wikilink")や[植生](../Page/植生.md "wikilink")などの自然の構造物、ビルや家屋、橋、ダムなどの[建造物](https://ja.wikipedia.org/wiki/建造物 "wikilink")の被害状況に基づいて判断される。

[アメリカ地質調査所](../Page/アメリカ地質調査所.md "wikilink")（USGS）が地震情報の中で用いているMM震度分布図「シェイクマップ\[4\]」では、算出式に最大速度（PGV）および[最大加速度](https://ja.wikipedia.org/wiki/表面最大加速度 "wikilink")（PGA）の値が用いられている。これは推定値の算出用であり、場所によって変わる地震波速度の分布データ、マグニチュード、観測点での加速度などをもとに発表される。震度とは正反対に、各階級に後付けで数値をあてはめたものである。

## 震度階級表

| 震度階級                                 | 揺れによる影響 | 速度                                                          | 加速度           |
| ------------------------------------ | ------- | ----------------------------------------------------------- | ------------- |
| I                                    | 極めて弱い   | ほとんどの人は揺れを感じない。                                             | 0.1cm/s以下     |
| II                                   | 非常に弱い   | 高い建物の上層階におり、安静にしている状態の人が揺れを感じる。                             | 0.1 - 1.1cm/s |
| III                                  | 弱い      | 高い建物の上層階にいる多くの人が揺れを感じる。駐車されている自動車がわずかに揺れる。                  | 2.1 - 5.0gal  |
| IV                                   | やや弱い    | 屋内にいる人の多くが揺れを感じる。眠っている人の一部が目を覚ます。食器棚がカタカタと揺れる。              | 1.1 - 3.4cm/s |
| V                                    | やや強い    | 多くの人が揺れを感じる。眠っている人の多くが目を覚ます。食器棚から食器が滑り落ちる。                  | 3.4 - 8.1cm/s |
| VI                                   | 強い      | ほぼすべての人が揺れを感じる。多くの人が不安を感じ、まっすぐに歩くことができない。本棚から本が滑り落ちる。       | 8.1 - 16cm/s  |
| VII                                  | 非常に強い   | 立っていることが難しい。軽い家具が転倒し、造りの弱い建造物が一部損壊する。自動車を運転している人の多くが揺れを感じる。 | 16 - 31cm/s   |
| VIII                                 | 極めて強い   | 重い家具が転倒し、多くの建造物が一部損壊する。                                     | 31 - 60cm/s   |
| <span style="color:#fff;">IX</span>  | 破壊的     | 多くの人が混乱に陥る。頑丈な建造物が一部損壊し、多くの建造物が半壊する。                        | 60 - 116cm/s  |
| <span style="color:#fff;">X</span>   | 破滅的     | 頑丈な建造物が半壊し、多くの建造物が全壊する。                                     | 116cm/s以上     |
| <span style="color:#fff;">XI</span>  | 壊滅的     | 頑丈な建造物が全壊し、橋が崩落する。                                          |               |
| <span style="color:#fff;">XII</span> | 絶望的     | あらゆるものが崩壊する。                                                |               |
