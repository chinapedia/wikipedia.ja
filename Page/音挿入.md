**音挿入**（おんそうにゅう、）とは、発音の便宜のため、[語](../Page/語.md "wikilink")の途中に[子音](../Page/子音.md "wikilink")や[母音](../Page/母音.md "wikilink")、[音節](../Page/音節.md "wikilink")が挿入される現象をいう。

この現象が起こるのは、たとえば以下のような場合がある。

  - 母音接続がある場合に、母音と母音の間に子音を入れて発音をしやすくするため
      - 例：  → draw**r**ing （intrusive R）
      - 例： [フランス語](https://ja.wikipedia.org/wiki/フランス語 "wikilink")の動詞倒置形:  （t euphonique, t analogique）
  - [子音連結](https://ja.wikipedia.org/wiki/子音連結 "wikilink")（consonant cluster）で、[調音部位](https://ja.wikipedia.org/wiki/調音部位 "wikilink")の異なる子音同士が連結する場合、発音しやすいよう、間に子音をおく場合
      - 例：  → am**b**rotos（）
      - 例： （人） → homne → homre →
  - 子音連結の際、子音と子音の間に母音を入れて連結を断ち切る場合（Anaptyxis）
      - 例：  →  （宝）
      - 例： Hamtramck（フランス系の人名） → Hamtram**i**ck（アメリカの地名）
  - 語頭に子音連結が来ることのできない言語において、語頭に母音を加えることで回避する場合（Prosthesis）
      - 例： [ギリシア語](https://ja.wikipedia.org/wiki/ギリシア語 "wikilink")の地名スミルナ（Smyrna） → [トルコ語](../Page/トルコ語.md "wikilink")の地名[イズミル](../Page/イズミル.md "wikilink")（**İ**zmir）
      - 例：  → スペイン語および古フランス語:
      - 例：  → , （フランス語では古い段階で「s」が欠落した）

## 日本語

例えば、「はるさめ」/harusame/は「はる」/haru/と「あめ」/ame/の間に子音/s/が挿入されてできた語形である（[接合辞](https://ja.wikipedia.org/wiki/接合辞 "wikilink")とも見られる）。この他にも「あまり」/amari/から「あんまり」/ammari/ができたり、「まひるま」/mahiruma/から「まっぴるま」/mappiruma/（\[h\]→\[p\]は[子音交替](../Page/子音交替.md "wikilink")）ができたりしている。

また[漢語](../Page/漢語.md "wikilink")や[外来語](../Page/外来語.md "wikilink")で原語で子音で終わったり、子音が連続している場合には母音の/i/, /u/, /o/が挿入される。例えば[英語](../Page/英語.md "wikilink")の strike /straik/ はストライキ/sutoraiki/やストライク/sutoraiku/となる。[漢語](../Page/漢語.md "wikilink")では語末の[入声](../Page/入声.md "wikilink")（[内破音](../Page/内破音.md "wikilink")）に母音が付加され、肉 /njiuk/がニク/niku/、一 /jit/がイチ/ichi/・イツ/itsu/（歴史的には「イチ」「イツ」と書いてイティ/iti/・イトゥ/itu/のように発音された）、立 /lip/がリフ/rihu/（現代ではリュウ/rjuː/。リツ/ritsu/は慣用音）などとなった。

[活用](../Page/活用.md "wikilink")では存在を表す動詞だった「ある」(-ar-)が[動詞](../Page/動詞.md "wikilink")の[語幹](../Page/語幹.md "wikilink")につくことで受身・尊敬・可能・自発を表したが、母音語幹動詞につく場合には母音の連続を避けるため、rが挿入されている。現代語ではeが挿入されたことでrがさらに挿入された。

  - 書く：kak+ar+u（書かる）→kak+are+r+u（書かれる）
  - 起く（起きる）：oki+r+ar+u（起きらる）→oki+r+are+r+u（起きられる）

他に「る」「さす（させる）」「れば」「れども」なども同じ理由で母音語幹動詞にrやsが挿入されている。
