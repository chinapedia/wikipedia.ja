**Java Card**（ジャバカード）は、[サン・マイクロシステムズ](../Page/サン・マイクロシステムズ.md "wikilink")が開発した[ICカード](../Page/ICカード.md "wikilink")プラットフォーム。[1996年](../Page/1996年.md "wikilink")に発表された。[アメリカ国防総省](../Page/アメリカ国防総省.md "wikilink")の[身分証カードとして採用された事例もある](https://ja.wikipedia.org/wiki/身分証明書 "wikilink")\[1\]。

## 概要

Java Cardテクノロジーを使用すると、スマートカードなどのメモリがごく限られたデバイスで、[Java](https://ja.wikipedia.org/wiki/Java "wikilink")テクノロジーを利用した小さな[アプリケーション](../Page/アプリケーションソフトウェア.md "wikilink")（[アプレット](https://ja.wikipedia.org/wiki/アプレット "wikilink")と呼ばれる）を実行することができる\[2\]。

Java Cardアプリケーション環境はスマートカード製造業者に[ライセンス](../Page/ライセンス.md "wikilink")される。Java Card APIは正式な国際標準や業界固有の標準と互換性がある。

Java Card API仕様に従ったJava Cardアプレットであれば、Java Cardアプリケーション環境を使用して開発された他社製カードでも動作する。1枚のカードで1つ以上のアプレットが動作できる。カードが顧客に発行された後で、新たなアプレットをカードにインストールできる。
