## 曲線の求め方

水素イオン指数 \(\mathrm{pH} \,\) は、水素イオン[活量](../Page/活量.md "wikilink")を \(a\mathrm{H}^+ \,\) と表すとき、次式により定義される。なお、[対数](../Page/対数.md "wikilink") (log) は底を10とする[常用対数](https://ja.wikipedia.org/wiki/常用対数 "wikilink")を使用する。

\[\mathrm{pH} = -\log_{10} a\mathrm{H}^+ = \log_{10} \frac{1}{a\mathrm{H}^+}\]

なお、活量は[デバイ-ヒュッケルの式](https://ja.wikipedia.org/wiki/デバイ-ヒュッケルの式 "wikilink")で[近似](https://ja.wikipedia.org/wiki/近似 "wikilink")することが可能であるが、計算が極めて煩雑となるため活量計数はすべて1と仮定し、ここでは活量の代わりに[モル濃度](../Page/モル濃度.md "wikilink")を用いる。しかし0.1[mol/l](https://ja.wikipedia.org/wiki/mol/l "wikilink")程度の水溶液でも活量によるpHの変化は0.1〜0.5程度に及ぶことに注意する必要がある。一般に弱酸を滴定する場合、pHは計算値より低い方へシフトする。[イオン](../Page/イオン.md "wikilink")の[電荷](../Page/電荷.md "wikilink")が大きいとき程、この影響は著しい。[水素イオン](https://ja.wikipedia.org/wiki/水素イオン "wikilink")のモル濃度すなわち水素イオン濃度は \({[\mathrm{H}^+]} \,\) で表す。

\[\mathrm{pH} = -\log_{10} [\mathrm{H}^+] = \log_{10} \frac{1}{[\mathrm{H}^+]}\]

また、溶液のpHを求めるには、まず水素イオン濃度を求めることになる。これには以下のことを考慮する。

  - [溶質](https://ja.wikipedia.org/wiki/溶質 "wikilink")の[酸解離定数](https://ja.wikipedia.org/wiki/酸解離定数 "wikilink")
  - [水](../Page/水.md "wikilink")の[自己解離](https://ja.wikipedia.org/wiki/自己解離 "wikilink")定数

これらを用いてそれぞれの[電離平衡](https://ja.wikipedia.org/wiki/電離平衡 "wikilink")に関する[質量作用の法則](https://ja.wikipedia.org/wiki/質量作用の法則 "wikilink")の式を立てる。次に

  - 溶質の物質収支に関する式
  - [電離](https://ja.wikipedia.org/wiki/電離 "wikilink")により生成したイオンについて電気的中性原理の式

これらの[連立方程式](https://ja.wikipedia.org/wiki/連立方程式 "wikilink")を解くことになる。また滴定に伴い[溶液](../Page/溶液.md "wikilink")の[体積](https://ja.wikipedia.org/wiki/体積 "wikilink")が増加するため、濃度変化も考慮しなければならない。

## 1価の酸を1価の塩基で滴定

### 強酸を強塩基で滴定

[塩酸](https://ja.wikipedia.org/wiki/塩酸 "wikilink")を[水酸化ナトリウム](../Page/水酸化ナトリウム.md "wikilink")水溶液で滴定する場合を考える。 [強酸](../Page/強酸.md "wikilink")および[強塩基](https://ja.wikipedia.org/wiki/強塩基 "wikilink")は完全に電離しているものと仮定する。また水の自己解離の[化学平衡](../Page/化学平衡.md "wikilink")は以下のようになる。また[オキソニウムイオン](../Page/オキソニウムイオン.md "wikilink") (H<sub>3</sub>O<sup>+</sup>)は水素イオンとして扱う。

  -
    <ce>H2O \\ \<=\> \\ {H^+} + OH^-</ce>

\[K_w = [\mbox{H}^+][\mbox{OH}^-] = 1.008 \times 10^{-14}\]

物質収支を考慮し、塩酸の全濃度\(C_A \,\)とすると、塩酸はすべてイオン化しているから

  -
    <ce>C_{A} \\ = \\ \[Cl^{-}\]</ce>

また電気的中性の原理より

  -
    <ce>{\[H^+\]} + \[{Na}^+\] \\ = \\ {\[{OH}^-\]} + \[{Cl}^{-}\]</ce>

これらの式から水素イオン濃度\[H<sup>+</sup>\]に関する[二次方程式](../Page/二次方程式.md "wikilink")が得られる。

\[[\mbox{H}^+]^2 + [\mbox{H}^+]([\mbox{Na}^+] - [\mbox{Cl}^-]) - K_w = 0 \,\]

また塩酸の全濃度 \(C_A \,\) は、滴定前の塩酸の体積を \(V_A \,\) 、塩酸の初濃度を \(C_{A0} \,\) 、滴下した水酸化ナトリウム水溶液の体積を \(V_B \,\) 、水酸化ナトリウム水溶液の初濃度を \(C_{B0} \,\) とし、水酸化ナトリウムおよび、生成した[塩化ナトリウム](../Page/塩化ナトリウム.md "wikilink")はすべて電離しているとすると、ナトリウムイオンの物質量は \(C_{B0} V_B \,\) となるから

\[C_A = {[\mbox{Cl}^-]} = \frac{C_{A0} V_A} {V_A + V_B}\]

\[{[\mbox{Na}^+]} = \frac{C_{B0} V_B} {V_A + V_B}\]

この二次方程式を[解の公式](https://ja.wikipedia.org/wiki/解の公式 "wikilink")により解き、正の根が水素イオン濃度となり、水素イオン濃度 \({[\mathrm{H}^+]} \,\) を \(\mathrm{pH} \,\) に変換すればよい。

\[{[\mbox{H}^+]} = \frac{-[\mbox{Na}^+] + [\mbox{Cl}^-] + \sqrt{([\mbox{Na}^+] - [\mbox{Cl}^-])^2 + 4K_w}} {2}\]

#### 簡便近似法

[thumb](https://ja.wikipedia.org/wiki/画像:Titration-HCl-NaOH.jpg "wikilink") 以下のように近似してもほとんど同じ結果を与える。

**滴定開始から[当量点](https://ja.wikipedia.org/wiki/当量点 "wikilink")まで**は、二次方程式の\(K_w \,\)の項が無視し得るため

\[[\mbox{H}^+] + [\mbox{Na}^+] - [\mbox{Cl}^-] \fallingdotseq 0\] となり

滴定前の塩酸の[物質量](https://ja.wikipedia.org/wiki/物質量 "wikilink")は \(C_{A0} V_A \,\) [ミリ](https://ja.wikipedia.org/wiki/ミリ "wikilink")[モル](../Page/モル.md "wikilink")、滴下した水酸化ナトリウムの物質量が \(C_{B0} V_B \,\) ミリモルであるから、未反応の塩酸の水素イオンの物質量は \(C_{A0} V_A - C_{B0} V_B \,\) ミリモルとなり、滴定中の溶液の体積が \(V_A + V_B \,\) [ミリリットル](https://ja.wikipedia.org/wiki/ミリリットル "wikilink")であるから、これよりモル濃度を計算する。

\[{[\mbox{H}^+]} = \frac{C_{A0} V_A - C_{B0} V_B} {V_A + V_B}\]

\[\mathrm{pH} = -\log_{10} \frac{C_{A0} V_A - C_{B0} V_B} {V_A + V_B}\]

**当量点**は塩化ナトリウム水溶液となり[中性であるから](https://ja.wikipedia.org/wiki/中性_\(酸塩基\) "wikilink")

\[\mathrm{pH} = \frac{1} {2} \mbox{p}K_w = 7.0\]

**当量点以降**は、二次方程式の \([\mbox{H}^+]^2 \,\) の項は充分小さく

\[[\mbox{H}^+]([\mbox{Na}^+] - [\mbox{Cl}^-]) - K_w \fallingdotseq 0\] となるから

過剰の水酸化ナトリウムの物質量 \(C_{B0} V_B - C_{A0} V_A \,\) と濃度を考える。

\[{[\mbox{OH}^-]} = \frac{C_{B0} V_B - C_{A0} V_A} {V_A + V_B}\]

\[{[\mbox{H}^+]} = \frac{K_w} {[\mbox{OH}^-]}\] であるから

\[{[\mbox{H}^+]} = \frac{(V_A + V_B) K_w} {C_{B0} V_B - C_{A0} V_A}\]

\[\mathrm{pH} = -\log_{10} \frac{(V_A + V_B) K_w} {C_{B0} V_B - C_{A0} V_A}\] {{-}}

### 弱酸を強塩基で滴定

[酢酸](../Page/酢酸.md "wikilink")を水酸化ナトリウム水溶液で滴定する場合を考える。酢酸では当量点におけるpH変化は著しいが、極めて酸性の弱い[シアン化水素](../Page/シアン化水素.md "wikilink")酸では当量点のpH変化が不明瞭になる。 水酸化ナトリウムは完全に電離しているものと仮定する。また酢酸の[電離平衡](https://ja.wikipedia.org/wiki/電離平衡 "wikilink")は以下のようになる。

  -
    <ce>CH3COOH \<=\> {H+} + CH3COO^-</ce>
    \(K_a = \frac{[\mbox{H}^+][\mbox{CH}_3\mbox{COO}^-]} {[\mbox{CH}_3\mbox{COOH}]} = 1.75 \times 10^{-5}\)

<!-- end list -->

  -

    \=

物質収支を考慮し、酢酸の全濃度 \(C_A  \,\) とすると

  -
    <ce>C_{A} \\ = \\ {\[CH3COOH\]} + \[CH3COO^{-}\]</ce>

また電気的中性の原理より

  -
    <ce>\[H+\] + \[Na+\] \\ = \\ \[OH^{-}\] + \[CH3COO^{-}\]</ce>

これらの式および水の自己解離平衡から水素イオン濃度\[H<sup>+</sup>\]に関する[三次方程式](../Page/三次方程式.md "wikilink")が得られる。

\[[\mbox{H}^+]^3 + [\mbox{H}^+]^2(K_a + [\mbox{Na}^+]) + [\mbox{H}^+](K_a[\mbox{Na}^+] - K_aC_A - K_w) - K_aK_w = 0 \,\]

また酢酸の全濃度 \(C_A \,\) は、滴定前の酢酸の体積を \(V_A \,\) 、酢酸の初濃度を \(C_{A0} \,\) 、滴下した水酸化ナトリウム水溶液の体積を \(V_B \,\) 、水酸化ナトリウム水溶液の初濃度を \(C_{B0} \,\) とすると

\[C_A = \frac{C_{A0} V_A} {V_A + V_B}\]

\[{[\mbox{Na}^+]} = \frac{C_{B0} V_B} {V_A + V_B}\]

この三次方程式の正の[実数](../Page/実数.md "wikilink")根が水素イオン濃度となるが解法が複雑となるため、酸性領域では \(K_w \,\) の影響、塩基性領域では \([\mbox{H}^+]^3 \,\) の項は充分に小さく無視し得るため二次方程式で近似が可能となる。

\[[\mbox{H}^+]^2 + [\mbox{H}^+](K_a + [\mbox{Na}^+]) + K_a[\mbox{Na}^+] - K_aC_A \fallingdotseq 0\]

\[[\mbox{H}^+]^2(K_a + [\mbox{Na}^+]) + [\mbox{H}^+](K_a[\mbox{Na}^+] - K_aC_A - K_w) - K_aK_w \fallingdotseq 0\]

また以下のような近似が可能であるが、滴定初期および当量点付近で[誤差](../Page/誤差.md "wikilink")が大きくなる。

**滴定前**は酢酸の[電離度](https://ja.wikipedia.org/wiki/電離度 "wikilink")を考える。電離により生成した水素イオンと酢酸イオンの濃度が等しく、電離度が小さいため、未電離の酢酸の濃度 \([\mbox{CH}_3\mbox{COOH}] \,\) が、全濃度 \(C_{A0} \,\) にほぼ等しいと近似して

\[K_a \fallingdotseq \frac{[\mbox{H}^+]^2}{C_{A0}}\]

\[{[\mbox{H}^+]} \fallingdotseq \sqrt{C_{A0} K_a}\]

\[\mathrm{pH} \fallingdotseq \frac{1} {2} (\mbox{p}K_a - \log_{10}C_{A0})\]

**滴定開始から当量点まで**は、酢酸の電離平衡の式を変形して

\[\mathrm{pH} = \mbox{p}K_a + \log_{10} \frac{[\mbox{CH}_3\mbox{COO}^-]} {[\mbox{CH}_3\mbox{COOH}]}\]

また、生成した酢酸イオンの物質量は加えた水酸化ナトリウムの物質量にほぼ相当し \(C_{B0} V_B \,\) 、未電離の[分子](../Page/分子.md "wikilink")状態の酢酸の物質量はほぼ \(C_{A0} V_A - C_{B0} V_B \,\) であるから

\[\mathrm{pH} = \mbox{p}K_a + \log_{10} \frac{C_{B0} V_B} {C_{A0} V_A - C_{B0} V_B}\]

**当量点**は[酢酸ナトリウム](https://ja.wikipedia.org/wiki/酢酸ナトリウム "wikilink")水溶液であるから酢酸イオンの[加水分解](../Page/加水分解.md "wikilink")を考慮する。加水分解により生成した酢酸分子と水酸化物イオンの物質量はほぼ等しいから

  -
    <ce>{CH3COO^{-}} + H2O \\ \<=\> \\ {CH3COOH} + OH^{-}</ce>

\[K_b = \frac{[\mbox{CH}_3\mbox{COOH}][\mbox{OH}^-]} {[\mbox{CH}_3\mbox{COO}^-]} = \frac{K_w} {K_a} \fallingdotseq \frac{[\mbox{OH}^-]^2} {C_A}\]

これらの式と水の自己解離より

\[{[\mbox{H}^+]} \fallingdotseq \sqrt{\frac{K_a K_w} {C_A}}\]

\[C_A = \frac{C_{A0} V_A} {V_A + V_B}\]

\[\mathrm{pH} \fallingdotseq \frac{1} {2} (\mbox{p}K_a + \mbox{p}K_w + \log_{10}C_{A})\]

**当量点以降**は過剰の水酸化ナトリウムの物質量と濃度を考える。塩酸を水酸化ナトリウムで滴定した場合とほぼ等しい。

\[{[\mbox{OH}^-]} = \frac{C_{B0} V_B - C_{A0} V_A} {V_A + V_B}\]

\[{[\mbox{H}^+]} = \frac{K_w} {[\mbox{OH}^-]}\] であるから

\[{[\mbox{H}^+]} = \frac{(V_A + V_B) K_w} {C_{B0} V_B - C_{A0} V_A}\]

\[\mathrm{pH} = -\log_{10} \frac{(V_A + V_B) K_w} {C_{B0} V_B - C_{A0} V_A}\] {{-}}

### 強酸を弱塩基で滴定

塩酸を[アンモニア](../Page/アンモニア.md "wikilink")水で滴定する場合を考える。アンモニアでは当量点のpH変化が著しいが、より弱い塩基である[ピリジン](https://ja.wikipedia.org/wiki/ピリジン "wikilink")では当量点は不明瞭になる。 塩酸は完全に電離しているものと仮定する。またアンモニア水の電離平衡は以下のようになる。

  -
    <ce>NH3 + H2O \<=\> NH4+ + OH-</ce>

\[K_b = \frac{[\mbox{NH}_4^+][\mbox{OH}^-]} {[\mbox{NH}_3]} = 1.76 \times 10^{-5}\]

\[K_a = \frac{[\mbox{H}^+][\mbox{NH}_3]} {[\mbox{NH}_4^+]} = \frac{K_w} {K_b} = 5.7 \times 10^{-10}\]

  -

    \=

物質収支を考慮し、アンモニアの全濃度を \(C_B \,\) とすると

  -
    <ce>C_{B} = \[NH4+\] + \[NH3\]</ce>

また電気的中性の原理より

  -
    <ce>\[H+\] + \[NH4+\]\\ =\\ \[OH^{-}\] + Cl-</ce>

これらの式および水の自己解離平衡から水素イオン濃度\[H<sup>+</sup>\]に関する三次方程式が得られる。

\[[\mbox{H}^+]^3 + [\mbox{H}^+]^2(K_a + C_B - [\mbox{Cl}^-]) - [\mbox{H}^+](K_a[\mbox{Cl}^-] + K_w) - K_aK_w = 0 \,\]

また塩酸の全濃度 \(C_A \,\) は、滴定前の塩酸の体積を \(V_A \,\) 、塩酸の初濃度を \(C_{A0} \,\) 、とし、アンモニアの全濃度 \(C_B \,\) は、滴下したアンモニア水の体積を \(V_B \,\) 、アンモニア水の初濃度を \(C_{B0} \,\) とすると

\[C_A = {[\mbox{Cl}^-]} = \frac{C_{A0} V_A} {V_A + V_B}\]

\[C_B = \frac{C_{B0} V_B} {V_A + V_B}\]

酸性領域では \(K_w \,\) の影響は無視し得るため

\[[\mbox{H}^+]^2 + [\mbox{H}^+](K_a + C_B - [\mbox{Cl}^-]) - K_a[\mbox{Cl}^-] \fallingdotseq 0\]

塩基性領域では \([\mbox{H}^+]^3 \,\) の項は充分小さく

\[[\mbox{H}^+]^2(K_a + C_B - [\mbox{Cl}^-]) - [\mbox{H}^+](K_a[\mbox{Cl}^-] + K_w) - K_aK_w \fallingdotseq 0\]

**滴定開始から当量点**までは、未反応の塩酸の水素イオンの物質量と濃度を考える。塩酸を水酸化ナトリウムで滴定した場合にほぼ等しい。

**当量点**は[塩化アンモニウム](../Page/塩化アンモニウム.md "wikilink")水溶液であるからアンモニウムイオンの酸解離を考慮する。

  -
    <ce>NH4+ \\ \<=\> \\ {NH3} + H+</ce>

**当量点以降**はアンモニウムイオンの電離平衡の式を変形して

\[\mathrm{pH} = \mbox{p}K_a + \log_{10} \frac{[\mbox{NH}_3]} {[\mbox{NH}_4^+]}\]

また、生成したアンモニウムイオンの物質量は最初に存在した塩酸にほぼ相当し \(C_{A0} V_A \,\) 、分子状態のアンモニアの物質量はほぼ \(C_{B0} V_B - C_{A0} V_A \,\) であるから

\[\mathrm{pH} = \mbox{p}K_a + \log_{10} \frac{C_{B0} V_B - C_{A0} V_A} {C_{A0} V_A}\] {{-}}

### 弱酸を弱塩基で滴定

酢酸をアンモニア水で滴定する場合を考える。当量点はやや不明瞭になり、これより弱い酸および塩基の中和では当量点はさらに不明瞭なものになる。

また酢酸、アンモニア水および水の電離平衡は以下のようになる。

\[K_{a1} = \frac{[\mbox{H}^+][\mbox{CH}_3\mbox{COO}^-]} {[\mbox{CH}_3\mbox{COOH}]} = 1.75 \times 10^{-5}\]

\[K_{a2} = \frac{[\mbox{H}^+][\mbox{NH}_3]} {[\mbox{NH}_4^+]} = \frac{K_w} {K_b} = 5.7 \times 10^{-10}\]

\[K_w = [\mbox{H}^+][\mbox{OH}^-] = 1.008 \times 10^{-14}\]

物質収支を考慮し、酢酸の全濃度を \(C_A \,\) 、アンモニアの全濃度を \(C_B \,\) とすると

\[C_A = [\mbox{CH}_3\mbox{COOH}] + [\mbox{CH}_3\mbox{COO}^-] \,\]

\[C_B = [\mbox{NH}_4^+] + [\mbox{NH}_3] \,\]

また電気的中性の原理より

\[[\mbox{H}^+] + [\mbox{NH}_4^+] = [\mbox{OH}^-] + [\mbox{CH}_3\mbox{COO}^-] \,\]

これらの式および水の自己解離平衡から水素イオン濃度\[H<sup>+</sup>\]に関する[四次方程式](https://ja.wikipedia.org/wiki/四次方程式 "wikilink")が得られる。

\[[\mbox{H}^+]^4 + [\mbox{H}^+]^3(K_{a1} + K_{a2} + C_B) + [\mbox{H}^+]^2(K_{a1}K_{a2} - K_{a1}C_A + K_{a1}C_B - K_w) - [\mbox{H}^+](K_{a1}K_{a2}C_A + K_{a1}K_w + K_{a2}K_w) - K_{a1}K_{a2}K_w = 0\]

また酢酸の全濃度 \(C_A \,\) は、滴定前の酢酸の体積を \(V_A \,\) 、酢酸の初濃度を \(C_{A0} \,\) 、滴下したアンモニア水の体積を \(V_B \,\) 、とし、アンモニアの全濃度 \(C_B \,\) は、アンモニア水の初濃度を \(C_{B0} \,\) とすると

\[C_A = \frac{C_{A0} V_A} {V_A + V_B}\]

\[C_B = \frac{C_{B0} V_B} {V_A + V_B}\]

酸性領域ではアンモニウムイオンの解離および \(K_w \,\) の影響は無視し得るため

\[[\mbox{H}^+]^2 + [\mbox{H}^+](K_{a1} + K_{a2} + C_B) - K_{a1}C_A + K_{a1}C_B \fallingdotseq 0\]

当量点付近では \([\mbox{H}^+]^4 \,\) 項と[定数項](https://ja.wikipedia.org/wiki/定数項 "wikilink")の寄与は小さく

\[[\mbox{H}^+]^2(K_{a1} + K_{a2} + C_B) + [\mbox{H}^+](K_{a1}K_{a2} - K_{a1}C_A + K_{a1}C_B - K_w) - K_{a1}K_{a2}C_A - K_{a1}K_w - K_{a2}K_w = 0 \,\]

塩基性領域では \([\mbox{H}^+]^4 \,\) 項および \([\mbox{H}^+]^3 \,\) 項は充分小さく

\[[\mbox{H}^+]^2(K_{a1}K_{a2} - K_{a1}C_A + K_{a1}C_B - K_w) - [\mbox{H}^+](K_{a1}K_{a2}C_A + K_{a1}K_w + K_{a2}K_w) - K_{a1}K_{a2}K_w \fallingdotseq 0\]

#### 簡便近似法

[thumb](https://ja.wikipedia.org/wiki/画像:Titration-AcOH-NH3.jpg "wikilink") また以下のような近似が可能であるが、滴定初期および当量点付近で誤差が大きくなる。

**滴定前**は酢酸の電離度を考える。電離により生成した水素イオンと酢酸イオンの濃度が等しいと近似して

\[\mathrm{pH} \fallingdotseq \frac{1} {2} (\mbox{p}K_a - \log_{10}C_{A0})\]

**滴定開始から当量点まで**は、酢酸の電離平衡の式を変形して

\[\mathrm{pH} = \mbox{p}K_a + \log_{10} \frac{[\mbox{CH}_3\mbox{COO}^-]} {[\mbox{CH}_3\mbox{COOH}]}\]

また、生成した酢酸イオンの物質量は加えたアンモニアに相当し \(C_{B0} V_B \,\) 、分子状態の酢酸の物質量は \(C_{A0} V_A - C_{B0} V_B \,\) であるから

\[\mathrm{pH} = \mbox{p}K_a + \log_{10} \frac{C_{B0} V_B} {C_{A0} V_A - C_{B0} V_B}\]

**当量点**は[酢酸アンモニウム](https://ja.wikipedia.org/wiki/酢酸アンモニウム "wikilink")水溶液であり、アンモニウムイオンと酢酸イオンの平衡を考える。

  -
    <ce>NH4+ + CH3COO- \\ \<=\>\\ NH3 + CH3COOH</ce>

ここで生成する酢酸とアンモニアの物質量はほぼ等しい。また酢酸イオンとアンモニウムイオンの濃度もほぼ等しいから、酢酸およびアンモニウムイオンの酸解離定数の積は

\[K_{a1}K_{a2} = \frac{[\mbox{H}^+]^2[\mbox{CH}_3\mbox{COO}^-][\mbox{NH}_3]} {[\mbox{CH}_3\mbox{COOH}][\mbox{NH}_4^+]} = 1.00 \times 10^{-14}\]

これらより以下の式が導かれ、pHは濃度にほとんど依存しない。

\[{[\mbox{H}^+]} \fallingdotseq \sqrt{K_{a1} K_{a2}}\]

\[\mathrm{pH} \fallingdotseq \frac{1} {2} (\mbox{p}K_{a1} + \mbox{p}K_{a2})\]

**当量点以降**はアンモニウムイオンの電離平衡の式を変形して

\[\mathrm{pH} = \mbox{p}K_a + \log_{10} \frac{[\mbox{NH}_3]} {[\mbox{NH}_4^+]}\]

また、生成したアンモニウムイオンの物質量は最初に存在した酢酸にほぼ相当し \(C_{A0} V_A \,\) 、[分子](../Page/分子.md "wikilink")状態のアンモニアの物質量はほぼ \(C_{B0} V_B - C_{A0} V_A \,\)であるから

\[\mathrm{pH} = \mbox{p}K_a + \log_{10} \frac{C_{B0} V_B - C_{A0} V_A} {C_{A0} V_A}\] {{-}}

### 弱酸を強塩基で滴定

多段階で電離する酸の解離の計算は大変複雑である。[シュウ酸](../Page/シュウ酸.md "wikilink")は2価の酸であり、一段目がやや強く電離し、二段目もそれほど小さくないため、第一当量点は明瞭でなく第二当量点のpH変化が著しい。[炭酸](https://ja.wikipedia.org/wiki/炭酸 "wikilink")はより弱酸であるため当量点は不明瞭になる。[酒石酸](../Page/酒石酸.md "wikilink")は一段目および二段目の解離定数の差が小さいため、第一当量点は全く検出されず第二等量点のみ顕著に現れる。[硫化水素](https://ja.wikipedia.org/wiki/硫化水素 "wikilink")酸は第一当量点のみ観測され、二段目の解離定数が著しく小さいため第二等量点を検出することができない。

[リン酸](../Page/リン酸.md "wikilink")は3価であるが第一および第二当量点で著しいpH変化が見られ、三段目の解離定数が小さいため第三当量点は不明瞭でほとんど観測されない。[クエン酸](../Page/クエン酸.md "wikilink")も3価であるが、一段〜三段までの解離定数の差が小さいため、第一および第二当量点は不明瞭で第三当量点のみpHの著しい変化が見られる。

また以下のような近似が可能であるが、滴定初期および当量点付近で誤差が大きくなる。

**滴定前**は炭酸の電離度を考える。一段目のみの解離を考慮し、二段目は極めて小さいため無視し得る。電離により生成した水素イオンと炭酸水素イオンの濃度が等しいと近似して

\[{[\mbox{H}^+]} \fallingdotseq \sqrt{C_{A0} K_{a1}}\]

\[\mathrm{pH} \fallingdotseq \frac{1} {2} (\mbox{p}K_{a1} - \log_{10}C_{A0})\]

**滴定開始から第一当量点まで**は、炭酸の一段目の電離平衡の式を変形して

\[\mathrm{pH} = \mbox{p}K_{a1} + \log_{10} \frac{[\mbox{HCO}_3^-]} {[\mbox{H}_2\mbox{CO}_3]}\]

また、生成した炭酸水素イオンの物質量は加えた水酸化ナトリウムにほぼ相当し \(C_{B0} V_B \,\) 、分子状態の炭酸の物質量はほぼ \(C_{A0} V_A - C_{B0} V_B \,\) であるから

\[\mathrm{pH} = \mbox{p}K_{a1} + \log_{10} \frac{C_{B0} V_B} {C_{A0} V_A - C_{B0} V_B}\]

**第一当量点**は[炭酸水素ナトリウム](../Page/炭酸水素ナトリウム.md "wikilink")水溶液であり、炭酸水素イオンの[不均化](https://ja.wikipedia.org/wiki/不均化 "wikilink")を考える。

  -
    <ce>2HCO3^{-} \\ \<=\> \\ {H2CO3} + CO3^{2-}</ce>

ここで生成する炭酸および炭酸イオンの物質量はほぼ等しい。次に第一および第二段階の酸解離定数の積は

\[K_{a1}K_{a2} = \frac{[\mbox{H}^+]^2[\mbox{CO}_3^{2-}]} {[\mbox{H}_2\mbox{CO}_3]} = 2.13 \times 10^{-17}\]

これらより以下の式が導かれ、pHは濃度にほとんど依存しない。

\[{[\mbox{H}^+]} \fallingdotseq \sqrt{K_{a1} K_{a2}}\]

\[\mathrm{pH} \fallingdotseq \frac{1} {2} (\mbox{p}K_{a1} + \mbox{p}K_{a2})\]

**第一当量点から第二当量点まで**は、炭酸の二段目の電離平衡の式を変形して

\[\mathrm{pH} = \mbox{p}K_{a2} + \log_{10} \frac{[\mbox{CO}_3^{2-}]} {[\mbox{HCO}_3^-]}\]

また、生成した炭酸イオンの物質量は加えた水酸化ナトリウムから、第一当量点までに消費された分を差し引いた物質量に相当し \(C_{B0} V_B - C_{A0} V_A \,\) 、炭酸水素イオンの物質量は \(2C_{A0} V_A - C_{B0} V_B \,\) であるから

\[\mathrm{pH} = \mbox{p}K_{a2} + \log_{10} \frac{C_{B0} V_B - C_{A0} V_A} {2C_{A0} V_A - C_{B0} V_B}\]

**第二当量点**は[炭酸ナトリウム](../Page/炭酸ナトリウム.md "wikilink")水溶液であり、炭酸イオンの加水分解を考慮する。

  -
    <ce>{CO3^{2-}} + H2O\\ \<=\>\\ {HCO3^{-}} + OH-</ce>

\[K_b = \frac{[\mbox{HCO}_3^-][\mbox{OH}^-]} {[\mbox{CO}_3^{2-}]} = \frac{K_w} {K_{a2}}\]

\[{[\mbox{H}^+]} \fallingdotseq \sqrt{\frac{K_{a2} K_w} {C_A}}\]

\[C_A = \frac{C_{A0} V_A} {V_A + V_B}\]

\[\mathrm{pH} \fallingdotseq \frac{1} {2} (\mbox{p}K_{a2} + \mbox{p}K_w + \log_{10}C_{A})\]

**当量点以降**は過剰の水酸化ナトリウムの物質量 \(C_{B0} V_B - 2C_{A0} V_A \,\) と濃度を考える。

\[{[\mbox{H}^+]} = \frac{K_w} {[\mbox{OH}^-]}\] であるから

\[{[\mbox{H}^+]} = \frac{(V_A + V_B) K_w} {C_{B0} V_B - 2C_{A0} V_A}\]

\[\mathrm{pH} = -\log_{10} \frac{(V_A + V_B) K_w} {C_{B0} V_B - 2C_{A0} V_A}\] {{-}}

\]\] [水酸化バリウム](https://ja.wikipedia.org/wiki/水酸化バリウム "wikilink")を塩酸で滴定する場合を考える。水酸化バリウムは強い[二酸塩基](https://ja.wikipedia.org/wiki/二酸塩基 "wikilink")であるが二段目の電離はやや不完全である。しかし滴定曲線は2価の強塩基としての形に近くpHの急激な変化は第二当量点のみに現れる。 水酸化バリウムの一段目は完全に電離しているものと仮定する。また二段目の電離平衡は以下のようになる。

  -
    <ce>Ba(OH)+ \\ \<=\> \\ {Ba^{2+}} + OH-</ce>
    <ce>{Ba^{2+}} + H2O \\ \<=\> \\ {H+} + Ba(OH)+</ce>

\[K_{a1} = \frac{K_w} {K_{b2}} = \frac{[\mbox{H}^+][\mbox{Ba(OH)}^+]} {[\mbox{Ba}^{2+}]} = 4 \times 10^{-14}\]

  -

    \=

物質収支を考慮し、水酸化バリウムの全濃度を \(C_B \,\) とすると

  -
    <ce>C_{B} \\ = \\ \[Ba^{2+}\] + \[Ba(OH)+\]</ce>

また電気的中性の原理より

  -
    <ce>2\[Ba^{2+}\] + \[Ba(OH)+\] + \[H+\] \\ = \\ \[OH^{-}\] + \[Cl^{-}\]</ce>

これらの式および水の自己解離平衡から水素イオン濃度\[H<sup>+</sup>\]に関する三次方程式が得られる。

\[[\mbox{H}^+]^3 + [\mbox{H}^+]^2(K_{a1} + 2C_B - [\mbox{Cl}^-]) + [\mbox{H}^+](K_{a1}C_B - K_{a1}[\mbox{Cl}^-] - K_w) - K_{a1}K_w = 0 \,\]

また水酸化バリウムの全濃度 \(C_B \,\) は、滴定前の水酸化バリウムの体積を \(V_B \,\) 、水酸化バリウムの初濃度を \(C_{B0} \,\) 、滴下した塩酸の体積を \(V_A \,\) 、塩酸水溶液の初濃度を \(C_{A0} \,\) とすると

\[C_B = \frac{C_{B0} V_B} {V_A + V_B}\]

\[{[\mbox{Cl}^-]} = \frac{C_{A0} V_A} {V_A + V_B}\]

酸性領域では \(K_w \,\) の影響は無視し得るため

\[[\mbox{H}^+]^2 + [\mbox{H}^+](K_{a1} + 2C_B - [\mbox{Cl}^-]) + K_{a1}C_B - K_{a1}[\mbox{Cl}^-] \fallingdotseq 0\]

塩基性領域では \([\mbox{H}^+]^3 \,\) の項は充分小さく

\[[\mbox{H}^+]^2(K_{a1} + 2C_B - [\mbox{Cl}^-]) + [\mbox{H}^+](K_{a1}C_B - K_{a1}[\mbox{Cl}^-] - K_w) - K_{a1}K_w \fallingdotseq 0\] {{-}}

### 弱塩基を強酸で滴定

炭酸ナトリウム水溶液を塩酸で滴定する場合を考える。炭酸イオンは2価の塩基と考えることができる。 物質収支を考慮し、炭酸の全濃度を \(C_B \,\) とすると

  -
    <ce>C_{B} \\ = \\ \[H2CO3\] + \[HCO3^{-}\] + \[CO3^{2-}\]</ce>

また電気的中性の原理より

  -
    <ce>\[H+\] + \[Na+\] \\ = \\ \[OH^{-}\] + \[Cl^{-}\] + \[HCO3^{-}\] + 2\[CO3^{2-}\]</ce>

これらの式および水の自己解離平衡から水素イオン濃度\[H<sup>+</sup>\]に関する四次方程式が得られる。

\[[\mbox{H}^+]^4 + [\mbox{H}^+]^3(K_{a1} + 2C_B - [\mbox{Cl}^-]) + [\mbox{H}^+]^2(K_{a1}K_{a2} - K_{a1}[\mbox{Cl}^-] + K_{a1}C_B - K_w) - [\mbox{H}^+](K_{a1}K_{a2}[\mbox{Cl}^-] + K_{a1}K_w) - K_{a1}K_{a2}K_w = 0\]

また炭酸の全濃度 \(C_B \,\) は、滴定前の炭酸ナトリウム水溶液の体積を \(V_B \,\) 、炭酸ナトリウムの初濃度を \(C_{B0} \,\) 滴下した塩酸の体積を \(V_A \,\) 、塩酸の初濃度を \(C_{A0} \,\) とすると

\[C_B = \frac{C_{B0} V_B} {V_A + V_B}\]

\[{[\mbox{Cl}^-]} = \frac{C_{A0} V_A} {V_A + V_B}\]

塩基性領域では \([\mbox{H}^+]^4 \,\) 項および \([\mbox{H}^+]^3 \,\) 項は充分小さく

\[[\mbox{H}^+]^2(K_{a1}K_{a2} - K_{a1}[\mbox{Cl}^-] + K_{a1}C_B - K_w) - [\mbox{H}^+](K_{a1}K_{a2}[\mbox{Cl}^-] + K_{a1}K_w) - K_{a1}K_{a2}K_w \fallingdotseq 0\]

第一当量点付近では \([\mbox{H}^+]^4 \,\) 項と定数項の寄与は小さく

\[[\mbox{H}^+]^2(K_{a1} + 2C_B - [\mbox{Cl}^-]) + [\mbox{H}^+](K_{a1}K_{a2} - K_{a1}[\mbox{Cl}^-] + K_{a1}C_B - K_w) - K_{a1}K_{a2}[\mbox{Cl}^-] - K_{a1}K_w \fallingdotseq 0\]

酸性領域では炭酸の第二段階の解離 \(K_{a2} \,\) および \(K_w \,\) の影響は無視し得るため

\[[\mbox{H}^+]^2 + [\mbox{H}^+](K_{a1} + 2C_B - [\mbox{Cl}^-]) - K_{a1}[\mbox{Cl}^-] + K_{a1}C_B \fallingdotseq 0\]

\]\] また以下のような近似が可能であるが、滴定初期および当量点付近で誤差が大きくなる。

**滴定前**は炭酸イオンの加水分解を考慮する。

  -
    <ce>{CO3^{2-}} + H2O\\ \<=\>\\ {HCO3^{-}} + OH-</ce>

\[K_b = \frac{[\mbox{HCO}_3^-][\mbox{OH}^-]} {[\mbox{CO}_3^{2-}]} = \frac{K_w} {K_{a2}}\]

\[{[\mbox{H}^+]} \fallingdotseq \sqrt{\frac{K_{a2} K_w} {C_{B0}}}\]

\[\mathrm{pH} \fallingdotseq \frac{1} {2} (\mbox{p}K_{a2} + \mbox{p}K_w + \log_{10}C_{B0})\]

**滴定開始から第一当量点まで**は、炭酸の二段目の電離平衡の式を変形して

\[\mathrm{pH} = \mbox{p}K_{a2} + \log_{10} \frac{[\mbox{CO}_3^{2-}]} {[\mbox{HCO}_3^-]}\]

また、生成した炭酸水素イオンの物質量は加えた塩酸に相当し \(C_{A0} V_A \,\) 、炭酸水素イオンの物質量は \(C_{B0} V_B - C_{A0} V_A \,\) であるから

\[\mathrm{pH} = \mbox{p}K_{a2} + \log_{10} \frac{C_{B0} V_B - C_{A0} V_A} {C_{A0} V_A}\]

**第一当量点**は炭酸水素ナトリウムと塩化ナトリウムが生成しているから、炭酸水素イオンの不均化を考える。

  -
    <ce>2HCO3^{-} \\ \<=\> \\ {H2CO3} + CO3^{2-}</ce>

\[\mathrm{pH} \fallingdotseq \frac{1} {2} (\mbox{p}K_{a1} + \mbox{p}K_{a2})\]

**第一当量点から第二当量点まで**は、炭酸の一段目の電離平衡の式を変形して

\[\mathrm{pH} = \mbox{p}K_{a1} + \log_{10} \frac{[\mbox{HCO}_3^-]} {[\mbox{H}_2\mbox{CO}_3]}\]

また、生成した炭酸の物質量は加えた塩酸から、第一当量点までに消費された分を差し引いた物質量にほぼ相当し \(C_{A0} V_A - C_{B0} V_B \,\) 、炭酸水素イオンの物質量はほぼ \(2C_{B0} V_B - C_{A0} V_A \,\) であるから

\[\mathrm{pH} = \mbox{p}K_{a1} + \log_{10} \frac{2C_{B0} V_B - C_{A0} V_A} {C_{A0} V_A - C_{B0} V_B}\]

**第二当量点**は塩化ナトリウムと炭酸が生成しているから、炭酸の電離を考慮する。一段目のみの解離を考慮し、二段目は極めて小さいため無視し得る。

\[{[\mbox{H}^+]} \fallingdotseq \sqrt{C_{B} K_{a1}}\]

\[C_B = \frac{C_{B0} V_B} {V_A + V_B}\]

\[\mathrm{pH} \fallingdotseq \frac{1} {2} (\mbox{p}K_{a1} - \log_{10}C_{B})\]

**当量点以降**は過剰の塩酸の物質量 \(C_{A0} V_A - 2C_{B0} V_B \,\) と濃度を考える。

\[\mathrm{pH} = -\log_{10} \frac{C_{A0} V_A - 2C_{B0} V_B} {V_A + V_B}\] {{-}}
