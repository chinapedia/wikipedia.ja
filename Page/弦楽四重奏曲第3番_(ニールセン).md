**弦楽四重奏曲第3番 変ホ長調** [作品14は](../Page/作品番号.md "wikilink")、[カール・ニールセン](../Page/カール・ニールセン.md "wikilink")が1897年、1898年に作曲した[弦楽四重奏曲](../Page/弦楽四重奏曲.md "wikilink")。ニールセンが書いた公式の4曲の弦楽四重奏曲では3番目の作品にあたる。初演は1899年5月1日に『Vor Forening』にて[アントン・スヴェンスン](https://ja.wikipedia.org/wiki/アントン・スヴェンスン "wikilink")、[ルズヴィ・ホルム](https://ja.wikipedia.org/wiki/ルズヴィ・ホルム "wikilink")、フレズレク・マーガ（Marke）、アイラ・イェンスンの演奏により私的に行われた。

## 概要

1897年と1898年にニールセンは忙しく過ごしていた。[オペラ](../Page/オペラ.md "wikilink")『[サウルとダヴィデ](../Page/サウルとダヴィデ.md "wikilink")』の作曲準備をしていただけでなく[ユトランド半島](../Page/ユトランド半島.md "wikilink")[コリング](../Page/コリング.md "wikilink")に程近い義理の両親が営む農園の世話を見ながら過ごしていたのである。第3楽章と第4楽章を浄書に回そうとした際に不幸な経験に見舞われている。彼が写譜者のもとに到着したとき、1頭の馬が荷車の前で辛い姿勢で横たわっていた。彼は楽譜を巻いて少年に手渡し馬が起き上がるのを手伝った。しかしわずか数分の間に少年は楽譜ごと姿を消してしまっていたのである。ニールセンはもう一度これらを一から書き起こさねばならなかった\[1\]。

## 受容

[Georg_Høeberg.gif](https://ja.wikipedia.org/wiki/File:Georg_Høeberg.gif "fig:Georg_Høeberg.gif")。\]\] 私的な演奏は一度ならず行われたらしいことがわかっているが、第3弦楽四重奏曲のコンサート会場での公開演奏は1901年10月4日の[コペンハーゲン](../Page/コペンハーゲン.md "wikilink")のOdd Fellows Mansion小ホールが初めてとなる。これは[ギーオウ・フベア](https://ja.wikipedia.org/wiki/ギーオウ・フベア "wikilink")、Louis Witzansky、アントン・ブロク、エアンスト・フベアから成る新しく結成されたフベア四重奏団の初の室内楽コンサートであった\[2\]。

評論家は演奏家を褒めはしたものの、音楽に称賛を寄せるものはいなかった。『Nationaltidende』紙のGustav Hetschは第1楽章について「五線紙の上で読むには素晴らしい音楽というようなものに属するのだろうとみえるが、結局のところ響きは不愉快である。我々が音楽的な部分をつかみ取れた僅かな瞬間は、それ以外の不協和音の泡に飲まれて消えていた。」と評した。しかし彼はアンダンテは好んでおり「雰囲気によって非常に高尚なものとなっている（略）この音楽には広大な地平線、蒼穹の星々の豊かさ、そして高尚なる糸杉があり、その音楽は現代的であるが故に過去の音、簡素な色合いに手を届かせんとするのである。」他の評論家も同じように第1楽章に対して批判的であった\[3\]。

## その他の演奏

本作はニールセンの生前にフベア四重奏団やブロイニング＝ベイチュ四重奏団などにより何度も演奏された。第1ヴァイオリンを務めたのは[ピーザ・ムラ](../Page/ピーザ・ムラ.md "wikilink")や[テルマーニー・エミル](https://ja.wikipedia.org/wiki/テルマーニー・エミル "wikilink")らである。1925年にはニールセンの生誕60周年を祝うため大学の式典ホールで開催された3回のコンサートの1回目に演奏されている。『Politiken』紙の評論は「変ホ長調の四重奏曲とその反駁の余地のない熟達の度合い、彼の当音楽ジャンルにおけるこの上なく偉大な勝利、力強く雄々しく、深遠な詩情を持ち、優美に田園的、勇敢に野心的」であるところに言及している。1902年にドイツで行われた演奏にも同様に前向きな論評がなされた\[4\]。

## 演奏時間

約33分\[5\]。

## 楽曲構成

### 第1楽章

  - [Allegro](https://ja.wikipedia.org/wiki/wikt:allegro "wikilink") [con brio](https://ja.wikipedia.org/wiki/wikt:con_brio "wikilink") 3/4拍子 [変ホ長調](../Page/変ホ長調.md "wikilink")

[ロンドソナタ形式](../Page/ロンドソナタ形式.md "wikilink")\[6\]。序奏はなく冒頭から流麗な第1主題が提示される（譜例1）。

譜例1 <score> \\relative c''' {

`\key es \major \time 3/4 \set Score.tempoHideNote = ##t \tempo "Allegro con brio." 4=125`
`\override Score.NonMusicalPaperColumn #'line-break-permission = ##f`
`r8. bes16\f \downbow es,4.( d8) es4( f g) aes( f) bes( ~ bes8 g) es'4. es,16^\( ( g`
`bes4.) des8-.\) bes4~ bes8 a-. c4.-. es,16^`\(( ges bes4.) des8-.\)` c4-> ~ c8 b-. es4.-.`

} </score>

主題が簡潔にまとめられて緩やかな推移が始まるが、[コン・フォーコから速い動きを見せて盛り上がった後に第](https://ja.wikipedia.org/wiki/wikt:con_fuoco "wikilink")2主題がチェロから示される（譜例2）。

譜例2 <score> \\relative c \\new Staff \\with { \\remove "Time_signature_engraver" } {

`\key es \major \time 3/4 \set Score.tempoHideNote = ##t \tempo "" 4=125 \clef bass`
`r8_\markup { \italic { a tempo } } f\p ( bes2-> ~ _\markup { \italic { molto tranq. } } `
`bes8) f\< ( bes2~ bes8) f( bes4. c8) d4( es8 d bes d) f2.\mf \> ~ f\p \< ~`
`f4 r8 g( f es) d2( des4 c\mf )`

} </score>

2つのヴァイオリンが[ポリリズム](../Page/ポリリズム.md "wikilink")を刻む中でヴィオラとチェロが譜例2を[ユニゾンで奏し](../Page/一度.md "wikilink")、静まって提示部を終える。提示部の反復は指示されておらず、譜例1を奏してから展開が行われる。随所にアクセントを含む箇所に続いて弱音で進められ、ブリオーソからは[スタッカート](https://ja.wikipedia.org/wiki/スタッカート "wikilink")で強弱が極端に交代する。そのまま第1主題の再現となり、第2主題はヴィオラが受け持つ。[コーダでは第](https://ja.wikipedia.org/wiki/コーダ_\(音楽\) "wikilink")1ヴァイオリンが譜例1を回想する傍らで奏される急速な音型が全楽器に伝播し、勢いよく楽章の終わりまでたどり着く。

### 第2楽章

  - [Andante](https://ja.wikipedia.org/wiki/wikt:andante "wikilink") [sostenuto](https://ja.wikipedia.org/wiki/wikt:sostenuto "wikilink") - Andante 4/4拍子

[イ短調](https://ja.wikipedia.org/wiki/イ短調 "wikilink")で開始するが、まもなく変ホ長調に落ち着く\[7\]。[アンダンテの主部に入ると表情豊かに譜例](https://ja.wikipedia.org/wiki/wikt:andante "wikilink")3が歌われる。

譜例3 <score> \\relative c'' {

`\key es \major \time 4/4 \set Score.tempoHideNote = ##t \tempo "Andante." 4=60`
`bes2._\markup { \dynamic mp \italic espress. } \< es4~ es f2( g8 aes)`
`g4\mf `\(f8( es)\)` es`\(( des) des( bes)\)` bes2. es4`\(~ es\> bes8( aes)\)` aes`\(( g) g( es)\)` es2 s8\!`

} </score>

クライマックスが形成され、譜例3が再び奏されると[ウン・ポコ](https://ja.wikipedia.org/wiki/wikt:un_poco "wikilink")・[ピウ・モッソとなってヴィオラがリズミカルなリズムを導入する](https://ja.wikipedia.org/wiki/wikt:più_mosso "wikilink")。これに合わせて第2ヴァイオリンが譜例4を奏し、付点音符の掛け合いとなる。

譜例4 <score> \\relative c' \\new Staff \\with { \\remove "Time_signature_engraver" } {

`\key es \major \time 4/4 \set Score.tempoHideNote = ##t \tempo "" 4=60 \partial 4`
`f4\p( e8.[ d16-.) e8.-.( f16-.) ] d8.-.( f16-.) f4( ~ f8.[ e16-.) f8.-.( g16-.) ] e8.-.( g16-.) g4( ~`
`g8.\< [ fis16-.) g8.-.( a16-.) ] bes8.-.( c16-.) d4( ~`
`d8. e16\! \times 2/3 { f8\> d c) } bes4( \afterGrace a\trill { g16[ a] } g4\p )`

} </score>

低弦が16分音符のうねりを作り出して頂点を迎えてから落ち着くと、チェロが譜例3を再現する。最後は静まって弱音で楽章を閉じる。

### 第3楽章

  - Allegretto pastorale 2/4拍子 [ハ長調](../Page/ハ長調.md "wikilink")

広義の[スケルツォ](../Page/スケルツォ.md "wikilink")と看做される\[8\]。[ピッツィカート](../Page/ピッツィカート.md "wikilink")の伴奏の上に軽妙な主題が奏される（譜例5）。

譜例5 <score> \\relative c''' {

`\key c \major \time 2/4 \set Score.tempoHideNote = ##t \tempo "Allegretto pastorale." 4=88 \partial 4`
`\override Score.NonMusicalPaperColumn #'line-break-permission = ##f`
`g8\p \downbow ( f16 e dis4-.) e8( es16 d c4-.) d8( c16 d e8 d16 e f8 e16 f g4^\markup \italic ten. ) \<`
`c8( b16 bes a4-.) g8( f16 e g4*1/2-.) s8\! aes8_\markup { \italic poco \dynamic f }( g16 f es8\> d16 c) des8( c16 bes g4\! ^\markup \italic ten. )`

} </score>

譜例5を繰り返してから音量を下げて行くと、6/8拍子、[プレストに転じて勢いのある主題が登場する](https://ja.wikipedia.org/wiki/wikt:presto "wikilink")（譜例6）。

譜例6 <score> \\relative c'' {

`\key c \major \time 6/8 \set Score.tempoHideNote = ##t \tempo "Presto." 4=190 \partial 4.`
`<<`
` { e4.~ e8-> e-> e-> e-> e-> e-> e }`
` \\`
` { `
`  e4.\ff ~ e8 e e e e e e e^> fis^> g^> fis^> g^> e^> g^> e^> fis^> gis^> fis^>`
`  e e fis g! fis g e g e fis gis fis e fis gis a b c`
` }`
`>>`

} </score>

より歌謡的な中間楽節を経て譜例6が繰り返される。元の拍子とテンポに戻って譜例5を再現し、最後は静かな終わりを迎える。

### 第4楽章

  - Allegro coraggioso 2/2拍子 変ホ長調

発想表記の"coraggioso"は「勇敢に」などの意。冒頭から勢いよく譜例7の主題が飛び出す。

譜例7 <score> \\relative c''' {

`\key es \major \time 2/2 \set Score.tempoHideNote = ##t`
`\tempo \markup {`
`  \column {`
`   \line { FINALE. }`
`   \line { Allegro coraggioso. }`
`  }`
` } 2=116`
`r4 bes-> \downbow \f g4.-> ( f8-> ) es2.-> bes4-> es2-> bes-> es2.-> bes4-> es2-> f->`
`g-> aes4-> f-> g2( \< bes8-.\! ) r r4 bes1->`

} </score>

第2主題はまずチェロに出され、ヴァイオリンが歌い継いでいく（譜例8）。

譜例8 <score> \\relative c' \\new Staff \\with { \\remove "Time_signature_engraver" } {

`\key es \major \time 2/2 \set Score.tempoHideNote = ##t \tempo "" 2=116 \clef bass`
`r2^\markup { \italic { a tempo ma molto tranq. } } des2~ _\markup { \dynamic p \italic espress. }`
`des4. bes8( es4. bes8 ces4.( aes8) des2~ des4. bes8( es4. bes8 ces4. aes8) des2~ des4. bes8( es4. ces8)`

} </score>

その後クライマックスを形成すると落ち着きを取り戻し、チェロを起点に各楽器がピッツィカートで[フガートを奏する](../Page/フーガ.md "wikilink")。次に譜例7を用いた情熱的な展開、譜例8による落ち着いた展開が行われて、[クレッシェンドして第](../Page/強弱法.md "wikilink")1主題の再現に至る。第2主題の再現はヴィオラから始まり、ここでもヴァイオリンへと続いていく。大きな盛り上がりを見せるとアレグロ・[モルトとなり](https://ja.wikipedia.org/wiki/wikt:molto "wikilink")、譜例7の素材を使ったコーダで勢いを高めてそのまま全曲に終止符を打つ。
