**Sherlock**（シャーロック）とは、[アップルがかつて開発していた](../Page/アップル_\(企業\).md "wikilink")[インターネット](../Page/インターネット.md "wikilink")およびローカルディスクの検索[ソフトウェア](../Page/ソフトウェア.md "wikilink")である。

## 概要

Sherlockは、[Mac OS 8.5の新機能として登場](https://ja.wikipedia.org/wiki/Mac_OS_8 "wikilink")。インターネット上の検索（[インターネット検索](https://ja.wikipedia.org/wiki/インターネット検索 "wikilink")）とローカルディスク内の検索（ファイル検索と内容検索）を同じインターフェイスから利用できる統合検索ツールである。

名称は、[アーサー・コナン・ドイル](../Page/アーサー・コナン・ドイル.md "wikilink")の作品に登場する[名探偵](../Page/名探偵.md "wikilink")[シャーロック・ホームズ](../Page/シャーロック・ホームズ.md "wikilink")に由来\[1\]しており、[1998年](https://ja.wikipedia.org/wiki/1998年 "wikilink")10月17日のMac OS 8.5発売イベントには、当時のアップルコンピュータ社長の[原田永幸](https://ja.wikipedia.org/wiki/原田永幸 "wikilink")がホームズの[仮装](https://ja.wikipedia.org/wiki/仮装 "wikilink")をして参加している\[2\]。また[Mac OS X版の](https://ja.wikipedia.org/wiki/macOS "wikilink")[アイコン](../Page/アイコン.md "wikilink")は、[虫眼鏡とホームズの](https://ja.wikipedia.org/wiki/レンズ#ルーペ "wikilink")[トレードマーク](https://ja.wikipedia.org/wiki/トレードマーク "wikilink")にもなっている[鹿撃ち帽](https://ja.wikipedia.org/wiki/鹿撃ち帽 "wikilink")を組み合わせた画像が採用された。

### Mac OS 8.5

最初のバージョンでは、「インターネット検索」「ファイル検索」「内容で検索」の3つのタブを切り替えて検索を実行する。日本語解析機能により自然な日本語で検索することができた\[3\]。インターネット検索は、[プラグイン](../Page/プラグイン.md "wikilink")形式で組み込んだ各[検索サイト](https://ja.wikipedia.org/wiki/検索サイト "wikilink")から横断的に検索する。[Excite](../Page/Excite.md "wikilink")や[Infoseek](https://ja.wikipedia.org/wiki/Infoseek "wikilink")などがあらかじめ入っていたほか、ユーザー自身がプラグイン（[Yahoo\!](../Page/Yahoo!.md "wikilink")などの他の検索サイト、[Amazon.com](../Page/Amazon.com.md "wikilink")などの[オンラインショップ](https://ja.wikipedia.org/wiki/オンラインショップ "wikilink")内の検索、その他ユーザー自作の検索）を追加できた\[4\]。ファイル検索は、従来の検索機能に相当するもので、ファイル名の[文字列検索又はファイル属性を用いて検索する](https://ja.wikipedia.org/wiki/文字列探索 "wikilink")。内容の検索は、あらかじめ索引を作ったローカルディスク内のファイル内容をから[全文検索](../Page/全文検索.md "wikilink")する。また、検索に設定した条件を保存して、開くと自動的に検索が実行される検索用のファイルを作成することもできる。なお、検索結果を返す画面の下部に[バナー広告](https://ja.wikipedia.org/wiki/バナー広告 "wikilink")が表示される領域が設けられている。

### Mac OS 9

Mac OS 9では**Sherlock 2**\[5\]となり、「チャネル」と呼ばれるグループごと検索がまとめられた。最初に、チャネルアイコンを選択した後に、検索したいキーワードを入力する。チャネルはプラグインの追加によってカスタマイズが可能である。また、インターフェイスデザインも一新され、いわゆるブラシメタル調になった。

#### あらかじめ用意されているチャネル（提供されたプラグインの例）

  - ファイル
    ローカルディスク内の検索。前バージョンの「ファイル検索」「内容で検索」に相当。
  - インターネット
    インターネットの検索サイトを利用しての検索。前バージョンの「インターネット検索」に相当。
  - 人々
    友人・家族に関する情報の検索（[Bigfoot.com](https://ja.wikipedia.org/wiki/:en:Bigfoot_International "wikilink")\[6\]など）。
  - エンターテイメント
    [映画](../Page/映画.md "wikilink")・[テレビ番組](../Page/テレビ番組.md "wikilink")・[音楽](../Page/音楽.md "wikilink")・[イベント](../Page/イベント.md "wikilink")に関する情報の検索（[フジテレビ](https://ja.wikipedia.org/wiki/フジテレビジョン "wikilink")、[TBSなど](../Page/TBSテレビ.md "wikilink")）。
  - ニュース
    最新のニュース配信サイトの検索（[CNET News.com](../Page/CNET.md "wikilink")、[毎日新聞](https://ja.wikipedia.org/wiki/毎日新聞 "wikilink")）。
  - アップル
    アップルに関する情報の検索（Apple.com、Apple Macintosh Products Guidなど）。
  - リファレンス
    辞書・辞典の資料の検索（American Heritage Dictionary、[Britannica.comなど](../Page/ブリタニカ百科事典.md "wikilink")）。
  - ショッピング
    オンラインショップや[ネットオークション](https://ja.wikipedia.org/wiki/ネットオークション "wikilink")の検索（Amazon.com、[eBay](https://ja.wikipedia.org/wiki/eBay "wikilink")など）。

### Mac OS X

Mac OS Xが登場してからも、[Mac OS X v10.1まではSherlock](../Page/Mac_OS_X_v10.1.md "wikilink") 2（デザインは[Aqua調](../Page/Aqua_\(コンピュータ\).md "wikilink")）であったが、[Mac OS X v10.2より](../Page/Mac_OS_X_v10.2.md "wikilink")**Sherlock 3**\[7\]となり、オークションの[トラッキング](https://ja.wikipedia.org/wiki/トラッキング "wikilink")機能や画像検索機能\[8\]が強化された。さらに[Mac OS X v10.3では](../Page/Mac_OS_X_v10.3.md "wikilink")、ファイル検索機能が[Finder](../Page/Finder.md "wikilink")から利用できるようにメニュー組み込まれた。続く[Mac OS X v10.4でもSherlockは搭載されていたものの](../Page/Mac_OS_X_v10.4.md "wikilink")、10.4からは、Sherlockを置き換える新しいローカルディスクの検索機能「[Spotlight](../Page/Spotlight.md "wikilink")」と、デスクトップからインターネット資源を呼び出せる[ウィジェットエンジン](https://ja.wikipedia.org/wiki/ウィジェットエンジン "wikilink")「[Dashboard](../Page/Dashboard.md "wikilink")」が登場した。[Mac OS X v10.5からはSherlockは搭載されていない](../Page/Mac_OS_X_v10.5.md "wikilink")。
