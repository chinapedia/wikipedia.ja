**阿蘇山公園道路**（あそざんこうえんどうろ）は、[熊本県](https://ja.wikipedia.org/wiki/熊本県 "wikilink")[阿蘇郡](../Page/阿蘇郡.md "wikilink")[南阿蘇村](https://ja.wikipedia.org/wiki/南阿蘇村 "wikilink")から[阿蘇市](https://ja.wikipedia.org/wiki/阿蘇市 "wikilink")に至る[有料道路](../Page/有料道路.md "wikilink")である。

## 概要

[阿蘇山](../Page/阿蘇山.md "wikilink")の山上広場から[中岳](../Page/中岳_\(阿蘇山\).md "wikilink")[火口](../Page/火口.md "wikilink")付近の[駐車場](../Page/駐車場.md "wikilink")まで向かうことができる。[自然公園法](https://ja.wikipedia.org/wiki/自然公園法 "wikilink")及び「阿蘇山公園道路の設置及び使用料徴収条例」に基づく公園道路として阿蘇市が管理している\[1\]。

### 路線データ

  - 起点：熊本県阿蘇郡南阿蘇村中松（阿蘇山頂広場、[熊本県道298号阿蘇公園下野線](../Page/熊本県道298号阿蘇公園下野線.md "wikilink")・[熊本県道299号草千里浜栃木線](../Page/熊本県道299号草千里浜栃木線.md "wikilink")起点）
  - 終点：熊本県阿蘇市黒川

## 通行料金

現在の通行料金は以下の通り\[2\]。

| 区分     | 料金     |
| ------ | ------ |
| 自転車    | 無料     |
| 単車     | 200円   |
| 軽自動車   | 600円   |
| 普通車    | 800円   |
| マイクロバス | 2,500円 |
| 中型バス   | 3,000円 |

料金は往復料金。行き止まりの道なので、片道料金は存在しない。出入口が分離されており、入口で徴収される。

利用可能時間は季節によって異なる。夜間は閉鎖される\[3\]。

## 特記事項

  - [阿蘇登山道路](https://ja.wikipedia.org/wiki/阿蘇登山道路 "wikilink")が有料道路としていた頃（[2000年](../Page/2000年.md "wikilink")に無料開放）は存在するもののも含めて日本で唯一有料道路の中にある有料道路であった。
  - 公園道路に平行して歩行者専用道路が設置されており、歩行者はこの道路を経由して中岳まで登山することができる。
  - 阿蘇山周辺には強い[火山ガス](../Page/火山ガス.md "wikilink")（[二酸化硫黄](../Page/二酸化硫黄.md "wikilink")等）が発生しており、特に火山ガスが大量に発生している場合などには利用が禁止されることがある\[4\]。
  - [気管支喘息](https://ja.wikipedia.org/wiki/気管支喘息 "wikilink")等の呼吸器疾患のある者の通行も健康上の問題から禁止されている\[5\]。
  - 火口に近い位置にあるため、阿蘇山の[噴火警戒レベル](https://ja.wikipedia.org/wiki/噴火警戒レベル "wikilink")が2以上の場合は閉鎖される。
