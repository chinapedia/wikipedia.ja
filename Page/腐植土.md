**腐植土**（ふしょくど、\[1\]）とは、腐植を豊かに含む、黒く軽鬆な[土壌](../Page/土壌.md "wikilink")のことである。

腐植土と**腐葉土**（ふようど、）は、同様に扱う文献もあるが、前者は主に土壌を指し、後者は主に[林床](https://ja.wikipedia.org/wiki/林床 "wikilink")で腐熟したや落葉[堆肥](https://ja.wikipedia.org/wiki/堆肥 "wikilink")を指す。

## 腐植

**腐植**（ふしょく、）とは、[土壌微生物](../Page/土壌微生物.md "wikilink")の活動により動植物遺体が[分解](https://ja.wikipedia.org/wiki/分解 "wikilink")・変質した物質の総称である。広義には単に[土壌有機物](https://ja.wikipedia.org/wiki/土壌有機物 "wikilink")としてのそれを指し、狭義には腐植化作用と呼ばれる分解・[重合](https://ja.wikipedia.org/wiki/重合 "wikilink")を繰り返し経て生成された、暗褐色で[コロイド](../Page/コロイド.md "wikilink")状の[無定形](../Page/アモルファス.md "wikilink")[高分子化合物](https://ja.wikipedia.org/wiki/高分子化合物 "wikilink")群（腐植物質）を指す。

腐植のおおまかな構成は下記のようになっている。

  - **広義の腐植**（土壌有機物の部分）
      - 非腐植物質（、腐植化されていない糖やタンパク質など）
      - 腐植物質（、腐植化された高分子化合物群、**狭義の腐植**）
          - [フミン酸](https://ja.wikipedia.org/wiki/フミン酸 "wikilink")（「腐植酸」、[pH](../Page/水素イオン指数.md "wikilink")2以下で非水溶性の画分、暗褐色を呈する）

          - [フルボ酸](https://ja.wikipedia.org/wiki/フルボ酸 "wikilink")（すべてのpH域で水溶性の画分、黄褐色を呈する）

          - （非水溶性の画分、黒色を呈する）

自然環境において腐植は、単に生き物の栄養源となる以外にも様々に重要な働きを持ち、土壌の保水性や[団粒化を促進したり](https://ja.wikipedia.org/wiki/土壌改良 "wikilink")、土壌の[陽イオン交換容量](https://ja.wikipedia.org/wiki/陽イオン交換容量 "wikilink")や[pH緩衝能を増加させたり](../Page/水素イオン指数.md "wikilink")、[リン酸](../Page/リン酸.md "wikilink")の[土壌固定を抑制したり](https://ja.wikipedia.org/wiki/肥料の三要素#リン "wikilink")、植物の[生理活性](../Page/生理活性.md "wikilink")物質として振る舞ったり、河川・海洋へ鉄（フルボ酸鉄[錯体](../Page/錯体.md "wikilink")）を移動させたり、土壌の[ポドゾル](https://ja.wikipedia.org/wiki/ポドゾル "wikilink")化作用に関わったりしている。

土壌における腐植の量は、土質や、腐植の分解・供給の速度、[降雨](https://ja.wikipedia.org/wiki/降雨 "wikilink")による流出、[土壌動物](https://ja.wikipedia.org/wiki/土壌動物 "wikilink")による耕耘・撹拌など、様々な要因が関わり、時に相互に影響する。腐植の多い土壌は黒ずみ、土色は腐植含有量の主観的な判別の目安にされる。また人工的な[堆肥](https://ja.wikipedia.org/wiki/堆肥 "wikilink")の腐熟によっても腐植物質が生成し、堆肥を黒ずませる\[2\]。ただし通気不良による[硫化鉄](https://ja.wikipedia.org/wiki/硫化鉄 "wikilink")生成でも堆肥は黒ずむため、堆肥の色による腐熟度の判別は難しい\[3\]。

## 腐植土

[thumb](https://ja.wikipedia.org/wiki/ファイル:Histisol_profile.jpg "wikilink") 地盤及び建築の観点から説明すると、腐植土は、大きな川や湖の水性植物などの有機物が分解して土壌と混ざり合ってできた暗褐色の土のことで、土質分類上は、有機土質に区別される。普通の土は、固体・液体・気体の三相構造から成り立っているが、有機質土では、固体の部分が粘土や砂といった土粒子の部分と水性植物などの有機物が混ざり合って成り立っている。そのため、有機質土は、一般に含水比が高く、小さな荷重に対しても圧縮性が非常に高いため、地盤沈下に対する充分な注意が必要である。例えば、造成工事をしたり、建築物を建てる場合には、適切な[地盤改良](../Page/地盤改良.md "wikilink")を行う必要が有る。

農学などでは腐植を20%以上含む土壌と定義される。

| 区分     | 腐植含有量  | 土色[明度の目安](https://ja.wikipedia.org/wiki/マンセル・カラー・システム#明度_\(Value\) "wikilink") | 備考                  |
| ------ | ------ | ------------------------------------------------------------------------------ | ------------------- |
| なし     |        |                                                                                |                     |
| あり     | 2%以下   | 5-7                                                                            |                     |
| 含む     | 2-5%   | 4-5                                                                            |                     |
| 富む     | 5-10%  | 2-3                                                                            |                     |
| すこぶる富む | 10-20% | 1-2                                                                            | 古くは「腐植質土」（1930年代以前） |
| 腐植土    | 20%以上  | 2以下                                                                            | 「有機質土層」とも           |

土壌の腐植含有量の区分\[4\]

一般的に腐植土の範疇に収まる[土壌分類](https://ja.wikipedia.org/wiki/土壌分類 "wikilink")には、[泥炭](https://ja.wikipedia.org/wiki/泥炭 "wikilink")土とがある。前者は排水不良な湿地などで形成され植物組織が残り、後者は泥炭地の地下水位低下などで形成され植物組織が分解されている。[泥炭](https://ja.wikipedia.org/wiki/泥炭 "wikilink")土に含まれる泥炭ごけ（[ピートモス](../Page/ピートモス.md "wikilink")）は園芸土として用いられ、腐葉土と用途や性質が類似するが、酸性が強い。

日本では泥炭土（泥炭地）や黒泥土（黒泥地）は関東や東北に多く、泥炭土は北海道にも多い\[5\]。[国際連合食糧農業機関](https://ja.wikipedia.org/wiki/国際連合食糧農業機関 "wikilink")（FAO）や[アメリカ合衆国農務省](https://ja.wikipedia.org/wiki/アメリカ合衆国農務省 "wikilink")（USDA）の土壌分類では、これらはに分類され、[カナダ](https://ja.wikipedia.org/wiki/カナダ "wikilink")や[スカンディナヴィア半島](../Page/スカンディナヴィア半島.md "wikilink")、[西シベリア平原](https://ja.wikipedia.org/wiki/西シベリア平原 "wikilink")に多い\[6\]。ヒストソルは冷涼な湿地に多いが、熱帯のなどにもある。

## 腐葉土

[thumb](https://ja.wikipedia.org/wiki/ファイル:Compost.jpg "wikilink") 腐葉土とは、森林生態系において地上部の[植物](../Page/植物.md "wikilink")により生産された有機物が朽木や落葉・落枝となって地表部に堆積し、それを資源として利用する[バクテリア](https://ja.wikipedia.org/wiki/バクテリア "wikilink")などの[微生物](../Page/微生物.md "wikilink")や[ミミズ](https://ja.wikipedia.org/wiki/ミミズ "wikilink")などの[土壌動物](https://ja.wikipedia.org/wiki/土壌動物 "wikilink")による生化学的な代謝作用により[分解](https://ja.wikipedia.org/wiki/分解 "wikilink")（落葉分解）されて[土](https://ja.wikipedia.org/wiki/土 "wikilink")状になったものである。腐葉土は厳密には土ではない。

その名の通り腐熟しているので色は黒っぽい。しかし、不快感を持つようなにおいは無く、山林に入ったときと同じような香りがする（[カブトムシ](../Page/カブトムシ.md "wikilink")の匂いと例える人もいる）。

長い月日をかけて自然が作り出す天然の[肥料](../Page/肥料.md "wikilink")で、植物の栽培や[昆虫](../Page/昆虫.md "wikilink")の飼育に適した堆肥である。山林に行けば手に入る土ではあるが、一般的には[園芸](../Page/園芸.md "wikilink")店などで袋詰めで普通に販売されているため都会でも手軽に手に入れることが出来る。

### 成分

自然にできたものは成分が[窒素](../Page/窒素.md "wikilink")に偏っていることが多いが、[リン酸](../Page/リン酸.md "wikilink")、[カリウム](../Page/カリウム.md "wikilink")などはミミズ、その他の[動物](https://ja.wikipedia.org/wiki/動物 "wikilink")の[糞](../Page/糞.md "wikilink")や微生物などの働きによって補われることもある。人工的に作られた腐葉土は成分が人工的に調整されている。以下参照。

腐葉土になりやすい葉は[落葉樹](https://ja.wikipedia.org/wiki/落葉樹 "wikilink")や、[広葉樹](https://ja.wikipedia.org/wiki/広葉樹 "wikilink")など、油分が少なく[発酵](../Page/発酵.md "wikilink")しやすい種類で、[杉](https://ja.wikipedia.org/wiki/杉 "wikilink")、[松](https://ja.wikipedia.org/wiki/松 "wikilink")などの油分が多い葉は腐葉土になりにくい。

### 人工的に作られた腐葉土

自然にできた腐葉土は、発酵して出来上がるのに1 - 2年以上かかるが、人工的に作る場合は[米糠](https://ja.wikipedia.org/wiki/米糠 "wikilink")などを使って発酵しやすい環境を作る。そのため出来上がるまでの期間は2か月ほどまで縮まる。さらに自然にできた腐葉土は[成分](https://ja.wikipedia.org/wiki/成分 "wikilink")が偏っていることが多いが、人工的に作ったものは動物の糞などを使って成分調整をしている。

日本国内で売られている製品でも、海外産の原料を含む場合がある\[7\]。

### 昆虫の餌

カブトムシの幼虫などの[餌](../Page/餌.md "wikilink")としても使われる。昆虫飼育に使用する場合は、防虫・[防カビ](https://ja.wikipedia.org/wiki/防カビ "wikilink")処理がされていないものが使われる。
