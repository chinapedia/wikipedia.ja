確率論において、[確率変数](../Page/確率変数.md "wikilink")の**条件付き期待値**（じょうけんんつききたいち、）とは初等的には何らかの情報が与えられた場合の確率変数に期待される値のことである。しかし、より一般の場合の定義では、確率変数の条件付き期待値は新しい確率変数であり、元の確率変数より強い可測性をもつ。このことは新しい確率変数を決定するのに必要な情報が減少したということなので、情報を減らしたときに確率変数がどうなるかを計算したものとみることもできる。この方法で情報を最小のものにすると、条件付き期待値は定数になり[期待値](../Page/期待値.md "wikilink")と一致する。初等的な定義では、この最小の情報に情報を追加したときの挙動を見ているといってもよい。

## 初等的な定義

初等的な定義では条件付き期待値は[条件付き確率](../Page/条件付き確率.md "wikilink")による期待値である。*P*(*A*) \> 0 をみたす[事象](https://ja.wikipedia.org/wiki/事象 "wikilink") *A* が起きたことが分かったときに、事象 *B* が起きる条件付き確率は

\[\operatorname{P}(B \mid A) := \frac{\operatorname{P}(A \cap B)}{\operatorname{P}(A)}\] で定義され、事象 *A* が起きたことが分かったときの確率変数 *X* の条件付き期待値は

\[\operatorname{E}[X \mid A] := \operatorname{E}^{\operatorname{P}(\cdot \mid A)}[X] = \frac{\operatorname{E}[X,A]}{\operatorname{P}(A)}\] で与えられる。

初等的な場合の例でサイコロを投げるかわりに、 と  が平均 、分散  の[正規分布](https://ja.wikipedia.org/wiki/正規分布 "wikilink")に従う場合を考えてみると、

[条件付き確率密度関数を使い](https://ja.wikipedia.org/wiki/条件付き確率分布 "wikilink")、*f*<sub>*Y*</sub>(*y*) \> 0 ならば、以下のように計算できる。*f*<sub>*Y*</sub>(*y*) は *Y* の確率密度関数である。

\[\operatorname{E}[ X \mid Y=y ] = \int_{\mathcal{X}} x f_{X \mid Y} (x \mid y) \, \mathrm{d}x,\]

\[f_{X\mid Y} (x\mid y) = \frac{f_{X, Y}(x, y)}{f_Y(y)}\]

さらに、一般の場合は情報を事象でも確率変数の値でもなく、[完全加法族](https://ja.wikipedia.org/wiki/完全加法族 "wikilink")で与える。
