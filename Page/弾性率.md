**弾性率**（だんせいりつ、）は、[変形](https://ja.wikipedia.org/wiki/変形 "wikilink")のしにくさを表す[物性値](https://ja.wikipedia.org/wiki/物性値 "wikilink")であり、[弾性変形](https://ja.wikipedia.org/wiki/弾性変形 "wikilink")における[応力](../Page/応力.md "wikilink")と[ひずみ](https://ja.wikipedia.org/wiki/ひずみ "wikilink")の間の比例定数の総称である。**弾性係数**あるいは**弾性定数**とも呼ばれる\[1\]。

1807年に[トマス・ヤング](https://ja.wikipedia.org/wiki/トマス・ヤング "wikilink")によって導入された\[2\]。

## 種類

[単純な伸長変形のモデル.jpg](https://ja.wikipedia.org/wiki/File:単純な伸長変形のモデル.jpg "fig:単純な伸長変形のモデル.jpg") [剪断変形.jpg](https://ja.wikipedia.org/wiki/File:剪断変形.jpg "fig:剪断変形.jpg") [単純な体積変形のモデル.jpg](https://ja.wikipedia.org/wiki/File:単純な体積変形のモデル.jpg "fig:単純な体積変形のモデル.jpg") 弾性率は、[弾性変形](https://ja.wikipedia.org/wiki/弾性変形 "wikilink")における[応力](../Page/応力.md "wikilink")と[ひずみ](https://ja.wikipedia.org/wiki/ひずみ "wikilink")の間の比例定数(応力/ひずみ)であり、加えられた外力([応力](../Page/応力.md "wikilink"))を分子、応力によって引き起こされたひずみを分母とした商である\[3\]。

  -
    弾性率 = 応力/ひずみ

ひずみは無次元であるので、弾性率は応力と同じ[次元を持ち](https://ja.wikipedia.org/wiki/量の次元 "wikilink")、[SIにおける単位は](https://ja.wikipedia.org/wiki/国際単位系 "wikilink")[パスカル](../Page/パスカル_\(単位\).md "wikilink")（記号: Pa）、[ニュートン毎](../Page/ニュートン_\(単位\).md "wikilink")[平方メートル](../Page/平方メートル.md "wikilink")（記号: N/m<sup>2</sup>）が用いられる。また、弾性率の逆数を**弾性コンプライアンス定数**や単に**弾性コンプライアンス**という。単位は1/Pa、m<sup>2</sup>/N。

弾性変形は伸長(または圧縮)変形、剪断変形、体積変形の3つの種類に分けられ、従って弾性率も3種類ある。それぞれひずみの定義は異なる。

  - 引張弾性率\(E\) ：引張力や圧縮力などの単軸応力についての弾性率。[ヤング率](../Page/ヤング率.md "wikilink")（縦弾性係数）。

\[E = \frac{\sigma}{\varepsilon}\]

  -
    伸長ひずみ\(\varepsilon = \frac{L - L_0}{L_0}\)（\(L_0\)は元々の長さ、\(L\)は引張後長さ）
    伸長粘度\(\eta_E = \frac{\sigma}{d\varepsilon / dt}\)（は時間）

<!-- end list -->

  - 剪断弾性率\(G\) ：[せん断力](https://ja.wikipedia.org/wiki/せん断力 "wikilink")についての弾性率。[剛性率](https://ja.wikipedia.org/wiki/剛性率 "wikilink")（ずり弾性率・横弾性係数・せん断弾性係数・ラメの第二定数）。

\[G = \frac{\sigma}{\gamma}\]

  -
    剪断ひずみ\(\gamma = \frac{d}{h} = \tan \alpha\)（\(d\)は剪断により面が剪断力方向に移動した距離、\(h\)は剪断力方向と垂直な試料厚さ、\(\alpha\)は、試料の面が長方形から[平行四辺形](../Page/平行四辺形.md "wikilink")になるときの倒れ角）
    剪断粘度\(\eta = \frac{\sigma}{d\gamma/dt}\)

<!-- end list -->

  - [体積弾性率](https://ja.wikipedia.org/wiki/体積弾性率 "wikilink")\(K\) ：[静水圧](https://ja.wikipedia.org/wiki/静水圧 "wikilink")（直角3方向の力）についての弾性率。

\[K = \frac{\sigma}{\kappa}\]

  -
    体積ひずみ\(\kappa = \frac{V - V_0}{V_0}\)（\(V_0\)は元々の体積、\(V\)は変形後体積）
    体積粘度\(\eta_V = \frac{\sigma}{d\kappa/dt}\)

<!-- end list -->

  - λ：[ラメの第一定数](https://ja.wikipedia.org/wiki/ラメ定数 "wikilink")（ラメの弾性係数）

## テンソル量としての弾性率

2階のテンソル量である応力とひずみに対して、弾性率は4階の[テンソル](../Page/テンソル.md "wikilink")量で表すことができる\[4\]。

\[\boldsymbol{\sigma} = \boldsymbol{D} \boldsymbol{\epsilon},\]

\[\sigma_{ij}= D_{ijkl} \epsilon_{kl}\quad(i,j,k,l =1\sim3)\]\[5\]

弾性率はテンソルであるため、[物質客観性の原理により座標変換において](https://ja.wikipedia.org/wiki/材料の構成式#物質客観性の原理 "wikilink")*D*ε}}の関係を保たねばならない。座標系からへ変換するとき、弾性率テンソルの成分は

\[D'_{ijmn}=D_{pqrs} l_{ip} l_{jq} l_{mr} l_{ns}\] と変換される\[6\]。ここでは、軸と軸の[方向余弦](https://ja.wikipedia.org/wiki/方向余弦 "wikilink")である。

弾性率テンソルは81(= 3<sup>4</sup>)個の成分を持つが、応力テンソルとひずみテンソルは対称性、すなわち

\[\sigma_{ij}=\sigma_{ji},\quad \epsilon_{ij}=\epsilon_{ji}\] によりそれぞれ独立な6成分を持つので、弾性率テンソルも

\[D_{ijkl}=D_{jilk}\] の性質を持ち、独立な成分は36(= 6<sup>2</sup>)個となる。さらに単位体積あたりの[弾性ひずみエネルギー](https://ja.wikipedia.org/wiki/弾性ひずみエネルギー "wikilink")

\[dW\equiv \sigma_{ij}\,d\epsilon_{ij}\] を用いて弾性率が

\[D_{ijkl}=\frac{\partial^2 W}{\partial\epsilon_{ij} \partial\epsilon_{kl}}\] と表せることから

\[D_{ijkl}=D_{klij}\] が成り立つため、最終的に弾性率テンソルの独立な成分は21(= 6×(6+1)/2)個となる\[7\]。

## 等方均質材料の弾性率の相関関係

一般に、等方性物質（無定形ポリマー、非晶性・無配向ポリマーなど）では3種の弾性率（引張弾性率\(E\)、剪断弾性率\(G\)、体積弾性率\(K\)）の関係について次式が成り立つ\[8\]。

\[E = 2G(1 - \nu) = 3K(1 - 2\nu)\] ここで、\(\nu\)とは、縦方向のひずみと横方向のひずみとの比([ポアソン比](../Page/ポアソン比.md "wikilink"))である。結晶性ポリマー、繊維、フィルム、繊維充填複合材料、一般の射出成形物などは等方性物質ではない。高分子鎖、充填繊維、結晶相などに配向を持ち、その程度は内部と表面で異なる。これ異方性物質は、独立した2つ以上の弾性率を持つ\[9\]。

材料が等方均質弾性材料とすると、弾性率テンソル***D*** の独立な成分は2個まで絞られ\[10\]、次式のように書ける\[11\]。

\[D_{ijkl}=\lambda\delta_{ij}\delta_{kl}+G(\delta_{ik}\delta_{jl}+\delta_{il}\delta{jk})\] ここでδは[クロネッカーのデルタ](../Page/クロネッカーのデルタ.md "wikilink")である。

この場合、ヤング率*E* 、ポアソン比ν、体積弾性率*K* 、剛性率*G* 、ラメの第一定数λの5つの弾性率はそれぞれ、2つを用いて残りの3つを表すことができる。その関係を下に示す。ここで、 (*E*<sup>2</sup> + 9λ<sup>2</sup> + 2*E* λ)<sup>1/2</sup>}} とする。

| 等方均質弾性体における各弾性率間の変換式 |
| -------------------- |
|                      |
| \(E, \nu\)           |
| \(E, K\)             |
| \(E, G\)             |
| \(E, \lambda\)       |
| \(\nu, K\)           |
| \(\nu, G\)           |
| \(\nu, \lambda\)     |
| \(K, G\)             |
| \(K, \lambda\)       |
| \(G, \lambda\)       |

## 複素弾性率

[粘弾性体](https://ja.wikipedia.org/wiki/粘弾性体 "wikilink")に対しては、弾性率は[複素数](../Page/複素数.md "wikilink")で表される。複素弾性率の実部は貯蔵弾性率、虚部は損失弾性率と呼ばれる。
