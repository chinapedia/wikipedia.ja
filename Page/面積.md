**面積**（めんせき）とは、平面内の、あるいは曲面内の図形の大きさ、広さ、の[量](https://ja.wikipedia.org/wiki/量 "wikilink")である。[立体](https://ja.wikipedia.org/wiki/立体 "wikilink")物の表面の面積の合計を特に**[表面積](https://ja.wikipedia.org/wiki/表面積 "wikilink")**（ひょうめんせき）と呼ぶ。

## 定義

平面図形については、2次元空間内の部分集合（つまり図形）の[定義関数を](https://ja.wikipedia.org/wiki/指示関数 "wikilink")[積分](https://ja.wikipedia.org/wiki/積分 "wikilink")して面積を定義する。直感的にはまず長方形の面積を定義し、一般の図形に対しては小さな長方形の集まりでその図形を近似した[極限](https://ja.wikipedia.org/wiki/極限 "wikilink")を以って面積を定義する。

曲面については、定義関数の[面積分](https://ja.wikipedia.org/wiki/面積分 "wikilink")のほか、曲面を（3次元空間内で）小さな平面図形の集まりでその図形を近似した[極限](https://ja.wikipedia.org/wiki/極限 "wikilink")によって面積を定義することができる。

## 面積を求める公式

### 平面

[Area.svg](https://ja.wikipedia.org/wiki/File:Area.svg "fig:Area.svg")  基本的な面積を計算する公式をいくつか示す。

  - [正方形](https://ja.wikipedia.org/wiki/正方形 "wikilink"): *a*（*a* = 一辺の長さ）
  - [長方形](https://ja.wikipedia.org/wiki/長方形 "wikilink"): *ab*（*a* = 縦の長さ、*b* = 横の長さ）
  - [菱形](https://ja.wikipedia.org/wiki/菱形 "wikilink"): *ab*（*a*, *b* は2つの[対角線](https://ja.wikipedia.org/wiki/対角線 "wikilink")の長さ）
  - [台形](https://ja.wikipedia.org/wiki/台形 "wikilink"): (*B* + *b*)*h*（*B*, *b* は上底、下底の長さ、*h* = 高さ）
  - [平行四辺形](https://ja.wikipedia.org/wiki/平行四辺形 "wikilink"): *ah*（*a* = 底辺の長さ、*h* = 高さ）
  - 平行四辺形: |**A** × **B**| = |**A**||**B**|sin *θ*（**A**, **B** は平行四辺形を張る[独立な](../Page/線型結合.md "wikilink")[ベクトル](https://ja.wikipedia.org/wiki/空間ベクトル "wikilink")、"×" はベクトルの[クロス積](https://ja.wikipedia.org/wiki/クロス積 "wikilink")（外積）、"| |" はベクトルの大きさ、*θ* は **A** と **B** の[ベクトルのなす角](https://ja.wikipedia.org/wiki/ベクトルのなす角 "wikilink")）
  - [三角形](https://ja.wikipedia.org/wiki/三角形 "wikilink"): *ah*（*a* = 底辺の長さ、*h* = 高さ）、*ab*sin *θ*（*a*、*b* = 辺の長さ、*θ* = 2辺のなす角の大きさ（[ラジアン](https://ja.wikipedia.org/wiki/ラジアン "wikilink") (rad)）、[ヘロンの公式](https://ja.wikipedia.org/wiki/ヘロンの公式 "wikilink")
  - 各[頂点](https://ja.wikipedia.org/wiki/頂点 "wikilink")の[座標](https://ja.wikipedia.org/wiki/座標 "wikilink")が与えられた[多角形](https://ja.wikipedia.org/wiki/多角形 "wikilink"): [座標法](https://ja.wikipedia.org/wiki/座標法 "wikilink")を参照
  - [円](https://ja.wikipedia.org/wiki/円_\(数学\) "wikilink"): *r*（ = [円周率](https://ja.wikipedia.org/wiki/円周率 "wikilink")、*r* = 半径）
  - [扇形](https://ja.wikipedia.org/wiki/扇形 "wikilink"): *r**θ*（*θ* = 中心角の大きさ（ラジアン））
  - 扇形: *r**θ*/360（*θ* = 中心角の大きさ（度））
  - 扇形: *lr*（*l* = 弧の長さ (2*rθ*/360))
  - [楕円](https://ja.wikipedia.org/wiki/楕円 "wikilink"): *ab*（*a*、*b* = 半長軸および半短軸の長さ）
  - [正多角形](https://ja.wikipedia.org/wiki/正多角形 "wikilink"): *Pa*（*P* = 周辺の長さ、*a* = 多角形の辺心距離（中心から辺の中心までの長さ））
  - 格子[多角形](https://ja.wikipedia.org/wiki/多角形 "wikilink"):[ピックの定理](https://ja.wikipedia.org/wiki/ピックの定理 "wikilink")
  - [アステロイド曲線に囲まれた部分](https://ja.wikipedia.org/wiki/アステロイド_\(曲線\) "wikilink"): *a*（アステロイド曲線の方程式 *x* + *y* = *a*）
  - [カージオイド](https://ja.wikipedia.org/wiki/カージオイド "wikilink")曲線に囲まれた部分: *a*（カージオイド曲線の極方程式 *r* = *a*(1 + cos *θ*)）

### 立体

立体の表面積、側面積を求める公式を以下に示す。

  - [立方体](https://ja.wikipedia.org/wiki/立方体 "wikilink")の表面積: 6*s*（*s* = 一辺の長さ）
  - [直方体](https://ja.wikipedia.org/wiki/直方体 "wikilink")の表面積: 2(*lw* + *lh* + *wh*)（*l* = 縦の長さ、*w* = 横の長さ、*h* = 高さ）
  - [円柱の側面積](https://ja.wikipedia.org/wiki/円柱_\(数学\) "wikilink"): 2*rh*（*r* = 底面の半径、*h* = 高さ）
  - 斜切円柱の側面積: *r*(*h* + *h*)（*h* = 最大母線の長さ、*h* = 最小母線の長さ）
  - [円錐](https://ja.wikipedia.org/wiki/円錐 "wikilink")の側面積: *ar*（*a* = 母線の長さ、*r* = 底面の半径）
  - 円錐台の側面積: *a*(*R* + *r*)（*a* = 母線の長さ、*R*, *r* = 両底面の半径、*h* = 高さ）
  - 円柱の表面積: 2*r*(*h* + *r*)（*r* = 底面の半径、*h* = 高さ）
  - 円錐の表面積: *r*(*r* + *a*)（*r* = 底面の半径、*a* = 母線の長さ）
  - [球](https://ja.wikipedia.org/wiki/球 "wikilink")の表面積: 4*r*（*r* = 半径）

円以下の公式は、正確には[積分](https://ja.wikipedia.org/wiki/積分 "wikilink")を使って正当化される。さらに幅広い図形についてこの概念を定義するためには、積分を避けて通ることはできない。

## 定義不良な面積 Ill-defined areas

[選択公理](https://ja.wikipedia.org/wiki/選択公理 "wikilink")を受け入れると、「意味のある面積が定義できない図形」が存在することを証明できる （[ルベーグ測度](https://ja.wikipedia.org/wiki/ルベーグ測度 "wikilink")を参照）。 このような「図形」（簡単に図示することは出来ない）は[タルスキーの円積問題](https://ja.wikipedia.org/wiki/タルスキーの円積問題 "wikilink") ([:en:Tarski's circle-squaring problem](https://ja.wikipedia.org/wiki/:en:Tarski's_circle-squaring_problem "wikilink")) に関係している（三次元における類似の例として、「体積の定義できない図形」と[バナッハ＝タルスキーのパラドックス](https://ja.wikipedia.org/wiki/バナッハ＝タルスキーのパラドックス "wikilink")がある）。 このような集合は実用上現実の世界では生じない。
