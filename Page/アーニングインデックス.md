**アーニングインデックス**(*Average Earnings Index*)とは、[競馬](../Page/競馬.md "wikilink")において[種牡馬](../Page/種牡馬.md "wikilink")の成績を表す指標の1つで、種牡馬ごとの産駒の平均収得賞金額の大小を表す\[1\]。略して**AEI**、または**EI**とも記述される。

全競走における競走馬1頭あたりの収得賞金に対する、ある種牡馬の産駒1頭あたりの収得賞金の比率をあらわしたもので、以下の数式で表される\[2\]。

|            |   |            |
| ---------- | - | ---------- |
| （産駒の総収得賞金） |   | （全出走馬収得賞金） |
| ────────── | ÷ | ────────── |
| （産駒の出走頭数）  |   | （総出走頭数）    |

1が平均値となり、1より大きい場合は産駒1頭あたりの獲得賞金が平均より多く、1より小さい場合は産駒1頭あたりの獲得賞金が平均より少ないことを表している。[種牡馬リーディングは出走産駒の総獲得賞金で順位が決まるため](../Page/リーディングサイアー.md "wikilink")、産駒が多いほど有利になるが、アーニングインデックスは産駒の平均の質を評価するものである。

産駒の出走頭数が少ない種牡馬については、1頭の産駒の活躍により、AEIが極端に大きな値となる場合がある。例えば[Sharrood](../Page/シャルード.md "wikilink")\[3\]の日本における中央・地方合計AEIは1993年度が95.00、1994年度が95.43である\[4\]。これは日本で出走した唯一の産駒[ビワハヤヒデ](../Page/ビワハヤヒデ.md "wikilink")\[5\]が[菊花賞](../Page/菊花賞.md "wikilink")や[天皇賞](https://ja.wikipedia.org/wiki/天皇賞 "wikilink")（春）などを制して2年間で8億円以上を稼いだ\[6\]ことによる。同様に1992年度のMajestic Light\[7\]（出走産駒は[ニシノフラワー](../Page/ニシノフラワー.md "wikilink")\[8\]のみ）や、2003年度の[Pleasant Tap](../Page/プレザントタップ.md "wikilink")\[9\]（出走産駒2頭のうち1頭が[タップダンスシチー](../Page/タップダンスシチー.md "wikilink")\[10\]）なども、AEIが50を超える値となっている。
