**最大剰余方式**（さいだいじょうよほうしき、Largest remainder method）とは、比例配分方式の一つで、[名簿式](../Page/政党名簿比例代表.md "wikilink")[投票制度](https://ja.wikipedia.org/wiki/投票制度 "wikilink")における議席の各党への割り当てや[選挙区](https://ja.wikipedia.org/wiki/選挙区 "wikilink")への[定数の配分に用いられる](https://ja.wikipedia.org/wiki/議員定数#定数配分 "wikilink")。**ヘア＝ニーマイヤー式**（[Hare-Niemeyer-Verfahren](https://ja.wikipedia.org/wiki/:de:Hare-Niemeyer-Verfahren "wikilink")）あるいは**ハミルトン方式**と呼ばれることもある。[最高平均方式](https://ja.wikipedia.org/wiki/最高平均方式 "wikilink")（除数方式、[Highest averages method](https://ja.wikipedia.org/wiki/:en:Highest_averages_method "wikilink")）と対照される。

## 方式

\(h\) 議席をいくつかの政党へ得票数に応じて分配することを考える。各政党には \(1,\dots,s\) と番号をつけ、政党 \(i\) の得票数は \(p_i\)であるとする。なお、政党を都道府県に、得票数を人口にそれぞれ置き換えることで、議席配分についても同様に考えることができる。

最大剰余方式では次のようにして議席を割り当てる。

1.  基数 \(Q\) を決め(次項参照)、各政党 \(i\) の「取り分」\(q_i=p_i/Q\) を計算する。
2.  政党 \(i\) に \(\lfloor q_i\rfloor\) 議席を割り当てる。（ \(\lfloor x\rfloor\) は \(x\) の小数点以下切捨てを表す）
3.  2.で配分した議席の合計が \(h\) 議席に満たない場合、取り分の端数 \(q_i-\lfloor q_i\rfloor\) の大きい順に1議席ずつ割り当てを追加する。

## 基数

基数 \(Q\) の定め方にはいくつか方式がある。

ヘア基数（単純基数）\(Q_{\textrm{h
}}\) は以下のように定義される。

\[Q_{\textrm{h}}=\frac{\sum_{i=1}^sp_i}{h}.\]

ヘア基数を使う最大剰余方式は、[1792年](../Page/1792年.md "wikilink")に最大剰余方式を考案した[アレクサンダー・ハミルトン](https://ja.wikipedia.org/wiki/アレクサンダー・ハミルトン "wikilink")に因んでハミルトン方式と呼ばれる。[ロシア](../Page/ロシア.md "wikilink")（2007年からは阻止条項7%）、[ウクライナ](../Page/ウクライナ.md "wikilink")（足切り3%）、[ナミビア](https://ja.wikipedia.org/wiki/ナミビア "wikilink")、[香港](https://ja.wikipedia.org/wiki/香港特別行政区立法会 "wikilink")、の立法府選挙で使用されている。歴史的には[19世紀](../Page/19世紀.md "wikilink")の[アメリカ合衆国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")で[議席配分に採用されていた](https://ja.wikipedia.org/wiki/:en:United_States_congressional_apportionment "wikilink")。

一方、ドループ基数と呼ばれる基数 \(Q_{\textrm{d}}\) は文献によって差異があるが、

\[Q_{\textrm{d}}=\left\lfloor 1+\frac{\sum_{i=1}^sp_i}{1+h}\right\rfloor\ ,\frac{\sum_{i=1}^sp_i}{1+h}\]

などと定義される(厳密には異なる配分結果を生じうるが、極めてまれである)。南アフリカの選挙に採用されている。2番目はハーゲンバッハ=ビジョフ基数と呼ばれることもある。

ヘア基数は小政党に対して少し寛大な傾向があり、ドループ基数は大政党寄りである。ヘア基数は、過半数を獲得した名簿に半数未満の議席が与えられることがあるが、ドループ基数よりも比例的であると考えられている。[1](http://www.parl.gc.ca/information/library/PRBpubs/bp334-e.htm) [2](https://web.archive.org/web/20060901090418/http://polmeth.wustl.edu/polanalysis/vol/8/PA84-381-388.pdf) [3](https://web.archive.org/web/20070926202329/http://www.dur.ac.uk/john.ashworth/EPCS/Papers/Suojanen.pdf) [4](https://web.archive.org/web/20060516204603/http://users.ox.ac.uk/~sann2300/041102-ceg-electoral-consequences-lijphart.shtml) [5](http://janda.org/c24/Readings/Lijphart/Lijphart.html)

また、以下のように定義されるインペリアリ基数 \(Q_{\textrm{i}}\) がある。

\[Q_{\textrm{i}}=\frac{\sum_{i=1}^sp_i}{2+h}.\]

ただし、この基数を用いて計算すると配分の合計が \(h\) を超える可能性があるため使われることは少ない。

## 長所と短所

最大剰余方式の議席配分法は最高平均方式と比較して単純で分かりやすく、また議席の計算も簡便である。ヘア基数が使われるのなら、票の割合の多寡は名簿にとって有利な点にはならず、この点に関しては中立的である。しかし、端数処理の単純さゆえに次項で述べるような不合理な結果をもたらす場合がある。

## 配分パラドックス

最大剰余方式はアラバマのパラドックスや人口パラドックスと呼ばれる不合理な配分をもたらすことがある。

以下ではヘア基数を用いた場合の例を示すが、ドループ基数を用いても同様の現象が生じうる。

### アラバマのパラドックス

総配分議席が増加したのにも関わらず配分が減ってしまう現象を、アメリカ下院の議席配分での実例にちなみアラバマのパラドックスという。

以下の表は、総議席が10から11に増えるとC県への配分が減ってしまう実例を示している。

|     | 人口   | 10議席 | 11議席 |
| --- | ---- | ---- | ---- |
| 取り分 | 配分議席 | 取り分  | 配分議席 |
| A県  | 5665 | 5.15 | 5    |
| B県  | 3685 | 3.35 | 3    |
| C県  | 1650 | 1.50 | 2    |

### 人口パラドックス

人口が相対的に増加しているのに配分が減り、相対的に減少しているのに配分が増えてしまう現象を人口パラドックスという。

以下の表に例を示す。A県の人口増加率はB県よりも高いが、配分議席の変化はB県がA県から議席を奪う形になっている。

|    | 人口    | 取り分  | 配分議席 |       | 人口      | 人口増加率     | 取り分        | 配分議席 |
| -- | ----- | ---- | ---- | ----- | ------- | --------- | ---------- | ---- |
| A県 | 7800  | 15.6 | 16   | ⇒     | 8432    | \+8.1%    | 15.3309... | 15   |
| B県 | 1700  | 3.4  | 3    | 1836  | \+8.0%  | 3.3381... | 4          |      |
| C県 | 500   | 1.0  | 1    | 732   | \+46.4% | 1.3309... | 1          |      |
| 計  | 10000 | 20   |      | 11000 |         | 20        |            |      |
