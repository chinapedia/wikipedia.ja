**リュカ数**（リュカすう、）とは、[フランス](https://ja.wikipedia.org/wiki/フランス "wikilink")の[数学者](../Page/数学者.md "wikilink")[エドゥアール・リュカ](../Page/エドゥアール・リュカ.md "wikilink")にちなんで名付けられた数であり、*n* 番目のリュカ数を *L*<sub>*n*</sub> で表すと

  -
    \(L_0=2,\ L_1=1, \,\)
    \(L_{n+2}=L_n+L_{n+1} \,\)

で定義される[数列](../Page/数列.md "wikilink")にある項のことである。つまり、初項（最初のリュカ数）を 2、次の項を 1 と定義し、それ以降の項は前の2つの項の和になっている数列のことである。

## 負の番号への拡張

[漸化式](https://ja.wikipedia.org/wiki/漸化式 "wikilink") *L*<sub>*n*+2</sub> = *L*<sub>*n*</sub> + *L*<sub>*n*+1</sub> を全ての整数 *n* に対して適用すると、*n* が負の整数である場合に拡張できる。例えば、-5 ≤ *n* ≤ 5 に対するリュカ数は次の値になる。

  -
    \-11, 7, -4, 3, -1, 2, 1, 3, 4, 7, 11

さらに、一般には *L*<sub>-*n*</sub> = (-1)<sup>*n*</sup>*L*<sub>*n*</sub> となる。

## 数学的性質

リュカ数は、[フィボナッチ数](../Page/フィボナッチ数.md "wikilink")と共に自然界に多く存在する。またフィボナッチ数 *F*<sub>*n*</sub> との間に多くの関係式があり、例として

  -
    \(L_n = F_{n-1}+F_{n+1}\,\)
    \(F_{2n} = L_n F_n\,\)
    \(F_n = {L_{n-1}+L_{n+1} \over 5}\,\)

などが挙げられる。また同じ項番号のフィボナッチ数とリュカ数の比 *L*<sub>*n*</sub>/*F*<sub>*n*</sub> は、*n* が大きくなるにつれて√5 = 2.23606798…に収束する。

フィボナッチ数と同様に、リュカ数も隣接する2項の比 *L*<sub>*n*+1</sub>/*L*<sub>*n*</sub> は *n* が大きくなるにつれて[黄金比](../Page/黄金比.md "wikilink") \(\phi = \frac{1 + \sqrt{5}}{2}\) = 1.61803398…に近づく。

*n* 番目のリュカ数は以下の式で表される。 {2} \\right)^n + \\left( \\frac{1-\\sqrt{5}}{2} \\right)^n= \\phi^n + (1-\\phi)^n = {\\phi^n + (-\\phi)^{-n}}</math>}} ここで \(\phi\) は黄金比である。

## リュカ素数

**リュカ素数**（リュカそすう、）とは、リュカ数である[素数](../Page/素数.md "wikilink")である。

リュカ素数 *L*<sub>*n*</sub> の最初のいくつかの項は以下の通りである。

  -
    2, 3, 7, 11, 29, 47, 199, 521, 2207, 3571, 9349, 3010349, 54018521, 370248451, 6643838879, ….（）

*L*<sub>*n*</sub> の *n* は以下の通りである。

  -
    0, 2, 4, 5, 7, 8, 11, 13, 16, 17, 19, 31, 37, 41, 47, ….（）

*n* = 0, 4, 8, 16 の場合を除いて、*L*<sub>*n*</sub> が素数ならば *n* も素数である\[1\]。しかし、*n* が素数でも、*L*<sub>*n*</sub> が素数になるとは限らない。
