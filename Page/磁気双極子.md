**磁気双極子**（じきそうきょくし、[英語](../Page/英語.md "wikilink")：magnetic dipole）は、正負の磁極の対のことを言う。単独の磁極（[磁気単極子](../Page/磁気単極子.md "wikilink")）の存在は現在に至るまで確認されていないので、磁気についての基本的な要素はこの磁気双極子となる。

## 定義

正負の磁極の磁荷(磁荷は存在しない)の大きさは等しいとして、その磁荷の大きさをq<sub>m</sub>とし、正負の磁極の間の距離をdとすると、**[磁気双極子モーメント](https://ja.wikipedia.org/wiki/磁気双極子モーメント "wikilink")**は、

\[\mathbf{m} = q_m \mathbf{d}\]

となる。**m**：磁気双極子モーメント。**d**は距離dのベクトル表示である。この磁気双極子モーメントを使って、磁気双極子から**ｒ**だけ離れたところでの[磁場](../Page/磁場.md "wikilink")は、

\(\mathbf{H}_{dipole} = - {1 \over {4 \pi \mu_0} } \mathrm{grad} { \mathbf{m} \cdot \mathbf{r} \over {r^3} } = - {1 \over {4 \pi \mu_0} } \nabla { \mathbf{m} \cdot \mathbf{r} \over {r^3} }\)

と表すことができる。

## E-H対応とE-B対応の違い

磁化の表現には、\(\tilde{q}_m=\mu_0q_m\) とすると、\(\tilde{q}_m\)を磁気量とする場合と\(q_m\)を磁気量とする二種の表現方法がある。\(\tilde{q}_m\)を磁気量とすると**E-H対応**と言い単位がWb([ウェーバ](../Page/ウェーバ.md "wikilink"))となる。\(q_m\)を磁気量とすると**E-B対応**と言い単位がT([テスラ](../Page/テスラ_\(単位\).md "wikilink"))=Wb/m2([ウェーバ](../Page/ウェーバ.md "wikilink")/[平方メートル](../Page/平方メートル.md "wikilink"))となる。本によってはどちらかの表現が使われている。

## 磁気双極子が受ける力

**H**<sub>dipole</sub>は磁気双極子が**r**（位置ベクトル）に作る磁場。μ<sub>0</sub>は真空の[透磁率](https://ja.wikipedia.org/wiki/透磁率 "wikilink")である。∇は、[ナブラ](../Page/ナブラ.md "wikilink")を参照。これは、

\[\mathbf{H} = - \nabla \phi\]

の形（[勾配](https://ja.wikipedia.org/wiki/勾配 "wikilink")でのスカラーポテンシャル参照）であり、磁場に対するスカラーポテンシャルφ<sub>m</sub>（磁気ポテンシャル）は、

\[\phi_m = {1 \over {4 \pi \mu_0} }  { \mathbf{m} \cdot \mathbf{r} \over {r^3} }\]

となる。一様な外部磁場**H**中に磁気双極子が存在する時、その磁気双極子は、

\[\mathbf{P} = \mathbf{m} \times \mathbf{H}\]

の[偶力](https://ja.wikipedia.org/wiki/偶力 "wikilink")を受ける。**P**：偶力、×は[外積](https://ja.wikipedia.org/wiki/外積 "wikilink")を意味する。

## ループ電流との等価性

ループ状に流れる[電流](../Page/電流.md "wikilink")（ループ電流、或いは環状電流）は、十分距離が離れたところから見ると、それが作る磁場を磁気双極子の作る磁場と同じとみなせる。つまりループ電流を磁気双極子とみなすことができる。ループの面積をS、ループに流れる電流をIとすると、ループの中心を通りループ面に垂直な方向での磁気双極子モーメントは、

\[\mathbf{m} = \mu_0 I \mathbf{S}\]

となる。これは、ループ面に対する法線方向をz軸とすると、磁気双極子が作る磁場の大きさと、ループ電流の作る磁場の大きさが等しいとして、

\[H_{dipole} = {1 \over {2 \pi \mu_0} } { m \over {z^3} } = {1 \over {2 \pi}} {IS \over {z^3} }\]

から求まる。

磁気双極子モーメントを真空の透磁率で割った量**μ**、

\[\mathbf{\mu} = { \mathbf{m} \over {\mu_0} }\]

を**[磁気モーメント](../Page/磁気モーメント.md "wikilink")**ということが多い。
