**ブール関数**（ブールかんすう、）は、非負整数 *k* 個の[ブール領域](https://ja.wikipedia.org/wiki/ブール領域 "wikilink") **B** \(= \{0, 1\}\) の引数をとり、1個のブール領域の値となる[関数](https://ja.wikipedia.org/wiki/関数_\(数学\) "wikilink") f : **B**<sup>*k*</sup> → **B** である。*k* = 0 では、単に定数 **B** となる。

ブール関数を一般化すると、*f* : *X* → **B** という形式の関数において、*X* が任意の集合である場合を「[ブール値関数](https://ja.wikipedia.org/wiki/ブール値関数 "wikilink")」と呼ぶ。*X* = **M** = {1, 2, 3, …} であるとき、*f* は無限の「二値数列; 」すなわち 0 と 1 の無限[列である](https://ja.wikipedia.org/wiki/列_\(数学\) "wikilink")。*X* = \[*k*\] = {1, 2, 3, …, *k*} であるとき、*f* は長さ *k* の二値数列である。そのような関数は \(2^{2^k}\) 個存在する。これは[計算複雑性理論](https://ja.wikipedia.org/wiki/計算複雑性理論 "wikilink")における問題で基本的な役割を果たす。

## 効率的表現

（[命題論理](https://ja.wikipedia.org/wiki/命題論理 "wikilink")の）論理式で表現できるが、効率的な表現としては次のようなものがある。

  - [二分決定図](https://ja.wikipedia.org/wiki/二分決定図 "wikilink") (BDD)
  - [否定標準形](https://ja.wikipedia.org/wiki/否定標準形 "wikilink")
  - [Propositional Directed Acyclic Graph](https://ja.wikipedia.org/wiki/:en:Propositional_directed_acyclic_graph "wikilink") (PDAG)

## 標準形

[選言標準形](https://ja.wikipedia.org/wiki/選言標準形 "wikilink")と[連言標準形](https://ja.wikipedia.org/wiki/連言標準形 "wikilink")が代表的である。他に、リード-マラー標準形などがある。

### リード-マラー標準形

リード-マラー標準形（[:en:Algebraic normal form](https://ja.wikipedia.org/wiki/:en:Algebraic_normal_form "wikilink")）は、積（[AND](../Page/論理積.md "wikilink")）の排他的論理和（[XOR](../Page/排他的論理和.md "wikilink")）による標準形である。

|                                    |                                              |
| ---------------------------------- | -------------------------------------------- |
| \(f(x_1, x_2, \ldots , x_n) = \!\) | \(a_0 + \!\)                                 |
|                                    | \(a_1x_1 + a_2x_2 + \ldots + a_nx_n + \!\)   |
|                                    | \(a_{1,2}x_1x_2 + a_{n-1,n}x_{n-1}x_n + \!\) |
|                                    | \(\ldots + \!\)                              |
|                                    | \(a_{1,2,\ldots,n}x_1x_2\ldots x_n \!\)      |

ここで \(a_0, a_1, \ldots, a_{1,2,\ldots,n} \in \{0,1\}^*\) である。

従って、列 \(a_0,a_1,\ldots,a_{1,2,\ldots,n}\) の値の列もブール関数を一意に表している。ブール関数の代数的次数は、1つの（AND）項に現われる \(x_i\) の個数で表される。つまり、\(f(x_1,x_2,x_3) = x_1 + x_3\) の次数は 1（線形）であり、\(f(x_1,x_2,x_3) = x_1 + x_1x_2x_3\) の次数は 3（立方）である。
