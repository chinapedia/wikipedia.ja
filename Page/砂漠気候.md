[Koppen_World_Map_BWh.png](https://ja.wikipedia.org/wiki/File:Koppen_World_Map_BWh.png "fig:Koppen_World_Map_BWh.png") [Koppen_World_Map_BWk.png](https://ja.wikipedia.org/wiki/File:Koppen_World_Map_BWk.png "fig:Koppen_World_Map_BWk.png") **砂漠気候**（さばくきこう）とは[ケッペンの気候区分](https://ja.wikipedia.org/wiki/ケッペンの気候区分 "wikilink")における気候区のひとつで、[乾燥帯](https://ja.wikipedia.org/wiki/乾燥帯 "wikilink")に属する。記号は**BWh**,**BWk**。BWのWはドイツ語のWüste（[砂漠](../Page/砂漠.md "wikilink")）に由来する。

[アリソフの気候区分](https://ja.wikipedia.org/wiki/アリソフの気候区分 "wikilink")では気候帯3-1.熱帯大陸性季節風気候と3-3.海洋性高気圧の東縁気候に対応する\[1\]。

## 特徴

  - 年間を通して[降水量](https://ja.wikipedia.org/wiki/降水量 "wikilink")が少ない。
  - 気温は場所によって大きく異なるが、日較差（一日の温度差）が非常に大きい。
  - 植物がほとんど育たない砂漠となるが、水源のある地域には[オアシス](https://ja.wikipedia.org/wiki/オアシス "wikilink")として気温に応じた植物が群生する。

## 条件

最暖月の平均気温が10℃以上であり（[寒帯](https://ja.wikipedia.org/wiki/寒帯 "wikilink")では無い）、年降水量が[乾燥限界](https://ja.wikipedia.org/wiki/ケッペンの気候区分#乾燥帯（B）の判定 "wikilink")\(r\)の半分に満たない場所であり、以下の式から求められる。条件に当てはまれば砂漠気候となる。

年間平均気温を\(t\)（℃）とし、年平均降水量（mm）が\(0.5r=10(t+x)\)未満を満たす。

\(x\)は以下の降水パターンの条件によって決まる。

|          |               |                                             |
| -------- | ------------- | ------------------------------------------- |
| \(x=14\) | w（冬季乾燥/夏雨）    | 最多雨月が夏にあり、10×最少雨月降水量＜最多雨月降水量                |
| \(x=0\)  | s（夏季乾燥/冬雨）    | 最多雨月が冬にあり、3×最少雨月降水量＜最多雨月降水量かつ最少雨月降水量が30mm未満 |
| \(x=7\)  | f（年中湿潤/年平均降雨） | wでもsでもない                                    |

上記の条件を満たすとき年平均気温が18℃以上なら**BWh**(熱帯砂漠）、18℃未満なら**BWk**（温帯砂漠）となる。

## 気候の特徴

[Libyen-oase1.jpg](https://ja.wikipedia.org/wiki/File:Libyen-oase1.jpg "fig:Libyen-oase1.jpg") [Sandstorm_in_Al_Asad,_Iraq.jpg](https://ja.wikipedia.org/wiki/File:Sandstorm_in_Al_Asad,_Iraq.jpg "fig:Sandstorm_in_Al_Asad,_Iraq.jpg")

年間を通して[中緯度高圧帯](https://ja.wikipedia.org/wiki/中緯度高圧帯 "wikilink")に入る地域で常に大陸性の乾燥した高気圧が居座り、雨が極端に少ない。[日照時間](https://ja.wikipedia.org/wiki/日照時間 "wikilink")が長く昼間の気温上昇が大きいが、夜間の気温低下も著しい。最高気温は夏にはおおむね30℃を上回る地域が多く、40℃を超す地域も珍しくない。50℃を超えるような高温が観測される地点は、そのほとんどが砂漠気候である。最低気温は夏の最も暑い時期には20℃前後と涼しいところも多いが、アラビア半島の海岸地帯などでは30℃を下回らない夜も多い。一方冬の最低気温はサハラ砂漠でも氷点下にまで下がるときもあり、ゴビ砂漠やタクラマカン砂漠などの比較的高緯度の地域では、-20℃以下の酷寒となる。

風が強いため、[砂嵐](https://ja.wikipedia.org/wiki/砂嵐 "wikilink")の発生により被害が出ることもある。また、砂嵐が遠く離れた地域に運ばれて影響を及ぼすこともある。

年降水量はおよそ**250mm以下**で、それを大きく下回る場所も多い。

## 土壌と植生の特徴

[砂漠土](https://ja.wikipedia.org/wiki/砂漠土 "wikilink")と呼ばれる、[風化](https://ja.wikipedia.org/wiki/風化 "wikilink")した砂や岩石（[砂漠](../Page/砂漠.md "wikilink")・沙漠）が広がっている。砂漠土は[アルカリ性](https://ja.wikipedia.org/wiki/アルカリ性 "wikilink")である。

[塩害](https://ja.wikipedia.org/wiki/塩害 "wikilink")が起こりやすい。

乾燥のため、植生は乏しい。河川の周辺や湧水のあるところでは土壌の水分が豊富なため、植物が生育する。アフリカや中東の温暖な地域のオアシスでは[ヤシ科](https://ja.wikipedia.org/wiki/ヤシ科 "wikilink")の植物がよく見られる。

## 産業の特徴・その他

[オアシス](https://ja.wikipedia.org/wiki/オアシス "wikilink")の周辺で農業が行われるほか、[カナート](https://ja.wikipedia.org/wiki/カナート "wikilink")・カレーズ・フォガラ・[ファラジ](https://ja.wikipedia.org/wiki/ファラジ "wikilink")等と呼ばれる地下水道を掘って灌漑が行われている地域もある。また、[ワジ](https://ja.wikipedia.org/wiki/ワジ "wikilink")と呼ばれる[水無川](https://ja.wikipedia.org/wiki/水無川 "wikilink")（枯れ川）の周辺でも伏流水や深層地下水を利用して農業が行われる。農業に適した土地が水のある土地に限られるため、砂漠気候では必然的に集落や都市も水のある土地にまとまって存在する。

また[アフリカ大陸](../Page/アフリカ大陸.md "wikilink")の砂漠では[ラクダ](https://ja.wikipedia.org/wiki/ラクダ "wikilink")、[アジア](https://ja.wikipedia.org/wiki/アジア "wikilink")の砂漠では[ウマ](../Page/ウマ.md "wikilink")などを交通の手段として交易路となっていることがある。古くは、交易都市として栄えた地域もあった。

また日差しが強く、非常に乾燥して風も強い。そのため動植物のほとんどは生育不能となるが一部の河川やオアシスなどでは緑の景観が望め、人などが生活できる。砂漠気候の地域では水源が決定的に不足しており、都市はあまり多くは見られない。

しかし一方で[海水淡水化](https://ja.wikipedia.org/wiki/海水淡水化 "wikilink")や淡水輸送などで[水源](https://ja.wikipedia.org/wiki/水源 "wikilink")が確保された都市は人が集まって急激に人口が増える傾向にあり、近代では開発により大都市並みの生活が送れる場所もある（[ラスベガス](https://ja.wikipedia.org/wiki/ラスベガス "wikilink")など）。海岸沿いに都市が多い理由は海の影響で穏やかな[海洋性気候](https://ja.wikipedia.org/wiki/海洋性気候 "wikilink")となること、[海上輸送](https://ja.wikipedia.org/wiki/海上輸送 "wikilink")が発達できること、水源が確保しやすいことなどがあげられる。[ドバイ](https://ja.wikipedia.org/wiki/ドバイ "wikilink")、[アブダビ](https://ja.wikipedia.org/wiki/アブダビ "wikilink")、[ドーハ](https://ja.wikipedia.org/wiki/ドーハ "wikilink")、[クウェート](../Page/クウェート.md "wikilink")などのペルシャ湾岸の都市では[原油](https://ja.wikipedia.org/wiki/原油 "wikilink")採掘という特殊条件のもとで大量の資金が流入することによって自由な開発が進み[娯楽](https://ja.wikipedia.org/wiki/娯楽 "wikilink")や[観光](../Page/観光.md "wikilink")などの多様な産業が急速に発展している。

高温・乾燥に加えて砂嵐が多いという環境にあるため、それらから身を守ることができるような特徴的な衣装が見られる。女性が用いる[ブルカ](https://ja.wikipedia.org/wiki/ブルカ "wikilink")やエジプト男性が用いる[ガラビア](https://ja.wikipedia.org/wiki/ガラビア "wikilink")などが、その例である。
