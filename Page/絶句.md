**絶句**（ぜっく）は、[漢詩](../Page/漢詩.md "wikilink")における[近体詩](../Page/近体詩.md "wikilink")の代表的な詩型の一つ。4句から成る。

## 概要

原形となる[詩型は](https://ja.wikipedia.org/wiki/詞#詩型 "wikilink")、[六朝時代](https://ja.wikipedia.org/wiki/六朝時代 "wikilink")に作られはじめている。時代が下るにつれて[韻律の規則が次第に整備されて](../Page/韻律_\(韻文\).md "wikilink")、[唐代](https://ja.wikipedia.org/wiki/唐代 "wikilink")に入って詩型として完成された。一句が5文字の五言絶句と7文字の七言絶句がある。[起承転結](../Page/起承転結.md "wikilink")の構成を持つ。

絶句は、(1)[平仄](../Page/平仄.md "wikilink")の規則を厳密に適応した律絶と、(2)制約が比較的緩い古絶に分類される。(2)古絶は五言詩のみである。(1)律絶は格律という点から言えば、[律詩](../Page/律詩.md "wikilink")の前半4句に相当し、「小律詩」とも呼ばれている。

## 平と仄

第1句第2字が平字であるものを**平起**、仄字であるものを**仄起**という。五言では承および結、七言では起、承および結の句で[押韻](../Page/押韻.md "wikilink")する。起句で押韻しないのは**踏落し**という。押韻は平韻が多く、まれに仄韻である。

律詩と同じであるのは、五言の二四不同、七言ではさらに二六対（同）にすること、各句第2字を横に見てゆき平仄仄平（または仄平平仄）とすること、「孤平」、「下二連」を避けることなどの粘法である。

ふつうの平仄式、粘法に合わないのは「拗体」という。六言絶句は数は多くない。

## 杜甫の詩題としての「絶句」

「詩聖」[杜甫](../Page/杜甫.md "wikilink")が詠んだ無題の詩も「絶句」と呼ばれる。

| --------- |

| **原文**    |
| **江碧鳥逾白** |
| **山青花欲燃** |
| **今春看又過** |
| **何日是歸年** |
