[水](../Page/水.md "wikilink")などある種の[溶媒](https://ja.wikipedia.org/wiki/溶媒 "wikilink")の[分子](../Page/分子.md "wikilink")は、[プロトンの供与および受容の両方を行うことができる](../Page/水素イオン.md "wikilink")。このような溶媒中では、一部の溶媒が溶媒同士でプロトンを授受し、[イオン化](../Page/イオン化.md "wikilink")している。この[平衡を溶媒の](../Page/化学平衡.md "wikilink")**自己解離**（じこかいり）と呼ぶ\[1\]。

## 定義

溶媒分子を、プロトンを明らかにして HSol と書くと、この溶媒の自己解離平衡は

  -
    <ce>{2HSol} \<=\> {H2Sol^+} + Sol^-</ce>

と書くことができる。

例えば、

  -
    水 <ce>{2H2O} \<=\> {H3O^+} + OH^-</ce>
    メタノール <ce>2MeOH \<=\> {MeOH2^+} + MeO^-</ce>

である。ただし、遊離のプロトンが存在しているわけではなく常に[溶媒和](https://ja.wikipedia.org/wiki/溶媒和 "wikilink")している。プロトンを受容した溶媒[陽イオン](https://ja.wikipedia.org/wiki/陽イオン "wikilink")を[リオニウム](https://ja.wikipedia.org/wiki/ライオニウム "wikilink") (lyonium)、プロトンを供与した溶媒[陰イオン](https://ja.wikipedia.org/wiki/陰イオン "wikilink")を[ライエイト](https://ja.wikipedia.org/wiki/リエイト "wikilink") (lyate)と呼ぶ\[2\]。

### 酸塩基平衡

溶媒HSol中において以下のような[酸](../Page/酸.md "wikilink")HAの[電離平衡](https://ja.wikipedia.org/wiki/電離平衡 "wikilink")が右辺に著しく偏りリオニウムを定量的に生成する場合、HAは溶媒HSol中において[強酸](../Page/強酸.md "wikilink")であり、平衡が左辺に偏る場合は[弱酸](https://ja.wikipedia.org/wiki/弱酸 "wikilink")として挙動する\[3\]。

  -
    <ce>{HSol} + HA \<=\> {H2Sol^+} + A^-</ce>

また溶媒HSol中において以下のような[塩基](../Page/塩基.md "wikilink")Bの[電離](https://ja.wikipedia.org/wiki/電離 "wikilink")平衡が右辺に著しく偏りリエイトを定量的に生成する場合、Bは溶媒HSol中において[強塩基](../Page/強塩基.md "wikilink")であり、平衡が左辺に偏る場合は[弱塩基](https://ja.wikipedia.org/wiki/弱塩基 "wikilink")として挙動する\[4\]。

  -
    <ce>{HSol} + B \<=\> {HB^+} + Sol^-</ce>

## 自己解離定数

自己解離平衡において、生成したリオニウムとリエイトの濃度の積は温度と圧力に依存する一定の値であり、これを**自己解離定数**、または**イオン積**と呼び *K*<sub>ap</sub>で表す。厳密にはイオン濃度の代わりにイオン[活量](../Page/活量.md "wikilink")を用いるが、一般的に自己解離により生成するイオン濃度は小さいため無限希釈と見なされ濃度と活量はほぼ一致する。

  -
    \(K_{\mathrm{ap}} = [\mathrm{H_2Sol}^+] [\mathrm{Sol}^-] / \mathrm{mol^2~L^{-2}}\)

水の場合は一般に *K*<sub>W</sub>で表し、

  -
    \(K_{\mathrm{W}} = [\mathrm{H_3O}^+] [\mathrm{OH}^-] / \mathrm{mol^2~L^{-2}}\)

である。25 ℃の場合、約10である。また、この対数をとって符号を変えた p*K*<sub>ap</sub>（= −log<sub>10</sub> *K*<sub>ap</sub>）または p*K*<sub>W</sub> （水の場合約14）を自己解離定数または[イオン積](https://ja.wikipedia.org/wiki/イオン積 "wikilink")と呼ぶこともある。

自己解離定数は溶媒のプロトン供与性および受容性が高いほど大きくなり、また[比誘電率](../Page/比誘電率.md "wikilink")が高いほど解離しやすくなる。また溶媒の比[電気伝導度は自己解離により生成するイオンの濃度と移動度の積にほぼ比例し](../Page/電気伝導率.md "wikilink")、またイオンの移動度は[電気泳動](../Page/電気泳動.md "wikilink")的で特にイオン半径の大きなものは[ストークスの法則](https://ja.wikipedia.org/wiki/ストークスの法則 "wikilink")に支配され、溶媒和イオンの半径が小さいほど高く、溶媒の[粘度](https://ja.wikipedia.org/wiki/粘度 "wikilink")に[反比例](../Page/反比例.md "wikilink")する\[5\]\[6\]。ただし、この比伝導度は水分など極微量の不純物に著しく影響され、自己解離定数の小さい溶媒は特に[誤差](../Page/誤差.md "wikilink")が大きい。

## 温度および圧力依存性

水の自己解離に関する[熱力学](../Page/熱力学.md "wikilink")的諸量は以下の通りである\[12\]。

|      | \(\Delta H^\circ\)         | \(\Delta G^\circ\)         | \(\Delta S^\circ\)                       | \(\Delta C_p^\circ\)                     | \(\Delta V^\circ\)                     |
| ---- | -------------------------- | -------------------------- | ---------------------------------------- | ---------------------------------------- | -------------------------------------- |
| 自己解離 | 55.836 kJ mol<sup>−1</sup> | 79.885 kJ mol<sup>−1</sup> | −80.66 J K<sup>−1</sup> mol<sup>−1</sup> | −223.8 J K<sup>−1</sup> mol<sup>−1</sup> | −22.07 cm<sup>3</sup> mol<sup>−1</sup> |
|      |                            |                            |                                          |                                          |                                        |

### 温度依存性

水の自己解離は吸熱的であるため自己解離定数は[温度](../Page/温度.md "wikilink")の上昇と伴に増大する。[ギブス自由エネルギー](https://ja.wikipedia.org/wiki/ギブス自由エネルギー "wikilink")の温度変化と[エンタルピー](../Page/エンタルピー.md "wikilink")変化の間には以下の関係があり、

  -
    \(\left( {\partial (\Delta G / T) \over {\partial T} } \right)_{P} = - {\Delta H \over {T^2} }\)

またギブス自由エネルギーと平衡定数の関係 Δ*G* = －*RT*ln*K* から、平衡定数の温度依存性は以下のようになる。

  -
    \(\left( {\partial \ln K_{\mathrm{ap}} \over {\partial T} } \right)_{P} = {\Delta H^\circ \over {RT^2} }\)

### 圧力依存性

また解離によりイオンの[水和](https://ja.wikipedia.org/wiki/水和 "wikilink")による電縮が起こるため、[エントロピー](../Page/エントロピー.md "wikilink")および[部分モル体積](https://ja.wikipedia.org/wiki/部分モル体積 "wikilink")は減少し、[圧力](../Page/圧力.md "wikilink")の上昇と伴に自己解離定数は増大する。

  -
    \(\left( {\partial \Delta G \over {\partial P} } \right)_{T} = \Delta V\)

## 希薄塩基性溶液の pH

[pH](../Page/水素イオン指数.md "wikilink") は溶液中の水素イオンの[活量](../Page/活量.md "wikilink")、あるいはリオニウムの活量で、

  -
    \(\mathrm{pH} = -\log_{10} a_{\mathrm{H}^+}\)

と定義されるが、希薄溶液では活量を濃度で近似して、

  -
    \(\mathrm{pH} \approx -\log_{10} \left( [\mathrm{H}^+] / \mathrm{mol ~ dm^{-3}} \right) = \mathrm{p}K_{\mathrm{W}} + \log_{10} \left( [\mathrm{OH}^-] / \mathrm{mol ~ dm^{-3}} \right)\)

のように表すこともできる。このように、希薄塩基性溶液の pH を \[OH<sup>−</sup>\] で近似的に求める場合、水の自己解離定数に依存する。

## プロトン授受性から見た溶媒の分類

水に代表されるプロトン授受が可能な溶媒を「両性溶媒」と呼び、他にアルコール類や[過酸化水素](../Page/過酸化水素.md "wikilink")、[酢酸](../Page/酢酸.md "wikilink")などが挙げられる。プロトン供与性は強いが受容性の弱い溶媒を「酸性溶媒」あるいは「プロトン供与性溶媒」と呼び、酢酸や[硫酸](https://ja.wikipedia.org/wiki/硫酸 "wikilink")などがある。一方、プロトン受容性は強いが供与性が弱い、あるいはほとんどない溶媒を「プロトン受容性溶媒」あるいは「塩基性溶媒」と呼び、液体[アンモニア](../Page/アンモニア.md "wikilink")や[ピリジン](https://ja.wikipedia.org/wiki/ピリジン "wikilink")などが挙げられる。プロトン供与性溶媒やプロトン受容性溶媒も一部は純溶媒中でもプロトン授受を行っている。一方、プロトン解離がほとんど起こらない溶媒は一般に「非[プロトン性溶媒](https://ja.wikipedia.org/wiki/プロトン性溶媒 "wikilink")」として区別される。
