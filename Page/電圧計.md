**電圧計**（でんあつけい、）は、ある2点間の[電位](../Page/電位.md "wikilink")差を測る[電気計器](https://ja.wikipedia.org/wiki/電気計器 "wikilink")である。

デジタル電圧計や[自動車](../Page/自動車.md "wikilink")、[オートバイ](../Page/オートバイ.md "wikilink")に搭載される電圧計も、この項目で説明する。 [Volt-Frequenzmesser.jpg](https://ja.wikipedia.org/wiki/File:Volt-Frequenzmesser.jpg "fig:Volt-Frequenzmesser.jpg")

## 使用方法・接続方法

内部[電気抵抗](../Page/電気抵抗.md "wikilink")の大きな測定器であり、測定する回路の2点間に[並列に接続する](https://ja.wikipedia.org/wiki/並列回路 "wikilink")。

[直流](../Page/直流.md "wikilink")の場合、電流制限用の大きな抵抗と[可動コイル形計器を接続し](../Page/指示電気計器.md "wikilink")、流れる微小な[電流](../Page/電流.md "wikilink")で[ムービング・コイル](https://ja.wikipedia.org/wiki/ムービング・コイル "wikilink")（[電磁石](../Page/電磁石.md "wikilink")）を動かして[電圧](../Page/電圧.md "wikilink")を測る。測定範囲の拡大のためには、[倍率器](https://ja.wikipedia.org/wiki/倍率器 "wikilink")が利用される。

[交流](../Page/交流.md "wikilink")の場合は、測定範囲の拡大のために[計器用変圧器](../Page/計器用変圧器.md "wikilink")が利用される。また、交流用計器や、[整流器](../Page/整流器.md "wikilink")と直流用計器を組み合わせたもので測定される。

## アナログ電圧計

### 内部構造

[D.C.VOLTMETER(OPEN).JPG](https://ja.wikipedia.org/wiki/File:D.C.VOLTMETER\(OPEN\).JPG "fig:D.C.VOLTMETER(OPEN).JPG") [A.C.VOLTMETER(OPEN).JPG](https://ja.wikipedia.org/wiki/File:A.C.VOLTMETER\(OPEN\).JPG "fig:A.C.VOLTMETER(OPEN).JPG") 基本的な構造は電流計と同じであるが、回路上に[電気抵抗](../Page/電気抵抗.md "wikilink")があるため定電流が流れると[オームの法則](../Page/オームの法則.md "wikilink")により求められた電圧を検針可能な構造となっている。右に可動コイル型の図を示す。構成部品は永久磁石、コイル、指針となっている。さらにその下の図は可動鉄片形で構成部品は可動鉄片、固定鉄片、指針となる。

### 直流電圧計

[D.C.VOLTMETER.JPG](https://ja.wikipedia.org/wiki/File:D.C.VOLTMETER.JPG "fig:D.C.VOLTMETER.JPG") 直流においての電圧を測定するのに使用される。構造は可動コイル型 (Moving-Coil Type) であり永久磁石およびコイルで構成される。また、振れ角は電流に比例するので目盛り平等目盛りとなる。

### 交流電圧計

[A.C.VOLTMETER.JPG](https://ja.wikipedia.org/wiki/File:A.C.VOLTMETER.JPG "fig:A.C.VOLTMETER.JPG") 構造は可動鉄片形 (Moving-iron Type) であり、非常に簡単、堅牢、安価で[商用電源](https://ja.wikipedia.org/wiki/商用電源 "wikilink")周波数程度 (45 - 65Hz) の交流電圧を測定するのに広く使用されている。本来は実効値の2乗で針が振れるが固定鉄片の形を改良することにより、定格の10 - 20%以上はほぼ平等目盛りとすることができる（写真の目盛りの始まり付近のことである）。

### 電流力計型電圧計

構造的には電力計と同様だが内部配線が異なる。原理的には直流から商用周波数程度（DC - 1000Hzまで可能なものも）である。

### 静電電圧計

二つの金属板に加えられた電圧により力が生じることによって指示する。低電圧の場合金属板に生じる[トルク](../Page/トルク.md "wikilink")が小さいので、もっぱら高電圧 (1 - 500kV) で使用される。

### 熱電形電圧計

熱電形電圧計は15 - 150Vまでが製品化されている。

### 電子電圧計（バルボル、ミリバル）

[HP_RMS_Voltmeter_3400A.jpg](https://ja.wikipedia.org/wiki/File:HP_RMS_Voltmeter_3400A.jpg "fig:HP_RMS_Voltmeter_3400A.jpg") 交流電圧の[実効値](../Page/実効値.md "wikilink")を広い周波数帯域及び広い電圧の範囲を測ることができる装置である。バルボルとは[真空管](../Page/真空管.md "wikilink")電圧計（valve voltmeter, VTVM）の俗称である\[1\]。またミリバルはミリボルト・バルボルの俗称で、微小電圧測定のための測定器である。

## デジタル電圧計

[ADC_and_MCU_diagram.png](https://ja.wikipedia.org/wiki/File:ADC_and_MCU_diagram.png "fig:ADC_and_MCU_diagram.png") ここでは汎用の16bit ADCを例にあげ解説する。IC内部には入力[マルチプレクサ](../Page/マルチプレクサ.md "wikilink")(MUX)、プログラマブルゲインアンプ(PGA)、[アナログ-デジタル変換回路](../Page/アナログ-デジタル変換回路.md "wikilink")(ADC)とそれらを制御し[マイコン](../Page/マイコン.md "wikilink")(MCU)などとの通信をつかさどる制御通信部からなる。下記にMCUによる電圧値取得の例を書く。ピンの割り当てやリファレンス電圧、PGAの設定、サンプルレートなどを指定した後に電圧値を1秒ごとにコンピュータに送信するソフトウェアである。

``` c
SPI spi(p5, p6, p7); // mosi(out), miso(in), sclk(clock)
DigitalOut cs(p8); // cs (the chip select signal)
Serial pc(USBTX, USBRX);

int main() {

    spi.format(8,1); //(bits, mode) (mode1 (CPOL=0, CPHA=1)
    spi.frequency(1000000); //1MHz

    wait_ms(20);//Delay for minimum of 16ms to allow power supplies to settle and power-on reset to complete.

    cs = 0;// Select the device by seting chip select low
    wait_ms(1);//Delay for minimum of tsccs(=10ns)

    spi.write(0x06);//reset
    wait_ms(1);//Delay for minimum of 0.6ms

    spi.write(0x16);//SDATAC: Stop read data continuous mode.
    wait_ms(1);

    //WREG: Write to register rrrr
    spi.write(0x40);//WREG, MUX0
    spi.write(0x00);//
    spi.write(0x01);//AIN0 = + , AIN1 = -
    wait_ms(1);

    spi.write(0x42);//System Control Register0
    spi.write(0x00);//
    spi.write(0x30);//Internal Reference is always on. & Internal Reference selected.
    wait_ms(1);

    spi.write(0x43);//System Control Register0
    spi.write(0x00);//
    spi.write(0x12);//PGA=2, Data Output Rate = 20SPS
    wait_ms(1);

    spi.write(0x04);//SYNC: Synchronize ADC conversions.
    wait_ms(1);

    cs = 1;

    while(1) {
        cs = 0;
        spi.write(0x12);//RDATA: Read data once.
        wait_ms(1);

        unsigned char Volt_hex1 = spi.write(0xff);
        unsigned char Volt_hex2 = spi.write(0xff);
        unsigned short Volt_hex = (Volt_hex1 << 8) | Volt_hex2;
        float Volt = (float)Volt_hex / 0x7fff; //0 ~ 1.024V
        cs= 1;

        pc.printf("%f\r", Volt);
        wait(1);
    }
}
```

表示部を[LCD](https://ja.wikipedia.org/wiki/LCD "wikilink")や[7セグメントディスプレイ](../Page/7セグメントディスプレイ.md "wikilink")に置き換えればデジタルパネルメータとして仕上げることも可能になる。

## 分圧器・計器用変圧器

[Potential_Divider.JPG](https://ja.wikipedia.org/wiki/File:Potential_Divider.JPG "fig:Potential_Divider.JPG") 分圧器は、[直流](../Page/直流.md "wikilink")**電圧計**の測定範囲拡大に使われる[直流用測定範囲拡張器](https://ja.wikipedia.org/wiki/直流用測定範囲拡張器 "wikilink")である。電圧計に直列に接続し電圧計にかかる[電圧](../Page/電圧.md "wikilink")を低下させる[抵抗器](../Page/抵抗器.md "wikilink")である。

電圧計の最大目盛りの時の電圧をV<sub>m</sub>・内部[電気抵抗](../Page/電気抵抗.md "wikilink")をr・電圧降下をV<sub>r</sub>、倍率器の抵抗をR<sub>S</sub>、とすると次のようになる。

  - \(V_m = V_r + \frac{V_r}{r} =  V_r \left(1 + \frac{R_S}{r}\right)\)
  - \(\frac{V_m}{V_r} = \left(1 + \frac{R_S}{r}\right)\)

よって、最大目盛りの時の電圧が\(\left(1 + \frac{R_S}{r}\right)\)倍に拡大されたことになる。

また、端子の切り替えにより倍率器の倍率を変更できる多重目盛りの電圧計も製作されている。

計器用変圧器は高電圧を安全に測定するために用いられ、低電圧を取り出すのに用いられる。
