**荘園公領制**（しょうえんこうりょうせい）とは、[日本](https://ja.wikipedia.org/wiki/日本 "wikilink")の[中世における](https://ja.wikipedia.org/wiki/中世#日本 "wikilink")、[荘園と](https://ja.wikipedia.org/wiki/荘園_\(日本\) "wikilink")[公領](https://ja.wikipedia.org/wiki/公領 "wikilink")を土台とした、重層的土地支配構造のことである。歴史学者の[網野善彦](../Page/網野善彦.md "wikilink")が提唱した。[11世紀](../Page/11世紀.md "wikilink")中後期から[12世紀](../Page/12世紀.md "wikilink")初期にかけて成立し、[院政](../Page/院政.md "wikilink")期を通じて発展し、[鎌倉時代](../Page/鎌倉時代.md "wikilink")前後に最盛期を迎えた。その一方で、鎌倉時代には[地頭](../Page/地頭.md "wikilink")による侵食を受け、[室町時代](../Page/室町時代.md "wikilink")には[守護](../Page/守護.md "wikilink")（[守護大名](../Page/守護大名.md "wikilink")）によって蚕食されるなど、[武士](../Page/武士.md "wikilink")の進出に伴って次第に解体への道を進み、[戦国時代頃までにほぼ形骸化した](../Page/戦国時代_\(日本\).md "wikilink")。最終的には[太閤検地](../Page/太閤検地.md "wikilink")で完全に消滅する。

なお、この項目においての荘園・公領とは、それぞれ荘園公領制の土台となった[寄進地系荘園](https://ja.wikipedia.org/wiki/寄進地系荘園#荘園_\(日本\) "wikilink")・[国衙領](../Page/国衙領.md "wikilink")を指すものとする。

## 歴史

### 成立

[平安時代](../Page/平安時代.md "wikilink")中期以降、[開発領主](../Page/開発領主.md "wikilink")による墾田開発が盛んになる。彼らは[国衙](../Page/国衙.md "wikilink")から田地の私有が認められたが、その権利は危ういものであった。そこで、彼らはその土地を荘園として[受領](../Page/受領.md "wikilink")層に寄進することとなる。受領層は彼ら[大名田堵](../Page/田堵.md "wikilink")（＝開発領主）を[荘官](../Page/荘官.md "wikilink")に任命し、その土地の実効支配権を認める代わりに、荘官から一定の税収を受け取る。こうして荘園を持つようになった受領層を[領家](../Page/領家.md "wikilink")と呼ぶ。領家は次第に、権限を剥奪された補佐官である任用や大名田堵との対立を増すようになる。そこで彼らは自らの荘園をさらに[権門](../Page/権門.md "wikilink")層に寄進し、保護を求める代わりに一定の税収を納めた。こうして荘園を集積した権門層を[本家](../Page/本家.md "wikilink")と呼ぶ。

時を同じくして、[律令制](https://ja.wikipedia.org/wiki/律令制 "wikilink")の崩壊と共に、上級[貴族](../Page/貴族.md "wikilink")は政治への意欲を全くなくし、地方政治は国司に一任されていた（[王朝国家](../Page/王朝国家.md "wikilink")）。増え続ける荘園に対抗して、国司は大名田堵を[在庁官人](../Page/在庁官人.md "wikilink")に任命し、自らの手元に置き、さらには[郡司](../Page/郡司.md "wikilink")・[郷司](https://ja.wikipedia.org/wiki/郷司 "wikilink")・[保司](https://ja.wikipedia.org/wiki/保司 "wikilink")として、地方行政官とした。郡司・郷司・保司はそれぞれ一定額をそれぞれ国司、[朝廷](../Page/朝廷.md "wikilink")に納めればよい。その後、荘園の更なる増大で税収が減り、上級貴族に俸給を払えなくなった朝廷は、彼らに[知行国](../Page/知行国.md "wikilink")として国を与え、その国に関する国司の任命権と税収を上級貴族に与えた。同様に皇族にも[院宮分国制](../Page/院宮分国制.md "wikilink")が敷かれ、こうして権門層たる上級貴族、受領層たる国司、大名田堵たる郡司・郷司・保司が結びつき、荘園制とほとんど変わらぬ構造となった。

なお、この重層的支配構造は「[職（しき）の体系](../Page/職の体系.md "wikilink")」とも呼ばれ、それぞれの立場の職務と権限を「～職」と呼んだ。（領家職、郡司職、郷司職など）

| 土地／職   | 権門   | 受領 | 大名田堵     |
| ------ | ---- | -- | -------- |
| **荘園** | 本家   | 領家 | 荘官       |
| **公領** | 知行国主 | 国司 | 郡司・郷司・保司 |

### 変遷・衰退

次第に在地領主同士の土地争いが増えるに従って、彼らは武装し、[武士](../Page/武士.md "wikilink")となった。[鎌倉幕府](../Page/鎌倉幕府.md "wikilink")の成立に従い、主に東国の武士は鎌倉幕府に[奉公](https://ja.wikipedia.org/wiki/奉公 "wikilink")する、[御家人](../Page/御家人.md "wikilink")となった。彼らはその代償たる[御恩](https://ja.wikipedia.org/wiki/御恩 "wikilink")として、[地頭](../Page/地頭.md "wikilink")に補任され、所領の支配権が鎌倉幕府に保証された。彼らは地頭という全く新しい職に付いたわけではなく、あくまで荘官や郷司・保司のうち幕府と主従関係を結んだものが地頭と呼ばれる点に付いては注意したい。当然彼らは荘官や郷司・保司以上の職は持たないのである。

| 土地                                                                                                                      | 地頭等任命権                                                                      | 収入を得る権利 |
| ----------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------- | ------- |
| **[関東御成敗地](https://ja.wikipedia.org/wiki/関東御成敗地 "wikilink")**                                                           | **[関東御領](../Page/関東御領.md "wikilink")・[関東御分国](../Page/関東御分国.md "wikilink")** | 鎌倉幕府    |
| **[関東進止所領](https://ja.wikipedia.org/wiki/関東進止所領 "wikilink")・[関東御口入地](https://ja.wikipedia.org/wiki/関東御口入地 "wikilink")** | 鎌倉幕府                                                                        | 荘園領主・国司 |
| **[本所一円地](https://ja.wikipedia.org/wiki/本所一円地 "wikilink")**                                                             | 荘園領主・国司                                                                     |         |

関東御成敗地に関しては幕府が地頭の補任権を持つ。つまり、その内関東進止領・関東御口入地には鎌倉幕府とは異なる荘園領主・国司が存在し、しかも彼らは荘官や郷司・保司の任命権は持たないのである。ただ頼朝の頃から、守護が在庁官人を指揮して[大田文](https://ja.wikipedia.org/wiki/大田文 "wikilink")（土地台帳）を作成していた。そこで、地頭の年貢の滞納や領民の不法使用など、地頭職を越えた越権行為が行われるようになり、荘園領主や国司との間に紛争が生じた。その解決策が[地頭請](https://ja.wikipedia.org/wiki/地頭請 "wikilink")や[下地中分](../Page/下地中分.md "wikilink")である。このようにして地頭は領地の支配権を強めていった。

[室町時代](../Page/室町時代.md "wikilink")になると、[幕府の設立の経緯から](../Page/室町幕府.md "wikilink")、[守護](../Page/守護.md "wikilink")の権力が強大であった。彼らは[守護領国制](../Page/守護領国制.md "wikilink")の成立を目指して、在庁官人を含む鎌倉期の地頭、[国人](../Page/国人.md "wikilink")を被官化し、[国衙](../Page/国衙.md "wikilink")や彼らの所持していた領地を掌握した。それと前後して、守護は[半済](../Page/半済.md "wikilink")、[守護請](../Page/守護請.md "wikilink")等で、荘園領主等の権利をも侵食して行き、[戦国時代になると](../Page/戦国時代_\(日本\).md "wikilink")、[守護大名](../Page/守護大名.md "wikilink")に代わった[戦国大名](../Page/戦国大名.md "wikilink")はさらに土地の[一円知行](../Page/一円知行.md "wikilink")を進めていく。

その後、[太閤検地](../Page/太閤検地.md "wikilink")により、土地には直接の耕作者の権利しか認められなくなり（一地一作人）、以前までの重層的支配構造は名実共に解消された。
