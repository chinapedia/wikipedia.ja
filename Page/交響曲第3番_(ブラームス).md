[ヨハネス・ブラームス](../Page/ヨハネス・ブラームス.md "wikilink")の**交響曲第3番**[ヘ長調](https://ja.wikipedia.org/wiki/ヘ長調 "wikilink")作品90（)は、[1883年](../Page/1883年.md "wikilink")5月から10月にかけて作曲された。ブラームスの[交響曲](../Page/交響曲.md "wikilink")の中では演奏時間が最も短い。初演者[ハンス・リヒターは](../Page/ハンス・リヒター_\(指揮者\).md "wikilink")、「この曲は、ブラームスの[『英雄』だ](../Page/交響曲第3番_\(ベートーヴェン\).md "wikilink")。」と表現した。しかし、当のブラームスはこの曲の[標題的な要素についてはなにも語っていない](../Page/標題音楽.md "wikilink")。

## 作曲の経緯

[1877年](../Page/1877年.md "wikilink")に[第2交響曲を作曲したブラームスは](../Page/交響曲第2番_\(ブラームス\).md "wikilink")、その翌年から[ヴァイオリン協奏曲](../Page/ヴァイオリン協奏曲_\(ブラームス\).md "wikilink")（1878年）、『[大学祝典序曲](../Page/大学祝典序曲.md "wikilink")』及び『[悲劇的序曲](../Page/悲劇的序曲.md "wikilink")』（1880年）、[ピアノ協奏曲第2番](../Page/ピアノ協奏曲第2番_\(ブラームス\).md "wikilink")（1881年）といった管弦楽作品を発表する。

第2交響曲から6年後の[1883年](../Page/1883年.md "wikilink")5月、ブラームスは温泉地として知られる[ヴィースバーデン](../Page/ヴィースバーデン.md "wikilink")に滞在し、第3交響曲をこの地で作曲した。第1楽章については、[1882年](../Page/1882年.md "wikilink")の夏ごろからとりかかっていたという説もあるが確証はない。ヴィースバーデンでは、友人達との親交や、とりわけ若い[アルト](../Page/アルト.md "wikilink")歌手ヘルミーネ・シュピースとの恋愛感情がこの曲に影響を及ぼしたともいわれる。同年[10月2日](../Page/10月2日.md "wikilink")にブラームスは[ウィーン](https://ja.wikipedia.org/wiki/ウィーン "wikilink")に戻り、[11月9日](../Page/11月9日.md "wikilink")と[22日に](https://ja.wikipedia.org/wiki/11月22日 "wikilink")[イグナーツ・ブリュル](../Page/イグナーツ・ブリュル.md "wikilink")と2台のピアノ版による試演会を開いている。ブラームスは50歳だった。

## 初演

1883年[12月2日](../Page/12月2日.md "wikilink")、ハンス・リヒターの[指揮により](../Page/指揮_\(音楽\).md "wikilink")、[ウィーン・フィルハーモニー管弦楽団](../Page/ウィーン・フィルハーモニー管弦楽団.md "wikilink")の演奏会で初演された。結果は大成功で、ブラームスは再三カーテンコールを受けた。

当時のドイツ音楽界は、ブラームス派対[ワーグナー派という陣営の対立が激化していたが](../Page/リヒャルト・ワーグナー.md "wikilink")、1883年にワーグナーが没してまもない時期であり、ワーグナー派の強い反発のなかでの初演だった。この初演を聴いた批評家でブラームスの友人でもあった[エドゥアルト・ハンスリック](../Page/エドゥアルト・ハンスリック.md "wikilink")は、[第1交響曲](../Page/交響曲第1番_\(ブラームス\).md "wikilink")、[第2交響曲と比べても](../Page/交響曲第2番_\(ブラームス\).md "wikilink")「芸術的に完璧な作品として心を打つ。」と絶賛している。一方、ワーグナー派で反ブラームスの急先鋒でもあった[フーゴ・ヴォルフ](../Page/フーゴ・ヴォルフ.md "wikilink")は、「まったく独創性というものが欠けたできそこないの作品である」とほとんど中傷に近い批評を寄せた。

## モットーについて

ブラームスは、[第1番ではC](../Page/交響曲第1番_\(ブラームス\).md "wikilink")-C\#-D（[ハ](https://ja.wikipedia.org/wiki/ハ_\(音名\) "wikilink")－[嬰ハ](https://ja.wikipedia.org/wiki/嬰ハ "wikilink")－[ニ](https://ja.wikipedia.org/wiki/ニ_\(音名\) "wikilink")）、[第2番ではD](../Page/交響曲第2番_\(ブラームス\).md "wikilink")-C\#-D（ニ－嬰ハ－ニ）と、過去の2つの交響曲で統一的な基本動機を用いていたが、この曲では、基本動機からさらに一歩踏み込んだ形として、[モットー](https://ja.wikipedia.org/wiki/モットー "wikilink")を使用している。曲の冒頭で管楽器によって示されるF-A♭-F（[ヘ](https://ja.wikipedia.org/wiki/ヘ_\(音名\) "wikilink")－[変イ](https://ja.wikipedia.org/wiki/変イ "wikilink")－ヘ）がそれである。このモットーは第1楽章全体を支配し、他の楽章でもあちこちに顔を出す。それだけでなく、これまでの基本動機のように素材として展開されたり旋律を形作ることより、むしろ以下に述べるように、交響曲全曲の性格を決定づけるものとなっている。

モットーの音型について、ブラームスの友人であった伝記作者[カルベックは](https://ja.wikipedia.org/wiki/マックス・カルベック "wikilink")、ブラームスが好んだという“Frei aber froh”（自由だが喜ばしく）という言葉の頭文字と結び付けている。この言葉は、ブラームスの親友[ヨーゼフ・ヨアヒム](../Page/ヨーゼフ・ヨアヒム.md "wikilink")のモットーである“Frei aber einsam”（自由だが孤独に）と対をなす合い言葉のようなものであったらしい（[F.A.E.ソナタ](../Page/F.A.E.ソナタ.md "wikilink")を参照）。とはいえ、交響曲の主調がヘ長調であるなら単純にF-A-Fとなるところを、ブラームスがあえてF-A♭-F（ドイツ語ではF-As-F）という[ヘ短調](https://ja.wikipedia.org/wiki/ヘ短調 "wikilink")に属する音型を用いていることは注目される。ここから生じるヘ長調とヘ短調の葛藤は、全曲の性格に決定的な影響を与えている。

## 楽器編成

[フルート](../Page/フルート.md "wikilink")2、[オーボエ](../Page/オーボエ.md "wikilink")2、[クラリネット](../Page/クラリネット.md "wikilink")2、[ファゴット](../Page/ファゴット.md "wikilink")2、[コントラファゴット](../Page/コントラファゴット.md "wikilink")（１，4楽章）、[ホルン](../Page/ホルン.md "wikilink")4（3rdと4thは１，4楽章）、[トランペット](../Page/トランペット.md "wikilink")2（１，4楽章）、[トロンボーン](../Page/トロンボーン.md "wikilink")3（１，2，4楽章）、[ティンパニ](../Page/ティンパニ.md "wikilink")（１，4楽章）、[弦五部](https://ja.wikipedia.org/wiki/弦五部 "wikilink")。

## 演奏時間

約37分と、ブラームスの交響曲の中では最も短い（第1楽章の繰り返しを含む）。

## 楽曲構成

### 第1楽章 Allegro con brio

| [Orchesterwerke_Romantik_Themen.pdf](https://ja.wikipedia.org/wiki/File:Orchesterwerke_Romantik_Themen.pdf "fig:Orchesterwerke_Romantik_Themen.pdf") | {{試聴 | image = none | filename = Brahms, Symphony No. 3 in F Major, Op. 90 - I. Allegro con brio.ogg | title = 第1楽章　Allegro con brio | description = Musopenより }} |

ヘ長調。6/4拍子。[ソナタ形式](../Page/ソナタ形式.md "wikilink")(提示部反復指定あり)。

冒頭、管楽器のモットーにつづいて、[ヴァイオリン](../Page/ヴァイオリン.md "wikilink")が第1主題を示す。モットーの持つヘ短調の響きが、表情に陰りを与えている。この主題は、ほぼ同じ形の動機が一瞬登場する[シューマンの](../Page/ロベルト・シューマン.md "wikilink")[交響曲第1番「春」の第](../Page/交響曲第1番_\(シューマン\).md "wikilink")2楽章や、[交響曲第3番「ライン」の第](../Page/交響曲第3番_\(シューマン\).md "wikilink")1楽章との関連を指摘されることもある。静かな経過句を経て9/4拍子となり、クラリネットが第2主題を[イ長調](../Page/イ長調.md "wikilink")で出す。この主題は、ワーグナーの歌劇『[タンホイザー](../Page/タンホイザー.md "wikilink")』の「ヴェーヌスベルクの音楽」と共通点があるとも指摘される。主題の後には第2交響曲の基本動機も顔を出す。提示部は反復指定があるが、ブラームスの他の交響曲に比べて実行される頻度はやや高い。展開部は情熱的に始まり、低弦が第2主題を暗い[嬰ハ短調](https://ja.wikipedia.org/wiki/嬰ハ短調 "wikilink")で奏する。静まると、ホルンがモットーに基づく旋律を大きく示す。第1主題の動機を繰り返しながら高まって、再現部に達する。コーダでは、モットーと第1主題が絡み合うが、収拾されて静まる。モットーが響くなか、第1主題が消え入るように奏されて終わる。

[ハ長調](../Page/ハ長調.md "wikilink")。4/4拍子。自由な[三部形式](https://ja.wikipedia.org/wiki/三部形式 "wikilink")あるいは自由なソナタ形式と見られる。

第1主題は[クラリネット](../Page/クラリネット.md "wikilink")と[ファゴット](../Page/ファゴット.md "wikilink")のひなびた旋律。各フレーズの終わりでモットーが示される。この第1主題に含まれる、3度をゆらゆらと反復する動機も目立つ。第2主題は同じくクラリネットとファゴットが新たに[コラール](../Page/コラール.md "wikilink")風の旋律を奏する。ヴァイオリンの新しい旋律(コデッタ主題)に受け継がれてから、経過的な展開部に入る。この楽章を三部形式とみなす場合は第2主題およびコデッタの部分が中間部に相当する。展開部は比較的小規模で第1主題の断片を奏して再現部を導く。第2主題の再現は省略され、コーダでは第1主題が静かにクラリネットで奏されてから曲が終わる。

### 第4楽章 Allegro - Un poco sostenuto

|                                                                                                                                                        |      |              |                                                                      |                                          |                            |
| ------------------------------------------------------------------------------------------------------------------------------------------------------ | ---- | ------------ | -------------------------------------------------------------------- | ---------------------------------------- | -------------------------- |
| [Orchesterwerke_Romantik_Themen.pdf](https://ja.wikipedia.org/wiki/File:Orchesterwerke_Romantik_Themen.pdf "fig:Orchesterwerke_Romantik_Themen.pdf") | {{試聴 | image = none | filename = Brahms - Symphony No.3 in F major Op.90 - IV. Allegro.ogg | title = 第4楽章　Allegro - Un poco sostenuto | description = Musopenより }} |

ヘ短調－ヘ長調。2/2拍子。自由なソナタ形式。

ファゴットと弦が第1主題を示す。[トロンボーン](../Page/トロンボーン.md "wikilink")の同音反復に導かれて、第2楽章のコラール風動機が奏される。直後に、音楽は激しくなり情熱的にすすむ。第2主題はハ長調、チェロとホルンによる三連符を用いたもの。小結尾はハ短調となる。展開部は第1主題を専ら扱うもので第1主題の再現を兼ねている。ゆえにここから再現部と見なすこともある。コラール風の動機が強奏で繰り返され、ハ短調から半音ずつずり上がってヘ長調に達し、一転ヘ短調となる。コーダに入ると、第1主題が表情を変えながら繰り返され、やがてヘ長調に転じる。モットーが現れ、弦の細かな反復する動きに乗って、コラール風の動機が示される。最後には第1楽章第1主題が回想され、静かに曲を閉じる。
