**摂動**（せつどう、 ）とは、一般に力学系において、主要な力の寄与（**主要項**）による運動が、他の副次的な力の寄与（**摂動項**）によって乱される現象である。摂動という語は元来、[古典力学](../Page/古典力学.md "wikilink")において、ある天体の運動が他の天体から受ける引力によって乱れることを指していたが、その類推から[量子力学](https://ja.wikipedia.org/wiki/量子力学 "wikilink")において、粒子の運動が複数粒子の間に[相互作用](https://ja.wikipedia.org/wiki/相互作用 "wikilink")が働くことによって乱れることも指すようになった。なお、転じて摂動現象をもたらす副次的な力のことを摂動と呼ぶ場合がある。

## 摂動論

上記のような複数天体間、複数粒子間に相互作用が働くときの運動は数学的に厳密に解くことができないことが知られている（[多体問題](../Page/多体問題.md "wikilink")）。これらの数学的に厳密に解くことのできない問題の近似解を求める手法の1つに、**摂動論**（せつどうろん、 ）がある。具体的には、次のような手順で近似解を求める。

  - 考えている問題Aを、厳密に解ける問題Bに小さな変更（**摂動**）が加えられた問題であるとみなす。
  - 問題Aの近似解は、問題Bの厳密解に、摂動が加わったことによって生じる小さな補正（**摂動項**）を加えたものであると考える。
  - ここで求めるべき摂動項は、問題Bの厳密解の組み合わせ、すなわち一次結合の形で表現出来ると考え、その係数を与えられた条件から順次求める。

## 古典力学における摂動論

天体の運行において、[月](https://ja.wikipedia.org/wiki/月 "wikilink")と[地球](https://ja.wikipedia.org/wiki/地球 "wikilink")、[太陽](https://ja.wikipedia.org/wiki/太陽 "wikilink")と[地球](https://ja.wikipedia.org/wiki/地球 "wikilink")などを扱う二体問題は厳密に解くことができるが、三体以上の[多体問題](../Page/多体問題.md "wikilink")を厳密に解くことは不可能である。ただし、月と地球、太陽と地球の問題では、他の天体からの引力による相互作用の効果は近似的に非常に小さいとして、これら二体問題に他の天体からの効果を補正項として考慮することによって十分精度の高い近似解を得ることができる。

## 量子力学における摂動論

量子力学における多体問題を解く上においても摂動論は重要な近似解法である。

### 時間に依存せず、[縮退](https://ja.wikipedia.org/wiki/縮退 "wikilink")のない場合

#### 前提

無摂動部分（無摂動項）の[ハミルトニアン](../Page/ハミルトニアン.md "wikilink")を\(\mathcal{H}_0\)とし、摂動部分（摂動項）を\(\mathcal{H}'\)とすると、全体のハミルトニアン\(\mathcal{H}\)は、

\[\mathcal{H} = \mathcal{H}_0 + \mathcal{H}'\]

となる。この時、ゼロ次（無摂動項）のハミルトニアン\(\mathcal{H}_0\)については、すべての[固有値](https://ja.wikipedia.org/wiki/固有値 "wikilink")（固有エネルギー）\(\{\epsilon_n^{(0)}\}\)と、対応する[固有ベクトル](https://ja.wikipedia.org/wiki/固有ベクトル "wikilink")\(\{|\Psi_n^{(0)}\rangle\}\)が完全に分かっているとする。ここで「対応する」とは固有値方程式

\[\mathcal{H}_0 |\Psi_n^{(0)}\rangle = \epsilon_n^{(0)} |\Psi_n^{(0)}\rangle\] を満たす関係にあるという意味である。 \(\mathcal{H}_0\)は[エルミート演算子](https://ja.wikipedia.org/wiki/エルミート演算子 "wikilink")である（つまりエネルギーは[オブザーバブル](https://ja.wikipedia.org/wiki/オブザーバブル "wikilink")である）ので、その固有ベクトル\(\{|\Psi_n^{(0)}\rangle\}\)は[完全系](https://ja.wikipedia.org/wiki/完全系 "wikilink")を成している。また\(\{|\Psi_n^{(0)}\rangle\}\)は規格直交化されているとする。

ハミルトニアン\(\mathcal{H}\)の固有ベクトル\(\{|\Psi_n\rangle\} \\)と、対応する固有値\(\{\epsilon_n\}\)を求めたい。ここで\(\{|\Psi_n\rangle\} \\)と\(\{\epsilon_n\}\)は

\[\mathcal{H} |\Psi_n\rangle = \epsilon_n |\Psi_n\rangle\] つまり

\[(\mathcal{H}_0 + \mathcal{H}') |\Psi_n\rangle = \epsilon_n |\Psi_n\rangle \cdots (0)\] を満たさなければならない。

#### 摂動論

摂動論では、未知の\(\mathcal{H}'\)、\(|\Psi_n\rangle \\)、\(\epsilon_n\)を、既知の\(V \\)、\(|\Psi_n^{(0)}\rangle\)、\(\epsilon_n^{(0)}\)と、未知の\(\{ |\Psi_n^{(1)}\rangle, |\Psi_n^{(2)}\rangle ,\dotsc \}\)、\(\{\epsilon_n^{(1)}, \epsilon_n^{(2)} , \dotsc \}\)、微小係数\(\lambda \\)を用いて

\[\begin{align}
\mathcal{H}' &= \lambda V \\
|\Psi_n\rangle &= |\Psi_n^{(0)}\rangle + \lambda |\Psi_n^{(1)}\rangle + \lambda^2 |\Psi_n^{(2)}\rangle + \dotsb \\
\epsilon_n &= \epsilon_n^{(0)} + \lambda \epsilon_n^{(1)} + \lambda^2 \epsilon_n^{(2)} + \dotsb
\end{align}\]

と表す。べき級数の中で既知であるのは、第１項目だけであることに注意。 これで、\(|\Psi_n\rangle \\)、\(\epsilon_n\)を求める問題は\(\{ |\Psi_n^{(1)}\rangle, |\Psi_n^{(2)}\rangle ,\dotsc \}\)、\(\{\epsilon_n^{(1)}, \epsilon_n^{(2)} , \dotsc \}\)を求める問題に変換された。

これらを(0)式に代入し、任意の\(\lambda \\)で成立すると仮定すると、

  - 未知の\((|\Psi_n^{(1)}\rangle, \epsilon_n^{(1)})\)だけを含む方程式　　\(\cdots (1)\)
  - 未知の\((|\Psi_n^{(2)}\rangle, \epsilon_n^{(2)})\)と\((|\Psi_n^{(1)}\rangle, \epsilon_n^{(1)})\)だけを含む方程式　　\(\cdots (2)\)
  - 未知の\((|\Psi_n^{(3)}\rangle, \epsilon_n^{(3)})\)と\((|\Psi_n^{(2)}\rangle, \epsilon_n^{(2)})\)と\((|\Psi_n^{(1)}\rangle, \epsilon_n^{(1)})\)だけを含む方程式　　\(\cdots (3)\)

　　　\(\vdots\)

が得られ、未知数を分離することができる。 これらを(1)式、(2)式、・・・の順に解いていくと、\(\{ |\Psi_n^{(1)}\rangle, |\Psi_n^{(2)}\rangle ,\dotsc \}\)、\(\{\epsilon_n^{(1)}, \epsilon_n^{(2)} , \dotsc \}\)が求まる。

これらの式は、未知の\(\{ |\Psi_n^{(1)}\rangle, |\Psi_n^{(2)}\rangle ,\dotsc \}\)を、既知の完全系\(\{|\Psi_n^{(0)}\rangle\}\)の線形結合（重ね合わせ）で展開して、その展開係数\(c_i \\)を求める問題に変換することで解ける。

\[|\Psi_n^{1}\rangle = c_1 |\Psi_1^{(0)}\rangle + c_2 |\Psi_2^{(0)}\rangle + c_3 |\Psi_3^{(0)}\rangle + \dotsb = \sum_i c_i |\Psi_i^{(0)}\rangle\]

#### 結果

エネルギーの一次の摂動は、\(|\Psi_n^{(0)}\rangle = |n \rangle\)とすると、

\[\epsilon_n^{(1)} = \langle n|\mathcal{H}'|n\rangle\] 固有ベクトルの一次の摂動の展開係数は、\(i \neq n\)とすると

\[c_i = - { \langle i|\mathcal{H}'|n\rangle \over { \epsilon_i^{(0)} - \epsilon_n^{(0)} } }\] 二次の摂動エネルギーは、

\[\epsilon_n^{(2)} = - \sum_{\{m|m\neq n\}} { \langle n|\mathcal{H}'|m\rangle\langle m|\mathcal{H}'|n\rangle \over { \epsilon_m^{(0)} - \epsilon_n^{(0)} } }\] ここで、\(\lambda\langle n|V|n \rangle = \langle n|\lambda V|n \rangle = \langle n|\mathcal{H}'|n \rangle\)である（他の項も同様）。

### 縮退のある場合

固有値が縮退している場合は、*i* ≠ *n*、*m* ≠ *n*の場合でも*ε<sub>i</sub>* = *ε<sub>n</sub>*、*ε<sub>m</sub>* = *ε<sub>n</sub>*となる場合が存在し、この場合上式二次摂動エネルギーや、一次の摂動波動関数の係数の分母部分が零となり発散してしまう。従って、縮退のある場合には、このような発散を回避する手段を施す必要がある（[ほとんど自由な電子](../Page/ほとんど自由な電子.md "wikilink")参照）。

摂動は普通、一次の項まで考慮すれば十分であるが、より高次な項を考える必要がある場合も多い（例：[近藤効果](https://ja.wikipedia.org/wiki/近藤効果 "wikilink")は摂動の二次の項まで考慮しないと説明できない）。

### 縮退のある場合の一次摂動

(摂動のない)シュレディンガー方程式

\[H|\psi\rangle = E|\psi\rangle\]

の固有値\(E_{n}\)がk重縮退していて、その対応する固有状態を\(|n_{i}\rangle \; (i = 1,2,3,...,k, k\in N)\)と表す。

微小な摂動\(gV\)(\(g\)は無次元の微小項)を加えた後、エネルギー固有値\(E_{n}\)を持っていた状態に関するシュレディンガー方程式は

\[(H+gV)|\psi_{n}(g)\rangle=E_{n}(g)|\psi_{n}(g)\rangle\]

となる。

ここで

\[\begin{align}
E_{n}(g) &= E_{n}^{(0)} + gE_{n}^{(1)}+ \dotsb \\
|\psi_{n}(g)\rangle &= |\psi_{n}^{(0)}\rangle + g|\psi_{n}^{(1)}\rangle + \dotsb
\end{align}\]

と展開できるとして、前述のシュレディンガー方程式の0次項を取り出して、

\[H|\psi_{n}^{(0)}\rangle=E_{n}^{(0)}|\psi_{n}^{(0)}\rangle\]

を得るが、摂動がない時のシュレディンガー方程式より

\[|\psi_{n}^{(0)}\rangle=\sum_{i}c_i|n_{i}\rangle\]

とおくことができる。

次に、シュレディンガー方程式の1次項を取り出すと、

\[\begin{align}
H|\psi_{n}^{(1)}\rangle + V|\psi_{n}^{(0)}\rangle &= E_{n}^{(0)}|\psi_{n}^{(1)}\rangle+E_{n}^{(1)}|\psi_{n}^{(0)}\rangle \\
(V - E_{n}^{(1)}) |\psi_{n}^{(0)}\rangle &= (E_{n}^{(0)} - H) |\psi_{n}^{(1)}\rangle
\end{align}\]

これに左から\(\langle n_j|\)をかけて

\[\langle n_j|(V-E_{n}^{(1)})|\psi_{n}^{(0)}\rangle=0\]

よって

\[c_jE_{n}^{(1)}=\sum_{i}\langle n_j|V|n_i\rangle c_i\]

が成り立つ。

これをすべての\(j\)について出すと、\(n\)個の\(n+1\)元方程式が得られるが、規格化を考えていないため、この\(n+1\)個の方程式を解くて、エネルギーの一時摂動及び縮退が解ける様子がわかる。

### グリーン関数による方法

ここまでに挙げたのは状態ベクトルに対する摂動論であるが、系が時間に依存する場合など、演算子に対する摂動論も便利である。

演算子に対する摂動論として、[グリーン関数](../Page/グリーン関数.md "wikilink")を使う方法が知られている。
