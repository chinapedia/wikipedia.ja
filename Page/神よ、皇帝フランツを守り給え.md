『**神よ、皇帝フランツを守り給え**』（）あるいは『**オーストリア皇帝讃歌**』（）は、[神聖ローマ帝国](../Page/神聖ローマ帝国.md "wikilink")、[オーストリア帝国](../Page/オーストリア帝国.md "wikilink")（のち[オーストリア＝ハンガリー帝国](../Page/オーストリア＝ハンガリー帝国.md "wikilink")）の[国歌](../Page/国歌.md "wikilink")。[フランツ・ヨーゼフ・ハイドン](../Page/フランツ・ヨーゼフ・ハイドン.md "wikilink")の作曲による。ハイドンによるメロディーはその後、数多くの[讃美歌](https://ja.wikipedia.org/wiki/讃美歌 "wikilink")や[クラシック音楽](../Page/クラシック音楽.md "wikilink")の中で用いられ、またドイツの国歌『[ドイツの歌](../Page/ドイツの歌.md "wikilink")』として転用されている。

## 作曲に至る過程

作曲の経緯には、2つの説がある。いずれにせよ、[イングランド](../Page/イングランド.md "wikilink")の『[神よ国王を護り賜え](../Page/女王陛下万歳.md "wikilink")』の影響を強く受けて成立したことは確実である。

### 第1の説

[Heinrich_Friedrich_Füger_004.jpg](https://ja.wikipedia.org/wiki/File:Heinrich_Friedrich_Füger_004.jpg "fig:Heinrich_Friedrich_Füger_004.jpg") [Haydn_portrait_by_Thomas_Hardy_(small).jpg](https://ja.wikipedia.org/wiki/File:Haydn_portrait_by_Thomas_Hardy_\(small\).jpg "fig:Haydn_portrait_by_Thomas_Hardy_(small).jpg")\]\]

  -

    伯爵が作曲全体のイニシアティブをとったとする説。ゾーラウ伯爵の1820年2月28日付の書状の内容に基づく。

[フランス革命戦争](../Page/フランス革命戦争.md "wikilink")の最中である1796年の終わり、神聖ローマ帝国（オーストリア）はナポレオン軍に脅かされるなど政治的に不安定な情勢下にあった。そのため皇帝のもとでの国民の団結が強く求められていたが、時の神聖ローマ皇帝[フランツ2世](https://ja.wikipedia.org/wiki/フランツ2世 "wikilink")は優柔不断かつ悲観的な性格で、それほど国民に愛されてはいなかった。

そこで[ニーダーエスターライヒ州](../Page/ニーダーエスターライヒ州.md "wikilink")政府長官の伯爵は、国民の団結を促す契機となるものを作らんと、詩人のに、イングランドの『[神よ国王を護り賜え](../Page/女王陛下万歳.md "wikilink")』を手本として、翌年の皇帝の誕生日（＝1797年2月12日）に演奏する歌の歌詞を書くように依頼した。1796年10月11日、ハシュカは完成した歌詞を、ゾーラウ伯爵に送った。

ゾーラウ伯爵から作曲を依頼されたハイドンも、1791年から92年、また1794年から95年にかけてロンドンに赴いた時に『神よ国王を護り賜え』を耳にしており、それを大いに意識して作曲を行った。（ハイドンの英国訪問については「[ロンドン交響曲](https://ja.wikipedia.org/wiki/ロンドン交響曲_\(ハイドン\) "wikilink")」も参照）

### 第2の説

  -
    ウィーンの宮廷図書館に勤めていたという人物の記した書物に基づく。

『神よ国王を護り賜え』を聴いたハイドンが、ウィーンの音楽愛好家である男爵に、オーストリアにも同様の国歌があればと述べた。スウィーテンがゾーラウ伯爵にこれを伝え、ゾーラウ伯爵がハシュカに作詞を依頼した。

## 作曲

[Gott_erhalte_Caroline.jpg](https://ja.wikipedia.org/wiki/File:Gott_erhalte_Caroline.jpg "fig:Gott_erhalte_Caroline.jpg")  オーケストラ譜には、ハイドン自身の手で「Volck's Lied」という注記があり、その意味は「国民の歌」とも「民謡」ともとれる。これに関連し、作曲時にハイドンが参考にした可能性がある民謡がある。クロアチア民謡「悲しき花嫁（Zalostna zarucnice）」で、冒頭部分の音符とリズムの輪郭が完全に一致する。[H.C.ロビンス・ランドン](https://ja.wikipedia.org/wiki/H.C.ロビンス・ランドン "wikilink")は、ハイドンが1761年以降仕えていた[エステルハージ家](https://ja.wikipedia.org/wiki/エステルハージ家 "wikilink")の居城があったエステルハーザ付近でこの民謡が歌われていたと指摘している。

カトリックの『』に含まれる《天にましますわれらの父よ》とも、冒頭の旋律に含まれる音が一致するなど、類似が指摘されている。

## 初演

1797年1月28日にゾーラウ伯爵がハイドンの浄書譜に出版を許可しているため、それまでには作曲が完了していたと考えられる。印刷は大急ぎで行われ、1月30日には初版の印刷が完了したらしい。そして皇帝フランツ2世の29歳の誕生日である1797年2月12日、ウィーンの[ブルク劇場](https://ja.wikipedia.org/wiki/ブルク劇場 "wikilink")において、フランツ2世臨席のもとでオーケストラ伴奏によって初演された。初演後まもなく、ハイドンと交流があったが、歌詞を英訳してイングランドで紹介し、好評を博した。

## 歌詞

初演以降、この歌は歌詞が何度か変えられている。「皇帝賛美」というコンセプトは同じであるが、[フランツ2世](https://ja.wikipedia.org/wiki/フランツ2世 "wikilink")の時だけでも2度変えられており、皇帝の代替わりの際に歌詞が変えられることがあった。

1854年の改訂によって、皇帝の個人名が含まれていない一般的な内容による歌詞になった。ただしフランツ・ヨーゼフ1世が[エリーザベトを皇后に迎えた後に加筆された](../Page/エリーザベト_\(オーストリア皇后\).md "wikilink")5番、[ルドルフ皇太子の誕生後に加筆された](../Page/ルドルフ_\(オーストリア皇太子\).md "wikilink")6番には、個人名が含まれている。

[第一次世界大戦](../Page/第一次世界大戦.md "wikilink")のさなかに[カール1世が皇位を継承すると](https://ja.wikipedia.org/wiki/カール1世_\(オーストリア皇帝\) "wikilink")、歌詞を改訂しようとする動きがあったが、その前に[ハプスブルク帝国](../Page/ハプスブルク帝国.md "wikilink")は敗戦とともに瓦解した。すなわち、カール1世の治世下では最後までフランツ・ヨーゼフ1世の時の歌詞が用いられた。[1989年](../Page/1989年.md "wikilink")4月1日に催された最後の皇后[ツィタ・フォン・ブルボン＝パルマ](../Page/ツィタ・フォン・ブルボン＝パルマ.md "wikilink")の葬儀と、[2011年](../Page/2011年.md "wikilink")7月16日に催された最後の皇太子[オットー・フォン・ハプスブルク](https://ja.wikipedia.org/wiki/オットー・フォン・ハプスブルク "wikilink")の葬儀でも、この「1854年版」の第1番が唱和されている。

## 共和制移行後

[Otto_Habsburg_001.jpg](https://ja.wikipedia.org/wiki/File:Otto_Habsburg_001.jpg "fig:Otto_Habsburg_001.jpg")\]\] 最後の皇帝カール1世が1922年に亡命先の[マデイラ島](https://ja.wikipedia.org/wiki/マデイラ島 "wikilink")で崩御した後、その遺児である元皇太子オットーを「[オーストリア皇帝](https://ja.wikipedia.org/wiki/オーストリア皇帝 "wikilink")」と仰ぐ王党派によって、ハイドンの旋律に新たな歌詞が付けられた。

|  |
|  |
|  |
|  |

[オーストリア＝ハンガリー帝国](../Page/オーストリア＝ハンガリー帝国.md "wikilink")崩壊後に成立した[第一共和国は](../Page/第一共和国_\(オーストリア\).md "wikilink")、帝政時代の国歌を受け継がず、共和国初代首相[カール・レンナー](../Page/カール・レンナー.md "wikilink")の作詞、の作曲による『』を国歌とした。

しかしこの曲はあまり国民に受け入れられず、1930年にはハイドンの皇帝讃歌の旋律にの詩『』を乗せたものを国歌とするという布告が出された。[第二次世界大戦](https://ja.wikipedia.org/wiki/第二次世界大戦 "wikilink")後、それまでの国歌の使用が[連合国](https://ja.wikipedia.org/wiki/連合国 "wikilink")によって禁止され、結果的に[ヴォルフガング・アマデウス・モーツァルト](../Page/ヴォルフガング・アマデウス・モーツァルト.md "wikilink")の作曲とされる旋律を用いた『[山岳の国、大河の国](../Page/山岳の国、大河の国.md "wikilink")』が制定されたが、再びハイドンの曲をオーストリア国歌の位置に戻す努力もなされた。
