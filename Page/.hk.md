**.hk**は、[香港](https://ja.wikipedia.org/wiki/香港 "wikilink")の[国別コードトップレベルドメイン](../Page/国別コードトップレベルドメイン.md "wikilink")(ccTLD)である。

イギリスの植民地であった[1990年](https://ja.wikipedia.org/wiki/1990年 "wikilink")に導入された。[1997年](https://ja.wikipedia.org/wiki/1997年 "wikilink")に[中華人民共和国](../Page/中華人民共和国.md "wikilink")への[返還が行われてその一部となったが](../Page/香港返還.md "wikilink")、過渡的な[一国二制度](../Page/一国二制度.md "wikilink")の政策により、本土のTLDである[.cn](../Page/.cn.md "wikilink")とは別のTLDを使い続けることが許されている。

Hong Kong Internet Registration Corporation(HKIRC)が、.hkドメインの管理を行うことを[香港政府によって支持された唯一の組織である](../Page/香港特別行政区政府.md "wikilink")。HKIRCは非営利で、法令によって設立されたものではない会員制の団体で、2001年に設立された。

1990年から2002年までは、 Joint Universities Computer Centre（大学連合コンピュータセンター）が.hkドメインを管理していた\[1\]。2002年、香港政府による[了解覚書](https://ja.wikipedia.org/wiki/了解覚書 "wikilink")によって、HKIRCが.hkの管理を引き継いだ\[2\]。Hong Kong Domain Name Registration Corporation Limited（HKDNR。HKIRCの全額出資子会社）が登録サービスの管理を行っている\[3\]。.hkドメインは第2レベルと第3レベルのどちらにも登録ができる。

2010年6月25日、[ICANN](../Page/ICANN.md "wikilink")はHKIRCによって申請されていた[国際化国別コードトップレベルドメイン](https://ja.wikipedia.org/wiki/国際化国別コードトップレベルドメイン "wikilink")**.香港**（[繁体字](../Page/繁体字.md "wikilink")・[簡体字](https://ja.wikipedia.org/wiki/簡体字 "wikilink")とも同じ。Punycode: ）を承認した\[4\] This TLD was added to the DNS in July 2010。.香港ドメインの優先登録仮受付は2011年2月22日に始まった\[5\]。最初の .香港 ドメインは2011年3月23日に有効になった\[6\]。香港 ドメインの一般受付は2011年5月31日から行われている\[7\]。

## ドメインの種類

英語のドメイン名は[漢字](../Page/漢字.md "wikilink")を一切含んではならないが、中国語のドメイン名は少なくとも1つの漢字を含まなければならない。すべてのドメイン名は、英字、アラビア数字(0-9)、[ハイフン](../Page/ハイフン.md "wikilink")を含むことができる\[8\]。

| 英語のドメイン名 | 中国語のドメイン名    | 資格のある申込者 |
| -------- | ------------ | -------- |
| .hk      | .hk\[9\]     | .香港      |
| .com.hk  | .公司.hk\[10\] | .公司.香港   |
| .org.hk  | .組織.hk\[11\] | .組織.香港   |
| .net.hk  | .網絡.hk\[12\] | .網絡.香港   |
| .edu.hk  | .教育.hk\[13\] | .教育.香港   |
| .gov.hk  | .政府.hk\[14\] | .政府.香港   |
| .idv.hk  | .個人.hk\[15\] | .個人.香港   |
