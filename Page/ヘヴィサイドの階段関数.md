[Dirac_distribution_CDF.svg](https://ja.wikipedia.org/wiki/File:Dirac_distribution_CDF.svg "fig:Dirac_distribution_CDF.svg") は、正負の引数に対しそれぞれ 1, 0 を返す[階段関数](https://ja.wikipedia.org/wiki/階段関数 "wikilink")  である。名称は[オリヴァー・ヘヴィサイド](https://ja.wikipedia.org/wiki/オリヴァー・ヘヴィサイド "wikilink")にちなむ。ヘヴィサイド関数と呼ばれることもある。通常、*H*(*x*) や *Y*(*x*) などで表されることが多い。

[単位ステップ関数](https://ja.wikipedia.org/wiki/単位ステップ関数 "wikilink")と似ているが、こちらは  と *x* = 0 の時も1の値を持つものとして定義される。[切断冪関数](https://ja.wikipedia.org/wiki/切断冪関数 "wikilink")の0乗。

## 不連続性

階段関数は、*x* \< 0 または *x* \> 0 の範囲で連続であるが, *x* = 0 で値 *c* をとるものとして階段関数  を実数全体の集合 \(\mathbb R\) 上の関数 \(H_c: \mathbb R \to \mathbb R\) と考えるならば、*c* をどのように定めても原点 *x* = 0 で[不連続である](https://ja.wikipedia.org/wiki/不連続性の分類 "wikilink")。*c* の値は必要に応じて都合のよい値を選ぶことができるが、*c* = 0, 1/2 などがしばしば用いられ、それぞれ  である。また、  と表すことができる。関数 sgn は[符号関数](https://ja.wikipedia.org/wiki/符号関数 "wikilink")である。

## 階段関数の密度とデルタ関数

[ディラックのデルタ関数](../Page/ディラックのデルタ関数.md "wikilink") δと[区間](https://ja.wikipedia.org/wiki/区間_\(数学\) "wikilink") \((-\infty, x]\) の[定義関数](https://ja.wikipedia.org/wiki/指示関数 "wikilink") \(\chi_{(-\infin, x]}\) に対し  とおくと、これは *x* \< 0 のとき区間 \((-\infty, x]\) は 0 を含まず、*x* ≥ 0 のとき区間 \((-\infty, x]\) が 0 を含むことから  となる。つまり  と表される。この意味でヘヴィサイドの階段関数はディラックのデルタ関数を[確率密度関数](https://ja.wikipedia.org/wiki/確率密度関数 "wikilink")とするときの[累積分布関数](https://ja.wikipedia.org/wiki/累積分布関数 "wikilink")に相当する。
