## 概要

### 開発

BAC 1-11は、[ダグラス DC-3やその後継にあたる](../Page/ダグラス_DC-3.md "wikilink")[コンベア240](../Page/コンベア240.md "wikilink")などに代わる短中距離ジェット旅客機として、[1956年](../Page/1956年.md "wikilink")にハンチング社が計画していたH107が原型であり、ビッカースも同様の機体としてVC-11を研究していた。

両社ともBACに吸収されたため、H107が元になり開発が進められ[1963年](../Page/1963年.md "wikilink")に初飛行が行われた。実際には[ビッカース バイカウントや](../Page/ビッカース_バイカウント.md "wikilink")[ロッキード L-188などのターボプロップ機の後継となった](../Page/ロッキード_L-188.md "wikilink")。なお、ライバルには同じく短中距離用に開発された[ダグラス DC-9や](../Page/マクドネル・ダグラス_DC-9.md "wikilink")[ボーイング737](../Page/ボーイング737.md "wikilink")などがあったが、それらの機体よりも先行して開発が進んだ上に、機体が小型であった。

### 運航

[1965年](../Page/1965年.md "wikilink")に運行が開始された。BAC1-11はイギリスの航空会社の発注を受け生産が始まった機体であるが、[ヨーロッパ](../Page/ヨーロッパ.md "wikilink")諸国や[アメリカ](https://ja.wikipedia.org/wiki/アメリカ "wikilink")、[ブラジル](https://ja.wikipedia.org/wiki/ブラジル "wikilink")など世界各国の航空会社でも短中距離用機材として多数が運用され、イギリスのジェット旅客機としては商業的に最も成功した機体である。

ブリティッシュ・エアクラフト・コーポレーションを吸収した[ブリティッシュ・エアロスペース](../Page/ブリティッシュ・エアロスペース.md "wikilink")（BAe）による[BAe 146の生産開始に伴い](../Page/BAe_146.md "wikilink")、[1982年](../Page/1982年.md "wikilink")に生産が中止されたが、その後当時の東側諸国の一員である[ルーマニア](../Page/ルーマニア.md "wikilink")に生産設備が移管され、「ROMBAC 1-11」として[1989年](../Page/1989年.md "wikilink")までライセンス生産された。

### 現在

BAe 146やその後継のアブロRJ、[フォッカー100](https://ja.wikipedia.org/wiki/フォッカー100 "wikilink")などが代替機材となり、[1990年代](../Page/1990年代.md "wikilink")にはほとんどの機体が定期旅客運航から引退したものの、[2014年](../Page/2014年.md "wikilink")現在も少数機がプライベートジェットとして運行されている。

## 派生型

[BAC_1-11_401AK_N5018_AA_CLE_27.03.71_edited-2.jpg](https://ja.wikipedia.org/wiki/File:BAC_1-11_401AK_N5018_AA_CLE_27.03.71_edited-2.jpg "fig:BAC_1-11_401AK_N5018_AA_CLE_27.03.71_edited-2.jpg")の1-11-401\]\]

  - Model 200 : 最初の生産型　客席数最大79席。日本にもデモフライトで飛来した。
    Model 300 : エンジン改良型、航続距離延長型
    Model 400 : Model 300をアメリカン航空の要求に応じて仕様変更した北米向け仕様。
    Model 475 : ルーマニアでの生産型ROMBAC 1-11-475
    Model 500 : 胴体を延長して客席数を最大119席にした型で、最も販売機数が多い。
    Model 550 : ルーマニアでの生産型ROMBAC 1-11-550
    Model 475 : Model 400の胴体とModel 500の主翼・エンジンを組み合わせて離着陸性能を強化した型。少数機がフォーセット航空、[エア･パシフィック](https://ja.wikipedia.org/wiki/エア･パシフィック "wikilink")等で運用された。さらに日本にもYS-11の代替機として離着陸性能強化型の売り込みがあり、参考として本型がデモフライトで飛来した。
    Model 670 : Model 500のエコノミーエンジン搭載型
