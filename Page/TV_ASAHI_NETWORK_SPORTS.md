**TV ASAHI NETWORK SPORTS**は、[テレビ朝日](../Page/テレビ朝日.md "wikilink")系列のスポーツ番組ブランドである。

[2002年](../Page/2002年.md "wikilink")に「TV ASAHI NETWORK SPORTS」ブランドを発表し、スポーツ選手の影をモチーフとしたマスコット及び統一イメージソングも設けられた。イメージソングは基本的に年毎に変更されている。テレビ朝日系列のスポーツ番組においてオープニングでマスコットとともに「TV ASAHI NETWORK SPORTS」のロゴが現れていた。

[2004年](../Page/2004年.md "wikilink")を最後に「TV ASAHI NETWORK SPORTS」の看板を外したが、統一イメージソングは[2005年](https://ja.wikipedia.org/wiki/2005年 "wikilink")以降も採用されている。

## イメージソング

イメージソングは2004年までは[B'z](../Page/B'z.md "wikilink")（2004年上半期は[TMG](https://ja.wikipedia.org/wiki/TMG_\(バンド\) "wikilink")）、2005年7月以降は[SMAP](https://ja.wikipedia.org/wiki/SMAP "wikilink")が担当している。2005年上半期は設定せず。

  - 2002年「[熱き鼓動の果て](https://ja.wikipedia.org/wiki/熱き鼓動の果て "wikilink")」
  - [2003年](../Page/2003年.md "wikilink")「[IT'S SHOWTIME\!\!](https://ja.wikipedia.org/wiki/IT'S_SHOWTIME!! "wikilink")」
  - 2003年「[野性のENERGY](https://ja.wikipedia.org/wiki/野性のENERGY "wikilink")」
  - 2004年上半期「[OH JAPAN 〜OUR TIME IS NOW〜](https://ja.wikipedia.org/wiki/OH_JAPAN_〜OUR_TIME_IS_NOW〜 "wikilink")」
  - 2004年下半期「[ARIGATO](https://ja.wikipedia.org/wiki/ARIGATO_\(B'zの曲\) "wikilink")」
  - 2005年「[BANG\! BANG\! バカンス\!](https://ja.wikipedia.org/wiki/BANG!_BANG!_バカンス! "wikilink")」
  - 2005年11月「[Triangle](https://ja.wikipedia.org/wiki/Triangle_\(SMAPの曲\) "wikilink")」
  - [2006年](https://ja.wikipedia.org/wiki/2006年 "wikilink")「[buzzer beater](https://ja.wikipedia.org/wiki/Dear_WOMAN "wikilink")」
  - [2007年](../Page/2007年.md "wikilink")11月「[弾丸ファイター](https://ja.wikipedia.org/wiki/弾丸ファイター "wikilink")」
  - [2009年](../Page/2009年.md "wikilink")7月「[スーパースター★](https://ja.wikipedia.org/wiki/そっと_きゅっと/スーパースター★ "wikilink")」

### 例外

  - サッカー中継では[サラ・ブライトマン](https://ja.wikipedia.org/wiki/サラ・ブライトマン "wikilink")「[A Question Of Honour](https://ja.wikipedia.org/wiki/クエスチョン・オブ・オナー "wikilink")」が使われている。ただしハーフタイムにおいて統一イメージソングも使われている。
  - ISUグランプリシリーズでは2006年以降も「Triangle」が使われている。
  - [世界水泳2007及び](https://ja.wikipedia.org/wiki/2007年世界水泳選手権 "wikilink")[世界競泳2007では](../Page/インターナショナル・スイム・ミート2007.md "wikilink")「[mermaid](https://ja.wikipedia.org/wiki/Super.modern.artistic.performance "wikilink")」が使用された。
  - [全国高等学校野球選手権大会](https://ja.wikipedia.org/wiki/全国高等学校野球選手権大会 "wikilink")など系列局制作では、イメージソングは使用されない場合が多い。
  - [世界水泳2011では](https://ja.wikipedia.org/wiki/2011年世界水泳選手権 "wikilink")「[ultra soul](https://ja.wikipedia.org/wiki/ultra_soul "wikilink")」をリアレンジした「[ultra soul 2011](https://ja.wikipedia.org/wiki/C'mon "wikilink")」が使用された。
