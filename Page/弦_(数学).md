[Chord_in_mathematics.svg](https://ja.wikipedia.org/wiki/File:Chord_in_mathematics.svg "fig:Chord_in_mathematics.svg") [初等幾何学](https://ja.wikipedia.org/wiki/初等幾何学 "wikilink")における[円の](../Page/円_\(数学\).md "wikilink")**弦**（げん、）は、その円上に両端点を持つ[線分](../Page/線分.md "wikilink")を言う。弦を無限に延長して得られる[直線](../Page/直線.md "wikilink")を、[割線](../Page/割線.md "wikilink")と呼ぶ。より一般に、任意の[曲線](../Page/曲線.md "wikilink")（例えば[楕円](../Page/楕円.md "wikilink")）において、その曲線上の二点を結ぶ線分を、その曲線上の弦と総称する。円の中心を通る弦はその円の[直径](https://ja.wikipedia.org/wiki/直径 "wikilink")である。任意の直径は弦であるが、任意の弦が直径となるわけでない。

[CIRCLE_LINES.svg](https://ja.wikipedia.org/wiki/File:CIRCLE_LINES.svg "fig:CIRCLE_LINES.svg")

## 円の弦

円の弦に関する性質には、例えば以下のようなものがある:

1.  二つの弦が、円の中心から等距離にあるための必要十分条件は、それら弦の長さが等しいことである。
2.  長さの等しい弦を、円の中心から見込む角（中心角）は等しい。
3.  円の中心を通る弦は直径と呼ばれ、その円の最長の弦である。
4.  弦  および  を延長して得られる[割線](../Page/割線.md "wikilink")が点  で交わるならば、それらの長さは  を満足する（[方冪の定理](https://ja.wikipedia.org/wiki/方冪の定理 "wikilink")）。

## 楕円の弦

[楕円](../Page/楕円.md "wikilink")における互いに平行な弦の族が与えられたとき、それら弦の中点はすべて[同一直線上にある](https://ja.wikipedia.org/wiki/同一直線上にある "wikilink")\[1\]。

## 弦をもとにした三角法

[三角法](../Page/三角法.md "wikilink")の初期の段階では弦が手広く用いられていた。知られた最古の三角函数表は[ヒッパルコス](../Page/ヒッパルコス.md "wikilink")の編纂したで、それには7.5[°刻みで弦函数の値が書き並べられていた](../Page/度_\(角度\).md "wikilink")。AD 2世紀に、アレクサンドリアの[プトレマイオスは](../Page/クラウディオス・プトレマイオス.md "wikilink")、天文学に関する著書『[アルマゲスト](../Page/アルマゲスト.md "wikilink")』において、より詳細な弦の数表を編纂している（0.5°から180°まで0.5°刻みで値が与えられ、これは円の直径を120として小数点以下60進ふた桁まで正確であった）\[2\]。

[TrigonometricChord.svg](https://ja.wikipedia.org/wiki/File:TrigonometricChord.svg "fig:TrigonometricChord.svg") 弦函数  は幾何学的には（図のように）[中心角](https://ja.wikipedia.org/wiki/中心角 "wikilink")  の見込む弦の長さが （ は半径）となるように定義される。すなわち、弦函数の値  は、中心角  によって隔てられた[単位円](../Page/単位円.md "wikilink")上の二点間を結ぶ弦の[長さ](../Page/長さ.md "wikilink")である。ここでは角度  は正の向きに測るものとし、弧度法で区間  の範囲に入るものと考えている。この元函数  をより現代的な[正弦函数](https://ja.wikipedia.org/wiki/正弦函数 "wikilink")  と関連付けることができる。それには、一点  ともう一つの点  を結ぶ弦の長さを[三平方の定理](https://ja.wikipedia.org/wiki/三平方の定理 "wikilink")を用いて計算すればよい。すると \(\operatorname{crd}\theta = \sqrt{(1-\cos \theta)^2+\sin^2 \theta} = \sqrt{2-2\cos \theta} =2 \sin\!\Bigl(\frac{\theta }{2}\Bigr)\) を得る\[3\]。最後の等号は[半角公式による](https://ja.wikipedia.org/wiki/三角関数の公式の一覧#倍角・三倍角・半角の公式 "wikilink")。

現代的な三角法が正弦函数に基づいて構築されているのと同様に、古来の三角法はこの弦函数をもとに構築されていた。ヒッパルコスは（いまではもうすべて失われたけれども）12巻にも及ぶ弦についての文献を書き上げたというから、三角法についてはかなりのことが知られていたと考えられる。現代的な三角函数に関するよく知られた恒等式の弦函数版がある:

| 恒等式                                                   | 正弦版                                                          | 弦版                                                                                       |
| ----------------------------------------------------- | ------------------------------------------------------------ | ---------------------------------------------------------------------------------------- |
| 三平方の定理                                                | \(\sin^2 \theta + \cos^2 \theta = 1\)                        | \(\operatorname{crd}^2 \theta + \operatorname{crd}^2 (\pi- \theta) = 4\)                 |
| 半角公式                                                  | \(\sin\frac{\theta}{2} = \pm\sqrt{\frac{1-\cos \theta}{2}}\) | \(\operatorname{crd}\ \frac{\theta}{2} = \pm \sqrt{2-\operatorname{crd}(\pi - \theta)}\) |
| [辺心距離](https://ja.wikipedia.org/wiki/辺心距離 "wikilink") | \(c=2 \sqrt{r^2- a^2}\)                                      | \(c=\sqrt{D ^2-4 a^2}\)                                                                  |
| 中心角                                                   | \(c=2 r \sin\!\Big(\frac{\theta }{2}\Bigr)\)                 | \(c=\frac{D}{2}\operatorname{crd}\theta\)                                                |
| ただし、半径 （直径 ）の円の中心角  が見込む弦の長さを  とする。                   |                                                              |                                                                                          |

弦函数  の逆函数  もまた存在して、逆正弦函数とは \(\operatorname{acrd}(y) = 2\arcsin\!\Big(\frac{y}{2}\Bigr)\) の関係にある\[4\]。
