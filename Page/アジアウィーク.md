**アジアウィーク**とは、[1994年](../Page/1994年.md "wikilink")より\[1\][中央競馬](../Page/中央競馬.md "wikilink")で開催されている[交換競走](../Page/交換競走.md "wikilink")\[2\]\[3\]のシリーズである。[アジア](../Page/アジア.md "wikilink")6か国（[マカオ](../Page/マカオ.md "wikilink")\[4\]、[マレーシア](https://ja.wikipedia.org/wiki/マレーシア "wikilink")\[5\]、[シンガポール](https://ja.wikipedia.org/wiki/シンガポール "wikilink")\[6\]、[タイ王国](../Page/タイ王国.md "wikilink")\[7\]、[インド](../Page/インド.md "wikilink")\[8\]、[フィリピン](https://ja.wikipedia.org/wiki/フィリピン "wikilink")\[9\]）それぞれの競馬団体より寄贈された優勝カップをかけた競走が2日間にかけて実施される。

2016年現在は夏季に[中京競馬場](../Page/中京競馬場.md "wikilink")で開催\[10\]。[2011年](../Page/2011年.md "wikilink")には本シリーズに合わせて[セリ市の開催日程を調整](../Page/セリ市_\(競馬\).md "wikilink")\[11\]したり、地元の馬主協会の協賛によるイベントを実施\[12\]するなど、[馬主](../Page/馬主.md "wikilink")・生産者団体との連携も行われている\[13\]。

## 施行される競走

2012年現在、以下の6競走が[特別競走として施行されている](https://ja.wikipedia.org/wiki/ステークス方式 "wikilink")。

| 競走名                    | 寄贈賞                                                                          |
| ---------------------- | ---------------------------------------------------------------------------- |
| マカオジョッキークラブトロフィー\[14\] | [マカオジョッキークラブ](../Page/マカオジョッキークラブ.md "wikilink")賞\[15\]                      |
| シンガポールターフクラブ賞\[16\]    | [シンガポールターフクラブ](https://ja.wikipedia.org/wiki/シンガポールターフクラブ "wikilink")賞\[17\] |
| マレーシアカップ\[18\]         | セランゴールターフクラブ・ペナンターフクラブ・ペラターフクラブ賞\[19\]                                       |
| タイランドカップ\[20\]         | ロイヤルバンコックスポーツクラブ・ロイヤルターフクラブオブタイランド賞\[21\]                                    |
| インディアトロフィー\[22\]       | ターフオーソリティーオブインディア賞\[23\]                                                     |
| フィリピントロフィー\[24\]       | マニラジョッキークラブ・フィリピンレーシングクラブ賞\[25\]                                             |

  - タイランドカップは[1997年](https://ja.wikipedia.org/wiki/1997年 "wikilink")から施行。
  - シンガポールターフクラブ賞とマレーシアカップは[2008年](https://ja.wikipedia.org/wiki/2008年 "wikilink")からの施行（[2007年](../Page/2007年.md "wikilink")まではシンガポール・マレーシア両国の競馬の統括機関であるマラヤンレーシングアソシエーションが寄贈賞を提供し、「マラヤンレーシングアソシエーショントロフィー」という競走名で施行）。
  - 2007年までは[香港ジョッキークラブ](../Page/香港ジョッキークラブ.md "wikilink")（旧・ロイヤルホンコンジョッキークラブ）が寄贈賞を提供した「ホンコンジョッキークラブトロフィー」（香港の中国返還以前の[1996年](../Page/1996年.md "wikilink")までは「ロイヤルホンコンジョッキークラブトロフィー」）もあったが、2008年よりアジアウィークから離脱して、6月に[東京競馬場](../Page/東京競馬場.md "wikilink")で施行されるようになった。

## 施行競馬場の変遷

[1994年](../Page/1994年.md "wikilink")に[中京競馬場](../Page/中京競馬場.md "wikilink")での開催が6 - 7月に行われていたときに始まった\[26\]本シリーズは、[2000年](../Page/2000年.md "wikilink")以降[小倉競馬場](../Page/小倉競馬場.md "wikilink")でも施行されるようになり、2007年まで西暦奇数年は中京（6月）、偶数年は小倉（8月）で実施されていた。2008年は[第32回アジア競馬会議](https://ja.wikipedia.org/wiki/第32回アジア競馬会議 "wikilink")が東京で行われることを記念して11月に東京競馬場で開催された\[27\]。2009年は小倉競馬場に開催が戻ったが、[2010年](https://ja.wikipedia.org/wiki/2010年 "wikilink")から[2012年](../Page/2012年.md "wikilink")は札幌競馬場で開催され\[28\]\[29\]、[2013年](../Page/2013年.md "wikilink")は札幌競馬場改修工事の関係で小倉競馬場で開催。[2014年](../Page/2014年.md "wikilink")から2019年は中京競馬場での開催となる\[30\]\[31\]。2020年は[京都競馬場](../Page/京都競馬場.md "wikilink")のスタンド改築工事による変則日程により[阪神競馬場](../Page/阪神競馬場.md "wikilink")で開催される。

## その他

[1996年](../Page/1996年.md "wikilink")から[2000年](../Page/2000年.md "wikilink")までは、シリーズ参加各国の[競馬騎手を招待した交流戦](../Page/騎手招待競走.md "wikilink")「**アジアジョッキーズ**」が当該週に2競走実施された。2000年はマカオジョッキークラブトロフィーとホンコンジョッキークラブトロフィーの2競走で争われた\[32\]。
