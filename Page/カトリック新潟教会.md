**カトリック新潟教会**（かとりっくにいがたきょうかい）は、[新潟県](../Page/新潟県.md "wikilink")[新潟市](https://ja.wikipedia.org/wiki/新潟市 "wikilink")[中央区にある](https://ja.wikipedia.org/wiki/中央区_\(新潟市\) "wikilink")[キリスト教](../Page/キリスト教.md "wikilink")（[カトリック](../Page/カトリック教会.md "wikilink")）の[教会](../Page/教会_\(キリスト教\).md "wikilink")。[カトリック新潟教区](https://ja.wikipedia.org/wiki/カトリック新潟教区 "wikilink")の[司教座聖堂](../Page/司教座聖堂.md "wikilink")（[カテドラル](https://ja.wikipedia.org/wiki/カテドラル "wikilink")）であり、[教会堂](../Page/教会堂.md "wikilink")名（[聖堂](../Page/聖堂.md "wikilink")名）は「王である[キリスト](../Page/キリスト.md "wikilink")」である。

## 建物

現在の聖堂は[チューリッヒ](../Page/チューリッヒ.md "wikilink")出身の建築家[マックス・ヒンデル](https://ja.wikipedia.org/wiki/マックス・ヒンデル "wikilink")氏（1887年-1963年）によって[設計](../Page/設計.md "wikilink")され、[ロマネスク](../Page/ロマネスク.md "wikilink")様式を取り入れた[和洋折衷](../Page/和洋折衷.md "wikilink")の[木造建築として](../Page/木構造_\(建築\).md "wikilink")[1927年](../Page/1927年.md "wikilink")に献堂された。

## 歴史

| 年号          |                                                                                                     |
| ----------- | --------------------------------------------------------------------------------------------------- |
| 明治4（1871）年  | [パリ外国宣教会](../Page/パリ外国宣教会.md "wikilink")エヴラール神父により新潟での宣教始まる                                         |
| 明治8（1875）年  | ドルワール・ド・レゼー神父、現在地を購入                                                                                |
| 大正元（1912）年  | 新潟[使徒座](https://ja.wikipedia.org/wiki/使徒座 "wikilink")知牧区として創立、[神言修道会に委託](../Page/神言会.md "wikilink") |
| 昭和2（1927）年  | 現在の[聖堂](../Page/聖堂.md "wikilink")が献堂                                                                |
| 昭和37（1962）年 | 新潟知牧区が[司教区に昇格](../Page/教区.md "wikilink")、新潟教会に[司教座を設置](../Page/司教座聖堂.md "wikilink")                 |
| 昭和56（1981）年 | 小聖堂建設                                                                                               |
| 平成8（1996）年  | [聖堂](../Page/聖堂.md "wikilink")全面改修                                                                  |
| 平成24（2012）年 | [カトリック新潟教区](https://ja.wikipedia.org/wiki/カトリック新潟教区 "wikilink")創立100周年を祝う                           |
