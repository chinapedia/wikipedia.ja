**ハードディスク・レコーダー**（*Harddisk recorder* ）は、[デジタルオーディオ](../Page/デジタルオーディオ.md "wikilink")または[デジタルビデオ](../Page/デジタルビデオ.md "wikilink")を記録する為に大容量[ハードディスクを利用する記録システムのひとつ](https://ja.wikipedia.org/wiki/ハードディスクドライブ "wikilink")。

## 音声録音装置

ハードディスクを使用したデジタル[マルチトラック・レコーダー](../Page/マルチトラック・レコーダー.md "wikilink")としてのハードディスク・レコーダー。**HDR**と略される。

### 概要

ハードディスク・レコーディング・システムは、従来の[オープンリール](../Page/オープンリール.md "wikilink")式テープまたは[カセット](../Page/コンパクトカセット.md "wikilink") [マルチトラックシステムの代わりに相当し](https://ja.wikipedia.org/wiki/オーバーダビング "wikilink")、テープ・レコーダーでは得られない編集機能を提供する。単体機と、コンピューター上で動作するソフトからの運用方法があるが、どちらでも通常はデジタル・ミキシングと音声信号の処理機能を提供する。

[1980年代](../Page/1980年代.md "wikilink")より以前、大部分のレコーディング・スタジオはアナログ・マルチトラック・レコーダー、一般的にはオープン・リール式テープを利用していた。1980年代から1990年代にニューイングランドデジタル社はハードディスク・レコーディングシステムを提供し、映画の[ポストプロダクション](../Page/ポストプロダクション.md "wikilink")のような限定された用途において地位を確立したものの、高いコストと限られたディスク容量のため規模の大きなレコーディングスタジオの利用に限られていた。

その後[CD市場の成長でデジタル](../Page/コンパクトディスク.md "wikilink")・レコーディングは製造メーカーによる開発の主要分野となり、いくつかの製品が1980年代後期から1990年代初期にリリースされた。オープン・リールにおいて、またより扱いやすい[ビデオカセットにおいても](../Page/ビデオテープレコーダ.md "wikilink")、多くは継続してメタル・テープを使用した。しかし1990年代中頃までにはハードディスク価格の安定と供給の拡大及び単価の低下、容量の増加と小型化が進み、ハードディスク・レコーディング・システムのコストはさらに下がり、個人または小規模プロジェクト・スタジオなどの為に導入可能なシステムになってきた。他のデジタル・レコーダーの種類としては[ADAT](https://ja.wikipedia.org/wiki/ADAT "wikilink")、[DTRS](https://ja.wikipedia.org/wiki/DTRS "wikilink")等が使用されていたが、スタジオ・レコーディングにおいてハードディスク・レコーディング・システムはシステムの完成度と共に急加速的に増え、一気にスタンダード化した。

ハードディスクへのオーディオ・レコーディングの大きな長所は、[ノンリニア編集](../Page/ノンリニア編集.md "wikilink")を可能とすることであり、音声データは[ランダムアクセス](../Page/ランダムアクセス.md "wikilink")出来て、非破壊編集が可能な点である。つまり、元の素材はどんな方法にせよあくまでも編集素材であって元の素材自体は意図的に操作しない限りは変化しない。

ノンリニア編集がすべてのハードディスク・レコーディング・システムに備わっている訳ではなく、異なる手段を提供するメーカーもある。またハードディスク・レコーダーにはテープベースのシステムと比較してハードディスク・レコーダーの記録メディアであるハードドライブ耐久性の低下や容量の限界によるディスクの交換に比較的高いコスト必要とするなど、多少の欠点がある。

ハードディスク・レコーダーは、しばしばデジタル・[ミキシング・コンソール](../Page/ミキシング・コンソール.md "wikilink")と一体化され、[デジタル・オーディオ・ワークステーション](../Page/デジタル・オーディオ・ワークステーション.md "wikilink")のひとつの機能として備わる。このタイプは複雑な処理を自動化することも出来るため、[レコーディング・エンジニア](../Page/レコーディング・エンジニア.md "wikilink")はミキシング作業中に行っていたテープ編集作業などからミキシング・バランスを取る作業に向け時間を割く事が可能になった。

[パーソナルコンピューター](https://ja.wikipedia.org/wiki/パーソナルコンピューター "wikilink")を基軸とする[Pro Tools](../Page/Pro_Tools.md "wikilink") ソフトウェアと潤沢な周辺機材によってハードディスク・レコーダーとして使用することが標準化されてきたので、より柔軟な[インターフェースを](../Page/インタフェース_\(情報技術\).md "wikilink")[スタジオ](https://ja.wikipedia.org/wiki/レコーディングスタジオ "wikilink")・エンジニアにもたらした。

考慮すべき点としては。どのようなハードディスク・レコーディング・システムも、物理ディスクサイズ、転送レート、[プロセッサ](../Page/CPU.md "wikilink")・スピードなどが大きな制約となる。過去一部のシステムはディスクサイズと転送レートの問題を解決するため[非可逆デジタル音声圧縮を使用していたが](../Page/非可逆圧縮.md "wikilink")、ハードディスク容量の急速な増加と低コスト化が進んだ結果そのような製品はなくなった。

## 映像録画装置

[ハードディスクを](https://ja.wikipedia.org/wiki/ハードディスクドライブ "wikilink")[記録媒体とする](../Page/電子媒体.md "wikilink")[録画](../Page/録画.md "wikilink")装置としての**ハードディスクレコーダー**。

### 歴史

1980年代～1990年代には[VHS](../Page/VHS.md "wikilink")カセットによる[ビデオデッキが主流であった](../Page/ビデオテープレコーダ.md "wikilink")。

2000年6月、記録媒体をハードディスクに変え、大容量化が容易などの謳い文句で日本初のハードディスクレコーダー「SVR-715」（Clip-On）がソニーから発売された\[1\]。

その後 劣化が生じない[DVDレコーダー](../Page/DVDレコーダー.md "wikilink")が発売され、シェアを拡大していった。

2001年4月に[東芝](../Page/東芝.md "wikilink")が[RD-Style](../Page/RD-Style.md "wikilink")シリーズ（現:[REGZA](https://ja.wikipedia.org/wiki/REGZA "wikilink")←[VARDIA](../Page/VARDIA.md "wikilink")）でHDDレコーダーとDVDレコーダーを組み合わせた[ハイブリッド](../Page/ハイブリッド.md "wikilink")型の「HDD\&DVDレコーダー」を発売。2002年中は双方のドライブが高価であったためハイブリッド型は傍流であったが、2003年のパナソニック [ディーガ](https://ja.wikipedia.org/wiki/ディーガ "wikilink")でハイブリッド型中心の機種展開を図り、DVDレコーダーにHDDレコーダーが一体化したもの（先述のハイブリッドレコーダー型）が主流となる。

2004年11月、6チャンネルを同時に1週間分録画できる「[VAIO](../Page/VAIO.md "wikilink") typeX」を皮切りに、全録型レコーダーという「とりあえず全部録画しておいて後で検索する」新しいタイプの録画機が出始めた。特に[PTP](https://ja.wikipedia.org/wiki/PTP "wikilink")が販売している[SPIDER](../Page/スパイダー.md "wikilink") PROは、主にマスコミにおける露出状況をチェックする企業や政治団体が導入、活用されている。

一方DVDは、2006年頃に[AVCHD](../Page/AVCHD.md "wikilink")による録画データの圧縮機能（いわゆるフルハイビジョン記録）が実用化されるようになるまで、DVDレコーダーでは基本的に[DVD-VR](../Page/DVD-VR.md "wikilink")として[標準画質](https://ja.wikipedia.org/wiki/標準画質 "wikilink")へ[ダウンコンバート](https://ja.wikipedia.org/wiki/ダウンコンバート "wikilink")をしたものしかDVD-Rなどへ保存（ダビング・ムーブ）できなかったため、[デジタルチューナー](https://ja.wikipedia.org/wiki/デジタルチューナー "wikilink")と連携し[ハイビジョン](../Page/ハイビジョン.md "wikilink")記録に対応したHDDレコーダーは一定の需要があった。しかし、このケースでもDVD-VRや[BDMVのAVCHDでの記録で対処出来るようになった事からHDDレコーダーデッキ単体として売り出されているのは](../Page/Blu-ray_Disc.md "wikilink")[iVDR](https://ja.wikipedia.org/wiki/iVDR "wikilink")関連のみとなった。ただし、特定の[薄型テレビ](../Page/薄型テレビ.md "wikilink")や[デジタルチューナー](https://ja.wikipedia.org/wiki/デジタルチューナー "wikilink")・BDレコーダー・DVDレコーダーあるいは[PlayStation 3の](https://ja.wikipedia.org/wiki/PlayStation_3 "wikilink")[torne](https://ja.wikipedia.org/wiki/torne "wikilink")など、本体の[USB端子に汎用型の外付けHDDを接続することで記録できる機能を搭載した機種が](../Page/ユニバーサル・シリアル・バス.md "wikilink")2009年頃から発売されるようになり、別の活路を見いだしている。

### 録画モード

ハードディスクに録画する場合には大きく、[ハイビジョン](../Page/ハイビジョン.md "wikilink")画質のままで録画できるDR（放送当時の画質と全く同じもの）、AF、AN、AEと、標準画質での録画となるXP、SP、LP、EP、並びにDVDやBDの残量を踏まえて自動的に画質を調整して録画するFRとがある。画質が良いほど収録時間が短くなり、画質が落ちるほど長時間の収録となる。

記録メディアに録画・ダビングする場合は、DRモードの画質そのものはBDのみに記録できる\[2\]が、それ以外はDVD/BDの双方でダビングすることができる。

録画時間は、機種や性能にもよるが、1テラバイトのDRモードで録画した場合は地上デジタルで127時間、BS・CS\[3\]を録画すると90時間収録できる。またディスクにダビングする場合、ブルーレイ（一層）にDRモードのままで録画すると、地デジで3時間、BS・CSは2時間10分は記録できるという。また画質が最も落ちるAE/HE（ハイビジョン画質）・EP（標準画質）の場合、ディスクへのダビング収録時間の設定変更が可能な機種もある\[4\]。

なおHDDから記録メディアに高速ダビングで録画する場合は、記録するディスクの種類によって対応が異なる\[5\]。

| HD（ハイビジョン画質）        | DR          | ○     |
| AF/HG\[6\]/XR\[7\]  | ○           |       |
| AN/HX\[8\]/XSR\[9\] |             |       |
| AE/HE\[10\]         |             |       |
| SD（標準画質）            | XP/SR\[11\] | △     |
| SP/LSR\[12\]        |             |       |
| LP/LR\[13\]         |             |       |
| EP/ER\[14\]         |             |       |
| FR\[15\]            |             |       |

### 長所

  - 読み込みや検索時間が光メディアと比べ、多少早い。
  - データの編集や削除が容易である。
  - 光メディアやフラッシュメモリと比べ、大容量である。
  - [文字多重放送](https://ja.wikipedia.org/wiki/文字多重放送 "wikilink")を録画する場合、[字幕をそのまま録画できる](https://ja.wikipedia.org/wiki/字幕放送 "wikilink")\[16\]

### 短所

  - HDD自体は[リムーバブルメディア](../Page/リムーバブルメディア.md "wikilink")ではないため、増設などができない\[17\]\[18\]
  - 録画データの保存容量がHDD容量に逼迫した場合は、他の機器へムーブするか、上書き録画の形での消去の対応を迫られる。
  - バックアップ（複製）は[コピーワンス](https://ja.wikipedia.org/wiki/コピーワンス "wikilink")信号の入っているデジタル放送の録画番組では不可であり（アナログ放送の場合は[RCA端子](../Page/RCA端子.md "wikilink")接続による[コンポジット伝送で可能な機種もある](../Page/コンポジット映像信号.md "wikilink")）、ムーブバック機能でレコーダーへの書き戻し([ファイナライズ](../Page/ファイナライズ.md "wikilink")していないBDのみ、DVDは不可)、[iLink](https://ja.wikipedia.org/wiki/iLink "wikilink")ケーブル・LANケーブル（DTCP-IP対応のみ）で接続した他のレコーダー（TS接続に対応したRecPotとDIGA間など）への[ムーブ](../Page/ムーブ.md "wikilink")など限られた組み合わせによる方法しかない。HDDが故障するとかなりの高確率で内蔵データは全て消失する。基本的にはHDD内のデーター（ファイル）を直にコピーすることはできず、それが行えたとしても[コピープロテクト](https://ja.wikipedia.org/wiki/コピープロテクト "wikilink")によりデータが壊れる可能性が高い。
  - [ダビング10](../Page/ダビング10.md "wikilink")に則ったデータの移動や増設（[USBによる外付け](../Page/ユニバーサル・シリアル・バス.md "wikilink")）に対応したレコーダーも登場しているが、一部のメーカーや機種に限られる。
