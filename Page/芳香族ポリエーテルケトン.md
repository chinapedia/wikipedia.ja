**芳香族ポリエーテルケトン**（ほうこうぞくポリエーテルケトン）は、[ベンゼン環](https://ja.wikipedia.org/wiki/ベンゼン環 "wikilink")が[エーテルと](../Page/エーテル_\(化学\).md "wikilink")[ケトン](../Page/ケトン.md "wikilink")により結合した直鎖状[ポリマー構造を持つ](../Page/重合体.md "wikilink")、結晶性の[熱可塑性樹脂](../Page/熱可塑性樹脂.md "wikilink")に属するポリマーの総称。工業材料としてはポリエーテルエーテルケトン (**PEEK**) が有名である。

## 種類

[thumb](https://ja.wikipedia.org/wiki/画像:Polyetherketone01.png "wikilink") [thumb](https://ja.wikipedia.org/wiki/画像:Polyetheretherketone01.png "wikilink") エーテル結合とケトン結合を交互に配置した基本的な直鎖状構造を持つ**ポリエーテルケトン** (polyetherketone, **PEK**) と、エーテル・エーテル・ケトンの順に結合を配置した**ポリエーテルエーテルケトン** (polyetheretherketone, **PEEK**) が工業的に製造され、特に後者は用途を拡大している。これらの他に、結合の配置を変更した**ポリエーテルケトンケトン** (polyetherketoneketone, **PEKK**) や**ポリエーテルエーテルケトンケトン** (polyetheretherketoneketone, **PEEKK**) も芳香族ポリエーテルケトンの一種である。

また、現在は研究段階にある[エステル](../Page/エステル.md "wikilink")結合を加えたポリエーテルケトンエステル\[1\]もこの群に加わる。

## 製法

5種類の製法に大別される。ただし工業化されているものは求核置換反応法と求電子置換反応法の2種類のみであり、残る3種類は[特許](../Page/特許.md "wikilink")開示がなされている。

### 求核置換反応法

PEKは、[フッ素](../Page/フッ素.md "wikilink")と[水酸基](https://ja.wikipedia.org/wiki/水酸基 "wikilink")を置換体として各端に結合させた[ベンゾフェノン](../Page/ベンゾフェノン.md "wikilink")を[求核置換反応](../Page/求核置換反応.md "wikilink")で結合させて製造する。

  -
    n HO−C<sub>6</sub>H<sub>4</sub>−C(=O)−C<sub>6</sub>H<sub>4</sub>−F → \[−O−C<sub>6</sub>H<sub>4</sub>−C(=O)−C<sub>6</sub>H<sub>4</sub>−\]<sub>n</sub> + n HF

PEEKは、[ヒドロキノン](https://ja.wikipedia.org/wiki/ヒドロキノン "wikilink")と、フッ素を置換体として両端に結合させたベンゾフェノンを求核置換反応で結合させて製造する。[触媒](../Page/触媒.md "wikilink")には[炭酸カリウム](../Page/炭酸カリウム.md "wikilink")などを使用する。

  -
    n HO−C<sub>6</sub>H<sub>4</sub>−OH ＋ n F−C<sub>6</sub>H<sub>4</sub>−C(=O)−C<sub>6</sub>H<sub>4</sub>−F → \[−O−C<sub>6</sub>H<sub>4</sub>−O−C<sub>6</sub>H<sub>4</sub>−C(=O)−C<sub>6</sub>H<sub>4</sub>−\]<sub>n</sub> + 2n HF

フッ素は置換体としては高価なため、[塩素](../Page/塩素.md "wikilink")などの利用も研究されている。

### 求電子置換反応法

PEKは、片方にケトン基を介して[求電子剤](../Page/求電子剤.md "wikilink")として塩素を結合させた、すなわち[アシル基](../Page/アシル基.md "wikilink") −C(=O)Cl としたベンゾフェノンを[フリーデル・クラフツ反応](../Page/フリーデル・クラフツ反応.md "wikilink")（[求電子置換反応の一種](https://ja.wikipedia.org/wiki/親電子置換反応 "wikilink")）で結合させる。触媒には[塩化アルミニウム](../Page/塩化アルミニウム.md "wikilink")などを使用する。

  -
    n C<sub>6</sub>H<sub>5</sub>−O−C<sub>6</sub>H<sub>4</sub>−C(=O)−Cl → \[−O−C<sub>6</sub>H<sub>4</sub>−C(=O)−C<sub>6</sub>H<sub>4</sub>−\]<sub>n</sub> + n HCl

PEKKは、[ベンゾフェノン](../Page/ベンゾフェノン.md "wikilink")と、両端に求電子剤として塩素を結合させたケトン基を持つベンゼン環を、塩化アルミニウムなどを触媒として、フリーデル・クラフツ反応で結合させて製造する。

  -
    n Cl−C(=O)−C<sub>6</sub>H<sub>4</sub>−C(=O)−Cl ＋ n C<sub>6</sub>H<sub>5</sub>−C(=O)−C<sub>6</sub>H<sub>5</sub> → \[−O−C<sub>6</sub>H<sub>4</sub>−C(=O)−C<sub>6</sub>H<sub>4</sub>−C(=O)−C<sub>6</sub>H<sub>4</sub>−\]<sub>n</sub> + 2n HCl

この製法では[分岐構造や異種結合が起こりやすく](https://ja.wikipedia.org/wiki/分枝_\(化学\) "wikilink")、反応のコントロールが難しい。

### ルイス酸/強酸触媒下、求核性芳香族化合物とチオカルボン酸誘導体との反応

PEEKの製造法。工業化はされていない。

  -
    n C<sub>6</sub>H<sub>5</sub>−O−C<sub>6</sub>H<sub>4</sub>−O−C<sub>6</sub>H<sub>5</sub> ＋ n C<sub>2</sub>H<sub>5</sub>−S−C(=O)−Cl → \[−O−C<sub>6</sub>H<sub>4</sub>−O−C<sub>6</sub>H<sub>4</sub>−C(=O)−C<sub>6</sub>H<sub>4</sub>−\]<sub>n</sub> + n C<sub>2</sub>H<sub>5</sub>SH + n HCl

[タイコインターナショナル](https://ja.wikipedia.org/wiki/タイコインターナショナル "wikilink")が特許を所有している。出願はレイケム (Raychem Co.) だが、1999年の同社買収によって所有権が移転した。

### フルオロアルカンスルホン酸、あるいは五酸化二リン/メタンスルホン酸存在下での反応

PEEKおよびPEKKの製造法。工業化はされていない。

  -
    n C<sub>6</sub>H<sub>5</sub>−O−C<sub>6</sub>H<sub>4</sub>−O−C<sub>6</sub>H<sub>4</sub>−C(=O)−OH → \[−C<sub>6</sub>H<sub>4</sub>−O−C<sub>6</sub>H<sub>4</sub>−O−C<sub>6</sub>H<sub>4</sub>−C(=O)−\]<sub>n</sub> + n HCl
    n C<sub>6</sub>H<sub>5</sub>−O−C<sub>6</sub>H<sub>5</sub> ＋ HO−C(=O)−C<sub>6</sub>H<sub>4</sub>−C(=O)−OH → \[−C<sub>6</sub>H<sub>4</sub>−O−C<sub>6</sub>H<sub>4</sub>−C(=O)−C<sub>6</sub>H<sub>4</sub>−C(=O)−\]<sub>n</sub> + 2n H<sub>2</sub>O

[インペリアル・ケミカル・インダストリーズ](../Page/インペリアル・ケミカル・インダストリーズ.md "wikilink")(ICI)が特許を所有。

### ニッケル触媒を利用した炭素-炭素カップリング反応

工業化はされていない。[ユニオンカーバイド](../Page/ユニオンカーバイド.md "wikilink")社が特許を所有。

## 特徴

  - 熱可塑性樹脂としては非常に高い[耐熱性](../Page/耐熱性.md "wikilink")を有する。非強化状態でPEKの[荷重たわみ温度](https://ja.wikipedia.org/wiki/荷重たわみ温度 "wikilink")は約186℃、PEEKは約140℃。[ガラス繊維](../Page/ガラス繊維.md "wikilink")などフィラーによる強化グレードでは300℃を超える。
  - 耐疲労性に優れる。PEEKは耐磨耗性や寸法安定性も良好。
  - 耐薬品性に優れる。
  - 改質しない状態で[UL94V](https://ja.wikipedia.org/wiki/アメリカ保険業者安全試験所 "wikilink")-0の[難燃性を有する](../Page/耐燃性.md "wikilink")。
  - 絶縁性や耐放射線性に優れる。
  - PEEKは加工性に優れ、通常の射出成型機での加工や[フィルム](../Page/フィルム.md "wikilink")化・[不織布](../Page/不織布.md "wikilink")化も可能。
  - 非常に高価である。

## 改質

  - フィラー強化
    ガラス繊維や[炭素繊維](../Page/炭素繊維.md "wikilink")などを充填し、機械的強度とともに耐熱性を高める。高耐熱用途ではガラス繊維30%強化グレードが多用されている。
  - PEEK-HT
    ICI子会社のVictrex plcが開発した耐熱性向上グレード。通常のPEEK比[融点](../Page/融点.md "wikilink")が約34℃、[ガラス転移点](../Page/ガラス転移点.md "wikilink")が約14℃それぞれ上昇する。高温環境下での機械的特性保持力に優れ、また耐磨耗性はPEEKの3倍程度にまで向上している。
  - 耐衝撃性改良
    Victrex plcが開発したグレード。[氷点](../Page/氷点.md "wikilink")下から100℃を越える使用環境にて通常のPEEK比2倍以上の耐衝撃性を有する。[ポリマーアロイ](https://ja.wikipedia.org/wiki/ポリマーアロイ "wikilink")と推測されるが、詳細は明らかにされていない。

## 用途

### 歴史

芳香族ポリエーテルケトンの先駆けとなったPEEKは1978年にICIが開発し、2年後に工業化した。[電線](../Page/電線.md "wikilink")の絶縁被膜や[自動車](../Page/自動車.md "wikilink")用オイルシールリングでの利用から、その高い耐熱性や機械的特性の良さから用途を拡大して現在に至る。

### 使用例

電線被膜や電気・電子関連部品。エンジンなど高温部での使用を含む自動車関連部品。航空宇宙関連の材料部品。[鉛フリーはんだ](../Page/鉛フリーはんだ.md "wikilink")素材、電子回路[基板](../Page/基板.md "wikilink")など。また、耐薬品性や耐腐食性機能を生かし、薬品・溶剤・腐食性ガス製造ラインの部品にも活用される。
