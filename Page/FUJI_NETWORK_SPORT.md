**『FUJI NETWORK SPORT』**（フジ ネットワーク すぽると）は[フジテレビ系列](../Page/フジテレビ系列.md "wikilink")（地上波・BS・CS共通）で放送される[スポーツ中継](../Page/スポーツ中継.md "wikilink")及び[スポーツ](../Page/スポーツ.md "wikilink")関連番組で2001年10月から2016年3月まで使用した共通ロゴである。

番組の最初に**FUJI NETWORK SPORT**と表示され、同時に「FUJI NETWORK SPORT」\[1\]とコールされる。スポーツニュースやスポーツ中継のほか、『[ジャンクCUP 2010](../Page/ジャンクSPORTS.md "wikilink")』（2010年12月28日放送）のようなアスリート参加型のバラエティ番組にも用いられている。なお、過去のスポーツ中継番組の一部では[PERFECT SPORT SPECIALの表記](../Page/PERFECT_SPORT.md "wikilink")・コールが使用されていた時期があった。

2016年4月～2018年3月は『[スポーツLIFE HERO'S](https://ja.wikipedia.org/wiki/スポーツLIFE_HERO'S "wikilink")』、2018年4月からは『[S-PARK](https://ja.wikipedia.org/wiki/S-PARK "wikilink")』\[2\]のオープニング映像を使用している。

略すと「**[FNS](../Page/フジネットワーク.md "wikilink")**」になるが、また別のものになる。

## ロゴデザイン

  - 初代（「ハァハァハァSPORT」と表記） 2001年10月 - 2002年12月
  - 2代目（黒い丸に白くSを模ったロゴ） 2003年1月 - 2005年3月
  - 3代目（「[すぽると\!](../Page/すぽると!.md "wikilink")」の番組タイトルロゴと共用） 2005年4月 - 2016年3月
