**俗ラテン語**（ぞくラテンご、 セルモー・ウルガーリス、）は、[ロマンス語](https://ja.wikipedia.org/wiki/ロマンス語 "wikilink")の[祖語](https://ja.wikipedia.org/wiki/祖語 "wikilink")で、[ローマ帝国](../Page/ローマ帝国.md "wikilink")内で話されていた口語[ラテン語](../Page/ラテン語.md "wikilink")のことである。ローマ帝国崩壊後、地方ごとに分化し現在のロマンス諸語になった。

[古代ローマ](../Page/古代ローマ.md "wikilink")から現代にかけて使用されてきた[ラテン語](../Page/ラテン語.md "wikilink")は基本的に文献に残る文語（古典ラテン語）のことであるが、これに対し口語、つまり民衆の話し言葉があったことが文献に残されており、これを俗ラテン語という。ただし、「俗」を意味する**vulgar**という言葉には「下品な」、「粗野な」、「卑しい」などの意味があるため、「民衆ラテン語」 （Popular Latin） 、「[ロマンス祖語](https://ja.wikipedia.org/wiki/ロマンス祖語 "wikilink")」（Proto-Romance）などの用語を主張する学者も多い\[1\]。

なお、  とは「日常の言葉」の意味の古典ラテン語であり、下記の音韻の変化に従えば俗ラテン語では  （セルモー・ヴォルガレ）となる。

## 文法

### 格の消失

下は第一変化名詞の rosa（バラ）に上記の音韻の変化を加えた仮想的な表であるが、単数では[主格](https://ja.wikipedia.org/wiki/主格 "wikilink")・[対格](../Page/対格.md "wikilink")・[奪格](../Page/奪格.md "wikilink")と、[属格](../Page/属格.md "wikilink")・[与格](https://ja.wikipedia.org/wiki/与格 "wikilink")が、それぞれ同形になっているのが分かる。[名詞](../Page/名詞.md "wikilink")全体にこのような変化が起こったため次第に[格](../Page/格.md "wikilink")語尾の区別がつかなくなり消失した。また区別が消失する過程で複数形は[ラ・スペツィア＝リミニ線](../Page/ラ・スペツィア＝リミニ線.md "wikilink")を基準に[西](https://ja.wikipedia.org/wiki/西 "wikilink")の地方（[イベリア](../Page/イベリア半島.md "wikilink")・[フランス](https://ja.wikipedia.org/wiki/フランス "wikilink")）では[対格](../Page/対格.md "wikilink")、[東](../Page/東.md "wikilink")（[イタリア](../Page/イタリア.md "wikilink")・[ルーマニア](../Page/ルーマニア.md "wikilink")）では主格で代表されるようになった。

| -- | ------- | ---------- |
| 単数 |         |            |
| 主格 | rosa    | ros**a**   |
| 属格 | rosae   | ros**e**   |
| 与格 | rosae   | ros**e**   |
| 対格 | rosam   | ros**a**   |
| 奪格 | rosā    | ros**a**   |
| 複数 |         |            |
| 主格 | rosae   | ros**e**   |
| 属格 | rosārum | ros**aro** |
| 与格 | rosīs   | ros**is**  |
| 対格 | rosās   | ros**as**  |
| 奪格 | rosīs   | ros**is**  |

### 中性の消失

第二変化名詞が音韻の変化を被りもともと単数主格・複数主格・複数対格にしか違いのない -us 型と -um 型が混同され、 -um 型の大部分を占める中性名詞は男性名詞として扱われるようになった。また複数形で使われることの多い名詞は主格の -a が女性形と同じなので女性形として扱われるようになった。

### 複合前置詞

俗ラテン語では前置詞を二つ三つ合わせた複合前置詞が現れた。ロマンス語に受け継がれているものには下のようなものがある。 （<sup>F</sup> <sup>E</sup> <sup>P</sup> はそれぞれ[フランス語](https://ja.wikipedia.org/wiki/フランス語 "wikilink")、[スペイン語](https://ja.wikipedia.org/wiki/スペイン語 "wikilink")、[ポルトガル語](https://ja.wikipedia.org/wiki/ポルトガル語 "wikilink")の意味とする。）

  - **donde**<sup>E</sup> \< de + unde
  - **dans**<sup>F</sup> \< de + intus
  - **dentro**<sup>EP</sup> \< de + intro
  - **dès**<sup>F</sup> \< de + ex
  - **desde**<sup>EP</sup> \< de + ex + de
  - **depuis**<sup>F</sup> / **después**<sup>E</sup> / **depois**<sup>P</sup> \< de + ex + post
  - **dehors**<sup>F</sup> / **de fuera**<sup>E</sup> / **de fora**<sup>P</sup> \< de + foris

### 副詞

古典ラテン語では副詞を作るのに cārus（大事な） -\> cārē や acer（鋭い）-\> acriter のように -ē あるいは -iter をつけるが、俗ラテン語ではこの方法は失われ[形容詞](../Page/形容詞.md "wikilink")の女性形に mente をつけるようになった。この mente は、元は mens（心、女性名詞）の単数[奪格](../Page/奪格.md "wikilink")で、形容詞+mente で「～な気持ちで」の意味であったのが心の意味が無くなったものである（例：vēlōx「速い」-\> voloce mente「心が速く→急いで」）。この用法は[紀元前1世紀](../Page/紀元前1世紀.md "wikilink")の[カトゥルス](https://ja.wikipedia.org/wiki/カトゥルス "wikilink")の文章に散見される。
