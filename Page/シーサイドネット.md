**シーサイドネット**（CsideNet）は、[東京都](../Page/東京都.md "wikilink")[豊島区](https://ja.wikipedia.org/wiki/豊島区 "wikilink")に本社を置く、[レンタルサーバー](../Page/ホスティングサーバ.md "wikilink")（ホスティングサービス）事業を行う企業である。

## 沿革

  - [2000年](../Page/2000年.md "wikilink")2月15日 - [個人事業者としてCsideNetを設立](../Page/個人事業主.md "wikilink")。
  - [2002年](../Page/2002年.md "wikilink")2月15日 - 有限会社シーサイドネットを設立。
  - [2004年](../Page/2004年.md "wikilink")3月7日 - 株式会社シーサイドネットに商号変更。
  - 2004年4月1日 - セコムトラストネット（現[セコムトラストシステムズ](https://ja.wikipedia.org/wiki/セコムトラストシステムズ "wikilink")）との業務提携により、[セキュリティサービス付き](../Page/コンピュータセキュリティ.md "wikilink")[事業者向けレンタルサーバー](../Page/企業.md "wikilink") C'S SERVER Professionalを開始。
  - 2004年6月21日 - 本社を[豊島区](https://ja.wikipedia.org/wiki/豊島区 "wikilink")[東池袋](../Page/東池袋.md "wikilink")の[サンシャイン60ビルに移転](../Page/サンシャインシティ.md "wikilink")。
  - [2005年](../Page/2005年.md "wikilink")10月 - 1年間で100の新機能を追加する「サービス強化キャンペーン」を開始。
  - 2005年12月1日 - [バリューコマース](../Page/バリューコマース.md "wikilink")がホスティング事業を会社分割し、新会社データソリューションズ株式会社を設立。同社の全株式をシーサイドネットが取得。
  - [2006年](../Page/2006年.md "wikilink")1月-3月 - [データセンターをセコムトラストネット](../Page/インターネットデータセンター.md "wikilink")（現セコムトラストシステムズ）の「[セキュアデータセンター](http://www.secomtrust.net/service/datacenter/whatdata.html)」に移転。\[1\]
  - [2015年](../Page/2015年.md "wikilink")8月22日 - 本社を千代田区三番町のシーサイドビルに移転\[2\]。
  - 2019年9月17日 - 本社を[豊島区](https://ja.wikipedia.org/wiki/豊島区 "wikilink")[東池袋](../Page/東池袋.md "wikilink")の[サンシャイン60ビルに移転](../Page/サンシャインシティ.md "wikilink")\[3\]。

## レンタルサーバーサービス

[2007年](../Page/2007年.md "wikilink")現在、約20,000ユーザーが利用している。

  - C'S SERVER Personal サブドメイン/独自ドメイン - 共用サーバー
  - C'S SERVER Professional - 事業者向け共用サーバー
  - C'S SERVER Private Hosting - 専用サーバー
