**宮本・永井ポテンシャル**は、[銀河円盤がつくる](../Page/渦巻銀河.md "wikilink")[重力ポテンシャル](https://ja.wikipedia.org/wiki/重力ポテンシャル "wikilink")を表すモデルである。[1975年](../Page/1975年.md "wikilink")に Masahito Miyamoto と Ryuzaburo Nagai によって提案され\[1\]、その後の研究で広く用いられるようになった\[2\]。

## 概要

宮本・永井ポテンシャルは軸対称な銀河円盤のモデルであり、[円柱座標](https://ja.wikipedia.org/wiki/円柱座標 "wikilink") \(R\), \(\theta\), \(z\) を用いて次式で与えられる\[3\]。

\[\Phi_\mathrm{d} ( R, z ) = - \frac{ G M_\mathrm{d} }{ \sqrt{ R^2 + \left( a_\mathrm{d} + \sqrt{ z^2 + b_\mathrm{d}^2 } \right)^2 } }\] ここに \(M_\mathrm{d}\) は円盤の質量、\(G\) は[重力定数であり](../Page/万有引力定数.md "wikilink")、パラメータ \(a_\mathrm{d}\), \(b_\mathrm{d}\) がそれぞれ円盤の \(R\)-方向、\(z\)-方向のスケールを与える。\(a_\mathrm{d} = 0\) のとき宮本・永井ポテンシャルは球対称な[プラマーモデル](https://ja.wikipedia.org/wiki/プラマーモデル "wikilink")に帰着し、\(b_\mathrm{d} = 0\) のときには無限に薄い円盤を表す[クズミンモデル](https://ja.wikipedia.org/wiki/クズミンモデル "wikilink")に帰着する。そのため、宮本・永井ポテンシャルはこれらのパラメータを調整することで厚みを持った円盤のポテンシャルを表すことができる。

宮本・永井ポテンシャルは次の質量分布

\[\rho ( R, z ) = \frac{ b_\mathrm{d}^2 M_\mathrm{d} }{ 4 \pi } \frac{ a_\mathrm{d} R^2 + \left( a_\mathrm{d} + 3 \sqrt{ z^2 + b_\mathrm{d}^2 } \right) \left( a_\mathrm{d} + \sqrt{ z^2 + b_\mathrm{d} } \right)^2 }{ \left[ R^2 + \left( a_\mathrm{d} + \sqrt{ z^2 + b_\mathrm{d}^2 } \right)^2 \right]^\frac{5}{2} \left( z^2 + b_\mathrm{d}^2 \right)^\frac{3}{2} }\] がつくる[重力場](../Page/重力場.md "wikilink")に対応する。円盤面 (\(z = 0\)) においてこの密度場は動径 \(R\) が大きいとき \(\rho \propto R^{-3}\) という形で減少するが, 実際には円盤銀河の輝度プロファイルは \(\propto \exp \left( - R / R_\mathrm{d} \right)\) という形で減少することが知られている（[指数法則](https://ja.wikipedia.org/wiki/セルシック則 "wikilink")）\[4\]。

## 円盤銀河の重力ポテンシャル

円盤銀河全体の重力ポテンシャルは、宮本・永井ポテンシャルに加えて[バルジがつくる重力ポテンシャルと](https://ja.wikipedia.org/wiki/銀河バルジ "wikilink")[ダークマターハロー](https://ja.wikipedia.org/wiki/ダークマターハロー "wikilink")のポテンシャルを加えたものとなる。バルジの寄与としては[ハーンキストモデル](https://ja.wikipedia.org/wiki/ハーンキストモデル "wikilink")

\[\rho = - \frac{ G M_\mathrm{b} }{ 2 \pi a^3 } \frac{ 1 }{ r + a }\] が\[5\]、ダークマターハローの寄与としては[NFWプロファイル](https://ja.wikipedia.org/wiki/NFWプロファイル "wikilink")

\[\rho = \frac{ 4 \rho_\mathrm{s} r_\mathrm{s}^3 }{ r ( r_\mathrm{s} + r )^2 }\] が\[6\]しばしば仮定される。
