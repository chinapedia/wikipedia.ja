**リトマス試験紙**（リトマスしけんし、\[1\]\[2\]）とは、[リトマス](../Page/リトマス.md "wikilink")を染み込ませた、[水溶液](../Page/水溶液.md "wikilink")の酸性・アルカリ性の反応を試験するものである\[3\]。**リトマス紙**とも呼ぶ\[4\]。

## 概要

[リトマス](../Page/リトマス.md "wikilink")水溶液に[ろ紙](../Page/ろ紙.md "wikilink")を浸して乾燥させたもので、青色と赤色の2種類がある\[5\]。変色範囲はpH4.5 - 8.3で\[6\]、青色の試験紙は[酸性](https://ja.wikipedia.org/wiki/酸性 "wikilink")の溶液で赤に、赤色の試験紙は[アルカリ性](https://ja.wikipedia.org/wiki/アルカリ性 "wikilink")の溶液で青に変化する\[7\]。この性質を使い、青色の試験紙が赤くなると酸性、赤色の試験紙が青くなるとアルカリ性、どちらも変化しないと[中性](https://ja.wikipedia.org/wiki/中性 "wikilink")というように水溶液が何性かを簡単に判別できる\[8\]\[9\]。[pH試験紙](https://ja.wikipedia.org/wiki/pH試験紙 "wikilink")の一種\[10\]。

## 色の変化

色の変化は次のようなものになる\[11\]\[12\]。

|    |          |       |
| -- | -------- | ----- |
| 酸性 | 中性（出荷状態） | アルカリ性 |
| 赤  | 青        | 青     |
| 赤  | 赤        | 青     |

### 変色の原理

リトマス試験紙に染み込ませてある[リトマスゴケ](https://ja.wikipedia.org/wiki/リトマスゴケ "wikilink")の主成分の[アゾリトミン](https://ja.wikipedia.org/wiki/アゾリトミン "wikilink")や[レカノール酸](https://ja.wikipedia.org/wiki/レカノール酸 "wikilink")という成分が、アルカリと反応すると青色の塩を作り出し、酸と反応すると赤く変色する特徴があるため、リトマス試験紙は変色する\[13\]\[14\]。

## 作り方

リトマス試験紙の原料のリトマスゴケや[ウメノキゴケ](https://ja.wikipedia.org/wiki/ウメノキゴケ "wikilink")などの種々の[地衣類](../Page/地衣類.md "wikilink")から[紫色](https://ja.wikipedia.org/wiki/紫色 "wikilink")の液体[リトマス](../Page/リトマス.md "wikilink")を抽出し、それを[アルコール](https://ja.wikipedia.org/wiki/アルコール "wikilink")に溶かす。これに[アンモニア](../Page/アンモニア.md "wikilink")または[塩酸](https://ja.wikipedia.org/wiki/塩酸 "wikilink")(もしくは[硫酸](https://ja.wikipedia.org/wiki/硫酸 "wikilink")、[塩化水素](../Page/塩化水素.md "wikilink"))を添加してそれぞれ赤色と青色に変色させる。これを[ろ紙](../Page/ろ紙.md "wikilink")の小片に浸して乾燥させて切り分けるとリトマス試験紙となる。近年は赤色や青色に変色させずに紫色のまま使う｢リトマスニュートラル｣と呼ばれるリトマス試験紙もある。\[15\]\[16\]\[17\]

## 保管方法

高温､多湿､直射日光を避けて\[18\]、密閉して保存する\[19\]。
