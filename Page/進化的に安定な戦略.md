**進化的に安定な戦略**（しんかてきにあんていなせんりゃく、、**ESS**）は、[進化生物学](../Page/進化生物学.md "wikilink")および[ゲーム理論](https://ja.wikipedia.org/wiki/ゲーム理論 "wikilink")の重要な概念で、[ジョン・メイナード＝スミス](../Page/ジョン・メイナード＝スミス.md "wikilink")と[ジョージ・プライスによって](https://ja.wikipedia.org/wiki/ジョージ・プライス_\(科学者\) "wikilink")1973年に提唱された\[1\]。

これは、[生物](../Page/生物.md "wikilink")の母集団のとる、「侵略されない戦略」の概念を基礎としている。仮に[突然変異](../Page/突然変異.md "wikilink")で[対立遺伝子](../Page/対立遺伝子.md "wikilink")が発生し、別の戦略を取って他の生物に働きかけようとしても、母集団を侵略することはできず、逆に[自然淘汰](https://ja.wikipedia.org/wiki/自然淘汰 "wikilink")で排除されてしまうような戦略である。メイナード＝スミスらはこの概念によってゲーム理論の有効性を広く示し、[行動生態学](https://ja.wikipedia.org/wiki/行動生態学 "wikilink")、[経済学](../Page/経済学.md "wikilink")、[心理学](../Page/心理学.md "wikilink")などに影響を与えた。

## 概要

具体例をもとに進化的安定性を説明する\[2\]。動物が交尾相手や餌といった資源を同じ種の個体と争う場合、互いに殺し合うような闘争を避け、威嚇などの儀式的な闘争をする事で決着をつける事がある。 こうした儀式的闘争が発達した原因として、進化的安定性の概念が登場する以前は、「闘争の際に殺し合いを行なう種は絶滅してしまうので、儀式的闘争をする種だけが生き残った」といった[群淘汰](https://ja.wikipedia.org/wiki/群淘汰 "wikilink")的な理由づけ\[3\]がなされがちであった。

しかし自然選択の対象が個々の個体である事を考えると、群淘汰的な理由づけでは儀式的闘争が数多くの種で発達した事をうまく説明できない。また、実際の動物の闘争を観察すると、戦いがエスカレートして傷つけ合ったり殺し合ったりする事も珍しくない\[4\]事も前述した理由づけとは合致しない。

そこで、儀式的闘争のような現象を群淘汰に頼らず、生物進化の基本的な原則である「[自然選択によって繁殖成功率が高い適応戦略が種に広がっていく](../Page/自然選択説.md "wikilink")」という事によって説明する為の枠組みが、本稿の主題である進化的安定性である。

話を簡単にするため、動物の戦略が「タカ戦略」と「ハト戦略」の２つのみからなる場合を考える。タカ戦略とは、闘争がエスカレートした場合に戦う戦略であり、ハト戦略は闘争がエスカレートした場合には逃げる戦略である。

もし同じ動物種に属する全ての個体が常にハト戦略を取るのであれば、儀式的なものであれ実際的なものであれ、闘争は生じないであろう。しかしこのような種に突然変異などによって生まれた、タカ戦略を取る個体が少しでも侵略してくれば、周囲にいるハト戦略の個体は全て逃げ出すわけだから、タカ戦略を持つ個体が圧倒的に有利となり、子孫を残す事で種にタカ戦略が広がる事となる。したがってハト戦略を取る個体だけからなる種は安定しない。

逆に全ての個体が常にタカ戦略を取るとすれば、闘争は常にエスカレートする。ここにハト戦略の個体が侵入してくると、他の個体が闘争により著しく疲弊している中、闘争から逃げているハト戦略の個体だけが有利となり、ハト戦略が種の中に広まっていく。したがってタカ戦略を取る個体だけからなる種もやはり安定しない。

こうして、ハト戦略の個体とタカ戦略の個体が混じり合った状態で種は安定する事になる。この状態では、闘争相手がハト戦略を取るかタカ戦略を取るかを見極める事が重要となる為、儀式的闘争が発達する事になる。

進化的安定性は、上で述べたような複数の戦略が入り混じった状態での安定性概念である。

### 混合戦略

前節で説明した例をはじめとして、生物による多くの駆け引きは、自身の利得を最大化しようとする個体の同士による一種のゲーム(**[進化ゲーム](../Page/進化ゲーム.md "wikilink")**)とみなす事ができる為、生物の駆け引きを[ゲーム理論](https://ja.wikipedia.org/wiki/ゲーム理論 "wikilink")により記述する事ができる。

進化的安定性の概念もゲーム理論の枠組みで記述でき、その定式化にはゲーム理論における混合戦略の概念が有用となる。

前節で説明した例を使って説明すると、闘争が必要になった時、各個体が取りうる選択肢として、｢タカ戦略｣と｢ハト戦略｣という二種類の戦略(**純粋戦略**)があった。しかし各個体はこれらの純粋戦略のうちひとつを常に取り続けるわけではなく、｢30%の確率でタカ戦略を取り、70%の確率でハト戦略を取る｣といった戦略をも取りうる。

**混合戦略**とは、このように個々の純粋戦略の上に確率を付与した戦略を指す。進化的安定性の概念は、この混合戦略の概念に対して定式化される。

### 進化的安定性の直観的な定式化

進化的安定性とは、何らかの混合戦略が集団の中で支配的になるための条件である。すなわち、混合戦略 *σ* が進化的に安定であるとは、直観的には、集団の中に戦略*σ*がすでに広まっている状況下において、 別の混合戦略*τ* を取る個体が少数侵入してきたとしても、それが排除される事をいう。

より詳しく言うと、たとえ*σ* に近い別の混合戦略 *τ* を取る個体群が集団に少数侵入してきたとしても、戦略*σ* を取る個体と戦略*τ*を取る個体が2者間で戦った際、前者の個体の方がより高い利得が[期待できるため](../Page/期待値.md "wikilink")、戦略*τ*を取る個体は[自然選択](https://ja.wikipedia.org/wiki/自然選択 "wikilink")により、いつしか集団から消えてしまう、という事である。

## ゲーム理論からの準備

進化的安定性はゲーム理論の概念に基づいて定式化することができる。そこで本節では、必要なゲーム理論の概念を導入し、次節で進化的安定性を定式化する。

### 利得関数

#### 定義

進化的安定性を定義するには、まず個々の個体の利得をゲーム理論的に定義する必要がある。ゲーム理論において利得はほかの個体とゲームを行ったときに得られる実数値として定義され、得られる利得は自分が取った戦略と対戦相手がとった戦略の結果として決まる。

すなわち、純粋戦略*i* を取る個体*P* が、純粋戦略 *j* を取る別の個体*Q*とゲームを行ったとき、個体*P*は**利得**と呼ばれる実数値

\[E(i,j)\]

を獲得する。そして*i*、*j*に\(E(i,j)\)を対応させる関数*E*を個体*P*に関する**利得関数**と呼ぶ。

利得関数はゲームが始まる前の段階で、外界の状況等により事前に定まっており、個々の個体が変えることはできない。個々の個体にできるのは、与えられた利得関数から得られる利得を最大化するよう自身の戦略を選ぶことだけである。

#### 対称性

進化的安定性を定義する際には、全ての個体に対して同一の利得関数が適用される事が前提となる。したがって純粋戦略*i* を取る個体*P* が、純粋戦略 *j* を取る別の個体*Q*と戦った時、個体*Q*が得る利得を

\[E'(i,j)\]

とすると、

\[E'(i,j)=E(j,i)\]

が任意の*i*、*j*に対して成立する事が要請される。利得関数がこのような性質を満たすゲームを**対称なゲーム**という。

#### 混合戦略の利得

混合戦略を取る個体の利得は、純粋戦略に対する利得の期待値として定義される。すなわち、各個体が取りうる純粋戦略にと番号をつけ、純粋戦略を取る確率がである混合戦略を\((p_i)_{i=1,\ldots,n}\)と書く事にすると、個体*P* 、*Q*がそれぞれ混合戦略\(\sigma=(p_i)_{i=1,\ldots,n}\)、\(\xi=(q_i)_{i=1,\ldots,n}\)を取る際の*P*の利得は、

\[E(\sigma,\tau)=\sum_{i,j}p_iq_jE(i,j)\]

により定義される。

### 進化ゲームの定式化

進化的安定性を定義するためのゲーム（**進化ゲーム**）は以下のようなものである。なお、このゲームはゲーム理論の言葉で言えば「**対象な2人[戦略型ゲーム](../Page/標準型ゲーム.md "wikilink")**」に相当する。

進化的安定性を定義するための進化ゲームでは、対戦する2個体*A*、*B*が選択肢として取りうる純粋戦略*1*、*2*、…、および利得関数が「ゲームのルール」として事前に定まっている。そして*A*、*B*は以下の手順でゲームを行なう：

1.  *A*、*B*はそれぞれ、与えられた選択肢の中から１つの純粋戦略*i*、*j*を秘密裏に選ぶ
2.  *A*、*B*は*i*、*j*を同時に公表する
3.  *A*、*B*はそれぞれ利得\(E(i,j)\)、\(E(j,i)\)を得る。

*A*、*B*の目的は、自身の利得を最大化する事である。

前節でも述べたように、進化的安定性の文脈では全ての個体に対して同一の利得関数が適用される事が前提とされるため、上述したゲームにおいて*A*、*B*が得られる利得はそれぞれ\(E(i,j)\)、\(E(j,i)\)と対称な形になっている。

上述した進化ゲームは、ゲームに参加する2個体*A*、*B*取りうる純粋戦略をそれぞれ行、列として*A*、*B*の利得を[行列](https://ja.wikipedia.org/wiki/行列 "wikilink")の形にまとめた**利得表**により特徴づけられる。

#### 具体例

下に上げたのは、前述したタカ戦略、ハト戦略からなる進化ゲーム（[タカハトゲーム](../Page/チキンゲーム.md "wikilink")）の利得表である\[5\]：

|    |                                              |                                        |
| -- | -------------------------------------------- | -------------------------------------- |
|    | タカ                                           | ハト                                     |
| タカ | \(\left({V-C \over 2},{V-C \over 2}\right)\) | \((V,0)\)                              |
| ハト | \((0,V)\)                                    | \(\left({V\over 2},{V\over 2}\right)\) |

ここでは2個体が争っている資源（例えば餌）を得た時に得られる利得を表し、は闘争によって怪我を追う事による損失を表す。

また利得表で縦軸は個体*A*の取る戦略、横軸は個体*B*の戦略であり、表内の (○, △)は、A、Bの利得がそれぞれ○、△である事を意味する。例えば表の左下のマスにかかれているは個体*A*がハト戦略、個体*B*がタカ戦略を撮った時、*A*、*B*の利得がそれぞれ、である事を意味する。表の左上と右下で値が2で割られているのは、2個体で資源を分け合った為である。

### 混合戦略の線形結合

最後に、進化的安定性を定義する際に記法を簡単にするため、混合戦略の「線形結合」を定義する。

以下、話を簡単にするため、各個体が取れる純粋戦略の種類が有限個である事を仮定するが、無限個の場合にも自然に定義を拡張できる。

まず、記号を定義する。各個体が取りうる純粋戦略にと番号をつける。そして純粋戦略を取る確率がである混合戦略を\((p_i)_{i=1,\ldots,n}\)と書く事にする。

2つの混合戦略の\(\sigma=(p_i)_{i=1,\ldots,n}\)、\(\xi=(q_i)_{i=1,\ldots,n}\)、および実数とが与えられた時、、の、による**[線形結合](https://ja.wikipedia.org/wiki/線形結合 "wikilink")**を

\[a\sigma+b \xi=(ap_i+b q_i)_{i=1,\ldots,n}\] により定義する。\(a+b =1\)であれば、混合戦略の線形結合\(a\sigma+b \xi\)もまた、混合戦略である。

## 進化的安定性の厳密な定義

有限個の純粋戦略を持つ戦略型ゲームの事を(利得表が有限サイズの行列の形に書けるので)**行列ゲーム**といい、これはもっとも典型的な進化ゲームの一つである。本節では対称な行列ゲームに対する進化的安定性を３つの異なる視点から定義づける。これら３つの定義は対称な行列ゲームにおいては同値であるが、より一般的な進化ゲームにおいては必ずしも同値ではない。

### 定義

対称な行列ゲームにおける進化的安定性は以下のように定義される\[6\]\[7\]。のを**侵入障壁**という。では侵入障壁が混合戦略に依存する事を許容するバージョンの定義を採用したが、がに依存しないバージョンの定義も存在し、これを**一様な侵入障壁をもつ進化的安定性**（*ESS with uniform invasion barrier*）と呼ぶ\[8\]。一般には一様なもののほうがそうでないものより強い定義であり、純粋戦略が無限個あるゲームの場合には進化的安定であるにもかかわらず一様な侵入障壁をもつ進化的安定ではない混合戦略が存在する事が知られている\[9\]。しかしで考えているゲーム（＝有限個の純粋戦略を持つ対象な戦略型ゲーム）の範囲では、両者の定義は同値である\[10\]\[11\]。

#### 定義の解釈

混合戦略\(\sigma_*\)を取る個体の集団に、混合戦略\(\sigma\)を取る個体群が侵入し、集団全体の中で後者の割合がになったとする。このとき、対戦相手がランダムに選ばれるとすれば、混合戦略\(\sigma_*\)を取る個体の利得の期待値は

\[(1-\varepsilon)E(\sigma_*,\sigma_*)+\varepsilon E(\sigma_*,\sigma)=E(\sigma_*,(1-\varepsilon)\sigma_*+\varepsilon \sigma)\] となり、で登場する不等式の左辺と一致する。同様の理由により混合戦略\(\sigma\)を取る個体の利得の期待値は

\[E(\sigma,(1-\varepsilon)\sigma_*+\varepsilon \sigma)\] となり、で登場する不等式の右辺と一致する。

したがっては混合戦略\(\sigma\)を取る個体群が\(\varepsilon_0\)以下の割合だけ侵入したとしても、混合戦略\(\sigma_*\)を取る個体の利得の期待値の方が混合戦略\(\sigma\)を取る個体の利得の期待値よりも真に大きくなる事を示している\[12\]。

### 局所優位性(local superiority)

２つの混合戦略\(\sigma_*=(p_i)_{i=1,\ldots,n}\)、\(\sigma=(q_i)_{i=1,\ldots,n}\)の**[距離](../Page/距離空間.md "wikilink")**を

\[\mathrm{d}(\sigma_*,\sigma):=\sqrt{\sum_{i=1}^n|p_i-q_i|^2}\]　　　…

により定義するとき\[13\]、進化的安定性を以下のように異なる角度から特徴づける事ができる\[14\]\[15\]\[16\]：なお、上では距離を式に従って定義したが、に書いたESSの別定義で本質的なのは距離そのものではなく、距離から定まる[位相構造なので](https://ja.wikipedia.org/wiki/位相空間 "wikilink")\[17\]、式の代わりに以下の[ℓ<sup>1</sup>距離](https://ja.wikipedia.org/wiki/Lp空間 "wikilink")

\[\mathrm{d}(\sigma_*,\sigma):=\sum_{i=1}^n|p_i-q_i|\]　　　

を用いて定義してものものと同値になる。

に書いたESSの別定義はより広範な進化ゲームに対して進化的安定性の概念を一般化する場合に有益であり\[18\]、一般化のさせかたにより、neighborhood invader strategy、neighborhood superiorなどとも呼ばれる\[19\]。

### 簡便な特徴づけ

は進化的安定性の直観的な意味を自然に定式化したものになっているものの、この定義に基づいて混合戦略の進化的安定性を直接チェックするのは容易ではない。そこで進化的安定性をより簡単にチェックする事を可能にする、別の特徴付けを紹介する\[20\]：

与えられた混合戦略に対し、\(E(\xi,\tau)\)を最大にする混合戦略をの**最適反応**(*best reply*)という\[21\]。

均衡条件は、\(\sigma_*\)が\(\sigma_*\)自身の最適反応である事を意味しており、\(E(\xi,\sigma_*)\)の最大値は\(M=E(\sigma_*,\sigma_*)\)である\[22\]。一方、安定条件は、\(M=E(\sigma,\sigma_*)\)を満たす\(\sigma\neq \sigma_*\)、すなわち\(\sigma_*\)に対する最適反応のうち\(\sigma_*\)以外の混合戦略\(\sigma\)は\(E(\sigma_*,\sigma)> E(\sigma,\sigma)\)を満たしている事を意味している\[23\]。

### ナッシュ均衡との関係

ゲーム理論における重要な均衡概念として[ナッシュ均衡](../Page/ナッシュ均衡.md "wikilink")があり、進化的安定性は\((\sigma_*,\sigma_*)\)のナッシュ均衡性と関係がある。本項で考えているゲーム（2人対称戦略型ゲーム）の場合、混合戦略の組\((\sigma_*,\sigma_*)\)が**ナッシュ均衡**であるとは任意の混合戦略に対し、

  -
    \(E(\sigma_*,\sigma_*)\ge E(\sigma,\sigma_*)\)　　…

が成立する事を言う。特に任意の混合戦略に対して式の不等号がイコールなしで成り立つ場合、\((\sigma_*,\sigma_*)\)は**狭義ナッシュ均衡**であるという。

前述したタカハトゲーム対してを適用する事で次が成立する事が分かる\[26\]：

## 進化的安定性概念の一般化

### モチベーション

これまで本稿では、行列ゲームに対する進化的安定性を議論してきたが、行列ゲームは下記のような条件を満たす場合にしか現実世界の生物の闘争をモデル化できない：

1.  ゲームは一度しか行われない
2.  各個体がとれる純粋戦略の個数は有限である
3.  各個体がどの純粋戦略を取るのかはゲーム開始時点にランダムに選ぶ事ができる
4.  ゲームは常に２個体で行われる
5.  全ての個体に対して同一の利得関数が適用される事が前提としている

しかし実際の生物学における応用では、以上の条件を満たさない事も多い：

1.  多くの状況では各個体はその生涯において何度も他の個体と闘争を繰り返すので、ゲームを繰り返し行う形でモデル化した方が自然な場合が多い
2.  「植物が種を飛ばす飛距離」や「動物が行動を起こすまでの時間」のように純粋戦略として連続量を取る事ができるケースでは純粋戦略の個数は無限にある
3.  哺乳類の配偶戦略のように「オスかメスか？」という生まれた段階で決定する戦略は、ゲーム開始時点にランダムに選ぶ事はできない
4.  草むらで種をばらまくいて近くにいる他の全ての個体と種のばらまく位置を争うケースのように、複数個体と争うゲームも多い
5.  「オスとメス」、「テリトリーを守る個体とそこに侵入する個体」のように非対称な闘争では、闘争する個体がどちらの立場にいるのかで利得関数が異なるはずである

本章の目標は上で述べたような、行列ゲームの範疇に収まらないより一般的なゲームに対して進化的安定性を定義する事である。

### 一般化の手法

本節では上述した1,...,5の制約を外すための手法を順に述べていく。

#### 繰り返しゲーム

まず1に関しては、ゲーム理論の言葉で言えば[繰り返しゲーム](https://ja.wikipedia.org/wiki/繰り返しゲーム "wikilink")を考える必要がある、という事である。一回のゲームの利得と繰り返し行ったゲームの利得の平均値とを区別する為、一回のゲームの利得はこれまで通り\(E(\cdot,\cdot)\)で表し、繰り返し行ったゲームの利得の平均値を\(\mathcal{E}(\cdot,\cdot)\)と表す事に事にする。

ゲームを行うたびに闘争相手が毎回異なると仮定できる場合には、繰り返しゲームの平均利得\(\mathcal{E}(\cdot,\cdot)\)は\(E(\cdot,\cdot)\)と一致する（詳細後述）。一方、同一の闘争相手と何度もゲームを繰り返す場合はより複雑で、（有限繰り返しゲームの場合）や[フォーク定理](../Page/フォーク定理.md "wikilink")（無限繰り返しゲームの場合）など、ゲーム理論の手法を用いて\(\mathcal{E}(\cdot,\cdot)\)を求める必要がある。

#### 戦略空間

2および3に関しては、**戦略空間**(strategy space)という概念を導入する事で一般化を図る\[27\]\[28\]。戦略空間とは、そのゲームにおいて各個体が取りうる戦略全体の集合の事である。例えば行列ゲームの場合は混合戦略全体の集合が戦略空間である。すなわち、

\[\Delta_n=\left\{(p_i)_{i=1,\ldots,n} \Bigg| 0\le p_1,\ldots,p_n \le 1,~ \sum_{i=1}^np_i=1 \right\}\]　　　　...()

が戦略空間である。ここでは取りうる純粋戦略の個数である。

また「個体の体長」のように連続量の純粋戦略が取れる（が闘争の際ランダムに体長を変える事はできないので混合戦略は取れない）ゲームの場合、戦略空間は正の実数全体の集合

\[\mathbf{R}_+=\{x>0 \mid x\in \mathbf{R} \}\]

である。一方「動物が行動を起こすまでの時間」のように純粋戦略は連続量であり、混合戦略も取りうるゲームの場合には、戦略空間は

\[\{\]上の確率分布\(\}\)

である\[29\]。なお進化的安定性の議論では、戦略間の「近さ」の概念が定義できる事が望ましいので、戦略空間が[位相空間](https://ja.wikipedia.org/wiki/位相空間 "wikilink")である事を求める事も多い\[30\]。

#### 個体群

4に関しては、個体vs.個体だけでなく個体vs.**[個体群](../Page/個体群.md "wikilink")**(population)の闘争を考える事で一般化を図る\[31\]。個体群に属する個体数が有限である場合には数学的解析が難しくなるので、以下本説では個体数が無限であるものと仮定する\[32\]。より厳密に言うと、戦略空間に属する戦略を取る個体の割合を区間に属する実数として定義できるものと仮定する。現実には無限の個体を含む個体群は存在しないが、個体群に属する個体が十分大きい場合には、近似的にこのような仮定を置いて議論を進める事ができる。

個体群の各々の個体は戦略空間に属するいずれかの戦略を取る。個体群において、「戦略を（確率1で）取る個体の割合がそれぞれである」という状態を

\[\varepsilon_1\delta_{\sigma_1}+\cdots+\varepsilon_1\delta_{\sigma_m}\]

もしくは

\[\sum_{i=1}^m\varepsilon_i\delta_{\sigma_i}\]

と表記し\[33\]、これをの**個体群戦略**(population strategy\[34\])と呼ぶ。個体群戦略と区別するため、個々の個体の戦略の事を強調して**個体戦略**(player strategy\[35\])と呼ぶ。

なお上の式における記号「」は、混合戦略の和「\(\varepsilon_1\sigma_1+\cdots+\varepsilon_1\sigma_m\)」と区別する為につけられた単なる記号であると解釈して差し支えない。この場合、上記の式は数学的には[形式和である](https://ja.wikipedia.org/wiki/:En:Formal_sum "wikilink")。しかしこのを上で定義された[ディラックのデルタ関数](../Page/ディラックのデルタ関数.md "wikilink")であると解釈する事で、上式を上の分布を表す式とみなす事もできる。

また上では個体群が有限個の戦略のいずれかしか取らない場合を考えたが、が[性質のよい空間であれば無限個の戦略を取る場合も考える事ができる](https://ja.wikipedia.org/wiki/測度空間 "wikilink")。しかし進化的安定性を定義する上では有限個の戦略を取る場合のみを考察すれば十分であるので、本稿では以下、無限個の戦略を取る場合は考えない。

本稿では個体群の性質として主として考えるのは個体群戦略のみなので、紛れがなければ個体群とその個体群戦略とで記号を混用し、

\[\Pi=\varepsilon_1\delta_{\sigma_1}+\cdots+\varepsilon_1\delta_{\sigma_m}\]

という表記も用いるものとする。

#### 対称化

5で述べたように実際の生物では２つの個体の立場が非対称なゲームも起こりうるが、進化ゲーム理論では２つの個体が対称な場合のみに対して進化的安定性を定義し\[36\]、非対称なゲームには**対称化**を施す事により対称なゲームに対する進化的安定性の概念を利用する。例えば「オス」と「メス」という２つの立場がある状況では、個体が受精した際「オス」か「メス」かをランダムに選べる事を考慮する事により、全ての個体が「オス」になる可能性も「メス」になる可能性もある対称なゲームとして定式化する。

そこで本章では以下、対称なゲームに対する進化的安定性のみを議論するものとし、非対称なゲームに対する進化的安定性は後の章で議論するものとする。

### 行列ゲームの再定式化

以上までで述べた一般的なフレームワークにおける進化的安定性の定義を述べる前に、行列ゲームを上述のフレームワークにおいて再定式化する。このために通りの純粋戦略が取れる行列ゲームを考え、その利得関数をとする。さらにを個体群とし、を個体群の中にいる一匹の個体とし、が取る混合戦略を\(\sigma=(p_i)_{i=1,\ldots,n}\)とする。

前章で述べた行列ゲームでは、は個体群の中のいずれか一匹の個体と一度だけゲームを行う事を前提としていた。しかし本章で述べる一般的フレームワークにおいては、の中の複数の個体と闘争する事を前提としている。より正確に言うと、定数を固定し、以下のようなゲームを回繰り返す：

1.  の中から[一様ランダムに一匹の個体](https://ja.wikipedia.org/wiki/一様分布 "wikilink")を選ぶ（は回行う各ゲームで毎回独立に選ばれる）。

2.  とが利得関数を持つ行列ゲームを行う。

そしてこのようなゲームにおけるの平均利得を\(\mathcal{E}(\sigma,\Pi)\)と表記する（ここで我々は前節で述べたように記号を混用して個体群の個体群戦略にもという記号を用いている）。

の個体群戦略が混合戦略により

\[\Pi=\varepsilon\delta_{\sigma}+(1-\varepsilon)\delta_{\tau}\]

と書けるとき、の対戦相手の戦略は確率でであり、確率でである。行列ゲームは回行われるが、我々は個体群には無限に多い個体が含まれていると仮定していたので（前述）、が同一の個体と複数回ゲームを行う事はありえない。よって回の行列ゲームの平均利得\(\mathcal{E}(\sigma,\Pi)\)は明らかに

\[\mathcal{E}(\sigma,\varepsilon\delta_{\sigma}+(1-\varepsilon)\delta_{\tau})=
{1\over k}\sum_{i=1}^k \left(\varepsilon E(\sigma,\sigma) + (1-\varepsilon) E(\sigma,\tau)\right)
=E(\sigma,\varepsilon\sigma + (1-\varepsilon)\tau)\]

を満たす\[37\]。すなわち行列ゲームの場合は複数回のゲームの平均利得\(\mathcal{E}(\cdot,\cdot)\)と個体群戦略\(\Pi=\varepsilon\delta_{\sigma}+(1-\varepsilon)\delta_{\tau}\)で考えようが、一回の行列ゲームの利得\(E(\cdot,\cdot)\)と1個体の混合戦略\(\varepsilon\sigma + (1-\varepsilon)\tau\)で考えようが実質的な差はない。

しかし行列ゲーム以外のゲームではこのような単純な関係が成立するとは限らず、そもそも「２個体間の一回のゲームの利得」\(E(\cdot,\cdot)\)が定義できない場合もある（例えば、２個体間の闘争ではなくの全ての個体が闘争する場合）ので、本章で述べる一般的なフレームワークにおいて改めて進化的安定性の概念を定式化する必要がある。
\=== 進化的安定性の定義 === 以上の準備の元、進化的安定性の概念を一般化する。集合を一つ固定し、これを**戦略空間**と呼び、の元を**戦略**ないし**個体戦略**と呼ぶ。そして任意の戦略に対し、形式和

\[\sum_{i=1}^m\varepsilon_i\delta_{\sigma_i}\]　　\((0\le \varepsilon \le 1, ~~ \textstyle\sum_i\varepsilon_i=1)\)

を**個体群戦略**と呼ぶ。さらに戦略と個体群戦略\(\textstyle\sum_{i=1}^n\varepsilon_i\delta_{\sigma_i}\)の組に**利得**と呼ばれる実数を対応させる関数

\[\mathcal{E}~:~(\tau,{\textstyle \sum_{i=1}^n\varepsilon_n\delta_{\sigma_n}}) \mapsto\mathbf{R}\]

を一つ固定し、この関数を**利得関数**と呼ぶ。直感的には\(\mathcal{E}\)の第一変数の個体戦略を取るある個体が、個体群戦略\(\textstyle\sum_{i=1}^n\varepsilon_i\delta_{\sigma_i}\)を取る個体群の中で（一般には複数回）闘争したときのが得られる利得（ないしその平均値）が\(\mathcal{E}(\tau;{\textstyle \sum_{i=1}^n\varepsilon_n\delta_{\sigma_n}})\)になるという事である。

以上のフレームワークにおいて、**ゲーム**は戦略空間と利得関数\(\mathcal{E}\)の組\((X,\mathcal{E})\)として定義される。

ゲーム\((X,\mathcal{E})\)に関する進化的安定性は以下のように定義される\[38\]：上の定義でははに依存する事を許容しているが、に依存しないが取れる場合には、**一様な侵入障壁をもつ進化的安定性**（*ESS with uniform invasion barrier*）と呼ぶ\[39\]。

\(\mathcal{E}\)の定義より、任意の戦略\(\sigma\in X\)は\(\mathcal{E}(\sigma,\cdot)\)のように個体戦略として\(\mathcal{E}\)の第一変数としてする事も、\(\mathcal{E}(\cdot,\delta_{\sigma})\)のように個体群戦略として\(\mathcal{E}\)の第二変数として登場する事も可能である。したがって「オス」と「メス」のように立場の異なる個体が存在したとしても、第一変数を「オス」の戦略、第二変数を「メス」の戦略といったふうに2つの変数を使い分ける事はできない。すなわち前述したように、立場の異なる個体間の非対称なゲームに対する進化的安定性を上記の定義では記述できず、何らかの「対称化」の操作を行う事によって非対称ゲームを記述する必要がある。対称化に関しては後の章でより詳しく説明する。

### \(\mathcal{E}\)の性質

すでに述べたように行列ゲームでは

\[\mathcal{E}(\sigma,\varepsilon\delta_{\sigma}+(1-\varepsilon)\delta_{\tau})
=E(\sigma,\varepsilon\sigma + (1-\varepsilon)\tau)\]

という単純な関係があり、しかも行列ゲームの利得関数は行列を用いて簡単に表記できるので、線形性

\[\mathcal{E}(\sum_i\nu_i\sigma_i,\sum_j\varepsilon_j\delta_{\tau_j})
=\sum_{i,j}\nu_i\varepsilon_j\mathcal{E}(\sigma_i,\delta_{\tau_j})\]

が成立した。

しかしこうした性質は行列ゲーム以外のゲームでは必ずしも成立するとは限らない。実際我々は現段階では一般のゲームにおける\(\mathcal{E}\)には一切仮定をおいていない為、線形性どころか連続性すら成り立つとは限らない。

このため行列ゲームに対して示した性質は一般のゲームに対しては無条件に成り立つとは限らず、線形性など何らかの仮定をおいた上でこうした性質（の類似物）を示す必要がある。

そこで本節では、線形性など\(\mathcal{E}\)に関する性質をいくつか導入し、これらの性質を元に進化的安定性の満たす性質を示す。
\==== 個体群戦略に関する線形性と個体戦略に関する線形性 ==== 本節では\(\mathcal{E}\)の線形性の概念を定義する。}</math>に対し、\(\mathcal{E}\)が

  -
    \(\mathcal{E}\left(\tau,\textstyle\sum _{i=1}^m\varepsilon _{i}\delta _{\sigma _{i}}\right)=\textstyle\sum _{i=1}^m\varepsilon _{i}\mathcal{E}(\tau,\delta _{\sigma _{i}})\)

を満たすとき、\(\mathcal{E}\)は**個体群戦略に関して線形**である、もしくは**右線形**であるという\[40\]。|name=（個体群戦略に関する線形性）}}

行列ゲームにおける混合戦略のように、戦略空間上に線形和が定義できる場合には、左線形性も同様に定義できる：}</math>に対し、\(\mathcal{E}\)が

  -
    \(\mathcal{E}\left(\textstyle\sum_{j=1}^m u_j\tau_j,\sum _{i=1}^{\ell}\varepsilon _{i}\delta _{\sigma _{i}}\right)=\textstyle\sum_{j=1}^m u_j\mathcal{E}\left(\tau_j,\textstyle\sum _{i=1}^{\ell}\varepsilon _{i}\delta _{\sigma _{i}}\right)\)

を満たすとき、\(\mathcal{E}\)は**個体戦略に関して線形**である、もしくは**左線形**であるという\[41\]。|name=（個体戦略に関する線形性）}}

これら２つの性質は行列ゲームの場合は明らかに満たされる。

多くのゲームにおいて、戦略空間は行列ゲームの場合と同様、何らかの混合戦略全体の空間であり、混合戦略\(\sigma=(p_i)_{i=1,\ldots,n}\)の利得は\(\mathcal{E}(\sigma,\Pi)=\sum_ip_i\mathcal{E}(i,\Pi)\)のように純粋戦略の利得の期待値として定義されるので、個体戦略に対する線形性は多くのゲームで成立する\[42\]。

それに対し個体群戦略に対する線形性は満たさないゲームも多く、例えば以下の３つの状況では満たされない事が多い：

  - ゲームが１：１の闘争でないとき\[43\]
  - （１：１の闘争であったとしても）同じ個体と複数回闘争しなければならないとき\[44\]
  - 取りうる戦略が連続量であるとき\[45\]

2013年現在、「線形性が満たされないゲームに関する一般的な理論はまだ十分に発展しているとは言い難い」\[46\]状況にあり、個別のゲームに応じた議論を行う必要がある。

#### 多型-単型同値性

行列ゲームにおける戦略空間は混合戦略全体の集合なので、戦略同士の線形和が定義できる。このように戦略空間上に何らかの和の概念が定義できている場合、以下の概念を定式化できる：}</math>に対し、\(\mathcal{E}\)が

  -
    \(\mathcal{E}\left(\tau,\textstyle\sum _{i=1}^{m}\varepsilon _{i}\delta _{\sigma _{i}}\right)=\mathcal{E}(\tau,\delta _{\textstyle\sum _{i=1}^{m}\varepsilon _{i}\sigma _{i}})\)

を満たすとき、\(\mathcal{E}\)は**多型-単型同値**(polymorphic-monomorphic equivalent)であるという\[47\]。|name=（多型-単型同値性）}}多型-単型同値性は行列ゲームでは明らかに成立する：

\[\mathcal{E}(\sigma,\sum_i\varepsilon_i\delta_{\sigma_i})
=E(\sigma,\sum_i\varepsilon_i\sigma_i)
=\mathcal{E}(\sigma,\delta_{\sum_i\varepsilon_i\sigma_i})\]

多型-単型同値性の直観的な意味を説明する。の式の左辺では\(\mathcal{E}\)の第２変数が\(\textstyle\sum_{i=1}^n\varepsilon_i\delta_{\sigma_i}\)であるので、個体群の中には戦略\(\sigma_1\)を取る個体が割合\(\varepsilon_1\)だけ存在し、...、戦略\(\sigma_n\)を取る個体が割合\(\varepsilon_n\)だけ存在するという状況を左辺は意味している。一方右辺では\(\mathcal{E}\)の第２変数が\(\textstyle\delta_{\sum_{i=1}^n\varepsilon_i\sigma_i}\)であるので、個体群に属する全ての個体が全く同一の戦略\(\textstyle\sum_{i=1}^n\varepsilon_i\sigma_i\)を取っている状況を右辺は意味している。

多型-単型同値性は\(\sigma_1,\ldots,\sigma_n\)が純粋戦略であるケースを考えると理解しやすい。上で述べた事から、の式の左辺は純粋戦略\(\sigma_1\)を取る個体が割合\(\varepsilon_1\)だけ存在し、...、純粋戦略\(\sigma_n\)を取る個体が割合\(\varepsilon_n\)だけ存在するという状況である。すなわち全ての個体は何らかの純粋戦略を取っており、個体毎にどの純粋戦略を取るのかが決まっている状況である。これは例えば、遺伝的多型により、個体が生まれた段階でどの純粋戦略を取るのかが決まる場合がこの状況に相当する。

一方、の式の右辺は、全ての個体が全く同一の混合戦略\(\textstyle\sum_{i=1}^n\varepsilon_i\sigma_i\)を取っている状況である。これは例えば、「混合戦略\(\textstyle\sum_{i=1}^n\varepsilon_i\sigma_i\)を取る事」が遺伝的に単型な形で刷り込まれており、ゲーム開始の段階でランダムに\(\sigma_1,\ldots,\sigma_n\)のうちどれかを行う場合がこの状況に相当する。

多型-単型同値性はこの多型のケースと単型のケースが\(\mathcal{E}\)の第一変数の戦略を取る個体の平均利得という観点から見るとこの「多型」の状況と「単型」の状況に差がない事を意味する。

の闘争相手がゲームのたびに個体群から毎回ランダムに選ばれるケース（個体群の個体数は無限大なのでこれは闘争相手が毎回異なる事を意味する）における繰り返し行列ゲームの場合には、明らかに多型-単型同値性が成立する。しかしゲームによっては多型-単型同値性が成り立たないものもあり、次章以降でそうしたゲームについて見る。

### 進化的安定性の簡便な特徴づけ

行列ゲームにおける進化的安定性の概念が均衡条件と安定条件により特徴づけられる事をで見た。この定理は本章で述べた一般的なゲームに関する進化的安定性に対しては常に成立するわけではないが、適切な条件下ではの類似物を示す事が可能である。

前節までと同様、を戦略空間とし、\(\mathcal{E}(\cdot,\cdot)\)を上の個体戦略と個体群戦略に「利得」とよばれる実数値を対応させる関数とする。さらに戦略\(\sigma_*,\sigma\in X\)を固定し、**インセンティブ関数**\(h_{\sigma_*,\sigma}\)を

\[h_{\sigma_*,\sigma}~:~[0,1] \to \mathbf{R},
~\varepsilon \mapsto
\mathcal{E}(\sigma_*,\varepsilon\delta_{\sigma_*}+(1-\varepsilon)\delta_{\sigma})
- \mathcal{E}(\sigma,\varepsilon\delta_{\sigma_*}+(1-\varepsilon)\delta_{\sigma})\]

により定義する\[48\]。ここで\([0,1]\)は0以上1以下の実数全体の集合である。このとき次が成立する\[49\]：多くの生物学上の応用では、

\[{\partial \over \partial \varepsilon}h_{\sigma_*,\sigma}(0)= 0\]

を満たす\((\sigma_*,\sigma)\in X^2\)の集合は[零集合](../Page/測度論.md "wikilink")（≒面積0の集合）であるので、上記偏微分が0になる確率が0である事を多くのケースでは仮定できる（これをgeneric payoff assumptionという\[50\]）\[51\]。この仮定の元では進化的安定性は均衡条件と安定条件が両方成立する事と[ほとんど至る所で同値である](https://ja.wikipedia.org/wiki/ほとんど_\(数学\) "wikilink")。

#### との関係

本節ではとの関係を見るため、を行列ゲームに適用してみる。すでに述べたように行列ゲームでは

\[\mathcal{E}(\sigma,\varepsilon\delta_{\sigma}+(1-\varepsilon)\delta_{\tau})
=E(\sigma,\varepsilon\sigma + (1-\varepsilon)\tau)\]

であり、\(E(\cdot,\cdot)\)は右線形かつ左線形であるので、インセンティブ関数\(h_{\sigma_*,\sigma}\)は

\[h_{\sigma_*,\sigma}(\varepsilon)=\varepsilon E(\sigma_*-\sigma,\sigma_*) + (1-\varepsilon)E(\sigma_*-\sigma,\sigma)\]

である。よってにおける偏微分は

\[{\partial\over\partial \varepsilon}h_{\sigma_*,\sigma}(0)= E(\sigma_*-\sigma,\sigma_*-\sigma)\]

である。の安定条件の仮定\(\mathcal{E}(\sigma_*,\delta_{\sigma_*})=\mathcal{E}(\sigma,\delta_{\sigma_*})\)が成り立つ条件下では

\[E(\sigma_*-\sigma,\sigma_*-\sigma)=E(\sigma,\sigma)-E(\sigma_*,\sigma)\]

であるので、の安定条件はのそれと一致する。すなわちは、\(E(\sigma_*-\sigma,\sigma_*-\sigma)\neq 0\)を要求する事以外はと一致している。

### 個体群ゲーム(Population Game)

本章では**個体群ゲーム**というゲームを定義し、このゲームにおける進化的安定性の性質を述べる。

#### モチベーション

多くのゲームにおいて、戦略空間は行列ゲームの場合と同様、有限個の純粋戦略を混合した混合戦略全体の空間であり、混合戦略\(\sigma=(p_i)_{i=1,\ldots,n}\)の利得は

\[\mathcal{E}(\sigma,\Pi)=\sum_ip_i\mathcal{E}(i,\Pi)\]

のように純粋戦略の利得の期待値として定義される。ここでさらに多型-単型同値が成り立てば、任意の個体群戦略\(\Pi=\textstyle\sum_j\varepsilon_j\delta_{\sigma_j}\)は

\[\Pi=\delta_{\tau}\]

のようにたった一つの混合戦略\(\tau=\textstyle\sum_i\varepsilon_j\sigma_j\)により記述できる。ここで\(f_i(\tau):=\mathcal{E}(i,\tau)\)と定義すれば、

\[\mathcal{E}(\sigma,\delta_{\tau})=\sum_ip_if_i(\tau)\]

が成立する事になる。利得関数\(\mathcal{E}\)が上式のように書けるゲームが**個体群ゲーム**である。

#### 定義

以上をまとめると次のようになる\[52\]\[53\]。なお以下では()式で定義される集合であり、直観的には有限個（個）の純粋戦略を混合した混合戦略全体の空間を意味する。

#### 応用例：場を通じる型(playing the field)

行列ゲームでは個体が個体群からランダムに選ばれた個体と１：１の闘争を行うケース（**１対１型**\[54\]）を想定していた。しかし生物学における実際の状況は、このようなは１：１の闘争を行うものばかりではなく、が個体群に属する全ての他の個体と闘争しなければならないものも存在する。

このようなの全ての他の個体との闘争を行われる状況を**場を通じる型**\[55\]（**playing the field**\[56\]）という。例えば植物が種を飛散させる状況下では、近くにいる他の全ての個体と土地を争わなければならないので、場を通じる型の類型に属する\[57\]。

場を通じる型のセッティングでは、そもそも１：１の闘争は行われないので行列ゲームのような１：１の闘争を前提とした利得関数は定義できず、を使わずに直接\(\mathcal{E}(\sigma,\Pi)\)を定義する必要がある事になる。

この際利用できるのが、個体群ゲームのフレームワークである\[58\]\[59\]。で述べたように、左線形性や多型-単型同値などの条件が成立しさえすれば、場を通じる型の状況を個体群ゲームとして記述できるので、個体群ゲームは有益な概念である。

#### 性質

以下の２つの性質が成立する\[60\]： ここでは式により定義される\(X=\Delta_n\)上の距離であるが、と同様、と同一の位相を定める距離であれば他のものでもよい。

### 非対称なゲーム

これまで全ての個体が対等である状況を考察してきたが、実際の生物学では「オス vs. メス」、「テリトリーの所有者 vs. テリトリーへの侵入者」、「体の大きい個体 vs. 体の小さい個体」のように２つの**非対称**な立場がある個体同士が闘争する。しかし前章でも述べたように、進化ゲーム理論ではこうした非対称なゲームに関しては何らかの「対称化」を施すことにより、対象なゲームとして進化的安定性を定義する\[61\]。

#### 非対称なゲームの記述

本節では非対称なゲームを定式化し、対称化を方法を述べる。今各個体には2つの**立場\[62\]**(role)があり、どちらの立場にいるかにより取れる戦略が異なるものとする。立場0、立場1にいる時に取れる戦略全体の集合をそれぞれ、と表記する。このとき、非対称なゲームの戦略空間は

\[X_0\times X_1\]

である。戦略空間の元\((\sigma,\tau)\in X_0\times X_1\)の直観的意味は「もし自分が立場0であれば戦略を取り、立場1であれば戦略を取る」というものである。

このゲームにおける**個体群戦略**は\((\sigma_1,\tau_1),\ldots,(\sigma_m,\tau_m)\in X_0\times X_1\)と\(\varepsilon_1,\ldots,\varepsilon_m \in [0,1]\)（\(\varepsilon_1+\cdots+\varepsilon_m =1\)）を用いて

\[\sum_{i=1}^m \varepsilon_i\delta_{(\sigma_i,\tau_i)}\]

と書けるものを指す。ゲームは非対称であるので、利得関数も自分が立場0にいるときと立場1にいるときで異なる。自分が立場\(k =0,1\)にいるときの利得関数を

\[\mathcal{E}_k(\xi,\textstyle\sum_{i=1}^m\varepsilon_i\delta_{\sigma_i})\]

と書く。ここではの元であり、\(\sigma_1,\ldots,\sigma_m\)は\(X_{1-k}\)の元である。非対称なゲームは組

\[((X_0,\mathcal{E}_0),(X_1,\mathcal{E}_1))\]

により定義される。

#### 対称化

以上のように定義された非対称なゲーム\(((X_0,\mathcal{E}_0),(X_1,\mathcal{E}_1))\)に対し、利得関数の対称化を行う。このために記号を導入する。個体群戦略

\[\Pi=\sum_i\varepsilon_i\delta_{(\sigma_i,\tau_i)}\]

に対し、

\[\pi_0(\Pi)=\sum_i\varepsilon_i\delta_{\sigma_i}\]、

\[\pi_1(\Pi)=\sum_i\varepsilon_i\delta_{\tau_i}\]

と書くことにする。関数

\[\rho~:~X_0\times X_1 \to [0,1]\]

を一つ固定するとき、利得関数の組\((\mathcal{E}_0,\mathcal{E}_1)\)を\(\rho\)**により対称化した利得関数**を

\[\mathcal{E}((\xi_0,\xi_1),\Pi)=
\rho(\xi_0,\xi_1)\mathcal{E}_0(\xi_0,\pi_1(\Pi))
+
(1-\rho(\xi_0,\xi_1))\mathcal{E}_1(\xi_1,\pi_0(\Pi))\]

により定義する\[63\]。直観的には\(\rho(\xi_0,\xi_1)\)は個体戦略\((\xi_0,\xi_1)\in X_0\times X_1\)を取っている個体が立場0になる確率である。

なお、対称化が定数関数

\[\rho=\text{const.}\]

を用いて行われた場合、この対称化は**戦略-立場独立**(strategy-role independent\[64\])であるという。

#### 進化的安定性

非対称なゲームに関する進化的安定性は、対称化したゲームの進化的安定性により定義する。すなわち個体戦略\((\xi_*,\nu_*)\in X_1\times X_2\)が**進化的安定**であるとは、戦略空間が\(X_1\times X_2\)であり利得関数が\(\mathcal{E}\)であるゲームに関して進化的安定である事を指す\[65\]。もちろんこの進化的安定性の概念は関数\(\rho\)に依存しており、\(\rho\)が異なれば進化的安定性の概念も異なる。

#### 一般化

これまで非対称なゲームを考察するに当たって、同じ立場にいる個体同士が闘争しないことを暗に仮定していた。すなわち、自分が立場0にいる時は立場1にいる個体と闘争し、立場1にいるときは立場0にいる個体と闘争する、という事である。しかし一般にはこれが成立しない場合もある。この場合には、4つの利得関数\(\mathcal{E}_{00},~
\mathcal{E}_{10},~
\mathcal{E}_{01},~
\mathcal{E}_{11}\)を考え、

\[\mathcal{E}((\xi_0,\xi_1),\Pi)=
\sum_{i,j}
\rho_{i,j}(\xi_0,\xi_1)\mathcal{E}_{i,j}(\xi_i,\pi_j(\Pi))\]

として対称化をはかる\[66\]。ここで\(\rho_{i,j}~:~X_0\times X_1 \to [0,1]\)は\(\textstyle\sum_{i,j}\rho_{i,j}(\xi_0,\xi_1)=1\)を満たす関数である。

直観的には\(\mathcal{E}_{ij}\)は自分が立場、闘争相手が立場にいるときの利得関数で、\(\rho_{i,j}(\xi_0,\xi_1)\)は自分が個体戦略\((\xi_0,\xi_1)\in X_0\times X_1\)を取っている際に、自分が立場、闘争相手が立場になる確率である。



## レプリケーター方程式(Replicator Equation)と進化的安定性

**レプリケーターダイナミクス**(replicator dynamics、**自己複製子動学**\[67\])は与えられた個体群内の各個体が取る戦略の頻度分布（すなわち、前章までの言葉で言えば個体群戦略）がどのように時間発展するかを定式化したモデルで、このモデルにおいて頻度分布の時間発展を記述する方程式を**レプリケーター方程式**([replicator equation](https://ja.wikipedia.org/wiki/:En:Replicator_equation "wikilink"))という。本節では「離散型」、「連続型」の２種類のレプリケーター方程式を紹介し、行列ゲームにおいて連続レプリケーター方程式の解の収束先と進化的安定性の関係を述べる。



### レプリケーター方程式

本節では以下の２種類のレプリケーター方程式を紹介する：

  - **離散レプリケーター方程式**（discrete replicator equation）：無性生殖する個体群の戦略の頻度分布を（オーバーラップのない）「世代」という離散的な時間で記述できると仮定した場合の方程式\[68\]
  - **連続レプリケーター方程式**（continuous replicator equation）：個体数が十分大きいため世代がオーバーラップし、連続的な時間によって（無性生殖する）個体群の戦略の頻度分布を記述できると近似した場合における方程式\[69\]

#### 離散レプリケーター方程式

離散レプリケーター方程式を定式化するために、以下のような個体群を考える：

1.  個体群の構成が世代によって記述でき、各世代にはオーバーラップがない。すなわち世代に生きた個体はには全て死滅し、世代は世代に生まれた個体の子供のみから構成される\[70\]。
2.  個体群内の各個体は有限個の純粋戦略のいずれかを取り、混合戦略は取らない\[71\]
3.  この個体群は無性生殖によって子孫を残す\[72\]
4.  この個体群には突然変異が生じないもの\[73\]

この個体群において世代で（純粋）戦略を取る個体の割合を\(p_i(t)\)と表記すると、この個体群における戦略の分布

\[\mathbf{p}(t)=(p_1(t),\ldots,p_n(t))\]

と記述できる\[74\]。

この個体群で戦略を取る各個体の利得を\(f_i(\mathbf{p}(t))\)と表記し、\(f_i(\mathbf{p}(t))\)に関して以下の仮定を置く：

  - この個体群で世代において戦略を取る個体が残す事ができる子供の数は利得\(f_i(\mathbf{p}(t))\)に等しい

このように仮定すると、個体群のうち割合\(p_i(t)\)の個体が、それぞれ\(f_i(\mathbf{p}_i(t))\)の子供を残すのだから、世代において戦略を取る個体の比率は

\[p_1(t)f_1(\mathbf{p}(t))~:~\cdots~:~p_n(t)f_n(\mathbf{p}(t))\]

となる。ここで我々は

  - 仮定3.により、（突然変異を例外とすれば）子供は親と同じ遺伝子を持つため、親と同じ戦略を取り
  - 仮定4.により突然変異が起こらない

事を利用した。以上より世代世代において戦略を取る個体の割合は、以下の**離散レプリケーター方程式**に従う\[75\]：

ここで

\[\bar{f}(\mathbf{p}(t))=\sum_jp_j(t)f_j(\mathbf{p}(t))\]

である\[76\]。

#### 離散レプリケーター方程式の分母

分数は分母だと意味を持たないので、最後に離散レプリケーター方程式の分母について触れておく。離散レプリケーター方程式の直観的な意味から、利得の期待値\(f_i(\mathbf{p}(t))\)は

\[f_i(\mathbf{p}(t))\ge 0\]

を満たす必要がある。また\(p_i(t)\)は割合であったので\(1\ge p_i(0) \ge 0\)であり、数学的帰納法により、離散レプリケーター方程式の分母が0になる世代の直前までは

\[1\ge p_i(t) \ge 0\]

が成立する事も示せる。したがって離散レプリケーター方程式の分母が0になる場合、すなわち

\[\bar{f}(\mathbf{p}(t))=\sum_jp_j(t)f_j(\mathbf{p}(t))=0\]

の場合は、正の数の和が0である事になるので、

\[p_1(t)f_1(\mathbf{p}(t))=\cdots=p_n(t)f_n(\mathbf{p}(t))=0\]

が成立する。これは各に対し、\(p_i(t)\)か\(f_i(\mathbf{p}(t))\)のいずれかが0である事を意味する。\(p_i(t)=0\)であれば、純粋戦略を取る個体は絶滅した事になるので、任意のに対し、\(p_i(s)=0\)である。また\(f_i(\mathbf{p}(t))=0\)であれば、純粋戦略を取る個体が世代で残せた子供の数\(f_i(\mathbf{p}(t))\)が0である事を意味するので、やはり任意のに対し、\(p_i(s)=0\)である。結局、離散レプリケーター方程式の分母が0になるという事は個体群の全ての個体が絶滅した場合に相当する。

#### 連続レプリケーター方程式

連続レプリケーター方程式を定式化する為、離散レプリケーター方程式の節の2～4の仮定と以下の1'の仮定を満たす個体群を考える：

  -
    1'. 個体数が十分大きいため世代がオーバーラップし、連続的な時間によって個体群の戦略の頻度分布を記述できる\[77\]

前節同様、（純粋）戦略を取る個体の割合を\(p_i(t)\)と表記し、\(\mathbf{p}(t)=(p_1(t),\ldots,p_n(t))\)とし、この個体群で戦略を取る各個体の利得を\(f_i(\mathbf{p}(t))\)と表記する。

利得\(f_i(\mathbf{p}(t))\)に関して前節のものと似た以下の仮定を置く：

  - この個体群で時刻において戦略を取る個体の増加率は利得\(f_i(\mathbf{p}(t))\)に等しい

個体群に属する個体数が十分に大きいと仮定しているので、個体数はに関して微分可能な連続量であるとみなして差し支えないので\[78\]、\(N_i(t)=p_i(t)N(t)\)とすると、上述の仮定から、

\[{\mathrm{d}\over \mathrm{d} t}N_i(t)=f_i(\mathbf{p}(t))N_i(t)\] …

が成立する\[79\]。記号を簡単にするため、時間微分を\(\dot{N}_i(t)\)のようにドットで書くことにすると、と\(N_i(t)=p_i(t)N(t)\)より、

\[\dot{p}_i(t)={\mathrm{d}\over \mathrm{d} t}\left({N_i(t)\over N(t)}\right)
={\dot{N}_i(t)-p_i(t)\dot{N}(t)\over N(t)}\]\(={f_i(\mathbf{p}(t))N_i(t)-p_i(t)\dot{N}(t)\over N(t)}
=p_i(t)\left(f_i(\mathbf{p}(t))-{\dot{N}(t)\over N(t)}\right)\)

が成立し、しかもから

\[{\dot{N}(t)\over N(t)}
=\sum_j{\dot{N}_j(t)\over N(t)}
=\sum_jf_j(\mathbf{p}(t))p_j(t)\]

でもあるので、以下の**連続レプリケーター方程式**が成立する\[80\]：

ここで

\[\bar{f}(\mathbf{p}(t))=\sum_jp_j(t)f_j(\mathbf{p}(t))\]

である\[81\]。なお、適切な条件下では離散レプリケーター方程式の極限として連続レプリケーター方程式が得られる事が知られている\[82\]\[83\]。

### 行列ゲームの連続レプリケーター方程式と進化的安定性

本節の目標は、行列ゲームに対し、レプリケーター方程式の解が進化的安定な状態へと収束する条件を見る事である。なお、行列ゲーム以外のゲームに関してはこのような収束性は成り立つとは限らない\[84\]。その理由の一端は、（後述するように）レプリケーター方程式が純粋戦略のみをとる個体群を想定しているのに対し、進化的安定性の定義では混合戦略をも考慮する事が多いからである\[85\]。したがって単型-多型同値が成り立たない系では、レプリケーター方程式による解析と進化的安定性とが一致しない可能性がある\[86\]。

#### 行列ゲームにおける連続レプリケーター方程式

まず行列ゲームに対する連続レプリケーター方程式を記述する。の行列\(A=(a_{ij})_{i,j}\)と行の縦ベクトルに対し、積の第行を

\[(A\mathbf{p})_i\]

という記号で書くことにすると、利得関数が

\[E(i,j)=a_{ij}\]

と記述できる行列ゲームにおいて、純粋戦略を取る個体の利得の期待値\(f_i(\mathbf{p}(t))\)は明らかに

\[f_i(\mathbf{p}(t))=(A\mathbf{p}(t))_i\]

なので、行列ゲームにおける連続レプリケーター方程式は

\[{\mathrm{d}p_i\over \mathrm{d}t}(t)=((A\mathbf{p}(t))_i-\mathbf{p}(t)^{T}A\mathbf{p}(t))p_i(t)~~~~~~\text{for }i=1,\ldots,n\]　 …

と記述できる\[87\]\[88\]。ここで\(\mathbf{p}(t)^{T}\)は\(\mathbf{p}(t)\)を[転置した横ベクトルである](../Page/転置行列.md "wikilink")。

本節ではと進化的安定性の関係性を調べるため、に関する性質を述べる。まず\(p_i(t)\)は純粋戦略を取る個体の割合であったから、\(\mathbf{p}(t)\)の初期値\(\mathbf{p}(0)\)は

\[\Delta_n=\left\{(p_i)_{i=1,\ldots,n} \Bigg| 0\le p_1,\ldots,p_n \le 1,~ \sum_{i=1}^np_i=1 \right\}\]　　　　...(、再掲)

の元である。は行列によって記述できる[常微分方程式](../Page/常微分方程式.md "wikilink")であるので、（少なくとも初期値の近傍では）解が存在し、しかもその解は一意である（[Picard–Lindelöf theorem](https://ja.wikipedia.org/wiki/:En:Picard–Lindelöf_theorem "wikilink")）\[89\]。

解の一意性から、における時間発展で２つの超平面

\[\{(p_i)_{i=1,\ldots,n} | \textstyle \sum_{i=1}^np_i=1 \}\]

\[\{(p_i)_{i=1,\ldots,n} | p_i=0 \}\]

### 行列ゲームの混合戦略に対する連続レプリケーター方程式と進化的安定性

#### 混合戦略に対する連続レプリケーター方程式

これまで我々は、着目している個体が純粋戦略を取る場合の連続レプリケーター方程式に関して考察してきたが、より一般に、有限個の混合戦略\(\mathbf{q}_1,\ldots,\mathbf{q}_m\in \Delta_n\)を取る個体がそれぞれ割合\(x_1(t),\ldots,x_m(t)\)で存在する個体群に対する連続レプリケーター方程式を考える事もできる\[99\]：

\[{\mathrm{d}x_i\over \mathrm{d}t}(t)=\left(\mathbf{q}_i-\mathbf{q}_{\mathbf{x}}(t)\right)^{T}A\mathbf{q}_{\mathbf{x}}(t))x_i(t)~~~~~~\text{for }i=1,\ldots,m\]　 …

ここで\(\mathbf{q}_{\mathbf{x}}(t)\)は**平均混合戦略\[100\]**

\[\mathbf{q}_{\mathbf{x}}(t)=\textstyle\sum_{j=1}^m x_j(t)\mathbf{q}_j\]　 …

である\[101\]。の導出はのそれと同様なので省略する。

#### 進化的安定性との関係

においてであれば、\(x=x_1(t)\)、\(\mathbf{q}=\mathbf{q}_1\)、\(\hat{\mathbf{q}}=\mathbf{q}_2\)と略記すると、\(x_2=1-x\)なので、に登場する本の式はいずれも

\[\dot{x}
=x(1-x)\{x(\mathbf{q}-\hat{\mathbf{q}})^{T}A\mathbf{q}+(1-x)(\mathbf{q}-\hat{\mathbf{q}})^{T}A\hat{\mathbf{q}})\}\]　 …　

に同値である事が簡単な計算から確かめられる\[102\]。このとき、次が成立する事が知られている\[103\]：</math>が混合戦略\(\mathbf{q}\)に対して進化的安定である必要十分条件はが漸近的に安定である事である。|name=}}

### 行列ゲームの混合戦略に対する離散レプリケーター方程式と進化的安定性

#### 行列ゲームの純粋戦略に対する離散レプリケーター方程式

混合戦略に関して考察する前に、まず本節では行列ゲームの純粋戦略に対する離散レプリケーターを導出する。純粋戦略を取る個体の割合を\(p_i(t)\)と表記し、\(\mathbf{p}(t)=(p_1(t),\ldots,p_n(t))\)とし、この個体群で戦略を取る各個体の利得を\(f_i(\mathbf{p}(t))\)と表記する。

の行列\(A=(a_{ij})_{i,j}\)を用いて利得関数が

\[E(i,j)=a_{ij}\]

と書ける行列ゲームにおいて、純粋戦略を取る個体の利得の期待値\(f_i(\mathbf{p}(t))\)は明らかに

\[f_i(\mathbf{p}(t))=(A\mathbf{p}(t))_i\]

なので、これを利用して離散レプリケーター方程式の具体的な形を書き下す事ができる。より一般に各個体が行列ゲームの利得以外に「背景利得」(background payoff)を得られる場合、すなわち

\[f_i(\mathbf{p}(t))=(A\mathbf{p}(t))_i+\beta\]

の場合には、離散レプリケーター方程式の具体的な形は

\[p_i(t+1)={(A\mathbf{p}(t))_i +\beta \over \mathbf{p}(t)^TA\mathbf{p}(t) +\beta }p_i(t)~~~~~~\text{for }i=1,\ldots,n\]　　 …　

である\[104\]\[105\]。

#### 行列ゲームの混合戦略に対する離散レプリケーター方程式

連続レプリケーター方程式の「純粋戦略版」であるから「混合戦略版」のを導いたのと同様の方法で、離散レプリケーター方程式の「混合戦略版」を「純粋戦略版」であるから導くことができる。

すなわち、有限個の混合戦略\(\mathbf{q}_1,\ldots,\mathbf{q}_m\in \Delta_n\)を取る個体が世代においてそれぞれ割合\(x_1(t),\ldots,x_m(t)\)だけ存在する個体群を考え、\(\mathbf{x}(t)=(x_1(t),\ldots,x_m(t))^T\)とするとき、混合戦略\(\mathbf{q}_i\)を取る個体の利得の期待値\(f_i(\mathbf{x}(t))\)は**平均混合戦略\[106\]**

\[\mathbf{q}_{\mathbf{x}}(t)=\textstyle\sum_{j=1}^m x_j(t)\mathbf{q}_j\]　 …

を用いて

\[f_i(\mathbf{x}(t))
=\sum_jx_j(t)E(\mathbf{q}_i(t),\mathbf{q}_j(t)))+\beta
=\mathbf{q}_i(t)^TA\mathbf{q}_{\mathbf{x}}(t)+\beta\]

と表記できるので、

\[x_i(t+1)={\mathbf{q}_i^TA\mathbf{q}_{\mathbf{x}}(t)+\beta \over\mathbf{q}_{\mathbf{x}}^TA\mathbf{q}_{\mathbf{x}}(t)+\beta}x_i(t)~~~~~~\text{for }i=1,\ldots,m\]　　 …　

となる。なおより明らかに比の等式

\[{x_1(t+1)\over x_1(t)}
~:~\cdots~:~
{x_m(t+1)\over x_m(t)}
=
E(\mathbf{q}_1,\mathbf{q}_{\mathbf{x}}(t))
~:~\cdots~:~
E(\mathbf{q}_m,\mathbf{q}_{\mathbf{x}}(t))\]　　 …　

が成立する。ここで

\[E(\mathbf{p},\mathbf{q})
=\mathbf{p}^TA\mathbf{q}+\beta\] …　

である。上の比の等式は左辺に登場する分母\(x_i(t)\)が0である場合は意味を持たないが、前節でも述べたのと同様の議論により、\(x_i(t)\)が0になるのは混合戦略\(\mathbf{q}_i\)を取る個体が個体群から絶滅した事を意味するので、以降のに関しては常に\(x_i(s)=0\)であるものと解釈する。

#### 進化的安定性との関係性

離散レプリケーター方程式と進化的安定性との関係を見るため、でであるケースを考え、\(x(t)=x_1(t)\)、\(\mathbf{q}=\mathbf{q}_1\)、\(\hat{\mathbf{q}}=\mathbf{q}_2\)と略記すると、\(x_2(t)=1-x(t)\)なので

\[{x(t+1)\over x(t)}
~:~
{1-x(t+1)\over 1-x(t)}
=
E(\mathbf{q},\mathbf{q}_{\mathbf{x}}(t))
~:~E(\hat{\mathbf{q}},\mathbf{q}_{\mathbf{x}}(t))\]　 …　

である\[107\]。ここではのように定義されており、

\[\mathbf{q}_{\mathbf{x}}(t)
=x(t)\mathbf{q}
+(1-x(t))\hat{\mathbf{q}}\]　 …

であり、の左辺の分母が0である場合の解釈は前節と同様であるものとする。また離散レプリケーター方程式の利得は子供の数を示していたので、

\[E(\mathbf{q},\mathbf{q}), ~\hat{E(\mathbf{q}},\hat{\mathbf{q}})
\ge 0\]　

が成立する事を仮定する。このとき、次が成立する\[108\]：</math>が進化的安定であり、\(\hat{\mathbf{q}}\)の \(\mathbf{q}\neq\hat{\mathbf{q}}\)に対する侵入障壁 \(\varepsilon_0\) と初期値 \(x(0)\)とが

\[x(0)<\varepsilon_0\] を満たせば、漸化式を満たす\(x(t)\)は\(t\to\infty\)のとき0に収束する。|name=}}

上述の定理は、個体群において\(\mathbf{q}\neq\hat{\mathbf{q}}\)を取る個体の割合が進化的安定戦略\(\hat{\mathbf{q}}\)の侵入障壁よりも小さい時は、世代を重ねる事で\(\mathbf{q}\)を取る個体の割合が0に収束していく事を意味する。

(t)) \\over E(\\mathbf{q}_{\\mathbf{x}}(t),\\mathbf{q}_{\\mathbf{x}}(t))}\\cdot x(t)

</math> 　…

が従う。これをさらに変形して

\[x(t)-x(t+1)
={E(\mathbf{q}_{\mathbf{x}}(t),\mathbf{q}_{\mathbf{x}}(t))
- E(\mathbf{q},\mathbf{q}_{\mathbf{x}}(t))
\over E(\mathbf{q}_{\mathbf{x}}(t),\mathbf{q}_{\mathbf{x}}(t))}\cdot x(t)\]　　…

を得る。の分母は

\[E(\mathbf{q}_{\mathbf{x}}(t),\mathbf{q}_{\mathbf{x}}(t))
=x(t)^2E(\mathbf{q},\mathbf{q})
+x(t)(1-x(t))E(\mathbf{q},\hat{\mathbf{q}})
+x(t)(1-x(t))E(\hat{\mathbf{q}},\mathbf{q})
+(1-x(t))^2E(\hat{\mathbf{q}},\hat{\mathbf{q}}) \ge 0\]

を満たし、\(E(\mathbf{q}_{\mathbf{x}}(t),\mathbf{q}_{\mathbf{x}}(t))
= 0\)の場合は既に述べたようにに対し\(x(s)=0\)であるので、以下

\[E(\mathbf{q}_{\mathbf{x}}(t),\mathbf{q}_{\mathbf{x}}(t))
>0\]

であるとしてよい。

次に我々は式を利用する事で、\(x(t)\)が[狭義単調減少であるための条件を調べる](https://ja.wikipedia.org/wiki/単調減少 "wikilink")。\([0,1]\)区間に属する\(x(t)\)が単調減少であれば[\(x(t)\)は何らかの非負の実数に収束するので](https://ja.wikipedia.org/wiki/単調収束定理 "wikilink")、我々が知りたかった\(x(t)\)の収束性に関する情報が得られるからである。

解析を簡単にするため、しばらくの間\(x(t)\neq 1\)を仮定する。

狭義単調減少性\(x(t+1)<x(t)\)を満たす必要十分条件は、式の右辺がによらず正になる事である。我々は式の分母\(E(\mathbf{q}_{\mathbf{x}}(t),\mathbf{q}_{\mathbf{x}}(t))\)が正である事をすでに示した。したがって式の右辺が正であるのは、\((E(\mathbf{q}_{\mathbf{x}}(t),\mathbf{q}_{\mathbf{x}}(t))
- E(\mathbf{q},\mathbf{q}_{\mathbf{x}}(t))
) x(t)\)の場合である。割合\(x(t)\)は\(0\le x(t)\le 1\)を満たしており、我々は\(x(t)\neq 0\)のケースを考えていたので、結局\(x(t+1)<x(t)\)となる必要十分条件は

\[E(\mathbf{q}_{\mathbf{x}}(t),\mathbf{q}_{\mathbf{x}}(t))
- E(\mathbf{q},\mathbf{q}_{\mathbf{x}}(t))
 >0\]　…　　…

となる事である。\(\tau_t\)の定義より、式の左辺は

\[(1-x(t))(E(\hat{\mathbf{q}},\mathbf{q}_{\mathbf{x}}(t))
- E(\mathbf{q},\mathbf{q}_{\mathbf{x}}(t)))
 >0\]

に等しい。前述のように\(0\le x(t)\le 1\)で、\(x(t)\neq 1\)を仮定していたので、が成立する必要十分条件は

\[E(\hat{\mathbf{q}},\mathbf{q}_{\mathbf{x}}(t))
- E(\mathbf{q},\mathbf{q}_{\mathbf{x}}(t))
 >0\]　

となる事である。再び\(\mathbf{q}_{\mathbf{x}}(t)\)の定義より、上式が成り立つ必要十分条件は

\[E(\hat{\mathbf{q}},(1-x(t))\hat{\mathbf{q}}+x(t)\mathbf{q})>E(\mathbf{q},(1-x(t))\hat{\mathbf{q}}+x(t) \mathbf{q})\]　…　

が成立する事である。

以上の議論から\(x(t)\neq 1\)であれば、\(x(t+1)<x(t)\)である必要十分条件はが成立する事である事がわかった。

よって特に、

  -
    「\(x(t)\neq 1\)かつが成立」⇒「\(x(t+1)<x(t)\)が成立」　　　…　

が結論づけられる。

さて、仮定より、\(x(0)\)は進化的安定戦略\(\hat{\mathbf{q}}\)の\(\mathbf{q}\neq \hat{\mathbf{q}}\)に対する参入障壁\(\varepsilon_0\)に対して\(0\le x(0)< \varepsilon_0\)を満たしていた。これはすなわち\(t=0\)に対しが成立する事を意味し、当然\(x(t)\neq 1\)でもあるので、より、\(x(1) < x(0)< \varepsilon_0\)が成立する。よって再び進化的安定性よりが成立し、\(x(1)\neq 1\)も成立するのでより、\(x(2)<x(1)<x(0)<\varepsilon_0\)も成立する。以下同様に考える事で[数学的帰納法](../Page/数学的帰納法.md "wikilink")により、\(x(t+1)<x(t)\)が任意のに対して成立する事がわかる。すなわち\(x(t)\)は狭義単調減少である。

[非負の単調減少列は収束するので](https://ja.wikipedia.org/wiki/単調収束定理 "wikilink")、\(x(t)\)はの極限\(x_\infty\)を持ち、\(0\le x_\infty<\varepsilon\)である。

\(0\le x_\infty<\varepsilon\)である事から、再び進化的安定性より

\[E(\hat{\mathbf{q}},(1-x_{\infty})\hat{\mathbf{q}}+x_{\infty}\mathbf{q})>E(\mathbf{q},(1-x_{\infty})\hat{\mathbf{q}}+x_{\infty} \mathbf{q})\]

である。したがって

\[\mathbf{r}_\infty :=(1-x_\infty)\hat{\mathbf{q}}+ x_\infty\mathbf{q}\]　

とすれば、

\[E(\hat{\mathbf{q}},\mathbf{r}_\infty)>E(\mathbf{q},\mathbf{r}_\infty)\]

である。

\(0\le x_\infty<\varepsilon\)より、

\[E(\mathbf{r}_\infty,\mathbf{r}_\infty)
=(1-x_\infty)E(\hat{\mathbf{q}},\mathbf{r}_\infty)+ x_\infty E(\mathbf{q},\mathbf{r}_\infty)
>(1-x_\infty)E(\mathbf{q},\mathbf{r}_\infty)+x_\infty E(\mathbf{q},\mathbf{r}_\infty)
= E(\mathbf{q},\mathbf{r}_\infty)\]　…　

である。一方の分母を払って極限を取ると

  -
    <math>E(\\tau_\\infty,\\tau_\\infty)\\cdot x_{\\infty}

\=E(\\sigma,\\tau_\\infty) \\cdot x_\\infty

</math>　

が成立する。\(x_\infty\neq 0\)であれば上式両辺を\(x_\infty\)で割る事ができるが、割った結果はと矛盾する。したがって\(x_t\)のにおける極限は\(x_\infty= 0\)である事が結論付けられる。|drop=yes}}
