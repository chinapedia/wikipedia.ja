**福井空港**（ふくいくうこう、[英語](../Page/英語.md "wikilink"):Fukui Airport）は、[福井県](../Page/福井県.md "wikilink")[坂井市](../Page/坂井市.md "wikilink")[春江町](../Page/春江町.md "wikilink")江留中にある[地方管理空港である](https://ja.wikipedia.org/wiki/日本の空港#地方管理空港 "wikilink")。 [通称](../Page/通称.md "wikilink")は春江空港。

## 概要

定期便の発着は[1976年](../Page/1976年.md "wikilink")以降ないが、自家用[プロペラ機](https://ja.wikipedia.org/wiki/プロペラ機 "wikilink")、[グライダー](../Page/グライダー.md "wikilink")などの発着が行われている\[1\]。

グライダースポーツが盛んに行なわれており\[2\]、多くの大学の航空部がここで訓練を行なっている。国内で数少ない航空機曳航（グライダーを他の[航空機](../Page/航空機.md "wikilink")で曳航して離陸させる方法）での訓練が行なえることがグライダー愛好家が多く利用することでも知られている。[同立戦](../Page/同立戦.md "wikilink")のグライダー競技会\[3\]は毎年ここで行なわれている（例外あり）。

[福井県警察](../Page/福井県警察.md "wikilink")の[ヘリコプター](../Page/ヘリコプター.md "wikilink")「くずりゅう」（[ユーロコプター](https://ja.wikipedia.org/wiki/ユーロコプター "wikilink")[EC135 T2i](https://ja.wikipedia.org/wiki/ユーロコプター_EC_135 "wikilink")）、福井県防災航空隊の[消防防災ヘリコプター](../Page/消防防災ヘリコプター.md "wikilink")「Blue Arrow」（ブルーアロー、[川崎重工業](../Page/川崎重工業.md "wikilink")[BK117](../Page/BK117_\(航空機\).md "wikilink")）\[4\]の基地にもなっている\[5\]。

## 歴史

[Fukui_Airport_Aerial_photograph.1975.jpg](https://ja.wikipedia.org/wiki/File:Fukui_Airport_Aerial_photograph.1975.jpg "fig:Fukui_Airport_Aerial_photograph.1975.jpg") [1964年](../Page/1964年.md "wikilink")に工事が開始され、[1966年](../Page/1966年.md "wikilink")開港した\[6\]。[全日本空輸](../Page/全日本空輸.md "wikilink")が[フォッカー F27を投入](../Page/フォッカー_F27.md "wikilink")（[1972年](../Page/1972年.md "wikilink")4月1日からは[YS-11](../Page/YS-11.md "wikilink")）し、[東京国際空港](https://ja.wikipedia.org/wiki/東京国際空港 "wikilink")（羽田）線を1日1 - 2便就航させた。

しかし[1973年](../Page/1973年.md "wikilink")に近隣の[小松空港がジェット化された影響で乗客数が減少](../Page/小松飛行場.md "wikilink")\[7\]\[8\]、滑走路が短く機体を大型化できない福井空港の発着便は採算が悪化し[1974年](../Page/1974年.md "wikilink")12月1日からは[名古屋乗継となり](../Page/名古屋飛行場.md "wikilink")、[1976年](../Page/1976年.md "wikilink")4月に定期航路が廃止された\[9\]\[10\]\[11\]。

これに対し、[1985年](https://ja.wikipedia.org/wiki/1985年 "wikilink")、福井県が滑走路の延長を2,000 mに拡張する計画を決定した\[12\]。調査が行われるものの、地元住民からの強硬な反対運動に遭う\[13\]。[1986年](../Page/1986年.md "wikilink")の第5次空港整備5箇年計画に福井空港が組み入れられ、滑走路延長へ向けた動きもあったが依然住民の反対は根強く、具体的な取り組みができないまま計画は事実上放置された。一方で小松空港へは[空港連絡バスの整備が行われ](../Page/小松空港連絡バス.md "wikilink")、アクセス改善が進んだ。

[2000年](../Page/2000年.md "wikilink")、**[公共事業](../Page/公共事業.md "wikilink")の抜本的見直しに関する三党合意**による公共事業の見直しで、福井空港拡張整備計画は地元住民の意見が醸成されていないとの理由で唯一「保留」扱いとされた。翌[2001年](../Page/2001年.md "wikilink")には全国的な公共事業の見直しの気運もあり、当時福井県知事の[栗田幸雄](../Page/栗田幸雄.md "wikilink")が福井空港拡張整備計画の凍結を表明した\[14\]。事実上、ジェット化の構想は白紙となった。[2003年](../Page/2003年.md "wikilink")6月27日、[県議会で福井県知事の](https://ja.wikipedia.org/wiki/福井県議会 "wikilink")[西川一誠](../Page/西川一誠.md "wikilink")が福井空港拡張整備計画の断念を表明した。「[北陸新幹線](https://ja.wikipedia.org/wiki/北陸新幹線 "wikilink")・小松空港重視」に政策を転換し、現滑走路の利用を軸に有効活用を模索することになった。

[2006年](../Page/2006年.md "wikilink")[9月30日](../Page/9月30日.md "wikilink")、[国土交通省](../Page/国土交通省.md "wikilink")[大阪航空局福井空港出張所と](../Page/地方航空局.md "wikilink")[気象庁](../Page/気象庁.md "wikilink")東京航空地方気象台福井空港出張所が閉鎖\[15\]。同年10月、空港管制業務が[中部国際空港](../Page/中部国際空港.md "wikilink")（中部FSC）からの遠隔管制（RAG）に移行した（[リモート空港](../Page/リモート空港.md "wikilink")）。

## 定期路線

現在、国内外の定期路線は設定されていない。上記の通り[1976年](../Page/1976年.md "wikilink")までは[羽田便が運航されていた](https://ja.wikipedia.org/wiki/東京国際空港 "wikilink")。ほかにも、[小松](../Page/小松飛行場.md "wikilink")・[名古屋](../Page/名古屋飛行場.md "wikilink")・[新潟便が設定されていた時期がある](https://ja.wikipedia.org/wiki/新潟空港 "wikilink")。

## 利用状況

国土交通省「空港管理状況調書」によると、福井空港の乗降客数および着陸数は以下のとおりとなっている\[16\]。

| 年度    | 乗降客数 | 着陸数   |
| ----- | ---- | ----- |
| 2006年 | 49   | 3,902 |
| 2007年 | 41   | 4,046 |
| 2008年 | 40   | 4,397 |
| 2009年 | 26   | 4,485 |
| 2010年 | 0    | 3,764 |
| 2011年 | 0    | 3,197 |
| 2012年 | 0    | 3,987 |
| 2013年 | 0    | 4,385 |
| 2014年 | 0    | 4,466 |
| 2015年 | 0    | 2,976 |

乗降客数は[2009年](../Page/2009年.md "wikilink")の26人を最後に記録されていない。一方、着陸数は年度に相違はあるが[富山空港](../Page/富山空港.md "wikilink")（[富山県](https://ja.wikipedia.org/wiki/富山県 "wikilink")[富山市](https://ja.wikipedia.org/wiki/富山市 "wikilink")）の国内線とほぼ同じレベルとなっている\[17\]。

## エピソード

英『[インデペンデント](../Page/インデペンデント.md "wikilink")』紙が2009年6月18日に発表した「世界で最もおかしい空港名」(World's funniest airport names) の「最も野蛮な名前」(The rudest) 部門で、福井空港は第1位に選ばれている\[18\]。その理由としては、2008年に[大リーグ](https://ja.wikipedia.org/wiki/大リーグ "wikilink")に移籍した[福留孝介](https://ja.wikipedia.org/wiki/福留孝介 "wikilink")の名前が「FUCK YOU DO ME」と聞こえるとして論議が巻き起こったのと同様に、英語圏の人々にとって「Fukui」という単語も際どい発音であると見られている（一種のダジャレと見る向きもあるが）\[19\]。なお、「Fukui」に近い発音のものでは「[Fak Fak空港](https://ja.wikipedia.org/wiki/:en:Fakfak_Airport "wikilink")」（インドネシア）も選ばれている\[20\]。
