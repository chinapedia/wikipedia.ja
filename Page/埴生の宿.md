## ホーム・スイート・ホーム

[Home_Sweet_Home_-_Project_Gutenberg_eText_21566.png](https://ja.wikipedia.org/wiki/File:Home_Sweet_Home_-_Project_Gutenberg_eText_21566.png "fig:Home_Sweet_Home_-_Project_Gutenberg_eText_21566.png") [1823年](../Page/1823年.md "wikilink")に作詞・作曲され、同年初演の[オペラ](../Page/オペラ.md "wikilink")『ミラノの乙女クラリ』（*Clari, Maid of Milan*）の中で歌われた。

### 作曲

[イギリス](https://ja.wikipedia.org/wiki/イギリス "wikilink")の[ヘンリー・ローリー・ビショップ](../Page/ヘンリー・ローリー・ビショップ.md "wikilink")作曲。なお、ビショップがこの曲の着想を得たのは[イタリア](../Page/イタリア.md "wikilink")（[シチリア](https://ja.wikipedia.org/wiki/シチリア "wikilink")）民謡からと言われている。

### 作詞

[アメリカ合衆国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")の[ジョン・ハワード・ペイン](https://ja.wikipedia.org/wiki/ジョン・ハワード・ペイン "wikilink")（, [1791年](../Page/1791年.md "wikilink") - [1852年](../Page/1852年.md "wikilink")）作詞。

## その後

アメリカでは、[バディ・ボールデン](https://ja.wikipedia.org/wiki/バディ・ボールデン "wikilink")楽団の演奏会のラスト・ナンバーとしてよく演奏されており、ジャズ・ナンバー化されている。

## 埴生の宿

<score vorbis="1"> \\relative c' { \\new PianoStaff \<\< \\new Staff { \\key c \\major \\time 4/4 r2. c4| e4. f8 f4. g8| g2 e4 g4| f4. e8 f4. d8| e2 r4 c4| e4. f8 f4. g8| g2 e4 g4| f4. e8 f4 d4| c2 r4 g'4| c4. b8 a4. g8| g2 e4 g4| f4. e8 f4 d4| e2 r4 g4| c4. b8 a4. g8| g2 e4 g4| f2 f4 d4| c2. r4| g'1| f2 d2| c2 d2| e2 r4 g4| c4. b8 a4. g8| g2 e4 g4| g4 a4 f4 d4| c2 r2 \\bar "|." } \\addlyrics { は にゅ う の や ど も わ が ー や ー ど た ま ー の よ そ い う ら ー や ま じ の ど ー か な り や は る ー の そ ら は な ー は あ る じ と り は と も おゝ わ が や ど よ た の ー し と も ー た の ー も し や } \>\> } </score> 日本では、[里見義](https://ja.wikipedia.org/wiki/里見義 "wikilink")訳詞の唱歌『埴生の宿』\[2\]として広く知られている。

<poem></poem>

この訳詞は[1889年](../Page/1889年.md "wikilink")（明治22年）12月に[東京音楽学校が出版した](https://ja.wikipedia.org/wiki/東京音楽学校_\(旧制\) "wikilink")『中等唱歌集』に収載された\[3\]。[太平洋戦争](https://ja.wikipedia.org/wiki/太平洋戦争 "wikilink")勃発に伴い、洋楽レコードが「敵性レコード」として廃棄が呼びかけられる中でも、『埴生の宿』や『[庭の千草](https://ja.wikipedia.org/wiki/庭の千草_\(民謡\) "wikilink")』など歌詞を邦訳にしたものは、国民生活になじんでいる\[4\]として敵性レコードから除外された\[5\]。2006年（平成18年）には[日本の歌百選](../Page/日本の歌百選.md "wikilink")の一つに選ばれている。

小説『[ビルマの竪琴](../Page/ビルマの竪琴.md "wikilink")』では、日本兵と敵兵がともに歌うという象徴的なシーンでこの歌が使われている。

映画、ドラマなどで使われた例として前述の『[ビルマの竪琴](../Page/ビルマの竪琴.md "wikilink")』をはじめ、『[二十四の瞳](https://ja.wikipedia.org/wiki/二十四の瞳_\(映画\) "wikilink")』（[木下恵介監督版](../Page/木下惠介.md "wikilink")）、『[火垂るの墓](../Page/火垂るの墓.md "wikilink")』、『[純情きらり](../Page/純情きらり.md "wikilink")』、『[ゲゲゲの女房](https://ja.wikipedia.org/wiki/ゲゲゲの女房 "wikilink")』、『[仮面ライダーV3](../Page/仮面ライダーV3.md "wikilink")』、『[マッサン](https://ja.wikipedia.org/wiki/マッサン "wikilink")』、『[ガールズ&パンツァー 劇場版](https://ja.wikipedia.org/wiki/ガールズ&パンツァー_劇場版 "wikilink")』などがある。

近年では、[2004年](../Page/2004年.md "wikilink")（平成16年）に[B-DASH](https://ja.wikipedia.org/wiki/B-DASH "wikilink")（アルバム『[ビッグ ブラック ストア (連絡しろ)](https://ja.wikipedia.org/wiki/ビッグ_ブラック_ストア_\(連絡しろ\) "wikilink")』に収録）が、[2007年](../Page/2007年.md "wikilink")（平成19年）に[キリンジ](https://ja.wikipedia.org/wiki/キリンジ "wikilink")（参加したアルバム『にほんのうた 第一集』に収録）がカバーしている。また、[三城ホールディングス](../Page/三城ホールディングス.md "wikilink")が「[パリミキ](https://ja.wikipedia.org/wiki/パリミキ "wikilink")」「[メガネの三城](https://ja.wikipedia.org/wiki/メガネの三城 "wikilink")」名義で出稿している[波瑠](../Page/波瑠.md "wikilink")出演の[テレビCM](https://ja.wikipedia.org/wiki/テレビCM "wikilink")では、[2019年](../Page/2019年.md "wikilink")（令和元年）[6月21日](../Page/6月21日.md "wikilink")から放送中の「知的好奇心（夏）」篇・「スマホ時代の目に」篇で[ハンバートハンバート](https://ja.wikipedia.org/wiki/ハンバートハンバート "wikilink")によるカバーバージョンが用いられている\[6\]。

[JR](../Page/九州旅客鉄道.md "wikilink")[三角線](https://ja.wikipedia.org/wiki/三角線 "wikilink")[石打ダム駅](https://ja.wikipedia.org/wiki/石打ダム駅 "wikilink")の列車接近放送や、[北大阪急行電鉄](../Page/北大阪急行電鉄.md "wikilink")[千里中央駅](https://ja.wikipedia.org/wiki/千里中央駅 "wikilink")の到着放送チャイム、[近鉄特急](https://ja.wikipedia.org/wiki/近鉄特急 "wikilink")の[桑名駅](../Page/桑名駅.md "wikilink")到着時の[車内チャイム](https://ja.wikipedia.org/wiki/車内チャイム "wikilink")、[埼玉県](https://ja.wikipedia.org/wiki/埼玉県 "wikilink")[羽生市](../Page/羽生市.md "wikilink")の[防災行政無線](https://ja.wikipedia.org/wiki/防災行政無線 "wikilink")の正午のチャイム（現在は途中切りになっている）や[神奈川県](https://ja.wikipedia.org/wiki/神奈川県 "wikilink")[愛甲郡](../Page/愛甲郡.md "wikilink")[愛川町](https://ja.wikipedia.org/wiki/愛川町 "wikilink")の防災行政無線の夕方のチャイムなどでも使用されている。かつては、埼玉県羽生市や[東京都](https://ja.wikipedia.org/wiki/東京都 "wikilink")[羽村市](../Page/羽村市.md "wikilink")の防災行政無線の夕方のチャイムでも使用されていた。
