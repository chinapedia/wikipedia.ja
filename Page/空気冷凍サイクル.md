**空気冷凍サイクル**（くうきれいとうサイクル）は、[気体](../Page/気体.md "wikilink")を[圧縮機](../Page/圧縮機.md "wikilink")で圧縮し、[熱交換器](../Page/熱交換器.md "wikilink")で冷却した後、膨張タービンで動力回収して[圧力](../Page/圧力.md "wikilink")・[温度](../Page/温度.md "wikilink")を下げ低温を得る、[冷凍機](../Page/冷凍機.md "wikilink")の[熱サイクルである](../Page/熱機関の理論サイクル.md "wikilink")。

小型軽量であるため、開放型のものが[航空機](../Page/航空機.md "wikilink")の客室用[空調に用いられている](../Page/空気調和.md "wikilink")。また、-150℃以下で[蒸気圧縮冷凍サイクル](../Page/蒸気圧縮冷凍サイクル.md "wikilink")より効率がよくなることを生かして、密閉型のものが[空気](https://ja.wikipedia.org/wiki/大気 "wikilink")・[天然ガス](../Page/天然ガス.md "wikilink")の液化に用いられている。

## 膨張弁サイクル

1 - 圧縮機が気体に l<sub>c</sub> の仕事を加えて圧縮 →2

2 - 熱交換器で冷却 →3

3 - 膨張タービンで l<sub>t</sub> の仕事をして膨張 →4

4 - r の熱量を吸収 →1

## 理想的な場合のサイクル

圧縮機などの損失がない場合を考えると、[逆カルノーサイクル](../Page/逆カルノーサイクル.md "wikilink")と同等である。

  -
    l<sub>t</sub> = c<sub>p</sub> ( T<sub>3</sub> - T<sub>4</sub> )
    r<sub>o</sub> = c<sub>p</sub> ( T<sub>1</sub> - T<sub>4</sub> )
    l<sub>c</sub> = c<sub>p</sub> ( T<sub>2</sub> - T<sub>1</sub> )

よって

  -
    l<sub>o</sub> = l<sub>c</sub> - l<sub>t</sub> = c<sub>p</sub> ( T<sub>2</sub> - T<sub>1</sub> ) - c<sub>p</sub> ( T<sub>3</sub> - T<sub>4</sub> )
    (COP)<sub>R</sub> = r<sub>o</sub> / l = ( T<sub>1</sub> - T<sub>4</sub> ) / {( T<sub>2</sub> - T<sub>1</sub> ) - ( T<sub>3</sub> - T<sub>4</sub> )}

断熱圧縮時と断熱膨張時の圧力比 P<sub>h</sub> / P<sub>l</sub> は等しいため

  -
    T<sub>2</sub> / T<sub>1</sub> = ( P<sub>h</sub> / P<sub>l</sub> )<sup>(κ-1)κ</sup> = T<sub>3</sub> / T<sub>4</sub>

よって

  -
    (COP)<sub>R</sub> = T<sub>1</sub> / ( T<sub>2</sub> - T<sub>1</sub> ) = T<sub>4</sub> / ( T<sub>3</sub> - T<sub>4</sub> )

<!-- end list -->

  -
    (COP)<sub>R</sub> : 冷凍サイクルの理論成績係数
    r<sub>o</sub> : 理想的な場合の気体1kgあたり冷却効果
    l<sub>o</sub> : 理想的な場合の気体1kgあたりに必要な仕事
    κ : 気体の比熱比 T : 気体の[温度](../Page/温度.md "wikilink") P : 気体の[圧力](../Page/圧力.md "wikilink")

## 損失を考慮したサイクル

圧縮機・膨張タービンでの損失を考慮すると

  -
    η<sub>t</sub> = ( T<sub>3</sub> - T<sub>4</sub>' ) / ( T<sub>3</sub> - T<sub>4</sub> )
    η<sub>c</sub> = ( T<sub>2</sub> - T<sub>1</sub> ) / ( T<sub>2</sub>' - T<sub>1</sub> )
    r = c<sub>p</sub> ( T<sub>1</sub> - T<sub>4</sub>' )
    l = ( T<sub>2</sub> - T<sub>1</sub> ) c<sub>p</sub> / η<sub>c</sub> - η<sub>t</sub> c<sub>p</sub> ( T<sub>3</sub> - T<sub>4</sub> )
    (COP)<sub>R</sub> = r / l = η<sub>c</sub> ( T<sub>1</sub> - T<sub>4</sub>' ) / {( T<sub>2</sub> - T<sub>1</sub> ) - η<sub>c</sub> η<sub>t</sub> ( T<sub>3</sub> - T<sub>4</sub> )}

<!-- end list -->

  -
    η<sub>c</sub> : 圧縮機効率
    η<sub>t</sub> : 膨張タービン効率
    T<sub>4</sub>' : 実際の膨張タービンの出口温度
    T<sub>2</sub>' : 実際の圧縮機の出口温度
    r : 実際の気体1kgあたり冷却効果
    l : 実際の気体1kgあたりに必要な仕事
