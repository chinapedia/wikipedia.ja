**経理実務士（補）**（けいりじつむし（ほ））は、日本経理実務士会が認定する資格である。

## 受験資格

制限なし。ただし、資格取得認定要件を満たしていること。

## 資格取得認定要件

### 経理実務士

|                                                                         |                                                                 |                       |
| ----------------------------------------------------------------------- | --------------------------------------------------------------- | --------------------- |
| 必須                                                                      | [全経簿記](http://www.zenkei.or.jp/license/bookkeeping.html)1級以上    | 会計・工業簿記の両方に合格していること。  |
| [全経税務会計](http://www.zenkei.or.jp/license/tax.html)3級以上                  | 所得税法・法人税法・消費税法のうち2科目に合格していること。                                  |                       |
| 選択                                                                      | [全経電卓](http://www.zenkei.or.jp/license/deskcalculator.html)2級以上 | 左記4検定のうち1検定に合格していること。 |
| [全経計算実務](http://www.zenkei.or.jp/license/calculate.html)2級以上            |                                                                 |                       |
| 全経珠算2級以上                                                                |                                                                 |                       |
| [全経コンピュータ会計](http://www.zenkei.or.jp/license/computer_account.html)2級以上 |                                                                 |                       |

## 認定料

4,000円

## 認定方法

  - 全経の会員校に在籍している学生又は卒業生

<!-- end list -->

  -
    [日本経理実務士会](http://www.zenkei.or.jp/nkj/download/KeiriJitsumuShinsei_3rd.pdf)から申込書を印刷し、必要事項に記載した上で、￥4,000を持って、在籍校で申請すること

<!-- end list -->

  - 一般（会員校に在籍していない者）

<!-- end list -->

  -
    [日本経理実務士会](http://www.zenkei.or.jp/nkj/download/KeiriJitsumuShinsei_3rd.pdf)から申込書を印刷し、必要事項に記載した上で、￥4,000分の定額小為替と認定資格要件の全経検定合格証書の写しを同封し、日本経理実務士会へ送ること

## 認定月

日本経理実務士会に3月末までに到着するように、郵送すること。
