**湯川ポテンシャル**（**ゆかわポテンシャル**, Yukawa potential、湯川型ポテンシャルとも言う）は以下の式で表現される形をしている。

\[\alpha {1 \over {r}} e^{ - \kappa r }\]

r はポテンシャル中心からの動径座標。αは適当な係数。κは、1/κがポテンシャルの（実効的な）到達距離に相当する量となる。このポテンシャルは[中心力](../Page/中心力.md "wikilink")ポテンシャルである。

## 素粒子物理学での湯川ポテンシャル

このポテンシャルは元々、[湯川秀樹](../Page/湯川秀樹.md "wikilink")が短距離力である[核力](https://ja.wikipedia.org/wiki/核力 "wikilink")の説明のために導入したものである（[1935年](../Page/1935年.md "wikilink")）。核力は、核子間でやりとりされる未知の粒子によるとし（交換力）、その力を表すポテンシャルの形が湯川ポテンシャルとなる。この場合、上記式に相当する核力でのポテンシャルの形は、

\[- { f^2 \over { 4 \pi \hbar c} } mc^2 { 1 \over { {1 \over \lambda } r } } e^{ - {1 \over \lambda} r }  = - { f^2 \over { 4 \pi} } {1 \over \lambda}  { 1 \over { {1 \over \lambda } r } } e^{ - {1 \over \lambda} r } = - { f^2 \over { 4 \pi} } { 1 \over { r } } e^{ - {1 \over \lambda} r }\]

となる。ここで、

\[\kappa = {1 \over \lambda} = { mc \over {\hbar} }\]

である。c は真空での光速、\(\hbar = { h \over {2 \pi} }\) で、h はプランク定数である。mは未知の粒子の質量である。質量ゼロの[光子](../Page/光子.md "wikilink")のやりとりによる交換力（つまり[クーロン力](https://ja.wikipedia.org/wiki/クーロン力 "wikilink")）によるポテンシャルは、1/rの形であるが、湯川は核力は短距離力であり上記のような短距離にしか及ばないポテンシャルを考え、その交換力を担う粒子は有限の質量を持つと考えた。そして、その未知の粒子の質量を、およそ電子の200倍の重さと予想した（[1934年](../Page/1934年.md "wikilink")頃）。この未知の粒子が、[中間子](../Page/中間子.md "wikilink")であり、上式でのλ(=1/κ)を中間子のコンプトン波長と言う。f は、核子と中間子の結合定数である。

## 物性物理学での湯川ポテンシャル

湯川ポテンシャルは、他にも[金属](../Page/金属.md "wikilink")中の不純物の有効静電ポテンシャルにも使用される。これは金属内の不純物の電荷をqとすると、それは金属の[伝導電子](https://ja.wikipedia.org/wiki/伝導電子 "wikilink")によって遮蔽されるので、不純物の静電ポテンシャルは通常のクーロンポテンシャルの形ではなく、湯川ポテンシャルの形となる。この時の遮蔽された静電ポテンシャルの形は、

\[U = {q \over { 4 \pi \epsilon_0 } } {1 \over r} e^{- \kappa r }\]

である。ここで、十分に低温なら、

\[\kappa^2 = {1 \over {\epsilon_0}} e^2 N(\epsilon_F)\]

であり、e は[素電荷](https://ja.wikipedia.org/wiki/素電荷 "wikilink")、N(ε<sub>F</sub>) は、[フェルミレベル](https://ja.wikipedia.org/wiki/フェルミレベル "wikilink")での電子の[状態密度](../Page/状態密度.md "wikilink")、ε<sub>0</sub> は真空の誘電率である。

また同様にして、強[電解質](https://ja.wikipedia.org/wiki/電解質 "wikilink")溶液や[プラズマ](https://ja.wikipedia.org/wiki/プラズマ "wikilink")の理論においても**デバイ遮蔽**の結果としてこの型のポテンシャルが現れ、**デバイ-ヒュッケルのポテンシャル**と呼ばれる。　またその特性距離を[デバイの長さ](https://ja.wikipedia.org/wiki/デバイの長さ "wikilink")と言う。
