**イータイピング** (e-typing) とは、イータイピング株式会社が運営する[タッチ・タイピング](../Page/タッチ・タイピング.md "wikilink")練習のための各種ツールをブラウザ上で無料提供している[ウェブサイト](../Page/ウェブサイト.md "wikilink")である。

## 概要

タイピング練習は[ウェブブラウザ](../Page/ウェブブラウザ.md "wikilink")のみで行うことができる。さまざまな例文が登録されており、楽しみながら練習できるよう工夫されている。そのため、[タッチ・タイピング](../Page/タッチ・タイピング.md "wikilink")を習得しようとしている多くの人に利用されており、登録会員数は81万人（2009年6月時点）でタイピングサイトとしては国内最多の登録数である。

無料タイピング練習のほか、以下のサービスを提供している。

  - イータイピング・マスター : ウェブ上で24時間受験できるタイピング技能資格試験
    法人向けタイピングウェブサービス : 法人向けに同社のタイピング教材をカスタマイズして提供（有料）

1週間に1回例文が更新される。（腕試しタイピング）※登録必要・無料

タイピングの結果を保存し、グラフ化することができる。※登録必要・無料

この e-typing ではタイピングの速度より正確さを最重視しているため、タイプミスによる成績下落が激しい傾向にある。

### 階級

8級〜1級,特級の9段階になっている。

### 合格ライン

1つの級に3つの項目があり3つとも合格ラインに達していれば合格となる。合格ラインは級が上がるほど難しくなり、文も長くなっていく。

### 3つの項目

下の表は簡単な順に並んでいる。また使用級も表示する。

| 項目名    | 使用級       |
| ------ | --------- |
| 単キーテスト | 8級〜6級     |
| かなテスト  | 8級〜4級     |
| 単語テスト  | 全級(8級〜特級) |
| 短文テスト  | 5級〜特級     |
| 長文テスト  | 3級〜特級     |

## 試験の方法

検定料を支払ってから2週間以内に[パソコン](https://ja.wikipedia.org/wiki/パソコン "wikilink")などで検定を受ける。そのとき発行された[ID](../Page/ID.md "wikilink")と[パスワード](../Page/パスワード.md "wikilink")を入力する。

### 8級

e-typingの8級は無料で受けられる。さらにどのくらいの級相当かが分かる。また有料で合格証明書(1,575円(郵送費・消費税含))も発行できる。ただし、その相当のレベルにかかわらず、8級の合格証となる。合格証は8級の合格ラインを越えないと発行できない。
