**三育学院中学校**は、[千葉県](../Page/千葉県.md "wikilink")[大多喜町](../Page/大多喜町.md "wikilink")にある[全寮制](https://ja.wikipedia.org/wiki/全寮制 "wikilink")で[キリスト教](../Page/キリスト教.md "wikilink")主義の[私立の](../Page/私立学校.md "wikilink")[中学校](../Page/中学校.md "wikilink")である。[沖縄三育中学校](../Page/沖縄三育中学校.md "wikilink")、[広島三育学院中学校とともに](../Page/広島三育学院高等学校・中学校・大和小学校.md "wikilink")、[広島三育学院高等学校と連携型](https://ja.wikipedia.org/wiki/広島三育学院中学校・高等学校 "wikilink")[中高一貫教育](../Page/中高一貫教育.md "wikilink")を行っている。

## 校風

三育学院中学校は、[千葉県](../Page/千葉県.md "wikilink")では数少ない[全寮制](https://ja.wikipedia.org/wiki/全寮制 "wikilink")[共学](https://ja.wikipedia.org/wiki/共学 "wikilink")の[私立中学校](https://ja.wikipedia.org/wiki/私立中学校 "wikilink")である。全国に中学校は他に、[広島県](../Page/広島県.md "wikilink")[三原市](../Page/三原市.md "wikilink")に[広島三育学院中学校](https://ja.wikipedia.org/wiki/広島三育学院中学校 "wikilink")、[沖縄県](../Page/沖縄県.md "wikilink")[名護市](../Page/名護市.md "wikilink")に[沖縄三育中学校](../Page/沖縄三育中学校.md "wikilink")がある。その他、日本国内に、高校1、大学1、短大・カッレジ１、[小学校](../Page/小学校.md "wikilink")10、[幼稚園](../Page/幼稚園.md "wikilink")5の系列校を持っている。

## 特徴

全寮制であり、教職員も同じ敷地内に住み、昼夜を問わず生徒たちの指導にあたっている。キリスト教教育を実践しており、教職員も全員がクリスチャンである。

卒業生の多くは、広島県三原市にある[広島三育学院高等学校](https://ja.wikipedia.org/wiki/広島三育学院高等学校 "wikilink")へ進学する。三育学院中学校から[広島三育学院高等学校に進学するには](../Page/広島三育学院高等学校・中学校・大和小学校.md "wikilink")、通常の[入学試験](https://ja.wikipedia.org/wiki/入学試験 "wikilink")[受験](https://ja.wikipedia.org/wiki/受験 "wikilink")が必要である。

全世界に数多くの系列校を持ち、広島三育学院高校へ進学後留学する者も多い。

  - 系列大学・大学院数：99校
  - 中学・高等学校数：1214校
  - 小学校：1262校

## 沿革

| 1898年（明治31年） | キリスト教宣教師ウィリアム・C・グレンジャー宣教師により、東京麻布に芝和英聖書学校を開設。                                                          |
| ------------ | ------------------------------------------------------------------------------------------------------ |
| 1914年（大正 3年） | 東京府豊多摩郡杉並村（現:杉並区）天沼に移転。日本伝道学校と改称。                                                                      |
| 1919年（大正 8年） | 小学校・中学部・高等部を設置し、天沼学院と改称。                                                                               |
| 1926年（大正15年） | 千葉県君津郡神納村（現袖ヶ浦市）に移転。 日本三育学院と改称。                                                                        |
| 1928年（昭和 3年） | 東京衛生病院看護婦学校（3年課程）を天沼に設立。                                                                               |
| 1943年（昭和18年） | 9月20日、キリスト教信仰のゆえの治安維持法違反の容疑で、特高警察により院長以下主要教員が検挙・連行。女学院も同様。学院業務を停止。 12月26日、千葉県例により学院閉鎖。学院施設は陸軍療養所として接収。 |
| 1947年（昭和22年） | 1月23日、日本三育学院再開。                                                                                        |
| 1953年（昭和28年） | 神学科・教育学科・セクレタリー科を設置。日本三育学院カレッジと改称。                                                                     |
| 1969年（昭和44年） | 北浦三育中学校設立                                                                                              |
| 1971年（昭和46年） | シャロンホーム（女子寮） カナンホーム（男子寮）完成                                                                             |
| 1975年（昭和50年） | 北浦三育幼稚園閉園式                                                                                             |
| 1977年（昭和52年） | 校舎増築、女子寮新館増築 食堂、旧チャペル新築 新校舎 校歌披露 広島県大和町　広島三育学院中学校・高等学校開校                                               |
| 1979年（昭和54年） | 第二グラウンド完成・男子寮増築                                                                                        |
| 1984年（昭和59年） | 体育館、テニスコート完成                                                                                           |
| 1987年（昭和62年） | 講堂、特別教室校舎新築 チャペル竣工                                                                                     |
| 1989年（平成 元年） | 牧師館完成                                                                                                  |
| 1992年（平成 4年） | 新男子寮竣工 新カナン寮                                                                                           |
| 2000年（平成12年） | ニュージーランド海外研修開始                                                                                         |
| 2004年（平成16年） | 教職員集合住宅完成                                                                                              |
| 2011年（平成23年） | 東日本大震災 女子寮耐震補強工事完了                                                                                     |
| 2012年（平成24年） | 震災復旧工事完了                                                                                               |
| 2013年（平成25年） | パイプオルガン設置                                                                                              |
| 2017年（平成29年） | 法人理事会にて大多喜町への移転統合計画承認                                                                                  |
| 2019年（令和元年）  | 千葉県での設置認可計画承認 50周年全体同窓会                                                                                |
| 2020年（令和2年）  | 千葉県夷隅郡大多喜町に移転　三育学院中学校開校                                                                                |

## 教育方針

  - キリスト教を柱とした全寮制による24時間の全人教育
  - 知育（知的教育）・徳育（信仰教育）・体育（健康教育）のバランスある全人的な三育教育の実践を目指す。
