**Apple Remote Desktop** (アップル・リモート・デスクトップ, 略称ARD) は、[アップルの](../Page/アップル_\(企業\).md "wikilink")[Mac](https://ja.wikipedia.org/wiki/Mac "wikilink")用[リモート管理ソフトである](https://ja.wikipedia.org/wiki/リモートデスクトップ "wikilink")。

## 概要

その前身は[Apple Network Assistant](https://ja.wikipedia.org/wiki/Apple_Network_Assistant "wikilink")\[1\]。Apple Remote Desktop(ARD)が2002年8月にリリース\[2\]された当初は[Carbon](../Page/Carbon.md "wikilink")化されたMac OS X対応のApple Network Assitant機能限定版ともいえる機能であった。2004年6月21日発表されたARD 2で[VNCベースとなるとともにMac](../Page/Virtual_Network_Computing.md "wikilink") OS X専用ソフトとして生まれ変わった\[3\]。2006年4月11日に発表されたARD 3では [Mac OS X v10.4を活かす機能も搭載され](../Page/Mac_OS_X_v10.4.md "wikilink")、利便性の向上とともにパフォーマンスアップがはかられている\[4\]。[2007年](../Page/2007年.md "wikilink")[10月18日](../Page/10月18日.md "wikilink")リリースのARD 3.2からは[Mac OS X v10.5に](../Page/Mac_OS_X_v10.5.md "wikilink")、[2009年](../Page/2009年.md "wikilink")[8月21日](../Page/8月21日.md "wikilink")リリースのARD 3.3からは[Mac OS X v10.6に対応している](../Page/Mac_OS_X_v10.6.md "wikilink")。

2015年1月27日、[アイコン](../Page/アイコン.md "wikilink")が（[通称](../Page/通称.md "wikilink")）[フラットデザイン](https://ja.wikipedia.org/wiki/:en:Flat_design "wikilink")\[5\]となったARD 3.8からは、[OS X Mavericks](https://ja.wikipedia.org/wiki/OS_X_Mavericks "wikilink")\~[OS X Yosemite対応](https://ja.wikipedia.org/wiki/OS_X_Yosemite "wikilink")、[Mac App Storeでのみの配布となり](https://ja.wikipedia.org/wiki/Mac_App_Store "wikilink")、無制限クライアント版が従来の57,000円\[6\]から9800円\[7\]へと大幅に値下げされた。従来のユーザに対しては、シリアル番号不要だがApple IDと関連付けされたRemote Desktop.appがアップデートとしてダウンロードされる仕組みとなった。それにより、複数のMacに管理ソフトをインストールすることができるようになった。ARD 3.9 (2017年2月21日) では、通信セキュリティ強化やTouch Barのサポートが追加され、OS X Yosemite 10.10.5以降の対応となった\[8\]。

バージョン３以前のARDによる暗号化は、パスワード、マウスイベント、キーストロークのみで、画面やファイル転送は対応していなかった。これらの利用には、アップル自身が、VPNを推奨していた。ARD 3.0ではAES128bitが利用できるようになった。
