**TestLink**は[PHPで作成された](../Page/PHP_\(プログラミング言語\).md "wikilink")[オープンソース](../Page/オープンソース.md "wikilink")のWebベース[テスト管理システム](https://ja.wikipedia.org/wiki/テスト管理システム "wikilink")である。TestLink単体でも動作可能だが、[Mantis](https://ja.wikipedia.org/wiki/Mantis "wikilink")や[JIRA](https://ja.wikipedia.org/wiki/JIRA "wikilink")、[Bugzilla](../Page/Bugzilla.md "wikilink"),[Trac](../Page/Trac.md "wikilink"),[Redmine](https://ja.wikipedia.org/wiki/Redmine "wikilink")、[FogBugz](https://ja.wikipedia.org/wiki/FogBugz "wikilink")などの[バグトラッキングシステム](https://ja.wikipedia.org/wiki/バグトラッキングシステム "wikilink")と連携可能である。バージョン1.7.0から日本語対応。

## 主な機能

  - テストケースの登録・管理・評価実行・評価結果の集計を行うことが可能
  - 登録したテストケースを仕様書の形で出力
  - テストケースをユーザーに割り当て
  - 要求仕様を登録しテストケースと関連づけ
  - Mantis,Bugzilla,Trac,RedMine等のバグ管理ツールと連携も可能

## 利点

  - 実行済みのテストケース数と結果をリアルタイムで把握
  - 登録済みのテストケース数をリアルタイムで把握
  - テストケースの管理手順を標準化
  - 要求仕様とテストケース、バグ票を関連付けて管理できる
  - ネットで情報共有、分散評価が可能

  - テストケースの変更履歴を自動で管理できる
  - テストプランに基づいてテストケースの流用ができる
  - テスト結果の履歴を自動管理できる
  - テストケース表記にHTMLが使える
  - テストケース作成者と実行者の権限が分けられる

## TestLink Ver1.7.0の新機能

  - Test Case history/ versioning
  - Unlimited structure of Test Specification
  - Support for other databases (SQL standard, generalized DB interdace)
  - Improved user management (editable)
  - Attachments
  - Revision of Standard terminology
  - Enhanced reports
  - XML import/export
  - Custom fields for Test Cases
