**WikiZnanie**（）は[オンライン百科事典](https://ja.wikipedia.org/wiki/オンライン百科事典 "wikilink")の一つ。言語は[ロシア語](https://ja.wikipedia.org/wiki/ロシア語 "wikilink")で、[ウィキ](../Page/ウィキ.md "wikilink")を採用しており、ライセンスは[FreeBSD Documentation Licenseである](https://ja.wikipedia.org/wiki/FreeBSD_Documentation_License "wikilink")。2003年創立。[MediaWiki](https://ja.wikipedia.org/wiki/MediaWiki "wikilink")のソフトウェアを使っている。名前は、wikiと（ラテン翻字例：znanie。ロシア語で「知識」の意）の2語の合成。

2006年、項目数が1000を越えた。2009年8月現在で、11万2000の項目がある（対する[ロシア語版ウィキペディア](../Page/ロシア語版ウィキペディア.md "wikilink")は42万7000項目）。記事の大半（推計95から98%）が、『ブロックハウス&エフロン百科事典』（；帝政ロシア時代の1890年から1906年に刊行された）の内容を含む。

[GFDL](https://ja.wikipedia.org/wiki/GFDL "wikilink")に基づく[ウィキペディア](https://ja.wikipedia.org/wiki/ウィキペディア "wikilink")とは異なり、WikiZnanieはFreeBSD Documentation Licenseに基づいて運営されている。また、投稿者個人の独自研究を容認しており、執筆姿勢としては「中立的な視点」より「多面的な視点」を推奨している。[サーバー](https://ja.wikipedia.org/wiki/サーバー "wikilink")は[モスクワ](https://ja.wikipedia.org/wiki/モスクワ "wikilink")にあり、ロシア国内からは迅速にアクセス可能。財源を確保するために商業広告を入れている。

創設者・[ウェブマスター](https://ja.wikipedia.org/wiki/ウェブマスター "wikilink")はアンドレイ・ヴォフク(；翻字例 Andrey Vovk）。
