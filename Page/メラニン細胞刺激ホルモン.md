**メラニン細胞刺激ホルモン** (メラニンさいぼうしげきホルモン, , **MSH**) または**メラノトロピン**は、[ペプチドホルモン](https://ja.wikipedia.org/wiki/ペプチドホルモン "wikilink")に属する[神経ペプチド](https://ja.wikipedia.org/wiki/神経ペプチド "wikilink")の一つ。α-MSH、β-MSH、γ-MSHに分類でき、いずれも[下垂体](https://ja.wikipedia.org/wiki/下垂体 "wikilink")中葉から分泌される。分子量は1664.9g/mol。

## 作用

[皮膚](../Page/皮膚.md "wikilink")や[髪](https://ja.wikipedia.org/wiki/髪 "wikilink")の[メラニン細胞](https://ja.wikipedia.org/wiki/メラニン細胞 "wikilink")のチロシナーゼを活性化させ、メラニンの合成を促進する。α-MSHは視床下部で作用し、食欲および性的興奮を刺激する。

**両生類**

[アフリカツメガエル](../Page/アフリカツメガエル.md "wikilink")などの一部の動物は、薄暗い環境でMSHの合成速度が増加する。それにより、色素が皮膚の[色素細胞](https://ja.wikipedia.org/wiki/色素細胞 "wikilink")に分散し、体色が暗くなることで捕食者から身を守る。この色素細胞は黒色素胞と呼ばれるため、両生類ではしばしば黒色素胞刺激ホルモンとも呼ばれる。

**ヒト**

MSHはエストロゲンと共に[妊娠](../Page/妊娠.md "wikilink")中に増加し、妊婦の色素沈着を引き起こす。過剰な[副腎皮質刺激ホルモン](../Page/副腎皮質刺激ホルモン.md "wikilink")の分泌に起因する[クッシング症候群](../Page/クッシング症候群.md "wikilink")は、色素沈着過剰を引き起こすことがある。[アジソン病](https://ja.wikipedia.org/wiki/アジソン病 "wikilink")患者の多くは、日光に頻繁に晒されていない領域を含む皮膚の黒ずみ（色素沈着過剰）を持つ。

## 構造

各種MSHは、メラノコルチンと呼ばれるグループに属する。このグループには、ACTH、α-MSH、β-MSH、γ-MSHが含まれる。MSHは次のようなアミノ酸配列をとる。

|                  |                                                                                         |
| ---------------- | --------------------------------------------------------------------------------------- |
| α-MSH:           | Ac-Ser-Tyr-Ser-Met-Glu-His-Phe-Arg-Trp-Gly-Lys-Pro-Val                                  |
| β-MSH (human):   | Ala-Glu-Lys-Lys-Asp-Glu-Gly-Pro-Tyr-Arg-Met-Glu-His-Phe-Arg-Trp-Gly-Ser-Pro-Pro-Lys-Asp |
| β-MSH (porcine): | Asp-Glu-Gly-Pro-Tyr-Lys-Met-Glu-His-Phe-Arg-Trp-Gly-Ser-Pro-Pro-Lys-Asp                 |
| γ-MSH:           | Tyr-Val-Met-Gly-His-Phe-Arg-Trp-Asp-Arg-Phe-Gly                                         |
|                  |                                                                                         |
