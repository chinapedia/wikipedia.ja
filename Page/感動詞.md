**感動詞**（かんどうし / 英: interjection） とは、感動、応答、呼び掛けを表す。[大和言葉](../Page/大和言葉.md "wikilink")においてはかつて「なげきことば」と呼ばれていた\[1\]。[主語](../Page/主語.md "wikilink")、[述語](../Page/述語.md "wikilink")、[修飾語](../Page/修飾語.md "wikilink")になることも他の語に修飾されることもない。**間投詞**（かんとうし）、**感嘆詞**（かんたんし）、**嘆詞**（たんし）とも言う。[口語](../Page/口語.md "wikilink")においては頻繁に用いられるが、[文語](../Page/文語.md "wikilink")において用いられることは少ない。

## 日本語

[活用](../Page/活用.md "wikilink")がなく、[自立語](https://ja.wikipedia.org/wiki/自立語 "wikilink")である。

|                                                   | 働き                              | 相当する語 |
| ------------------------------------------------- | ------------------------------- | ----- |
| 感動                                                | 話し手の感動を表す。                      | 口語    |
| 文語                                                | 「ああ」「あっぱれ」「あはれ」「あな」など。          |       |
| 呼び掛け                                              | 相手に呼びかける。                       | 口語    |
| 文語                                                | 「いかに」「いざ」「やあやあ」など。              |       |
| [応答](https://ja.wikipedia.org/wiki/応答 "wikilink") | 相手に応答する。                        | 口語    |
| 文語                                                | 「いな」「おうおう」など。                   |       |
| [挨拶](../Page/挨拶.md "wikilink")                    | 相手に挨拶する。                        | 口語    |
| 掛け声                                               | 話し手や相手に呼びかけて、調子をとったり、勢いをつけたりする。 | 口語    |

## 英語

  - ah（ああ）
  - aha（なるほど）
  - eh（そうだろう？、～だよな？）
  - er（あの－）
  - hmm（うーん）
  - oh（おお）
  - uh（えー、あのー）
  - wow （うあー、わお）
  - well（えーと、まあ）

## 中国語

  - a ㄚ˙ (返事や頷いた際の「ああ」や、驚いた際の「あっ」)

  - en ㄣ˙ (頷いた際の「うん」、考えている時や悩んでいる際の「ん…」。三点リーダと併用することもある)

  - wéi ㄨㄟˊ ([電話](../Page/電話.md "wikilink")応対の決まり文句「もしもし」)

  - aiya ㄞˋㄧㄚ˙ (驚いた時の言葉「あらま」)

  - o ㄛ˙ (驚いた際の「おお」など)

## 朝鮮語

  - aigo (感極まった時、溜め息をつく時の言葉)
