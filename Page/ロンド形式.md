**ロンド形式**（ロンドけいしき、rondo）は、[楽曲](https://ja.wikipedia.org/wiki/楽曲 "wikilink")の[形式の一つ](https://ja.wikipedia.org/wiki/楽式 "wikilink")。異なる旋律を挟みながら、同じ旋律（ロンド主題）を何度も繰り返す形式。

[日本語](../Page/日本語.md "wikilink")([漢字](../Page/漢字.md "wikilink"))では、ロンドを「**輪舞曲**」([大辞泉](../Page/大辞泉.md "wikilink"))もしくは「**回旋曲**」([大辞林](../Page/大辞林.md "wikilink"))と表記する。

## 一般的なスタイル

[古典派以降では](../Page/古典派音楽.md "wikilink")、[複合三部形式](https://ja.wikipedia.org/wiki/複合三部形式 "wikilink")の一種とみなすことのできる、次の2つの形式が一般的になった。

  - それぞれ**大ロンド形式**（だい――）、**小ロンド形式**（しょう――）と呼ばれる。

**複合三部形式**

| 主部 | 中間部 | 主部 |
| -- | --- | -- |
| A  | B   | A  |

**大ロンド形式**

|   |   |   |   |   |   |   |
| - | - | - | - | - | - | - |
| A | B | A | C | A | B | A |

**小ロンド形式**

|   |   |   |   |   |
| - | - | - | - | - |
| A | B | A | C | A |

なお、古くは以下のような形をとっていた。

|   |   |   |   |   |   |   |
| - | - | - | - | - | - | - |
| A | B | A | C | A | D | A |
