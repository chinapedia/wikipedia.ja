 **iSync**（アイシンク）は、かつて[アップルが提供していた](../Page/アップル_\(企業\).md "wikilink")[ソフトウェア](../Page/ソフトウェア.md "wikilink")である。

## 概要

動作環境は[Mac OS Xにのみ対応し](https://ja.wikipedia.org/wiki/macOS "wikilink")、[iCal](https://ja.wikipedia.org/wiki/iCal "wikilink")や[Address Bookなどのツールのデータを](https://ja.wikipedia.org/wiki/連絡先_\(アップル\) "wikilink")、[.Mac](https://ja.wikipedia.org/wiki/.Mac "wikilink")や[iPod](https://ja.wikipedia.org/wiki/iPod "wikilink")などのその他のデバイスと同期させる機能を持つ。その他にも[SyncML](https://ja.wikipedia.org/wiki/SyncML "wikilink")準拠の[携帯電話](../Page/携帯電話.md "wikilink")や[Palm OSなどの携帯型情報ツールにも対応する](https://ja.wikipedia.org/wiki/Palm_OS "wikilink")。Mac OS X v10.4以前では、ユーザの[Safari](../Page/Safari.md "wikilink")のブックマークデータや.Macの購読サービスなどとの同期にも対応していた。

Mac OS X 10.4以降では、この機能のうちの大部分はSyncサービス・フレームワークに移行され、他の開発者が自身のアプリケーション用の同期機能に使えるようになった。iSyncはモバイル・ハンドセット、Palm OSオーガナイザー、互換スマートフォンなどへの対応機能は残している。iTunes 4.8のリリースにともない、Mac OS X v10.4以降におけるiPodの同期機能は[iTunes](https://ja.wikipedia.org/wiki/iTunes "wikilink")に移管された。.Macとの同期機能は現在では.Mac Syncが対応するようになっており、システム設定ペインからアクセスできる。[Mac OS X Lionでは](https://ja.wikipedia.org/wiki/Mac_OS_X_Lion "wikilink")[OSに同梱されなくなったため](../Page/オペレーティングシステム.md "wikilink")、現時点では[Mac OS X v10.6に同梱の](../Page/Mac_OS_X_v10.6.md "wikilink")3.1が最新版である。
