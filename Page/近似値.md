**近似値**（きんじち）とは、必要とされる[誤差](../Page/誤差.md "wikilink")の範囲内で、ある[数](../Page/数.md "wikilink")を表していると思って構わない数値のこと。あるいはある数の情報を一部削って得られる値、すなわちある数値に対して[端数処理](../Page/端数処理.md "wikilink")を施した値（数値を「丸め」たもの）である。

## 代表例

### 円周率

学校[教育などで](../Page/数学_\(教科\).md "wikilink")[円周率](https://ja.wikipedia.org/wiki/円周率 "wikilink") [π](https://ja.wikipedia.org/wiki/π "wikilink") の値として用いられる 3.14 が、近似値の最も代表的な例である。円周率は広く知られている代表的な[無理数](../Page/無理数.md "wikilink")であり、整数の[商として表されることは決してない](../Page/除法.md "wikilink")。すなわち、その小数表示は有限桁で途切れたり循環したりすることはなく、

\[\pi=3.1415926535897932384626433832795\cdots\] () のようになる。

この円周率の近似値として、3.14 や 3.14159 が度々用いられるが、このような「区切りのいい数字」で書かれる値ばかりが近似値ではない。例えば、[アルキメデス](../Page/アルキメデス.md "wikilink")が正九十六角形を用いて円周率の詳しい値を計算したという話は有名であるが、それにより円周率の近似値 [{{sfrac](https://ja.wikipedia.org/wiki/22/7 "wikilink") や  が得られる。さらに精度の高い近似値として  が用いられる。他にも √ や √ + √ などの無理数を円周率の近似値として用いることもある。(平方根は[代数的数](../Page/代数的数.md "wikilink")なので、[超越数](../Page/超越数.md "wikilink")である円周率よりはまだ計算に向いているため、このような近似も意味があるわけである。)

### 平方根

[平方数](../Page/平方数.md "wikilink")でない整数の[平方根](../Page/平方根.md "wikilink")も無理数であり、現実的な計算に用いる時にはしばしば近似値が用いられる。[2 の正の平方根](../Page/2の平方根.md "wikilink") √ の場合ではその小数展開が

\[\sqrt{2}=1.4142135623730950488016887242097\cdots\] となるため、一般的に[十進法](../Page/十進法.md "wikilink")では 1.41 や 1.414 などが近似値として用いられる。同様に、√ は 1.732、√ は 2.236 などが近似値として使われる。

## 数値計算における近似値の必要性

上記に挙げた例のように、有限の資源で表示できない値では正確な計算ができない、または即時に結果が出ないということが起きる。例えば[コンピュータ](../Page/コンピュータ.md "wikilink")で計算する場合は、[桁あふれが発生し全体の処理に影響を及ぼすことがある](https://ja.wikipedia.org/wiki/算術オーバーフロー "wikilink")。

この問題を解消するためにある程度まで情報を削除し、計算を簡略させるために近似値を用いる。

当然、得られた結果は正確なものではなく、本来の数値からは誤差を生ずる。したがって、同時に誤差の評価もきちんと行うことが要求される。

## 異なるN進法での互換

異なる[N進法に置き換えた場合でも](https://ja.wikipedia.org/wiki/位取り記数法 "wikilink")、近似値は発生する。特に、[十進法](../Page/十進法.md "wikilink")では割り切れない「[1/3](https://ja.wikipedia.org/wiki/1/3 "wikilink")」や「[1/9](https://ja.wikipedia.org/wiki/1/9 "wikilink")」を、[六進法](../Page/六進法.md "wikilink")や[十二進法](../Page/十二進法.md "wikilink")など「1/3」や「1/9」が割り切れるN進法に換算すると、近似値が以下のようになる。併わせて、六進法では割り切れない「[1/5](https://ja.wikipedia.org/wiki/1/5 "wikilink")」を、十進法や[二十進法](../Page/二十進法.md "wikilink")など「1/5」が割り切れるN進法に換算した近似値も掲載する。この他、「三の倍数」進法と「五の倍数」進法で近似値を出す場合には、[十二進法](../Page/十二進法.md "wikilink")⇔[二十進法](../Page/二十進法.md "wikilink")よりも、[六進法](../Page/六進法.md "wikilink")⇔[十進法](../Page/十進法.md "wikilink")で換算した方が循環節が短く、倍数の判別も易しい。

  - 十分の一（一[割](https://ja.wikipedia.org/wiki/割 "wikilink")）

([10](../Page/10.md "wikilink"))－1 = [9](../Page/9.md "wikilink") = [3](../Page/3.md "wikilink") なので、「三分の一」が割り切れるN進法では、「[十分の一](https://ja.wikipedia.org/wiki/1/10 "wikilink")」の近似値は「[九分の一](https://ja.wikipedia.org/wiki/1/9 "wikilink")」になる。

  - 十分の一
      - 十進法：1÷10 = 0.1
      - 六進法：1÷(14) = 0.0<u>3333</u>…
      - 二十進法：1÷A = 0.2
      - 十二進法：1÷A = 0.1<u>2497</u>…
  - 九分の一
      - 十進法：1÷9 = 0.<u>1111</u>…
      - 六進法：1÷(13) = 0.04
      - 二十進法：1÷9 = 0.<u>248 HFB</u>…
      - 十二進法：1÷9 = 0.14

  - 三割

「三分の一」が割り切れるN進法では、十進法「[十分の三](https://ja.wikipedia.org/wiki/3/10 "wikilink")」の近似値は、「[二十七分の](../Page/27.md "wikilink")[八](https://ja.wikipedia.org/wiki/8 "wikilink")」(= 2/3) または「三分の一」を意味する小数になるが、「二十七分の八」の方が近い。

  - 十分の三 = 二十分の六
      - 十進法：3÷10 = 0.3
      - 六進法：3÷(14) = 0.1<u>4444</u>…
      - 二十進法：3÷A = 0.6
      - 十二進法：3÷A = 0.3<u>7249</u>…
  - 二十七分の八（2÷3）
      - 十進法：8÷27 = 0.<u>296</u>…
      - 六進法：(12)÷(43) = 0.144
      - 二十進法：8÷(17) = 0.<u>5IA 782 J53 E19 CBH 0EG</u>…
      - 十二進法：8÷(23) = 0.368
  - 三分の一
      - 十進法：1÷3 = 0.<u>3333</u>…
      - 六進法：1÷3 = 0.2
      - 二十進法：1÷3 = 0.<u>6D6D</u>…
      - 十二進法：1÷3 = 0.4

  - 3パーセント

3[パーセント](https://ja.wikipedia.org/wiki/百分率 "wikilink")（十進法の0.03）を十進法の[単位分数](https://ja.wikipedia.org/wiki/単位分数 "wikilink")にすると 1/[33](../Page/33.md "wikilink").333… になるため、1/3が割り切れるN進法では、3パーセントの近似値は 1/[36](../Page/36.md "wikilink") の小数になる。十進法の100÷36 = 2.777…である「2と77/100」を「2と77/99」に丸めると、約分した値が「2と7/9」になる。

  - 100÷36
      - 十進法：100÷36 = 2.<u>777</u>…
      - 六進法：(244)÷(100) = 2.44
      - 二十進法：(50)÷(1G) =2.<u>FB2 48H</u>…
      - 十二進法：(84)÷(30) = 2.94
  - 1/36
      - 十進法：1÷36 = 0.02<u>777</u>…
      - 六進法：1÷(100) = 0.01
      - 二十進法：1÷(1G) = 0.0<u>B24 8HF</u>…
      - 十二進法：1÷(30) = 0.04

<!-- end list -->

  - 56パーセント

4[パーセントは十進法で](https://ja.wikipedia.org/wiki/百分率 "wikilink")1/[25](../Page/25.md "wikilink")になるので、4の倍数ではm/25となる。このため、十進法の「56/100」、これを約分した「[14](../Page/14.md "wikilink")/25」の近似値は、五桁で[311](../Page/311.md "wikilink")の倍数で「4354/[7776](https://ja.wikipedia.org/wiki/7776 "wikilink")」を意味する小数になり、四桁に縮めると[52](../Page/52.md "wikilink")の倍数で「728/[1296](https://ja.wikipedia.org/wiki/1296 "wikilink")」を意味する小数になる。1/3が割り切れるN進法では、「九分の五」または「十二分の七」が割り切れる近似値となるが、「九分の五」の方がより近い小数になる。

  - 二十五分の十四
      - 十進法：14÷25 = 0.56
      - 六進法：(22)÷(41) = 0.<u>32054</u>…
      - 二十進法：(E)÷(15) = 0.B4
      - 十二進法：(12)÷(21) = 0.<u>68781 B0591 5343A 0B62A</u>…
  - 九分の五
      - 十進法：5÷9 = 0.<u>555</u>…
      - 六進法：5÷(13) = 0.32
      - 二十進法：5÷9 = 0.<u>B24 8HF</u>…
      - 十二進法：5÷9 = 0.68
  - 十二分の七
      - 十進法：7÷12 = 0.58<u>333</u>…
      - 六進法：(11)÷(20) = 0.33
      - 二十進法：7÷(C) = 0.B<u>D6D6</u>…
      - 十二進法：7÷(10) = 0.7
