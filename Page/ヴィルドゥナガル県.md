**ヴィルドゥナガル県（விருதுநகர் மாவட்டம் ; Virudhunagar district）**は、[インド](../Page/インド.md "wikilink")南部の[タミル・ナードゥ州](https://ja.wikipedia.org/wiki/タミル・ナードゥ州 "wikilink")に属する30の県の一つ。県庁所在地は[ヴィルドゥナガル](https://ja.wikipedia.org/wiki/ヴィルドゥナガル "wikilink")。

## 概要

[2001年](../Page/2001年.md "wikilink")の国勢調査では、面積4283平方キロメートル、人口175万1301人、人口密度は409人/[km<sup>2</sup>](https://ja.wikipedia.org/wiki/平方キロメートル "wikilink")。 北は[マドゥライ県](https://ja.wikipedia.org/wiki/マドゥライ県 "wikilink")、北東は[シヴァガンガイ県](../Page/シヴァガンガイ県.md "wikilink")、東は[ラーマナータプラム県](../Page/ラーマナータプラム県.md "wikilink")、南東は[トゥーットゥックディ県](https://ja.wikipedia.org/wiki/トゥーットゥックディ県 "wikilink")、南西は[ティルネルヴェーリ県](https://ja.wikipedia.org/wiki/ティルネルヴェーリ県 "wikilink")に接し、西で僅かに[テーニ県](../Page/テーニ県.md "wikilink")と[ケーララ州](../Page/ケーララ州.md "wikilink")に接する。

## 歴史

[1985年](https://ja.wikipedia.org/wiki/1985年 "wikilink")3月8日に[ラーマナータプラム県](../Page/ラーマナータプラム県.md "wikilink")から[カーマラージャル県](https://ja.wikipedia.org/wiki/カーマラージャル県 "wikilink")として分離して誕生。知事の名を県名にしてはならないとするタミル・ナードゥ州政府の通達を受け、[1996年](../Page/1996年.md "wikilink")に現在の名称に改称した。

## 行政区分

### 郡

ヴィルドゥナガル県は、以下の8つの郡（[タミル語](../Page/タミル語.md "wikilink") வட்டம் 「円」 ; 英訳 taluk）に区分けされている。

  - ティルッチュリ郡（திருச்சுழி ; Tiruchuli）
  - カーリヤーッパッティ郡（காரியாப்பட்டி ; Kariapattai）
  - アルップッコーッタイ郡（அருப்புக்கோட்டை ; Arupukottai）
  - ヴィルドゥナガル郡（விருதுநகர் ; Virudhunagar）
  - サーットゥール郡（சாத்தூர் ; Sathur）
  - シヴァカーシ郡（சிவகாசி ; Sivakasi）
  - ティルヴィッリプットゥール郡（திருவில்லிபுத்தூர் ; Srivilliputhur）
  - ラージャパーライヤム郡（இராஜபாளையம் ; Rajapalayam）

### 区

ヴィルドゥナガル県は、以下の11の区（英訳 block）に区分けされている。

  - ナリックディ区（நரிக்குடி ; Narikudi）
  - カーリヤーッパッティ区（காரியாப்பட்டி ; Kariapattai）
  - ティルッチュリ区（திருச்சுழி ; Tiruchuli）
  - アルップッコーッタイ区（அருப்புக்கோட்டை ; Arupukottai）
  - ヴィルドゥナガル区（விருதுநகர் ; Virudhunagar）
  - サーットゥール区（சாத்தூர் ; Sathur）
  - シヴァカーシ区（சிவகாசி ; Sivakasi）
  - ヴェンバッコーッタイ区（வெம்பக்கோட்டை ; Vembakottai）
  - ヴァッティラーイルップ区（வத்திராயிருப்பு ; Watrap）
  - ティルヴィッリプットゥール区（திருவில்லிபுத்தூர் ; Srivilliputhur）
  - ラージャパーライヤム区（இராஜபாளையம் ; Rajapalayam）
