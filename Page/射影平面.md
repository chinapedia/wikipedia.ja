[Railroad-Tracks-Perspective.jpg](https://ja.wikipedia.org/wiki/File:Railroad-Tracks-Perspective.jpg "fig:Railroad-Tracks-Perspective.jpg")で交わる。\]\] [数学](../Page/数学.md "wikilink")における**射影平面**（しゃえいへいめん、）とは、初等的な[平面](../Page/平面.md "wikilink")の概念を拡張する幾何学的な構成のことである。通常の平面においては、二直線は典型的には一つの交点を持つが、特定の直線の組（平行線）は交点を持たない。一方、射影平面においては、通常の平面に「無限遠点」が追加され、平行線は無限遠点で交点を持つ。従って、射影平面では**任意**の相異なる二直線がただ一点において交わる。

射影平面の定義としてよく用いられるものが二種類ある。ひとつは[線型代数学](../Page/線型代数学.md "wikilink")から来るもので、この場合の射影平面は、適当なに対する[等質空間](https://ja.wikipedia.org/wiki/等質空間 "wikilink")として与えられる。この場合の重要な例として、\[1\]\[2\]  および  が挙げられる。もうひとつは、もっと一般のおよび[有限幾何学](https://ja.wikipedia.org/wiki/有限幾何学 "wikilink")の立場により与えられる定義である。これは[平面幾何学](https://ja.wikipedia.org/wiki/平面幾何学 "wikilink")の[接続的性質の研究に適している](https://ja.wikipedia.org/wiki/接続_\(幾何学\) "wikilink")。

射影平面の概念は、もっと高次元の[射影空間](../Page/射影空間.md "wikilink")の概念に一般化される。射影平面は二次元の射影空間である。

## 線型代数学的な定義

線型代数学的には、射影平面は「三次元空間内の原点を通る直線全体の成す集合」として与えられる。射影平面上の直線は三次元空間内の原点を通る平面から生じる。きちんと述べれば、以下のようになる\[3\]。

*K* を任意の[可除環](https://ja.wikipedia.org/wiki/可除環 "wikilink")（斜体）とし、 *K* <sup>3</sup> を *K* の元の三つ組 *x* = (*x* <sub>0</sub>, *x* <sub>1</sub>, *x* <sub>2</sub>) 全体の成す集合（[直積集合](../Page/直積集合.md "wikilink")）とする。*K* <sup>3</sup> の零ベクトルでない任意の点 *x* に対し、原点と *x* を通る *K* <sup>3</sup> 内の「直線」とは、 *K* <sup>3</sup> の部分集合

\[\{k x : k \in K\}\] のことである。同様に *K* <sup>3</sup> の線型独立な点 *x*, *y* （つまり *kx* + *ly* = 0 ならば必ず *k* = *l* = 0）に対し、原点と *x*, *y* を通る「平面」とは、 *K* <sup>3</sup> の部分集合

\[\{k x + l y : k, l \in K\}\] のことであり、この平面は無数の直線を含む。

可除環 *K* 上の**射影平面** ''K***P**<sup>2</sup> とは、*K'' <sup>3</sup> の原点を通る直線全体の成す集合をいう。*K***P**<sup>2</sup> の部分集合 ''L'' が、射影平面 ''K***P**<sup>2</sup> 内の（射影）**直線**であるとは、*K'' <sup>3</sup> における平面で、それが含む直線全体の成す集合が *K***P**<sup>2</sup> においてちょうど *L* と一致するものが存在するときにいう。

少し異なる定義の仕方もあって、射影平面というのは集合 *K* <sup>3</sup> ∖ {(0, 0, 0)} を

\[x \sim k x,\quad k \in K\] で与えられる[同値関係](../Page/同値関係.md "wikilink")で割ったものである、ということもできる。この場合も射影平面内の直線は先ほどとまったく同じように定義できる。*K* が[位相空間](https://ja.wikipedia.org/wiki/位相空間 "wikilink")ならば *K***P**<sup>2</sup> にも（[直積位相](https://ja.wikipedia.org/wiki/積位相 "wikilink")、[部分空間の位相](../Page/相対位相.md "wikilink")、[商位相](https://ja.wikipedia.org/wiki/商位相 "wikilink")を通じて）内在的な位相が入る。

*K***P**<sup>2</sup> における座標系 (''x'' <sub>0</sub>, ''x'' <sub>1</sub>, ''x'' <sub>2</sub>) は**斉次座標系*'  と呼ばれる。各三つ組 (*x* <sub>0</sub>, *x* <sub>1</sub>, *x* <sub>2</sub>) は ''K***P'''<sup>2</sup> の点を[矛盾無く表すが](https://ja.wikipedia.org/wiki/well-defined "wikilink")、三つ組 (0, 0, 0) だけは例外で ''K***P**<sup>2</sup> のどの点にも対応しない。*K'' が[有限体](https://ja.wikipedia.org/wiki/有限体 "wikilink")でない限り *K***P**<sup>2</sup> の各点に対応する三つ組は無数に存在しうる。

### 例

  - *K* として[実数](../Page/実数.md "wikilink")体 **R** を取れば、[実射影平面](https://ja.wikipedia.org/wiki/実射影平面 "wikilink") **RP**<sup>2</sup> が生じる。これは位相幾何学において、向きを持たない実二次元の[多様体](../Page/多様体.md "wikilink")の基本的な例を与えるものである\[4\]。

<!-- end list -->

  - *K* として[複素数](../Page/複素数.md "wikilink")体 **C** を取れば、[複素射影平面](https://ja.wikipedia.org/wiki/複素射影平面 "wikilink") **CP**<sup>2</sup> が生じる。これは複素二次元の閉多様体であり、従って向きを持つ実四次元の多様体である。他の[体上の射影平面ともども](https://ja.wikipedia.org/wiki/可換体 "wikilink")[代数幾何学](../Page/代数幾何学.md "wikilink")の基本的な例を与える\[5\]。

<!-- end list -->

  - [四元射影平面](https://ja.wikipedia.org/wiki/四元射影平面 "wikilink")もまた別な意義を持つ対象である。[ケーリー平面](https://ja.wikipedia.org/wiki/ケーリー平面 "wikilink")は[八元数](https://ja.wikipedia.org/wiki/八元数 "wikilink")環上の射影平面と考えられるが、八元数環が斜体を成さないため、きちんとした構成を十分に記述することはできない\[6\]。

<!-- end list -->

  - *K* として位数 *p* <sup>*n*</sup> の[有限体](https://ja.wikipedia.org/wiki/有限体 "wikilink")を取れば、*p* <sup>2*n*</sup> + *p* <sup>*n*</sup> + 1 個の点を持つ射影平面が得られる。後述するファノ平面は *p* <sup>*n*</sup> = 2 とした場合にあたる。

### 通常平面との関係

体 *K* 上の通常の平面 *K* <sup>2</sup> は射影平面 *K***P**<sup>2</sup> へ写像

\[(x_1, x_2) \mapsto (1, x_1, x_2)\]

によって埋め込こまれる。この写像の像の補集合は (0, *x* <sub>1</sub>, *x* <sub>2</sub>) なる形の点全体の成す集合であり、このような埋め込みが与えられているという観点によって、補集合の点は[無限遠点](../Page/無限遠点.md "wikilink")を表している。無限遠点の全体は *K***P**<sup>2</sup> における直線を成す（つまり、この直線は *K* <sup>3</sup> における平面

\[\{k (0, 0, 1) + l (0, 1, 0) : k, l \in K\}\]

から生じる）。直観的には、無限遠点というのは平行線の交わる所としての「余分な」点であり、点 (0, *x* <sub>1</sub>, *x* <sub>2</sub>) というのは傾きが *x* <sub>2</sub>/*x* <sub>1</sub> であるような直線すべての交点に対応する。例えば、通常の平面 *K* <sup>2</sup> における二直線

\[\begin{align}
 a &= \{(x_1, 0) : x_1 \in K\},\\
 b &= \{(x_1, 1) : x_1 \in K\}
\end{align}\]

を考えれば、これらの傾きはともに 0 であってこれらは交わらない。これらを先ほどの埋め込みによって ''K'**'P**<sup>2</sup> の部分集合と見なせば、これらは *K***P**<sup>2</sup> における直線とはならないが、それぞれに点 (0, 1, 0) を加えた

\[\begin{align}
 \bar a = \{(1, x_1, 0) : x_1 \in K\} \cup \{(0, 1, 0)\}\\
 \bar b = \{(1, x_1, 1) : x_1 \in K\} \cup \{(0, 1, 0)\}
\end{align}\] は *K***P**<sup>2</sup> における直線となる。<span style="text-decoration:overline>''a''</span> は ''K''<sup>3</sup> における平面
:<math>\{k (1, 0, 0) + l (0, 1, 0) : k, l \in K\}</math>
から生じ、<span style="text-decoration:overline>*b*</span> は平面

\[\{k (1, 0, 1) + l (0, 1, 0) : k, l \in K\}\] から生じる。これらの射影直線 <span style="text-decoration:overline>''a''</span>, <span style="text-decoration:overline>*b*</span> は点 (0, 1, 0) において交わる。実は、*K* <sup>2</sup> における傾き 0 の直線はすべて、この方法で射影化したとき、*K***P**<sup>2</sup> の点 (0, 1, 0) において交わる。

先ほど与えた平面 *K* <sup>2</sup> の射影平面 *K***P**<sup>2</sup> への埋め込みは一意ではなく、それぞれの埋め込みごとにその無限遠点となる点は変わってくる。例えば、埋め込み

\[(x_1, x_2) \mapsto (x_2, 1, x_1)\] を考えれば、像の補集合に属する (*x* <sub>0</sub>, 0, *x* <sub>2</sub>) なる形の点が無限遠点と見なされる。

### 射影変換

射影平面 ''K***P**<sup>2</sup> における（射影）変換とは、*K'**'P**<sup>2</sup> からそれ自身への可逆な写像であって、直線を直線に写すものをいう。斉次座標系を用いれば、これは 3-次[正則行列](../Page/正則行列.md "wikilink") *M* を用いて

  -
    \(y = Mx\)

と書くことができる。二つの行列が同じ射影変換を定めるのは、一方が他方の定数倍であるときに限る。従って、[射影変換全体の成す群は](../Page/射影線型群.md "wikilink")[一般線型群](https://ja.wikipedia.org/wiki/一般線型群 "wikilink")の剰余群である。

### 高次元化

いくつか線型代数学的な定義を挙げたが、いずれもより高次元の場合に拡張するのは容易である。*d*-次元[射影空間](../Page/射影空間.md "wikilink") ''K***P**<sup>*d*</sup> は、*K'' <sup>*n*+1</sup> 内の原点を通る直線全体の成す集合であり、*K***P**<sup>*d*</sup> 内の直線は *K* <sup>*n*+1</sup> の原点を通る平面に対応する。射影空間の概念はさらに[グラスマン空間の概念へ一般化することができる](https://ja.wikipedia.org/wiki/グラスマン多様体 "wikilink")。

## 組合せ論的な定義

より一般な組合せ論的定義によれば、**射影平面**は**直線**の集合と**点**の集合から成り、点と直線との間の**結合**あるいは**接続**  と呼ばれる以下のような性質を持つ関係を備えるものである。

1.  任意の異なる二点に対し、それらを接続する直線がただ一つ存在する。
2.  任意の異なる二直線に対し、それらのいずれとも接続する点がただ一つ存在する。
3.  平面上の四点で、そのうちの二点よりも多くに接続するような直線は一つも存在しない、というものが存在する。

条件2は[平行線](https://ja.wikipedia.org/wiki/平行線 "wikilink")が存在しないことを意味する。また条件3は退化する場合（[後述](https://ja.wikipedia.org/wiki/#退化平面 "wikilink")）を除くためだけにある。

### 性質

射影平面においては、それが含む直線の数と点の数とが同じであることを示すことができる（有限でも無限でも）。有限射影平面は

  - *N*<sup>2</sup> + *N* + 1 個の点と、
  - *N*<sup>2</sup> + *N* + 1 本の直線を持ち、
  - 各直線上に *N* + 1 個の点が載っていて、
  - 各点を *N* + 1 本の直線が通る。

ここで、*N* ≥ 2 は射影平面の**位数**  と呼ばれる[整数](../Page/整数.md "wikilink")である（[有限幾何学](https://ja.wikipedia.org/wiki/有限幾何学 "wikilink")も参照）。

前節に述べたような線型代数学的定義から生じる射影平面はどれも、本節に言う組合せ論的な定義に基づく射影平面として記述することができる。従って、位数 *p*<sup>*n*</sup> の有限体から、位数 *N* = *p*<sup>*n*</sup> の射影平面を与えることができる。

[fano_plane.svg](https://ja.wikipedia.org/wiki/File:fano_plane.svg "fig:fano_plane.svg") [ファノ平面](https://ja.wikipedia.org/wiki/ファノ平面 "wikilink")は二元体から生じる射影平面である。これは最小の射影平面で、わずか七つの点と七つの直線から成る。右図では、七つの「点」を小さな黒丸で表し、七つの「直線」は六本の線分と一つの円で表したものだが、[射影平面の双対性](https://ja.wikipedia.org/wiki/射影平面の双対性 "wikilink")により（点と直線とを互いに入れ替えたものもやはり射影平面となるから）、黒丸を「直線」、線分と円を「点」と思うこともできる。この七つの点に対して置換を施すと、[共線](https://ja.wikipedia.org/wiki/共線 "wikilink")点（同一直線上にある点）を共線点へ移す変換を引き起こす。このことを以って、ファノ平面は「対称」であるという。

### 一般化された座標

組合せ論的に定義される任意の射影平面に対して、[平面三項環](https://ja.wikipedia.org/wiki/平面三項環 "wikilink")と呼ばれる座標「環」（ただし、本当に[環となっているとは限らない](https://ja.wikipedia.org/wiki/環_\(数学\) "wikilink")）を対応させることができる。平面三項環は体または斜体になっている必要はなく、また斜体から構成することができない射影平面というものも多く存在する。そのような射影平面は[非デザルグ的な射影平面と呼ばれ](https://ja.wikipedia.org/wiki/非デザルグ平面 "wikilink")、未だ活発な研究の成されている研究対象である。

この平面三項環が持つ代数的性質は、それに対応して平面が持つ幾何学的な接続関係の性質に読み替えることができる。例えば、[デザルグの定理](../Page/デザルグの定理.md "wikilink")に対応するのは座標環が[斜体から得られるということであり](../Page/斜体_\(数学\).md "wikilink")、また[パップスの定理](https://ja.wikipedia.org/wiki/パップスの定理 "wikilink")が対応するのは座標環が[可換体](https://ja.wikipedia.org/wiki/可換体 "wikilink")から得られるということである。あるいは（たとえば[八元数](https://ja.wikipedia.org/wiki/八元数 "wikilink")体のように）必ずしも[結合的でない](../Page/結合法則.md "wikilink")[交代的な可除代数から得られるようなものには](https://ja.wikipedia.org/wiki/交代代数 "wikilink")[ムーファン平面](https://ja.wikipedia.org/wiki/ムーファン平面 "wikilink")が対応する。

有限射影平面においてデザルグの定理がパップスの定理を含むという純粋に幾何学的な主張の、しかし唯一知られている証明は、斜体に成分を持つ座標をとって[ウェダーバーンの定理](https://ja.wikipedia.org/wiki/ウェダーバーンの定理 "wikilink")（有限可除環は必ず可換体である）を用いるという代数的な方法によってなされる（逆にパップスの定理がデザルグの定理を含むことは、（有限とは限らない）任意の射影平面において真であり、しかもこれは幾何学的に証明することができる）。

### 素数位数の平面の構成

素数位数 *N* の射影平面は、位数 *N* の有限体上の線型代数学的構成を用いて構成することができる。あるいは以下のようにしても構成できる。

  -
    一点 *P* を用意する。
    *N* 個の点を用意して、*c* = 0, ..., *N* − 1 に応じて、*P*(*c*) と書く。
    同様に *N*<sup>2</sup> 個の点を *P*(*r*, *c*) (*r*, *c* = 0, ..., *N* − 1) とする。

これらの点に対して、以下のように直線を構成する

  -
    一本の直線が *L* = {*P*, *P*(0), ..., *P*(*N* − 1)} で与えられる。
    *N* 本の直線を、*c* = 0, ..., *N* − 1 に応じて、*L*(*c*) = {*P*, *P*(0, *c*), ..., *P*(*N* − 1, *c*)} とする。
    *N*<sup>2</sup> 本の直線を *L*(*r*, *c*) = {*P*(*c*) および P((*r* + *ci*) mod *N*, *i*)} (*i*, *r*, *c* = 0, ..., *N* − 1) とする。

ただし、ここで書いた式

  -
    (*r* + *ci*) mod *N*

で *i* が 0 から *N* − 1 までの全ての値を亘るのは、*N* が素数の場合に限る。

この構成から、二つの[退化平面が生じる](https://ja.wikipedia.org/wiki/#退化平面 "wikilink")。すなわち、*N* = 0 に対してただ一つの点がただ一つの直線に接続したものが得られ、*N* = 1 に対して三点と三直線からなる三角形が得られる。素数 *N* に対しては、この方法で構成される任意の平面が[上記定義の条件3を満足する](https://ja.wikipedia.org/wiki/#組合せ論的な定義 "wikilink")。

例えば *N* = 2 とすると、

  -
    一直線 *L* = { *P*, *P*(0), *P*(1)}
    二直線 *L*(*c*) = {*P*, *P*(0,*c*), *P*(1,*c*)} (*c* = 0, 1)
    四直線 *L*(*r*, *c*) = {*P*(*c*), P((*r* + *ci*) mod *2*, *i*) | *i* = 0, 1} (*r*, *c* = 0, 1)

となる。

### 有限位数の存在

既に説明したように、各素数冪 *p*<sup>*n*</sup> に対して同位数の射影平面が存在する。事実として、「知られている」全ての有限射影平面はその位数が素数冪である。

それ以外の位数についても有限射影平面が存在するかどうかというのは、未解決の問題である（有限射影平面の基本予想 ）。位数に関する一般的な制限として知られているのは、位数 *N* が法 4 に関して 1 または 2 と合同ならば、それは二つの平方数の和にならなければいけないという[ブルック＝ライザー＝チョウラの定理](https://ja.wikipedia.org/wiki/ブルック＝ライザー＝チョウラの定理 "wikilink")である。これにより *N* = 6 が除外できる。次の場合は *N* = 10 が、大規模計算機の計算により除外された。それ以上の場合については知られていない（特に *N* = 12 も未解決である）。

位数 *N* の射影平面が存在するための必要十分条件は、位数 *N* の[アフィン平面が存在することである](https://ja.wikipedia.org/wiki/アフィン平面_\(接続幾何\) "wikilink")。位数 *N* のアフィン平面がただ一つ存在するならば、位数 *N* の射影平面もただ一つ存在するが、逆は必ずしも真でない。

位数 *N* の射影平面はスタイナーの S(2, *N* + 1, *N*<sup>2</sup> + *N* + 1)-系である（[スタイナー系](https://ja.wikipedia.org/wiki/スタイナー系 "wikilink")参照）。逆に *N* ≥ 2 に対するこの形のスタイナー系が射影平面となることが証明できる。

位数 *N* の[互いに直交するラテン方格の総数は高々](https://ja.wikipedia.org/wiki/直交ラテン方格 "wikilink") *N* − 1 である。これが *N* − 1 となりうる必要十分条件は、その位数の射影平面が存在することである。

一方、射影平面の分類は全然終わっていない。いくつかの結果を位数の順に以下に示す。

  - 2 : 全て PG(2,2) に同型
  - 3 : 全て PG(2,3) に同型
  - 4 : 全て PG(2,4) に同型
  - 5 : 全て PG(2,5) に同型
  - 6 : この位数の射影平面は存在しない（[オイラーの](../Page/レオンハルト・オイラー.md "wikilink")として、[タリーにより示された](https://ja.wikipedia.org/wiki/ガストン・タリー "wikilink")）。
  - 7 : 全て PG(2,7) に同型
  - 8 : 全て PG(2,8) に同型
  - 9 : PG(2,9) および三種類の異なる（同型でない）非デザルグ平面
  - 10 : この位数の射影平面は存在しない（計算機による膨大な計算の結果として証明された）。
  - 11 : すくなくとも PG(2,11) が挙げられる。他は知られていないが可能性はある。
  - 12 : この位数の射影平面は存在しないと予想されているが証明はされていない。

### 退化平面

退化平面とは、上述の定義における[条件3を満足しない場合である](https://ja.wikipedia.org/wiki/#組合せ論的な定義 "wikilink")。退化平面には二つの系列がある。

1.  任意個数の点 *P*<sub>1</sub>, ..., *P*<sub>*n*</sub> および直線 *L*<sub>1</sub>, ..., *L*<sub>*m*</sub> が以下を満たすもの。
      -
        *L*<sub>1</sub> = { *P*<sub>1</sub>, *P*<sub>2</sub>, ..., *P*<sub>*n*</sub>}
        *L*<sub>2</sub> = { *P*<sub>1</sub> }
        *L*<sub>3</sub> = { *P*<sub>1</sub> }
        …
        *L*<sub>*m*</sub> = { *P*<sub>1</sub> }
2.  任意個数の点 *P*<sub>1</sub>, ..., *P*<sub>*n*</sub> および直線 *L*<sub>1</sub>, ..., *L*<sub>*n*</sub> が以下を満たすもの（点の数と直線の数は同数である）。
      -
        *L*<sub>1</sub> = { *P*<sub>2</sub>, *P*<sub>3</sub>, ..., *P*<sub>*n*</sub> }
        *L*<sub>2</sub> = { *P*<sub>1</sub>, *P*<sub>2</sub> }
        *L*<sub>3</sub> = { *P*<sub>1</sub>, *P*<sub>3</sub> }
        …
        *L*<sub>*n*</sub> = { *P*<sub>1</sub>, *P*<sub>*n*</sub> }

### 高次元化

接続関係を用いた定義も、二次元より高次元の射影幾何における対応物を考えることができる。高次元化したものは、平面の場合ほどには興味深い性質を示さず（平面の場合よりもよく振舞うといってもよいが）、つまり斜体上の古典的な射影空間となる。そうなる理由は別な場所で述べるほうが適切であろうけれども、高次元射影空間における接続の性質を用いて幾何学的に[デザルグの定理](../Page/デザルグの定理.md "wikilink")を示せる（つまり、座標環が斜体となる）ことによる。
