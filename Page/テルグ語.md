**テルグ語**（テルグご、*తెలుగు* Telugu）は、[ドラヴィダ語族](../Page/ドラヴィダ語族.md "wikilink")に属する言語で、[インド](../Page/インド.md "wikilink")南東部の[アーンドラ・プラデーシュ州](../Page/アーンドラ・プラデーシュ州.md "wikilink")および[テランガーナ州](../Page/テランガーナ州.md "wikilink")の[公用語](../Page/公用語.md "wikilink")である。[タミル・ナードゥ州](../Page/タミル・ナードゥ州.md "wikilink")、[カルナータカ州](../Page/カルナータカ州.md "wikilink")などでも話され、ドラヴィダ語族の諸言語のなかでは、約8,000万人ともっとも多数の使用者がいる。またインド連邦レベルでも[憲法の](https://ja.wikipedia.org/wiki/インド憲法 "wikilink")[第8付則に定められた22の指定言語のひとつである](../Page/インドの公用語の一覧.md "wikilink")。しかし言語的求心性が弱く、標準化も進んでいない\[1\]。

## 文字

[ブラーフミー文字](../Page/ブラーフミー文字.md "wikilink")より発展した[テルグ文字](https://ja.wikipedia.org/wiki/テルグ文字 "wikilink")を使用する。ブラーフミー文字は元々[サンスクリット](../Page/サンスクリット.md "wikilink")を表記するための文字であったため、発音に無用な混乱が生じている（音声上同じ音であっても、それらを表記するテルグ文字が異なる場合には発音上でもそれらを区別することが求められたので、そのような単語がテルグ文字で書かれている場合、それを読む者によって違う発音となることがある）。

[フランス語](https://ja.wikipedia.org/wiki/フランス語 "wikilink")の[リエゾン](../Page/リエゾン.md "wikilink")をもっと一般化したような「[連音](../Page/連音.md "wikilink")」があり、発音で聞くと、個々の単語の境界が不明である。[19世紀](../Page/19世紀.md "wikilink")の[イギリス](https://ja.wikipedia.org/wiki/イギリス "wikilink")人が東洋の[イタリア語](../Page/イタリア語.md "wikilink")と呼んだように、[母音](../Page/母音.md "wikilink")に特徴があり、母音の数は20以上あるとはいえ、実用的に使用される主要な単母音は4種類で、日本人には馴染みのある発音しやすい言葉である\[2\]。

## 歴史と文法構造

[ドラヴィダ語を代表すると共に](../Page/ドラヴィダ語族.md "wikilink")、外来の要素が最も少なくその古典的な形を維持しているのは、インド最南端とも言える場所に位置する[タミル・ナードゥ州](../Page/タミル・ナードゥ州.md "wikilink")の公用語である[タミル語](../Page/タミル語.md "wikilink")である。テルグ語は、ドラヴィダ語ではあっても、西暦1世紀頃に、[南インド](../Page/南インド.md "wikilink")に勢力を確立した[アーリヤ系の](../Page/アーリア人.md "wikilink")[アーンドラ朝](https://ja.wikipedia.org/wiki/アーンドラ朝 "wikilink")（サータヴァーハナ朝）の文化影響を大きく受けている。その文字ともあいまって、[サンスクリット語](https://ja.wikipedia.org/wiki/サンスクリット語 "wikilink")からの語彙の借用や造語がきわめて多い。また[ヒンドゥスターニー語](../Page/ヒンドゥスターニー語.md "wikilink")（[ヒンディー語](../Page/ヒンディー語.md "wikilink")、[ウルドゥー語](../Page/ウルドゥー語.md "wikilink")）や[ペルシア語](https://ja.wikipedia.org/wiki/ペルシア語 "wikilink")、[アラビア語](../Page/アラビア語.md "wikilink")からの語の混入もある。

また[北インド](https://ja.wikipedia.org/wiki/北インド "wikilink")にイスラム諸王国が築かれると共に、逆に南における[ヒンドゥー文化](https://ja.wikipedia.org/wiki/ヒンドゥー文化 "wikilink")の正統維持者の自負を持ち、[カースト制](https://ja.wikipedia.org/wiki/カースト制 "wikilink")を重視した。この結果、テルグ語の古典詩人たちは多く、サンスクリットでも作品を造ったため、テルグ語にはさらに多くの[サンスクリット](../Page/サンスクリット.md "wikilink")系の語彙が流入した。またこのようなヒンドゥー文化の重視のため、テルグ語話者自身にテルグ語伝統への求心性が薄く、多数の[方言](../Page/方言.md "wikilink")に分化して、20世紀になるまで[共通語](../Page/共通語.md "wikilink")の試みも自然的な収束もなかった。20世紀になってようやく共通語化が進められたが、それでも方言分化があり、逆に後退しているようにも見える。英語や近隣のドラヴィダ語などを話者が併用するため、膨大な使用人口を持ちながら、消滅へと向かっている可能性があるとさえ指摘する者もいる\[3\]。

テルグ語は、名詞の格変化を表現するのに、すべての[名詞](../Page/名詞.md "wikilink")の[単数形](https://ja.wikipedia.org/wiki/単数形 "wikilink")・[複数形](https://ja.wikipedia.org/wiki/複数形 "wikilink")に適用される決まった接尾音節を加える。この接尾音節は日本語の「[格助詞](https://ja.wikipedia.org/wiki/格助詞 "wikilink")」に似ている。そのため、[インド・ヨーロッパ語](https://ja.wikipedia.org/wiki/インド・ヨーロッパ語 "wikilink")のような[屈折言語](https://ja.wikipedia.org/wiki/屈折言語 "wikilink")というより、[日本語](../Page/日本語.md "wikilink")や、[アルタイ諸語](../Page/アルタイ諸語.md "wikilink")の言語に似ており、[膠着語](../Page/膠着語.md "wikilink")に近い。複数形を厳密に区別しなければならない点を除くと、日本人には理解しやすく修得しやすい文法構造である。

### 格変化

例えば、Ramudu （男性名詞，単数・主格）の屈折変化は次のようになるが、ここで示された「接尾音節」が、すべての名詞に対し、単数・複数に関係なく付いて、格を表示するというのは、屈折言語における格変化というより、日本語などの**膠着語**における「格助詞」に近いものだということになる。

|                                                          |              |  |             |
| -------------------------------------------------------- | ------------ |  | ----------- |
| [主格](../Page/主格.md "wikilink")　（－は）　：                    | Ramudu       |  | ( －du )     |
| [対格](../Page/対格.md "wikilink")　（－を） 　：                   | Ramuni       |  | ( －ni )     |
| [造格](https://ja.wikipedia.org/wiki/造格 "wikilink")　（－で）　： | Ramuniki     |  | ( －ki )     |
| [与格](../Page/与格.md "wikilink")　（－に）　：                    | Ramuniki     |  | ( －ki )     |
| [奪格](../Page/奪格.md "wikilink")　（－で）　：                    | Ramudininchi |  | ( －ninchi ) |
| [属格](../Page/属格.md "wikilink")　（－の）　：                    | Ramunidi     |  | ( －di )     |

### 語順

また、文章を構成する要素単語の語順で見ると、テルグ語では、「行為主体（**主格**）－行為対象（**対格**）－行為・述語詞」というような順序である。「ラーマはボールを打つ」という[日本語](../Page/日本語.md "wikilink")での言葉の語順は、「ラーマ**は**」－「ボール**を**」－「打つ」という風に、テルグ語と同じである。

  -
    Ramu**du**　banti**ni**　kottadu
    ラーマ**は**　ボール**を**　打つ

## 文化的背景

テルグ語の話者は、南インドの大穀倉地帯に住み、[米](../Page/米.md "wikilink")を主食とするので、文化的に日本に似た面もある。日本語で「[稲](../Page/イネ.md "wikilink")」「[米](../Page/米.md "wikilink")」「[ごはん](https://ja.wikipedia.org/wiki/ごはん "wikilink")」が、独立した単語としてあるのと同様に、テルグ語でも、これらは独立単語である。米一粒一粒に女神が宿っているとされ、そのため、米一粒一粒も、大事に扱わねばならないという文化がある。米はまた繁栄の象徴で、常に複数で表現される\[4\]。
