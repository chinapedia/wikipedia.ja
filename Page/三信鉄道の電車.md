**三信鉄道の電車**（さんしんてつどうのでんしゃ）

本項では、[三信鉄道](../Page/三信鉄道.md "wikilink")（現在の[東海旅客鉄道](../Page/東海旅客鉄道.md "wikilink")[飯田線](https://ja.wikipedia.org/wiki/飯田線 "wikilink")の一部）が保有した[電車](../Page/電車.md "wikilink")について記述する。

## 概要

三信鉄道は[天竜川](../Page/天竜川.md "wikilink")の電源開発を目的に、[豊川鉄道](https://ja.wikipedia.org/wiki/豊川鉄道 "wikilink")・[鳳来寺鉄道](https://ja.wikipedia.org/wiki/鳳来寺鉄道 "wikilink")と[伊那電気鉄道](https://ja.wikipedia.org/wiki/伊那電気鉄道 "wikilink")を結ぶ鉄道として建設されたもので、開業当初から[直流](../Page/直流電化.md "wikilink")1500Vによる[電化鉄道であった](../Page/鉄道の電化.md "wikilink")。建設は、伊那電気鉄道側の[天竜峡と](../Page/天竜峡駅.md "wikilink")、鳳来寺鉄道側の[三河川合の南北両側から進められ](https://ja.wikipedia.org/wiki/三河川合駅 "wikilink")、北線は[1932年](../Page/1932年.md "wikilink")（昭和7年）10月の天竜峡 - [門島間](https://ja.wikipedia.org/wiki/門島駅 "wikilink")、南線は1年遅れて[1933年](../Page/1933年.md "wikilink")（昭和8年）10月の三河川合 - [三河長岡間を皮切りに順次延伸され](https://ja.wikipedia.org/wiki/東栄駅 "wikilink")、[1937年](../Page/1937年.md "wikilink")（昭和12年）8月の[大嵐](https://ja.wikipedia.org/wiki/大嵐駅 "wikilink") - [小和田間を最後に全通した](https://ja.wikipedia.org/wiki/小和田駅 "wikilink")。

三信鉄道は、[1943年](../Page/1943年.md "wikilink")（昭和18年）8月1日付けで、接続する豊川鉄道、鳳来寺鉄道、伊那電気鉄道とともに[戦時買収され](../Page/戦時買収私鉄.md "wikilink")、[鉄道省](https://ja.wikipedia.org/wiki/鉄道省 "wikilink")飯田線となった。この時点で鉄道省籍を得たのは、[電気機関車](../Page/電気機関車.md "wikilink")1両、電車9両、[貨車](../Page/貨車.md "wikilink")69両である。これらは、貨車を除いて買収後も私鉄時代の形式番号のまま使用され、国鉄形式が与えられたのは、[1953年](https://ja.wikipedia.org/wiki/1953年 "wikilink")（昭和28年）6月1日に施行された、[車両形式称号規程改正の際である](../Page/国鉄旧形電車の車両形式.md "wikilink")。電車は後述のように、かつて鉄道省に在籍したものばかりであったが、再国有化にあたって旧番に復することはなかった。

## 形式

1943年の国有化時点で在籍した電車は、デニ201形1両およびデ301形8両の2形式9両である。これらはいずれも鉄道省払下げのモハ1系電動車（[モハ1形](https://ja.wikipedia.org/wiki/国鉄モハ1形電車 "wikilink")、[モニ3形](https://ja.wikipedia.org/wiki/国鉄モニ3形電車 "wikilink")）を鋼体化、あるいは木造車体のまま両運転台化したものであった。

### デニ201形

**デニ201形**は、[1936年](../Page/1936年.md "wikilink")（昭和11年）および[1939年](../Page/1939年.md "wikilink")（昭和14年）に製造された17m級両運転台式の[三等](https://ja.wikipedia.org/wiki/三等車 "wikilink")[荷物合造](../Page/荷物車.md "wikilink")[制御](../Page/制御車.md "wikilink")[電動車で](../Page/動力車.md "wikilink")、2両（デニ201, デニ202）が在籍していたが、デニ201は国有化前の[1942年](../Page/1942年.md "wikilink")（昭和17年）に事故により[廃車されたため](https://ja.wikipedia.org/wiki/廃車_\(鉄道\) "wikilink")、国有化されたのはデニ202のみである。

デニ201は1936年[日本車輌製造](../Page/日本車輌製造.md "wikilink")東京支店で北線開業時に導入したデ2（旧鉄道省モハ1022 ← [デハ6362](https://ja.wikipedia.org/wiki/国鉄デハ6340系電車 "wikilink")）を鋼体化したものであり、デニ202は1939年に[木南車輌製造](../Page/木南車輌製造.md "wikilink")で製造された三信鉄道唯一の新製電車であるが、メーカー手持ちのモハ1形の台枠と電装品を使用したものである。

製造メーカーは異なるものの両車の形態はほぼ同様で、側面窓は2段式で配置はdD（荷）4D6D2d、車内は6組の[ボックスシートを備え](../Page/鉄道車両の座席.md "wikilink")、荷物室に食い込む形で[便所も装備されていた](https://ja.wikipedia.org/wiki/列車便所 "wikilink")。最大長16,800[mm](../Page/メートル.md "wikilink")（車体長16,000mm）、最大幅2,740mm、最大高4,250mm（車体高3,782mm）で、自重35.00[t](../Page/トン.md "wikilink")。総定員は76人（うち座席40人）、荷物室荷重は1tである。

デニ202は、国有化後も飯田線にあって天竜峡以南で使用されていたが、[1951年](https://ja.wikipedia.org/wiki/1951年 "wikilink")（昭和26年）10月12日に三河川合 - 池場間で[旧鶴見臨港鉄道モハ313](https://ja.wikipedia.org/wiki/鶴見臨港鉄道の電車#モハ310形 "wikilink")、[旧伊那電気鉄道サハフ311とともに事故により焼失し](https://ja.wikipedia.org/wiki/伊那電気鉄道の電車#サハフ310形 "wikilink")、翌年モハ70801（[モハ71002](https://ja.wikipedia.org/wiki/国鉄70系電車#モハ71形 "wikilink")）の名義上の改造種車となった。

### デ301形

**デ301形**は、デニ201形と同じく鉄道省モハ1形同等の走行装置を持つ17m級両運転台式の三等制御電動車で、8両（デハ301 - デハ308）が在籍した。この形式は

  - 1936年に日本車輌製造東京支店で鋼体化されたデハ301 - デハ303
  - 1939年および1941年に木南車輛で鋼体化されたデハ304, デハ305
  - 1938年に木造車体のまま両運転台化のうえ入線したデハ306 - デハ308

の3グループに分かれる。形式の記号は「デ」であるが、現車標記は「デハ」であった。

#### 鋼体化グループ

鋼体化されたグループは、長距離運転に備えて、いずれも車内中央部にボックスシート6組と後位扉と運転台の間に便所を有し、窓配置はd2D<span style="text-decoration:underline;">12</span>D2dであった。車体の基本寸法、自重はデニ200形と同様で、定員は100人（うち座席52人）である。

1936年に鋼体化されたグループは、鋼体化当初は**デ100形**（デハ101 - デハ103）と称していた。デハ101, デハ102は前年に鉄道省を除籍されたモニ3形（モニ3009, モニ3010）を、103は[1934年](../Page/1934年.md "wikilink")（昭和9年）に入線したデ3（旧モハ1020）を種車としている。

木南車輌製造で鋼体化された2両（デハ304, デハ305）は、いずれも自社所有車を鋼体化したもので、旧番号は304がデユ1（← デユニ1 ← デ4 ← 鉄道省モハ1058）、305がデ1（← モハ1023）である。デハ304落成の際に、デハ101 - デハ103はデハ301 - デハ303に[改番](../Page/改番.md "wikilink")されたものと推定される。

これらは、国有化後も飯田線で使用されたが、1951年から[1952年](../Page/1952年.md "wikilink")（昭和27年）にかけて更新修繕が施行され、後位[運転台の撤去および客室化](https://ja.wikipedia.org/wiki/運転台撤去車 "wikilink")、貫通扉と幌の新設ならびに電装解除が行なわれ、記号が「**クデハ**」となった。さらに省形電車との併結のため、連結器が密着式に変更され、塗装も窓回りを[ベージュ](https://ja.wikipedia.org/wiki/ベージュ "wikilink")、腰板・幕板を[えんじ色](../Page/えんじ色.md "wikilink")に変更されている。

1953年の車両形式称号規程改正では、**クハ5800形**（5800 - 5804）となったが、[1959年](../Page/1959年.md "wikilink")（昭和34年）2月に全車が廃車され、私鉄に譲渡された。

各車の番号の変遷は次のとおりである。

  - 鉄道院デハニ6465（1914年） → デニ6458 → モニ3009（1928年改番） → 三信鉄道**デハ101**（1936年譲渡・鋼体化） → **デハ301**（1941年改番） → 鉄道省デハ301（1943年買収） → クデハ301（1951年）→ クハ5800（1953年改番）
  - 鉄道院デロハ6136（1914年） → デハ6370（1925年改造） → デニ6459 → モニ3010（1928年改番） → 三信鉄道**デハ102**（1936年譲渡・鋼体化） → **デハ302**（1941年改番） → 鉄道省デ302（1943年買収） → クデハ302（1951年）→ クハ5801（1953年改番）
  - 鉄道院デハ6360（1918年） → モハ1020（1928年改番） → 三信鉄道デ3（1934年譲渡） → **デハ103**（1936年鋼体化） → **デハ303**（1941年改番） → 鉄道省デハ303（1943年買収） → クデハ303（1951年）→ クハ5802（1953年改番）
  - 鉄道省デハ33534（1923年） → モハ1058（1928年改番） → 三信鉄道デ4（1934年譲渡） → **デユニ1** → **デユ1** → **デハ304**（1939年鋼体化） → 鉄道省デハ304（1943年買収） → クデハ304（1951年）→ クハ5803（1953年改番）
  - 鉄道院デハ6363（1918年） → モハ1023（1928年改番） → 三信鉄道デ1（1933年譲渡） → **デハ305**（1941年鋼体化） → 鉄道省デハ305（1943年買収） → クデハ305（1951年）→ クハ5804（1953年改番）

#### 木製車グループ

1938年8月に入線したグループ（デハ306 - デハ308）は、後位に運転台を増設したのみで鉄道省時代のモニター屋根木造車体のまま使用され、便所やボックスシートの装備はなく、3扉ロングシートのままであった。側面窓配置はd2D121D121D2、総定員は100人（うち座席48人）である。旧番号は、モハ1033, 1035, 1036で、いずれも[1911年](../Page/1911年.md "wikilink")（大正10年）[汽車製造](../Page/汽車製造.md "wikilink")東京支店製の旧[デハ33500形である](https://ja.wikipedia.org/wiki/国鉄デハ33500系電車 "wikilink")。

1943年の国有化後も飯田線南部で使用されたが、306が[1945年](../Page/1945年.md "wikilink")（昭和20年）2月17日に[三河槙原](https://ja.wikipedia.org/wiki/三河槙原駅 "wikilink") - 三河川合間で落石に乗り上げて宇連川に[旧伊那電気鉄道サハユニフ100とともに転落](https://ja.wikipedia.org/wiki/伊那電気鉄道の電車#サ100形 "wikilink")、大破して廃車となり、残る2両は1952年7月まで使用され、[大井川鉄道に払下げられている](../Page/大井川鐵道.md "wikilink")。

各車の番号の変遷は次のとおりである。

  - 鉄道省デハ33512（1922年） → モハ1033（1928年改番） → 三信鉄道**デ306**（1938年譲渡）→ 鉄道省デ306（1943年買収）
  - 鉄道省デハ33509（1922年） → モハ1035（1928年改番） → 三信鉄道**デ307**（1938年譲渡）→ 鉄道省デ307（1943年買収）
  - 鉄道省デハ33510（1922年） → モハ1036（1928年改番） → 三信鉄道**デ308**（1938年譲渡）→ 鉄道省デ308（1943年買収）

## 譲渡

旧三信鉄道の電車は、事故廃車となった2両（202, 306）を除く7両が私鉄に払下げられている。その状況は、次のとおりである。 [thumbが廃車後も保管するキハ](https://ja.wikipedia.org/wiki/ファイル:Type_KiHa_5800_Kominato_Railway\(2\).jpg "wikilink")5800（旧・国鉄クハ5800←三信デ301）。気動車化に伴う改造を諸所に受けているが車体や台車に多く原形を残す\]\]

  - クデハ307 → 大井川鉄道モハ301（1952年） → [モハ3829](https://ja.wikipedia.org/wiki/大井川鉄道310系電車 "wikilink")（1970年、[名古屋鉄道](https://ja.wikipedia.org/wiki/名古屋鉄道 "wikilink")[モ3829と振替](../Page/名鉄3800系電車.md "wikilink")）
  - クデハ308 → 大井川鉄道モハ302（1952年） → モハ3302（1970年、名古屋鉄道モ3302と振替）
  - クハ5800 → [小湊鐵道](../Page/小湊鐵道.md "wikilink")[キハ5800](https://ja.wikipedia.org/wiki/小湊鉄道キハ5800形気動車 "wikilink")（1959年） → 廃車（1997年3月）
  - クハ5801 → 小湊鐵道キハ5801（1959年） → 廃車（1978年）
  - クハ5802 → [伊豆箱根鉄道](../Page/伊豆箱根鉄道.md "wikilink")モハ47（1961年） → 廃車
  - クハ5803 → 大井川鉄道クハ503（1959年） → 廃車（1974年）
  - クハ5804 → 大井川鉄道クハ506（1959年） → 廃車（1978年）

上記のうち、大井川鉄道に払下げられた307（旧モハ1035）の車体は、振替後も解体されないまま[千頭駅](https://ja.wikipedia.org/wiki/千頭駅 "wikilink")構内に放置されていたが、東海旅客鉄道に譲渡の上、同社の発足10周年記念事業として[名古屋工場で昭和初期の姿に復元された](../Page/東海旅客鉄道名古屋工場.md "wikilink")。その後は[伊那松島運輸区](https://ja.wikipedia.org/wiki/伊那松島運輸区 "wikilink")に保管され、同区のイベント時に公開されていたが、2011年に開館した[リニア・鉄道館](https://ja.wikipedia.org/wiki/リニア・鉄道館 "wikilink")に移設された。

小湊鐵道に譲渡された2両は、床下に[ディーゼルエンジン](../Page/ディーゼルエンジン.md "wikilink")を装架して[気動車](../Page/気動車.md "wikilink")に改造された。新型の[キハ200形の増備に伴い予備車となったが](../Page/小湊鉄道キハ200形気動車.md "wikilink")、5800は車籍を有したまま同鉄道の五井機関区に保管された。同車は1997年に除籍されたものの、五井機関区内に現存している。2019年(平成31年）3月29日、同車は「小湊鉄道キハ5800形式気動車 附気動車台帳及び修繕表」として、[市原市](../Page/市原市.md "wikilink")の[指定文化財](https://ja.wikipedia.org/wiki/有形文化財 "wikilink")（歴史資料）に指定された\[1\]。
