**X11.app** (**XQuartz**) は、[X.Org Serverに基づく](../Page/X.Org_Server.md "wikilink")[macOS](https://ja.wikipedia.org/wiki/macOS "wikilink")向けの[X Window Systemの実装](../Page/X_Window_System.md "wikilink")。[ハードウェア](../Page/ハードウェア.md "wikilink")による[2次元コンピュータグラフィックス](https://ja.wikipedia.org/wiki/2次元コンピュータグラフィックス "wikilink")と[OpenGL](../Page/OpenGL.md "wikilink")のアクセラレーションを追加し、macOSの[グラフィカルユーザインタフェース](https://ja.wikipedia.org/wiki/グラフィカルユーザインタフェース "wikilink") (GUI) である[Aquaとの統合を行っている](../Page/Aqua_\(コンピュータ\).md "wikilink")。

X11.appは当初[Mac OS X v10.2向け](../Page/Mac_OS_X_v10.2.md "wikilink")[ベータ版](../Page/ベータ版.md "wikilink")として公開され、[Mac OS X v10.3では標準パッケージとなり](../Page/Mac_OS_X_v10.3.md "wikilink")、アップルのウェブサイトからダウンロード可能となった。[Mac OS X v10.4では](../Page/Mac_OS_X_v10.4.md "wikilink")、インストールDVDにオプションとしてインストーラが同梱された。[Mac OS X v10.5](../Page/Mac_OS_X_v10.5.md "wikilink")〜10.7では、X11.app はデフォルトでインストールされていた。[OS X v10.8からは標準搭載ではなくなり](https://ja.wikipedia.org/wiki/OS_X_v10.8 "wikilink")、別途インストールが必要となった。

Mac OS X v10.4でのX11.appはX11R6.6の[Xプロトコル](https://ja.wikipedia.org/wiki/Xプロトコル "wikilink")を実装している。この実装にはXFree86 4.4ベースのXサーバ、[Quartz](../Page/Quartz.md "wikilink")ルートレス・ウィンドウマネージャ、ライブラリ、[xterm](https://ja.wikipedia.org/wiki/xterm "wikilink")などの基本ユーティリティが含まれる\[1\]。「ルートレス」とは、X WindowのアプリケーションをQuartzデスクトップ上に表示でき、他のQuartzアプリケーションのウィンドウと同じに扱えることを意味する。quartz-wmによりX11のトップレベルウィンドウをmacOSのトップレベルウィンドウにマッピングした、「マルチウィンドウ」も扱える。Mac OS X v10.5 になるとX11.appはXFree86よりも一般化してきた[X.Org Serverを使うよう書き換えられた](../Page/X.Org_Server.md "wikilink")\[2\]。

ライセンスは[X.Org Server同様](../Page/X.Org_Server.md "wikilink")、基本的に[X11 Licenseだが](https://ja.wikipedia.org/wiki/X11_License "wikilink")、quartz-wmなど一部が[Apple Public Source Licenseで配布されている](https://ja.wikipedia.org/wiki/Apple_Public_Source_License "wikilink")。

X Window Systemの実装としては、Mac OS 9までは[MacX](https://ja.wikipedia.org/wiki/:w:MacX "wikilink")、Mac OS X v10.2までは[XDarwin](https://ja.wikipedia.org/wiki/XDarwin "wikilink")などがあった。
