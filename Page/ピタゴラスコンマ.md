**ピタゴラスコンマ**（）、あるいはダイトニックコンマ\[1\]は、[ピタゴラス音律](../Page/ピタゴラス音律.md "wikilink")における[異名同音](../Page/異名同音.md "wikilink")の差である小さな[音程](../Page/音程.md "wikilink")（あるいはコンマ）であり、例えば C と B、あるいは D と Cなどの差である\[2\]。これは531441:524288の周波数比に等しく、約23.46[セント](../Page/セント.md "wikilink")であり、おおむね半音の1/4である（75:74 と 74:73の間\[3\]）。

ピタゴラスコンマは、ピタゴラス音律のアポトメ（apotome）とリンマ（limma）の差\[4\]（すなわちピタゴラス音律によって定義される半音階的[半音](../Page/半音.md "wikilink")と全音階的半音の差）、あるいは12の純正な[完全五度](https://ja.wikipedia.org/wiki/完全五度 "wikilink")と7[オクターヴ](../Page/オクターヴ.md "wikilink")との差、また3つのピタゴラス音律のダイトーン（ditone）と1オクターヴとの差としても定義できる（これがダイトニックコンマと呼ばれる理由である）。

ピタゴラス音律における[減二度](https://ja.wikipedia.org/wiki/減二度 "wikilink")はリンマとアポトメの差と定義される。これはピタゴラスコンマの逆に一致し、下向きのピタゴラスコンマと見なすことができ（例：C から D）、約−23.46コンマである。

## 導出

上述のようにピタゴラスコンマは様々な方法によって導出される。

  - [ピタゴラス音律](../Page/ピタゴラス音律.md "wikilink")における2つの[異名同音](../Page/異名同音.md "wikilink")の差、 すなわち C と B、あるいは D と Cの差など。
  - ピタゴラス音律のアポトメとリンマの差。
  - 12の純正な[完全五度](https://ja.wikipedia.org/wiki/完全五度 "wikilink")と7[オクターヴ](../Page/オクターヴ.md "wikilink")との差<ref>例えばCから12回の完全5度の上昇によってCと異名同音関係にあるB♯が得られる(C–G–D–A–E–B–F♯–C♯–G♯–D♯–A♯–E♯–B♯)。これを純正な完全5度(3/2)で行った場合、このB♯と、元のCの7オクターヴ上のCとの間にピタゴラスコンマの差が生じる。

\(\frac{\left ( \frac{3}{2} \right )^{12}}{2^7}
= \frac{3^{12}}{2^{19}}
= \frac{531441}{524288}\)</ref>。

  - 3つのピタゴラス音律のダイトーン（[長三度](../Page/長三度.md "wikilink")）と1オクターヴとの差。

純正な完全五度は周波数比3/2である。これはピタゴラス音律において、与えられた最初の音から他の音に対する周波数比を定める基準としてオクターヴとともに用いられる。

アポトメとリンマはピタゴラス音律で定義される2種類の[半音](../Page/半音.md "wikilink")である。アポトメ（約113.69セント、例：C から C）は半音階的半音、あるいは[増一度](https://ja.wikipedia.org/wiki/増一度 "wikilink")であり、リンマ（約90.23セント、 例：C から D) は全音階的半音、あるいは[短二度](https://ja.wikipedia.org/wiki/短二度 "wikilink")である。

ダイトーン（あるいは長三度）は2つの[全音](https://ja.wikipedia.org/wiki/全音 "wikilink")からなる音程である。ピタゴラス音律では全音は約203.9セント（周波数比9:8）であり、ダイトーンは約407.8セントである。

8400\) と12の完全五度 (12 × 701.96 {{=}} 8423.52)、赤は1200、黒は701.96。|400px|center}}  1200) と3ダイトーン (3 × 407.82 {{=}} 1223.46)、赤は1200、紫は407.82。|400px|center}}

## 大きさ

ピタゴラスコンマの大きさを[セント](../Page/セント.md "wikilink")値で表すと

\[\hbox{apotome} - \hbox{limma} \approx 113.69 - 90.23 \approx 23.46 ~\hbox{cents} \!\]

より正確に周波数比で表すと

\[\frac{\hbox{apotome}}{\hbox{limma}}
=\frac{3^7/2^{11}}{2^8/3^5}
= \frac{3^{12}}{2^{19}}
= \frac{531441}{524288}
= 1.0136432647705078125
\!\]
