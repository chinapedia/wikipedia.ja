**脳動静脈奇形**（のうどうじょうみゃくきけい、）は、10万人に1人と言われる脳の[血管](../Page/血管.md "wikilink")が[動脈](../Page/動脈.md "wikilink")と[静脈](../Page/静脈.md "wikilink")の異常吻合を生じている先天性疾患。動脈と静脈との異常吻合部にはナイダス nidus と呼ばれる異常血管塊が認められる。

1.  ＡVM

若年者の[クモ膜下出血](https://ja.wikipedia.org/wiki/クモ膜下出血 "wikilink")の原因として重要。

## 臨床像・症状

  - ナイダスが破れると、[クモ膜下出血](https://ja.wikipedia.org/wiki/クモ膜下出血 "wikilink")や[脳内出血](https://ja.wikipedia.org/wiki/脳内出血 "wikilink")を引き起こす。
  - 最も代表的な症状は、クモ膜下出血に伴う[髄膜刺激症状](https://ja.wikipedia.org/wiki/髄膜刺激症状 "wikilink")と脳内出血に伴う脳局所症状である。その他に起こりやすいのが痙攣発作である。また病変が非常に大きい場合には、心拍出量増加や左心拡張を来して[心不全](../Page/心不全.md "wikilink")に陥ることもある。心不全は特に新生児で見られる。[サムネイル](https://ja.wikipedia.org/wiki/ファイル:AVM1_HE.jpg "wikilink")

## 破裂率・予後

  - 未破裂の脳動静脈奇形が出血する確率は2%/年前後と報告されており、増大する率は0.2〜2.8%/年と言われている。
  - [脳動脈瘤](https://ja.wikipedia.org/wiki/脳動脈瘤 "wikilink")破裂によるクモ膜下出血よりは再出血が少なく、生命予後は比較的良好と考えられている。

## 重症度分類

治療方針を決定するためにSpetzlerの重症度分類が用いられることが一般的である。

| 特徴         | 点数      |
| ---------- | ------- |
| nidusの大きさ  | 小（〜3cm） |
| 中（3〜6cm）   | 2       |
| 大（＞6cm）    | 3       |
| 周囲脳の機能的重症性 | 重要でない   |
| 重要である      | 1       |
| 導出静脈の型     | 表在性のみ   |
| 深在性        | 1       |

AVMの重症度分類（Spetzlerら、1986）

  - 重症度(Grade)=（大きさ）+（機能的重要性）+（導出静脈）=(1,2,3)+(0,1)+(0,1)
  - 機能的に重要な部位とは、運動野、知覚野、[言語野](https://ja.wikipedia.org/wiki/言語野 "wikilink")、[視覚野](https://ja.wikipedia.org/wiki/視覚野 "wikilink")、[視床](../Page/視床.md "wikilink")、[視床下部](https://ja.wikipedia.org/wiki/視床下部 "wikilink")、[内包](https://ja.wikipedia.org/wiki/内包 "wikilink")、[脳幹](../Page/脳幹.md "wikilink")部、大脳脚、深部[小脳](../Page/小脳.md "wikilink")核を指す。

## 治療

以前に比べると積極的治療の傾向が強くなってきている。出血例に対しては何らかの治療が必要と考えられる。治療の基本は手術によるナイダスの全摘出である。血管内治療もおこなわれるが、殆どの場合、手術治療或いは放射線治療の前処置として行われることが多い。ナイダスの部位的に手術が困難な症例でナイダスの径が3cm以下であった場合には、[ガンマナイフ](https://ja.wikipedia.org/wiki/ガンマナイフ "wikilink")治療のみで良好な成績を挙げられる。
