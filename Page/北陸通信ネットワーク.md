**北陸通信ネットワーク株式会社**（ほくりくつうしんネットワーク）は、[石川県](https://ja.wikipedia.org/wiki/石川県 "wikilink")[金沢市](https://ja.wikipedia.org/wiki/金沢市 "wikilink")にある電気通信会社。

## 概要

[北陸電力](../Page/北陸電力.md "wikilink")の関連会社である。

かつては、[ISDN](../Page/ISDN.md "wikilink")サービスや[アステル北陸](../Page/アステル北陸.md "wikilink")より継承した[PHS](../Page/PHS.md "wikilink")サービスを手掛けていたが、[ISDN](../Page/ISDN.md "wikilink")収容局のリプレースが困難であることを理由に撤退している。

なお、[北海道総合通信網](../Page/北海道総合通信網.md "wikilink")とともに、ISDNに割り当てられた[電話番号](../Page/電話番号.md "wikilink")を返上している（他地域でISDNから撤退した会社は、[IP電話](../Page/IP電話.md "wikilink")番号用途に継承している）。

## 沿革

  - [1993年](../Page/1993年.md "wikilink")5月 - 設立
  - [1994年](../Page/1994年.md "wikilink")2月 - 第一種電気通信事業許可取得
  - 1994年10月 - 営業開始
  - [1996年](../Page/1996年.md "wikilink")6月 - アステル北陸向けISDNサービス開始
  - [1997年](https://ja.wikipedia.org/wiki/1997年 "wikilink")4月 - 電力系NCC相互連係による全国サービス開始
  - 1997年6月 - フレームリレーサービス開始
  - 1997年7月 - 東京事務所開設
  - [1998年](https://ja.wikipedia.org/wiki/1998年 "wikilink")7月 - HTCN（HTNetコンピュータ通信網サービス）開始
  - 1998年10月 - ATM専用サービス、一般向けISDNサービス開始
  - [2001年](../Page/2001年.md "wikilink")6月 - DSL接続サービス開始
  - [2001年](../Page/2001年.md "wikilink")12月1日 - [アステル北陸](../Page/アステル北陸.md "wikilink")より、PHS事業の譲渡
  - [2003年](../Page/2003年.md "wikilink")4月 - 広域イーサネット網（HTNet-Ether）サービス開始
  - [2003年](../Page/2003年.md "wikilink")11月30日 - PHS新規申し込み終了。
  - [2004年](../Page/2004年.md "wikilink")5月26日 - PHS事業終了。[ISDN](../Page/ISDN.md "wikilink")サービス終了。
  - [2005年](https://ja.wikipedia.org/wiki/2005年 "wikilink")9月 - IP電話サービス開始
  - [2007年](../Page/2007年.md "wikilink")4月 - 「ISO/IEC27001:2005」、「JIS Q27001:2006」の認証取得
  - 2008年4月 - 遠隔データバックアップサービス「リモートバックアップサービス」開始
  - 2008年9月 - ネットワーク機器 監視サービス「ウォッチドッグ」開始
  - 2009年2月 -「ファイアウォールゲートウェイサービス」開始
  - 2010年2月 - ファイル転送サービス「HTNet File Transfer」開始
  - 2010年11月 - 「IPv6インターネット接続サービス」開始
  - 2011年2月 - 一般建設業許可
  - 2011年9月 - HTNet Cloud サービス開始
  - 2011年11月 - Web会議システムサービス｢HTNet Web Meeting｣ サービス開始
  - 2012年3月 - クラウドルーティング サービス開始
  - 2013年12月 - HTNetクラウドWi-Fi サービス開始
  - 2014年1月 - 「HTNet Mail Gateway」サービス開始
  - 2015年10月 - SaaS 型Web 会議サービス「HTNet Web Meeting V-CUBE」開始
  - 2016年6月 - SaaS型 IT資産管理サービス「LanScope on HTNet Cloud」開始
