**対称式**（たいしょうしき、*symmetric polynomial*）あるいは**対称多項式**（たいしょうたこうしき）とは、変数を入れ替えても変わらない[多項式](../Page/多項式.md "wikilink")のことである。

## 概要

2 変数の多項式

  -
    *f*(*x*,*y*) = *x*<sup>2</sup> + *x y* + *y*<sup>2</sup>

において、*x* と *y* を入れ替えた式

  -
    *f*(*y*,*x*) = *y*<sup>2</sup> + *y x* + *x*<sup>2</sup> = *x*<sup>2</sup> + *x y* + *y*<sup>2</sup>

は、元の *f*(*x*,*y*) とは全く変わらない多項式である。このように、変数を入れ替えても変わらない多項式の事を**対称式**という。

似たようなものに**[交代式](https://ja.wikipedia.org/wiki/交代式 "wikilink")**がある。交代式は

  -
    *g*(*x*,*y*) = *x*<sup>2</sup> − *y*<sup>2</sup>

のように、変数を入れ替えると、もとの式と符号が変わる

  -
    *g*(*y*,*x*) = *y*<sup>2</sup> − *x*<sup>2</sup> = − *g*(*x*,*y*)

という性質を持つ式である。符号が変わるだけなので、偶数個の交代式の積や、交代式を 2乗した式などは対称式となる。例えば

  -
    *g*(*x*,*y*)<sup>2</sup> = (*x*<sup>2</sup> − *y*<sup>2</sup>)<sup>2</sup>

は対称式である。

任意の対称式は、基本対称式

  -
    *s*<sub>1</sub> = *x* + *y*
    *s*<sub>2</sub> = *x y*

の多項式で書ける。例えば

  -
    *f*(*x*,*y*) = *x*<sup>2</sup> + *x y* + *y*<sup>2</sup> = (*x*+*y*)<sup>2</sup> − *x y* = *s*<sub>1</sub><sup>2</sup> − *s*<sub>2</sub>

である。

こういった対称式の概念は、 2 変数に留まらず、3 変数以上の多項式にも拡張される。例えば

  -
    *f*(*x*,*y*,*z*) = *x*<sup>3</sup> + *y*<sup>3</sup> + *z*<sup>3</sup>
    *f*(*x*,*y*,*z*,*w*) = 2 *x* + 2 *y* + 2 *z* + 2 *w* + 3 *y*<sup>2</sup> *z*<sup>2</sup> *w*<sup>2</sup> + 3 *z*<sup>2</sup> *w*<sup>2</sup> *x*<sup>2</sup> + 3 *w*<sup>2</sup> *x*<sup>2</sup> *y*<sup>2</sup> + 3 *x*<sup>2</sup> *y*<sup>2</sup> *z*<sup>2</sup>

は、それぞれ、3 変数と 4 変数の対称式であり、どの 2 つの変数を入れ替えても、元の多項式と変わらない式である。

は、[1629年](../Page/1629年.md "wikilink")に「代数学の新しい発明」(*Invention Nouvelle en l'Algèbre*) おいて、*n* 次の[代数方程式](../Page/代数方程式.md "wikilink")の[根と係数の関係](../Page/根と係数の関係.md "wikilink")を発見した。代数方程式の係数は *n* 個の根の基本対称式と呼ばれる対称式により書かれるというこの関係は、一般の次数の代数方程式の構造を調べるための重要な足掛かりの一つとなった。さらに、ジラールは、これらの関係を用いて虚数の有用性を説いた。

[18世紀](../Page/18世紀.md "wikilink")の後半になると、任意の対称式は基本対称式によって書くことができる事が、[ウェアリングや](../Page/エドワード・ウェアリング.md "wikilink")らによって示され、[ラグランジュによる](../Page/ジョゼフ＝ルイ・ラグランジュ.md "wikilink")、代数方程式の根の置換の研究へとつながっていった。

## 定義

### 対称式

Λ<sub>*n*</sub> = {1,2,3,…,*n*} とし、*S*<sub>*n*</sub> は Λ<sub>*n*</sub> に[作用する](https://ja.wikipedia.org/wiki/群の作用 "wikilink") *n* 次の[対称群](../Page/対称群.md "wikilink")とする。

*n* 変数の多項式 *f*(*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>) が、任意の σ ∈ *S*<sub>*n*</sub> に対して

  -
    *f*(*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>)<sup>σ</sup> = *f*(*x*<sub>σ(1)</sub>,*x*<sub>σ(2)</sub>,…,*x*<sub>σ(*n*)</sub>) = *f*(*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>)

を満たすとき、*f*(*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>) を、**対称多項式**あるいは**対称式**という。

  -
    要は *f* は変数をどのように入れ替えても不変な多項式である。

同様に[有理式](https://ja.wikipedia.org/wiki/有理式 "wikilink")

\[f(x_1,x_2,\cdots x_n) = {h(x_1,x_2,\cdots x_n) \over g(x_1,x_2,\cdots x_n)}\] が、任意の σ ∈ *S*<sub>*n*</sub> に対して

\[f(x_1,x_2,\cdots x_n)^{\sigma} = {h(x_1,x_2,\cdots x_n)^{\sigma} \over g(x_1,x_2,\cdots x_n)^{\sigma}} = f(x_1,x_2,\cdots x_n)\] であるとき、有理式 *f* は**対称的**であるという。

  -
    対称多項式や対称有理式は、*S*<sub>*n*</sub> という[群の作用](https://ja.wikipedia.org/wiki/群の作用 "wikilink")によって不変な式であるため、*S*<sub>*n*</sub> [不変式](https://ja.wikipedia.org/wiki/不変式 "wikilink")ともいう。

有理式として対称的でも、分母や分子に現れる *g* や *h* は、対称式でないこともある。この場合 *g* の変数を置換して現れる異なる多項式 *g*<sub>1</sub>, *g*<sub>2</sub>, …, *g*<sub>*m*</sub> を分母分子にかけて

\[f(x_1,x_2,\cdots ,x_n) = { h g_1 g_2 \cdots g_m \over g g_1 g_2 \cdots g_m}\] という有理式にすることで、分母分子ともに対称式となるような表示が得られる。

ここで分母に行ったような、対称式とは限らない一般の多項式に対して、置換を作用して得られる、多項式の組から、対称式を作る手法は、しばしば有用である。例えば、対称式とは限らない 2 変数の多項式 *f*(*x*<sub>1</sub>,*x*<sub>2</sub>) と、その変数を置換して得られる多項式 *f*(*x*<sub>2</sub>,*x*<sub>1</sub>) の和や積

  -
    *g*(*x*<sub>1</sub>,*x*<sub>2</sub>) = *f*(*x*<sub>1</sub>,*x*<sub>2</sub>) + *f*(*x*<sub>2</sub>,*x*<sub>1</sub>)
    *h*(*x*<sub>1</sub>,*x*<sub>2</sub>) = *f*(*x*<sub>1</sub>,*x*<sub>2</sub>) *f*(*x*<sub>2</sub>,*x*<sub>1</sub>)

は、いずれも対称式である。

[単項式](../Page/多項式.md "wikilink")

\[T(x_1,x_2, \cdots , x_n) = c \  x_1^{a_1} x_2^{a_2} \cdots x_n^{a_n}\] に、適当な置換 σ ∈ *S*<sub>*n*</sub> を作用させて単項式

\[T(x_1,x_2, \cdots , x_n)^{\sigma} = c \  x_{\sigma (1)}^{a_1} \  x_{\sigma (2)}^{a_2} \cdots x_{\sigma (n)}^{a_n}\] が作られるとき、この 2 つの単項式 *T*(*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>) と *T*(*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>)<sup>σ</sup> は**同型**（どうけい）であるという。

*T* と同型な単項式の全ての和

  -
    *T* + *T*<sub>1</sub> + *T*<sub>2</sub> + … *T*<sub>*m*</sub>

は、対称式であり、このように、同型な単項式の和として得られる対称式のことを、**単型**（たんけい）の対称式という。

### 基本対称式

[集合](../Page/集合.md "wikilink") *A* の[濃度を](../Page/集合.md "wikilink") |*A*| とするとき

\[\sigma_k(x_1,\cdots ,x_n) = \sum_{\lambda \sub \Lambda_n ,| \lambda |=k} \left( \  \prod_{t \in \lambda} x_t \right)\] という対称式の事を *k* 次の**基本対称式**（きほんたいしょうしき、*elementary symmetric polynomial*）という。変数を省略して σ<sub>*k*</sub> とも書く。変数の数 *n* に混乱の恐れがあれば、σ<sub>*n*,*k*</sub> のように *n* を明示する。

σ<sub>*k*</sub> は、*x*<sub>1</sub> *x*<sub>2</sub> … *x*<sub>*k*</sub> という *k* 次の単項式と同型な単項式からなる、単型の対称式である。

  -
    すなわち、*n* 個の変数 {*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>} から、*k* 個の変数を選んで掛け合わせて *k* 次の単項式を作る。この時、 *k* 個の変数の組み合わせを全て考えて、*k* 次の単項式を足し合わせてできた対称式である。
    *k* 個の要素の選び方は、[二項係数](../Page/二項係数.md "wikilink")を用いて <sub>*n*</sub>*C*<sub>*k*</sub> 通りあることが分かるので、 σ<sub>*k*</sub> は、 <sub>*n*</sub>*C*<sub>*k*</sub> 個の *k* 次の単項式の和である。

たとえば、2 変数なら

  -
    σ<sub>1</sub> = *x*<sub>1</sub> + *x*<sub>2</sub>
    σ<sub>2</sub> = *x*<sub>1</sub> *x*<sub>2</sub>

3 変数なら

  -
    σ<sub>1</sub> = *x*<sub>1</sub> + *x*<sub>2</sub> + *x*<sub>3</sub>
    σ<sub>2</sub> = *x*<sub>1</sub> *x*<sub>2</sub> + *x*<sub>1</sub> *x*<sub>3</sub> + *x*<sub>2</sub> *x*<sub>3</sub>
    σ<sub>3</sub> = *x*<sub>1</sub> *x*<sub>2</sub> *x*<sub>3</sub>

が基本対称式である。[根と係数の関係](../Page/根と係数の関係.md "wikilink")により、基本対称式は、 *x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub> を根とするモニックな *n* 次多項式の係数として現れる。

\[t^n - \sigma_{1} \  t^{n-1} + \sigma_{2} \  t^{n-2} -\cdots +(-1)^{n-1} \  \sigma_{n-1} \  t +(-1)^n \  \sigma_{n} = \prod_{k= 1}^n (t - x_k) = (t - x_1)(t - x_2)\cdots (t - x_n)\]

### ニュートン多項式

基本対称式の他に対称式の重要な例として、各自然数 *k* に対し *p*<sub>*k*</sub> = *x*<sub>1</sub><sup>*k*</sup> + ... + *x*<sub>*n*</sub><sup>*k*</sup> によって定義されるk次**ニュートン多項式**があげられる。基本対称式の場合と同じように、n変数に関するニュートン多項式 *p*<sub>*k*</sub>(*x*<sub>1</sub>, .., *x*<sub>*n*</sub>) について、*x*<sub>*n*</sub>*に* 0 を代入すると *n* - 1 変数に関するニュートン多項式 *p*<sub>*k*</sub>(*x*<sub>1</sub>, .., *x*<sub>*n* - 1</sub>) がえられる。

上記の根と係数の関係から、1 ≤ *i* ≤ nなる iについて

  -
    \(x_i^n - \sigma_{1} \  x_i^{n-1} + \sigma_{2} \  x_i^{n-2} -\cdots +(-1)^{n-1} \  \sigma_{(n-1)} \  x_i +(-1)^n \  \sigma_{n} = \prod_{k= 1}^n (x_i - x_k) = 0\)

が成り立っているが、これらの式の左辺を *x*<sub>*i*</sub> たちについてすべて足し合わせることで

  -
    \(p_n - \sigma_{1} \  p_{n-1} + \sigma_{2} \  p_{n-2} -\cdots +(-1)^{n-1} \  \sigma_{(n-1)} \  p_1 +(-1)^n n \  \sigma_{n} = 0\)

が得られる。この関係式から、*n* についての帰納的な考察により各自然数 *k* について *k* 変数の整係数多項式 *P*(s<sub>1</sub>, ..., *s*<sub>*k*</sub>) が存在して *p*<sub>*k*</sub> = *P*<sub>*k*</sub>(σ<sub>1</sub>,..., σ<sub>*k*</sub>) となっていることや、おなじく *k* 変数の有理係数多項式 *S*<sub>*k*</sub>(*q*<sub>1</sub>, ..., *q*<sub>*k*</sub>) が存在して σ<sub>*k*</sub> = *S*<sub>*k*</sub>(*p*<sub>1</sub>, ..., *p*<sub>*k*</sub>) となっていることがしたがう。たとえば、

  -
    *p*<sub>1</sub> = σ<sub>1</sub>, *p*<sub>2</sub> = σ<sub>1</sub><sup>2</sup> - 2 σ<sub>2</sub>, *p*<sub>3</sub> = σ<sub>1</sub><sup>3</sup> - 3 σ<sub>1</sub> σ<sub>2</sub> + 3 σ<sub>3</sub>
    \(\sigma_2 = \frac{1}{2} ( p_1^2 - p_2), \ \sigma_3 = \frac{1}{3}(p_3 - p_1 p_2 ) + \frac{1}{6} (p_1^3 - p_1 p_2)\)

## 対称式の基本定理

任意の対称式 *f*(*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>) に対し、基本対称式を変数にとる多項式 *g*(σ<sub>1</sub>,σ<sub>2</sub>,…,σ<sub>*n*</sub>) が一意に存在して

  -
    *f*(*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>) = *g*(σ<sub>1</sub>,σ<sub>2</sub>,…,σ<sub>*n*</sub>)

となる。これを**対称式の（第一）基本定理**という。この *g*(σ<sub>1</sub>,σ<sub>2</sub>,…,σ<sub>*n*</sub>) という多項式は一意に決まる 。この基本対称式に関する多項式 g を具体的に見いだすアルゴリズムは複数知られており、それらは、基本定理の証明を与えてもいる。ここでは、そのような方法のいくつかを述べる。

### ウェアリングによる方法

[1762年](../Page/1762年.md "wikilink")に[ウェアリングは](../Page/エドワード・ウェアリング.md "wikilink")、対称式に現れる単項式の指数の組に、[辞書式順序を入れて](../Page/単項式順序.md "wikilink")、単項式の次数を下げていく方法で、対称式の基本定理の証明を行った。

0でない係数 *c* を持つ単項式

\[r(x_1,x_2,\cdots, x_n) = c \  \prod_{k=1}^n x_k^{a_k} = c \  x_1^{a_1} \  x_2^{a_2} \cdots x_n^{a_n}\] に対して、*n* 個の指数の組

  -
    *deg*(*r*) = (*a*<sub>1</sub>,*a*<sub>2</sub>, …, *a*<sub>*n*</sub>)

を**次数**（じすう）という。ここで、積に用いていない変数の指数は 0 である。この次数に、辞書式順序を入れる。

  -
    すなわち、2 つの単項式 *s* と *t* を比べ、指数を *a*<sub>1</sub> から順に見ていき、最初の異なる指数の[整数](../Page/整数.md "wikilink")としての大小を *deg*(*s*) と *deg*(*t*) の大小とし、全ての指数が等しいときは *deg*(*s*) = *deg*(*t*) とする。たとえば、 (3,2,1,2) \> (3,1,0,3) \> (2,5,0,2) \> (0,5,2,2) \> (0,2,2,5) である。

多項式 *f*(*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>) に対しては、*n* 変数の多項式として

\[f( x_1,x_2, \cdots ,x_n )=\sum_{(p_1,p_2,\ldots,p_n) \in \mathbb{N}^n} a_{p_{1} p_{2} \ldots p_{n}} \  x_1^{p_1} \  x_2^{p_2} \  \cdots \  x_n^{p_n}\] と表したとき、係数が 0 でない項の中で最も次数の高い項の次数を *deg*(*f*) とする。

*f*(*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>) が対称式の時、その次数 *deg*(*f*) = (*a*<sub>1</sub>,*a*<sub>2</sub>, …, *a*<sub>*n*</sub>) は、任意の添字 1 ≤ *j* ≤ *k* ≤ *n* に対して、広義単調減少 *a*<sub>*j*</sub> ≥ *a*<sub>*k*</sub> となる。

  -
    対称式では、広義単調減少でない (0,1,3,2,2) のような次数の項があれば、(3,2,2,1,0) という次数で係数の等しい項が必ずある。

*f* の項のうちで、次数が *deg*(*f*) に等しい項の係数を *c*<sub>0</sub> とすると、*f* が定数でなければ

\[deg(f) > deg\left\{f - c_0 \left(\prod_{k = 1}^{n-1} \sigma_k^{a_k - a_{(k+1)}}\right)\sigma_n^{a_n} \right\}\] が成り立つ。適当な基本対称式の積を、*f* から引くと *f* よりも次数を下げる事ができるということである。得られた式の次数を調べ、同じように適当な基本対称式の積を引いていくことにより、多項式の次数を下げていくことができる。*f* と基本対称式の多項式の差は、この操作を有限回繰り返すことによって、定数になる。

  -
    *f*(*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>) − *h*(σ<sub>1</sub>,σ<sub>2</sub>,…,σ<sub>*n*</sub>) = 定数

となるような、基本対称式についての多項式 *h* が得られ、

  -
    *f*(*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>) = *h*(σ<sub>1</sub>,σ<sub>2</sub>,…,σ<sub>*n*</sub>) + 定数

と表すことができる。

  - 計算例1

2 変数 *x*<sub>1</sub>, *x*<sub>2</sub> の基本対称式は

  -
    σ<sub>1</sub> = *x*<sub>1</sub> + *x*<sub>2</sub>
    σ<sub>2</sub> = *x*<sub>1</sub> *x*<sub>2</sub>

である。

  -
    *f*(*x*<sub>1</sub>, *x*<sub>2</sub>) = *x*<sub>1</sub><sup>3</sup> *x*<sub>2</sub> + *x*<sub>1</sub> *x*<sub>2</sub><sup>3</sup> + *x*<sub>1</sub><sup>2</sup> + *x*<sub>1</sub> *x*<sub>2</sub> + *x*<sub>2</sub><sup>2</sup>

は対称式であり、それぞれの項の次数は順に (3,1), (1,3), (2,0), (1,1), (0,2) であるから、 *deg*(*f*) = (3,1) で、*x*<sub>1</sub><sup>3</sup> *x*<sub>2</sub> の係数は 1 である。したがって

  -
    *h*<sub>1</sub> = *f* − σ<sub>1</sub><sup>3−1</sup> σ<sub>2</sub><sup>1</sup> = *f* − σ<sub>1</sub><sup>2</sup> σ<sub>2</sub> = − 2 *x*<sub>1</sub><sup>2</sup> *x*<sub>2</sub><sup>2</sup> + *x*<sub>1</sub><sup>2</sup> + *x*<sub>1</sub> *x*<sub>2</sub> + *x*<sub>2</sub><sup>2</sup>

となり、 *deg*( *h*<sub>1</sub> ) = (2,2) である。 *h*<sub>1</sub> の *x*<sub>1</sub><sup>2</sup> *x*<sub>2</sub><sup>2</sup> の係数は − 2 であるから

  -
    *h*<sub>2</sub> = *h*<sub>1</sub> − ( − 2 σ<sub>1</sub><sup>2−2</sup> σ<sub>2</sub><sup>2</sup> ) = *h*<sub>1</sub> + 2 σ<sub>2</sub><sup>2</sup> = *x*<sub>1</sub><sup>2</sup> + *x*<sub>1</sub> *x*<sub>2</sub> + *x*<sub>2</sub><sup>2</sup>

さらに *deg*( *h*<sub>2</sub> ) = (2,0) より

  -
    *h*<sub>3</sub> = *h*<sub>2</sub> − ( σ<sub>1</sub><sup>2−0</sup> σ<sub>2</sub><sup>0</sup> ) = *h*<sub>2</sub> − σ<sub>1</sub><sup>2</sup> = − *x*<sub>1</sub> *x*<sub>2</sub> = − σ<sub>2</sub>

であるから

  -
    *f* = σ<sub>1</sub><sup>2</sup> σ<sub>2</sub> + *h*<sub>1</sub> = σ<sub>1</sub><sup>2</sup> σ<sub>2</sub> − 2 σ<sub>2</sub><sup>2</sup> + *h*<sub>2</sub> = σ<sub>1</sub><sup>2</sup> σ<sub>2</sub> − 2 σ<sub>2</sub><sup>2</sup> + σ<sub>1</sub><sup>2</sup> − σ<sub>2</sub>

が得られる。

### コーシーによる方法

[1829年](../Page/1829年.md "wikilink")に[コーシーは](../Page/オーギュスタン＝ルイ・コーシー.md "wikilink")、 1 つの変数に着目した方法を用いた。 *x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub> を変数とする *n* 変数の対称式は、*x*<sub>1</sub> だけを定数と思えば、*x*<sub>2</sub>,…,*x*<sub>*n*</sub> を変数とする *n*−1 変数の対称式でもある。 *x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub> を変数とする *k* 次の基本対称式を σ<sub>*k*</sub> とし、 *x*<sub>2</sub>,…,*x*<sub>*n*</sub> を変数とする *k* 次の基本対称式を τ<sub>*k*</sub> とすると

  -
    σ<sub>1</sub> = *x*<sub>1</sub> + τ<sub>1</sub>
    σ<sub>2</sub> = τ<sub>1</sub> *x*<sub>1</sub> + τ<sub>2</sub>
    … … …
    σ<sub>*n*−1</sub> = τ<sub>*n*−2</sub> *x*<sub>1</sub> + τ<sub>*n*−1</sub>
    σ<sub>*n*</sub> = τ<sub>*n*−1</sub> *x*<sub>1</sub>

という関係が成り立つ。

  -
    τ<sub>1</sub> = σ<sub>1</sub> − *x*<sub>1</sub>
    τ<sub>2</sub> = σ<sub>2</sub> − τ<sub>1</sub> *x*<sub>1</sub> = σ<sub>2</sub> − (σ<sub>1</sub> − *x*<sub>1</sub>) *x*<sub>1</sub> = σ<sub>2</sub> − σ<sub>1</sub> *x*<sub>1</sub> + *x*<sub>1</sub><sup>2</sup>
    …

と、順に代入を繰り返してみると、τ<sub>1</sub>, τ<sub>2</sub>,…,τ<sub>*n*−1</sub> は、*x*<sub>1</sub> と σ<sub>1</sub>, …, σ<sub>*k*</sub> の多項式で表されることが分かる。

また、*x*<sub>1</sub>, *x*<sub>2</sub>, …, *x*<sub>*n*</sub> は、

\[t^n - \sigma_{1} \  t^{n-1} + \sigma_{2} \  t^{n-2} -\cdots +(-1)^{n-1} \  \sigma_{(n-1)} \  t +(-1)^n \  \sigma_{n} = 0\] の根であるから、これらを変数とする多項式は、次数下げなどにより、どの *x*<sub>*m*</sub> (1 ≤ *m* ≤ *n*) に関しても *n*−1 次以下となるような多項式にすることができる。

対称式 *f*(*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>) を *x*<sub>1</sub> について整理し、*x*<sub>1</sub><sup>*k*</sup> の係数が *g*<sub>*k*</sub>(*x*<sub>2</sub>,…,*x*<sub>*n*</sub>, σ<sub>1</sub>, σ<sub>2</sub>, …, σ<sub>*n*</sub>) 、すなわち

\[f(x_1,x_2,\cdots ,x_n) = \sum_{k=0}^{n-1} g_k (x_2,x_3,\cdots , x_n,\sigma_1,\sigma_2,\cdots , \sigma_n) \  x_1^k\] であるとすると *g*<sub>*k*</sub> は、*x*<sub>2</sub>,…,*x*<sub>*n*</sub> に関しての対称式になる。

*n*−1 変数の対称式は、基本対称式の多項式で表せるということを仮定すると、 *g*<sub>*k*</sub> は、τ<sub>1</sub>, τ<sub>2</sub>,…,τ<sub>*n*−1</sub> の多項式で書くことができる。すなわち、どの *g*<sub>*k*</sub> も、*x*<sub>1</sub> と σ<sub>1</sub>, …, σ<sub>*k*</sub> の多項式で表される。*x*<sub>1</sub> の次数を下げつつ

\[f(x_1,x_2,\cdots ,x_n) = \sum_{k=0}^{n-1} h_k (\sigma_1,\sigma_2,\cdots , \sigma_n) \  x_1^k\] の形に整理することができる。左辺は対称式なので、*x*<sub>1</sub> と任意の *x*<sub>*p*</sub> を入れ替えても変わらないので、任意の *p*, *q* ∈ {1,2,…,*n*} について

\[\sum_{k=0}^{n-1} h_k (\sigma_1,\sigma_2,\cdots , \sigma_n) \  x_p^k = \sum_{k=0}^{n-1} h_k (\sigma_1,\sigma_2,\cdots , \sigma_n) \  x_q^k\] が成り立つことから、 1 ≤ *k* ≤ *n-1* のとき *h*<sub>*k*</sub> ≡ 0 となり

  -
    *f*(*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>) = *h*<sub>0</sub>(σ<sub>1</sub>,σ<sub>2</sub>,…,σ<sub>*n*</sub>)

と表され、*n* 変数の対称式は、基本対称式の多項式で表せることがわかる。*n* = 1 の時は対称式の基本定理は明らかに成り立つので、数学的帰納法により、*n* 変数の対称式について基本定理が成り立つ。

### 斉重対称式

基本対称式の単項式

\[T = \prod_{k=1}^{n} \sigma_k^{a_k} = \sigma_1^{a_1} \  \sigma_2^{a_2} \  \cdots \  \sigma_n^{a_n}\] に対して、**重さ**

\[w(T) = \sum_{k=1}^{n} k \  a_k = a_1 + 2 \  a_2 + \cdots + n \  a_n\] を定義する。 0 でない定数 *c* をかけた *c T* の重さも、*T* と同じとする。σ<sub>1</sub>,σ<sub>2</sub>,…,σ<sub>*n*</sub> の多項式で、重さが同じ単項式の和になっているものを、**斉重多項式**（せいじゅうたこうしき、*isobaric polynomial*）という。

基本対称式 σ<sub>*k*</sub> は、*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub> に関して、*k* 次の[斉次多項式であるので](../Page/多項式.md "wikilink")、単項式 *T* は、*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub> に関して、 *w*(*T*) 次の斉次多項式となり、*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub> に関する斉次多項式の次数と、σ<sub>1</sub>,σ<sub>2</sub>,…,σ<sub>*n*</sub> に関する斉重多項式の次数が対応している。

したがって、対称式を基本対称式で表すためには、対称式を、次数の異なる斉次多項式にわけ、それぞれの斉次多項式を、その次数と同じ重さをもつ、斉重多項式で表していけばよい。

*s* 次の対称式 *f*(*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>) を、

\[f(x_1,x_2, \cdots , x_n) = \sum_{t = 1}^s f_t (x_1,x_2, \cdots , x_n)\] のように、 *t* 次の斉次多項式 *f*<sub>*t*</sub>(*x*<sub>1</sub>,*x*<sub>2</sub>,…,*x*<sub>*n*</sub>) の和として表せば、それぞれの斉次多項式 *f*<sub>*t*</sub> は対称式である。

*f*<sub>*t*</sub> は、基本対称式を変数とする、重さが *t* の単項式の全て *T*<sub>*t*,1</sub>, *T*<sub>*t*,2</sub>, …, *T*<sub>*t*,m</sub> の線型結合によって

\[f_t (x_1,x_2, \cdots , x_n) = \sum_{i=1}^m c_i \  T_{t,i} = c_1 \  T_{t,1} + c_2 \  T_{t,2} + \cdots + c_m \  T_{t,m}\] の形で表すことができる。この式に現れる係数 *c*<sub>1</sub>, …, *c*<sub>*m*</sub> を求めることで、 *f*<sub>*t*</sub> が、基本対称式を変数とする多項式で表される。

このようにして、*f*<sub>*t*</sub> の和である *f* も、基本対称式を変数とする多項式で表されることになる。

の形になり、係数の、 *c*<sub>1</sub>, *c*<sub>2</sub>, *c*<sub>3</sub> の 3 つを定めればよい。用いる方法はなんでもよく、例えば、ウェアリングによる方法で定義した次数を使うと、*deg*(*g*) = (3,1) であるから、 σ<sub>1</sub><sup>2</sup> σ<sub>2</sub> の項が最高次数となり、これより大きな次数の σ<sub>1</sub><sup>4</sup> は現れないので、 *c*<sub>1</sub> = 0 である。恒等式なので、*x*<sub>1</sub> = 1, *x*<sub>1</sub> = −1 という特別な値を代入しても成り立ち、この時、σ<sub>1</sub> = 0, σ<sub>2</sub> = −1 であるから、*c*<sub>3</sub> = −2 が分かる。 *x*<sub>1</sub> = *x*<sub>2</sub> = 1 とすると、σ<sub>1</sub> = 2, σ<sub>2</sub> = 1 なので、*c*<sub>2</sub> = 1 とわかり

  -
    *g* = σ<sub>1</sub><sup>2</sup> σ<sub>2</sub> − 2 σ<sub>2</sub><sup>2</sup>

である。 *h* の方も同様に、重さが 2 の斉重多項式

  -
    *h*(*x*<sub>1</sub>, *x*<sub>2</sub>) = *d*<sub>1</sub> σ<sub>1</sub><sup>2</sup> + *d*<sub>2</sub> σ<sub>2</sub>

の形になり、係数を求めると *d*<sub>1</sub> = 1, *d*<sub>2</sub> = −1 となるので

  -
    *h*(*x*<sub>1</sub>, *x*<sub>2</sub>) = σ<sub>1</sub><sup>2</sup> − σ<sub>2</sub>

となり

  -
    *f* = *g* + *h* = σ<sub>1</sub><sup>2</sup> σ<sub>2</sub> − 2 σ<sub>2</sub><sup>2</sup> + σ<sub>1</sub><sup>2</sup> − σ<sub>2</sub>

となることが分かる。

  - 計算例2
    σ<sub>1</sub> = *x*<sub>1</sub> + *x*<sub>2</sub> + *x*<sub>3</sub>
    σ<sub>2</sub> = *x*<sub>3</sub> *x*<sub>1</sub> + *x*<sub>1</sub> *x*<sub>2</sub> + *x*<sub>2</sub> *x*<sub>3</sub>
    σ<sub>3</sub> = *x*<sub>1</sub> *x*<sub>2</sub> *x*<sub>3</sub>

を用いて、三次方程式の判別式

  -
    *D* = { (*x*<sub>3</sub> − *x*<sub>1</sub>)(*x*<sub>1</sub> − *x*<sub>2</sub>)(*x*<sub>2</sub> − *x*<sub>3</sub>)}<sup>2</sup>

を基本対称式の多項式で表す。

*D* は 6 次の斉次対称式なので、σ<sub>1</sub>,σ<sub>2</sub>,σ<sub>3</sub> の多項式で書くと、重さが 6 の斉重多項式

  -
    *D* = *c*<sub>1</sub> σ<sub>1</sub><sup>6</sup> + *c*<sub>2</sub> σ<sub>1</sub><sup>4</sup> σ<sub>2</sub> + *c*<sub>3</sub> σ<sub>1</sub><sup>3</sup> σ<sub>3</sub> + *c*<sub>3</sub> σ<sub>1</sub><sup>2</sup> σ<sub>2</sub><sup>2</sup> + *c*<sub>4</sub> σ<sub>1</sub> σ<sub>2</sub> σ<sub>3</sub> + *c*<sub>5</sub> σ<sub>2</sub><sup>3</sup> + *c*<sub>6</sub> σ<sub>3</sub><sup>2</sup>

になる。 *D* は、 *x*<sub>1</sub> に関して 4 次なので、右辺もそうなるように、 *c*<sub>1</sub> = *c*<sub>2</sub> = 0 でなければならない。

*x*<sub>1</sub> = 0, *x*<sub>2</sub> = 1, *x*<sub>3</sub> = −1 という特別な値を入れても、この等式は成り立つはずで、 σ<sub>1</sub> = σ<sub>3</sub> = 0, σ<sub>2</sub> = −1 だから、 *c*<sub>5</sub> = −4 とわかる。以下、特殊な値を入れたり、係数を比較したりすることにより、

  -
    *D* = −4σ<sub>1</sub><sup>3</sup> σ<sub>3</sub> + σ<sub>1</sub><sup>2</sup> σ<sub>2</sub><sup>2</sup> + 18σ<sub>1</sub>σ<sub>2</sub>σ<sub>3</sub> −4σ<sub>2</sub><sup>3</sup> + 27 σ<sub>3</sub><sup>2</sup>

が得られる。

### 基本対称式の代数的独立性

*n* 個の変数 *x*<sub>1</sub>, ... , *x*<sub>*n*</sub> に関する対称式 *f*(x<sub>1</sub>, ..., *x*<sub>*n*</sub>) は基本対称式たち σ<sub>1</sub>, ... , σ<sub>*n*</sub> に関する多項式 *P* によって表すことができるが、この *P* は *f* に対して一意的に定まる。つまり、基本対称式たちは係数環上代数的独立だということになる。

証明は変数 *x*<sub>1</sub>, ... , *x*<sub>*n*</sub> の数 *n* と、対称式の次数に関する帰納法によって行われる。*n* - 1変数の多項式 *Q*(s<sub>1</sub>, ..., *s*<sub>*n* - 1</sub>) *で*、基本対称式 σ<sub>1</sub>(*x*<sub>1</sub>, ... , *x*<sub>*n* - 1</sub>), ..., σ<sub>*n* - 1</sub>(*x*<sub>1</sub>, ... , *x*<sub>*n* - 1</sub>) を代入して 0 になるようなものは *s*<sub>*i*</sub> たちの多項式としてすでに 0 になることが示せていたとする。

P(*s*<sub>1</sub>, ..., *s*<sub>*n*</sub>) *を* n 変数の多項式で基本対称式を移入したときに零になる *P*(σ<sub>1</sub>,..., σ<sub>*n*</sub>) = 0 ようなものとする。このとき特に、*P*(σ<sub>1</sub>,..., σ<sub>*n*</sub>) をx<sub>*i*</sub>たちに関する多項式と見なして *x*<sub>*n*</sub>*に*0を代入したものも 0 になる。 *x*<sub>*n*</sub>*に* 0 を代入することでσ<sub>1</sub>,..., σ<sub>*n* - 1</sub> *は* n - 1変数x<sub>1</sub>, ... , *x*<sub>*n* - 1</sub>についての基本対称式になり、一方 σ<sub>*n*</sub> *は* 0になる。したがって *P*(σ<sub>1</sub>, .., σ<sub>*n*-1</sub>, 0) であり、帰納法の仮定によって *n* - 1 変数の多項式 *P*(s<sub>1</sub>, ..., *s*<sub>*n* - 1</sub>, 0) *は* 0*に*、つまりP(*s*<sub>1</sub>, ..., *s*<sub>*n*</sub>) *は* s<sub>*n*</sub>で割りきれるということになる。従ってP = *R*(s<sub>1</sub>, .., *s*<sub>*n*</sub>) *s*<sub>*n*</sub> なる多項式 Rが得られるが、*R* に基本対称式たちを代入すると 0 になる。この操作を続けると任意の自然数 *m* について *P* は *s*<sub>*n*</sub><sup>*m*</sup> で割りきれるということになるが、そうなっているためには *P* は 0 でなければならない。
