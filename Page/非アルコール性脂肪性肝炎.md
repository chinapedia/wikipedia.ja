**非アルコール性脂肪性肝炎**（ひアルコールせいしぼうせいかんえん、Non-alcoholic steatohepatitis：**NASH**）とは、肝臓に脂肪が蓄積することで起こる[肝炎](../Page/肝炎.md "wikilink")である。[非アルコール性脂肪性肝疾患](https://ja.wikipedia.org/wiki/非アルコール性脂肪性肝疾患 "wikilink")のうちで最も極端な形態であり、NASH は原因不明の[肝硬変](../Page/肝硬変.md "wikilink")の重要な原因だとみなされ\[1\]\[2\]、画像診断あるいは組織診断で脂肪肝を認め、アルコール性肝障害などの疾患を除外した病態である\[3\]。最終的に[肝細胞癌](../Page/肝細胞癌.md "wikilink")に進行することもある\[4\]。

## 臨床像

発生に至る機序はまだはっきりとはわかっていないが、[肥満](../Page/肥満.md "wikilink")に伴う[脂肪肝](../Page/脂肪肝.md "wikilink")に加え、肝臓に何らかのストレスがかかることによって発生するのではないかと考えられている。ストレスは具体的には[活性酸素](../Page/活性酸素.md "wikilink")による[酸化ストレス](https://ja.wikipedia.org/wiki/酸化ストレス "wikilink")、[過酸化脂質](../Page/過酸化脂質.md "wikilink")、[鉄](../Page/鉄.md "wikilink")、[インスリン](../Page/インスリン.md "wikilink")抵抗性、[サイトカイン](../Page/サイトカイン.md "wikilink")の放出などがある\[5\]。日本では2005年より[メタボリックシンドローム](../Page/メタボリックシンドローム.md "wikilink")の増加により、NASHへの注目も高まっている\[6\]。

脂肪肝においては、血清[フェリチン](https://ja.wikipedia.org/wiki/フェリチン "wikilink")の増加がしばしばみられ、脂肪肝のなかでも非アルコール性脂肪性肝炎 (NASH) を含んだ非アルコール性脂肪性肝疾患では、肝組織内の[鉄の過剰が肝障害の増悪因子と考えられている](https://ja.wikipedia.org/wiki/鉄過剰症 "wikilink")\[7\]。非アルコール性脂肪性肝炎では理由は不明であるが[鉄](../Page/鉄.md "wikilink")の代謝異常が起こりフェリチンの増加が認められる。また肝臓には鉄が蓄積されていることが知られている\[8\]。ヘプシジン（[:en:Hepcidin](https://ja.wikipedia.org/wiki/:en:Hepcidin "wikilink")）は肝臓で産生される一種の[ペプチドホルモン](https://ja.wikipedia.org/wiki/ペプチドホルモン "wikilink")であり、鉄代謝制御を行っている。ヘプシジンは腸からの鉄の過剰な吸収を抑制する作用を有する。ヘプシジン産生障害は鉄過剰症を引き起こす。なお、多くの[病原体](https://ja.wikipedia.org/wiki/病原体 "wikilink")はその増殖に多量の鉄を要するため、ヘプシジンが[血清鉄](https://ja.wikipedia.org/wiki/血清鉄 "wikilink")濃度を低下させることは[炎症](../Page/炎症.md "wikilink")の原因となる菌の増殖を抑制して[抗菌作用](https://ja.wikipedia.org/wiki/抗菌作用 "wikilink")も発揮することになる\[9\]。自由な鉄原子は[過酸化物](../Page/過酸化物.md "wikilink")と反応し[フリーラジカル](https://ja.wikipedia.org/wiki/フリーラジカル "wikilink")を生成し、これが [DNA](../Page/デオキシリボ核酸.md "wikilink") や[タンパク質](../Page/タンパク質.md "wikilink")、および[脂質](../Page/脂質.md "wikilink")を破壊するためである。細胞中で鉄を束縛する[トランスフェリン](https://ja.wikipedia.org/wiki/トランスフェリン "wikilink")の量を超えて鉄を摂取すると、これによって自由な鉄原子が生じ、鉄中毒となる。余剰の鉄は[フェリチン](https://ja.wikipedia.org/wiki/フェリチン "wikilink")や[ヘモジデリン](https://ja.wikipedia.org/wiki/ヘモジデリン "wikilink")にも貯蔵隔離される。過剰の鉄はこれらの[タンパク質](../Page/タンパク質.md "wikilink")に結合していない自由鉄を生じる。自由鉄が[フェントン反応](https://ja.wikipedia.org/wiki/フェントン反応 "wikilink")を介して[ヒドロキシラジカル](https://ja.wikipedia.org/wiki/ヒドロキシラジカル "wikilink")（OH•）等の[活性酸素](../Page/活性酸素.md "wikilink")を発生させる。発生した活性酸素は細胞のタンパク質や[DNA](https://ja.wikipedia.org/wiki/DNA "wikilink")を損傷させる。活性酸素が各臓器を攻撃し、肝臓には肝炎、肝硬変、[肝臓がん](https://ja.wikipedia.org/wiki/肝臓がん "wikilink")を、[膵臓](../Page/膵臓.md "wikilink")には[糖尿病](../Page/糖尿病.md "wikilink")、[膵臓癌](https://ja.wikipedia.org/wiki/膵臓癌 "wikilink")を、[心臓](../Page/心臓.md "wikilink")には[心不全](../Page/心不全.md "wikilink")を引き起こす\[10\]。[ヒト](../Page/ヒト.md "wikilink")の体には鉄を排出する効率的なメカニズムがなく、粘膜や粘液に含まれる1-2mg/日程度の少量の鉄が排出されるだけであるため、ヒトが吸収できる鉄の量は1-2mg/日程度と非常に少ない\[11\]。しかし血中の鉄分が一定限度を超えると、鉄の吸収をコントロールしている消化器官の細胞が破壊される。この為、高濃度の鉄が蓄積すると、ヒトの心臓や肝臓に恒久的な損傷が及ぶ事があり\[12\]、最悪の場合は[死](../Page/死.md "wikilink")に至ることもある。

## 症状

自覚症状はほとんどない。検査で発覚することがほとんどである。

## 診断

[アルコール性肝障害](https://ja.wikipedia.org/wiki/アルコール性肝障害 "wikilink")との鑑別が最も重要となる。アルコール性肝障害ではAST/ALT比が1.0以上となることと、問診によってアルコール摂取量を把握することで鑑別する。肝線維化の進行度合いが予後に大きく影響するため「小葉内炎症」「風船様変性細胞」の程度が重要視される\[13\]\[14\]。

  - 診断基準

観察者の判断差異や施設間差異の低減のため、下記表によるスコアリングによる病理診断(NAS: NAFLD Activity Score)が行われる事がある\[15\]。

| 項目               | 程度   | 点数 |
| ---------------- | ---- | -- |
| 肝脂肪化             | 5%未満 | 0点 |
| 5～33%            | 1点   |    |
| 33～66%           | 2点   |    |
| 66%以上            | 3点   |    |
| 小葉内炎症            | 病巣なし | 0点 |
| 200倍の視野で2箇所の病巣以下 | 1点   |    |
| 200倍の視野で2～4箇所の病巣 | 2点   |    |
| 200倍の視野で4箇所以上の病巣 | 3点   |    |
| 肝細胞の風船様変化        | なし   | 0点 |
| 少数の風船様変性細胞       | 1点   |    |
| 多数の風船様変性細胞       | 2点   |    |
|                  |      |    |
| 診断               |      | 合計 |
| 脂肪肝 (NAFL)       | 0〜2点 |    |
| 境界型 NASH         | 3〜4点 |    |
| 非アルコール性脂肪肝炎 NASH | 0〜8点 |    |

NAS (NAFLD Activity Score)

更に、下記 Younossiの診断基準を併用する事がある\[16\]。

1.  肝細胞の脂肪化（程度は問わない）に加え小葉中心性の肝細胞の風船様変性（centrilobular ballooning）やMallory-Denk体を認めるもの。
2.  肝細胞の脂肪化に加え小葉中心性の細胞周囲/類洞周囲（pericellular/perisinusoidal）の線維化または架橋形成（bridging fibrosis）を認めるもの。

以上 1.または 2.を満たす場合NASHと定義する。

## 治療

食生活の改善と運動療法が基本。肝臓病に対する薬が投与されることもある。

[ピオグリタゾン](https://ja.wikipedia.org/wiki/ピオグリタゾン "wikilink")と[ビタミンE](../Page/ビタミンE.md "wikilink")はNASHの改善に有用であるとの報告がある\[17\]\[18\]。 5%の体重減少でもNASHの改善がみられるが、減量幅は多いほど良いとの報告がある\[19\]\[20\]。

## 予後

肝炎から[肝硬変](../Page/肝硬変.md "wikilink")、[肝細胞癌](../Page/肝細胞癌.md "wikilink")へと進展することがあるため、肝機能を検査して常に確認しておくことが肝要である。

## 疫学

中年以降の女性に好発する\[21\]。一般人口の罹患率はアメリカで2 - 3%と推定されており、日本においても同程度存在すると考えられている\[22\]。

## 歴史

  - [1979年](../Page/1979年.md "wikilink")、アドラーらが肥満患者にアルコール肝障害に類似した症例が見られることを報告。
  - [1980年](https://ja.wikipedia.org/wiki/1980年 "wikilink")、ルドウィッヒらが飲酒歴がないにも関わらずアルコール性肝障害に類似した症例が見られる疾患を非アルコール性脂肪性肝炎と命名\[23\]。
