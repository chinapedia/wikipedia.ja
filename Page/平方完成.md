\[ax^2 + bx + c = a(x - h)^2 + k \quad (a \neq 0)\] \(x-h\) の \(h\) を除けば、つまり \(x-h=t\) と変換すれば

\[a t^2 +h\] の形に帰着される。このことより、以下のことが導出できる：

  - [二次方程式](../Page/二次方程式.md "wikilink")の解を求める（→[二次方程式の解の公式](https://ja.wikipedia.org/wiki/二次方程式の解の公式 "wikilink")）
  - 二次関数のグラフの頂点を求める
  - [微分積分学](../Page/微分積分学.md "wikilink")で、冪指数に一次の項を含む[ガウス積分](../Page/ガウス積分.md "wikilink")の計算
  - [ラプラス変換](https://ja.wikipedia.org/wiki/ラプラス変換 "wikilink")の計算

また、平方完成の考え方を応用して解く手法も見られる（[\#類似の手法](https://ja.wikipedia.org/wiki/#類似の手法 "wikilink")）。

## 概観

二次式 \(ax^2 + bx + c \ (a \neq 0)\) において、一次の項「\(+ \; \! \; \! bx\)」があるのとないのでは、応用上の取り扱いが大きく異なる。

変数 \(x\) が \(x-h\) の形になる代わりに一次の項がなくなれば、\(h\) の違いだけで済むことができる。

ここでは、**二次の係数（最高次係数）**が  の場合とそうでない場合に分けてみる。

  - 二次の係数（最高次係数）が  の場合
    \(x^2 + bx + c\)

の一次の項「\(+ \; \! \; \! bx\)」をなくして \(x^2\) を \((x-h )^2\) の形にする。

\[(x-h )^2 = x^2 -2hx + h^2\] より、一次の係数を比較すると

\[b = -2h\]

\[h=-\frac{b}{2}\] これにより、 の平方完成は次の式になる：

\[x^2 + bx + c = \left( x+ \frac{b}{2} \right)^2 - \frac{b^2}{4} + c\]

  - 二次の係数（最高次係数）が  でない場合
    \(ax^2 + bx + c \quad (a \neq 0)\)

の一次の項「\(+ \; \! \; \! bx\)」をなくして \(x\) を \(x-h\) にする。

二次の係数が  の場合で得られた等式

\[x^2 + bx = \left( x+ \frac{b}{2} \right)^2 - \frac{b^2}{4}\] を利用する。

\[\begin{align}
ax^2 + bx + c &= a \left( x^2 + \frac{b}{a} x \right)+c \\
 &= a \left\{ \left( x + \frac{b}{2a} \right)^2 - \left( \frac{b}{2a} \right)^2 \right\} +c \\
 &= a \left( x + \frac{b}{2a} \right)^2 - \frac{b^2}{4a} +c \\
 &= a \left( x + \frac{b}{2a} \right)^2 - \frac{b^2 -4ac}{4a}
\end{align}\]\[1\] つまり、一次以上の項を二次の係数  で括ることにより、二次の係数が  の場合を利用している。

## 二次形式の平方完成

1変数の二次式の平方完成を踏まえて、一般の  変数二次式に対しても、平方完成ができる。例えば二変数なら

\[a x^2 +bxy + cy^2 +dx+ey+f \quad (abc \neq 0)\] である。これは[二次形式](https://ja.wikipedia.org/wiki/二次形式 "wikilink")

\[\begin{pmatrix}
x &y
\end{pmatrix} \begin{pmatrix}
a            &\frac{b}{2} \\
\frac{b}{2} &c
\end{pmatrix} \begin{pmatrix}
x \\
y
\end{pmatrix} + \begin{pmatrix}
x &y
\end{pmatrix} \begin{pmatrix}
d \\
e
\end{pmatrix} +f \quad (abc \neq 0)\] の形で書ける。

一般の  変数二次式は、 を[対称行列](../Page/対称行列.md "wikilink")として

  -
    \(x^{\top} Ax + x^{\top} b + c = (x - h)^{\top} A(x - h) + k \quad \left( h = -\frac{1}{2} A^{-1}b, \quad k = c - \frac{1}{4} b^{\top} A^{-1} b \right)\)

で書ける。

が対称でないときは  と  の式が

\[h = -(A+A^{\top} )^{-1} b, \quad k = c - h^{\top} Ah = c - b^{\top} (A+A^{\top} )^{-1} A(A+A^{\top} )^{-1} b\] とやや一般になるが同じ式で書ける。

## 類似の手法

平方完成とは、 の形の式に第三項  を加えて完全平方式を作る操作である。 が先に与えられていても、中間項  または  を加えることにより完全平方式を得ることができる。

### 相反式の平方完成

正の実数  に対して、自身とその逆数の和は

\[\begin{align}
x + \frac{1}{x} &= \left(x - 2 + \frac{1}{x} \right) + 2 \\[5pt]
                &= \left( \sqrt{x} - \frac{1}{\sqrt{x}} \right)^2 + 2
\end{align}\] このように平方完成すると、正の数とその逆数の和は常に  以上であることが示される。

### 複二次式の因数分解

複二次式

\[x^4 + 324\] を[因数分解](../Page/因数分解.md "wikilink")することを考える。この式は \((x^2)^2 + 18^2\) と見ることができるから、中間項  36*x*}} を考え、

\[\begin{align}
x^4 + 324 &= (x^4 + 36x^2 + 324 ) - 36x^2 \\
 &= (x^2 + 18)^2 - (6x)^2 \\
 &= (x^2 + 18 + 6x)(x^2 + 18 - 6x) \\
 &= (x^2 + 6x + 18)(x^2 - 6x + 18)
\end{align}\] と因数分解できる。

## 応用

### 積分

[不定積分](../Page/不定積分.md "wikilink")

\[\int \frac{dx}{ax^2+bx+c}\] の被積分関数に平方完成を適用すれば、より基本的な積分

\[\int \frac{dx}{x^2 - a^2} = \frac{1}{2a} \ln \left| \frac{x-a}{x+a} \right| +C\] または

\[\int \frac{dx}{x^2 + a^2} = \frac{1}{a} \arctan \frac{x}{a} +C\] に帰着できる。

### 複素数

を[複素数](../Page/複素数.md "wikilink")とするとき、

\[|z|^2 - b^* z - bz^* + c\]

  -
    （ は複素数、 は[実数](../Page/実数.md "wikilink")）
    （ はそれぞれ  の[複素共役](../Page/複素共役.md "wikilink")）

は常に実数である。このことは、複素数に対する恒等式  *uu\**}} を用いて、式を以下のように変形すると分かる：

\[\begin{align}
|z|^2 - b^* z - bz^* + c &= zz^* - b^* z - bz^* + bb^* - bb^* + c \\
 &= z( z^* - b^* ) - b(z^* - b^* ) - |b|^2 + c \\
        &= (z-b)(z^* -b^* ) - |b|^2 + c \\
        &= (z-b)(z-b)^* - |b|^2 + c \\
        &= |z-b|^2 - |b|^2 + c
\end{align}\]

別の例として、 を実数とするとき、

\[ax^2 + by^2\] は、 のとき、[複素数の絶対値](https://ja.wikipedia.org/wiki/複素数の絶対値 "wikilink")の平方を用いて書くことができる。実際に、\(z = \sqrt{a}\,x + i \sqrt{b}\,y\) と置けば

\[\begin{align}
|z|^2 &= z z^* \\
      &= (\sqrt{a}\,x + i \sqrt{b}\,y)(\sqrt{a}\,x - i \sqrt{b}\,y) \\
      &= ax^2 + by^2
\end{align}\] となる。

### 冪等行列

[正方行列](../Page/正方行列.md "wikilink")  が[冪等とは](https://ja.wikipedia.org/wiki/冪等行列 "wikilink")  *M*}} が成り立つことである。

\[M = \begin{pmatrix}
a & b \\
b & 1-a
\end{pmatrix}\] は、\(a^2 + b^2 = a\) ならば冪等行列である。平方完成により

\[(a - \tfrac{1}{2})^2 + b^2 = \tfrac{1}{4}\]  が実行列なら、これは -平面において中心 、半径  の円の方程式である。角度  を用いて書けば、

\[M = \frac{1}{2}\begin{pmatrix}
1 - \cos \theta &\sin\theta \\
\sin \theta     &1 + \cos \theta
\end{pmatrix}\] と媒介変数表示できる。
