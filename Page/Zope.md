**Zope**（ゾープ、**Z Object Publishing Environment**）は、プログラミング言語[Python](../Page/Python.md "wikilink")で書かれた、[オブジェクト指向](../Page/オブジェクト指向.md "wikilink")[ウェブアプリケーションサーバ](https://ja.wikipedia.org/wiki/アプリケーションサーバ "wikilink")、もしくは[Webアプリケーションフレームワーク](https://ja.wikipedia.org/wiki/Webアプリケーションフレームワーク "wikilink")である。Zope 2系列および、[Zope 3が並行してリリースされている](https://ja.wikipedia.org/wiki/Zope_3 "wikilink")。[Zope Public Licenseに基づき配布される](https://ja.wikipedia.org/wiki/Zope_Public_License "wikilink")[フリーソフトウェア](../Page/フリーソフトウェア.md "wikilink")である。

## 概要

アプリケーションサーバ、[ウェブサーバ](https://ja.wikipedia.org/wiki/Webサーバ "wikilink")、[データベース](../Page/データベース.md "wikilink")の機能をすべて包含しているため、Zopeのみでの環境構築が可能である。導入時点で、[ブログ](https://ja.wikipedia.org/wiki/ブログ "wikilink")や[ウィキ](https://ja.wikipedia.org/wiki/ウィキ "wikilink")といった[ウェブアプリケーション](https://ja.wikipedia.org/wiki/ウェブアプリケーション "wikilink")が同梱されており、これらを組み合わせて容易に[CMSが構築できる](https://ja.wikipedia.org/wiki/コンテンツ管理システム "wikilink")。より高機能なCMS、[RDBMSとの連携など](https://ja.wikipedia.org/wiki/関係データベース管理システム "wikilink")、様々な外部プロダクトによる機能拡張が可能である。

Zope の管理はウェブインターフェースの管理画面から行う。Pythonによる高度なカスタマイズが可能である。Zopeにおいてはすべては Python のオブジェクトであり、[ZODBと呼ばれる](https://ja.wikipedia.org/wiki/Zope_Object_Database "wikilink")[オブジェクトデータベース](https://ja.wikipedia.org/wiki/オブジェクトデータベース "wikilink")に格納される。

## Zope 3

Zope 3は、Zope 2から全面的に再実装され、Webアプリケーションフレームワークとなった。Zope 2.8.1より、Zope 3 の機能を利用することができる。

## Plone

オブジェクト指向CMSとも言える[Plone](https://ja.wikipedia.org/wiki/Plone "wikilink")は、Windows（XP以上）、Linux（CentOS、Ubuntuなど）、FreeBSD、MacOSXで動作する。最新版であるPlone3.3.5（2010年6月現在）のインストールには200MB以上のディスクスペースを必要とするが、これにはZope2.10.7およびPython2.4.5が含まれている。なお日本語文字コードは[UTF-8](https://ja.wikipedia.org/wiki/UTF-8 "wikilink")となる。サイトの管理にはオブジェクト指向の考え方が必要で、ファイルマネージャーに近い感覚でコンテンツを取り扱うところに特徴がある。独自のWebサーバーを内蔵しているので、[Apacheによる通常のWebサイトと共存するには](https://ja.wikipedia.org/wiki/Apache_HTTP_Server "wikilink")、ポートの設定などに工夫が必要となる。
