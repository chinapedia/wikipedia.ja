**文字参照**（もじさんしょう、）とは[HTMLなどの](../Page/HyperText_Markup_Language.md "wikilink")[SGML文書においては](../Page/Standard_Generalized_Markup_Language.md "wikilink")、直接記述できない[文字](../Page/文字.md "wikilink")や[記号](../Page/記号.md "wikilink")（[マークアップで使われる](../Page/マークアップ言語.md "wikilink")、[半角](https://ja.wikipedia.org/wiki/半角 "wikilink")の[不等号](https://ja.wikipedia.org/wiki/不等号 "wikilink")「`<`」や「`>`」など）を表記する際に用いられる方法である。SGML構成素のひとつとして定義されており、文書[文字集合](../Page/文字集合.md "wikilink")中の文字を参照する為の手段を提供する。HTMLにおける文字参照には、表記方法により**数値文字参照**\[1\]と**文字実体参照**\[2\]の二種が存在する。[XMLにおいては](../Page/Extensible_Markup_Language.md "wikilink")、HTMLにおける「数値文字参照」を「文字参照」と呼ぶ。なおHTMLにおける「文字実体参照」は、XMLでは**実体参照**\[3\]と呼び区別する。

## 数値文字参照（文字参照）

数値文字参照は、[10進数もしくは](../Page/十進法.md "wikilink")[16進数によって文書文字集合における該当文字を](../Page/十六進法.md "wikilink")[ISO 10646の文字番号で指定する方法である](https://ja.wikipedia.org/wiki/ISO_10646 "wikilink")。以下はその例示であり、同一の8分[音符](../Page/音符.md "wikilink")記号「♪」を示している。16進数による指定は元々の[SGMLの規格には存在しなかったが](../Page/Standard_Generalized_Markup_Language.md "wikilink")、HTML 4.0で規格化された。16進数表記の際には大文字小文字の区別がない。

  - 「`♪`」（10進数による指定）
  - 「`♪`」（16進数による指定）

## 文字実体参照（実体参照）

文字実体参照は、特定の文字列（名前のようなもの）によって文書文字集合における該当文字を指定する方法である。

文字実体参照の方が数値文字参照に比べて直感的に文字参照を行えるため、一般的に用いられる事が多い。しかしながら、文字実体参照として定義されていない特殊な文字（例えば前述の8分音符「♪」）については、数値文字参照を行うより他に参照方法がない。

### HTMLで使用可能な文字参照

以下は、HTMLマークアップにおいて頻繁に用いられると思われる文字実体参照である。

| 文字  | 文字実体参照 | 説明                                                                                             |
| --- | ------ | ---------------------------------------------------------------------------------------------- |
| ` ` |        | [ノーブレークスペース](https://ja.wikipedia.org/wiki/ノーブレークスペース "wikilink") - 折り返しを起こさない（ホワイトスペースではない）空白 |
| `<` | \<     | 小なり記号（半角）                                                                                      |
| `>` | \>     | 大なり記号（半角）                                                                                      |
| `&` | &      | [アンパサンド](../Page/アンパサンド.md "wikilink")（半角）                                                     |
| `"` | "      | 二重[引用符](https://ja.wikipedia.org/wiki/引用符 "wikilink")（半角）                                      |

数値文字参照の16進数による表現では[大文字](https://ja.wikipedia.org/wiki/大文字 "wikilink")と[小文字](https://ja.wikipedia.org/wiki/小文字 "wikilink")の区別がなかったのに対し、文字実体参照では大文字小文字の区別が存在する。よって、以下のような大文字小文字の違いによる文字実体参照の差異が存在する場合がある。

  - ä (`ä`) 小文字 a（[ウムラウト](../Page/ウムラウト.md "wikilink")付）
  - Ä (`Ä`) 大文字 A（ウムラウト付）

### XMLで使用可能な実体参照

XMLの仕様においては、定義済み実体として可能な文字は次に示す5種類のみである。これ以外の実体参照は、別に指定した[DTDを使用しない限りは使用できない](../Page/Document_Type_Definition.md "wikilink")。

| 定義済み実体 | 表示  | 使用用途（他の用途に使っても構わない）                                                                                                                                                     |
| ------ | --- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| &      | `&` | コメントや[CDATAセクションの内部に書く場合を除いて](https://ja.wikipedia.org/wiki/Extensible_Markup_Language#CDATAセクション "wikilink")、すべてこのように表記する。（属性値も含む\[4\]）                                |
| \<     | `<` | 上と同じ。（タグとして書く場合はもちろん除く。）                                                                                                                                                |
| \>     | `>` | SGMLとの互換性のために、要素の内容でテキスト"\]\]\>"を表記する時に"\]\]\>"のように使う。（[CDATAセクションの内部で使用するわけではないことに注意](https://ja.wikipedia.org/wiki/Extensible_Markup_Language#CDATAセクション "wikilink")） |
| "      | `"` | 属性値ではこの表記を行う（<タグ 属性="'''ここで使用する'''">〜</タグ>）                                                                                                                             |
| '      | `'` | 属性値ではこの表記を行う（\<タグ 属性=**'ここで使用する**'\>〜</タグ>）                                                                                                                             |

### タイプセット・スタイル句読法

見やすくするため頭文字「O」の隣に文字を配置している。

| 文字 | 文字実体参照 | 説明                 |
| -- | ------ | ------------------ |
| ‘O | `‘`    | left single quote  |
| O’ | `’`    | right single quote |
| O‚ | `‚`    | single low-9 quote |
| “O | `“`    | left double quote  |
| O” | `”`    | right double quote |
| O„ | `„`    | double low-9 quote |
| —O | `—`    | em dash            |
| –O | `–`    | en dash            |
