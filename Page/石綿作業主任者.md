**石綿作業主任者**（いしわたさぎょうしゅにんしゃ）は、[労働安全衛生法](../Page/労働安全衛生法.md "wikilink")に定められた[作業主任者](../Page/作業主任者.md "wikilink")([国家資格](https://ja.wikipedia.org/wiki/国家資格 "wikilink"))のひとつであり、石綿作業主任者技能講習を修了した者の中から事業者により選任される。

また、主任者となるための技能講習を修了した者すなわち資格取得者のこと、あるいは資格そのものを指すこともある。

## 概要

  - [石綿](../Page/石綿.md "wikilink")による身体的な被害防止の指揮・監督を行う。また、労働安全衛生上の労働者の衛生の確保にも配慮する。

## 受講資格

  - なし。（ただし18歳に満たないものは石綿での就労や、作業主任者として選任できない。）

## 技能講習

  - 制度改正により[2006年](../Page/2006年.md "wikilink")[4月1日](../Page/4月1日.md "wikilink")から新設された。
  - [2006年](../Page/2006年.md "wikilink")[3月31日](../Page/3月31日.md "wikilink")までに旧・[特定化学物質等作業主任者](https://ja.wikipedia.org/wiki/特定化学物質等作業主任者 "wikilink")技能講習を修了した者は、従来どおり、石綿作業主任者にも選任されることができる（新制度下での補講等を受ける必要はない）。現・石綿作業主任者技能講習は、旧・[特定化学物質等作業主任者](https://ja.wikipedia.org/wiki/特定化学物質等作業主任者 "wikilink")技能講習から独立した形になる（下記の表を参照）。

| 講習名                    | 作業主任者  | 備考 |
| ---------------------- | ------ | -- |
| 石綿                     | 特定化学物質 |    |
| 石綿作業主任者技能講習            | ○      | ×  |
| 旧・特定化学物質**等**作業主任者技能講習 | ○      | ○  |

*' 新旧制度の比較 （○は選任可能）*'

### 講習科目

1.  健康障害及びその予防措置に関する知識
2.  作業環境の改善方法に関する知識
3.  保護具に関する知識
4.  関係法令
5.  修了試験
