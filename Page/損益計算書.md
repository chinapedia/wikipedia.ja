**損益計算書**（そんえきけいさんしょ、等）は、[財務諸表](../Page/財務諸表.md "wikilink")の1つである。企業のある一定期間における[収益](../Page/収益.md "wikilink")（revenue）と[費用](../Page/費用.md "wikilink")（expense）の状態を表すために、[複式簿記](../Page/複式簿記.md "wikilink")で記録されたデータを集計することによって、[貸借対照表](../Page/貸借対照表.md "wikilink")などと同時に作成される。企業内において経営判断のための情報として用いるほか、[株主](../Page/株主.md "wikilink")や[債権者](https://ja.wikipedia.org/wiki/債権者 "wikilink")などに経営成績に関する情報を提供する。

## 名称

日本語では「損益計算書」で一貫している。**P/L** と略称されることもある。

英語では一般的に "Income Statement" や "Statement of Income"、"Profit and Loss Statement" と呼ばれるが、書類の名称としてはそのほかにもいくつかある。 以下に示す単語の前、又は後に "Statement" や "Statement of" が付く。括弧内の数字は米国での2005年の会社数の例\[1\]。

  - Income (255社)
  - Operations (254社)
  - Earnings (86社)
  - その他 (5社)\[2\]

英国や英連邦の企業では "Profit and Loss Statement" とする場合があるが、米国ではあまり無い。ただし、米国でも通称では日本同様に "P/L" と呼ぶことがある。

独語では、[Gewinn- und Verlustrechnung](https://ja.wikipedia.org/wiki/:de:Gewinn-_und_Verlustrechnung "wikilink")。

## 米国の損益計算書の構成

ARB43の「Comparative Financial Statements」規則S-X210.5-03で開示項目が示されている。

  - 総売上高と総収入（Net sales and gross revenue）
      - 有形製品の純売上高（総売上高から割引、返品、値引きを引いたもの）
      - 公益事業等からの収入
      - レンタル収入
      - サービス収入
      - その他の収入
  - 売上原価と収入に対する費用（Costs and expenses applicable to sales and revenue）
      - 有形製品の売上原価
      - 公益事業等の費用
      - レンタル収入対応費用
      - サービス費用
      - その他の収入に対応した費用を個別に示す

上記の各項目が総売上高と総収入や売上原価と収入に対する費用の10%未満であれば1つのグループとして表示が許される。

  - その他の損益（Miscellaneous other income and Miscellaeous income deduction）
  - 持分法損益（Equity in earnings of 50% or less owned persons）\[3\]

## 中国の損益計算書の構成

| 营业成本（営業原価）         | b |                         | [売上原価](../Page/売上原価.md "wikilink")                                                                                                                                     |
| 营业税金及附加（営業税）       | c |                         | [販売管理費](https://ja.wikipedia.org/wiki/販売管理費 "wikilink")の内の租税公課                                                                                                         |
| 销售费用（販売費用）         | d |                         | 販売管理費の内の販売費                                                                                                                                                            |
| 管理费用（管理費用）         | e |                         | 販売管理費の内の一般管理費                                                                                                                                                          |
| 财务费用（財務費用）         | f |                         | [営業外損益の内の支払](https://ja.wikipedia.org/wiki/営業外収益 "wikilink")[利息](https://ja.wikipedia.org/wiki/利息 "wikilink")（＋）、[為替](../Page/為替.md "wikilink")差損（＋）、受取利息（－）、為替差益（－）など |
| 资产减值损失（資産減損損失）     | g |                         | [特別損失の内の](https://ja.wikipedia.org/wiki/wikt:特別損失 "wikilink")[減損損失](../Page/減損会計.md "wikilink")                                                                        |
| 公允价值变动收益（公正価値変動収益） | h |                         | [営業外収益](https://ja.wikipedia.org/wiki/営業外収益 "wikilink")の内の[有価証券](../Page/有価証券.md "wikilink")含み損益など                                                                     |
| 投资收益（投資収益）         | i |                         | [営業外収益](https://ja.wikipedia.org/wiki/営業外収益 "wikilink")の内の有価証券売却損益、受取配当など                                                                                              |

| 营业外支出（営業外支出）       | l |                         | 特別損失の内の固定資産売却損、固定資産除却損など                                                                                                                                               |
| 利润总额（利潤総額）         | m | m=j+k-l                 | 税引前利益                                                                                                                                                                  |
| 所得税费用（所得税費用）       | n |                         | [法人税等](https://ja.wikipedia.org/wiki/法人税等 "wikilink")                                                                                                                  |
| 净利润（純利潤）           | o | o=m-n                   | [当期純利益](https://ja.wikipedia.org/wiki/当期純利益 "wikilink")                                                                                                                |

利润表（損益計算書）\[4\]

中国の上場株式会社の業績を把握するために財務報告書の損益計算書を利用する場合、「营业利润（営業利潤）」の項目は日本基準の「[営業利益](https://ja.wikipedia.org/wiki/営業利益 "wikilink")」と計算方法が異なるので注意を要する。
