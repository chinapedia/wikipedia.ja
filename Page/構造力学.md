**構造力学**（こうぞうりきがく、[英語](../Page/英語.md "wikilink")：structural mechanics）は[連続体力学](../Page/連続体力学.md "wikilink")の一分野であり、[橋](../Page/橋.md "wikilink")梁、[建築物](../Page/建築物.md "wikilink")、[ヴィークル類](../Page/乗り物.md "wikilink") などの構造物が[荷重](../Page/荷重.md "wikilink")を受けたときに生じる[応力](../Page/応力.md "wikilink")や[変形](https://ja.wikipedia.org/wiki/変形 "wikilink")などを解析するための[力学](https://ja.wikipedia.org/wiki/力学 "wikilink")である。一つの物体のときは[材料力学](../Page/材料力学.md "wikilink")という。[土木工学](../Page/土木工学.md "wikilink")の分野では根幹を成す学問分野であり、[水理学](../Page/水理学.md "wikilink")、[地盤力学と合わせて](../Page/土質力学.md "wikilink")「3力（さんりき）」と呼ばれることがある。

## 前提条件

[物理学](../Page/物理学.md "wikilink")における力学では変形しない物体を取り扱うが、構造力学では[変形](https://ja.wikipedia.org/wiki/変形 "wikilink")する現実的な物体を取り扱う。変形しても微小なものであり、目に見えるほど大きな変形は構造力学の適用範囲を超える。

変形発生時には力の作用する向きは変わらず、変形前に平面だった部分は変形後も平面と見なすこと（平面保持の仮定）ことで微小な変形とする。また、微小な変形においては加える力と変形量が比例する[線形であり](../Page/線型性.md "wikilink")、力を除くと元の力に戻る[弾性](../Page/弾性.md "wikilink")であると仮定する。

## 構造力学の問題を解くための三つの条件

次の三つの条件を式にすることで、あらゆる構造を解くことができる。

  - [力および](../Page/力_\(物理学\).md "wikilink")[モーメント](../Page/モーメント.md "wikilink")の平衡条件
  - [変位](../Page/変位.md "wikilink")の適合条件
  - 力と変位の関係

### 力およびモーメントの平衡条件

部材には水平力、鉛直力、モーメントがはたらくが、これらはつり合わなければならない。すなわち、次のつり合い条件式を満たさなければならない。（以下、変形を2次元で考える）

  -
    水平力 \(\sum H = 0\)
    鉛直力 \(\sum V = 0\)
    モーメント \(\sum M = 0\)

### 変位の適合条件

部材や支持部は、それらに適った変位でなければならない。連続した部材は変位も（場合によってはたわみ角なども）連続している。

支持形式には、ローラー支点、ヒンジ支点、固定支点、自由端がある。これらが満たす境界条件を下にまとめる。*u* は水平変位、*v* は鉛直変位、*dv*/*dx* = θ はたわみ角である。

| 支持形式   | 変位に関する境界条件                      | 支点反力に関する境界条件              |
| ------ | ------------------------------- | ------------------------- |
| ローラー支点 | *v* = 0                         | *H* = 0, *M* = 0          |
| ヒンジ支点  | *u* = 0, *v* = 0                | *M* = 0                   |
| 固定支点   | *u* = 0, *v* = 0, *dv*/*dx* = 0 | \-                        |
| 自由端    | \-                              | *H* = 0, *V* = 0, *M* = 0 |
