**[マジックソフトウェア・ジャパン株式会社](https://www.magicsoftware.com/ja/)**（Magic Software Japan K.K.）は、イスラエルに本社を置く、ローコード開発ツール[「Magic xpa Application Platform」](https://www.magicsoftware.com/ja/app-development-platform/xpa/)とデータ連携ツール[「Magic xpi Integration Platform」](https://www.magicsoftware.com/ja/integration-platform/xpi/)のソフトウェアベンダー、[Magic Software Enterprises](https://www.magicsoftware.com/)（NASDAQ：MGIC）の日本法人である。

本社は東京都新宿区にあり、その他の事業拠点として、札幌、仙台、信越、名古屋、大阪、岡山、広島、福岡にそれぞれ営業所がある。

## 事業内容

アプリケーションプラットフォーム分野：ローコード開発ツール「Magic xpa Application Platform」、および関連製品の仕入/製造、販売業務

システム連携ソリューション分野：データ連携ツール「Magic xpi Integration Platform」 の仕入/製造、販売業務
