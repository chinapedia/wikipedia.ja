[thumb](https://ja.wikipedia.org/wiki/ファイル:Huang_Tingjian.jpg "wikilink") **黄 庭堅**（こう ていけん、[慶暦](https://ja.wikipedia.org/wiki/慶暦 "wikilink")5年[6月12日](https://ja.wikipedia.org/wiki/6月12日_\(旧暦\) "wikilink")（[1045年](../Page/1045年.md "wikilink")[7月28日](../Page/7月28日.md "wikilink")） - [崇寧](https://ja.wikipedia.org/wiki/崇寧 "wikilink")4年[9月30日](https://ja.wikipedia.org/wiki/9月30日_\(旧暦\) "wikilink")（[1105年](../Page/1105年.md "wikilink")[11月8日](../Page/11月8日.md "wikilink")））は、[中国](../Page/中国.md "wikilink")[北宋](../Page/北宋.md "wikilink")時代の[書家](../Page/書家.md "wikilink")・[詩人](../Page/詩人.md "wikilink")・[文学者](https://ja.wikipedia.org/wiki/文学者 "wikilink")である。[洪州](https://ja.wikipedia.org/wiki/洪州 "wikilink")分寧県（現在の[江西省](../Page/江西省.md "wikilink")[九江市](../Page/九江市.md "wikilink")[修水県](https://ja.wikipedia.org/wiki/修水県 "wikilink")）の人。

[字](../Page/字.md "wikilink")は**魯直**（ろちょく）、[号は山谷道人](../Page/号_\(称号\).md "wikilink")（さんこくどうじん）、涪翁（ふうおう）などがある。**黄山谷**と呼ばれることが多い。宋代の詩人においては[蘇軾](../Page/蘇軾.md "wikilink")・[陸游](../Page/陸游.md "wikilink")と並び称され、書家としては蘇軾・[米芾](../Page/米芾.md "wikilink")・[蔡襄](https://ja.wikipedia.org/wiki/蔡襄 "wikilink")とともに[宋の四大家](https://ja.wikipedia.org/wiki/宋の四大家 "wikilink")に数えられる。「詩書画三絶」と讃えられ、師の蘇軾と名声を等しくして**蘇黄**と呼ばれた。蘇軾とともに、宋時代の詩を代表する人物であり、とくに、中国文学史上最初の自覚的な詩派である江西派は、12世紀前半に形成されて南宋期の文学界に大きな勢力を持つのだが、この派が開祖とあおいだのは黄山谷だったのである。子孫に[清](../Page/清.md "wikilink")代の詩人の[黄景仁](../Page/黄景仁.md "wikilink")がいる。

## 生涯

[治平](https://ja.wikipedia.org/wiki/治平_\(宋\) "wikilink")3年（1066年）に23歳で[進士](../Page/進士.md "wikilink")に及第した。黄庭堅は[王安石](../Page/王安石.md "wikilink")の[新法派と意見を対立させたため](../Page/新法・旧法の争い.md "wikilink")、[河南](../Page/河南省.md "wikilink")・[北京](../Page/北京市.md "wikilink")・[江西](../Page/江西省.md "wikilink")・[山東などの地方に赴任させられている](../Page/山東省.md "wikilink")。[元豊](https://ja.wikipedia.org/wiki/元豊_\(宋\) "wikilink")8年（1085年）に新法派を支えた[神宗が没すると旧法派が権力を奪取し](../Page/神宗_\(宋\).md "wikilink")、黄庭堅は首都[汴京](https://ja.wikipedia.org/wiki/汴京 "wikilink")にて校書郎・著作左郎・起居舎人など中央官僚として活躍した。34歳のころ蘇軾と知り合い、[張耒](https://ja.wikipedia.org/wiki/張耒 "wikilink")（ちょうらい）・[晁補之](https://ja.wikipedia.org/wiki/晁補之 "wikilink")（ちょうほし）・[秦観](https://ja.wikipedia.org/wiki/秦観 "wikilink")とともに蘇軾門下となり**蘇門四学士**と称された。汴京ではこの他にも多くの[文人](../Page/文人.md "wikilink")達と交友した。

しかし、[紹聖](https://ja.wikipedia.org/wiki/紹聖 "wikilink")元年（1094年）以降、新法派が再度実権を握ると黄は左遷され、[四川の](../Page/四川省.md "wikilink")[涪州](../Page/フ州.md "wikilink")（ふうしゅう）・[黔州](https://ja.wikipedia.org/wiki/黔州 "wikilink")（けんしゅう）・[戎州](https://ja.wikipedia.org/wiki/戎州 "wikilink")（じゅうしゅう）に貶謫されてしまう。崇寧2年（1103年）讒言を受けて[宜州](https://ja.wikipedia.org/wiki/宜州 "wikilink")（現在の[広西チワン族自治区](../Page/広西チワン族自治区.md "wikilink")）へ流刑となりその地で病没した。享年61。[南宋](../Page/南宋.md "wikilink")の[度宗](../Page/度宗.md "wikilink")のとき名誉が回復され文節の[諡号](https://ja.wikipedia.org/wiki/諡号 "wikilink")が贈られた。

地方への赴任は、自然を愛でて詩書画に耽溺する時間が許され、必ずしも不幸であったとはいえない。また[仏門](https://ja.wikipedia.org/wiki/仏門 "wikilink")に帰依し[老荘思想](../Page/老荘思想.md "wikilink")に傾倒するような自由な精神活動が行えた。むしろ黄庭堅の革新的な芸術を開花させるに理想的な環境であったといえる。

また親孝行でも有名であったらしく、[二十四孝](../Page/二十四孝.md "wikilink")の一人として知られている。

## 書

[right](https://ja.wikipedia.org/wiki/ファイル:Huang_Tingjian_Fu_Bo_Shen_Ci3..JPG "wikilink")』黄庭堅書（部分）紙本33.6×820.6cm、[永青文庫](../Page/永青文庫.md "wikilink")蔵
【釈文】経伏波／神祠／蒙蒙篁竹／下、有路上壷／頭。漢塁／麏鼯闘、／蠻溪霧\]\] 黄庭堅は草書をもっとも得意とした。若いときから草書が好きで、初め宋代の[周越を師としたが](https://ja.wikipedia.org/wiki/中国の書家一覧#周越 "wikilink")、越に学んでから20年ほどの間は古人の[用筆](https://ja.wikipedia.org/wiki/用筆 "wikilink")の妙を悟れず、俗気にとらわれてそれを脱することができなかった時期である。黄庭堅がもっとも苦しんだのがこの俗気を脱することであった\[1\]。

その後[顔真卿](../Page/顔真卿.md "wikilink")・[楊凝式](https://ja.wikipedia.org/wiki/楊凝式 "wikilink")・[懐素](../Page/懐素.md "wikilink")などの影響を受け、また[潤州](https://ja.wikipedia.org/wiki/潤州 "wikilink")の焦山岸辺にある[六朝時代](https://ja.wikipedia.org/wiki/六朝時代 "wikilink")の碑文『[瘞鶴銘](../Page/エイ鶴銘.md "wikilink")』の書体から啓発を受けて、丸みの有る文字が連綿と繋がる独自の[草書体](../Page/草書体.md "wikilink")を確立した。明らかに懐素の影響を受けていながら、筆跡の曲折は手厚く懐素のリズムと完全に異なっている。

[行書は洗練されてなお力強くて](../Page/行書体.md "wikilink")、独特の創造的書法をもつ。これらの書法は後世に対して大きい影響を与えた。そのため北宋の[書道界](../Page/書道界.md "wikilink")の傑出した存在となり、蘇軾と並び評価が高い。黄庭堅と蘇軾・米芾・蔡襄をして[宋の四大家](https://ja.wikipedia.org/wiki/宋の四大家 "wikilink")と称される。

### 書作品

作品には『[伏波神祠詩巻](https://ja.wikipedia.org/wiki/#伏波神祠詩巻 "wikilink")』・『[黄州寒食詩巻跋](https://ja.wikipedia.org/wiki/#黄州寒食詩巻跋 "wikilink")』・『[松風閣詩巻](https://ja.wikipedia.org/wiki/#松風閣詩巻 "wikilink")』・『[李白憶旧遊詩巻](https://ja.wikipedia.org/wiki/#李白憶旧遊詩巻 "wikilink")』などが知られる。

#### 伏波神祠詩巻

『伏波神祠詩巻』（ふくはしんししかん）は、[建中靖国](https://ja.wikipedia.org/wiki/建中靖国 "wikilink")元年（1101年）5月、[荊州](../Page/荊州.md "wikilink")で[劉禹錫](../Page/劉禹錫.md "wikilink")の「経伏波神祠詩」（ふくはしんしをへるのし）一首を楷書に近い行書で書いたもので、晩年の傑作として著名である。毎行3から5字、46行にわたる大作で、[張孝祥や](https://ja.wikipedia.org/wiki/中国の書家一覧#張孝祥 "wikilink")[文徴明](../Page/文徴明.md "wikilink")らの多くの跋がある。紙本33.6×820.6cm。[永青文庫](../Page/永青文庫.md "wikilink")蔵\[2\]\[3\]\[4\]。

#### 黄州寒食詩巻跋

『黄州寒食詩巻跋』（こうしゅうかんじきしかんばつ）は、[元符](https://ja.wikipedia.org/wiki/元符 "wikilink")3年（1100年）の書で、蘇軾の『[黄州寒食詩巻](https://ja.wikipedia.org/wiki/行書黄州寒食詩巻 "wikilink")』に彼が[題跋したものである](https://ja.wikipedia.org/wiki/書道用語一覧#題跋 "wikilink")。蘇軾の書も彼の会心の作であるが、この題跋も黄庭堅の作品の中で特にすぐれたものである。

行草体で9行、[落款](../Page/落款.md "wikilink")はない。内容は蘇軾の書を評して、「[顔真卿](../Page/顔真卿.md "wikilink")・[楊凝式](https://ja.wikipedia.org/wiki/楊凝式 "wikilink")・[李建中の筆意を兼ねており](https://ja.wikipedia.org/wiki/中国の書家一覧#李建中 "wikilink")、蘇軾に再び書かせてもこれほどの出来ばえにはならないであろう。」と讃えている。が、それにもまして黄庭堅の跋は尊敬する蘇軾の書を前にして堂々たる気構えをもって書している。そこには顔真卿と楊凝式の[書法](../Page/書法.md "wikilink")を学んだ跡が見られ、しかも禅僧のような気魄に満ちている\[5\]\[6\]\[7\]\[8\]。

#### 松風閣詩巻

『松風閣詩巻』（しょうふうかくしかん）は、崇寧元年（1102年）の流謫中の書で、晩年の作として特に重視されている。自詠の詩を行書で29行に書いている。この詩巻には顔真卿の他に、[柳公権](../Page/柳公権.md "wikilink")の筆意をも兼ねあわせた[筆致が伺え](https://ja.wikipedia.org/wiki/書道用語一覧#筆致 "wikilink")、一段と円熟した境地に達している。紙本。[台北](https://ja.wikipedia.org/wiki/台北 "wikilink")・[国立故宮博物院](../Page/国立故宮博物院.md "wikilink")蔵\[9\]\[10\]。

#### 李白憶旧遊詩巻

『李白憶旧遊詩巻』（りはくおくきゅうゆうしかん、李太白憶旧遊詩巻とも）は、紹聖元年（1094年）以後の書で、李白の「憶旧遊寄譙郡元参軍詩」（きゅうゆうをおもい しょうぐんげんさんぐんによするのし）一首を草書で書いたものである。紙本37cm×39.2cm。[藤井斉成会有鄰館](../Page/藤井斉成会有鄰館.md "wikilink")蔵\[11\]\[12\]\[13\]\[14\]。

### 日本への影響

[書道史](https://ja.wikipedia.org/wiki/書道史 "wikilink")家の財津永次によれば、[臨済宗](../Page/臨済宗.md "wikilink")の[宗峰妙超](../Page/宗峰妙超.md "wikilink")（大燈国師）には黄庭堅の書風の影響が見られ、また宗峰を介し[後醍醐天皇](https://ja.wikipedia.org/wiki/後醍醐天皇 "wikilink")にも影響を与えたという\[15\]。2020年時点で、宗峰妙超の墨跡は少なくとも5点が\[16\]\[17\]\[18\]\[19\]\[20\]、後醍醐天皇の墨跡は少なくとも3点が\[21\]\[22\]\[23\]、[国宝](../Page/国宝.md "wikilink")に指定されている。

## 詩文

黄庭堅は詩文にも優れ、[杜甫](../Page/杜甫.md "wikilink")の詩と[韓愈](../Page/韓愈.md "wikilink")の文に造詣が深い。杜甫の詩に影響されたのは、黄庭堅の父・黄庶が『伐檀集』（ばつたんしゅう）二巻の著で、杜甫の詩を好んだとしている。これも理由の一つとされる\[24\]。古人の詩文をもととし、これに創意を加えて新しい作品をつくることをいう「[換骨奪胎](https://ja.wikipedia.org/wiki/換骨奪胎 "wikilink")」の語で知られる詩論を確立し\[25\]、後世[江西詩派](https://ja.wikipedia.org/wiki/江西詩派 "wikilink")の開祖とされた。

代表作『山谷詩集』でも、書道芸術に対し重要な見解をいくつか発表している。優れた伝統の継承と個性の創造を強調して、その作品で実証してみせている。

| ---------- |
| **荊江亭即事**  |
| **原文**     |
| '''翰墨場中老伏波 |
| '''菩提坊裏病維摩 |
| '''近人積水無鷗鷺 |
| '''時有帰牛浮鼻過 |
|            |

山谷は多くの中国の詩人たちと同じく、公式的には儒教的立場を標榜しつつ、しかも老荘および仏教（禅学）思想に多大の興味と関心を払っている。かれは特に「荘子」を好んだ。それは、少年時代からの親友黄介（? - 1088年）の影響によるもので、この友人は、恐らく山谷よりも年長であったと考えられるが、読書人たちのあいだにまだそれほど流行していなかった時期に、早くもこの痛快な超俗の書を推賞していたのである。

山谷の作品における「荘子」の引用は、枚挙にいとまがないが、なかでも見落とすことのできないのは、「斉物論篇」に説かれる万物一体観にもとづく、自然への甚だしい愛着、大自然のうちに潑剌として生きる動物たちとの融合への願望であろう。

のちに、「二十四孝」のひとりに名を連ねられたように、温厚篤実の典型的な君子人とされている山谷ではあるが、かれのなかにも、世俗への嫌悪とそして野性へのあこがれが、実は根深く存在しつづけていたのである。

このように、大自然を愛した山谷の作品に、自然詩が、なかんずく純粋の叙景詩ー唐代の詩人たちが完成した山水の美の忠実な描写ーが、ほとんど見られない。かれは、自然をえがくことには、あまり、興味を持たぬように思われる。かれの作品に時としてあらわれる風景描写は、だいがいはむしろ紋切り型といってよく、類似表現の繰りかえされる場合が多い。
