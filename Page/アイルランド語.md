[right](https://ja.wikipedia.org/wiki/ファイル:Cainteoirí_Gaeilge_-_Irish_Speakers.svg "wikilink") **アイルランド語**（アイルランドご、 、）は、[インド・ヨーロッパ語族](../Page/インド・ヨーロッパ語族.md "wikilink")[ケルト語派](../Page/ケルト語派.md "wikilink")に属する[言語](../Page/言語.md "wikilink")である。現存する[ゲール語](../Page/ゲール語.md "wikilink")の一つであり、しばしば**アイルランド・ゲール語**や**アイリッシュ・ゲール**\[1\]、あるいは**西ゲール語**とも呼ばれる。[アイルランド](../Page/アイルランド.md "wikilink")共和国の[第一公用語であり](../Page/公用語.md "wikilink")、[2007年](../Page/2007年.md "wikilink")以降[欧州連合の公用語の一つである](../Page/欧州連合の言語.md "wikilink")\[2\]。

現代のアイルランド人の多くは[英語](../Page/英語.md "wikilink")を母語とするが（2002年の[国勢調査](../Page/国勢調査.md "wikilink")によると、41.9%がアイルランド語話者）、アイルランド語と英語は、同じインド・ヨーロッパ語族(ヨーロッパから南アジア、北アジア、アフリカ、南アメリカ、北アメリカ、オセアニアにかけて話者地域が広がる語族)であり、言語学上、言語系統的には同じ語族に分類されるが、[ケルト語派](../Page/ケルト語派.md "wikilink")であるアイルランド語と[ゲルマン語派](../Page/ゲルマン語派.md "wikilink")である英語とは、完全に別言語であり当然意思の疎通はできない。

## 現況

アイルランド語は元来は[アイルランド人](../Page/アイルランド人.md "wikilink")の固有の言語であり、被支配民の下層階級が中心とはいえ、アイルランドの人口の大部分をアイルランド語話者が占めていた。しかし、[イギリス](https://ja.wikipedia.org/wiki/イギリス "wikilink")によってアイルランド島全土のほとんどが植民地化されていた時代に[英語](../Page/英語.md "wikilink")にとって代わられ、話者の数が激減した。これには[ジャガイモ飢饉](https://ja.wikipedia.org/wiki/ジャガイモ飢饉 "wikilink")の影響も大きい。19世紀のアイルランド民族運動の高揚の中で、[ゲール語連盟](https://ja.wikipedia.org/wiki/ゲール語連盟 "wikilink")および初代大統領[ダグラス・ハイド](../Page/ダグラス・ハイド.md "wikilink")らによる復興活動がおこなわれた。

今日ではこの言語を日常的に使う人の数は非常に少なく、アイルランド国内においても「[ゲールタハト](../Page/ゲールタハト.md "wikilink")」() と呼ばれる一部の地域に限られるが、公共向けの掲示や交通標識の多くにはアイルランド語の併記が行われている。また政府の公職（[首相や](../Page/アイルランドの首相.md "wikilink")[議会](../Page/ウラクタス.md "wikilink")、政党、議員など）の名称をアイルランド語で表記し、[国防軍においては号令にアイルランド語を使用するなど](../Page/アイルランド国防軍.md "wikilink")、[民族主義](../Page/民族主義.md "wikilink")的な観点からも使用が推奨されている。

アイルランド共和国では[義務教育](../Page/義務教育.md "wikilink")においてアイルランド語は必修となっており、ゲールタハトに英語禁止の[修学旅行](../Page/修学旅行.md "wikilink")に行き、うっかり英語を使った生徒を一人で自宅に帰らすという厳しい措置をとった学校も存在したほどである\[3\]。また、[公務員](https://ja.wikipedia.org/wiki/公務員 "wikilink")試験などにおいてもアイルランド語の試験が必須とされるため、学習者は多いものの、ゲールタハトを除くほとんどの地域においては義務教育終了後、または就職後には忘れ去られ、使われなくなってしまうことが多い。

アイルランド政府は様々な保護策を採っており、ゲールタハトのネイティブなアイルランド語話者の家庭には、その土地に居住し続けることを条件に政府から補助金などが支給されている。しかし、すでに英語がアイルランドの優勢言語であり、英語に長けていない者は社会的に不利・不便な立場にあるという現実から、アイルランド語を母語とする親も子供の将来を考え、子供にはあえて英語で話しかける傾向にある\[4\]。このような状況から、西部の一部の海岸地域に点在するゲールタハトを除いては、アイルランドの日常生活においてアイルランド語の会話はほとんど聞かれないのが実情である。またゲールタハトも人口過疎の僻地に偏っており、比較的話者が多いと思われる唯一の都市（シティ）は西部の[ゴールウェイ](../Page/ゴールウェイ.md "wikilink")のみである。

多くのアイルランド国民にとってアイルランド語は、ヨーロッパ世界の古典語である[ラテン語](../Page/ラテン語.md "wikilink")の学習と同様に、非常に退屈なものである。英語化が進行していく過程においては辺境の言語・貧者の言語・劣等者の言語とまで貶められていたアイルランド語であるが、母語・日常語としては絶滅寸前となった近現代になってから国策的に保護される対象となると、状況は一変した。今日では、ゲールタハト以外であえてアイルランド語を用いるのは、むしろ権威主義的な、やや衒学的で気取った人間だという偏見さえある。

## アイルランド語におけるアルファベット

## 音声

[母音](../Page/母音.md "wikilink")[音素](../Page/音素.md "wikilink")は/a/, /e/, /i/, /o/, /u/という五つの[短母音](https://ja.wikipedia.org/wiki/短母音 "wikilink")とそれぞれに対応する[長母音](../Page/長母音.md "wikilink")、加えて[曖昧母音](https://ja.wikipedia.org/wiki/曖昧母音 "wikilink")/ə/を持つ。基本的に[強勢](https://ja.wikipedia.org/wiki/強勢 "wikilink")は第一音節に落ち、[強勢](https://ja.wikipedia.org/wiki/強勢 "wikilink")を持たない[短母音](https://ja.wikipedia.org/wiki/短母音 "wikilink")は[曖昧母音](https://ja.wikipedia.org/wiki/曖昧母音 "wikilink")に中和される。

[子音](../Page/子音.md "wikilink")[音素](../Page/音素.md "wikilink")は[破裂音](../Page/破裂音.md "wikilink")の/p/, /b/, /t/, /d/, /k/, /ɡ/、[摩擦音](../Page/摩擦音.md "wikilink")の/f/, /s/, /h/、[鼻音](../Page/鼻音.md "wikilink")の/m/, /n/, /ŋ/、[流音](../Page/流音.md "wikilink")の/r/, /l/を持つ。これらに加え、後述する[緩音現象により](../Page/子音弱化.md "wikilink")、[摩擦音](../Page/摩擦音.md "wikilink")または[接近音](../Page/接近音.md "wikilink")の/v～w/, /ɣ～j/, /x/が音素に加わる。また、[無声声門摩擦音](https://ja.wikipedia.org/wiki/無声声門摩擦音 "wikilink")/h/以外の[子音](../Page/子音.md "wikilink")は、それぞれ[口蓋化](../Page/口蓋化.md "wikilink")されたもの（狭子音：）とそうでないもの（広子音：）の二つに分かれるため、[子音](../Page/子音.md "wikilink")[音素](../Page/音素.md "wikilink")の数は比較的多い。これらの区別は、[スラヴ語](https://ja.wikipedia.org/wiki/スラヴ語 "wikilink")学において軟音・硬音と呼ばれるものに相当する。

## 文法

基本語順は[VSO](https://ja.wikipedia.org/wiki/VSO "wikilink")。例えば、

  -

    　私は彼を見た。

という文は「見た・私は・彼を」という順で構成されている。

[ケルト語派](../Page/ケルト語派.md "wikilink")に共通する特徴として、特定の統語環境において語の頭音が変化する[緩音現象](https://ja.wikipedia.org/wiki/緩音現象 "wikilink")を持つ。

### 緩音現象

[ケルト語派](../Page/ケルト語派.md "wikilink")に共通して見られる現象であるが、アイルランド語には軟音化（）\[5\]と暗音化（）の2種類が存在する。

#### 軟音化

軟音化を受けた[子音](../Page/子音.md "wikilink")は以下のように変化する。

  -
    元表記 \> 変化後表記 /音/ の順に記す。ここに挙げた子音は非口蓋化音（広子音）で代表させた。

|             |             |             |
| ----------- | ----------- | ----------- |
| p \> ph /f/ | b \> bh /w/ | f \> fh（消失） |
| t \> th /h/ | d \> dh /ɣ/ | s \> sh /h/ |
| c \> ch /x/ | g \> gh /ɣ/ | m \> mh /w/ |

おおむね、[破裂音](../Page/破裂音.md "wikilink")が[摩擦音](../Page/摩擦音.md "wikilink")ないしは[接近音](../Page/接近音.md "wikilink")に変化する。dとg、bとmはこの軟音化の結果、同じ音を表すことになる。

一例として、[名詞](../Page/名詞.md "wikilink")の女性[単数](https://ja.wikipedia.org/wiki/単数 "wikilink")[主格](../Page/主格.md "wikilink")形に[定冠詞](https://ja.wikipedia.org/wiki/定冠詞 "wikilink")anが付く場合にこの軟音化が起こる。

  -
    [男性名詞](https://ja.wikipedia.org/wiki/男性名詞 "wikilink")　fear (a man) \> an fear (the man)
    [女性名詞](https://ja.wikipedia.org/wiki/女性名詞 "wikilink")　bean (a woman) \> an **bh**ean (the woman)

#### 暗音化

暗音化を受けた[子音](../Page/子音.md "wikilink")は以下のように変化する。

  -
    元表記 \> 変化後表記 /音/ の順に記す。ここに挙げた子音は非口蓋化音（広子音）で代表させた。

|             |             |              |
| ----------- | ----------- | ------------ |
| p \> bp /b/ | b \> mb /m/ | f \> bhf /w/ |
| t \> dt /d/ | d \> nd /n/ |              |
| c \> gc /ɡ/ | g \> ng /ŋ/ |              |

おおむね、[無声音](../Page/無声音.md "wikilink")が[有声音](https://ja.wikipedia.org/wiki/有声音 "wikilink")に、[有声音](https://ja.wikipedia.org/wiki/有声音 "wikilink")が[鼻音](../Page/鼻音.md "wikilink")に変化する。一例として、[前置詞](https://ja.wikipedia.org/wiki/前置詞 "wikilink")i（）が用いられると続く[名詞](../Page/名詞.md "wikilink")にこの暗音化が起こる。なお、[大文字](../Page/大文字.md "wikilink")で書き始める[固有名詞](https://ja.wikipedia.org/wiki/固有名詞 "wikilink")の場合、その前に添える暗音化を示す文字は[小文字](../Page/小文字.md "wikilink")で書く。

  -
    Tóiceo (Tokyo) \> i **dT**óiceo (in Tokyo)

### 名詞

アイルランド語の[名詞](../Page/名詞.md "wikilink")は基本的に、[男性名詞](https://ja.wikipedia.org/wiki/男性名詞 "wikilink")・[女性名詞](https://ja.wikipedia.org/wiki/女性名詞 "wikilink")の二性、[単数](https://ja.wikipedia.org/wiki/単数 "wikilink")・[複数](../Page/複数.md "wikilink")の二数、[主格](../Page/主格.md "wikilink")・[属格](../Page/属格.md "wikilink")の二格を区別する。

#### 性

前述の通り、アイルランド語の[名詞](../Page/名詞.md "wikilink")は[男性名詞](https://ja.wikipedia.org/wiki/男性名詞 "wikilink")・[女性名詞](https://ja.wikipedia.org/wiki/女性名詞 "wikilink")の二つに分けられる。大まかに言って、語末が非[口蓋化](../Page/口蓋化.md "wikilink")音（広子音）で終わるものは[男性名詞](https://ja.wikipedia.org/wiki/男性名詞 "wikilink")、[口蓋化](../Page/口蓋化.md "wikilink")音（狭子音）で終わるものは[女性名詞](https://ja.wikipedia.org/wiki/女性名詞 "wikilink")という傾向がある。

  -
    [男性名詞](https://ja.wikipedia.org/wiki/男性名詞 "wikilink")：teach（家）、leabhar（本）など
    [女性名詞](https://ja.wikipedia.org/wiki/女性名詞 "wikilink")：cathair（街）、spéir（空）など

ただし、これには例外が数多く存在する。例えば、srón（鼻）は非口蓋化音で終わるが、女性名詞である。

#### 数

前述の通り、アイルランド語の[名詞](../Page/名詞.md "wikilink")は[単数](https://ja.wikipedia.org/wiki/単数 "wikilink")・[複数](../Page/複数.md "wikilink")の二つを区別する。複数形の作り方は多種多様であり、基本的には一つ一つ記憶しなければならない。また、[方言](../Page/方言.md "wikilink")によって複数形が異なることもあるので注意が必要である。そのうちのいくつかを以下に挙げる。

  - 語末の[子音](../Page/子音.md "wikilink")を[口蓋化](../Page/口蓋化.md "wikilink")
      -
        bád \> bá**id**（船）
  - 語末に-aを付加
      -
        clann \> clann**a**（子供、家族）
  - 語末に-annaを付加
      -
        am \> am**anna**（時）
  - 語末に-achaを付加
      -
        teanga \> teang**acha**（舌、言語）

#### 格

前述の通り、アイルランド語の[名詞](../Page/名詞.md "wikilink")は[主格](../Page/主格.md "wikilink")・[属格](../Page/属格.md "wikilink")の二つを区別する。ただし、一部の名詞はこれとは別に[与格](../Page/与格.md "wikilink")形を持つ。また多くの場合に他の[格](../Page/格.md "wikilink")と同一の形態を取るものの、[呼格](https://ja.wikipedia.org/wiki/呼格 "wikilink")も存在する。

主格は[動詞](../Page/動詞.md "wikilink")の[主語](../Page/主語.md "wikilink")や[直接目的語](https://ja.wikipedia.org/wiki/直接目的語 "wikilink")などに用いられ、属格は[名詞](../Page/名詞.md "wikilink")同士の[修飾](https://ja.wikipedia.org/wiki/修飾 "wikilink")関係を示すために用いられる。なお、与格は伝統的にそう呼ばれているが、単独で使用することはなく、特定の[前置詞](https://ja.wikipedia.org/wiki/前置詞 "wikilink")との組み合わせで用いる。 単数属格は大きく分けて、以下の5通りの作り方がある。

  - 語末の[子音](../Page/子音.md "wikilink")を[口蓋化](../Page/口蓋化.md "wikilink")
      -
        bád \> bá**id**（船）
        この場合、多くは単数属格と複数主格が同じ形となる。
  - （語末の子音が非口蓋化音なら口蓋化して）-eを付加
      -
        súil \> sú**ile**（目）
        fuinneog \> fuinneo**ige**（窓）
  - （語末の子音を口蓋化音なら非口蓋化して）-aを付加
      -
        rud \> ru**da**（もの）
        dochtúir \> dochtú**ra**（医者）
  - 不変化
      -
        teanga \> teanga（舌、言語）
  - 語末の子音を非口蓋化（または、非口蓋化音を付加）
      -
        athair \> atha**r**（父）
        caora \> caora**ch**（羊）

実際にはこれらに様々なバリエーションが存在するため、[複数形](https://ja.wikipedia.org/wiki/複数形 "wikilink")と並んで一つ一つ記憶しなければならない。

複数属格は基本的に複数主格をそのまま用いる。ただし、特別な複数属格形を用いる名詞もある。

### 動詞

[動詞](../Page/動詞.md "wikilink")は[時制](../Page/時制.md "wikilink")と法によって活用する。活用のタイプには大きく分けて2種類あり、辞書形（二人称単数命令形）が1[音節](../Page/音節.md "wikilink")かそれ以上かで見分けることができる。

|      | タイプI      | タイプII      |
| ---- | --------- | ---------- |
| 命令   | glac（取る）  | codail（眠る） |
| 習慣現在 | glacann   | codlaíonn  |
| 習慣過去 | ghlacadh  | chodlaíodh |
| 過去   | ghlac     | chodail    |
| 未来   | glacfaidh | codlóidh   |
| 条件法  | ghlacfadh | chodlódh   |
| 接続法  | go nglaca | go gcodlaí |

過去に関連する時制（過去、習慣過去および[過去未来](https://ja.wikipedia.org/wiki/過去未来 "wikilink")に相当する[条件法](../Page/条件法.md "wikilink")）では、動詞の語頭に軟音化が起きる。母音で始まる動詞、および軟音化の結果頭子音を失う、f-で始まる動詞にはd'-を付すことでこれらの形態を作る。

  -
    ól（飲む）＞**d**'ól-
    fan（待つ）＞**d'fh**an-

#### 人称形

主語の人称・数に一致した人称形も存在するが、それらは通常、[人称代名詞](../Page/人称代名詞.md "wikilink")との結合形であると解釈される。故に、人称形と主語人称代名詞を同時に用いることはできない。例えば、上のglac（取る）の習慣現在一人称単数形は

\*:glacaim のようになるが、 ＊glacaim mé のように対応する人称代名詞を置くことはできない。

この人称形は動詞活用のパラダイムにおいて全てが揃っている訳ではなく、またその使用範囲も方言によって異なる。

#### 自立形

これらの他に、自立形（）と呼ばれる形態が各時制・法に存在する。これは、主語が不特定多数であることを示す非人称形である。[フランス語](https://ja.wikipedia.org/wiki/フランス語 "wikilink")のon、[ドイツ語](../Page/ドイツ語.md "wikilink")のmanを用いた表現に類似した用法の他、ある種の受動文としての用法も持つ。

|      | タイプI        | タイプII          |
| ---- | ----------- | -------------- |
| 命令   | glac（取る）    | codail（眠る）     |
| 習慣現在 | glactar     | codlaítear     |
| 習慣過去 | ghlactaí    | chodlaítí      |
| 過去   | glacadh     | codlaíodh      |
| 未来   | glacfar     | codlófar       |
| 条件法  | ghlacfaí    | chodlófaí      |
| 接続法  | go nglactar | go gcodlaítear |

過去に関連する形態のうち、過去形だけはこの自立形において語頭の軟音化が起きない。
