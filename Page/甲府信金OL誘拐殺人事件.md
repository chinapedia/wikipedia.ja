**甲府信金OL誘拐殺人事件** （こうふしんきんオーエルゆうかいさつじんじけん）は[1993年](../Page/1993年.md "wikilink")（[平成](../Page/平成.md "wikilink")5年）[8月10日](../Page/8月10日.md "wikilink")に[山梨県](../Page/山梨県.md "wikilink")[甲府市](../Page/甲府市.md "wikilink")で発生した[身代金](https://ja.wikipedia.org/wiki/身代金 "wikilink")目的[誘拐](https://ja.wikipedia.org/wiki/誘拐 "wikilink")[殺人事件](https://ja.wikipedia.org/wiki/殺人事件 "wikilink")。

## 事件の概要

### 事件発生から遺体発見まで

被害者女性（当時19歳）は甲府市にある[甲府信用金庫](../Page/甲府信用金庫.md "wikilink")の支店に勤務する新人OLであった。この日も窓口業務が終了する時間になった時、本店を経由して地元マスメディアを名乗る男から被害者女性を指名して取材依頼がくる。被害者女性および上司は応諾、勤務時間終了後被害者女性は、電話の男が差し向けたタクシーで待ち合わせ場所の[小瀬スポーツ公園に向かったきり行方不明となる](https://ja.wikipedia.org/wiki/山梨県小瀬スポーツ公園 "wikilink")。

翌日、被害者女性の父親が帰宅していないことを支店に問い合わせた時、身代金を要求する一本の電話が入ったことから誘拐が発覚。支店側はすぐさま[山梨県警に通報](https://ja.wikipedia.org/wiki/山梨県警察 "wikilink")、山梨県警は犯人を刺激しないよう非公開としつつ、その後もかかってくる犯人からの電話に[逆探知](https://ja.wikipedia.org/wiki/逆探知 "wikilink")で犯人の居場所を特定しようとする。

犯人は映画「[天国と地獄](https://ja.wikipedia.org/wiki/天国と地獄_\(映画\) "wikilink")」の手法で[中央自動車道](../Page/中央自動車道.md "wikilink")の104[キロポストから身代金](../Page/距離標.md "wikilink")4500万円を投下するよう指示するも身代金奪取に失敗。しかし、山梨県警は身代金受取場所に遅れるなどのミスを犯したほか、犯人も1キロ離れた105キロポストで待機するミスを犯していた\[1\]。その後犯人からの連絡は途絶え、誘拐されてから1週間後の8月17日、被害者女性は[静岡県](../Page/静岡県.md "wikilink")[富士宮市](../Page/富士宮市.md "wikilink")の[富士川](../Page/富士川.md "wikilink")で遺体として発見された。

### 情報公開から犯人逮捕まで

遺体発見後の同日午後6時41分に山梨県警とマスメディアは[報道協定](https://ja.wikipedia.org/wiki/報道協定 "wikilink")を解除し、一斉公開に踏み切る。同時に遺体を発見した[静岡県警と共に合同捜査本部を設置した](https://ja.wikipedia.org/wiki/静岡県警察 "wikilink")。 メディアでは「犯人は男3人組」「共犯者に女性がいる」など根拠のない情報が流され、また報道協定を解除したことで山梨県警には一日700件の情報が寄せられたが、いずれも有力な情報に繋がらなかった。

そこで逆探知装置に残っていた声をもとに音声・音響分野の研究を行なっている[鈴木松美](https://ja.wikipedia.org/wiki/鈴木松美 "wikilink")に声紋鑑定を依頼、鈴木の声紋鑑定により以下のことが割り出された。これらは逮捕された際ほぼ一致しており、鈴木の声紋鑑定が正確であったことが伺える\[2\]。

| 特徴  | 鈴木による鑑定                                                                                                                                                                    | 実際の犯人                                                                      |
| --- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| 身長  | 音の高さは身長が高くなるほど低くなる傾向がある。逆探知装置に残された音の高さから身長170cm前後である。                                                                                                                      | 犯人の身長は172cmであり、ほぼ一致している。                                                   |
| 年齢  | 年を取るごとに声帯の筋肉が劣ることから、おおよその年齢がわかる。逆探知装置に残された音をもとに年齢は40歳から55歳の間である。                                                                                                           | 逮捕時の犯人の年齢は38歳であり、鈴木の判断より年齢が若いが、大幅に間違ってはいない。                                |
| 在住地 | 約束を「やぐぞぐ」と濁音の訛りで言ったり、「そうですか」を「ほうすっか」と言うなど、[甲州弁](https://ja.wikipedia.org/wiki/甲州弁 "wikilink")（国中弁）を多用していることから[甲府盆地](https://ja.wikipedia.org/wiki/甲府盆地 "wikilink")に在住している。 | 犯人は甲府盆地出身・在住であり、鈴木の鑑定が一致したことになる。                                           |
| 職業  | 身代金要求時に「無地の帯封」を求めている。これは現金で大金を扱っている者にしかわからない特徴であり、高額なものを取り扱う営業職ではないか。                                                                                                      | 大型トラック等の自動車販売会社に勤めるセールスマンであり、封帯が必要な数百万単位の高額なものを取り扱っている。よって鈴木の鑑定が一致したことになる。 |
|     |                                                                                                                                                                            |                                                                            |

さらに鈴木は電話を切る時に出る電気信号をもとにどこから脅迫電話をかけたかの特定を試みる（当時の[公衆電話](https://ja.wikipedia.org/wiki/公衆電話 "wikilink")はアナログ式であり、距離および交換機の数によってパルス波長が異なる）。その結果中央自動車道の[境川パーキングエリア](../Page/境川パーキングエリア.md "wikilink")の公衆電話からかけたことがわかり、そこは犯人が身代金を投下するよう指示した104キロポストの至近であった\[3\]。

鈴木による鑑定の内容が連日報道され、また鈴木自身も作業場を公開したりテレビ出演する等協力の姿勢を見せた中、犯人の知人である建設会社社長が鈴木による鑑定内容と音声を報道で聞いて間違いないと判断し、犯人を呼び出し[自首](https://ja.wikipedia.org/wiki/自首 "wikilink")するよう説得する。犯人は当初否認していたが説得に折れ、8月24日の早朝に犯人の男が建設会社社長に連れられ山梨県警所轄の警察署に出頭し、逮捕された\[4\]。

### 経緯

犯人の男は、販売実績アップのために数多くでっち上げた架空契約の支払いや、愛人であった[韓国人](https://ja.wikipedia.org/wiki/韓国人 "wikilink")[ホステス](https://ja.wikipedia.org/wiki/ホステス "wikilink")との交際費等で約7000万円の借金を抱えており、その返済目的で犯行に及んだとされる。また被害者女性を指名した理由は支店を訪れた際、名札をつけていた被害者女性が当時デビューしたばかりの女優・歌手と同姓同名だったため名前を覚えており、その女性を誘拐の対象にし、犯行に及んだ。先輩職員が名札をつけていない中、新入社員の被害者女性だけまじめに名札をつけていたこともあり、これが仇になってしまったのである。

尚、被害者女性は誘拐されたその日に殺害され、富士川上流の[笛吹川](https://ja.wikipedia.org/wiki/笛吹川 "wikilink")から流されていた。

## 裁判

[甲府地方裁判所](../Page/甲府地方裁判所.md "wikilink")で一審が開かれ、犯人の男は犯行を全面に認めたことから争点は自首の有効性について争われた。弁護側は自首は有効であること主張する一方、[甲府地検はすでに犯人がこの男と特定](../Page/甲府地方検察庁.md "wikilink")（さらに境川パーキングエリアの駐車していた自動車のナンバーを控えており、そのうちの1台が犯人の自宅に停まっていたなど割り出されていた\[5\]）されており、自首は無効という理由で[死刑](../Page/死刑.md "wikilink")を求刑した。そして審議の結果、甲府地裁は自首の成立を否定したものの、[無期懲役](https://ja.wikipedia.org/wiki/無期懲役 "wikilink")の判決を下した。

これに不服として甲府地検は[東京高等裁判所](../Page/東京高等裁判所.md "wikilink")に控訴したものの東京高裁は甲府地裁の判決を支持し、控訴を棄却した。その後地検側、弁護側ともに期日までに上告しなかったため、1996(平成8)年5月1日に犯人の無期懲役が確定した。

## 影響

### テレビ番組

[1993年](../Page/1993年.md "wikilink")[8月26日](../Page/8月26日.md "wikilink")に放送される予定であったタモリのドラマ番組『[If もしも](https://ja.wikipedia.org/wiki/If_もしも "wikilink")』の「誘拐するなら男の子か女の子か」が放送中止になる。その後現在も放送されていない（[世にも奇妙な物語](https://ja.wikipedia.org/wiki/世にも奇妙な物語 "wikilink")と同様の権利関係の問題とも考えられる）。

### デマ

犯人逮捕後の10月に「被害者の父親が犯人と顔見知りであり、共謀して被害者を殺害した」というデマが流れた。根も葉もない嘘であったが、当初の複数犯人説を信じ続ける者がいた。
