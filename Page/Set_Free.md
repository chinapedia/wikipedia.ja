Set Free（セットフリー）は、日本の5人組ロックバンド。京都を中心に活動している。ジャンルはポップ。

## メンバー

  - **清田尚吾**（きよた しょうご）：Vo.（ボーカル）担当
  - **ワイニー**：パフォーマンス・パーカッション担当
  - **三輪昇太郎**（みわ しょうたろう）：Dr.（ドラム）担当
  - **土居龍月**（どい たつき）：Gt.（ギター）担当
  - **東木塲龍矢**（ひがしこば りゅうや）：Ba.（ベース）担当

### 旧メンバー

  - **緒川謙吾**（おがわ けんご）：Ba.（ベース）担当
      - 2018年12月 - 2019年2月在籍

## 略歴

  - **2014年**
      - 9月、活動開始\[1\]。当初はVo.清田尚吾とワイニーの2人体制であった。
  - **2018年**
      - 12月、Ba.緒川謙吾、Dr.三輪昇太郎、Gt.土居龍月が加入。5人体制になる\[2\]。
  - **2019年**
      - 2月、Ba.緒川謙吾が卒業\[3\]。
      - 3月、RO JACKに入賞\[4\]。
      - 3月28日、ライブオーディション「TOKYO BIG UP\!」（第3回）にてグランプリを獲得\[5\]。
      - 7月、サポートメンバーのBa.東木塲龍矢がSet Freeに加入\[6\]。
  - **2020年**
      - 3月11日、初の全国流通盤であるミニアルバム『トースター』をリリース\[7\]。
