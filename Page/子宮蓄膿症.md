[Pyometra_annotated.JPG](https://ja.wikipedia.org/wiki/File:Pyometra_annotated.JPG "fig:Pyometra_annotated.JPG") **子宮蓄膿症**（しきゅうちくのうしょう、英:pyometra）とは[子宮腔](https://ja.wikipedia.org/wiki/子宮腔 "wikilink")内に[膿](https://ja.wikipedia.org/wiki/膿 "wikilink")汁が貯留して排出しない状態。[牛と](https://ja.wikipedia.org/wiki/ウシ "wikilink")[犬](https://ja.wikipedia.org/wiki/犬 "wikilink")に多い。

## 原因

  - 牛
      - [胚](https://ja.wikipedia.org/wiki/胚 "wikilink")あるいは[胎子](https://ja.wikipedia.org/wiki/胎子 "wikilink")が早期死滅し、これが子宮内に留まり、細菌感染を起こすことに原因することが多い。
  - 犬
      - 犬では高齢の未経産あるいは長く繁殖を休止している経産の犬で、[黄体期](https://ja.wikipedia.org/wiki/黄体期 "wikilink")に発生しやすい。

## 症状、診断

  - 牛
      - 概して軽症であり、全身性の症状、血液性状の変化は示さない。子宮蓄膿症である牛の多くで遺残[黄体](https://ja.wikipedia.org/wiki/黄体 "wikilink")が存在することにより[発情](https://ja.wikipedia.org/wiki/発情 "wikilink")が現れないこと、膿汁により妊娠2 - 3ヶ月ほどの子宮のように触知されることから[妊娠](../Page/妊娠.md "wikilink")との鑑別診断が必要である。

| 鑑別点    | 妊娠        | 子宮蓄膿症        |
| ------ | --------- | ------------ |
| 子宮の膨満  | 左右不対称     | 左右対称         |
| 子宮の収縮性 | 妊娠3か月まであり | なし・弛緩        |
| 子宮の波動感 | 水様性       | 粘稠性          |
| 子宮壁    | 柔軟で弾力あり   | 菲薄または肥厚、弾力なし |
| 子宮動脈   | 妊角側が発達肥大  | 左右同等         |
| 胎子     | 触知可       | 触知不可         |

**直腸検査によるウシの妊娠と子宮蓄膿症との鑑別点**

  - 犬
      - 食欲不振、[多飲多尿](https://ja.wikipedia.org/wiki/多飲多尿 "wikilink")、嘔吐、腹部膨満、外陰部から悪臭のある膿汁の排出、[白血球](../Page/白血球.md "wikilink")増多、[好中球の左方移動](https://ja.wikipedia.org/wiki/好中球の左方移動 "wikilink")がみられる。

## 治療

  - 牛
      - 子宮頚管を弛緩させるために[エストロゲン](../Page/エストロゲン.md "wikilink")を投与する、あるいは子宮収縮により膿汁を排出させるためにPGF<sub>2α</sub>の投与する。
  - 犬
      - 卵巣子宮全摘出が最良の方法とされ、治療後の繁殖を希望する場合[外科手術](https://ja.wikipedia.org/wiki/外科手術 "wikilink")の代わりPGF<sub>2α</sub>およびその類縁物質の投与を行われるが、[副作用](../Page/副作用.md "wikilink")、再発などが生じることがある。

## その他

卵巣を摘出したイヌにおいてエストロゲンおよびプロゲステロンを周期的に長期間投与すると子宮蓄膿症を誘発することができる。
