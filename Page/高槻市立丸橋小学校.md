**高槻市立丸橋小学校**（たかつきしりつ まるばししょうがっこう）は、[大阪府](../Page/大阪府.md "wikilink")[高槻市](../Page/高槻市.md "wikilink")にある[公立](../Page/公立学校.md "wikilink")[小学校](../Page/小学校.md "wikilink")。略称は「マルハシ」「マルバシ」「マルショウ」などがある。

|       |        |        |        |
| ----- | ------ | ------ | ------ |
|       | 2011年度 | 2016年度 | 2021年度 |
| 児童数   | 640    | 574    | 514    |
| 通常学級数 | 18     | 17     | 17     |
| 支援学級数 | 4      | 6      | 6      |

## 著名な出身者

・二川孝広\[2\] - プロサッカー選手（[FCティアモ枚方](../Page/FCティアモ枚方.md "wikilink")）

## 沿革

  - [1976年](../Page/1976年.md "wikilink")4月1日　開校　
  - 7月14日　簡易[プール](../Page/プール.md "wikilink")完成
  - [1979年](../Page/1979年.md "wikilink")4月2日　[養護学級](https://ja.wikipedia.org/wiki/養護学級 "wikilink")（[知的障害](https://ja.wikipedia.org/wiki/知的障害 "wikilink")）設置
  - [1987年](https://ja.wikipedia.org/wiki/1987年 "wikilink")6月12日　プール竣工
  - [1997年](https://ja.wikipedia.org/wiki/1997年 "wikilink")3月31日　[コンピュータ](../Page/コンピュータ.md "wikilink")ルーム完工
  - 2009年4月　コンピューター各クラス配置完工

## 交通アクセス

  - [JR京都線](https://ja.wikipedia.org/wiki/JR京都線 "wikilink")[摂津富田駅](https://ja.wikipedia.org/wiki/摂津富田駅 "wikilink")および[阪急京都本線](../Page/阪急京都本線.md "wikilink")[富田駅下車後](https://ja.wikipedia.org/wiki/富田駅_\(大阪府\) "wikilink")、[高槻市営バス](https://ja.wikipedia.org/wiki/高槻市営バス "wikilink")「芝生東口」停留所下車後すぐ。
