「**工業**」（こうぎょう）は、[産業としての](../Page/第二次産業.md "wikilink")[工業](../Page/工業.md "wikilink")の各分野に関する知識と技術を習得させることなどを目的とする[教科](../Page/教科.md "wikilink")。後期[中等教育](../Page/中等教育.md "wikilink")（[高等学校](https://ja.wikipedia.org/wiki/高等学校 "wikilink")、[中等教育学校](../Page/中等教育学校.md "wikilink")の[後期課程](https://ja.wikipedia.org/wiki/後期課程 "wikilink")、[特別支援学校](../Page/特別支援学校.md "wikilink")の[高等部](https://ja.wikipedia.org/wiki/高等部 "wikilink")）における専門教育に関する各教科（専門教科）の1つである。

教科「工業」は、「[工業に関する学科](https://ja.wikipedia.org/wiki/#工業に関する学科 "wikilink")」（工業学科）や、「[総合学科](../Page/総合学科.md "wikilink")」などで主に開講・学習される。

教科「工業」に属する科目の数は60(2013年入学者以降は61)にのぼり、そのいくつかと普通教科を組み合わせて[教育課程](../Page/教育課程.md "wikilink")を編成することで、主に専門学科や総合学科においては、学科の特色が活きるように配慮されている。

## 教科の目的

工業の各分野に関する基礎的・基本的な知識と技術を習得させ、現代社会における工業の意義や役割を理解させるとともに、環境に配慮しつつ、工業技術の諸問題を主体的、合理的に解決し、社会の発展を図る創造的な能力と実践的な態度を育てる。（高等学校学習指導要領から）

### 平成25年度からの学年進行によるおもな科目変更

従前の60科目から61科目に再編される（文部科学省『高等学校学習指導要領』（平成21年3月告示）による改訂）。

  - (新設)　→　「環境工学基礎」
  - マルチメディア応用　→　「コンピュータシステム技術」に変更

### 平成15年度からの学年進行によるおもな科目変更

従前の74科目から60科目に再編された（文部省『高等学校学習指導要領』（平成12年3月告示）による改訂）。

  - 工業基礎　→　「工業技術基礎」に変更
  - 工業数理　→　「工業数理基礎」に変更
  - 工業英語　→　「工業技術英語」に変更
  - 造船工学　→　廃止
  - 電力応用　→　「電力技術」に統合
  - コンピュータ応用　→　「マルチメディア応用」に変更
  - 工業計測技術　→　廃止
  - 設備施工　→　「設備計画」「空気調和設備」「衛生・防災設備」に整理統合
  - 土木設計　→　「土木基礎力学」「土木構造設計」に再構成
  - 水理　　　→　「土木基礎力学」に統合
  - 土質力学　→　「土木基礎力学」に統合
  - 土木計画　→　「社会基盤工学」に変更
  - 地質工学　→　廃止
  - 化学工業　→　「工業化学」に統合
  - 化学システム技術　→　「化学工学」「生産システム技術」に統合
  - 化学工業安全　→　「地球環境化学」に統合
  - 環境工学　→　「地球環境化学」に統合
  - 環境保全　→　「地球環境化学」に統合
  - セラミック材料　→　「セラミック化学」に統合
  - 繊維技術　→　「繊維・染色技術」に統合
  - 染色技術　→　「繊維・染色技術」に統合

  - 電子基礎　→　「生産システム技術」に統合

以上の他、各学校で学校設定科目が設置されることもある。

## 工業に関する学科

**工業に関する学科**（こうぎょうにかんするがっか）は、[高等学校設置基準](https://ja.wikipedia.org/wiki/高等学校設置基準 "wikilink")（平成16年文部科学省令第20号）に規定されている[専門教育を主とする学科](../Page/専門教育を主とする学科.md "wikilink")の1つ。専門教科「工業」を中心に履修し、[高等学校学習指導要領](https://ja.wikipedia.org/wiki/高等学校学習指導要領 "wikilink")（平成11年文部省告示第58号）により「工業技術基礎」及び「課題研究」が原則履修科目として指定されている。

### 設置状況

「工業に関する学科」は、ほぼ全ての[工業高等学校](../Page/工業高等学校.md "wikilink")及び[商工高等学校](../Page/商工高等学校.md "wikilink")に設置されているほか、一部の[実業高等学校](../Page/実業高等学校.md "wikilink")に設置されている。この他に、窯業科などが設置されている[窯業高等学校](https://ja.wikipedia.org/wiki/窯業高等学校 "wikilink")がある。かつては「農業に関する学科」を併設する[農工高等学校](../Page/農工高等学校.md "wikilink")も存在した。

実習のために高度な設備を必要とするため、[普通高等学校への設置は比較的少数であるが](../Page/普通科_\(学校\).md "wikilink")、それでも工業高等学校などから転換した学校を中心に設置例が見られる。

## 備考

[教育職員免許法](../Page/教育職員免許法.md "wikilink")第5条の規定によって[教育職員の](https://ja.wikipedia.org/wiki/教員免許 "wikilink")[普通免許を取得する場合には](https://ja.wikipedia.org/wiki/教育職員免許状#普通免許 "wikilink")、基本的に[教育実習](../Page/教育実習.md "wikilink")を行うことが義務付けられているが、[高等学校の](https://ja.wikipedia.org/wiki/高等学校#日本の高等学校 "wikilink")「工業」の教科に関する[教員](https://ja.wikipedia.org/wiki/高等学校教諭 "wikilink")[免許を](../Page/教育職員免許状.md "wikilink")[取得する際には](https://ja.wikipedia.org/wiki/高等学校教諭#免許状を取得する方法 "wikilink")、特例として教育実習は不要である。しかも、「工業」の教科に関する[高等学校教員の](https://ja.wikipedia.org/wiki/高等学校教諭 "wikilink")[普通免許は](https://ja.wikipedia.org/wiki/教育職員免許状#普通免許 "wikilink")、数ある教育職員の普通免許で唯一教育実習がなくても、取得することができる教育職員の普通免許状であることがうかがえる。
