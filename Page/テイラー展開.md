[数学](../Page/数学.md "wikilink")において、**テイラー級数** () は[関数のある一点での](../Page/関数_\(数学\).md "wikilink")[導関数](https://ja.wikipedia.org/wiki/導関数 "wikilink")たちの値から計算される項の[無限和](https://ja.wikipedia.org/wiki/無限和 "wikilink")として関数を表したものである。そのような級数を得ることを**テイラー展開**という。

テイラー級数の概念はスコットランドの数学者[ジェームズ・グレゴリー](https://ja.wikipedia.org/wiki/ジェームズ・グレゴリー "wikilink")により定式化され、フォーマルにはイギリスの数学者[ブルック・テイラー](../Page/ブルック・テイラー.md "wikilink")によって1715年に導入された。0 を中心としたテイラー級数は、**マクローリン級数** () とも呼ばれる。これはスコットランドの数学者[コリン・マクローリン](../Page/コリン・マクローリン.md "wikilink")にちなんでおり、彼は18世紀にテイラー級数のこの特別な場合を積極的に活用した。

関数はそのテイラー級数の有限個の項を用いて近似することができる。[テイラーの定理](https://ja.wikipedia.org/wiki/テイラーの定理 "wikilink")はそのような近似による誤差の定量的な評価を与える。テイラー級数の最初のいくつかの項として得られる多項式はと呼ばれる。関数のテイラー級数は、その関数のテイラー多項式で次数を増やした[極限が存在すればその極限である](https://ja.wikipedia.org/wiki/数列の極限 "wikilink")。関数はそのテイラー級数がすべての点で[収束するときでさえもテイラー級数に等しいとは限らない](https://ja.wikipedia.org/wiki/収束_\(数学\) "wikilink")。[開区間](https://ja.wikipedia.org/wiki/開区間 "wikilink")（あるいは[複素平面](https://ja.wikipedia.org/wiki/複素平面 "wikilink")の[開円板](https://ja.wikipedia.org/wiki/開円板 "wikilink")）でテイラー級数に等しい関数はその区間上の[解析関数](../Page/解析関数.md "wikilink")と呼ばれる。

点  を含む[開区間](https://ja.wikipedia.org/wiki/開区間 "wikilink")  上で[無限回微分可能](https://ja.wikipedia.org/wiki/無限回微分可能 "wikilink")な実数値関数  が与えられたとき、[べき級数](https://ja.wikipedia.org/wiki/べき級数 "wikilink")

\[\sum_{n=0}^{\infin} \frac{f^{(n)}(a)}{n!} (x - a)^{n}\] を関数  の点  まわりの**テイラー級数**という。ここで  は  の[階乗](../Page/階乗.md "wikilink")、 は  *a*}} における  の  次[微分係数](https://ja.wikipedia.org/wiki/微分係数 "wikilink")である\[1\]。また、便宜的に  は 1 であると定義する\[2\]。テイラー級数が[収束](https://ja.wikipedia.org/wiki/収束 "wikilink")し、元の関数  に一致するとき、 は**テイラー展開可能**であるという。テイラー展開がある大域的な[領域の各点で可能な関数は](https://ja.wikipedia.org/wiki/領域_\(解析学\) "wikilink")、その領域において**解析的** () である、またはその領域上の**[解析関数](../Page/解析関数.md "wikilink")** () であるという。

ここで一般には関数  が無限回微分可能であってもそのテイラー級数が  で収束するとは限らず、たとえ収束しても一致するとは限らないことに注意が必要である。一致するかどうかは、[テイラーの定理](https://ja.wikipedia.org/wiki/テイラーの定理 "wikilink")における[剰余項](https://ja.wikipedia.org/wiki/剰余項 "wikilink")  が 0 に収束するかどうかによって判定できる；ここで剰余項  は、ある  が存在して、

\[R_n (x) = {f^{(n)}(c) \over n!} (x - a)^n\]

と書ける。または積分を用いて、次のように表せる。

\[R_n (x) =\frac{1}{(n-1)!}\int_a^x (x-t)^{n-1} f^{(n)}(t)\, \mathrm dt\] また、この剰余項を評価することで関数の近似値を精度保証つきで数値的に求めることもできる（[テイラーの定理\#例](https://ja.wikipedia.org/wiki/テイラーの定理#例 "wikilink")を参照）。

特に  0}} における以下の様な展開

\[\sum_{n=0}^{\infin} \frac{f^{(n)}(0)}{n!} x^{n}\] を**マクローリン展開**（マクローリンてんかい、; 名称は数学者[コリン・マクローリン](../Page/コリン・マクローリン.md "wikilink")に由来する）と呼ぶ。

### マクローリン級数の一覧

いくつかの重要な関数のテイラー展開を以下に示す。これらはすべて複素解析的な関数であり、複素変数であると考えても成り立つ。

  - [多項式](../Page/多項式.md "wikilink")
    多項式をマクローリン展開したものは元の多項式自身である。
  - [指数関数](../Page/指数関数.md "wikilink"):
    \(e^{x} = \sum^{\infin}_{n=0} \frac{x^n}{n!}\quad\mbox{ for all }x\)
  - [自然対数](../Page/自然対数.md "wikilink"):
    \(\log(1+x) = \sum^{\infin}_{n=1} \frac{(-1)^{n+1}}n x^n\quad\mbox{ for all } |x| < 1\)
    \(\log(1-x) = - \sum^{\infty}_{n=1} \frac{x^n}n\quad\text{ for } |x| < 1\)
  - [幾何級数](https://ja.wikipedia.org/wiki/幾何級数 "wikilink")
    \(\frac{1}{1-x} = \sum^{\infin}_{n=0} x^n\quad\mbox{for all } |x| < 1\)
    \(\frac{1}{(1-x)^2} = \sum^\infty_{n=1} nx^{n-1}\quad\text{ for }|x| < 1\!\)
    \(\frac{x}{(1-x)^2} = \sum^\infty_{n=0} nx^n\quad\text{ for }|x| < 1\!\)
    \(\frac{2}{(1-x)^3} = \sum^\infty_{n=2} (n-1)nx^{n-2}\quad\text{ for }|x| < 1\!\)
    \(\frac{2x^2}{(1-x)^3} = \sum^\infty_{n=0} (n-1)nx^n\quad\text{ for }|x| < 1\!\)
  - [二項定理](../Page/二項定理.md "wikilink")
    \((1+x)^\alpha = \sum^{\infin}_{n=0} \binom{\alpha}{n} x^n\quad\mbox{for all } |x| < 1 \mbox{ and any complex } \alpha\)
  - [三角関数](../Page/三角関数.md "wikilink")
    \(\sin x = \sum^{\infin}_{n=0} \frac{(-1)^n}{(2n+1)!} x^{2n+1}\quad\mbox{ for all } x\)
    \(\cos x = \sum^{\infin}_{n=0} \frac{(-1)^n}{(2n)!} x^{2n}\quad\mbox{ for all } x\)
    \(\tan x = \sum^{\infin}_{n=1} \frac{B_{2n} (-4)^n (1-4^n)}{(2n)!} x^{2n-1}\quad\mbox{ for } |x| < \frac{\pi}{2}\)
    \(\csc x = \sum^{\infin}_{n=0} \frac{ (-1)^n (2-2^{2n}) B_{2n}}{(2n)!} x^{2n-1}\quad\mbox{ for } 0 < |x| < \pi\)
    \(\sec x = \sum^{\infin}_{n=0} \frac{(-1)^n E_{2n}}{(2n)!} x^{2n}\quad\mbox{ for } |x| < \frac{\pi}{2}\)
    \(\cot x = \sum^{\infin}_{n=0} \frac{ (-1)^n 2^{2n} B_{2n}}{(2n)!} x^{2n-1}\quad\mbox{ for } 0 < |x| < \pi\)
    \(\sin^{-1} x = \sum^{\infin}_{n=0} \frac{(2n)!}{4^n (n!)^2 (2n+1)} x^{2n+1}\quad\mbox{ for } |x| < 1\)
    \(\cos^{-1} x = {\pi\over 2}- \sum^{\infin}_{n=0} \frac{(2n)!}{4^n (n!)^2 (2n+1)} x^{2n+1}\quad\mbox{ for } |x| < 1\)
    \(\tan^{-1} x = \sum^{\infin}_{n=0} \frac{(-1)^n}{2n+1} x^{2n+1}\quad\mbox{ for } |x| < 1\)
  - [双曲線関数](../Page/双曲線関数.md "wikilink")
    \(\sinh x = \sum^{\infin}_{n=0} \frac{1}{(2n+1)!} x^{2n+1}\quad\mbox{ for all } x\)
    \(\cosh x = \sum^{\infin}_{n=0} \frac{1}{(2n)!} x^{2n}\quad\mbox{ for all } x\)
    \(\tanh x = \sum^{\infin}_{n=1} \frac{B_{2n} 4^n (4^n-1)}{(2n)!} x^{2n-1}\quad\mbox{ for } |x| < \frac{\pi}{2}\)
    \(\sinh^{-1} x = \sum^{\infin}_{n=0} \frac{(-1)^n (2n)!}{4^n (n!)^2 (2n+1)} x^{2n+1}\quad\mbox{ for } |x| < 1\)
    \(\tanh^{-1} x = \sum^{\infin}_{n=0} \frac{1}{2n+1} x^{2n+1}\quad\mbox{ for } |x| < 1\)
  - [ランベルトのW関数](../Page/ランベルトのW関数.md "wikilink")
    \(W_0(x) = \sum^{\infin}_{n=1} \frac{(-n)^{n-1}}{n!} x^n\quad\mbox{ for } |x| < \frac{1}{e}\)

の展開に現われる  は[ベルヌーイ数](../Page/ベルヌーイ数.md "wikilink")である。 二項展開の \(\textstyle \binom{\alpha}{n}\) は[二項係数](../Page/二項係数.md "wikilink")である。 の展開に現われる  は[オイラー数](../Page/オイラー数.md "wikilink")である。

## 一変数複素関数のテイラー展開

点  を含む[開集合](../Page/開集合.md "wikilink")  上で微分可能、すなわち[正則な](../Page/正則関数.md "wikilink")[複素関数](../Page/複素解析.md "wikilink")  が与えられたとき、[べき級数](https://ja.wikipedia.org/wiki/べき級数 "wikilink")

\[\sum_{n=0}^{\infin} \frac{f^{(n)}(a)}{n!} (z - a)^{n}\]

を関数  の点  まわりの**テイラー級数**という。[正則関数の解析性](https://ja.wikipedia.org/wiki/正則関数の解析性 "wikilink")から、点  を中心として  に包含されるような任意の[開円板](https://ja.wikipedia.org/wiki/開円板 "wikilink")  { *z* ∈ **C** {{\!}}  \< *r* } ⊆ *D* }} 上でこの級数は  に収束する。

剰余項  は[複素線積分](https://ja.wikipedia.org/wiki/複素線積分 "wikilink")を用いて、次のように表せる；

\[R_n (z) = (z-a)^n \left [ \frac{n!}{2 \pi i}\int_C \frac{f(w)}{(w-a)^{n-1}(w-z)} \mathrm dw \right ]\]

ここで  は、点  を囲み、周および内部が  に含まれるような反時計回りの[円周](../Page/円周.md "wikilink")である。

## 多変数関数のテイラー展開

テイラー展開は一変数関数のみならず、多変数関数にも適用できる。 変数関数  のテイラー展開は以下の式である。

\[f(x_1,\dots,x_d) = \sum_{n_1=0}^\infty \sum_{n_2=0}^\infty \cdots \sum_{n_d = 0}^\infty
\frac{(x_1-a_1)^{n_1}\cdots (x_d-a_d)^{n_d}}{n_1!\cdots n_d!}\,\left(\frac{\partial^{n_1 + \cdots + n_d}f}{\partial x_1^{n_1}\cdots \partial x_d^{n_d}}\right)(a_1,\dots,a_d).\!\]

[多重指数](https://ja.wikipedia.org/wiki/多重指数 "wikilink")記法を用いれば、 変数関数  のテイラー展開は次式で表現される。

\[f(\mathbf{x}) = \sum_{\alpha\in\mathbb{N}^d_0}^{}\frac{(\mathbf{x}-\mathbf{a})^{\alpha}}{\alpha !}\,({\mathrm{\partial}^{\alpha}}\,f)(\mathbf{a})\]

[アインシュタインの縮約記法](../Page/アインシュタインの縮約記法.md "wikilink")を用いれば、多変数関数  のテイラー展開は次式である。

\[f(x^{\mu}) = \sum^{\infin}_{n=0} \frac{1}{n!} \left[ (x^{\mu} - \alpha^{\mu}) \partial_{\mu} \right]^n f(\alpha^{\mu})\] 上式の  は[微分演算子であり](https://ja.wikipedia.org/wiki/微分作用素 "wikilink")、[ベクトル解析](../Page/ベクトル解析.md "wikilink")の記法では  に置き換えられる。一番後ろに  があるが、これは  に左の演算子を作用させてから  の引数として  を与えることを表していることに注意する。
