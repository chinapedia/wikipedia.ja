[J_M_W_Turner-La_bataile_de_Trafalgar.JPG](https://ja.wikipedia.org/wiki/File:J_M_W_Turner-La_bataile_de_Trafalgar.JPG "fig:J_M_W_Turner-La_bataile_de_Trafalgar.JPG")『トラファルガーの戦い』より“[英国は各員がその義務を尽くすことを期待する](https://ja.wikipedia.org/wiki/英国は各員がその義務を尽くすことを期待する "wikilink")”\]\] **国際信号旗**（こくさいしんごうき、international maritime signal flags）は、海上において船舶間での[通信](../Page/通信.md "wikilink")に利用される世界共通の旗である。その使い方は[国際信号書](https://ja.wikipedia.org/wiki/国際信号書 "wikilink")（こくさいしんごうしょ、[International Code of Signals](https://ja.wikipedia.org/wiki/:en:International_Code_of_Signals "wikilink")、INTERCO）によって定められており、国際信号旗による信号を**旗旒信号**（きりゅうしんごう、Flag Signalling）と呼ぶ。

## 概要

[US_Navy_021014-N-6268K-001_Sailor_prepares_signal_flags_for_at_sea_operations.jpg](https://ja.wikipedia.org/wiki/File:US_Navy_021014-N-6268K-001_Sailor_prepares_signal_flags_for_at_sea_operations.jpg "fig:US_Navy_021014-N-6268K-001_Sailor_prepares_signal_flags_for_at_sea_operations.jpg")の乗組員。この艦では3組が用意されている。\]\] 中世ヨーロッパでは、船舶間の通信に旗を用いることが行われてきていた。[18世紀](../Page/18世紀.md "wikilink")にはイギリスの[リチャード・ハウ](../Page/リチャード・ハウ.md "wikilink")が主に数字を意味する複数の信号旗を用い、その数字を符号として単語に置き換え、旗の掲揚を繰り返すことで、文章も含めた通信を行なう方法を考案した。これはホーム・リッグス・ポップハムによって改良され、[トラファルガーの海戦](../Page/トラファルガーの海戦.md "wikilink")でも通信に使用された。これらが発展し、[1857年](../Page/1857年.md "wikilink")に国際信号書として、国際信号旗が定められた。

旗旒信号は、1つの旗が一つの[アルファベット](https://ja.wikipedia.org/wiki/アルファベット "wikilink")、[数字](../Page/数字.md "wikilink")（または[ひらがな](https://ja.wikipedia.org/wiki/ひらがな "wikilink")）に対応している。さらに符字として、1つの旗にある特定の意味をもたせており、例としては、[ダイビング](../Page/ダイビング.md "wikilink")支援船は“A旗”を掲げて、自船が現在潜水作業を実施している旨を他船に知らせる。このほか、[ヨット](../Page/ヨット.md "wikilink")や[ディンギー](../Page/ディンギー.md "wikilink")競走においては特別なコードが用いられる。例としてP旗は「準備せよ」、S旗は「コースの短縮」を意味している。また、[砕氷船](../Page/砕氷船.md "wikilink")と被援助船の通信においても、特別なコードが用いられる。例として砕氷船によるA旗は「前進せよ（氷間水路を進行せよ）」、被援助船によるA旗は「本船は前進し（ようとし）ている、本船は氷間水路を進行し（ようとし）ている」を意味している。

取り出しやすいように通常は甲板上に専用のラックを設けている。

[NATOでは別途](../Page/北大西洋条約機構.md "wikilink")、専用のコードを用意している。

### 1字信号の補足的用法

1字信号には、以下のように数字を伴って使用する補足的な用法がある\[1\]。

| 文字 | NATOフォネティックコードによる発音 | 直後に置かれる数字                                             | 意味                                                |
| -- | ------------------- | ----------------------------------------------------- | ------------------------------------------------- |
| A  | アルファ (*Alfa*)       | 3桁                                                    | 方位角または方位                                          |
| C  | チャーリー (*Charlie*)   | 進路（[針路](https://ja.wikipedia.org/wiki/針路 "wikilink")） |                                                   |
| D  | デルタ (*Delta*)       | 2桁、4桁、6桁                                              | 日付                                                |
| G  | ゴルフ (*Golf*)        | 4桁、5桁                                                 | 経度（最後の2けたは分、前の2けたまたは3けたは度を表す）                     |
| L  | リマ (*Lima*)         | 4桁                                                    | 緯度（最後の2けたは分、前の2けたは度を表す）                           |
| R  | ロメオ (*Romeo*)       | 1桁またはそれ以上                                             | 距離（単位：[海里](../Page/海里.md "wikilink")）             |
| S  | シエラ (*Sierra*)      | 速度（単位：[ノット](../Page/ノット.md "wikilink")）               |                                                   |
| T  | タンゴ (*Tango*)       | 4桁                                                    | 地方時 (Local Time)                                  |
| V  | ヴィクター (*Victor*)    | 1桁またはそれ以上                                             | 速度（単位：[キロメートル毎時](../Page/キロメートル毎時.md "wikilink")） |
| Z  | ズールー (*Zulu*)       | 4桁                                                    | [協定世界時](../Page/協定世界時.md "wikilink")              |

ただし、発音のイントネーションは通常の英単語でのものとは大きく異なる。

### 2字信号

  - “C”“B”の順に並べて掲げると「至急救援を求める」の意になる\[2\]。
  - “M”“J”の順に並べて掲げると「貴船の医師の派遣を求む」の意になる\[3\]。
  - “N”“C”の順に並べて掲げると「本船は遭難した。至急救援を求める」（[遭難信号](https://ja.wikipedia.org/wiki/遭難信号 "wikilink")）の意になる\[4\]。
  - “U”“W”の順に並べて掲げると「貴船の安全なる航海（御安航）を祈る」の意になる（民間船から[敬礼](../Page/敬礼.md "wikilink")を受けた[軍艦](../Page/軍艦.md "wikilink")が答礼と共に掲げる。また盛大に出航式を行なった船に向けて、港が掲げる）\[5\]。
  - “S”“N”の順に並べて掲げると即時停船要求。航行を続けたり「R」旗を掲げないなど、無視した場合は攻撃を受ける\[6\]。

### 3字信号

  - 信号「UW」に“1”を追加して掲げると「協力に感謝する。御安航を (Thank your cooperation, bon voyage)」\[7\]の意に。“2”を追加して掲げると「ようこそ (Welcome)」、"3"を追加して掲げると「お帰りなさい (Welcome home)」となるが、この2つは1990年前後の改訂で追加されている。
  - 救難信号「CB」に“4”を追加して掲げると原因が座礁\[8\]、“6”を追加して掲げると原因が火災の意になる\[9\]。
  - Mから始まる英字の3字信号は医療支援に使われる。
      - “M”“A”“A”の順に並べて掲げると「本船は緊急の医学的な助言を求める」の意になる\[10\]。
      - “M”“C”“C”の順に並べて掲げると「患者は脈拍微弱、測定不能」の意\[11\]、最後の“C”が“X”になると「意識混濁」の意になる\[12\]。
      - “M”“P”“F”の順に並べて掲げると「患者は改善している」の意\[13\]、“F”ではなく“R”になると「患者は死んだ」の意になる\[14\]。
