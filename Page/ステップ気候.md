[ステップ気候の分布図  ](https://ja.wikipedia.org/wiki/File:Koppen_World_Map_BSh_BSk.png "fig:ステップ気候の分布図  ") [The_semi-arid_terrain_of_Machakos_(5112045372).jpg](https://ja.wikipedia.org/wiki/File:The_semi-arid_terrain_of_Machakos_\(5112045372\).jpg "fig:The_semi-arid_terrain_of_Machakos_(5112045372).jpg")）\]\] [Brazilian_semi-arid_region.jpg](https://ja.wikipedia.org/wiki/File:Brazilian_semi-arid_region.jpg "fig:Brazilian_semi-arid_region.jpg")）\]\] **ステップ気候**（すてっぷきこう）とは[ケッペンの気候区分](https://ja.wikipedia.org/wiki/ケッペンの気候区分 "wikilink")における気候区のひとつで、[乾燥帯](../Page/乾燥帯.md "wikilink")に属する。記号は**BS**で、BShとBSkに分かれる。BSのSはドイツ語のSteppe（[ステップ](https://ja.wikipedia.org/wiki/ステップ_\(植生\) "wikilink")）に由来する。

[アリソフの気候区分](https://ja.wikipedia.org/wiki/アリソフの気候区分 "wikilink")では気候帯4-1.亜熱帯大陸性気候に相当する\[1\]。

## 特徴

BSk気候の地域では，黒や栗色の土壌（[チェルノーゼム](https://ja.wikipedia.org/wiki/チェルノーゼム "wikilink")）に、[ステップとよばれる丈の短い草原が広がる](https://ja.wikipedia.org/wiki/ステップ_\(植生\) "wikilink")。年間を通して[降水量](https://ja.wikipedia.org/wiki/降水量 "wikilink")は少なく[雨季](https://ja.wikipedia.org/wiki/雨季 "wikilink")には少量の雨が降る。昼と夜の気温差が激しい。

## 条件

最暖月の平均気温が10℃以上であり（[寒帯](../Page/寒帯.md "wikilink")では無い）、年平均降水量が[乾燥限界](https://ja.wikipedia.org/wiki/ケッペンの気候区分#乾燥限界_r_（mm/樹木生育に必要な最低降水量）の計算 "wikilink") \(r\)には満たないが\(r\)の半分以上である地域であり、

以下の式から求められる。条件に当てはまればステップ気候となる。

年間平均気温を\(t\)（℃）とし、年平均降水量（mm）が\(0.5r=10(t+x)\)以上かつ\(r=20(t+x)\)未満を満たす。

\(x\)は以下の降水パターンの条件によって決まる。

|          |               |                                             |
| -------- | ------------- | ------------------------------------------- |
| \(x=14\) | w（冬季乾燥/夏雨）    | 最多雨月が夏にあり、10×最少雨月降水量＜最多雨月降水量                |
| \(x=0\)  | s（夏季乾燥/冬雨）    | 最多雨月が冬にあり、3×最少雨月降水量＜最多雨月降水量かつ最少雨月降水量が30mm未満 |
| \(x=7\)  | f（年中湿潤/年平均降雨） | wでもsでもない                                    |

上記の条件を満たすとき年平均気温が18℃以上なら**BSh**、18℃未満なら**BSk**となる。

## 気候

一年中あるいは乾季のみの間[中緯度高圧帯](https://ja.wikipedia.org/wiki/中緯度高圧帯 "wikilink")の影響下に入るため、日本で言う[秋晴れ](https://ja.wikipedia.org/wiki/秋晴れ "wikilink")のような乾燥した晴天が続く。雨季にはある程度の雨が降るが、年間平均降水量は250 - 750mmとやや少なめである。

[植物](../Page/植物.md "wikilink")の自生は可能であるが[森林](../Page/森林.md "wikilink")ではなく、[草原](https://ja.wikipedia.org/wiki/草原 "wikilink")になる。また、特に[サヘル](https://ja.wikipedia.org/wiki/サヘル "wikilink")や[アラル海](https://ja.wikipedia.org/wiki/アラル海 "wikilink")周辺などでは[過放牧](https://ja.wikipedia.org/wiki/過放牧 "wikilink")や過耕作、水源の枯渇などにより[砂漠化](../Page/砂漠化.md "wikilink")（土地の不毛化）が進んでいる。

## 土壌と植生

[黒色土](https://ja.wikipedia.org/wiki/黒色土 "wikilink")（[チェルノーゼム](https://ja.wikipedia.org/wiki/チェルノーゼム "wikilink")、[プレーリー土](https://ja.wikipedia.org/wiki/プレーリー土 "wikilink")）、[栗色土](https://ja.wikipedia.org/wiki/栗色土 "wikilink")などの肥沃な土壌が分布している。そのため、[灌漑](https://ja.wikipedia.org/wiki/灌漑 "wikilink")によって耕地化されている地域もある。

## 産業・その他

農業が主産業となっている地域が多い。企業による計画的な大規模農業もあるが、個人や集落単位での農業も盛んである。
