**圧力**（あつりょく、）とは、

  - [物体](../Page/物体.md "wikilink")の[表面](https://ja.wikipedia.org/wiki/表面 "wikilink")あるいは内部の任意の[面](https://ja.wikipedia.org/wiki/面 "wikilink")に向かい[垂直](../Page/垂直.md "wikilink")に押す[力のこと](../Page/力_\(物理学\).md "wikilink")\[1\]。
  - （比喩）人を威圧して従わせようとする力のこと\[2\]。

## 概説

圧力の[大きさ](https://ja.wikipedia.org/wiki/大きさ "wikilink")を表す場合は、[単位](../Page/単位.md "wikilink")[面積](../Page/面積.md "wikilink")あたりに働く力で表わし\[3\]、[国際単位系](https://ja.wikipedia.org/wiki/国際単位系 "wikilink")における圧力の[単位](../Page/単位.md "wikilink")は、[パスカルである](../Page/パスカル_\(単位\).md "wikilink")。

*P* = *F* / *S*、すなわち 1 Pa = 1 N/m である。

また歴史的には、[水銀柱](https://ja.wikipedia.org/wiki/水銀柱 "wikilink")の（水銀面の）高さ、[バール](../Page/バール.md "wikilink")などでも表している。

[Pressure_exerted_by_collisions.svg](https://ja.wikipedia.org/wiki/File:Pressure_exerted_by_collisions.svg "fig:Pressure_exerted_by_collisions.svg")的、[分子運動論](https://ja.wikipedia.org/wiki/分子運動論 "wikilink")的な[モデルで圧力の発生のしくみを表現した図](../Page/モデル_\(自然科学\).md "wikilink")。容器内に気体や液体がある場合に、その多数の分子が高速で容器内を運動、壁面に衝突し跳ね返り、分子の運動方向が変わった分壁面に対して垂直な、押しこむ方向の[力が生じる](../Page/力_\(物理学\).md "wikilink")、ということが非常に多数起きている、と考える。\]\] 静止している流体内の1点で働く圧力は方向によらず一定であり、これを「[静水圧](https://ja.wikipedia.org/wiki/静水圧 "wikilink")」と言う\[4\]。

圧力には、絶対[真空](../Page/真空.md "wikilink")（絶対零圧力）を基準（ゼロ）とする[絶対圧](https://ja.wikipedia.org/wiki/絶対圧 "wikilink")と、[大気圧を基準](../Page/気圧.md "wikilink")（ゼロ）とする[ゲージ圧](https://ja.wikipedia.org/wiki/ゲージ圧 "wikilink")（相対圧）とがある。

## 圧力計

[WPGaugeFace.jpg](https://ja.wikipedia.org/wiki/File:WPGaugeFace.jpg "fig:WPGaugeFace.jpg") 圧力を測定する装置を「圧力計」（英:[pressure meter](https://ja.wikipedia.org/wiki/:en:pressure_meter "wikilink")、pressure gauge）といい、次のようなものが代表的である。また、圧力計は[液位計](https://ja.wikipedia.org/wiki/液位計 "wikilink")としても利用される。

  - ブルドン管圧力計
    [ブルドン管](https://ja.wikipedia.org/wiki/ブルドン管 "wikilink")の[弾性変形](https://ja.wikipedia.org/wiki/弾性変形 "wikilink")を用いて測定する。
  - ダイヤフラム圧力計
    流体とブルドン管圧力計との間に[ダイアフラム](https://ja.wikipedia.org/wiki/ダイアフラム "wikilink")を設け、受圧部とブルドン 管の間に封入液を充満して圧力の伝達媒体としたもの。

## 圧力に関する現象

圧力をかけたり変化させたりすることで、以下のような性質を示す物質がある。

  - [沸点](../Page/沸点.md "wikilink")、[融点](../Page/融点.md "wikilink")の変化
    → [相図](../Page/相図.md "wikilink")
  - [結晶構造](../Page/結晶構造.md "wikilink")の変化
    圧力によって物質の構造が変化する物質がある。→ [構造相転移](https://ja.wikipedia.org/wiki/構造相転移 "wikilink")
  - [電荷](../Page/電荷.md "wikilink")の発生、[電気抵抗](../Page/電気抵抗.md "wikilink")の変化
    圧力を加えることで物体の表面に電荷が生じたり（[圧電効果](../Page/圧電効果.md "wikilink")）、電気抵抗が変化したりする。[圧電素子はガスレンジの着火装置などに利用される](https://ja.wikipedia.org/wiki/ピエゾ素子 "wikilink")。
  - [磁化](../Page/磁化.md "wikilink")の変化
    圧力によって磁化の強さが変化する。→ [ビラリ現象](https://ja.wikipedia.org/wiki/ビラリ現象 "wikilink")

## 分子運動論での圧力の求め方

[ビリアルの定理](https://ja.wikipedia.org/wiki/ビリアルの定理 "wikilink")から求める。圧力*P*は、

  -
    \(P = \left\langle {1 \over 3V} \left(\sum_i { {P_i}^2 \over m_i } - \sum_i r_i { {\partial \phi } \over {\partial \vec{r}_i } } \right) \right\rangle\)

で求まる。上式で、\(< >\)は統計的な平均、右辺()内第一項は、粒子の全運動エネルギーの和、第二項は粒子間に働く力(= \(- { {\partial \phi } \over {\partial \vec{r}_i } }\))と粒子の座標との積の和である。*i*は粒子の指標。'' m<sub>i</sub> *は粒子の質量。\(\vec{r}_i\)は粒子の座標の[位置ベクトル](https://ja.wikipedia.org/wiki/位置ベクトル "wikilink")。\(\phi \,\)は適当なポテンシャル。* V ''は系の体積。

## バンド計算での圧力の求め方

熱力学の知識から\[5\]、平衡状態であれば、系の全エネルギーの体積微分が、圧力（静水圧）である。歪に対する微分がストレスになる。圧力*P*の表式は（*V*を系の体積として）、

  -
    \(P = - { {\partial E_{\rm tot} } \over {\partial V} }\)
    \(E_{\rm tot} \,\) ： 系の[全エネルギー](https://ja.wikipedia.org/wiki/全エネルギー "wikilink")

となる。[バンド計算](../Page/バンド計算.md "wikilink")でのより具体的な圧力（ストレス）の求め方は、ニールセンとマーティンによる論文\[6\]\[7\]が参考になる。

## 超高圧の実験

超高圧実験装置にはプレス型と[ダイヤモンドアンビルセル](https://ja.wikipedia.org/wiki/ダイヤモンドアンビルセル "wikilink")を使ったものとに二分できる。

プレス型は、ピストンシリンダーなどを使って生じた圧力を油圧（直接加圧する場合もあり）で伝達して試料を押す。発生可能な圧力の大まかな目安は数万気圧（数GPa）である。比較的広い圧力発生空間を確保することができ、多彩な物性測定実験が可能となっている。

ダイヤモンドアンビルセル(Diamond Anvil Cell: DAC)は、天然または人工合成の[ダイヤモンド](../Page/ダイヤモンド.md "wikilink")を使って超高圧を実現するもので、小型（手のひらサイズ）で、透明（光学的な観測が可能）であり、サブテラパスカル（数百万気圧、数百GPa）までの加圧が可能である。一方、ダイヤモンドそのものが大型化できないので、試料は大変小さなものにしなければならない。ダイヤモンド以外に、[サファイア](../Page/サファイア.md "wikilink")、[炭化ケイ素](../Page/炭化ケイ素.md "wikilink")を使ったアンビルセルもあるが、加圧できる圧力はダイヤモンドよりも劣る。

上記の高圧実験は通常、静水圧が前提であるが、動的に圧力を加える実験として衝撃圧縮実験がある。

他に非等方的な圧力実験の試みもある。

## 比喩

「圧力」は、比喩的に、人を威圧して従わせようとする力も指すことがある。
