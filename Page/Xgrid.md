**Xgrid**（エックスグリッド）とは[アップルがかつて開発していた](../Page/アップル_\(企業\).md "wikilink")[Ｍac OS X Server](https://ja.wikipedia.org/wiki/macOS_Server "wikilink") / [Mac OS X用の](https://ja.wikipedia.org/wiki/macOS "wikilink")[分散コンピューティング](../Page/分散コンピューティング.md "wikilink") / [コンピュータ・クラスター](../Page/コンピュータ・クラスター.md "wikilink")環境。

ローカルネットワーク及び[インターネット](https://ja.wikipedia.org/wiki/インターネット "wikilink")を介して複数台のMacに計算作業を分担させる事が可能。[Bonjour](https://ja.wikipedia.org/wiki/Bonjour "wikilink")テクノロジーを用いて、ネットワーク上で主コントローラのマシンがエージェントとして使えるマシンを自動検知することができる。

Mac OS X v10.3 以降で動作\[1\]し、Mac OS X Server / Mac OS X v10.4から標準機能として搭載されて、[OS X Mountain Lion](https://ja.wikipedia.org/wiki/OS_X_Mountain_Lion "wikilink")/[OS X Server 2.0で廃止された](https://ja.wikipedia.org/wiki/MacOS_Server "wikilink")\[2\]。

[アメリカ合衆国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")内では、安価かつ容易に[グリッド・コンピューティング](../Page/グリッド・コンピューティング.md "wikilink")を実現できるシステムとして主に大学の研究機関の間では一定の人気があった。
