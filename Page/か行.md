**か行**（かぎょう）とは、[日本語](../Page/日本語.md "wikilink")の[五十音](../Page/五十音.md "wikilink")の第2番目の行である。[仮名の](../Page/仮名_\(文字\).md "wikilink")「[か](../Page/か.md "wikilink")」「[き](../Page/き.md "wikilink")」「[く](../Page/く.md "wikilink")」「[け](../Page/け.md "wikilink")」「[こ](../Page/こ.md "wikilink")」が含まれる。どの仮名も[子音](../Page/子音.md "wikilink")と[母音](../Page/母音.md "wikilink")からなる音（[モーラ](../Page/モーラ.md "wikilink")）を表す。

## 発音

### 清音

[清音](../Page/清音.md "wikilink")か行の頭子音の[音素](../Page/音素.md "wikilink")は  である。音声学的には、か行各音の頭子音は[無声軟口蓋破裂音](../Page/無声軟口蓋破裂音.md "wikilink")  で、舌の後部を[口蓋](../Page/口蓋.md "wikilink")の奥の部分（軟口蓋）に押しあて一旦閉鎖した上で破裂させることで発する無声子音である。ただし、「き」の頭子音は[硬口蓋化](https://ja.wikipedia.org/wiki/硬口蓋化 "wikilink")するため、調音点が他のか行音より前方推移し、後部硬口蓋になる。[ローマ字](https://ja.wikipedia.org/wiki/ローマ字 "wikilink")では日本式・ヘボン式ともに ka ki ku ke ko と表記される。

### 濁音

か行の仮名に濁点をつけて表記される[濁音](../Page/濁音.md "wikilink")が行（が・ぎ・ぐ・げ・ご）の頭子音の音素は  である。日常的に[鼻濁音](../Page/鼻濁音.md "wikilink")を使う話者の場合は、この頭子音は文節の頭では清音のそれを有声化した有声軟口蓋破裂音 、それ以外では[軟口蓋鼻音](https://ja.wikipedia.org/wiki/軟口蓋鼻音 "wikilink")  に、日常的に鼻濁音を使わない話者の場合は、この頭子音は文節の頭および撥音「ん」の後では、[有声軟口蓋破裂音](../Page/有声軟口蓋破裂音.md "wikilink") 、撥音の後を除く文節中・文節尾では[有声軟口蓋摩擦音](https://ja.wikipedia.org/wiki/有声軟口蓋摩擦音 "wikilink")  またはそれに近い音（閉鎖密着の度合いが弱い有声軟口蓋破裂音）に発音される。ただし、いずれの場合も「ぎ」の頭子音は「き」の場合と同様に硬口蓋化して調音点が前方推移する。ローマ字では日本式・ヘボン式ともに ga gi gu ge go と表記される。

日常的に鼻濁音を使う地域は共通語の基盤となった[東京方言](../Page/東京方言.md "wikilink")が話される地域を中心に東日本から以北に広がっている。一方、中国、四国以西の地域ではほとんど鼻濁音は使われない。

### 拗音

  - か行とが行のい段音を第一文字とする開[拗音](../Page/拗音.md "wikilink") 「きゃ」「きゅ」「きょ」と「ぎゃ」「ぎゅ」「ぎょ」 の頭子音の音素はそれぞれ  と  である。その発音は「き」および「ぎ」の頭子音とまったく同様である。ローマ字では日本式・ヘボン式ともに kya kyu kyo および gya gyu gyo と表記される。
  - 伝統的に使われた合拗音「くゎ」「ぐゎ」は、現在はほとんど使われない。これらは、現在ではほとんどの地域で「か」「が」と発音され、[現代仮名遣い](../Page/現代仮名遣い.md "wikilink")では「か」「が」と書かれる。ローマ字では日本式・ヘボン式ともに ka ga である。

### 外来語の表記

  - 「クァ」（クヮ）「クィ」「クェ」「クォ」は『[外来語の表記](https://ja.wikipedia.org/wiki/外来語の表記 "wikilink")』第2表に挙げられており、[外来語](../Page/外来語.md "wikilink")の原音または原綴りになるべく近づけて書き表そうとする場合に使われる仮名である。日本語では「クア」「クイ」「クエ」「クオ」や「クワ」「クウィ」「クウェ」「クウォ」または「カ」「キ」「ケ」「コ」と発音され、またそのように表記される場合もある。クァルテット（カルテット）・クェスチョン（クエスチョン）など。
  - 「グァ」（グヮ）は『外来語の表記』第2表に挙げられており、外来語の原音または原つづりになるべく近づけて書き表そうとする場合に使われる仮名である。日本語では「グア」または「ガ」と発音され、またそのように表記される場合もある。パラグァイ（パラグアイ）・グァテマラ（グアテマラ・ガテマラ）など。

  - 「キェ」は『外来語の表記』の表中では挙げられていないが、たまに使われることがある。例えば[キェルケゴール](https://ja.wikipedia.org/wiki/キェルケゴール "wikilink")（キルケゴール）など

## か行の仮名の由来

[平仮名](../Page/平仮名.md "wikilink")、[片仮名](../Page/片仮名.md "wikilink")のか行はそれぞれ次の漢字に由来している。

| 平仮名 | 漢字 |   | 片仮名        | 漢字 |
| --- | -- | - | ---------- | -- |
| か   | 加  | カ | 加の偏から      |    |
| き   | 幾  | キ | 幾の上部の転じたもの |    |
| く   | 久  | ク | 久の最初の二画    |    |
| け   | 計  | ケ | 介を略したもの    |    |
| こ   | 己  | コ | 己の最初の二画    |    |

[変体仮名](../Page/変体仮名.md "wikilink")として以下に示す漢字に由来するものも用いられた。

  - か - 可、閑、家、我
  - き - 支、起、貴、喜
  - く - 具、倶、九、求
  - け - 介、希、遣、気
  - こ - 古、故、許、期

## 上代特殊仮名遣

[万葉仮名](https://ja.wikipedia.org/wiki/万葉仮名 "wikilink")では「き」「け」「こ」とそれらの濁音には甲乙二種の使い分けが見られる。それぞれ次の二群に分けられる（[上代特殊仮名遣い](https://ja.wikipedia.org/wiki/上代特殊仮名遣い "wikilink")）。

|   | 甲類       | 乙類       |
| - | -------- | -------- |
| き | 伎岐吉棄支企など | 紀幾貴奇寄綺など |
| ぎ | 伎岐陣藝蟻など  | 疑宜義擬     |
| け | 家計鶏介結啓など | 気既該戒階居など |
| げ | 下牙雅夏     | 気宜など     |
| こ | 古故胡孤姑高など | 許去虚興己居など |
| ご | 吾呉胡後悟虞など | 其期碁語御馭   |
