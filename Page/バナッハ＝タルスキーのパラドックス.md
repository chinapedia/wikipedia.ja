[Doubling_of_a_sphere,_as_per_the_Banach-Tarski_Theorem.png](https://ja.wikipedia.org/wiki/File:Doubling_of_a_sphere,_as_per_the_Banach-Tarski_Theorem.png "fig:Doubling_of_a_sphere,_as_per_the_Banach-Tarski_Theorem.png") **バナッハ＝タルスキーのパラドックス** (Banach-Tarski paradox) は、[球](../Page/球.md "wikilink")を3次元[空間](../Page/空間.md "wikilink")内で、[有限](https://ja.wikipedia.org/wiki/有限 "wikilink")個の部分に分割し、それらを[回転](https://ja.wikipedia.org/wiki/回転 "wikilink")・[平行移動](https://ja.wikipedia.org/wiki/平行移動 "wikilink")操作のみを使ってうまく組み替えることで、元の球と同じ半径の球を2つ作ることができるという[定理](../Page/定理.md "wikilink")（ただし、各断片は通常の意味で体積を定義できない）。この操作を行うために球を最低5つに分割する必要がある。

バナッハ＝タルスキーの証明では、[ハウスドルフのパラドックス](https://ja.wikipedia.org/wiki/ハウスドルフのパラドックス "wikilink")が援用され、その後、多くの人により証明の最適化、様々な空間への拡張が行われた。

結果が直観に反することから、定理であるが「パラドックス」と呼ばれる。[証明](../Page/証明.md "wikilink")の1箇所で[選択公理](../Page/選択公理.md "wikilink")を使うため、選択公理の不合理性を論じる文脈で引用されることがある。[ステファン・バナフ](https://ja.wikipedia.org/wiki/ステファン・バナフ "wikilink")（バナッハ）と[アルフレト・タルスキ](https://ja.wikipedia.org/wiki/アルフレト・タルスキ "wikilink")が1924年に初めてこの定理を述べたときに選択公理を肯定的にとらえていたか、否定的にとらえていたか、判断することは難しい（「この研究に対する選択公理の果たす役割は注目に値する。」(Le rôle que joue cet axiome dans nos raisonnements nous semble mériter l'attention.)としか述べていない）。なお、選択公理よりも真に弱い[ハーン–バナッハの定理](https://ja.wikipedia.org/wiki/ハーン–バナッハの定理 "wikilink")からバナッハ＝タルスキーのパラドックスを導くことができる。\[1\]

この定理は次のように述べることも出来る。

  - 球は、それ自身と同じ球二つと分割合同である。

ただし、分割合同とは以下のように定義される: *A* と *B* を[ユークリッド空間](../Page/ユークリッド空間.md "wikilink")の[部分集合](../Page/部分集合.md "wikilink")とする。 *A* と *B* が有限個の互いに交わらない部分集合の[合併として](https://ja.wikipedia.org/wiki/合併_\(集合論\) "wikilink")

\[A=\bigcup_{i=1}^n A_i,\quad B=\bigcup_{i=1}^n B_i\] つまり、

  -
    *A* = *A*<sub>1</sub> ∪ ... ∪ A<sub>*n*</sub> , *B* = *B*<sub>1</sub> ∪ ... ∪ B<sub>*n*</sub>

と表すことができ、全ての *i* について、\(A_i\) と \(B_i\) が合同であるとき、*A* と *B* を分割合同という。

さらに、この定理から次のより強い形の系を導くことが出来る。

  - 3次元ユークリッド空間の[有界](../Page/有界.md "wikilink")な部分集合で、[内部が](https://ja.wikipedia.org/wiki/内部_\(位相空間論\) "wikilink")[空でないもの](../Page/空集合.md "wikilink")(つまり、有限の拡がりを持ち、[曲線](https://ja.wikipedia.org/wiki/曲線 "wikilink")や[曲面](https://ja.wikipedia.org/wiki/曲面 "wikilink")ではないもの)を任意に二つ選んだとすると、それらは分割合同である。

言い換えると、ビー玉を有限個に分割して組み替えることで月を作ったり、電話を組み替えて睡蓮を作ったり出来る（当然のごとく材質は変えられない）、ということである。 この定理の証明で、点集合は[選択公理](../Page/選択公理.md "wikilink")を使ってつくられる選択集合で構成されており、各断片は[ルベーグ可測](https://ja.wikipedia.org/wiki/ルベーグ可測 "wikilink")では**ない**。すなわち、各断片は明確な[境界](../Page/境界.md "wikilink")や通常の意味での[体積](https://ja.wikipedia.org/wiki/体積 "wikilink")を持たない。物理的な分割では可測な集合しか作れないので、現実にはこのような分割は不可能である。 しかしながら、それらの**幾何学的な形状**に対してはこのような変換が可能なのである。

この定理は 3次元以上の全ての次元においても成り立つ。2次元[ユークリッド平面](https://ja.wikipedia.org/wiki/ユークリッド平面 "wikilink")においては成り立たないものの、2次元においても分割に関するパラドックスは存在する: [円を有限個の部分に分割して組替える事で](../Page/円_\(数学\).md "wikilink")、同じ面積の[正方形](../Page/正方形.md "wikilink")を作ることが出来るのである。これはタルスキーの円積問題([:en:Tarski's circle-squaring problem](https://ja.wikipedia.org/wiki/:en:Tarski's_circle-squaring_problem "wikilink"))として知られている。

2次元[ユークリッド平面](https://ja.wikipedia.org/wiki/ユークリッド平面 "wikilink")においては、合同変換ではなく面積を保つ変換に条件をゆるめると、バナッハ＝タルスキーのパラドックスと同様な定理が成立することを、[1929年](../Page/1929年.md "wikilink")に[ジョン・フォン・ノイマン](../Page/ジョン・フォン・ノイマン.md "wikilink")が証明した。この定理は次のように述べることが出来る。

> *A* と *B* を2次元ユークリッド空間の内点を持つ有界な部分集合とする。 *A* と *B* が有限個の互いに交わらない部分集合の合併として
>
> \[A=\bigcup_{i=1}^n A_i,\quad B=\bigcup_{i=1}^n B_i\] と表すことが出来る。ここで、全ての *i* について、面積を保つ変換 \(f_i\) が存在して
>
> \[B_i = f_i ( A_i )\] とする事が出来る。

## 証明の概要

定理の証明を与える。ここでの方法はバナッハとタルスキーによるものと似ているが全く同一ではない。証明は本質的に4つのステップに分かれる。

1.  2つの生成元を持つ[自由群](../Page/自由群.md "wikilink")\(F_2\)の「パラドキシカルな分割」を見つける。
2.  自由群\(F_2\)と同型な3次元の回転群を見つける。
3.  2で作った回転群のパラドキシカルな分割と選択公理を用いて2次元球面の分割を作る。
4.  3の2次元球面の分割を3次元球の分割に拡張する。

それぞれのステップの詳細について述べる。

### ステップ1

2つの生成元*a*と*b*から生成される自由群は4つの文字*a*、*a*<sup>−1</sup>、*b*、*b*<sup>−1</sup>からなる有限の長さを持つ文字列から構成される。ここで*a*が*a*<sup>−1</sup>の直後に現れるような文字列は許されない。*b*についても同様である。2つのこのような文字列があったとき、それらの積をそれらの文字列をつなげたものと定義する。ただしそれにより「許されない文字列」が生じたときは、その部分を「空の文字列」で置き換えることで対処する。例えば*abab*<sup>−1</sup>*a*<sup>−1</sup>と*abab*<sup>−1</sup>*a*の積は*abab*<sup>−1</sup>*a*<sup>−1</sup>*abab*<sup>−1</sup>*a*となるが、これは*a*<sup>−1</sup>*a*という「許されない文字列」を含むため、この部分を「空の文字列」で置き換えて*abaab*<sup>−1</sup>*a*となる。このような文字列の集合はここで定義した演算によって、「空の文字列」を単位元*e*に持つ群になることが確かめられる。この群を*F*<sub>2</sub>と書く。

[Paradoxical_decomposition_F2.svg](https://ja.wikipedia.org/wiki/File:Paradoxical_decomposition_F2.svg "fig:Paradoxical_decomposition_F2.svg")における*S*(*a*<sup>−1</sup>) と*aS*(*a*<sup>−1</sup>)の集合\]\]

群\(F_2\)は以下のようにして「パラドキシカルな分割」が可能である: *S*(*a*)を*a*で始まる\(F_2\)の文字列全体の集合とする。*S*(*a*<sup>−1</sup>)、*S*(*b*)、*S*(*b*<sup>−1</sup>)についても同様である。明らかに、

\[F_2=\{e\}\cup S(a)\cup S(a^{-1})\cup S(b)\cup S(b^{-1})\] 一方

\[F_2=aS(a^{-1})\cup S(a), \,\] および

\[F_2=bS(b^{-1})\cup S(b). \,\] である。 *aS*(*a*<sup>−1</sup>)という表記は、*S*(*a*<sup>−1</sup>)の元の左に*a*をかけた文字列の全体である。

最後の行がこの証明の核心である。例えば集合\(aS(a^{-1})\)は\(aa^{-1}b\)という文字列を含む。\(a\)は\(a^{-1}\)の直後に現れてはいけないというルールにより、この文字列は\(b\)となる。このように\(aS(a^{-1})\)は\(b\)で始まるすべての文字列を含む。同様に\(bS(b^{-1})\)は\(a\), \(a^{-1}\), \(b^{-1}\)で始まるすべての文字列も含む。

### ステップ2

3次元空間の回転群でちょうど\(F_2\)と同じように振る舞う(\(F_2\)と同型な)群を見つけるために、直交する2つの軸、*x*および*z*をとる。そして*a*で*x*軸を回転軸とした1[ラジアン](../Page/ラジアン.md "wikilink")の回転、*b*で*z*軸を回転軸とした1ラジアンの回転を表すとする(回転の角度は1ラジアンでなくても、円周率πの無理数倍であれば何でもよい)。2つの回転*a*、*b*が操作の合成を積として\(F_2\)と同型になることの証明はやや煩雑だが難しくはないので、この部分は省略する。*a*と*b*によって生成される回転群を**H**とする。すると、ステップ１で得たパラドキシカルな分割を**H**に対しても適用することが出来る。

### ステップ3

[単位球面](https://ja.wikipedia.org/wiki/単位球面 "wikilink")*S*<sup>2</sup>は群**H**の作用を考えることにより軌道の集合に分けることが出来る。すなわち、*S*<sup>2</sup>の2つの点は、一方の点を他方に移すような回転が**H**に存在するとき、またそのときに限り同じ軌道に属すると定めるのである(ある点の軌道が*S*<sup>2</sup>の[稠密集合](https://ja.wikipedia.org/wiki/稠密集合 "wikilink")になることに注意)。[選択公理](../Page/選択公理.md "wikilink")を用いて、すべての軌道からちょうど1個の点を選んで新たな集合を作ることが出来る。この集合を*M*とする。今*S*<sup>2</sup>のすべての点は、ある*M*の点に、ある**H**の元を作用させることによって得ることが出来る。したがって、**H**のパラドキシカルな分割は以下のように*S*<sup>2</sup>の4つの部分集合*A<sub>1</sub>*, *A<sub>2</sub>*, *A<sub>3</sub>*, *A<sub>4</sub>*への分割を与える。

\[A_1=S(a)M \cup M \cup B\]

\[A_2=S(a^{-1})M \setminus B\]

\[\displaystyle A_3=S(b)M\]

\[\displaystyle A_4=S(b^{-1})M\] ここで

\[B = \bigcup_{i=1}^{\infty} a^{-i} M\] である。

今、球面は4つの部分集合に分割されている。以下のように、これらのうち2つの集合を回転させることで最初の2倍の球面を得ることが出来る。

\[aA_2 = A_2 \cup A_3 \cup A_4\]

\[bA_4 = A_1 \cup A_2 \cup A_4\] したがって

\[A_1 \cup aA_2 = S^{2}\] および

\[A_3 \cup bA_4 = S^{2}\]

### ステップ4

最後に、*S*<sup>2</sup>上のすべての点と原点とを結ぶ線分を考えると、ステップ3で考えた*S*<sup>2</sup>の分割は自然に球から中心点を除いた集合の分割へと拡張される。(この中心点はもう少し注意して扱う必要がある。同様に、この概要では省略したが、*S*<sup>2</sup> の点の内 **H** に含まれる何らかの回転の軸上にあるものも特殊な扱いをする必要がある。)
