**ウィノナ州立大学**（Winona State University）は[アメリカ合衆国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")[ミネソタ州](https://ja.wikipedia.org/wiki/ミネソタ州 "wikilink")・[ミネソタ州立大学機構](https://ja.wikipedia.org/wiki/ミネソタ州立大学機構 "wikilink")（Minnesota State College and University）を構成する最古の大学（州立大学機構が設立される前からあった）で、主な4年制大学である。

ミネソタ州立大学機構の1つだが独特のカラーがあり、私立大学のようなイメージがある。これは州立機構が発足する前から学校があった為だという。メインキャンパスはミネソタ州南東部の[ウィノナ](../Page/ウィノナ.md "wikilink")にあり、[ロチェスターのユニバーシティ](../Page/ロチェスター_\(ミネソタ州\).md "wikilink")・センター・ロチェスター（UCR）\[1\]にも分校を置いている。UCR内には、他にミネソタ大学ロチェスター校や、ロチェスターの地元[コミュニティ・カレッジ](https://ja.wikipedia.org/wiki/コミュニティ・カレッジ "wikilink")であるロチェスター・コミュニティ・アンド・テクニカル・カレッジも置かれている。

## 歴史

教員養成大学として[1858年](../Page/1858年.md "wikilink")に設立された、今日ここは多くの分野の学科を提供している。教育に関連するいくつかの修士課程を持っているが、文系の一学部を除いて博士課程は提供していない。他の州立大学と比べると独特の雰囲気があり、州立大学というよりもミネソタ大学や私立大学に近い雰囲気がある。最近ではミネソタ大学に次ぐ学力レベル（ミネソタ州内）と安い学費で全米の「ベスト100」大学に選ばれている。

### 名称変更

  - Winona State Normal School (1858年)
  - Winona State Teachers' College (1870年)
  - Winona State College (1930年代)
  - Winona State University (1976年)
