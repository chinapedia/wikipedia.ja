[thumb](https://ja.wikipedia.org/wiki/ファイル:JRA_WINS_Isawa.JPG "wikilink")）\]\] **石和温泉**（いさわおんせん）は、[山梨県](../Page/山梨県.md "wikilink")[笛吹市](../Page/笛吹市.md "wikilink")の旧[東八代郡](../Page/東八代郡.md "wikilink")[石和町](../Page/石和町.md "wikilink")地域にある[温泉](../Page/温泉.md "wikilink")である。

石和温泉に隣接する形で、同市の旧[東山梨郡](../Page/東山梨郡.md "wikilink")[春日居町](../Page/春日居町.md "wikilink")地域にある[春日居温泉](https://ja.wikipedia.org/wiki/春日居温泉 "wikilink")とあわせて、**[石和・春日居温泉郷](https://ja.wikipedia.org/wiki/石和温泉郷 "wikilink")**を形成している。

## 泉質

アルカリ性単純泉の無色無臭であり、[pHは](../Page/水素イオン指数.md "wikilink")8.5～9.5程度とされている。 また、源泉は主に[山梨県企業局](https://ja.wikipedia.org/wiki/山梨県企業局 "wikilink")が管理しており、パイプを使って各温泉宿に供給している。

| 源泉名   | 泉温    | 湧出量    |
| ----- | ----- | ------ |
| 第1号源泉 | 28.0℃ | 毎分360L |
| 第2号源泉 | 59.0℃ | 毎分240L |
| 第3号源泉 | 41.0℃ | 毎分504L |
| 第4号源泉 | 25.9℃ | 毎分12L  |
| 第5号源泉 | 63.0℃ | 毎分372L |
| 第6号源泉 | 70.3℃ | 毎分348L |

山梨県企業局管轄源泉一覧\[1\]

山梨県企業局分のみで換算すると源泉は6つ、毎分湧出量は1,836Lであるが、石和温泉内にある一部[温泉宿](../Page/温泉宿.md "wikilink")では自前で源泉を掘削・供給している所もあり\[2\]、実際はこれ以上の源泉数・毎分湧出量がある。これに加え春日居温泉の湧出量を含めると毎分8,080Lが湧出していると推測されている\[3\]。また、山梨県企業局分は資源確保を目的に[循環風呂](https://ja.wikipedia.org/wiki/循環風呂 "wikilink")で供給しているが\[4\]、温泉宿独自で源泉を保有している所は[掛け流し](../Page/掛け流し.md "wikilink")を謳う所もある。

## 歴史

[Isawa_Hot_Springs_Creek_Aug.2013.JPG](https://ja.wikipedia.org/wiki/File:Isawa_Hot_Springs_Creek_Aug.2013.JPG "fig:Isawa_Hot_Springs_Creek_Aug.2013.JPG") [1961年](https://ja.wikipedia.org/wiki/1961年 "wikilink")（昭和36年）に「いすみ荘」で温泉掘削を行った際、毎分2,000L、泉温49℃の温泉が湧出し、周辺の川や田畑に流れ出した。その後即席の[露天風呂](../Page/露天風呂.md "wikilink")が作られ、「青空温泉」と称したのがはじまりである\[5\]\[6\]。その後山梨県企業局や地域内の温泉宿によって掘削が行われ現在に至る\[7\]。最初に温泉が湧出した場所は現在[日本中央競馬会](../Page/日本中央競馬会.md "wikilink")の[場外勝馬投票券発売所](../Page/場外勝馬投票券発売所.md "wikilink")「[ウインズ石和](../Page/ウインズ石和.md "wikilink")」になっており、入口付近に発祥の地の碑が建っている。

[高度経済成長](../Page/高度経済成長.md "wikilink")時代になってから出来た温泉であるが、[新宿駅](https://ja.wikipedia.org/wiki/新宿駅 "wikilink")から[特急](https://ja.wikipedia.org/wiki/特別急行列車 "wikilink")「[あずさ](../Page/あずさ_\(列車\).md "wikilink")」「[かいじ](../Page/かいじ_\(列車\).md "wikilink")」で1時間半前後と近く、また[中央自動車道](../Page/中央自動車道.md "wikilink")開通などアクセスが容易になったことから[京浜](../Page/京浜.md "wikilink")地区の奥座敷として発展。団体客向けの歓楽温泉として知られ、[コンパニオンを配置した](../Page/コンパニオン_\(接客業\).md "wikilink")[温泉旅館が作られたほか](../Page/温泉宿.md "wikilink")、[スナックや](../Page/スナックバー_\(飲食店\).md "wikilink")[バーなどの](../Page/バー_\(酒場\).md "wikilink")[酒場](../Page/酒場.md "wikilink")、さらには[ストリップ劇場などの大規模な](https://ja.wikipedia.org/wiki/ストリップティーズ "wikilink")[風俗街](../Page/風俗街.md "wikilink")も作られた。しかし風俗街というイメージ低下の懸念に加え、団体旅行の低迷により需要が落ち込んだため、近年では個人客や外国人向けに力を入れるようになり、風俗店の取り締まりを強化する一方で[笛吹川](../Page/笛吹川.md "wikilink")の[鵜飼い](../Page/鵜飼い.md "wikilink")や八幡神社の太鼓演奏など歴史、文化を強調したイベントを開催している。また周辺が[果樹園](../Page/果樹園.md "wikilink")であることから特産物の[ブドウ](../Page/ブドウ.md "wikilink")や[モモ](../Page/モモ.md "wikilink")とも結びつけて、[ワイナリー](../Page/ワイナリー.md "wikilink")が至る所に点在する。

## 温泉街

[The_night_of_Isawa_spa_town_Aug.2013.JPG](https://ja.wikipedia.org/wiki/File:The_night_of_Isawa_spa_town_Aug.2013.JPG "fig:The_night_of_Isawa_spa_town_Aug.2013.JPG") [石和温泉駅](https://ja.wikipedia.org/wiki/石和温泉駅 "wikilink")から[笛吹川](../Page/笛吹川.md "wikilink")まで[近津用水](https://ja.wikipedia.org/wiki/近津用水 "wikilink")路を中央に東西約1kmにわたり温泉街が伸びており、全盛期の宿泊施設は大小合わせて120軒超とその規模は[熱海に次ぐともいわれていたが](../Page/熱海温泉.md "wikilink")、現在は50程度の温泉旅館が営業している。宿泊施設は団体客向けの大型施設から家族向けの[旅館](../Page/旅館.md "wikilink")、純和風の高級旅館など様々である。以前は宿泊前提であったが日帰り旅行向けの施設も増えてきており、[健康ランド](../Page/健康ランド.md "wikilink")をはじめ有料の「石和源泉足湯ひろば」という[足湯](../Page/足湯.md "wikilink")及び[手湯](https://ja.wikipedia.org/wiki/手湯 "wikilink")も存在する。

[共同浴場](../Page/共同浴場.md "wikilink")は石和温泉駅を南下し、平等川を渡った市部地区に「石和温泉」という名前の浴場が1件存在する。

## その他

### 温泉偽装

[2004年](../Page/2004年.md "wikilink")の[温泉偽装問題](../Page/温泉偽装問題.md "wikilink")では、石和温泉にある宿泊施設の一つが、井戸水を沸かしながら「温泉」と称して営業していたことが石和温泉旅館協同組合の内部調査で判明。温泉旅館として営業を始めた[1983年](https://ja.wikipedia.org/wiki/1983年 "wikilink")当初から地下水を沸かしていた。

### 宿への酒類持ち込み

温泉宿への酒類の持ち込みは原則禁止となっている（但し温泉宿や旅行プランによっては条件付きで持ち込みが可能の所もあり、事前に宿泊先へ連絡が望ましい）。

なお、2012年11月1日から3月31日までの間に限り「石和温泉旅館協同組合」に加盟している全35施設で県内産ワインに限り1本当たり500円を徴収することで持ち込みを許可していたことがある\[8\]。
