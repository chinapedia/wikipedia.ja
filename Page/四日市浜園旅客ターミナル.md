**四日市浜園旅客ターミナル**（よっかいちはまぞのりょかくターミナル）は、[三重県](https://ja.wikipedia.org/wiki/三重県 "wikilink")[四日市市](../Page/四日市市.md "wikilink")浜園にあった高速[旅客船](../Page/旅客船.md "wikilink")ターミナルである。

[2006年](https://ja.wikipedia.org/wiki/2006年 "wikilink")[4月1日](../Page/4月1日.md "wikilink")、[中部国際空港](../Page/中部国際空港.md "wikilink")（セントレア）への航路の運航を開始した。

ターミナルには[高潮](../Page/高潮.md "wikilink")対策も施され、災害時には緊急物資輸送や市民の交通機関、四日市市の防災基地としても利用される予定であったが、[2011年](../Page/2011年.md "wikilink")4月、所有者である四日市市において取り壊しの予算が成立したのち、ターミナルおよび桟橋など付帯施設のすべてが取り壊された。

## 沿革

  - 2006年4月1日 : 旅客ターミナルが開業。[セラヴィ観光汽船](../Page/セラヴィ観光汽船.md "wikilink")により、中部国際空港への航路が開設される\[1\]。
  - 2006年9月2日 : セラヴィ観光汽船により、名古屋港ガーデン埠頭への航路が開設される\[2\]。
  - 2008年4月 : 経営破綻したセラヴィ観光汽船から、YALが中部国際空港航路の事業を承継する。
  - 2008年7月4日 : YAL社長（当時）が贈賄で逮捕されたことに伴い、四日市市からの燃料費補助が凍結される\[3\]。
  - 2008年8月1日 : ダイヤ改正により減便する。
      - 従来は1日10往復、所要時間35分、運賃1,690円\[4\]だったものを、1日8往復、所要時間39分、運賃1,800円に変更した。
  - 2008年9月5日 : 同年10月4日の運行をもって航路廃止を決定する\[5\]。
  - 2008年10月1日 : エンジントラブルを理由に同日の2便より欠航。修理に時間がかかることから運航は再開されず、そのまま廃止となった。\[6\]
  - 2008年10月5日 : 航路が廃止される。\[7\]
      - 航路廃止後も、残務処理のためとして同年内の船舶係留が許可されていた\[8\]。
  - 2011年4月 : 所有者である四日市市の「平成24年度当初予算」において施設撤去のための予算が成立\[9\]。後に、ターミナルおよび桟橋など付帯施設のすべてが撤去される。

## 航路

\[10\] 全ての航路は廃止済みである。なお、各航路ともセラヴィ観光汽船により運航されていたが、同社の破綻後は中部国際空港への航路のみYALが承継した。

  - 中部国際空港（[愛知県](https://ja.wikipedia.org/wiki/愛知県 "wikilink")[常滑市](../Page/常滑市.md "wikilink")） 行き
      - 就航船 : **シーワープ** （かつては[明淡高速船](https://ja.wikipedia.org/wiki/明淡高速船 "wikilink")（現 [淡路ジェノバライン](https://ja.wikipedia.org/wiki/淡路ジェノバライン "wikilink")）で運航していた）
          - 就航船の要目 : [双胴船](../Page/双胴船.md "wikilink")、134総トン、定員 - 旅客200名、最大速力 - 31.6ノット（約58.8km）\[11\]
  - [名古屋港](../Page/名古屋港.md "wikilink")ガーデン埠頭 行き
      - 就航船 : **いなば2**
      - 運航要目 : 1日4便、所要時間50分 ※土・日・祝日のみ運航。
  - 港内遊覧
      - 就航船 : **いなば2**
      - 運航要目 : 1日1便、所要時間 約50分 ※土・日・祝日のみ運航。
