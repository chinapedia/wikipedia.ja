**トゥーットゥックディ県（தூத்துக்குடி மாவட்டம் ; Thoothukudi district）**は、[インド](../Page/インド.md "wikilink")南部の[タミル・ナードゥ州](https://ja.wikipedia.org/wiki/タミル・ナードゥ州 "wikilink")に属する30の県の一つ。県庁所在地は[トゥーットゥックディ](https://ja.wikipedia.org/wiki/トゥーットゥックディ "wikilink")。

## 概要

[2001年](../Page/2001年.md "wikilink")の国勢調査では、面積4621平方キロメートル、人口157万2273人、人口密度は340人/[km<sup>2</sup>](https://ja.wikipedia.org/wiki/平方キロメートル "wikilink")。 北は[ヴィルドゥナガル県](../Page/ヴィルドゥナガル県.md "wikilink")、北東は[ラーマナータプラム県](../Page/ラーマナータプラム県.md "wikilink")、西は[ティルネルヴェーリ県](https://ja.wikipedia.org/wiki/ティルネルヴェーリ県 "wikilink")に接し、東は[マンナール湾](https://ja.wikipedia.org/wiki/マンナール湾 "wikilink")、南は[インド洋](../Page/インド洋.md "wikilink")である。

## 歴史

[1986年](https://ja.wikipedia.org/wiki/1986年 "wikilink")に[ティルネルヴェーリ県](https://ja.wikipedia.org/wiki/ティルネルヴェーリ県 "wikilink")が分離して成った[チダンバラナール県](https://ja.wikipedia.org/wiki/チダンバラナール県 "wikilink")が、[1996年](../Page/1996年.md "wikilink")に改称して誕生。

## 行政区分

### 郡

トゥーットゥックディ県は、以下の8つの郡（[タミル語](../Page/タミル語.md "wikilink") வட்டம் 「円」 ; 英訳 taluk）に区分けされている。

  - ヴィラーッティックラム郡（விளாத்திக்குளம் ; Vilathikulam）
  - イェッタイヤープラム郡（எட்டையாபுரம் ; Ettayapuram）
  - コーイルパッティ郡（கோயில்பட்டி ; Kovilpattai）
  - オッタッピダーラム郡（ஒட்டப்பிடாரம் ; Ottapidaram）
  - トゥーットゥックディ郡（தூத்துக்குடி ; Thoothukkudi）
  - ティルヴァイクンダム郡（திருவைகுண்டம் ; Srivaikundam）
  - ティルッチェンドゥール郡（திருச்செந்தூர் ; Tiruchendur）
  - サーッタンクラム郡（சாத்தன்குளம் ; Sathankulam）

### 区

トゥーットゥックディ県は、以下の12の区（英訳 block）に区分けされている。

  - プドゥール区（புதூர் ; Pudur）
  - ヴィラーッティックラム区（விளாத்திக்குளம் ; Vilathikulam）
  - コーイルパッティ区（கோயில்பட்டி ; Kovilpattai）
  - オッタッピダーラム区（ஒட்டப்பிடாரம் ; Ottapidaram）
  - カヤッタール区（கயத்தார் ; Kayattar）
  - トゥーットゥックディ区（தூத்துக்குடி ; Thoothukkudi）
