[分子生物学](../Page/分子生物学.md "wikilink")において**相補性**（そうほせい、）とは、2つの構造が互いに鍵と鍵穴の関係にあるような関係性を意味している。自然界において、相補性は[DNAの](../Page/デオキシリボ核酸.md "wikilink")[複製や](https://ja.wikipedia.org/wiki/DNA複製 "wikilink")[転写の際の基本原則であり](https://ja.wikipedia.org/wiki/転写_\(生物学\) "wikilink")、**相補的**（complementary）な[塩基対](../Page/塩基対.md "wikilink")の形成によって、細胞は情報を次の世代へコピーすることが可能となり、配列中に保存されている情報の損傷を見つけて修復することも可能となる。

2つの[核酸](../Page/核酸.md "wikilink")鎖の間でみられる相補性は完全な相補性から全く相補性がないものまでさまざまであり、相補性の程度によってこれらの配列が結合した際の安定性が決定される。さらに、DNA修復など、さまざまな調節機能が塩基対の相補性に基づいて行われている。塩基対の相補性の原則に基づいてDNA鎖と[RNA鎖のハイブリッド二本鎖を形成することも可能であり](../Page/リボ核酸.md "wikilink")、[cDNAライブラリなどの現代的な](../Page/相補的DNA.md "wikilink")[生物工学](../Page/生物工学.md "wikilink")のツールでこうした性質が利用されている。相補性の多くは2つの異なるDNA/RNA鎖の間で見られるものであるが、配列内部に相補性を有する配列では自分自身と相補的な結合を行って折りたたまれた構造をとることも可能である。

## DNAとRNAの塩基対の相補性

[DNA_chemical_structure.svg](https://ja.wikipedia.org/wiki/File:DNA_chemical_structure.svg "fig:DNA_chemical_structure.svg") [Complementarity_(DNA).png](https://ja.wikipedia.org/wiki/File:Complementarity_\(DNA\).png "fig:Complementarity_(DNA).png")

相補性は、[核酸塩基](https://ja.wikipedia.org/wiki/核酸塩基 "wikilink")（[アデニン](../Page/アデニン.md "wikilink")、[チミン](../Page/チミン.md "wikilink")（RNAでは[ウラシル](../Page/ウラシル.md "wikilink")）、[グアニン](../Page/グアニン.md "wikilink")、[シトシン](https://ja.wikipedia.org/wiki/シトシン "wikilink")）の間の相互作用によって達成されている。アデニンとグアニンは[プリンであり](https://ja.wikipedia.org/wiki/プリン塩基 "wikilink")、チミン、シトシンとウラシルは[ピリミジンである](../Page/ピリミジン塩基.md "wikilink")。プリンはピリミジンよりも大きい。どちらも、異なるタイプの核酸塩基とだけ塩基対を形成する。核酸では塩基は[水素結合](../Page/水素結合.md "wikilink")によって結びつくが、水素結合はアデニンとチミン（ウラシル）の間、グアニンとシトシンの間でのみ効果的に機能する。相補的な塩基対A=T（A=U）は2つの水素結合を持ち、G≡C塩基対は3つの水素結合を持つ。他の核酸塩基の配置は、すべて[二重らせん](../Page/二重らせん.md "wikilink")の形成を阻害する。DNAの二本鎖は逆の方向を向いており、と呼ばれる\[1\]。

|        |                                 |            |
| ------ | ------------------------------- | ---------- |
| **核酸** | **核酸塩基**                        | **相補的塩基対** |
| DNA    | アデニン(A)、チミン(T)、グアニン(G)、シトシン(C)  | A=T、G≡C    |
| RNA    | アデニン(A)、ウラシル(U)、グアニン(G)、シトシン(C) | A=U、G≡C    |

DNAまたはRNAの相補鎖は、核酸塩基の相補性に基づいて構築される\[2\]。各塩基対、A=TとG≡Cはほぼ同じ位置を占めるため、立体的な歪みが生じることなくDNAの二重らせんが形成される。核酸塩基間の水素結合もDNA二重らせんを安定化する\[3\]。

二重らせんを形成するDNA鎖の相補性によって、一方の鎖を鋳型として利用して他方を構築することが可能となる。この原理は[DNA複製](https://ja.wikipedia.org/wiki/DNA複製 "wikilink")に重要な役割を果たし、遺伝情報がどのように次の世代へ受け渡されるかについての基礎をなしている。相補性はDNAの[転写でも利用され](https://ja.wikipedia.org/wiki/転写_\(生物学\) "wikilink")、DNAの鋳型からRNA鎖が生成される\[4\]。

などの[DNA修復](../Page/DNA修復.md "wikilink")過程も相補性に基づいており、相補的でない核酸塩基を除去することでDNA複製の差異のエラーを修正することができる\[5\]。

一本鎖DNAは相補的なDNAまたはRNAと容易に結合し、[ハイブリダイゼーション](../Page/ハイブリダイゼーション.md "wikilink")を行う。この原理は、[ポリメラーゼ連鎖反応](../Page/ポリメラーゼ連鎖反応.md "wikilink")（PCR）などの一般的な実験技術の基礎をなしている\[6\]。

相補的な配列を持つ2本の鎖は、、アンチセンスと呼ばれる。一般的に、センス鎖は転写されるDNA配列または転写によって生成されたRNA鎖である。一方、アンチセンス鎖はセンス鎖に相補的な鎖である。

## ヘアピンループの自己相補性

[Stem-loop.svg](https://ja.wikipedia.org/wiki/File:Stem-loop.svg "fig:Stem-loop.svg") 自己相補性（self-complementarity）とは、DNAまたはRNAの配列が折り返されて二本鎖に似た構造が形成されることを指している。配列のどの部分が自己相補性を示すかによって、鎖は[ヘアピンループ](https://ja.wikipedia.org/wiki/ヘアピンループ "wikilink")、ジャンクション、バルジ、を形成する\[7\]。RNAは、グアニンとウラシルの間の結合などDNAではみられない塩基対が形成されるため、こうした構造を形成しやすい\[8\]。

## 調節機能

転写された遺伝子配列中に存在する相補的な核酸配列は塩基対を形成する。こうした構造は自然界でも一般的にみられ、[遺伝子サイレンシング](https://ja.wikipedia.org/wiki/遺伝子サイレンシング "wikilink")などの調節機能を持っている\[9\]。

### アンチセンス転写産物

アンチセンス転写産物は、[コーディング配列に相補的な](../Page/コーディング領域.md "wikilink")[ノンコーディングRNA](../Page/ノンコーディングRNA.md "wikilink")である\[10\]。ゲノムワイド解析によって、は自然界で一般的に生じていることが示されている。これらは、遺伝暗号のコーディング能力を増大させ、遺伝子調節に新たな複雑性を加えるものであると考えられている。これまで、[ヒトゲノム](../Page/ヒトゲノム.md "wikilink")の40%が双方向に転写されることが知られており、逆方向の転写の重要性が示唆されている\[11\]。センス転写産物とアンチセンス転写産物間の相補性領域は二本鎖RNAハイブリッドを形成し、遺伝子調節に重要な役割を果たす可能性がある。例えば、[HIF1αと](../Page/低酸素誘導因子.md "wikilink")[β-セクレターゼのmRNAは双方向に転写されており](https://ja.wikipedia.org/wiki/Β-セクレターゼ1 "wikilink")、アンチセンス転写産物がセンス転写産物の安定化因子として機能していることが示されている\[12\]。

### miRNAとsiRNA

[MiRNA.svg](https://ja.wikipedia.org/wiki/File:MiRNA.svg "fig:MiRNA.svg") [miRNA](https://ja.wikipedia.org/wiki/miRNA "wikilink")は転写される遺伝子領域に相補的なRNA配列で、調節機能を持つ。現在の研究からは、体内を循環するmiRNAは新たな[バイオマーカーとして利用できる可能性が示唆されており](https://ja.wikipedia.org/wiki/バイオマーカー_\(薬学\) "wikilink")、疾患診断での利用の有望性が示されている\[13\]。miRNAは調節遺伝子に由来するより長いRNA配列から形成され、などによって切断される。こうして形成された短い鎖は（RISC）に結合する。そして、転写された遺伝子のmRNAへ相補性を利用して結合し、3つの方法で遺伝子のサイレンサーとして機能する。1つ目は、[リボソーム](../Page/リボソーム.md "wikilink")の結合を阻害し、[翻訳の開始を防ぐことである](../Page/翻訳_\(生物学\).md "wikilink")。2つ目は、結合したmRNAを分解することである。3つ目は、Dicerが作用する2本鎖RNAを形成し、より多くのmiRNAを作り出すことでより多くのコピーを分解することである。[siRNA](https://ja.wikipedia.org/wiki/siRNA "wikilink")はmiRNAと似た機能を持つ。siRNAは異なるRNAに由来するが、miRNAと同様な目的を果たす\[14\]。これらの配列の短さを考えると、相補性の規則は選択された標的を非常に特異的に認識できることを意味している。各塩基の位置ごとに4つの選択肢があるとすると、mi/siRNAの20–22塩基で通り以上の組み合わせを作り出すことが可能である。ヒトゲノムの長さが31億塩基対であるすると\[15\]、各miRNAがゲノム全体で偶然に一致するのは1ヶ所だけであることを意味している。

## バイオインフォマティクス

相補性はDNAやRNAに存在する情報を一本の鎖に保存することを可能にする。鋳型鎖から相補鎖を決定することができ、逆にからは鋳型鎖を決定することができる。ミスマッチが存在する配列の簡略表記法（曖昧コード）、相補鎖を速く読み取るための表記法（[アンビグラム](https://ja.wikipedia.org/wiki/アンビグラム "wikilink")）が開発されている。

### cDNAライブラリ

cDNAライブラリは発現している遺伝子のコレクションであり、遺伝子の同定やクローニングの際に有用な参照ツールである。cDNAライブラリはRNA依存性DNAポリメラーゼ（[逆転写酵素](../Page/逆転写酵素.md "wikilink")）を用いて、mRNAからDNAへの逆転写によって構築される。この過程はDNA/RNA間の相補性の原則に基づいている。ライブラリの最終産物は二本鎖DNAであり、[プラスミド](../Page/プラスミド.md "wikilink")へ挿入される場合もある。cDNAライブラリは現代の研究において強力なツールである\[16\]\[17\]。

### 曖昧コード

| シンボル\[18\] | 意味                                                                  | 塩基 |
| ---------- | ------------------------------------------------------------------- | -- |
| **A**      | [アデニン](../Page/アデニン.md "wikilink")（**a**denine）                     | A  |
| **C**      | [シトシン](https://ja.wikipedia.org/wiki/シトシン "wikilink")（**c**ytosine） |    |
| **G**      | [グアニン](../Page/グアニン.md "wikilink")（**g**uanine）                     |    |
| **T**      | [チミン](../Page/チミン.md "wikilink")（**t**hymine）                       |    |
| **U**      | [ウラシル](../Page/ウラシル.md "wikilink")（**u**racil）                      |    |
| **W**      | 弱い相互作用（**w**eak）                                                    | A  |
| **S**      | 強い相互作用（**s**trong）                                                  |    |
| **M**      | アミノ（a**m**ino）                                                      | A  |
| **K**      | ケト（**k**eto）                                                        |    |
| **R**      | プリン（pu**r**ine）                                                     | A  |
| **Y**      | ピリミジン（p**y**rimidine）                                               |    |
| **B**      | A以外（Aの後の**B**）                                                      |    |
| **D**      | C以外（Cの後の**D**）                                                      | A  |
| **H**      | G以外（Gの後の**H**）                                                      | A  |
| **V**      | T（U）以外（T、Uの後の**V**）                                                 | A  |
| **N**      | 任意（a**n**y）                                                         | A  |

[体系学](https://ja.wikipedia.org/wiki/体系学 "wikilink")において配列を表記する際、「2つのうちのいずれか」や「3つのうちのいずれか」を意味するを用いる必要がある場合がある。IUPAC表記法の**R**（AまたはG）は**Y**（CまたはT）と相補的であり、**M**（AまたはC）は**K**（GまたはT）と相補的である。**W**（AまたはT）と**S**（CまたはG）は相補的ではない。WとSは[ヌクレオチド](https://ja.wikipedia.org/wiki/ヌクレオチド "wikilink")が相補的なパートナーと対合する際に用いる水素結合の数を示しており、パートナーは同数の結合を持っていなければらない\[19\]。

特定の1つのヌクレオチドを除去したコードは、そこで除去されたヌクレオチドと相補的なヌクレオチドを除去したコードと相補的となる。例えば、**V**（T以外）は**B**（A以外）と相補的となる。

### アンビグラム

特定の文字（アンビグラム）を核酸の表記に用いることで、例えば、グアニンをb、シトシンをq、アデニンをn、チミンをuで表記すると、テキストをひっくり返すだけで相補的なDNA配列を作ることができる\[20\]。buqn（GTCA）はひっくり返すと、ubnq（TGAC）となる。

  -

アンビグラムを用いた表記は、[回文配列](https://ja.wikipedia.org/wiki/回文配列 "wikilink")などの相補的な核酸配列の可視化を容易にする\[21\]。通常のASCIIやUnicode文字ではなくカスタムフォントを用いることで、この特徴はより明確になる\[22\]。
