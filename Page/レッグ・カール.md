**レッグ・カール**(leg curl)とは[ウェイトトレーニング](https://ja.wikipedia.org/wiki/ウェイトトレーニング "wikilink")の基本的種目の一つ。

道具として専用の[レッグカールマシン](https://ja.wikipedia.org/wiki/レッグカールマシン "wikilink")や[トレーニングチューブ](https://ja.wikipedia.org/wiki/トレーニングチューブ "wikilink")、[アンクルウエイト](https://ja.wikipedia.org/wiki/アンクルウエイト "wikilink")などを必要とし、[ハムストリングス](https://ja.wikipedia.org/wiki/ハムストリングス "wikilink")の筋力をアップさせるのに大きな効果がある。[エアロビクス](https://ja.wikipedia.org/wiki/エアロビクス "wikilink")などではウエイトを使わない下腿の重量のみでのレッグ・カールも存在する。

ライイング・レッグカールでは[半腱様筋](../Page/半腱様筋.md "wikilink")・[半膜様筋](../Page/半膜様筋.md "wikilink")と[大腿二頭筋](../Page/大腿二頭筋.md "wikilink")が同程度に刺激される。

シーテッド・レッグカールでは[半腱様筋](../Page/半腱様筋.md "wikilink")と[半膜様筋](../Page/半膜様筋.md "wikilink")への刺激が大きく、[大腿二頭筋](../Page/大腿二頭筋.md "wikilink")への刺激が抑えられる。

[ハムストリングス](https://ja.wikipedia.org/wiki/ハムストリングス "wikilink")の内側の種目として行われるのはシーテッド・レッグカールである。

[ハムストリングス](https://ja.wikipedia.org/wiki/ハムストリングス "wikilink")の上部の種目として行われるのはスタンディング・レッグカールである。

## 具体的動作

### ライイング・レッグカール

1.  自分の脚の長さに合わせてレッグカールマシンのパッドの位置を調節する。膝をマシンの回転軸の位置に合わせ、うつ伏せになり、ハンドルを掴む。
2.  息を吐きながら脚を曲げる。足首を曲げて行うとハムストリングスを収縮させやすい。このとき、腰が浮かないように注意する。
3.  十分に曲げたら、息を吸いながら元の姿勢に戻る。
4.  2\~3を繰り返す。

### シーテッド・レッグカール

1.  自分の脚の長さに合わせてレッグカールマシンのパッドと背もたれの位置を調節する。膝をマシンの回転軸の位置に合わせ、シートに座り、ハンドルを掴む。
2.  息を吐きながら脚を曲げる。足首を曲げて行うとハムストリングスを収縮させやすい。
3.  十分に曲げたら、息を吸いながら元の姿勢に戻る。
4.  2\~3を繰り返す。

### スタンディング・レッグカール

1.  自分の脚の長さに合わせてレッグカールマシンのパッドの位置を調節する。膝をマシンの回転軸の位置に合わせ、直立する。
2.  息を吐きながら片脚を曲げる。足首を曲げて行うとハムストリングスを収縮させやすい。
3.  十分に曲げたら、息を吸いながら元の姿勢に戻る。
4.  2\~3を繰り返す。反対側も同様に行う。

### プローン・レッグカール

1.  上体はシートから浮かせて背筋をまっすぐに保ち、パッドの下にふくらはぎをひっかける。動作は股関節を伸ばした状態で始める。
2.  息を吐きながら両足を曲げる。フィニッシュポジションでは、ハムストリングが最大収縮しているため、大腿がシートから浮いた状態となる。
3.  十分に曲げたら、息を吸いながら元の姿勢に戻る。
4.  2\~3を繰り返す。

### バランスボール・レッグカール

1.  床に仰向けになり、[バランスボール](../Page/バランスボール.md "wikilink")に足を乗せる。腋を軽く開き、両手で床を押さえる。
2.  息を吐きながら、腰を床につけたまま、ひざを曲げてボールを自分の方へ引き寄せる。
3.  息を吸いながら元の姿勢に戻る。
4.  2\~3を繰り返す。

### バランスボール・レッグカール(高負荷)

1.  床に仰向けになり、[バランスボール](../Page/バランスボール.md "wikilink")に足を乗せる。腋を軽く開き、両手で床を押さえる。尻を上げ、肩で体重を支える。肩から膝までをまっすぐに保つ。
2.  息を吐きながら、ひざを曲げてボールを自分の方へ引き寄せる。
3.  息を吸いながら元の姿勢に戻る。
4.  2\~3を繰り返す。
