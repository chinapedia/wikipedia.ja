**.in**は[インド](../Page/インド.md "wikilink")の[国別コードトップレベルドメイン](https://ja.wikipedia.org/wiki/国別コードトップレベルドメイン "wikilink")(ccTLD)。インド国立インターネット交換局（NIXI）の下、政府に指定されたINRegistryが管理を行っている。

[2005年](https://ja.wikipedia.org/wiki/2005年 "wikilink")の時点で.inドメインの方針はかなり自由化されており、.inに続くセカンドレベルドメインの登録は無制限。以下のような、自由化以前に作られた分類も利用可能。

  - （インドに存在する企業・個人・組織の誰もが利用可能）

  - （銀行・企業・商標用）

  - （小売店・組合・事務局・個人事業主用）

  - （[ISP](https://ja.wikipedia.org/wiki/ISP "wikilink")用）

  - （[NPO](../Page/NPO.md "wikilink")用）

  - （その他）

  - （個人用）

公的な組織のために6つの分類が用意されている。

  - （学術施設）

  - （教育施設）

  - （調査施設）

  - （教育調査施設）

  - （政府）

  - （軍）

.inドメインの自由化以前の[1992年](../Page/1992年.md "wikilink")から[2004年](../Page/2004年.md "wikilink")の間はわずか7千の登録しかなかったが、[2006年](https://ja.wikipedia.org/wiki/2006年 "wikilink")の終わりには、150カ国以上のユーザーから登録され、20万以上まで登録者を伸ばした。[インド](../Page/インド.md "wikilink")・[ドイツ](https://ja.wikipedia.org/wiki/ドイツ "wikilink")・[アメリカの登録者は](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")、全登録者のおよそ80パーセントに及ぶ。

.nic.inドメインはインドの国立情報センター(National Informatics Centre )のために予約されたものであったが、実際にはインドの政府組織のほとんどがnic.inドメインを取得するに至った。

## 国際化ccTLD

インドでは、インド国内で使われている22の言語についての[国際化ドメイン名](../Page/国際化ドメイン名.md "wikilink")を導入しようとした。

以下の2つは利用可能になっている。

  - （[デーヴァナーガリー](https://ja.wikipedia.org/wiki/デーヴァナーガリー "wikilink")）は以下のドメインが利用可能である。\[1\]

|               |                  |
| :-----------: | :--------------: |
| **デーヴァナーガリー** |     **英字転記**     |
|     भारत      |    *.bharat*     |
|  कंपनी.भारत   | *company.bharat* |
|  विद्या.भारत  |  *vidya.bharat*  |
|  सरकार.भारत   | *sarkar.bharat*  |
|               |                  |

  - （[タミル語](../Page/タミル語.md "wikilink")）は2015年に利用可能になった。\[2\]

以下のものは、割当はされたが、2015年11月現在で利用できない。

  - （[ベンガル語](../Page/ベンガル語.md "wikilink")）

  - （[グルムキー文字](../Page/グルムキー文字.md "wikilink")）

  - （[グジャラート語](https://ja.wikipedia.org/wiki/グジャラート語 "wikilink")）

  - （[テルグ語](https://ja.wikipedia.org/wiki/テルグ語 "wikilink")）

  - （[ウルドゥー語](../Page/ウルドゥー語.md "wikilink")）
