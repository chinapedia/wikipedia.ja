**Movable Type**（ムーバブル・タイプ、**MT**、エムティー）は、[シックス・アパート](../Page/シックス・アパート.md "wikilink")が開発・提供する、[ブログ](../Page/ブログ.md "wikilink")で用いられることの多い[コンテンツ管理システム](../Page/コンテンツ管理システム.md "wikilink") (CMS) 製品である。[プラグイン](../Page/プラグイン.md "wikilink")機構により機能を拡張できることが特徴である。

## 概要

Movable Typeは主に[Perl](../Page/Perl.md "wikilink")と[PHPで書かれており](../Page/PHP_\(プログラミング言語\).md "wikilink")、動作に必要なのはPerlのみとなっている\[1\]。データストアには[リレーショナルデータベースを採用しており](https://ja.wikipedia.org/wiki/関係データベース "wikilink")、[MySQL](https://ja.wikipedia.org/wiki/MySQL "wikilink")、[PostgreSQL](https://ja.wikipedia.org/wiki/PostgreSQL "wikilink")、[SQLite](../Page/SQLite.md "wikilink")などに対応していたのだが、Movable Type 5からはMySQLのみ対応になった。

また、企業向けには別製品としてMovable Type Advancedが存在する。これはMovable Typeを基に、[イントラネット](../Page/イントラネット.md "wikilink")での利用を想定した大量のブログやユーザ、組織や権限などの管理、[LDAP対応](../Page/Lightweight_Directory_Access_Protocol.md "wikilink")、[Oracle Databaseや](../Page/Oracle_Database.md "wikilink")[Microsoft SQL Serverといった](https://ja.wikipedia.org/wiki/Microsoft_SQL_Server "wikilink")[エンタープライズ](../Page/エンタープライズ.md "wikilink")向けのデータベース対応などの追加機能を有償で提供するものである。

さらに、[2007年](../Page/2007年.md "wikilink")12月12日には[ライセンス](../Page/ライセンス.md "wikilink")を[GPLに改変し](../Page/GNU_General_Public_License.md "wikilink")[オープンソース](../Page/オープンソース.md "wikilink")化するMTOS (Movable Type Open Source) プロジェクトが開始された。MTおよびMTOSのソースコードに大きな差異はなく、機能的には全く同じものである。しかし、[2013年](../Page/2013年.md "wikilink")[7月10日](../Page/7月10日.md "wikilink")にMovable Type 6が発表された時にMovable Type 6以降のMTOSの配布をしないことを発表し、プラグインなどは[MIT Licenseに改変した](../Page/MIT_License.md "wikilink")。現時点で提供されているMTOS (Movable Type 5) は同じライセンスで提供される。

これにより、2013年11月13日現在、

  - オープンソース版 Developer向け（Movable Type 5）
  - 開発者ライセンス版 Developer向け（Movable Type 6）
  - 商用ライセンス (Pro) 版 Blogger向け（個人無償）、Business向け（有償、無料トライアル有）
  - クラウド版（有償）
  - Movable Type Advanced（有償）
  - [Amazon Web Services版](https://ja.wikipedia.org/wiki/Amazon_Web_Services "wikilink")

に分かれている。

なお、英語で[活字](../Page/活字.md "wikilink")による印刷の仕組みのことをと呼ぶ。

## 主な機能

  - ブログ記事の作成
  - ウェブページの作成
  - [テンプレートエンジン](../Page/テンプレートエンジン.md "wikilink")(モジュールテンプレートによる作業効率化)
  - [フィード](../Page/フィード.md "wikilink")の提供
  - [トラックバック](https://ja.wikipedia.org/wiki/トラックバック "wikilink")
  - [Atom出版プロトコルによる投稿](https://ja.wikipedia.org/wiki/Atom_\(ウェブコンテンツ配信\)#Atom_Publishing_Protocol "wikilink")
  - 画像や動画、音声といったアイテムの管理
  - 複数ブログの管理
  - 複数ユーザと権限の管理
  - [タグ](../Page/タグ.md "wikilink")やカテゴリによるブログ記事の管理
  - [スパム](../Page/スパム_\(メール\).md "wikilink")[検疫](../Page/検疫.md "wikilink")
  - [OpenID](../Page/OpenID.md "wikilink")やTypeKeyなどの外部認証によるコメント投稿者の管理
  - スタイルや[ウィジェット](https://ja.wikipedia.org/wiki/ウィジェット "wikilink")による[デザイン](../Page/デザイン.md "wikilink")
  - ブログ内の検索
  - プラグイン
