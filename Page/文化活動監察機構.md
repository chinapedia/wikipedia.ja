**文化活動監察機構**（ぶんかかつどうかんさつきこう、*<span lang="pt">Inspecção Geral das Actividades Culturais</span>*）は、[ポルトガル](../Page/ポルトガル.md "wikilink")政府・文化省が設置している公的倫理審査団体。略称・**IGAC**。

[コンピュータゲーム](../Page/コンピュータゲーム.md "wikilink")及び映像作品の[レイティング](../Page/レイティング.md "wikilink")を実施している。

## レイティング表示

コンピュータゲームの審査については[Pan European Game Information](https://ja.wikipedia.org/wiki/Pan_European_Game_Information "wikilink")（PEGI）が行っている場合、PEGIのレイティングがほぼそのまま適用される。但し、IGACがPEGIとの間で[イギリス](https://ja.wikipedia.org/wiki/イギリス "wikilink")の[全英映像等級審査機構](https://ja.wikipedia.org/wiki/British_Board_of_Film_Classification "wikilink")（BBFC）とPEGIの間で結ばれているような審査権限移譲を行っている訳ではない。

PEGIのレイティング中「3+」と「7+」は、ポルトガル国内ではIGACの区分に合わせてそれぞれ「4+」「6+」に置き換えて使用される。

| 表示   | 説明                                                                                                         |
| ---- | ---------------------------------------------------------------------------------------------------------- |
| A    | 子供のため部分的に適した。                                                                                              |
| M/3  | **3歳以上対象**。視聴時間が短く平易に理解が可能で、視聴者に恐怖感や幻惑感を与えないもの。                                                            |
| M/6  | **6歳以上対象**。M/4よりもやや教育的要素が強いもの。                                                                             |
| M/12 | **12歳以上対象・保護者同伴推奨**。M/4-6よりも視聴時間が長く、複雑なストーリー性を有し内容によっては視聴者に[心的外傷](../Page/心的外傷.md "wikilink")を与えるおそれがあるもの。 |
| M/14 | **14歳以上対象・保護者同伴推奨**                                                                                        |
| M/16 | **16歳以上対象・保護者同伴推奨**。M/12よりも暗喩的な性表現や直接的な暴力表現が多いもの。                                                          |
| M/18 | **18歳未満視聴禁止**。直接的な性表現や過激な、或いは猟奇的な暴力表現が中心のもの。対象年齢に達しない者の視聴（販売・譲渡・貸与・映写）や視聴機会の提供が禁じられる。                      |
