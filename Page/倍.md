**倍**（ばい）は、[数学](../Page/数学.md "wikilink")上の概念であるが、その定義は[東洋](../Page/東洋.md "wikilink")数学と[西洋](../Page/西洋.md "wikilink")数学では異なっている。

  - [東洋](../Page/東洋.md "wikilink")…（いずれも 0 でない）自然数 *m* と *n* に対して、*m* に同一量 *m* を *n* 個分加えた数（*m* × (*n* + 1)を求める）。
  - [西洋](../Page/西洋.md "wikilink")…（いずれも 0 でない）自然数 *m* と *n* に対して、*m* を *n* 個分加えた数(*m* × *n* を求める、即ち[乗法](../Page/乗法.md "wikilink")である）。

## 日本における定義・用法

[日本](https://ja.wikipedia.org/wiki/日本 "wikilink")では、[江戸時代](../Page/江戸時代.md "wikilink")以前においては東洋数学の定義が用いられてきた（例えば、「一倍」とは今日で言うところの2倍に該当する。また同じく「半倍」とは、今日で言うところの1.5倍に該当する）が、[近代](../Page/近代.md "wikilink")以後に西洋数字が用いられるようになるとその意味合いも変化して、今日のように乗法を指すようになった。

政府は、[1875年](../Page/1875年.md "wikilink")（明治8年）12月2日以降の公文書における倍数表記を西洋式に改める旨の布告を出している（[太政官布告](https://ja.wikipedia.org/wiki/太政官布告 "wikilink")第183号\[1\]）\[2\]。

もっとも、こうした理解には異説がある。[養老律令](../Page/養老律令.md "wikilink")の雑令に記された出挙の利息制限について記された「一倍」は現在と同じ1倍（＝100％）を指しており、『[法曹至要抄](../Page/法曹至要抄.md "wikilink")』（中巻87条）や[鎌倉幕府](../Page/鎌倉幕府.md "wikilink")の[追加法](https://ja.wikipedia.org/wiki/追加法 "wikilink")でもこれを踏襲している。一方、『[今昔物語集](../Page/今昔物語集.md "wikilink")』（巻14第38）には、利息としての「一倍」（すなわち100％）と元本と合わせた「一倍」（すなわち、100+100＝200％）が併用して用いられている。そして、[江戸時代](../Page/江戸時代.md "wikilink")の『[日葡辞書](../Page/日葡辞書.md "wikilink")』や[井原西鶴](../Page/井原西鶴.md "wikilink")『[日本永代蔵](../Page/日本永代蔵.md "wikilink")』では、「一倍」は2倍（200％）の意味で記されている。こうした変遷から、本来は100％の利息を指して「一倍」の利息と称していたものが、中世になると元利合計の200％をもって「一倍」と称するようになり、中世後期から江戸時代にかけて「一倍」＝200％とする考えが社会に定着したとするものである\[3\]。

「**人一倍**」という言葉などに近代以前の用法の名残が見出せる。単独で「倍」と使われた場合、大抵は「二倍」を意味する（例：倍になる）。

## 西洋数学における n 倍を表す表現

  -
    *「各[自然数](https://ja.wikipedia.org/wiki/自然数#100までの自然数（非負整数）一覧 "wikilink")」の項目も参照*

double、triple、quadruple は''' *n* 倍**を表す[倍数詞である](https://ja.wikipedia.org/wiki/数詞#倍数詞 "wikilink")。数学などでは主に** *n* 個を[組とする表現](https://ja.wikipedia.org/wiki/順序組 "wikilink")'''や ''' *n* 個重なっている（*n* 重）という表現**（例：二個重ねを指す場合は double）として使用されることが多い。triple 以 降は** *n* 個を組**や** *n* 個が重なるとする場合**、***n* -[tuple](https://ja.wikipedia.org/wiki/tuple "wikilink")**（**''n**'' の部分は[ラテン語](../Page/ラテン語.md "wikilink")を[由来](https://ja.wikipedia.org/wiki/由来 "wikilink")とする[数を表す接頭辞](https://ja.wikipedia.org/wiki/接頭辞#数を表す接頭辞 "wikilink")）をつなげて一部形を変化させた表現となり、倍数詞でも** *n* 倍'''を表すとき同様に表現できる。

  - [1](https://ja.wikipedia.org/wiki/1 "wikilink")倍 - single（[シングル](../Page/シングル_\(曖昧さ回避\).md "wikilink")）
  - [2](../Page/2.md "wikilink")倍 - double（[ダブル](../Page/ダブル.md "wikilink")）、duple（デュプル）
  - [3](../Page/3.md "wikilink")倍 - triple（[トリプル](https://ja.wikipedia.org/wiki/トリプル "wikilink")）、treble（トレブル）
  - [4](https://ja.wikipedia.org/wiki/4 "wikilink")倍 - quadruple（[クアドラプル](https://ja.wikipedia.org/wiki/クアドラプル "wikilink")、カドラプル、クォドラプル、クワドループル）
  - [5](https://ja.wikipedia.org/wiki/5 "wikilink")倍 - quintuple（[クインタプル](https://ja.wikipedia.org/wiki/クインタプル "wikilink")、クインティプル、クインテュープル）
  - [6](../Page/6.md "wikilink")倍 - sextuple（セクスタプル、セクステュープル）
  - [7](../Page/7.md "wikilink")倍 - septuple（セプタプル、セプテュプル）
  - [8](https://ja.wikipedia.org/wiki/8 "wikilink")倍 - octuple（オクタプル、オクテュプル）
  - [9](../Page/9.md "wikilink")倍 - nonuple（ノナプル、ノニュプル）
  - [10](../Page/10.md "wikilink")倍 - decuple（デカプル、デキュプル、デキャプル、ディカプル）
  - [12](../Page/12.md "wikilink")倍 - duodecuple（デュデカプル）
  - [16](../Page/16.md "wikilink")倍 - sedecuple（セデカプル）
  - [20](../Page/20.md "wikilink")倍 - vigintuple（ヴィジンタプル、ヴァイジンタプル）
  - [30](../Page/30.md "wikilink")倍 - trigintuple（トリジンタプル）
  - [60](../Page/60.md "wikilink")倍 - sexagintuple（セキサジンタプル）
  - [100](../Page/100.md "wikilink")倍 - centuple（センタプル）
