**千葉県道121号成東鳴浜線**（ちばけんどう121ごう なるとうなるはません）は、[千葉県](../Page/千葉県.md "wikilink")[山武市](../Page/山武市.md "wikilink")津辺の[国道126号](../Page/国道126号.md "wikilink")との交差点を起点とし、同市本須賀（本須賀海岸）の[千葉県道30号飯岡一宮線](../Page/千葉県道30号飯岡一宮線.md "wikilink")との交差点を終点とする[県道である](../Page/都道府県道.md "wikilink")。

## 起点・終点

  - 起点：千葉県山武市津辺（津辺交差点、国道126号交点・千葉県道76号成東酒々井線起点）
  - 終点：千葉県山武市本須賀（本須賀海岸交差点、千葉県道30号飯岡一宮線交点）

## 路線状況

[平成22年](https://ja.wikipedia.org/wiki/平成22年 "wikilink")（[2010年](https://ja.wikipedia.org/wiki/2010年 "wikilink")）度時点での成東鳴浜線の交通量について、観測地点の山武市殿台での自動車類交通量はそれぞれ、

|                   |                 |
| ----------------- | --------------- |
| **昼間12時間自動車類交通量** | **24時間自動車類交通量** |
| style= |小型車       | style= |大型車     |
| 6885台             | 236台            |

また道路部幅員・車道部幅員6.75m、車道幅員5.75m。車線数は2車線であるが歩道幅員は存在しない。昼夜率は1.35%、昼間12時間ピーク比率11.0%、24時間大型車混入率3.3%である\[1\]。
