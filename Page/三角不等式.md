[TriangleInequality.svg](https://ja.wikipedia.org/wiki/File:TriangleInequality.svg "fig:TriangleInequality.svg") [数学](../Page/数学.md "wikilink")における**三角不等式**（さんかくふとうしき、）は、任意の[三角形](../Page/三角形.md "wikilink")に対してその任意の二辺の和が残りの一辺よりも大きくなければならないことを述べるものである\[1\]。三角形の三辺が  で最大辺が  とすれば、三角不等式は

  -
    \(z \leq x + y\)

が成り立つことを主張している。等号が成立するのは三角形が面積  に退化したときに限る。[ユークリッド幾何学](../Page/ユークリッド幾何学.md "wikilink")ほかいくつかの幾何学において、三角不等式は距離に関する定理であって、ベクトルやベクトルの長さ（[ノルム](../Page/ノルム.md "wikilink")）を用いて

  -
    \(\|\mathbf x + \mathbf y\| \leq \|\mathbf x\| + \|\mathbf y\|\)

と書くことができる。ここで、第三辺の長さ  がベクトルの和  で置き換わっていることに注意。 が[実数](../Page/実数.md "wikilink")のとき、それを  のベクトルと見れば、三角不等式は[絶対値](https://ja.wikipedia.org/wiki/絶対値 "wikilink")の間の関係を記述するものとなる。

ユークリッド幾何学において、[直角三角形](../Page/直角三角形.md "wikilink")に対する三角不等式は[三平方の定理の帰結であり](../Page/ピタゴラスの定理.md "wikilink")、一般の三角形の場合は[余弦定理](../Page/余弦定理.md "wikilink")の帰結である（もちろんそれらの定理によらない証明は可能であるけれども）。三角不等式は  や  の何れかにおいて直観的に見ることができる。右図は明らかに不等号が成り立つもの (上) から等号に近いもの (下) までの三例である。ユークリッド幾何学の場合では、等号が成立するには一つの角が  で二つの角が  の場合、したがって三[頂点](https://ja.wikipedia.org/wiki/頂点 "wikilink")が場合に限られる。したがって、ユークリッド幾何学において二点間の最短距離は直線である。

### 折線不等式

三角不等式は[数学的帰納法](../Page/数学的帰納法.md "wikilink")により、任意の折線に関する命題に拡張することができる。すなわち、そのような折線の全ての辺の長さの和は、その折線の二端点を直線で結んだ長さよりも小さくなることはない。特にその帰結として、多角形のどんな長さの辺も残り全ての辺の長さの和より必ず小さいことが言える。 [Arclength.svg](https://ja.wikipedia.org/wiki/File:Arclength.svg "fig:Arclength.svg") このように折線に対して一般化すれば、ユークリッド幾何において二点間を結ぶ最短曲線が直線であることが示せる。

二点間を結ぶ折線がその二点間を結ぶ線分よりも短くならないことから、曲線の[弧長](https://ja.wikipedia.org/wiki/弧長 "wikilink")がその曲線の両端点の間の距離より短くなることはないことが従う。実際、定義により曲線の弧長はそれを近似する折線の長さの上限で、折線に対する結果は端点間を結ぶ線分が全ての折線近似の中で最短ということであった。曲線の弧長は任意の折線近似の長さ以上であるから、曲線それ自身が直線経路より短くなることはない。

## 逆三角不等式

三角不等式が上からの評価であるのに対し、下からの評価を与える**逆向きの三角不等式** (*reverse triangle inequality*) は三角不等式からの初等的な帰結として得られる。それは平面幾何の言葉で言えば「三角形の任意の辺は、その他の二辺の差よりも大きい」ということができる。ノルム空間の場合には

  -
    \(\Bigl|\|x\|-\|y\|\Bigr| \leq \|x-y\|,\)

あるいは距離空間の場合には  ということになる。これはノルム  や距離函数  が の[リプシッツ連続](https://ja.wikipedia.org/wiki/リプシッツ連続 "wikilink")函数となることを示すもので、したがって特に[一様連続](../Page/一様連続.md "wikilink")である。

逆三角不等式は通常の三角不等式を用いて証明できる:

  -
    \(\|x\| = \|(x-y) + y\| \leq \|x-y\| + \|y\| \implies \|x\| - \|y\| \leq \|x-y\|,\)
    \(\|y\| = \|(y-x) + x\| \leq \|y-x\| + \|x\| \implies \|x\| - \|y\| \geq -\|x-y\|\)

に注意すれば

  -
    \(-\|x-y\| \leq \|x\|-\|y\| \leq \|x-y\| \implies \bigl|\,\|x\|-\|y\|\,\bigr| \leq \|x-y\|.\)

## ミンコフスキー空間における不等号の反転

[ミンコフスキー空間](../Page/ミンコフスキー空間.md "wikilink")において  がともに未来光錐内にある時間的ベクトルならば、三角不等式は逆向きの評価

  -
    \(\|x+y\| \geq \|x\| + \|y\|\)

になる。この不等式の物理学的例が[特殊相対論](https://ja.wikipedia.org/wiki/特殊相対論 "wikilink")における[双子のパラドックス](../Page/双子のパラドックス.md "wikilink")である。二つのベクトルがともに過去光錐内にある場合や、少なくとも一方がヌルベクトルである場合にも、同じくこの逆向きの不等号を持つ三角不等式が成り立つ。この結果は、任意の自然数  に対する  次元において成立する。

がともに空間的ベクトルの場合は、通常通りの三角不等式が満足される。
