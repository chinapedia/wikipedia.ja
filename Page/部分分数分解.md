**部分分数分解**（ぶぶんぶんすうぶんかい、*partial fraction decomposition*）とは、[有理式](https://ja.wikipedia.org/wiki/有理式 "wikilink")（あるいは分数式ともいう、[多項式](../Page/多項式.md "wikilink")の商で表される式のこと）に対し、その有理式の分母が[互いに素](../Page/互いに素.md "wikilink")な多項式の積で表されるとき、その有理式を多項式と複数の有理式（ただし、分子の次数は分母の次数より小さい）の和で表すことをいう。このとき分解された各々の有理式の分母を通分すれば、当然ながら元の有理式の分母となる。

有理式からその部分分数分解を得ることを 「部分分数に分解する」 と言いまわすことがあるが、部分分数という実体があるわけではないことに注意。

  - 例：

<!-- end list -->

  - \(\frac{1}{x(x+1)}=\frac{1}{x}-\frac{1}{x+1}\)
  - <math>\\begin{align}

\\frac{2(x - 1)}{x^2(x^2 + 1)} &= \\frac{2}{x} - \\frac{2}{x^2} - \\frac{2(x - 1)}{x^2 + 1} \\qquad \\big(\\text{ in } \\mathbb{R}(x)\\big) \\\\

`                                        &= \frac{2}{x} - \frac{2}{x^2}  - \frac{1 - i}{x + i} - \frac{1 + i}{x - i} \qquad \big(\text{ in }\mathbb{C}(x)\big)`

\\end{align}</math>

有理式の和分や積分においては、部分分数に分解することで計算が楽になることがある。

## 原理

以下、多項式 *h*(*x*) に対し、deg *h* で *h*(*x*) の次数を表すことにする。ただし、*h*(*x*) が多項式として 0 （つまり恒等的に *h*(*x*) = 0 ）であるなら deg *h* = -∞ とする。

### 除法の原理

有理式 *f*(*x*)/*g*(*x*) に対し、deg *f* ≥ deg *g* ならば、一変数[多項式環](../Page/多項式環.md "wikilink")の[除法](../Page/除法.md "wikilink")の原理より、

\[f(x) = Q(x)g(x) + R(x), \quad \deg R < \deg g\] となる多項式 *Q*(*x*), *R*(*x*) が存在するから、

\[\frac{f(x)}{g(x)} = Q(x) + \frac{R(x)}{g(x)}\] と分解することができる。

### 互除法

また、一変数多項式環は[単項イデアル整域](https://ja.wikipedia.org/wiki/単項イデアル整域 "wikilink")だから、多項式 *p*(*x*) と *q*(*x*) が[互いに素](../Page/互いに素.md "wikilink")（つまり共通因数を含まない）ならば *a*(*x*)*p*(*x*) + *b*(*x*)*q*(*x*) = 1 を満たす多項式 *a*(*x*), *b*(*x*) が存在する。したがって、*g*(*x*) = *g*<sub>1</sub>(*x*)*g*<sub>2</sub>(*x*) で、*g*<sub>1</sub>(*x*), *g*<sub>2</sub>(*x*) が互いに素ならば

\[\frac{f(x)}{g(x)} = \frac{\psi_1(x)}{g_1(x)} + \frac{\psi_2(x)}{g_2(x)}\] と分解される。

### 分母が冪の場合

また、*g*(*x*) がある多項式の冪になっているとき、それを *g*(*x*) = (*g*<sub>0</sub>(*x*))<sup>*m*</sup> と書けば、除法の原理より

  -
    *f*(*x*) = *Q*<sub>1</sub>(*x*)(*g*<sub>0</sub>(*x*))<sup>*m*-1</sup> + *R*<sub>1</sub>(*x*), deg *R*<sub>1</sub> \< (*m* - 1)deg *g*<sub>0</sub>

となる多項式 *Q*<sub>1</sub>(*x*), *R*<sub>1</sub>(*x*) がとれる。この *R*<sub>1</sub>(*x*) をさらに (*g*<sub>0</sub>(*x*))<sup>*m*-2</sup> で割り算すれば

  -
    *R*<sub>1</sub>(*x*) = *Q*<sub>2</sub>(*x*)(*g*<sub>0</sub>(*x*))<sup>*m*-2</sup> + *R*<sub>2</sub>(*x*), deg *R*<sub>2</sub> \< (*m* - 2)deg *g*<sub>0</sub>

となり、以下帰納的に

  -
    *R*<sub>*i*-1</sub>(*x*) = *Q*<sub>*i*</sub>(*x*)(*g*<sub>0</sub>(*x*))<sup>*m*-*i*</sup> + *R*<sub>*i*</sub>(*x*), deg *R*<sub>*i*</sub> \< (*m* - *i*)deg *g*<sub>0</sub>

となるものがとれるから、

\[\frac{f(x)}{g(x)} = \sum_{i=1}^{m-1} \frac{Q_i(x)}{(g_0(x))^{i}} + \frac{R_{m-1}(x)}{(g_0(x))^m}\] が成り立つ。特に、deg *Q*<sub>*i*</sub> ≤ deg *R*<sub>*i*-1</sub> - (*m* - *i*)deg *g*<sub>0</sub> ≤ deg *g*<sub>0</sub> となる。

## 複素数係数有理式の分解

任意の[複素数](../Page/複素数.md "wikilink")係数の一変数有理式は、その極（分母となる多項式の零点）が分かれば[因数定理](../Page/因数定理.md "wikilink")を用いて一次式の積に分解されるから、上で見た三つの原理を使うと、多項式の項以外は、分子が定数で分母が一次式の[冪であるような項からなる部分分数分解をもつことが示せる](https://ja.wikipedia.org/wiki/冪乗 "wikilink")。

## 実数係数有理式の分解

実数係数多項式が虚根を持てばその[複素共役も根であることから](https://ja.wikipedia.org/wiki/共役複素数 "wikilink")、任意の実数係数の一変数多項式は実数の範囲で一次式と二次式の積に分解される。したがって、実数係数の一変数有理式の部分分数分解は、分子が定数で分母が一次式の冪である項と、分子が高々一次式で分母が二次式の冪である項および多項式の項からなる。

## 有理型関数の展開

有理式の部分分数分解と同様のことは[有理型関数](../Page/有理型関数.md "wikilink")にも拡張される。一般に有理型関数の極は有限個とは限らないから、この分解は[無限和](https://ja.wikipedia.org/wiki/無限和 "wikilink")すなわち、級数への展開となるので、これを**部分分数への展開**あるいは**部分分数展開** (*partial fraction expansion*) と呼ぶことが多い。

例えば、1/sin<sup>2</sup> *z* は、 sin *z* が整関数であるから、有理型関数である。これは

\[\frac{1}{\sin^2 z} = \sum_{n=-\infty}^\infty \frac{1}{(z - n\pi)^2}\] という部分分数に展開される。
