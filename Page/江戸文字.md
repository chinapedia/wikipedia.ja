## 文字の種類

| 名称                                                              | 説明                                                                                                                                                                                                                                                                                                                                                                                                                                             | 書体                                                                                                                |
| --------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| **[芝居](https://ja.wikipedia.org/wiki/芝居 "wikilink")文字**（しばいもじ）  | 通称、**勘亭流**（かんていりゅう）。[歌舞伎](../Page/歌舞伎.md "wikilink")の看板、[番付](../Page/番付.md "wikilink")に使われる書体。[岡崎屋勘六](https://ja.wikipedia.org/wiki/岡崎屋勘六 "wikilink")が[1779年](../Page/1779年.md "wikilink")（安永8年）に考案。勘亭流の名は彼の号「勘亭」に由来する。歌舞伎界で勘亭流が使われたのは、台本、俳優の楽屋表示などの、外に向けた宣伝物以外のものと、芝居小屋の表の看板、街頭の宣伝物である番付(ポスター)などである。文字の大きさは用途に応じて当然変わる。外向けでないものはそこそこ読みやすい大きさ、看板は大書、番付では外題を遠目にも映りが良いように太く大きめ。[河竹黙阿弥](../Page/河竹黙阿弥.md "wikilink")以降は、狂言台本の本文にも使われた。            | [Kanteiryu_sample.png](https://ja.wikipedia.org/wiki/File:Kanteiryu_sample.png "fig:Kanteiryu_sample.png")       |
| **[寄席文字](https://ja.wikipedia.org/wiki/寄席文字 "wikilink")**（よせもじ） | 通称、**橘流**（たちばなりゅう）。伝統師は、[橘右近](../Page/橘右近.md "wikilink")、[ビラ清](https://ja.wikipedia.org/wiki/ビラ清 "wikilink")、[ビラ辰](https://ja.wikipedia.org/wiki/ビラ辰 "wikilink")。客を寄せるための書体で、客が集まるよう縁起をかついで、字が詰まり加減になっているのが特徴。勘亭流と提灯文字を元に創始された。[落語](../Page/落語.md "wikilink")の看板や[めくり](https://ja.wikipedia.org/wiki/めくり "wikilink")、[番付](../Page/番付.md "wikilink")や、[ビラ](https://ja.wikipedia.org/wiki/ビラ "wikilink")や[千社札](../Page/千社札.md "wikilink")に使用される。 | [Yosemoji_sample.png](https://ja.wikipedia.org/wiki/File:Yosemoji_sample.png "fig:Yosemoji_sample.png")          |
| **籠文字**（かごもじ)                                                   | 字画が厚く、やや四角い書体。反転文字として使われることが多いが、輪郭線として使われることもある。[千社札](../Page/千社札.md "wikilink")に基本的に使用される書体。[田てう](https://ja.wikipedia.org/wiki/里見田蝶 "wikilink")、[田キサが知られる](https://ja.wikipedia.org/wiki/梅素亭玄魚 "wikilink")。                                                                                                                                                                                                                                  | [Kagomoji_sample.png](https://ja.wikipedia.org/wiki/File:Kagomoji_sample.png "fig:Kagomoji_sample.png")          |
| **髭文字**（ひげもじ）                                                   | 文字に「[髭](../Page/髭.md "wikilink")」がついている書体。                                                                                                                                                                                                                                                                                                                                                                                                     | [Higemoji_sample.png](https://ja.wikipedia.org/wiki/File:Higemoji_sample.png "fig:Higemoji_sample.png")          |
| **相撲字**（すもうじ）                                                   | 通称、**[根岸流](../Page/根岸流_\(書風\).md "wikilink")**（ねぎしりゅう）。[大相撲](https://ja.wikipedia.org/wiki/大相撲 "wikilink")の[番付](../Page/番付.md "wikilink")・広告などに使われる書体。                                                                                                                                                                                                                                                                                         | [Sumomoji_sample.png](https://ja.wikipedia.org/wiki/File:Sumomoji_sample.png "fig:Sumomoji_sample.png")          |
| **提灯文字**（ちょうちんもじ）                                               |                                                                                                                                                                                                                                                                                                                                                                                                                                                | [Chochinmoji_sample.png](https://ja.wikipedia.org/wiki/File:Chochinmoji_sample.png "fig:Chochinmoji_sample.png") |
| **角字**（かくじ）                                                     | 極太の四角い書体。[印鑑](https://ja.wikipedia.org/wiki/印鑑 "wikilink")などに使用される。[楷書体](../Page/楷書体.md "wikilink")や、[漢字](../Page/漢字.md "wikilink")そのものの事を角字と言う事もある。                                                                                                                                                                                                                                                                                           | [Kakuji_sample.png](https://ja.wikipedia.org/wiki/File:Kakuji_sample.png "fig:Kakuji_sample.png")                |
