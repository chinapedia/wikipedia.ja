[thumb](https://ja.wikipedia.org/wiki/ファイル:Intel_80486DX-33.jpg "wikilink") **Intel486 DX**は、[インテル](https://ja.wikipedia.org/wiki/インテル "wikilink")が[1989年](../Page/1989年.md "wikilink")にリリースした[Intel486](../Page/Intel486.md "wikilink")シリーズの[32ビット](../Page/32ビット.md "wikilink")[マイクロプロセッサ](../Page/マイクロプロセッサ.md "wikilink")([CPU](../Page/CPU.md "wikilink"))である。**i486DX**、**486DX**とも表記される（Intelを略すときは通常はDXの前にスペースを開けない）。

[80386の後継であり](../Page/Intel_80386.md "wikilink")、486シリーズの最初に登場した。当初は「**80486**」ないし略して486としていたが、廉価版の486SXの出現以降は486DXとされた。

## 386との相違点

  - 386ではプロセッサには内蔵されていなかった[キャッシュ](../Page/キャッシュ.md "wikilink")がプロセッサに内蔵された。容量は8Kiバイト。
  - Pipelining- This allows the processor to handle a LocateFetchExecute each clock cycle. The pipeline is offset meaning the execute step required information from the previous two clock cycles. A locate would be to feed the next fetch, the fetch would be to feed the next execute. The 386 needs to do each step separately.
  - Virtual Memory Handler- [Hardwired](https://ja.wikipedia.org/wiki/Hardwired "wikilink") programming to handle swapping memory to hard drive.
  - Integrated [FPU](../Page/FPU.md "wikilink")- Added accelerated high end math functions.

486は32ビットのデータバスを持つ。これはメモリを、30ピンの[SIMM](../Page/SIMM.md "wikilink")を複数か72ピンのSIMMを1つ使用できる。486は32ビットのアドレスバスを4GBまでメモリに持つことができる。ただしこの当時このような大容量のメモリは存在していない。

初期の486を搭載した[PC/AT互換機](https://ja.wikipedia.org/wiki/PC/AT互換機 "wikilink")は、[VESA ローカルバスを](../Page/VESA_ローカルバス.md "wikilink")[ビデオカード](../Page/ビデオカード.md "wikilink")や[SCSI経由の](../Page/Small_Computer_System_Interface.md "wikilink")[ハードディスクドライブ](https://ja.wikipedia.org/wiki/ハードディスクドライブ "wikilink")の接続に用たものが多かった。このバスの転送速度は[マザーボード](https://ja.wikipedia.org/wiki/マザーボード "wikilink")の周波数に依存した。
