**血球貪食症候群**（Hemophagocytic syndrome; HPS,欧米ではHemophagocytic lymphohistiocytosis; HLHなど、国際的に統一の呼び名はまだない）は、医学の発達に伴って明らかとなってきた疾患のひとつで、本来人間をまもるべき[マクロファージ](https://ja.wikipedia.org/wiki/マクロファージ "wikilink")や[好中球](https://ja.wikipedia.org/wiki/好中球 "wikilink")といった[免疫細胞](https://ja.wikipedia.org/wiki/免疫細胞 "wikilink")が暴走し、自らの血球（とくに血小板）を食べてしまう病気である。小児で先に報告されたが、成人に起こることは現在ではわかっている。きわめて重篤な致死的疾患である。突然健常者におこることもある。

## 病型

### 一次性（遺伝性）

  - 家族性血球貪食症候群（familial hemophagocytic lymphohistiocytosis; FHL）
      - [常染色体劣性遺伝](https://ja.wikipedia.org/wiki/常染色体劣性遺伝 "wikilink")による。血球貪食症候群を唯一の表現型として発症する。Perforin、syntaxin11、Munc13-4、Munc18-2などが原因遺伝子として報告されているが、遺伝子が特定されていないものもある。

| 型    | 蛋白     | 責任遺伝子      |
| ---- | ------ | ---------- |
| FHL1 | HPLH1  | 不明         |
| FHL2 | PRF1   | Perforin   |
| FHL3 | UNC13D | Munc13-4   |
| FHL4 | STX11  | Syntaxin11 |
| FHL5 | STXBP2 | Munc18-2   |

  - X連鎖リンパ増殖性疾患（Duncan病、X-linked lymphoproliferative disorder; XLP）
      - [伴性遺伝](https://ja.wikipedia.org/wiki/伴性遺伝 "wikilink")による。EBウイルスに対する異常な免疫応答を特徴とする。

| 型    | 蛋白                                    | 責任遺伝子  |
| ---- | ------------------------------------- | ------ |
| XLP1 | SLAM-associated protein               | SH2D1A |
| XLP2 | X-linked inhibitor of apoptosis（XIAP） | BIRC4  |

  - [白子症](https://ja.wikipedia.org/wiki/白子症 "wikilink")を伴う免疫不全症候群
      - 2型Griscelli症候群（Griscelli syndrome, type 2; GS II）
          - 常染色体劣性遺伝による。
      - Chediak-Higashi症候群（Chédiak-Higashi syndrome; CHS）
          - 常染色体劣性遺伝による。白子症と血球貪食症候群の発症、並びに細胞内巨大顆粒の存在を特徴とする。
      - 2型Hermansky-Pudlak症候群（Hermansky-Pudlak syndrome, type 2; HPS II）
          - 常染色体劣性遺伝による。白子症と血小板機能障害による出血傾向を特徴とする。

| 型                            | 蛋白                                    | 責任遺伝子     |
| ---------------------------- | ------------------------------------- | --------- |
| Griscelli Syndrome ll        | Rab27A                                | RAB27A    |
| Chédiak-Higashi Syndrome     | LYST（Lysosomal trafficking regulator） | CHS1/LYST |
| Hermansky-Pudlak Syndrome ll | AP3のβサブユニット                           | AP3B1     |

### 二次性（反応性）

  - ウイルス関連血球貪食症候群（Virus-associated hemophagocytic syndrome; VAHSヴァース）
      - EBV-AHS (イービーヴァース）：[EBウイルス](https://ja.wikipedia.org/wiki/EBウイルス "wikilink")による、最重症型のHPS
      - そのほか[サイトメガロウイルス](https://ja.wikipedia.org/wiki/サイトメガロウイルス "wikilink")、[パルボウイルス](../Page/パルボウイルス.md "wikilink")、[単純ヘルペスウイルス](../Page/単純ヘルペスウイルス.md "wikilink")、[帯状疱疹](../Page/帯状疱疹.md "wikilink")ウイルス、[麻疹](../Page/麻疹.md "wikilink")ウイルス、ヒトヘルペスウイルス8型、[HIVウイルスなどの](../Page/ヒト免疫不全ウイルス.md "wikilink")[ウイルス](../Page/ウイルス.md "wikilink")や、[ブルセラ](../Page/ブルセラ症.md "wikilink")、[結核菌](../Page/結核菌.md "wikilink")、[リーシュマニア](../Page/リーシュマニア.md "wikilink")、また[グラム陰性桿菌](https://ja.wikipedia.org/wiki/グラム陰性桿菌 "wikilink")や[真菌](https://ja.wikipedia.org/wiki/真菌 "wikilink")によって生じたとする報告がある。
  - 自己免疫疾患関連血球貪食症候群（Autoimmune disease-associated hemophagocytic syndrome; AAHSアース）
      - [全身性エリテマトーデス](https://ja.wikipedia.org/wiki/全身性エリテマトーデス "wikilink")にともなうもの
      - そのほか[関節リウマチ](../Page/関節リウマチ.md "wikilink")、[成人スティル病](../Page/成人スティル病.md "wikilink")などたいていの[膠原病](../Page/膠原病.md "wikilink")に合併する報告があるが、なぜかこの病型のHPSは軽症で、[ステロイド](../Page/ステロイド.md "wikilink")が奏効することが多い。
  - 悪性腫瘍関連血球貪食症候群（Malignancy-associated hemophagocytic syndrome; MAHSマーズ）
      - 節外性NK-T細胞性リンパ腫をはじめとする[悪性リンパ腫](../Page/悪性リンパ腫.md "wikilink")や[白血病](../Page/白血病.md "wikilink")
  - その他
      - [東海村JCO臨界事故](https://ja.wikipedia.org/wiki/東海村JCO臨界事故 "wikilink")の犠牲者A（35歳）が発症した\[1\]が、メカニズムは現在のところ不明である。

## 症状

[発熱](https://ja.wikipedia.org/wiki/発熱 "wikilink")、[汎血球減少](https://ja.wikipedia.org/wiki/汎血球減少 "wikilink")、肝脾腫、[播種性血管内凝固症候群](https://ja.wikipedia.org/wiki/播種性血管内凝固症候群 "wikilink")（DIC）、全身倦怠感、リンパ節腫脹、腹水、出血などである。

## 検査

血球を食べてしまうので、[赤血球](../Page/赤血球.md "wikilink")、[白血球](../Page/白血球.md "wikilink")、[血小板](../Page/血小板.md "wikilink")が激減する。特に血小板の減少があまり激しいと、治療するまもなく重篤な出血（脳出血、消化管出血など）をひきおこし命を落とす。凝固能も低下する。[トリグリセリド](https://ja.wikipedia.org/wiki/トリグリセリド "wikilink")が異常に上昇するのは特徴的といわれる。[フィブリノーゲン](https://ja.wikipedia.org/wiki/フィブリノーゲン "wikilink")は低下する。[フェリチン](https://ja.wikipedia.org/wiki/フェリチン "wikilink")、[可溶性IL-2受容体](https://ja.wikipedia.org/wiki/可溶性IL-2受容体 "wikilink")の上昇は（発熱+血球減少という検査前情報を含めれば）診断的な情報である。[腫瘍マーカー](https://ja.wikipedia.org/wiki/腫瘍マーカー "wikilink")のひとつである[NSE](https://ja.wikipedia.org/wiki/NSE "wikilink")も上昇するという\[2\]。

## 診断

骨髄生検による。異常な[マクロファージ](https://ja.wikipedia.org/wiki/マクロファージ "wikilink")が自らの血球を貪食している像が見られる。

## 治療

基本的には基礎疾患の治療であるが、EBウイルスによる最重症型などでは早期に[エトポシド](../Page/エトポシド.md "wikilink")のようなきわめて強力な（そして副作用の強い）治療に踏み切らなければならないといわれている（HLH-94プロトコル）。新しい治療法ではさらに[シクロスポリン](../Page/シクロスポリン.md "wikilink")を早期から併用する（HLH-2004プロトコル）。
