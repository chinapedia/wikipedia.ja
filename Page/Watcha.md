**Watcha, Inc.**（）は、[大韓民国](../Page/大韓民国.md "wikilink")に本社を置くソフトウェア会社。2011年9月に設立された。 レコメンドサービス「WATCHA PEDIA ()」と、 [動画配信サービス](../Page/定額制動画配信サービス.md "wikilink")「WATCHA ()」を運営する。

日本での運営は、**株式会社Watcha Japan**（Watcha Japan, inc.）が行っている。

## 沿革

  - 2011年9月 - Frograms, Inc. (）設立\[1\]。
  - 2012年8月 ‐ レコメンドサービス「WATCHA()」のベータ版を公開\[2\]。
  - 2013年5月 ‐ 「WATCHA 」の正式版を公開\[3\]。
  - 2015年9月 ‐ 「 WATCHA」の日本語版を公開。
  - 2016年1月 ‐ [定額制動画配信サービス](../Page/定額制動画配信サービス.md "wikilink")「WATCHA PLAY()」を公開\[4\]。
  - 2018年3月 ‐ Watcha, Inc.**（**)に社名を変更\[5\]。
  - 2020年7月9日 ‐ 「WATCHA」を「WATCHA PEDIA ()」に、「WATCHA PLAY」を「WATCHA ()」に改名\[6\]。

## サービス

### WATCHA PEDIA

2013年に韓国で公開され、2015年に日本で公開されたレコメンドサービス。旧名称は、「WATCHA」。

映画やドラマの評価、レビュー機能のほか、ユーザーの記録データを分析し、一人ひとりの好みに合った作品をおすすめするサービス\[7\]。

総評価件数は5億件を突破している。

### WATCHA

2016年1月に韓国でサービスを開始した動画配信サービス。日本でもサービス開始予定。旧名称は、「WATCHA PLAY」。

ベーシックプラント、プレミアムプランがある。

|             | ベーシック プラン | プレミアム プラン |
| ----------- | --------- | --------- |
| 月額料金        | 790円(税抜)  | 1200円(税抜) |
| 同時視聴可能台数    | 1台        | 4台        |
| ダウンロード可能作品数 | 5作        | 100作      |
