**ルジャンドルの微分方程式**（るじゃんどるのびぶんほうていしき）とは、[アドリアン＝マリ・ルジャンドル](../Page/アドリアン＝マリ・ルジャンドル.md "wikilink")にその名をちなむ、以下の形の[常微分方程式](../Page/常微分方程式.md "wikilink")の事である\[1\]\[2\]。

\[\frac{d}{dx} \left[ \left( 1 - x^2 \right) y' \right] + \nu (\nu + 1) y = 0\]

これは[ガウスの微分方程式](../Page/ガウスの微分方程式.md "wikilink")において、*α* = *ν* + 1, *β* = -*ν*, *γ* = 1 と選び、x → (1-x)/2 と置き換えた場合と同じである\[3\]。

この解は[偶関数](https://ja.wikipedia.org/wiki/偶関数 "wikilink")と[奇関数](https://ja.wikipedia.org/wiki/奇関数 "wikilink")になる事が知られていて、それぞれ以下のようになる。

  - \(y_e(x) = \sum_{n = 0}^\infty \frac{2^{2n}}{(2n)!} \left( -\frac{\nu}{2} \right)_n \left( \frac{\nu + 1}{2} \right)_n x^{2n}\)
  - \(y_o(x) = \sum_{n = 0}^\infty \frac{2^{2n}}{(2n+1)!} \left( \frac{1 - \nu}{2} \right)_n \left( 1 + \frac{\nu}{2} \right)_n x^{2n + 1}\)

また特別なケースとして *ν* = 0, 1, 2, ... の場合に解は *ν* 次多項式となる。この多項式のことを[ルジャンドルの多項式](https://ja.wikipedia.org/wiki/ルジャンドルの多項式 "wikilink")と呼ぶ\[4\]\[5\]。
