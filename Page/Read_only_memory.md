**Read only memory**（リードオンリーメモリ、ROM: ロム）は、記録されている情報を読み出すことのみ可能な[メモリである](../Page/記憶装置.md "wikilink")。**読み出し専用メモリ**ともいう。

## 概説

[ROM](https://ja.wikipedia.org/wiki/ROM "wikilink")は読み出し専用メモリであり、[ROM](https://ja.wikipedia.org/wiki/ROM "wikilink")に記録されている内容は本来の意味では書き換えできない。しかし、「[RAM](https://ja.wikipedia.org/wiki/RAM "wikilink")」が基本的にランダムアクセスで読み書きできるものを指す\[1\]のに対し、それの対義語のように使われているため、[EPROM](https://ja.wikipedia.org/wiki/EPROM "wikilink")の類など、何らかの条件の元一時的に書きこみできるものも[ROM](https://ja.wikipedia.org/wiki/ROM "wikilink")と呼ばれるが通常は読み出し専用である。

書き換えできない[ROM](https://ja.wikipedia.org/wiki/ROM "wikilink")は、配線パターンによって（ICの製造過程の「マスク」から「マスクドROM」という）特定の情報を予め記録した状態で製造される。その他、読み出し専用・書き換え不可という動作や生産時に情報を記録しておくという性質が似ている[ストレージ](https://ja.wikipedia.org/wiki/ストレージ "wikilink")メディア等を指す場合もある（[CD-ROM](../Page/CD-ROM.md "wikilink")など）。

## 用途

  - ・ファームウェアを記憶し[CPU](https://ja.wikipedia.org/wiki/CPU "wikilink")の[主記憶装置](../Page/主記憶装置.md "wikilink")として使用される。汎用のコンピュータにおいてはIPLや  といった一部のプログラムやデータがROM上に置かれ、アプリケーションプログラムは  上で実行されるのに対し、[組み込み用途のコンピュータの場合](https://ja.wikipedia.org/wiki/組み込みシステム "wikilink")、システムのプログラム全体を  上に配置することもある。

  - [補助記憶装置](https://ja.wikipedia.org/wiki/補助記憶装置 "wikilink")用の外部記憶媒体としての用途。

  - 着脱可能な[カートリッジによる](https://ja.wikipedia.org/wiki/ロムカセット "wikilink")[アプリケーションの供給](https://ja.wikipedia.org/wiki/アプリケーションソフトウェア "wikilink")。主記憶装置としてのアドレス空間が割り当てられる。[ファミリーコンピュータ](https://ja.wikipedia.org/wiki/ファミリーコンピュータ "wikilink")や[{{langなどの据え置き型ゲーム機](https://ja.wikipedia.org/wiki/NINTENDO64 "wikilink")、また、ほとんどの[携帯型ゲーム](https://ja.wikipedia.org/wiki/携帯型ゲーム "wikilink")機などで使用される。
