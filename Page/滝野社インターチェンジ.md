**滝野社インターチェンジ**（たきのやしろインターチェンジ）は、[兵庫県](../Page/兵庫県.md "wikilink")[加東市](../Page/加東市.md "wikilink")にある[中国自動車道](../Page/中国自動車道.md "wikilink")の[インターチェンジ](../Page/インターチェンジ.md "wikilink")である。

加東市および[西脇市](../Page/西脇市.md "wikilink")の最寄りインターチェンジ。旧[加東郡](../Page/加東郡.md "wikilink")の3町（[社町](../Page/社町.md "wikilink")、[滝野町](../Page/滝野町.md "wikilink")、[東条町](../Page/東条町.md "wikilink")）の合併による加東市の成立までは、社町と滝野町の境界に位置していた。

## バス停留所

料金所そばに高速バス専用の[バス停留所](../Page/バス停留所.md "wikilink")が設けられている。停留所設備は高速道路外に設けられており、停車するバスはいったん料金所を出て客扱いを行う形になる。

隣接する国道175号社バイパスの側道には、[中国ハイウェイバス](../Page/中国ハイウェイバス.md "wikilink")社線・西脇線・[西脇急行線](../Page/西脇急行線.md "wikilink")の滝野社インターバス停（[明石市](../Page/明石市.md "wikilink")方面のみに設置）がある。こちらに停車するバスは、いったん北池交差点に設けられた専用の転回場を通り、滝野社インターバス停で客扱いを行う形になる。

  - 中国ハイウェイバス社線（上り便）：国道175号社バイパス → 滝野社インター入口交差点 → 北池交差点（[Uターン](../Page/Uターン.md "wikilink")） → 滝野社インターバス停（客扱い） → 滝野社インター入口交差点 → 北野交差点 → 滝野社IC（兵庫県道17号西脇三田線からの出入口） → 中国自動車道
  - 中国ハイウェイバス社線（下り便）：中国自動車道 → 滝野社IC（国道175号社バイパスからの出入口） → 北池交差点（Uターン） → 滝野社インターバス停（客扱い） → 滝野社インター入口交差点 → 国道175号社バイパス
  - 中国ハイウェイバス西脇線（上り便）：兵庫県道17号西脇三田線 → 滝野社インター入口交差点 → 北池交差点（[Uターン](../Page/Uターン.md "wikilink")） → 滝野社インターバス停（客扱い） → 滝野社インター入口交差点 → 北野交差点 → 滝野社IC（兵庫県道17号西脇三田線からの出入口） → 中国自動車道
  - 中国ハイウェイバス西脇線（下り便）：中国自動車道 → 滝野社IC（国道175号社バイパスからの出入口） → 北池交差点（Uターン） → 滝野社インターバス停（客扱い） → 滝野社インター入口交差点 → 兵庫県道17号西脇三田線
  - 西脇急行線：兵庫県道17号西脇三田線 → 滝野社インター入口交差点 → 北池交差点（Uターン） → 滝野社インターバス停（客扱い） → 滝野社インター入口交差点 → 兵庫県道17号西脇三田線
