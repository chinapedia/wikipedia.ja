『**パンとシランクス**』（）は、[カール・ニールセン](../Page/カール・ニールセン.md "wikilink")が作曲した[交響詩](../Page/交響詩.md "wikilink")。1918年2月11日に[コペンハーゲン](../Page/コペンハーゲン.md "wikilink")で開催された自作曲によるコンサートのために書かれた。

## 概要

1918年1月23日に[スウェーデン](https://ja.wikipedia.org/wiki/スウェーデン "wikilink")の作曲家[ヴィルヘルム・ステーンハンマル](../Page/ヴィルヘルム・ステーンハンマル.md "wikilink")に宛てた手紙の中で、ニールセンは2月11日の演奏会に『パンとシランクス』を加えたもののまだ一音も書いていないと説明している。しかし、前年に妻の[アネ・マリーイとこの作品を作曲するきっかけとなった](../Page/アネ・マリーイ・カール＝ニールセン.md "wikilink")[オウィディウス](../Page/オウィディウス.md "wikilink")の『[変身物語](../Page/変身物語.md "wikilink")』について議論して以来、ニールセンはしばらく楽曲について思いを巡らせていたようである。総譜は2月6日までに完成された\[1\]。

約20年にわたるニールセンの作品のみで構成されたこのコンサートでは、本作に加えて[オペラ](../Page/オペラ.md "wikilink")『[サウルとダヴィデ](../Page/サウルとダヴィデ.md "wikilink")』第2幕への前奏曲、『[眠り](https://ja.wikipedia.org/wiki/眠り_\(ニールセン\) "wikilink")』、[交響曲第4番が全て作曲者自身の](../Page/交響曲第4番_\(ニールセン\).md "wikilink")[指揮で演奏された](../Page/指揮_\(音楽\).md "wikilink")。さらにピアノ曲『[シャコンヌ](https://ja.wikipedia.org/wiki/シャコンヌ_\(ニールセン\) "wikilink")』がクレスチャン・クレスチャンスンの演奏で披露されている\[2\]。

## 評価

本作はとりわけよい評価を受けた。『ポリティケン』紙のチャーレス・ケアウルフによる論評はまず作品が「ガリア風\[3\]、極めて[ドビュッシー風の](../Page/クロード・ドビュッシー.md "wikilink")」特徴を有することをほのめかし、それから再生、発展、円熟が感られることを称賛した。彼は一層饒舌な調子で締めくくっている。「音がひとつ加わるごとにますます崇高なものへとなっていくのである。そして最後に最高音と再低音がヴァイオリンのハーモニクスとコントラバスから双方立ち上がってくる（略）そして実に自然と喜びが生まれ出るのだ。」その他の論評も前向きであった\[4\]。

本作はニールセンの生前の[スカンジナビア](https://ja.wikipedia.org/wiki/スカンジナビア "wikilink")では頻繁に演奏された。彼がコンサートプログラムを考える際は、『[サガの夢](https://ja.wikipedia.org/wiki/サガの夢 "wikilink")』と『パンとシランクス』を同時に演奏するように組むことが多かった\[5\]。

## 楽器編成

[フルート](../Page/フルート.md "wikilink")2（[ピッコロ](../Page/ピッコロ.md "wikilink")持ち替え）、[オーボエ](../Page/オーボエ.md "wikilink")2（[コーラングレ](../Page/コーラングレ.md "wikilink")持ち替え）、[クラリネット](../Page/クラリネット.md "wikilink")（A）2、[ファゴット](../Page/ファゴット.md "wikilink")2、[ホルン](../Page/ホルン.md "wikilink")4、[トランペット](../Page/トランペット.md "wikilink")（F）2、[ティンパニ](../Page/ティンパニ.md "wikilink")、[シンバル](../Page/シンバル.md "wikilink")、[タンブリン](https://ja.wikipedia.org/wiki/タンブリン "wikilink")、[トライアングル](../Page/トライアングル.md "wikilink")、[スネアドラム](../Page/スネアドラム.md "wikilink")、[グロッケンシュピール](../Page/グロッケンシュピール.md "wikilink")、[シロフォン](../Page/シロフォン.md "wikilink")、[弦五部](../Page/弦楽合奏.md "wikilink")\[6\]。

## 演奏時間

約8分半\[7\]。

## 楽曲構成

好色な神[パーンが](../Page/パーン_\(ギリシア神話\).md "wikilink")[ニュンペー](../Page/ニュンペー.md "wikilink")のを追っていた際に[パンパイプ](../Page/パンパイプ.md "wikilink")を生み出したという古代の伝説が基になっている。シューリンクスは川岸まで走り着き、川のニュンペーに助けを求めた。願いに応じる形で彼女は川の窪みに群生する葦に姿を変えられ、神が悔しがる吐息がその間を吹き抜けると忘れがたい音を発した。パーンは葦を刈り取って形を整えて初めてのパンパイプを拵え、これが以来「シューリンクス」として知られるようになった\[8\]。

木管楽器に大きな独奏部があることは、楽曲がシューリンクスを取り上げているという理由で理解しやすい。曲はニールセンが[交響曲第4番直後の絶頂期に書かれている](../Page/交響曲第4番_\(ニールセン\).md "wikilink")。活発で可憐、そして詩的な作品となっている\[9\]。

チェロの独奏によって幕を開ける（譜例1）。

譜例1 <score> \\relative c, {

`\key f \major \time 12/8 \set Score.tempoHideNote = ##t \tempo "Andantino. (quasi allegretto)" 4.=63 \clef bass`
`r4^\markup { (Vc. solo) } r8 f4.\f ~ f4 e8( f4 e8) d4.->( c4.~ c4) f8\> ( a4 c8 \bar "||" \time 9/8f8\p )`

} </score>

ざわついた楽想が差しはさまれて譜例1が繰り返される。非旋律的な展開が勢いよく行われ、落ち着きを取り戻すとグロッケンシュピールを伴って譜例2が出される。

譜例2 <score> \\relative c' {

`\key a \minor \time 3/4 \set Score.tempoHideNote = ##t \tempo "l'stesso tempo." 4=63`
`f8._\markup { \dynamic mp \italic espressivo } ^\markup { (Cor. ingl.) }( aes16 c4. b8 f' d des4.-> c8`
`b des c4. b8 f' d! des[ \times 2/3 { c16 des c] } b4)`

} </score>

コーラングレの旋律を中心に展開していき、打楽器が鋭いリズムを刻むと次々速度を上げていく。その後、静まり返ると[ポコ](https://ja.wikipedia.org/wiki/wikt:poco "wikilink")・[アダージョから譜例](https://ja.wikipedia.org/wiki/wikt:adagio "wikilink")2が回想され、すぐさま譜例1が奏されると不協和音とともに音量を落として幕引きとなる。
