[固体物理学](https://ja.wikipedia.org/wiki/固体物理学 "wikilink")において、**強結合近似**（きょうけつごうきんじ、）は[電子バンド計算の際に用いられる近似の一つで](../Page/バンド構造.md "wikilink")、系の[波動関数](../Page/波動関数.md "wikilink")を各原子の場所に位置する孤立[原子](https://ja.wikipedia.org/wiki/原子 "wikilink")に対する波動関数の[重ね合わせ](../Page/重ね合わせ.md "wikilink")により近似する手法である。この手法は量子化学で用いられる[LCAO法](../Page/LCAO法.md "wikilink")と密接な関係がある。さまざまな固体に対して用いることができ、多くの場合で定量的に良い結果を得ることができる。そうでない場合は他の手法と組み合せることもできる。強結合近似は一電子近似であるが、[表面準位](https://ja.wikipedia.org/wiki/表面準位 "wikilink")計算や様々な[多体問題](https://ja.wikipedia.org/wiki/多体問題_\(量子論\) "wikilink")、[準粒子](https://ja.wikipedia.org/wiki/準粒子 "wikilink")の計算などの進んだ計算の叩き台として用いられる。**強束縛近似**、**タイトバインディング近似**とも。

## 概要

「強結合」という名前は、この[電子バンド構造モデルが固体に強く結合した電子の](../Page/バンド構造.md "wikilink")[量子力学](https://ja.wikipedia.org/wiki/量子力学 "wikilink")的物性を記述することから来ている。このモデルにおける[電子](https://ja.wikipedia.org/wiki/電子 "wikilink")は、その属する[原子](https://ja.wikipedia.org/wiki/原子 "wikilink")に強く束縛されており、隣接する原子の[状態やそれの作るポテンシャルや相互作用は限定されたものとなる](../Page/量子状態.md "wikilink")。結果として、電子の[波動関数](../Page/波動関数.md "wikilink")はその属する原子が[遊離](https://ja.wikipedia.org/wiki/遊離 "wikilink")状態にある時の[原子軌道](../Page/原子軌道.md "wikilink")に似たものとなり、エネルギーも遊離原子およびイオンにおける[イオン化エネルギー](../Page/イオン化エネルギー.md "wikilink")に近くなる。

強結合近似の下の一粒子ハミルトニアンの数学的形式\[1\]を初めて見るときは複雑に見えるかもしれないが、このモデルはまったく複雑ではなく、直感的理解が極めて容易である。この理論で重要な役割を果たすのは三種類の行列要素だけである。このうち二種類はゼロに近いことが多く、しばしば無視される。最も重要なのは原子間行列要素であり、化学の分野では単に[結合エネルギー](../Page/結合エネルギー.md "wikilink")と呼 ばれる。

一般に、このモデルではいくつかの原子[エネルギー準位](../Page/エネルギー準位.md "wikilink")と原子軌道が用いられる。ここで、各軌道は異なるの表現に属することがあり、その場合はバンド構造が複雑になりがちである。[逆格子および](../Page/逆格子ベクトル.md "wikilink")[ブリュアンゾーン](../Page/ブリュアンゾーン.md "wikilink")はしばしば格子の[空間群](../Page/空間群.md "wikilink")とは異る空間群の表現に属することになる。ブリュアンゾーンの高対称点は異った点群表現に属する。単純な化合物を対象とする場合、高対称点の固有状態を解析的に計算するのは難しくない。そのため、強結合モデルは[群論](https://ja.wikipedia.org/wiki/群論 "wikilink")について学ぶ際の好例として挙げられることがある。

強結合モデルはその長い歴史上、様々な方法で様々な目的に用いられており、それぞれ異った結果をもたらしている。このモデルは自己完結的ではなく、部分的に[ほとんど自由な電子](../Page/ほとんど自由な電子.md "wikilink")モデルなどの他のモデルや別の方法による計算の結果を組込む必要がある。このモデル全体、もしくは一部分が他の計算の基として用いられることがある\[2\]。たとえば、[導電性高分子](../Page/導電性高分子.md "wikilink")や[有機半導体](../Page/有機半導体.md "wikilink")、[分子エレクトロニクス](https://ja.wikipedia.org/wiki/分子エレクトロニクス "wikilink")の分野においては、もともとの強結合モデルでは原子軌道を用いるところに[共役系](https://ja.wikipedia.org/wiki/共役系 "wikilink")の[分子軌道](../Page/分子軌道.md "wikilink")を用い、原子間行列成分を分子内・分子間ホッピング・[トンネリングパラメータにおきかえたものが用いられている](../Page/トンネル効果.md "wikilink")。これらの導電体のほぼ全ては非常に非等方性が強く、完全に一次元的であると見做せることもある。

## 歴史的背景

1928年までに、[フントの成果に影響された](https://ja.wikipedia.org/wiki/フリードリッヒ・フント "wikilink")[マリケンは](../Page/ロバート・マリケン.md "wikilink")、分子軌道というアイデアを得ていた。B. N. FinklesteinとG. E. Horowitzにより分子軌道を近似する手法としてLCAO法が考案され、同時かつ独立に、固体に対するLCAO法が[ブロッホにより開発され](../Page/フェリックス・ブロッホ.md "wikilink")、彼の1928年の博士論文として発表された。特に[遷移金属のdバンドを近似するために](../Page/遷移元素.md "wikilink")、さらに単純なパラメトライズされたタイトバインディングモデルが1954年に[スレイターとコスターにより提案された](../Page/ジョン・クラーク・スレイター.md "wikilink")\[3\]。これはSKタイトボンディングモデルと呼ばれることもある。このモデルでは、固体の電子バンド構造計算をもともとの[ブロッホの定理](../Page/ブロッホの定理.md "wikilink")ほど厳密に行わず、[ブリュアンゾーン](../Page/ブリュアンゾーン.md "wikilink")の高対称点の計算のみを行って残りの点でのバンド構造は高対称点間の補間により求める。

この手法では、別の原子サイトとの相互作用はとして扱われる。とり入れるべき相互作用として、数種類のものがある。結晶の[ハミルトニアン](../Page/ハミルトニアン.md "wikilink")を各原子のハミルトニアンの和として表わすのはあくまで近似であり、また隣接する原子同士の波動関数は重なりを持つことから、真の波動関数を精度よく表現できるわけではない。 詳細な数学的形式については後述する。

3d[遷移金属電子のように極めて局在化している電子は](../Page/遷移元素.md "wikilink")[強相関と呼ばれる振舞いを示すことがあり](../Page/強相関電子系.md "wikilink")、強相関電子系についての最近の研究には基礎的な近似として強結合近似が用いられる。この場合、電子電子相互作用のふるまいはを用いて記述する必要がある。

強結合近似モデルは静的な[電子バンド構造計算および](../Page/バンド構造.md "wikilink")[バンドギャップ](../Page/バンドギャップ.md "wikilink")計算に用いられることが多いが、[乱雑位相近似](../Page/乱雑位相近似.md "wikilink") (RPA) モデルなどの手法と組み合わせることにより系の動的応答の研究にも用いられることがある。

## 数学的形式

[原子軌道](../Page/原子軌道.md "wikilink") \(\varphi_m( \boldsymbol{r} )\) を単一孤立原子の[ハミルトニアン](../Page/ハミルトニアン.md "wikilink")  の[固有関数](../Page/固有関数.md "wikilink")とする。原子が結晶中にある場合、原子の波動関数は隣接する原子サイトと重なりをもち、したがって原子軌道は結晶ハミルトニアンの真の固有関数にはならない。この近似を「強結合近似」と呼ぶのは、原子サイト間の相互作用は電子が原子により強く結合しているほど弱くなり、この近似が有効となるためである。結晶ハミルトニアン  を得るために必要な原子ポテンシャルへからのずれは全て  で表わされ、かつ微小量と仮定する。

\[H (\boldsymbol{r}) = \sum_{\boldsymbol{R}_n} H_{\mathrm{at}} (\boldsymbol{r} - \boldsymbol{R}_n) + \Delta U (\boldsymbol{r})\] 非時間依存一電子[シュレーディンガー方程式](../Page/シュレーディンガー方程式.md "wikilink")の解 \(\psi_r\) は、原子軌道 \(\varphi_m(\boldsymbol{r} - \boldsymbol{R}_n)\) の[線形結合により以下のように近似される](../Page/LCAO法.md "wikilink")。

\[\psi(\boldsymbol{r}) = \sum_{m,\boldsymbol{R_n}} b_m (\boldsymbol{R}_n) \ \varphi_m (\boldsymbol{r} - \boldsymbol{R}_n)\] ここで  は原子エネルギー準位の添字であり、  は[結晶格子上の原子サイトを表わす](../Page/結晶構造.md "wikilink")。

### 並進対称性と規格化

[ブロッホの定理](../Page/ブロッホの定理.md "wikilink")により、並進によって結晶の波動関数は位相因子分しか変わらない。

\[\psi(\boldsymbol{r} + \boldsymbol{R}_{\ell}) = e^{i \boldsymbol{k} \cdot \boldsymbol{R}_{\ell}} \psi(\boldsymbol{r})\] ここで  は波動関数のである。したがって、上の線形結合の係数は以下の式を満たす。

\[\sum_{m,\boldsymbol{R}_n} b_m (\boldsymbol{R_n}) ~ \varphi_m (\boldsymbol{r} - \boldsymbol{R}_n + \boldsymbol{R}_{\ell}) = e^{i \boldsymbol{k} \cdot \boldsymbol{R}_{\ell}} \sum_{m,\boldsymbol{R}_n} b_m (\boldsymbol{R_n}) ~ \varphi_m (\boldsymbol{r} - \boldsymbol{R}_n)\]

\(\boldsymbol{R}_p= \boldsymbol{R}_n - \boldsymbol{R}_\ell\) のように置き換えると、

\[b_m ( \boldsymbol{R}_p + \boldsymbol{R}_{\ell}) = e^{i \boldsymbol{k} \cdot \boldsymbol{R}_{\ell}} b_m (\boldsymbol{R}_p)\] （ここで右辺はダミー添字 \(\boldsymbol{R}_n\) を \(\boldsymbol{R}_p\) で置き換えてある）
または

\[b_m (\boldsymbol{R}_p) = e^{i \boldsymbol{k} \cdot \boldsymbol{R}_{p}} b_m (\boldsymbol{0})\] 波動関数を[規格化](../Page/規格化.md "wikilink")する。波動関数のノルムは、

\[\begin{align}
\int d^3 r ~ \psi^* (\boldsymbol{r}) \psi (\boldsymbol{r})
&= \sum_{\boldsymbol{R}_n} b^* (\boldsymbol{R}_n) \sum_{\boldsymbol{R}_{\ell}} b(\boldsymbol{R}_{\ell}) \int d^3 r ~ \varphi^* (\boldsymbol{r} - \boldsymbol{R}_n) \varphi (\boldsymbol{r} - \boldsymbol{R}_{\ell}) \\
&= b^*(0) b(0) \sum_{\boldsymbol{R}_n} e^{-i \boldsymbol{k} \cdot \boldsymbol{R}_n} \sum_{\boldsymbol{R}_{\ell}} e^ {i \boldsymbol{k} \cdot \boldsymbol{R}_{\ell}} \int d^3 r ~ \varphi^* (\boldsymbol{r} - \boldsymbol{R}_n) \varphi (\boldsymbol{r} - \boldsymbol{R}_{\ell}) \\
&=N b^*(0) b(0) \sum_{\boldsymbol{R}_p} e^{-i \boldsymbol{k} \cdot \boldsymbol{R}_p} \int d^3 r ~ \varphi^* (\boldsymbol{r} - \boldsymbol{R}_p) \varphi (\boldsymbol{r}) \\
&=N b^*(0) b(0) \sum_{\boldsymbol{R}_p} e^{i \boldsymbol{k} \cdot \boldsymbol{R}_p} \int d^3 r ~ \varphi^* (\boldsymbol{r}) \varphi (\boldsymbol{r} - \boldsymbol{R}_p)
\end{align}\]

よって規格化条件より *b*(0) は次のように定まる。

\[b^*(0) b(0) = \frac{1}{N} \frac{1}{1 + \sum_{\boldsymbol{R}_p \neq \boldsymbol{0}} e^{i \boldsymbol{k} \cdot \boldsymbol{R}_p} \alpha (\boldsymbol{R}_p)}\] α (***R***<sub>p</sub> ) は原子重なり積分で、しばしば無視されて次のように近似される\[4\]。

  -
    \(b_n (0) \approx \frac {1} {\sqrt{N}}\)

すると波動関数は以下のようになる。

\[\psi (\boldsymbol{r}) \approx \frac{1}{\sqrt{N}}  \sum_{m,\boldsymbol{R}_n} e^{i \boldsymbol{k} \cdot \boldsymbol{R}_n} \varphi_m (\boldsymbol{r} - \boldsymbol{R}_n)\]

### 強結合ハミルトニアン

波動関数に強結合近似を適用するとき、番目のエネルギーバンドには番目の原子[エネルギー準位](../Page/エネルギー準位.md "wikilink")のみが重要となり、ブロッホエネルギー \(\varepsilon_m\) は次のような表式となる。

\[\begin{align}
\varepsilon_m &= \int d^3 r ~ \psi^*(\boldsymbol{r}) H(\boldsymbol{r}) \psi (\boldsymbol{r}) \\
&= \sum_{\boldsymbol{R}_n} b^*(\boldsymbol{R}_n) \int d^3 r ~ \varphi^*(\boldsymbol{r} - \boldsymbol{R}_n) H(\boldsymbol{r}) \psi (\boldsymbol{r}) \\
&= \sum_{\boldsymbol{R}_{\ell}}  \sum_{\boldsymbol{R}_n} b^*(\boldsymbol{R}_n) \int d^3 r ~ \varphi^*(\boldsymbol{r} - \boldsymbol{R}_n) H_{\mathrm{at}}(\boldsymbol{r} - \boldsymbol{R}_{\ell}) \psi (\boldsymbol{r}) + \sum_{\boldsymbol{R}_n} b^*(\boldsymbol{R}_n) \int d^3 r ~ \varphi^*(\boldsymbol{r} - \boldsymbol{R}_n) \Delta U (\boldsymbol{r}) \psi(\boldsymbol{r}) \\
&\approx E_m + b^*(0) \sum_{\boldsymbol{R}_n} e^{-i \boldsymbol{k} \cdot \boldsymbol{R}_n} \int d^3 r ~ \varphi^* (\boldsymbol{r} - \boldsymbol{R}_n) \Delta U (\boldsymbol{r}) \psi (\boldsymbol{r})
\end{align}\] さらに、他のサイト上の原子ハミルトニアンを含む項は無視する。するとこのエネルギーは以下のようになる。

\[\begin{align}
\varepsilon_m(\boldsymbol{k}) &= E_m - N |b (0)|^2 \left(\beta_m + \sum_{\boldsymbol{R}_n\neq 0} \sum_l \gamma_{m,l}(\boldsymbol{R}_n) e^{i \boldsymbol{k} \cdot \boldsymbol{R}_n} \right) \\
&= E_m - \frac{\beta_m + \sum_{\boldsymbol{R}_n \neq 0} \sum_l e^{i \boldsymbol{k} \cdot \boldsymbol{R}_n} \gamma_{m,l}(\boldsymbol{R}_n)}{1 + \sum_{\boldsymbol{R}_n \neq 0} \sum_l e^{i \boldsymbol{k} \cdot \boldsymbol{R}_n} \alpha_{m,l} (\boldsymbol{R}_n)}
\end{align}\] ここで、  は  番目の原子準位であり、  は強結合行列要素と呼ばれる。

### 強結合行列要素

行列要素

\[\beta_m = -\int d^3 r ~ \varphi_m^*(\boldsymbol{r}) \Delta U(\boldsymbol{r}) \varphi_m(\boldsymbol{r})\] は隣接する原子のポテンシャルによる原子準位のシフトに由来する。この項はほとんどの場合比較的小さく、もしこれが大きいときは隣接する原子が原子準位に大きな影響を与えることを意味する。

次に、行列要素

\[\gamma_{m,l}(\boldsymbol{R_n}) = -\int d^3 r ~ \varphi_m^*(\boldsymbol{r}) \Delta U(\boldsymbol{r}) \varphi_l(\boldsymbol{r} - \boldsymbol{R}_n)\] は隣接する原子上の原子軌道  と  の間の[原子間行列要素と呼ばれる](https://ja.wikipedia.org/wiki/#原子間行列要素一覧 "wikilink")。結合エネルギー、または二中心積分とも呼ばれ、強結合模型上で**最も重要な行列要素**である。

最後に、行列要素

\[\alpha_{m,l}(\boldsymbol{R_n}) = \int d^3 r ~ \varphi_m^*(\boldsymbol{r}) \varphi_l(\boldsymbol{r} - \boldsymbol{R}_n)\] は隣接する原子上の原子軌道  と  の間の[重なり積分](https://ja.wikipedia.org/wiki/重なり積分 "wikilink")である。

## 行列要素の計算

上述のとおり、隣接原子の作るポテンシャルの中心原子への影響は限られているので、行列要素  はイオン化エネルギーに比してあまり大きくない。もし、  があまり小さくないならば、それは隣接原子の作るポテンシャルの中心原子への影響が小さくないことを意味する。そのような場合、何らかの理由でその系の電子構造には強結合模型があまりよくあてはまらないということである。例えば、原子間距離が近すぎたり、格子上の原子もしくはイオンの電荷が異ったりする場合が挙げられる。

原子間行列要素  は、原子軌道が詳しく分かっているならば直接計算することができる。しかし、ほとんどの場合でこれは不可能である。この行列要素をパラメトライズする方法は数多く存在する。のデータからパラメトライズする方法などが挙げられる。[ブリュアンゾーン](../Page/ブリュアンゾーン.md "wikilink")内の対称性の高い点におけるエネルギーと固有状態を計算し、別途調べたバンド構造と整合するように行列要素の積分内に表われる値を決めることができる。

原子間重なり行列要素  は小さいか、無視できる。この要素が大きいことはやはり強結合近似がうまくあてはまらないことを意味する。大きな重なりはたとえば原子間距離が小さすぎるときなどに見られる。典型金属や遷移金属のブロードなsバンドやspバンドは、第二近傍原子の影響を含めた行列要素および重なり積分を導入することでよりよく現実のバンドを再現することができるが、金属の波動関数を表わすための模型としてはあまり有用だとはいえない。凝集系におけるブロードなバンドは[ほとんど自由な電子](../Page/ほとんど自由な電子.md "wikilink")模型のほうがより良く説明できる。

強結合模型はバンド幅が小さく、電子が強く局在しているdバンドやfバンドの場合に特によい近似となる。また、[ダイヤモンド](../Page/ダイヤモンド.md "wikilink")や[シリコン](https://ja.wikipedia.org/wiki/シリコン "wikilink")などの隣接する原子の少ない結晶構造の場合にもよくあてはまる。この模型とほとんど自由な電子モデルを組み合わせることは簡単にでき、NFE-TBハイブリッド模型と呼ばれる\[5\]。

## ワニエ関数との関連

[ブロッホ関数は周期的結晶格子における電子状態を説明する](../Page/ブロッホの定理.md "wikilink")。ブロッホ関数は次の[フーリエ級数](../Page/フーリエ級数.md "wikilink")により表現される\[6\]。

\[\psi_m(\boldsymbol{k},\boldsymbol{r}) = \frac{1}{\sqrt{N}} \sum_{n} a_m(\boldsymbol{R}_n,\boldsymbol{r}) e^{i\boldsymbol{k}\cdot \boldsymbol{R}_n}\] ここで、  は周期的結晶格子における原子サイト、  はブロッホ波の、  は電子の位置座標、  はバンド添字、そして  個の原子サイトの総和を取るものとする。ブロッホ波は周期的結晶ポテンシャル中の電子についての、エネルギー固有値  に対応する厳密解であり、結晶全体に広がっている。

[フーリエ変換](../Page/フーリエ変換.md "wikilink")を用いて、複数のブロッホ関数から  番目のエネルギーバンドに対応する空間的に局在した波動関数を構築することができる。

\[a_m(\boldsymbol{R}_n,\boldsymbol{r}) = \frac{1}{\sqrt{N}} \sum_{\boldsymbol{k}}{e^{i\boldsymbol{k}\cdot \boldsymbol{R}_n} \psi_m(\boldsymbol{k},\boldsymbol{r})} = \frac{1}{\sqrt{N}} \sum_{\boldsymbol{k}} e^{i\boldsymbol{k}\cdot \boldsymbol{R}_n}\] この実空間上の関数 \({a_m(\boldsymbol{R}_n,\boldsymbol{r})}\) は[ワニエ関数](https://ja.wikipedia.org/wiki/ワニエ関数 "wikilink")と呼ばれ、原子サイト  に強く局在している。もちろん、厳密なワニエ関数が求まれば逆フーリエ変換によりブロッホ関数も求まる。

しかし、ブロッホ関数もワニエ関数も、直接に計算するのは簡単ではない。固体の電子構造を計算するためには、何らかの近似を導入する必要がある。ここで、孤立原子極限を考えれば、ワニエ関数は原子軌道に一致するはずである。この極限からワニエ関数の近似として原子軌道が有効であろうことが示唆され、この近似を強結合近似と呼ぶ。

## 第二量子化

や[ハバード模型のような新しい電子構造理論は](https://ja.wikipedia.org/wiki/ハバードモデル "wikilink")、強結合近似を基礎としている\[7\]。強結合近似を理解するために、[第二量子化](../Page/第二量子化.md "wikilink")表示を用いることができる。

原子軌道を基底状態として用いると、強結合模型における第二量子化されたハミルトニアンは以下のように書ける。

\[H = -t \sum_{\langle i,j \rangle,\sigma} \left( c^{\dagger}_{i,\sigma} c_{j,\sigma} + c_{i,\sigma} c^{\dagger}_{j,\sigma} \right)\]

  -
    \(c^\dagger_{i\sigma} , c_{j\sigma}\) - 生成消滅演算子
    \(\displaystyle\sigma\) - スピン偏極
    \(\displaystyle t\) - ホッピング積分
    \(\displaystyle \langle i,j \rangle\) - 最近傍添字

ここで、ホッピング積分  は強結合模型における移動積分  に相当する。\(t\rightarrow 0\) の極限は電子が隣のサイトに移れないことに相当する。この極限は孤立原子系と一致する。ホッピング項が存在する (\(\displaystyle t>0\)) とき、電子はどちらのサイトにも存在でき、[運動エネルギー](../Page/運動エネルギー.md "wikilink")が下がる。

強相関電子系では、電子電子相互作用を考慮する必要がある。この項は次のように書ける。

\[H_{\mathrm{ee}} = \frac{1}{2} \sum_{n,m,\sigma} \left\langle n_1 m_1, n_2 m_2 \left| \frac{e^2}{|\boldsymbol{r}_1 - \boldsymbol{r}_2|} \right| n_3 m_3, n_4 m_4 \right\rangle c^\dagger_{n_1 m_1 \sigma_1}c^\dagger_{n_2 m_2 \sigma_2}c_{n_4 m_4 \sigma_2} c_{n_3 m_3 \sigma_1}\] このハミルトニアンの相互作用項は直接[クーロン相互作用および](https://ja.wikipedia.org/wiki/クーロンの法則 "wikilink")[交換相互作用](../Page/交換相互作用.md "wikilink")を含む。この項により (MIT) や[高温超伝導](../Page/高温超伝導.md "wikilink")、などの新しい物理が生まれる。

## 例: 一次元sバンド

以下に、強結合模型をs軌道を一つだけ持つ原子が間隔  で直線状に並び、[σ結合](https://ja.wikipedia.org/wiki/σ結合 "wikilink")した**sバンド模型**に適用した例を示す。

ハミルトニアンの近似固有状態を探すため、次のような原子軌道の線形結合を用いる。

  -
    \(|k\rangle =\frac{1}{\sqrt{N}}\sum_{n=1}^N e^{inka} |n\rangle\)

ここで  はサイトの総数、  は \(-\pi/a \le k \le \pi/a\) を満たす実数とする（原子軌道の重なりを無視すれば、この波動関数は規格化定数  により規格化される）。 最近接原子軌道のみが重なりを持つものとすると、ハミルトニアンの非零要素は以下のようになる。

  -
    \(\langle n|H|n\rangle= E_0 = E_i - U\)

<!-- end list -->

  -
    \(\langle n\pm 1|H|n\rangle=-\Delta\)

\[\langle n|n\rangle= 1, \langle n \pm 1|n\rangle= S\]  

エネルギー  は原子軌道に対応するイオン化エネルギーであり、  は隣接する原子の作るポテンシャルによる軌道エネルギーシフトである。\(\langle n\pm 1|H|n\rangle=-\Delta\) という要素は[スレーター・コスター原子間行列要素と呼ばれ](https://ja.wikipedia.org/wiki/#原子間行列要素一覧 "wikilink")、[結合エネルギー](../Page/化学結合.md "wikilink")  と一致する。この一次元sバンド模型ではs軌道同士の\(\sigma\)結合しか存在せず、その結合エネルギーを  とする。隣接原子間の重なり積分は  とする。ここで、状態 \(|k\rangle\) のエネルギーを計算すると次のようになる。

\[\begin{align}
H|k\rangle &= \frac{1}{\sqrt{N}}\sum_n e^{inka} H |n\rangle \\
\langle k|H|k\rangle &= \frac{1}{N}\sum_{m,n} e^{i(n-m)ka} \langle m|H|n\rangle \\
&= \frac{1}{N} \sum_n \langle n|H|n\rangle + \frac{1}{N} \sum_n \langle n-1|H|n\rangle e^{+ika} + \frac{1}{N}\sum_n\langle n+1|H|n\rangle e^{-ika} \\
&= E_0 -2\Delta \cos ka
\end{align}\]

したがってこの状態 \(|k\rangle\) のエネルギーは次のようなよく知られたエネルギー分散を持つ。

\[E(k)= \frac{E_0-2\Delta \cos ka}{1 + 2 S \cos ka}\]

  - \(k = 0\) のときのエネルギーは \(E = (E_0 - 2 \Delta)/ (1 + 2 S)\) となり、波動関数は全ての原子軌道の和となる。この状態は[結合性軌道の連なりと見ることができる](../Page/分子軌道.md "wikilink")。
  - \(k = \pi / 2a\) のときのエネルギーは \(E = E_0\) となり、波動関数は位相因子 \(e^{i \pi / 2}\) のついた原子軌道の和となる。この状態は[非結合性軌道](https://ja.wikipedia.org/wiki/非結合性軌道 "wikilink")の連なりと見ることができる。
  - \(k = \pi / a\) のときのエネルギーは \(E = (E_0 + 2 \Delta) / (1 - 2 S)\) となり、波動関数は原子軌道を交互に足し引きしたものとなる。この状態は[反結合性軌道](https://ja.wikipedia.org/wiki/反結合性軌道 "wikilink")の連なりと見ることができる。

この例はすぐに三次元に拡張することができる。例えば、体心立方格子ならば単純に  の部分を最近接サイトの位置ベクトルに置き換えればよい\[8\]。同様に、各サイトに原子軌道を複数導入すれば複数のバンドを扱うことができる。

## 原子間行列要素一覧

[1954年](../Page/1954年.md "wikilink")、 スレーターとコスターは主に[遷移金属のdバンドについて原子間行列要素の一覧を発表した](../Page/遷移元素.md "wikilink")\[9\]。

\[E_{i,j}(\boldsymbol{r}_{n,n'}) = \langle n,i|H|n',j\rangle\] これは忍耐力と努力があれば [cubic harmonic](https://ja.wikipedia.org/wiki/:en:Cubic_harmonic "wikilink") 軌道から愚直に計算できる。この一覧は二つの隣接する原子上のcubic harmonic 軌道 ,  の間の[LCAO二中心](../Page/LCAO法.md "wikilink")[結合積分を表わしている](../Page/化学結合.md "wikilink")。結合積分は例えば[σ結合](https://ja.wikipedia.org/wiki/σ結合 "wikilink")、[π結合](https://ja.wikipedia.org/wiki/π結合 "wikilink")、[δ結合](https://ja.wikipedia.org/wiki/δ結合 "wikilink")に対してそれぞれ  のように表記する。

原子間ベクトルは次のように表わされる。

\[\boldsymbol{r}_{n,n'} = (r_x,r_y,r_z) = d (l,m,n)\] ここで、  は原子間の距離、 は隣接原子へのである。

  -
    \(E_{s,s} = V_{ss\sigma}\)
    \(E_{s,x} = l V_{sp\sigma}\)
    \(E_{x,x} = l^2 V_{pp\sigma} + (1 - l^2) V_{pp\pi}\)
    \(E_{x,y} = l m V_{pp\sigma} - l m V_{pp\pi}\)
    \(E_{x,z} = l n V_{pp\sigma} - l n V_{pp\pi}\)
    \(E_{s,xy} = \sqrt{3} l m V_{sd\sigma}\)
    \(E_{s,x^2-y^2} = \frac{\sqrt{3}}{2} (l^2 - m^2) V_{sd\sigma}\)
    \(E_{s,3z^2-r^2} = [n^2 - (l^2 + m^2) / 2] V_{sd\sigma}\)
    \(E_{x,xy} = \sqrt{3} l^2 m V_{pd\sigma} + m (1 - 2 l^2) V_{pd\pi}\)
    \(E_{x,yz} = \sqrt{3} l m n V_{pd\sigma} - 2 l m n V_{pd\pi}\)
    \(E_{x,zx} = \sqrt{3} l^2 n V_{pd\sigma} + n (1 - 2 l^2) V_{pd\pi}\)
    \(E_{x,x^2-y^2} = \frac{\sqrt{3}}{2} l (l^2 - m^2) V_{pd\sigma} + l (1 - l^2 + m^2) V_{pd\pi}\)
    \(E_{y,x^2-y^2} = \frac{\sqrt{3}}{2} m(l^2 - m^2) V_{pd\sigma} - m (1 + l^2 - m ^2) V_{pd\pi}\)
    \(E_{z,x^2-y^2} = \frac{\sqrt{3}}{2} n(l^2 - m^2) V_{pd\sigma} - n(l^2 - m^2) V_{pd\pi}\)
    \(E_{x,3z^2-r^2} = l[n^2 - (l^2 + m^2)/2]V_{pd\sigma} - \sqrt{3} l n^2 V_{pd\pi}\)
    \(E_{y,3z^2-r^2} = m [n^2 - (l^2 + m^2) / 2] V_{pd\sigma} - \sqrt{3} m n^2 V_{pd\pi}\)
    \(E_{z,3z^2-r^2} = n [n^2 - (l^2 + m^2) / 2] V_{pd\sigma} + \sqrt{3} n (l^2 + m^2) V_{pd\pi}\)
    \(E_{xy,xy} = 3 l^2 m^2 V_{dd\sigma} + (l^2 + m^2 - 4 l^2 m^2) V_{dd\pi} + (n^2 + l^2 m^2) V_{dd\delta}\)
    \(E_{xy,yz} = 3 l m^2 nV_{dd\sigma} + l n (1 - 4 m^2) V_{dd\pi} + l n (m^2 - 1) V_{dd\delta}\)
    \(E_{xy,zx} = 3 l^2 m n V_{dd\sigma} + m n (1 - 4 l^2) V_{dd\pi} + m n (l^2 - 1) V_{dd\delta}\)
    \(E_{xy,x^2-y^2} = \frac{3}{2} l m (l^2 - m^2) V_{dd\sigma} + 2 l m (m^2 - l^2) V_{dd\pi} + [l m (l^2 - m^2) / 2] V_{dd\delta}\)
    \(E_{yz,x^2-y^2} = \frac{3}{2} m n (l^2 - m^2) V_{dd\sigma} - m n [1 + 2(l^2 - m^2)] V_{dd\pi} + m n [1 + (l^2 - m^2) / 2] V_{dd\delta}\)
    \(E_{zx,x^2-y^2} = \frac{3}{2} n l (l^2 - m^2) V_{dd\sigma} + n l [1 - 2(l^2 - m^2)] V_{dd\pi} - n l [1 - (l^2 - m^2) / 2] V_{dd\delta}\)
    \(E_{xy,3z^2-r^2} = \sqrt{3} \left[ l m (n^2 - (l^2 + m^2) / 2) ]V_{dd\sigma} - 2 l m n^2 V_{dd\pi} + [l m (1 + n^2) / 2] V_{dd\delta} \right]\)
    \(E_{yz,3z^2-r^2} = \sqrt{3} \left[ m n (n^2 - (l^2 + m^2) / 2) V_{dd\sigma} + m n (l^2 + m^2 - n^2) V_{dd\pi} -[ m n (l^2 + m^2) / 2 ]V_{dd\delta} \right]\)
    \(E_{zx,3z^2-r^2} = \sqrt{3} \left[ l n (n^2 - (l^2 + m^2) / 2) V_{dd\sigma} + l n (l^2 + m^2 - n^2) V_{dd\pi} - [l n (l^2 + m^2) / 2] V_{dd\delta} \right]\)
    \(E_{x^2-y^2,x^2-y^2} = \frac{3}{4} (l^2 - m^2)^2 V_{dd\sigma} + [l^2 + m^2 - (l^2 - m^2)^2] V_{dd\pi} + [n^2 + (l^2 - m^2)^2 / 4] V_{dd\delta}\)
    \(E_{x^2-y^2,3z^2-r^2} = \sqrt{3} \left[ (l^2 - m^2) [n^2 - (l^2 + m^2) / 2] V_{dd\sigma} / 2 + n^2 (m^2 - l^2) V_{dd\pi} + [(1 + n^2)(l^2 - m^2) / 4 ]V_{dd\delta}\right]\)
    \(E_{3z^2-r^2,3z^2-r^2} = [n^2 - (l^2 + m^2) / 2]^2 V_{dd\sigma} + 3 n^2 (l^2 + m^2) V_{dd\pi} + \frac{3}{4} (l^2 + m^2)^2 V_{dd\delta}\)

ここに示さなかった行列成分もあるが、それらはここに示した行列成分の添字と方向余弦を並べ変えれば得られる。
