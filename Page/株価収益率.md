## 概要

株価収益率は株価を[一株当たり当期純利益](../Page/一株当たり当期純利益.md "wikilink")で割ったものであり、次の式で求められる。

|       |   |    |   |            |
| ----- | - | -- | - | ---------- |
| 株価収益率 | ＝ | 株価 | ÷ | 一株当たり当期純利益 |

[株主](../Page/株主.md "wikilink")の側から見れば、「[利益](../Page/利益.md "wikilink")が全て[配当](../Page/配当.md "wikilink")に回された場合に何年で元本を回収できるか」という指標として見ることができる。一方[企業](../Page/企業.md "wikilink")の側から見れば、「株主からの出資をどれくらいの利回りで運用しているか」という指標の逆数と見ることができる。

また株価収益率は[時価総額](../Page/時価総額.md "wikilink")を[当期純利益](https://ja.wikipedia.org/wiki/当期純利益 "wikilink")で割ったものと言い換えることもでき、次の式で求められる。

|       |   |                                    |   |                                                         |
| ----- | - | ---------------------------------- | - | ------------------------------------------------------- |
| 株価収益率 | ＝ | 株価                                 | ÷ | 一株当たり当期純利益                                              |
|       | ＝ | （株価×発行済株式数）                        | ÷ | （一株当たり当期純利益×発行済株式数）                                     |
|       | ＝ | [時価総額](../Page/時価総額.md "wikilink") | ÷ | [当期純利益](https://ja.wikipedia.org/wiki/当期純利益 "wikilink") |

利益が減ると、株価収益率は増加することとなる。一般に株価収益率が業界平均値と比較して高いときは、当該企業の株価は割高とされる。

## 用いられ方

株価収益率には[決算](../Page/決算.md "wikilink")により確定した純利益を元に算出される数値（前期実績PER）と期末で予想される純利益を元に算出される数値（予想PER）があり、[株式市場](https://ja.wikipedia.org/wiki/株式市場 "wikilink")では一般に予想PERが重視される。予想PERは当該企業が半期・四半期毎に公表する決算予測をもとに算定されることが多く、その的中性や信憑性に対するスタンスの違いにより利用には慎重さが求められる。一般的には予想PERの算定根拠として[アナリストの予測が採用されることが多い](../Page/証券アナリスト.md "wikilink")。

株価収益率の逆数が[益回り](https://ja.wikipedia.org/wiki/益回り "wikilink")である。株価水準がPER＝20にあるとき、その時点での益回りは1/20（5パーセント）となる。また益回りに[配当性向](https://ja.wikipedia.org/wiki/配当性向 "wikilink")を乗したものが[配当利回り](https://ja.wikipedia.org/wiki/配当利回り "wikilink")である。益回り、配当利回りについても過去実績値による計算と予測値によるものがあり、算定の根拠には注意が必要である。

## 適正とされる値

株価収益率の標準値は14から20の間 (≒益回り5 - 7%)が適正とされるが当該企業の成長性に楽観的な場合は高PERまで買われ（利益の高成長が見込める場合、現行では高いPERも数年後には低い水準になることがある）、将来に不透明感が高い場合は低PERで取引される。また[石油](https://ja.wikipedia.org/wiki/石油 "wikilink")や[鉄鋼](https://ja.wikipedia.org/wiki/鉄鋼 "wikilink")、[海運](../Page/海運.md "wikilink")など国際商品市況に業績が大きく影響をうける業種は過渡的な経緯や国際的な落ち着き所としてセクター全体が低PERで取引されていることがある。[不動産](../Page/不動産.md "wikilink")に対する投資収益率（不動産価格/年間賃料）が構造的に20近辺であり（地価の変動と賃料相場が連動する特性があることから）、[証券市場](../Page/証券市場.md "wikilink")全体の株価収益率が20を越えて買い上げられている場合は[株価バブル](https://ja.wikipedia.org/wiki/株価バブル "wikilink")を警戒する必要がある（個別銘柄では前述のように将来性を期待して買い進められたり、また（予想）純利益が下方修正されることで高PERとなることがある）。

[ニューヨーク証券取引所](../Page/ニューヨーク証券取引所.md "wikilink")では伝統的に14 - 20程度をコアとした株価収益率の推移を示してきたが、日本では[バブル景気](../Page/バブル景気.md "wikilink")崩壊後に株式の相互持合いが解消される[1990年代](../Page/1990年代.md "wikilink")頃までは40～60内外の相当割高な株価で推移してきた。また[新興市場](https://ja.wikipedia.org/wiki/新興市場 "wikilink")では成長性を期待した取引が中心となることから、[NASDAQ](../Page/NASDAQ.md "wikilink")では60 - 80程度 (≒益回り1.25 - 1.7%)をコアとした株価収益率の推移が見られる。市場間、あるいは[投資](../Page/投資.md "wikilink")対象（企業・[債券](../Page/債券.md "wikilink")・不動産・[商品先物](https://ja.wikipedia.org/wiki/商品先物 "wikilink")等）間での[投資収益率](https://ja.wikipedia.org/wiki/投資収益率 "wikilink")の大幅な違いは価格変動の大きな波乱要素であり投資対象、たとえば企業収益などへの期待が正・負いずれかの方向に大きく裏切られることなどをきっかけに急激な[相場](https://ja.wikipedia.org/wiki/相場 "wikilink")変動や[市場](../Page/市場.md "wikilink")の混乱をもたらす要因となる。
