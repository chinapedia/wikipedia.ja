**ブルース**\[1\]（Blues）は、[米国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")[深南部で](https://ja.wikipedia.org/wiki/ディープサウス "wikilink")[アフリカ系アメリカ人](https://ja.wikipedia.org/wiki/アフリカ系アメリカ人 "wikilink")の間から発生した[音楽](../Page/音楽.md "wikilink")の一種およびその[楽式](https://ja.wikipedia.org/wiki/楽式 "wikilink")。[19世紀](../Page/19世紀.md "wikilink")後半頃に米国深南部で[黒人霊歌](https://ja.wikipedia.org/wiki/霊歌 "wikilink")、フィールドハラー（農作業の際の叫び声）や、ワーク・ソング（[労働歌](https://ja.wikipedia.org/wiki/労働歌 "wikilink")）などから発展したものといわれている。兄弟とも言える[ジャズ](../Page/ジャズ.md "wikilink")が楽器による演奏が主役なのに対して、ギターを用いた歌が主役である \[2\]。[アコースティック・ギター](https://ja.wikipedia.org/wiki/アコースティック・ギター "wikilink")の弾き語りを基本とした[デルタ・ブルース](https://ja.wikipedia.org/wiki/デルタ・ブルース "wikilink")\[3\]、[カントリー・ブルース](https://ja.wikipedia.org/wiki/カントリー・ブルース "wikilink")\[4\]、エレクトリック・ギターを使用した[バンド形式に発展した](https://ja.wikipedia.org/wiki/バンド_\(音楽\) "wikilink")[シカゴ・ブルース](https://ja.wikipedia.org/wiki/シカゴ・ブルース "wikilink")\[5\]など多様に展開している。

## 概要：音楽の特徴

「ブルース」とは、孤独感や悲しみを表現する独唱歌であり、悲しみや孤独の感情は、[英語](../Page/英語.md "wikilink")ではしばしば「[ブルー](https://ja.wikipedia.org/wiki/青 "wikilink")（*blue*）」の色でたとえられることに由来している\[6\]。[20世紀](../Page/20世紀.md "wikilink")以降の[ポピュラー音楽](https://ja.wikipedia.org/wiki/ポピュラー音楽 "wikilink")に幅広く影響を与え、[ジャズ](../Page/ジャズ.md "wikilink")や[ロックンロール](../Page/ロックンロール.md "wikilink")のルーツのひとつとしても知られている。

ブルースの基本的な構成として、12[小節](https://ja.wikipedia.org/wiki/小節 "wikilink")形式（[ブルース形式](https://ja.wikipedia.org/wiki/ブルース形式 "wikilink")）で綴られる詩が多い。12小節形式の基本はA・A・Bの形式をとる。つまり、4小節の同じ歌詞を二度繰り返し、最後の4小節で締めの歌詞を歌う。これがワンコーラスとなる。

ブルース形式（12小節形式）のコード進行

|    |          |   |         |
| :-: | :------: | :-: | :-----: |
| I  | I または IV | I |    I    |
| IV |    IV    | I |    I    |
| V  | IV または V | I | I または V |

これらのローマ数字は、[コード度数を表している](https://ja.wikipedia.org/wiki/和音 "wikilink")。これが例えばキーがC (ハ長調)だとすると以下の通りとなる。

|   |         |   |         |
| :-: | :-----: | :-: | :-----: |
| C | C または F | C |    C    |
| F |    F    | C |    C    |
| G | F または G | C | C または G |

歌詞は、身近な出来事、感情を表現したものが多い。日常の幸せなことや憂鬱なこと(blues)を12小節に乗せて歌う。[アメリカ南部の黒人たちにとって身近な存在だったギターは](https://ja.wikipedia.org/wiki/アメリカ合衆国南部 "wikilink")、伴奏楽器として適していたこともあり、初期のブルースはギターの弾き語りによるものが多かった。

[thumb](https://ja.wikipedia.org/wiki/ファイル:A_minor_pentatonic_scale.svg "wikilink") [ペンタトニックスケール](https://ja.wikipedia.org/wiki/五音音階 "wikilink")\]\] 旋律に独特の節回しがあり、一般に[ブルー・ノート・スケール](https://ja.wikipedia.org/wiki/ブルー・ノート・スケール "wikilink")と呼ばれている5音階（ペンタトニック・スケール）で[即興](https://ja.wikipedia.org/wiki/即興 "wikilink")的に演奏される。特に短3度、減5度、短7度の音に用いられる微妙な音の「訛り」はクオーターと呼ばれ、ブルース独特の「音」であり、カントリーやジャズには基本的にみられない音である。[ギター](https://ja.wikipedia.org/wiki/ギター "wikilink")では、「クオーター・チョーキング」=1/4音上げで演奏されることが多いが、厳密には1/4音と決まっているわけではなく、人それぞれ感覚的に上げて使っている。[ピアノ](../Page/ピアノ.md "wikilink")ではこの音を出すのが不可能のため、3度の音の場合は、長短の二つの鍵盤を[トリル](https://ja.wikipedia.org/wiki/トリル "wikilink")したり同時に打鍵したりなどのテクニックを用いる[奏者もいる](https://ja.wikipedia.org/wiki/演奏者 "wikilink")。また[1950年代](../Page/1950年代.md "wikilink")からはジャズの影響によって、（[ブルー・ノート・スケール](https://ja.wikipedia.org/wiki/ブルー・ノート・スケール "wikilink")5音だけではない）[テンション・ノート](https://ja.wikipedia.org/wiki/テンション・ノート "wikilink")や[テンション・コード](https://ja.wikipedia.org/wiki/テンション・コード "wikilink")が用いられるなど、洗練さを加えてきた。

[シャッフルと呼ばれる](https://ja.wikipedia.org/wiki/シャッフル_\(音楽\) "wikilink")、はねるリズムパターンが主流である。譜面上は便宜上、12ビート（12/8）とするか、はねた8ビート（8/8）の1拍3連の2拍目を休符にするか、1拍目と2拍目をつないだ形で表記されることが多いが、実際には均等に3等分したものではない。人によって微妙にリズムが異なり、ほぼ3連符の場合もあれば、16ビートに近いくらい後拍が遅れたもの（4連の16分音符の2拍目と3拍目を休符にしたもの）や、8分音符に近いものもある。ジャズの「スウィング」と似た部分もあるが、独特な後拍の強調があり同じではない。

ブルース形式でなくとも、タイトルに「〜ブルース」と付く曲も多く存在する。

ちなみに日本の歌謡曲の一つのスタイルとしてブルースと呼ばれるものもあるが、この項目のブルースとは音楽性は全く違い、[メロディ](https://ja.wikipedia.org/wiki/メロディ "wikilink")ーや[アレンジは](../Page/編曲.md "wikilink")[ムード歌謡](https://ja.wikipedia.org/wiki/ムード歌謡 "wikilink")の一種で、「憂鬱（Blue）な気持ちを歌った曲」という意味合いが強い。（[青江三奈](https://ja.wikipedia.org/wiki/青江三奈 "wikilink")「[伊勢佐木町ブルース](https://ja.wikipedia.org/wiki/伊勢佐木町ブルース "wikilink")」、[美川憲一](https://ja.wikipedia.org/wiki/美川憲一 "wikilink")「[柳ヶ瀬ブルース](https://ja.wikipedia.org/wiki/柳ヶ瀬ブルース "wikilink")」など）

## 歴史

### アメリカのブルース史

[250px](https://ja.wikipedia.org/wiki/ファイル:W._C._Handy_-_The_"St._Louis_Blues"_-_First_page.jpg "wikilink")"の譜面(1914)\]\] [19世紀](../Page/19世紀.md "wikilink")後半頃に[米国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")[深南部で](https://ja.wikipedia.org/wiki/ディープサウス "wikilink")[黒人霊歌](https://ja.wikipedia.org/wiki/霊歌 "wikilink")、フィールドハラー（労働歌）などから発展したものと言われている\[7\]。

[1903年](../Page/1903年.md "wikilink")、[ミシシッピ州](https://ja.wikipedia.org/wiki/ミシシッピ州 "wikilink")の[デルタ地帯](https://ja.wikipedia.org/wiki/デルタ地帯 "wikilink")を旅行中だった白人、が同州[タトワイラー](https://ja.wikipedia.org/wiki/タトワイラー "wikilink")で、黒人によるブルースの生演奏に遭遇。この後、彼は楽曲を楽譜にして発表し、ブルースは広く世間に知られることになった。だが、ハンディは、ブルースを楽譜におこしただけであり、ブルースの父というのは過大な評価と言える。この年を「ブルースの生誕の年」とする見方もあり、[2003年](../Page/2003年.md "wikilink")はブルース生誕100年を記念して[アメリカ合衆国議会](https://ja.wikipedia.org/wiki/アメリカ合衆国議会 "wikilink")により、「ブルースの年」と宣言された。\[8\]

[1920年](../Page/1920年.md "wikilink")、[メイミー・スミス](https://ja.wikipedia.org/wiki/メイミー・スミス "wikilink")がオーケー・レーベルに初レコーディング。これがブルースのレコーディングとしては初と言われている。彼女の"Crazy Blues"は、初年度75,000枚を売り上げるヒットを記録した。\[9\]また現在、知名度の高い戦前ブルース・シンガーはロバート・ジョンソン\[10\]だが、当時はチャーリー・パットン\[11\]の方が、黒人の間での人気が高かった。

戦前のアメリカにおいて、ブルースは[米国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")[深南部から](https://ja.wikipedia.org/wiki/ディープサウス "wikilink")[セントルイス](https://ja.wikipedia.org/wiki/セントルイス "wikilink")、[シカゴ](../Page/シカゴ.md "wikilink")、[ニューヨーク](https://ja.wikipedia.org/wiki/ニューヨーク "wikilink")などへ北上し、各地でスタイルを変えながら発展した。元々ギターの弾き語り中心であったが、都市部に展開するにつれ、ピアノとギターのデュオ形式、バンド形式など、より都会的な洗練された形式へと変わって行った。都市部で展開されたブルースのスタイルをシティ・ブルースという。代表的なミュージシャンは、[リロイ・カー](https://ja.wikipedia.org/wiki/リロイ・カー "wikilink")など。しかし都会にあこがれる反面、故郷への想いが強く詩に影響を与えている歌が多い。\[12\]

シカゴでは、[1950年](../Page/1950年.md "wikilink")頃からエレクトリックのバンドによるブルースが登場した。[デルタ・ブルース](https://ja.wikipedia.org/wiki/デルタ・ブルース "wikilink")を基調とした泥臭いサウンドで、戦前のシティ・ブルースとは一線を画すものであった。このサウンドはシカゴ・ブルースと呼ばれるようになった。その代表格となるのが、[マディ・ウォーターズ](https://ja.wikipedia.org/wiki/マディ・ウォーターズ "wikilink")である。ロックンロールの巨匠、[チャック・ベリー](https://ja.wikipedia.org/wiki/チャック・ベリー "wikilink")もこの頃のブルースに大きく影響を受け、後のロックバンドにも受け継がれているといえる。\[13\]

[1960年代](../Page/1960年代.md "wikilink")初めになると、[イギリス](https://ja.wikipedia.org/wiki/イギリス "wikilink")でアメリカから多くのブルースのレコードが輸入され、同国でブルースのブームが起きた。その流れの中で、[ローリング・ストーンズ](https://ja.wikipedia.org/wiki/ローリング・ストーンズ "wikilink")、[フリートウッド・マック](https://ja.wikipedia.org/wiki/フリートウッド・マック "wikilink")、クリーム、ジェスロ・タル（初期）、アニマルズなど、ブルースに影響を受けたバンドが多く登場し、ブルース・ロックが隆盛となった。\[14\]

### 日本のブルース・シーン

日本では、1970年代にブルース・ブームが起こった。 [1971年](https://ja.wikipedia.org/wiki/1971年 "wikilink")、[B.B.キング](https://ja.wikipedia.org/wiki/B.B.キング "wikilink")が初来日を果たす。[1973年](../Page/1973年.md "wikilink")に[スリーピー・ジョン・エスティス](https://ja.wikipedia.org/wiki/スリーピー・ジョン・エスティス "wikilink")の「スリーピー・ジョン・エスティスの伝説(The Legend of Sleepy John Estes)」がオリコン・チャートに食い込む大ヒットとなる。

[1974年](../Page/1974年.md "wikilink")、「第1回ブルース・フェスティバル」開催。同フェスティバルは第3回まで開催され、エスティスを始め[ロバート・ロックウッド・ジュニア](https://ja.wikipedia.org/wiki/ロバート・ロックウッド・ジュニア "wikilink")&[エイセズ](https://ja.wikipedia.org/wiki/エイセズ "wikilink")、[オーティス・ラッシュ](https://ja.wikipedia.org/wiki/オーティス・ラッシュ "wikilink")らの来日が実現した。

京都、大阪を中心に[ウエスト・ロード・ブルース・バンド](https://ja.wikipedia.org/wiki/ウエスト・ロード・ブルース・バンド "wikilink")、[憂歌団](https://ja.wikipedia.org/wiki/憂歌団 "wikilink")など、ブルース・バンドが登場。日本の独自のブルース・シーンが形成されて行く。
