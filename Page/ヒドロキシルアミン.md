| Section8 =  }} **ヒドロキシルアミン** (hydroxylamine) は示性式が NH<sub>2</sub>OH と表される[無機化合物](https://ja.wikipedia.org/wiki/無機化合物 "wikilink")である。[水](../Page/水.md "wikilink")と[アンモニア](../Page/アンモニア.md "wikilink")が互いに一部分を共有したような構造を持っているので、それらの混成体と見ることもできる。純粋なヒドロキシルアミンは室温で不安定な結晶性の固体であり、吸湿性を持つ。[潮解](https://ja.wikipedia.org/wiki/潮解 "wikilink")性がある。一般的に[水溶液](../Page/水溶液.md "wikilink")、または塩酸塩などの[塩として取り扱われる](../Page/塩_\(化学\).md "wikilink")。

ヒドロキシルアミンは生合成的[硝化](https://ja.wikipedia.org/wiki/硝化 "wikilink")の中間体である。アンモニアの酸化はヒドロキシルアミン酸化還元酵素によって媒介される。

## 生産

いくつかの合成法が知られている\[2\]。

ラシヒ法 (Raschig Synthesis) では、まず[亜硝酸アンモニウム](https://ja.wikipedia.org/wiki/亜硝酸アンモニウム "wikilink")水溶液を 0 {{℃}} において <chem>HSO4^- /SO2</chem> で還元し、ヒドロキシルアミド-*N*,*N*-ジスルフェートとする。これを[加水分解](../Page/加水分解.md "wikilink")して[硫酸](https://ja.wikipedia.org/wiki/硫酸 "wikilink")塩 (NH3OH)2SO4 を得る。

固体のヒドロキシルアミンはこの硫酸塩を液体アンモニアで処理することによって得られる。[硫酸アンモニウム](../Page/硫酸アンモニウム.md "wikilink")は液体アンモニアに不溶なのでろ別でき、アンモニアは減圧下に留去される。

他の合成法として、ヒドロキシルアンモニウム塩を経るものがある。[亜硝酸](../Page/亜硝酸.md "wikilink")または[亜硝酸ナトリウム](../Page/亜硝酸ナトリウム.md "wikilink")を亜硫酸イオンで還元する。生成したヒドロキシルアミド-*N*-スルフェートを加水分解し、ヒドロキシルアンモニウム塩としたのち、ナトリウムブトキシドで中和し遊離のヒドロキシルアミンを得る。

## 反応

アルキル化剤のような求電子試薬と反応する。酸素と窒素はどちらも攻撃を受ける。

  -
    <chem>R-X\\ + NH2OH -\> R-ONH2\\ + HX</chem>
    <chem>R-X\\ + NH2OH -\> R-NHOH\\ + HX</chem>

[アルデヒド](../Page/アルデヒド.md "wikilink")や[ケトン](../Page/ケトン.md "wikilink")との反応では[オキシム](../Page/オキシム.md "wikilink")が生成する。

  -
    <chem>R2C=O\\ + NH2OH \\cdot HCl, NaOH -\> R2C=NOH\\ + NaCl\\ + H2O</chem>

この反応はケトン、アルデヒドの精製に有用である。またオキシム類は[ジメチルグリオキシム](https://ja.wikipedia.org/wiki/ジメチルグリオキシム "wikilink")のような配位子としても使われる。

クロロ硫酸と反応してヒドロキシルアミン-*O*-スルホン酸を与える。これは[カプロラクタム](https://ja.wikipedia.org/wiki/カプロラクタム "wikilink")の合成に用いられる試薬である。

  -
    <chem>HOSO2Cl\\ + NH2OH -\> NH2OSO2OH\\ + HCl</chem>

ヒドロキシルアミン-*O*-スルホン酸は 0 ℃ 以下で保存する必要があり、ヨウ素滴定で確認できる。

## 利用

ヒドロキシルアミンおよびその塩類は多くの[有機化学](../Page/有機化学.md "wikilink")、[無機化学](../Page/無機化学.md "wikilink")反応において[還元剤](../Page/還元剤.md "wikilink")として一般的に用いられる。[脂肪酸](../Page/脂肪酸.md "wikilink")の[酸化防止剤](../Page/酸化防止剤.md "wikilink")としての作用もある。化学以外での利用法としては、[獣皮](https://ja.wikipedia.org/wiki/獣皮 "wikilink")の[脱毛](../Page/脱毛.md "wikilink")や写真の現像液などがある\[3\]。[半導体](../Page/半導体.md "wikilink")の洗浄剤として利用される。[農薬](../Page/農薬.md "wikilink")の原料にもなる。

[硝酸ヒドロキシルアンモニウム](https://ja.wikipedia.org/wiki/硝酸ヒドロキシルアンモニウム "wikilink")は[ロケットの推進剤](https://ja.wikipedia.org/wiki/ロケットの推進剤 "wikilink")として、一液推進薬（水溶液）、[固体燃料](https://ja.wikipedia.org/wiki/固体燃料 "wikilink")の両方が研究されている。

## 安全性

ヒドロキシルアミンは爆発性の化合物であるが、その危険性の度合いについては完全にはわかっていない。火気や高温体に触れると、爆発的に燃焼する。[紫外線](../Page/紫外線.md "wikilink")を受けると爆発する。1999年以来、ヒドロキシルアミンを扱う工場での死者を伴う事故が何度か起こっている。

[鉄](../Page/鉄.md "wikilink")(II), (III) イオンによって 50 % ヒドロキシルアミン溶液の分解が加速されることが知られている。ヒドロキシルアミンとその誘導体は塩の形で安全に取り扱うことができる。

呼吸器、皮膚、目、そして他の粘膜を刺激する。皮膚から吸収される可能性があり、飲み込んだ場合も有害であり、変異誘発物質である可能性がある。蒸気が大量に体内に入ると、血液の酸素吸収力が低下し、死亡することがある。
