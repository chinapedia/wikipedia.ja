**速度定数**（そくどていすう）は[化学反応](https://ja.wikipedia.org/wiki/化学反応 "wikilink")において生成物または反応物が増減する速さを表す量である。

反応速度に関する全般的な理論については[反応速度論](../Page/反応速度論.md "wikilink")を、定式化の詳細は[反応速度](../Page/反応速度.md "wikilink")を参照のこと。

## 速度則

[速度式または速度則は反応速度の数学的表現であり](../Page/反応速度式.md "wikilink")、反応速度論において反応速度と基質の濃度を次数を使って関連付けるのに用いられる。例えば、化学反応  については以下のように書き表される。

液体、気体、または[溶液](../Page/溶液.md "wikilink")中での反応について、\[X\] は 基質 X の[濃度](https://ja.wikipedia.org/wiki/濃度 "wikilink")を示し、[界面](../Page/界面.md "wikilink")上での反応の場合では単位面積あたりの X の量を示す。気体反応ならば[圧力](../Page/圧力.md "wikilink")で記述してもよい。

上の式で、*k* (*T* ) が反応速度の係数、すなわち速度定数である。[温度](../Page/温度.md "wikilink")*T* 、[イオン強度](../Page/イオン強度.md "wikilink")、光照射など、基質の濃度以外のさまざまな要因によって変化するため、真の意味で定数（一定不変）ということではない。

速度式の指数 ''n' '' と ''m' '' は反応の次数と呼ばれ、反応機構によって値が変わる。

## 温度依存性

反応速度の係数（*k*<sub>1</sub>, *k*<sub>2</sub> など）は温度に依存し、一般的に次に示す[アレニウスの式](../Page/アレニウスの式.md "wikilink")によって与えられる。

ここで *E<sub>a</sub>* は[活性化エネルギー](../Page/活性化エネルギー.md "wikilink")、*R* は[気体定数](../Page/気体定数.md "wikilink")である。温度 *T* において分子は[ボルツマン分布に基づくエネルギーを持つことから](https://ja.wikipedia.org/wiki/マクスウェル=ボルツマン分布 "wikilink")、*E<sub>a</sub>* を超えるエネルギーが供給される衝突の回数は exp (-*E*<sub>a</sub> /*RT* ) に比例することがわかる。*A* は頻度因子または前指数項と呼ばれる。

*A* や *E<sub>a</sub>* の値は反応によって異なる（よって、例えば *k*<sub>1</sub> と *k*<sub>2</sub> は異なる値をとる）。アレニウスの式は[経験式](https://ja.wikipedia.org/wiki/経験式 "wikilink")である\[1\]ため、より複雑な、この形に従わない、他の速度定数の温度依存性を記述する式も存在する。たとえば、分子衝突説、遷移状態説などの理論によれば、指数 0 ≤ *m* ≤ 1 を用いて次のように表される\[2\]。

\[k \propto T^m \exp\left(-\frac{E_a}{RT}\right)\] ただし一般に *m* \<\< *E* /*RT* であるため、アレニウスの式が成り立つとみなしてよいとされる。

## 単位

速度定数*k* の[次元および](https://ja.wikipedia.org/wiki/量の次元 "wikilink")[単位](../Page/単位.md "wikilink")は速度式に依存する\[3\]。*n* 次反応に対して速度定数の次元は

  -
    \[濃度\]<sup>1-*n*</sup> \[時間\]<sup>-1</sup>

となる。濃度の単位を mol/l、時間をsで表すと、速度定数の単位はたとえば1次反応ではs<sup>-1</sup>に、2次反応ではl/(mol·s)となる。
