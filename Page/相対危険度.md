**相対危険度**（そうたいきけんど）とは[疫学](https://ja.wikipedia.org/wiki/疫学 "wikilink")における指標の1つで、「相対リスク」とも呼ばれ、[暴露](https://ja.wikipedia.org/wiki/暴露 "wikilink")群と非暴露群における[疾病](https://ja.wikipedia.org/wiki/疾病 "wikilink")の頻度を比で表現したもの。そのまま比率として表すが、百分率で表す場合もある。相対危険度は暴露群の発生率を非暴露群の発生率で割ることにより求めることができ、暴露因子と疾病発生との関連の強さを示す指標となる。主に[コホート研究](https://ja.wikipedia.org/wiki/コホート研究 "wikilink")で用いられる。

|      | 疾病あり | 疾病なし | 計   |
| ---- | ---- | ---- | --- |
| 暴露あり | A    | B    | A+B |
| 暴露なし | C    | D    | C+D |
| 計    | A+C  | B+D  | T   |

疾病と暴露の比較

\(R = \cfrac{\cfrac{A}{A+B}}{\cfrac{C}{C+D}}\)

R:相対危険度

「相対危険度」が1（100％）であれば暴露群も非暴露群も発生率は等しいが、「相対危険度」が1（100％）より大きければ暴露群の方が非暴露群よりも発生率が高くなり、「相対危険度」が1（100％）よりも小さければ暴露の方が非暴露群よりも発生率が低くなる。

「相対危険度」が1を上回る場合は「過剰相対危険度（余剰リスク，相対リスク増加）」、「相対危険度」が1を下回る場合は「相対リスク減少」を求められる。

暴露群の発生率ではなく、集団全体の発生率を用いたリスクの比は、「人口相対危険度」となる。
