**宇宙の加速膨張**()とは、[宇宙](../Page/宇宙.md "wikilink")の膨張が加速しているように観測されることである。[1998年](https://ja.wikipedia.org/wiki/1998年 "wikilink")に、[Ia型超新星](https://ja.wikipedia.org/wiki/Ia型超新星 "wikilink")の観測によって、宇宙の膨張が加速しているのではないかとの疑問が浮かび上がった。\[1\]\[2\]

## 観測的証拠

現在までに宇宙の加速膨張は以下に述べる複数の独立な観測からその証拠が得られている。

### Ia型超新星

[Ia型超新星](https://ja.wikipedia.org/wiki/Ia型超新星 "wikilink")は[白色矮星](../Page/白色矮星.md "wikilink")の質量が[チャンドラセカール限界](../Page/チャンドラセカール限界.md "wikilink")を超えたときに発生する爆発現象で、極めて明るく、かつ（適切な[較正](../Page/較正.md "wikilink")のもとで）[光度](https://ja.wikipedia.org/wiki/光度 "wikilink")が常に一定であると考えられている。このためIa型超新星は宇宙の[標準光源](https://ja.wikipedia.org/wiki/標準光源 "wikilink")として理想的な対象であり、高[赤方偏移](../Page/赤方偏移.md "wikilink")宇宙でのIa型超新星の[見かけの等級](https://ja.wikipedia.org/wiki/見かけの等級 "wikilink")と赤方偏移を比較することで、その[光度距離](https://ja.wikipedia.org/wiki/光度距離 "wikilink")の赤方偏移依存性を測定することができる\[3\]。これは[平坦な宇宙モデルのもとで赤方偏移](https://ja.wikipedia.org/wiki/宇宙の形 "wikilink") \(z\) が1より小さいとき

\[d_L ( z ) = \frac{ c z }{ H_0 } \left[ 1 + \frac{ 1 }{ 2 } ( 1 - q_0 ) z + \mathcal{O} ( z^2 ) \right]\] という形で[減速パラメータ](https://ja.wikipedia.org/wiki/減速パラメータ "wikilink") \(q_0\) に依存する\[4\]から、Ia型超新星の観測によって減速パラメータを決定することができる。[1998年](https://ja.wikipedia.org/wiki/1998年 "wikilink")に[ハイゼット超新星探索チーム](https://ja.wikipedia.org/wiki/ハイゼット超新星探索チーム "wikilink")\[5\]と[超新星宇宙論計画](https://ja.wikipedia.org/wiki/超新星宇宙論計画 "wikilink")\[6\]は独立に遠方のIa型超新星の観測を行い、減速膨張 (\(q_0 > 0\)) が棄却されることを示した。なおこの業績で[ソール・パールマッター](https://ja.wikipedia.org/wiki/ソール・パールマッター "wikilink")、[ブライアン・P・シュミット](https://ja.wikipedia.org/wiki/ブライアン・P・シュミット "wikilink")、[アダム・リースは](https://ja.wikipedia.org/wiki/アダム・リース_\(天体物理学者\) "wikilink")[2011年](../Page/2011年.md "wikilink")の[ノーベル物理学賞](../Page/ノーベル物理学賞.md "wikilink")を受賞した\[7\]。

### 宇宙マイクロ波背景放射

[宇宙マイクロ波背景放射](../Page/宇宙マイクロ波背景放射.md "wikilink") (CMB) は[宇宙の晴れ上がり](https://ja.wikipedia.org/wiki/宇宙の晴れ上がり "wikilink")時に宇宙を満たしていた[光子](../Page/光子.md "wikilink")が等方な 2.725 K の熱的な[マイクロ波](../Page/マイクロ波.md "wikilink")として現在観測されるものである。CMB にはわずかな温度の[異方性があり](https://ja.wikipedia.org/wiki/等方的と異方的 "wikilink")、その角度依存性や[偏光](../Page/偏光.md "wikilink")の解析から[宇宙論パラメータ](https://ja.wikipedia.org/wiki/宇宙論パラメータ "wikilink")を高い精度で決定することができる\[8\]。これは[21世紀](../Page/21世紀.md "wikilink")に入ってから人工衛星によるCMB温度異方性の高い精度の観測によって可能となり、[WMAP](https://ja.wikipedia.org/wiki/WMAP "wikilink")衛星は[2003年](../Page/2003年.md "wikilink")にIa型超新星の結果と矛盾しない結果を報告し\[9\]、宇宙の加速膨張が裏付けられた。その後の[Planck衛星の結果](https://ja.wikipedia.org/wiki/プランク_\(人工衛星\) "wikilink")\[10\]も加速膨張を支持している。

### バリオン音響振動

[宇宙の晴れ上がり](https://ja.wikipedia.org/wiki/宇宙の晴れ上がり "wikilink")以前の初期宇宙ではすべての物質は[電離](https://ja.wikipedia.org/wiki/電離 "wikilink")状態にあり、[光子](../Page/光子.md "wikilink")との[トムソン散乱](https://ja.wikipedia.org/wiki/トムソン散乱 "wikilink")によりバリオンと光子は一体の[流体](../Page/流体.md "wikilink")として振る舞う。時刻（宇宙年齢） \(t\) でのこの流体の音速 \(c_s\) は

\[c_s ( t ) = \frac{ c }{ \sqrt{ 3 ( 1 + R ( t ) } } , \ \ R ( t ) = \frac{ 3 \bar{\rho}_b ( t ) }{ 4 \bar{\rho}_\gamma ( t ) } = a ( t ) \times \frac{ 3 \Omega_{m 0} }{ 4 \Omega_{\gamma 0} }\] により与えられる\[11\]（\(\bar{\rho}_b\), \(\bar{\rho}_\gamma\) はバリオンと光子の平均エネルギー密度、\(a(t)\)は[スケール因子](https://ja.wikipedia.org/wiki/スケール因子_\(宇宙論\) "wikilink")）。バリオン-光子流体はこの音波が時間 \(t\) の間に伝播する距離

\[r_s ( t ) = \int_0^t c_s ( t' ) a ( t' ) dt' = \frac{ 1 }{ \sqrt{ 3 } } \int_0^t \frac{ c }{ \sqrt{ 1 + R ( t' ) } } dt'\] を典型的な波長とする特徴的な密度ゆらぎを持つ\[12\]（[バリオン音響振動](https://ja.wikipedia.org/wiki/バリオン音響振動 "wikilink")、BAO）。この密度ゆらぎは晴れ上がり後も残り、[宇宙の大規模構造](../Page/宇宙の大規模構造.md "wikilink")にもこの波長に対応する密度ゆらぎの痕跡が残される\[13\]。物質の[パワースペクトルに含まれるバリオン音響振動の波長は物質の密度パラメータ](https://ja.wikipedia.org/wiki/パワースペクトル_\(宇宙論\) "wikilink") \(\Omega_{m 0}\) で決まる「宇宙のものさし」として機能するため、その見かけのサイズ（[角径距離](https://ja.wikipedia.org/wiki/角径距離 "wikilink")）から宇宙論パラメータを観測的に制限する有用なツールとなる\[14\]。BAOは[2005年](https://ja.wikipedia.org/wiki/2005年 "wikilink")に[SDSSによって銀河分布の相関から検出されており](https://ja.wikipedia.org/wiki/スローン・デジタル・スカイサーベイ "wikilink")\[15\]、[2017年](../Page/2017年.md "wikilink")のによるBAOの解析\[16\]もまた加速膨張と無矛盾である。

## 理論的帰結

[フリードマン・ルメートル・ロバートソン・ウォーカー計量](https://ja.wikipedia.org/wiki/フリードマン・ルメートル・ロバートソン・ウォーカー計量 "wikilink")を仮定するとき、[スケール因子](https://ja.wikipedia.org/wiki/スケール因子_\(宇宙論\) "wikilink") \(a\) に関する力学方程式は[アインシュタイン方程式](../Page/アインシュタイン方程式.md "wikilink")から得られる

\[\frac{ \ddot{a} }{ a } = - \frac{ 4 \pi G }{ 3 c^2 } \sum_a ( \rho_a + 3 p_a ) + \frac{ \Lambda }{ 3 }\] である。ここで \(\Lambda\) は[宇宙定数](../Page/宇宙定数.md "wikilink")、\(\rho_a\), \(p_a\) は成分 \(a\) に関するエネルギー密度および圧力である。ここで宇宙定数がなく、すべての成分が

\[\rho_a + 3 p_a > 0\] を満足するならば \(\ddot{a} < 0\) となり減速膨張宇宙となる\[17\]。それ故に宇宙の加速膨張は、アインシュタイン重力を仮定すると、正の宇宙定数が存在するか、不等式 \(\rho_a + 3 p_a > 0\) を破る何らかのエキゾチックなエネルギー成分が存在しなければならないことを示している。両者の可能性を合わせて、このエネルギー成分は[ダークエネルギー](https://ja.wikipedia.org/wiki/ダークエネルギー "wikilink")と呼ばれる\[18\]。

### 宇宙の終焉

ダークエネルギー優勢宇宙では加速膨張が継続し、その振る舞いはダークエネルギーの圧力と密度の比（[状態方程式パラメータ](https://ja.wikipedia.org/wiki/状態方程式_\(宇宙論\) "wikilink")） \(w = p_\Lambda / \rho_\Lambda\) により特定される。宇宙の加速膨張を引き起こすためには \(w < - 1/3\) でなければならない。宇宙定数の場合厳密に \(w = - 1\) であり、そのとき宇宙膨張はやがて指数関数的膨張となり、宇宙は永遠に存在し[熱的死](../Page/熱的死.md "wikilink")を迎えると予想される。しかし \(-1 < w < - 1/3\) となる[ファントムエネルギー](https://ja.wikipedia.org/wiki/ファントムエネルギー "wikilink")と呼ばれるモデルでは、宇宙は有限の時間で無限の大きさへと膨張し、あらゆる構造が宇宙膨張により破壊されると予想されている（[ビッグリップ](../Page/ビッグリップ.md "wikilink")）\[19\]。[Hyper Suprime-Camチームによる](https://ja.wikipedia.org/wiki/Hyper_Suprime-Cam "wikilink")[2018年](../Page/2018年.md "wikilink")の報告\[20\]では[すばる望遠鏡](https://ja.wikipedia.org/wiki/すばる望遠鏡 "wikilink")のHSCを用いた弱い[重力レンズ](../Page/重力レンズ.md "wikilink")効果の観測に加えてPlanck衛星やBAOのデータを合わせた解析により \(w = -1.37^{+0.43}_{-0.36}\) と結論されており、\(w = -1\) と無矛盾だがそこからずれている可能性は棄却されていない。
