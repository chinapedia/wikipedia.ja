**長期暦**（ちょうきれき、, ）とは[メソアメリカ文明](https://ja.wikipedia.org/wiki/メソアメリカ文明 "wikilink")でおそくとも[紀元前1世紀](../Page/紀元前1世紀.md "wikilink")頃から使用されはじめた[暦](../Page/暦.md "wikilink")である。周期が非常に長いために、実質上循環しない暦として利用できる。碑文に長期暦が記されている場合にはその絶対年代を知ることができるために重要である。[マヤ諸都市の石碑に刻まれていたことでひろく知られるようになったために](https://ja.wikipedia.org/wiki/マヤ文明 "wikilink")[マヤ暦](../Page/マヤ暦.md "wikilink")とされることもあるが、マヤ以外の土地で古くから使われており、マヤ文明に起源するわけではない。

## 長期暦のサイクル

長期暦は、1日である[キン](https://ja.wikipedia.org/wiki/キン_\(マヤ暦\) "wikilink")、20キンの「月」である[ウィナル](https://ja.wikipedia.org/wiki/ウィナル "wikilink")、18ウィナルの「年」である[トゥン](../Page/トゥン.md "wikilink")、更に20トゥンを1[カトゥン](https://ja.wikipedia.org/wiki/カトゥン "wikilink")、20カトゥンを1[バクトゥン](../Page/バクトゥン.md "wikilink")とする単位で構成される。石碑に長期暦の日が刻まれたり墓の壁画に長期暦の日付けが刻まれたりしていると、その日を特定できるため、歴史学上又は考古学上の貴重な資料となる。マヤ研究者は通常バクトゥンから順に「.」で区切って算用数字で長期暦を表す。

| 名称    | 日数     | 備考     |
| ----- | ------ | ------ |
| バクトゥン | 144000 | 約394年  |
| カトゥン  | 7200   | 約19.7年 |
| トゥン   | 360    | 約1年    |
| ウィナル  | 20     |        |
| キン    | 1      |        |

[トンプソンによると](https://ja.wikipedia.org/wiki/ジョン・エリック・シドニー・トンプソン "wikilink")、計算のためにバクトゥンより大きな単位も存在し、20バクトゥンを1ピクトゥン、20ピクトゥンを1カラブトゥン、20カラブトゥンを1キンチルトゥン、20キンチルトゥンを1アラウトゥンとした\[1\]。1アラウトゥンは約6308万年に相当する。

長期暦の[元期](../Page/元期.md "wikilink")は13.0.0.0.0 4アハウ 8クムクであり、これは[先発グレゴリオ暦](https://ja.wikipedia.org/wiki/先発グレゴリオ暦 "wikilink")で紀元前3114年8月11日にあたるとされる。バクトゥンで0のかわりに13が使われているのは、おそらく[ツォルキン](../Page/ツォルキン.md "wikilink")の日付の計算で13を0として扱うのが拡大解釈されたものであろうという\[2\]。[キリグア](../Page/キリグア.md "wikilink")石碑Cではこの日付に創造説話を関連させている\[3\]。一部の碑文では長期暦の5つの数の上にずらっと13を並べたものがある\[4\]。

下記のGMT対照法によれば、2012年12月21日にこの13.0.0.0.0（4アハウ 3カンキン）に至る。ただし、近年騒がれている周期に終わりがあるかのような解釈については、マヤ古来のものでなく[西洋](../Page/西洋.md "wikilink")的な見方であるとして否定的な見解もある\[5\]。[ドレスデン絵文書](../Page/ドレスデン絵文書.md "wikilink")には元期より以前の紀元前3115年3月20日の日付も扱われており\[6\]、元期に終末が来るように考えられていたようではない。

### 西暦との対応関係

[西暦](../Page/西暦.md "wikilink")との対応については以前は[スピンデン対照法](https://ja.wikipedia.org/wiki/スピンデン対照法 "wikilink")（Spinden Correlation）という換算法と[GMT対照法](../Page/GMT対照法.md "wikilink")（GMT Correlation。GMTは、[グッドマン](https://ja.wikipedia.org/wiki/ジョゼフ・グッドマン "wikilink")-マルティネス-[トンプソンの略](https://ja.wikipedia.org/wiki/ジョン・エリック・シドニー・トンプソン "wikilink")）という2つの換算法の間で論争があったが、現在はGMT対照法が2,3日のズレはあるがほぼ正しいという結論に落ち着いている。

## 起源及び最古の資料

長期暦の最も古い資料は7バクトゥンを表すもので（cycle7の資料と呼ばれる）、7バクトゥンのはじまる[紀元前354年頃から](../Page/紀元前4世紀.md "wikilink")[紀元前30年](../Page/紀元前30年.md "wikilink")くらいの時期にcycle7の資料が集中する付近に住む[チャパス州北部と](../Page/チアパス州.md "wikilink")[タバスコ州](../Page/タバスコ州.md "wikilink")に住む先住民[ソケー](https://ja.wikipedia.org/wiki/ソケー "wikilink")や[オアハカ州](../Page/オアハカ州.md "wikilink")北東部に住む先住民[ミヘー](../Page/ミヘー.md "wikilink")の人々によって発明されたと推察されている。[ラ・モハーラ](../Page/ラ・モハーラ.md "wikilink")や[トゥシュトラの小像](https://ja.wikipedia.org/wiki/トゥシュトラの小像 "wikilink")に刻まれている碑文や銘文がかれらの言語を当てはめると解読できると主張する学者もいるからである（[ラ・モハラの文字](https://ja.wikipedia.org/wiki/ラ・モハラの文字 "wikilink")を参照）。

確実に最古の日付けが刻まれているとされるのは、メキシコ・チャパス州の[チャパ・デ・コルソ](../Page/チャパ・デ・コルソ.md "wikilink")(Chiapa de Corzo)の石碑2号である〔7バクトゥン16カトゥン〕3トゥン2ウィナル13キン6[ベン](https://ja.wikipedia.org/wiki/ベン_\(マヤ暦\) "wikilink")〔18[シュル](https://ja.wikipedia.org/wiki/シュル "wikilink")〕（7.16.3.2.13 = [紀元前36年](../Page/紀元前36年.md "wikilink")。角括弧内は計算により補った箇所。以下、長期暦の単位を「.」で略す）の日付けが刻まれている。ただし[グアテマラ](../Page/グアテマラ.md "wikilink")の[アバフ・タカリク](../Page/アバフ・タカリク.md "wikilink")の石碑2号の日付けはカトゥン以下が欠損しているためはっきりしないが、カトゥンの「位」が6であれば7.6.0.0.0.として一番古くて[紀元前235年](../Page/紀元前235年.md "wikilink")までさかのぼることになる。また仮に16であっても7.16.0.0.0.であれば、[紀元前38年](../Page/紀元前38年.md "wikilink")になるので最も古くなる可能性を持っている。7.16.19.17.19であれば[紀元前18年](../Page/紀元前18年.md "wikilink")に当たる。

## マヤの長期暦の資料

マヤで長期暦を記した最古の資料とされるものに[ハウバーグの石碑](https://ja.wikipedia.org/wiki/ハウバーグの石碑 "wikilink")（[紋章文字](../Page/紋章文字.md "wikilink")が刻まれているがどの遺跡のものか比定できない）があり、〔8.8.0.7.0.〕3[アハウ](https://ja.wikipedia.org/wiki/アハウ "wikilink")13シュル （紀元[199年](https://ja.wikipedia.org/wiki/199年 "wikilink")）とされるが、欠損が多い。これを除くともっとも古い日付は[ティカル](../Page/ティカル.md "wikilink")石碑29号の8.12.14.8.15（292年）になる。

マヤ中心部の都市国家は9世紀に次々に崩壊し、10世紀にはいるとほとんど長期暦は記されなくなった。[トニナ](https://ja.wikipedia.org/wiki/トニナ "wikilink")のモニュメント101号 10.4.0.0.0（909年1月15日）が確実なものではもっとも遅い日付を表している\[7\]。イツィムテ石碑6号は10.4.1.0.0（910年）の日付が記されているともいう。

## 短期暦

スペイン人がマヤの地を訪れたときにはもはや長期暦は使われていなかったが、それを簡単にした短期暦が[後古典期](https://ja.wikipedia.org/wiki/後古典期 "wikilink")のマヤ低地で使われていた。この暦は『[チラム・バラムの書](../Page/チラム・バラムの書.md "wikilink")』のような後世の書物にも使われている。短期暦ではバクトゥンを使わず、カトゥンをその最後の日の[ツォルキン](../Page/ツォルキン.md "wikilink")による日付で呼んだ。この方式では13カトゥン（93600日、256年強）で一周する\[8\]。カトゥンの日数（7200日）は20の倍数なので、20日周期は常にアハウになる。また、 13 × 553 + 11}} なので、13日の数字は11ずつ進む。
