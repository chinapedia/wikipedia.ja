『**欽ちゃん&香取慎吾の全日本仮装大賞**』（きんちゃんアンドかとりしんごのぜんにほんかそうたいしょう、英字表記:*ALL JAPAN KASOH GRAND-PRIX*\[1\]）は、[日本テレビ](../Page/日本テレビ放送網.md "wikilink")[系列で年](../Page/日本テレビネットワーク協議会.md "wikilink")1回、[1月](https://ja.wikipedia.org/wiki/1月 "wikilink")あるいは[2月](https://ja.wikipedia.org/wiki/2月 "wikilink")に放送される[日本](https://ja.wikipedia.org/wiki/日本 "wikilink")の[視聴者参加型番組](https://ja.wikipedia.org/wiki/視聴者参加型番組 "wikilink")で、[萩本欽一](../Page/萩本欽一.md "wikilink")と[香取慎吾](https://ja.wikipedia.org/wiki/香取慎吾 "wikilink")の冠番組でもある。これまでに97回放送されている。

## 概要

### 番組内容

視聴者が試行錯誤した「[仮装](https://ja.wikipedia.org/wiki/仮装 "wikilink")作品」を競う番組である。

初期は「[ズームイン\!\!朝\!](../Page/ズームイン!!朝!.md "wikilink")」番組内で募集告知したが、現在は電話、はがき、インターネットで応募受付している。全国各地の予選から約30組\[2\]が[東京国際フォーラム](../Page/東京国際フォーラム.md "wikilink")、[TOKYO DOME CITY HALL](https://ja.wikipedia.org/wiki/ミーツポート#TOKYO_DOME_CITY_HALL "wikilink")、[東京ビッグサイト](../Page/東京国際展示場.md "wikilink")、[日本工学院専門学校片柳アリーナ](https://ja.wikipedia.org/wiki/日本工学院専門学校#片柳アリーナ "wikilink")\[3\]や[幕張メッセ](../Page/幕張メッセ.md "wikilink")\[4\]などで行われる本選へ出場する。

番組テーマ曲は独自曲だが、[マイケル・ジャクソン](https://ja.wikipedia.org/wiki/マイケル・ジャクソン "wikilink")の「Ease on Down The Road」をアレンジしている。

### 番組の歴史

[コント55号](../Page/コント55号.md "wikilink")の[萩本欽一](../Page/萩本欽一.md "wikilink")が司会で、[1975年](../Page/1975年.md "wikilink")から3年連続で[大晦日](../Page/大晦日.md "wikilink")に放送された『[コント55号の紅白歌合戦をぶっ飛ばせ\!なんてことするの\!?](../Page/コント55号の紅白歌合戦をぶっ飛ばせ!なんてことするの!?.md "wikilink")』を基に企画されている。初放送は[1979年](../Page/1979年.md "wikilink")12月31日21:00から23:30まで『**欽ちゃんの[紅白歌合戦をぶっ飛ばせ](../Page/NHK紅白歌合戦.md "wikilink")\!第1回全日本仮装大賞〜なんかやら仮そう\!〜**』として[新宿コマ劇場](../Page/新宿コマ劇場.md "wikilink")から[生放送](../Page/生放送.md "wikilink")され、78組が出場した。当初は[仮装行列](https://ja.wikipedia.org/wiki/仮装行列 "wikilink")的作品が多数であったが、後に錬成された作品の比率が増加していく。

萩本は[1985年](https://ja.wikipedia.org/wiki/1985年 "wikilink")3月に当時のレギュラー番組を全て降板して休養を宣言したが、本番組は例外としてその時期に放送された第15回・第16回放送分も司会を務めた。本番組も降板するつもりが、日本テレビの[齋藤太朗](../Page/齋藤太朗.md "wikilink")から「欽ちゃんの番組は欽ちゃんがやめると言えば終わりだけど、これは素人の番組だ。俺が一生懸命作って、素人ががんばったんだよ。欽ちゃんはその司会をしているんであって、勝手にやめるって言えないんじゃないの」と説得されて降板を思い留まった\[5\]、と述懐している。

[1988年](../Page/1988年.md "wikilink")1月2日放送の第23回大会は海外予選が[ハワイ](https://ja.wikipedia.org/wiki/ハワイ "wikilink")で行われ、以後[ロサンゼルス](../Page/ロサンゼルス.md "wikilink")、[香港](https://ja.wikipedia.org/wiki/香港 "wikilink")、[台湾](https://ja.wikipedia.org/wiki/台湾 "wikilink")などで地区予選や放送が行われる国際的番組となる。

放送開始から[2001年](../Page/2001年.md "wikilink")までの23年間は萩本が単独で司会するも、[2002年](../Page/2002年.md "wikilink")1月1日放送の第65回以降は萩本と香取慎吾の共同司会になり、香取は萩本のアシスタントとして進行している。

番組名は、第64回までは『**欽ちゃんの爆笑仮装コンテスト 第○回全日本仮装大賞**』、香取が参加以降の第65回と第70-76回までは『**欽ちゃん&香取慎吾の新\!仮装大賞**』、第66-69回と第77回以降は『**欽ちゃん&香取慎吾の全日本仮装大賞**』と変遷している。

第1回から第4回までは年末年始と春期の年2回放送だが、応募者の増加と第4回の視聴率20.8%により、[1982年](../Page/1982年.md "wikilink")の第5回から[2003年](../Page/2003年.md "wikilink")の第70回までは[正月](../Page/正月.md "wikilink")・春期・秋期\[6\]の年3回放送された。[2004年](../Page/2004年.md "wikilink")の第71回から[2013年](../Page/2013年.md "wikilink")の第90回までは、正月と春期\[7\]の年2回放送された。[2014年](../Page/2014年.md "wikilink")の第91回からは年1回放送されている。

放送時間は原則として19:00から20:54までの2時間枠である。過去には『[スーパースペシャル](../Page/スーパースペシャル.md "wikilink")』や『[木曜スペシャル](../Page/木曜スペシャル.md "wikilink")』\[8\]などの枠で放送され、[1982年](../Page/1982年.md "wikilink")から[2001年](../Page/2001年.md "wikilink")の正月放送は18:30から20:54までの2時間半枠\[9\]、香取が参加した[2002年](../Page/2002年.md "wikilink")から[2004年](../Page/2004年.md "wikilink")の正月放送\[10\]は18:00から20:54までの3時間枠で放送された。

## 番組の流れ

### 本選までの道のり

1.  申し込み　申込書を郵送あるいはFAXで「日本テレビ・仮装大賞本部」へ送付する。
2.  書類審査　合格すると電話で予選日程日時を通知される。
3.  作品製作　セットや道具を自費で製作する。
4.  地区予選　現在は[札幌](../Page/札幌テレビ放送.md "wikilink")（北海道）、[仙台](../Page/宮城テレビ放送.md "wikilink")（東北地方で尚且つ太平洋側）、[新潟](../Page/テレビ新潟放送網.md "wikilink")（東北地方で尚且つ日本海側・新潟県も含む）、[東京](../Page/日本テレビ放送網.md "wikilink")（関東・海外）、[福井](https://ja.wikipedia.org/wiki/福井放送 "wikilink")（北陸）、[長野](../Page/テレビ信州.md "wikilink")（甲信）、[静岡](../Page/静岡第一テレビ.md "wikilink")（静岡）、[名古屋](../Page/中京テレビ放送.md "wikilink")（東海）、[大阪](../Page/讀賣テレビ放送.md "wikilink")（近畿）、[広島](../Page/広島テレビ放送.md "wikilink")（中国）、[高松](../Page/西日本放送.md "wikilink")（四国）、[福岡](../Page/福岡放送.md "wikilink")（九州・沖縄）で開催されている。
5.  本選会場　番組へ出場して[公開審査](../Page/公開審査.md "wikilink")を受ける。

### 本選の基本的な構成

1.  オープニング、審査員紹介、欽ちゃん 慎吾ちゃんの掛け声。
2.  作品発表、審査
3.  合格者紹介\[16\]
4.  各賞表彰式
5.  エンディング、視聴者投票結果発表\[17\]、優勝作品紹介\[18\]

## 審査

### 審査方法の変遷

|               | 第1回  | 第2-80回 | 第81-83回 | 第84-85回 | 第86回- |
| ------------- | ---- | ------ | ------- | ------- | ----- |
| **審査員の人数**    | 15人  | 10人    | 5人      | 10人     |       |
| **1人あたりの持ち点** | 1点   | 2点     | 4点      | 2点      |       |
| **合格点**       | 8点以上 | 15点以上  | 12点以上   | 15点以上   |       |
| **満点**        | 15点  | 20点    |         |         |       |

  - 第1回の審査方法は、15人の審査員が、15点満点から持ち点1点を減らしていく減点方式で、出場者の演技終了時に得点が8点以上なら合格であった。
  - 第2-80回と第86回以降は、10名の審査員が、1人あたり2点の持ち点で計20点を投票する加点方式。演技が終了した時点から審査が始まる。得点が15点以上になった時点で合格となり、ファンファーレが鳴る。逆に得点が14点以下で加点が途切れた状態が3 - 5秒（近年では10秒）程度続くと不合格となり、鐘の音が鳴り響く。但し、鐘が鳴ったあとも出場者が退場するまでは加点が可能なため、演技終了後に萩本、香取のフォローなどにより、審査員のおまけで加点され合格となることがある。
  - 第81-85回の審査員は5人で、1人あたりの持ち点は4点だった。
  - 合格点も第81-83回は20点満点中12点以上に引き下げられたが、第84回から再び15点以上に戻された。
  - 第89回からは番組初の試みとして、視聴者生投票を実施している。テレビの[データ放送](../Page/データ放送.md "wikilink")や[パソコン](https://ja.wikipedia.org/wiki/パソコン "wikilink")\[19\]、[携帯電話](../Page/携帯電話.md "wikilink")、[スマートフォン](https://ja.wikipedia.org/wiki/スマートフォン "wikilink")からリアルタイムで作品を会場の審査員同様に持ち点2点で審査することができる。

## 表彰

  - 出場者は、本選大会で合格した時点で「入賞」扱いとなり、メダルが授与される。メダルは徐々にデザインが変化している。
  - 最終審査で、合格者の中からアイデア賞などの各賞や第3位・準優勝・優勝を選定し、表彰を行う。合格者のみ表彰式に参加することができる。各賞の発表は第1-48回と第55-64回は審査員長が行っていたが、第65回からは香取慎吾が担当している\[20\]。
