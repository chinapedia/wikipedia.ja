**大野の語彙法則**（おおののごいほうそく）あるいは単に大野の法則（おおののほうそく）とは、[日本](https://ja.wikipedia.org/wiki/日本 "wikilink")の[国語学者](https://ja.wikipedia.org/wiki/日本語学者 "wikilink")、[大野晋](../Page/大野晋.md "wikilink")が[1956年](../Page/1956年.md "wikilink")に発表した、日本の9古典作品の[品詞](../Page/品詞.md "wikilink")の構成比に関して見出した統計的法則。\[1\]

## 大野の語彙法則（原版）

[ohnos-vocabulary-law-fig1-ja.jpg](https://ja.wikipedia.org/wiki/File:ohnos-vocabulary-law-fig1-ja.jpg "fig:ohnos-vocabulary-law-fig1-ja.jpg") [万葉集](../Page/万葉集.md "wikilink")と[源氏物語](https://ja.wikipedia.org/wiki/源氏物語 "wikilink")の[名詞](../Page/名詞.md "wikilink")、[動詞](../Page/動詞.md "wikilink")、[形容詞](../Page/形容詞.md "wikilink")、[形容動詞](../Page/形容動詞.md "wikilink")、その他の[百分率](https://ja.wikipedia.org/wiki/百分率 "wikilink")を[グラフ](https://ja.wikipedia.org/wiki/グラフ "wikilink")に目盛り、名詞、動詞、形容詞の万葉と源氏の値を端点として各々を結ぶと、名詞は単調減少の[直線](../Page/直線.md "wikilink")となり、他は単調増加の直線となる。ここに、他の七作品の百分率を、上記と同一のグラフの上に目盛ると、その各点は上記の三直線上に、ほぼ[垂直](../Page/垂直.md "wikilink")に並ぶ\[2\]。

## 水谷静夫による大野の語彙法則（改訂版）

任意の3作品甲・乙・丙の[語彙](https://ja.wikipedia.org/wiki/語彙 "wikilink")での名詞構成比を\(X_0, x, X_1\)とし、しかるべき語類の構成比を\(Y_0, y, Y_1\)とすると、3点\((X_0, Y_0), (x, y), (X_1, Y_1)\)がほぼ一直線をなす：すなわち\[3\] \[4\] \[5\]

|  |                                                                 |
|  | --------------------------------------------------------------- |
|  | \(\frac{y - Y_0}{Y_1 - Y_0} \approx \frac{x - X_0}{X_1 - X_0}\) |
