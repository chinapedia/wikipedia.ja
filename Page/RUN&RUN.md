『**RUN\&RUN**』（**ラン アンド ラン**）は、[1980年](https://ja.wikipedia.org/wiki/1980年 "wikilink")[3月15日](../Page/3月15日.md "wikilink")に全国劇場公開された[矢沢永吉](https://ja.wikipedia.org/wiki/矢沢永吉 "wikilink")の[ドキュメンタリー映画](https://ja.wikipedia.org/wiki/ドキュメンタリー映画 "wikilink")\[1\]。矢沢の[1979年](../Page/1979年.md "wikilink")[9月15日](../Page/9月15日.md "wikilink")の[ナゴヤ球場](../Page/ナゴヤ球場.md "wikilink")ライブと、普段見せない私生活を追う\[2\]。

ファンからの問い合わせが[富士映画に殺到したが](https://ja.wikipedia.org/wiki/松竹富士 "wikilink")、当時の矢沢は徹底したマスコミ嫌いを標榜し\[3\]、富士映画が何度頭を下げても矢沢は表舞台に出て来てくれず\[4\]。このため製作記者会見も出来ず、完成披露試写会での出席も拒否され、映画の宣伝に一切協力しなかった\[5\]。ところが1980年2月10日に[NHK教育の](https://ja.wikipedia.org/wiki/NHK教育テレビジョン "wikilink")『[若い広場](https://ja.wikipedia.org/wiki/若い広場 "wikilink")』に突如出演し\[6\]、富士映画の関係者をビックリさせた\[7\]。この放送が大きな反響を呼んだため、これ以上ない映画の宣伝になった\[8\]。

[なぎら健壱](../Page/なぎら健壱.md "wikilink")は、公開時に映画館に本作を観に行き、劇場内で矢沢タオルを宙に放るYAZAWA信者を目撃し、衝撃を受けたと話している\[9\]。
