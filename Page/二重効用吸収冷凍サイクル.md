**二重効用吸収冷凍サイクル**（にじゅうこうようきゅうしゅうれいとうサイクル）は、[単効用吸収冷凍サイクル](../Page/単効用吸収冷凍サイクル.md "wikilink")に高温[再生器](https://ja.wikipedia.org/wiki/再生器 "wikilink")を追加し高温再生器で分離した[冷媒](../Page/冷媒.md "wikilink")[蒸気](https://ja.wikipedia.org/wiki/蒸気 "wikilink")の凝縮熱で低温再生器を動作させ、効率の向上を図った[冷凍機](../Page/冷凍機.md "wikilink")の[熱サイクルである](../Page/熱機関の理論サイクル.md "wikilink")。

再生器の必要熱量・凝縮器の冷却熱量の減少により、[燃料](../Page/燃料.md "wikilink")の節約・[冷却塔](../Page/冷却塔.md "wikilink")の容量低減が可能である。

## 直列流サイクル

直列流サイクルは、高温再生器から低温再生器へ吸収液を直列に流すものである。

1 - [蒸発器](https://ja.wikipedia.org/wiki/蒸発器 "wikilink")で液体の冷媒を低い圧力に保った低温で気化させ Q<sub>E</sub> 冷却する。 →1'

1',6 - [吸収器](https://ja.wikipedia.org/wiki/吸収器 "wikilink")で吸収液に冷媒蒸気を吸収させる。吸収液濃度 ξ<sub>1</sub> →2

2 - 吸収液[ポンプ](../Page/ポンプ.md "wikilink")で希吸収液を低温液[熱交換器](../Page/熱交換器.md "wikilink")へ送り Q<sub>H</sub> 熱交換する →7

7 - 吸収液ポンプで希吸収液を高温液熱交換器へ送り Q<sub>HH</sub> 熱交換する →7H

7H - 高温[再生器](https://ja.wikipedia.org/wiki/再生器 "wikilink")で Q<sub>G</sub> の熱で希吸収液を加熱する。 →5H

5H - 濃吸収液と冷媒蒸気に分離する。 →4H,4H'

4H - 再生器から高温液熱交換器へ[圧力](../Page/圧力.md "wikilink")差で濃吸収液を送り Q<sub>HH</sub> 熱交換する。吸収液濃度 ξ<sub>3</sub> →8H

4H' -低温再生器で Q<sub>CH</sub> 冷却し冷媒を液化する。 →3H

8H - 低温再生器で Q<sub>CH</sub> の熱で希吸収液を加熱する。 →5

5 - 濃吸収液と冷媒蒸気に分離する。 →4,4'

4 - 再生器から低温吸収液熱交換器へ圧力差で濃吸収液を送り Q<sub>H</sub> 熱交換する。吸収液濃度 ξ<sub>2</sub> →8

8 - 吸収器で吸収液を Q<sub>A</sub> 冷却する。 →6

4' - [凝縮器](https://ja.wikipedia.org/wiki/凝縮器 "wikilink")で Q<sub>C</sub> 冷却し冷媒を液化する。 →3

3,3H - 凝縮器から蒸発器へ圧力差で液化した冷媒を移送する。 →1

### 理論成績係数・冷凍能力

熱交換器などの損失がない場合を考えると。

  -
    Q<sub>E</sub> + Q<sub>G</sub> = Q<sub>A</sub> + Q<sub>C</sub>
    Q<sub>E</sub> = D ( h<sub>1</sub>' - h<sub>3</sub> )
    Q<sub>C</sub> = D ( h<sub>4</sub>' - h<sub>3</sub> )
    Q<sub>CH</sub> = D ( h<sub>4H</sub>' - h<sub>3H</sub> )
    Q<sub>G</sub> = a D h<sub>4H</sub> + D ( a / a<sub>1</sub> )( h<sub>4</sub>' - h<sub>4H</sub> ) - a D h<sub>7H</sub>
    Q<sub>A</sub> = ( a -1 ) D h<sub>8</sub> + D h<sub>1</sub>' - a D h<sub>2</sub>
    Q<sub>H</sub> = a D ( h<sub>7</sub> - h<sub>2</sub> )
    Q<sub>HH</sub> = a D ( h<sub>7H</sub> - h<sub>7</sub> )
    (COP)<sub>R</sub> = Q<sub>E</sub> / Q<sub>G</sub> = ( h<sub>1</sub>' - h<sub>3</sub> ) / { a ( h<sub>4</sub> - h<sub>7H</sub> ) + ( h<sub>4</sub>' - h<sub>4H</sub> ) a / a<sub>1</sub> }

<!-- end list -->

  -
    (COP)<sub>R</sub> : 冷凍サイクルの理論成績係数
    Q : 熱量
    h : 吸収液の[エンタルピー](../Page/エンタルピー.md "wikilink") 　h' : 吸収液と平衡する冷媒蒸気のエンタルピー
    a : 吸収液循環比、1kgの冷媒蒸気を再生器で発生させるために必要な希吸収液の循環量 a = ξ<sub>2</sub> / ( ξ<sub>2</sub> - ξ<sub>1</sub> )
    a<sub>1</sub> : 高温再生器の吸収液循環比 a<sub>1</sub> = ξ<sub>3</sub> / ( ξ<sub>3</sub> - ξ<sub>1</sub> )
    D : 再生器の発生冷媒蒸気量

## 並列流サイクル

並列流サイクルは、高温再生器と低温再生器へ吸収液を分割して流すものである。直列流サイクルと比較して、高圧再生器の最高圧力を低くでき、より[熱効率](../Page/熱効率.md "wikilink")を高めることが可能である。

1 - [蒸発器](https://ja.wikipedia.org/wiki/蒸発器 "wikilink")で液体の冷媒を低い圧力に保った低温で気化させ Q<sub>E</sub> 冷却する。 →1'

1',6 - [吸収器](https://ja.wikipedia.org/wiki/吸収器 "wikilink")で吸収液に冷媒蒸気を吸収させる。吸収液濃度 ξ<sub>1</sub> →2

2 - 吸収液[ポンプ](../Page/ポンプ.md "wikilink")で希吸収液を加圧2分流し、低温液[熱交換器](../Page/熱交換器.md "wikilink")で Q<sub>H</sub>、ドレン熱回収器で Q<sub>CL</sub> 熱交換する →7、7L

7 - 希吸収液を2分流し、高温液熱交換器で Q<sub>HH</sub>、排気ガス熱回収器で Q<sub>GL</sub> 熱交換する →7H

7H - 高温[再生器](https://ja.wikipedia.org/wiki/再生器 "wikilink")で Q<sub>G</sub> の熱で希吸収液を加熱する。 →5H

5H - 濃吸収液と冷媒蒸気に分離する。 →4H,4H'

4H - 再生器から高温液熱交換器へ[圧力](../Page/圧力.md "wikilink")差で濃吸収液を送り Q<sub>HH</sub> 熱交換する。吸収液濃度 ξ<sub>3</sub> →8H

4H' -低温再生器で Q<sub>CH</sub> 冷却し冷媒を液化する。 →3H

3H -ドレン熱回収器で Q<sub>CL</sub> 冷媒を冷却する。 →3L

7L - 低温再生器で Q<sub>CH</sub> の熱で希吸収液を加熱する。 →5

5 - 濃吸収液と冷媒蒸気に分離する。 →4,4'

8H,4 - 再生器から低温吸収液熱交換器へ圧力差で濃吸収液を送り Q<sub>H</sub> 熱交換する。吸収液濃度 ξ<sub>2</sub> →8

8 - 吸収器で吸収液を Q<sub>A</sub> 冷却する。 →6

4' - [凝縮器](https://ja.wikipedia.org/wiki/凝縮器 "wikilink")で Q<sub>C</sub> 冷却し冷媒を液化する。 →3

3,3L - 凝縮器・ドレン熱回収器から蒸発器へ圧力差で液化した冷媒を移送する。 →1

## 二段蒸発・吸収サイクル

**二段蒸発・吸収サイクル**は、蒸発器・吸収器を上下二分割し、下部を圧力が高い第1蒸発器・吸収器、上部を第2蒸発器・吸収器とするものである。冷却水・冷水は下部から上部へ、冷媒・吸収液は上部から下部への対向流となっている。

吸収液をより希薄な濃度まで使用でき、大温度差の冷水供給が可能であり、同じ[冷凍能力](https://ja.wikipedia.org/wiki/冷凍能力 "wikilink")での吸収液循環量・加熱量の低減が可能である。

## 二重効用第一種吸収ヒートポンプ

このサイクルを利用し、低温排熱を高温熱源で汲み上げて加熱するものを**二重効用第一種吸収ヒートポンプ**と呼ぶ。

  -
    (COP)<sub>H</sub> = (COP)<sub>R</sub> + 1

  -
    (COP)<sub>H</sub> : ヒートポンプサイクルの理論成績係数
