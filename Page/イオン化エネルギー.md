[First_ionization_energies.png](https://ja.wikipedia.org/wiki/File:First_ionization_energies.png "fig:First_ionization_energies.png")で最も小さく、[希ガス](https://ja.wikipedia.org/wiki/希ガス "wikilink")で最も大きくなる周期的な変化が見られる。\]\] **イオン化エネルギー**（イオンかエネルギー、、電離エネルギー、イオン化ポテンシャルとも言う）とは、[原子](https://ja.wikipedia.org/wiki/原子 "wikilink")、[イオン](../Page/イオン.md "wikilink")などから[電子](https://ja.wikipedia.org/wiki/電子 "wikilink")を取り去って[イオン化](../Page/イオン化.md "wikilink")するために要する[エネルギー](https://ja.wikipedia.org/wiki/エネルギー "wikilink")\[1\]。ある原子がその電子をどれだけ強く結び付けているのかの目安である。

気体状態の単原子（または分子の基底状態）の中性原子から取り去る電子が1個目の場合を**第1イオン化エネルギー**（IE<sup>1</sup>）、2個目の電子を取り去る場合を**第2イオン化エネルギー**（IE<sup>2</sup>）、3個目の電子を取り去る場合を**第3イオン化エネルギー**（IE<sup>3</sup>）・・・（以下続く）と言う\[2\]。単にイオン化エネルギーといった場合、第1イオン化エネルギーのことを指すことがある。

\[\rm M(g) \rarr M^+ (g) + e^-\] 　IE<sup>1</sup>

\[\rm M^+(g) \rarr M^{2+} (g) + e^-\] 　IE<sup>2</sup>

\[\rm M^{2+}(g) \rarr M^{3+} (g) + e^-\] 　IE<sup>3</sup>

イオン化エネルギーの一般的な傾向は、[s軌道](https://ja.wikipedia.org/wiki/s軌道 "wikilink")と[p軌道](https://ja.wikipedia.org/wiki/p軌道 "wikilink")の相対的エネルギーとともに、電子の結合に対する[有効核電荷](../Page/有効核電荷.md "wikilink")の効果を考えることによって説明できる。

原子核の正電荷が増すにつれ、与えられた軌道にある負に荷電した電子はより強い[クーロン引力を受け](https://ja.wikipedia.org/wiki/クーロン力 "wikilink")、より強く保持される。[ヘリウム](https://ja.wikipedia.org/wiki/ヘリウム "wikilink")の1s電子を除去するには[水素](https://ja.wikipedia.org/wiki/水素 "wikilink")の1s電子を除去するよりも多くのエネルギーを必要とする。

\[\rm H(g) \rarr H^+ (g) + e^-\] 　\(IE^1=1312\;\rm{kJ\;mol^{-1}}\)

\[\rm He(g) \rarr He^+ (g) + e^-\]　　\(IE^1=2372\;\rm{kJ\;mol^{-1}}\)

周期表の同じ周期の中で最高のイオン化エネルギーは[希ガス](https://ja.wikipedia.org/wiki/希ガス "wikilink")のものであり、希ガスは安定な[閉殻](https://ja.wikipedia.org/wiki/閉殻 "wikilink")電子配置をもつといわれる。

[主量子数](https://ja.wikipedia.org/wiki/主量子数 "wikilink")*ｎ*の値が小さい内殻電子のイオン化エネルギーは価電子に比べ格段に大きい\[3\]。たとえば電子3個の[リチウム](https://ja.wikipedia.org/wiki/リチウム "wikilink")ではIE<sup>1</sup>は5.32eV であるが、1sからのIE<sup>2</sup>は75.6eVである\[4\]。2s軌道の電子は1s軌道の電子ほど強く保持されていない。

最低のイオン化エネルギーは[周期表](../Page/周期表.md "wikilink")の左端にある第1族元素のものである。これらの原子のひとつから電子1個を除くと希ガス原子と同じ閉殻電子配置を持つイオンになる。

どの原子からも最も容易に失われる電子は最高エネルギー軌道にある電子からである。

## イオン化エネルギーについての補足

[アルカリ金属](https://ja.wikipedia.org/wiki/アルカリ金属 "wikilink")などでIEが低く、希ガスに近づくにつれ値が高まる傾向があることは前述のとおりだが、[ベリリウム](../Page/ベリリウム.md "wikilink")と[ホウ素](../Page/ホウ素.md "wikilink")、[窒素](../Page/窒素.md "wikilink")と[酸素](../Page/酸素.md "wikilink")などではその傾向が少しだけ逆転している\[5\]。この理由については[原子軌道](https://ja.wikipedia.org/wiki/原子軌道 "wikilink")や[フントの規則](../Page/フントの規則.md "wikilink")を考慮する必要がある。

窒素原子と酸素原子を例に考える。二つの電子配置は次の表のようになる。（IEの単位はeV）

N : 1s<sup>2</sup> 2s<sup>2</sup> 2p<sup>3</sup> 　　IE<sup>1</sup>：14.53, IE <sup>2</sup>：29.60

O : 1s<sup>2</sup> 2s<sup>2</sup> 2p<sup>4</sup> 　　IE<sup>1</sup>：13.61, IE <sup>2</sup>：35.12

|       | 1s | 2s | 2p<sub>x</sub> | 2p<sub>y</sub> | 2p<sub>z</sub> |
| ----- | -- | -- | -------------- | -------------- | -------------- |
| **N** | ↑↓ | ↑↓ | ↑              | ↑              | ↑              |
| **O** | ↑↓ | ↑↓ | ↑↓             | ↑              | ↑              |

窒素原子より酸素原子のほうが第一イオン化エネルギーが小さいのは、2p軌道に入る4個目の電子が三重に縮重したp軌道のいずれかの軌道に異なる[スピンをもって入り](https://ja.wikipedia.org/wiki/スピン角運動量 "wikilink")、電子間の静電的な反発エネルギーが電子を不安定にするためである。

ちなみに、第2イオン化エネルギーの場合は、どちらも区別のつかない2p軌道からひとつずつ取り去るので、[有効核電荷](../Page/有効核電荷.md "wikilink")が大きい酸素原子のほうがIE<sup>2</sup>は大きくなる。このことは他の周期でもみられる。

また[電気陰性度](https://ja.wikipedia.org/wiki/電気陰性度 "wikilink")（マリケンの電気陰性度）は、[電子親和力](https://ja.wikipedia.org/wiki/電子親和力 "wikilink")とイオン化エネルギーの[相加平均](https://ja.wikipedia.org/wiki/相加平均 "wikilink")であるが、前者に比べ後者のほうがかなり大きいため、電気陰性度はほぼイオン化エネルギーに比例する。
