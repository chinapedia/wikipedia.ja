**サセプタンス**（）は、[交流](../Page/交流.md "wikilink")回路において位相を変化させる要素である。[アドミタンス](../Page/アドミタンス.md "wikilink")の[虚数](../Page/虚数.md "wikilink")成分と等しい。単位は[国際単位系](https://ja.wikipedia.org/wiki/国際単位系 "wikilink")では、[ジーメンス](../Page/ジーメンス.md "wikilink") (S) が用いられる。かつては、パーミタンス\[1\]という言葉が用いられた。[アドミタンス](../Page/アドミタンス.md "wikilink")との関係は以下のように表される。

\(Y = G + jB\)

|     |   |                                                             |
| --- | - | ----------------------------------------------------------- |
| *j* | : | [虚数](../Page/虚数.md "wikilink")単位                            |
| *Y* | : | [アドミタンス](../Page/アドミタンス.md "wikilink")                      |
| *G* | : | [コンダクタンス](https://ja.wikipedia.org/wiki/コンダクタンス "wikilink") |
| *B* | : | **サセプタンス**                                                  |

[アドミタンス](../Page/アドミタンス.md "wikilink")は[並列回路](https://ja.wikipedia.org/wiki/並列回路 "wikilink")において回路全体のアドミタンスを和で表すことができるので、**サセプタンス**は並列回路の計算に用いられる。また、インピーダンス*Z* = 1/*Y* であるので、[コンダクタンス](https://ja.wikipedia.org/wiki/コンダクタンス "wikilink")が0の場合はリアクタンス *X* = −1/*B* となる。

## 誘導性サセプタンス

\(B_L = \frac{1}{\omega L}\)

| --- | - | --------------------------------------------------------------- |
