[Prime-gap-frequency-distribution.svg](https://ja.wikipedia.org/wiki/File:Prime-gap-frequency-distribution.svg "fig:Prime-gap-frequency-distribution.svg")。ピークは6の倍数で生じている\[1\]。\]\] **素数の間隔**(そすうのかんかく、prime gap)は、連続する2つの[素数](../Page/素数.md "wikilink")の差。*g*<sub>*n*</sub>もしくは*g*(*p*<sub>*n*</sub>)で表されるn番目の素数の間隔は、n+1番目の素数とn番目の素数の差である。すなわち

\[g_n = p_{n + 1} - p_n.\\]

*g*<sub>1</sub> = 1, *g*<sub>2</sub> = *g*<sub>3</sub> = 2, *g*<sub>4</sub> = 4である。素数の間隔の[列は広く研究されてきたが](../Page/列_\(数学\).md "wikilink")、多くの疑問や仮説が残っている。

始めから60個の素数の間隔は

  -
    1, 2, 2, 4, 2, 4, 2, 4, 6, 2, 6, 4, 2, 4, 6, 6, 2, 6, 4, 2, 6, 4, 6, 8, 4, 2, 4, 2, 4, 14, 4, 6, 2, 10, 2, 6, 6, 4, 6, 6, 2, 10, 2, 4, 2, 12, 12, 4, 2, 4, 6, 2, 10, 6, 6, 6, 2, 6, 4, 2, ... .

*g*<sub>*n*</sub>の定義により、全ての素数は次のように書ける。

\[p_{n+1} = 2 + \sum_{i=1}^n g_i.\]

## 簡単な観察

最初で最小で唯一奇数の素数の間隔は、唯一偶数の素数である2と最初の奇数の素数である3の間の1である。他の素数の間隔はすべて偶数である。値2の間隔が連続しているのは素数3,5,7の間の間隔である*g*<sub>2</sub>と*g*<sub>3</sub>の1組だけである。

任意の整数nに対して、[階乗](../Page/階乗.md "wikilink")n\!はn含むnまでの全ての正の整数の[積](https://ja.wikipedia.org/wiki/積 "wikilink")である。 このとき数列

  -
    \(n!+2, n!+3,\ldots,n!+n\)

において1番目の項は2で割り切れ、2番目の項は3で割り切れ、これが続く。よって、これはの連続した合成整数の数列であり、少なくともn以上の長さを持つ間隔に属していなければならない。任意に大きい素数間に間隔が存在すること、すなわち、任意の整数Nに対してとなる整数mが存在することがわかる。

しかし、n個の数の素数の間隔は、n\!よりもずっと小さい数で生じることがある。例えば、14よりも大きい素数の間隔は最初523と541の間で生じるが、15\!は非常に大きな数1307674368000である。

素数の平均間隔は整数の[自然対数](../Page/自然対数.md "wikilink")が大きくなるにつれて長くなり、したがって関係する整数に対する素数の間隔の比は小さくなる（そして漸近的にゼロになる）。これは[素数定理](../Page/素数定理.md "wikilink")の結果である。ヒューリスティックな観点から見ると、自然対数に対する間隔の長さの比が固定の正数*k*以上である確率はであると予想される。結果として比は任意に大きくなる。実際、整数の桁数に対する間隔の比は際限なく増加する。これはWestzynthiusによる結果の帰結である\[2\]。

逆に、[双子素数](../Page/双子素数.md "wikilink")の推論は、無限に多い整数nに対してを仮定している。

## さらなる結果

### 上限

1852年に証明された[ベルトランの仮説](../Page/ベルトランの仮説.md "wikilink")は、*k*と2*k*の間には必ず素数があり、よって特に*p*<sub>*n*+1</sub> \< 2*p*<sub>*n*</sub>であることは、*g*<sub>*n*</sub> \< *p*<sub>*n*</sub>を意味するという内容である。

1896年に証明された[素数定理](../Page/素数定理.md "wikilink")は、十分大きい素数では素数pと次の素数との間の間隔の平均長は漸近的にln(*p*)に近づくという内容である。実際の間隔の長さはこれよりもずっと大きいことや小さいことがあるが、素数定理から素数の間隔の長さの上限を推論することができる。

すべての\(\epsilon > 0\)に対して、すべての\(n > N\)で

\[g_n < p_n\epsilon\] であるような数\(N\)がある。 また、素数に比例して間隔が任意に小さくなることも推論できる。比

\[\lim_{n\to\infty}\frac{g_n}{p_n}=0.\] となる。 [Hoheisel](https://ja.wikipedia.org/wiki/Guido_Hoheisel "wikilink") (1930) は

\[\pi(x + x^\theta) - \pi(x) \sim \frac{x^\theta}{\log(x)} \text{ as } x \to \infty,\] であるような定数θ \< 1が存在することを初めて示し\[12\]、それゆえ十分大きい*n*に対して

\[g_n<p_n^\theta,\,\] であることを示した。

Hoheiselはθで可能な値32999/33000を得た。これは[Heilbronnにより](https://ja.wikipedia.org/wiki/Hans_Heilbronn "wikilink")249/250と改善され\[13\]、[Chudakovにより任意のε](https://ja.wikipedia.org/wiki/Nikolai_Chudakov "wikilink") \> 0に対してθ = 3/4 + εとした\[14\]。

主な進歩は[Inghamによる](https://ja.wikipedia.org/wiki/Albert_Ingham "wikilink")\[15\]。彼はいくつかの正の定数*c*に対して

\[\zeta(1/2 + it)=O(t^c)\,\]であるとき、任意の\(\theta > (1 + 4c)/(2 + 4c)\)に対して\(\pi(x + x^\theta) - \pi(x) \sim \frac{x^\theta}{\log(x)}\)

と示した。ここで*O*は[ランダウの記号](../Page/ランダウの記号.md "wikilink")、ζは[リーマンゼータ関数](../Page/リーマンゼータ関数.md "wikilink")、πは[素数計数関数](../Page/素数計数関数.md "wikilink")である。任意の*c* \> 1/6が許容されることが分かっていれば、θが5/8より大きい任意の数値であることが分かる。

Inghamの結果の直接の結果は、nが十分大きい場合、*n*<sup>3</sup>と(*n* + 1)<sup>3</sup>の間に必ず素数が存在するということである\[16\]。Lindelöfの仮説はInghamの式が*c*の任意の正の数に対しても成り立つことを暗に示すが、十分大きい*n*に対して*n*<sup>2</sup>と(*n* + 1)<sup>2</sup>の間に素数が存在することを暗示するには十分ではないであろう（[ルジャンドル予想](https://ja.wikipedia.org/wiki/ルジャンドル予想 "wikilink")参照）。これを証明するには[クラメールの予想](https://ja.wikipedia.org/wiki/クラメールの予想 "wikilink")のようなより強い結果が必要であろう。

[Huxleyは](https://ja.wikipedia.org/wiki/Martin_Huxley "wikilink")1972年にθ = 7/12 = 0.58(3)を選択してもよいことを示した\[17\]。

2001年のBaker、[ハーマン](https://ja.wikipedia.org/wiki/グリン・ハーマン "wikilink")、[Pintzによる結果では](https://ja.wikipedia.org/wiki/János_Pintz "wikilink")、θは0.525ととられる可能性があることを示した\[18\]。

2005年、[Daniel Goldston](https://ja.wikipedia.org/wiki/Daniel_Goldston "wikilink"), [János Pintz](https://ja.wikipedia.org/wiki/János_Pintz "wikilink"), [Cem Yıldırımは](https://ja.wikipedia.org/wiki/Cem_Yıldırım "wikilink")

\[\liminf_{n\to\infty}\frac{g_n}{\log p_n}=0\]

を証明し、2年後これを改良し\[19\]、

\[\liminf_{n\to\infty}\frac{g_n}{\sqrt{\log p_n}(\log\log p_n)^2}<\infty.\] とした。 2013年、[張益唐](https://ja.wikipedia.org/wiki/張益唐 "wikilink")は

\[\liminf_{n\to\infty} g_n < 7\cdot 10^7,\]

を証明した。これは7000万を超えない間隔が無限にあるという意味である\[20\]。張の境界を最適化するPolymathプロジェクトの共同作業により、2013年7月20日に境界を4680まで下げることに成功した\[21\]。2013年11月、[James MaynardはGPYふるいを新たに改善したものを導入し](https://ja.wikipedia.org/wiki/James_Maynard_\(mathematician\) "wikilink")、境界を600まで下げ、任意の*m*について、それぞれがm個の素数を含む解釈が無限である境界間隔が存在することを示した\[22\]。Maynardの考えを用いて、Polymathプロジェクトは境界を246に改良した\[23\]\[24\]。Elliott–Halberstam予想とその一般形を仮定すると、Nはそれぞれ12と6に減少される\[25\]。

### 下限

1931年、Erik Westzynthius は極大の素数の間隔は対数的よりも大きくなることを証明した。つまり\[26\]

\[\limsup_{n\to\infty}\frac{g_n}{\log p_n}=\infty.\]

である。1938年、[Robert Rankinは](https://ja.wikipedia.org/wiki/Robert_Alexander_Rankin "wikilink")、無限に大きいnに対して

\[g_n > \frac{c\log n\log\log n\log\log\log\log n}{(\log\log\log n)^2}\]

が成り立つ定数*c* \> 0が存在することを示し、Westzynthiusと[ポール・エルデシュ](../Page/ポール・エルデシュ.md "wikilink")の結果を改良した。彼はのちに任意の定数*c* \< *e*<sup>γ</sup>（γは[オイラーの定数](../Page/オイラーの定数.md "wikilink")）を取ることができることを示した。1997年に定数*c*の値は2*e*<sup>γ</sup>以下の任意の値に改良された\[27\]。

ポール・エルデシュは、上記の不等式の定数*c*が任意に大きく取れることの証明および反証に対して$10,000の賞金を提供した\[28\]。これは2014年にFord–Green–Konyagin–Taoと[James Maynardにより独立に正しいことが証明された](https://ja.wikipedia.org/wiki/James_Maynard_\(mathematician\) "wikilink")\[29\]\[30\]。

この結果はさらにFord–Green–Konyagin–Maynard–Taoにより、無限に大きいnに対して

\[g_n > \frac{c\log n\log\log n\log\log\log\log n}{\log\log\log n}\]

と改良された\[31\]。

エルデシュの最初の賞の精神で、[テレンス・タオ](../Page/テレンス・タオ.md "wikilink")はこの不等式で*c*が任意に大きくとられる可能性があるという証明に対して10,000 USDを提供した\[32\]。

素数の連鎖の下限も決定されている\[33\]。

## 素数の間隔の予想

[Wikipedia_primegaps.png](https://ja.wikipedia.org/wiki/File:Wikipedia_primegaps.png "fig:Wikipedia_primegaps.png") [リーマン予想](https://ja.wikipedia.org/wiki/リーマン予想 "wikilink")のもとではさらに良い結果が得られる。[ハラルド・クラメール](https://ja.wikipedia.org/wiki/ハラルド・クラメール "wikilink")はリーマン予想が、間隔*g*<sub>*n*</sub>は[ランダウの記号](../Page/ランダウの記号.md "wikilink")を用いて

\[g_n = O(\sqrt{p_n} \log p_n),\] であることを暗示していることを証明した\[34\]。のちに、この間隔はさらに小さいと予想した。おおまかに言うと、[クラメールの予想](https://ja.wikipedia.org/wiki/クラメールの予想 "wikilink")は

\[g_n = O\left((\log p_n)^2\right).\]

という内容である。[Firoozbakhtの予想](https://ja.wikipedia.org/wiki/Firoozbakhtの予想 "wikilink")は\(p_{n}^{1/n}\)（\(p_n\)はn番目の素数）はnの厳密に減少する関数である。すなわち、

\[p_{n+1}^{1/(n+1)} < p_n^{1/n} \text{ for all } n \ge 1.\]

である。この予想が真である場合、関数\(g_n = p_{n+1} - p_n\)は \(g_n < (\log p_n)^2 - \log p_n \text{ for all } n > 4.\)を満たす\[35\]。これはクラメールの予想の強い形を暗示しているが、[Granville](https://ja.wikipedia.org/wiki/Andrew_Granville "wikilink")[Pintzのヒューリスティックとは矛盾している](https://ja.wikipedia.org/wiki/János_Pintz "wikilink")\[36\]\[37\]\[38\]。これは任意の\(\varepsilon>0\)に対して\(g_n > \frac{2-\varepsilon}{e^\gamma}(\log p_n)^2\)が無限回起こる（\(\gamma\)は[オイラーの定数](../Page/オイラーの定数.md "wikilink")）

その一方、[Oppermannの予想](https://ja.wikipedia.org/wiki/Oppermannの予想 "wikilink")はクラメールの予想より弱い。Oppermannの予想で予想される間隔は

\[g_n < \sqrt{p_n}.\] のオーダーである。結果としてOppermannの予想の元では、全ての自然数\(n>m\)に対して\(g_n < \sqrt{p_n}.\)を満たす\(m\)（おそらく\(m=30\)）が存在する。

Oppermannの予想よりも弱い[Andricaの予想](https://ja.wikipedia.org/wiki/Andricaの予想 "wikilink")は

\[g_n < 2\sqrt{p_n} + 1.\]

という内容である\[39\]。これは連続する平方数の間には素数が必ずあるという[ルジャンドル予想](https://ja.wikipedia.org/wiki/ルジャンドル予想 "wikilink")よりは少し強い。

Polignacの予想は、全ての正の偶数*k*が無限の頻度で素数の間隔として生じるという内容である。*k* = 2の場合は[双子素数予想](https://ja.wikipedia.org/wiki/双子素数予想 "wikilink")である。この予想は特定のkの値についてはまだ証明されておらず、反証もされていないが、[張益唐](https://ja.wikipedia.org/wiki/張益唐 "wikilink")の結果は少なくとも1つ（現在のところ未知）の7千万より小さいkの値については真であることが証明されている。上で議論されたように、この上限は246に改良された。

## 数論的関数として

n番目の素数と(n+1)番目の素数の間の間隔*g*<sub>*n*</sub>は[数論的関数](https://ja.wikipedia.org/wiki/数論的関数 "wikilink")の1例である。この文脈では通常*d*<sub>*n*</sub>で表され、素数差分関数()と呼ばれる\[40\]。この関数は[乗法的関数](https://ja.wikipedia.org/wiki/乗法的関数 "wikilink")でも[加法的関数](https://ja.wikipedia.org/wiki/加法的関数 "wikilink")でもない。
