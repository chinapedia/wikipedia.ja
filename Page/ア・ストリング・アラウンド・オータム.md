『**ア・ストリング・アラウンド・オータム**』（*A String Around Autumn* ）は、[日本](https://ja.wikipedia.org/wiki/日本 "wikilink")の[作曲家](../Page/作曲家.md "wikilink")[武満徹](https://ja.wikipedia.org/wiki/武満徹 "wikilink")により、[フランス革命](https://ja.wikipedia.org/wiki/フランス革命 "wikilink")200周年記念行事の一環として開かれた「[パリの秋](https://ja.wikipedia.org/wiki/パリの秋 "wikilink")」フェスティバルの委嘱により[作曲](../Page/作曲.md "wikilink")された[ヴィオラ](../Page/ヴィオラ.md "wikilink")と[オーケストラ](../Page/オーケストラ.md "wikilink")のための作品。

## 作曲・初演

作曲は[1989年](../Page/1989年.md "wikilink")。初演は同年[11月29日](../Page/11月29日.md "wikilink")に、[今井信子](https://ja.wikipedia.org/wiki/今井信子 "wikilink")のヴィオラ独奏、[ケント・ナガノ](../Page/ケント・ナガノ.md "wikilink")[指揮](../Page/指揮_\(音楽\).md "wikilink")、[パリ管弦楽団](https://ja.wikipedia.org/wiki/パリ管弦楽団 "wikilink")によって行われた。

## 曲の構成

作品の[タイトル](https://ja.wikipedia.org/wiki/タイトル "wikilink")は[大岡信](../Page/大岡信.md "wikilink")の英詩集『**秋をたたむ紐**』（A String Around Autumn）の中の「**沈め 詠うな ただ黙して 秋景色をたたむ 紐となれ**」（Be simple:A String Around Autumn）からとられた。

曲は[ドビュッシーの影響をうかがわせる](../Page/クロード・ドビュッシー.md "wikilink")[旋法](../Page/旋法.md "wikilink")的な[動機](https://ja.wikipedia.org/wiki/動機 "wikilink")の扱い方や、半透明の薄い膜を重ね上げていくような[楽器の使用法によって独自の効果を生んでいる](https://ja.wikipedia.org/wiki/オーケストレーション "wikilink")。風に揺れる森の木のざわめきのようなオーケストラの響きの中を独奏ヴィオラがミステリアスにかつ自由に逍遥する趣があり、豊かな[パースペクティブ](https://ja.wikipedia.org/wiki/パースペクティブ "wikilink")と抑揚を兼ね備え、音の鳴っている間にすら静寂を感じさせる。

**E-F♯-A-h-D**という完全4度の[音程](../Page/音程.md "wikilink")の音列と、**A-F**および**C-A♭**の2つの短6度によって構成された”**E-F♯-A-h-C-D-F-A♭**”の動機を武満は「糸のような流れ」と呼び、「さまざまな断片的な旋律的動機によって織り上げられた想像風景である」と述べている。

独奏ヴィオラは、秋景色の中で自然を観照する人間の役割を果たし、武満に最も強い影響を与えたドビュッシーと[メシアンを生んだ](../Page/オリヴィエ・メシアン.md "wikilink")[フランス](https://ja.wikipedia.org/wiki/フランス "wikilink")の人々のために捧げる、との武満自身の言葉が残されている。

[1990年](https://ja.wikipedia.org/wiki/1990年 "wikilink")5月に「[国際モーリス・ラヴェル賞](https://ja.wikipedia.org/wiki/国際モーリス・ラヴェル賞 "wikilink")」を受賞。
