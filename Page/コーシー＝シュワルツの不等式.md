[数学](../Page/数学.md "wikilink")における**コーシー＝シュワルツの不等式**（コーシーシュワルツのふとうしき、）、**シュワルツの不等式**、**シュヴァルツの不等式**あるいは**コーシー＝ブニャコフスキー＝シュワルツの不等式**  とは、[内積空間](https://ja.wikipedia.org/wiki/内積空間 "wikilink")における二つのベクトルの間の[内積](../Page/内積.md "wikilink")がとりうる値をそれぞれのベクトルの[ノルム](../Page/ノルム.md "wikilink")によって評価する不等式である。[線型代数学](../Page/線型代数学.md "wikilink")や[関数解析学](../Page/関数解析学.md "wikilink")における有限次元および無限次元のベクトルに対するさまざまな内積や、[確率論](../Page/確率論.md "wikilink")における[分散や](../Page/分散_\(確率論\).md "wikilink")[共分散](../Page/共分散.md "wikilink")に適用されるなど、様々な異なる状況で現れる有用な不等式である。

数列に対する不等式は[オーギュスタン＝ルイ・コーシー](../Page/オーギュスタン＝ルイ・コーシー.md "wikilink")によって1821年に、積分系での不等式はまず[ヴィクトール・ブニャコフスキー](../Page/ヴィクトール・ブニャコフスキー.md "wikilink")によって1859年に発見された後[ヘルマン・アマンドゥス・シュワルツ](https://ja.wikipedia.org/wiki/ヘルマン・アマンドゥス・シュワルツ "wikilink")によって1888年に再発見された。

## 定理の内容といくつかの事実

*x* や *y* が[実または](../Page/実数.md "wikilink")[複素](../Page/複素数.md "wikilink")[内積空間](https://ja.wikipedia.org/wiki/内積空間 "wikilink") \((X,\langle \cdot,\cdot\rangle)\) の元であるとき、シュワルツの不等式は次のように述べられる：

\[|\langle x,y\rangle|^2 \leq \langle x,x\rangle \cdot \langle y,y\rangle.\]

左辺は内積 \(\langle x,y\rangle\) の[絶対値](https://ja.wikipedia.org/wiki/絶対値 "wikilink")の平方である。ここに、等号は *x* と *y* が[線型従属](https://ja.wikipedia.org/wiki/線型従属 "wikilink")であるとき、つまり *x*, *y* のいずれか一方が 0 であるか、さもなくば一方が他方の適当なスカラー倍であるときであり、かつそのときに限る。内積の導くノルム \(\|x\|^2:=\langle x,x\rangle\) を用いればこれは

  -
    \(|\langle x, y \rangle| \leq \Vert x\Vert\cdot\Vert y\Vert\)

とも表せる。

コーシー・シュワルツの不等式の重要な帰結には、内積が2変数の関数と見て[連続であるということ](https://ja.wikipedia.org/wiki/連続関数 "wikilink")、従って特にひとつのベクトル *x* を決めるごとに内積が一つの連続汎関数 \(\langle x,\cdot\rangle\) あるいは \(\langle \cdot,x\rangle\) を定めるということである。さらに、ベクトル *x* に対して汎関数 \(x^* \colon y \mapsto \langle y,x\rangle\)を与える対応が等長作用素になっていることも従う。

また、この定理の系として[内積ノルムに関する](https://ja.wikipedia.org/wiki/内積空間 "wikilink")[三角不等式](../Page/三角不等式.md "wikilink")

\[\Vert x+y \Vert \le \Vert x\Vert + \Vert y\Vert\] が導かれる。ここで等号が成立するのは、*x* と *y* の一方が他方の非負実数倍であるとき、かつそのときに限る。

## 証明に関する話題

定理には数多くの証明が知られている。

### 判別式による証明

実内積空間におけるシュワルツの不等式の特徴的な証明の一つに、二次式とその判別式を用いるものがある。実際、 なる内積を考えるとき、*t* を実変数（あるいは任意の実定数）として

  -
    \(0 \le \langle x + t|\langle x,y\rangle|y, x + t|\langle x,y\rangle|y \rangle = \langle x, x \rangle + 2 |\langle x, y \rangle|^2 t + |\langle x,y\rangle|^2 \langle y, y\rangle t^2\)

は（内積の性質により）*t* の如何にかかわらず成立する *t* の二次の絶対不等式となる。ゆえに、二次の絶対不等式に関してよく知られた事実により、この *t* に関する二次式の判別式

  -
    \((|\langle x, y \rangle |^2)^2 - \langle x, x\rangle \langle y, y \rangle |\langle x,y\rangle |^2\)

は半負定値（非正）でなければならない。これを整理してシュワルツの不等式を得る。

### 数学的帰納法による証明

別の観点に立った証明として、直交射影の概念を用いる以下のものがある： = 0 のときは、*x* と *y* との内積が 0 になり、問題の不等式は自明な形で等号として成立する。 \> 0 のときは、

  -
    \(t=\frac{\langle x,y\rangle}{\|y\|^2}\)

に対して *t y* を *x* の *y* 方向への直交射影と見なすことができる。実際、この *t* について *z* := *x* - *t y* は *y* に直交している。

  -
    \(\|z\|^2 = \frac{\|x\|^2\|y\|^2-|\langle x,y \rangle|^2}{\|y\|^2}\)

が非負であることよりコーシー＝シュワルツの不等式が従う。さらに、*x* と *y* とが線型従属のときかつそのときに限り *z* = 0 であり、不等式において等号が成立することがわかる。

標準内積に関する内積空間と考えたときの[ユークリッド空間](../Page/ユークリッド空間.md "wikilink") **R**<sup>*n*</sup> の場合に書き下すと、

\[\left(\sum_{i=1}^n x_i y_i\right)^2\leq \left(\sum_{i=1}^n x_i^2\right) \left(\sum_{i=1}^n y_i^2\right)\] となるが、この不等式は *n* に関する数学的帰納法で証明することができる。各 \(x_i,y_i\) が負でない場合を示せばよい。*n* = 1 のときは明らかに成立。*n* = 2 のときは、

\[(x_1^2+x_2^2)(y_1^2+y_2^2)-(x_1 y_1+x_2 y_2)^2=(x_1 y_2-x_2 y_1)^2\ge 0\] より成り立つ。*n* = *m* で成立すると仮定する。*n* = *m* + 1 のとき、

\[\left(\sum_{i=1}^{m+1} x_i y_i\right)^2=\left( \sum_{i=1}^m x_i y_i+x_{m+1} y_{m+1}\right)^2\]

\[\leq\left(\left(\sum_{i=1}^m x_i^2 \right)^\frac{1}{2}\left(\sum_{i=1}^m y_i^2 \right)^\frac{1}{2}+x_{m+1} y_{m+1} \right)^2\] (∵帰納法の仮定より)

\[\leq\left(\sum_{i=1}^m x_i^2+x_{m+1}^2\right)\left(\sum_{i=1}^m y_i^2+y_{m+1}^2 \right)\]     (∵*n*=2のときより)

\[=\left(\sum_{i=1}^{m+1}x_i^2\right) \left(\sum_{i=1}^{m+1}y_i^2\right)\] となって成立する。

## 具体的な例

標準内積に関する内積空間と考えたときの[ユークリッド空間](../Page/ユークリッド空間.md "wikilink") **R**<sup>*n*</sup> の場合に書き下すと、

\[\left(\sum_{i=1}^n x_i y_i\right)^2\leq \left(\sum_{i=1}^n x_i^2\right) \left(\sum_{i=1}^n y_i^2\right)\] となる。特に *n* = 2, 3 のときには

  -
    \((x_1y_1 + x_2y_2)^2 \leq (x_1^2 + x_2^2)(y_1^2 + y_2^2)\)
    \((x_1y_1 + x_2y_2 + x_3y_3)^2 \leq (x_1^2 + x_2^2 + x_3^2)(y_1^2 + y_2^2 + y_3^2)\)

と書ける。これは有限次元の内積空間における例である。無限次元の内積空間の例として、[二乗可積分関数](https://ja.wikipedia.org/wiki/二乗可積分関数 "wikilink")の空間の場合には内積が積分の形で与えられ、2つの自乗可積分関数 *f*, *g* に対して

\[\left|\int f(x)g(x)^*\,dx\right|^2\leq\int \left|f(x)\right|^2\,dx \cdot \int\left|g(x)\right|^2\,dx\]

というのがシュワルツの不等式を表している式である。これらは[ヘルダーの不等式](https://ja.wikipedia.org/wiki/ヘルダーの不等式 "wikilink")に一般化される。
