**音節**（おんせつ）または**シラブル**（）は、連続する[言語音](https://ja.wikipedia.org/wiki/言語音 "wikilink")を区切る分節単位の一種である。典型的には、1個の[母音](https://ja.wikipedia.org/wiki/母音 "wikilink")を中心に、その母音単独で、あるいはその母音の前後に1個または複数個の[子音](https://ja.wikipedia.org/wiki/子音 "wikilink")を伴って構成する[音声](https://ja.wikipedia.org/wiki/音声 "wikilink")（群）で、音声の[聞こえ](https://ja.wikipedia.org/wiki/聞こえ "wikilink")の一種のまとまりをいう。

[中国語](https://ja.wikipedia.org/wiki/中国語 "wikilink")などの声調言語（トーン言語）では、母音と子音の組合せに、さらに母音の音程の高低変化による[声調](../Page/声調.md "wikilink")を加えて一つの音節を構成する。

[日本語](../Page/日本語.md "wikilink")の場合、音節とは区切り方が必ずしも一致しない「[モーラ](https://ja.wikipedia.org/wiki/モーラ "wikilink")」（拍）という分節単位が重要である。

## 音節の構成

### 基本

典型的な音節は「母音を中心とした音のまとまり」であり、次の4種類がある。

1.  母音 (V)
2.  子音＋母音 (CV)
3.  母音＋子音 (VC)
4.  子音＋母音＋子音 (CVC)

中心となる母音 (V) を、**音節主音**（<small>おんせつしゅおん、</small>）または**音節核**（<small>おんせつかく、</small>）と呼ぶ。子音 (C) は、母音の前後にそれぞれ複数個がありうる。母音 (V) は、[二重母音](https://ja.wikipedia.org/wiki/二重母音 "wikilink")もしくは[半母音](https://ja.wikipedia.org/wiki/半母音 "wikilink")を伴う多重母音、または[長母音](https://ja.wikipedia.org/wiki/長母音 "wikilink")でありうる。1音節内に音節主音となる母音が複数個存在することはない（その場合は別の音節に分ける）。

上記の例のうち、1.と2.のように母音で終わる音節を、**開音節**（）という。3.と4.のように子音で終わる音節を、**閉音節**（）という。閉音節の末尾にある子音は、**末尾子音**（）と呼ばれる。

言語によっては、長母音または二重母音を伴う音節が他の音節よりも時間的にやや長く発音されることがある。この場合には、長い音節を**重音節**（）、短い音節を**軽音節**（）と呼ぶ。この区別は[アクセント](../Page/アクセント.md "wikilink")の規則などで重要になることもある（[英語](../Page/英語.md "wikilink")、[ラテン語](../Page/ラテン語.md "wikilink")や一部の[ロマンス語](https://ja.wikipedia.org/wiki/ロマンス語 "wikilink")、[アラビア語](../Page/アラビア語.md "wikilink")、日本語など）。日本語では、重音節が2つの[モーラ](https://ja.wikipedia.org/wiki/モーラ "wikilink")として扱われる。

言語によっては、複数の子音が連続すること（**[子音結合](https://ja.wikipedia.org/wiki/子音結合 "wikilink")**）があり、例えば、**CCC**VC や V**CC** という音節もある。このときの連続した子音のかたまりを**子音群**（）と呼ぶ。

### 音節主音的子音

言語によっては、母音の代わりに、「[聞こえ](https://ja.wikipedia.org/wiki/聞こえ "wikilink")度」の相対的に高い子音を音節主音にし、それ単独で、またはその前後に1個以上のより聞こえ度の低い子音を伴って音節を構成する場合がある。例えば、英単語 needle （[IPA](../Page/国際音声記号.md "wikilink"): ）では、語末の  を音節主音とし、 でひとつの音節が構成されている。英語では、 のほかに、、、 が音節主音になりうる\[1\]。このような音節主音として用いられる子音を、**音節主音的**な子音または**成節子音**という。

[スラヴ諸語](https://ja.wikipedia.org/wiki/スラヴ諸語 "wikilink")では、 や  のほかに、 や  も音節主音的子音として用いられる。したがって、綴り字の上では母音の数が子音に比べて極端に少ないか、または母音を全く含まない単語も多く存在する。例えば、[クロアチア語](https://ja.wikipedia.org/wiki/クロアチア語 "wikilink")の[早口言葉](../Page/早口言葉.md "wikilink") “” や、[チェコ語](../Page/チェコ語.md "wikilink")における母音字が皆無の “****” のような文がある。

中国語の主に南方の方言では、 や  が単語を成すことがある。例えば、「五」は[広東語](../Page/広東語.md "wikilink")で （[イェール式表記](https://ja.wikipedia.org/wiki/イェール式#広東語 "wikilink"): ng5）である。 [中古期の日本語で漢語由来の](https://ja.wikipedia.org/wiki/中古日本語 "wikilink")「馬」「梅」を「むま」「むめ」などと書いた例があるのは、  などと発音したものであるとされる。

[国際音声記号](../Page/国際音声記号.md "wikilink")（IPA）では、子音の下に \[''' **\] （または上に \[** '''\]）を書き加えることによって、その子音が音節主音であることを示す。

### 声調言語

[中国語](https://ja.wikipedia.org/wiki/中国語 "wikilink")、[ベトナム語](https://ja.wikipedia.org/wiki/ベトナム語 "wikilink")、[ハウサ語](../Page/ハウサ語.md "wikilink")など、音節声調をもつ言語においては、母音と子音の組合せ以外に、さらに[声調](../Page/声調.md "wikilink")が加わって一つの音節を構成する。 例えば、[標準中国語](https://ja.wikipedia.org/wiki/標準中国語 "wikilink")（[普通話](https://ja.wikipedia.org/wiki/普通話 "wikilink")）の漢字「光」の発音は、[ピンインで](../Page/ピン音.md "wikilink")  と表記され、音節の構成は次のとおりである。

|                                                   |
| ------------------------------------------------- |
| 音節                                                |
| [声母](https://ja.wikipedia.org/wiki/声母 "wikilink") |
| 韻頭                                                |
| [介音](https://ja.wikipedia.org/wiki/介音 "wikilink") |
| g                                                 |

音節声調には、次の2種がある。

  - **曲線声調** - 音節内で音の高低が変化する。
  - **段位声調** - 音節間での相対的な音程の差がある（音節内での高低変化はない）。

中国語やベトナム語は、曲線声調である。

## 日本語の音節

[日本語](../Page/日本語.md "wikilink")では、閉音節は「[ん](../Page/ん.md "wikilink")」（鼻母音で発音される「ん」を除く）および「[っ](https://ja.wikipedia.org/wiki/っ "wikilink")」で終わる音節だけで、あとは開音節である。 標準的な日本語では、例えば仮名1文字「あ・か・さ・た・な」 /a/ /ka/ /sa/ /ta/ /na/のように「子音＋母音」の開音節を基本とする。 但し、長音、促音、撥音（ん）だけは、音節区切りでは、前の音といっしょに数える。 [日本語\#音韻](https://ja.wikipedia.org/wiki/日本語#音韻 "wikilink")に標準的な日本語で表記での音節の一覧表がある。

特殊な開音節の[拗音](https://ja.wikipedia.org/wiki/拗音 "wikilink")節「きゃ・きゅ・きょ」 /kya/ /kyu/ /kyo/などは仮名2文字で1モーラ1音節である。\[2\]

また、閉音節の「ん」を含む[撥音](https://ja.wikipedia.org/wiki/撥音 "wikilink")節と閉音節の「っ」を含む[促音](https://ja.wikipedia.org/wiki/促音 "wikilink")節は、単独で[モーラ](https://ja.wikipedia.org/wiki/モーラ "wikilink")を構成し、前置された仮名との2文字で2モーラ1音節である。詳細は[捨て仮名](https://ja.wikipedia.org/wiki/捨て仮名 "wikilink")を参照。

「か*あ*さん」「に*い*さん」のような[長音](https://ja.wikipedia.org/wiki/長音 "wikilink")や[長音符](https://ja.wikipedia.org/wiki/長音符 "wikilink")の「ー」で示される長音は、日本語では音声学的には[長母音](https://ja.wikipedia.org/wiki/長母音 "wikilink")である。これも仮名または符号と前置された仮名との2文字で2モーラ1音節である。

また、「ん?」、「んだ」、「ん万円」などのように「ん」が語頭にある場合は「ん」だけで1つの音節を構成し、この場合、「ん」は鼻母音に発音されない限り（即ち子音である限り）「音節主音的」な子音である。

ネイティブの日本語話者には、音節よりも[モーラ](https://ja.wikipedia.org/wiki/モーラ "wikilink")のほうが直感的単位である（たとえば、詩の「七五調」や「五七調」はモーラで数えている）。

[音節](https://ja.wikipedia.org/wiki/音節 "wikilink")、[モーラ](https://ja.wikipedia.org/wiki/モーラ "wikilink")方言（[拍](https://ja.wikipedia.org/wiki/拍 "wikilink")）、[シラビーム方言](https://ja.wikipedia.org/wiki/シラビーム方言 "wikilink")(音節音素)比較の例

また、日本語では語末などで[無声化して聞こえない母音が現れることも多い](https://ja.wikipedia.org/wiki/無声音 "wikilink")（例えば、「です」が des、「ました」が mashta\[3\] のように聞こえるなど）が、[モーラ](https://ja.wikipedia.org/wiki/モーラ "wikilink")数に変化はない（des は2モーラ、mashta は3モーラ）。日本語を外国語として習った者が「ました」と言うとき、日本人の耳に違和感が生じることがあるのは、日本人が無意識に mashta とするところをきちんと mashita と発音しているからである\[4\]。

日本語の中には、[近畿方言](https://ja.wikipedia.org/wiki/近畿方言 "wikilink")のように声調言語としての特徴があり、声調を加えて音節を考えるべきものもある。

## 綴り字と発音

言語によっては、[綴り字](https://ja.wikipedia.org/wiki/綴り字 "wikilink")（スペリング）と発音の対応が一定でなく、見かけ上の音節数が実際のそれと異なる場合がある。その場合、単に文字を数えるだけで正確な音節数がわかるとは限らない。しかし、ある程度の法則性は存在する。

### 英語における無音の “e”

英語では一般的に、無音の  が閉音節の末尾に付く場合、当該音節の核となる母音は、長母音か二重母音として発音され、付かない場合は短母音として発音される。例えば、mad と made では、前者が短母音、後者が二重母音である。bit 対 bite、mod 対 mode なども同様である。長母音の例としては、pet に対する Pete、cut に対する cute などがある。

これらの例は、綴り字上は CVCV（2音節）に見えるが、実際の発音では CVC（1音節）である。なお、このように無音の e を末尾に伴う閉音節における二重母音・長母音は、アルファベットの文字そのものの名称と発音が同じになる（例えば  は 、 は）。

## 音節文字

音節をそのままひとつの文字として表記する文字を[音節文字](https://ja.wikipedia.org/wiki/音節文字 "wikilink")と呼び、日本語における[仮名](https://ja.wikipedia.org/wiki/仮名 "wikilink")をはじめ、中国南西部に居住する[イ族](https://ja.wikipedia.org/wiki/イ族 "wikilink")の[彝文字](https://ja.wikipedia.org/wiki/彝文字 "wikilink")（ロロ文字）や、[19世紀](../Page/19世紀.md "wikilink")前半に[シクウォイア](https://ja.wikipedia.org/wiki/シクウォイア "wikilink")によって発明された[アメリカ](https://ja.wikipedia.org/wiki/アメリカ "wikilink")の[チェロキー文字](https://ja.wikipedia.org/wiki/チェロキー文字 "wikilink")、その影響を受けて発明された[リベリア](../Page/リベリア.md "wikilink")の[ヴァイ文字](https://ja.wikipedia.org/wiki/ヴァイ文字 "wikilink")など、世界中にいくつかの文字体系が存在する。ただし、現代における大言語において音節文字を使用しているのは日本語のみであり、表音文字の大部分は[音素](../Page/音素.md "wikilink")を表す[音素文字](https://ja.wikipedia.org/wiki/音素文字 "wikilink")に属する。また、日本語においても完全な音節文字体系ではなく、[表意文字](https://ja.wikipedia.org/wiki/表意文字 "wikilink")である[漢字](../Page/漢字.md "wikilink")と併用されて用いられる\[5\]。日本語における仮名は、本来表意文字である漢字を、その意味にかかわらず日本語の一音節をあらわすために用いる、いわゆる[万葉仮名](https://ja.wikipedia.org/wiki/万葉仮名 "wikilink")から変化したものである。
