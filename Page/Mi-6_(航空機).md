}} **Mi-6**(ミル6；[ロシア語](https://ja.wikipedia.org/wiki/ロシア語 "wikilink"):****<small>ミー・シェースチ</small>)は、[ソ連の](https://ja.wikipedia.org/wiki/ソビエト連邦 "wikilink")[MVZミーリの開発した多目的輸送](../Page/M・L・ミーリ記念モスクワ・ヘリコプター工場.md "wikilink")[ヘリコプター](../Page/ヘリコプター.md "wikilink")。実用化された中では、後継機である[Mi-26の登場まで世界最大のヘリコプターであった](../Page/Mi-26_\(航空機\).md "wikilink")。大量生産され、軍民問わず多数が運用された。[北大西洋条約機構](../Page/北大西洋条約機構.md "wikilink")(NATO)は、「[フック](../Page/フック.md "wikilink")」()という[NATOコードネーム](../Page/NATOコードネーム.md "wikilink")を付与した。

## 概要

Mi-6の開発は、[1954年](../Page/1954年.md "wikilink")に始められた。[Mi-4中型ヘリコプターの発展型として開発された開発機](https://ja.wikipedia.org/wiki/Mi-4_\(航空機\) "wikilink")**VM-6**(****)が、その試作機となった。なお、この名称は「ミルの6 tヘリコプター」(<small>ヴィルタリョート・ミーリャ・シスチトーンヌィイ</small>)という意味であり、のちの制式名称とは直接は関係ない。機体は**Mi-6**の公式名称を得て、1957年に初飛行した。

機体には特大のギアボックスと強力な[ターボシャフトエンジン](../Page/ターボシャフトエンジン.md "wikilink")が用意され、[固定翼機](../Page/固定翼機.md "wikilink")のような「主翼」が胴体上部に取り付けられた。この「主翼」は、状況に応じ、取り外すことができる。Mi-6はヘリコプターとしては高速の300km/hの最大飛行速度、12000 kgの積載量を発揮し、速度における世界記録を達成するなどした。Mi-6の輸送キャビンの12x2 m、65x2.5 mという寸法は、大型[輸送機](../Page/輸送機.md "wikilink")の[An-8や](https://ja.wikipedia.org/wiki/An-8_\(航空機\) "wikilink")[An-12のものに近いたいへんに大型のものであった](../Page/An-12_\(航空機\).md "wikilink")。機体は、[1957年](../Page/1957年.md "wikilink")[9月](../Page/9月.md "wikilink")になってようやく初飛行にこぎつけた。

完成されたMi-6は、[1965年](../Page/1965年.md "wikilink")に[ル・ブルジェ空港](../Page/ル・ブルジェ空港.md "wikilink")で開催された[パリ国際エアショー](https://ja.wikipedia.org/wiki/パリ国際エアショー "wikilink")で成功裏に展示飛行を行った。

[1971年](https://ja.wikipedia.org/wiki/1971年 "wikilink")には、離陸可能な積載量を44 tに引き上げた改良型**Mi-6A**が初飛行に成功した。この機体の最大飛行速度は、300 km/hを超えた。Mi-6AはMi-6シリーズの第2世代の基本型となり、多くの派生型が製作された。

Mi-6シリーズは、後継機のMi-26が配備されてからは徐々に退役を始め[ソ連崩壊](../Page/ソ連崩壊.md "wikilink")後急速に姿を消したが、現在も一部では使用が続けられている。

## 運用

数機のMi-6の機体は[チェルノブイリ原発事故](https://ja.wikipedia.org/wiki/チェルノブイリ原発事故 "wikilink")の事後処理で投入され、作業終了後放射能汚染の危険があるため、現場近くの空き地に[軍用車](https://ja.wikipedia.org/wiki/軍用車 "wikilink")、[バス](../Page/バス_\(交通機関\).md "wikilink")、[トラック](../Page/トラック.md "wikilink")、[消防車](https://ja.wikipedia.org/wiki/消防車 "wikilink")と共に放置されている。

## 派生型

  - Mi-6 /
    1957年に初飛行した基本型多目的輸送ヘリコプター。
  - Mi-6A /
    1971年に初飛行した発展型多目的輸送ヘリコプター。
  - Mi-6TZ /
    Mi-6の発展型の燃料輸送ヘリコプター()。地上及び航空機への燃料7400 [lを輸送できた](../Page/リットル.md "wikilink")。最初の量産シリーズは**Mi-6TZ-SV**(****)として、[1960年代](../Page/1960年代.md "wikilink")末より配備に入った。
  - Mi-6ATZ /
    [1972年](../Page/1972年.md "wikilink")にMi-6TZの姉妹機で、Mi-6Aの発展型。
  - Mi-6VKP /
    1972年に初飛行した空中[コマンドポスト](https://ja.wikipedia.org/wiki/コマンドポスト "wikilink")型。Mi-6の発展型で、40 名からなる司令部の搭乗が可能であった。
  - Mi-6M /
    Mi-6の派生型の[対潜ヘリコプター](https://ja.wikipedia.org/wiki/対潜ヘリコプター "wikilink")。[1958年](../Page/1958年.md "wikilink")に初飛行を果たしたが、その後[PLAT](https://ja.wikipedia.org/wiki/PLAT_\(魚雷\) "wikilink")()[魚雷](../Page/魚雷.md "wikilink")4 発または[コーンドル](https://ja.wikipedia.org/wiki/コーンドル_\(空対艦ミサイル\) "wikilink")()[空対艦ミサイル](../Page/空対艦ミサイル.md "wikilink")4 発の運用能力を付与された強化型が[1963年](../Page/1963年.md "wikilink")に製造された。機体は試験に入ったものの、国家試験には到らずに開発は中止となった。
  - Mi-6P /
    [1965年](../Page/1965年.md "wikilink")に初飛行した[旅客ヘリコプター](../Page/旅客機.md "wikilink")。70 - 80名の乗客を輸送できた。[アエロフロートなどで長らく運用された](../Page/アエロフロート・ロシア航空.md "wikilink")。
  - Mi-6PZh /
    [消防機](https://ja.wikipedia.org/wiki/消防機 "wikilink")型。[1967年](../Page/1967年.md "wikilink")に初飛行。12t の消火剤を運ぶことができた。[南フランス](https://ja.wikipedia.org/wiki/南フランス "wikilink")で起きた大火災の際に出動したことでも知られる。
  - Mi-6PZh2 /
    Mi-6PZhの改良型で、1972年に初飛行した。新しい消火剤噴射器を搭載した。
  - Mi-6PP /
    [電子偵察機](https://ja.wikipedia.org/wiki/電子偵察機 "wikilink")への妨害活動を行う[電子戦機](../Page/電子戦機.md "wikilink")として開発された。[1962年](../Page/1962年.md "wikilink")に初飛行し、[1980年代](../Page/1980年代.md "wikilink")には再びMi-6PPと命名された機体が製作されたが、いずれも配備には到らなかった。
  - Mi-6PRTBV /
    機動ロケット技術ベース・ヘリコプター()。[1960年](../Page/1960年.md "wikilink")に初飛行した。作業用戦術ロケット・システム8K11()や8K14()または戦術ロケットR-9(やR-10()を搭載できた。Mi-6の発展型として製作されたが、配備には到らなかった。
  - Mi-6RVK /
    ロケット・ヘリコプター・コンプレックス()。1963年に2 機が試験に入った。すなわち、9K53(と9K73()あまたはR-17V()ロケット・コンプレックスである。この機体は地上戦術ロケット・システム9M21ルナー()と作業用戦術ロケット・システム8K114()を搭載した。試験は順調に進み1965年に試験部隊へ配備となり、試験研究が実施された。量産はされなかった。
  - Mi-6PS /
    [1966年](../Page/1966年.md "wikilink")に初飛行した[捜索救難ヘリコプター](https://ja.wikipedia.org/wiki/救難機 "wikilink")()。
  - Mi-6APS /
    Mi-6PSの姉妹機で、Mi-6Aの派生型。
  - Mi-6AYa /
    Mi-6Aの派生型で、[1973年](../Page/1973年.md "wikilink")に初飛行した空中コマンドポスト。40 名までの司令部が搭乗できた。
  - Mi-6VzPU /
    Mi-6AYaの別名。
  - [Mi-22 / {{lang](../Page/Mi-22_\(航空機\).md "wikilink")
    [1975年](../Page/1975年.md "wikilink")に実戦配備に就いたMi-6AYa/VzPUの量産型。
