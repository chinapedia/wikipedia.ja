**射影空間**（しゃえいくうかん、<em lang="en">projective space</em>） とは、その次元が *n* であるとき、(*n* + 1)個の「数」の比全体からなる空間の事をさす。比を構成する「数」をどんな[体](https://ja.wikipedia.org/wiki/可換体 "wikilink")（あるいは[環](https://ja.wikipedia.org/wiki/環_\(数学\) "wikilink")）にとるかによって様々な空間が得られる。[非ユークリッド幾何学](../Page/非ユークリッド幾何学.md "wikilink")のひとつである[射影幾何学](../Page/射影幾何学.md "wikilink")がその概念の端緒であるが、射影空間は[位相幾何学](../Page/位相幾何学.md "wikilink")、[微分幾何学](../Page/微分幾何学.md "wikilink")、[代数幾何学](../Page/代数幾何学.md "wikilink")など幾何学のあらゆる分野にわたって非常に重要な概念である。

## 定義

*K* を[体とする](https://ja.wikipedia.org/wiki/可換体 "wikilink")。*K* 上の *n* 次元の射影空間 *KP<sub>n</sub>* は集合としては、(*n* + 1)個の *K* の要素の比 \[*x*<sub>0</sub> : *x*<sub>1</sub> : ... : *x<sub>n</sub>*\] の全体として定義される。すなわち、[ベクトル空間](https://ja.wikipedia.org/wiki/ベクトル空間 "wikilink") *V* = *K*<sup>*n*+1</sup> の 0 でないベクトルに対して、[同値関係](../Page/同値関係.md "wikilink") (*a*<sub>0</sub>, *a*<sub>1</sub>, ..., *a<sub>n</sub>*) ∼ (*b*<sub>0</sub>, *b*<sub>1</sub>, ..., *b<sub>n</sub>*) を、0 でない *K* の元 *t* が存在して任意の *i* = 0, 1, ..., *n* に対して *b<sub>i</sub>* = *t a<sub>i</sub>* であることとして定義するとき、

である。*K* が[実数](../Page/実数.md "wikilink")体 **R** や[複素数](../Page/複素数.md "wikilink")体 **C** など[位相体であるとき](https://ja.wikipedia.org/wiki/位相環 "wikilink")、その積位相から定まる *K*<sup>*n*+1</sup>  {0} の位相の商位相でもって*KP<sub>n</sub>*は自然に[位相空間](https://ja.wikipedia.org/wiki/位相空間 "wikilink")になる。ベクトル空間 *K*<sup>*n*+1</sup> の座標をひとつ定めると、射影空間の点を比として表す表し方 \[*x*<sub>0</sub> : *x*<sub>1</sub> : ... : *x<sub>n</sub>*\] がひとつ定まる。これを射影空間の**斉次座標**（あるいは同次座標; ）と呼ぶ。

## 多様体の構造

ある斉次座標 \[*x*<sub>0</sub> : *x*<sub>1</sub> : ... : *x<sub>n</sub>*\] に対して、*x*<sub>0</sub> ≠ 0 となる射影空間 *KP<sub>n</sub>* の点全体 *U*<sub>0</sub> は、斉次座標の最初の成分を *x*<sub>0</sub> で割って \[1 : *x*<sub>1</sub> / *x*<sub>0</sub> : ... : *x<sub>n</sub>* / *x*<sub>0</sub>\] とただ一通りに書けるので、*U*<sub>0</sub> は、[アフィン空間](../Page/アフィン空間.md "wikilink") *K<sup>n</sup>* と自然な全単射がある。同様に *x<sub>i</sub>* ≠ 0 となる点全体 *U<sub>i</sub>* も同様にしてアフィン空間との間の全単射 φ<sub>*i*</sub>: *K<sup>n</sup>* → *U<sub>i</sub>* がある。*K* が位相体であるときは全単射 φ<sub>*i*</sub> は位相同型となり、*U*<sub>0</sub>, ..., *U<sub>n</sub>* は *KP<sub>n</sub>* の開被覆となる。φ<sub>*j*</sub><sup>−1</sup> ◦ φ<sub>*i*</sub> が

で与えられる（1/a<sub>j</sub> は第 *i* 成分）ことから、この開被覆は **R***P<sub>n</sub>* に[多様体](../Page/多様体.md "wikilink")の構造（**実射影空間**）を、**C***P<sub>n</sub>* に[複素多様体](https://ja.wikipedia.org/wiki/複素多様体 "wikilink")の構造（**複素射影空間**）を与える。変換関数が有理関数で与えられる事から、任意の体 *K* に対しても、[ザリスキ位相](https://ja.wikipedia.org/wiki/ザリスキ位相 "wikilink")を考える事によって *KP<sub>n</sub>* は[代数多様体](../Page/代数多様体.md "wikilink")となる。

射影空間の概念は純粋に代数的であり非常に標準的であるため、適切な枠組みを用いる事によって、その性質は体 *K* の取り方によらず共通しているものが多い。以下の記述は特に断らない限り、[スキーム論の枠組みを用いる事で任意の体上の代数多様体としての射影空間に対して成り立つが](../Page/概型.md "wikilink")、[代数幾何学](../Page/代数幾何学.md "wikilink")以外で重要な場合は体 *K* が実数体 **R** または複素数体 **C** の場合であるので、実射影空間および複素射影空間の場合に則した記述を行う。

## コンパクト性

体 *K* が実数体 **R** または複素数体 **C** であるとき、これらの位相から定まる位相（ユークリッド位相・古典位相）に関して、射影空間 *KP<sub>n</sub>* は[コンパクトな](../Page/コンパクト空間.md "wikilink")[ハウスドルフ空間](../Page/ハウスドルフ空間.md "wikilink")である。*K* = **R** のときは、射影空間の定義によって、性質のよい自然な写像 *S<sup>n</sup>* → **R***P<sub>n</sub>* があるので、**R***P<sub>n</sub>* のコンパクト性およびハウスドルフ性は *n* 次元球面 *S<sup>n</sup>* のそれから従う。 *K* = **C** の時も同様に *S*<sup>1</sup> [ファイバー束](../Page/ファイバー束.md "wikilink") *S*<sup>2*n*+1</sup> → **C***P<sub>n</sub>* （ホップ束; Hopf bundle と呼ばれる）があることから従う。

一般の体 *K* に関しては射影空間にはザリスキ位相をいれて考えるが、この位相について *KP<sub>n</sub>* は基礎体 *K* 上**固有**  になる。代数多様体の固有性はユークリッド位相に関するコンパクト・ハウスドルフ性の概念の対応物である。

## モジュライ空間としての射影空間

射影空間 *KP<sub>n</sub>* の点 *p* = \[*a*<sub>0</sub> : *a*<sub>1</sub> : ... : *a<sub>n</sub>*\] は、アフィン空間 *K*<sup>*n*+1</sup> 内で、原点と点 (*a*<sub>0</sub>, *a*<sub>1</sub>, ..., *a<sub>n</sub>*) を結ぶ直線 *l<sub>p</sub>* と1対1に対応している。従って、射影空間は *K*<sup>*n*+1</sup> 内の原点を通る直線（あるいは1次元の部分ベクトル空間）をパラメータ付けする空間（[モジュライ空間](https://ja.wikipedia.org/wiki/モジュライ空間 "wikilink")）と見なせる。このモジュライ論的観点からは、射影空間は[グラスマン多様体](https://ja.wikipedia.org/wiki/グラスマン多様体 "wikilink")や[旗多様体](https://ja.wikipedia.org/wiki/旗多様体 "wikilink")の特別な場合と見なせる。

## 射影変換群

一般線形群 *GL*(*n* + 1, *K*) はベクトル空間 *V* = *K*<sup>*n*+1</sup> に原点を固定して作用し、原点を通る直線を原点を通る直線に写すので、射影空間 *KP<sub>n</sub>* には *GL*(*n* + 1, *K*) が作用する。単位行列の定数倍は射影空間に自明に作用するので、この作用は剰余群 *PGL*(*n*, *K*) = *GL*(*n* + 1, *K*)/*K*<sup>×</sup> を経由する。群 *PGL*(*n*, *K*) を*KP<sub>n</sub>* の**[射影変換群](../Page/射影線型群.md "wikilink")**  と言う。射影変換群は、代数多様体としての（あるいは *K* = **C** のときは、複素多様体としての）*KP<sub>n</sub>* の自己同型群にほかならない。\[1\]

## 超平面と双対射影空間

射影空間 *KP<sub>n</sub>* の斉次座標 \[*x*<sub>0</sub> : *x*<sub>1</sub> : ... : *x<sub>n</sub>*\] に対して、方程式

上述の一次方程式は、係数 (*a*<sub>0</sub>, *a*<sub>1</sub>, ..., *a<sub>n</sub>*) を定数倍しても解集合は不変である。従って、*KP<sub>n</sub>* の超平面は比 \[*a*<sub>0</sub> : *a*<sub>1</sub> : ... : *a<sub>n</sub>*\] と1対1に対応している。*KP<sub>n</sub>* の超平面全体をパラメータ付けする空間はこの対応で *KP<sub>n</sub>* と同一視できる。これを**双対射影空間**  という。

同様の理由で、射影空間 *KP<sub>n</sub>* の点 *p* = \[*a*<sub>0</sub> : *a*<sub>1</sub> : ... : *a<sub>n</sub>*\] に方程式 *a*<sub>0</sub>*y*<sub>0</sub> + *a*<sub>1</sub>*y*<sub>1</sub> + *a<sub>n</sub>y<sub>n</sub>* = 0 で定まる *K*<sup>*n*+1</sup> の *n* 次元部分ベクトル空間 *V<sub>p</sub>*を対応させる対応は1対1の対応である。*KP<sub>n</sub>* の自明なベクトル束 ''''K<sup>*n*+1</sup> × *KP<sub>n</sub>* の部分ベクトル束 \(\mathcal V\) を

## 斉次座標環とスキーム論的定義

本節ではまず、複素射影空間 **C***P<sub>n</sub>* について考える。自明束 **C**<sup>*n*+1</sup> × **C***P<sub>n</sub>* の正則切断は *c*<sub>0</sub>*e*<sub>0</sub> + ... + *c<sub>n</sub>e<sub>n</sub>* 、ただし *c<sub>i</sub>* は定数、*e<sub>i</sub>* は *i* 番目の標準基底に値を取る定数切断、と書ける。これが誘導する超平面直線束 \(\mathcal O(1)\) の正則切断を σ で表すことにすると、**C***P<sub>n</sub>* の点 *p* = \[*a*<sub>0</sub> : *a*<sub>1</sub> : ... : *a<sub>n</sub>*\] が σ(*p*) = 0 を満たすことは、点 (*c*<sub>0</sub>, ..., *c<sub>n</sub>*) が平面 *V<sub>p</sub>* に含まれている、すなわち *a*<sub>0</sub>*c*<sub>0</sub> + ... + *a<sub>n</sub>c<sub>n</sub>* = 0 を満たすことを意味しているので、方程式 σ = 0 は、斉次一次式 *c*<sub>0</sub>*x*<sub>0</sub> + ... + *c<sub>n</sub>x<sub>n</sub>* = 0 にほかならない。したがって、超平面直線束 \(\mathcal O(1)\) の正則切断全体の空間 \(\Gamma (\mathbb{C}P_n, \mathcal O(1))\) には斉次一次式の空間（すなわち、*V* = **C**<sup>*n*+1</sup> の双対空間）*V*^ からの単射がある。一方、\(\mathcal O(1)\) の任意の正則切断で定まる[因子は超平面と線形同値になることからこの写像は全射でもある](../Page/因子_\(代数幾何学\).md "wikilink")、すなわち、\(\Gamma (\mathbb{C}P_n, \mathcal O(1))\cong V^{\wedge}\) がわかる。

\(\mathcal O(n)\) を \(\mathcal O(1)\) の *n* 階のテンソル積 \(\mathcal O(1)^{\otimes n}\) として定めると、同様の議論で\(\Gamma (\mathbb{C}P_n, \mathcal O(n)) \cong \mbox{Sym} ^n V^{\wedge}\)が証明できる。射影空間 **C***P<sub>n</sub>* の**斉次座標環**  を

で定義すると、以上の議論から *R* は *V*^ の[対称代数](../Page/多重線型代数.md "wikilink")、すなわち、(*n* + 1)-変数の多項式環になることがわかる。

[スキーム論では](../Page/概型.md "wikilink")、以上の議論の逆をたどって一般の環（より一般には、任意のスキーム）上の射影空間を定義する。*A* を任意の[可換環として](https://ja.wikipedia.org/wiki/環_\(数学\) "wikilink")、 *R* を *A* 上の (*n* + 1) 変数の多項式環 *A*\[*x*<sub>0</sub> , ... , *x<sub>n</sub>*\] とする。*R* を *A* 上の次数環とみて、Proj(*R*) を *R* の無縁イデアル（定数項を持たない多項式全体のなすイデアル）を含まない斉次[素イデアル](../Page/素イデアル.md "wikilink")全体の集合とすると、これは自然に *A* 上のスキームになり、これを *A* 上の射影空間と呼び、スキーム論では通常 \(\mathbb{P}^n_A\) で表す。環 *A* が体 *K* であるときは \(\mathbb{P}^n_K\) の *K* 値点全体は、ザリスキ位相を入れた上記 *KP<sub>n</sub>* と一致し、*R* の次数づけをずらした環 *R*(*n*) に対応する[可逆層](https://ja.wikipedia.org/wiki/可逆層 "wikilink")が上に現れた \(\mathcal O(n)\) と一致する。
