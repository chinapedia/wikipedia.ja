**時間依存密度汎関数法**（じかんいそんみつどはんかんすうほう、、）は、[密度汎関数理論](../Page/密度汎関数理論.md "wikilink")にない[時間発展](https://ja.wikipedia.org/wiki/時間発展 "wikilink")、[時間](../Page/時間.md "wikilink")に依存した概念を導入したもの、及びその試み。[2003年](../Page/2003年.md "wikilink")現在、発展途上であり、まだ確立された理論手法となってはいない。類義語として**TD[LDA](../Page/局所密度近似.md "wikilink")**（）がある。

## 詳細

[波動関数](../Page/波動関数.md "wikilink")をΨ(*t* )、*t* を時間、[ハミルトニアン](../Page/ハミルトニアン.md "wikilink")を*H* とする。この時、出発点としての時間を含む[シュレーディンガー方程式](../Page/シュレーディンガー方程式.md "wikilink")は、

\[i \hbar { \partial \Psi (t) \over {\partial t}} = H \Psi (t)\] となる。ここで、\(\hbar = h / 2 \pi\)（*h* は[プランク定数](../Page/プランク定数.md "wikilink")）である。時刻*t*<sub>0</sub> 、*t* でのそれぞれの波動関数の関係をシュレーディンガー表示で表すと、

\[\Psi (t) = e^{ -{ i \over {\hbar}}  H (t - t_0) } \Psi(t_0)\] となる。少々厳密ではないが、\(t \to t + \Delta t\)、\(t_0 \to \, t\)とし、波動関数の時間発展を、Δ*t* の時間刻みによる逐次的な発展として考えると上式は、

\[\Psi (t + \Delta t) = e^{ - {i \over {\hbar}} H (t + \Delta t - t ) } \Psi (t) = e^{ - {i \over {\hbar}} H \Delta t } \Psi(t)\] となる。問題となるのは、\(e^{ - {i \over {\hbar}} H \Delta t }\)の部分の処理で、*H*Δ*t* に関して冪展開したり、指数関数部分に関して分解（鈴木-トロッター分解）を施すなどしてΔ*t* に関しての逐次計算が行われ方程式が解かれる。

TDDFT (TDLDA)は、ポテンシャル部分が時間に依存する場合、例として時間によって変動する動的な[電場](../Page/電場.md "wikilink")、[磁場](../Page/磁場.md "wikilink")中での電子の振舞いや、[断熱近似](../Page/断熱近似.md "wikilink")が成り立たないような[化学反応](https://ja.wikipedia.org/wiki/化学反応 "wikilink")を扱うような場合などに適用される。但し、この手法は密度汎関数理論が前提であり、正しい結果を与えることが保証されるのは[基底状態](https://ja.wikipedia.org/wiki/基底状態 "wikilink")に対してのみである。上記のような時間依存する系は、準位の交差など[励起状態](https://ja.wikipedia.org/wiki/励起状態 "wikilink")を扱う計算となっている。これまでの実際の計算例などから経験的にこのような励起状態をTDDFTは良く記述できていることが分かっているが、どのような場合でも正しい結果を与える保証はない。
