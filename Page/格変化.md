**格変化**（かくへんか、）とは、[格](../Page/格.md "wikilink")の区別を表す、[名詞](../Page/名詞.md "wikilink")、[形容詞](../Page/形容詞.md "wikilink")、[限定詞](../Page/限定詞.md "wikilink")などの[語形変化](../Page/語形変化.md "wikilink")のことである。

語形変化による格は、ふつう[接尾辞](../Page/接尾辞.md "wikilink")によって表される。による調査では、名詞が格変化する 505 の言語のうち、452 言語は接尾辞によって格の区別を表す\[1\]。たとえば (1) の[バツ語](https://ja.wikipedia.org/wiki/バツ語 "wikilink")の例では、名詞 *Lek’iv* が[能格](../Page/能格.md "wikilink")であることが接尾辞 *-v* によって表されている。  接尾辞以外の手段で格を表す言語は珍しい。ドライヤーの調査では 38 言語が[接頭辞](../Page/接頭辞.md "wikilink")、5 言語が[声調](../Page/声調.md "wikilink")の変化、1 言語が[語幹](../Page/語幹.md "wikilink")の母音の変化によって格を区別する\[2\]。(2) は接頭辞で格を区別する[トンガ語の例である](https://ja.wikipedia.org/wiki/トンガ語_\(ザンビア\) "wikilink")。名詞 *aJoni* が ‘with John’ の意味であることが接頭辞 *a-* によって表されている。

## 格組織の衰退

[言語変化](https://ja.wikipedia.org/wiki/言語変化 "wikilink")によって格組織が衰退し、ごく一部の語だけが格変化を痕跡的に残しているという場合がある\[3\]。たとえば、[インド・ヨーロッパ祖語](../Page/インド・ヨーロッパ祖語.md "wikilink")（印欧祖語）は語形変化によって 8 つの格を区別していたと推定されている。しかし、印欧祖語から派生した言語の一つである[ヒンディー語](../Page/ヒンディー語.md "wikilink")では、表 1 のように名詞が格変化によって区別する格は 3 つだけである。लड़का (laṛkā)「少年」は[男性名詞](../Page/性_\(文法\).md "wikilink")、लड़की (laṛkī)「少女」は[女性名詞の例である](../Page/性_\(文法\).md "wikilink")。

|    | 単数        | 複数        | 単数    | 複数       |
| -- | --------- | --------- | ----- | -------- |
| 直格 | laṛkā     | laṛke     | laṛkī | laṛkiyā̃ |
| 斜格 | laṛke     | laṛkõ     | laṛkī | laṛkiyõ  |
| 呼格 | laṛke     | laṛko     | laṛkī | laṛkiyo  |
|    | लड़का「少年」 | लड़की「少女」 |       |          |

表 1. ヒンディー語の名詞の格変化\[4\]

また、ヒンディー語の[形容詞](../Page/形容詞.md "wikilink")は一部を除いてほとんどが格変化をしない。ヒンディー語では、衰退した格組織に代わって、[後置詞](https://ja.wikipedia.org/wiki/後置詞 "wikilink")が様々な格の区別を表すのに用いられている。

[ドイツ語](../Page/ドイツ語.md "wikilink")も印欧祖語から派生した言語の一つであるが、その格組織はヒンディー語とは別種の変化を遂げている。ドイツ語では、[限定詞](../Page/限定詞.md "wikilink")や句頭の形容詞は格変化によって[主格](../Page/主格.md "wikilink")、[属格](../Page/属格.md "wikilink")、[与格](../Page/与格.md "wikilink")、[対格](../Page/対格.md "wikilink")の 4 つを区別するが、名詞自体はほとんどの場合語形変化しない。すなわち、格の区別はもっぱら限定詞や形容詞の語形変化によって表されており、名詞の格変化はわずかに痕跡として残っているに過ぎない\[5\]。
