**国鉄気動車の車両形式**（こくてつきどうしゃのしゃりょうけいしき）

本項目では、[日本国有鉄道](../Page/日本国有鉄道.md "wikilink")（国鉄）及び[国鉄分割民営化](https://ja.wikipedia.org/wiki/国鉄分割民営化 "wikilink")により国鉄の車両を引き継いだJR各社が保有する[気動車](../Page/気動車.md "wikilink")の[形式番号の付与法則について記述する](https://ja.wikipedia.org/wiki/形式称号 "wikilink")。

## 概要

[thumb](https://ja.wikipedia.org/wiki/ファイル:Dc100ar-fk.jpg "wikilink")200番台\]\]

[気動車](../Page/気動車.md "wikilink")の場合、[キハ181系気動車以降の](https://ja.wikipedia.org/wiki/国鉄キハ181系気動車 "wikilink")**新系列気動車**と、それ以前の在来形気動車とで形式の付番方法が異なり、電車と同様に在来形のものは2桁、新系列（電車の場合は[新性能と呼称](../Page/新性能電車.md "wikilink")）のものは3桁となっている。国鉄時代には、新系列気動車は[特急形気動車と](../Page/特急形車両.md "wikilink")[事業用車](https://ja.wikipedia.org/wiki/事業用車 "wikilink")しか製造されず、特急形以外（[急行形](../Page/急行形車両.md "wikilink")・[一般形](../Page/一般形車両_\(鉄道\).md "wikilink")\[1\]）の車両は在来車との連結運転を優先して2桁の形式の車両が製造された。民営化後は特急形以外にも3桁の形式が登場している。

また、[JR九州キハ72系気動車](https://ja.wikipedia.org/wiki/JR九州キハ72系気動車 "wikilink")や[JR東海キハ75形気動車](../Page/JR東海キハ75形気動車.md "wikilink")、[キハ85系気動車などのように](../Page/JR東海キハ85系気動車.md "wikilink")、JR化後もあえて2桁の形式を用いている場合もある。

このうち、**キ**については気動車(JR東日本ではキハE200形を除き液体式)、**HB**についてはハイブリッド気動車を、**GV**は電気式気動車をそれぞれ指し、○については主に付随車の車種を表す記号を表し、●については用途を、Aは機関方式（JR東日本では駆動方式\[2\]）、Bは車両用途・構造を、CについてはA・Bに付随して定義される数字を指し、Xは製造番号を表す。JR九州（一部の車両）とJR西日本（一部の車両）、JR東海ではAが省かれた以外は、新系列気動車と同じ表記となっているが、HC85系については電車と同じく**クモ**・**モ**で表記される。

| 記号   | 車種  | 補足                           |

| （なし） | 動力車 | 運転台の有無を問わず、動力（走行用エンジン）の有る車両。 |
| ク    | 制御車 | 運転台は有るが、動力を有さない。             |
| サ    | 付随車 | 運転台も無く、且つ動力を有さない。            |

## 在来形気動車

この現在に連なる称号規程は、[1957年](../Page/1957年.md "wikilink")（昭和32年）4月に制定されたものである。それ以前は、[客車](../Page/客車.md "wikilink")の番号体系の中に含まれており、[二軸車および](../Page/二軸車_\(鉄道\).md "wikilink")[片ボギー](https://ja.wikipedia.org/wiki/片ボギー "wikilink")車は5000番台、[ボギー車は](https://ja.wikipedia.org/wiki/ボギー台車 "wikilink")40000番台を称した（いずれも例外あり）。

| 例     | キ | ○ | ● | A | B | _ | XXX  | 車両形式                                                 |
| ----- | - | - | - | - | - | -- | ---- | ---------------------------------------------------- |
| 付随車の例 | キ | ク | ハ | 3 | 5 |    | 301  | [キハ35系気動車キクハ](../Page/国鉄キハ35系気動車.md "wikilink")35形車両 |
| 動力車の例 | キ |   | ハ | 5 | 8 |    | 1107 | [キハ58系気動車キハ](../Page/国鉄キハ58系気動車.md "wikilink")58形気動車 |

このうち、キについては気動車を指し、○については車種を表す記号、●については用途を、Aは車両用途・構造を、Bについては運転台による区別を表す形式番号を指し、Xは製造番号を表す。また、_は空白を表す。

  - 気動車の車種を表す記号
      -
        新系列気動車と同じ。

  - 用途を表す記号
      -
        新系列気動車と同じ。

| 形式番号  | 動力装置方式                                                     |
| ----- | ---------------------------------------------------------- |
| 0     | [機械式、電気式](../Page/気動車・ディーゼル機関車の動力伝達方式.md "wikilink")（例外あり） |
| 1 - 4 | 液体式機関1台                                                    |
| 5     | 液体式機関2台                                                    |
| 6・7   | 大出力機関                                                      |
| 8     | 特急形                                                        |

      -
        [車両番号はおおむね製造順に](https://ja.wikipedia.org/wiki/鉄道の車両番号 "wikilink")1から番号が付けられるが\[11\]、仕様や用途の違いによって番台区分されることがある。（2000番台など）

## JR四国の車両

[JR四国が所有している車両のうち](https://ja.wikipedia.org/wiki/JR四国の車両形式#気動車 "wikilink")、JR化後に新製された車両（電車・気動車）には、形式と車両番号が一体となった4桁の数字のみの表記が用いられており、気動車には千位が1か2の形式番号が使用されている（国鉄から継承した車両とJR他社から購入した車両、およびそれらの改造車は国鉄時代の付番方法のままである）。
