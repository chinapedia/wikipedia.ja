『**And so this is Xmas**』（アンド ソー ディス イズ クリスマス）は、[秦建日子](../Page/秦建日子.md "wikilink")による[長編小説](../Page/長編小説.md "wikilink")。[河出書房新社](../Page/河出書房新社.md "wikilink")より[2016年](../Page/2016年.md "wikilink")[11月24日](../Page/11月24日.md "wikilink")に刊行された\[1\]。[東京](../Page/東京.md "wikilink")を舞台とした連続爆弾[テロ事件を巡る](https://ja.wikipedia.org/wiki/テロリズム "wikilink")[クライム](https://ja.wikipedia.org/wiki/犯罪 "wikilink")[サスペンス](../Page/サスペンス.md "wikilink")。

刊行にあわせて秦の脚本・演出で[舞台化され](../Page/演劇.md "wikilink")、劇団「秦組」によりHATAGUMI vol.7公演として同年9月から10月に上演された\[2\]。

[2019年](../Page/2019年.md "wikilink")[12月3日](../Page/12月3日.md "wikilink")には『**サイレント・トーキョー And so this is Xmas**』と改題され、[河出文庫](../Page/河出文庫.md "wikilink")より文庫化\[3\]。また、[2020年](../Page/2020年.md "wikilink")には『**サイレント・トーキョー**』のタイトルで実写映画化され\[4\]\[5\]\[6\]、同年[12月4日](../Page/12月4日.md "wikilink")に公開予定\[7\]。

## 舞台

劇団「秦組」により、HATAGUMI vol.7公演として[シアターグリーン](../Page/シアターグリーン.md "wikilink")BOX in BOX THEATERにて[2016年](../Page/2016年.md "wikilink")[9月24日](../Page/9月24日.md "wikilink")から[10月23日](../Page/10月23日.md "wikilink")に全25公演にて上演された\[8\]。併演は秦組vol.8公演『月の子供 the Musical』\[9\]。
