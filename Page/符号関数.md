[Signum_function.svg](https://ja.wikipedia.org/wiki/File:Signum_function.svg "fig:Signum_function.svg") **符号関数** (ふごうかんすう、) は、[実数](../Page/実数.md "wikilink")に対しその[符号に応じて](https://ja.wikipedia.org/wiki/プラス記号とマイナス記号 "wikilink")[1](https://ja.wikipedia.org/wiki/1 "wikilink")、[−1](../Page/−1.md "wikilink")、[0](../Page/0.md "wikilink")のいずれかを返す[関数](../Page/関数_\(数学\).md "wikilink")

\[\sgn x = \begin{cases}
  1 & :\ x > 0 \\
  0 & :\ x = 0 \\
 -1 & :\ x < 0 \end{cases}\]

およびそれを拡張した[複素関数](../Page/複素数.md "wikilink")。

記号は  のほかに、   なども使われる。

[英語](../Page/英語.md "wikilink")から「**サイン関数**」とも呼ぶが、この名は[正弦関数](https://ja.wikipedia.org/wiki/正弦関数 "wikilink")  と非常に紛らわしい。区別するために sign の[ラテン語](../Page/ラテン語.md "wikilink")形の signum（シグヌム、英語読みはシグナム）から「**シグナム関数**」(signum function) と呼ぶことがある。英語以外でも[ドイツ語](../Page/ドイツ語.md "wikilink")などいくつかの言語で signum 系の名前で呼ばれる。

## 複素数への拡張

実数に対する符号関数は

\[\sgn x = \begin{cases}
  {x}/{|x|} & :\ x \ne 0 \\
  0         & :\ x = 0 \end{cases}\] と書くこともできる。符号関数の複素数への拡張は、この式を複素数へも適用することで得られる。複素数に対する符号関数は、[複素数平面](https://ja.wikipedia.org/wiki/複素数平面 "wikilink")上で[ベクトルに対し同方向の](https://ja.wikipedia.org/wiki/ベクトル_\(数学\) "wikilink")[単位ベクトル](../Page/単位ベクトル.md "wikilink")を求める操作と同等である（ただし[零ベクトル](https://ja.wikipedia.org/wiki/零ベクトル "wikilink")以外のとき）。

なおこのほかに、[gnuplot](https://ja.wikipedia.org/wiki/gnuplot "wikilink")では、複素数に対し符号関数を

\[\sgn x = \sgn \operatorname{Re}\,x\] と定義している。また[Maple](../Page/Maple.md "wikilink") Vでは

\[\operatorname{csgn}\,x = \begin{cases}
       \sgn \operatorname{Re}\,x & :\ \operatorname{Re}\,x \ne 0 \\
       \sgn \operatorname{Im}\,x & :\ \operatorname{Re}\,x = 0 \end{cases}\] という関数を定義している（\(\operatorname{Re}\) と \(\operatorname{Im}\) はそれぞれ複素数の実部と虚部）。しかしこれ以降は、これらの定義は使わず、最初の定義の符号関数について述べる。

## 性質

符号関数は、以下のような性質を持つ（これらは複素数に対し成り立つ）。

極形式との関係:

  - <math>|\\sgn x| = \\begin{cases}

` 1 & :\ x \neq 0 \\`
` 0 & :\ x = 0 \end{cases}`</math>

  - \(\arg \sgn x = \arg x \,\) 、ただし \(\arg\) は偏角
  - \(x = (\sgn x) |x| \,\)
  - \(\sgn x = e^{i \arg x} \quad (x \ne 0)\)

符号の演算:

  - \(\sgn -x = -\sgn x \,\)
  - \(\sgn \frac{1}{x} = \frac{1}{\sgn x} \quad (x \ne 0)\)
  - \(\sgn xy = \sgn x \sgn y \,\)
  - \(\sgn \frac{x}{y} = \frac{\sgn x}{\sgn y} \quad (y \ne 0)\)
  - \(\sgn x^y = (\sgn x)^y\,\)

実数に対しては、加えて次のような性質を持つ。

  - \(\frac d {d x} \sgn x = 2 \operatorname{\delta}(x)\) 、ただし \(\operatorname{\delta}\) は[ディラックのデルタ関数](../Page/ディラックのデルタ関数.md "wikilink")
  - \(\sgn x = \frac d {d x} |x| \quad (x \ne 0)\)
  - \(\sgn x = 2 \operatorname{H}_{1/2}(x) - 1 \,\) 、ただし \(\operatorname{H_{1/2}}\) は[ヘヴィサイドの階段関数](../Page/ヘヴィサイドの階段関数.md "wikilink")
  - \(\sgn x = [x > 0] - [x < 0] \,\) 、ただし \([\,]\) は[アイヴァーソンのブラケット](https://ja.wikipedia.org/wiki/アイバーソンの記法 "wikilink")
  - \(\lim_{k \to \infty} \tanh kx = \frac \pi 2 \sgn x\)
