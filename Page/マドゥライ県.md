**マドゥライ県（மதுரை மாவட்டம் ; Madurai district）**は、[インド](../Page/インド.md "wikilink")南部の[タミル・ナードゥ州](../Page/タミル・ナードゥ州.md "wikilink")に属する30の県の一つ。県庁所在地は[マドゥライ](../Page/マドゥライ.md "wikilink")。

## 概要

[2001年](../Page/2001年.md "wikilink")の国勢調査では、面積3452平方キロメートル、人口257万8201人、人口密度は747人/[km<sup>2</sup>](https://ja.wikipedia.org/wiki/平方キロメートル "wikilink")。 北は[ティンドゥッカル県](../Page/ティンドゥッカル県.md "wikilink")、東は[シヴァガンガイ県](../Page/シヴァガンガイ県.md "wikilink")、南は[ヴィルドゥナガル県](../Page/ヴィルドゥナガル県.md "wikilink")、西は[テーニ県](../Page/テーニ県.md "wikilink")に接する。北では[ティルッチラーッパッリ県](https://ja.wikipedia.org/wiki/ティルッチラーッパッリ県 "wikilink")にも僅かに接している。 [マドゥライ](../Page/マドゥライ.md "wikilink")の市内には、著名な歴史的建造物である[ヒンドゥー教](../Page/ヒンドゥー教.md "wikilink")寺院の[ミーナークシ寺院](https://ja.wikipedia.org/wiki/ミーナークシ寺院 "wikilink")がある。

## 歴史

古くは[マドゥライ](../Page/マドゥライ.md "wikilink")は、[パーンディヤ朝](../Page/パーンディヤ朝.md "wikilink")の首都であった。*詳細は[マドゥライ](../Page/マドゥライ.md "wikilink")を参照。*

マドゥライ県は、[1968年](https://ja.wikipedia.org/wiki/1968年 "wikilink")の[タミル・ナードゥ州](../Page/タミル・ナードゥ州.md "wikilink")発足以前の、[イギリス領インド帝国](../Page/イギリス領インド帝国.md "wikilink")[マドラス管区](https://ja.wikipedia.org/wiki/マドラス管区 "wikilink")時代から存在する。マドラス管区時代は、**マドゥラ県**（Madura district）という英語名が正式名称であった。

[1985年](https://ja.wikipedia.org/wiki/1985年 "wikilink")9月15日、北部が[ティンドゥッカル県](../Page/ティンドゥッカル県.md "wikilink")として分離。[1997年](https://ja.wikipedia.org/wiki/1997年 "wikilink")1月1日には西側が[テーニ県](../Page/テーニ県.md "wikilink")として分離した。

## 行政区分

### 郡

マドゥライ県は、以下の7つの郡（[タミル語](../Page/タミル語.md "wikilink") வட்டம் 「円」 ; 英訳 taluk）に区分けされている。

  - メールール郡（மேலூர் ; Melur）
  - マドゥライ北郡（வட மதுரை ; Madurai-North）
  - マドゥライ南郡（தென் மதுரை ; Madurai-South）
  - ヴァーディッパッティ郡（வாடிப்பட்டி ; Vadipatti）
  - ウシランパッティ郡（உசிலம்பட்டி ; Usilampatti）
  - ティルマンガラム郡（திருமங்கலம் ; Thirumangalam）
  - ペーライユール郡（பேரையூர் ; Peraiyur）

### 区

マドゥライ県は、以下の13の区（英訳 block）に区分けされている。

  - コッターンパッティ区（கொட்டாம்பட்டி ; Kottampatti）
  - メールール区（மேலூர் ; Melur）
  - アランカーナッルール区（அலங்காநல்லூர் ; Alanganallur）
  - マドゥライ西区（மேல் மதுரை ; Madurai-West）
  - マドゥライ東区（கீழ் மதுரை ; Madurai-East）
  - ヴァーディッパッティ区（வாடிப்பட்டி ; Vadipatti）
  - ティルッパランクンダム区（திருப்பரங்குன்றம் ; Thirupparankundram）
  - セッランパッティ区（செல்லம்பட்டி ; Chellampatti）？
  - ティルマンガラム区（திருமங்கலம் ; Thirumangalam）
  - カッリクディ区（கள்ளிகுடி ; Kallikudi）
  - ウシランパッティ区（உசிலம்பட்டி ; Usilampatti）
  - セーダッパッティ区（சேடப்பட்டி ; Sedapatti）
  - ティー・カッルッパッティ区（டி கல்லுப்பட்டி ; T Kallupatti）
