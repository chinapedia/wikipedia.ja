**ロブレス**（Robles）は、スペイン語圏の姓。

  - [アルフォンソ・ガルシア・ロブレス](https://ja.wikipedia.org/wiki/アルフォンソ・ガルシア・ロブレス "wikilink")(1911 - 1991) - [メキシコ](../Page/メキシコ.md "wikilink")の[外交官](../Page/外交官.md "wikilink")、[政治家](https://ja.wikipedia.org/wiki/政治家 "wikilink")。[1982年](../Page/1982年.md "wikilink")に[ノーベル平和賞](../Page/ノーベル平和賞.md "wikilink")受賞。

  - (1976 - ) - メキシコの[野球](../Page/野球.md "wikilink")選手（[内野手](../Page/内野手.md "wikilink")）。MLB通算110安打。

  - [ジョエル・ロブレス](https://ja.wikipedia.org/wiki/ジョエル・ロブレス "wikilink")(1990 - ) - [スペイン](https://ja.wikipedia.org/wiki/スペイン "wikilink")の[サッカー](https://ja.wikipedia.org/wiki/サッカー "wikilink")選手（[GK](https://ja.wikipedia.org/wiki/ゴールキーパー_\(サッカー\) "wikilink")）。

  - [ジョン＝ジョン・ロブレス](https://ja.wikipedia.org/wiki/ジョン＝ジョン・ロブレス "wikilink")(1986 - ) - [フィリピン](https://ja.wikipedia.org/wiki/フィリピン "wikilink")の野球選手。

  - (1946 - ) - メキシコの野球選手（[捕手](../Page/捕手.md "wikilink")）。MLB通算2安打。

  - [ダイロン・ロブレス](https://ja.wikipedia.org/wiki/ダイロン・ロブレス "wikilink")(1986 - ) - [キューバ](../Page/キューバ.md "wikilink")の[陸上競技](../Page/陸上競技.md "wikilink")選手（[障害走](https://ja.wikipedia.org/wiki/障害走 "wikilink")）。[2008年北京オリンピック金メダリスト](https://ja.wikipedia.org/wiki/2008年北京オリンピックの陸上競技 "wikilink")。

  - [ダニエル・アロミア・ロブレス](https://ja.wikipedia.org/wiki/ダニエル・アロミア・ロブレス "wikilink")(1871 - 1942) - [ペルー](https://ja.wikipedia.org/wiki/ペルー "wikilink")の[作曲家](../Page/作曲家.md "wikilink")。『[コンドルは飛んでいく](https://ja.wikipedia.org/wiki/コンドルは飛んでいく "wikilink")』他。

  - [マリサ・ロブレス](https://ja.wikipedia.org/wiki/マリサ・ロブレス "wikilink")(1937 - ) - スペインの[ハープ](../Page/ハープ.md "wikilink")奏者。

  - [ハンセル・ロブレス](https://ja.wikipedia.org/wiki/ハンセル・ロブレス "wikilink")(1990 - ) - アメリカの野球選手（[投手](../Page/投手.md "wikilink")）。

  - [ビクター・ロブレス](https://ja.wikipedia.org/wiki/ビクター・ロブレス "wikilink")(1997 - ) - アメリカの野球選手（[外野手](../Page/外野手.md "wikilink")）。

  - [アルマンド・ロブレス・ゴドイ](https://ja.wikipedia.org/wiki/アルマンド・ロブレス・ゴドイ "wikilink")(1923 - 2010) - ペルーの[脚本家](../Page/脚本家.md "wikilink")、[映画監督](../Page/映画監督.md "wikilink")。『[みどりの壁](https://ja.wikipedia.org/wiki/みどりの壁 "wikilink")』、『[砂のミラージュ](https://ja.wikipedia.org/wiki/砂のミラージュ "wikilink")』他。

  - [フランシスコ・ロブレス・オルテガ](https://ja.wikipedia.org/wiki/フランシスコ・ロブレス・オルテガ "wikilink")(1949 - ) - メキシコの[カトリック教会](https://ja.wikipedia.org/wiki/カトリック教会 "wikilink")の[司教](../Page/司教.md "wikilink")、[枢機卿](../Page/枢機卿.md "wikilink")。

  - (1981 - ) - スペインのサッカー選手（[MF](../Page/ミッドフィールダー.md "wikilink")）。[リーガ・エスパニョーラ](../Page/リーガ・エスパニョーラ.md "wikilink")（1部）通算30試合出場。

  - [ホセ・マリア・ヒル＝ロブレス・イ・ヒル＝デルガド](https://ja.wikipedia.org/wiki/ホセ・マリア・ヒル＝ロブレス・イ・ヒル＝デルガド "wikilink")(1935 - ) - スペインの[弁護士](../Page/弁護士.md "wikilink")、政治家。

  - (1989 - ) - ベネズエラの野球選手（[投手](../Page/投手.md "wikilink")）。通算3試合登板。

  - [マヌエル・ロブレス・ペスエラ](https://ja.wikipedia.org/wiki/マヌエル・ロブレス・ペスエラ "wikilink")(1817 - 1862) - メキシコの軍人、[大統領](https://ja.wikipedia.org/wiki/メキシコの大統領 "wikilink")。

  - (1947 - ) - [ドミニカ共和国](../Page/ドミニカ共和国.md "wikilink")の野球選手（[遊撃手](../Page/遊撃手.md "wikilink")）。MLB通算25安打。

  - [ルイス・ロブレス](https://ja.wikipedia.org/wiki/ルイス・ロブレス "wikilink")(1984 - ) - アメリカのサッカー選手（[GK](https://ja.wikipedia.org/wiki/ゴールキーパー_\(サッカー\) "wikilink")）。[同国代表通算](https://ja.wikipedia.org/wiki/サッカーアメリカ代表 "wikilink")3試合出場。
