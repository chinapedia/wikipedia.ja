**KDE 1**は、[KDE](../Page/KDE.md "wikilink")（**K Desktop Environment**）の最初のシリーズである。このシリーズには2つのメジャーリリースが含まれる。

## プレリリース

[Screenshot_of_KDE_0.1_A.jpg](https://ja.wikipedia.org/wiki/File:Screenshot_of_KDE_0.1_A.jpg "fig:Screenshot_of_KDE_0.1_A.jpg")

[マティアス・エトリッヒ](../Page/マティアス・エトリッヒ.md "wikilink")の*Kool Desktop Environment*創設の宣言\[1\]（[1996年](../Page/1996年.md "wikilink")[10月14日](../Page/10月14日.md "wikilink")）のすぐあとに、開発は開始された。*Kool*の語はのちに落ち、単純に*K Desktop Environmet*となる\[2\]。

最初の調整されたリリースは*Beta 1*（[1997年](https://ja.wikipedia.org/wiki/1997年 "wikilink")[10月20日](../Page/10月20日.md "wikilink")）である\[3\]。これは当初の宣言から1年後のものであった。追加のベータリリースは、1997年[11月23日](../Page/11月23日.md "wikilink")、[1998年](https://ja.wikipedia.org/wiki/1998年 "wikilink")[2月1日](../Page/2月1日.md "wikilink")、同年[4月19日](../Page/4月19日.md "wikilink")にも行われている。

## K Desktop Environment 1.0

[KDE_1.0.png](https://ja.wikipedia.org/wiki/File:KDE_1.0.png "fig:KDE_1.0.png")

1998年[7月12日](../Page/7月12日.md "wikilink")、K Desktop Environment 1.0の完成バージョンがリリースされた\[4\]。このバージョンに対する評価は分かれた。多くの批評は、[フリーソフトウェア](../Page/フリーソフトウェア.md "wikilink")ではなかった[Qt](../Page/Qt.md "wikilink")の利用に関するもので、[Motifや](../Page/Motif_\(GUI\).md "wikilink")[LessTif](https://ja.wikipedia.org/wiki/LessTif "wikilink")を代わりに使うようアドバイスしていた。このような批判にも拘わらず、KDEは多くのユーザによって受け入れられ、[Linuxディストリビューション](../Page/Linuxディストリビューション.md "wikilink")でも用いられた\[5\]。

## K Desktop Environment 1.1

[KDE_1.1.jpg](https://ja.wikipedia.org/wiki/File:KDE_1.1.jpg "fig:KDE_1.1.jpg")

アップデートバージョンであるK Desktop Environment 1.1は、より高速で安定していて、多くの小さな改善点を含んでいた\[6\]。また、このリリースには、新しいアイコンや背景、テクスチャのセットが含まれている。

### KDE Restorationプロジェクト

KDEの20周年を記念して、KDEと[Fedora](../Page/Fedora.md "wikilink")に対する貢献者であるHelio Chissini de Castroは、1.1.2を[2016年](../Page/2016年.md "wikilink")[10月14日](../Page/10月14日.md "wikilink")に再リリースした\[7\]。

この再リリース版は現代のLinuxとの互換性のために必要となるいくつかの変更を含んでいる。Castroは現代のLinuxシステムでのQt 1.45のコンパイルも示している\[8\]。
