**力率**（りきりつ、Power factor）は、[交流](../Page/交流.md "wikilink")[電力](../Page/電力.md "wikilink")の効率に関して定義された値であり、[皮相電力に対する](https://ja.wikipedia.org/wiki/電力#皮相電力\(apparent_power\) "wikilink")[有効電力の](https://ja.wikipedia.org/wiki/電力#有効電力（effective_power） "wikilink")[割合](../Page/割合.md "wikilink")である。[料金計算などの電力の管理では](https://ja.wikipedia.org/wiki/電気料金 "wikilink")、[パーセント](../Page/パーセント.md "wikilink")で表される。

## 概要

力率は0から1の値をとり、[抵抗](../Page/電気抵抗.md "wikilink")[負荷](https://ja.wikipedia.org/wiki/負荷 "wikilink")では全て有効電力のため1（= 100 %）、電力を消費しない[誘導負荷および](../Page/インダクタンス.md "wikilink")[容量負荷では全て](../Page/静電容量.md "wikilink")[無効電力のため](https://ja.wikipedia.org/wiki/電力#無効電力 "wikilink")0となる。

[正弦波](../Page/正弦波.md "wikilink")交流において、[RLCからなる](https://ja.wikipedia.org/wiki/RLC回路 "wikilink")[線形負荷では](../Page/線型性.md "wikilink")、[インピーダンス](../Page/インピーダンス.md "wikilink")に応じて[電圧](../Page/電圧.md "wikilink")と[電流](../Page/電流.md "wikilink")がずれ、その[位相](../Page/位相.md "wikilink")差の[余弦](https://ja.wikipedia.org/wiki/余弦 "wikilink")となる。

正弦波[電圧源](https://ja.wikipedia.org/wiki/電圧源 "wikilink")に対し、電流が正弦波でなくなる非[線形負荷では](../Page/線型性.md "wikilink")、力率は1未満となる。

## 電力との関係

[計算](../Page/計算.md "wikilink")の簡易化のため、総合[インピーダンス](../Page/インピーダンス.md "wikilink")をZ、[電気抵抗](../Page/電気抵抗.md "wikilink")成分をR、[リアクタンス](../Page/リアクタンス.md "wikilink")成分をX、総合[アドミタンス](../Page/アドミタンス.md "wikilink")をY、[電気伝導](../Page/電気伝導.md "wikilink")成分をG、[サセプタンス](../Page/サセプタンス.md "wikilink")成分をB、加える[電圧](../Page/電圧.md "wikilink")の[実効値](../Page/実効値.md "wikilink")をV<sub>e</sub>、流れる[電流](../Page/電流.md "wikilink")の実効値をI<sub>e</sub>、電流と電圧の位相差をθとする。また、[皮相電力](https://ja.wikipedia.org/wiki/皮相電力 "wikilink")をS、有効電力をP、無効電力をQとする。

有効電力の皮相電力に対する割合で表すと

\= \\frac{R}{\\sqrt{R^2 + X^2}} = \\frac{G}{\\sqrt{G^2 + B^2}}</math>}}

また、位相角を利用して表すと

{{Indent|\(\theta = \tan^{-1} \frac{Q}{P} = \tan^{-1} \frac{{V_e}^2 X}{{V_e}^2 R} = \tan^{-1} \frac{X}{R} = \tan^{-1} \frac{{I_e}^2 B}{{I_e}^2 G}  = \tan^{-1} \frac{B}{G}\)}}

このときのcosθ=P/Sである。

| 回路      | 力率                                                                        |
| ------- | ------------------------------------------------------------------------- |
| RL直列回路  | \(\frac{R}{\sqrt{R^2 + (\omega L)^2}}\)                                   |
| RC直列回路  | \(-\cfrac{R}{\sqrt{R^2 + \left(\cfrac{1}{\omega C}\right)^2}}\)           |
| RLC直列回路 | \(\cfrac{R}{\sqrt{R^2 + \left(\omega L - \cfrac{1}{\omega C}\right)^2}}\) |

## 力率の管理

[Cosfimeter.jpg](https://ja.wikipedia.org/wiki/File:Cosfimeter.jpg "fig:Cosfimeter.jpg") インピーダンスのリアクタンス成分によって、交流電圧と交流電流の位相に差が生じ、負荷回路で[消費される電力よりも多くの電力の](https://ja.wikipedia.org/wiki/消費電力 "wikilink")[配電](../Page/配電.md "wikilink")が必要となる。そのため、[電気工作物](../Page/電気工作物.md "wikilink")の必要容量の増大や損失の増加、[電圧の低下](https://ja.wikipedia.org/wiki/電圧低下 "wikilink")・上昇による障害などが起こる。それを軽減するため[コンデンサ](../Page/コンデンサ.md "wikilink")や[リアクトル](https://ja.wikipedia.org/wiki/リアクトル "wikilink")などを回路に接続し、力率の管理をおこなう。

[エネルギー](https://ja.wikipedia.org/wiki/エネルギー "wikilink")として電力を必要とする装置は、容量性負荷のものより誘導性負荷のもの（[インダクタンス](../Page/インダクタンス.md "wikilink")成分）が多く、誘導性負荷の力率改善のためのにコンデンサを用いることが多い。この用途で用いるコンデンサのことを[進相コンデンサ](../Page/進相コンデンサ.md "wikilink")と言う\[1\]。 誘導性負荷では電圧に対して電流の位相が遅れるため、この位相を合わせる方向に調整することで力率が向上する。具体的な誘導性負荷の装置としては、[モーターが代表的なものであり](../Page/電動機.md "wikilink")、[照明](../Page/照明.md "wikilink")用の[蛍光灯](../Page/蛍光灯.md "wikilink")（の安定器）も誘導性を示す。交流電源を使用する[家庭電化製品では](../Page/家庭用電気機械器具.md "wikilink")、純粋な抵抗負荷（力率1）である[白熱電球](../Page/白熱電球.md "wikilink")・[電気ストーブ](../Page/電気ストーブ.md "wikilink")・[トースター](../Page/トースター.md "wikilink")等の例外を除いて [変圧器](../Page/変圧器.md "wikilink")（あるいは[スイッチング電源](../Page/スイッチング電源.md "wikilink")）やモーターを内蔵しているため、、ほとんどが誘導性負荷である。

[高圧](../Page/高圧.md "wikilink")受電の場合、[低圧](https://ja.wikipedia.org/wiki/低圧 "wikilink")回路に力率改善回路を分散設置すると高価になるが、高圧[受電設備](https://ja.wikipedia.org/wiki/受電設備 "wikilink")の容量が低減でき[高調波](../Page/高調波.md "wikilink")障害の局在化につながる。また、高圧回路に接続すると受電設備の容量は大きくなるが安価である。

日本では、一定以上の[受電設備](https://ja.wikipedia.org/wiki/受電設備 "wikilink")容量の需要家は、力率によって電気[料金](https://ja.wikipedia.org/wiki/料金 "wikilink")の割引・割増が決まる制度がある\[2\]。

力率は有効電力と無効電力から計算により求めることも可能であるが、力率計という測定器もある。
