**Microsoft Windows Neptune**（マイクロソフト ウインドウズ ネプチューン）は、[マイクロソフト](../Page/マイクロソフト.md "wikilink")のベータ版[オペレーティングシステム](../Page/オペレーティングシステム.md "wikilink")（OS）のコードネームである。当時はまだビジネス向けと位置づけられていた[Windows 2000をベースに](../Page/Microsoft_Windows_2000.md "wikilink")[Windows NT系のカーネルで](../Page/Windows_NT系.md "wikilink")、一般家庭向けOSを作る計画であったが頓挫し、代わりに9x系列カーネルで[Windows Meがリリースされた](../Page/Microsoft_Windows_Millennium_Edition.md "wikilink")。

バージョン5.1として開発されていたが、[2000年](../Page/2000年.md "wikilink")1月頃に、Microsoft Neptune開発チームとWindows Odyssey（後述）開発チームが結合し、Whistler（[Windows XP](../Page/Microsoft_Windows_XP.md "wikilink")）へと方針転換された。

## 機能

  - ログイン画面
    Welcome（ようこそ）と共にユーザーの画像と名前を表示するログイン画面。
  - Internet Connection Firewall（後の[Windows ファイアウォール](https://ja.wikipedia.org/wiki/Windows_ファイアウォール "wikilink")）
    [ファイアウォール](../Page/ファイアウォール.md "wikilink")が搭載されており、ネットワークに安全に接続できるような対策が施されている。

## コンセプト

  - スタイル
    [GUIのコンセプトはWindows](https://ja.wikipedia.org/wiki/グラフィカルユーザインタフェース "wikilink") XPというよりも、[Windows Vistaに近い](https://ja.wikipedia.org/wiki/Microsoft_Windows_Vista "wikilink")。
  - UI設計
    [ユーザインタフェース](../Page/ユーザインタフェース.md "wikilink")の設計は、Neptuneでは[HTMLベースのマークアップ言語で構築できるような構想であった](../Page/HyperText_Markup_Language.md "wikilink")。
    Windows Vistaでは[XAML](../Page/Extensible_Application_Markup_Language.md "wikilink")（[XML](../Page/Extensible_Markup_Language.md "wikilink")）でユーザインタフェースを構築できる。

## ビルド番号

Windows NeptuneはWindows 2000の正式リリースの約2ヶ月前にすでに5000番台になっていた（Windows 2000はBuild 2195）。

また、初版であるビルド5000と後に発表されたビルド5111ではインターフェースが異なっている。

## Microsoft Windows Odyssey

**Microsoft Windows Odyssey**（マイクロソフト ウィンドウズ オデッセイ）は、[マイクロソフト](../Page/マイクロソフト.md "wikilink")のベータ版[オペレーティングシステム](../Page/オペレーティングシステム.md "wikilink")（OS）のコードネームである。[1999年](../Page/1999年.md "wikilink")頃に開発中止した。ビルド番号は5154。 ビルドがコンパイルされたかどうかは不明だが、おそらく計画段階でキャンセルされた。
