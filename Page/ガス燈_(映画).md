『**ガス燈**』（ガスとう、***Gaslight***）はの[戯曲](../Page/戯曲.md "wikilink")。それを元にした[サスペンス映画](../Page/サスペンス映画.md "wikilink")として、[1940年](../Page/1940年.md "wikilink")の[英国版と](https://ja.wikipedia.org/wiki/イギリスの映画 "wikilink")[1944年](../Page/1944年.md "wikilink")の[米国版があり](../Page/アメリカ合衆国の映画.md "wikilink")、[イングリッド・バーグマン](../Page/イングリッド・バーグマン.md "wikilink")が[アカデミー主演女優賞](../Page/アカデミー主演女優賞.md "wikilink")を受賞した後者がよく知られている。

オリジナルの『ガス燈』は[1939年](../Page/1939年.md "wikilink")[1月](https://ja.wikipedia.org/wiki/1月 "wikilink")に[ロンドン](../Page/ロンドン.md "wikilink")で舞台上演されテレビ中継もされた。米国では[1941年](https://ja.wikipedia.org/wiki/1941年 "wikilink")[12月](https://ja.wikipedia.org/wiki/12月 "wikilink")に『*Angel Street*』と改題されて[ブロードウェイで上演され](../Page/ブロードウェイ_\(ニューヨーク\).md "wikilink")、3年に渡るロングランとなった。そのブロードウェイ公演では主人公ジャック・マニンガムを[ヴィンセント・プライス](../Page/ヴィンセント・プライス.md "wikilink")、ラフ刑事を[レオ・G・キャロル](https://ja.wikipedia.org/wiki/レオ・G・キャロル "wikilink")が演じた\[1\]。

この作品（特に1944年の米国版映画）の内容から、[1970年代](../Page/1970年代.md "wikilink")後半以降「[ガスライティング](https://ja.wikipedia.org/wiki/ガスライティング "wikilink")」が[心理的虐待](https://ja.wikipedia.org/wiki/心理的虐待 "wikilink")を表す用語として使われるようになった\[2\]。

## あらすじ

霧深い[ロンドン](../Page/ロンドン.md "wikilink")に、ある夫婦が暮らしている。夫に「[物忘れや盗癖が目立つ](https://ja.wikipedia.org/wiki/忘却 "wikilink")」と指摘された妻は自分がおかしくなったのだと思い込み、不安に苛まれるようになる。しかし、それは夫がそう言い聞かせることで妻を精神的に追い込んでいたからだった。そこにはかつてアリスという裕福な女性が殺害され、宝石が盗まれた事件が関係しており、その犯人はまだ捕まっていなかったという事実があった。

[第二次世界大戦](https://ja.wikipedia.org/wiki/第二次世界大戦 "wikilink")参戦直前の作品であるため日本では公開されなかった（[キネマ旬報](../Page/キネマ旬報.md "wikilink")には『**ガスの灯り**』として紹介されているので輸入された可能性はある）。また、[MGMが米国版を製作した際](../Page/メトロ・ゴールドウィン・メイヤー.md "wikilink")、この英国版のプリントを市場から一掃しようとしたため稀少な作品となったこともあるが、現在[DVD](../Page/DVD.md "wikilink")などでは1944年版と併せて収録されている。

## ガス燈（1944年）

[メトロ・ゴールドウィン・メイヤー](../Page/メトロ・ゴールドウィン・メイヤー.md "wikilink")の製作で、と、が脚色し、[ジョージ・キューカー](../Page/ジョージ・キューカー.md "wikilink")が監督した。

[イングリッド・バーグマン](../Page/イングリッド・バーグマン.md "wikilink")が[アカデミー主演女優賞](../Page/アカデミー主演女優賞.md "wikilink")と[ゴールデングローブ賞 主演女優賞 (ドラマ部門)を受賞している](https://ja.wikipedia.org/wiki/ゴールデングローブ賞_主演女優賞_\(ドラマ部門\) "wikilink")。また、[アンジェラ・ランズベリー](../Page/アンジェラ・ランズベリー.md "wikilink")（撮影当時17歳）の映画デビュー作であり、[アカデミー助演女優賞](../Page/アカデミー助演女優賞.md "wikilink")に[ノミネート](https://ja.wikipedia.org/wiki/ノミネート "wikilink")された。

## 作品ごとの違い

:\*オリジナルの戯曲と2本の映画でメインとなる登場人物3人の名前が異なっている。

| オリジナル戯曲    | 1940年版      | 1944年版         |
| ---------- | ----------- | -------------- |
| ジャック・マニンガム | ポール・マレン     | グレゴリー・アントン     |
| ベラ・マニンガム   | ベラ・マレン      | ポーラ・アルキスト・アントン |
| ラフ刑事       | B.G.ラフ（元刑事） | ブライアン・キャメロン警部補 |
|            |             |                |

:\*1944年版では殺されたアリスの設定が違う。オリジナルおよび1940年版ではただの裕福な女性という設定だが、1944年版では国際的に有名な[オペラ](../Page/オペラ.md "wikilink")歌手であり、ヒロインの伯母（または叔母）で育ての親である。またアリスの名字はオリジナルおよび1940年版ではバーロウだが、1944年版ではアルキストになっている。

:\*1944年版は開始3分弱、ポーラの歌のレッスンの場面では[ドニゼッティの](../Page/ガエターノ・ドニゼッティ.md "wikilink")[オペラ](../Page/オペラ.md "wikilink") [ランメルモールのルチア](../Page/ランメルモールのルチア.md "wikilink") 第1部の二重唱 "Verranno a te sull'aure"\[5\] の冒頭が歌われ、ポーラの今後の運命が暗示される。
