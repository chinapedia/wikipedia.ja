**四次方程式**（よじほうていしき、quartic equation）とは、次数が 4 であるような[代数方程式](../Page/代数方程式.md "wikilink")の事である。この項目では主に一変数の四次方程式を扱う。

## 概要

[サムネイル](https://ja.wikipedia.org/wiki/ファイル:四次方程式の解の公式.png "wikilink") [代替文=](https://ja.wikipedia.org/wiki/ファイル:四次方程式の解の公式（部分式置き換え）.png "wikilink") 一変数の四次方程式は

  -

    0}}, (*a*<sub>4</sub> ≠ 0)

の形で表現される。 *a*<sub>4</sub> で割り

  -

    0}}, ( *A*<sub>*n*</sub> = *a*<sub>*n*</sub> / *a*<sub>4</sub>)

の形にしても根は変わらないのでこの形で論じられることが多い。

一般的な四次方程式の解法は、[ジェロラモ・カルダーノ](../Page/ジェロラモ・カルダーノ.md "wikilink")の弟子である[ルドヴィコ・フェラーリ](../Page/ルドヴィコ・フェラーリ.md "wikilink")によって発見され、カルダノの著書『[アルス・マグナ](https://ja.wikipedia.org/wiki/アルス・マグナ_\(カルダーノの著書\) "wikilink")』で概要が述べられた。カルダノは *x*, *x*<sup>2</sup>, *x*<sup>3</sup> を、[線分の長さ](../Page/直線.md "wikilink")、一辺の長さが *x* の[正方形](../Page/正方形.md "wikilink")の[面積](../Page/面積.md "wikilink")、一辺の長さが *x* の[立方体](https://ja.wikipedia.org/wiki/立方体 "wikilink")の[体積](https://ja.wikipedia.org/wiki/体積 "wikilink")と対応させてとらえ、 4 次以上の方程式には意味がないと考えていたため、三次方程式と違って詳細には述べられていない。

しかし、カルダノの死後、[ルネ・デカルト](../Page/ルネ・デカルト.md "wikilink")は著書『[方法序説](../Page/方法序説.md "wikilink")』の試論の1つである『幾何学』において[定規とコンパスによる作図](../Page/定規とコンパスによる作図.md "wikilink")を論じ、長さ *x* の線分、長さ *y* の線分、長さ 1 の線分から長さ *x y* の線分が得られることを示している。これによると、長さ *x* の線分と長さ 1 の線分から長さ *x*<sup>*n*</sup> （ *n* は任意の[自然数](../Page/自然数.md "wikilink")）の線分の作図が可能であることが分かるため 4 次以上の方程式を解くことにも幾何学的な意味を与えることは可能であり、カルダノのとらえ方は不十分であったことが分かる。

その後、四次方程式は三次方程式と同様に様々な解法が発見され、[五次方程式](../Page/五次方程式.md "wikilink")の代数的解法の探索と合わせて詳細な研究が進められた。

## 複二次式

四次方程式の内奇数次の項が無い

  -
    *a*<sub>4</sub> *x*<sup>4</sup> + *a*<sub>2</sub> *x*<sup>2</sup> + *a*<sub>0</sub> = 0 (*a*<sub>4</sub> ≠ 0)

の形の式は *x*<sup>2</sup> を変数とする[二次方程式](../Page/二次方程式.md "wikilink")と見ることができ、**複二次方程式** (*biquadratic equation*) あるいは単に**複二次式**と呼ばれる。二次方程式の解法を知っていれば簡単に解くことができる。

*y* = *x*<sup>2</sup> と変換することで *y* に関する二次方程式

  -
    *a*<sub>4</sub> *y*<sup>2</sup> + *a*<sub>2</sub> *y* + *a*<sub>0</sub> = 0

を得ることができ、この二次方程式を解くことによって根を求められる。

また、[実数](../Page/実数.md "wikilink")を係数とする複二次式

  -
    *x*<sup>4</sup> + *A*<sub>2</sub> *x*<sup>2</sup> + *A*<sub>0</sub> = 0

に対して、次のような二次式の積への因数分解もよく行われる。 *x*<sup>2</sup> の二次方程式とみたときの[判別式](../Page/判別式.md "wikilink")

  -
    *D* = *A*<sub>2</sub><sup>2</sup> − 4 *A*<sub>0</sub>

の[符号によって](../Page/符号_\(数学\).md "wikilink")

*D* \> 0 であれば *x*<sup>2</sup> について[平方完成](../Page/平方完成.md "wikilink")することにより

\[x^4 +A_2 x^2 + A_0 = \left(x^2 + {A_2 \over 2}\right)^2 - {A_2^2 - 4 A_0 \over 4}\] *D* \< 0 であれば *A*<sub>0</sub> \> 0 であることに注意して

\[x^4 +A_2 x^2 + A_0 = \left(x^2 + \sqrt{A_0} \right)^2 - \left(2 \sqrt{A_0} - A_2 \right) x^2\] と変形すれば、いずれの場合も[因数分解](../Page/因数分解.md "wikilink")の公式

  -
    α<sup>2</sup> − β<sup>2</sup> = (α + β) (α − β)

を利用して実数を係数とする二次式の積に因数分解できる。

## 根の様子

四次方程式は、高々4個の根を持つ。

四次方程式0}}の判別式は

\[\begin{align}
\Delta=&256a^3e^3-192a^2bde^2-128a^2c^2e^2+144a^2cd^2e-27a^2d^4\\
&\ +144ab^2ce^2-6ab^2d^2e-80abc^2de+18abcd^3+16ac^4e\\
&\ -4ac^3d^2-27b^4e^2+18b^3cde-4b^3d^3-4b^2c^3e+b^2c^2d^2
\end{align}\] によって与えられ、係数によって定まる以下の4個の定数によって更に詳細な情報が得られる。

\[\begin{align}
P&=8ac-3b^2\\
R&=b^3+8a^2d-4abc\\
\Delta_0&=c^2-3bd+12ae\\
D&=64a^3e-16a^2c^2+16ab^2c-16a^2bd-3b^4
\end{align}\]

に関して、以下の事実が成立する。\[1\]

1.  のとき、2つの異なる実数解と2つの互いに共役である複素数解を持つ。

2.  のとき、

    1.  かつならば、4つの異なる実数根を持つ。

    2.  またはならば、2つの互いに共役である複素数根を2組持つ。

3.  のときにのみ、方程式は重根を持ち、

    1.  かつかつならば、実数に1つの二重根と2つの異なる根を持つ。

    2.  または(かつ(D, Rのどちらかが0でない))ならば、実数に1つの二重根を持ち、2つの互いに共役である複素数根を持つ。

    3.  0}}かつならば、実数に1つの三重根と1つの重根でない根を持つ。

    4.  0}}のとき、

        1.  ならば、実数に2つの異なる二重根を持つ。

        2.  かつR=0ならば、それぞれが二重根である互いに共役である複素数根を2つ持つ。

        3.  0}}ならば、を四重根として持つ。

以上には、例えばかつである場合などが記されていない。しかし、このような組み合わせは実際には存在しない。

## フェラーリの方法

**[フェラーリの方法](../Page/ルドヴィコ・フェラーリ.md "wikilink")**は、一般的な四次方程式の解法のうちで最初に与えられた解法である。四次方程式

  -
    *a*<sub>4</sub> *x*<sup>4</sup> + *a*<sub>3</sub> *x*<sup>3</sup> + *a*<sub>2</sub> *x*<sup>2</sup> + *a*<sub>1</sub> *x* + *a*<sub>0</sub> = 0 (*a*<sub>4</sub> ≠ 0)

を *a*<sub>4</sub> で割り

  -
    *x*<sup>4</sup> + *A*<sub>3</sub> *x*<sup>3</sup> + *A*<sub>2</sub> *x*<sup>2</sup> + *A*<sub>1</sub> *x* + *A*<sub>0</sub> = 0

の形にする。( *A*<sub>*n*</sub> = *a*<sub>*n*</sub> / *a*<sub>4</sub>)

\[B_3 := \frac{A_3}{4}\] とし

  -
    *x* = *y* − *B*<sub>3</sub>

によって変数変換を行うと

  -
    *y*<sup>4</sup> + (*A*<sub>2</sub> − 6 *B*<sub>3</sub><sup>2</sup>) *y*<sup>2</sup> + (*A*<sub>1</sub> − 2 *A*<sub>2</sub> *B*<sub>3</sub> + 8 *B*<sub>3</sub><sup>3</sup>) *y* + (*A*<sub>0</sub> - *A*<sub>1</sub> *B*<sub>3</sub> + *A*<sub>2</sub> *B*<sub>3</sub><sup>2</sup> − 3 *B*<sub>3</sub><sup>4</sup>) = 0

のように三次の項が消えた方程式が得られる。見やすいように

  -

    0}}

と書く。 *q* = 0 の時は、 複二次式として解けばよいので、以後は *q* ≠ 0 とする。

[媒介変数](../Page/媒介変数.md "wikilink") *u* ≠ 0 を用い

\[\left(y^2 + \frac{p + u}{2}\right)^2 - u \left(y - \frac{q}{2u}\right)^2 = 0\] と変形する。ここで上式を展開し係数を比較すると、*u* の[三次方程式](../Page/三次方程式.md "wikilink")

  -

    *q*<sup>2</sup>}}

が得られる。このような補助的な方程式を、与えられた四次方程式に関する**三次分解方程式**(*resolvent cubic equation*) という。 *q* ≠ 0 なので、この分解方程式の解は *u* ≠ 0 を満たしており、この解の一つを *u* として取る。また、求める四次方程式は

\[\left\{ \left(y^2 + \frac{p + u}{2}\right) + \sqrt{u} \left(y - \frac{q}{2u}\right) \right\} \left\{ \left(y^2 + \frac{p + u}{2}\right) - \sqrt{u} \left(y - \frac{q}{2u}\right) \right\} = 0\] となり、この 2 つの二次方程式から、四次方程式の根を求めることができる。

## デカルトの方法

[デカルトは](../Page/ルネ・デカルト.md "wikilink")、著書『[方法序説](../Page/方法序説.md "wikilink")』の試論の1つである『幾何学』において四次方程式

  -
    *y*<sup>4</sup> + *p y*<sup>2</sup> + *q y* + *r* = 0

の解を求めるために、二次式による因数分解

  -
    *y*<sup>4</sup> + *p y*<sup>2</sup> + *q y* + *r* = (*y*<sup>2</sup> + *c*<sub>1</sub> *y* + *c*<sub>0</sub>) (*y*<sup>2</sup> + *d*<sub>1</sub> *y* + *d*<sub>0</sub>)

を仮定した方法を奨めた。係数比較によって

  -
    *c*<sub>1</sub> + *d*<sub>1</sub> = 0
    *c*<sub>0</sub> + *d*<sub>0</sub> + *c*<sub>1</sub> *d*<sub>1</sub> = *p*
    *c*<sub>1</sub> *d*<sub>0</sub> + *c*<sub>0</sub> *d*<sub>1</sub> = *q*
    *c*<sub>0</sub> *d*<sub>0</sub> = *r*

が得られる。上の 3 つの式から

  -
    *d*<sub>1</sub> = − *c*<sub>1</sub>
    2 *c*<sub>0</sub> *c*<sub>1</sub> = *c*<sub>1</sub><sup>3</sup> + *p c*<sub>1</sub> − *q*
    2 *d*<sub>0</sub> *c*<sub>1</sub> = *c*<sub>1</sub><sup>3</sup> + *p c*<sub>1</sub> + *q*

が得られる。

  -
    4 *c*<sub>1</sub><sup>2</sup> *r* = 4 *c*<sub>1</sub><sup>2</sup> *c*<sub>0</sub> *d*<sub>0</sub> = (2 *c*<sub>0</sub> *c*<sub>1</sub>)(2 *d*<sub>0</sub> *c*<sub>1</sub>) = (*c*<sub>1</sub><sup>3</sup> + *p c*<sub>1</sub> − *q*)(*c*<sub>1</sub><sup>3</sup> + *p c*<sub>1</sub> + *q*)

であるから

  -
    *c*<sub>1</sub><sup>2</sup>( *c*<sub>1</sub><sup>2</sup> + *p*)<sup>2</sup> - *q*<sup>2</sup> = 4 *c*<sub>1</sub><sup>2</sup> *r*

という *c*<sub>1</sub> に関して 6 次の方程式が得られる。偶数次の項しか無いので *u* = *c*<sub>1</sub><sup>2</sup> とでもおけば

  -
    *u*( *u* + *p*)<sup>2</sup> - *q*<sup>2</sup> = 4 *r u*

という *u* に関する三次方程式が得られる。この方程式は、フェラーリの方法で得たのと同じ**三次分解方程式**であり、これを解くことによって、元の方程式の根が得られる。

## オイラーの方法

[レオンハルト・オイラー](../Page/レオンハルト・オイラー.md "wikilink")は、三次方程式や四次方程式の解法をいくつか発見した。ここに述べる方法も**オイラーの方法**と呼ばれる解法の一つである。

  -
    (*x* + *a* + *b* + *c*) (*x* + *a* − *b* − *c*) (*x* − *a* + *b* − *c*) (*x* − *a* − *b* + *c*)
    \= ((*x* + *a*)<sup>2</sup> − (*b* + *c*)<sup>2</sup>) ((*x* − *a*)<sup>2</sup> − (*b* − *c*)<sup>2</sup>)
    \= (*x*<sup>2</sup> − *a*<sup>2</sup>)<sup>2</sup> + (*b*<sup>2</sup> − *c*<sup>2</sup>)<sup>2</sup> − (*x* + *a*)<sup>2</sup> (*b* − *c*)<sup>2</sup> − (*x* − *a*)<sup>2</sup> (*b* + *c*)<sup>2</sup>
    \= *x*<sup>4</sup> + *a*<sup>4</sup> + *b*<sup>4</sup> + *c*<sup>4</sup> − 2 (*x*<sup>2</sup> *a*<sup>2</sup> + *x*<sup>2</sup> *b*<sup>2</sup> + *x*<sup>2</sup> *c*<sup>2</sup> + *a*<sup>2</sup> *b*<sup>2</sup> + *b*<sup>2</sup> *c*<sup>2</sup> + *c*<sup>2</sup> *a*<sup>2</sup>) + 8 *x a b c*
    \= *x*<sup>4</sup> − 2 (*a*<sup>2</sup> + *b*<sup>2</sup> + *c*<sup>2</sup>) *x*<sup>2</sup> + 8 *a b c x* + *a*<sup>4</sup> + *b*<sup>4</sup> + *c*<sup>4</sup> − 2 (*a*<sup>2</sup> *b*<sup>2</sup> + *b*<sup>2</sup> *c*<sup>2</sup> + *c*<sup>2</sup> *a*<sup>2</sup>)

という等式を用いると *x* を変数とする四次方程式

  -
    *x*<sup>4</sup> − 2 (*a*<sup>2</sup> + *b*<sup>2</sup> + *c*<sup>2</sup>) *x*<sup>2</sup> + 8 *a b c x* + *a*<sup>4</sup> + *b*<sup>4</sup> + *c*<sup>4</sup> − 2 (*a*<sup>2</sup> *b*<sup>2</sup> + *b*<sup>2</sup> *c*<sup>2</sup> + *c*<sup>2</sup> *a*<sup>2</sup>) = 0

の根は *a*, *b*, *c* を用いて

  -
    − *a* − *b* − *c*, − *a* + *b* + *c*, *a* − *b* + *c*, *a* + *b* − *c*

の 4 つであることが分かる。

この方程式と、 3 次の項の消えた四次方程式

  -
    *x*<sup>4</sup> + *p x*<sup>2</sup> + *q x* + *r* = 0

の係数を比べ、 *p*, *q*, *r* から *a*, *b*, *c* を求めることができれば、 3 次の項の消えた四次方程式の根は上にあるように 4 つ求まる。

実際に係数を比べてみれば

  -
    *p* = − 2 (*a*<sup>2</sup> + *b*<sup>2</sup> + *c*<sup>2</sup>)
    *q* = 8 *a b c*
    *r* = *a*<sup>4</sup> + *b*<sup>4</sup> + *c*<sup>4</sup> − 2 (*a*<sup>2</sup> *b*<sup>2</sup> + *b*<sup>2</sup> *c*<sup>2</sup> + *c*<sup>2</sup> *a*<sup>2</sup>) = (*a*<sup>2</sup> + *b*<sup>2</sup> + *c*<sup>2</sup>)<sup>2</sup> − 4 (*a*<sup>2</sup> *b*<sup>2</sup> + *b*<sup>2</sup> *c*<sup>2</sup> + *c*<sup>2</sup> *a*<sup>2</sup>)

ここで *f*<sub>0</sub> = (2 *a*)<sup>2</sup>, *f*<sub>1</sub> = (2 *b*)<sup>2</sup>, *f*<sub>2</sub> = (2 *c*)<sup>2</sup> とおけば

  -
    *f*<sub>0</sub> + *f*<sub>1</sub> + *f*<sub>2</sub> = − 2 *p*
    *f*<sub>0</sub> *f*<sub>1</sub> + *f*<sub>1</sub> *f*<sub>2</sub> + *f*<sub>2</sub> *f*<sub>0</sub> = *p*<sup>2</sup> − 4 *r*
    *f*<sub>0</sub> *f*<sub>1</sub> *f*<sub>2</sub> = *q*<sup>2</sup>

となり、[根と係数の関係](../Page/根と係数の関係.md "wikilink")により *f*<sub>0</sub>, *f*<sub>1</sub>, *f*<sub>2</sub> は三次方程式

  -
    *u*<sup>3</sup> + 2 *p u*<sup>2</sup> + (*p*<sup>2</sup> − 4 *r*) *u* − *q*<sup>2</sup> = 0

の解であり、これもフェラーリの方法に現れた**三次分解方程式**である。この三次方程式を解くことによって *a*, *b*, *c* が得られる。

## ラグランジュの方法

[ジョゼフ＝ルイ・ラグランジュ](../Page/ジョゼフ＝ルイ・ラグランジュ.md "wikilink")は、既に知られていた三次方程式や四次方程式の解法を、いろいろな視点から詳しく調べ上げた。ここで述べるのは、ラグランジュによるフェラーリの方法の解釈であり、現代的に言えば[対称群](../Page/対称群.md "wikilink")を用いた方法である。

フェラーリの方法において、四次方程式は

  -
    *y*<sup>4</sup> + *p y*<sup>2</sup> + *q y* + *r* = 0

の形に変形される。この方程式の 4 つの根を *r*<sub>0</sub>, *r*<sub>1</sub>, *r*<sub>2</sub>, *r*<sub>3</sub> とする。三次分解式を解くことで四次方程式は、 2 つの二次方程式

\[y^2 + {p + u \over 2} = \pm \sqrt{u} \left(y - {q \over 2u}\right)\] に分解することができた。

\[y^2 + {p + u \over 2} = \sqrt{u} \left(y - {q \over 2u}\right)\] は、元の四次方程式の 4 つの根のうちの 2 つを根とするが、これをとりあえず *r*<sub>0</sub>, *r*<sub>1</sub> の 2 つとしたとき、

\[y^2 + {p + u \over 2} = - \sqrt{u} \left(y - {q \over 2u}\right)\] の根は *r*<sub>2</sub>, *r*<sub>3</sub> となり、[根と係数の関係](../Page/根と係数の関係.md "wikilink")から

\[r_0 + r_1 = \sqrt{u}\]

\[r_2 + r_3 = - \sqrt{u}\] したがって

  -
    (*r*<sub>0</sub> + *r*<sub>1</sub>) (*r*<sub>2</sub> + *r*<sub>3</sub>) = − *u*

便宜上

\[y^2 + {p + u \over 2} = \sqrt{u} \left(y - {q \over 2u}\right)\] の根を *r*<sub>0</sub>, *r*<sub>1</sub> としたが、根の並び方はいろいろ考えられる。 *r*<sub>*m*</sub> と *r*<sub>*n*</sub> を入れ替える[互換を](../Page/対称群.md "wikilink") σ<sub>*m,n*</sub> と書けば、例えば

  -
    σ<sub>*0,1*</sub> (*r*<sub>0</sub> + *r*<sub>1</sub>) (*r*<sub>2</sub> + *r*<sub>3</sub>) = (*r*<sub>0</sub> + *r*<sub>1</sub>) (*r*<sub>2</sub> + *r*<sub>3</sub>)
    σ<sub>*0,2*</sub> (*r*<sub>0</sub> + *r*<sub>1</sub>) (*r*<sub>2</sub> + *r*<sub>3</sub>) = (*r*<sub>2</sub> + *r*<sub>1</sub>) (*r*<sub>0</sub> + *r*<sub>3</sub>)

など、一般には異なる値を取ることになる。このように調べていくと 4 つの根の並び方は 4\! = 24 通りあるが

  -
    (*r*<sub>0</sub> + *r*<sub>1</sub>) (*r*<sub>2</sub> + *r*<sub>3</sub>) = − *u*

の値は、最初の根の並べ方によって

  -
    *s*<sub>0</sub> = (*r*<sub>0</sub> + *r*<sub>1</sub>) (*r*<sub>2</sub> + *r*<sub>3</sub>)
    *s*<sub>1</sub> = (*r*<sub>0</sub> + *r*<sub>2</sub>) (*r*<sub>1</sub> + *r*<sub>3</sub>)
    *s*<sub>2</sub> = (*r*<sub>0</sub> + *r*<sub>3</sub>) (*r*<sub>1</sub> + *r*<sub>2</sub>)

の 3 通りとなる。

例えば、互換 σ<sub>0,1</sub> を作用させると、

  -
    σ<sub>*0,1*</sub> *s*<sub>0</sub> = *s*<sub>0</sub>
    σ<sub>*0,1*</sub> *s*<sub>1</sub> = *s*<sub>2</sub>
    σ<sub>*0,1*</sub> *s*<sub>2</sub> = *s*<sub>1</sub>

となる。

一般に、互換 σ<sub>*m,n*</sub> は *s*<sub>0</sub>, *s*<sub>1</sub>, *s*<sub>2</sub> の並び替えしかしないため *s*<sub>0</sub>, *s*<sub>1</sub>, *s*<sub>2</sub> に関する[基本対称式](https://ja.wikipedia.org/wiki/基本対称式 "wikilink")

  -
    *s*<sub>0</sub> + *s*<sub>1</sub> + *s*<sub>2</sub>
    *s*<sub>0</sub> *s*<sub>1</sub> + *s*<sub>1</sub> *s*<sub>2</sub> + *s*<sub>2</sub> *s*<sub>0</sub>
    *s*<sub>0</sub> *s*<sub>1</sub> *s*<sub>2</sub>

は、互換 σ<sub>*m,n*</sub> によって不変であり、 *r*<sub>0</sub>, *r*<sub>1</sub>, *r*<sub>2</sub>, *r*<sub>3</sub> の基本対称式で書けることになる。

  -
    すなわち *s*<sub>0</sub>, *s*<sub>1</sub>, *s*<sub>2</sub> の基本対称式は、最初に考えた四次方程式の係数 *p*, *q*, *r* で書ける。

以上の事から

  -
    *u* = − (*r*<sub>0</sub> + *r*<sub>1</sub>) (*r*<sub>2</sub> + *r*<sub>3</sub>)

は、根の並べ方によって 3 つの値 − *s*<sub>0</sub>, − *s*<sub>1</sub>, − *s*<sub>2</sub> をとり、これらを根とする方程式

  -
    (*u* + *s*<sub>0</sub>) (*u* + *s*<sub>1</sub>) (*u* + *s*<sub>2</sub>) = 0

の左辺は *u* についての多項式として展開すると、その係数が *p*, *q*, *r* の多項式として書ける式である。この *u* に関する三次方程式こそ、フェラーリの方法で**三次分解方程式**として求められた方程式に他ならない。

このようにしてラグランジュは、四次方程式を解くための補助方程式である三次分解方程式の根が、元の四次方程式の根の整式で書ける事を発見し、補助方程式の次数が三次である理由を、根の置換という立場からはっきりと示した。

このような式は他にもあり

\[t_0 = \left(r_0 + r_1\right) - \left( r_2 +r_3\right)\]

\[t_1 = \left(r_0 + r_2\right) - \left( r_1 +r_3\right)\]

\[t_2 = \left(r_0 + r_3\right) - \left( r_1 +r_2\right)\] とすれば、\(t_0^2, t_1^2, t_2^2\) を根とする三次方程式で四次方程式を解くこともできる。 ラグランジュは補助方程式の根を用いて、問題の方程式の根の公式を表現するのとは逆に、補助方程式の根を、元の方程式の根の整式（あるいは一般に有理式）として書ける事が代数的に解ける理由と考え、特に

\[u = r_0 +i r_1 - r_2 - i r_3\] の形の式、さらに一般に、*n*次方程式であれば [1の原始*n*乗根](https://ja.wikipedia.org/wiki/1の冪根 "wikilink")\(\zeta_n\) を用いて

\[u = \sum_{k=0}^{n-1} \zeta_n^k r_k = r_0 +\zeta_n r_1 + \zeta_n^2 r_2 + \cdots + \zeta_n^{n-2} r_{n-2} + \zeta_n^{n-1} r_{n-1}\] の形の式の性質を詳しく調べたが、五次以上の代数方程式の代数的解法の発見には至らなかった。この形の式を**ラグランジュの分解式**(*Lagrange resolvent*)という。 五次以上の代数方程式の代数的解法の存在については、[パオロ・ルフィニ](../Page/パオロ・ルフィニ.md "wikilink")、[オーギュスタン＝ルイ・コーシー](../Page/オーギュスタン＝ルイ・コーシー.md "wikilink")、[ニールス・アーベル](../Page/ニールス・アーベル.md "wikilink")らの研究が[アーベル–ルフィニの定理](https://ja.wikipedia.org/wiki/アーベル–ルフィニの定理 "wikilink")として結実し、否定されることになるが、彼らの研究は、このようなラグランジュの研究を源流としている。
