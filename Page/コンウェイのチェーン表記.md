**コンウェイのチェーン表記**とは、[1995年](https://ja.wikipedia.org/wiki/1995年 "wikilink")に[イギリス](https://ja.wikipedia.org/wiki/イギリス "wikilink")の[数学者](../Page/数学者.md "wikilink")[ジョン・ホートン・コンウェイ](../Page/ジョン・ホートン・コンウェイ.md "wikilink")によって導入された[巨大数](../Page/巨大数.md "wikilink")の表記法の一つである。

[クヌースの矢印表記](../Page/クヌースの矢印表記.md "wikilink")や[アッカーマン関数](../Page/アッカーマン関数.md "wikilink")などより比較的強い演算である。

## アッカーマン関数

[アッカーマン関数](../Page/アッカーマン関数.md "wikilink")はコンウェイのチェーン表記へ置き換えられる:

  -
    *m* \> 2の際 *A*(*m*, *n*) = (2 → (*n* + 3) → *(m* − 2)) − 3 （[ハイパー演算の角括弧表記を用いると](https://ja.wikipedia.org/wiki/:en:Hyperoperation#Notations "wikilink") *A*(*m*, *n*) = 2 \[*m*\] (*n* + 3) - 3）

よって

  -
    *n* \> 2の際 2 → *n* → *m* = *A*(*m* + 2,*n* − 3) + 3

(*n* = 1 と *n* = 2 は *A*(*m*, −2) = −1 と *A*(*m*, −1) = 1に対応し、論理的に加算できる。)

ちなみに多変数アッカーマン関数で\(\{a,b,c,d\}\approx A(d,c,b)\)くらいである。

## グラハム数

[グラハム数](../Page/グラハム数.md "wikilink") \(G\) それ自体を、コンウェイのチェーン表記を用い簡潔に表す事は出来ない。が、仲介させる関数を \(g(n) = 3 \rightarrow 3 \rightarrow n =  \begin{matrix}
    3\underbrace{\uparrow \uparrow \cdots \uparrow}3 \\
    \text{n 本 の 矢 印}
 \end{matrix}\) と定義することで \(G = g^{64}(4)\) と表記できる。([写像の冪を参照](https://ja.wikipedia.org/wiki/写像の合成#写像の冪 "wikilink")。)又、 \(3 \rightarrow 3 \rightarrow 64 \rightarrow 2 < G < 3 \rightarrow 3 \rightarrow 65 \rightarrow 2\) である。

\[3 \rightarrow 3 \rightarrow 3 \rightarrow 3\]

\[= 3 \rightarrow 3 \rightarrow (3 \rightarrow 3 \rightarrow 27 \rightarrow 2) \rightarrow 2\,\]

\[= g^ {3 \rightarrow 3 \rightarrow 27 \rightarrow 2}(1)\]

\[= g^{g^{27}(1)}(1)\] \(\left.
 \begin{matrix}
=   &3\underbrace{\uparrow \uparrow \uparrow \cdots\cdots\cdots\cdot\cdot \uparrow \uparrow \uparrow}_{}3 \\
    &3\underbrace{\uparrow \uparrow \uparrow \cdots\cdots\cdots\cdot \uparrow \uparrow \uparrow}_{}3\text{本} \\
    &3\underbrace{\uparrow \uparrow \uparrow \cdots\cdots\cdots \uparrow \uparrow \uparrow}_{}3\text{本} \\
    &\underbrace{\qquad\;\; \vdots \qquad\;\;} \\
    &3\underbrace{\uparrow \uparrow \uparrow \cdots\cdot \uparrow \uparrow \uparrow}_{3\uparrow 3\text{本}}3\text{本}
 \end{matrix}
\right\}
\left.

    3\underbrace{\uparrow \uparrow \cdots\cdots\cdots\cdot \uparrow}3\text{層 } \\

    \underbrace{\qquad\;\; \vdots \qquad\;\;} \\

 \end{matrix}
\right\} \ 3\uparrow 3 = 27\text{層 }\)

数\(3 \rightarrow 3 \rightarrow 27 \rightarrow 2\) = g<sup>27</sup>(1)は65よりもはるかに大きいので、\(3 \rightarrow 3 \rightarrow 3 \rightarrow 3\)はグラハム数よりはるかに大きい。 さらに末尾の数字を増やしたりチェーンを伸ばしたりすることで極めて巨大な数を表記可能である。3→3→3→3はコンウェイのテトラトリと呼ばれる。

## ピーター・ハーフォードによる拡張

Web[デベロッパー](https://ja.wikipedia.org/wiki/デベロッパー "wikilink")で統計学者のピーター・ハーフォードは、この表記法の拡張を定義した。

\(a \rightarrow_b c = \underbrace{a\rightarrow_{b-1} a\rightarrow_{b-1} a\rightarrow_{b-1} \dots \rightarrow_{b-1} a\rightarrow_{b-1} a\rightarrow_{b-1} a}_{c \text{ 本の矢}}\)（あくまで\(\rightarrow_{b-1}\)が\(c\)本、という意味）

\(a \rightarrow_2 (a-1)\)は既に cg(a) と同等であり、関数\(f(n) = n \rightarrow_n n\)は cg(n) よりも大きな増加速度を持つ。

### 定義

\(a \rightarrow_1b\)=\(a^b\)

\(a \rightarrow_1b \rightarrow_1n\)=\(a\uparrow^nb\)

\(a \rightarrow_nb  \rightarrow_nc\dots \rightarrow_nz \rightarrow_n1\)=\(a \rightarrow_nb  \rightarrow_nc\dots \rightarrow_nz\)

\(a \rightarrow_nb  \rightarrow_nc\dots \rightarrow_nz \rightarrow_n1 \rightarrow_n\dots\)=\(a \rightarrow_nb  \rightarrow_nc\dots \rightarrow_nz\)

\(a \rightarrow_nb \rightarrow_n\dots\rightarrow_ny \rightarrow_nz\)=\(a \rightarrow_nb \rightarrow_n\dots\rightarrow_n(a \rightarrow_nb \rightarrow_n\dots(y-1)\rightarrow_nz)\rightarrow_n(z-1)\)

\(a\rightarrow_nb=\underbrace{a\rightarrow_{n-1}a\rightarrow_{n-1}\dots\rightarrow_{n-1}a}_{\text{長さ}b+1}\)

この表記は拡張チェーン表記と呼ばれ、[回転矢印表記](https://ja.wikipedia.org/wiki/回転矢印表記 "wikilink")と同じくらいの強さである。

この表記は急増加関数で\(x\rightarrow_xx\approx f_{\omega^3}(x)\)と近似できる。

ちなみに多変数[アッカーマン関数](../Page/アッカーマン関数.md "wikilink")で\(x\rightarrow_xx\approx A(1,0,0,x)\)程度である。

彼は上記以外の規則は変更しなかった。（ひとつのチェーンにおいては1種類の矢印のみを使用できた。つまりb≠dの際\(a \rightarrow_b c \rightarrow_d e\)は規則違反になる）

もし更に規則を若干訂正し

\(a \rightarrow_b c \rightarrow_d e = a \rightarrow_b \underbrace{c \rightarrow_{d-1} c \rightarrow_{d-1} c \rightarrow_{d-1} \dots \rightarrow_{d-1} c \rightarrow_{d-1} c \rightarrow_{d-1} c}_{e \text{ 本の矢}}\)

とすると、\(a \rightarrow_b c \rightarrow_d e\)は規則違反では無くなる上に表記法が更に強くなる(あまり意味は無い)。\[3\]
