**Servoy**はJavaベースの[Windows XP](../Page/Microsoft_Windows_XP.md "wikilink") / [Vista](https://ja.wikipedia.org/wiki/Microsoft_Windows_Vista "wikilink")、[macOS](https://ja.wikipedia.org/wiki/macOS "wikilink")、そして[Linux](../Page/Linux.md "wikilink")で使用できる [RAD](https://ja.wikipedia.org/wiki/RAD_\(計算機プログラミング環境\) "wikilink")（Rapid Application Development：アプリケーション開発ツール）。[iPhone](https://ja.wikipedia.org/wiki/iPhone "wikilink")、[Windows CEなどの多くの](https://ja.wikipedia.org/wiki/Windows_CE "wikilink")[モバイルプラットフォーム](https://ja.wikipedia.org/wiki/モバイルプラットフォーム "wikilink")で動作させることが可能。Servoyは Servoy Server、Servoy Developer Eclipse、Smart Client、Web Client、及びオプションの Servoy Runtime を含む。

Servoyは多くのデータベースと併用が可能。例えば[SQL Anywhere](https://ja.wikipedia.org/wiki/SQL_Anywhere "wikilink")、[Oracle](../Page/Oracle_Database.md "wikilink")、[Microsoft SQL Server](https://ja.wikipedia.org/wiki/Microsoft_SQL_Server "wikilink")、[IBM DB2](https://ja.wikipedia.org/wiki/DB2 "wikilink")、[IBM Informix](../Page/IBM_Informix.md "wikilink")、[Sybase](../Page/Sybase.md "wikilink")、[OpenBase](https://ja.wikipedia.org/wiki/OpenBase "wikilink")、[MySQL](https://ja.wikipedia.org/wiki/MySQL "wikilink")、[PostgreSQL](https://ja.wikipedia.org/wiki/PostgreSQL "wikilink")、[Firebird](../Page/Firebird.md "wikilink")など。

Servoy用にプログラムを書く場合、[JavaScript](../Page/JavaScript.md "wikilink")を使用することができ、[Java](https://ja.wikipedia.org/wiki/Java "wikilink")言語を使用して新規機能のプラグイン化・追加を行うことができる。

これらのバランス、セキュリティレベル、そして迅速な開発が可能なことで、Servoyの人気は確立されている。各国語へ容易にローカライズすることができ、ほとんどのSQLバックエンド・データベースで動作し、それらのモジュールの開発力と柔軟な拡張性の発揮を可能にしているためである。

わずかだが大企業の導入事例も報告がある、例えば米[SONY](../Page/ソニー.md "wikilink")、[モトローラ](../Page/モトローラ.md "wikilink")、[ウェルズ・ファーゴ](../Page/ウェルズ・ファーゴ.md "wikilink")、[シマンテック](../Page/シマンテック.md "wikilink")、[ハネウェル](../Page/ハネウェル.md "wikilink")、[ワーナー・ブラザース](../Page/ワーナー・ブラザース.md "wikilink")など。各社はServoyを選び、利用している。

最新のバージョンである Servoy 3.5 では、HTTP Tunneling、TableEvents、Column Validators、[モーダルウィンドウ](https://ja.wikipedia.org/wiki/モーダルウィンドウ "wikilink")、Enhanced Relations などを採用すると同時に、[Ajax](https://ja.wikipedia.org/wiki/Ajax "wikilink")による機能追加で利用者へのユーザビリティ、ニーズにも対応している。
