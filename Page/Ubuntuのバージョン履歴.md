[Ubuntu](../Page/Ubuntu.md "wikilink")のリリースは、[Canonical](https://ja.wikipedia.org/wiki/Canonical "wikilink")によって半年毎に行われており、リリース年の数字とリリース月の数字をバージョン番号として使っている。最初のUbuntuのリリースを例とすれば、Ubuntu 4.10は[2004年](../Page/2004年.md "wikilink")[10月20日](../Page/10月20日.md "wikilink")にリリースされている\[1\]。

Canonicalは、Ubuntuのリリースを、[GNOME](../Page/GNOME.md "wikilink")のリリースの約1ヵ月後（これは[X.Org](https://ja.wikipedia.org/wiki/X.Org "wikilink")のリリースの1ヵ月後である）にスケジューリングしており、これはUbuntuのリリースがGNOMEやXのより新しいバージョンを含むという結果に繋がっている\[2\]\[3\]。

4回のリリースにつき、そのうち1回のリリースは長期サポート（LTS）版のリリースとしてデザインされている\[4\]。12.04から後のLTSのバージョンは5年間のサポートがフリーに提供されている。また、ESM有料オプションが存在しており、これを利用するとさらに長期間のサポート（20.04では10年）を受けることができる\[5\]\[6\]。LTSでないバージョンのサポート期間は9ヶ月間である。

## リリースの歴史

### Ubuntu 4.10 (Warty Warthog)

[Ubuntu-desktop-2-410-20080706.png](https://ja.wikipedia.org/wiki/File:Ubuntu-desktop-2-410-20080706.png "fig:Ubuntu-desktop-2-410-20080706.png")

Ubuntu 4.10（Warty Warthog）は、[2004年](../Page/2004年.md "wikilink")[10月20日](../Page/10月20日.md "wikilink")にリリースされ、これはCanonicalによるUbuntuの最初のリリースであった\[7\]。Ubuntuは[Debian](../Page/Debian.md "wikilink")をベースに設計され、6ヵ月おきのリリースと、18ヵ月のサポートを提供する計画であった。また、Ubuntu 4.10はフリーにダウンロードで提供されるほか、CanonicalのShipItサービス\[8\]で入手することもできた。

### Ubuntu 5.04 (Hoary Hedgehog)

[Ubuntu-desktop-2-504-20080706.png](https://ja.wikipedia.org/wiki/File:Ubuntu-desktop-2-504-20080706.png "fig:Ubuntu-desktop-2-504-20080706.png")

Ubuntu 5.04（Hoary Hedgehog）は、[2005年](../Page/2005年.md "wikilink")[4月8日](../Page/4月8日.md "wikilink")にリリースされた\[9\]。Ubuntu 5.04には多数の新機能が搭載され、これにはアップデートマネージャ、アップデート通知、readaheadやgrepmap、[サスペンド](https://ja.wikipedia.org/wiki/サスペンド "wikilink")とハイバネートのサポート、プロセッサの動的なフリークエンシーのスケーリング、Ubuntu hardware dataaseなどが含まれる\[10\]。

### Ubuntu 5.10 (Breezy Badger)

[Ubuntu-desktop-2-510-20080706.png](https://ja.wikipedia.org/wiki/File:Ubuntu-desktop-2-510-20080706.png "fig:Ubuntu-desktop-2-510-20080706.png")

Ubuntu 5.10（Breezy Badger）は、2005年[10月12日](https://ja.wikipedia.org/wiki/10月12日 "wikilink")にリリースされた\[11\]。Ubuntu 5.10は、Usplashやアプリケーションの追加/削除のためのツール、Alacarteメニューエディター、簡単に利用できる言語のセレクター、[LVM](https://ja.wikipedia.org/wiki/LVM "wikilink")のサポートなどが含まれる\[12\]。

### Ubuntu 6.06 LTS (Dapper Drake)

[Ubuntu-desktop-2-606-20080706.png](https://ja.wikipedia.org/wiki/File:Ubuntu-desktop-2-606-20080706.png "fig:Ubuntu-desktop-2-606-20080706.png")

Ubuntu 6.06（Dapper Drake）は、[2006年](../Page/2006年.md "wikilink")[6月1日](../Page/6月1日.md "wikilink")にリリースされた\[13\]。これは、Ubuntuの最初の長期サポート（LTS）版である。Ubuntu 6.06は当初Ubuntu 6.04として計画されていたが、この当初のスケジュールに遅れてのリリースとなった\[14\]。Ubuntu 6.06では、[Live CDとインストール用CDが](../Page/Live_CD.md "wikilink")1枚に統合された\[15\]ほか、[Ubiquity](https://ja.wikipedia.org/wiki/Ubiquity "wikilink")インストーラーが含まれ、Humanlooksテーマの改善などが行われた。

### Ubuntu 6.10 (Edgy Eft)

[Ubuntu-desktop-2-610-20080708.png](https://ja.wikipedia.org/wiki/File:Ubuntu-desktop-2-610-20080708.png "fig:Ubuntu-desktop-2-610-20080708.png")

Ubuntu 6.10（Edgy Eft）は、2006年[10月26日](../Page/10月26日.md "wikilink")にリリースされた\[16\]。Ubuntu 6.10においては、Humanテーマに大きく改良が加えられ、[Tomboy](https://ja.wikipedia.org/wiki/Tomboy "wikilink")ノートアプリケーションや[F-Spot](../Page/F-Spot.md "wikilink")フォトマネージャなどが含まれた。

### Ubuntu 7.04 (Feisty Fawn)

[Ubuntu_7.04_Feisty_Fawn.png](https://ja.wikipedia.org/wiki/File:Ubuntu_7.04_Feisty_Fawn.png "fig:Ubuntu_7.04_Feisty_Fawn.png")

Ubuntu 7.04（Feisty Fawn）は、[2007年](../Page/2007年.md "wikilink")[4月19日](../Page/4月19日.md "wikilink")にリリースされた\[17\]。Ubuntu 7.04では、[KVM](https://ja.wikipedia.org/wiki/KVM "wikilink")のサポートや、[Adobe Flash](../Page/Adobe_Flash.md "wikilink")、[Java](https://ja.wikipedia.org/wiki/Java "wikilink")、[MP3](../Page/MP3.md "wikilink")などの制限されたコーデックのサポート、[Nvidia](https://ja.wikipedia.org/wiki/Nvidia "wikilink")や[ATI](https://ja.wikipedia.org/wiki/ATI "wikilink")のドライバのより簡単なインストールなどが含まれた。また、Sudokuやchessゲーム、ディスク使用量アナライザーが追加された\[18\]。

### Ubuntu 7.10 (Gusty Gibbon)

[Ubuntu-desktop-2-710-20080708.png](https://ja.wikipedia.org/wiki/File:Ubuntu-desktop-2-710-20080708.png "fig:Ubuntu-desktop-2-710-20080708.png")

Ubuntu 7.10（Gusty Gibbon）は、2007年[10月18日](../Page/10月18日.md "wikilink")にリリースされた\[19\]。Ubuntu 7.10は、[AppArmor](https://ja.wikipedia.org/wiki/AppArmor "wikilink")セキュリティフレームワーク、より高速なデスクトップ検索、[NTFS-3G](https://ja.wikipedia.org/wiki/NTFS-3G "wikilink")によるより完全な[NTFS](https://ja.wikipedia.org/wiki/NTFS "wikilink")のサポートなどが追加され、[Compiz Fusionが標準で有効化された](../Page/Compiz_Fusion.md "wikilink")。

### Ubuntu 8.04 LTS (Hardy Heron)

[Ubuntu-desktop-2-804-20080708.png](https://ja.wikipedia.org/wiki/File:Ubuntu-desktop-2-804-20080708.png "fig:Ubuntu-desktop-2-804-20080708.png")

Ubuntu 8.04（Hardy Heron）は、[2008年](https://ja.wikipedia.org/wiki/2008年 "wikilink")[4月24日](../Page/4月24日.md "wikilink")にリリースされた\[20\]。本リリースはLTS版である。本リリースでは、Trackerデスクトップ検索の統合や、[Brasero](../Page/Brasero.md "wikilink")ディスクバーナー、[Transmission](https://ja.wikipedia.org/wiki/Transmission "wikilink") [BitTorrent](../Page/BitTorrent.md "wikilink")クライアント、Vinagre [VNC](https://ja.wikipedia.org/wiki/VNC "wikilink")クライアントの追加、[PulseAudio](https://ja.wikipedia.org/wiki/PulseAudio "wikilink")経由のシステムサウンドなどを含む。本リリースから、[Wubi](../Page/Wubi.md "wikilink")インストーラーが追加されたほか、Ubuntu Netbook Remixも導入された\[21\]。

### Ubuntu 8.10 (Intrepid Ibex)

[Ubuntu_screenshot.png](https://ja.wikipedia.org/wiki/File:Ubuntu_screenshot.png "fig:Ubuntu_screenshot.png")

Ubuntu 8.10（Intrepid Ibex）は、2008年[10月30日](../Page/10月30日.md "wikilink")にリリースされた\[22\]。本リリースでは、Ubuntu Live USB creatorが追加されたほか、ゲストアカウントでの利用がサポートされた。

### Ubuntu 9.04 (Jaunty Jackalope)

[Ubuntu_9.04_Jaunty_Jackalope.png](https://ja.wikipedia.org/wiki/File:Ubuntu_9.04_Jaunty_Jackalope.png "fig:Ubuntu_9.04_Jaunty_Jackalope.png")

Ubuntu 9.04（Jaunty Jackalope）は、[2009年](../Page/2009年.md "wikilink")[4月23日](../Page/4月23日.md "wikilink")にリリースされた\[23\]。このリリースでは、起動時間が短縮され、usplash画面が刷新されたほか、ログイン画面が変更された。本バージョンより[ARM](https://ja.wikipedia.org/wiki/ARM "wikilink")アーキテクチャのネイティブサポートが追加されている\[24\]。

### Ubuntu 9.10 (Karmic Koala)

[Ubuntu_9.10.png](https://ja.wikipedia.org/wiki/File:Ubuntu_9.10.png "fig:Ubuntu_9.10.png")

Ubuntu 9.10（Karmic Koala）は、2009年[10月29日](../Page/10月29日.md "wikilink")にリリースされた\[25\]。当初、本リリースでは新しいテーマを含む予定だったが、これは10.04に延期された\[26\]。しかし、デフォルトのテーマに対するマイナーな変更がなされている。また、このバージョンでは、[Pidgin](../Page/Pidgin.md "wikilink")が[Empathy](https://ja.wikipedia.org/wiki/Empathy "wikilink")に置き換えられたほか、デフォルトのファイルシステムが[ext4](https://ja.wikipedia.org/wiki/ext4 "wikilink")に変更されている。

### Ubuntu 10.04 LTS (Lucid Lynx)

[Ubuntu_10.04_screenshot.png](https://ja.wikipedia.org/wiki/File:Ubuntu_10.04_screenshot.png "fig:Ubuntu_10.04_screenshot.png")

Ubuntu 10.04（Lucid Lynx）は、[2010年](https://ja.wikipedia.org/wiki/2010年 "wikilink")[4月29日](../Page/4月29日.md "wikilink")にリリースされた\[27\]。本リリースはLTS版である。このリリースでは、[Nvidia](https://ja.wikipedia.org/wiki/Nvidia "wikilink")グラフィックスドライバのデフォルトが[Nouveau](../Page/Nouveau.md "wikilink")ドライバに変更されたほか、[Plymouth](https://ja.wikipedia.org/wiki/Plymouth "wikilink")が導入された。また、[F-Spot](../Page/F-Spot.md "wikilink")が通常のユーザーレベルでの画像編集に十分な機能を提供しているという理由から[GIMP](../Page/GIMP.md "wikilink")がインストールCDから除かれた\[28\]。

また、2010年[3月4日](../Page/3月4日.md "wikilink")には、本リリースが、新しいUbuntuのヴィジュアルスタイルを反映した新しいロゴを含む新しいテーマを含むことがアナウンスされた\[29\]。

### Ubuntu 10.10 (Maverick Meerkat)

[UbuntuMaverickDesktop.png](https://ja.wikipedia.org/wiki/File:UbuntuMaverickDesktop.png "fig:UbuntuMaverickDesktop.png")'')\]\]

Ubuntu 10.10（Maverick Meerkat）は、2010年[10月10日](../Page/10月10日.md "wikilink")にリリースされた\[30\]。このリリースは、[ネットブック](https://ja.wikipedia.org/wiki/ネットブック "wikilink")エディションのために[Unity](https://ja.wikipedia.org/wiki/Unity "wikilink")インターフェースを含むほか、F-Spotが[Shotwell](https://ja.wikipedia.org/wiki/Shotwell "wikilink")によって置き換えられた。また、公式に[Ubuntuフォントがデフォルトで使われるようになった](../Page/Ubuntu_\(書体\).md "wikilink")\[31\]。

### Ubuntu 11.04 (Natty Narwhal)

[Ubuntu-11-04-cat.png](https://ja.wikipedia.org/wiki/File:Ubuntu-11-04-cat.png "fig:Ubuntu-11-04-cat.png")

Ubuntu 11.04（Natty Narwhal）は、[2011年](../Page/2011年.md "wikilink")[4月28日](../Page/4月28日.md "wikilink")にリリースされた\[32\]。Ubuntu 11.04は、[GNOME Shellの代わりにUnityをユーザーインターフェースとして使っている](https://ja.wikipedia.org/wiki/GNOME_Shell "wikilink")。その他、[Rhythmbox](../Page/Rhythmbox.md "wikilink")を[Banshee](https://ja.wikipedia.org/wiki/Banshee "wikilink")が置き換え、これがデフォルトの音楽プレーヤーになったほか、[OpenStack](https://ja.wikipedia.org/wiki/OpenStack "wikilink")のプラットフォームが加えられた\[33\]。

### Ubuntu 11.10 (Oneiric Ocelot)

[Ubuntu_11.10_Final.png](https://ja.wikipedia.org/wiki/File:Ubuntu_11.10_Final.png "fig:Ubuntu_11.10_Final.png")

Ubuntu 11.10（Oneric Ocelot）は、2011年[10月13日](../Page/10月13日.md "wikilink")にリリースされた\[34\]。このリリースから、[PiTiVi](https://ja.wikipedia.org/wiki/PiTiVi "wikilink")動画エディタがUbuntuのISOファイルから除かれたほか、[Synaptic](../Page/Synaptic.md "wikilink")パッケージマネージャも削除され、オプションで提供される形となった。また、[Mozilla ThunderbirdはGNOME](../Page/Mozilla_Thunderbird.md "wikilink") [Evolution](https://ja.wikipedia.org/wiki/Evolution "wikilink")に置き換えられている\[35\]。

### Ubuntu 12.04 LTS (Precise Pangolin)

[Ubuntu_12.04.png](https://ja.wikipedia.org/wiki/File:Ubuntu_12.04.png "fig:Ubuntu_12.04.png")

Ubuntu 12.04（Precise Pangolin）は、[2012年](../Page/2012年.md "wikilink")[4月26日](../Page/4月26日.md "wikilink")にリリースされた\[36\]。本リリースはLTS版である。Ubuntu Software CenterやUnityに改良が加えられたほか、[Banshee](https://ja.wikipedia.org/wiki/Banshee "wikilink")から[Rhythmbox](../Page/Rhythmbox.md "wikilink")にデフォルトの音楽プレーヤーが戻された。Ubuntu 12.04は、[IPv6](https://ja.wikipedia.org/wiki/IPv6 "wikilink") privacy extensionsがデフォルトで有効化されている最初のバージョンである\[37\]。

### Ubuntu 12.10 (Quantal Quetzal)

[Ubuntu_Desktop_12.10.png](https://ja.wikipedia.org/wiki/File:Ubuntu_Desktop_12.10.png "fig:Ubuntu_Desktop_12.10.png")

Ubuntu 12.10（Quantal Quetzal）は、2012年[10月18日](../Page/10月18日.md "wikilink")にリリースされた\[38\]。このリリースには、Ubuntu Web Appsが含まれ、[Amazon.com](../Page/Amazon.com.md "wikilink")の検索機能を標準で含んでいた（この機能はユーザーによって[GUI](https://ja.wikipedia.org/wiki/GUI "wikilink")で設定できるようになっている）。この機能の是非については、様々な議論が行われた\[39\]。

### Ubuntu 13.04 (Raring Ringtail)

[Ubuntu_13.04_Desktop.png](https://ja.wikipedia.org/wiki/File:Ubuntu_13.04_Desktop.png "fig:Ubuntu_13.04_Desktop.png")

Ubuntu 13.04（Raring Ringtail）は、[2013年](../Page/2013年.md "wikilink")[4月25日](../Page/4月25日.md "wikilink")にリリースされた。このリリースでは、[Wubi](../Page/Wubi.md "wikilink")インストーラーが[Windows 8環境との互換性のなさなどから削除された](https://ja.wikipedia.org/wiki/Windows_8 "wikilink")\[40\]\[41\]。

### Ubuntu 13.10 (Saucy Salamander)

[Ubuntu-13.10.png](https://ja.wikipedia.org/wiki/File:Ubuntu-13.10.png "fig:Ubuntu-13.10.png")

Ubuntu 13.10（Saucy Salamander）は、2013年[10月17日](../Page/10月17日.md "wikilink")にリリースされた\[42\]。このリリースでは、デフォルトのブラウザを[Mozilla Firefoxから](../Page/Mozilla_Firefox.md "wikilink")[Chromium](https://ja.wikipedia.org/wiki/Chromium "wikilink")に変更することが考慮されたが、パッケージアップデートのタイミングの問題などから、Firefoxが継続して用いられることになった\[43\]。また、[Mir](https://ja.wikipedia.org/wiki/Mir "wikilink")ディスプレイサーバーによる[X11](https://ja.wikipedia.org/wiki/X11 "wikilink")の置き換えが計画されたが、標準での置き換えは延期されている。

### Ubuntu 14.04 LTS (Trusty Tahr)

[Ubuntu_14.04-English-25.04.2014.png](https://ja.wikipedia.org/wiki/File:Ubuntu_14.04-English-25.04.2014.png "fig:Ubuntu_14.04-English-25.04.2014.png")

Ubuntu 14.04 LTS（Trusty Tahr）は、[2014年](../Page/2014年.md "wikilink")[4月17日](../Page/4月17日.md "wikilink")にリリースされた。本リリースはLTS版である。このリリースでは、[Nexus 7や](https://ja.wikipedia.org/wiki/Nexus_7 "wikilink")[Nexus 10のようなタブレットへのインターフェースの対応にフォーカスが当てられ](https://ja.wikipedia.org/wiki/Nexus_10 "wikilink")、デスクトップへの変更は少数に留まった。また、本リリースではGNOME Control CenterをフォークしたUnity Control Ceterが導入されたほか、[SSD](../Page/SSD.md "wikilink")のTRIMが標準でサポートされた\[44\]。

### Ubuntu 14.10 (Utopic Unicorn)

[Ubuntu_14.10_Utopic_Desktop.png](https://ja.wikipedia.org/wiki/File:Ubuntu_14.10_Utopic_Desktop.png "fig:Ubuntu_14.10_Utopic_Desktop.png")

Ubuntu 14.10（Utopic Unicorn）は、2014年[10月23日](../Page/10月23日.md "wikilink")にリリースされた。本リリースは、公式にバグ修正とクオリティの向上に焦点が当てられ、新機能の追加は小規模なものとなった\[45\]。

### Ubuntu 15.04 (Vivid Vervet)

[Ubuntu_15.04.png](https://ja.wikipedia.org/wiki/File:Ubuntu_15.04.png "fig:Ubuntu_15.04.png")

Ubuntu 15.04（Vivid Vervet）は、[2015年](../Page/2015年.md "wikilink")[4月23日](../Page/4月23日.md "wikilink")にリリースされた\[46\]。このリリースでは、[systemd](https://ja.wikipedia.org/wiki/systemd "wikilink")が[upstart](https://ja.wikipedia.org/wiki/upstart "wikilink")に変わってデフォルトとなった\[47\]ほか、以前のグローバルメニューがローカルに統合されたメニューで置き換えられた。また、本リリースは、Intel [Haswell](https://ja.wikipedia.org/wiki/Haswell "wikilink")に対するグラフィック性能の改善や、[AMD](https://ja.wikipedia.org/wiki/AMD "wikilink") [Radeon](https://ja.wikipedia.org/wiki/Radeon "wikilink")カードに対するサポートの改善などが含まれている。

### Ubuntu 15.10 (Wily Werewolf)

[Ubuntu_15.10_English.png](https://ja.wikipedia.org/wiki/File:Ubuntu_15.10_English.png "fig:Ubuntu_15.10_English.png")

Ububtu 15.10（Wily Werewolf）は、2015年[10月22日](../Page/10月22日.md "wikilink")にリリースされた\[48\]。このリリースでは、アップストリームのGNOMEのスクロールバーに合わせて、スクロールバーに対する変更が加えられている\[49\]。また、本リリースは[Mir](https://ja.wikipedia.org/wiki/Mir "wikilink")を含んでリリースされることが意図されていたが、結局[Mir](https://ja.wikipedia.org/wiki/Mir "wikilink")なしでのリリースが行われた。

### Ubuntu 16.04 LTS (Xenial Xerus)

[Ubuntu_16.04_Desktop.png](https://ja.wikipedia.org/wiki/File:Ubuntu_16.04_Desktop.png "fig:Ubuntu_16.04_Desktop.png")

Ubuntu 16.04 (Xenial Xerus)は、[2016年](../Page/2016年.md "wikilink")[4月21日](../Page/4月21日.md "wikilink")にリリースされた\[50\]。本リリースはLTS版である。このリリースでは、デフォルトの[デスクトップ環境](https://ja.wikipedia.org/wiki/デスクトップ環境 "wikilink")はUnity 7のままだが、Unity 8がオプションとして提供された。また、本リリースでは、[Ceph](https://ja.wikipedia.org/wiki/Ceph "wikilink")と[ZFS](../Page/ZFS.md "wikilink")のサポートや、[LXD](https://ja.wikipedia.org/wiki/LXD "wikilink")、[Snapパッケージのサポートなども追加されている](https://ja.wikipedia.org/wiki/Snappy "wikilink")\[51\]\[52\]。

### Ubuntu 16.10 (Yakkety Yak)

[Ubuntu_16.10_English.png](https://ja.wikipedia.org/wiki/File:Ubuntu_16.10_English.png "fig:Ubuntu_16.10_English.png")

Ubuntu 16.10（Yakkety Yak）は、2016年[10月13日](../Page/10月13日.md "wikilink")にリリースされた\[53\]。本リリースは依然としてUnity 7のメンテナンスバージョンを用いているが、Unity 8のパッケージがISOに含まれているため、ユーザーはこれをテストすることができる。

### Ubuntu 17.04 (Zesty Zapus)

[Ubuntu_17.04_ukr.png](https://ja.wikipedia.org/wiki/File:Ubuntu_17.04_ukr.png "fig:Ubuntu_17.04_ukr.png")

Ubuntu 17.04（Zesty Zapus）は、[2017年](../Page/2017年.md "wikilink")[4月13日](../Page/4月13日.md "wikilink")にリリースされた\[54\]。このリリースでは、アップストリームの[Debian](../Page/Debian.md "wikilink")プロジェクトでの変更に倣い、[PowerPC](../Page/PowerPC.md "wikilink")（32ビット）アーキテクチャのサポートが廃止された。また、標準の[DNS](https://ja.wikipedia.org/wiki/DNS "wikilink")リゾルバーがsystemd-resolvedに変更されている\[55\]。

### Ubuntu 17.10 (Artful Aardvark)

[Ubuntu-17.10-ca.png](https://ja.wikipedia.org/wiki/File:Ubuntu-17.10-ca.png "fig:Ubuntu-17.10-ca.png")

Ubuntu 17.10（Artful Aardvark）は、2017年[10月19日](../Page/10月19日.md "wikilink")にリリースされた\[56\]。このリリースは、Ubuntuが[GNOME Shellインターフェースを使う最初のリリースであり](https://ja.wikipedia.org/wiki/GNOME_Shell "wikilink")、また[X11](https://ja.wikipedia.org/wiki/X11 "wikilink")が[Wayland](https://ja.wikipedia.org/wiki/Wayland "wikilink")に置き換えられている。また、このリリースでは32ビットのデスクトップイメージが廃止されたが、minimal ISOを用いることにより、依然として32ビットシステムをインストールすることはできる\[57\]。

### Ubuntu 18.04 LTS (Bionic Beaver)

[Ubuntu_18.04.png](https://ja.wikipedia.org/wiki/File:Ubuntu_18.04.png "fig:Ubuntu_18.04.png")

Ubuntu 18.04（Bionic Beaver）は、[2018年](../Page/2018年.md "wikilink")[4月26日](../Page/4月26日.md "wikilink")にリリースされた\[58\]。本リリースはLTS版である。本リリースでは、新しいテーマが導入されることが計画されていたが、バグの修正などが未完了のため、新しいテーマの導入は見送られた\[59\]。また、新しいTo-Doアプリが追加されたほか、インストーラーに"Minimal Install"オプション（ブラウザや基本的なツールのみをインストールする）が追加されている。

### Ubuntu 18.10 (Cosmic Cuttlefish)

[Ubuntu_18.10.png](https://ja.wikipedia.org/wiki/File:Ubuntu_18.10.png "fig:Ubuntu_18.10.png")

Ubuntu 18.10（Cosmic Cuttlefish）は、2018年[10月18日](../Page/10月18日.md "wikilink")にリリースされた\[60\]。本リリースでは、Zstandardとして知られる新たなロスレス圧縮方式を利用することにより、インストールのスピードが改善されている。また、プリインストールされているSnapアプリケーションの起動速度が改善されている\[61\]。

### Ubuntu 19.04 (Disco Dingo)

[Ubuntu_19.04_"Disco_Dingo".png](https://ja.wikipedia.org/wiki/File:Ubuntu_19.04_"Disco_Dingo".png "fig:Ubuntu_19.04_\"Disco_Dingo\".png")

Ubuntu 19.04（Disco Dingo）は、[2019年](../Page/2019年.md "wikilink")[4月18日](../Page/4月18日.md "wikilink")にリリースされた\[62\]。本リリースは、[Linuxカーネル](../Page/Linuxカーネル.md "wikilink") 5.0を用いており、AMD FreeSyncテクノロジや、Adiantum暗号化、[Btrfs](https://ja.wikipedia.org/wiki/Btrfs "wikilink")スワップファイル、多くのUSB 3.2などのサポート、USB Type-C対応の改善などが実施されている\[63\]。

### Ubuntu 19.10 (Eoan Ermine)

[Ubuntu_19.10_Eoan_Ermine.png](https://ja.wikipedia.org/wiki/File:Ubuntu_19.10_Eoan_Ermine.png "fig:Ubuntu_19.10_Eoan_Ermine.png")

Ubuntu 19.10（Eoan Ermine）は、2019年[10月17日](../Page/10月17日.md "wikilink")にリリースされた。本リリースは、[Ryzen](https://ja.wikipedia.org/wiki/Ryzen "wikilink")マザーボードへの対応などを含むLinuxカーネル 5.3を含む\[64\]ほか、[ZFS](../Page/ZFS.md "wikilink")の試験的なサポートなどの機能が追加された。

### Ubuntu 20.04 (Focal Fossa)

[Desktop_Ubuntu_20.04.png](https://ja.wikipedia.org/wiki/File:Desktop_Ubuntu_20.04.png "fig:Desktop_Ubuntu_20.04.png")

Ubuntu 20.04（Focal Fossa）は、2019年秋時点で[2020年](../Page/2020年.md "wikilink")[4月23日](../Page/4月23日.md "wikilink")にリリースされる予定\[65\]であったが、予定通り同日にリリースが行われた\[66\]\[67\]。本リリースは、18.04以来2年ぶりとなるLTSリリースで5年間のサポートが提供されるほか、ESM（Extended Security Maintenance）プログラムを利用すると、最長10年のサポートが受けられる\[68\]。機能的には、新たな[VPN](https://ja.wikipedia.org/wiki/VPN "wikilink")の方式である[WireGuard](../Page/WireGuard.md "wikilink")への対応が行われたほか、サーバーの自動インストール機能の追加が行われた\[69\]。また、デスクトップ関連では[GNOME](../Page/GNOME.md "wikilink") 3.36が採用され、パフォーマンスが改善されたほか、ロック画面やフォルダアイコン、ブートスプラッシュのデザインが変更され、デスクトップテーマはライトテーマとダークテーマを選択できるようになった\[70\]。
