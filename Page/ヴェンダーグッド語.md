**ヴェンダーグッド語** (**Vendergood**) とは、[数学者](../Page/数学者.md "wikilink")[ウィリアム・ジェイムズ・サイディズ](../Page/ウィリアム・ジェイムズ・サイディズ.md "wikilink")が幼い頃に発明した[人工言語](https://ja.wikipedia.org/wiki/人工言語 "wikilink")。

サイディズは、8歳で著した2冊目の著書*Book of Vendergood*で、この言語を説明している。ヴェンダーグッド語の大半は、[ラテン語](../Page/ラテン語.md "wikilink")や[ギリシア語](https://ja.wikipedia.org/wiki/ギリシア語 "wikilink")、[ドイツ語](../Page/ドイツ語.md "wikilink")、[フランス語](https://ja.wikipedia.org/wiki/フランス語 "wikilink")、そしてその他の[ロマンス諸語](../Page/ロマンス諸語.md "wikilink")に基づいている。ヴェンダーグッド語には、[直説法](https://ja.wikipedia.org/wiki/直説法 "wikilink")、[仮定法](https://ja.wikipedia.org/wiki/仮定法 "wikilink")、[絶対命令法](https://ja.wikipedia.org/wiki/絶対命令法 "wikilink")、[接続法](../Page/接続法.md "wikilink")、[命令法](https://ja.wikipedia.org/wiki/命令法 "wikilink")、[不定法](https://ja.wikipedia.org/wiki/不定法 "wikilink")、[願望法](https://ja.wikipedia.org/wiki/願望法 "wikilink")、そしてサイディズが独自に考案した強調法の8つの活用形がある。[冠詞](../Page/冠詞.md "wikilink")は[性](https://ja.wikipedia.org/wiki/性 "wikilink")によって変化し、ある評者からは「[日本語](../Page/日本語.md "wikilink")の[動詞](../Page/動詞.md "wikilink")の活用より複雑である」と評された。

ヴェンダーグッド語では[12進法を採用している](../Page/十二進法.md "wikilink")。その理由についてサイディズは「物を売るときは12個（1ダース）が単位になっているし、12は4つの[因数](https://ja.wikipedia.org/wiki/因数 "wikilink")を持つ最小の数だからだ\!」と説明している。

| ヴェンダーグッド語        | 英語       |
| ---------------- | -------- |
| eis              | one      |
| duet             | two      |
| tre              | three    |
| quar             | four     |
| quin             | five     |
| sex              | six      |
| sep              | seven    |
| oo/oe            | eight    |
| non              | nine     |
| ecem             | ten      |
| elevenos         | eleven   |
| dec              | twelve   |
| eidec (eis, dec) | thirteen |

現存する僅かなヴェンダーグッド語の例文は以下の通りである。

| ヴェンダーグッド語                 | 英語                         |
| ------------------------- | -------------------------- |
| Amevo (-)ne the neania?   | Do you love the young man? |
| The toxoteis obscurit.    | The bowman obscures.       |
| Quen diseois-nar?         | What do you learn? (sing.) |
| (Euni) disceuo.           | I am learning Vendergood.  |
| Obscureuo ecem agrieolai. | I obscure ten farmers.     |
