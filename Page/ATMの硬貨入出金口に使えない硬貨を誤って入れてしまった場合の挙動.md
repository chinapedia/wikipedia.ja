> この記事は[ATMの硬貨入出金口に使えない硬貨を誤って入れてしまった場合の挙動](https://ja.wikipedia.org/wiki/ATMの硬貨入出金口に使えない硬貨を誤って入れてしまった場合の挙動)から翻訳されています。


この項では、日本の[現金自動預け払い機](../Page/現金自動預け払い機.md "wikilink")（ATM）において、「お預入れ」や「お振込み」などの取引でその硬貨入出金口に外国硬貨や[変造・毀損硬貨](https://ja.wikipedia.org/wiki/硬貨#偽造・変造 "wikilink")、古銭などの使えない硬貨を誤って入れてしまった場合にどう動くかを述べる。

そのような場合の挙動は、場合によって、大きく次の表に示す5パターンに分けられる。

| 挙動・事象                                                                                                   | 該当例                                                                                                                                                                                                                                                                                                                                                                                                                       |
| ------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 当該硬貨がATMの硬貨入出金口に戻ってくる。                                                                                  | 外国硬貨・近代銭のうち、材質が純[ニッケル](../Page/ニッケル.md "wikilink")以外で、サイズが現代の日本で通用する[500円玉](https://ja.wikipedia.org/wiki/500円玉 "wikilink")のサイズ（26.5mm）から[1円玉](https://ja.wikipedia.org/wiki/1円玉 "wikilink")のサイズ（20mm）までの範囲に収まるものは大抵こうなる（最も新しい日本通貨（[記念硬貨](https://ja.wikipedia.org/wiki/記念硬貨 "wikilink")除く）としては500円白銅貨）。それに該当しなくてもこのように戻ってくる例としては、直径が1円玉より少し小さい小型50銭黄銅貨・10銭銀貨や、機械でなく手作業で作られた[寛永通宝](../Page/寛永通宝.md "wikilink")銅銭などが挙げられる。 |
| 当該硬貨がATMの下の方にある返却口に返却される。                                                                               | 竜1銭銅貨（直径が500円玉より若干大きい）、[一分銀](../Page/一分銀.md "wikilink")（長方形でそれなりの大きさがある）など                                                                                                                                                                                                                                                                                                                                                |
| ATMが動作を停止し、「お取り扱いできなくなりました」等のメッセージが表示され、係員呼び出しとなる。                                                      | [天保通宝](../Page/天保通宝.md "wikilink")のように大きすぎるものや、50円ニッケル貨のように磁力の強いものなど。                                                                                                                                                                                                                                                                                                                                                     |
| ATMの内部処理的には金額的に当該硬貨が無視されるだけで何事もなかったかのように動作するものの、当該硬貨はATMに飲み込まれ、それっきり戻ってこず、二度と取り出すことができなくなってしまい、事実上消滅する。 | 1銭錫貨、1円金貨、[一朱銀](../Page/一朱銀.md "wikilink")、[二朱金](../Page/二朱金.md "wikilink")、明治[二分金](../Page/二分金.md "wikilink")など、サイズが小さく薄いものはそのリスクが高い。                                                                                                                                                                                                                                                                                    |
| 正規判定されて処理される。                                                                                           | [韓国の変造500ウォン硬貨が混ざっていた場合など](https://ja.wikipedia.org/wiki/大韓民国ウォン#日本における変造500ウォン硬貨問題 "wikilink")。処理中止（返金処理、返却レバー）した場合には機械内部に[備蓄された正規硬貨が戻って来る場合や](https://ja.wikipedia.org/wiki/硬貨#偽造・変造 "wikilink")、投入されたものが保留されていてそのまま戻る場合がある。                                                                                                                                                                                            |
