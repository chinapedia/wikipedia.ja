[thumb](https://ja.wikipedia.org/wiki/ファイル:JGR-3300SLa.jpg "wikilink") **3300形**は、かつて[日本国有鉄道](../Page/日本国有鉄道.md "wikilink")およびその前身である[鉄道院・鉄道省に在籍した](https://ja.wikipedia.org/wiki/鉄道省 "wikilink")[タンク式](../Page/タンク機関車.md "wikilink")[蒸気機関車](../Page/蒸気機関車.md "wikilink")である。

## 概要

[アメリカ合衆国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")の[ボールドウィン・ロコモティブ・ワークス](../Page/ボールドウィン・ロコモティブ・ワークス.md "wikilink")から輸入された、[車軸配置](https://ja.wikipedia.org/wiki/車軸配置 "wikilink")[2-6-2(1C1)の単式](https://ja.wikipedia.org/wiki/車輪配置_2-6-2 "wikilink")2気筒の[飽和式のタンク機関車である](https://ja.wikipedia.org/wiki/水蒸気#飽和蒸気と過熱蒸気 "wikilink")。[3030形や](https://ja.wikipedia.org/wiki/国鉄3030形蒸気機関車 "wikilink")[3250形とは寸法がやや大きい程度の同系車で](../Page/国鉄3250形蒸気機関車.md "wikilink")、メーカー規格では10-24 1/4Dである。また、山陽鉄道32形（後の[鉄道院3360形](https://ja.wikipedia.org/wiki/国鉄3360形蒸気機関車 "wikilink")）は、本形式を[ヴォークレイン4気筒複式にした同系機である](https://ja.wikipedia.org/wiki/複式機関#ボークレイン式 "wikilink")。

大きさが手頃で貨物用や勾配線用に適したため、[筑豊鉄道](https://ja.wikipedia.org/wiki/筑豊鉄道 "wikilink")（初代）、[播但鉄道](https://ja.wikipedia.org/wiki/播但鉄道 "wikilink")、[豊州鉄道](https://ja.wikipedia.org/wiki/豊州鉄道 "wikilink")（初代）および[山陽鉄道](https://ja.wikipedia.org/wiki/山陽鉄道 "wikilink")により計24両が導入された。[1906年](../Page/1906年.md "wikilink")（明治39年）に制定された[鉄道国有法](../Page/鉄道国有法.md "wikilink")により、[九州鉄道](../Page/九州鉄道.md "wikilink")（初代）、山陽鉄道を経由して全車が官設鉄道に編入されている。全車が全く同形態というわけでなく、導入した会社によって諸元や形態に若干の差があった。

## 筑豊鉄道

筑豊鉄道では、[1890年](../Page/1890年.md "wikilink")（明治23年）から[1895年](../Page/1895年.md "wikilink")（明治28年）にかけて、12両が導入された。[1897年](../Page/1897年.md "wikilink")（明治30年）に筑豊鉄道が九州鉄道に営業譲渡したのにともない、九州鉄道の**73形**となった。その状況は、次のとおりである。

  - 1890年製（2両）
      - [製造番号](../Page/製造番号.md "wikilink")11123 - 筑豊鉄道**3** → 九州鉄道**73**
      - 製造番号11124 - 筑豊鉄道**4** → 九州鉄道**74**
  - 1893年製（5両）
      - 製造番号13766 - 筑豊鉄道**10** → 九州鉄道**80**
      - 製造番号13767 - 筑豊鉄道**11** → 九州鉄道**81**
      - 製造番号13770 - 筑豊鉄道**12** → 九州鉄道**82**
      - 製造番号13771 - 筑豊鉄道**13** → 九州鉄道**83**
      - 製造番号13779 - 筑豊鉄道**14** → 九州鉄道**84**
  - 1894年製（3両）
      - 製造番号14034 - 筑豊鉄道**17** → 九州鉄道**87**
      - 製造番号14035 - 筑豊鉄道**18** → 九州鉄道**88**
      - 製造番号14037 - 筑豊鉄道**19** → 九州鉄道**89**
  - 1895年製（2両）

      - 製造番号14389 - 筑豊鉄道**28** → 九州鉄道**98**

また、[1898年](../Page/1898年.md "wikilink")（明治31年）[4月8日](../Page/4月8日.md "wikilink")に[幸袋駅](https://ja.wikipedia.org/wiki/幸袋駅 "wikilink")構内で入換作業中の13号（九州鉄道83号）のボイラー破裂事故が発生しているが、修復された。*[日本の鉄道事故 (1949年以前)\#九州鉄道蒸気機関車ボイラー破裂事故も参照](https://ja.wikipedia.org/wiki/日本の鉄道事故_\(1949年以前\)#九州鉄道蒸気機関車ボイラー破裂事故 "wikilink")。*

## 播但鉄道・山陽鉄道

播但鉄道では[1893年](../Page/1893年.md "wikilink")（明治26年）に4両（1 - 4）が開業用として用意されたが、翌年に2が山陽鉄道に譲渡され**6形**とされるとともに、山陽鉄道では2両を増備し、同形式に編入した。播但鉄道では、欠番を埋めるため、4を2代目の2としている。その後、[1903年](../Page/1903年.md "wikilink")（明治36年）に播但鉄道が山陽鉄道に営業譲渡したのにともない、播但鉄道に残っていたL1形（1 - 3）が**23形**として山陽鉄道に編入された。[1894年](../Page/1894年.md "wikilink")（明治27年）に譲り受けた1両と1903年に編入された3両は同形のはずであるが、形式が分かれた理由は不明である。その状況は、次のとおりである。

  - 1893年製（4両）
      - 製造番号13838 - 播但鉄道**1** → 山陽鉄道**113**
      - 製造番号13840 - 播但鉄道**2** → 山陽鉄道**34**
      - 製造番号13842 - 播但鉄道**3** → 山陽鉄道**115**
      - 製造番号13843 - 播但鉄道**4** → **2**\[II\] → 山陽鉄道**114**
  - 1894年製（2両）
      - 製造番号14108 - 山陽鉄道**32**
      - 製造番号14109 - 山陽鉄道**33**

## 豊州鉄道

豊州鉄道では、1894年と[1896年](../Page/1896年.md "wikilink")（明治29年）に3両ずつ計6両が導入された。豊州鉄道は1901年に九州鉄道へ営業譲渡したため、同社の**167形**とされた。先に編入されていた旧筑豊鉄道の同形車と同じ形式にならなかったが、その理由は不明である。

  - 1894年製（3両）
      - 製造番号14061 - 豊州鉄道**2** → 九州鉄道**167**
      - 製造番号14062 - 豊州鉄道**3** → 九州鉄道**168**
      - 製造番号14063 - 豊州鉄道**4** → 九州鉄道**169**
  - 1896年製（3両）
      - 製造番号14800 - 豊州鉄道**6** → 九州鉄道**171**
      - 製造番号14801 - 豊州鉄道**7** → 九州鉄道**172**
      - 製造番号14802 - 豊州鉄道**8** → 九州鉄道**173**

## 廃車と譲渡

国有化後の本形式は、旧山陽鉄道のものは姫路から梅小路、広島、下関に移り、旧九州鉄道のものは大里、若松、行橋、大分、熊本などに分散して配置され、[貨物列車](https://ja.wikipedia.org/wiki/貨物列車 "wikilink")牽引用や入換用として使用された。

[廃車は](../Page/廃車_\(鉄道\).md "wikilink")[1918年](../Page/1918年.md "wikilink")（大正7年）の3310\[1\]、[1922年](../Page/1922年.md "wikilink")（大正11年）の3314から始まった。1926年には、神戸鉄道局に4両、門司鉄道局に14両、建設局に4両の配置であった。[1927年](../Page/1927年.md "wikilink")（昭和2年）から[1932年](../Page/1932年.md "wikilink")（昭和7年）にかけて、次の13両が民間に払下げられている。下記のうち、[佐久鉄道](https://ja.wikipedia.org/wiki/佐久鉄道 "wikilink")に払下げられたものは、[1934年](../Page/1934年.md "wikilink")（昭和9年）の同鉄道の[国有化により再び国有鉄道籍となった](../Page/鉄道敷設法.md "wikilink")。佐久鉄道では3301 - 3303に番号を揃えていたため、書類上は国有鉄道時代の番号に戻されたが、2両は佐久鉄道時代のナンバープレートを外さないまま、再び払い下げられた。

  - 3300（1927年） → [相模鉄道](https://ja.wikipedia.org/wiki/相模鉄道 "wikilink")3300（1932年廃車）
  - 3301（1927年） → 相模鉄道3301（1932年廃車）
  - 3302（1927年） → [片上鉄道](https://ja.wikipedia.org/wiki/同和鉱業片上鉄道 "wikilink")4（1951年廃車）
  - 3303（1927年） → 佐久鉄道3303 → 国有鉄道3303
  - 3306（1927年） → 相模鉄道3306（1932年廃車）
  - 3308（1928年） → 片上鉄道5 → 鉄道車両工業5（1940年譲受）
  - 3309（1927年） → 佐久鉄道3302 → 国有鉄道3309 → 片上鉄道8（1936年譲受。1953年廃車）
  - 3311 → [明治鉱業](https://ja.wikipedia.org/wiki/明治鉱業 "wikilink")平山鉱業所（1959年廃車）
  - 3313（1927年） → 佐久鉄道3301 → 国有鉄道3313 → 片上鉄道9（1935年譲受。1955年廃車）
  - 3315（1932年） → 片上鉄道6（1951年廃車）
  - 3316（1927年） → [王子製紙豊原工場](https://ja.wikipedia.org/wiki/王子製紙_\(初代\) "wikilink")
  - 3317（1929年） → [横荘鉄道](https://ja.wikipedia.org/wiki/横荘鉄道 "wikilink")5（1950年廃車）
  - 3319（1927年） → 相模鉄道3319（1932年廃車）

本形式が最終的に国有鉄道から除籍されたのは、[1950年](../Page/1950年.md "wikilink")（昭和25年）の3323（建設局所属）であった。
