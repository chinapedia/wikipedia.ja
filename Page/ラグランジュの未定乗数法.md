**ラグランジュの未定乗数法**（ラグランジュのみていじょうすうほう、）とは、束縛条件のもとで[最適化を行うための](../Page/最適化問題.md "wikilink")[数学](../Page/数学.md "wikilink")（[解析学](../Page/解析学.md "wikilink")）的な方法である。いくつかの[変数に対して](../Page/変数_\(数学\).md "wikilink")、いくつかの[関数の値を固定するという束縛条件のもとで](../Page/関数_\(数学\).md "wikilink")、別のある1つの関数の極値を求めるという問題を考える。各束縛条件に対して定数（**未定乗数**、）を用意し、これらを係数とする[線形結合](https://ja.wikipedia.org/wiki/線形結合 "wikilink")を新しい関数（未定乗数も新たな変数とする）として考えることで、束縛問題を普通の[極値](../Page/極値.md "wikilink")問題として解くことができる方法である。

## 定理

ラグランジュの未定乗数法は、次のような定理として記述される。

### 2次元の場合

束縛条件 *g*(*x*, *y*) = 0 のもとで、*f*(*x*, *y*)が最大値となる点 (*a* , *b*) を求める問題，つまり

  -
    maximize \(f(x, y)\)
    subject to \(g(x, y) = 0\)

という問題を考える。**ラグランジュ乗数**をλとし、  とおく。点 (*a*, *b*) で ∂*g*/∂*x* と ∂*g*/∂*y* の少なくとも一方が0でないならば、*α* が存在して点 (*a*, *b*, *α*) で  が成り立つ\[1\] 。

### 一般の多次元の場合

*n* 次元空間の点 ***x**'' = (*x*,..., *x*) のある領域 *R* を定義域とする被評価関数 *z* = *f*(***x**'') が、同じ領域を定義域とする *m* 次元ベクトル値関数  のもとで、*R* 内の点 ***x*** において極値をとるための[必要条件](https://ja.wikipedia.org/wiki/必要条件 "wikilink")は、その点における*f* の[勾配](https://ja.wikipedia.org/wiki/勾配 "wikilink")ベクトル  が、その点で、*m* 個の*g* それぞれの勾配ベクトルが張る*m* 次元[線型部分空間](../Page/線型部分空間.md "wikilink")に含まれること、すなわち、スカラーの組 ***λ**'' = (*λ*,..., *λ* ) を用いて、  が成り立つことである。移項して∇を取れば、  が[停留点をとることである](https://ja.wikipedia.org/wiki/極値#極値点の判定 "wikilink")。ただし、{∇*g*,...,∇*g* } は[一次独立](../Page/線型独立.md "wikilink")、すなわち  でなければならない。式(1)の*m* 本と式(2)の*n* 本の式を連立させて、***x**'' と**λ**の (*n* + *m* ) 個の未知数について解けば、*f* の極値を与える候補点が得られる\[2\]。

## 解釈

### 幾何学的な説明

[LagrangeMultipliers3D.png](https://ja.wikipedia.org/wiki/File:LagrangeMultipliers3D.png "fig:LagrangeMultipliers3D.png") [LagrangeMultipliers2D.svg](https://ja.wikipedia.org/wiki/File:LagrangeMultipliers2D.svg "fig:LagrangeMultipliers2D.svg") 簡単のため2[次元](../Page/次元.md "wikilink")の場合を考えよう。*g* (*x*,*y* ) = *c*（ここで*c* は与えられた定数である）という条件のもと、関数*f* (*x*,*y* ) を最大化するものとしよう。*f* の値を高さとしたグラフを考えると、高さが *d* の *f* の等高線は *f* (*x*,*y* ) = *d* で与えられる。ここで、任意の曲線に沿って移動する点を考えると、この点が等高線を横切る場合、必ず *f* (*x*,*y* ) は増加、もしくは減少するが、この点が等高線に沿って移動する場合は *f* (*x*,*y* ) は変化しないことが分かる。この条件と通常の極値の条件を合わせて考えれば、曲線上で *f* (*x*,*y* ) が最大をとる点では、*f* の等高線の接線と曲線の接線が平行となっているか、*f* の勾配がゼロとなっていることが分かる。 ここで *g* (*x*,*y* ) = *c* の接線は、*g* の勾配ベクトル \(\nabla_{x,y} g\)と直交し、また *f* の等高線 *f* (*x*,*y* ) = *d* の接線は *f* の勾配ベクトル \(\nabla_{x,y} f\) と直交することをふまえると、前述の条件は

\[\nabla_{x,y} f = \lambda \nabla_{x,y} g\] と書ける。 ここで

\[\nabla_{x,y} f= \left( \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y} \right), \qquad \nabla_{x,y} g= \left( \frac{\partial g}{\partial x}, \frac{\partial g}{\partial y} \right).\]

である。定数は *f* の勾配ベクトルと *g* の勾配ベクトルが平行ではあるが長さが一般に異なるために必要である。 がゼロの場合、*f* (*x*,*y* ) の勾配がゼロとなる条件になる。これは *g* (*x*,*y* ) = *c* の曲線上にちょうど *f* の最大値があるため、曲線上で *f* (*x*,*y* )が最大を取る点と通常の *f* (*x*,*y* ) の最大値が一致するケースである。

前述の式を変形すると

\[\nabla_{x,y}(f-\lambda g)=0\] となることから、\(f-\lambda g\)の極値を求めればいいことになる。

### 物理的な解釈

[物理学](../Page/物理学.md "wikilink")の問題を解くとき、ラグランジュの未定乗数は単なる方便ではなく、ある[物理量](../Page/物理量.md "wikilink")を表すことが多い。

たとえば、[流体力学](../Page/流体力学.md "wikilink")において、[非圧縮性流れ](https://ja.wikipedia.org/wiki/非圧縮性流れ "wikilink")の[ナビエ-ストークス方程式](https://ja.wikipedia.org/wiki/ナビエ-ストークス方程式 "wikilink")を解く場合、[圧力](../Page/圧力.md "wikilink")は速度ベクトル場が[連続の式](https://ja.wikipedia.org/wiki/連続の式 "wikilink")という束縛条件を満たすための未定乗数として求められる\[3\]。
