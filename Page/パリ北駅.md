**パリ北駅**（パリ・ノール、パリ・ガール・デュ・ノール、、ガール・デュ・ノール、、、）は、[フランス](https://ja.wikipedia.org/wiki/フランス "wikilink")の首都・[パリ](https://ja.wikipedia.org/wiki/パリ "wikilink")市[10区にある](https://ja.wikipedia.org/wiki/10区_\(パリ\) "wikilink")[フランス国鉄](../Page/フランス国鉄.md "wikilink") (SNCF)・[パリ交通公団](../Page/パリ交通公団.md "wikilink") (RATP) の[駅](../Page/鉄道駅.md "wikilink")。SNCFにおけるパリの主要[ターミナル駅](../Page/ターミナル駅.md "wikilink")の一つである。[RERと](../Page/RER_\(イル＝ド＝フランス\).md "wikilink")[メトロの路線では](https://ja.wikipedia.org/wiki/パリメトロ "wikilink")**ガール・デュ・ノール** () という駅名が使用され、[TGV](https://ja.wikipedia.org/wiki/TGV "wikilink")などの遠距離路線、国際路線では**パリ・ノール** () である。

## 駅構造

### フランス国鉄・RER

[thumb](https://ja.wikipedia.org/wiki/ファイル:Gare_du_Nord_Nouveau_pole_échange_banlieue.JPG "wikilink") [thumb](https://ja.wikipedia.org/wiki/ファイル:ThalysTGVparis1.JPG "wikilink") 地上部分に南北方向に[頭端式ホーム](../Page/頭端式ホーム.md "wikilink")28線、地下にRER用の[島式ホーム](https://ja.wikipedia.org/wiki/島式ホーム "wikilink")2面4線を有する。

地上ホームは西から順に1 - 21番線および30 - 36番線で、以下のように使われている。

| ホーム     | 発着列車\<\!--        | 行先--\> |
| ------- | ----------------- | ------ |
| 1・2     | 業務用であり旅客列車は使用しない。 |        |
| 3 - 6   | ユーロスター            |        |
| 7・8     | タリス               |        |
| 9 - 18  | TGVなど長距離列車、一部のTER |        |
| 19 - 21 | TERピカルディ          |        |
| 30 - 40 | トランジリアン           |        |
| 41 - 44 | RER（地下ホーム）        |        |

**パリ北駅ホーム**

ユーロスター用ホームの入口には[出入国審査場が設けられており](../Page/出入国管理.md "wikilink")、チェックインを済ませた乗客以外はホームに入ることができない。これは[イギリス](https://ja.wikipedia.org/wiki/イギリス "wikilink")が[シェンゲン協定](../Page/シェンゲン協定.md "wikilink")に加盟していないためである。

またトランジリアン用のホームの入口には[自動改札機](../Page/自動改札機.md "wikilink")が設置されており、他のホームとは分離されている。地下のRER駅にも自動改札機があり、近郊列車ホームとは改札内で連絡している。

地上ホームの南端に接して1865年に完成した駅舎がある。これは[ジャック・イトルフ](https://ja.wikipedia.org/wiki/ジャック・イトルフ "wikilink")の設計によるもので、正面にはパリおよびの沿線の諸都市を象徴する23の女神像がある。駅舎内には出札窓口や案内所のほか、レストラン、カフェ、郵便局、警察署などがある。

### パリメトロ

4号線・5号線ともに[相対式ホーム](https://ja.wikipedia.org/wiki/相対式ホーム "wikilink")2面2線を有する[地下駅](../Page/地下駅.md "wikilink")である。

## 利用状況

SNCFにおける利用者数は年間約1億9千万人で、フランス国鉄の駅のみならず、ヨーロッパにある鉄道駅の中でも最多である\[1\]。

## 歴史

北駅は[北部鉄道のターミナル駅として](https://ja.wikipedia.org/wiki/北部鉄道_\(フランス\) "wikilink")[1846年](../Page/1846年.md "wikilink")[6月14日](../Page/6月14日.md "wikilink")に開業し、同日に開業したパリ - [アミアン](../Page/アミアン.md "wikilink") - リール線の起点となった。駅はすぐに手狭となったため、[1860年](../Page/1860年.md "wikilink")には駅舎の一部を解体して敷地を拡張した。

[1861年](../Page/1861年.md "wikilink")からイトルフの設計による現駅舎の建設が始まり、[1864年](../Page/1864年.md "wikilink")には一部工事中のまま使用が開始され、[1865年](../Page/1865年.md "wikilink")に完成した。ホーム全体は大きな[トレイン・シェッド](https://ja.wikipedia.org/wiki/トレイン・シェッド "wikilink")で覆われており、これを支える柱は[鋳鉄](../Page/鋳鉄.md "wikilink")製である。この柱は[スコットランド](../Page/スコットランド.md "wikilink")製であるが、これは当時そのような大きな柱を作れる鋳造所がスコットランドにしかなかったためである。

その後も利用者の増加にともない数度にわたってホームの増設や内装の変更が行なわれ、さらに東側に近郊列車用のホームが増設された。また[1906年](../Page/1906年.md "wikilink")にメトロ5号線が、[1908年](../Page/1908年.md "wikilink")に4号線がそれぞれ開業した。

北駅はドイツ北部から[北ヨーロッパ](../Page/北ヨーロッパ.md "wikilink")、[東ヨーロッパ](../Page/東ヨーロッパ.md "wikilink")北部、東ヨーロッパ東部、[ロシア](../Page/ロシア.md "wikilink")へ至る鉄道網のターミナルとなり、また[鉄道連絡船](../Page/鉄道連絡船.md "wikilink")を介してイギリスへの列車も運転された。[ロシア](../Page/ロシア.md "wikilink")方面への「[北急行](https://ja.wikipedia.org/wiki/北急行 "wikilink")」やロンドン行「[フレッシュ・ドール](https://ja.wikipedia.org/wiki/黄金の矢_\(列車\) "wikilink")」といった列車の始発駅であった。[日本](https://ja.wikipedia.org/wiki/日本 "wikilink")から[シベリア鉄道](../Page/シベリア鉄道.md "wikilink")経由でフランスに向かった場合も、多くは当駅に到着した。

[1981年](../Page/1981年.md "wikilink")にRER B線の地下駅が開業したが、当駅を境として[電化方式が北側は](../Page/鉄道の電化.md "wikilink")[交流](../Page/交流電化.md "wikilink")25,000[V](../Page/ボルト_\(単位\).md "wikilink")50[Hz](../Page/ヘルツ.md "wikilink")、南側は[直流](../Page/直流電化.md "wikilink")1,500Vと異なることから[直通運転](../Page/直通運転.md "wikilink")は行なわれず、運行系統は当駅で分断され、乗り換えが必要であった。[1983年](https://ja.wikipedia.org/wiki/1983年 "wikilink")以降、交直切替のための地上設備（[デッドセクション](../Page/デッドセクション.md "wikilink")）が設けられ、順次[交直流電車](../Page/交直流電車.md "wikilink")が導入され直通運転が行なわれるようになった。[1987年](https://ja.wikipedia.org/wiki/1987年 "wikilink")にはRER D線が開業した。これらにより地上駅を起点としていた近郊列車の一部が地下線経由となり、パリ中心部を経て南郊外まで直通するようになった。

[1993年](../Page/1993年.md "wikilink")には[LGV北線](https://ja.wikipedia.org/wiki/LGV北線 "wikilink")が開業し、フランス北部方面へのTGVの始発駅となった。[1994年](../Page/1994年.md "wikilink")のユーロスター乗り入れに伴い、3 - 6番線はその専用ホームとなった。
