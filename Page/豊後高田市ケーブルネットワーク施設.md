**豊後高田市ケーブルネットワーク施設**は、2008年にサービスを開始した市営の[ケーブルテレビ](../Page/ケーブルテレビ.md "wikilink")局である。

## 概要

大分県[豊後高田市](https://ja.wikipedia.org/wiki/豊後高田市 "wikilink")が市内の情報格差是正を目的としてインフラを整備し、一部のサービスを民間事業者に委託する形式で行っている。地理的に[皿倉山](https://ja.wikipedia.org/wiki/皿倉山 "wikilink")や[大坂山](https://ja.wikipedia.org/wiki/大坂山 "wikilink")から電波が届きやすい状況にあったことから、県内では数少ない福岡民放全社の再放送サービスを提供している。

| CATV加入者間無料電話      |                                               |
| 衛星波系多チャンネルテレビサービス |                                               |
| インターネットサービス       | [NTT西日本](../Page/西日本電信電話.md "wikilink")（大分支店） |
| ひかり電話             |                                               |

過去にはテレビサービスの運用を[大分ケーブルテレコム](../Page/大分ケーブルテレコム.md "wikilink")に委託していたことがあったが、一部在福民放の再放送停止問題や委託先の[J:COMグループ入りに伴い委託を解消](../Page/ジュピターテレコム.md "wikilink")。多チャンネルテレビサービスに関しては市が直接[大分県デジタルネットワークセンター](https://ja.wikipedia.org/wiki/大分県デジタルネットワークセンター "wikilink")から供給を受ける形式に変わった。
