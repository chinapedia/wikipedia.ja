**自由民主党総裁**（じゆうみんしゅとうそうさい、英：Leader of the Liberal Democratic Party）は、[自由民主党の](../Page/自由民主党_\(日本\).md "wikilink")[党首](../Page/党首.md "wikilink")。[自由民主党の国会議員および](../Page/自由民主党国会議員一覧.md "wikilink")[党員](../Page/党員.md "wikilink")、[党友](../Page/党友.md "wikilink")などによる[自由民主党総裁選挙](../Page/自由民主党総裁選挙.md "wikilink")によって選出される。「[総裁](../Page/総裁.md "wikilink")」の役職名は、前身の[立憲政友会](../Page/立憲政友会.md "wikilink")、[立憲民政党](../Page/立憲民政党.md "wikilink")から引き継いだもの。

## 概要

自由民主党は、[1955年](../Page/1955年.md "wikilink")11月の結党以降ほぼ全期間にわたって[衆議院](https://ja.wikipedia.org/wiki/衆議院 "wikilink")で比較第1党を保っているため、歴代の総裁のほとんどが在任中に[内閣総理大臣](https://ja.wikipedia.org/wiki/内閣総理大臣 "wikilink")を兼任しており、両者を合わせて**総理総裁**と呼ばれることもある。[マスメディア](../Page/マスメディア.md "wikilink")での報道においてもほとんどは「首相」の肩書で紹介されており、「総裁」として報道されるのは、党首の資格で出演する国政選挙の関連番組や党首候補者として取り上げられる総裁選関連のニュースなどに限られる。過去に何度か、「総理・総裁分離論」（総裁と[総理をそれぞれ別の人物が務める](https://ja.wikipedia.org/wiki/内閣総理大臣 "wikilink")）が案として出たことがあるが、実現した例は一度もない（[後述](https://ja.wikipedia.org/wiki/#総総分離論 "wikilink")）。

なお、総裁が首相を兼務する場合、党務は[幹事長が主に担当する](../Page/自由民主党幹事長.md "wikilink")。

### 選出

総裁は、自由民主党則6条1項が引用する総裁公選規程第1条により「党所属国会議員、党員、[自由国民会議会員および](../Page/自由社会を守る国民会議.md "wikilink")[国民政治協会](../Page/国民政治協会.md "wikilink")会員」による公選が原則だが、党則6条2項により、総裁が任期中に欠けた場合で緊急の事態により正規の総裁選挙が行えない場合には、「党大会に代わる両院議員総会」において、所属する全ての現職国会議員及び都道府県連合の代表者による投票によって新総裁を選出する場合もある。また、党の有識者や幹部等による話し合い調整に基づいて新総裁候補者を1本化し、[両院議員総会](https://ja.wikipedia.org/wiki/両院議員総会 "wikilink")での承認を受けて新総裁を決定する場合もある。なお、自由民主党総裁に立候補できる者は、総裁公選規程9条により、党所属国会議員に限定される。

### 任期規定

総裁任期は党則80条1項により、現在3年である。総裁任期はたびたび変更されている。

また、1974年以降には前任者の途中退任による残任期間を除く任期を連続2期までとする規定が追加され、2017年には3期までと変更された。

| 期間            | 任期               | 再選規定 |
| ------------- | ---------------- | ---- |
| 1955年 - 1972年 | 2年               | 制限なし |
| 1972年 - 1974年 | 3年               |      |
| 1974年 - 1978年 | 連続2期まで（再々任を認めない） |      |
| 1978年 - 2003年 | 2年               |      |
| 2003年 - 2017年 | 3年               |      |
| 2017年-        | 連続3期まで           |      |

1974年に連続3選を禁止する規定が導入されて以降、規定の上限まで務めて任期満了に伴い退任した総裁の例は、2例ある。

  - [中曽根康弘](https://ja.wikipedia.org/wiki/中曽根康弘 "wikilink") - 1986年までに2期4年の任期を全うするも、同年の衆参同日選挙での圧勝などの理由で、特例により一年延長。計5年の任期を務めた。
  - [小泉純一郎](https://ja.wikipedia.org/wiki/小泉純一郎 "wikilink") - 2003年までに1期目（2年）を、2006年までに2期目（3年）を満了。中曽根と同じく特例による任期延長が検討されたが、小泉本人がこれを固辞。2期5年（残任期間を含めると3期5年5ヶ月）の任期を全うした。

総裁を一旦退任した人物の再任を制限する規定はなく、[安倍晋三](https://ja.wikipedia.org/wiki/安倍晋三 "wikilink")が唯一の再任例となっている。

## その他

### 総総分離論

総裁以外の自民党国会議員が内閣総理大臣に就任することについて、自民党議員から首相を選出する場合、過去の特殊な例外を除き総裁を首相に選出しているが、権力の分散、責任の分担、党内融和の観点から、しばしば総理と総裁の分離案が浮上している。しかし、過去に何度か分離案が浮上しても調整段階で失敗している。著名な例としては[大福戦争](https://ja.wikipedia.org/wiki/大福戦争 "wikilink")時の「[大平総理](../Page/大平正芳.md "wikilink")・[福田総裁](../Page/福田赳夫.md "wikilink")」案があるが、これは大平が「福田総裁代行」とすることを要求し、いずれも成案とならず[ハプニング解散](../Page/ハプニング解散.md "wikilink")に至る。[鈴木善幸](../Page/鈴木善幸.md "wikilink")退陣後の党内調整では「[中曽根総理](https://ja.wikipedia.org/wiki/中曽根康弘 "wikilink")・福田総裁」案がまとまりかけたが、これも中曽根と[田中角栄](../Page/田中角栄.md "wikilink")の拒否により流れ総裁公選となる。

総裁以外の自民党議員が首相に選出された例は、1957年[2月25日](../Page/2月25日.md "wikilink")の石橋湛山の総裁時代における岸信介の首相選出や、1964年[11月9日](../Page/11月9日.md "wikilink")の池田勇人の総裁時代における佐藤栄作の首相選出がある。また、[麻生太郎](https://ja.wikipedia.org/wiki/麻生太郎 "wikilink")の総裁時代である[2009年](../Page/2009年.md "wikilink")[9月16日](../Page/9月16日.md "wikilink")には、首相に選出はされなかったものの両院議員総会長の[若林正俊](../Page/若林正俊.md "wikilink")が自民党の首相候補となった。

しかし、これらは総理総裁であった石橋、池田が病気のために首相はおろか自民党総裁など政治家としての公務が難しい状況であったこと、3、4ヶ月前の総裁選で岸、佐藤が現総裁に次ぐ2位であったこと、岸、佐藤両者とも総裁から後継総裁に指名され次期総裁就任が目されていたこと、岸、佐藤両者とも首相就任から1ヶ月して自民党総裁に正式に就任していることから、また麻生の総裁時代に若林が首相候補となったのは、衆議院総選挙大敗によって自民党議員が首相になれないことが確実視されていたこと、衆議院総選挙大敗の責任を取る形で麻生執行部の退任がすでに決定している中で後継総裁はまだ選出されていなかったという事情によるもので、総総分離体制が持続されていた、もしくはそれを視野に入れた選出とはみなされていない。

なお、総裁を退くと首相も辞任することと、首相を辞任したら総裁も退くことが慣例化しているため、自民党において総総分離体制が持続されたことはない。

### 総理総裁の条件

党則上、国会議員の党員全員に総裁の資格はあるが、現実問題として実績がない者が総裁選挙に出馬したところで選出される役職ではなく、田中角栄は総理総裁の条件として、「[党三役のうち](../Page/自由民主党執行部.md "wikilink")[幹事長を含む二役](../Page/自由民主党幹事長.md "wikilink")（つまり、他に[総務会長か](../Page/自由民主党総務会.md "wikilink")[政調会長のいずれか](https://ja.wikipedia.org/wiki/自由民主党政務調査会 "wikilink")）、内閣で[外務](../Page/外務大臣_\(日本\).md "wikilink")・[大蔵](https://ja.wikipedia.org/wiki/財務大臣#日本 "wikilink")・[通産のうち二](../Page/経済産業大臣.md "wikilink")[閣僚](../Page/国務大臣.md "wikilink")」（の経験者であること）を挙げていた。それらの要職を歴任しさえすれば必ず総理総裁になれるというわけではないが、総理総裁候補の実力者なら経験しているのは当然と考えていたと思われる。[野田聖子](../Page/野田聖子.md "wikilink")も、[2009年](../Page/2009年.md "wikilink")に自民党が下野した際に総裁選に出ようとして、後見役の[古賀誠](https://ja.wikipedia.org/wiki/古賀誠 "wikilink")から同様に「資格は小選挙区出で三役経験者だ」と言われ止められたという\[1\]（この時は[谷垣禎一](../Page/谷垣禎一.md "wikilink")が候補に名乗りを上げ当選した）。[三角大福](../Page/三角大福.md "wikilink")（三木武夫、田中、大平正芳、福田赳夫）の時代はこの条件を一応充足していた。

しかし、鈴木善幸以降は条件に該当しない総理総裁が多く、田中が挙げた条件全てを満たした自民党の議員は、[安倍晋太郎](../Page/安倍晋太郎.md "wikilink")、[三塚博](../Page/三塚博.md "wikilink")、[桜内義雄](https://ja.wikipedia.org/wiki/櫻内義雄 "wikilink")、橋本龍太郎、[麻生太郎](https://ja.wikipedia.org/wiki/麻生太郎 "wikilink")の5人、条件に該当した状態で総理総裁に就任したのは橋本だけである。麻生については、総理総裁を退任した後に財務大臣に就任したことによって、条件全てを満たした。逆に海部俊樹、小泉純一郎、福田康夫は条件として挙げられた党の役職を、また[安倍晋三](https://ja.wikipedia.org/wiki/安倍晋三 "wikilink")は省庁大臣職を一つも経験しない\[2\]状態で総理総裁となった。

自民党歴代総裁の在職日数上位5名（池田勇人、佐藤栄作、中曽根康弘、小泉純一郎、安倍晋三）の内、田中の条件に該当している者はいない（ただし自民党の前身の自由党時代を含めると池田勇人と佐藤栄作は充足している）。田中は[ロッキード事件](../Page/ロッキード事件.md "wikilink")で無罪を勝ち取った後の復帰を念頭に、三角大福世代からの世代交代の動きに釘を刺すことに執心しており、その一環としての条件主張でもあった。田中自身より年上の鈴木善幸（条件ポストの経験は総務会長のみ\[3\]）や、田中と同年齢の中曽根（外相および蔵相を未経験\[4\]）については、総裁に擁立する立場に回っている。

条件からは除外されている[内閣官房長官](../Page/内閣官房長官.md "wikilink")は、当時から内閣の要職であったが、田中が[田中派内の世代交代を抑えようとする上で念頭にあった](../Page/木曜クラブ.md "wikilink")[竹下登](../Page/竹下登.md "wikilink")が官房長官経験者であるのに対して、田中自身は未経験であった。2000年以降は、特に[副総理](../Page/副総理.md "wikilink")扱いの閣僚がなければ、官房長官を[内閣総理大臣臨時代理](../Page/内閣総理大臣臨時代理.md "wikilink")第1位事前指定者とする慣行がある。また、[中央省庁再編](../Page/中央省庁再編.md "wikilink")で新設された、官邸機能の一部も引き受ける巨大省庁の[総務省](../Page/総務省.md "wikilink")を管轄する[総務大臣](../Page/総務大臣.md "wikilink")は、田中時代には存在しなかったポストである。

ちなみに現職の自民党総裁で[重複立候補](https://ja.wikipedia.org/wiki/重複立候補 "wikilink")した者は[2000年の衆院選の](https://ja.wikipedia.org/wiki/第42回衆議院議員総選挙 "wikilink")[石川2区で圧勝した森だけである](../Page/石川県第2区.md "wikilink")（現職の総理大臣が重複立候補した例は森と[民主党政権時代の](../Page/民主党_\(日本_1998-2016\).md "wikilink")[野田佳彦](../Page/野田佳彦.md "wikilink")のみ。森は小選挙区比例代表並立制導入以降、自身が比例候補定年73歳未満だった1996・2000・2003・2005・2009年と5回連続で重複立候補し全て小選挙区勝利している）。小泉の総裁時代に行われた[2005年の衆院選では](../Page/第44回衆議院議員総選挙.md "wikilink")、自民党神奈川県連会長の[河野太郎](../Page/河野太郎.md "wikilink")が小泉に[比例南関東ブロック](https://ja.wikipedia.org/wiki/比例南関東ブロック "wikilink")と[神奈川第11区へ重複立候補することを要請していた](../Page/神奈川県第11区.md "wikilink")。理由は「比例での自民票上積み」とされる。しかし、重複すると、小泉の顔を使った自民党のポスターが同ブロックの神奈川、千葉、山梨3県で貼れなくなるという公職選挙法の問題があったため、重複立候補は取り止めとなった。

### 肖像画

自民党本部の8階ホールには、歴代総裁肖像画が展示されている。ただし、[1994年](../Page/1994年.md "wikilink")6月に自民党執行部が村山を首班指名し、海部がそれに反対する形で自らの首班指名に意欲を示して離党した時には、総裁としての海部の肖像画が外された。その後、[2003年](../Page/2003年.md "wikilink")11月に海部が自民党に復党した際、海部の肖像画が再び展示されるようになった。

この肖像画は自分で好きな画家を指名することが可能で、1枚数百万円ともいわれる。8階ホールでは26人目まで飾る部分が確保されている（前職の谷垣で24人目）。
