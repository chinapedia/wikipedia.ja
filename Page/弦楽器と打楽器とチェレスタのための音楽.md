## 概説

[1926年](../Page/1926年.md "wikilink")にバーゼル室内管弦楽団を結成して活動していた指揮者パウル・ザッハーは、[1934年](../Page/1934年.md "wikilink")に世界的製薬会社[エフ・ホフマン・ラ・ロシュ](../Page/エフ・ホフマン・ラ・ロシュ.md "wikilink")のオーナー未亡人と結婚したことにより巨万の富を相続した。彼はそれ以降製薬会社の業績立て直しにも貢献しつつ、音楽家としても同時代の作曲家たちへのおよそ200作品もの委嘱、[バーゼル・スコラ・カントルム](https://ja.wikipedia.org/wiki/バーゼル・スコラ・カントルム "wikilink")やパウル・ザッハー財団の創設など、その財力を音楽界のために惜しみなく費やした人物として知られている。バルトークとも[1929年](../Page/1929年.md "wikilink")4月に[ジュネーブ](https://ja.wikipedia.org/wiki/ジュネーブ "wikilink")で行われた[国際現代音楽協会](../Page/国際現代音楽協会.md "wikilink")で出会って以来\[2\]の交友があった。

1936年6月23日、ザッハーは当時夏季休暇でスイスのブラウンヴァルトに滞在していたバルトークに、翌年1月21日に行う予定のバーゼル室内管弦楽団・創立10周年演奏会のために新作\[3\]を書いてもらえないだろうかと手紙を送った。バルトークはすぐさま承諾する返事を送り、しかも『弦楽器と打楽器のための（つまり弦楽器に加えて[ピアノ](../Page/ピアノ.md "wikilink")・[チェレスタ](../Page/チェレスタ.md "wikilink")・[ハープ](../Page/ハープ.md "wikilink")・[シロフォン](../Page/シロフォン.md "wikilink")・その他の打楽器のための）作品を書く』と、具体的な楽器構成についても言及している。

委嘱から完成まで実質2か月弱とかなりの速筆で書かれた。当時演奏家としての活動や民族音楽研究\[4\]などで多忙だったバルトークが何故この規模の曲をそれだけ順調に完成させられたのかについては、草稿などの研究やザッハーへの返事にほぼ完成稿と同様の楽器編成で語っている点などから、既に書き始められていた弦楽合奏のアイディアに、ザッハーの依頼を受けて打楽器や鍵盤楽器を加えて書いたことで可能になったものだと考えられている。\[5\]

当初『*Musik für Saiteninstrumente*（弦楽器のための音楽）』と題されたこの作品\[6\]の世界初演は予定通り、[1937年](../Page/1937年.md "wikilink")[1月21日](../Page/1月21日.md "wikilink")にザッハーが指揮するバーゼル室内管弦楽団により[バーゼル](../Page/バーゼル.md "wikilink")で行われた。ザッハーは前年11月末からリハーサルを開始するという熱の入れようで、直前リハーサルから立ち会ったバルトークは予想以上に彼らの準備が進んでいたことに喜び、夫人へ「指揮者もオーケストラも、僕との練習に大変な情熱と献身を示してくれていて、皆この作品に熱狂している（僕もだ！）」と手紙に書くほどだった。そして初演は成功し、第4楽章がアンコールされ、批評家たちもバルトークの代表作の一つとして高く評価した\[7\]。なお、日本初演は[1939年](../Page/1939年.md "wikilink")[5月10日](../Page/5月10日.md "wikilink")に[ヨーゼフ・ローゼンシュトック](../Page/ヨーゼフ・ローゼンシュトック.md "wikilink")指揮の[新交響楽団により行われた](../Page/NHK交響楽団.md "wikilink")。

[総譜](../Page/総譜.md "wikilink")は同年に[オーストリア](../Page/オーストリア.md "wikilink")・[ウィーン](https://ja.wikipedia.org/wiki/ウィーン "wikilink")の[ウニヴェルザール出版社](https://ja.wikipedia.org/wiki/ウニヴェルザール出版社 "wikilink")より出版され、タイトルの上には「この曲をバーゼル室内管弦楽団とパウル・ザッハー氏に捧ぐ」と記されている。

## 特徴

その題名が示すように、[弦楽器](../Page/弦楽器.md "wikilink")（[ヴァイオリン](../Page/ヴァイオリン.md "wikilink")、[ヴィオラ](../Page/ヴィオラ.md "wikilink")、[チェロ](../Page/チェロ.md "wikilink")、[コントラバス](../Page/コントラバス.md "wikilink")、[ハープ](../Page/ハープ.md "wikilink")）、[打楽器](../Page/打楽器.md "wikilink")（[木琴](https://ja.wikipedia.org/wiki/木琴 "wikilink")、[スネア付きドラム](../Page/スネアドラム.md "wikilink")、スネア無しドラム、[シンバル](../Page/シンバル.md "wikilink")、[タムタム](../Page/タムタム.md "wikilink")、[バスドラム](../Page/バスドラム.md "wikilink")、[ティンパニ](../Page/ティンパニ.md "wikilink")、[チェレスタ](../Page/チェレスタ.md "wikilink")）、そして[ピアノ](../Page/ピアノ.md "wikilink")を用いる。

几帳面なバルトークの性格を反映して、[総譜](../Page/総譜.md "wikilink")上には楽器の配置位置が明確に指定されている。それによれば弦楽器群は弦楽5部を2つに分けて[指揮者](../Page/指揮者.md "wikilink")の左右に対向配置し、中央にその他の楽器が配置されるように指示されている。

チェレスタが楽器編成に入っている（しかも、使用頻度の高いピアノを差し置いて、他の打楽器とは別に題名に登場している）理由としては、バルトークが民族音楽の研究に没頭していたときに接した[インドネシア](../Page/インドネシア.md "wikilink")の[バリ](../Page/バリ島.md "wikilink")・[ガムラン](../Page/ガムラン.md "wikilink")によるものだという説が最有力である。確かにこの作品で頻用されるチェレスタのグリッサンドは、インドネシアの雰囲気をかもし出してはいる。ただし、この時代にはミュステル社の鍵盤アクションに不備の多いチェレスタしか存在しておらず、初演当時のソロパートの演奏はかなり困難であったことが想像される。

## 楽曲構成

曲は緩―急―緩―急の4楽章形式。第1楽章冒頭の主題が変形され、[循環主題](https://ja.wikipedia.org/wiki/循環主題 "wikilink")として各楽章に用いられる。なお各楽章の調、構造についてはバルトーク本人の指示に従って、ウニフェルザル社が総譜に掲載していた「楽曲分析」に従った。

### 第1楽章 Andante tranquillo

イ調。[変拍子](https://ja.wikipedia.org/wiki/変拍子 "wikilink")の主題に基づく変則的な[フーガ](../Page/フーガ.md "wikilink")。静穏な中に高い緊張を感じさせる。

[弱音器](../Page/弱音器.md "wikilink")つきのヴィオラの半音階的な主題から始まる。（譜例1）

譜例1 <score> { \\key a \\minor \\tempo "Andante tranquillo" 8=116-112 \\time 8/8 \\relative c' {\\clef alto \\partial 8 a8\\pp( bes-- cis4 c b8) r a( \\time 12/8 bes-- cis d es c4 b bes8) r cis( e \\time 8/8 dis-- d c cis b) r c( es \\time 7/8 d4-- cis8 b c-- bes4)} } </score> 続いて5、13、27小節目に完全5度ずつ上で主題が登場する。また、8、16小節目に完全5度下（変則的なのはここで、通常のフーガならまず主調の完全4度下である）に主題が登場し、弦楽器群は次第に音域を扇のように広げていく。34小節目にティンパニが登場。55から56小節目に基音から一番遠い関係（[増四度](https://ja.wikipedia.org/wiki/増四度 "wikilink")）になる[変ホ](https://ja.wikipedia.org/wiki/変ホ "wikilink")音のクライマクスを迎えた後は主題の転回形を基にだんだん鎮まって行き、88小節で開始と同じ[イ音で静かに閉じる](https://ja.wikipedia.org/wiki/イ_\(音名\) "wikilink")。以上に登場する8、13、21、34、55といった数字は[フィボナッチ数列](https://ja.wikipedia.org/wiki/フィボナッチ数列 "wikilink")に現れる\[8\]\[9\]。

### 第2楽章 Allegro

ハ（長）調。2つの主題を持つ[ソナタ形式](../Page/ソナタ形式.md "wikilink")。一楽章と雰囲気は対照的で明るく動きのある楽章。特に弦楽器の対向配置を生かしたステレオ効果の掛け合いが特徴。ときにピアノや弦楽器も打楽器的（[バルトーク・ピッツィカート](https://ja.wikipedia.org/wiki/バルトーク・ピッツィカート "wikilink")）に用いられる。

まず第1主題の要素から導き出されているものの、明るい印象を与える第1主題（譜例2）がハ調で提示される。

譜例2 <score> { \\key c \\major \\tempo "Allegro" 4=138-144 \\time 2/4 \\relative c' { \\partial 8 a8-. \\f c8.( des16 c d e f) fis8-. es-. des4 c8} } </score>

推移主題を経て第2主題（譜例3）はト調で提示される。

譜例3 <score> { \\key c \\major \\time 2/4 \\relative c'' { \\partial 8 fis8-. \\p g,4._\\markup { \\italic leggero } bes8-. as4. cis8-. g( f-.) g-. bes (as-.) e-. r d (g-.) c-. r cis (as-.) fis-. r es (e-.) a-. b-. fis (c'-.) e-. r es} } </score> 展開部では第1主題の要素を展開した後に第4楽章の第1主題を示唆する主題が挟まれ、第1主題を変奏した主題によるフガートを経由して再現部に入る。再現部は全体的に圧縮されているが、第2主題がハ調になるなど古典的ソナタ形式の約束を踏まえている。

### 第3楽章 Adagio

嬰へ調。作曲者の静穏な楽章の一つの典型である「夜の歌」の好例。

A（譜例4）-B（譜例5）-C-B-Aの5部分に分かれたアーチ形式（作曲者自身は「ブリッジ構造」と呼んだ）。各部の経過に第1楽章の主題が断片的に配置され、効果的に用いられる。

譜例4 <score> { \\key a \\minor \\tempo "Adagio molto" 4=40 \\time 4/4 \\relative c' {\\clef alto e32-\>\\p cis8..\~cis8. r16 dis32-\> eis8..\~eis8. r16 \\acciaccatura gis \\tuplet 3/2 {e32-\> cis dis\~} dis8.\~dis8. r16 ais64-\> (b bis d) cis4. r16 \\acciaccatura e \\tuplet 3/2 {c32-\> b a} ais4. r16 e64-\> (eis fis gis) fisis4.\~ fisis32 eis64(dis) d32-\> e4...} } </score>

譜例5 <score> { \\tempo "Più andante" 4=56 \\time 4/4 \\relative c'' {r2 r8 e\\mf ( a_\\markup { \\italic espr. } b ais gis fisis4 fis8 a4 eis8 gis4 ais) b8(cis cisis bis ais gis fisis4 a8 fis4 f8 e4 cis4)} } </score>

特にAの部分のティンパニの[グリッサンド](../Page/グリッサンド.md "wikilink")や木琴の[拍子木](../Page/拍子木.md "wikilink")のような即興的な音形が印象的で、全体的にはいささか不気味な雰囲気も感じさせる。

### 第4楽章 Allegro molto

イ（長）調。舞曲風アレグロ。バルトーク本人は形式をA（譜例6）+B（譜例7）+A,C（譜例8）+D（譜例9）+E+D+F,G,Aと書いているが、下記の譜例でも明快なようにDはAの変奏であるなど、[ロンド形式](../Page/ロンド形式.md "wikilink")の要素が強い。

譜例6（主題A） <score> { \\tempo "Allegro molto" 2=130 \\time 2/2 \\relative c*' {r8 a\\f gis4 fis8 e4(dis8) cis b cis4 dis8 e4(fis8) gis a b4 a8 gis4(fis8) e dis cis4 b8 a4(b8) a4} } </score> 譜例7（主題B） <score> { \\tempo "Allegro molto" 2=130 \\time 2/2 \\relative c' {r4 cis-\!\\p cis-\!-\>_\\markup {\\italic marcato} cis-\! cis-\! cis-\! cis-\!-\> cis-\! r bis-\! b-\! cis-\! b ais-\! a-\!-\> fis-\!} } </score> 譜例8（主題C） <score> { \\tempo "Un poco meno mosso" 2=120 \\time 2/2 \\relative c*' {r4 a8\\ff (g) a (g) a (g) a (g) a (g) f(e f g) a(b c4)d,8(e f g a4)d c8(b a g) a4 b8(c) g(f e) c d(e) f(g)} } </score> 譜例9（主題D） <score> { \\tempo "Ancora meno mosso" 2=112 \\time 2/2 \\relative c''' {fis8\\mf e4 cis8 b cis fis,4 gis4 b cis8 dis8 e4 fis8 e4 cis8 b cis gis4 e a fis8 e fis4 } } </score> 片方の弦楽器グループが[イ長調](../Page/イ長調.md "wikilink")の和音をピチカートでかき鳴らすのに乗って、もう一群が下降音型で提示するメイン主題A\[10\]が短くなったり、上下転回したりと何度も変奏されながら\[11\]出現する。後半はバルトークがGと書いている第1楽章のテーマの音階が全音階に拡張された主題が登場し、チェロの[カデンツァ](../Page/カデンツァ.md "wikilink")風ソロを経て、テンポがめまぐるしく変わる中、熱狂的に閉じる。

以上のように全曲を通じて精密に技巧を凝らして作曲されているが、聴衆にはまったくそれを意識させない。バルトークのそれまでの[民族音楽](../Page/民族音楽.md "wikilink")研究の成果が、[バロック音楽](../Page/バロック音楽.md "wikilink")の[コンチェルト・グロッソ](../Page/コンチェルト・グロッソ.md "wikilink")を思わせる古典的な形式の中に昇華したバルトーク円熟期の代表作で、録音も多い。
