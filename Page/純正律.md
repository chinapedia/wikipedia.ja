[Moodswingerscale.svg](https://ja.wikipedia.org/wiki/File:Moodswingerscale.svg "fig:Moodswingerscale.svg") **純正律**（じゅんせいりつ Just intonation)は、[周波数](../Page/周波数.md "wikilink")の比が単純な[整数](../Page/整数.md "wikilink")比である[純正音程](https://ja.wikipedia.org/wiki/純正音程 "wikilink")のみを用いて規定される[音律](https://ja.wikipedia.org/wiki/音律 "wikilink")である。

例えば純正律による[長調](https://ja.wikipedia.org/wiki/長調 "wikilink")の[全音階](../Page/全音階.md "wikilink")は、純正完全5度（3/2）と純正長3度（5/4）を用いて各音が決定される。 すなわち、Cを基準とした場合、Cの3度上がE、5度上がG、次にGの3度上がB、5度上がD、さらにCの5度下がF、Fの3度上がAとなり、これらを1[オクターヴ](../Page/オクターヴ.md "wikilink")内に配列することで[ハ長調](https://ja.wikipedia.org/wiki/ハ長調 "wikilink")の全音階が得られる。

上述の音階を以下に示す。大文字のTは大全音（9/8）、小文字のtは小全音（10/9）、sは半音（16/15）の音程を表す。

| Note  | Name | C    | D     | E   | F    | G   | A     | B    | C |
| ----- | ---- | ---- | ----- | --- | ---- | --- | ----- | ---- | - |
| Ratio | 1/1  | 9/8  | 5/4   | 4/3 | 3/2  | 5/3 | 15/8  | 2/1  |   |
| Cents | 0    | 204  | 386   | 498 | 702  | 884 | 1088  | 1200 |   |
| Step  | Name |      | T     | t   | s    | T   | t     | T    | s |
| Ratio | 9/8  | 10/9 | 16/15 | 9/8 | 10/9 | 9/8 | 16/15 |      |   |
| Cents | 204  | 182  | 112   | 204 | 182  | 204 | 112   |      |   |

純正律の長所は、[倍音](https://ja.wikipedia.org/wiki/倍音 "wikilink")の[うなり](https://ja.wikipedia.org/wiki/うなり "wikilink")を伴わない、単純な整数比による純正な[和音](../Page/和音.md "wikilink")が得られることである。 上記の例であれば、C-E-G、F-A-C、G-B-Dの[三和音](https://ja.wikipedia.org/wiki/三和音 "wikilink")は4:5:6の比となり、三和音として最も単純な比を持つ。

短所は、音の組によっては、純正音程から著しく外れることである。上記の例ではD-Aの音程は純正完全5度（3/2）よりも81/80（[シントニックコンマ](https://ja.wikipedia.org/wiki/シントニックコンマ "wikilink")）狭い40/27となり、この音程を含む和音は非常に響きが悪くなる。そのため純正律では[転調](../Page/転調.md "wikilink")や[移調](../Page/移調.md "wikilink")が困難である。

もう一つの短所は、旋律の演奏に際しては、純正律では大全音（9/8）と小全音（10/9）の2種類の[全音](https://ja.wikipedia.org/wiki/全音 "wikilink")が存在するため、音階が不均等な印象を与え、また演奏が難しいことである。

## 純正律を可能とする楽器

日本の楽器メーカーである[ヤマハ](https://ja.wikipedia.org/wiki/ヤマハ "wikilink")では、学校における[合奏](https://ja.wikipedia.org/wiki/合奏 "wikilink")および[合唱](https://ja.wikipedia.org/wiki/合唱 "wikilink")の指導において、特にロングトーン（持続音）の[ハーモニー](../Page/ハーモニー.md "wikilink")の改善を目的とした、**ハーモニーディレクター**および**ハーモニーキーボード**という名称の[電子キーボード](https://ja.wikipedia.org/wiki/電子キーボード "wikilink")を供給している。これらの製品では主キーの長調と短調を選択して、その三和音を平均律と純正律に近いチューニングに分けて、発音させることができるようになっている。また[ローランド](../Page/ローランド.md "wikilink")でも[シンセサイザー](../Page/シンセサイザー.md "wikilink")の一部製品（XPシリーズ、RSシリーズなど）において、平均律をセント単位で補正することで、純正律に近いチューニングを可能としている。

なおヤマハ、ローランドとも[電子ピアノ](../Page/電子ピアノ.md "wikilink")と[電子オルガン](https://ja.wikipedia.org/wiki/電子オルガン "wikilink")の高級製品においては、純正律（あるいは平均律以外の古典的な音律）の選択が標準的に装備されている。
