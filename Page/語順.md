**語順**（ごじゅん、word order\[1\]\[2\]）とは、[文](../Page/文.md "wikilink")や[句](https://ja.wikipedia.org/wiki/句 "wikilink")のなかで句や[語](https://ja.wikipedia.org/wiki/語 "wikilink")が並ぶ順番のこと\[3\]である。語順は、ただ一つに決まっていることもあれば、複数の語順が可能なこともある。

音声言語を使って会話をするとき、単語は、一つ一つ順番に発音される。音声言語では（[手話](../Page/手話.md "wikilink")と違って）二つの単語を同時に発音することはできない。このため、単語は、文のなかで時間の流れにそって並ぶことになる。これは、[形態素](../Page/形態素.md "wikilink")や[句](https://ja.wikipedia.org/wiki/句 "wikilink")など、語よりも小さい単位や大きい単位にも当てはまる。形態素も句も、一度に一つしか発音できないので、ある順番に並べて発音する必要がある。

## 語順の類型論

言語類型論では複数の語順が可能な場合には、そのうちの一つが**基本語順** (basic word order)\[4\]または**支配的語順** (dominant \[word\] order)\[5\]とされる。

[ラテン語](../Page/ラテン語.md "wikilink")や[フィンランド語](../Page/フィンランド語.md "wikilink")は定型的な語順を持たない。しかし傾向として、前者が SOV 型、後者が SVO 型と見なせる。このように柔軟な語順で使用される言語は、文中における名詞の役割を表示する[格](../Page/格.md "wikilink")を有するのが普通である。同様に語順が比較的自由な日本語では[格助詞](https://ja.wikipedia.org/wiki/格助詞 "wikilink")がこの役割を担っている。また[ハンガリー語](https://ja.wikipedia.org/wiki/ハンガリー語 "wikilink")や[チェコ語](../Page/チェコ語.md "wikilink")のように、語順が[主題](https://ja.wikipedia.org/wiki/主題 "wikilink")・[焦点など](https://ja.wikipedia.org/wiki/焦点_\(言語学\) "wikilink")[情報構造](https://ja.wikipedia.org/wiki/情報構造 "wikilink")を示す手段となる言語もある。

同じくらいよく使われる語順が二つ以上あるために基本語順を一つに決められない言語にも、いくつかのタイプがある。オーストラリアの[ヌングブユ語](https://ja.wikipedia.org/wiki/ヌングブユ語 "wikilink")\[6\]のようにほとんど全ての語順が同じくらいよく使われる言語もあれば、[ドイツ語](../Page/ドイツ語.md "wikilink")や[オランダ語](../Page/オランダ語.md "wikilink")で主節ではSVOが優勢だが従属節ではSOVとなるように、統語的環境によって基本語順が変わる言語もある\[7\]。

### 主語・目的語・動詞の語順

[他動詞](https://ja.wikipedia.org/wiki/他動詞 "wikilink")節に現れる主語\[8\] (S: subject)・目的語 (O: object)・動詞 (V: verb) の3つの要素の語順を考えると3\!＝6通りの可能性がある。他動詞の[平叙文](https://ja.wikipedia.org/wiki/平叙文 "wikilink")で[主語](https://ja.wikipedia.org/wiki/主語 "wikilink")と[目的語](https://ja.wikipedia.org/wiki/目的語 "wikilink")の両方が[代名詞](../Page/代名詞.md "wikilink")でない完全な[名詞句](https://ja.wikipedia.org/wiki/名詞句 "wikilink")の場合では、6通りの語順全てが自然言語の基本語順として確認されているが、目的語が最初に来る[OSV型](https://ja.wikipedia.org/wiki/OSV型 "wikilink")と[OVS型](https://ja.wikipedia.org/wiki/OVS型 "wikilink")は珍しい\[9\]。

Dryer (2011a) は世界1377の言語を調べ、可能な語順が複数ある場合には使用頻度によって基本語順を決めた。この調査によれば、[SOV型](https://ja.wikipedia.org/wiki/SOV型 "wikilink")が一番多く565言語、次いで[SVO型](https://ja.wikipedia.org/wiki/SVO型 "wikilink")が488言語であった。他の4つのタイプはいずれも100言語以下で、[VSO型](https://ja.wikipedia.org/wiki/VSO型 "wikilink")が95言語、[VOS型](https://ja.wikipedia.org/wiki/VOS型 "wikilink")が25、OVS型が11、OSV型が4であった。同じくらいよく使われる語順が二つ以上ある言語は189あり、これらは頻度によって基本語順を決定できないため分類からは除かれている。

| 類型              | 言語の数 |
| --------------- | ---- |
| 主語・目的語・動詞 (SOV) | 565  |
| 主語・動詞・目的語 (SVO) | 488  |
| 動詞・主語・目的語 (VSO) | 95   |
| 動詞・目的語・主語 (VOS) | 25   |
| 目的語・動詞・主語 (OVS) | 11   |
| 目的語・主語・動詞 (OSV) | 4    |
| 基本語順なし          | 189  |
| **計：**          | 1377 |

主語、目的語、動詞の語順の各類型にあてはまる言語の数

### 句レベルの語順

主語・目的語・動詞のような[節](https://ja.wikipedia.org/wiki/節 "wikilink")レベルの要素だけでなく、[句](https://ja.wikipedia.org/wiki/句 "wikilink")のレベルにも基本的な語順がある。

取り上げられることが多いのは以下のような要素の語順である。

  - [接置詞](https://ja.wikipedia.org/wiki/接置詞 "wikilink")（[前置詞](https://ja.wikipedia.org/wiki/前置詞 "wikilink")や[後置詞](https://ja.wikipedia.org/wiki/後置詞 "wikilink")）と、それが取る[名詞句](https://ja.wikipedia.org/wiki/名詞句 "wikilink")
  - [名詞](../Page/名詞.md "wikilink")と、それを修飾する[形容詞](../Page/形容詞.md "wikilink")
  - 名詞と、その所有者を意味する要素（[属格](../Page/属格.md "wikilink")名詞句など）
  - 名詞と、それを修飾する[関係節](https://ja.wikipedia.org/wiki/関係節 "wikilink")

それぞれの例の和訳から分かるように、日本語は、《名詞句 - 後置詞》、《形容詞 - 名詞》、《所有者 - 名詞》、《関係節 - 名詞》の語順となる。

## 句の語順

  - 修飾語 + 被修飾語: 形容詞 + 名詞、副詞 + 動詞など。SOV 型に多い。
  - 被修飾語 + 修飾語: 名詞 + 形容詞、動詞 + 副詞など。VSO 型に多く、SVO 型にも比較的多い。

日本語は総合的に見て常に「修飾語 + 被修飾語」の語順である。例えば、この「**日本語は総合的に見て常に「修飾語 + 被修飾語」の語順である**」と言う文を分析すると、

  - 「**である**」 - 何が「である」なのか？
  - 「**語順**である」 - どのような語順か？
  - 「**「修飾語 + 被修飾語」の**語順である」 - どのような時そのような語順か？
  - 「**常に**「修飾語 + 被修飾語」の語順である」 - どのような見地からか？
  - 「**総合的に見て**常に「修飾語 + 被修飾語」の語順である」 - どの言語がそうなのか？
  - 「**日本語は**総合的に見て常に「修飾語 + 被修飾語」の語順である」

と、常に修飾語（限定語・句）が被修飾語（被限定語・句）に先行する。「金を失くした！」を「失くした、金を！」と[倒置](https://ja.wikipedia.org/wiki/倒置 "wikilink")するのは、話者が昂奮していてしどろもどろになっている時や、強調する効果を狙って意図的に行う時ぐらいしかない。
