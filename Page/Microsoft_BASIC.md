**Microsoft BASIC**（マイクロソフト ベーシック）とは、[マイクロソフト](../Page/マイクロソフト.md "wikilink")が開発・販売する[BASIC](../Page/BASIC.md "wikilink")言語の処理系である。

## 概要

[マイクロソフト](../Page/マイクロソフト.md "wikilink")の基盤を築いた製品である。最初に登場したのは[1975年](../Page/1975年.md "wikilink")、ホビースト向け[マイクロコンピュータ](../Page/マイクロコンピュータ.md "wikilink")・[MITS](../Page/Micro_Instrumentation_and_Telemetry_Systems.md "wikilink") [Altair 8800用の](../Page/Altair_8800.md "wikilink")[Altair BASICであった](../Page/Altair_BASIC.md "wikilink")。

Altair BASIC [インタプリタ](../Page/インタプリタ.md "wikilink")は、マイクロソフト社を設立した[ポール・アレン](../Page/ポール・アレン.md "wikilink")と[ビル・ゲイツ](../Page/ビル・ゲイツ.md "wikilink")（およびゲイツの同級生の[モンティ・ダビドフ](../Page/モンティ・ダビドフ.md "wikilink")）が開発した。

開発には[ミニコンピュータ](../Page/ミニコンピュータ.md "wikilink") [PDP-10](../Page/PDP-10.md "wikilink") を使用した。[紙テープ](../Page/紙テープ.md "wikilink")の形で販売され、初期バージョンは 4Kバイトのメモリを使用した。拡張した8Kバイトバージョンは、いろいろなプラットフォームに移植された。BASIC-80（8080/85と[Z80](../Page/Z80.md "wikilink")）、BASIC-68 ([MC6800](../Page/MC6800.md "wikilink"))、BASIC-69 ([MC6809](../Page/MC6809.md "wikilink"))、[6502](https://ja.wikipedia.org/wiki/6502 "wikilink")-BASIC、そして 16ビットの BASIC-86 ([8086](../Page/Intel_8086.md "wikilink")/[8088](../Page/Intel_8088.md "wikilink"))である。

Altair BASICでの成功の後、Microsoft BASIC は儲かるソフトウェアライセンスビジネスの基盤となっていく。1970年代から(特に)1980年代にかけて、様々な[ホームコンピュータや](../Page/ホビーパソコン.md "wikilink")[パーソナルコンピュータ](../Page/パーソナルコンピュータ.md "wikilink")に移植され、拡張されていった。オリジナルのAltair BASIC とは対照的に、多くのパソコン用のBASICは[ROM-BASIC](../Page/ROM-BASIC.md "wikilink")の形で[ROMに格納され](https://ja.wikipedia.org/wiki/Read_Only_Memory "wikilink")、マシンの電源を入れると"READY."（または"Ok"）などのプロンプトを表示して直ちにBASICが起動するようになっていた。つまり、Microsoft BASIC とその派生品は初歩的な[オペレーティングシステム](../Page/オペレーティングシステム.md "wikilink") (OS) としての役割を担っており（[スタンドアロンBASIC](../Page/スタンドアロンBASIC.md "wikilink")）、そのような機種は"BASICターンキーモデル"などと呼ばれていた。

## Microsoft発売の主なバージョン

当初DOS向けではインタプリタとコンパイラが別系統の商品として販売された後、[QuickBASIC](../Page/QuickBASIC.md "wikilink")によって統一され、 その後プロフェッショナル向けが分離された後、[Microsoft Visual BasicのStandard](https://ja.wikipedia.org/wiki/Microsoft_Visual_Basic "wikilink")/Professional版の形で再統一されている。 一時期Macintosh向けにもインタプリタとQuickBASICが発売されていた。

### インタプリタ系

  - BASIC-80 : 8ビット（8080/85とZ80）版
  - BASIC-86 : 16ビット(8086/8088)版、DOS向けの最終版は V5.28(1983)。
  - [GW-BASIC](https://ja.wikipedia.org/wiki/:en:GW-BASIC "wikilink") : IBMが[PC DOSにバンドルしたBASICAのMicrosoft版](https://ja.wikipedia.org/wiki/IBM_PC_DOS "wikilink")。IBM PCのROM BASIC無しで動作し、[CGAと](../Page/Color_Graphics_Adapter.md "wikilink")[EGAのグラフィックをサポートする](../Page/Enhanced_Graphics_Adapter.md "wikilink")。最終版は [V3.23](http://www.gw-basic.com/downloads.html)(1988)。[VGAはサポートされていないので](../Page/Video_Graphics_Array.md "wikilink")、[DOS/V](https://ja.wikipedia.org/wiki/DOS/V "wikilink")機ではグラフィックが使えない。
  - [QBasic](https://ja.wikipedia.org/wiki/QBasic "wikilink") : MS-DOS 5以降にバンドルされた。QuickBASICからコンパイル・リンク等の機能を除いた物。

### コンパイラ系

  - Microsoft BASIC Compiler (BASCOM) : CP/M-80やMS-DOS用に販売されており、DOS向けの最終版は5.36(1984)。
  - QuickBASIC (1985) : BASCOMの後継で4.5が最終版、この後Visual BASICに移行する。
  - Microsoft Visual Basic (1991) : 先にWindows 3.0向けが発売され、その後DOS向けの最終版が発売された。

### プロフェッショナル向け

  - Microsoft BASIC Compiler 6.0 (1988) : QuickBASIC 4.0から派生した。MS-DOS,OS/2両対応で後のVisual Basic Professional版の先駆け。
  - Microsoft BASIC PDS 7.x
  - Microsoft Visual Basic with Professional Toolkit
  - Microsoft Visual Basic for DOS Professional Edition : DOS版のプロフェッショナル向けの最終版
