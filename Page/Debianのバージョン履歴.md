[Debian](../Page/Debian.md "wikilink")は、固定されたスケジュールでリリースされていない。近年のリリースは、おおよそ2年ほどをかけて、Debianプロジェクトによって作られている。

Debianディストリビューションのコードネームは、[トイ・ストーリー](../Page/トイ・ストーリー.md "wikilink")の[映画](../Page/映画.md "wikilink")に登場するキャラクターの名前から取られている。Debianの非安定版（unstable）はSidと名付けられており、これは彼の玩具を破壊するキャラクターの名前である\[1\]。

## リリースの歴史

Debian 1.0は、リリースされていない。これは、ベンダーが誤って開発版のリリースをこのバージョンナンバーで出荷してしまったからである。パッケージ管理システムの[dpkg](https://ja.wikipedia.org/wiki/dpkg "wikilink")とそのフロントエンドdeslectは、Debianにおいては以前のリリースで開発、実装済みであった。a.outバイナリから[ELF](https://ja.wikipedia.org/wiki/ELF "wikilink")バイナリへの移行は、計画されていた1.0のリリース以前から既に始められていた。また、唯一サポートされていたアーキテクチャはIntel 80386（[i386](https://ja.wikipedia.org/wiki/i386 "wikilink")）であった\[2\]。

### Debian 1.1 (Buzz)

Debian 1.1（Buzz）は、[1996年](../Page/1996年.md "wikilink")[6月16日](https://ja.wikipedia.org/wiki/6月16日 "wikilink")にリリースされた。このリリースには474個のパッケージが含まれている。Debianは、ELFバイナリフォーマットへの移行を完了させ、[Linuxカーネル](../Page/Linuxカーネル.md "wikilink")2.0を使っていた\[3\]。

### Debian 1.2 (Rex)

Debian 1.2（Rex）は、1996年12月12日にリリースされた。このリリースには848個のパッケージが含まれており、これは120人の開発者によりメンテナンスされていた\[4\]。

### Debian 1.3 (Bo)

Debian 1.3（Bo）は、[1997年](https://ja.wikipedia.org/wiki/1997年 "wikilink")[6月5日](https://ja.wikipedia.org/wiki/6月5日 "wikilink")にリリースされた。このリリースには974個のパッケージが含まれており、これは200人の開発者によりメンテナンスされていた\[5\]。

### Debian 2.0 (Hamm)

Debian 2.0（Hamm）は、[1998年](https://ja.wikipedia.org/wiki/1998年 "wikilink")[7月24日](../Page/7月24日.md "wikilink")にリリースされた。このリリースには1500好パッケージが含まれ、これは400人を越す開発者によってメンテナンスされたいた。libc6への移行が行われ、DebianはMotorola 68000シリーズ（m68k）アーキテクチャへの移植が行われた\[6\]。

### Debian 2.1 (Slink)

Debian 2.1（Slink）は、[1999年](../Page/1999年.md "wikilink")[3月9日](../Page/3月9日.md "wikilink")にリリースされ、これは2250個のパッケージを含んでいた。パッケージ管理のフロントエンドとして[APT](../Page/APT.md "wikilink")が導入され、[Alpha](https://ja.wikipedia.org/wiki/Alpha "wikilink")と[SPARC](../Page/SPARC.md "wikilink")への移植が行われた\[7\]\[8\]。

### Debian 2.2 (Potato)

Debian 2.2（Potato）は、[2000年](../Page/2000年.md "wikilink")8月14日から15日にかけてリリースされ、450以上の開発者によってメンテナンスされる2600のパッケージを含んでいた。新しく含まれたパッケージには、ディスプレイマネージャの[GDM](https://ja.wikipedia.org/wiki/GDM "wikilink")や、ディレクトリサービスの[OpenLDAP](https://ja.wikipedia.org/wiki/OpenLDAP "wikilink")、セキュリティソフトの[OpenSSH](../Page/OpenSSH.md "wikilink")また、メール転送エージェントの[Postfix](../Page/Postfix.md "wikilink")などが含まれる。Debianは、[PowerPC](../Page/PowerPC.md "wikilink")と[ARM](https://ja.wikipedia.org/wiki/ARM "wikilink")アーキテクチャに移植された\[9\]\[10\]\[11\]。

### Debian 3.0 (Woody)

Debian 3.0（Woody）は、[2002年](../Page/2002年.md "wikilink")[7月19日](../Page/7月19日.md "wikilink")にリリースされ、900を超える開発者によってメンテナンスされる8500前後のパッケージを含んでいた。[KDE](../Page/KDE.md "wikilink")が導入され、[IA-64](../Page/IA-64.md "wikilink")、[PA-RISC](../Page/PA-RISC.md "wikilink")、[MIPS](https://ja.wikipedia.org/wiki/MIPS "wikilink")、[s390](https://ja.wikipedia.org/wiki/s390 "wikilink")のアーキテクチャに新たに移植された\[12\]\[13\]\[14\]</ref>。

### Debian 3.1 (Sarge)

Debian 3.1（Sarge）は、[2005年](../Page/2005年.md "wikilink")[6月6日](../Page/6月6日.md "wikilink")にリリースされ、おおよそ15400のパッケージを含んでいた。このリリースでは、debian-installerと[OpenOffice.org](../Page/OpenOffice.org.md "wikilink")が新たに含まれた\[15\]\[16\]。

### Debian 4.0 (Etch)

[Debian-etch-gnome.png](https://ja.wikipedia.org/wiki/File:Debian-etch-gnome.png "fig:Debian-etch-gnome.png")

Debian 4.0（Etch）は、[2007年](../Page/2007年.md "wikilink")[4月8日](../Page/4月8日.md "wikilink")にリリースされ、1030人以上の開発者によってメンテナンスされる、おおよそ18000のパッケージを含んでいた。Debianは、[x86-64](https://ja.wikipedia.org/wiki/x86-64 "wikilink")に移植されたが、m86kアーキテクチャへのサポートは廃止された\[17\]\[18\]。このバージョンでは、[UTF-8](../Page/UTF-8.md "wikilink")と[udev](https://ja.wikipedia.org/wiki/udev "wikilink")によるデバイス管理がデフォルトとなった。

### Debian 5.0 (Lenny)

[DebianLenny.png](https://ja.wikipedia.org/wiki/File:DebianLenny.png "fig:DebianLenny.png")

Debian 5.0（Lenny）は、[2009年](../Page/2009年.md "wikilink")[2月14日](../Page/2月14日.md "wikilink")にリリースされ、23000個以上のパッケージを含んでいた。Debianは、armelアーキテクチャに移植された\[19\]\[20\]\[21\]</ref>。

### Debian 6.0 (Squeeze)

[Debian_6.0.2.1.png](https://ja.wikipedia.org/wiki/File:Debian_6.0.2.1.png "fig:Debian_6.0.2.1.png")

Debian 6.0（Squeeze）は、[2011年](../Page/2011年.md "wikilink")[2月6日](../Page/2月6日.md "wikilink")にリリースされ、29000個以上のパッケージを含んでいた。ウェブブラウザの[Chromium](https://ja.wikipedia.org/wiki/Chromium "wikilink")が新たに含まれ、[kfreebsd](https://ja.wikipedia.org/wiki/Debian_GNU/kFreeBSD "wikilink")（i386とamd64、後にサポートは中止される）への移植が行われた。また、Intel 486、[Alpha](https://ja.wikipedia.org/wiki/Alpha "wikilink")、[PA-RISC](../Page/PA-RISC.md "wikilink")アーキテクチャへのサポートが中止されている\[22\]\[23\]\[24\]\[25\]。

Squeezeは、ノンフリーなファームウェアのコンポーネント（"バイナリーブロブ"として知られる）がポリシーの問題により、mainレポジトリから除かれた最初のリリースである\[26\]\[27\]\[28\]\[29\]。

### Debian 7 (Wheezy)

[MATE_1.6_on-Debian-Wheezy.png](https://ja.wikipedia.org/wiki/File:MATE_1.6_on-Debian-Wheezy.png "fig:MATE_1.6_on-Debian-Wheezy.png")

Debian 7（Wheezy）は、[2013年](../Page/2013年.md "wikilink")[5月4日](https://ja.wikipedia.org/wiki/5月4日 "wikilink")にリリースされ、36000個を超えるパッケージを含んでいた。[UEFI](https://ja.wikipedia.org/wiki/UEFI "wikilink")のサポートが追加され、[armhf](https://ja.wikipedia.org/wiki/armhf "wikilink")、[s390x](https://ja.wikipedia.org/wiki/s390x "wikilink")アーキテクチャのサポートが追加された\[30\]\[31\]\[32\]。

### Debian 8 (Jessie)

[Debian_8.2_GNOME_desktop.png](https://ja.wikipedia.org/wiki/File:Debian_8.2_GNOME_desktop.png "fig:Debian_8.2_GNOME_desktop.png")

Debian 8（Jessie）は、[2015年](../Page/2015年.md "wikilink")4月25日から26日にかけてリリースされ、43000個を超えるパッケージを含んでいた。このリリースでは、[systemd](https://ja.wikipedia.org/wiki/systemd "wikilink")が[init](https://ja.wikipedia.org/wiki/init "wikilink")の代わりにデフォルトでインストールされている。（sysvinitと[upstart](https://ja.wikipedia.org/wiki/upstart "wikilink")パッケージも代替の選択肢として提供されている）。Debianは[ARM64](https://ja.wikipedia.org/wiki/ARM64 "wikilink")と[ppc64le](https://ja.wikipedia.org/wiki/ppc64le "wikilink")アーキテクチャに移植されたが、IA-64、kfreebsd、s390、SPARCのサポートが中止された\[33\]\[34\]\[35\]。

### Debian 9 (Stretch)

[Debian_9_(Stretch)_with_GNOME_Shell_3.22.png](https://ja.wikipedia.org/wiki/File:Debian_9_\(Stretch\)_with_GNOME_Shell_3.22.png "fig:Debian_9_(Stretch)_with_GNOME_Shell_3.22.png")

Debian 9（Stretch）は、[2017年](../Page/2017年.md "wikilink")[6月17日](../Page/6月17日.md "wikilink")にリリースされ、51000個を超えるパッケージを含んでいた\[36\]。Intel 586（[Pentium](https://ja.wikipedia.org/wiki/Pentium "wikilink")）、Intel586/686 hybrid、PowerPCアーキテクチャはStretchではもうサポートされていない\[37\]。

### Debian 10 (Buster)

[Debian_10_Buster_GNOME.png](https://ja.wikipedia.org/wiki/File:Debian_10_Buster_GNOME.png "fig:Debian_10_Buster_GNOME.png") Debian 10（Buster）は、[2019年](../Page/2019年.md "wikilink")[7月6日](../Page/7月6日.md "wikilink")にリリースされ、57703個のパッケージを含んでいた\[38\]。このリリースでは、UEFIセキュアブートのサポートが追加、[AppArmor](https://ja.wikipedia.org/wiki/AppArmor "wikilink")のデフォルトでの有効化、LUKSのデフォルトフォーマットとしてLUKS2の使用、[GNOME](../Page/GNOME.md "wikilink")での標準での[Wayland](https://ja.wikipedia.org/wiki/Wayland "wikilink")の使用などの変更が行われている。

### Debian 11 (Bullseye)

Debian 11（Bullseye）は、次にリリース予定のOSであり、まだリリース日などは決まっていない。
