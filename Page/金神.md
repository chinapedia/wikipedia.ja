[God_of_Konjin.png](https://ja.wikipedia.org/wiki/File:God_of_Konjin.png "fig:God_of_Konjin.png") **金神**（こんじん）とは[方位神](https://ja.wikipedia.org/wiki/方位神 "wikilink")の1つである。

金神の在する[方位](../Page/方位.md "wikilink")に対してはあらゆることが凶とされ、特に土を動かしたり造作・修理・移転・旅行などが忌まれる。この方位を犯すと家族7人に死が及び、家族が7人いない時は隣の家の者まで殺される（これを七殺（ななさつ）という）と言われて恐れられた。

金神の中でも、「うしとらの金神」は「久遠国」という夜叉国の王である巨旦大王の精魂とされる。巨旦大王の[眷属](https://ja.wikipedia.org/wiki/眷属 "wikilink")の精魂も（普通の）金神と呼ばれる凶神となっている。

またその精魂の抜けた屍は[牛頭天王](../Page/牛頭天王.md "wikilink")によって5つに引き裂かれ、[五節句](https://ja.wikipedia.org/wiki/五節句 "wikilink")に合わせて祭った（巨旦調伏の祭礼）。

すなわち、

  - 1月1日----紅白の[鏡餅](../Page/鏡餅.md "wikilink")（巨旦の骨肉）
  - 3月3日----[蓬](https://ja.wikipedia.org/wiki/蓬 "wikilink")の[草餅](../Page/草餅.md "wikilink")（巨旦の皮膚）
  - 5月5日----[菖蒲の](../Page/ショウブ.md "wikilink")[ちまき](../Page/ちまき.md "wikilink")（巨旦の髭と髪）
  - 7月7日----[小麦の](../Page/コムギ.md "wikilink")[素麺](../Page/素麺.md "wikilink")（巨旦の筋）
  - 9月9日----黄菊の酒（巨旦の血）

である。

## 金神の巡行

[24directions.png](https://ja.wikipedia.org/wiki/File:24directions.png "fig:24directions.png") 金神の在する方位は、その年の[十干](../Page/十干.md "wikilink")によって変わる。

  - 甲・己の年：午・未・申・酉
  - 乙・庚の年：辰・巳
  - 丙・辛の年：子・丑・寅・卯・午・未
  - 丁・壬の年：寅・卯・戌・亥
  - 戊・癸の年：子・丑・申・酉

## 金神の遊行・間日

金神は以下のように遊行する。すなわち、これらの期間においては金神が遊行した方角以外は犯しても構わない。これは、金神の在する方位が1年間も塞がったままでは都合が悪いために設けられたものである。

| \!遊行の期間                                                        | 遊行する方位                                      |
| -------------------------------------------------------------- | ------------------------------------------- |
| 通年                                                             | [甲寅](../Page/甲寅.md "wikilink")（51番目）の日から5日間 |
| [丙寅](../Page/丙寅.md "wikilink")（3番目）の日から5日間                     | 酉の方へ                                        |
| [戊寅](../Page/戊寅.md "wikilink")（15番目）の日から5日間                    | 中央（家の中）へ                                    |
| [庚寅](../Page/庚寅.md "wikilink")（27番目）の日から5日間                    | 子の方へ                                        |
| [壬寅](https://ja.wikipedia.org/wiki/壬寅 "wikilink")（39番目）の日から5日間 | 卯の方へ                                        |
| 春（立春から春の土用まで）                                                  | [乙卯](../Page/乙卯.md "wikilink")（52番目）の日から5日間 |
| 夏（立夏から夏の土用まで）                                                  | [丙午](../Page/丙午.md "wikilink")（43番目）の日から5日間 |
| 秋（立秋から秋の土用まで）                                                  | [辛酉](../Page/辛酉.md "wikilink")（58番目）の日から5日間 |
| 冬（立冬から冬の土用まで）                                                  | [壬子](../Page/壬子.md "wikilink")（49番目）の日から5日間 |

（数字は[甲子](../Page/甲子.md "wikilink")から数えた日数）

また春の丑の日、夏の申の日、秋の未の日、冬の酉の日は金神の間日（まび）として金神の方向を犯しても差し支えないとした。

## 大金神・姫金神

後に、金神に似た性格の**大金神**・**姫金神**という方位神が設けられた。それに伴い、元の金神は**巡金神**と呼ばれるようになった。

大金神・姫金神の在する方位は、その年の[十二支](../Page/十二支.md "wikilink")によって変わる。大金神と姫金神は常に正反対の方位に在する。

## 金神と宗教

金神は人々に大変恐れられており、[江戸時代](../Page/江戸時代.md "wikilink")の末には[岡山県](https://ja.wikipedia.org/wiki/岡山県 "wikilink")地方では、「金神封じ」と称して祈祷を行う修験者もいたり、後述するが、「出雲屋敷」と称して[出雲](../Page/出雲.md "wikilink")の神々に家を供え、金神から守ってもらおうとした家も多数あったが、金神を強力な神として迎え信仰した者も多い。

確実に記録\[1\]に残っているのは、現在の[倉敷市](../Page/倉敷市.md "wikilink")[船穂町](../Page/船穂町.md "wikilink")の「堅磐谷の元金神」（現在の[歳徳金神社](https://ja.wikipedia.org/wiki/歳徳金神社 "wikilink")）通称「堅磐谷の婆さん」と呼ばれた[小野うた](https://ja.wikipedia.org/wiki/小野うた "wikilink")である。彼女は8人の子供の内7人まで亡くし、最後に残った娘・小野はるも盲目になり、今の倉敷市[連島町](https://ja.wikipedia.org/wiki/連島町 "wikilink")の文十郎が金神を拝んでいると聞き、参詣し、金神信仰に入った。金神を祈祷して金神に抗議したところ、逆に金神が降臨し、その口を通じて金神が話をしたことから、信仰が始まったとされる。その後、祭典日には100人以上の参拝があり、門前に菓子屋などが建った。このように、「何々金神」と呼ばれる信仰者が多数生まれた。

その中でも、金神の祟りで幾度も転居を余儀なくされた[香取繁右衛門](https://ja.wikipedia.org/wiki/香取繁右衛門 "wikilink")（亀山の金神）と、その実兄であり金神七殺の祟りで家族を何人も失い自らも瀕死の大病になった川手文治郎（後に赤沢文治そして[金光大神](../Page/金光大神.md "wikilink")、大谷の金神）は、金神の啓示と自らの体験、例えば土地の主人である金神の留守を狙うような事をする人間に金神が祟るのは当たり前で、金神が巡って塞がりの土地は逆に金神に許しを得て使わせていただければよく、金神は昔から有る神であり、神として立て仰ぐ人間を待っていたと説き、それぞれ信仰すれば逆にすばらしい加護があったなどという信仰体験から金神信仰を一般化、体系化し、宗教化することにより、繁右衛門は[香取金光教](https://ja.wikipedia.org/wiki/香取金光教 "wikilink")を、そして文治は[金光教](../Page/金光教.md "wikilink")を啓いた。

また[明治](../Page/明治.md "wikilink")初頭、[京都](https://ja.wikipedia.org/wiki/京都 "wikilink")の[出口なお](../Page/出口なお.md "wikilink")は神懸かりの時、最も恐ろしいと言われる艮の金神の突然の降臨とその[自動書記](https://ja.wikipedia.org/wiki/自動書記 "wikilink")等の啓示により、金神こそがこの世の根本の神であり、信仰すれば大きな恩恵が与えられると伝え、最初は金光教の傘下で活動したが、後に[出口王仁三郎](../Page/出口王仁三郎.md "wikilink")と共に[大本](../Page/大本.md "wikilink")を興した。

なお、先述のように、中国地方や全国の出雲大社の分社・講社・分祠等周辺には「出雲屋敷」と言う信仰がある。現在でも出雲大社や出雲大社教の神職等が用いる「出雲屋敷祝詞」には、金神の障りを防ぐ「…**金神**之障里在羅志米須方位之神之祟里無久…」の一文がある\[2\]。
