[thumb](https://ja.wikipedia.org/wiki/ファイル:Print_version_ja.PNG "wikilink") ウィキペディアのページは左サイドバーを除くなど**印刷に適したレイアウトで表示できます**。通常は、ブラウザの「印刷」メニューがどこにあるかだけ分かっていれば十分です\[1\]。

詳しく説明すると、[外装の設定を変更していない標準の状態で](https://ja.wikipedia.org/wiki/Help:外装の詳細設定 "wikilink")、印刷用の[スタイルシート](../Page/Cascading_Style_Sheets.md "wikilink")（CSS）が用意されています。最近のブラウザでは、ページの印刷時には自動的にこのスタイルシートが選択されるからです。

以前の外装に設定しているとか、ブラウザがうまく判別しない場合には、印刷用バージョンのリンクを使うか、URLアドレスを変更します。

## リンク

URLに `&printable=yes` という文字列を追加します。

URLアドレスの変更例： `https://ja.wikipedia.org/w/index.php?title=Help:印刷用バージョン&printable=yes`

この印刷用バージョンへのURLは、以下のように表記できます。

{{fullurl:{{FULLPAGENAME}}|printable=yes}}

モノブックスキンを使っている時には、ページを印刷すると画面に表示していなくても、自動的にこの印刷用バージョンが印刷されます。ただし、ブラウザの印刷設定によっては、ページ番号が追加されるなど若干の違いがあるでしょう。実際に印刷されるであろう内容を正しく確認するには、ブラウザの「印刷プレビュー」機能を利用してください。

## CSS

昔はベースとしてMediaWikiに独立したCSSファイルがありましたが、バージョン1.22以降は独立したCSSファイルではなく、[`mediawiki.legacy.commonPrint`](https://ja.wikipedia.org/w/load.php?debug=false&lang=ja&modules=mediawiki.legacy.commonPrint%2Cshared&only=styles&printable=1&skin=vector)というモジュールで指定されます。このモジュールの内容はMediaWikiの開発者しか変更できません。

ウィキペディア内部でのCSSは[MediaWiki:Print.cssにあります](https://ja.wikipedia.org/wiki/MediaWiki:Print.css "wikilink")。こちらのCSSは上記のベースでの設定を上書きすることができ、[インターフェース管理者のみが編集できます](https://ja.wikipedia.org/wiki/Wikipedia:インターフェース管理者 "wikilink")。

### クラス

[MediaWiki:Print.cssで印刷用バージョンに表示しないクラスを指定できます](https://ja.wikipedia.org/wiki/MediaWiki:Print.css "wikilink")。例えば、などnavboxクラスを使用したものは印刷する意味がないので、表示されないよう指定しています。

### @media print

CSSにおいてメディアタイプを指定することで、スタイルシートを使い分けることができます。モニターでの表示に`@media screen`が使用される一方、印刷用バージョンでは`@media print`が使われます\[2\]。

## カスタムCSS

[カスタムCSSの中で](https://ja.wikipedia.org/wiki/Wikipedia:カスタムCSS "wikilink")`@media print`を指定したルールを追加することで、印刷用バージョンでの表示をカスタマイズすることができます。下記に例をいくつか記します。

  - 出典テンプレのURLを表示する

:

``` CSS
/* 参考としてURLを印刷 */
# content cite a.external.text:after {display: inline;}
```

:\* なお、出典テンプレ以外のURLはデフォルトで表示されます。

  -

  - 脚注を印刷しない

:

``` CSS
/* 脚注を印刷しない */
@media print {
    ol.references {display: none;}
}
```

  -

  - フォントサイズを変更する

:

``` CSS
/* とても小さい文字でインクを節約する */
@media print {
    #footer,
    #content,
    body { font-size: 8pt !important; }
    h1 { font-size: 17pt }
    h2 { font-size: 14pt }
    h3 { font-size: 11pt }
    h4 { font-size: 9pt }
    h5 { font-size: 8pt }
    h6 {
        font-size: 8pt;
        font-weight: normal;
    }
}
```

  -

  - 記事名、「出典: フリー百科事典『ウィキペディア（Wikipedia）』」、記事のURL、カテゴリ、最終更新日時、ライセンス表記を消す

:

``` CSS
/* 本文以外をほぼ全て表示しない */
@media print {
    .firstHeading { margin-top: 0; padding-top: 0; }
    #siteSub { display: none; }
    .printfooter { display: none; }
    #catlinks, .catlinks { display: none; }
    #footer { display: none; }
}
```
