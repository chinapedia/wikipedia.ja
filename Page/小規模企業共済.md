**小規模企業共済**（しょうきぼきぎょうきょうさい）は、[中小企業基盤整備機構](../Page/中小企業基盤整備機構.md "wikilink")の運営する、小規模企業の役員や[個人事業主](../Page/個人事業主.md "wikilink")が[退職](../Page/退職.md "wikilink")等で事業をやめた場合における、生活の安定や事業の再建を図るための資金をあらかじめ準備しておく[共済](../Page/共済.md "wikilink")制度。小規模企業の役員や個人事業主の[退職金](../Page/退職金.md "wikilink")制度と言えるもの。毎月の掛金額は、1,000円から7万円までの範囲内（500円単位）で自由に選択できる。

掛金は税法上、全額を[小規模企業共済等掛金控除](https://ja.wikipedia.org/wiki/小規模企業共済等掛金控除 "wikilink")として課税所得から控除できる。また、1年以内の前納掛金も同様に控除できる。なお、掛金は、役員や個人事業主等である共済契約者自身の収入の中から払い込むため、事業上の損金または必要経費には計上はできない\[1\]。

事業廃業時は、60歳以上など色々と条件があるが、それらを満たすと、一括受取りと分割受取り（10年または15年）の併用にすることが出来、一括受取り分は[退職所得](../Page/退職所得.md "wikilink")扱い、分割受取り分は[公的年金](https://ja.wikipedia.org/wiki/公的年金 "wikilink")等の[雑所得](../Page/雑所得.md "wikilink")扱いとなる。\[2\]

払い込み済み掛け金の7～9割程度の範囲で事業資金等を借り入れる事ができる\[3\]。

預かり資産は国内債券を中心に運用がされている\[4\]。予定利率は少しずつ下げられていて、以下の通り\[5\]。

| 開始日     | 予定利率 |
| ------- | ---- |
| 1965年4月 | 6.6% |
| 1996年4月 | 4.0% |
| 2000年4月 | 2.5% |
| 2004年4月 | 1.0% |

予定利率

自分で運用方針を決めたい場合は、[確定拠出年金](../Page/確定拠出年金.md "wikilink")がある。同じように小規模企業共済等掛金控除として課税控除され、退職所得および公的年金等の雑所得として受け取れる。
