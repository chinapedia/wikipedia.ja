[thumb](https://ja.wikipedia.org/wiki/画像:爆風圧力の時間特性.JPG "wikilink") **爆風**（ばくふう、Blast Wave）とは、[爆発](../Page/爆発.md "wikilink")に伴い空気中を伝播する圧縮波と、圧縮波の背後に出来る波形構造を含む[負圧のこと](../Page/真空.md "wikilink")。

爆風には三つの特徴がある。

  - 伝播速度が[音速](../Page/音速.md "wikilink")以上である
  - 波面が不連続で、波面の前後で[密度](../Page/密度.md "wikilink")と[圧力](../Page/圧力.md "wikilink")の急激な変化がある。
  - 爆風の後には[負圧](https://ja.wikipedia.org/wiki/負圧 "wikilink")を伴う。

伝播速度が[音速](../Page/音速.md "wikilink")以下にまで減衰したものは爆音になる。 一般に、爆風による過圧 (Over pressure) は[爆薬](https://ja.wikipedia.org/wiki/爆薬 "wikilink")量の3乗根に比例して大きくなり、爆源からの距離に反比例して小さくなる。

爆風による圧力は[ブラストメーター](https://ja.wikipedia.org/wiki/ブラストメーター "wikilink")という測定装置で測定することが出来る。

## 空中爆発

[thumb](https://ja.wikipedia.org/wiki/画像:Shoke_wabe_hige.JPG "wikilink") 爆薬が空中で爆発した場合には爆風の入射波が地面に当たって地上反射波となって進む。入射波と地上反射波が合わさる融合波面（マッハステム）\[1\]では圧力が2倍近くにまで高まる（実際には地面が100%全てを反射しないので2倍にはならない、反射しない圧力は地面に穴を穿つのに消費されたりする）。

[原爆が高度](../Page/原子爆弾.md "wikilink")580mで爆発するように設定されていた理由は融合波面を形成することによって、爆風の効果を高めるためである。地表の構造物を吹き飛ばす[デイジーカッター](../Page/デイジーカッター.md "wikilink")と呼ばれる大型爆弾が地上1mで起爆するように作られているのも同じ理由である。

逆に爆薬が地面に接触した状態で爆発すると地面に触れている方向の力は爆風にならずに地面への振動としてエネルギーが消費され地面に穴を穿ってしまうだけになる。

## 爆風による構造物の損傷

| 圧力                                                                                | 被害状況                                                                          |
| --------------------------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| 0.06～0.08kgf/[cm<sup>2</sup>](https://ja.wikipedia.org/wiki/平方センチメートル "wikilink") | [窓ガラス](../Page/窓ガラス.md "wikilink")が割れる。                                       |
| 0.1～0.35kgf/cm<sup>2</sup>                                                        | 窓枠が破損する。                                                                      |
| 0.4～0.5kgf/cm<sup>2</sup>                                                         | [瓦](../Page/瓦.md "wikilink")が落ちたり、[トタン](../Page/トタン.md "wikilink")屋根がはがれたりする。 |
| 0.6～0.7kgf/cm<sup>2</sup>                                                         | 柱が歪んだり、折れたりする。                                                                |
| 1.4～6.0kgf/cm<sup>2</sup>                                                         | 家屋倒壊                                                                          |
| 10kgf/cm<sup>2</sup>                                                              | 強固な家屋や[鉄筋コンクリート](../Page/鉄筋コンクリート.md "wikilink")などの構造物が損壊する。                  |
| 20kgf/cm<sup>2</sup>                                                              | 鉄筋コンクリートや[岩](https://ja.wikipedia.org/wiki/岩 "wikilink")などが割れる。               |
| 30kgf/cm<sup>2</sup>                                                              | 重構造物が倒壊する、特殊な対爆構造物以外では耐えられない。                                                 |

## 爆風による致死率

  - なお、この致死率は負傷後に適切な治療を受けられると言う前提での値である。
  - 致死率は[即死](../Page/即死.md "wikilink")または48時間以内に死亡する確率である。
  - 動物実験と事故事例からの統計に基づいて推測された値であり、状況次第で結果は異なる。

| 致死率               | 圧力                    |
| ----------------- | --------------------- |
| 1%以下（死亡することは稀）    | 1kgf/cm<sup>2</sup>   |
| 数%                | 2～3kgf/cm<sup>2</sup> |
| 10%（生存者も重傷か軽傷を負う） | 4kgf/cm<sup>2</sup>   |
| 50%（生存者も重傷を負う）    | 6kgf/cm<sup>2</sup>   |
| 80%（生存者も重傷を負う）    | 12kgf/cm<sup>2</sup>  |
| ほぼ100%（まず助からない）   | 32kgf/cm<sup>2</sup>  |

## 爆風による人体の損傷

人体が爆風に暴露した場合、最初に負傷するのが[耳](../Page/耳.md "wikilink")と[眼である](../Page/目.md "wikilink")。 次に損傷を受けるのは[表皮](../Page/表皮.md "wikilink")と[肺](../Page/肺.md "wikilink")である。表皮は厚手の服などで守られていれば軽症で済むが、[呼吸器](../Page/呼吸器.md "wikilink")から出血すると大事に到る場合がある。 それ以上になると、[臓器](https://ja.wikipedia.org/wiki/臓器 "wikilink")破裂などの深刻な怪我を負う。 さらに大きな12.0kgf/cm<sup>2</sup>以上の爆風に晒されると、肉が剥がれ、人間としての原形を留めなくなってくる。

この所見は純粋に爆風に晒された場合のみであり、破片などが飛んできて当たった場合の怪我については考慮していない。

これらのデータは[ウサギ](../Page/ウサギ.md "wikilink")と[ブタ](https://ja.wikipedia.org/wiki/ブタ "wikilink")を使用した動物実験と爆発事故の事例を元にした統計的なものであり、状況によって結果は大きく異なる場合がある。
