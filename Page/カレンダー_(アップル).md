カレンダーは、[アップルが開発](../Page/アップル_\(企業\).md "wikilink")・提供している[macOS](https://ja.wikipedia.org/wiki/macOS "wikilink")用のカレンダー兼スケジュール[アプリケーションである](../Page/アプリケーションソフトウェア.md "wikilink")。同社の公式サイトから無償でダウンロードが可能で、[Mac OS X v10.3以降にはバンドルされている](../Page/Mac_OS_X_v10.3.md "wikilink")。以前は、**iCal**（アイキャル）という名称だったが、[OS X Mountain Lionで今の名称に変更された](https://ja.wikipedia.org/wiki/OS_X_Mountain_Lion "wikilink")。

## 歴史

iCalは[2002年](../Page/2002年.md "wikilink")[7月18日](../Page/7月18日.md "wikilink")に発表され、同年9月11日から無償でダウンロードが可能となった。

iCalのデータを同期アプリケーション**[iSync](https://ja.wikipedia.org/wiki/iSync "wikilink")**（アイシンク）を利用して、[iPod](https://ja.wikipedia.org/wiki/iPod "wikilink")、[Bluetooth](../Page/Bluetooth.md "wikilink")搭載の[携帯電話](../Page/携帯電話.md "wikilink")、[Palm OS搭載の](https://ja.wikipedia.org/wiki/Palm_OS "wikilink")[携帯情報端末](../Page/携帯情報端末.md "wikilink") (PDA) に転送することができる。[2002年](../Page/2002年.md "wikilink")[10月4日](https://ja.wikipedia.org/wiki/10月4日 "wikilink")にパブリック[ベータが公開され](../Page/ベータ版.md "wikilink")、[2003年](../Page/2003年.md "wikilink")[1月6日](../Page/1月6日.md "wikilink")からは無償で提供されている。

発表当初は起動が遅い、動作が緩慢であるといった批判が多かったが、その後のアップデートによって徐々に改善され、ハードウェアの高速化に伴って問題にならなくなってきている。

[Mac OS X v10.5に付属するバージョン](../Page/Mac_OS_X_v10.5.md "wikilink")3.0では、[CalDAV](https://ja.wikipedia.org/wiki/CalDAV "wikilink")に対応し、[Mac OS X Serverの](https://ja.wikipedia.org/wiki/macOS_Server "wikilink")[iCal Serverと共に利用することで](https://ja.wikipedia.org/wiki/iCal_Server "wikilink")[グループウェア](../Page/グループウェア.md "wikilink")的な活用が可能となった。

[Mac OS X v10.6付属するバージョン](../Page/Mac_OS_X_v10.6.md "wikilink")4.0では、加えて[Microsoft Exchange 2007をネイティブサポートしている](https://ja.wikipedia.org/wiki/Microsoft_Exchange_Server#Exchange_Server_2007 "wikilink")。[Mac OS X Server v10.6のiCal](https://ja.wikipedia.org/wiki/macOS_Server#Mac_OS_X_Server_v10.6 "wikilink") Server 2と共に利用することで、より柔軟な運用が可能となった。

リリース当初は（リマインダーやチェックリストなどとも呼ばれる）の機能も備えていたが、後に**リマインダー**というアプリケーションに分離された。[OS X v10.11では同様の機能が](https://ja.wikipedia.org/wiki/OS_X_El_Capitan "wikilink")**メモ**というアプリケーションにも組み込まれる予定である。

## 共有

アップルの公式サイトに設けられた「カレンダーライブラリ」から、各種イベント、各国や[三大宗教の](https://ja.wikipedia.org/wiki/世界三大宗教 "wikilink")[祝祭日](../Page/祝祭日.md "wikilink")をダウンロードできる。

iCalで作成したカレンダーは[インターネット](../Page/インターネット.md "wikilink")上に公開することができる。よって他人が作成したデータを利用すること（購読する、という）も可能である。この時のプロトコル ([iCalendar](https://ja.wikipedia.org/wiki/iCalendar "wikilink")) はRFC 5545として公開されている。一般のiCalユーザーが作成したサイトでは各国の祝祭日、スポーツやコンサートの日程などが記されたカレンダーが公開されている。こうしたサイトから購読したカレンダーはiCalが定期的・自動的にアップデートするため、その内容が常に最新の状態となるよう工夫されている。なお、カレンダーを公開するにはアップルが提供するサービスの一つである**[MobileMe](../Page/MobileMe.md "wikilink")**、または[WebDAV](../Page/WebDAV.md "wikilink")機能を有効にした[Webサーバ](../Page/Webサーバ.md "wikilink")が必要となる。購読するための制限は特にない。
