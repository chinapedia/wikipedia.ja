**I-16**（[露](https://ja.wikipedia.org/wiki/ロシア語 "wikilink"):**** <small>イー・シヂスャート</small>、 *Polikarpov I-16*）は、[ソビエト連邦](https://ja.wikipedia.org/wiki/ソビエト連邦 "wikilink")の[パリカールパフ設計局が開発した](https://ja.wikipedia.org/wiki/ポリカルポフ設計局 "wikilink")[単葉戦闘機](../Page/単葉機.md "wikilink")。

[戦間期](../Page/戦間期.md "wikilink")から[第二次世界大戦](https://ja.wikipedia.org/wiki/第二次世界大戦 "wikilink")の初期にかけて[赤色空軍](../Page/赤色空軍.md "wikilink")の主力戦闘機を務めた、世界最初の実用的な引込式主脚を備えた[戦闘機](../Page/戦闘機.md "wikilink")である。

## 概要

木製の太く短い[胴体を持つ本機の外観は](https://ja.wikipedia.org/wiki/機体 "wikilink")、[アメリカ合衆国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")製の高速レース機である[ジービー・レーサーに類似して](https://ja.wikipedia.org/wiki/ジービー_モデルZ "wikilink")、極度に寸詰まりな形態となっている。機首のエンジン[カウリング](https://ja.wikipedia.org/wiki/カウリング "wikilink")前面に[シャッター](../Page/シャッター.md "wikilink")を設け、厳寒時にエンジンが[オーバークール](https://ja.wikipedia.org/wiki/オーバークール "wikilink")となることを防止している\[1\]。[1933年](../Page/1933年.md "wikilink")に試作機のTsKB-12が完成、12月に初飛行した。

時代に先駆けた機構的な特徴は、[パイロットの人力によって](https://ja.wikipedia.org/wiki/パイロット_\(航空\) "wikilink")[ワイヤ](https://ja.wikipedia.org/wiki/ワイヤ "wikilink")駆動で作動する引き込み[脚で](../Page/降着装置.md "wikilink")、[速度](../Page/速度.md "wikilink")は配備当時世界最速であり、実戦でも[九五式戦闘機](../Page/九五式戦闘機.md "wikilink")や[He 51など](../Page/ハインケルHe51.md "wikilink")[複葉戦闘機を性能的に圧倒した](../Page/複葉機.md "wikilink")\[2\]。

[スペイン内戦](../Page/スペイン内戦.md "wikilink")、[ノモンハン事件](../Page/ノモンハン事件.md "wikilink")、[独ソ戦](../Page/独ソ戦.md "wikilink")の初期、[ソ連・フィンランド戦争](https://ja.wikipedia.org/wiki/ソ連・フィンランド戦争 "wikilink")に使用されたが、この時期の[航空機](../Page/航空機.md "wikilink")の進歩は目覚しく、いずれの[戦闘](../Page/戦闘.md "wikilink")でも[敵](https://ja.wikipedia.org/wiki/敵 "wikilink")方により新しい高[性能](https://ja.wikipedia.org/wiki/性能 "wikilink")の戦闘機が現れたことで、不運にもある意味で「やられ役」を演じることとなってしまった。それでも、ソ連ではI-16を操縦する[撃墜王](https://ja.wikipedia.org/wiki/撃墜王 "wikilink")が幾人も誕生した。だが、[ドイツ国防軍](../Page/ドイツ国防軍.md "wikilink")がソ連に侵攻した[1941年](https://ja.wikipedia.org/wiki/1941年 "wikilink")時点ですでにI-16は相対的に旧式化しきってしまっており、その後も戦闘機や[戦闘爆撃機](../Page/戦闘爆撃機.md "wikilink")として運用が続けられたものの、より高性能な[Yak-1や](../Page/Yak-1_\(航空機\).md "wikilink")[LaGG-3](https://ja.wikipedia.org/wiki/LaGG-3_\(航空機\) "wikilink")、[MiG-3が登場すると](https://ja.wikipedia.org/wiki/MiG-3_\(航空機\) "wikilink")、徐々にそれらに取って代わられていった。

[1932年](../Page/1932年.md "wikilink")の[設計](../Page/設計.md "wikilink")着手時には、[複葉機](../Page/複葉機.md "wikilink")全盛の中で、純[片持式低翼](../Page/カンチレバー.md "wikilink")[単葉](../Page/単葉機.md "wikilink")、純[モノコック](../Page/モノコック.md "wikilink")構造の胴体、引込脚、スライド式の[風防](https://ja.wikipedia.org/wiki/風防 "wikilink")、[推力](../Page/推力.md "wikilink")式単[排気](https://ja.wikipedia.org/wiki/排気 "wikilink")管など、新機軸を盛り込んだ意欲的な設計であった。各種の派生型を合わせ、[1941年](https://ja.wikipedia.org/wiki/1941年 "wikilink")までに8,644機（9,450機とも）が作られた。[第二次世界大戦](https://ja.wikipedia.org/wiki/第二次世界大戦 "wikilink")勃発時には旧式化していたが、新たなタイプが再生産され、[1943年](../Page/1943年.md "wikilink")頃まで対地攻撃任務などに運用された。

そのずんぐりした機体からソ連兵からは**イシャク**（Ishak、[ロバ](../Page/ロバ.md "wikilink")）スペイン内戦の兵士からは**モスカ**（Mosca、[ハエ](../Page/ハエ.md "wikilink")）、**ラタ**（Rata、[ハツカネズミ](../Page/ハツカネズミ.md "wikilink")）などの愛称で呼ばれた。

[ノモンハン事件](../Page/ノモンハン事件.md "wikilink")で[日本軍](../Page/日本軍.md "wikilink")が鹵獲し、その後鹵獲機展示会などで飛行させた。

## 派生型

<small>''一部の型に関する説明には文献によって相当な差異が見られる。このリストは次の文献を基にしている\[3\]\[4\]\[5\]</small>。

  - TsKB-12
    最初の試作機。336 kW (450 hp) [M-22 エンジン](../Page/ブリストル_ジュピター.md "wikilink") 、翼内に非同調式の[ShKAS機関銃](https://ja.wikipedia.org/wiki/ShKAS_\(機関銃\) "wikilink")2丁と弾薬900発を搭載していた。
  - TsKB-12bis
    2番目の試作機。533 kW (715 hp) [Wright SGR-1820-F-3 サイクロン](../Page/ライト_R-1820.md "wikilink") エンジンを搭載していた。
  - TsKB-12P (I-16P)
    翼内に[ShVAK機関砲](https://ja.wikipedia.org/wiki/ShVAK_\(機関砲\) "wikilink") 2丁と弾薬150発を搭載した試作機。
  - TsKB-18
    M-22エンジンと装甲コックピットを備えた地上攻撃機。ShKAS または[PV-1機関銃](https://ja.wikipedia.org/wiki/PV-1_\(機関銃\) "wikilink")4丁と100 kg (220 lb)の爆弾を搭載した。さらに2機のI-16 Type 5には6丁のShKAS機関銃が搭載され、そのうち4丁は地上掃射のために20°下に向けることができた。
  - TsKB-29 (SPB)
    空圧式の降着装置とフラップ、ライトサイクロンエンジン、ShKAS機関銃二丁を搭載しており、[ズヴェノー・プロジェクトにて高速急降下爆撃機として使用された](https://ja.wikipedia.org/wiki/ズヴェノー_\(航空機\) "wikilink")。
  - I-16 Type 1
    量産試作型。358 kW (480 hp)M-22エンジンを搭載していた。.
  - I-16 Type 4
    最初の量産型。M-22エンジンを搭載していた。
  - I-16 Type 5
    Type 4に流線形でテーパーのかかったエンジンカウルと522 kW (700 hp) [シュベツォフ M-25エンジンを搭載した型](https://ja.wikipedia.org/wiki/シュベツォフ_M-25 "wikilink")。2機の試作機がM-62エンジンを搭載して試験された。量産された。
  - I-16 Type 6
    545 kW (730 hp) [シュベツォフ M-25Bエンジンを搭載した](https://ja.wikipedia.org/wiki/シュベツォフ_M-25 "wikilink")。重量は1383 kgまで減少した。
  - I-16 Type 10
    4丁のShKAS機関銃（同調式のものを胴体に2丁搭載し、翼内にあと2丁搭載した）と560 kW (750 hp)M-25Bエンジンを搭載し、スライド式キャノピーに代わって風防を設置、冬季には引き込み式スキーを装着可能とした型。[イスパノスイザ](https://ja.wikipedia.org/wiki/イスパノスイザ "wikilink")製の機体はライトサイクロンR-1820-F-54エンジンを搭載していた。
  - I-16 Type 12
    主翼にShKAS機関銃2丁搭載のType 5に、2門のShVAK機関砲を追加した型。少数がスペイン内戦に投入されたが、機関砲搭載による機動性の低下が大きく自衛することもままならないとして、対地攻撃用として使われた。
  - I-16 Type 16
    Type 10に同調式ShVAK 12.7mm試作型を搭載した型。1939年に16211-16213のシリアルナンバーの3機のみ製作された。工場での試験を通過し空軍での試験のために輸送された\[6\]。
  - I-16 Type 17
    機首にShKAS機関銃2丁搭載のType 10の主翼に、2門のShVAK機関砲を搭載、尾輪をゴム製タイヤに変更、560 kW (750 hp)M-25Vエンジンを搭載した型。数機に地上掃射のための12.7 mm (0.5 in) [UB機関銃が搭載された](https://ja.wikipedia.org/wiki/UB_\(機関銃\) "wikilink")。
  - I-16 Type 18
    Type 10に620 kW (830 hp) [シュベツォフ M-62エンジンと二速スーパーチャージャー](https://ja.wikipedia.org/wiki/シュベツォフ_M-62 "wikilink")、可変ピッチプロペラを搭載した型。 翼下に2つの100 l (26 US gal)増槽を搭載できた。
  - I-16 Type 19
    翼内に搭載した機銃を、ShKASからより発射速度の高いSN（サヴィン–ノロフ）に換装した以外はType 10と同じ型。プロペラ同調機銃は交換されていない。1939年に19211-19213のシリアルナンバーで、SN機関銃が実用化されなかったこともあり3機の製造に終わった。最初は新型機銃の試験に使用され、その後空軍にI-16SNとして送られた。[冬戦争](../Page/冬戦争.md "wikilink")において活動が見られた\[7\]\[8\]。
  - I-16 Type 20
    この名称は最初に1939年2月に第21工場で製造された、プロペラ同調式のSN機関銃を装備した4機の試作機に与えられた。このタイプは1939年8月に却下され、その後この名称は落下式増槽を装備できる最初の型（増槽以外はtype 10と同じ）に再度使用された。この93 l (25 US gal)増槽はPSB-21と名付けられた。80機が空軍に引き渡された。加えて、1940年1月以降に製造されたすべてのI-16はこれらの増槽を搭載することができた\[9\]。これらの増槽は日本の[九七式戦闘機](../Page/九七式戦闘機.md "wikilink")が使用したものを基にしていた。
  - I-16 Type 21 and Type 22
    これらの型は4丁のプロペラ同調機銃を搭載するように計画された。Type 21はShKASのみを搭載する計画で、一方type 22はShKASとSN機関銃を混載する計画だった。これらのタイプはどちらも机上のみの存在で、使用されることはなかった\[10\]。
  - I-16 Type 23
    Type 10に[RS-82ロケット弾を追加した型](https://ja.wikipedia.org/wiki/RS-82_\(ロケット弾\) "wikilink")。1939年5月から35機が製造された。それ以上の生産は1939年8月に中止された\[11\]。
  - I-16 Type 24
    4丁のShKAS、左右のエルロンを同時に下げる方式を廃しフラップを設置、尾輪を追加、2枚目のドアをコックピット右側に設置、670 kW (900 hp) [シュベツォフ M-63エンジンを搭載した型](https://ja.wikipedia.org/wiki/シュベツォフ_M-63 "wikilink")。
  - I-16 Type 27
    Type 17にM-62エンジンを搭載した型。
  - I-16 Type 28
    Type 24に2丁のShKASと2門のShVAKを搭載した型。
  - I-16 Type 29
    機首に2丁の同調式ShKAS、胴体下部に1丁の[UBSを搭載した型](https://ja.wikipedia.org/wiki/UB_\(機関銃\) "wikilink")。翼内には機銃を搭載せず、地上攻撃用の装備のために取っておかれた。ロケット弾架を両方の翼に3つずつ搭載した。さらに1941年からは外部増槽用のハードポイントが多目的なものに変更され、新型落下式増槽PLBG-100またはFAB-100爆弾を搭載できるようになった。1941年夏に撮影された戦時中の写真から2つの搭載例が確認できる。1つは6発の[RS-82ロケット弾と](https://ja.wikipedia.org/wiki/RS-82_\(ロケット弾\) "wikilink")2発のFAB-100爆弾、もう一つの例は4発のRS-132ロケット弾\[12\]。
  - I-16 Type 30
    1941-42年に再び量産された型。M-63エンジンを搭載していた
  - I-16TK
    Type 10に高高度での性能を高めるため[ターボチャージャー](../Page/ターボチャージャー.md "wikilink")を搭載した型。8,600 m (28,200 ft)で494 km/h (307 mph)に達したが、量産はされなかった。
  - UTI-1
    Type 1の複座練習機型。
  - UTI-2
    UTI-1の改良型。固定脚を装備していた。
  - UTI-4 (I-16UTI) または I-16 Type 15
    Type 5の複座練習機型。ほとんどが固定脚を装備した状態で製造された。この型はかなりの数が製造されており、おおよそ3,400機が製造された\[13\]。

1993年から、ニュージーランドのパイロットであり事業家の[ティム・ウォリス氏の所有する](https://ja.wikipedia.org/wiki/w:en:_Tim_Wallis "wikilink")[アルパイン・ファイター・コレクションが](https://ja.wikipedia.org/wiki/w:en:_Alpine_Fighter_Collection "wikilink")、ロシアで発見された6機のI-16と3機のI-153をノヴォシビルスクのソヴィエト航空宇宙研究所(Soviet Aeronautical Research Institute、Sibnia)によって飛行可能な状態にレストアすることを計画した\[14\]。レストアされた最初の機体 (I-16 *9*) の初飛行は1995年に行われた。レストアされた機体は鉄道でヴラディヴァストークに輸送され、そこから香港経由でニュージーランドまで船で輸送された。この計画は3機目のI-153がニュージーランドに到着した1999年に完了した。さらに7機目のI-16が後にアメリカ人収集家のジェリー・イェイゲン氏(Jerry Yagen)のためにレストアされた。
