**材料規格**（ざいりょうきかく）とは、金属やゴムなどの工業材料の特性や形状などを識別するための[規格](../Page/工業規格.md "wikilink")。

## 概要

工業材料の材料規格を定める[工業規格](../Page/工業規格.md "wikilink")には、国際規格（ISO）、地域規格、国家規格、業界（団体）規格、社内規格がある\[1\]。

工業材料に関しては業界（団体）規格のほうが通用している場合もあり、民間の規格を政府が可能な限り踏襲して法律で規格化していることも多い\[2\]。

規格化された製品名を簡略に扱うための[記号](../Page/記号.md "wikilink")を材料記号という。[日本工業規格](https://ja.wikipedia.org/wiki/日本工業規格 "wikilink")(JIS)の材料記号はアルファベット文字と数字で表記される。記号には材質を示す文字記号、製品規格を示す文字記号、種類を表す数字、製法を示す文字記号で構成される。

## 主な材料規格

### 日本（日本工業規格）

#### 鉄鋼系材料

代表的な[日本工業規格](https://ja.wikipedia.org/wiki/日本工業規格 "wikilink")(JIS)における[鉄鋼](https://ja.wikipedia.org/wiki/鉄鋼 "wikilink")系材料記号の一覧の一例（最新の情報は、最新の『**JISハンドブック 鉄鋼**』を確認の事 ）。

第1位の文字は鋼を表すS(Steel)、または鉄を表すF(Ferrum)。

| 記号     | 記号例      | 説明                                                                                                      |
| ------ | -------- | ------------------------------------------------------------------------------------------------------- |
| S□□C   | S45Cなど   | 機械構造用[炭素鋼](../Page/炭素鋼.md "wikilink")（□□は炭素含有量）                                                         |
| SCM□□□ | SCM435   | [クロムモリブデン鋼](../Page/クロムモリブデン鋼.md "wikilink")                                                            |
| SGP□□  | SGP25Aなど | 配管用炭素鋼管（Gas Pipe、□□は大きさ）                                                                                |
| SK□□□  | SK120    | [炭素工具鋼](../Page/炭素工具鋼.md "wikilink")(Kogu、□は炭素含有量)                                                      |
| SKH□□  | SKH51    | [高速度工具鋼](https://ja.wikipedia.org/wiki/高速度工具鋼 "wikilink")                                               |
| SS□□□  | SS400など  | 一般構造圧延鋼（Structural、□□□は最低[引張強さ](https://ja.wikipedia.org/wiki/引張強さ "wikilink")を示す。）                     |
| SUS□□□ | SUS304など | [ステンレス鋼](../Page/ステンレス鋼.md "wikilink")                                                                  |
| FC□□□  | FC200など  | 普通[鋳鉄](../Page/鋳鉄.md "wikilink")（[ねずみ鋳鉄](https://ja.wikipedia.org/wiki/ねずみ鋳鉄 "wikilink")）□□□は最低引張強さを示す。 |
|        |          |                                                                                                         |

### アメリカ合衆国

アメリカ合衆国で材料規格を定めている工業規格には、国家規格の[ANSI](../Page/米国国家規格協会.md "wikilink")（米国規格）や、業界（団体）規格の[ASTM](https://ja.wikipedia.org/wiki/ASTMインターナショナル "wikilink")（アメリカ材料試験協会規格）、AISI（アメリカ鉄鋼協会規格）、[SAE](../Page/SAE規格.md "wikilink")（アメリカ自動車技術会規格）などがある\[3\]。

### 中国

中国で材料規格を定めている工業規格には、国家規格のGB（[中華人民共和国国家標準](https://ja.wikipedia.org/wiki/中華人民共和国国家標準 "wikilink")）や、業界（団体）規格のYB（中国黒色冶金分野標準）などがある\[4\]。
