**マルコフの不等式**（）は[確率論](../Page/確率論.md "wikilink")で、[確率変数](../Page/確率変数.md "wikilink")の非負値[関数の値が](../Page/関数_\(数学\).md "wikilink")、ある正の[定数](../Page/定数.md "wikilink")以上になる[確率](https://ja.wikipedia.org/wiki/確率 "wikilink")の[上限](https://ja.wikipedia.org/wiki/上限 "wikilink")を与える不等式である。[アンドレイ・マルコフ](../Page/アンドレイ・マルコフ.md "wikilink")が証明した。

マルコフの不等式は確率と[期待値](../Page/期待値.md "wikilink")の関係を述べたもので、ランダム変数の[累積分布関数](../Page/累積分布関数.md "wikilink")に関して大まかではあるが有用な限界を与える。

## 定式化

マルコフの不等式は、[測度論](../Page/測度論.md "wikilink")的には、(*X*,Σ,μ) を[測度空間](https://ja.wikipedia.org/wiki/測度空間 "wikilink")とし、*f* を拡張[実数](../Page/実数.md "wikilink")値（無限大もとりうる）[可測関数](https://ja.wikipedia.org/wiki/可測関数 "wikilink")とし、 *t* \> 0 とすれば、

\[\mu(\{x\in X|\,|f(x)|\geq t\}) \leq {1\over t}\int_X |f|\,d\mu\]

であることを述べる。空間の測度が 1 である特別な場合（つまり確率空間である）には、次のように言い換えられる：

*X* を任意の確率変数とし、*a* \> 0 とすると、

\[\textrm{Pr}(|X| \geq a) \leq \frac{\textrm{E}(|X|)}{a}\]

## 確率論における証明

測度空間が確率空間である場合は証明が単純でわかりやすいので、この場合の証明をまず別に示そう。

任意の事象 *E* に対して、*I*<sub>*E*</sub> を *E* の特性確率変数、つまり *E* が起きるならば *I*<sub>*E*</sub> = 1 、そうでないならば = 0 であるとする。すると、事象 *X* ≥ *a* が起きるならば *I*<sub>(*X* ≥ *a*)</sub> = 1 であり、*X* \< *a* ならば *I*<sub>(*X* ≥ *a*)</sub> = 0 である。そこで

\[aI_{(|X| \geq a)} \leq |X|\,\]

ゆえに

\[\operatorname{E}(aI_{(|X| \geq a)}) \leq \operatorname{E}(|X|)\,\]

ここでこの不等式の左辺は

\[a\operatorname{E}(I_{(|X| \geq a)})=a\Pr(|X| \geq a)\,\]

と同じであることがわかる。従って

\[a\Pr(|X| \geq a) \leq \operatorname{E}(|X|)\,\]

となり、 *a* \> 0 だから、両辺を *a* で割ればよい。

## 一般的証明

任意の可測集合 *A* に対して、 1<sub>*A*</sub> をその[特性関数](../Page/指示関数.md "wikilink")、つまり *x* ∈ *A* ならば 1<sub>*A*</sub>(*x*) = 1 、そうでなければ 0 としよう。*A*<sub>*t*</sub> を *A*<sub>*t*</sub> = {*x* ∈ *X*| |*f*(*x*)| ≥ *t*} として定義すれば、

\[0\leq t\,1_{A_t}\leq |f|1_{A_t}\leq |f|\]

となり、ゆえに

\[\int_X t\,1_{A_t}\,d\mu\leq\int_{A_t}|f|\,d\mu\leq\int_X |f|\,d\mu\]

ここで、この不等式の左辺は

\[t\int_X 1_{A_t}\,d\mu=t\mu(A_t)\]

と同じであることに注意しよう。すると

\[t\mu(\{x\in X|\,|f(x)|\geq t\}) \leq \int_X|f|\,d\mu\]

であり、また *t* \> 0 であるから、両辺を *t* で割れば

\[\mu(\{x\in X|\,|f(x)|\geq t\}) \leq {1\over t}\int_X|f|\,d\mu\]

となる。

## 応用例

  - マルコフの不等式は、[チェビシェフの不等式](../Page/チェビシェフの不等式.md "wikilink")の証明に用いられる。
  - *X* を非負[整数](../Page/整数.md "wikilink")値確率変数とする（[組合せ論](https://ja.wikipedia.org/wiki/組合せ論 "wikilink")でよくあるように）と、マルコフの不等式で *a* = 1 とすることにより \(\textrm{Pr}(X \neq 0) \leq \textrm{E}(X)\) が得られる。*X* をある[集合](../Page/集合.md "wikilink")の[濃度とすると](../Page/濃度_\(数学\).md "wikilink")、これからこの集合は[空集合](../Page/空集合.md "wikilink")ではないことが証明される。このように存在証明への応用も可能である。
