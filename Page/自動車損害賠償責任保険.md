[Jidousha_baisho_sekinin_hoken_shomeisho.JPG](https://ja.wikipedia.org/wiki/File:Jidousha_baisho_sekinin_hoken_shomeisho.JPG "fig:Jidousha_baisho_sekinin_hoken_shomeisho.JPG") **自動車損害賠償責任保険**（じどうしゃそんがいばいしょうせきにんほけん）とは、[自動車損害賠償保障法](../Page/自動車損害賠償保障法.md "wikilink")によって、[自動車](../Page/自動車.md "wikilink")および[原動機付自転車](../Page/原動機付自転車.md "wikilink")\[1\]を使用する際、全ての車の所有者に加入が[義務](../Page/義務.md "wikilink")づけられている[損害保険](../Page/損害保険.md "wikilink")である。略称**自賠責保険**（じばいせきほけん）。公道で走行する際に、加入が義務付けられていることから、俗に「[強制保険](https://ja.wikipedia.org/wiki/強制保険 "wikilink")」といわれる。

なお、[農業協同組合](../Page/農業協同組合.md "wikilink")・[消費生活協同組合](https://ja.wikipedia.org/wiki/消費生活協同組合 "wikilink")・[中小企業等協同組合](../Page/中小企業等協同組合.md "wikilink")が[共済](../Page/共済.md "wikilink")として扱う\[2\]**自動車損害賠償責任共済**も存在するが、制度区分を除けば概ね同じ制度であり、以下では自動車損害賠償責任保険と合わせて単に「～保険」のように記述する。

## 概要

自賠責保険は、[自動車損害賠償保障法](../Page/自動車損害賠償保障法.md "wikilink")が施行された[1955年](../Page/1955年.md "wikilink")（[昭和](../Page/昭和.md "wikilink")30年）に、「[交通事故](../Page/交通事故.md "wikilink")が発生した場合の[被害者](../Page/被害者.md "wikilink")の補償」を目的として開始された対人保険制度である。

あらかじめ全ての自動車保有者が自賠責保険に加入することで、交通事故の被害者は「被害者請求制度」を使い、加害者を介さずに「最低限の[損害賠償](../Page/損害賠償.md "wikilink")金」を直接受け取ることができる。

自賠責保険の支払いは、国土交通大臣および金融庁長官が定めた支払基準に基づいてなされることになっており、民間の[自動車保険](../Page/自動車保険.md "wikilink")（任意保険）で導入されている「[ゴールド免許](../Page/ゴールド免許.md "wikilink")・無事故・無違反に対する割引」が一切なく、慰謝料などについては「裁判所算定基準」より低い金額で、画一的な基準で行われることになっている\[3\]。

その代わり自賠責保険においては、交通事故により負傷した者は、自動車保有者及び運転者に過失が無い場合を除き、過失割合にかかわらず\[4\]被害者として扱われ、相手側の自賠責保険から保険金が支払われる。ただし、[過失割合が](../Page/交通事故の過失割合.md "wikilink")70%以上の場合は[重過失減額](https://ja.wikipedia.org/wiki/重過失減額 "wikilink")として、過失割合に応じて一定の割合の減額が適用される（[\#重過失減額](https://ja.wikipedia.org/wiki/#重過失減額 "wikilink")参照）。また、「最低限の補償」の確保を目的としているので、[保険金](https://ja.wikipedia.org/wiki/保険金 "wikilink")の限度額（上限）が被害者1人につき死亡3,000万円まで・[後遺障害](../Page/後遺障害.md "wikilink")は段階に応じて、75万円～最大3,000万円（[介護](../Page/介護.md "wikilink")を要する重度の後遺障害は4,000万円まで）・傷害120万円までと低いうえ、傷害で補償される下限の金額が法律上においても一切明文化されていない。

また、運転者自身の死傷（自損事故）によって損害が発生した場合、および自賠責保険における**「記名被保険者」**または**「自賠責保険の対象自動車等の運転者」**が死傷者の場合、自賠責保険は支払われない\[5\]。

また**死傷者のいない[物損事故](https://ja.wikipedia.org/wiki/物損事故 "wikilink")のみの事故**の場合にも自賠責保険は適用されず、車両や建造物、[鉄道車両](../Page/鉄道車両.md "wikilink")などが破損した分には一切適用されない。そのため多くの自動車所有者は、自賠責保険における補償額の少なさを補い、かつ自損事故や物損事故にも対応出来るよう、任意の[任意保険（自動車保険）](../Page/自動車保険.md "wikilink")（特に対人保険で補償できない事故に対する人身傷害保険）にも別途加入することが一般的になっている。

[自動車検査登録制度](../Page/自動車検査登録制度.md "wikilink")のある自動車や、250ccを超える[オートバイ](../Page/オートバイ.md "wikilink")の場合は、継続検査の際、新しく交付される車検証の有効期間を満たす自動車損害賠償保険証明書を提示しなければ、[自動車検査証](../Page/自動車検査証.md "wikilink")の有効期間の更新はできない。しかし、車検のない原動機付自転車を含む250cc以下のオートバイは、契約期間を1年から最長5年までの期間で任意に契約でき、[コンビニエンスストア](../Page/コンビニエンスストア.md "wikilink")や[郵便局](../Page/郵便局.md "wikilink")でも加入や更新手続きができる場合もあり、契約期間を長くすれば1年あたりの単価が割安になる。車両を廃車・ナンバープレートを返還した際は、契約者の申請により自賠責保険の掛金が返金される。

なお、自動車損害賠償保障法第10条と同法施行令第1条の2の規定により、[自衛隊](../Page/自衛隊.md "wikilink")（道路運送車両法が適用されない車両に限る）・[国連軍](../Page/国連軍.md "wikilink")・[在日米軍](https://ja.wikipedia.org/wiki/在日米軍 "wikilink")の車両には、自賠責保険の付保は要しない\[6\]\[7\]。[外交官](../Page/外交官.md "wikilink")については、自動車損害賠償保障法上の明文規定はないが、外交特権により適用がない。ただし外務省は外交官ナンバーの発給の際に、任意保険の加入を義務付けているため、保障の問題はない。また、農耕作業の用に供することを目的として製作した小型[特殊自動車](../Page/特殊自動車.md "wikilink")は加入できない。

検査対象外軽自動車、原動機付自転車及び締約国登録自動車など、[自動車検査登録制度](../Page/自動車検査登録制度.md "wikilink")のない車両は、自賠責保険に加入すると、保険会社から「保険標章」と呼ばれる自賠責保険の満了年月を記したステッカーが交付される。これらの車両には、保険標章の貼り付けが義務付けられており\[8\]、貼り付けられていない場合は公道の走行が認められない。保険標章を貼り付ける位置は、自動車がフロントガラス、オートバイが[ナンバープレートとなっている](https://ja.wikipedia.org/wiki/日本のナンバープレート "wikilink")。

自賠責保険に加入しないまま、自動車や原動機付自転車を運行させた場合は『**無保険運行**』となり、1年以下の[懲役](../Page/懲役.md "wikilink")又は50万円以下の[罰金](../Page/罰金.md "wikilink")に処せられるほか、[道路交通法](../Page/道路交通法.md "wikilink")上の違反点数6点が加算され、**即[運転免許証](../Page/運転免許証.md "wikilink")停止**の[行政処分になる](https://ja.wikipedia.org/wiki/行政処分_\(運転免許\) "wikilink")（ただし過失の場合はその限りではない）。

### 運営・料率

自賠責保険・共済は、各保険会社や協同組合に加入することとなる\[9\]が、保険金額は[政令](../Page/政令.md "wikilink")で定められ\[10\]、支払い基準も[国土交通大臣](../Page/国土交通大臣.md "wikilink")及び[内閣総理大臣](https://ja.wikipedia.org/wiki/内閣総理大臣 "wikilink")が定めることになっている\[11\]、自賠責の契約申し込みは拒否できない\[12\]など、契約した会社によらず、[日本国政府](https://ja.wikipedia.org/wiki/日本国政府 "wikilink")（[国土交通省](../Page/国土交通省.md "wikilink")）で定めた契約内容が適用されることとなる。自賠責事業による剰余金は[特別会計](../Page/特別会計.md "wikilink")でプールされ、自賠責を扱う損害保険会社などの間で分配される\[13\]。

保険料率についても、「能率的な経営の下における適正な原価を償う範囲内で、できる限り低いもの」として\[14\]、[損害保険料率算出機構](../Page/損害保険料率算出機構.md "wikilink")が算出する料率も通常の範囲料率ではなく、固定の値となっている\[15\]。なお、具体的な料率は車種・契約期間だけでなく、一般の車両については[本土](https://ja.wikipedia.org/wiki/日本列島 "wikilink")・[離島](../Page/離島.md "wikilink")\[16\]・[沖縄県](../Page/沖縄県.md "wikilink")・沖縄県の離島\[17\]の別によって、[タクシー](../Page/日本のタクシー.md "wikilink")・[ハイヤー](../Page/ハイヤー.md "wikilink")は、さらに細かな地域ごとに料率が異なっている\[18\]。

### 保障

#### 重過失減額

交通事故において被害者に過失がある場合であっても、原則として[無過失責任](https://ja.wikipedia.org/wiki/無過失責任 "wikilink")主義に基づき、一定の割合までは[過失相殺](https://ja.wikipedia.org/wiki/過失相殺 "wikilink")による減額を認めず、損害に対する保険金は100%支払われる。ただし、[過失割合が](../Page/交通事故の過失割合.md "wikilink")70%かそれ以上となる場合は**重過失減額**とし、下記に応じて保険金の支払いがなされる。なお、被害者の過失割合が100%と認定される場合は「加害者に責任なし」として、自賠責保険からの保険金支払いはなされない\[19\]。

| 過失割合  | 死亡・後遺障害に関する保険金 | 傷害に関する保険金\[20\] |
| ----- | -------------- | --------------- |
| 70%未満 | 減額なし           |                 |
| 70%以上 | 20%減額          |                 |
| 80%以上 | 30%減額          | 20%減額           |
| 90%以上 | 50%減額          | 20%減額           |
| 100%  | 支払いなし（無責）      |                 |
|       |                |                 |

重過失減額

上記の限度額は保険金の支払限度額に対する減額であるため、保険金が限度額に達していない場合は依然として損害に対する保険金は100%が支払われることになる\[21\]。

## 政府保障事業

政府保障事業は自賠責保険を補完する国の事業で、正式名称は「自動車損害賠償保障事業」である。加害者を特定できない[ひき逃げ](../Page/ひき逃げ.md "wikilink")事故や、加害車両が無保険車であった場合には、被害者が自賠責保険による損害賠償を加害者から受けられないため、自動車損害賠償保障法に基づき政府が自賠責保険の支払基準に準じた損害額を被害者に支払う。政府が損害賠償金を立替払いしているに過ぎないため、加害者が特定される無保険車事故の場合には、後から政府は立替払いした金額を加害者に請求する。損害保険会社であれば、どこの窓口でも政府保障事業に対する被害者からの請求を受け付けている。

### 賠償金未回収問題

この政府保障事業については、交通事故の[加害者](../Page/加害者.md "wikilink")が国に対し、立て替えられた賠償金を弁済する義務がある。にもかかわらず、加害者が国に弁済されないまま回収されないことが多く、平成23年（[2011年](../Page/2011年.md "wikilink")）[3月](https://ja.wikipedia.org/wiki/3月 "wikilink")末現在での未回収残高が458億円にも及び過去最大となったことが、2011年の[会計検査院](../Page/会計検査院.md "wikilink")の指摘によって判明している。会計検査院は[国土交通省](../Page/国土交通省.md "wikilink")に対し、無保険車を減らすための対策が不十分であることを指摘している\[26\]。

## その他の問題

### 人身事故の見込み診断書

人身事故の場合、医師は警察に「見込み診断書」を提出するが、この診断書上の治療期間と行政処分の重さは[道路交通法](../Page/道路交通法.md "wikilink")により関連付けられている\[27\]。そのため、自動車保険ジャーナルの記事によれば、医師側には診断書上の要治療期間の記載は3週間以内に留めることが望ましいと考える習慣があり、実際に、警察提出用の診断書は殆どが3週間以内となっているという\[28\]。交通事故は年々減少しているが、2014年の時点では、年間の受傷者数は71万人以上である（死亡者を除く）\[29\]。

### 人身事故の素因減額

1988年の[最高裁判所は](../Page/最高裁判所_\(日本\).md "wikilink")、受傷者に心理的要因による被害があると[推定](../Page/推定.md "wikilink")できる場合は賠償額を減額（素因減額）することができると自ら判決したが\[30\]、初期の医療機関での症状の見落としなどにより受傷の程度が過少に算定されてしまっていた場合に、セカンド・オピニオンを求めるなどして診断を正当に是正することは容易であるとは言えない。

また最高裁判所は1992年には、事故以前からあった受傷者の体質的疾患を考慮して素因減額を行えるということも自ら判決している\[31\]。他方、体型的要因については1996年、受傷者に通常とは異なる身体的特徴があったとしても、それが疾患に当たらない場合は特別の事情がない限り、その身体的特徴をもって素因減額はできないとしている\[32\]。

### 人身事故と自殺との因果関係

1968年から1993年までの30年間に､交通事故受傷者が自殺したことに起因する損害賠償請求裁判の判例は50件以上ある\[33\]｡

最高裁判所は1975年には､交通事故の後遺障害が残存しており自殺した21歳男性の事件について、相当因果関係を否定する判決をした\[34\]。

ただ､その後の1993年には､事故から約4年後に自殺した47才男性の事件について､事故が加害者の一方的過失であったこと､被害者に与えた精神的衝撃が長年に渡り残ったこと､その後の補償交渉が円滑に進行しなかったことなどから交通事故と自殺との間に相当因果関係を認めた判例もある\[35\]。

### 運用益の一般会計化による未返還問題

[2017年](../Page/2017年.md "wikilink")[10月20日](../Page/10月20日.md "wikilink")の[毎日新聞](https://ja.wikipedia.org/wiki/毎日新聞 "wikilink")の報道によると、自賠責保険の運用益は、従来は[運輸省](../Page/運輸省.md "wikilink")（現・[国土交通省](../Page/国土交通省.md "wikilink")）の[特別会計](../Page/特別会計.md "wikilink")に繰り入れられていたが、[1994年](../Page/1994年.md "wikilink")に当時の[大蔵省](../Page/大蔵省.md "wikilink")（現・[財務省](https://ja.wikipedia.org/wiki/財務省 "wikilink")）が、国の財政の逼迫を理由として、[一般会計](https://ja.wikipedia.org/wiki/一般会計 "wikilink")に繰り入れた。その後両省は返還に向けて覚書を締結したが、返還期限となっている[2000年](../Page/2000年.md "wikilink")度を過ぎても元本分以外は返還されておらず、交通事故被害者への補償が滞る虞が指摘された\[43\]。その後財務省と国土交通省は、特別会計に残った運用益の枯渇を防ぐため一定額の返還を実施することなどを盛り込んだ覚書を、[2017年](../Page/2017年.md "wikilink")[12月18日](../Page/12月18日.md "wikilink")に締結することになり、これを受け財務省は[2003年](../Page/2003年.md "wikilink")以来15年ぶりに返還を再開する\[44\]。
