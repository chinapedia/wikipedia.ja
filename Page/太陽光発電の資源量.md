当記事では、[太陽光発電](../Page/太陽光発電.md "wikilink")に関する事柄のうち、特に**太陽光発電の資源量**（たいようこうはつでんのしげんりょう）について述べる。

## 概要

地表面に到達する[太陽光](../Page/太陽光.md "wikilink")のエネルギーは人類のエネルギー消費量に比べて桁違いに大きく、砂漠などの陸地のごく一部に[太陽光発電](../Page/太陽光発電.md "wikilink")設備（もしくは[太陽熱発電](../Page/太陽熱発電.md "wikilink")設備）を設置するだけで全世界のエネルギー需要量を上回るエネルギーが得られるとされる。また太陽電池の製造に必要なシリコンなどの資源も人類の全エネルギー需要を供給するに足りるだけの量があり、事実上無限とされる。

一般的に、[太陽光発電](../Page/太陽光発電.md "wikilink")は設置に必要な条件が少ないため、潜在的に設置できる量は多い。太陽光発電は電力の消費地に直接設置するのが効率がよい設置方法とされる。太陽光を十分に受けることができ、設置場所がその重量に耐えることができる場所であれば様々な建造物に設置が可能である。日本においても、必要量を置く場所は十分にあるとされる。

## 太陽エネルギーから見た資源量

[thumbより](https://ja.wikipedia.org/wiki/ファイル:Solar_land_area.png "wikilink")）\]\] [thumbで発電するのに必要な面積](https://ja.wikipedia.org/wiki/ファイル:Fullneed.jpg "wikilink")\[1\]\]\] [太陽光](../Page/太陽光.md "wikilink")の[エネルギー](https://ja.wikipedia.org/wiki/エネルギー "wikilink")は薄く広く分布するが、[地球](https://ja.wikipedia.org/wiki/地球 "wikilink")全体では膨大な量となる。[太陽](https://ja.wikipedia.org/wiki/太陽 "wikilink")から地球全体に照射されている光エネルギーは、[ワット](../Page/ワット.md "wikilink")数にして約180PW（P=[ペタ](https://ja.wikipedia.org/wiki/ペタ "wikilink")=10の15乗）である。そのうち、地上で実際に利用可能な量は約1PWといわれる\[2\]。また、仮に[ゴビ砂漠](../Page/ゴビ砂漠.md "wikilink")に太陽電池を敷き詰めると、全人類のエネルギー需要量に匹敵する発電量が得られるという\[3\]。設置場所における年間の日射量は緯度や気候によって異なる。日本では約1200kWh/m<sup>2</sup>である。欧州では中部で約1000kWh/m<sup>2</sup>、南部で約1700kWh/m<sup>2</sup>である。また赤道付近の国々では最大約2600kWh/m<sup>2</sup>に達する\[4\]。

## 原料から見た資源量

太陽光発電システムの生産に必要な原料も基本的に豊富であり、少なくとも2050年頃までに予測される需要は十分に満たせるとされる\[5\]。シリコンを用いる太陽電池では、資源量は事実上無限とされる。またシリコンを用いない太陽電池についてはインジウムなどの資源が将来的に制約になる可能性があるが、技術的に使用量を節約することで2050年以降も利用可能ではないかと見られている\[6\]。

[太陽電池](../Page/太陽電池.md "wikilink")は、種類により下記のような原料をその[半導体](../Page/半導体.md "wikilink")層（[pn接合](https://ja.wikipedia.org/wiki/pn接合 "wikilink")部分周辺）に用いる。

  - [結晶シリコン太陽電池](https://ja.wikipedia.org/wiki/結晶シリコン太陽電池 "wikilink")…[シリコン](../Page/ケイ素.md "wikilink")（結晶シリコン）
  - [薄膜シリコン太陽電池](https://ja.wikipedia.org/wiki/薄膜シリコン太陽電池 "wikilink")…シリコン（[シランガス](../Page/シラン_\(化合物\).md "wikilink")）
  - [CIS系太陽電池](https://ja.wikipedia.org/wiki/CIS系太陽電池 "wikilink")…[銅](../Page/銅.md "wikilink")、[インジウム](../Page/インジウム.md "wikilink")、[ガリウム](../Page/ガリウム.md "wikilink")、[セレン](../Page/セレン.md "wikilink")、[硫黄](../Page/硫黄.md "wikilink")など
  - [CdTe太陽電池](https://ja.wikipedia.org/wiki/CdTe太陽電池 "wikilink")…[カドミウム](../Page/カドミウム.md "wikilink")、[テルル](../Page/テルル.md "wikilink")
  - [III-V族太陽電池](https://ja.wikipedia.org/wiki/III-V族太陽電池 "wikilink")…[ガリウム](../Page/ガリウム.md "wikilink")、[砒素](https://ja.wikipedia.org/wiki/砒素 "wikilink")、[リン](../Page/リン.md "wikilink")、[ゲルマニウム](../Page/ゲルマニウム.md "wikilink")など
  - [色素増感太陽電池](https://ja.wikipedia.org/wiki/色素増感太陽電池 "wikilink")、[有機薄膜太陽電池](https://ja.wikipedia.org/wiki/有機薄膜太陽電池 "wikilink")…有機色素、[チタン](../Page/チタン.md "wikilink")など

このほか、電極に[銀](https://ja.wikipedia.org/wiki/銀 "wikilink")や[インジウム](../Page/インジウム.md "wikilink")を用いる場合がある。

現在の市場の主流は[結晶シリコン太陽電池](https://ja.wikipedia.org/wiki/結晶シリコン太陽電池 "wikilink")である。[シリコン](https://ja.wikipedia.org/wiki/シリコン "wikilink")（ケイ素）の主要原料である[二酸化ケイ素](../Page/二酸化ケイ素.md "wikilink")（シリカ）の資源量は事実上無限であり、全世界の需要を今後も長期に亘って満たせる資源量がある\[7\]。工業用の高純度のものも世界中に広く分布する\[8\]。このため少なくとも現在見通せる範囲では、結晶シリコン太陽電池や薄膜シリコン太陽電池では資源的な心配は無いとされる\[9\]。

一方、[インジウム](../Page/インジウム.md "wikilink")、[ガリウム](../Page/ガリウム.md "wikilink")、[セレン](../Page/セレン.md "wikilink")、[テルル](../Page/テルル.md "wikilink")、[ゲルマニウム](../Page/ゲルマニウム.md "wikilink")などはシリコンほど潤沢でなく、現在の技術水準のままだとすればそれぞれ対応する太陽電池は2050年までに何らかの資源的制約を受ける可能性がある\[10\]。しかしそれまでの技術の進展によって原料の増産や節約、他の材料による代替も期待できるため、2050年までだけでなく、それ以降も十分に需要を満たせるのではないかと推測されている\[11\]。

太陽電池用シリコン原料の供給は2008年までは逼迫して価格も高止まりしていたが、各社の増産が追いつくことで2009年からは価格の低下が予測されている\[12\]。 結晶シリコン太陽電池の生産には微細シリコン半導体デバイスほどの原料純度（11N〜）は必要なく、7N程度で足りる。太陽電池専用の[ソーラーグレードシリコン](https://ja.wikipedia.org/wiki/ソーラーグレードシリコン "wikilink")（SOG-Si）原料の増産の動きが活発であり\[13\]、今後は高純度シリコン市場の大部分を占めるようになると予測されている\[14\] 。ソーラーグレード専用の生産技術も様々なものが実用化されており、精製に必要なエネルギーやコストも大幅に削減されると見られている。（詳しくは[ソーラーグレードシリコン](https://ja.wikipedia.org/wiki/ソーラーグレードシリコン "wikilink")を参照。）

## 日本における資源量

太陽光発電は集中型発電所などに比べれば比較的大きな設置面積を必要とする。しかし設置する場所に対する制約が少ないことから、国土の比較的狭い日本においても、下記のように電力需要よりも遙かに多い量が設置できるだけの場所があるとされる。このため導入量は設置可能な面積ではなく、電力供給の構成上の観点から決まるとされる\[18\]。日本国内で導入可能な量は、下記のように見積もられている\[19\]。

|                        |
| ---------------------- |
| **日本における太陽光発電の設置可能量**  |
| **設置場所**               |
| 戸建て住宅                  |
| 集合住宅                   |
| 公共施設                   |
| 大型産業施設                 |
| その他（未利用地における水素製造などを含む） |
| 合計                     |
|                        |

日本国内の標準的な環境においては、設備量1kWpあたりの発電量は約1000kWh/年である\[20\]。100GWpの設備量の年間発電量は約100TWhとなり、日本の年間総発電量（約1064TWh\[21\]）の約10%となる。[太陽光発電](../Page/太陽光発電.md "wikilink")の導入により、昼間のピーク電力需要の緩和と、火力発電に由来する[温暖化ガス](https://ja.wikipedia.org/wiki/温暖化ガス "wikilink")の排出量削減が期待されている\[22\]。

2004年に[NEDOの示したロードマップにおいては](https://ja.wikipedia.org/wiki/新エネルギー産業技術総合開発機構 "wikilink")、2030年までに100GWpの導入を目標としている\[23\]。[福田ビジョン](../Page/福田ビジョン.md "wikilink")においては2008年時点での導入量の40倍が目標として掲げられた。2008〜2009年にかけて与野党や関係省庁からも導入ペースの前倒しが打ち出され（[固定価格買取制度\#歴史的経過](https://ja.wikipedia.org/wiki/固定価格買取制度#歴史的経過 "wikilink")を参照）、2009年11月からの新たな買取制度\[24\]に於いては2020年までに2005年の20倍(29GWp)の導入目標が掲げられている\[25\]。

世界的に見ると、日本における平均年間日照量は最も日照の多い地域の半分程度である。[米国](https://ja.wikipedia.org/wiki/米国 "wikilink")の平均とほぼ同等であり、また導入量世界一の[ドイツ](https://ja.wikipedia.org/wiki/ドイツ "wikilink")よりは多い（右上図参照）。国内で見ると、冬期に晴天が少なく積雪の多い日本海側では日照量・発電量が少なく、太平洋側で多くなる\[26\]。

## 懐疑論と反論

日本において、下記のような懐疑論が散見される。

  - 集中型発電所に比べて設置面積が大きいことを理由に、必要量の導入可能性に否定的な意見が見られる。しかし上述のように実際には設置できる場所自体は十分にあり設置可能面積は導入量の制約要因ではないとされているが、中短期的には実現が現実的でない設置数を元にしている、一番発電量が多い昼間ピーク時の値に基づいた見積もりを行っている等の指摘がある。
  - 高純度シリコン原料の供給逼迫を理由に、将来生産できる設備量が不足するかのように主張する意見が見られた。しかしこれは原料の精製能力が逼迫していたことが原因であり\[27\]、供給量増大によって解消に向かっているとされる\[28\]。
