**韻母**（いんぼ）とは、[中国語](https://ja.wikipedia.org/wiki/中国語 "wikilink")の[音節](../Page/音節.md "wikilink")の構成要素の一つ。1[音節](../Page/音節.md "wikilink")内で[声母](https://ja.wikipedia.org/wiki/声母 "wikilink")（頭子音）の後に続き、[声調](../Page/声調.md "wikilink")を除いた[母音](../Page/母音.md "wikilink")を中心とした部分をいう（声調も韻母に含める場合がある）。

韻母は、**[介音](../Page/介音.md "wikilink")**（[半母音](https://ja.wikipedia.org/wiki/半母音 "wikilink")）・**主母音**・**[尾音](https://ja.wikipedia.org/wiki/尾音 "wikilink")**（鼻音か二重母音を構成する補助的な母音）で構成される。韻母の中における部位を表す用語としては、介音を**[韻頭](https://ja.wikipedia.org/wiki/韻頭 "wikilink")**、主母音を**韻腹**、尾音を**[韻尾](https://ja.wikipedia.org/wiki/韻尾 "wikilink")**と呼ぶ。

また[韻文](https://ja.wikipedia.org/wiki/韻文 "wikilink")の[押韻](../Page/押韻.md "wikilink")や[韻書](../Page/韻書.md "wikilink")の分類には、介音を除いた韻腹と韻尾部分および音節全体にかかっている声調（[四声](../Page/四声.md "wikilink")）を区別したものが用いられ、これを**韻**（いん）と呼ぶ。

## 特徴

例として、「東」字の発音は[標準中国語](https://ja.wikipedia.org/wiki/標準中国語 "wikilink")（[普通話](../Page/普通話.md "wikilink")）では  と表記されるが、このうち d は声母、-ong が韻母、¯ が声調である。この場合、介音はゼロで、-o-が主母音、-ngが尾音である。なおこの韻母は[注音字母にしたがって分けると](../Page/注音符号.md "wikilink")、介音 -u- と、韻腹と韻尾 -eng からなる。

また、「光」字の発音の例では、標準中国語（普通話）では  と表記されるが、音節の構成部分に分けると以下のようになる。

|    |
| -- |
| 音節 |
| 声母 |
| 韻頭 |
| 介音 |
| g  |

これらの構成は、[r化しない音節では](https://ja.wikipedia.org/wiki/児化 "wikilink")、[普通話](../Page/普通話.md "wikilink")・[方言](../Page/方言.md "wikilink")を問わず共通している。r化した音節では上記から外れている音節の例がみられる。

また、伝統的な音韻学では韻母を介音の特性から、[開口呼](https://ja.wikipedia.org/wiki/開口呼 "wikilink")・[斉歯呼](https://ja.wikipedia.org/wiki/斉歯呼 "wikilink")・[合口呼](https://ja.wikipedia.org/wiki/合口呼 "wikilink")・[撮口呼](https://ja.wikipedia.org/wiki/撮口呼 "wikilink")の [四呼](../Page/四呼.md "wikilink")によって4つに分類し、韻尾の特性から[陰声韻](https://ja.wikipedia.org/wiki/陰声韻 "wikilink")（母音で終わるもの）・[陽声韻](https://ja.wikipedia.org/wiki/陽声韻 "wikilink")（[鼻音](../Page/鼻音.md "wikilink")韻尾）・[入声](../Page/入声.md "wikilink")韻（[閉鎖音](https://ja.wikipedia.org/wiki/閉鎖音 "wikilink")韻尾）の3つに分類する。

## 中世中国語

「[韻書](../Page/韻書.md "wikilink")」で使われる韻字には平・上・去・入の[声調](../Page/声調.md "wikilink")の違いによって異なる字が使われた。語尾子音のうち[内破音](../Page/内破音.md "wikilink")の、、は尾音\[m\]、\[n\]、\[ŋ\]の[入声](../Page/入声.md "wikilink")とされ、\[m\]と、\[n\]と、\[ŋ\]とはそれぞれ同じ尾音と考えられている。

なお[韻](https://ja.wikipedia.org/wiki/韻 "wikilink")と韻母は正確には同じではなく、韻とは韻母のうち主母音と尾音、それに声調を加えたものである。このため、[漢詩](../Page/漢詩.md "wikilink")で[押韻](../Page/押韻.md "wikilink")する場合、韻母をそろえることが必要であるが、介音は必ずしも一致する必要は無く、主母音と韻尾、声調が一致していなければならない。

[六朝](../Page/六朝.md "wikilink")から[唐](https://ja.wikipedia.org/wiki/唐 "wikilink")代の中国語についてまとめた『[広韻](../Page/広韻.md "wikilink")』では、206韻が設けられ、その内訳は、上平声28韻、下平声29韻（平声を表す字が多いので上下2巻に分けられた）、上声55韻、去声60韻、入声34韻であった。その後、漢詩の押韻の規範となる「[平水韻](../Page/平水韻.md "wikilink")」では、これを106韻に整理し、上平声15韻、下平声15韻、上声29韻、去声30韻、入声17韻の韻目が建てられた。

## 近世中国語

[元代](https://ja.wikipedia.org/wiki/元代 "wikilink")に[北曲](../Page/北曲.md "wikilink")をもとに作られた『[中原音韻](../Page/中原音韻.md "wikilink")』では、中原地方における中国語では[入声](../Page/入声.md "wikilink")が消滅したことを反映して、入声が平声・上声・去声のどれに変わったかを示している。また、韻目を19韻にまとめて、同韻の字を集めた上で、それぞれがどういう声調であるかを示している。
