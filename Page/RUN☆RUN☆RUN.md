## 解説

初回盤には｢キラ☆キラ☆ ハイカラステッカー｣が封入されている。

表題曲「RUN☆RUN☆RUN」は、[ロッテ](../Page/ロッテ.md "wikilink")[ガム](https://ja.wikipedia.org/wiki/ガム "wikilink")『ピュアホワイトシトラス』CMソングに起用された\[1\]。
