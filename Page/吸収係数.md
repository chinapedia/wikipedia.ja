**吸収係数**（きゅうしゅうけいすう、）または**吸光係数**（きゅうこうけいすう）とは[光](../Page/光.md "wikilink")がある[媒質](https://ja.wikipedia.org/wiki/媒質 "wikilink")に[入射](https://ja.wikipedia.org/wiki/入射 "wikilink")したとき、その媒質がどれくらいの光を吸収するのかを示す定数。[長さの逆数](https://ja.wikipedia.org/wiki/長さの逆数 "wikilink")の[次元を持つ](https://ja.wikipedia.org/wiki/量の次元 "wikilink")。[ランベルト・ベールの法則](../Page/ランベルト・ベールの法則.md "wikilink")に従えば、媒質をある距離通過した光の強度と入射した光の強度の比の対数（[吸光度](../Page/吸光度.md "wikilink")）は、通過距離と比例関係にあり、その比例係数を吸収係数と呼ぶ。

記号としては[自然対数](../Page/自然対数.md "wikilink")の形式ではα、[常用対数](https://ja.wikipedia.org/wiki/常用対数 "wikilink")の形式ではβを用いる場合が多い。また、[溶液](../Page/溶液.md "wikilink")などの吸収を観測する場合、αまたはβを溶液の単位[モル濃度](../Page/モル濃度.md "wikilink")で[規格化](../Page/規格化.md "wikilink")した**モル吸光係数**εが用いられる。

[音波](../Page/音波.md "wikilink")についても同様に、**音波吸収係数**などの概念が用いられる\[1\]。

## 定義

媒質に入射する前の光の[強度を](../Page/放射照度.md "wikilink")*I*<sub>0</sub> としたとき、入射後の光の強度*I* はランベルト・ベールの法則から吸収係数α、βおよびεを用いて以下の式で示される。

\[I = I_0 e^{-\alpha x} = I_0 10^{-\beta x} = I_0 10^{-\epsilon cx}\]

ここで*x*は媒質の距離、*c* は溶液のモル濃度である。したがって、

\[\alpha = -\frac{1}{x}\ln\frac{I}{I_0},\quad
\beta = -\frac{1}{x}\log_{10}\frac{I}{I_0},\quad
\epsilon= -\frac{1}{cx}\log_{10}\frac{I}{I_0}\] となる。

## 吸光度との関係

[吸光度](../Page/吸光度.md "wikilink")*A* = -log<sub>10</sub>(*I* /*I*<sub>0</sub> ) をこれらの式を用いて表すと、

\[A = 0.434\alpha x = \beta x = \epsilon cx\]

となる。ここで、0.434 = log<sub>10</sub>*e* である。
