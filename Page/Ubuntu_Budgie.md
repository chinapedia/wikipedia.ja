**Ubuntu Budgie**（以前の名称として**Budgie-Remix**）は、[Ubuntu](../Page/Ubuntu.md "wikilink")の公式派生版であり、[Budgie](https://ja.wikipedia.org/wiki/Solus#Budgie "wikilink")[デスクトップ環境](https://ja.wikipedia.org/wiki/デスクトップ環境 "wikilink")を用いている。Ubuntu Budgieは、安定していてよくテストされたUbuntuの中核的部分を、現代的で軽量で伝統的な見た目の[Solus](https://ja.wikipedia.org/wiki/Solus "wikilink")プロジェクトによって開発されているBudgie デスクトップに結び付けている。

## 歴史

Ubuntu Budgieは、Ubuntu 16.04 LTSの開発に合わせて、非公式な派生版として開発が始められた（budgie-remixと呼ばれた）\[1\]。Budgie Remix 16.10は、Ubuntu 16.10の公開にすぐ続いて公開された\[2\]。Budgie-Remixは、Ubuntuの公式派生版となった際、Ubuntu Budgieに改名されている\[3\]。

Ubuntu Budgie 17.04は[2017年](../Page/2017年.md "wikilink")4月に公開\[4\]され、また同年10月には17.10にアップデートされた\[5\]。Ubuntu Budgie 18.10では、32ビット版が廃止された\[6\]。32ビット版は[Ubuntu MATEでも同時に廃止されている](https://ja.wikipedia.org/wiki/Ubuntu_MATE "wikilink")。

## 開発履歴

### budgie-remix 16.04

budgie-remix 16.04は、[2016年](../Page/2016年.md "wikilink")[4月25日](../Page/4月25日.md "wikilink")に公開された\[7\]\[8\] 。これは、Ubuntu 16.04の公開の4日後である。システムは、Budgie Desktop 10.2.5を含み、[Mutter](https://ja.wikipedia.org/wiki/Mutter "wikilink")に合わせたウィンドウマネージャや設定可能なパネル、通知センターの設定なども含む。また、Arc-GTK+テーマが使われている。

### budgie-remix 16.10

budgie-remix 16.10は、2016年[10月16日](../Page/10月16日.md "wikilink")に公開された\[9\]\[10\]\[11\]。これは、Ubuntu 16.10の公開の3日後である。システムは、Budgie Desktop 10.2.7、GTK+ 3.22、[Linuxカーネル](../Page/Linuxカーネル.md "wikilink") 4.8を含む。このバージョンでは多くの機能改善がなされ、ディスク全体の[暗号化](https://ja.wikipedia.org/wiki/暗号化 "wikilink")、ホームフォルダーの[暗号化](https://ja.wikipedia.org/wiki/暗号化 "wikilink")、導入過程の多言語対応などがなされた。

### Ubuntu Budgie 17.04

Ubuntu Budgie 17.04は[2017年](../Page/2017年.md "wikilink")[10月19日](../Page/10月19日.md "wikilink")に公開された\[12\]\[13\]。上述のように、budgie-remixは、Ubuntuコミュニティの公式製番となったことにあわせ、Ubuntu Budgieに改名されている。システムは、Budgie Desktop 10.2.9、Linux 4.10などを含む。

### Ubuntu Budgie 17.10

Ubuntu Budgie 17.10は、2017年[10月19日](../Page/10月19日.md "wikilink")に公開された\[14\]\[15\]。システムは、Budgie Desktop 10.4、Linuxカーネル4.13を含む。Budgie Desktop 10.4は、新たなAlt-Tabや、Ravenでの[Spotify](https://ja.wikipedia.org/wiki/Spotify "wikilink")対応、ウィンドウコントロールの左配置と右配置の変更に対応、[SMB](https://ja.wikipedia.org/wiki/SMB "wikilink")ファイルや暗号化された[ボリュームの](../Page/論理ボリュームマネージャ.md "wikilink")[アプレット](https://ja.wikipedia.org/wiki/アプレット "wikilink")追加による対応、日付と時刻の見た目の変更などの新しい機能を含む。

### Ubuntu Budgie 18.04

Ubuntu Budgie 18.04は[2018年](../Page/2018年.md "wikilink")[4月26日](../Page/4月26日.md "wikilink")に公開された\[16\]\[17\]\[18\]。システムはLinuxカーネル4.15を含む。このバージョンでは、OpenVNCをネットワークマネージャから導入できるようになった。また、Quick NoteやDropByなどの新しいアップレットの追加などが行われた。
