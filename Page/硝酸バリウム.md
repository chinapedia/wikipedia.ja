**硝酸バリウム**（しょうさんバリウム、Barium nitrate）は化学式**Ba(NO<sub>3</sub>)<sub>2</sub>**で表される[バリウム](../Page/バリウム.md "wikilink")の[硝酸塩](https://ja.wikipedia.org/wiki/硝酸塩 "wikilink")である。加熱すると[融点](../Page/融点.md "wikilink")付近で[亜硝酸バリウム](https://ja.wikipedia.org/wiki/亜硝酸バリウム "wikilink")を経て[酸化バリウム](../Page/酸化バリウム.md "wikilink")に分解する。

## 合成法

\(\mathrm{Ba(OH)_2 + 2 \ HNO_3 \longrightarrow Ba(NO_3)_2 + 2 \ H_2O}\)

\(\mathrm{Ba(OH)_2 + N_2O_5 \longrightarrow Ba(NO_3)_2 + H_2O}\)

\(\mathrm{BaO + 2 \ HNO_3(verd.) \longrightarrow Ba(NO_3)_2 + H_2O}\)

\(\mathrm{Ba + 2 \ HNO_3(verd.) \longrightarrow Ba(NO_3)_2 + H_2}\)

\(\mathrm{BaCO_3 + 2 \ HNO_3(verd.) \longrightarrow Ba(NO_3)_2 + H_2O + CO_2}\)

また、[硫酸バリウム](../Page/硫酸バリウム.md "wikilink")の粉末を[木炭](../Page/木炭.md "wikilink")とともに混合し、1,000℃で加熱処理すると[硫化バリウム](https://ja.wikipedia.org/wiki/硫化バリウム "wikilink")となり、これを[硝酸](../Page/硝酸.md "wikilink")と反応させることで硝酸バリウムが得られる。
\(\mathrm{BaSO_4 + 2 \ C \longrightarrow BaS + CO_2}\)

\(\mathrm{BaS + 2 \ HNO_3 \longrightarrow Ba(NO_3)_2 + H_2S}\)

## 用途

  - 火薬類の酸化剤として用いられる。
  - [トリニトロトルエン](../Page/トリニトロトルエン.md "wikilink")と混合すると[バラトール](../Page/バラトール.md "wikilink")という爆薬になる。

## 毒性

他のバリウム化合物と同じく有[毒](../Page/毒.md "wikilink")であり、中毒の徴候は嘔吐、下痢、腹部の苦痛、筋肉の震えなどがある。

## 法規制

日本では[毒物及び劇物取締法](https://ja.wikipedia.org/wiki/毒物及び劇物取締法 "wikilink")および[毒物及び劇物指定令](https://ja.wikipedia.org/wiki/毒物及び劇物指定令 "wikilink")により**バリウム化合物**として[劇物](https://ja.wikipedia.org/wiki/劇物 "wikilink")に指定されている。また[消防法](../Page/消防法.md "wikilink")により、硝酸塩類として[危険物](https://ja.wikipedia.org/wiki/危険物 "wikilink")第1類に指定されている。他に、[船舶安全法](https://ja.wikipedia.org/wiki/船舶安全法 "wikilink")、[航空法](https://ja.wikipedia.org/wiki/航空法 "wikilink")に規定があり、バリウム化合物として[大気汚染防止法](https://ja.wikipedia.org/wiki/大気汚染防止法 "wikilink")の、バリウムの水溶性化合物として[労働安全衛生法](https://ja.wikipedia.org/wiki/労働安全衛生法 "wikilink")、[PRTR法の](https://ja.wikipedia.org/wiki/特定化学物質の環境への排出量の把握等及び管理の改善の促進に関する法律 "wikilink")、アンモニウム化合物として[水質汚濁防止法](https://ja.wikipedia.org/wiki/水質汚濁防止法 "wikilink")の規制を受ける。
