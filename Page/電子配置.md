**電子配置**（でんしはいち、）とは、多電子系である[原子](https://ja.wikipedia.org/wiki/原子 "wikilink")や[分子](../Page/分子.md "wikilink")の[電子状態](../Page/電子状態.md "wikilink")が「[一体近似](https://ja.wikipedia.org/wiki/一体近似 "wikilink")で得られる[原子軌道](../Page/原子軌道.md "wikilink")あるいは[分子軌道](../Page/分子軌道.md "wikilink")に複数の[電子](https://ja.wikipedia.org/wiki/電子 "wikilink")が詰まった状態」として近似的に表すことができると考えた場合に、電子がどのような軌道に配置しているのか示したもので、これによって各[元素](../Page/元素.md "wikilink")固有の性質が決定される。

## 量子数と軌道

原子を構成している[電子](https://ja.wikipedia.org/wiki/電子 "wikilink")の振舞いは[原子核](../Page/原子核.md "wikilink")による静電ポテンシャル中の3次元[シュレーディンガー方程式](../Page/シュレーディンガー方程式.md "wikilink")を解くことで得られる（特に[水素原子におけるシュレーディンガー方程式の解](https://ja.wikipedia.org/wiki/水素原子におけるシュレーディンガー方程式の解 "wikilink")は解析的に解ける）。電子のとり得る軌道は[主量子数](https://ja.wikipedia.org/wiki/量子数#主量子数 "wikilink") n、[方位量子数](https://ja.wikipedia.org/wiki/方位量子数 "wikilink") l、[磁気量子数](https://ja.wikipedia.org/wiki/磁気量子数 "wikilink") m の3つによって指定される。

  - 主量子数 n は軌道の大きさとエネルギーを決定している。1, 2, 3, ... と整数値をとり、これは[電子殻](../Page/電子殻.md "wikilink") K殻、L殻、M殻、…に対応している。
  - 方位量子数 *l* は軌道の形を決定している。0, 1, 2, ..., n−1 の整数値をとる。これは[s軌道](https://ja.wikipedia.org/wiki/s軌道 "wikilink")、[p軌道](https://ja.wikipedia.org/wiki/p軌道 "wikilink")、[d軌道](https://ja.wikipedia.org/wiki/d軌道 "wikilink")、f軌道、g軌道…に対応している。
  - 磁気量子数mは各軌道を決定している。−*l*, −*l*+1, ..., 0, ..., *l*−1, *l* の整数値をとる。

例えば、主量子数2、方位量子数1の軌道を総称して2p軌道と呼ぶ。2p軌道は −1, 0, 1 の3つの磁気量子数をとり得るが、これらに対応して、2p<sub>x</sub>, 2p<sub>y</sub>, 2p<sub>z</sub> の異なる配位をもつ3つの軌道が存在する。

## 軌道と電子対

このように電子の配置は軌道と対応付けられる。そして電子の属する軌道の種類に応じて、電子も分類されて呼称される。

  - **s電子** - [s軌道](https://ja.wikipedia.org/wiki/s軌道 "wikilink")上の電子。1s電子、2s電子、3s電子、4s電子、5s電子、6s電子、7s電子が存在する。
  - **p電子** - [p軌道](https://ja.wikipedia.org/wiki/p軌道 "wikilink")上の電子。2p電子、3p電子、4p電子、5p電子、6p電子、7p電子が存在する。
  - **d電子** - [d軌道](https://ja.wikipedia.org/wiki/d軌道 "wikilink")上の電子。3d電子、4d電子、5d電子、6d電子が存在する。
  - **f電子** - [f軌道](https://ja.wikipedia.org/wiki/f軌道 "wikilink")上の電子。4f電子、5f電子が存在する。

また、電子は[フェルミ粒子](../Page/フェルミ粒子.md "wikilink")なので1つの軌道には、お互いに逆向きの[スピンをもつ](https://ja.wikipedia.org/wiki/スピン角運動量 "wikilink")2個の電子しか入ることができない（「[パウリの排他原理](../Page/パウリの排他原理.md "wikilink")」）。このように軌道が2つの電子によって占有された状態を**電子対**と呼ぶ。

言い換えると、2p軌道には最大6個の電子が収容される。同様に3d, 4d等のd軌道には最大10個、4f等のf軌道には最大14個の電子が収容される。

以上をまとめると下表のようになる。

| 主量子数 (電子殻) | 方位量子数         | 磁気量子数 | 軌道名 | 収容できる電子数 |
| ---------- | ------------- | ----- | --- | -------- |
| 1（K殻）      | 0             | 0     | 1s  | 2        |
| 2（L殻）      | 0             | 0     | 2s  | 2        |
| 1          | 0, ±1         | 2p    | 6   |          |
| 3（M殻）      | 0             | 0     | 3s  | 2        |
| 1          | 0, ±1         | 3p    | 6   |          |
| 2          | 0, ±1, ±2     | 3d    | 10  |          |
| 4（N殻）      | 0             | 0     | 4s  | 2        |
| 1          | 0, ±1         | 4p    | 6   |          |
| 2          | 0, ±1, ±2     | 4d    | 10  |          |
| 3          | 0, ±1, ±2, ±3 | 4f    | 14  |          |

[水素](../Page/水素.md "wikilink")のような1電子系では電子の持つエネルギーは主量子数 n によってのみ決まるが、一般に原子は多電子系であり、電子同士の反発により各軌道のエネルギーに差が生じる。すなわち、方位量子数 *l* が大きくなるほど軌道は原子核から遠くに分布するため、電子間相互作用の影響が大きくなる。したがって、多電子系の電子軌道は 1s → 2s → 2p → 3s → 3p → 4s → 3d → 4p → 5s → 4d → 5p → 6s → 4f → … の順にエネルギーが高くなり、この順に電子が配置されていく（各軌道内での配置については「[フントの規則](../Page/フントの規則.md "wikilink")」を参照）。ただし、d電子の充填などではスピン間相互作用の寄与も入ってくるため、この規則に従わない場合もある。

## 電子配置によって決まる元素の性質

各元素の物理的・化学的な性質は、主に外側の軌道にどのように電子が充填されているかできまる。例えば、[ヘリウム](../Page/ヘリウム.md "wikilink")や[ネオン](../Page/ネオン.md "wikilink")などの[希ガス](https://ja.wikipedia.org/wiki/希ガス "wikilink")は最外殻の電子がいっぱいになって安定な電子殻になっていることで、[価電子](../Page/価電子.md "wikilink")による結合力がなく、いわゆる弱い[ファンデルワールス力](../Page/ファンデルワールス力.md "wikilink")しか結合力がなくなっていることがその性質をきめている。

[ナトリウム](https://ja.wikipedia.org/wiki/ナトリウム "wikilink")や[カリウム](../Page/カリウム.md "wikilink")と言った[アルカリ金属](https://ja.wikipedia.org/wiki/アルカリ金属 "wikilink")は希ガスに電子を1個追加した配置になっている。電子を1個取り出して、希ガスと同じ電子配置になった方が、中性でいるよりも安定である。このため、1価の陽[イオン](../Page/イオン.md "wikilink")になりやすい。同様の理由で[カルシウム](../Page/カルシウム.md "wikilink")や[ストロンチウム](../Page/ストロンチウム.md "wikilink")などの[アルカリ土類金属](https://ja.wikipedia.org/wiki/アルカリ土類金属 "wikilink")は2価の陽[イオン](../Page/イオン.md "wikilink")に、[フッ素](../Page/フッ素.md "wikilink")や[塩素](../Page/塩素.md "wikilink")などの[ハロゲン](https://ja.wikipedia.org/wiki/ハロゲン "wikilink")は1価の陰[イオン](../Page/イオン.md "wikilink")になりやすい。

### 内殻電子

**最外殻電子**（[価電子](../Page/価電子.md "wikilink")）の軌道より内側にあるものは**内殻電子** (Core electron) と呼ばれる。

通常は、原子間の[化学結合](../Page/化学結合.md "wikilink")や、[物性](https://ja.wikipedia.org/wiki/物性 "wikilink")に影響を与えることは少ないが、比較的浅い軌道の内殻電子（例：[ガリウム](../Page/ガリウム.md "wikilink")の3d電子）や、内殻励起のような現象では、内殻電子が重要な寄与をする場合がある。
