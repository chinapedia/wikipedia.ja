**アトラスオオカブト**とは、[昆虫綱](https://ja.wikipedia.org/wiki/昆虫綱 "wikilink")[甲虫目](https://ja.wikipedia.org/wiki/甲虫目 "wikilink")[カブトムシ亜科](../Page/カブトムシ亜科.md "wikilink")に属するカブトムシの1種。[フィリピン](https://ja.wikipedia.org/wiki/フィリピン "wikilink")・[インドネシア](../Page/インドネシア.md "wikilink")など[東南アジア](../Page/東南アジア.md "wikilink")の低地に分布する。[コーカサスオオカブト](https://ja.wikipedia.org/wiki/コーカサスオオカブト "wikilink")に類似しているが、やや小型で胸角が細く、頭角の突起がない（ただし頭角の先端がやや膨らみ、槍状になっている）。名前は、[ギリシア神話](../Page/ギリシア神話.md "wikilink")に登場する世界を支える巨人[アトラスが由来で](../Page/アトラース.md "wikilink")、[アトラス山脈](https://ja.wikipedia.org/wiki/アトラス山脈 "wikilink")に生息しているわけではない（アトラス山脈もアトラスオオカブトと同じく巨人アトラスに由来する名前である）。

個体数が多いことから、外国産カブトムシでありながら極めて安価で流通しており（小型のものは国産カブトムシより数百円高い程度）、[オオヒラタクワガタ](https://ja.wikipedia.org/wiki/オオヒラタクワガタ "wikilink")と並んで毎年大量に輸入されている。夏場は一般のペットショップ(主に熱帯魚等を扱う)やホームセンターでも普通に売られていることが多く、外国産昆虫の中では最も身近な存在であると言える。但し、大量に入荷されているため、大型個体と小型個体の選別をあまりしないため、雑に販売していることが多く、中には国産カブトムシより小さい個体もいる。

種間変異が著しく、雄の小型個体では角が殆どなく（**短角型**）、[亜種](https://ja.wikipedia.org/wiki/亜種 "wikilink")の判別も困難となる。また、そのような小型個体しか見られない亜種もある。

## 亜種

Nagai (2004) によれば、7亜種に分類されている。

| 和名        | 雄の体長         | 体形     | 頭角の突起 | 胸角の湾曲 |
| --------- | ------------ | ------ | ----- | ----- |
| アトラスオオカブト | 50mm - 100mm | 幅広い    | なし    | 強い    |
| フィリピンアトラス | 50mm - 110mm | 普通     | あり    | 強い    |
| ネシアアトラス   | 40mm - 100mm | 長細く直線的 | ややあり  | かなり弱い |
| インドアトラス   | 50mm - 90mm  | 胸部が幅広い | なし    | 弱い    |
| ペレンアトラス   | 50mm前後       | 不明     | 不明    | 不明    |
| ブトンアトラス   | 50mm前後       | 不明     | 不明    | 不明    |
| シムルエアトラス  | 50mm - 70mm  | 長細い    | 不明    | 不明    |
|           |              |        |       |       |

  - **アトラスオオカブト ssp. *atlas***
      -
        [スラウェシ島](https://ja.wikipedia.org/wiki/スラウェシ島 "wikilink")・[サンギヘ島](https://ja.wikipedia.org/wiki/サンギヘ島 "wikilink")・[シアウ島](https://ja.wikipedia.org/wiki/シアウ島 "wikilink")・[トギアン島](https://ja.wikipedia.org/wiki/トギアン島 "wikilink")に生息する。
        [ロンポバッタン山](https://ja.wikipedia.org/wiki/ロンポバッタン山 "wikilink")周辺には、やや小型でずんぐりとした体形のものが、その北方には大型個体の胸角の先端が反り返る個体がそれぞれ確認されている。

  - **フィリピンアトラス ssp. *hesperus***
      -
        [フィリピン諸島](https://ja.wikipedia.org/wiki/フィリピン諸島 "wikilink")の[ルソン島](https://ja.wikipedia.org/wiki/ルソン島 "wikilink")・[マリンドッケ島](https://ja.wikipedia.org/wiki/マリンドッケ島 "wikilink")・[ミンドロ島](https://ja.wikipedia.org/wiki/ミンドロ島 "wikilink")・[ロンブトン島](https://ja.wikipedia.org/wiki/ロンブトン島 "wikilink")・[レイテ島](https://ja.wikipedia.org/wiki/レイテ島 "wikilink")・[サマール島](https://ja.wikipedia.org/wiki/サマール島 "wikilink")・[ネグロス島](https://ja.wikipedia.org/wiki/ネグロス島 "wikilink")・[パナオン島](https://ja.wikipedia.org/wiki/パナオン島 "wikilink")・[ディナガット島](../Page/ディナガット島.md "wikilink")・[ミンダナオ島](https://ja.wikipedia.org/wiki/ミンダナオ島 "wikilink")・[パラワン島](https://ja.wikipedia.org/wiki/パラワン島 "wikilink")に生息する。

  - **ネシアアトラス ssp. *keyboh***
      -
        [マレー半島](https://ja.wikipedia.org/wiki/マレー半島 "wikilink")・[シンガポール島](https://ja.wikipedia.org/wiki/シンガポール島 "wikilink")・[シンケップ島](https://ja.wikipedia.org/wiki/シンケップ島 "wikilink")・[カリマンタン島](https://ja.wikipedia.org/wiki/カリマンタン島 "wikilink")・[ラウト島](https://ja.wikipedia.org/wiki/ラウト島 "wikilink")・[スマトラ島](../Page/スマトラ島.md "wikilink")・[ウェ島](https://ja.wikipedia.org/wiki/ウェ島 "wikilink")・[トアンク島](https://ja.wikipedia.org/wiki/トアンク島 "wikilink")・[シベルト島](https://ja.wikipedia.org/wiki/シベルト島 "wikilink")・[シポラ島](https://ja.wikipedia.org/wiki/シポラ島 "wikilink")・[ニアス島](https://ja.wikipedia.org/wiki/ニアス島 "wikilink")・[パガイウタラ島](https://ja.wikipedia.org/wiki/パガイウタラ島 "wikilink")・[タナマサ島](https://ja.wikipedia.org/wiki/タナマサ島 "wikilink")・[リンガ島に生息する](https://ja.wikipedia.org/wiki/リンガ島_\(インドネシア\) "wikilink")。
        記載者永井信二の友人[上田正樹](../Page/上田正樹.md "wikilink")のあだ名「キーボー」から。
