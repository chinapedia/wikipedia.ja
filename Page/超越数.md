## 超越数かどうかが未解決の例

\(e+\pi ,e-\pi ,e\pi,\frac{\pi}{e} ,{\pi}^{\pi} ,e^e ,\pi^e ,\pi^{\sqrt{2}} ,e^{\pi^2}\) などの円周率  やネイピア数  の大抵の和、積、べき乗は、有理数であるのか無理数であるのか超越的であるのか否かは証明されていない\[3\]。一方で、

\[\pi +e^{\pi} ,\pi e^{\pi} ,e^{\pi \sqrt{n}}\]（ は正の整数） は、超越的であると証明されている\[4\]\[5\]。

## 代数的独立性

複数の超越数が[代数的独立である例を挙げる](https://ja.wikipedia.org/wiki/体の拡大#代数性・超越性 "wikilink")。

  - \(\alpha_1,\ \alpha_2,\cdots,\ \alpha_n\) を有理数体上線形独立な代数的数としたとき、\(e^{\alpha_1},\ e^{\alpha_2},\cdots,\ e^{\alpha_n}\) は、代数的独立である。（リンデマン、ワイエルシュトラス）
  - \(\scriptstyle \pi,\ \Gamma(\tfrac{1}{4})\)、および、\(\scriptstyle \pi,\ \Gamma(\tfrac{1}{3})\) は、それぞれ代数的独立である。（チュドノフスキー）
  - \(\scriptstyle \pi,\ e^\pi,\ \Gamma(\tfrac{1}{4})\) は、代数的独立である。（ネステレンコ）
  - 代数的数 \(\scriptstyle \alpha_1,\cdots,\ \alpha_n\ (0 < |\alpha_i| < 1\ (i = 1,\cdots,\ n))\) を、\(\tfrac{\alpha_i}{\alpha_j}\) が[1の冪根](https://ja.wikipedia.org/wiki/1の冪根 "wikilink")ではないようにとったとき、\(\textstyle\sum\limits_{k=1}^\infty {\alpha_1}^{-k!},\cdots,\ \sum\limits_{k=1}^\infty {\alpha_n}^{-k!}\) は、代数的独立である。（西岡）
  - 相異なる 2 以上の整数 \(\scriptstyle d_1,\cdots,\ d_n\) と、\(\scriptstyle 0 <|\alpha| < 1\) である代数的数 \(\scriptstyle \alpha\) に対して、\(\textstyle\sum\limits_{k=1}^\infty \alpha^{{d_1}^k},\cdots,\ \sum\limits_{k=1}^\infty \alpha^{{d_n}^k}\) は、代数的独立である。（西岡）
      - 2次の無理数 \(\scriptstyle \omega >0\) と、相異なる代数的数 \(\scriptstyle \alpha_1,\cdots,\ \alpha_n\ (0 < |\alpha_i| < 1\ (i = 1,\cdots,\ n))\) に対して、\(\textstyle\sum\limits_{k=1}^\infty [h\omega]{\alpha_1}^{-k} ,\cdots,\ \sum\limits_{k=1}^\infty [k\omega]{\alpha_n}^{-k}\) は、代数的独立である。（マッサー (D. W. Masser)）
      - 相異なる 2 以上の整数 \(\scriptstyle d_1,\cdots,\ d_n\) と、\(\scriptstyle 0 <|\alpha| < 1\) である代数的数 \(\scriptstyle \alpha\) に対して、\(\scriptstyle\textstyle\prod\limits_{k=0}^\infty \bigl( 1-\alpha^{{d_1}^k} \bigr),\cdots,\ \prod\limits_{k=0}^\infty \bigl(1-\alpha^{{d_n}^k} \bigr)\) は、代数的独立である。（西岡）
          - 相異なる 2 以上の整数 \(\scriptstyle d_1,\cdots ,\ d_n\) に対して、\(\textstyle\sum\limits_{k=0}^\infty \frac{1}{{F_{2^k +1}}^{d_1}},\cdots,\ \sum\limits_{k=0}^\infty \frac{1}{{F_{2^k +1}}^{d_n}}\) は、代数的独立である。（西岡）
      - \(\scriptstyle m_1,\ m_2,\ m_3\) を少なくとも1つは偶数である正整数としたとき、\(\textstyle\sum\limits_{n=1}^\infty \frac{1}{{F_n}^{2m_1}},\ \sum\limits_{n=1}^\infty \frac{1}{{F_n}^{2m_2}}, \ \sum\limits_{n=1}^\infty \frac{1}{{F_n}^{2m_3}}\) は、代数的独立である。（エルスナー (C. Elsner)、下村、塩川）
    代数的独立性には、**シャヌエルの予想** (Schanuel's conjecture) と呼ばれる有名な予想があり、現在でも解決されていない。（\(n=2\) のときでさえも、未解決である。）
    シャヌエルの予想
    \[\alpha_1,\ \alpha_2,\cdots,\ \alpha_n\] を有理数体上線形独立な複素数としたとき、
    \[\mathbf{trans}.\deg_{\mathbb{Q}}\mathbb{Q}(\alpha_1,\ \alpha_2,\cdots,\ \alpha_n,\ e^{\alpha_1},\ e^{\alpha_2},\cdots,\ e^{\alpha_n})\ge n\]\[6\]。
    注意：\(\alpha_1,\ \alpha_2,\cdots,\ \alpha_n\) が代数的数のときは、[リンデマン＝ワイエルシュトラスの定理である](https://ja.wikipedia.org/wiki/リンデマンの定理 "wikilink")。
    この予想が解決すると、様々な数が代数的独立になることが知られている。 その中の一例を挙げる：
    以下の 17 個の数は、代数的独立である：
    \[e,\ e^{\pi},\ e^e,\ e^i,\ \pi,\ \pi^{\pi},\ \pi^e,\ \pi^i,\ 2^\pi,\ 2^e,\ 2^i,\ \log\pi,\ \log 2,\ \log 3,\ \log\log 2,\ (\log 2)^{\log 3},\ 2^{\sqrt{2}}\]。

注意：上記の数のうち、\(e^e,\ \pi^\pi,\ \pi^e,\ \pi^i,\ 2^{\pi},\ 2^e,\ \log\pi,\ \log\log 2,\ (\log 2)^{\log 3}\) は、それ自身が超越数であるかについても今のところ未解決である。

## マーラーの分類

複素数 \(\alpha\) に対して、

\[w_{n, H}(\alpha) = \min \Bigl\{ |f(\alpha)| \; \Bigl| \; f(z) = {\textstyle\sum\limits_{i=0}^n a_i z^i}, f(\alpha) \ne 0, |a_0|,\cdots, |a_n|\le H \Bigr\},\]
\(w_n(\alpha) = \limsup_{H\to\infty} \frac{-\log w_{n, H}(\alpha)} {\log H},\)
\(w(\alpha) = \limsup_{n\to\infty} \frac{w_n(\alpha)}{n}\) として、\(w(\alpha)\) を定める。このとき、\(\scriptstyle 0\le w(\alpha)\le\infty\) が成立する。

また、\(\scriptstyle \mu(\alpha) = \inf\{ n | w_n(\alpha) = \infty \}\) とする。ただし、\(\scriptstyle w(\alpha) < \infty\) の場合、\(\scriptstyle \mu(\alpha) = \infty\) とする。

この \(\scriptstyle w(\alpha),\ \mu(\alpha)\) を用いて、マーラーは、複素数を以下のように分類した。これを**マーラーの分類** (Mahler's classification) と呼ぶ。

  - \(\alpha\) は、*A* 数 (A-number) である。\(:\Leftrightarrow w(\alpha) = 0,\ \mu(\alpha) = \infty\)。
  - \(\alpha\) は、*S* 数 (S-number) である。\(:\Leftrightarrow 0 < w(\alpha) < \infty,\ \mu(\alpha) = \infty\)。
  - \(\alpha\) は、*T* 数 (T-number) である。\(:\Leftrightarrow w(\alpha) = \infty,\ \mu(\alpha) = \infty\)。
  - \(\alpha\) は、*U* 数 (U-number) である。\(:\Leftrightarrow w(\alpha) = \infty,\ \mu(\alpha) < \infty\)。

以下の性質がある。

  - *A* 数からなる集合、*S* 数からなる集合、*T* 数からなる集合、*U* 数からなる集合は、いずれも空集合ではない。
  - \(\alpha,\ \beta\) を代数的従属である複素数としたとき、\(\alpha,\ \beta\) は同じクラス(同じ分類の数)である。
  - *A* 数からなる集合は、代数的数全体の集合に等しい。
  - ほとんど全て\[7\]の複素数は、*S* 数である。
      - さらに、ほとんど全ての実数は、タイプ\[8\]1 の *S* 数であり、ほとんど全ての複素数は、タイプ 1/2 の *S* 数である。
  - 全ての[リウヴィル数](../Page/リウヴィル数.md "wikilink") は、*U* 数である。
  - 任意の正整数 \(n\) に対して、\(\mu(\alpha) = n\) を満たす *U* 数が存在する。

また、いくつかの具体的な超越数に対して、どのクラスに属するかについては、例えば、以下のことが知られている。

  - 自然対数の底  は、タイプ 1 の *S* 数である。

  - は、*U* 数ではない。

  - チャンパーノウン定数は、*S* 数である。

  - を 1 以外の正の有理数としたとき、\(\log r\) は、*U* 数ではない。

## 超越測度

超越数 \(\alpha\) に対して、\(T(\alpha, n, H)\) を、\(\scriptstyle n\ge 1,\ H\ge 1\) で定義された実数を値にとる関数とする。

次数が \(n\) 以下で、各係数の絶対値が \(H\) 以下である、0 以外の整数係数多項式に対して、

\[|f(\alpha)| \ge T(\alpha, n, H)\] が、任意の \(\scriptstyle n\ge 1,\ H\ge 1\) で成立するとき、\(T(\alpha, n, H)\) を \(\alpha\) の**超越測度** (transcendence measure) という。

マーラーの分類のところで与えられた、\(w_{n,H}(\alpha)\) は、超越測度の１つであり、その定義から、最良の評価を与えるものである。

## 歴史

[リウヴィルは](https://ja.wikipedia.org/wiki/ジョゼフ・リウヴィル "wikilink")、[1844年](../Page/1844年.md "wikilink")に超越数の最初の例を与えた（[リウヴィル数](../Page/リウヴィル数.md "wikilink")）。さらに[1873年](../Page/1873年.md "wikilink")に[シャルル・エルミート](../Page/シャルル・エルミート.md "wikilink")によって、自然対数の底 *e* が超越数であることが証明された。

[カントールは](https://ja.wikipedia.org/wiki/ゲオルグ・カントール "wikilink")[1874年](../Page/1874年.md "wikilink")に、実数全体の集合が非可算集合である一方で代数的数全体の集合が[可算集合](../Page/可算集合.md "wikilink")であることを示すことにより、ほとんどの実数や複素数は超越数であることを示した。

その後、[リンデマンは](https://ja.wikipedia.org/wiki/フェルディナント・フォン・リンデマン "wikilink")、[1882年](../Page/1882年.md "wikilink")に円周率  が超越数であることを証明した。これによって[古代ギリシア](../Page/古代ギリシア.md "wikilink")数学以来の難問であった[円積問題](../Page/円積問題.md "wikilink")が否定的に解かれた。また、彼は、任意の0でない代数的数 *a* に対する *e* が超越数であることも証明した（[リンデマンの定理](https://ja.wikipedia.org/wiki/リンデマンの定理 "wikilink")）。

[ヒルベルトは](../Page/ダフィット・ヒルベルト.md "wikilink")、[1900年](../Page/1900年.md "wikilink")にパリで行われた[国際数学者会議](../Page/国際数学者会議.md "wikilink")において、[ヒルベルトの23の問題](../Page/ヒルベルトの23の問題.md "wikilink")と呼ばれる23個の問題を提出したが、そのうちの 7番目の問題「*a* が 0 でも 1 でもない[代数的数](../Page/代数的数.md "wikilink")で、*b* が代数的無理数であるとき、*a* は超越数であるか」は、1934-1935年にゲルフォントとシュナイダーによって肯定的に解決された（[ゲルフォント＝シュナイダーの定理](https://ja.wikipedia.org/wiki/ゲルフォント＝シュナイダーの定理 "wikilink")）。

[1968年](../Page/1968年.md "wikilink")[ベイカーは](../Page/アラン・ベイカー.md "wikilink")、ゲルフォント＝シュナイダーの定理を含む、代数的数の1次形式の超越性および、1次形式の値が計算可能な下限で与えられることを証明した（[ベイカーの定理](https://ja.wikipedia.org/wiki/ベイカーの定理 "wikilink")を参照）。特に後者の結果は、[ディオファントス方程式](../Page/ディオファントス方程式.md "wikilink")の整数解の上限を求めるための基本的な定理として重要なものである。この功績により、彼は、[1970年](../Page/1970年.md "wikilink")、[フィールズ賞](../Page/フィールズ賞.md "wikilink")を受賞した。

[1996年](../Page/1996年.md "wikilink")、ネステレンコにより、長い間懸案であった、 と、（[ゲルフォントの定数](https://ja.wikipedia.org/wiki/ゲルフォントの定数 "wikilink")）の代数的独立性が証明された。
