**断熱過程**（だんねつかてい、）とは、外部との[熱](https://ja.wikipedia.org/wiki/熱 "wikilink")のやりとり（熱接触）がない状況で、[系をある状態から別の状態へと変化させる熱力学的な過程である](https://ja.wikipedia.org/wiki/系_\(自然科学\) "wikilink")。

## 概要

[エントロピー](../Page/エントロピー.md "wikilink")は断熱過程における不可逆性を特徴付ける状態量であり、系が断熱的に状態を遷移する前後でのエントロピーの変化は  である。等号が成り立つのは逆向きの遷移が可能な場合に限られ、不可逆な遷移ではエントロピーが増加する。これは[エントロピー増大則](https://ja.wikipedia.org/wiki/エントロピー増大則 "wikilink")と呼ばれ、[熱力学第二法則](../Page/熱力学第二法則.md "wikilink")の表現の一つである。

[熱力学第一法則](https://ja.wikipedia.org/wiki/熱力学第一法則 "wikilink")から、[閉鎖系](https://ja.wikipedia.org/wiki/閉鎖系 "wikilink")が状態を遷移する間に外部から流入する熱  は、系が外部に行う仕事  と、状態を遷移する前後での[内部エネルギー](../Page/内部エネルギー.md "wikilink")の変化  との間に  が成り立つ。断熱過程においては  0}} なので  となる。外部との熱の移動を遮断した状態で系が外部に仕事をすると内部エネルギーが減少し、逆に外部から系に仕事をすると内部エネルギーが増加することを意味している。多くの場合、内部エネルギーが増加すると温度は上昇し、内部エネルギーが減少すると温度は低下する。

特に流体の場合、圧縮することは外部から系に仕事をすることを意味し、温度が上昇する（**断熱圧縮**）。また、系が膨張して外部に仕事をすると、系の温度が低下する（**断熱膨張**）。ただし、膨張する際に仕事をしないようにもできて、これは**断熱自由膨張**と呼ばれる。

[Fire_piston.jpg](https://ja.wikipedia.org/wiki/File:Fire_piston.jpg "fig:Fire_piston.jpg")による発火実験の観察\]\] 瞬間的な圧縮では熱が殆ど移動しないため、断熱圧縮で温度が上昇する。この原理を利用した[発火装置として](../Page/発火法.md "wikilink")[圧気発火器](https://ja.wikipedia.org/wiki/圧気発火器 "wikilink")がある。この装置は密閉した[シリンダ](https://ja.wikipedia.org/wiki/シリンダ "wikilink")に[ピストン](../Page/ピストン.md "wikilink")が入った構造をしており、ピストンでシリンダ内の空気を急激に圧縮することで温度が上昇して[火口](../Page/火口.md "wikilink")に火をつける。東南アジアなどで用いられていたほか、アクリルやガラスなど透明の筒で内部の発火を確認できる実験観察用の圧気発火器も存在する。[ディーゼルエンジン](../Page/ディーゼルエンジン.md "wikilink")では燃料の点火に用いている。[大気圏(再)突入で宇宙機が加熱されたり隕石などが燃え尽きる現象も周囲の空気の断熱圧縮による温度上昇の影響が大きく](../Page/大気圏再突入.md "wikilink")、しばしば用いられる空気との[摩擦](../Page/摩擦.md "wikilink")という説明は間違いである。

## 準静的断熱過程

[準静的過程](../Page/準静的過程.md "wikilink")では系が常に平衡にあるとみなされるため、系の変化の無限小の極限をとることができて[状態量](../Page/状態量.md "wikilink")の[微分](../Page/微分.md "wikilink")を考えることができる。特にエントロピーの微分  は無限小の過程で系に流入する無限小の熱  と  で関係付けられる。準静的な断熱過程では  であり、積分によりエントロピーの変化もない。したがって準静的断熱過程は[可逆](../Page/可逆.md "wikilink")である。このため[等エントロピー過程](https://ja.wikipedia.org/wiki/等エントロピー過程 "wikilink")と呼ばれることもある。

準静的断熱過程と準静的[等温過程](../Page/等温過程.md "wikilink")からなる[理論サイクルである](../Page/熱機関の理論サイクル.md "wikilink")[カルノーサイクル](../Page/カルノーサイクル.md "wikilink")は、[熱機関](https://ja.wikipedia.org/wiki/熱機関 "wikilink")の[熱効率](../Page/熱効率.md "wikilink")の理論的な上限を与え、熱力学の理論構成によっては[熱力学温度](../Page/熱力学温度.md "wikilink")やエントロピーの定義とも密接に関わる重要な役割を果たす。

## 断熱火炎温度

[アセチレン](../Page/アセチレン.md "wikilink")の燃焼

  -
    C<sub>2</sub>H<sub>2</sub> + (5/2) O<sub>2</sub> → 2 CO<sub>2</sub> + H<sub>2</sub>O

を考える。アセチレン1 molに対し、[二酸化炭素](../Page/二酸化炭素.md "wikilink")2 mol、[水](../Page/水.md "wikilink")1 molが生成する。燃焼前のガスの温度を*T*<sub>0</sub> = 25 °C = 298.15 Kとすると、[標準燃焼熱](https://ja.wikipedia.org/wiki/標準燃焼熱 "wikilink")は Δ<sub>c</sub>*H* (*T*<sub>0</sub>) = -1256.8 kJ/molである。

定圧熱容量*C<sub>P</sub>* は温度に対し[線型](https://ja.wikipedia.org/wiki/線型 "wikilink")に変化し、*C<sub>P</sub>* = *a* + *b T* と表せると仮定する。さらに系外に熱は出ていかないとみなすと、

\[\Delta_\text{c} H(T_0) + \int_{T_0}^T C_P \mathrm{d}T = 0\] となるから、

\[\frac{b}{2}T^2 + aT + c = 0, \quad c = \Delta_\text{c} H(T_0) - \frac{b}{2}T_0^2 - aT_0\]

\[\therefore T = \frac{-a+\sqrt{D}}{b}, \quad D = a^2-2bc\] を得る。したがって、係数*a* , *b* が分かれば炎の温度*T* が計算できる。

ここで係数は下記の表から、*C<sub>P</sub>* = 2*C<sub>P</sub>*(CO<sub>2</sub>(g)) + *C<sub>P</sub>*(H<sub>2</sub>O(g)) の関係を用いて求める。実際に計算すると、

  -
    *a* = 2×*a*(CO<sub>2</sub>(g)) + *a*(H<sub>2</sub>O(g)) = 82.36 JK<sup>-1</sup>mol<sup>-1</sup>
    *b* = 2×*b*(CO<sub>2</sub>(g)) + *b*(H<sub>2</sub>O(g)) = 0.09661 JK<sup>-2</sup>mol<sup>-1</sup>
    *c* = Δ<sub>c</sub>*H*(*T*<sub>0</sub>) - (*b*/2)*T*<sub>0</sub><sup>2</sup> - *aT*<sub>0</sub> = -1285691 Jmol<sup>-1</sup>
    \= 505.18 JK<sup>-1</sup>mol<sup>-1</sup>

より、

  -
    *T* = 4377 K ≒ 4100℃

つまりアセチレンの炎は約4100 °Cであると分かる。
