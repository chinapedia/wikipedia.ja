**タンクレード・ド・オートヴィル**（Tancrède de Hauteville, [990年](../Page/990年.md "wikilink")頃 - [1041年](../Page/1041年.md "wikilink")頃）は、[オートヴィル（アルタヴィッラ）家の始祖](../Page/オートヴィル朝.md "wikilink")。イタリア語名では**タンクレーディ・ダルタヴィッラ**（Tancredi d'Altavilla）となる。

## 生涯

タンクレードの生涯についてはほとんど知られていない。[コタンタン半島](https://ja.wikipedia.org/wiki/コタンタン半島 "wikilink")の街[クタンス](https://ja.wikipedia.org/wiki/クタンス "wikilink")の[ノルマン系の一貴族であった彼は](../Page/ノルマン人.md "wikilink")、おそらく[オートヴィル＝ラ＝ギシャール](https://ja.wikipedia.org/wiki/オートヴィル＝ラ＝ギシャール "wikilink")（Hauteville-la-Guichard）の村を所有していた。タンクレードは2人の妻によって、12人もの息子を残し、その多くが[南イタリア](../Page/南イタリア.md "wikilink")において重要な諸侯となった。

[1010年](../Page/1010年.md "wikilink")頃、ミュリエル（Murielle）との結婚では以下の子をもうけた（／の後はイタリア名）。

  - 鉄腕ギヨーム／グリエルモ（Guillaume Bras-de-Fer / Guglielmo Braccio di Ferro） - [プッリャ伯](../Page/プッリャとカラブリアの伯爵と公爵一覧.md "wikilink")（在位：1042年 - 1046年）
  - ベアトリス（1010年頃 - 1101年頃） - [ウー伯](https://ja.wikipedia.org/wiki/ウー伯 "wikilink")ロベールの息子アルマン（[ノルマンディー家](https://ja.wikipedia.org/wiki/ノルマンディー家 "wikilink")、モンテスカリオーゾ伯家祖）と結婚
  - ドロゴン／ドロゴーネ（Drogon / Drogone） - プッリャ伯（在位：1046年 - 1051年）
  - オンフロワ／ウンフレード（Onfroi / Umfredo） - プッリャ伯（在位：1051年 - 1057年）
  - ゴドフロワ／ゴッフレード（Godefroi / Goffredo） - ロリテッロ伯
  - セルロン／セルローネ（Serlon / Serlone） - ノルマンディーに残留、息子セルローネ2世（1072年没）は叔父[ルッジェーロのもとで戦う](../Page/ルッジェーロ1世_\(シチリア伯\).md "wikilink")\[1\]。

[1025年](../Page/1025年.md "wikilink")にミュリエルが死ぬと、タンクレードはフレデゼンド（Frédésende）と2度目の結婚を行い、以下の子をもうけた。

  - ロベール・ギスカール／[ロベルト・イル・グイスカルド](../Page/ロベルト・イル・グイスカルド.md "wikilink")（Robert Guiscard / Roberto il Guiscardo） - プッリャ伯、のちにプッリャ公、カラブリア公、シチリア公。
  - モジェ／マルジェーロ（Mauger / Malgero）
  - ギヨーム／グリエルモ（Guillaume / Guglielmo） - プリンチパート伯
  - アルフレッド（Alfred） - ノルマンディーに残留
  - ユベール（Hubert） - ノルマンディーに残留
  - タンクレード（Tancrède） - ノルマンディーに残留
  - エマ（Emma）
  - フレデゼンド（Frédésende） - アヴェルサ伯・カプア公リッカルド1世と結婚\[2\]
  - ロジェ／[ルッジェーロ](../Page/ルッジェーロ1世_\(シチリア伯\).md "wikilink")（Roger / Ruggero I） - 大伯爵（*Gran Conte*）と呼ばれたシチリア伯

残る幾人かの女子の名前は知られていない。
