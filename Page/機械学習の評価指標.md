## 回帰に対する評価指標

以下のものがある\[1\]\[2\]

| 略称      | 名称（英語）                              | 名称（日本語）   | 定義                                                                                                                 |
| ------- | ----------------------------------- | --------- | ------------------------------------------------------------------------------------------------------------------ |
| RMSE    | Root Mean Squared Error             | 平均平方二乗誤差  | \(\sqrt{{1\over N}\sum_{i=1}^N(y_i-\hat{y}_i)^2}\)                                                                 |
| MAE     | Mean Absolute Error                 | 平均絶対誤差    | \({1\over N}\sum_{i=1}^N|y_i-\hat{y}_i|\)                                                                          |
| MAPE    | Mean Absolute Percentage Error      | 平均絶対誤差率   | \({1\over N}\sum_{i=1}^N{y_i\over |y_i-\hat{y}_i|}\)                                                               |
| RMSPE   | Root Mean Squared Persentage Error  | 平均平方二乗誤差率 | \(\sqrt{{1\over N}\sum_{i=1}^N\left({y_i\over y_i-\hat{y}_i}\right)^2}\)                                           |
| RMSLE   | Root Mean Squared Logarithmic Error |           | \(\sqrt{{1\over N}\sum_{i=1}^N(\log(y_i+1)-\log(\hat{y}_i+1))^2}\)                                                 |
| \(R^2\) |                                     | 決定係数      | \(1-{\sum_{i=1}^N (y_i-\hat{y}_i)^2 \over \sum_{i=1}^N (y_i-\bar{y})^2}\)、ここで\(\bar{y}={1\over N}\sum_{i=1}^Ny_i\) |
