**日本学校農業クラブ連盟**（にほんがっこうのうぎょうくらぶれんめい、**** = ）は、[東京都](https://ja.wikipedia.org/wiki/東京都 "wikilink")に[事務局](https://ja.wikipedia.org/wiki/事務局 "wikilink")をおく、[農業科](https://ja.wikipedia.org/wiki/農業科 "wikilink")や[総合学科](https://ja.wikipedia.org/wiki/総合学科 "wikilink")をもつ[高等学校](https://ja.wikipedia.org/wiki/高等学校 "wikilink")に属する[生徒](https://ja.wikipedia.org/wiki/生徒 "wikilink")の[団体](../Page/団体.md "wikilink")である。略して、**農業クラブ**、**農ク**と生徒及び関係者から呼ばれている。

## 歴史

[戦後](../Page/戦後.md "wikilink")、[学制改革](../Page/学制改革.md "wikilink")に伴い農業高等学校ごとに学校農業クラブの組織づくりが進められ、都道府県単位での組織も成立するようになった。元々[アメリカ合衆国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")には、()があり、これが手本にされた。

[1950年](../Page/1950年.md "wikilink")春に[東京都立園芸高等学校](https://ja.wikipedia.org/wiki/東京都立園芸高等学校 "wikilink")（当時：東京都立新制園芸高等学校）が設立し、全国組織が成立した。同年11月2日には東京都の[日比谷公会堂](https://ja.wikipedia.org/wiki/日比谷公会堂 "wikilink")で  の結成大会が行われた。

## プロジェクト

[1908年](../Page/1908年.md "wikilink")、米国マサチューセッツ州の農学校で、学校で学んだことを家庭の農業に実際に適用して学習するために**ホームプロジェクト**という手法が採用された。そもそも、農業クラブとは、この農家子弟の学習活動のために作られたものである。現在の  でもホームプロジェクト法を主として学習活動を行っている。

戦後日本の農業教育においてもこれに倣い、ホームプロジェクト法が導入された。教職員が各家庭を回って、親とともに農家の子弟に農業の技術を指導していく、体験を重視する手法である。ただ、今の日本ではこんな学校はだめだということで、取り壊し作業に取りかかっている。

## 全国大会

年に一度、全国大会が行われる。様々なプロジェクト（研究）の発表や、測量競技や農業鑑定競技などが行われ、各県、ブロック代表がその技術を競うものとなっている。

  - 全国大会で行なわれる競技

:; 意見発表会

:: 農業や農業学習に関する自分の考え等を主張し合う競技で、形式はスピーチで、決められた時間内に自分の意見や主張、経験して得たこと等を発表する。競技の内容自体はシンプルなものだが、発表者は発表内容は元より、高度なスピーチ能力から発表後にある質疑応答など、高レベルな発表が要求される。

:; プロジェクト発表会

:: プロジェクト学習の成果を発表する競技である。ここで述べられているプロジェクト学習は、学校ごとに『卒業研究』『卒論』『課題研究』『プロ研』など、名称が異なるがスクールプロジェクトに基づいている。内容は、いわゆるプレゼンテーションの形式で、ジャンルは生物学的な分野から社会学・農業政策的な分野まで幅広い。

:; 農業鑑定競技会

:: 専門分野で学習する機器・材料・生物などの名称や使い方、計算問題などが出題される。問題は農業、園芸、畜産、生活科学、農業土木、林業、食品科学科、造園、農業機械に分かれている。

:; 平板測量競技会

:: 平板測量の早さ・正確さを争う競技である。

:; 農業情報処理競技会（公開競技）

:: コンピューターを使った情報処理の能力を競う競技である。

:; 家畜審査競技会（公開競技）

:: 乳用牛（ホルスタイン）・肉用牛（黒毛和種）・豚（[ランドレース種](https://ja.wikipedia.org/wiki/ランドレース種 "wikilink")と、その子豚数匹）の外貌等の審査をする競技。[1990年](https://ja.wikipedia.org/wiki/1990年 "wikilink")の東京大会では開催されなかった。乳用牛の部で畜産科（農業土木科）以外の生徒が優勝した事もある。

:; フラワーデザイン競技会（公開競技）

:: 個人またはチームで、園芸作物の知識や栽培技術、活用方法、フラワーデザインに関する学習の成果を競う競技である。

:; クラブ員代表者会議

        全国の単位クラブの会長（代表者）が参加して行われる会議。毎年3つのテーマについて話し合いを行う。

## 旗と歌

### 旗

「  の歌」の冒頭でも歌われている通り、「みのる稲穂に 富士と鳩」をシンボルマークとしている。マークにおいて、鳩は平和を、富士山は日本を、稲穂は日本の農業・主食を表している。

### 歌

「*'  の歌*'」（[1951年](https://ja.wikipedia.org/wiki/1951年 "wikilink")制作、作詞：吉沢義之\[1\]、作曲：[堀内敬三](https://ja.wikipedia.org/wiki/堀内敬三 "wikilink")）は[農業高等学校](https://ja.wikipedia.org/wiki/農業高等学校 "wikilink")など加盟校では校歌と同等あるいはそれ以上に斉唱することが多く、卒業生はこの歌を歌える者が多い。

また、 踊りと称される踊りも存在している。復活版として第44回  大会（愛知大会）にてメイン会場であった[名古屋国際会議場](https://ja.wikipedia.org/wiki/名古屋国際会議場 "wikilink")センチュリーホール（3,000名収容）にて、[愛知県立佐屋高等学校](https://ja.wikipedia.org/wiki/愛知県立佐屋高等学校 "wikilink")の農業クラブ員にて構成されたおよそ20名にて踊りが披露された。

[2019年](../Page/2019年.md "wikilink")度前期放送の[NHK](https://ja.wikipedia.org/wiki/日本放送協会 "wikilink")[連続テレビ小説](../Page/連続テレビ小説.md "wikilink")『[なつぞら](https://ja.wikipedia.org/wiki/なつぞら "wikilink")』では、ヒロイン・奥原なつ（演・[広瀬すず](https://ja.wikipedia.org/wiki/広瀬すず "wikilink")）をはじめとした農高生が劇中で歌唱されて注目を集め、第78話では会社の余興でなつが独唱しているシーンも描かれた。
[NHK札幌放送局](../Page/NHK札幌放送局.md "wikilink")では「みんなでつくる\!『FFJの歌』」として視聴者投稿の歌唱動画をつなぎ合わせてミュージックビデオを制作、2019年7月より同局ウェブサイトや動画サイトで公開され、北海道内のローカル番組で放送\[2\]\[3\]。2019年9月にはNHK[プレマップ](https://ja.wikipedia.org/wiki/プレマップ "wikilink")で1ヶ月限定の全国放送を実施している\[4\]。

[2000年](../Page/2000年.md "wikilink")、 50周年を記念して新しいシンボルマークと[さだまさし](../Page/さだまさし.md "wikilink")補作詞・作曲の新テーマソング「**美しい未来へ〜Future Farmers of Japan〜**」が発表された。
