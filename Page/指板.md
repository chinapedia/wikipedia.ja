[Gfingerboard.JPG](https://ja.wikipedia.org/wiki/File:Gfingerboard.JPG "fig:Gfingerboard.JPG") [Violinfingerboard.JPG](https://ja.wikipedia.org/wiki/File:Violinfingerboard.JPG "fig:Violinfingerboard.JPG") **指板**（しばん、フィンガーボード、フレットボードとも）は弦楽器の一部分である。ネックの表面に貼られた薄く長い木片で、この上に弦が張られる。演奏時に弦を指板に押さえつける位置を変えることにより、振動長を変え、結果的に音程が変わる。

## フレット

[trastes.jpg](https://ja.wikipedia.org/wiki/File:trastes.jpg "fig:trastes.jpg") 指板には[フレット](../Page/フレット.md "wikilink")が打たれる場合がある。これは指板上の弦を押さえる位置に、弦に垂直に取り付ける堅い突起である。フレットによって演奏者は弦の同じ位置を押さえることが出来、また、指だけで押さえた場合のような振動の減衰を抑えられる。[弓を使う楽器では通常フレットは打たれない](https://ja.wikipedia.org/wiki/弓_\(楽器\) "wikilink")。これらの楽器では弓でこすり続けることにより持続的に音を出すため、振動の減衰は問題にはならない。何番目のフレットであるかを容易に確認するため、指板上にはインレイでマークされることが多い。

長期間の使用に伴いフレットは摩耗し、その結果「ビビり」が発生し、楽器は使い物にならなくなる。この問題の解消のためには、フレットの打ち直しが必要となる。

## 素材

[擦弦楽器](https://ja.wikipedia.org/wiki/擦弦楽器 "wikilink")（[ヴァイオリン](../Page/ヴァイオリン.md "wikilink")、[ヴィオラ](../Page/ヴィオラ.md "wikilink")、[チェロ](../Page/チェロ.md "wikilink")、[コントラバス](../Page/コントラバス.md "wikilink")）では主に[エボニー](https://ja.wikipedia.org/wiki/コクタン "wikilink")、[ローズウッド](../Page/シタン.md "wikilink")、その他の[広葉樹](https://ja.wikipedia.org/wiki/広葉樹 "wikilink")が使われる。[ギター](../Page/ギター.md "wikilink")では[メイプルの指板](../Page/サトウカエデ.md "wikilink")（貼り/ワンピース）やローズウッドのワンピース指板等も用いられる。近年の先進的ギター製造者は[炭素繊維強化プラスチック](https://ja.wikipedia.org/wiki/炭素繊維強化プラスチック "wikilink")等の非木材のより軽い素材を指板に使用することもある\[1\]。

## パラメーター

[Fingerboard_scheme.svg](https://ja.wikipedia.org/wiki/File:Fingerboard_scheme.svg "fig:Fingerboard_scheme.svg")

一般的に、指板は長い長方形の板である。ギター、マンドリン、ウクレレやこれらに類する[撥弦楽器](https://ja.wikipedia.org/wiki/撥弦楽器 "wikilink")では指板は広く平らだが、指板の幅に対して半径の大きい円筒面あるいは円錐面であるものもある。楽器仕様に示される「指板半径」はナット部分での[曲率](../Page/曲率.md "wikilink")半径である。

多くの[擦弦楽器](https://ja.wikipedia.org/wiki/擦弦楽器 "wikilink")では個々の弦に対する弓へのゆとりを与えるため、目に見てわかるほどのカーブが指板、ナット、ブリッジに付けられている

指板の長さ、幅、厚さおよび密度は楽器の音色に影響を与えうる。

多くの指板は以下のようなパラメータによって完全に説明される:

  - *w<sub>1</sub>* — ナット部での幅（ヘッド側）
  - *w<sub>2</sub>* — スケール半分の位置での幅（フレット付き楽器の場合は、通常12フレット）
  - *h<sub>1</sub>* — ナット部での側面の高さ（厚さ）
  - *h<sub>2</sub>* — スケール半分の位置での側面の高さ（厚さ）
  - *r* — 半径（一定とは限らない）

### 指板半径

[Fretboard-radii-graph.svg](https://ja.wikipedia.org/wiki/File:Fretboard-radii-graph.svg "fig:Fretboard-radii-graph.svg") 半径*r*と、その指板上での変化により、指板は次の4種類に分類される:

|   |         |                                                                                                              |                                         |
| - | ------- | ------------------------------------------------------------------------------------------------------------ | --------------------------------------- |
| 1 | フラット    | ナット側、ブリッジ側の両方で平らである。すべての弦は同一平面上にあり、楽器には指板半径が無い（言い方を変えれば半径は無限である）。                                            | \(r = \infty\)                          |
| 2 | シリンドリカル | 指板は一定の半径を持ち、指板、ナット、ブリッジの半径は同一である（厳密に言うと、指板の半径はブリッジ、ナットより僅かに小さい）。                                             | \(r = r_1 = r_2 = const\)               |
| 3 | コニカル    | 指板上の半径が変化する。普通は\(r_1\)から\(r_2\)に向かってリニアに増える。これはコンパウンド・ラジウスと呼ばれることもある\[2\]。ナットとブリッジの両方がカーブしているが、ナットの方が半径は小さい。 | \(r(x) = r_1 + \frac{x}{l}(r_2 - r_1)\) |
| 4 | コンパウンド  | 厳密なコニカルでなく、カーブしたナットと平らなブリッジを持つ。指板上のすべての部分がカーブしているが、厳密な円錐形ではない。                                               | \(r(x) = f(x)\)、通常は \(r(l) = \infty\)   |

ノート:

  - \(l\)はスケール。
  - \(x\)は指板上を 0（ナット側）から \(l\)まで変化する、指板上の位置をあらわす指標。
  - \(r(x)\)は\(x\)における指板上の半径。
  - \(f(x)\)は非線形関数。

[クラシック・ギター](../Page/クラシック・ギター.md "wikilink")やいくつかの[12弦ギター](../Page/12弦ギター.md "wikilink")、また[スティール弦アコースティックギター](../Page/スティール弦アコースティックギター.md "wikilink")のうち僅かな機種ではフラット指板が使用されている。その他のほとんどのギターでは何らかのカーブがかかった指板だが、最近の5弦ベース、6弦ベースにはフラット指板の物もある。

ギターでは小さな半径（9-10"）がコード弾きやリズムギターに向き、大きな半径（12"-16"、あるいはそれ以上）は速弾きのソロに向くと言われている。コニカル、コンパウンド指板はこの両特性を求めた物である。ナット側の半径が小さくなっているのでコードを押さえやすく、半径の大きなブリッジ側ではソロプレイをしやすく、また「フレット・アウト（[チョーキング](../Page/チョーキング.md "wikilink")中に高フレット側に指がずれてしまう）」を防ぐことが出来る。

[擦弦楽器](https://ja.wikipedia.org/wiki/擦弦楽器 "wikilink")では隣接する弦の[重音奏法](../Page/重音奏法.md "wikilink")を可能にするため、カーブした指板を使用する傾向がある。現代的な[ヴァイオリン属](../Page/ヴァイオリン属.md "wikilink")や[コントラバス](../Page/コントラバス.md "wikilink")では特に強くカーブしている。しかし、若干の古典的な擦弦楽器ではフラットである。
