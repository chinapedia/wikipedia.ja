**旧制中等教育学校の一覧**（きゅうせいちゅうとうきょういくがっこうのいちらん）とは、[学制改革](../Page/学制改革.md "wikilink")前（[第二次世界大戦](https://ja.wikipedia.org/wiki/第二次世界大戦 "wikilink")前）の[旧学制の中等教育学校を各](../Page/旧制中等教育学校.md "wikilink")[都道府県](../Page/都道府県.md "wikilink")ごとにまとめた一覧。

後身の[新制高校への移行の変遷](https://ja.wikipedia.org/wiki/高等学校 "wikilink")・系譜についても合わせてまとめた。

なお、[ナンバースクール](../Page/ナンバースクール.md "wikilink")については[旧制中等学校・新制高校のナンバースクール一覧](../Page/旧制中等学校・新制高校のナンバースクール一覧.md "wikilink")（東京は[ナンバースクール (東京都)](https://ja.wikipedia.org/wiki/ナンバースクール_\(東京都\) "wikilink")）を参照のこと。

## 旧制中等学校と新制高等学校（変遷と系譜）

当一覧での公立・私立の区分は現在の学校の区分を基準にしている。

※一覧中の略称例　（○○：自治体名、固有名詞／△：数字）

  - ○○第△中学校　→　○○△中　（第△中学校　→　△中、○○中学校　→　○○中）
  - ○○第△高等女学校　→　○○第△高女　（第△高等女学校　→　第△高女、○○高等女学校　→　○○高女）
  - ○○第△商業学校　→　○○△商　（第△商業学校　→　△商、○○商業学校　→　○○商）
  - ○○第△高等学校　→　○○△高　（第△高等学校　→　△高、○○高等学校　→　○○高）
  - ○○第△新制高等学校　→　○○第△新制高　（第△新制高等学校　→　第△新制高、○○新制高等学校　→　○○新制高）
  - ○○第△女子高等学校　→　○○第△女子高　（第△女子高等学校　→　第△女子高、○○女子高等学校　→　○○女子高）
  - ○○第△商業高等学校　→　○○第△商業高　（第△商業高等学校　→　第△商業高、○○商業高等学校　→　○○商業高）

  - ○○工業高等学校　→　○○工業高
