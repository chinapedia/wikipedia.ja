**ガムラ・ウプサラ**（**ガムラ・ウップサーラ**とも）(*Gamla Uppsala*)（「Old Uppsala、古ウプサラ」の意）は、[スウェーデン](https://ja.wikipedia.org/wiki/スウェーデン "wikilink")の[ウプサラ](https://ja.wikipedia.org/wiki/ウプサラ "wikilink")郊外にある町の名である。 [1991年](https://ja.wikipedia.org/wiki/1991年 "wikilink")の人口は16,231人であった。

[3世紀](../Page/3世紀.md "wikilink")から[4世紀](../Page/4世紀.md "wikilink")にかけて、そこは宗教面、経済面、政治面で重要な中心地であった。<ref name="Hadenius1">Hadenius, Stig; Nilsson, Torbjörn & Åselius, Gunnar. (1996年). *Sveriges historia*. Centraltryckeriet, Borås. ISBN 91-34-51-857-6 p. 83:

|  |                                                                                                                      |  |                                                                               |
|  | -------------------------------------------------------------------------------------------------------------------- |  | ----------------------------------------------------------------------------- |
|  | **Uppsala**, från 200- och 300-talen ett betydande ekonomiskt, religiöst och politiskt centrum i Mälardalen \[...\]. |  | *英訳の和訳*: **ウプサラ**は、3世紀から4世紀にかけて、メーラレン(*Mälaren*)盆地において経済的・宗教的・政治的に重要な中心地であった。 |

</ref>

また、古い時代の文書の伝えるところでは、先史時代のガムラ・ウプサラは、神話伝説時代の[ユングリング王朝に属するスウェーデン王が居住する場所として](https://ja.wikipedia.org/wiki/ユングリング家 "wikilink")、すでに北欧では有名だった。<ref name="Nationalencyklopedin1">*[ナショナルエンスィークロペディーン](https://ja.wikipedia.org/wiki/:en:Nationalencyklopedin "wikilink")*の「*Gamla Uppsala*」の記事の細目「*Historia*」(1992年):

|  |                                                                                                                                                 |  |                                                                                            |
|  | ----------------------------------------------------------------------------------------------------------------------------------------------- |  | ------------------------------------------------------------------------------------------ |
|  | Tidiga skriftliga källor visar att G. redan under forntiden var vittberömt i Norden som säte för sveakungarna av den mytomspunna Ynglingaätten. |  | *英訳の和訳*: 初期に書かれた文献により、先史時代という早い時期、G.は伝説的なユングリング王朝のスウェーデン王の居住地として北欧の国々では広く有名であったことが明らかとなった。 |

</ref>

実際に、[北欧](../Page/北欧.md "wikilink")についての最古記録、例えば『[ユングリンガ・タル](https://ja.wikipedia.org/wiki/ユングリンガ・タル "wikilink")』（[:en:Ynglingatal](https://ja.wikipedia.org/wiki/:en:Ynglingatal "wikilink")）、『[Westrogothic law](https://ja.wikipedia.org/wiki/:en:Västgötalagen "wikilink")』\[1\]、『[ギュータサガ](https://ja.wikipedia.org/wiki/ギュータサガ "wikilink")』([:en:Gutasaga](https://ja.wikipedia.org/wiki/:en:Gutasaga "wikilink"))\[2\]は、スウェーデンの王を指して「ウプサラの王」といっている。<ref name="MuseumofNationalAntiquities">*[Gamla Uppsala](http://www.historiska.se/histvarld/artikel.asp?id=10358)*（*[スウェーデン国立歴史博物館](https://ja.wikipedia.org/wiki/スウェーデン国立歴史博物館 "wikilink")のオフィシャルサイト内*より2007年1月19日に検索された記事）:

|  |                                                                                                                              |  |                                                                                     |
|  | ---------------------------------------------------------------------------------------------------------------------------- |  | ----------------------------------------------------------------------------------- |
|  | Svearikets kung omtalas som kungen i Uppsala bl a i Ynglingatal, Gutasagan och Äldre Västgötalagens bihang om gränsdragning. |  | *英訳の和訳*: ユングリンガ・タルで、ギュータサガで、そして国境の確定に関するWestrogothic法の付表で、スウェーデン王は必ずウプサラの王として言及される。 |

</ref>

[中世](../Page/中世.md "wikilink")の時代、そこは[ウップランド](https://ja.wikipedia.org/wiki/ウップランド地方 "wikilink")(*Uppland*)で最も大きな村落であった。その東部が、初期の頃にスウェーデン王家の地所の複合体の核を構成していただろうと考えられる。 そしていわゆる**[Uppsala öd](https://ja.wikipedia.org/wiki/:en:Uppsala_öd "wikilink")**は、**kungsgården**、つまり国王の地所から成る地域であった。<ref name="Nationalencyklopedin2">*[ナショナルエンスィークロペディーン](https://ja.wikipedia.org/wiki/:en:Nationalencyklopedin "wikilink")*の「*Gamla Uppsala*」の記事の細目「*Historia*」(1992年):

|  |                                                                                                                                                                                               |  |                                                                                                              |
|  | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |  | ------------------------------------------------------------------------------------------------------------ |
|  | Under medeltiden var G. Upplands största by, vars östra del ursprungligen torde ha bildat kärnan i det s.k. Uppsala öd, kronans godskomplex, och vars västra del utgjorde själva kungsgården. |  | *英訳の和訳*: 中世の時代、G.はUpplandで最も大きな村であり、その東部の一部は初期は君主の地所の複合体、いわゆるUppsala ödの中心部であったはずで、また西部はその地域自体が国王の地所から成っていた。 |

</ref>

そこはまた、先史時代から中世にかけて、2月下旬から3月上旬に開催される総会「全スウェーデン集会」([:en:Thing of all Swedes](https://ja.wikipedia.org/wiki/:en:Thing_of_all_Swedes "wikilink"))の場所でもあった。\[3\] 集会は、**[Disting](https://ja.wikipedia.org/wiki/:en:Disting "wikilink")**と呼ばれる盛大な市場とともに開催された。また、古い神々を信仰する人々の祝賀の祭は**[Dísablót](https://ja.wikipedia.org/wiki/:en:Dísablót "wikilink")**（女神への供犠）と呼ばれた。\[4\]\[5\]

Upplandの法律([:en:Law of Uppland](https://ja.wikipedia.org/wiki/:en:Law_of_Uppland "wikilink"))の伝えるところでは、王が夏の間の**[leidang](https://ja.wikipedia.org/wiki/:en:leidang "wikilink")**（税として労働を提供するシステム）として戦争のために人々を召集すると宣言したのは、この集会でのことだった。 また集会において船の乗組員、漕ぎ手、指揮官が決定された。\[6\]

この場所が古い神々への信仰の中心地であったので、[キリスト教](../Page/キリスト教.md "wikilink")化の進む中、[1164年](../Page/1164年.md "wikilink")に最初のスウェーデン[大司教](https://ja.wikipedia.org/wiki/大司教 "wikilink")座が置かれた。\[7\]
