**クハ6形**は、かつて[日本国有鉄道](../Page/日本国有鉄道.md "wikilink")およびその前身である[鉄道省](https://ja.wikipedia.org/wiki/鉄道省 "wikilink")等に在籍した木造[直流用](../Page/直流電化.md "wikilink")[電車](../Page/電車.md "wikilink")である。

## 概要

[1936年](../Page/1936年.md "wikilink")（昭和11年）、国に買収され鉄道省に編入された[福塩線](../Page/福塩線.md "wikilink")の[改軌](../Page/改軌.md "wikilink")完成にともない、[クハ15形から改造製作された木製](../Page/国鉄クハ15形電車.md "wikilink")[制御車](../Page/制御車.md "wikilink")である。[1943年](../Page/1943年.md "wikilink")（昭和18年）までに8両（6001 - 6008）が本形式とされた。本形式は、同時に改造された[モハ1形と同じく](https://ja.wikipedia.org/wiki/国鉄モハ1形電車 "wikilink")、[直流](../Page/直流電化.md "wikilink")600[Vに対応させたもので](../Page/ボルト_\(単位\).md "wikilink")、福塩線の他、同じく買収線区である[可部線](https://ja.wikipedia.org/wiki/可部線 "wikilink")にも投入されており、[1945年](../Page/1945年.md "wikilink")（昭和20年）[8月6日](../Page/8月6日.md "wikilink")、[広島市](https://ja.wikipedia.org/wiki/広島市 "wikilink")に[原子爆弾](../Page/原子爆弾.md "wikilink")が投下された際、1両（6006）が[横川駅構内で犠牲となっている](https://ja.wikipedia.org/wiki/横川駅_\(広島県\) "wikilink")。

その他の車両は、いずれも広島[鉄道管理局](../Page/鉄道管理局.md "wikilink")管内で[1952年](../Page/1952年.md "wikilink")（昭和27年）まで使用されたほか、1両（6008）が[事業用](../Page/事業用車.md "wikilink")（[救援車](../Page/救援車.md "wikilink")）に転用され、[1953年](https://ja.wikipedia.org/wiki/1953年 "wikilink")（昭和28年）6月1日に施行された[車両称号規程改正により](https://ja.wikipedia.org/wiki/国鉄旧形電車の車両形式#1928年（昭和3年）車両称号規程 "wikilink")**クエ9100形**（9100）に[改番](../Page/改番.md "wikilink")されている。この車両は、大阪鉄道管理局管内で[1965年](../Page/1965年.md "wikilink")（昭和40年）まで在籍していた。

## 番号区分

形式的にはクハ15形からの改造車が全車を占めるが、もともとクハ15形自体が雑多な車両の寄せ集めであり、様々な出自を持つ車両が本形式にも含まれている。

  - クハ6001 ← クハ15001 ← クハ23515 ← サハ33734（[デハ33400系](https://ja.wikipedia.org/wiki/国鉄デハ33400系電車 "wikilink")）
  - クハ6002 ← クハ15002 ← クハ23516 ← サハ23629 ← デハ23416（デハ33400系）
  - クハ6003 ← クハ15003 ← クハ23517 ← サハ23630 ← デハ23417（デハ33400系）
  - クハ6004 ← クハ15011 ← サハ26125 ← サハ33551（[デハ63100系](https://ja.wikipedia.org/wiki/国鉄デハ63100系電車 "wikilink")）
  - クハ6005 ← クハ15047 ← クハ23508 ← デハ23513（[デハ33500系](https://ja.wikipedia.org/wiki/国鉄デハ33500系電車 "wikilink")）
  - クハ6006 ← クハ15049 ← クハ23512 ← デハ23514（デハ33500系）
  - クハ6007 ← クハ15045 ← クハ23501 ← デハ23501（デハ33500系）
  - クハ6008 ← クハ15019 ← サハ26133 ← サハ33559（デハ63100系）
