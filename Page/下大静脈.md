**下大静脈**（かだいじょうみゃく、inferior vena cava(IVC)）とはヒトの体の中でいちばん大きな[静脈](../Page/静脈.md "wikilink")で、その直径は約3.5cmである。下半身からの血液を集めて[心臓](../Page/心臓.md "wikilink")に流れ込んでいる。

下大静脈は[腹腔](../Page/腹腔.md "wikilink")の後部で脊柱の右側を上行しており、右下方の後面から[右心房](https://ja.wikipedia.org/wiki/右心房 "wikilink")に入る。

## 還流静脈

下大静脈は左右の総腸骨静脈が合流してできる。下大静脈は[脊柱](https://ja.wikipedia.org/wiki/脊柱 "wikilink")と並走する[奇静脈](https://ja.wikipedia.org/wiki/奇静脈 "wikilink")系や脊柱内外を網目のように存在する[椎骨静脈叢](https://ja.wikipedia.org/wiki/椎骨静脈叢 "wikilink")と[吻合](../Page/吻合.md "wikilink")しており、下大静脈が閉塞した際にはこれらの静脈が副側路となる。

下大静脈はT8の高さにある、[横隔膜](../Page/横隔膜.md "wikilink")の[大静脈孔](https://ja.wikipedia.org/wiki/大静脈孔 "wikilink")を通って[胸腔](https://ja.wikipedia.org/wiki/胸腔 "wikilink")内に入るとすぐに[右心房](https://ja.wikipedia.org/wiki/右心房 "wikilink")に合流する。下大静脈に流入する主な根には次のようなものがある。

|                                                                                                             |        |
| ----------------------------------------------------------------------------------------------------------- | ------ |
| **静脈**                                                                                                      | **高さ** |
| [下横隔静脈](https://ja.wikipedia.org/wiki/下横隔静脈 "wikilink")                                                     | T8     |
| [肝静脈](https://ja.wikipedia.org/wiki/肝静脈 "wikilink")                                                         | T8     |
| [腎静脈](https://ja.wikipedia.org/wiki/腎静脈 "wikilink")                                                         | L1     |
| [精巣静脈](https://ja.wikipedia.org/wiki/精巣静脈 "wikilink")・[卵巣静脈](https://ja.wikipedia.org/wiki/卵巣静脈 "wikilink") | L2     |
| [腰静脈](https://ja.wikipedia.org/wiki/腰静脈 "wikilink")                                                         | L1-L5  |
| [総腸骨静脈](https://ja.wikipedia.org/wiki/総腸骨静脈 "wikilink")                                                     | L5     |

下大静脈が体の中心から少し外れているために還流の形は非対称なところがある。たとえば、右精巣・卵巣静脈は直接下大静脈に流れ込んでいるが、左側は左腎静脈と合流してから下大静脈に流れ込んでいる。その一方で、肝静脈や腰静脈は通常左右対称で直接下大静脈に流れ込んでいる。

なお、上半身の静脈環流は[上大静脈](https://ja.wikipedia.org/wiki/上大静脈 "wikilink")が担っており、上大静脈と下大静脈は[奇静脈](https://ja.wikipedia.org/wiki/奇静脈 "wikilink")を介して連絡している。

## 関連疾患

管腔内圧は低く破裂を起こしにくいため下大静脈に関する病気はほとんどの場合、その狭窄に関するものである。下大静脈を圧迫する典型的な原因に[大動脈](../Page/大動脈.md "wikilink")の拡張、妊娠による[子宮](../Page/子宮.md "wikilink")の拡大、腹部の[悪性腫瘍](../Page/悪性腫瘍.md "wikilink")（[大腸がん](https://ja.wikipedia.org/wiki/大腸がん "wikilink")・[腎細胞がん](https://ja.wikipedia.org/wiki/腎細胞がん "wikilink")・[卵巣がん](https://ja.wikipedia.org/wiki/卵巣がん "wikilink")）がある。まれに排便時のいきみで下大静脈の血流が悪くなり[失神](../Page/失神.md "wikilink")することがある\[1\]。

下大静脈の閉塞は稀であるが、命にかかわり緊急性が高いとされる。[深部静脈血栓症](../Page/深部静脈血栓症.md "wikilink")や肝移植、[大腿静脈](https://ja.wikipedia.org/wiki/大腿静脈 "wikilink")の[カテーテル](../Page/カテーテル.md "wikilink")などの医療器具で閉塞を起こすことがある。\[2\]

## 発生

発生において、[右心房](https://ja.wikipedia.org/wiki/右心房 "wikilink")と下大静脈は[オイスタシイ弁](https://ja.wikipedia.org/wiki/オイスタシイ弁 "wikilink")によって分けられる。この構造は成体においては完全に退行しているか小さな心内膜ひだとして残っているかである\[3\] 。下大静脈の他の部分は、体幹を縦走する左右有対称の主な静脈のいろいろな部分が連なって形成される。比較的複雑な発生過程を経るために下大静脈には異常や変異が多い。
