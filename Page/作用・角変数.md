**作用・角変数** (さよう・かくへんすう, action-angle variable) とは、[解析力学](../Page/解析力学.md "wikilink")において[可積分な正準力学系に対して導入される](https://ja.wikipedia.org/wiki/可積分系 "wikilink")、作用変数と角変数の組からなる正準変数のこと。

## 定義

### 可積分系

\(n\)自由度の自励正準力学系がLiouvilleの意味で可積分であるとは、\(n\)個の関数的に独立\[1\]な[孤立積分](https://ja.wikipedia.org/wiki/第一積分 "wikilink") \(F_i\) (\(i = 1, 2, \cdots, n\)) が存在し、互いに[Poisson可換であること](../Page/ポアソン括弧.md "wikilink")、すなわち

\[\left[ F_i , F_j \right] = 0\] を満足することである\[2\]\[3\]。このとき、[リウヴィル＝アーノルドの定理](https://ja.wikipedia.org/wiki/リウヴィル＝アーノルドの定理 "wikilink")は、各積分 \(F_i\) が値 \(f_i\) を取る超曲面 \(\bigcap_{i = 1}^n F_i^{-1} ( f_i )\) が連結かつコンパクトであるならば、この曲面はトーラス \(\mathbb{T}^n\) と同相であること（Arnoldトーラスと呼ばれる）、そしてArnoldトーラスを含む近傍で定義された正準変数 \(( \mathbf{J}, \boldsymbol{\theta} )\) が存在し[ハミルトニアン](../Page/ハミルトニアン.md "wikilink") \(H\) が \(\mathbf{J}\) だけの関数になることを主張する\[4\]\[5\]。この定理により保証される正準変数 \(( \mathbf{J}, \boldsymbol{\theta} )\) が作用・角変数である\[6\]\[7\]。

### 変数分離系

変数分離可能 (separable) な系に関しては、作用・角変数をより明示的に導入することができる。このような系では、適切な正準変数 \(( \mathbf{p}, \mathbf{q} )\) を用いると、[ハミルトンの特性関数](https://ja.wikipedia.org/wiki/ハミルトンの特性関数 "wikilink") \(S ( \mathbf{q}, \boldsymbol{\alpha} )\) を \(S = S_1 ( q_1, \mathbf{\alpha} ) + S_2 ( q_2, \boldsymbol{\alpha} ) + \cdots + S_n ( q_n, \boldsymbol{\alpha} )\) という形に書くことができる\[8\]。積分定数 \(\boldsymbol{\alpha} = ( \alpha_1, \alpha_2, \cdots, \alpha_n )\) の値が特定されると、各座標 \(q_i\) が周期運動をするならば、その運動のパターンは次の二通りが可能である\[9\]\[10\]。

  - ある有界な範囲を周期的に運動する秤動 (libration)
  - 運動量が座標の周期関数となる回転 (rotaion)

このとき、定数 \(\mathbf{\alpha}\) により定まる解軌道に沿った一周期に関する次の積分

\[J_i := \frac{ 1 }{ 2 \pi } \oint p_i \, dq_i = \frac{ 1 }{ 2 \pi } \oint \frac{ \partial S_i }{ \partial q_i } ( q_i, \boldsymbol{\alpha} ) dq_i\] により**作用変数** (action variable) \(J_i = J_i ( \boldsymbol{\alpha} )\) が定義できる\[11\]\[12\]\[13\]\[14\]\[15\]。この定義のもとでハミルトンの特性関数は \(S = S ( \mathbf{q}, \mathbf{J} )\) という関数に読み替えることができ、この特性関数を母関数とする[正準変換](https://ja.wikipedia.org/wiki/正準変換 "wikilink") \(( \mathbf{p}, \mathbf{q} ) \mapsto ( \mathbf{J}, \boldsymbol{\theta} )\) により**角変数** (angle variable)

\[\theta_i := \frac{ \partial S }{ \partial J_i }\] が導入される\[16\]\[17\]\[18\]。角変数 \(\theta_i\) は運動の一周期の間に \(2 \pi\) 変化する\[19\]\[20\]。

## 性質

### Kronecker軌道

作用・角変数 \(( \mathbf{J}, \boldsymbol{\theta} )\) を用いるとき、系のハミルトニアンは \(H = H ( \mathbf{J} )\) であるため、[正準方程式](https://ja.wikipedia.org/wiki/正準方程式 "wikilink")は

\[\frac{ d J_i }{ d t } = - \frac{ \partial H }{ \partial \theta_i } = 0 , \ \ \frac{ d \theta_i }{ d t } = \frac{ \partial H }{ \partial J_i } =: \omega_i ( \mathbf{J} )\] となる。従ってその解はただちに

\[J_i = \mathrm{Const.}, \ \ \theta_i = \omega_i ( \mathbf{J} ) t + \beta_i\] と求まる (\(\beta_i\) は定数)。従って \(\omega_i = \frac{ \partial H }{ \partial J_i }\) は運動の角振動数である。この解がArnoldトーラス上に描く軌道をKronecker軌道と呼ぶ\[21\]。

振動数 \(\boldsymbol{\omega} = ( \omega_1, \omega_2, \cdots, \omega_n )\) がすべて互いに有理数比にある場合には、解軌道 \(\boldsymbol{\theta} = \boldsymbol{\omega} t + \boldsymbol{\beta}\) はArnoldトーラス上の周期軌道となる\[22\]。一方、そうでない場合には、解軌道はArnoldトーラスを稠密に埋め尽くし、準周期軌道 (qusi-periodic orbit) または条件周期軌道 (conditionally periodic orbit) と呼ばれる\[23\]。

### 正準摂動論

可積分ハミルトニアン \(H_0\) に摂動 \(\epsilon H_1\) が加わったハミルトニアン

\[H = H_0 ( \mathbf{J} ) + \epsilon H_1 ( \mathbf{J}, \boldsymbol{\theta} )\] を取り扱うことはしばしばある。このような近可積分系に対して適用される[正準摂動論](https://ja.wikipedia.org/wiki/正準摂動論 "wikilink")は作用・角変数に立脚して定式化される。これは、非摂動ハミルトニアン \(H_0\) に関する作用・角変数 \(( \mathbf{J}, \boldsymbol{\theta} )\) から摂動後のハミルトニアンに関する作用・角変数 \(( \mathbf{J}^*, \boldsymbol{\theta}^* )\) への正準変換 \(S = S ( \boldsymbol{\theta}, \mathbf{J}^* )\) を摂動的に決定するというアイデアに基づいている\[24\]\[25\]。

### 断熱不変量

作用変数 \(J\) は、ハミルトニアンの断熱的な（運動の時間スケールに比べてゆっくりとした）変化に際して保存する[断熱不変量](https://ja.wikipedia.org/wiki/断熱不変量 "wikilink")になる\[26\]。

## 具体例

### 調和振動子

1次元調和振動子は次のハミルトニアンにより記述される。

\[H = \frac{ p^2 }{ 2 m } + \frac{ 1 }{ 2 } m \omega_0^2 q^2\] この系はエネルギー \(E\) が保存するため可積分であり、ハミルトンの特性関数 \(S\) はエネルギーを積分定数とする

\[S = \int \pm \sqrt{ 2 m E - m^2 \omega_0^2 q^2 } \, dq\] という形に求まる。ここから調和振動子の作用・角変数は \(J = E / \omega_0\), \(\theta = \arcsin \left( \sqrt{ \frac{ m \omega_0 }{ 2 J } } q \right)\) と計算できる\[27\]。

\[q = \sqrt{ \frac{ 2 J  }{ m \omega_0 } } \, \sin \theta , \ \ p = \sqrt{ 2 m \omega_0 J } \, \cos \theta\]

\[H = \omega_0 J\]

### ケプラー問題

3次元ケプラー問題のハミルトニアンは、球座標 \(( r, \theta, \phi )\) を用いるとき変数分離系となる。

\[H = \frac{ 1 }{ 2 } \left( p_r^2 + \frac{ 1 }{ r^2 } p_\theta^2 + \frac{ 1 }{ r^2 \sin^2 \theta } p_\phi^2 \right) - \frac{ \mu }{ r }\] 対応するハミルトンの特性関数は次式で与えられる。

\[S = \int \pm \sqrt{ 2 C + \frac{ 2 \mu }{ r } - \frac{ G^2 }{ r^2 } } dr + \int \pm \sqrt{ G^2 - \frac{ G_z^2 }{ \sin^2 \theta } } d\theta + G_z \phi\] 系のエネルギーが負であるときには運動は有界であり、作用・角変数 \(( J_r, J_\theta, J_\phi , w_r, w_\theta, w_\phi )\) は次のように求められる\[28\]。

\[J_r = \sqrt{ \mu a } \left[ 1 - \sqrt{ 1 - e^2 } \right] , \ \ J_\theta = \sqrt{ \mu a ( 1 - e^2 ) } \, ( 1 - \cos I ) , \ \ J_\phi = \sqrt{ \mu a ( 1 - e^2 ) } \, \cos I\]

\[w_r = M , \ \ w_\theta = M + \omega , \ \ w_\phi = M + \omega + \Omega\] ここに \(a\) は軌道長半径、\(e\) は軌道離心率、\(I\) は軌道傾斜角、\(M\) は平均近点離角、\(\omega\) は近点引数、\(\Omega\) は昇交点黄経である。このときハミルトニアンは \(H = - \frac{ \mu^2 }{ 2 ( J_r + J_\theta + J_\phi )^2 }\) と表示される。なお、天体力学において用いられる[ドローニー変数](https://ja.wikipedia.org/wiki/ドローニー変数 "wikilink")や[ポアンカレ変数](https://ja.wikipedia.org/wiki/ポアンカレ変数 "wikilink")は、この作用・角変数に対して接触変換を施すことで得られる正準変数である\[29\]。
