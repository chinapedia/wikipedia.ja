[数学](../Page/数学.md "wikilink")、特に[線型代数学](https://ja.wikipedia.org/wiki/線型代数学 "wikilink")において、**対角行列**（たいかくぎょうれつ、）とは、[正方行列](../Page/正方行列.md "wikilink")であって、その対角成分（-要素）以外が零であるような行列のことである。

\[\begin{pmatrix}
  c_1 &&&0\\ & c_2 &&\\ && \ddots &\\ 0&&&c_n
\end{pmatrix}\] この対角行列は、[クロネッカーのデルタ](https://ja.wikipedia.org/wiki/クロネッカーのデルタ "wikilink")を用いて (*c*<sub>*i*</sub> δ<sub>*ij*</sub>) と表現できる。また、しばしば

  -
    diag(*c*<sub>1</sub>, *c*<sub>2</sub>, ..., *c*<sub>*n*</sub>)

のようにも書かれる。

[単位行列](../Page/単位行列.md "wikilink")や[スカラー行列](https://ja.wikipedia.org/wiki/スカラー行列 "wikilink")は対角行列の特殊例である。

## 性質

  - 対角行列の[行列式](https://ja.wikipedia.org/wiki/行列式 "wikilink")は、各対角成分の[総乗](https://ja.wikipedia.org/wiki/総乗 "wikilink") Π*c*<sub>*i*</sub> に等しい。対角行列の行列式は、対角成分が等しい[上三角行列](https://ja.wikipedia.org/wiki/三角行列 "wikilink")、[下三角行列の行列式とも等しくなる](https://ja.wikipedia.org/wiki/三角行列 "wikilink")。
  - 対角行列の[転置行列](../Page/転置行列.md "wikilink")は同一である。そのため対角行列は[対称行列](../Page/対称行列.md "wikilink")でもある。
  - 対角行列の[逆行列](https://ja.wikipedia.org/wiki/逆行列 "wikilink")は対角成分の[逆数](https://ja.wikipedia.org/wiki/逆数 "wikilink")を並べた対角行列である。
    :<math> \\begin{pmatrix}

` c_1 &&&0\\ & c_2 &&\\ && \ddots &\\ 0&&&c_n`

\\end{pmatrix}^{-1} = \\begin{pmatrix}

` c_1^{-1} &&&0\\ & c_2^{-1} &&\\ && \ddots &\\ 0&&&c_n^{-1}`

\\end{pmatrix}</math>

## 例

\(\begin{pmatrix}
1 & 0 \\
0 & 2 \\
\end{pmatrix}\)

\(\begin{pmatrix}
1 & 0 & 0 & 0\\
0 & 10 & 0 & 0\\
0 & 0 & -8 & 0\\
0 & 0 & 0 & 7
\end{pmatrix}\)

## 三重対角行列

**三重対角行列**（さんじゅう-、[tridiagonal matrix](https://ja.wikipedia.org/wiki/:en:Tridiagonal_matrix "wikilink")）とは、主対角線とその上下に隣接する対角線にだけ非零の成分を持つ行列であり\[1\]、[疎行列](https://ja.wikipedia.org/wiki/疎行列 "wikilink")の一種である。

\[\begin{bmatrix}
   {b_1} & {c_1} & {   } & {   } & { 0 } \\
   {a_2} & {b_2} & {c_2} & {   } & {   } \\
   {   } & {a_3} & {b_3} & \ddots & {   } \\
   {   } & {   } & \ddots & \ddots & {c_{n-1}}\\
   { 0 } & {   } & {   } & {a_n} & {b_n}\\
\end{bmatrix}\]

[数値解析](https://ja.wikipedia.org/wiki/数値解析 "wikilink")においてしばしば三重対角行列を含む方程式が現れる。このような方程式はトーマスアルゴリズムあるいは (TDMA) と呼ばれる、計算量のオーダーが*O* (*n* ) の解法を用いて解かれる。

与えられた行列を三重対角行列に変換する方法（三重対角化）には、[ハウスホルダー変換](https://ja.wikipedia.org/wiki/ハウスホルダー変換 "wikilink")や[ランチョス法](https://ja.wikipedia.org/wiki/ランチョス法 "wikilink")が知られている。
