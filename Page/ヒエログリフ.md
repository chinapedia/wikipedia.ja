[Egyptian_funerary_stela.jpg](https://ja.wikipedia.org/wiki/File:Egyptian_funerary_stela.jpg "fig:Egyptian_funerary_stela.jpg") **ヒエログリフ**（hieroglyph、**聖刻文字**、**神聖文字**）とは、[ヒエラティック](../Page/ヒエラティック.md "wikilink")、[デモティック](../Page/デモティック.md "wikilink")と並んで[古代エジプト](../Page/古代エジプト.md "wikilink")で使われた3種の[エジプト文字](https://ja.wikipedia.org/wiki/エジプト文字 "wikilink")のうちの1つ。エジプトの遺跡に多く記されており、紀元[4世紀](../Page/4世紀.md "wikilink")頃までは読み手がいたと考えられているが、その後読み方は忘れ去られてしまった。[19世紀](../Page/19世紀.md "wikilink")になって、フランスの[シャンポリオンの](../Page/ジャン＝フランソワ・シャンポリオン.md "wikilink")[ロゼッタ・ストーン](https://ja.wikipedia.org/wiki/ロゼッタ・ストーン "wikilink")解読以降読めるようになった。ロゼッタストーンによるとくずし字もあるとされている。

一般には古代エジプトの象形文字あるいはその書体を指す\[1\]が、広義には[アナトリア・ヒエログリフ](https://ja.wikipedia.org/wiki/アナトリア・ヒエログリフ "wikilink")（、の[象形文字](../Page/象形文字.md "wikilink")）、[クレタ・ヒエログリフ](../Page/クレタ聖刻文字.md "wikilink")（、[Eteocypriot languageの象形文字](https://ja.wikipedia.org/wiki/:en:Eteocypriot_language "wikilink")）、[マヤ・ヒエログリフ](../Page/マヤ文字.md "wikilink")（、[マヤ語の象形文字](../Page/マヤ語族.md "wikilink")）、[ミクマク・ヒエログリフ](https://ja.wikipedia.org/wiki/ミクマック象形文字 "wikilink")（、[ミクマク語](https://ja.wikipedia.org/wiki/ミクマク語 "wikilink")の象形文字）など、他の象形文字\[2\]に対しても用いられることがある\[3\]。　

## 名称

[ギリシア語](https://ja.wikipedia.org/wiki/ギリシア語 "wikilink")の （, ヒエログリュピカ）に由来する。これは、（, ヒエロス。「聖なる」）＋（, グリフォ。「[彫る](https://ja.wikipedia.org/wiki/グリフ "wikilink")」）を意味する。古代エジプト遺跡で主に[碑銘](https://ja.wikipedia.org/wiki/碑銘 "wikilink")に用いられていたためこう呼ばれた\[4\]。

## 歴史

### 文字の歴史

[Egypt_Hieroglyphe2.jpg](https://ja.wikipedia.org/wiki/File:Egypt_Hieroglyphe2.jpg "fig:Egypt_Hieroglyphe2.jpg") ヒエログリフがいつ頃使われ始めたかについてはまだ解明されていない。[エジプト原始王朝時代以前の](https://ja.wikipedia.org/wiki/エジプトの歴史#エジプト原始王朝時代 "wikilink")[紀元前4000年の](https://ja.wikipedia.org/wiki/紀元前41世紀 "wikilink")[Gerzeh cultureの壷に描かれたシンボルがヒエログリフに似ていることが知られている](https://ja.wikipedia.org/wiki/:en:Gerzeh_culture "wikilink")。[紀元前3200年頃](../Page/紀元前33世紀.md "wikilink")、[上エジプト](../Page/上エジプト.md "wikilink")にあった[:en:Nekhen](https://ja.wikipedia.org/wiki/:en:Nekhen "wikilink")の遺構から[1890年](../Page/1890年.md "wikilink")に出土したの文字を最古のヒエログリフとする見解が長い間一般的であった。

[紀元前3000年](https://ja.wikipedia.org/wiki/紀元前3000年 "wikilink")頃にはヒエログリフとヒエラティックが使い分けられていた。ヒエログリフは神聖なものとされ、神や、それと同等であるとされたファラオを称える石碑や神殿、墓などに刻まれた。神聖文字とも言われる。言わば漢字における[楷書](https://ja.wikipedia.org/wiki/楷書 "wikilink")に相当する。一方、[パピルス](../Page/パピルス.md "wikilink")へ手書きするときには[ヒエラティック](../Page/ヒエラティック.md "wikilink")（神官文字）が使われ、これは[行書](https://ja.wikipedia.org/wiki/行書 "wikilink")に例えられる。

この当時、文字というものはその王朝の文化や学問がいかに発展しているかを示す象徴であった。[古代エジプト](../Page/古代エジプト.md "wikilink")では、こうした背景からヒエログリフは特に重要視され、学習するものはごく限られた高い経歴をもつ者に限られた。また、一般の民衆には簡単に読み書きが不可能であるように、文字や文法も故意に複雑化された（[文字の歴史\#青銅器時代の文字](https://ja.wikipedia.org/wiki/文字の歴史#青銅器時代の文字 "wikilink")も参照）。

[エジプト中王国](../Page/エジプト中王国.md "wikilink")時代（[紀元前2040年](https://ja.wikipedia.org/wiki/紀元前2040年 "wikilink")-[紀元前1782年](https://ja.wikipedia.org/wiki/紀元前1782年 "wikilink")）にヒエログリフの改革が行われ、使用する文字の数を750程度に抑え、単語の綴りも一定化された。当時、[古代エジプト語は](../Page/エジプト語.md "wikilink")に移行した時期で、よりも細かいニュアンスを表現出来る文章語としての完成度が求められたことも要因として上げられる。この改革は、同時代の[古代オリエント](https://ja.wikipedia.org/wiki/古代オリエント "wikilink")世界において[楔形文字](../Page/楔形文字.md "wikilink")でも使用する文字数を減らす改革と、起こった時期が一致している。

[末期王朝時代の](https://ja.wikipedia.org/wiki/:en:Late_Period_of_ancient_Egypt "wikilink")[エジプト第26王朝](../Page/エジプト第26王朝.md "wikilink")（[紀元前650年](../Page/紀元前650年.md "wikilink")）頃にはヒエラティックの簡略化が進み、[草書体](../Page/草書体.md "wikilink")とも言うべき[デモティック](../Page/デモティック.md "wikilink")（民衆文字）となった。

その後、古代ローマ帝国統治下において徐々にギリシア文字が浸透、[4世紀](../Page/4世紀.md "wikilink")を境にして使用されなくなっていった。2018年現在、ヒエログリフの使用が最後に確認されているのは、[フィラエのイシス神殿内にある礼拝所の壁面に書かれたもので](../Page/フィラエ神殿.md "wikilink")、紀元後[394年](../Page/394年.md "wikilink")8月24日の日付がデモティックで残っている。

### 解読の歴史

中世を通じてもヒエログリフは多くの人々の関心を惹き付けていた。近代に入ると多くの学者達がヒエログリフの解読に挑んだ。特に有名なのは16世紀のと17世紀の[アタナシウス・キルヒャー](../Page/アタナシウス・キルヒャー.md "wikilink")であるが、解読に失敗したり、全く根拠のない独自の解釈に終わった。初めて解読に成功したのは19世紀のフランス人学者[ジャン＝フランソワ・シャンポリオン](../Page/ジャン＝フランソワ・シャンポリオン.md "wikilink")であり、彼はキルヒャーの収拾した資料を研究し、[ロゼッタ・ストーン](https://ja.wikipedia.org/wiki/ロゼッタ・ストーン "wikilink")の解読を行うことで読み方を解明した。これが突破口になり、その後も研究が進んだため、現代ではヒエログリフは比較的簡単に読むことができる。

## 文字の特徴

ヒエログリフは[象形文字](../Page/象形文字.md "wikilink")と呼ばれるように絵に似ているが、その見かけに反して、[表意文字](https://ja.wikipedia.org/wiki/表意文字 "wikilink")よりも[表音文字](https://ja.wikipedia.org/wiki/表音文字 "wikilink")が多い。表意文字の音を借りることもある。漢字でいえば[仮借](https://ja.wikipedia.org/wiki/仮借 "wikilink")の使用法に近い。表音文字では通常母音は無視され、子音だけが利用される。

その単語のカテゴリーを示すために、発音されない文字が表意的に単語に付け加えられることがある。これを[限定符](../Page/限定符.md "wikilink")という。限定符の違いによって同音異義語を区別することができる。漢字でいえば[形声文字](https://ja.wikipedia.org/wiki/形声文字 "wikilink")の[部首](../Page/部首.md "wikilink")に近いものである。例えば "pr" という発音には「建物、家、王宮」等という単語と、「出る」という単語がある。そこでヒエログリフでは建物を表す下記の象形文字でこれを表し、pr と読む。この場合は[表語文字](https://ja.wikipedia.org/wiki/表語文字 "wikilink")として使用していると言える。

|                   |          |
| ----------------- | -------- |
| <hiero>O1</hiero> | （pr『建物』） |
|                   |          |

pr「出る」を書きたい場合には、音だけを借りて同様に表す事も出来る。しかし、これでは「建物」等の意味に誤読される恐れがあるので、普通は下記の様な歩みだしている足の形をした文字を限定符として付け加える。これは、その単語が「歩行」に関する事を示す限定符である。

|                    |       |               |
| ------------------ | ----- | ------------- |
| <hiero>D54</hiero> | (D54) | (𓂻 : U+130BB) |

また、表意的に使われている事を示す為に "r" の音を表すヒエログリフを、以下のようにいわば送りがなとして添える事もある。これでprと発音し、送りがなのrや限定符は発音しない。

|                           |          |
| ------------------------- | -------- |
| <hiero>O1:D21-D54</hiero> | （pr『出る』） |
|                           |          |

[Egypt-Hieroglyphs.jpg](https://ja.wikipedia.org/wiki/File:Egypt-Hieroglyphs.jpg "fig:Egypt-Hieroglyphs.jpg")の像\]\] 右からでも左からでも書け、縦書き横書きも同様に行える。読む方向は、生物の形をしたヒエログリフの頭の向きで判断し、頭が向いている方向が文頭になる。

ヒエログリフで表される音は1子音から4子音だけで、母音は表記されない。実際にどう発音されていたかについては、[ギリシャ語](https://ja.wikipedia.org/wiki/ギリシャ語 "wikilink")や[コプト・エジプト語](https://ja.wikipedia.org/wiki/コプト・エジプト語 "wikilink")など、他の言語に借用された単語などから[再建](https://ja.wikipedia.org/wiki/再建 "wikilink")される場合もあるが、わからないことの方が多く、現代では以下のような仮の発音法が取り決められている。

  - 子音一つの単語の場合、前に「エ」音を補って読む。
      -
        例: s → エス（男）
  - 子音が二つ以上続く単語の場合は、各子音間に「エ」音を補って読む。
      -
        例: nfr → ネフェル（美しい）
  - Ȝ、‘、ỉ、w は本来子音文字だが、それぞれ母音「ア」、「アー」、「イ」、「ウ」として読む。ただし、語頭の ỉ は「ア」と読む場合もある。
      -
        例: sȜ → サ（息子）。 R‘ → ラー（太陽神[ラー](../Page/ラー.md "wikilink")）、Wsỉr → ウシル（[オシリス](../Page/オシリス.md "wikilink")）、Ỉtn → アテン（太陽神[アテン](../Page/アテン.md "wikilink")）、またはイテン。

しかし、この取り決めも絶対のものではなく、研究者によって様々な読み方がされている。例えばフランス式では「エ」の代わりに「オ」を補い、Ỉtn をアトンとする場合もある。
