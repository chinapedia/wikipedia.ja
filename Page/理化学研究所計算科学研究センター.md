**理化学研究所計算科学研究センター** (りかがくけんきゅうじょけいさんかがくけんきゅうセンター、、R-CCS) は、[兵庫県](../Page/兵庫県.md "wikilink")[神戸市](../Page/神戸市.md "wikilink")の[ポートアイランド](../Page/ポートアイランド.md "wikilink")にある[理化学研究所](https://ja.wikipedia.org/wiki/理化学研究所 "wikilink")の基盤センターである。 [日本](https://ja.wikipedia.org/wiki/日本 "wikilink")における[計算科学](../Page/計算科学.md "wikilink")及び[計算機科学](../Page/計算機科学.md "wikilink")の先導的研究開発機関であり、中核拠点である\[1\]。

に**理化学研究所計算科学研究機構** (りかがくけんきゅうじょけいさんかがくけんきゅうきこう、、AICS) から名称が変更された\[2\]。

## 概要

計算科学研究機構はに次世代スーパーコンピュータの共用に係る業務と、計算資源の利活用を通じた計算科学と計算機科学の連携による研究と発展のために設立された\[3\]。 から次世代スーパーコンピュータである[京の共用を開始し](https://ja.wikipedia.org/wiki/京_\(スーパーコンピュータ\) "wikilink")、の共用終了までに多くの[研究機関](https://ja.wikipedia.org/wiki/研究機関 "wikilink")、[大学](../Page/大学.md "wikilink")及び[企業](../Page/企業.md "wikilink")が利用し、多くの成果を生み出した\[4\]。

、理化学研究所は[文部科学省](../Page/文部科学省.md "wikilink")が2014年度から開始した京の後継機の開発プロジェクトの開発主体となり、翌年4月に計算科学研究機構内に開発プロジェクトが設置され、ポスト京の開発が開始された\[5\]。 にポスト京は[富岳と命名された](https://ja.wikipedia.org/wiki/富岳_\(スーパーコンピュータ\) "wikilink")\[6\]。 富岳は頃からの共用開始を予定している\[7\]。

## 沿革

詳細は及びを参照:

  - : 第3期科学技術基本計画において次世代スーパーコンピュータが「国家基幹技術」に指定され、理化学研究所が開発主体となる。

  - : 次世代スーパーコンピュータが[特定先端大型研究施設の共用の促進に関する法律](https://ja.wikipedia.org/wiki/特定先端大型研究施設の共用の促進に関する法律 "wikilink")に基づく共用施設とされ、理化学研究所が設置者となる。

  - : 次世代スーパーコンピュータの設置場所が神戸に決定される。

  - : 計算科学研究機構が設置され、次世代スーパーコンピュータの名称が「京」に決定される。

  - : 京が[TOP500](https://ja.wikipedia.org/wiki/TOP500 "wikilink")で世界第1位となる。

  - : 京の成果によって[ゴードン・ベル賞](../Page/ゴードン・ベル賞.md "wikilink")で最高性能賞を受賞する。

  - : 京の共用を開始する。

  - : 理化学研究所がポスト京の開発プロジェクトの開発主体となる。

  - : 計算科学研究機構内にポスト京の開発プロジェクトが設置され、開発が開始される。

  - : 名称が「計算科学研究センター」に変更される。

  - : ポスト京の名称が「富岳」に決定される。

  - : 京の共用が終了する。
