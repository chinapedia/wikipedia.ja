**BeeCut**は、初心者向けの多機能動画編集・加工ソフトである。

## 概要

BeeCutは動画編集ソフトであり、主な機能はトリミング、回転反転、逆再生、ズーム、テキスト・音楽・モザイクの追加、フィルター・エフェクト・オーバーレイ・切り替え効果の適用、PIP（ピクチャーインピクチャー）、フリーズフレーム、写真スライドショーの作成などである。ドラッグ&ドロップで操作できるので、使い方は比較的に簡単な編集ソフトである。

## 特徴

### ５つの画面比率

1：1（正方形）、3：4、4：3（スタンダード）、9：16（スマホ縦画面）：16：9（ワイドスクリーン）

### 表示時間の設定

画像、テキスト、トランジション、フィルター、オーバーレイの持続時間を一気に設定できる

### アフレコ録音

アフレコ録音機能が内蔵している、録音後オーディオトラックに保存される。

### 音声文字変換

文字は音声に、音声はテキストに変換できる。（現在英語と中国語しかサポートできない）

### オープニング素材と効果音・BGM

直接に利用可能なオープニング素材と大量の効果音、BGMが提供してる。

### 画面録画

録画ツールがある。選択すればオンライン画面録画ツールのページへ飛びます。

## 対応フォーマット

|      |                                            |
| ---- | ------------------------------------------ |
| 音声形式 | WAV、MP3、M4A、MID、FLAC、AAC、WMA、AU、AIFF、OGG、他 |
| 動画形式 | MP4、MOV、AVI、MPEG、WMV 、その他                  |
| 画像形式 | BMP、JPG、PNG、TIFF、HEIC                      |

＊動画、音声形式変換ツールがある。フォーマットを自由に変換できる。

CPU - Intel i5以上

メモリー - 4GB以上
