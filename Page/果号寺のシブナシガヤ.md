[Kagoji_no_Shibunashigaya_B.jpg](https://ja.wikipedia.org/wiki/File:Kagoji_no_Shibunashigaya_B.jpg "fig:Kagoji_no_Shibunashigaya_B.jpg") **果号寺のシブナシガヤ**（かごうじのシブナシガヤ）は、[三重県](https://ja.wikipedia.org/wiki/三重県 "wikilink")[伊賀市](https://ja.wikipedia.org/wiki/伊賀市 "wikilink")西山にある国の[天然記念物](../Page/天然記念物.md "wikilink")に指定された[シブナシガヤ](https://ja.wikipedia.org/wiki/シブナシガヤ "wikilink")（渋無榧）である\[1\]。

[カヤ](../Page/カヤ.md "wikilink")（榧）の変種であるシブナシガヤの[種子](https://ja.wikipedia.org/wiki/種子 "wikilink")は通常のカヤの種子と異なり、[胚乳](../Page/胚乳.md "wikilink")全体を囲むように着いている**渋皮**が全く存在しない、もしくはあっても取れやすいといった特徴があり、種子の殻を割ると、中から渋皮のない白い胚乳が直接露出しているため、昔から「蜂の子ガヤ」「白米ガヤ」などと呼ばれ珍重されてきた\[2\]。

国の天然記念物に指定されたシブナシガヤは、枯死により指定解除\[3\]されたものも含めて日本全国に3件しか存在せず、果号寺のシブナシガヤはそのうちのひとつである。[1932年](../Page/1932年.md "wikilink")（[昭和](../Page/昭和.md "wikilink")7年）7月25日に国の[天然記念物](../Page/天然記念物.md "wikilink")に指定された\[4\]\[5\]\[6\]。[1997年](https://ja.wikipedia.org/wiki/1997年 "wikilink")（[平成](../Page/平成.md "wikilink")9年）に三重県が選定した『みえの樹木百選』のひとつにも選ばれている\[7\]。

## 解説

[Kagoji_no_Shibunashigaya_E.jpg](https://ja.wikipedia.org/wiki/File:Kagoji_no_Shibunashigaya_E.jpg "fig:Kagoji_no_Shibunashigaya_E.jpg")

果号寺のシブナシガヤのある西山地区は、三重県北西部の[伊賀盆地](https://ja.wikipedia.org/wiki/伊賀盆地 "wikilink")北西、[滋賀県](../Page/滋賀県.md "wikilink")[甲賀市](../Page/甲賀市.md "wikilink")との境界にある[御斎峠](https://ja.wikipedia.org/wiki/御斎峠 "wikilink")の南東麓に位置しており、シブナシガヤのある[浄土宗](../Page/浄土宗.md "wikilink")果号寺は家屋と畑地が混在する標高約220 - 230メートルの傾斜地に所在している。国の天然記念物に指定されたシブナシガヤは果号寺本堂に隣接した[境内](../Page/境内.md "wikilink")の東北隅に生育しており\[8\]\[9\]、地表からの高さ50センチメートルでの幹囲は370センチメートル、地表高90センチメートル付近より3本の幹に分岐している\[10\]。枝張りは東西約15メートル、南北約18メートル、樹高約20メートルの高木であり、樹勢は旺盛で毎年、夏の終り頃に多くの実をつける\[11\]。

樹齢は不明であるものの、本樹は後述する江戸期の史料中にも記載が見られることから、古くから知られた古木である。生育する[伊賀地方](https://ja.wikipedia.org/wiki/伊賀地方 "wikilink")から西隣の[奈良県](../Page/奈良県.md "wikilink")北東部の[山辺郡](https://ja.wikipedia.org/wiki/山辺郡 "wikilink")一帯にはカヤの古木が多く見られるが、これはカヤの種子の[胚乳](../Page/胚乳.md "wikilink")に多く含まれる[油脂](../Page/油脂.md "wikilink")から搾り取った[植物油](../Page/植物油.md "wikilink")が最上級の食用油であったことから、この地方に多数植樹されたことによるものであるという\[12\]。

これらの中でも果号寺のシブナシガヤは渋皮のない珍しいものとして古くより知られており、江戸時代中期の[宝暦](../Page/宝暦.md "wikilink")年間（[1751年](../Page/1751年.md "wikilink") - [1764年](../Page/1764年.md "wikilink")）に、[藤堂藩](https://ja.wikipedia.org/wiki/藤堂藩 "wikilink")[上野城](../Page/上野城.md "wikilink")[城代](../Page/城代.md "wikilink")藤堂元甫によって編纂された[地誌](https://ja.wikipedia.org/wiki/地誌 "wikilink")『三国地誌』の中では、

|                                     |
| ----------------------------------- |
| 白榧實、西山村、此實尋常ニカハリ渋外皮ニツキテ實白シ故ニ白米榧トヨブ。 |
| 藤堂元甫『三国地誌』                          |

と記されており\[13\]、また、著名な[本草学者](https://ja.wikipedia.org/wiki/本草学者 "wikilink")である[小野蘭山](../Page/小野蘭山.md "wikilink")による『本草網目啓蒙』の嘉永版（[嘉永](../Page/嘉永.md "wikilink")2年/[1849年](../Page/1849年.md "wikilink")）の巻37、第6枚の裏には、

|                                                                            |
| -------------------------------------------------------------------------- |
| 一種しぶなしガヤアリ、しぶ皮核ニ着イテ仁ニ着カズ、故ニはちのこトモ呼ブ、濃州多良\[14\]及び伊州上野\[15\]ニモアリ伊賀方言しろがやと言フ。 |
| 小野蘭山『本草網目啓蒙』                                                               |

このように記載されており\[16\]\[17\]、当時より渋皮のない珍しいカヤ（榧）が伊賀地方にあることが知られていたことがわかる\[18\]。

国の天然記念物に指定されたカヤは巨樹や変種を含め日本全国に15件あるが、そのうちの特徴的な珍しい種子を持つ個体について、[林野庁](../Page/林野庁.md "wikilink")所管の[国立研究開発法人](https://ja.wikipedia.org/wiki/国立研究開発法人 "wikilink")[森林研究・整備機構](../Page/森林研究・整備機構.md "wikilink")の関西育種場（[岡山県](https://ja.wikipedia.org/wiki/岡山県 "wikilink")[勝田郡](../Page/勝田郡.md "wikilink")[勝央町](../Page/勝央町.md "wikilink")）では、果号寺のシブナシガヤを含むヒダリマキガヤ・コツブガヤなど次に挙げる、

  - [果号寺のシブナシガヤ](https://ja.wikipedia.org/wiki/果号寺のシブナシガヤ "wikilink")（三重県伊賀市）
  - 高倉神社のシブナシガヤ（三重県伊賀市）
  - 庫蔵寺のコツブガヤ（三重県[鳥羽市](../Page/鳥羽市.md "wikilink")）
  - [熊野のヒダリマキガヤ](https://ja.wikipedia.org/wiki/熊野のヒダリマキガヤ "wikilink")（[滋賀県](../Page/滋賀県.md "wikilink")[蒲生郡](../Page/蒲生郡.md "wikilink")[日野町](../Page/日野町_\(滋賀県\).md "wikilink")）
  - [建屋のヒダリマキガヤ](https://ja.wikipedia.org/wiki/建屋のヒダリマキガヤ "wikilink")（[兵庫県](../Page/兵庫県.md "wikilink")[養父市](https://ja.wikipedia.org/wiki/養父市 "wikilink")）

国の天然記念物に指定された上記5件のカヤを保存種として[クローン](https://ja.wikipedia.org/wiki/クローン "wikilink")収集し、保存・定植している\[19\]。
