**原生花園**（げんせいかえん）とは、人為的な手を加えず自然をそのままにした状態でも色鮮やかな[花](https://ja.wikipedia.org/wiki/花 "wikilink")が咲く、湿地帯や草原地帯のこと。独特の植生が見られることで知られ、自然の花畑とも呼べる。

[北海道](https://ja.wikipedia.org/wiki/北海道 "wikilink")では[オホーツク海](../Page/オホーツク海.md "wikilink")沿岸をはじめ、[道東](https://ja.wikipedia.org/wiki/道東 "wikilink")から[道北](https://ja.wikipedia.org/wiki/道北 "wikilink")にかけて多く分布する。中には[小清水原生花園](https://ja.wikipedia.org/wiki/小清水原生花園 "wikilink")が全国的に有名になったことから、他の地区もイメージアップを図って同様に名乗り始めたものが多い。また、道内以外にも原生花園を名乗る地区があるが、基本的に○○湿原と名乗ることが多い。

小清水原生花園、[サロベツ原生花園](https://ja.wikipedia.org/wiki/サロベツ原野 "wikilink")、[ワッカ原生花園](https://ja.wikipedia.org/wiki/ワッカ原生花園 "wikilink")が著名。また、原生花園と名乗らないが、[霧多布湿原](https://ja.wikipedia.org/wiki/霧多布湿原 "wikilink")も性質上は原生花園である。

## 全国の原生花園

### 北海道の原生花園

  - [日本海](../Page/日本海.md "wikilink")沿岸

:; 金浦原生花園（かなうらげんせいかえん）（天塩郡[遠別町](../Page/遠別町.md "wikilink")）

:: 遠別町市街地より南下。[エゾカンゾウ](https://ja.wikipedia.org/wiki/エゾカンゾウ "wikilink")が中心。晴天時には遠く[利尻富士](https://ja.wikipedia.org/wiki/利尻富士 "wikilink")を望む。

:; 鏡沼海浜公園（かがみぬまかいひんこうえん）（天塩郡天塩町）

:: 原生花園と名乗らないが、性質上は原生花園。エゾスカシユリやエゾカンゾウ、ハマヒルガオなどが見られる。

:; サロベツ原生花園（天塩郡[豊富町](../Page/豊富町.md "wikilink")）

:: [サロベツ原野](https://ja.wikipedia.org/wiki/サロベツ原野 "wikilink")を参照。

:; 抜海原生花園（ばっかいげんせいかえん）（[稚内市](../Page/稚内市.md "wikilink")）

:: 稚内市西部抜海地区。日本海を隔てて利尻富士を望む。[ハマナス](../Page/ハマナス.md "wikilink")、[ハナショウブ](../Page/ハナショウブ.md "wikilink")などが見られる。近くに展望台。[利尻礼文サロベツ国立公園](../Page/利尻礼文サロベツ国立公園.md "wikilink")に含まれる。

:; メグマ原生花園（稚内市）

  -

      -
        稚内市中央部、野寒布岬と宗谷岬の中間に位置する。主の住む沼として畏れられていた海跡湖、メグマ沼一帯を指し、手付かずの自然が良好な状態で残されており、[ヒオウギアヤメ](https://ja.wikipedia.org/wiki/ヒオウギアヤメ "wikilink")、エゾカンゾウ、[ツルコケモモ](https://ja.wikipedia.org/wiki/ツルコケモモ "wikilink")など200種の植物が咲く。交通アクセスもよく、観光ガイドに載ることも多い。遊歩道はないが、国道からその自然を散策することができる。

  - オホーツク海沿岸

:; エサヌカ原生花園（宗谷郡[猿払村](../Page/猿払村.md "wikilink")）

:: [クッチャロ湖](https://ja.wikipedia.org/wiki/クッチャロ湖 "wikilink")北部の[モケウニ沼](https://ja.wikipedia.org/wiki/モケウニ沼 "wikilink")近辺の湿地帯。[エゾカンゾウ](https://ja.wikipedia.org/wiki/エゾカンゾウ "wikilink")、[ハマナス](../Page/ハマナス.md "wikilink")などが見られる。観光化はされていない穴場だが、[国道228号](https://ja.wikipedia.org/wiki/国道228号 "wikilink")が縦貫しているため探勝しやすい。

:; ベニヤ原生花園（枝幸郡[浜頓別町](https://ja.wikipedia.org/wiki/浜頓別町 "wikilink")）

:: クッチャロ湖の北西に展開する原生花園で、オホーツク海沿岸にある原生花園の中でもかなり大規模で観光化も進んでいる。[ノハナショウブ](https://ja.wikipedia.org/wiki/ノハナショウブ "wikilink")が特に知られ、夏場には緑の絨毯を一面紫色に染める。その他100種類以上の花が確認される。

:; オムサロ原生花園（紋別市）

:: 渚滑川河口域に位置する。ハマナスが中心だが、他に[エゾスカシユリ](https://ja.wikipedia.org/wiki/エゾスカシユリ "wikilink")、エゾカンゾウなど。オムサロ・ネイチャー・ビューセンターという休憩施設があり、利用しやすい。

:; コムケ原生花園（小向原生花園）（[紋別市](../Page/紋別市.md "wikilink")）

:: 紋別市東部にあるコムケ湖一帯にある。夏はハマナスや[コケモモ](https://ja.wikipedia.org/wiki/コケモモ "wikilink")、秋はサンゴソウが一面に繁茂し、真っ赤に染める。

:; 鶴沼原生花園（サンゴ岬とも）（紋別郡[湧別町](https://ja.wikipedia.org/wiki/湧別町 "wikilink")）

:: サロマ湖の西岸にあり、ワッカ原生花園とは海を隔てる。サンゴ岬とも呼ばれる。[サンゴソウ](https://ja.wikipedia.org/wiki/サンゴソウ "wikilink")が一面に繁茂し、真っ赤に染める。サロマ湖原生花園とも。

:;[ワッカ原生花園](https://ja.wikipedia.org/wiki/ワッカ原生花園 "wikilink")（北見市[常呂町](../Page/常呂町.md "wikilink")）

::

:; 能取原生花園（網走市）

:: 能取湖の北岸に位置する。同じくサンゴソウが有名。

:; [小清水原生花園](https://ja.wikipedia.org/wiki/小清水原生花園 "wikilink")（斜里郡小清水町）

::

:; 以久科原生花園（いくしなげんせいかえん）（斜里郡斜里町）

:: 斜里町市街地から東の外れにある原生花園。エゾスカシユリ、エゾカンゾウ、ハマナスなどが確認される。さほど観光化されていない。

:; [野付海岸原生花園](https://ja.wikipedia.org/wiki/野付半島 "wikilink")（野付郡別海町）

::

:; 北方原生花園（ほっぽう-）（根室市）

  -

      -
        根室半島の北岸にある原生花園。ヒオウギアヤメ、[ワタスゲ](https://ja.wikipedia.org/wiki/ワタスゲ "wikilink")、[クロユリ](../Page/クロユリ.md "wikilink")のほか、低木の[ミズナラ](https://ja.wikipedia.org/wiki/ミズナラ "wikilink")が見られる。

  - 太平洋岸

:; [霧多布湿原](https://ja.wikipedia.org/wiki/霧多布湿原 "wikilink")（厚岸郡浜中町）

::

:; [あやめが原](https://ja.wikipedia.org/wiki/あやめが原 "wikilink")（厚岸郡厚岸町）

::

:; 豊北原生花園（とよきたげんせいかえん）（十勝郡[浦幌町](../Page/浦幌町.md "wikilink")）

:: 十勝川と浦幌十勝川の間に挟まれた湿地帯。ハマナス、コケモモなど300種類以上の植物を確認。観光化もされていない穴場となっている。

:; 大津原生花園・長節原生花園（ちょうぶしげんせいかえん）（中川郡[豊頃町](../Page/豊頃町.md "wikilink")）

:: 十勝川と長節湖の間に広がる原生花園。[スズラン](https://ja.wikipedia.org/wiki/スズラン "wikilink")、クロユリ、ハマナスなどが咲く。

:; 晩成原生花園（広尾郡[大樹町](https://ja.wikipedia.org/wiki/大樹町 "wikilink")）

  -

      -
        海岸湖、ホロカヤントーの近くに分布する。ハマナス、ハナショウブ、[ガンコウラン](https://ja.wikipedia.org/wiki/ガンコウラン "wikilink")など。晩成温泉が湧出し、観光拠点となる。晩成海岸原生花園、晩成温泉原生花園とも。
