## 概要

[オール日本スーパーマーケット協会](../Page/オール日本スーパーマーケット協会.md "wikilink")会員。なお店頭で使用されるロゴの「関」の表記は、「[18px](https://ja.wikipedia.org/wiki/画像:RYAKUJI_2-0000.gif "wikilink")」の中に「关」が入った[略字](../Page/略字.md "wikilink")体で表されている。地元では「関スパ」「関スー」など と略される。

## 沿革

  - [1959年](../Page/1959年.md "wikilink")（昭和34年）
      - 7月 - [大阪市](../Page/大阪市.md "wikilink")[東淀川区](../Page/東淀川区.md "wikilink")に、北野祐次他9名にてスーパーマーケット開業を目的として「相互産業株式会社」を設立（資本金2,500千円）
      - 11月 - 本店所在地を大阪市東淀川区より[兵庫県](../Page/兵庫県.md "wikilink")[伊丹市](../Page/伊丹市.md "wikilink")に移転
      - 12月 - 兵庫県伊丹市に、スーパーマーケットの第1号店として伊丹店（その後、中央店）を開設
  - [1964年](../Page/1964年.md "wikilink")（昭和39年）10月 - 上記第1号店隣接地に伊丹ショッピングセンターを開設
  - [1971年](https://ja.wikipedia.org/wiki/1971年 "wikilink")（昭和46年）
      - 4月 - 兵庫県伊丹市の阪急伊丹駅前市街地再開発事業に参画し、伊丹ショッピングデパートを開設
      - 12月 - 不動産貸付業の相互開発株式会社を吸収合併
  - [1973年](../Page/1973年.md "wikilink")（昭和48年）4月 - 株式会社小僧寿し本部との間にフランチャイズ契約を取り交わし、外食事業の展開に着手
  - [1974年](../Page/1974年.md "wikilink")（昭和49年）8月 - 商号を「株式会社関西スーパーマーケット」に変更
  - [1987年](https://ja.wikipedia.org/wiki/1987年 "wikilink")（昭和62年）8月 - 大阪市[福島区](../Page/福島区.md "wikilink")に所在する[大阪中央卸売市場](https://ja.wikipedia.org/wiki/大阪中央卸売市場 "wikilink")の隣接地に、生鮮食品仕入業務用の事務所として「中央市場オフィス」を開設
  - [1988年](../Page/1988年.md "wikilink")（昭和63年）5月 - 小僧寿し店23か店を株式会社小僧寿し関西地区本部へ営業譲渡
  - [1991年](../Page/1991年.md "wikilink")（平成3年）11月 - [大阪証券取引所](https://ja.wikipedia.org/wiki/大阪証券取引所 "wikilink")市場第二部に上場
  - [1992年](../Page/1992年.md "wikilink")（平成4年）
      - 1月 - ホームセンター事業を目的とし、株式会社スマイルを設立。5月に商号を株式会社ビッグパワーに変更
      - 8月 - 商品の運送事業を目的とし、株式会社関西スーパー物流（その後、連結子会社化）を設立
  - [2005年](../Page/2005年.md "wikilink")（平成17年）3月 - 株式会社ビッグパワーを会社清算
  - [2008年](https://ja.wikipedia.org/wiki/2008年 "wikilink")（平成20年）3月 - [東京証券取引所](../Page/東京証券取引所.md "wikilink")市場第二部に上場
  - [2013年](../Page/2013年.md "wikilink")（平成25年）
      - 1月 - 大阪証券取引所市場第一部指定
      - 3月 - 株式会社スーパーナショナル（大阪市）と業務・資本提携。持分法適用会社に。
      - [7月16日](../Page/7月16日.md "wikilink") - 東京証券取引所と大阪証券取引所の現物株市場統合に伴い、東京証券取引所市場第一部上場となる。
  - [2016年](../Page/2016年.md "wikilink")（平成28年）
      - 8月 - 兵庫県伊丹市の本社社屋・中央店の建て替え竣工\[1\]。
      - 10月27日 - [エイチ・ツー・オー リテイリングとの間で資本業務提携契約の締結を発表](../Page/エイチ・ツー・オー_リテイリング.md "wikilink")。
      - 11月14日 - エイチ・ツー・オー リテイリングに対し[第三者割当増資](../Page/第三者割当増資.md "wikilink")を行い、同社が約10%の株式を取得。

### 関西スーパーおさいふカード

関西スーパーが発行する[Edyカード](../Page/楽天Edy.md "wikilink")。関西スーパーでの買い物月額によって、翌月の買い物が割引される。 ※なお、関西スーパーおさいふカード以外のEdyカードも利用できるが、上記の割引サービスの対象外となる。

| 月の買い物額     | 翌月の割引率（%） |
| ---------- | --------- |
| 1万円未満      | 0         |
| 1万円以上2万円未満 | 0.5       |
| 2万円以上3万円未満 | 1.0       |
| 3万円以上4万円未満 | 1.5       |
| 4万円以上5万円未満 | 2.0       |
| 5万円以上      | 2.5       |

## エイチ・ツー・オー リテイリングとの資本業務提携

2016年（平成28年）10月27日開催の取締役会において、[エイチ・ツー・オー リテイリングとの間で資本業務提携](../Page/エイチ・ツー・オー_リテイリング.md "wikilink")・第三者割当による新株式発行を行うことを決議。同日付けで資本業務提携契約の締結を発表した。11月14日には第三者割当増資により、エイチ・ツー・オー リテイリングが10.02%（議決権割合10.17%）の株式を取得した\[2\]。

業務提携により

  - 店舗で販売する商品の共同仕入れ
  - エイチ・ツー・オー リテイリンググループにおいて使用されている「Sポイント」の導入
  - 次世代型レジスターの共同開発

などを行うとしている。
