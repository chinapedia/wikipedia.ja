**Outlook Web Access**（**OWA**）は、[Microsoft Exchange Server](../Page/Microsoft_Exchange_Server.md "wikilink") 5.0 およびそれ以降での[Webメール](https://ja.wikipedia.org/wiki/Webメール "wikilink")サービスであり、以前は **Exchange Web Connect** (**EWC**) と呼ばれていた。Outlook Web Access の Web インタフェースは、[Microsoft Outlook](https://ja.wikipedia.org/wiki/Microsoft_Outlook "wikilink") のインタフェースに似せてある。Exchange 2010 では**Outlook Web App**と名称変更された。[Office Web Appsの一員ではない](https://ja.wikipedia.org/wiki/Office_Web_Apps "wikilink")。

OWA は、電子メール（[S/MIME](https://ja.wikipedia.org/wiki/S/MIME "wikilink")サポート）、カレンダー、連絡先一覧、タスク一覧、その他のメールボックスのコンテンツにアクセスでき、Outlook のデスクトップ版が使用できない環境でのアクセスを提供する。Exchange 2007 では、[Microsoft SharePoint](../Page/Microsoft_SharePoint.md "wikilink") のサイトに格納された文書へのアクセスなどもサポートしている。Exchange Server の一部として提供され、[ウェブブラウザ](../Page/ウェブブラウザ.md "wikilink")によるアクセスを可能とする。デスクトップ版 Outlook との大きな違いは、Outlook がネットワークから切断された状態でも電子メールやカレンダーを使えるのに対して、OWA ではネットワークに接続されていないと機能しない。

[インターネットカフェ](https://ja.wikipedia.org/wiki/インターネットカフェ "wikilink")など、Webアクセスが可能な場所ならどこからでもアクセスできる。Exchange 2000、Exchange 2003、Exchange 2007 での OWA インタフェースは2種類ある。"Premium" [UI](../Page/ユーザインタフェース.md "wikilink") は、[Internet Explorer](../Page/Internet_Explorer.md "wikilink") 5 （2000 および 2003）と Internet Explorer 6 SP1 以降（2007）でのみ利用可能である。一方 "Basic" UI は他のブラウザでも利用可能だが、一部機能が制限されている。

クライアント側のスクリプトから[HTTP要求](../Page/Hypertext_Transfer_Protocol.md "wikilink") (XMLHTTP) を発行できるようにしたのは、Outlook Web Access チームであった。これが間もなく Internet Explorer 5.0 に組み込まれ、[XMLHttpRequest](https://ja.wikipedia.org/wiki/XMLHttpRequest "wikilink") と改称され、[World Wide Web Consortium](../Page/World_Wide_Web_Consortium.md "wikilink") で標準化された\[1\]。これは、[Ajax](https://ja.wikipedia.org/wiki/Ajax "wikilink")技術によるWebアプリケーション構築の基盤の1つになっている。
