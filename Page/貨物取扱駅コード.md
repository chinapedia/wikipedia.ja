**貨物取扱駅コード**（かもつとりあつかいえきコード）は、[日本貨物鉄道](../Page/日本貨物鉄道.md "wikilink")が貨物取扱駅を識別する4桁の番号で 、[1973年](../Page/1973年.md "wikilink")に[日本国有鉄道](../Page/日本国有鉄道.md "wikilink")が作成している。 [貨車車票](../Page/貨車車票.md "wikilink")は発着駅を漢字表記と貨物取扱駅コードで記している。旅客鉄道各社の[駅名コード](https://ja.wikipedia.org/wiki/駅名コード "wikilink")とは異なる。

各貨物取扱駅コードが日本地図上に表現されている **貨物取扱駅コード図表**は社員や関係者に配布され『[貨物時刻表](../Page/貨物時刻表.md "wikilink")』に収録されている。

## 導入の背景

かつて、貨物列車を組成した駅は個々の車票から**貨車連結順序表**を作成して貨車それぞれの着駅を**組成通報**として主要停車駅へ伝達し、各駅は組成通報に従い構内作業を計画して列車到着に備えていた。

車票は漢字記載だが、組成通報は[電信](../Page/電信.md "wikilink")や[テレタイプ](https://ja.wikipedia.org/wiki/テレタイプ "wikilink")で片仮名記述に加え文字数制限により発駅が記述されず、誤報の抑止に窮していた。

1973年8月1日に、各駅に4桁の識別コードを設定し、車票は様式を縦書きから横書きヘ変更して駅名を漢字とコードで併記し、組成通報は発着駅をコードで表現する、本方式を使用開始している。

## 番号体系

導入時の貨物取扱駅数から4桁とされ、第1桁は北海道を1、九州を9として順に付与し地区を示す。第1桁を含めた上位2桁で総局・[鉄道管理局](../Page/鉄道管理局.md "wikilink")が識別され次の表に対応を示す。

|        | 第2桁の数字           |
| ------ | ---------------- |
| |0     | |1               |
| 第1桁の数字 | |1               |
| |2     | 盛岡局（20・21）       |
| |3     | 新潟局（30・31）       |
| |4     | ─                |
| |5     | 長野局（50・51）       |
| |6     | 大阪局（60・61）       |
| |7     | 四国総局（70）         |
| |8     | 広島局（80・81）       |
| |9     | 門司局（90・91・92・93） |

貨物取扱駅コードの上2桁と総局・鉄道管理局との対応

1973年の導入当時に定めた下位2桁の規則を次に記す。

1.  線区の起点または起点に近い局分界駅から下り方向に、[旅客駅](https://ja.wikipedia.org/wiki/旅客駅 "wikilink")を無視して01から始まる連番を付与する。例えば、広島鉄道管理局管内の[山陽本線](https://ja.wikipedia.org/wiki/山陽本線 "wikilink")の場合、[本郷駅は](https://ja.wikipedia.org/wiki/本郷駅_\(広島県\) "wikilink")8001、[西高屋駅](../Page/西高屋駅.md "wikilink")は8002とする。
2.  貨物営業をしている[国鉄自動車線および連絡社線は](../Page/国鉄バス.md "wikilink")、各駅には番号を付与せず接続駅コードの次コードを代表コードとして付与する。例えば、[三島駅](https://ja.wikipedia.org/wiki/三島駅 "wikilink")は5201とし、三島駅接続の[伊豆箱根鉄道](../Page/伊豆箱根鉄道.md "wikilink")は5202とする。
3.  [荷扱所](https://ja.wikipedia.org/wiki/荷扱所 "wikilink")は管理駅コードの次コードを付与する。例えば、[石岡駅](../Page/石岡駅.md "wikilink")は4207とし、柿岡荷扱所は4208とする。
4.  旅客駅であっても貨車による郵便または[手小荷物](https://ja.wikipedia.org/wiki/手小荷物 "wikilink")取扱駅はコードを付与する。例えば、[京都駅](https://ja.wikipedia.org/wiki/京都駅 "wikilink")は6010とする。
5.  異例取扱貨物の発着駅もコードを付与する。例えば、[上野駅](https://ja.wikipedia.org/wiki/上野駅 "wikilink")は4446とする。
6.  [小口扱貨物](https://ja.wikipedia.org/wiki/小口扱貨物 "wikilink")のみを取り扱う駅は、その線区の[車扱貨物](../Page/車扱貨物.md "wikilink")取扱駅へ連番を付与後にコードを付与する。例えば、仙台鉄道管理局管内の東北本線で車扱貨物取扱駅の最終が[石越駅](../Page/石越駅.md "wikilink")接続[栗原電鉄線で](https://ja.wikipedia.org/wiki/くりはら田園鉄道線 "wikilink")2347とすれば、当該線区で小口扱貨物のみ取扱駅の最初の[泉崎駅](../Page/泉崎駅.md "wikilink")は2438とする。
7.  期間を限定して貨物を取り扱う駅には、都度コードを通達する。

現在は連絡社線内の駅を識別に2桁の枝番号が使われている。例えば、[川崎貨物駅](../Page/川崎貨物駅.md "wikilink") (4506) で接続する[神奈川臨海鉄道](../Page/神奈川臨海鉄道.md "wikilink") (4507) の[水江町駅](../Page/水江町駅.md "wikilink")は4507-01\[1\]とする。車票のコード欄は4桁であり枝番号記入時は4桁数字の下に書き加えている。

旅客列車駅名コードと貨物取扱駅コードは、第1桁の地区表示以外に関連は無い。

## 番号の変遷

いくつかの駅コードは、次のように変遷している。

  - 4105は、[熊谷駅](https://ja.wikipedia.org/wiki/熊谷駅 "wikilink")で貨物営業を廃止して[熊谷貨物ターミナル駅](../Page/熊谷貨物ターミナル駅.md "wikilink")へ開業時に付与している。
  - 5423は、[岐阜駅](https://ja.wikipedia.org/wiki/岐阜駅 "wikilink")で貨物営業を廃止して[岐阜貨物ターミナル駅](../Page/岐阜貨物ターミナル駅.md "wikilink")へ開業時に付与している。
  - [米原駅](../Page/米原駅.md "wikilink")は、国鉄末期に分割民営化へ向け名古屋鉄道管理局から大阪鉄道管理局へ移管され、5429から6000へ変更されている。
  - [多気駅](../Page/多気駅.md "wikilink")は、[1983年](https://ja.wikipedia.org/wiki/1983年 "wikilink")の貨物営業廃止時は天王寺鉄道管理局内で6406だが、[1987年](https://ja.wikipedia.org/wiki/1987年 "wikilink")の貨物営業再開時は名古屋鉄道管理局内で5518を新規付与されている。
  - 6049は、[姫路駅](https://ja.wikipedia.org/wiki/姫路駅 "wikilink")で貨物営業を廃止して[姫路貨物駅](../Page/姫路貨物駅.md "wikilink")へ開業時に付与している。
  - 6654は、[浜田港駅](https://ja.wikipedia.org/wiki/浜田港駅 "wikilink")を[1982年](../Page/1982年.md "wikilink")に廃止し、[岡見駅](../Page/岡見駅.md "wikilink")の貨物営業再開時に新規付与している。
  - 7001は、[高松貨物ターミナル駅](../Page/高松貨物ターミナル駅.md "wikilink")開業時に付与し、[高松駅は](https://ja.wikipedia.org/wiki/高松駅_\(香川県\) "wikilink")7001から7000へ変更されている。
  - 8032は、[防府駅](../Page/防府駅.md "wikilink")で貨物営業を廃止して[防府貨物駅](../Page/防府貨物駅.md "wikilink")へ開業時に付与している。
