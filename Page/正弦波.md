[thumb](https://ja.wikipedia.org/wiki/ファイル:Sine_Cosine_Graph.png "wikilink") **正弦波**（せいげんは、<em lang="en">sine wave</em>、<em lang="en">sinusoidal wave</em>）は、[正弦関数として観測可能な周期的変化を示す](../Page/三角関数.md "wikilink")[波動](../Page/波動.md "wikilink")のことである。その波形は**正弦曲線**（せいげんきょくせん、<em lang="en">sine curve</em>）もしくは**シヌソイド** (<em lang="en">Sinusoid</em>) と呼ばれ、[数学](../Page/数学.md "wikilink")、[信号処理](https://ja.wikipedia.org/wiki/信号処理 "wikilink")、[電気工学](../Page/電気工学.md "wikilink")およびその他の分野において重要な働きをする。

## 基本形・一般形

[Simple_harmonic_oscillator.gif](https://ja.wikipedia.org/wiki/File:Simple_harmonic_oscillator.gif "fig:Simple_harmonic_oscillator.gif") 固定された観測位置における正弦波は次のような関数として記述することができる（基本形）：

\[y = A\cdot \sin(\omega t - \varphi)\]

ここで、*t* は[時刻](../Page/時刻.md "wikilink") 、*A* は[振幅](../Page/振幅.md "wikilink")（波の中心からの最大偏差）、ω は[角周波数](../Page/角周波数.md "wikilink")、−φ は初期[位相](../Page/位相.md "wikilink")（*t* = 0 における位相）という。

  -
    −φ は位相シフトとも関係がある。例えば、初期位相 −φが負の値であれば、波形全体が未来の時間へシフトされる、すなわち波の到達が遅れる。シフトされる時間は、φ / ω である。

基本形に、波動の発生源からの距離 *x* や[波数](../Page/波数.md "wikilink") *k* 、直流成分（振幅の中心となる値） *D* などを含めて

\[y = A\cdot \sin(kx - \omega t - \varphi) + D\]

という関数の形で波形を記述できるものを**正弦波**と総称する（一般形）。波数は角周波数と以下のような関係にある。

\[k = { \omega \over c } = { 2 \pi f \over c } = { 2 \pi \over \lambda }\]

ここで、λ は[波長](../Page/波長.md "wikilink")、*f* は[周波数](../Page/周波数.md "wikilink")、*c* は[位相速度](../Page/位相速度.md "wikilink")である。

この方程式は1次元の正弦波となるため、上記の一般化された方程式では、時刻 *t* における位置 *x* での波の振幅が導かれる。これは例えば、ワイヤーに沿った波の値と考えることが出来る\[1\]。

コサイン波形（余弦波）もシヌソイドと言われる。これは、正弦波が後方にシフトされたもので波形が同一だからである。

\[\cos(x) = \sin\left({x} + \frac{\pi}{2}\right)\]

なお、正弦関数は[波動方程式](../Page/波動方程式.md "wikilink")・[ヘルムホルツ方程式](../Page/ヘルムホルツ方程式.md "wikilink")を満たす最も基本的な関数である。

## 自然現象における正弦波

上記の通り、正弦波は単一の周波数成分のみを持つ波動であり、厳密な意味では自然界には存在しない。しかし、一般の[物理学](../Page/物理学.md "wikilink")や[電磁気学](../Page/電磁気学.md "wikilink")、[音響学](../Page/音響学.md "wikilink")などでは、観測されるべき波動（すなわち上記の基本形・一般形で表される波動）の振幅が、付随される[雑音](https://ja.wikipedia.org/wiki/雑音 "wikilink")に比べて十分に大きい場合、これを正弦波と見なすことが多い。 この広義の意味での正弦波は自然界でも海の波、音波、光波などで発生する。また、日々の平均気温を年間を通してプロットした際などにも荒いシヌソイドパターンが現れる。

商用電源など発電機から得られる[交流](../Page/交流.md "wikilink")電圧の波形は一般に正弦波形をとる。

## フーリエ級数・フーリエ解析

[1822年](../Page/1822年.md "wikilink")、フランス人数学者の[ジョゼフ・フーリエ](../Page/ジョゼフ・フーリエ.md "wikilink")は、周期的な波動をさまざまな（基本周波数の整数倍の）周波数の正弦波の重ね合わせとして表す方法を発見した。この方法は[フーリエ級数](../Page/フーリエ級数.md "wikilink")またはフーリエ級数展開と呼ばれ、信号処理におけるもっとも基礎的な手法の一つである。

また、単一のパルス波や人の声による不規則な音波といった周期的でない波形も、連続的に変化する異なった周波数の波を重ね合わせて表すことができる。このような一般的で複雑な波を様々な周波数の正弦波に分解して解析する手法は[フーリエ変換](../Page/フーリエ変換.md "wikilink")と呼ばれている。

## 応用

### 音波としての正弦波

人の[耳](../Page/耳.md "wikilink")は単一の正弦波を認識することが出来る。なぜなら、そのような波形を持つ音は人には純粋な音高の音としてはっきりと聞こえるからである。純粋な正弦波に近い音には、[口笛](../Page/口笛.md "wikilink")や、ぬれた指先で[クリスタルグラス](https://ja.wikipedia.org/wiki/クリスタルグラス "wikilink")の縁をなぞって振動させる際に発生する音、そして[音叉](../Page/音叉.md "wikilink")の音がある。このように正弦波として聞こえる音は[純音](../Page/純音.md "wikilink")と呼ばれる。

音波が2つ以上の正弦波によって構成される場合、その中で最も[周波数](../Page/周波数.md "wikilink")が低い正弦波を基準として、その他の正弦波の周波数が基準となる正弦波の周波数の整数倍で構成されるときは、その音波の波形は周期的な[交流](../Page/交流.md "wikilink")波形となる。この音は、人の耳には[楽音](../Page/楽音.md "wikilink")または単音として認識される。 それ以外の2つ以上の正弦波によって構成される音はノイズか[和音](../Page/和音.md "wikilink")、ないしは[うなり](../Page/うなり.md "wikilink")として聞こえる。

#### 音楽への応用

[1950年代](../Page/1950年代.md "wikilink")、正弦波音が[オルガン](../Page/オルガン.md "wikilink")の音に似ているということも好都合であり、[電子音楽](../Page/電子音楽.md "wikilink")の黎明期に愛用された。この時流に沿う形で、[フランス](https://ja.wikipedia.org/wiki/フランス "wikilink")の[作曲家](../Page/作曲家.md "wikilink")[アンリ・プッスール](../Page/アンリ・プッスール.md "wikilink")は「正弦波曲線が、楽曲の理想的な形式」と定義して話題となった。この理論を杓子定規に応用した作品に、[篠原眞](../Page/篠原眞.md "wikilink")の「タンダンス」がある。

また、[1980年代](../Page/1980年代.md "wikilink")には、正弦波に対し[変調を掛けることによって波形を生成する](../Page/周波数変調.md "wikilink")[FM音源](../Page/FM音源.md "wikilink")方式の楽器が発売され、少ないパラメータから算出される多彩な波形によって、一時代を築いた。一部の製品では、正弦波を単体で扱う機能や、予め変換された波形を再生することを支援する機能も実装された。

### 工学への応用

  - [電子工学](../Page/電子工学.md "wikilink")では、[増幅器やその他](../Page/増幅回路.md "wikilink")[電子回路](../Page/電子回路.md "wikilink")の電気的特性を測定するのに[標準信号発生器](../Page/標準信号発生器.md "wikilink")等の[発振器で生成した正弦波を用いることがある](../Page/発振回路.md "wikilink")。
  - [無線工学](https://ja.wikipedia.org/wiki/無線工学 "wikilink")では、[搬送波](https://ja.wikipedia.org/wiki/搬送波 "wikilink")として正弦波を用い、それに振幅や位相の角度で[変調をかけて](../Page/変調方式.md "wikilink")、[無線通信](../Page/無線通信.md "wikilink")を行う。[スーパーヘテロダイン方式の](../Page/スーパーヘテロダイン受信機.md "wikilink")[ラジオ受信機](../Page/受信機.md "wikilink")（[家電](https://ja.wikipedia.org/wiki/家電 "wikilink")や[オーディオ用として一般的に販売されている](../Page/音響機器.md "wikilink")[ラジオ](../Page/ラジオ.md "wikilink")でも一般的な方式）では、[局部発振器](https://ja.wikipedia.org/wiki/局部発振器 "wikilink")で生成した正弦波と受信した信号とを[混合器に入力して](https://ja.wikipedia.org/wiki/混合器_\(ヘテロダイン\) "wikilink")、中間周波数を得る。
