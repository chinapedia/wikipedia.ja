**北海道札幌北陵高等学校**（ほっかいどう さっぽろほくりょうこうとうがっこう、**Hokkaido Sapporo Hokuryo High School**）は、[北海道](https://ja.wikipedia.org/wiki/北海道 "wikilink")[札幌市](https://ja.wikipedia.org/wiki/札幌市 "wikilink")[北区](../Page/北区_\(札幌市\).md "wikilink")[屯田にある公立](../Page/屯田_\(札幌市北区\).md "wikilink")（道立）の[高等学校](https://ja.wikipedia.org/wiki/高等学校 "wikilink")。全日制普通科。

## 概要

  - 周辺は住宅街になっている。

### 学科

  - **普通科**（全日制課程）
      - 定員 320名
      - 45分授業で1日7コマ （7時間授業）
      - 理科の科目は[地学](https://ja.wikipedia.org/wiki/地学 "wikilink")を修学することはできない
      - スキー学習、水泳学習はない
      - 芸術科目は[音楽](../Page/音楽.md "wikilink")、[美術](../Page/美術.md "wikilink")、[書道](https://ja.wikipedia.org/wiki/書道 "wikilink")から選択
      - 男子体育の武道は[柔道](../Page/柔道.md "wikilink")

| 類型              | 第1学年・第2学年前期  | 第2学年後期        | 第3学年前期        | 第3学年後期       |
| --------------- | ------------ | ------------- | ------------- | ------------ |
| 理型              | 共通           | 理I            | 自然科学探求型（理I）   | サイエンスコース（理I） |
| 理II             | 自然科学探求型（理II） | サイエンスコース（理II） |               |              |
| 自然科学探求型（医I）     | メディカルコース（医I） |               |               |              |
| 文型              | 人文科学探求型      | 人文科学探求型       | 人文科学探求コース（文I） |              |
| 人文科学探求コース（文II）  |              |               |               |              |
| 人文科学探求コース（文III） |              |               |               |              |

  - 理I、理II、文I、文II、文IIIは志望する進学先により選択する。
  - 以前は理科3科目を必要とする医学部志願者向けのメディカルコース（医II）があったが、希望者が少数のため、事実上廃止された。

（2011年4月現在）

## 沿革

（沿革節の主要な出典は公式サイト\[1\] ）

  - [1971年](https://ja.wikipedia.org/wiki/1971年 "wikilink")（[昭和](../Page/昭和.md "wikilink")46年）
      - 11月27日 - 北海道教育委員会において、新設高等学校開設要綱を発表する。
      - 12月3日 - 校名を北海道札幌北陵高等学校に決定する。
  - [1972年](../Page/1972年.md "wikilink")
      - 3月13日 - 校旗の意匠を決定する。
      - 3月14日 - 校章と制服を決定する。
      - 4月10日 - 開校式並びに第1回入学式を挙行する。[札幌市立中央中学校](../Page/札幌市立中央中学校.md "wikilink")旧一条校舎（[中央区南](../Page/中央区_\(札幌市\).md "wikilink")2条東6丁目、現在の札幌市民ギャラリーの敷地内）を仮校舎として使用。
  - [1974年](../Page/1974年.md "wikilink")11月13日 - 現在地に校舎完成、移転。
  - [1981年](../Page/1981年.md "wikilink")
      - 7月24日 - 校訓を制定する。
      - 10月3日 - 創立10周年記念式典を挙行する。
  - [1991年](../Page/1991年.md "wikilink")（[平成](../Page/平成.md "wikilink")3年）10月6日 - 創立20周年式典を挙行する。
  - [1997年](https://ja.wikipedia.org/wiki/1997年 "wikilink")4月1日 - 2期制を導入する。
  - [2001年](../Page/2001年.md "wikilink")10月28日 - 創立30周年記念式典を挙行する。
  - [2002年](../Page/2002年.md "wikilink")4月1日 - 新制服に改定する。
  - [2011年](../Page/2011年.md "wikilink")4月1日 - 制服（女子）の一部改定。
  - [2017年](../Page/2017年.md "wikilink")7月 - 制服（夏服）の一部改定。
  - [2019年](../Page/2019年.md "wikilink")3月 - 学校教育目標の改定

### 購買

  - 平日の日中のみ営業。
  - 生徒ホールに設置されており、自動販売機（[キリンビバレッジ](../Page/キリンビバレッジ.md "wikilink")・[サントリー](../Page/サントリー.md "wikilink")）が常時使用可能。
  - 飲料は紙パック、缶、ペットボトルで販売している。
  - 学食はない。

## 通学

### 通学手段

  - バス
  - 自転車
  - 徒歩

[JR北海道](../Page/北海道旅客鉄道.md "wikilink") [学園都市線](https://ja.wikipedia.org/wiki/札沼線 "wikilink")、[札幌市営地下鉄](https://ja.wikipedia.org/wiki/札幌市営地下鉄 "wikilink")を利用する場合は降車後、上記の通学手段を必要とする。

### バス

学校から徒歩1分程の所に[北海道中央バス](../Page/北海道中央バス.md "wikilink")「北陵高校前」停留所が設置され、[麻生駅](https://ja.wikipedia.org/wiki/麻生駅 "wikilink")、[栄町駅](../Page/栄町駅_\(札幌市\).md "wikilink")、[札幌駅](https://ja.wikipedia.org/wiki/札幌駅 "wikilink")北口、[札幌ターミナルなどと結ばれる](../Page/北海道中央バス札幌ターミナル.md "wikilink")。

路線詳細は[北海道中央バス新川営業所](../Page/北海道中央バス新川営業所.md "wikilink")または[北海道中央バス石狩営業所](../Page/北海道中央バス石狩営業所.md "wikilink")を参照。

### 自転車

  - 敷地内に駐輪場があるが、屋根がないので雨が降った際は自転車が水浸しになる。
  - 駐輪場に積雪がある場合はその期間、自転車通学は禁止となる。(ただし積雪がない場合も教職員の駐車場となるため、冬期の自転車通学は原則禁止)

## 教育目標

（2019年3月まで）

  - 1、ひとりひとりを大切にし，その可能性を最大限に伸ばす教育
  - 2、あらゆる機会をとらえ，調和のとれた人間形成をめざす教育
  - 3、たえず向上しようとする旺盛な意欲のもと，充実した気力と強じんな体力を養う教育

（2019年3月から）

  - 1、　自己を開く　　他者の考えにふれ、自己を成長させようとする姿勢
  - 2、　知を啓(ひら)く　　先人の知に学び、視野をひろげようとする姿勢
  - 3、　未来を拓(ひら)く　　新時代の担い手としての使命を探り、社会に貢献しようとする姿勢

## 校訓

  - **自立**　人に甘えず自己の責任を自覚し，自己を確立する
  - **敬愛**　思いやりの心を持ち，自己中心性を脱却する
  - **進取**　常に前向きにとりくみ，自己を高める努力を怠らない
