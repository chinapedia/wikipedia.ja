[言語学](../Page/言語学.md "wikilink")の**ゼロ**は、[発音](https://ja.wikipedia.org/wiki/発音 "wikilink")上は存在しないが、特定の[理論](../Page/理論.md "wikilink")において存在すると見なすものである。記号には [∅](https://ja.wikipedia.org/wiki/∅ "wikilink") を用いる。代用表記としてラテン文字の [Ø](../Page/Ø.md "wikilink") も用いられる。

例えば、[英語](../Page/英語.md "wikilink")の cats  と fish  において、[複数](../Page/複数.md "wikilink")を表すのは前者は  だが、後者は何も無い。この何も無いのを、ゼロがあると見なし、複数形の[接尾辞](../Page/接尾辞.md "wikilink")がそれぞれ - と -∅ であるとする。

[本居宣長](../Page/本居宣長.md "wikilink")は、『詞の玉緒』（[1779年](../Page/1779年.md "wikilink")）で、「は」、「も」、「徒（ただ）」の[係り結び](../Page/係り結び.md "wikilink")が[終止形になることを示した](../Page/終止形_\(文法\).md "wikilink")。この「徒」は[助詞](../Page/助詞.md "wikilink")が付かない場合を意味し、ゼロに当たる。

ただし、ゼロを乱用してはならない。ゼロが唯一の形態であるような[形態素](../Page/形態素.md "wikilink")を設けることは避けるべきである<ref>{{ Citation

## ゼロ接辞

ある[接辞](../Page/接辞.md "wikilink")の体系の中で、一部に接辞が現れない場合、接辞 ∅ を設けるほうが簡潔である。以下に[ドイツ語](../Page/ドイツ語.md "wikilink")の[所有限定詞](../Page/所有限定詞.md "wikilink") mein の[格変化](../Page/格変化.md "wikilink")を示す。男性単数主格および中性単数主対格では mein が使われるが、これを接尾辞 -∅ が付いていると考える。

|    | 単数          | 複数          |
| -- | ----------- | ----------- |
| 男性 | 女性          | 中性          |
| 主格 | mein-**∅**  | mein-**e**  |
| 属格 | mein-**es** | mein-**er** |
| 与格 | mein-**em** | mein-**er** |
| 対格 | mein-**en** | mein-**e**  |

## ゼロ冠詞

[冠詞](../Page/冠詞.md "wikilink")の体系で、無冠詞をゼロと考えることがある。とくに、いわゆる名詞[句](../Page/句.md "wikilink")には必ず[限定詞](../Page/限定詞.md "wikilink")があるという立場では、限定詞としてゼロ冠詞を設ける。以下に英語と[フランス語](https://ja.wikipedia.org/wiki/フランス語 "wikilink")の冠詞を示す。英語の複数不定名詞句はゼロ冠詞を持つ。

|    | 英語          | フランス語         |
| -- | ----------- | ------------- |
| 不定 | 定           | 不定            |
| 単数 | **a** book  | **the** book  |
| 複数 | **∅** books | **the** books |

## ゼロ格

[日本語](../Page/日本語.md "wikilink")では、[文](../Page/文.md "wikilink")中の[名詞](../Page/名詞.md "wikilink")[句](../Page/句.md "wikilink")には[格助詞](https://ja.wikipedia.org/wiki/格助詞 "wikilink")が必要である。例えば「を」は[対格](../Page/対格.md "wikilink")の[標識](../Page/標識.md "wikilink")である。

  - ご飯**を** 食べた。

だが[口語](../Page/口語.md "wikilink")では格助詞を省略できる。これは、格助詞 ∅ が付いていると見なせる。これがゼロ格である\[1\]。

  - ご飯 **∅** 食べた。

## ゼロコピュラ

[コピュラ](../Page/コピュラ.md "wikilink")の変化の中で、一部コピュラを使わない場合がある。例えば[ロシア語](https://ja.wikipedia.org/wiki/ロシア語 "wikilink")では、現在時制ではコピュラを使わず単に主語と補語を並べる。これをゼロコピュラがあると見なす。同様に、[ハンガリー語](https://ja.wikipedia.org/wiki/ハンガリー語 "wikilink")では三人称現在でコピュラが省略されることがあり、これもゼロコピュラと見なせる。

日本語では、疑問文ではコピュラ「だ」が消えるが、これをゼロコピュラと見なせる。

|     | 常体            | 敬体            |
| --- | ------------- | ------------- |
| 平叙文 | これは本**だ**。    | これは本**である**。  |
| 疑問文 | これは本 **∅** か。 | これは本**である**か。 |

## ゼロ関係代名詞

英語で、人間でない[目的語](../Page/目的語.md "wikilink")を示す[関係代名詞](https://ja.wikipedia.org/wiki/関係代名詞 "wikilink")は that または which であるが、省略しても良い。省略した場合、関係代名詞 ∅ があると見なせる。これをゼロ関係代名詞と呼ぶ。

  - the book **that** I read yesterday
  - the book **∅** I read yesterday
