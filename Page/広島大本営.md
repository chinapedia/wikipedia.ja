[right御座所](https://ja.wikipedia.org/wiki/ファイル:Imperial_General_Headquarters_in_Hiroshima.JPG "wikilink")。左上に[広島城](https://ja.wikipedia.org/wiki/広島城 "wikilink")天守が見える。\]\] **広島大本営**（ひろしまだいほんえい）は、[1894年](../Page/1894年.md "wikilink")（[明治](../Page/明治.md "wikilink")27年）に勃発した[日清戦争](https://ja.wikipedia.org/wiki/日清戦争 "wikilink")の戦争指揮のために[広島県](../Page/広島県.md "wikilink")[広島市](https://ja.wikipedia.org/wiki/広島市 "wikilink")の[広島城](https://ja.wikipedia.org/wiki/広島城 "wikilink")（現[中区](../Page/中区_\(広島市\).md "wikilink")[基町](https://ja.wikipedia.org/wiki/基町 "wikilink")）内に設置された、[大日本帝国](https://ja.wikipedia.org/wiki/大日本帝国 "wikilink")軍の最高[統帥](https://ja.wikipedia.org/wiki/統帥 "wikilink")機関である[大本営](https://ja.wikipedia.org/wiki/大本営 "wikilink")である。

## 概要

[rightで結ばれている](https://ja.wikipedia.org/wiki/ファイル:Hiroshima_map_circa_1930.PNG "wikilink")。\]\] [大本営](https://ja.wikipedia.org/wiki/大本営 "wikilink")は[1893年](../Page/1893年.md "wikilink")（明治26年）[5月19日](../Page/5月19日.md "wikilink")に[勅令](https://ja.wikipedia.org/wiki/勅令 "wikilink")第52号戦時大本営条例によって法制化された制度であり、[日清戦争](https://ja.wikipedia.org/wiki/日清戦争 "wikilink")において初めて設置された。このときの大本営は1894年[6月5日](https://ja.wikipedia.org/wiki/6月5日 "wikilink")に[東京](../Page/東京.md "wikilink")の参謀本部内に設置され、同年[8月5日](../Page/8月5日.md "wikilink")に皇居内に移った。

その後、当時東京を起点とする鉄道（[山陽鉄道](https://ja.wikipedia.org/wiki/山陽鉄道 "wikilink")）網の西端であったこと（[広島駅](https://ja.wikipedia.org/wiki/広島駅 "wikilink")）、また大型船が運用出来る港（宇品港（現・[広島港](https://ja.wikipedia.org/wiki/広島港 "wikilink")））が有ったことで、前線に向かう[兵站](https://ja.wikipedia.org/wiki/兵站 "wikilink")基地となった広島市に移ることとなった\[1\]。

[9月13日](../Page/9月13日.md "wikilink")に大本営が[宮中](https://ja.wikipedia.org/wiki/宮中 "wikilink")からこの地に移転し、2日後の[15日には戦争指揮のために](../Page/9月15日.md "wikilink")[明治天皇](https://ja.wikipedia.org/wiki/明治天皇 "wikilink")が移った。このため、[行宮](https://ja.wikipedia.org/wiki/行宮 "wikilink")の役割も果たした。10月9日、非常時に備え[縮景園](https://ja.wikipedia.org/wiki/縮景園 "wikilink")は大本営副営と定められ、明治天皇の居所として清風館があてられた\[2\]。明治天皇は日清講和条約（[下関条約](../Page/下関条約.md "wikilink")）調印後の[1895年](../Page/1895年.md "wikilink")（明治28年）[5月30日](../Page/5月30日.md "wikilink")までの227日間この地で指揮を執った後、東京に還幸した。大本営はその後も[台湾](https://ja.wikipedia.org/wiki/台湾 "wikilink")の統治機構整備など戦後処理のために広島に留まり、[1896年](../Page/1896年.md "wikilink")（明治29年）[4月1日](../Page/4月1日.md "wikilink")に大本営解散の詔勅によって解散した。

この時期、1894年10月に招集された[第7回帝国議会](https://ja.wikipedia.org/wiki/第7回帝国議会 "wikilink")は広島の[広島臨時仮議事堂](https://ja.wikipedia.org/wiki/広島臨時仮議事堂 "wikilink")で開会された（議事堂は西練兵場内に建設された）。[国](../Page/国.md "wikilink")の[立法](../Page/立法.md "wikilink")・[行政](../Page/行政.md "wikilink")・[軍事](../Page/軍事.md "wikilink")の最高機関が一時的とはいえ[広島市](https://ja.wikipedia.org/wiki/広島市 "wikilink")に集積したことで、広島市は臨時の[首都](../Page/首都.md "wikilink")の機能を担った。これは[明治維新](../Page/明治維新.md "wikilink")以降、首都機能が東京から離れた唯一の事例である（[日本の首都](../Page/日本の首都.md "wikilink")を参照）。

## 施設と現況

### 大本営

{{ external media | topic = [広島県立文書館](https://ja.wikipedia.org/wiki/広島県立文書館 "wikilink")所有の絵葉書。 | align = | width = 200px | image1 = [飛行機より見たる広島大本営跡](http://www.pref.hiroshima.lg.jp/soshiki_file/monjokan/picture/images/200407-1418_01.jpg) 北側三ノ丸から撮影。 | image3 = [玉座](http://www.pref.hiroshima.lg.jp/soshiki_file/monjokan/picture/images/200407-1621_09.jpg) | image4 = [御椅子と御火鉢](http://www.pref.hiroshima.lg.jp/soshiki_file/monjokan/picture/images/200407-1621_08.jpg) | image5 = [御衣桁](http://www.pref.hiroshima.lg.jp/soshiki_file/monjokan/picture/images/200407-1621_07.jpg) | image6 = [大本営設置当時の広島城表門](http://www.pref.hiroshima.lg.jp/soshiki_file/monjokan/picture/images/200407-1391_01.jpg) }} {{ external media | topic = [アメリカ国立公文書記録管理局](https://ja.wikipedia.org/wiki/アメリカ国立公文書記録管理局 "wikilink")が所有する米軍撮影写真。 | align = | width = 200px | image1 = [Hiroshima aerial A3390](http://www.japanairraids.org/wp-content/uploads/2010/09/Hiroshima-aerial-A3390.jpg) 南側から撮影。 | image2 = [Hiroshima aerial A3393](http://www.japanairraids.org/wp-content/uploads/2010/09/Hiroshima-aerial-A3393.jpg) 東側から撮影。写真中央付近が現在の[中国放送](../Page/中国放送.md "wikilink")本社敷地、上半分が本丸にあたる。大本営および皇后御座所共に木材が散乱している状況がわかる。 }} 元々この地には本丸御殿があったが火災により全焼、1877年（明治10年）跡地に広島[鎮台](https://ja.wikipedia.org/wiki/鎮台 "wikilink")司令部として建設されたものである\[4\]。1888年（明治21年）[第5師団発足以降はその司令部として使われていた](../Page/第5師団_\(日本軍\).md "wikilink")\[5\]。そして、大本営として用いられた。

[洋風建築の](https://ja.wikipedia.org/wiki/西洋館 "wikilink")[木造](../Page/木構造_\(建築\).md "wikilink")2階建て、[広島城](https://ja.wikipedia.org/wiki/広島城 "wikilink")本丸におかれた\[6\]。1階に大臣室や侍従職室、2階に御座所・御召替所・侍従長室・軍議室などがあり\[7\]、御座所は執務室と寝室を兼ね、近くに[剣璽](https://ja.wikipedia.org/wiki/剣璽 "wikilink")が置かれた\[8\]。

明治天皇はほぼここに留まって深夜まで政務を行い、質素な生活をしていたと伝えられている\[9\]。この戦争中に見せた「精力的な活動」「質素な生活」つまり「聖徳」は明治後期における天皇神格化の中で重要なイメージ戦略として用いられており、例えば当時普通の古びた備品を使う様は戦前の[教科書](https://ja.wikipedia.org/wiki/教科書 "wikilink")に描かれている\[10\]。後に明治天皇は当時のことを詠んでいる\[11\]。

1896年（明治29年）大本営解散後は第5師団管理の下「大本営址」は文化財として保護され\[12\]、[史蹟名勝天然紀念物保存法](../Page/史蹟名勝天然紀念物保存法.md "wikilink")施行により「史蹟明治二十七八年戦役広島大本営」として[1926年](../Page/1926年.md "wikilink")（[大正](../Page/大正.md "wikilink")15年）に国の[史跡](../Page/史跡.md "wikilink")（旧史跡）に指定され\[13\]た。1928年（昭和3年）から広島城天守の一般開放が始まり、あわせて名所として公開された\[14\]。

[1945年](../Page/1945年.md "wikilink")（[昭和](../Page/昭和.md "wikilink")20年）8月6日、[広島市への原子爆弾投下](https://ja.wikipedia.org/wiki/広島市への原子爆弾投下 "wikilink")により建物は全て崩壊した\[15\]（[爆心地](https://ja.wikipedia.org/wiki/爆心地 "wikilink")から約900メートル）。1948年（昭和23年）、戦後の軍国主義排除の風潮の中で、史跡指定が解除され塗りつぶされた。\[16\]。現在では、建物の基礎および礎石と、一部文字が消された[石碑](https://ja.wikipedia.org/wiki/石碑 "wikilink")が残っているのみとなっている。

### 昭憲皇太后御座所

{{ external media | topic = [広島県立文書館](https://ja.wikipedia.org/wiki/広島県立文書館 "wikilink")所有の絵葉書。 | align = | width = 200px | image1 = [広島大本営跡と皇后御座所](http://www.pref.hiroshima.lg.jp/soshiki_file/monjokan/picture/images/200407-1417_01.jpg) カラー写真 | image2 = [昭憲皇太后御座所](http://www.pref.hiroshima.lg.jp/soshiki_file/monjokan/picture/images/200407-1621_06.jpg) }}

広島城天守から真南、大本営から西南西方向に位置した。元々は第5師団監査部の建物で、大本営設置後はその事務所として使われていた\[17\]。

当初は1895年（明治28年）3月13日[昭憲皇太后](../Page/昭憲皇太后.md "wikilink")広島行啓で進められていたが都合により延期、同年3月19日皇后行啓した際に御座所として用いられた\[18\]。この地で皇后も精力的に慰問を行っていたことが伝えられる\[19\]。

これも現在は基礎石のみが残る。

  -
    本丸内にあり大本営から東南東方向に位置する。1898年（明治31年）広島軍用水道布設の際に造られた池で、牛田水源地（現[牛田浄水場](https://ja.wikipedia.org/wiki/牛田浄水場 "wikilink")）から[神田橋水道橋を通って](../Page/神田橋_\(京橋川\).md "wikilink")[京橋川](https://ja.wikipedia.org/wiki/京橋川 "wikilink")を渡り城内へ上水道が引き込まれた時に一緒に整備された\[20\]。1925年（大正14年）に「桜の池」と命名\[21\]\[22\]。
    現在はコンクリートで覆われた形状は残っているが、水は入れられていない\[23\]。

<!-- end list -->

  - 大本営跡の南側は元々は前庭にあたり、車周りの植え込みの一つだった[クロガネモチ](https://ja.wikipedia.org/wiki/クロガネモチ "wikilink")（[広島市被爆樹木リスト](http://www.city.hiroshima.lg.jp/www/contents/1447323970372/index.html)）は被爆にも耐えた被爆樹木として現在も生息している\[24\]。
  - 当時御座所にあった[金屏風](https://ja.wikipedia.org/wiki/金屏風 "wikilink")は大本営解散後もそのまま置かれていた。[戦中](https://ja.wikipedia.org/wiki/戦中 "wikilink")に青崎国民学校（現[広島市立青崎小学校](https://ja.wikipedia.org/wiki/広島市立青崎小学校 "wikilink")）に疎開させていたため被爆による消失から免れ、現在は[広島県立文書館](https://ja.wikipedia.org/wiki/広島県立文書館 "wikilink")が所蔵している\[25\]
  - 従者は市中心部の旅館に宿泊している。最も多いのが大本営に近い[大手町](../Page/大手町_\(広島市\).md "wikilink")。これは帝国議会開催中の議員も同様で、市内のみならず周辺町村、現在の広島市域全域で宿泊した。彼ら宿泊所を提供した市民は後に、天覧の栄誉を受けている。
  - 嘉仁皇太子親王行啓の際には憲兵本部が宿泊所にあてられた。
