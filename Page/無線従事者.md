**無線従事者**（むせんじゅうじしゃ）とは、[電波法](../Page/電波法.md "wikilink")に定める[無線設備](https://ja.wikipedia.org/wiki/無線設備 "wikilink")の操作又はその監督を行う者であって、[総務大臣](../Page/総務大臣.md "wikilink")の[免許](../Page/免許.md "wikilink")を受けたものをいう。[業務独占資格](../Page/業務独占資格.md "wikilink")\[1\]\[2\]であり、[総務](../Page/総務省.md "wikilink")[省令](../Page/省令.md "wikilink")で定める簡易な操作\[3\]以外の操作を要する[無線局](https://ja.wikipedia.org/wiki/無線局 "wikilink")に対する[必置資格](https://ja.wikipedia.org/wiki/必置資格 "wikilink")としての性格も有する\[4\]。

## 定義

[電波法](../Page/電波法.md "wikilink")第2条では、次のとおり定義されている。

  - 「[無線設備](https://ja.wikipedia.org/wiki/無線設備 "wikilink")」とは、無線電信、無線電話その他電波を送り、又は**受ける**ための電気的設備をいう\[5\]。
  - 「[無線局](https://ja.wikipedia.org/wiki/無線局 "wikilink")」とは、無線設備及び無線設備の操作を行う者の総体をいう。**但し、受信のみを目的とするものを含まない**\[6\]。
  - 「**無線従事者**」とは、無線設備の操作又はその[監督を行う者であつて](https://ja.wikipedia.org/wiki/主任無線従事者 "wikilink")、[総務大臣](../Page/総務大臣.md "wikilink")の[免許](../Page/免許.md "wikilink")を受けたものをいう\[7\]。

ここでいう「無線局」は、電波法第4条ただし書にあるものを除き、総務大臣の免許を受けなければならない。また、**無線局**の無線設備の操作を行う者は、電波法第40条に定める無線従事者の免許を受けた者（あるいは[主任無線従事者](https://ja.wikipedia.org/wiki/主任無線従事者 "wikilink")の監督下にある者）でなければならない\[8\]\[9\]。

無線従事者は、電波法第40条の区分に従い、[政令](../Page/政令.md "wikilink")[電波法施行令](https://ja.wikipedia.org/wiki/電波法施行令 "wikilink")第3条に操作範囲が定められ、その技能の程度は[無線従事者規則](../Page/無線従事者規則.md "wikilink")に規定されている。 これは、電波法の目的が「電波の公平且つ能率的な利用を確保することによって、[公共の福祉](../Page/公共の福祉.md "wikilink")を増進すること」\[10\]であり、これを達成するために、無線設備の操作又はその[監督を行う者に](https://ja.wikipedia.org/wiki/主任無線従事者 "wikilink")、資格ごとにその最低限の技能・規範を証明し免許することとしているからである。

### 主任無線従事者

1990年（平成2年）より施行された制度であり、[無線従事者免許証](../Page/無線従事者免許証.md "wikilink")の資格を持たない者であっても、無線局に選任された**[主任無線従事者](https://ja.wikipedia.org/wiki/主任無線従事者 "wikilink")**の指揮監督のもと、その主任無線従事者の免許の操作範囲内に限り無線設備の操作を行うことができる\[11\]\[12\]。

主任無線従事者になるためには、一定の業務経歴を有すると共に、[主任無線従事者講習を受講しなければならない](https://ja.wikipedia.org/wiki/主任無線従事者#講習 "wikilink")。

この制度は、無線従事者の確保が難しい[免許人](https://ja.wikipedia.org/wiki/免許人 "wikilink")であっても、無線局の運用を維持することが出来るよう、無線従事者でないものについても主任無線従事者の指揮監督下で無線局の運用ができるようにするための措置である。

ただし、[モールス符号](../Page/モールス符号.md "wikilink")による無線電信操作・その他総務省令で定める無線設備の操作には適用されない\[13\]。また、[アマチュア無線局](https://ja.wikipedia.org/wiki/アマチュア無線局 "wikilink")の無線設備の操作に関しても、適用されない\[14\]。アマチュア無線におけるスクールコンタクトなどの特例は、電波法第39条の13に基づく電波法施行規則第34条の10\[15\]により許容されたものであり、主任無線従事者の制度によるものではない\[16\]。

## 種別

無線従事者は、1950年（昭和25年）6月の電波法制定時に定義されたもので、次の四種類に大別されていた。

  - **[無線通信士](../Page/無線通信士.md "wikilink")**
  - **[無線技術士](https://ja.wikipedia.org/wiki/無線技術士 "wikilink")**
  - **[アマチュア無線技士](https://ja.wikipedia.org/wiki/アマチュア無線技士 "wikilink")**
  - **[特殊無線技士](https://ja.wikipedia.org/wiki/特殊無線技士 "wikilink")**

この内、無線通信士は[無線電信法](../Page/無線電信法.md "wikilink")下におけるそれを、無線技術士は同法下における電気通信技術者を継承したものである。また、アマチュア無線技士と特殊無線技士は、電波法において制定されたものである。 変遷を含め、詳細は各項目を参照。

1990年（平成2年）5月の改正電波法令の施行により、資格が海上、航空、陸上の利用分野別に再編 \[17\] され、次の八種類に大別された。

  - **[総合無線通信士](https://ja.wikipedia.org/wiki/総合無線通信士 "wikilink")**
  - **[海上無線通信士](../Page/海上無線通信士.md "wikilink")**
  - **[海上特殊無線技士](../Page/海上特殊無線技士.md "wikilink")**
  - **[航空無線通信士](../Page/航空無線通信士.md "wikilink")**
  - **[航空特殊無線技士](../Page/航空特殊無線技士.md "wikilink")**
  - **[陸上無線技術士](../Page/陸上無線技術士.md "wikilink")**
  - **[陸上特殊無線技士](../Page/陸上特殊無線技士.md "wikilink")**
  - **アマチュア無線技士**

詳細は各項目を参照。

## 取得

無線従事者の免許を受けようとする者は、電波法第41条第2項各号に基づき、総務大臣の免許を受けなければならない\[52\]。取得にあたり年齢・経歴・国籍などの制限は無い。 ただし、国家試験、養成課程・認定講習課程修了試験の設問は[日本語](../Page/日本語.md "wikilink")のみである。

### 国家試験

全ての資格について[日本無線協会](../Page/日本無線協会.md "wikilink")により実施される。次に挙げる者は科目が免除される。

  - 無線通信士、陸上無線技術士の科目合格者は、合格の翌月から原則として3年間、その科目を免除される\[53\]。
  - 無線通信士の電気通信術の科目合格者は、合格の翌月から原則として3年間、同等またはそれ以下の能力の無線通信士の電気通信術の科目を免除される\[54\]。
  - 総務大臣が告示する学校等の卒業者は、卒業の日から原則として3年間、無線工学の基礎、電気通信術及び英語の一部または全部を免除される\[55\]。
  - 一定の無線従事者、またはその資格による一定の業務経歴を有する者は、一部の科目が免除される\[56\]。
  - [電気通信主任技術者](../Page/電気通信主任技術者.md "wikilink")、[工事担任者](https://ja.wikipedia.org/wiki/工事担任者 "wikilink")\[57\]は、一部の科目が免除される\[58\]。

最近\[59\]の国家試験問題及び合格速報\[60\]は日本無線協会の公式[サイトで公開される](../Page/ウェブサイト.md "wikilink")。但し、統一日程で実施しない第三級・第四級アマチュア無線技士は除く。

現在\[61\]実施される国家試験での[計算尺](../Page/計算尺.md "wikilink")の使用はできない。

### 養成課程

第三級・第四級海上無線通信士、航空無線通信士、海上・航空・陸上特殊無線技士、第二級・第三級・第四級アマチュア無線技士については、[養成課程を修了することによって](https://ja.wikipedia.org/wiki/無線従事者養成課程 "wikilink")、無線従事者の免許を受けることが出来る\[62\]。

#### 長期型養成課程

第三級・第四級海上無線通信士、航空無線通信士、海上・航空・陸上特殊無線技士については総務大臣認定の学校等が開設する教育課程を修了することによって、無線従事者の免許を受けることが出来る\[63\]。

### 学校卒業

次の[学校](https://ja.wikipedia.org/wiki/学校 "wikilink")\[64\]で、無線通信に関する所定の科目を履修して卒業すれば、無線従事者の免許を受けることが出来る\[65\]。

  - [大学](../Page/大学.md "wikilink")\[66\]:第一級陸上特殊無線技士、第二級海上特殊無線技士、第三級海上特殊無線技士
    [短期大学](../Page/短期大学.md "wikilink")、[高等専門学校](../Page/高等専門学校.md "wikilink"):第二級陸上特殊無線技士、第二級海上特殊無線技士、第三級海上特殊無線技士
    [高等学校](https://ja.wikipedia.org/wiki/高等学校 "wikilink")、[中等教育学校](../Page/中等教育学校.md "wikilink"):第三級陸上特殊無線技士、第二級海上特殊無線技士

### 認定講習課程

第一級・第二級総合無線通信士、海上無線通信士、陸上無線技術士は、所定の資格により業務経歴を有する者が[認定講習課程を修了することにより与えられる](https://ja.wikipedia.org/wiki/無線従事者認定講習課程 "wikilink")\[67\]。

### 資格、業務経歴等

第一級総合無線通信士、第二級海上無線通信士、第二級陸上特殊無線技士は、所定の資格により業務経歴を有する者に与えられる\[68\]。

### 欠格事由

下記の者には、無線従事者の免許を与えないことがある\[69\]。

1.  電波法に規定する罪を犯し罰金以上の刑に処せられ、その執行を終わり、又はその執行を受けることがなくなった日から2年を経過しない者
2.  無線従事者の免許を取り消され、取消しの日から2年を経過しない者
3.  著しく心身に欠陥があつて無線従事者たるに適しない者

3.の事例として無線従事者規則第45条により精神病者、耳の聞こえない者、口の利けない者又は目の見えない者は、与えられないとしている。ただし、

  - 総務大臣または[総合通信局](../Page/総合通信局.md "wikilink")長\[70\]が、その資格の無線従事者として無線設備の操作に支障の無いと認める者
  - 精神病者以外の下表の者

|                       |                                             |
| --------------------- | ------------------------------------------- |
| nowrap|耳の聞こえる者で口の利ける者 | nowrap|第一級・第二級・第三級・第四級アマチュア無線技士、第三級陸上特殊無線技士 |
| 目の見える者                | 第一級・第二級・第三級・第四級アマチュア無線技士                    |
| 上に掲げる以外の者             | 第一級・第二級・第三級アマチュア無線技士                        |

には、免許が与えられる。

### 無線従事者免許証

無線従事者の免許が与えられたときには、総務大臣または総合通信局長が免許証を交付する\[71\]。

詳細は**[無線従事者免許証](../Page/無線従事者免許証.md "wikilink")**を参照。

### 無線従事者原簿

総務大臣は、無線従事者原簿を備え付け、免許に関する事項を記載する\[72\]。

無線従事者原簿に記載される免許に関する事項は次のとおり\[73\]。

1.  無線従事者の資格別
2.  免許年月日及び免許証の番号
3.  氏名及び生年月日
4.  免許証を訂正され、又は再交付された者であるときは、その年月日
5.  免許を取り消され若しくは業務に従事することを停止された者又は電波法上の罪を犯し刑に処せられた者であるときはその旨並びに理由及び年月日
6.  その他総務大臣が必要と認める事項
