**千葉県道300号上高根北袖線**（ちばけんどう300ごう かみたかねきたそでせん）は、[千葉県](https://ja.wikipedia.org/wiki/千葉県 "wikilink")[市原市](../Page/市原市.md "wikilink")と[袖ケ浦市](https://ja.wikipedia.org/wiki/袖ケ浦市 "wikilink")を結ぶ[一般県道](../Page/都道府県道.md "wikilink")。

## 概要

[230px](https://ja.wikipedia.org/wiki/ファイル:Chiba_Route_143_and_Chiba_Route_300_at_Ichihara.JPG "wikilink")

  - 起点：市原市上高根（信号交差点、[千葉県道143号南総昭和線](../Page/千葉県道143号南総昭和線.md "wikilink")交点）
  - 終点：袖ケ浦市北袖（北袖インターチェンジ、[国道16号](../Page/国道16号.md "wikilink")交点）

### 交通状況

[平成22年](https://ja.wikipedia.org/wiki/平成22年 "wikilink")([2010年](https://ja.wikipedia.org/wiki/2010年 "wikilink"))時の上高根北袖線の各観測地点での自動車類交通量と幅員構成については以下の通りになっている\[1\]。

|                  |                       |            |
| ---------------- | --------------------- | ---------- |
| **観測地点**         | **24時間自動車類交通量（上下合計）** | **幅員構成**   |
| style= |小型車      | style= |大型車           | style= |合計 |
| 袖ヶ浦市川原井1847-24地先 | 4,693台                | 851台       |
| 市原市椎津3143地先      | 5,390台                | 1,928台     |

また車線数はいずれの地点も2車線区間である。
