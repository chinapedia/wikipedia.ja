**グジャラーティー文字**（グジャラーティーもじ、、）は、[インド](../Page/インド.md "wikilink")の[ブラーフミー文字](https://ja.wikipedia.org/wiki/ブラーフミー文字 "wikilink")から派生した文字の1つ。**グジャラート文字**ともいう。主にインドの[グジャラート州](https://ja.wikipedia.org/wiki/グジャラート州 "wikilink")で話されている[グジャラート語](https://ja.wikipedia.org/wiki/グジャラート語 "wikilink")を表記するのに用いられる。[アブギダ](https://ja.wikipedia.org/wiki/アブギダ "wikilink")の一種。書字方向は**左から右へ**の横書きである。単語と単語の間は英文のように分かち書きする。

## 概要

グジャラーティー文字は、現在インドにおいて[ヒンディー語](https://ja.wikipedia.org/wiki/ヒンディー語 "wikilink")などの多言語に用いられる[デーヴァナーガリー](https://ja.wikipedia.org/wiki/デーヴァナーガリー "wikilink")文字をもとに、インド・グジャラート州の地域で[16世紀](../Page/16世紀.md "wikilink")ごろに派生した文字である。したがって、デーヴァナーガリー文字における、単語中を貫通して書き記す横棒（シローレーカー）を取り除けばグジャラーティー文字とほぼ同じ字形となる字母も多い。

グジャラーティー文字はその音韻表記の手法により[アブギダ](https://ja.wikipedia.org/wiki/アブギダ "wikilink")という[文字体系に分類される](https://ja.wikipedia.org/wiki/文字#分類 "wikilink")。これは、子音文字が単独の[子音](../Page/子音.md "wikilink")を表さず、子音の直後につづく[母音](../Page/母音.md "wikilink")（[随伴母音](https://ja.wikipedia.org/wiki/随伴母音 "wikilink") inherent vowel とよばれる）をも含んだ音節を示すものである。グジャラーティー文字の場合、子音の基礎字母に母音 a () が随伴母音としてつく文字である。例を示すと、子音字母 **** は、ka () の音節を示す。随伴母音でない母音のついた音節を示すには、たとえば ki を示す場合、**** のように付加文字をつける。母音のつかない、子音だけの音を示すには、[ヴィラーマ](https://ja.wikipedia.org/wiki/ヴィラーマ "wikilink")という記号を用いて **** (k) と書く。ほかにも、二重子音を表すのに合成字母がつくられる（後述）。グジャラーティー文字やデヴァナーガリー文字のみならず[インド系文字はすべてこのような書字システムにしたがう](../Page/ブラーフミー系文字.md "wikilink")。なお、参考のために記すと、英語などの表記に用いられる[ラテン文字](../Page/ラテン文字.md "wikilink")は子音と母音が完全に分離して表記され、しかも母音記号（a など）が付加文字としてではなく単独の文字として使用される文字体系であり、アブギダと区別して[アルファベット](https://ja.wikipedia.org/wiki/アルファベット "wikilink")と呼ばれる。

### 母音aについて

અ (a) の母音は、単語内での位置によっては、文字上では表記されていても発音されない。

  - 語末では発音されない。
  - 直後の音節の子音につく母音が発音される場合は、発音しない。

ただしこれらの規則は、二重子音字の場合は必ずしも適用されない。

## 子音

グジャラーティー文字の子音字は次のようなものであり、（最後の二重子音の2文字を除いて）下表の順番を辞書順とする。さらに発音（ラテン文字への翻字と発音記号 ([IPA](../Page/国際音声記号.md "wikilink")) も併記する。概要で述べたように、これらの子音字は、単独ではそれぞれ「子音+a」を表している。子音に後続する母音がa以外のものを表すためには、各字母の上下左右に前述した母音付加記号を加える。

この配置は、似た発音同士を並べている点で合理的であり、[デーヴァナーガリー](https://ja.wikipedia.org/wiki/デーヴァナーガリー "wikilink")など他の北方[インド系文字](https://ja.wikipedia.org/wiki/インド系文字 "wikilink")の並べ方と同じである。

グジャラート語は母音間の子音の弱化に特徴があり、[帯気音](https://ja.wikipedia.org/wiki/帯気音 "wikilink")はしばしば摩擦音化する（有声帯気音ではさらに母音が[息もれ声](https://ja.wikipedia.org/wiki/息もれ声 "wikilink")をともなう）が、正書法上は閉鎖音として記される。

## 結合文字

グジャラーティー文字は（そしてデーヴァナーガリーなど他の主要なインド系文字もそうであるが）、"kt"とか"nt"などの二重子音を記述する場合、文字の一部を省いてつなぎ合わせたような新たな文字をつくる。これを「結合文字」という。例を挙げると、グジャラート語のあいさつ「こんにちは」はであるが、このうちのが結合文字である（この場合は）。

結合文字の作り方にはある程度の法則があるが、時にそれに従わない文字もある。この結合文字作成のややこしさゆえに、コンピュータによるグジャラーティー文字の入力システムの構築は難題である（[複雑なテキスト配置](../Page/複雑なテキスト配置.md "wikilink")の項目を参照）。結合文字に対応するすべのない[タイプライター](../Page/タイプライター.md "wikilink")などにより作成された文書では、随伴母音を取り去る補助記号が多用される傾向にある。

グジャラーティー文字における結合文字の作り方は、この文字の基になったデーヴァナーガリーのそれによく似ている。中には、結合するに際してデーヴァナーガリーの字形を使用するものもある。作り方はまず、二重子音のうちの最初の子音の文字が、次のうちどちらに属すかによって異なる。

1.  文字の右端が縦棒の字画で終わる文字（など。こちらが多数派）
2.  1.に属さないもの（など）

            例）<span class="Unicode"> <span style="font-size:20px;">ક્ર</span> kra (ક્ k + ર ra), <span style="font-size:20px;">ગ્ર</span> gra (ગ્ g + ર ra), <span style="font-size:20px;">ઝ્ર</span> jhra (ઝ્ jh + ર ra), <span style="font-size:20px;">ત્ર</span> tra</span>（<span class="Unicode">ત્ t + ર ra</span>、<span class="Unicode">ત્ </span>が特殊な形に変わる）,<span class="Unicode"> <span style="font-size:20px;">દ્ર</span> dra </span>(<span class="Unicode">દ્ d + ર ra</span>、<span class="Unicode">દ્</span> はデーヴァナーガリー形を使用）,<span class="Unicode"> <span style="font-size:20px;">ભ્ર</span> bhra (ભ્ bh + ર ra), <span style="font-size:20px;">શ્ર</span> śra</span>（<span class="Unicode">શ્ ś + ર ra</span>、<span class="Unicode">શ્</span> の変形は前述）, <span class="Unicode"><span style="font-size:20px;">હ્ર</span> hra</span>（<span class="Unicode">હ્ h + ર ra</span>、前述のとおり別形を用いる）。
    2.  1番目の子音字が<span class="Unicode"> ર (ra) </span>– 前述のレーパを用いる。<span class="Unicode"><span style="font-size:20px;">ર્ર</span> rra</span>（ર્ r + ર ra</span>、前述のとおり別形）
    3.  その他<span class="Unicode"> (ઙ ṅa, છ cha, જ ja, ટ ṭa, ઠ ṭha, ડ ḍa, ઢ ḍha) </span>– 下部に「へ」の字のようなものをつける。
          -
            例）<span class="Unicode"> <span style="font-size:20px;">જ્ર</span> jra (જ્ j + ર ra), <span style="font-size:20px;">ઢ્ર</span> ḍhra (ઢ્ ḍh + ર ra)</span>
  - 2番目の <span class="Unicode">ય (ya) </span>は決して下につかない。
      -
        例）<span class="Unicode"> <span style="font-size:20px;">ડ્ય</span> ḍya</span>（<span class="Unicode">ડ્ ḍ + ય ya</span>、前述の規則どおりならば下につくはずである）
  - 2番目が <span class="Unicode">ન (na)</span> のうち、次の2つの結合文字は不規則的である。
      -
        <span class="Unicode"> <span style="font-size:20px;">શ્ન</span> śna (શ્ ś + ન na); <span style="font-size:20px;">ન્ન</span> nna (ન્ n + ન na)</span>

上記以外のものでは、次の3例は特殊である。<span class="Unicode"><span style="font-size:20px;">ક્ષ</span> kṣa (ક્ k + ષ ṣa); <span style="font-size:20px;">જ્ઞ</span> jña </span>（<span class="Unicode">જ્ j + ઞ ña</span>、発音は前述のとおり ）;<span class="Unicode"> <span style="font-size:20px;">ત્ત</span> tta (ત્ t + ત ta)</span>

なお、これらの結合文字はグジャラート語本来の単語やサンスクリット由来の単語を記述するのに使い、英語などからの近代の外来語を記述するのにはほとんど用いられない。[母音\#母音aについて](https://ja.wikipedia.org/wiki/母音#母音aについて "wikilink")の節にて述べたように読まない短母音 a の規則を利用して記述するのが普通である。

| ૦ | ૧ | ૨ | ૩ | ૪ | ૫ | ૬ | ૭ | ૮ | ૯ |

なお、使用法は算用数字とまったく同じである。例：  = 906 = 九百六。
