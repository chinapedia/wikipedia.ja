[thumb](https://ja.wikipedia.org/wiki/ファイル:Kanachu_Fujisawa_branch.jpg "wikilink") **神奈川中央交通東・藤沢営業所**（かながわちゅうおうこうつうひがし・ふじさわえいぎょうしょ）とは、[神奈川県](https://ja.wikipedia.org/wiki/神奈川県 "wikilink")[藤沢市](../Page/藤沢市.md "wikilink")辻堂新町3-4-23に位置する[神奈川中央交通](../Page/神奈川中央交通.md "wikilink")の子会社・神奈川中央交通東の営業所である。車庫の別称は**高山車庫**で、主に藤沢市内の路線を管轄し、一部は[鎌倉市](../Page/鎌倉市.md "wikilink")、[横浜市](../Page/横浜市.md "wikilink")、[茅ヶ崎市](../Page/茅ヶ崎市.md "wikilink")に乗り入れる。最寄の停留所は「高山車庫」である。

元来は独立した「神奈川中央交通**藤沢営業所**」（略号「**ふ**」）であったが、[2005年](../Page/2005年.md "wikilink")[4月16日](../Page/4月16日.md "wikilink")に**[藤沢神奈交バス](../Page/藤沢神奈交バス.md "wikilink")**藤沢営業所へ全面委託するとともに、[茅ヶ崎営業所が管轄する操車所に組織変更されて](../Page/神奈川中央交通茅ヶ崎営業所.md "wikilink")「**茅ヶ崎営業所藤沢操車所**」となり神奈川中央交通の営業所としてはいったん廃止された。

[2017年](../Page/2017年.md "wikilink")[1月1日](../Page/1月1日.md "wikilink")、神奈川中央交通グループの乗合バス事業再編にともない、管理委託先を藤沢神奈交バスから神奈川中央交通東に変更し、「**神奈川中央交通東・藤沢営業所**」となる。所属車両に社番として表記される営業所の略号は、引き続き「**ふ**」を使用する。

## 現行路線

### 藤沢駅北口 - 神明町 - 高山車庫方面

  - 藤01：[藤沢駅](https://ja.wikipedia.org/wiki/藤沢駅 "wikilink")北口 - 神明町 - 高山車庫
  - 藤01：藤沢駅北口 → 神明町 → 高山車庫（深夜バス）

藤沢駅と高山車庫を結ぶ路線だが、出入庫路線としてだけでなく[日本精工](../Page/日本精工.md "wikilink")・[ソニー](../Page/ソニー.md "wikilink")などの工場を結ぶ通勤路線、「ソニー前」停留所および「高山車庫」停留所付近の大型ショッピングセンター「[湘南モールフィル](../Page/湘南モールフィル.md "wikilink")」「[MrMax湘南藤沢ショッピングセンター](../Page/ミスターマックス.md "wikilink")」へのアクセス路線としても機能している。藤沢駅を出ると[東海道線](https://ja.wikipedia.org/wiki/東海道線 "wikilink")沿いの市道を直進し、湘南モールフィルの先で右折して高山車庫へ向かう。

藤沢駅発着路線の出入庫としては最短で3分間隔の高頻度で運転され、日中時間帯も毎時6便程度が確保される。従来は神奈川中央交通と藤沢神奈交バスの共同運行路線とされ、藤が岡循環で使用される専用の[いすゞ・エルガミオ](https://ja.wikipedia.org/wiki/いすゞ・エルガミオ "wikilink")もこの路線を用いて出入庫を行う。平日の藤沢駅発に深夜バスが1便設定されているが、上村（かむら）経由の方が1便多く設定されている。

### 藤沢駅北口 - 上村 - 高山車庫・辻堂駅方面

  - 藤02：[藤沢駅](https://ja.wikipedia.org/wiki/藤沢駅 "wikilink")北口 - 上村 - 高山車庫
  - 藤02：藤沢駅北口 → 上村 → 高山車庫（深夜バス）
  - 藤03：藤沢駅北口 - 上村 - 明治中学校前 - [辻堂駅](https://ja.wikipedia.org/wiki/辻堂駅 "wikilink")北口

[日本精工](../Page/日本精工.md "wikilink")北側の上村地区を経由する路線で、「上村経由」と案内される。藤沢駅から県道戸塚茅ヶ崎線交差点を直進して日本精工藤沢工場を半周したのち右折し、上村橋を渡ってから高山交差点にて高山車庫方面、または辻堂駅方面へ分かれる。上記の藤01と異なり、平日藤沢駅発の深夜バスは2便が設定されているが、日中時間帯は10-15分間隔とやや間隔があいている。かつては[江ノ電バス](../Page/江ノ電バス.md "wikilink")が往路1便のみ乗り入れており、共通定期券を取り扱っていた。

### 藤沢駅北口 - 浜見山 - 辻堂団地・辻堂駅方面

  - 藤04：[藤沢駅](https://ja.wikipedia.org/wiki/藤沢駅 "wikilink")北口 - 相生町 - 鵠沼小学校前 - 浜見山 - 辻堂団地（[江ノ電バス湘南営業所](../Page/江ノ電バス湘南営業所.md "wikilink")と共同運行）
  - 藤06：藤沢駅北口 - 相生町 - 鵠沼小学校前 - 浜見山 - [辻堂駅](https://ja.wikipedia.org/wiki/辻堂駅 "wikilink")南口（江ノ電バス湘南営業所と共同運行）（土曜・休日運休）

相生町の先の交差点を左折し、[東海道線](https://ja.wikipedia.org/wiki/東海道線 "wikilink")の陸橋を超えて[県道戸塚茅ヶ崎線を進み](../Page/神奈川県道30号戸塚茅ヶ崎線.md "wikilink")、浜見山交差点で辻堂団地、または辻堂駅方面へ向かう。[江ノ電バス湘南営業所](../Page/江ノ電バス湘南営業所.md "wikilink")との共同運行路線で共通定期券の取扱いを行っているが、時刻表に藤04（両社）と藤06（神奈川中央交通担当便）が併記され、藤06（江ノ電バス担当便）だけが江ノ電バスの時刻表に掲載されており、揃えられていない。藤77（藤沢駅 - 江ノ島駅 - 江ノ島 - 鵠沼海岸 - 辻堂駅）廃止後は藤沢駅発着路線で唯一、東海道線より南側の海沿いを走る路線群で、当営業所管内の路線において最も海沿いを走る路線である。

藤04は日中時間帯に約20分間隔で毎時3便を交互に運行するが、平日朝夕は神奈川中央交通担当便が若干増えて約10分間隔で運行される。藤06は平日運行で、江ノ電バスは藤沢駅方向1便のみ担当しており、それ以外は神奈川中央交通による運行である。

### 藤沢駅北口 - 市民病院 - 辻堂駅方面

  - 藤09：[藤沢駅](https://ja.wikipedia.org/wiki/藤沢駅 "wikilink")北口 - 南仲通り - 引地橋 - 明治中学校前 - [辻堂駅](https://ja.wikipedia.org/wiki/辻堂駅 "wikilink")北口
  - 藤10：藤沢駅北口 → 南仲通り → 藤沢小学校前 → [市民病院](https://ja.wikipedia.org/wiki/藤沢市民病院 "wikilink") → 引地橋 → 明治中学校前 → 辻堂駅北口（土曜・休日運休）
  - 藤10：辻堂駅北口 → 明治中学校前 → 引地橋 → 市民病院 → 遊行通り四丁目 → 藤沢駅北口（土曜・休日運休）

引地橋から藤沢市民病院方面へ向かう路線で、藤09は引地橋まで藤07・08（茅ヶ崎営業所）と同一経路を走行し、[メルシャン](../Page/メルシャン.md "wikilink")前の手前で左折して辻堂駅北口へ向かう路線である。汲田から先は前述の藤03と同一経路として補完している。なお、独自区間である引地橋 - 汲田間の途中には「湘南台」停留所が存在するが、藤沢市北部にある[湘南台駅](https://ja.wikipedia.org/wiki/湘南台駅 "wikilink")周辺とは全くの無関係である。

藤10は[藤沢市民病院](https://ja.wikipedia.org/wiki/藤沢市民病院 "wikilink")へのアクセス路線で、藤沢駅 - 市民病院間は往復で経路が異なるのが特徴である。平日の日中時間帯は藤10に振り替えて運行され、土曜・休日は全便が藤09となる。

### 藤沢駅北口 - 市民病院 - 湘南ライフタウン方面

  - 藤12：[藤沢駅](https://ja.wikipedia.org/wiki/藤沢駅 "wikilink")北口 - 南仲通り - 引地橋 - 大庭隧道 - 駒寄 - 湘南ライフタウン
  - 藤13：藤沢駅北口 → 南仲通り → 藤沢小学校前 → 市民病院 → 引地橋 → 大庭隧道 → 駒寄 → 湘南ライフタウン（土曜・休日運休）
  - 藤13：湘南ライフタウン → 駒寄 → 大庭隧道 → 引地橋 → 市民病院 → 遊行通り四丁目 → 藤沢駅北口（土曜・休日運休）

藤沢駅からライフタウン中央部へのアクセス路線として新設されたもので、毎時1-2便程度が確保されている。藤13は上記の藤10と同様に藤沢市民病院へのアクセス路線で、平日に運行される藤12の一部を振り替えたものだが、藤10とは異なる午前中に1.5往復、午後に病院方向へ2便が運行されるのみである。

### 藤沢駅北口 - 湘南カントリークラブ - 湘南台駅方面

  - 藤15：[藤沢駅](https://ja.wikipedia.org/wiki/藤沢駅 "wikilink")北口 - 南仲通り - 引地橋 - 湘南カントリークラブ前 - 滝の沢小学校前 - 湘南ライフタウン（土曜・休日運休）
  - 藤39：藤沢駅北口 - 南仲通り - 引地橋 - 湘南カントリークラブ前 - 大辻 - 遠藤 - [湘南台駅](https://ja.wikipedia.org/wiki/湘南台駅 "wikilink")西口

ライフタウンを外周から回り込む路線で、1時間1便が運行される。湘南ライフタウンから先は和泉橋経由で運行されるが、藤沢駅から湘南台駅前の経路全体が遠回りのため、渋滞に巻き込まれやすく、定時運行が困難になることもしばしば発生する。藤15は前述の藤12と同様に湘南ライフタウンへのアクセス路線だが、藤39と同様に羽根沢団地第一住宅南側に位置する湘南カントリークラブ経由である点が異なる。2016年5月16日のダイヤ改正で土曜・休日の運行が廃止された。

### 藤沢駅北口 - 石川橋 - 湘南台駅方面

  - 藤34：[藤沢駅](https://ja.wikipedia.org/wiki/藤沢駅 "wikilink")北口 - 遊行通り四丁目 - 引地橋 - 石川橋 - 一色上 - [湘南台駅](https://ja.wikipedia.org/wiki/湘南台駅 "wikilink")西口

藤沢駅と湘南台駅を[県道藤沢厚木線で運行する路線で](../Page/神奈川県道43号藤沢厚木線.md "wikilink")、一色上停留所付近の工場街が独自経路となっており、湘南ライフタウンを経由しない。かつては藤沢駅から綾瀬車庫・本厚木駅方面の路線が多数運行されていたが、長距離路線のために大幅な遅延が恒常化していたため、[2002年](../Page/2002年.md "wikilink")7月に綾瀬営業所へ移管および短縮する再編を実施した結果、現在では藤34は県道藤沢厚木線を縦断する唯一の路線となった。土曜・休日の日中時間帯は、藤沢駅方向が南仲通り経由で運行される。

善行団地へ向かう短距離路線であるが、沿線には非常に起伏があり、善行駅から善行団地へ向かうには急坂が連続するため、一定の利用者がいる。以前は善01のみだったが、沿線開発の進展と団地北側を経由する新道が開通したため、北回りの善04が新設された。[2013年](../Page/2013年.md "wikilink")には善行団地の西側に位置する大庭・善行坂地区にも新道が開通したため、同年7月のダイヤ改正で善05・06が新設された。この区間はかつて藤37（藤沢駅 - 善行大橋 - 一色上 - 湘南台駅）が運行していた路線であるため、事実上の路線復活となっている。この循環線は善行団地周辺の需要状況によって両回りが設定されており、13時で運行が入れ替わる。

なお、善05は善行駅を出発すると善行坂方面へ向かうのに対して、善06は善行坂から善行二丁目経由で善行駅へ戻る経路となっているが、これは善行駅ロータリーにおいて善05の経路で戻ると、ロータリー周辺の道路が狭いために降車場へ到着出来なくなるためで、これを防ぐために善行二丁目経由で運行される。終車時間帯には善01が運行され、平日・土曜には深夜バスも運行される。

六会日大前駅西側に位置する天神町地区を循環する路線群で、[2003年](../Page/2003年.md "wikilink")10月16日に藤沢神奈交バス直轄の路線として開業した。その後、神奈川中央交通東への移管を経て前述の藤が岡循環線と同様に神奈川中央交通へ移管し、運行受託を行っている。以前は「てんじんミニバス」と表記された専用の小型車（[三菱ふそう・エアロミディME](../Page/三菱ふそう・エアロミディME.md "wikilink")）が専属で運用に入っていたが、神奈川中央交通東へ移管されてからすぐに舞岡営業所から[日野・ポンチョ](../Page/日野・ポンチョ.md "wikilink")が転入し、使用されている。ただし2台のみ配置で予備車が存在しないため、中型車による運行の場合もある。「湘南看護専門学校前」停留所は、2014年4月1日に「天神町会館前」に改称された。

他の藤沢営業所の路線から孤立しており、運用の前後は天神町9号到着後に高山車庫へ回送、あるいは高山車庫から六03の始発である天神町6号へ回送される。

[2020年](../Page/2020年.md "wikilink")3月14日のダイヤ改正で六01・02・04・06が一度に廃止され、往路で天神町会館前、復路で天神町4号をそれぞれ経由する六07が新設された。

## 車両

藤沢神奈交バスだった頃は「藤沢操車所」として茅ヶ崎営業所の管轄下だったが、現在は「神奈川中央交通東」の営業所として、全車が神奈川中央交通からの委託車両（車番は「0」が付かない「ふxxx」）の配置となっている。[2017年](../Page/2017年.md "wikilink")12月15日までは、神奈川中央交通からの管理委託車（車番は「0」が付かない「ふxxx」）と、神奈川中央交通東が所有する車両（「ふ600」番台\[8\])と区別されていたが、直轄路線の神奈川中央交通本体への移管に際して「ふ600」番台への改番が進められ、現在は消滅している。高速・観光車および特定車については、かつての神奈川中央交通藤沢営業所時代には在籍していたが、現在は配置されていない。

かつての[モノコック](../Page/モノコック.md "wikilink")車体の時代は日野・三菱・いすゞ・日産ディーゼルの4社が揃って在籍していたが、[神奈川三菱ふそう自動車販売](../Page/神奈川三菱ふそう自動車販売.md "wikilink")が神奈川中央交通の傘下となって以降は、他営業所と同様に三菱ふそう車の比率が一気に上昇した。その過程において日産ディーゼル車は全廃、日野自動車も[日野・リエッセ](../Page/日野・リエッセ.md "wikilink")が配置されるまでは姿を消していたが、藤沢市周辺は[いすゞ自動車](../Page/いすゞ自動車.md "wikilink")のお膝元という事情もあって、いすゞ車（純正）の投入が続けられていた。そのため、三菱車といすゞ車の比率は6:4で推移していたが、近年発覚したいわゆる「[三菱リコール隠し](../Page/三菱リコール隠し.md "wikilink")」で三菱車の信用が失墜すると、いすゞ自動車の牙城ともいえる茅ヶ崎営業所の管轄下で、前述のように藤沢市がいすゞ自動車のお膝元であることから容易に選択できる有利さがあったことで両社の比率が急速に縮まっていった。現在では本区のみならず、神奈川中央交通全体においても新車導入の際は、大型車が三菱、中型車がいすゞを選択する傾向がみられる。

ごく短い期間ではあったが、平成初頭から中期まで富士重工製車体のいすゞ車が在籍していた時期があった。

2019年3月現在、大型車は[三菱ふそう・エアロスター](../Page/三菱ふそう・エアロスター.md "wikilink")、[いすゞ・エルガ](https://ja.wikipedia.org/wiki/いすゞ・エルガ "wikilink")[ワンステップバス](../Page/ワンステップバス.md "wikilink")が中心である。ノンステップバスは2013年以降、藤沢市の補助によって導入が進められており、[三菱ふそう・エアロスター](../Page/三菱ふそう・エアロスター.md "wikilink")が23台、[いすゞ・エルガ](https://ja.wikipedia.org/wiki/いすゞ・エルガ "wikilink")が9台配置\[9\]、中型車は[いすゞ・エルガミオ](https://ja.wikipedia.org/wiki/いすゞ・エルガミオ "wikilink")が4台配置、小型車は天神町循環線用の[日野・ポンチョ](../Page/日野・ポンチョ.md "wikilink")が2台配置されている。以前は[三菱ふそう・エアロミディMJ](../Page/三菱ふそう・エアロミディMJ.md "wikilink")（KK-MJ27HL）が3台配置されていたが、2016年6月に[いすゞ・エルガミオ](https://ja.wikipedia.org/wiki/いすゞ・エルガミオ "wikilink")ノンステップバスに置き換えられた。小型車も[三菱ふそう・エアロミディME](../Page/三菱ふそう・エアロミディME.md "wikilink")（KK-ME17DF）が2台配置されていたが、2017年1月に[舞岡営業所から](../Page/神奈川中央交通舞岡営業所.md "wikilink")[日野・ポンチョ](../Page/日野・ポンチョ.md "wikilink")が転属され\[10\]、それを置き換えられた。

2013年6月より、本区の一部車両でも運転席後方にモニターが設置され、ニュース・天気予報・広告などが放映される「かなch.」のサービスが実施されている。

所属車両に搭載される[LED行先表示器は交通電業社製を](../Page/発光ダイオード.md "wikilink")、運賃箱・整理券発行機・LCD運賃表示器はレシップ製を使用する\[11\]。尚、この組合せは、当区から分区した綾瀬も同一である。

## その他

  - 綾瀬営業所を分離した上、所轄路線に藤沢市北部方面の路線も含まれていることから、開設後しばらくは綾瀬営業所発着便も存在した。
  - 路線再編が進むなか、綾瀬営業所との共同運行路線が当営業所単独運行となる路線が多数発生したが、その影響で最終便運行後、高山車庫までの回送や始発便運行前の高山車庫からの送り込み回送の距離が伸びる傾向にある。現在は藤沢市内の北限は湘南台駅から遠藤バス停付近を東西に結んだラインまでとなっていて、同じ藤沢市内でも長後地区や御所見地区には到達していない。この地域は綾瀬営業所の管轄となっていて、前述の北限の位置で営業エリアを分けている(湘南ライフタウン～辻堂駅方面は除く)。
  - 管内路線の車内放送は長らく綾瀬営業所･秦野営業所とともに8トラックテープによる車内を使用していたが、2007年10月29日より音声合成装置に切り替わった。テープ時代には、運賃境界の停留所のみ冒頭のチャイムが通常と異なるなどの特徴があった（通常の停留所では3打音であるが運賃境界停留所では4打音に変更されていた）。
  - 神奈川中央交通では全ての営業所で古いタイプの運賃箱を使用していたが、藤沢操車所は2007年10月21日より、主に海側の営業所で使用されているレシップ製の運賃箱に載せ替えた。
