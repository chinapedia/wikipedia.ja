[thumb](https://ja.wikipedia.org/wiki/画像:TamilNaduDistricts_ByNumber.png "wikilink") **ティルチラーパッリ県（திருச்சிராப்பள்ளி மாவட்டம் ; Tiruchirappalli district）**は、[インド](../Page/インド.md "wikilink")南部の[タミル・ナードゥ州](../Page/タミル・ナードゥ州.md "wikilink")に属する30の県の一つ。県庁所在地は[ティルチラーパッリ](../Page/ティルチラーパッリ.md "wikilink")。**ティルッチラーッパッリ県**とも呼ばれる。

## 概要

[2001年](../Page/2001年.md "wikilink")の国勢調査では、面積4511平方キロメートル、人口241万8366人、人口密度は536人/[km<sup>2</sup>](https://ja.wikipedia.org/wiki/平方キロメートル "wikilink")。

北は[セーラム県](../Page/セーラム県_\(インド\).md "wikilink")、北東は[ペランバルール県](../Page/ペランバルール県.md "wikilink")、東は[タンジャーヴール県](../Page/タンジャーヴール県.md "wikilink")、南東は[プドゥッコーッタイ県](https://ja.wikipedia.org/wiki/プドゥッコーッタイ県 "wikilink")、南は[シヴァガンガイ県](../Page/シヴァガンガイ県.md "wikilink")と[マドゥライ県](../Page/マドゥライ県.md "wikilink")、南西は[ティンドゥッカル県](../Page/ティンドゥッカル県.md "wikilink")、西は[カルール県](../Page/カルール県.md "wikilink")、北西は[ナーマッカル県](../Page/ナーマッカル県.md "wikilink")に接する。

本県は[タミル・ナードゥ州](../Page/タミル・ナードゥ州.md "wikilink")の真ん中にあってこのように多くの県に接し、道路や鉄道、河川などが集まる交通の要衝である。 河川の中でも、県北部を大河[カーヴェーリ川](https://ja.wikipedia.org/wiki/カーヴェーリ川 "wikilink")が横切る。

県庁所在地の[ティルチラーパッリ](../Page/ティルチラーパッリ.md "wikilink")は、略して**ティルチ**（திருச்சி ; Trichy）とも呼ばれる。

## 歴史

[1968年](https://ja.wikipedia.org/wiki/1968年 "wikilink")の[タミル・ナードゥ州](../Page/タミル・ナードゥ州.md "wikilink")発足以前の、[イギリス領インド帝国](../Page/イギリス領インド帝国.md "wikilink")[マドラス管区](https://ja.wikipedia.org/wiki/マドラス管区 "wikilink")時代から存在する。マドラス管区時代は、**トリチノポリ県**（Trichinopoly district）という英語名が正式名称であった。

[1974年](../Page/1974年.md "wikilink")[1月14日](../Page/1月14日.md "wikilink")、南東部が[タンジャーヴール県](../Page/タンジャーヴール県.md "wikilink")の一部と合併して[プドゥッコーッタイ県](https://ja.wikipedia.org/wiki/プドゥッコーッタイ県 "wikilink")として分立。[1995年](https://ja.wikipedia.org/wiki/1995年 "wikilink")11月1日には、西部が[カルール県](../Page/カルール県.md "wikilink")、北部が[ペランバルール県](../Page/ペランバルール県.md "wikilink")として分立した。

## 行政区分

### 郡

ティルチラーパッリ県は、以下の8つの郡（[タミル語](../Page/タミル語.md "wikilink") வட்டம் 「円」 ; 英訳 taluk）に区分けされている。

  - ラールクディ郡（லால்குடி ; Lalgudi）
  - マナッチャナッルール郡（மணச்சநல்லூர் ; Manachanallur）
  - トゥライユール郡（துறையூர் ; Thuraiyur）
  - ムシリ郡（முசிறி ; Musiri）
  - トッティヤム郡（தொட்டியம் ; Thottiyam）
  - ティルチラーパッリ郡（திருச்சிராப்பள்ளி ; Tiruchirappalli）
  - シュリーランガム郡（ஸ்ரீரங்கம் ; Srirangam）？
  - マナッパーライ郡（மணப்பாறை ; Manapparai）

### 区

ティルチラーパッリ県は、以下の14の区（英訳 block）に区分けされている。

  - プッランパーディ区（புள்ளம்பாடி ; Pullambadi）
  - ラールクディ区（லால்குடி ; Lalgudi）
  - マナッチャナッルール区（மணச்சநல்லூர் ; Manachanallur）
  - トゥライユール区（துறையூர் ; Thuraiyur）
  - ムシリ区（முசிறி ; Musiri）
  - ウッピリヤープラム区（உப்பிலியாபுரம் ; Uppiliyapuram）
  - ターッタイヤンガールペーッタイ区（தாத்தையங்கார்பேட்டை ; Thathiyengarpet）
  - トッティヤム区（தொட்டியம் ; Thottiyam）
  - アンダナッルール区（அந்தநல்லூர் ; Andanallur）？
  - ティルヴェーランブール区（திருவேரம்பூர் ; Tiruverambur）？
  - マニカンダム区（மணிகண்டம் ; Manikandam）
  - マナッパーライ区（மணப்பாறை ; Manapparai）
  - ヴァイヤンパッティ区（வையம்பட்டி ; Vaiyampattai）
