**三次方程式**（さんじほうていしき、*cubic equation*）とは、[次数が](https://ja.wikipedia.org/wiki/多項式の次数 "wikilink") 3 である[代数方程式](../Page/代数方程式.md "wikilink")のことである。本項目では主に、[実数](../Page/実数.md "wikilink")を[係数](../Page/係数.md "wikilink")とする一変数の三次方程式を扱う。

## 概要

[thumb](https://ja.wikipedia.org/wiki/画像:三次方程式の解の公式.png "wikilink") [thumb](https://ja.wikipedia.org/wiki/画像:三次方程式の解の公式（部分式置き換え）.png "wikilink") 一般に一変数の三次方程式は

  -
    \(a_3 x^3 + a_2x^2+a_1x + a_0 = 0\quad (a_3\ne 0)\)

の形で表現される。現代においては、三次方程式の解法といえば、主に[代数的解法のことを意味する](https://ja.wikipedia.org/wiki/代数方程式#代数方程式の解法 "wikilink")。

古代[バビロニアにおいて既に代数的に解かれていたと考えられている](https://ja.wikipedia.org/wiki/バビロニア数学 "wikilink")[二次方程式](../Page/二次方程式.md "wikilink")と違い、三次方程式が代数的に解かれたのは[16世紀](../Page/16世紀.md "wikilink")になってからである。[11世紀](../Page/11世紀.md "wikilink")頃、[円錐曲線による作図](https://ja.wikipedia.org/wiki/円錐曲線による作図 "wikilink")によって三次方程式の解を幾何学的に表した[ウマル・ハイヤーム](../Page/ウマル・ハイヤーム.md "wikilink")なども、三次方程式を代数的に解くことはできないと考えていた。

三次方程式の代数的解法は[ガロア理論](../Page/ガロア理論.md "wikilink")へと至る代数方程式論の始まりであり、[カルダノが著書](../Page/ジェロラモ・カルダーノ.md "wikilink")『[アルス・マグナ](https://ja.wikipedia.org/wiki/アルス・マグナ_\(カルダーノの著書\) "wikilink")』によって三次方程式と[四次方程式](../Page/四次方程式.md "wikilink")の代数的解法を公表した[1545年](../Page/1545年.md "wikilink")は、その影響の大きさから現代数学の始まりの年とされることもある。

まだ[負の数が数学者達にあまり受け入れられていなかった時代であり](https://ja.wikipedia.org/wiki/正の数と負の数 "wikilink")、全ての係数が正の数であるとして扱われたために、例えば、2次の項が無い三次方程式は

  -

の3つがあり、いずれも別の形の方程式とされた。

このように負の数ですら嫌悪された時代に、三次方程式の代数的解法は[虚数](../Page/虚数.md "wikilink")をもたらした。三次方程式の解が全て正の実数である場合に限っても、代数的解法にこだわる限り虚数を避けては通れないのである。虚数に対する不安は、[19世紀](../Page/19世紀.md "wikilink")に[コーシーや](../Page/オーギュスタン＝ルイ・コーシー.md "wikilink")[ガウスが活躍するようになるまで続いた](../Page/カール・フリードリヒ・ガウス.md "wikilink")。

また、三次方程式と四次方程式の代数的解法の発見を基に、数学者達は 5 次以上の一般の代数方程式の代数的解法を追い求めた。最終的にこの代数的解法の存在は、[アーベル-ルフィニの定理](https://ja.wikipedia.org/wiki/アーベル-ルフィニの定理 "wikilink")によって否定されるものの、[ガロア理論](../Page/ガロア理論.md "wikilink")として結実し、[群や](https://ja.wikipedia.org/wiki/群_\(数学\) "wikilink")[体などの基本的な](https://ja.wikipedia.org/wiki/可換体 "wikilink")[代数的構造](../Page/代数的構造.md "wikilink")の概念を生み出した。

## ラグランジュの方法

[ラグランジュは](../Page/ジョゼフ＝ルイ・ラグランジュ.md "wikilink")、三次方程式や四次方程式の代数的解法を分析し、根の置換という代数方程式論の方向性を決定づける重要な概念に到達した。この研究は[ガロア理論](../Page/ガロア理論.md "wikilink")の発見へと繋がっていった。

  -
    *x* + *A* *x* + *A* *x* + *A* = 0

の 3 つの解を *r*, *r*, *r* とし [1 の虚立方根の一つ](https://ja.wikipedia.org/wiki/1の冪根 "wikilink")

\[\omega = {-1 + i \sqrt{3} \over 2}\] を取る。

  -
    *s* = *r* + *r* + *r*
    *s* = *r* + ω *r* + ω *r*
    *s* = *r* + ω *r* + ω *r*

とおくと

\[r_0 = {s_0 + s_1 + s_2 \over 3}\]

\[r_1 = {s_0 + \omega^2 s_1 + \omega s_2 \over 3}\]

\[r_2 = {s_0 + \omega s_1 + \omega^2 s_2 \over 3}\] である。[根と係数の関係](../Page/根と係数の関係.md "wikilink")により *s* = −*A* であることが分かるので *s* と *s* の二つが分かれば解が求まることになる。ここで *r* と *r* を入れ替える[互換を](../Page/対称群.md "wikilink") σ と書けば

  -
    (σ *s*) = *r* + ω *r* + ω *r*
    ω (σ *s*) = *r* + ω *r* + ω *r* = *s*

が得られる。両辺を三乗することにより

  -
    σ *s* = *s*

同様に

  -
    σ *s* = *s*

σ σ も計算してみれば分かる通り、これらの互換は *s* と *s* の入れ替えしかない。つまり *s* + *s* と *s* *s* は *r*, *r*, *r* の[対称式](../Page/対称式.md "wikilink")であり、それらの[基本対称式で表される](https://ja.wikipedia.org/wiki/対称式#基本対称式 "wikilink")。すなわち *s* と *s* を解とする二次方程式

  -
    (*z* − *s*)(*z* − *s*) = *z* −(*s* + *s*) *z* + *s* *s* = 0

の係数は、元の三次方程式の係数 *A*, *A*, *A* で表されることになる。実際にこれは

\[z^2 - \left( -2 {A_2}^3 + 9 A_1 A_2 - 27 A_0 \right) z + \left( {A_2}^2 - 3 A_1 \right)^3 = 0\] という[二次方程式](../Page/二次方程式.md "wikilink")になり、この解は[解の様子を調べた時に定義した記号](https://ja.wikipedia.org/wiki/#解の様子 "wikilink") ⊿ と ⊿ によって

\[\frac{\Delta_2 \pm 3 \sqrt{-3\Delta}}{2}\] と書くことができる。

この根号は[二次方程式の解の差積](https://ja.wikipedia.org/wiki/二次方程式#実数係数の二次方程式 "wikilink") \(\pm\left( {s_1}^3 - {s_2}^3 \right)\)として得られ、ここに現れる \({s_1}^3, {s_2}^3\) も、3乗根は元の方程式の根 \(r_1,r_2,r_3\) と 1の3乗根\(\omega\) の四則演算で表されている。すなわち三次方程式を解く際に冪乗根を取って出てくる式は、元の方程式の解 \(r_1,r_2,r_3\) と1の冪乗根の有理式で表現できる。[ジョゼフ＝ルイ・ラグランジュ](../Page/ジョゼフ＝ルイ・ラグランジュ.md "wikilink")やは、これこそ三次方程式が代数的に解ける理由であると考えた。

## 円錐曲線による作図

代数的解法は重要であるものの、歴史的にはそれよりも先に、作図による三次方程式の幾何学的解法が模索されていた。このような解法は、[古代ギリシア](../Page/古代ギリシア.md "wikilink")の[メナイクモス](https://ja.wikipedia.org/wiki/メナイクモス "wikilink")に始まり、[セルジューク朝](../Page/セルジューク朝.md "wikilink")期[ペルシャ](https://ja.wikipedia.org/wiki/ペルシャ "wikilink")の[ウマル・ハイヤーム](../Page/ウマル・ハイヤーム.md "wikilink")によって一般化された。

[thumb](https://ja.wikipedia.org/wiki/画像:CubicEquation_1.png "wikilink") *xy* 平面上の 2 つの[放物線](../Page/放物線.md "wikilink")を表す式

  -
    *x* = *py* (*p* \> 0)
    *y* = *qx* (*q* \> 0)

において *y* を消去すると、

  -
    *x* = *p* *qx*

となり、この 2 つの放物線の交点の *x* 座標は、

  -
    \(x = 0, \sqrt[3]{p^2 q}\)

となり、x = 0 でない方の交点の位置によって

  -
    *x* = *p**q*

という形の三次方程式の解が得られることになる。特に *q* = 2*p* ととれば、[立方体倍積問題](../Page/立方体倍積問題.md "wikilink")と同値な三次方程式

  -
    *x* = 2 *p*

の実数解を、[線分](../Page/線分.md "wikilink")の長さとして得たことになる。

[thumb](https://ja.wikipedia.org/wiki/画像:CubicEquation_2.png "wikilink") また、放物線と[円を表す式](../Page/円_\(数学\).md "wikilink")

  -
    *x* = *p y* (*p* \> 0)
    *x* + *y* = *q x* (*q* \> 0)

において同様に *y* を消去すれば

  -
    *p* *x* + *x* = *q x*

であり、*x* = 0 以外の交点を求めることは

  -
    *x* + *p* *x* = *q*

という三次方程式の実数解を与えるのと同じである。

一般に、

  -
    *a* *x* + *a* *x* + *a* *x* + *a* = 0 (*a* ≠ 0)

という三次方程式は

  -
    *x* = *py* (*p* \> 0)
    *a* *p* *y* + *a* *p x y* + *a* *x* + *a* *x* = 0 (*a* ≠ 0)

というように、放物線と、もう 1 つの円錐曲線の組み合わせでも書けるし

  -
    *x* = *py*
    (*a* *x* + *a*) *py* + *a* *x* + *a* = 0

のように、放物線と双曲線の交点としても表すことができる。

の形の三次方程式を解くことと同じであり、メナイクモスによる方法は、三次方程式の幾何学的解法の一つと考えられ、円錐曲線の数表を計算しておけば、三次方程式の解の近似値も得ることができることになる。しかし、一般に円錐曲線は、[プラトンの束縛の下で作図できる](../Page/定規とコンパスによる作図.md "wikilink")[曲線](../Page/曲線.md "wikilink")ではないため、円錐曲線による幾何学的解法は立方体倍積問題の解法とは見なされない。このような円錐曲線の研究は、[アルキメデス](../Page/アルキメデス.md "wikilink")や[イブン・ハイサム](../Page/イブン・ハイサム.md "wikilink")等を経て、[セルジューク朝](../Page/セルジューク朝.md "wikilink")期[ペルシア](../Page/ペルシア.md "wikilink")の[ウマル・ハイヤーム](../Page/ウマル・ハイヤーム.md "wikilink")により拡張され、様々な形をした三次方程式の解が、円錐曲線同士の交点として調べられ、網羅された。

三次方程式の[代数的解法は](https://ja.wikipedia.org/wiki/代数方程式#代数方程式の解法 "wikilink")、16世紀頃に[ボローニャ大学](../Page/ボローニャ大学.md "wikilink")の[シピオーネ・デル・フェッロ](https://ja.wikipedia.org/wiki/シピオーネ・デル・フェッロ "wikilink")によって発見されたとされる。デル・フェロの解いた三次方程式は

という形の物である。当時はまだ、負の数はあまり認められていなかったため、[係数](../Page/係数.md "wikilink")を正に限った形をしている。

この方程式自体は特殊な形であるものの、一般の三次方程式はこの形に変形できるため、本質的には三次方程式はデル・フェロが解いたといっても過言ではない。また、この方程式の場合は係数の符号の制約から還元不能にはならない。

デル・フェロは、この解法を公開せず、何人かの弟子に託して[1526年](../Page/1526年.md "wikilink")に死んだ。そのうちの一人、アントニオ・マリア・フィオル (*Antonio Maria Fior*) は、この方法を、当時盛んに行われていた、金銭を賭けた計算勝負に使い、勝ち続けた。

三次方程式の解法があるという噂を元に[タルタリアは](https://ja.wikipedia.org/wiki/ニコロ・フォンタナ・タルタリア "wikilink")、独力かどうかは分からないが

の形の三次方程式を解くことに成功し、さらにはデル・フェロの三次方程式の解法にも辿り着いた。タルタリアが三次方程式を解いたとの噂を聞いたフィオルは噂を信用せずタルタリアに計算勝負を挑み、打ち負かして名声を上げようとしたものの、デル・フェロの三次方程式の解法しか知らなかったため、計算勝負に負けた。

タルタリアが三次方程式の代数的解法を知っていると聞いた[カルダノはタルタリアに頼み込み](../Page/ジェロラモ・カルダーノ.md "wikilink")、三次方程式の代数的解法を聞き出すことに成功した。カルダノは、弟子の[ルドヴィコ・フェラーリ](../Page/ルドヴィコ・フェラーリ.md "wikilink")が得た、一般的な[四次方程式](../Page/四次方程式.md "wikilink")の代数的解法と併せて、三次方程式の代数的解法を出版したいと考えるようになったが、タルタリアとの約束で秘密にすると誓ったために、出版することはできなかった。そこで、かつてデル・フェロが、三次方程式の代数的解法を得たという噂を頼りに、フェラーリとボローニャに行き、デル・フェロの養子のアンニバレ・デラ・ナーヴェ (*Annibale della Nave*) に会い、デル・フェロの遺稿を見せてもらった。それによってカルダノは、タルタリアが三次方程式を解いた最初の人ではないことを知ったので、タルタリアとの約束は無効とし[1545年](../Page/1545年.md "wikilink")に『アルス・マグナ』(*Ars Magna*) を出版し、様々な形の三次方程式の解法を公表した。以来、三次方程式の解法は**カルダノの方法**と呼ばれるようになった。このことはタルタリアを激怒させ論争に発展したが、カルダノは『アルス・マグナ』の中でデル・フェロとタルタリアの功績について賞賛しており、独自の方法と偽ったわけではない。また、タルタリアから解の導出方法までは聞いておらず、色々な形の三次方程式について解を表したことはカルダノ自身の業績である。
