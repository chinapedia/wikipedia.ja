[LogisticMap_BifurcationDiagram.png](https://ja.wikipedia.org/wiki/File:LogisticMap_BifurcationDiagram.png "fig:LogisticMap_BifurcationDiagram.png")。パラメータ値を変化させていくと非常に複雑な挙動が現れる。\]\] **ロジスティック写像**（ロジスティックしゃぞう、）とは、[1次元](https://ja.wikipedia.org/wiki/1次元 "wikilink")の離散[力学系](../Page/力学系.md "wikilink")の一種。[ロジスティック方程式](../Page/ロジスティック方程式.md "wikilink")の離散化からも得られるため**離散型ロジスティック方程式**とも呼ばれる\[1\]。[変数を](../Page/変数_\(数学\).md "wikilink") *x* としたとき、次の1変数2次差分方程式（[漸化式](https://ja.wikipedia.org/wiki/漸化式 "wikilink")）で示される。

\[x_{n+1} = a x_n (1-x_n)\]

ロジスティック写像は、パラメータ *a* にどのような値を与えるかによって、*n* を増やすに連れた*x<sub>n</sub>*の値の変化（振る舞いや軌道と呼ぶ）が、一定値への[収束](https://ja.wikipedia.org/wiki/収束 "wikilink")、複数の値を繰り返し取り続ける周期的な振動、[カオスと呼ばれる非周期的な極めて複雑な振る舞い](../Page/カオス理論.md "wikilink")、へと変化する。

この複雑な振る舞いについて多くの研究がされてきたが、特に[ロバート・メイ](https://ja.wikipedia.org/wiki/ロバート・メイ "wikilink")(他にジム・ヨーク、ジョージ・オスター)の研究によって広く知られるようになった。

カオスを生み出す[系は](../Page/力学系.md "wikilink")[非線形性](https://ja.wikipedia.org/wiki/非線形性 "wikilink")を持つ必要があるが、このような非線形関数の中でも、ロジスティック写像は最も単純なものの1つである[二次関数](../Page/二次関数.md "wikilink")の差分方程式からカオスを生成する。この単純さと、他のカオスとも共通する現象がいくつも現れることから、[カオス理論](../Page/カオス理論.md "wikilink")の入り口としてよく採り上げられる。

## 写像の基本的特徴

### 変数とパラメータ

ロジスティック写像を再掲する。

ここで、*n* はステップ数、*n* = 0, 1, 2... である\[2\]。離散時間の発展を意味する。記号 *t* と表記する場合もある。*n* = 1 を初期状態として *n* = 1, 2, 3... と数える場合もあるが、本記事中では前者で統一する。

*x<sub>n</sub>* は第 *n* 項目の *x* の値を示す。式()は、*x<sub>n</sub>* の値とその次の値である *x*<sub>*n*+1</sub> の関係を示すものである。すなわち、*x<sub>n</sub>* から *x*<sub>*n*+1</sub> が一意的に決定されることを示している。一方で、*x<sub>n</sub>* の値からその前の値である *x*<sub>*n*−1</sub> の値は一意的に決定できない。[二次関数](../Page/二次関数.md "wikilink")の解が2つあることからも明らかなように、*x<sub>n</sub>* の値に辿り着く *x*<sub>*n*-1</sub> の値については2つの値が有り得る。このような現在の値から過去の値への一意性が無いことは、1変数差分方程式でカオスを生み出す[必要条件](https://ja.wikipedia.org/wiki/必要条件 "wikilink")でもある。2変数以上になるとこの条件は必要ではなくなる。

*x*<sub>0</sub> を初期値として、ロジスティック写像では一般的に 0 ≤ *x*<sub>0</sub> ≤ 1 の範囲で初期値を取る。文献によっては 0 \< *x*<sub>0</sub> ≤ 1 の範囲であるが、本記事中では前者で統一する。

*a* は[定数](../Page/定数.md "wikilink")で、パラメータとも呼ばれ、非線形性の強さを表すとされる。*x<sub>n</sub>* をある環境下における生物個体数とし、ロジスティック写像を[個体群](../Page/個体群.md "wikilink")の成長モデルとみなすような文脈では、*a* は最大の生物個体数増加率 (biotic potential) とも呼ばれる\[3\]。記号には *r* や *μ* なども割り当てることもある\[4\]。パラメータは一般的に 0 ≤ *a* ≤ 4 の範囲で選択される。文献によっては 0 \< *a* ≤ 4 の範囲であるが、本記事中では前者で統一する。

### 写像のグラフ

[Quadratic_function_(Logistic_map).png](https://ja.wikipedia.org/wiki/File:Quadratic_function_\(Logistic_map\).png "fig:Quadratic_function_(Logistic_map).png") 式()について、縦軸を *x*<sub>*n*+1</sub>、横軸を *x<sub>n</sub>* としてグラフを描くと、単純な[二次関数](../Page/二次関数.md "wikilink")の[放物線](../Page/放物線.md "wikilink")が得られる。0 ≤ *x<sub>n</sub>* ≤ 1 の範囲でこのグラフを描くと、両端の *x<sub>n</sub>* = 0, 1 で、*x*<sub>*n*+1</sub> = 0 となる。真ん中の *x<sub>n</sub>* = 0.5 で、*x*<sub>*n*+1</sub> = *a* / 4 となる[極大値](https://ja.wikipedia.org/wiki/極大値 "wikilink")かつ[最大値](https://ja.wikipedia.org/wiki/最大値 "wikilink")を取る。このような *x<sub>n</sub>* から *x*<sub>*n*+1</sub> への写像を[リターン・マップと呼ぶ](https://ja.wikipedia.org/wiki/ポアンカレプロット "wikilink")。

ロジスティック写像のような山形の曲線を描く写像は単峰写像と呼ばれる写像の一種で、その中でもロジスティック写像はが負であるという条件を満たす。これらの条件を満たす写像では、後述するような[分岐のパターンが発生する](https://ja.wikipedia.org/wiki/分岐_\(力学系\) "wikilink")。他の例としては、[正弦波](../Page/正弦波.md "wikilink")を用いた *x*<sub>*n*+1</sub> = *a* sin *x<sub>n</sub>* という写像でもこの条件を満たし、周期倍分岐、カオス、窓など、ロジスティック写像同様な振る舞いを見せる。このような曲線の写像ではカオスを生み出す基本機構である引き伸ばしと折り畳みという操作を備えているのが特徴である。*a* = 4 のときの *x<sub>n</sub>* から *x<sub>n+1</sub>* への変換を考えると、まず区間 \[0, 1\] の *x<sub>n</sub>* が全体として2倍に引き伸ばされ、伸びた区間が半分に折り畳まれ、区間 \[0, 1\] の *x<sub>n+1</sub>* として与えられる、というような操作を行っていることになる。このような引き伸ばしと折り畳みを繰り返す操作は、料理の[パイ](../Page/パイ.md "wikilink")をこねる操作に似ていることから[パイこね変換](https://ja.wikipedia.org/wiki/パイこね変換 "wikilink")と呼ばれ、低次元カオスの典型的な発生機構である。

### 定義域と値域

一般的には、ロジスティック写像は、式()による[単位区間](https://ja.wikipedia.org/wiki/単位区間 "wikilink") \[0, 1\] から \[0, 1\] への[写像](../Page/写像.md "wikilink")とされる。この区間の写像であるために必要な初期値 *x*<sub>0</sub> とパラメータ *a* の範囲を確認する。*x<sub>n</sub>* が区間 \[0,1\] 内の値であれば、*x*<sub>*n*+1</sub> が取り得る最小値は *x<sub>n</sub>* = 0, 1 のとき *x*<sub>*n*+1</sub> = 0、最大値は *x<sub>n</sub>* = 0.5 のとき *x*<sub>*n*+1</sub> = *a*/4 である。したがって、初期値が 0 ≤ *x*<sub>0</sub> ≤ 1 を満たし、なおかつパラメータが 0 ≤ *a* ≤ 4 を満たせば、次の値は 0 ≤ *x*<sub>2</sub> ≤ 1 となる。よって、その次の値も 0 ≤ *x*<sub>3</sub> ≤ 1 となり、[数学的帰納法](../Page/数学的帰納法.md "wikilink")からすべての *n* で常に 0 ≤ *x<sub>n</sub>* ≤ 1 が成立する。

*a* が 4 を超えると、後述のように初期値に関わらず負の無限大へ発散する。*a* が負のときは、*x<sub>n</sub>* が負の値を取るようになるが、*a* = −2 までは有限な範囲で変動して発散はしない。ただし、多くの文献では前述の 0 ≤ *a* ≤ 4 の範囲を扱い、ロジスティック写像が生態学上のモデルとして研究された経緯もあってパラメータ *a* が負の場合を論じることは少ない\[5\]。

## パラメータ *a* による振る舞いの変化

[Logistic_map_with_parameter_from_0.02_to_4_t_from_0_to_200.gif](https://ja.wikipedia.org/wiki/File:Logistic_map_with_parameter_from_0.02_to_4_t_from_0_to_200.gif "fig:Logistic_map_with_parameter_from_0.02_to_4_t_from_0_to_200.gif")

### 0 ≤ *a* ≤ 1 のとき

パラメータ *a* が大きくなるに従った、変数 *x<sub>n</sub>* の振る舞いの変化を順に説明する。まず、*a* が 0 \< *a* ≤ 1 の範囲では、初期値 *x*<sub>0</sub> が 0 から 1 までのどの値を取ったとしても、*x<sub>n</sub>* は *n* の増加と共に単調減少し、最終的に 0 に[収束](https://ja.wikipedia.org/wiki/収束 "wikilink")するこの収束する点を安定な[固定点](https://ja.wikipedia.org/wiki/固定点 "wikilink")、安定な[不動点](../Page/不動点.md "wikilink")、あるいは吸引的不動点、沈点などと呼ぶ。一般に、十分な時間経過後にその[力学系](../Page/力学系.md "wikilink")が漸近する[集合](../Page/集合.md "wikilink")は[アトラクタ](https://ja.wikipedia.org/wiki/アトラクタ "wikilink")や吸引集合と呼ばれる。この場合は、*x* = 0 がアトラクタに相当する。

1次元写像の軌道を視覚的に表すのには、[クモの巣図法](https://ja.wikipedia.org/wiki/クモの巣図法 "wikilink")と呼ばれる手法が利用される。これはグラフ上に原点を通る傾き1の直線を引き、この対角線と写像の曲線を交互に行き来することで1次元写像の軌道を2次元的に視覚化する方法である。*a* = 0.9 におけるクモの巣図と、*n* の増加に伴った*x<sub>n</sub>*の[時系列](../Page/時系列.md "wikilink")データを示す。 [Logistic_map_cobweb_and_time_evolution_a=0.9.png](https://ja.wikipedia.org/wiki/File:Logistic_map_cobweb_and_time_evolution_a=0.9.png "fig:Logistic_map_cobweb_and_time_evolution_a=0.9.png") 上記のようにある1点に落ち着いて、その値から動かなくなったとき、写像は常に *x*<sub>*n*+1</sub> = *x<sub>n</sub>* を満たす状態にある。よって、このときの値を *x<sub>f</sub>* とすると、  という関係式を満たす。単に不動点あるいは固定点と言えば、式()を満たす値 *x<sub>f</sub>* を指す。式()を解くと、*x<sub>f</sub>* は、  となる。0 \< *a* ≤ 1 のとき、不動点 1 − 1/*a* は負または0の値となるので、0 ≤ *a* ≤ 1 のときは不動点1 − 1/*a*は現れず、不動点は0のみである。

式()から不動点の値は求められるが、*x<sub>n</sub>* ≠ *x<sub>f</sub>* のときに、*n* の増加と共に *x<sub>n</sub>* = *x<sub>f</sub>* に収束するかは保証されない。*x<sub>f</sub>* の近傍にある *x*<sub>n</sub> が最終的に *x*<sub>f</sub> に収束するならば、この不動点 *x<sub>f</sub>* は安定であるいう。逆に *n* の増加と共に *x<sub>f</sub>* から離れていくならば、この不動点 *x<sub>f</sub>* は不安定であるという。また、離れも近づきもしない場合は中立安定であるという。0 \< *a* ≤ 1 のとき、0 は上記通り安定不動点に該当する。一方の不動点 1 − 1/*a* は、0 ≤ *x<sub>n</sub>* ≤ 1 の範囲では現れない不安定不動点である。

### 1 \< *a* ≤ 2 のとき

1 \< *a* ≤ 2 のときは、初期値が 0 または 1 である場合を除き、*x<sub>n</sub>* が単調増加あるいは単調減少しながら 1 − 1/*a* に収束する。以下にそれぞれの例を示す。  *a* \< 1 から *a* を増加させていくと、安定不動点 0 と不安定不動点 1 − 1/*a* が *a* の増加と共に近づき、*a* = 1 で衝突し、1 − 1/*a* が安定不動点に、0 が不安定不動点に変わる安定性の交替が発生する。ただし、*a* = 1 ではまだ 0 へ収束し、*a* = 1 を超えたところで漸近的な振る舞いは変化する。このようなパラメータの変化により[力学系](../Page/力学系.md "wikilink")の解の性質が変わることを[分岐と呼ぶ](https://ja.wikipedia.org/wiki/分岐_\(力学系\) "wikilink")。分岐が発生するパラメータの値を、この場合は *a* = 1 を分岐点と呼ぶ。*a* = 1 における分岐は[トランスクリティカル分岐](https://ja.wikipedia.org/wiki/トランスクリティカル分岐 "wikilink")と呼ばれるタイプの分岐に相当する。このようなパラメータの変化に伴った分岐の発生の様子を示すために、横軸に写像のパラメータを取り、縦軸に長期間後の変数の値を示した、[分岐図と呼ばれる図が用いられる](https://ja.wikipedia.org/wiki/分岐図_\(力学系\) "wikilink")。以下にパラメータ *a* が 0 から 3 までの分岐図を示す。 [Bifurcation_diagram_logistic_map_a_0_to_3.svg](https://ja.wikipedia.org/wiki/File:Bifurcation_diagram_logistic_map_a_0_to_3.svg "fig:Bifurcation_diagram_logistic_map_a_0_to_3.svg") 0も不動点として残っているが、不安定なので0に収束するのは初期値0、1のときだけである。これは以下の1 \< *a* \< 4 の範囲でも同様である。

### 3.4494897... \< *a* ≤ 3.5699456... のとき

2周期軌道の状態から *a* を増加させていくと、*a* = 1 + √6 = 3.4494897... で安定な2周期軌道から安定な4周期軌道の状態に変化する。これは2周期から4周期への周期倍分岐である。この4周期点の値も2周期点と同様に *a* に依存して決まるが、2周期点のように[陽関数](https://ja.wikipedia.org/wiki/陽関数 "wikilink")の形で得ることはできない。さらに *a* の値を大きくしていくと4周期から8周期、8周期から16周期、16周期から32周期... と周期倍分岐が続けて発生していく。

この周期倍分岐は無限に続くが、一方で、周期倍分岐が発生する分岐点の間隔は[等比数列](../Page/等比数列.md "wikilink")的に減少する。1周期から2周期への分岐を1番目の周期分岐として数えれば、*k* 番目の分岐点で 2<sup>*k*</sup> 周期が発生する。*k* 番目の分岐点パラメータ *a* を *a*<sub>*k*</sub> と表すと、実際に16周期までの分岐点は *a*<sub>1</sub> = 3, *a*<sub>2</sub> = 3.44949..., *a*<sub>3</sub> = 3.54409..., *a*<sub>4</sub> = 3.56441... と、分岐点の間隔は減少していく。このため、*k* → ∞ の極限における分岐点を *a<sub>c</sub>* とすると、*a<sub>c</sub>* = 3.5699456... という有限な値に収束する。*a<sub>c</sub>* は[集積点](https://ja.wikipedia.org/wiki/集積点 "wikilink")やファイゲンバウム点と呼ばれる。*a<sub>k</sub>* の減少の割合の極限は次式で示すような定数値となる。

{a_{k+1}-a_k} = 4.6692\\ldots </math>|}}

この *δ* の値は、ミッチェル・ファイゲンバウムにより発見されたことから[ファイゲンバウム定数](../Page/ファイゲンバウム定数.md "wikilink")と呼ばれる。このような周期倍分岐を無限回繰り返した末にカオスへと遷移する現象はカオスへ至る道筋の一つで、ファイゲンバウムのシナリオや周期倍分岐ルートと呼ばれる。 [Bifurcation_diagram_logistic_map_a_3_to_3.6.png](https://ja.wikipedia.org/wiki/File:Bifurcation_diagram_logistic_map_a_3_to_3.6.png "fig:Bifurcation_diagram_logistic_map_a_3_to_3.6.png") *a* = *a<sub>c</sub>* のときのロジスティック写像は、無限個の周期点が存在する閉じることの無い周期軌道を取る。この値における軌道はファイゲンバウム・アトラクタ\[6\]や臨界2<sup>∞</sup>アトラクタと呼ばれる。ファイゲンバウム・アトラクタの構造は[フラクタル](../Page/フラクタル.md "wikilink")になっており、[容量次元](https://ja.wikipedia.org/wiki/容量次元 "wikilink")は約0.54である。一方で、カオスの要件の一つである初期値鋭敏性は持たない\[7\]。他のカオスではフラクタル構造と初期値鋭敏性を同時に備えるのに対し、ファイゲンバウム・アトラクタではフラクタル構造は持つが初期値鋭敏性は持たないのが特徴である。

### 3.5699456... \< *a* \< 4 のとき

*a<sub>c</sub>* = 3.5699456... を超えるとカオスが発生し、収束も周期性も無い不規則で複雑な挙動を示すようになる。初期値鋭敏性の指標である[リアプノフ指数](../Page/リアプノフ指数.md "wikilink") *λ* を計算すると、*a* \< *a<sub>c</sub>* の範囲では *λ* は負または 0 の値の範囲に留まっていたが、*a* \> *a<sub>c</sub>* の範囲から *λ* が正の値も取るようになる。ロジスティック写像のような1次元の区間力学系が生み出すカオスは、特に1次元カオスと呼ばれる。

*a<sub>c</sub>* \< *a* \< 4 の範囲で分岐図を見ると、例えば *a* = 3.8 などでは1つの区間内に軌道が収まっているが、*a* = 3.65 などでは2つの区間に軌道が分かれている。このような区間はバンドと呼ばれる。*a* がバンドを複数持つ領域にあるとき、軌道は規則的に順番に各バンドを巡り、しかし各バンド内での取る値は不規則である。バンドが2つのときを例にすると、*n* が偶数のときは下側のバンド内に存在し、奇数のときは上側のバンド内に存在するが、偶数、奇数それぞれのときの値はバンド内で不規則に決まる。このような状態のカオス軌道を、バンドカオスや周期的カオスと呼ぶ。*a* = 3.65 からさらに値を小さくしていくと、周期倍加分岐のときと同じように、4, 8, 16, ...2<sup>*k*</sup> というふうにバンドの数が2倍に増えていく。バンドが増えるときのパラメータ *a* の値を *e<sub>k</sub>* で表すとする。*e<sub>k</sub>* の値の間隔も、周期倍加分岐と同じく、バンドの増加と共に急激に小さくなっていき、*k* → ∞ で *e<sub>∞</sub>* = *a<sub>c</sub>* となり、周期倍加分岐の集積点と一致する。バンドとバンドが一緒になることをバンド融合、分かれることをバンド分裂と呼び、これらもカオスの典型的な分岐の1つである。

### 4 \< *a* のとき

*a* が 4 を超えるとほとんど全ての点 *x*<sub>0</sub> から始まる挙動はマイナス無限大へ発散する。この分岐は、窓領域の終わりで発生していた内部クライシスと同じ現象だが、アトラクタが不安定化しても外側の大きなアトラクタに移行することなく軌道が無限遠に発散することが特徴である。内部クライシスと呼び分けるために境界クライシスと呼ぶ。あるいはアトラクタ破壊のクライシスとも呼ぶ。

4 \< *a* のときでも、マイナス無限大へ発散しない挙動を取る初期値 *x*<sub>0</sub> の集合が存在する。この集合を *A* とすると、[単位区間](https://ja.wikipedia.org/wiki/単位区間 "wikilink") *I* から差し引いた *I* − *A* 内にある点が発散の一途をたどることになる。一方、発散しない集合 *A* は、長さが 0 でありながら[非可算無限の点が存在する](https://ja.wikipedia.org/wiki/非可算集合 "wikilink")[カントール集合](../Page/カントール集合.md "wikilink")の一種を形成する。

### *a* \< 0 のとき

以上まではパラメータ *a* が[正](https://ja.wikipedia.org/wiki/正 "wikilink")の場合について説明してきた。ロジスティック写像は生態学上のモデルとして研究された経緯もあり、一般的に *a* が[負](https://ja.wikipedia.org/wiki/負 "wikilink")の場合については論じられることは少ない\[8\]。*a* が負の範囲のときの写像の分岐を確認すると、正の場合と似たような分岐を経て発散に至る。

*a* の値を0から減少させていくと、−1 までは *x<sub>f</sub>* = 0 の安定不動点に漸近するが、−1 を超えたところから2周期点に分岐し、正のときと同じく周期倍分岐を経てカオスへ至る\[9\]。*a* = −2 のときはカオスだが、*a* = 4 のときと同じように時間発展の厳密解を得ることができ、式()で表される。最終的には、*a* が −2 を下回るとプラス無限大へ発散するようになる\[10\]。以下に −2 から 4 までのロジスティック写像分岐図の全体像を示す。 [Bifurcation_diagram_logistic_map_(-2_to_4).png](https://ja.wikipedia.org/wiki/File:Bifurcation_diagram_logistic_map_\(-2_to_4\).png "fig:Bifurcation_diagram_logistic_map_(-2_to_4).png")

## ロジスティック方程式離散化からの導出

ロジスティック方程式は、[ピエール＝フランソワ・フェルフルスト](https://ja.wikipedia.org/wiki/ピエール＝フランソワ・フェルフルスト "wikilink")により、ある環境における生物個体数の時間変化を表すために発表されたもので、次式で表される。

ここで、*t* は連続時間、*N* は生物個体数で *t* の関数 *N*(*t*)、d*N*/d*t* は *N* の[時間微分](../Page/時間微分.md "wikilink")で時刻*t* における *N* の増加率、*K* は環境収容力で、*r* は[内的自然増加率](../Page/内的自然増加率.md "wikilink")である。*K* と *r* が[定数](../Page/定数.md "wikilink")である。 [Logistic_curve_in_some_initial_conditions.png](https://ja.wikipedia.org/wiki/File:Logistic_curve_in_some_initial_conditions.png "fig:Logistic_curve_in_some_initial_conditions.png") ロジスティック方程式（式()）の形式は、ロジスティック写像（式()）と同様に変数が大きくなると負の[フィードバック](../Page/フィードバック.md "wikilink")が働く点など一見似ているが、振る舞いは相当に異なる。式()では、パラメータの値に寄らず、十分時間経過後の *N* は常に *K* に収束する。

このようなロジスティック方程式に、1階[常微分方程式](../Page/常微分方程式.md "wikilink")の[数値解法](https://ja.wikipedia.org/wiki/数値解法 "wikilink")の一つである[オイラー法](../Page/オイラー法.md "wikilink")による差分化を行うことで、ロジスティック写像を以下のように求めることができる。後述の[\#研究史](https://ja.wikipedia.org/wiki/#研究史 "wikilink")に記す通り、[ロバート・メイ](https://ja.wikipedia.org/wiki/ロバート・メイ "wikilink")がロジスティック方程式から差分化して得たロジスティック写像を用いた研究を行い、ロジスティック写像が広く知られるようになった。

式()でパラメータ *K* と *r* の値は固定し、*Δt* を変化させながら振る舞いを観察することは、式()におけるパラメータ *a* を変化させながら振る舞いを観察することに対応する。前述の[\#パラメータ *a* による振る舞いの変化で見た通り](https://ja.wikipedia.org/wiki/#パラメータ_''a''_による振る舞いの変化 "wikilink")、*a* を変化させることでその振る舞いは複雑な変化を遂げる。これを言い換えると、ロジスティック方程式を時間離散化したときに導入した時間刻み幅 *Δt* を大きくすることは、元の方程式との誤差を単に大きくするだけでなく、カオス的振る舞いを生み出すということである。これはロジスティック方程式に限らず、式()で差分化を行い、なおかつ *Δt* が十分に大きければ、一般的な[自励系](https://ja.wikipedia.org/wiki/自励系 "wikilink")の方程式 d*x*/d*t*=*f*(*x*) においても同様にカオス的振る舞いが発生することも明らかにされている。

## 応用

### 個体群成長モデル

ロジスティック写像は、生物の個体数が世代ごとにどう変化するか予測する[計算モデル](../Page/計算モデル.md "wikilink")として[ロバート・メイ](https://ja.wikipedia.org/wiki/ロバート・メイ "wikilink")により提案された。後述の[\#研究史](https://ja.wikipedia.org/wiki/#研究史 "wikilink")に記す通り、それ以前からも式()については既に研究されていたが、この1970年代前半のメイの研究によりロジスティック写像が広く知られるようになった。

[数理生物学](../Page/数理生物学.md "wikilink")的には、ロジスティック写像は、生物の[個体群](../Page/個体群.md "wikilink")における個体数あるいは個体群密度の変動モデルを意味する。細かくは、離散世代型の単一種個体群モデルに相当する。ロジスティック写像の数理生物学的な導出の仕方は様々だが、まず最も単純な指数成長型の個体群モデルを導入して、それの拡張として生物学的条件を与えて導入する場合について説明する。

生物の個体数を *N* として、繁殖過程に関して[世代](../Page/世代.md "wikilink")が重ならないような生物種とする。このようなとき、*n* 世代における *N* の値に[比例](https://ja.wikipedia.org/wiki/比例 "wikilink")して増えていく単純なモデルが考えられる。

ここで *c* は、*n* によらない定数とし、正味の増殖率を意味する。個体当たりの結実率・出産率と繁殖可能になるまでの生存率との積とも考えられる。また、*c* が *n* によらない定数とは、環境条件が時間的に変化しない場合である。

この単純なモデルでは、個体数 *N* は指数関数的に無制限に増加することになる。一方で、[密度効果](https://ja.wikipedia.org/wiki/密度効果 "wikilink")を考えると、個体数が多くなると資源の奪い合いが激しくなり、繁殖率が低下すると仮定できる。これも単純なモデルの1つとして、次のような増殖率 *c* と個体数 *N* の関係を考えることができる。

ここで、*a* が最大繁殖率、*b* が繁殖率に対する密度効果の強さを表している。 式()と式()を合成すると、

となり、*x<sub>n</sub>* = *b* *N<sub>n</sub>*/*a* と変数変換すれば、ロジスティック写像と同形の次式が得られる。

*N* の値が負のとき、個体数としての意味が成さないので、この制約から、*a* ≤ 4 という制約が生まれる。

1941年、昆虫学者の[内田俊郎](https://ja.wikipedia.org/wiki/内田俊郎 "wikilink")は離散世代型の生物種である[マメゾウムシ](https://ja.wikipedia.org/wiki/マメゾウムシ "wikilink")の密閉容器中の繁殖実験で、個体数変動が振動しながら収束していくデータが得ている。これは[ロジスティック方程式](../Page/ロジスティック方程式.md "wikilink")では再現できないものだったが、ロジスティック写像の 2 \< *a* ≤ 3 の場合で再現できる。また、ロジスティック写像の 3 \< *a* ≤ 3.44949... に相当するような実験データとしては、内田俊郎がヨツモンマメゾウムシによる実験で、昆虫学者のニコルソンがヒツジキンバエによる実験で2周期振動型の個体数変化のデータを得ている。

ただし、ロジスティック写像のみが離散世代型単一種個体群モデルに適用できるモデルというわけではない。1953年に内田俊郎らも自身の実験データを別のモデルで再現している。また、ロジスティック写像ではパラメータの生物学的解釈が難しいとする意見もある\[14\]。ロジスティック写像型のモデル以外にも、様々な種類のモデルが数理生物学で研究されており。[リッカーモデル](https://ja.wikipedia.org/wiki/リッカーモデル "wikilink")などもパラメータ変化によるカオス的振る舞いも見せる。また、実験室の厳密に管理された条件では実際の生物個体群がカオス的変動を示した例はあるが、野外環境での個体群が本当にカオス的変動を起こすのかどうかは、様々な環境変動の影響があることからまだ論争が続いている\[15\]。

### 擬似乱数生成器

コンピュータのソフトウェア計算から[擬似乱数](../Page/擬似乱数.md "wikilink")を得るための手法の１つとして、カオスを用いたものがある。ロジスティック写像もそのための1つとして研究されている。擬似乱数生成器用には、ピュアカオスの *a* = 4 のときを対象にすることが多い。研究最初期としては、後述の[\#研究史](https://ja.wikipedia.org/wiki/#研究史 "wikilink")に示す通り、1947年に[スタニスワフ・ウラム](../Page/スタニスワフ・ウラム.md "wikilink")と[ジョン・フォン・ノイマン](../Page/ジョン・フォン・ノイマン.md "wikilink")が、*a* = 4 のロジスティック写像により擬似乱数が得られることを指摘している。写像の初期値鋭敏性により、種と呼ばれる初期値が非常に近い2つの値から計算しても異なる時系列を得ることができる。また、ロジスティック写像の計算では、区間 (0, 1) の中で常に値を取るので、[浮動小数点](https://ja.wikipedia.org/wiki/浮動小数点 "wikilink")だけでなく[固定小数点](https://ja.wikipedia.org/wiki/固定小数点 "wikilink")でも問題無く計算できる利点がある。

一方で次のような欠点もある。*a* = 4 のロジスティック写像の密度関数は式()で示された分布になっており、区間 (1, 0) 内の数値の発生確率は一様ではなくU字型になっている。そのため、一様分布の数列にするために何らかの処理が必要となる。

その方法としては、

1.  得られた実数列を一様乱数列に直接変換する方法
2.  得られた実数列に対して[閾値](https://ja.wikipedia.org/wiki/閾値 "wikilink")を使って 0, 1 に振り分け、これを繰り返して[ベルヌーイ試行](../Page/ベルヌーイ試行.md "wikilink")列による一様乱数を得る方法
3.  得られた実数列の[仮数](https://ja.wikipedia.org/wiki/仮数 "wikilink")の一部のビット数列を一様乱数と見なす方法

以上のようなものがある。また、*x<sub>n</sub>* と*x*<sub>*n*+1</sub> には強い相関があるので、上記の1、2番目の方法では多重化すなわち反復合成を繰り返しが必須である。そのために必要な多重度は、1番目の方法に対するファタックらの研究によると10回以上、2番目の方法に対する香田らの研究によると16回以上である。

さらに、*a* = 4 のロジスティック写像であっても初期値によっては不動点に収束する。また、デジタルコンピュータを用いてカオスを計算する一般的問題として、コンピュータでは有限計算精度で計算されるため、原理的に真にカオスな値を得ることができないという点がある。このようなカオスは擬似カオスと呼ばれる。この有限計算精度によって、どんな初期値から不動点に収束するかの判別は実際に数値計算させるしか方法が無くなってしまう。

以上のようなロジスティック写像などのカオスを応用した擬似乱数生成の研究は続けられているが、2006年時点で、カオスを乱数源とする方法で乱数性、高速性に優れた擬似乱数生成器の実現はまだ報告されていないという指摘もある\[16\]。上記で説明した方法以外に、連立3変数のロジスティック写像や整数化したロジスティック写像などによる擬似乱数生成研究がある\[17\]。

## 研究史

式()がロジスティック写像と呼ばれる前の時代としては、1947年に[スタニスワフ・ウラム](../Page/スタニスワフ・ウラム.md "wikilink")と[ジョン・フォン・ノイマン](../Page/ジョン・フォン・ノイマン.md "wikilink")が、  を使用すれば[疑似乱数](https://ja.wikipedia.org/wiki/疑似乱数 "wikilink")を得ることができることを指摘している\[18\]。 [Stanislaw_Ulam_ID_badge.png](https://ja.wikipedia.org/wiki/File:Stanislaw_Ulam_ID_badge.png "fig:Stanislaw_Ulam_ID_badge.png")\]\] [John_von_Neumann_ID_badge.png](https://ja.wikipedia.org/wiki/File:John_von_Neumann_ID_badge.png "fig:John_von_Neumann_ID_badge.png")\]\]

式()は *a* = 4 のロジスティック写像に相当する。この頃はカオスという言葉もまだ存在していない。また、ウラムとノイマンはこの式と[テント写像](../Page/テント写像.md "wikilink")が[位相同型](../Page/位相同型.md "wikilink")の関係にあること、この式の[不変測度](https://ja.wikipedia.org/wiki/不変測度 "wikilink")が[一様分布](../Page/一様分布.md "wikilink")ではないことも明らかにしていた。

1950年にプルキン(S. P. Pulkin) が連続写像や区分的に連続な写像から振動の繰り返し系列が生じることを示したが、この論文中にもロジスティック写像と同じ式が現れる。その後、1950年代前半から1960年代後半にかけて、他の研究者からも任意のパラメータ *a* を備えた形式についても研究されている\[19\]。ミュルバーク(Pekka Juhana Myrberg) は1958年から1963年にかけて、*x*<sub>*n*+1</sub> = *x*<sub>*n*</sub><sup>2</sup> − *λ* という形式の1次元2次写像の分岐現象について調べ、1963年にはこの写像による周期倍加分岐も研究している。カオス現象自体の発見（可視化）は、1961年の[上田睆亮](../Page/上田睆亮.md "wikilink")や1963年の[エドワード・ローレンツ](../Page/エドワード・ローレンツ.md "wikilink")などで達成されるが、まだこの時点でもカオスという共通語は存在していない。 [BobMayHarvard.jpg](https://ja.wikipedia.org/wiki/File:BobMayHarvard.jpg "fig:BobMayHarvard.jpg")\]\] その後、数理生物学者の[ロバート・メイ](https://ja.wikipedia.org/wiki/ロバート・メイ "wikilink")が、[生態学](../Page/生態学.md "wikilink")の問題に取り組む過程で式()を用いた研究を1970年初頭から開始する。1973年に、メイは[ロジスティック方程式](../Page/ロジスティック方程式.md "wikilink")を離散時間化により差分方程式に変換させて、式()を求めた。メイはこの式を基に数値実験を行い、パラメータ*a*による変数の振る舞いを変化を研究した。1974年と1976年にメイはこれらの研究成果を発表し、これらの論文によって、カオスと関係の深い写像としてロジスティック写像が広く知られるようになる。さらに、ロジスティック写像型の関数が持つ非周期軌道と周期軌道の存在について独立に研究していたジェームス・ヨーク([James A.
