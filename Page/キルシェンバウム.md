**キルシェンバウム**（Kirshenbaum）または**ASCII IPA**（アスキーアイピーエー）は、[国際音声記号](../Page/国際音声記号.md "wikilink")（IPA）を元に作られた [ASCII](../Page/ASCII.md "wikilink") 文字のみで表現された[音声記号](https://ja.wikipedia.org/wiki/音声記号 "wikilink")である。国際音声記号には特殊な文字が多く、そのままでは[コンピュータ](../Page/コンピュータ.md "wikilink")上で扱うことが困難なので、ASCIIの7ビット文字のみで国際音声記号を表現する手段として考案された。類似の音声記号体形に[SAMPA](../Page/SAMPA.md "wikilink")及びそれを改訂した[X-SAMPA](../Page/X-SAMPA.md "wikilink")があり、。

キルシェンバウムの名は、この方式の作成を主導したエヴァン・キルシェンバウムの名に基づく。元来、[Usenet](https://ja.wikipedia.org/wiki/Usenet "wikilink")、特に sci.lang と alt.usage.english の両ニュースグループにおいて音声記号を表現する必要に迫られたことから作られた。

## 特徴

キルシェンバウムでは、まず音を[弁別素性](https://ja.wikipedia.org/wiki/弁別素性 "wikilink")に分析し、各素性に3文字の略称を与える。たとえば「rnd」で円唇性を、「blb」で[両唇音](../Page/両唇音.md "wikilink")を表す。ひとつの音を素性の束として表すには、複数の素性を[コンマ](../Page/コンマ.md "wikilink")で区切って[波括弧](https://ja.wikipedia.org/wiki/波括弧 "wikilink") {……} で囲む。

よく使う音は、IPAと同様に字母的な記号を与える。たとえば、「b」は[有声両唇破裂音](https://ja.wikipedia.org/wiki/有声両唇破裂音 "wikilink")であるから、「{blb,vcd,stp}」と書いたのと同じことになる。

字母的記号のうしろに弁別素性を不等号 \<……\> で囲んで置くことができる。これによって記号の数を減らすことが可能になっている。たとえば[有声咽頭摩擦音](../Page/有声咽頭摩擦音.md "wikilink")は、[無声咽頭摩擦音](https://ja.wikipedia.org/wiki/無声咽頭摩擦音 "wikilink")を表す記号「H」と有声を表す素性「vcd」を使って「H<vcd>」と表される。

アルファベットの小文字は基本的にIPAと同じ音を表す。ただし、英語の音声の記述が簡単になることを重視している関係上、「r」はIPAの（[歯茎接近音](https://ja.wikipedia.org/wiki/歯茎接近音 "wikilink")）を表し、（[歯茎ふるえ音](../Page/歯茎ふるえ音.md "wikilink")）は「r<trl>」と書く。仕様書によれば、英語以外を主に使うニュースグループでは対応関係を変えてもよいとしている。ASCIIにない字は大文字やアルファベット以外の文字を使って表現する。

弁別素性以外にも、IPAでは使わないいくつかの記号を文字に後置することができる。

  - [終止符](../Page/終止符.md "wikilink") . は子音のうしろで[そり舌音](../Page/そり舌音.md "wikilink")を、母音のうしろで[円唇母音](../Page/円唇母音.md "wikilink")を表す。
  - [引用符](../Page/引用符.md "wikilink") " は子音のうしろで[口蓋垂音](../Page/口蓋垂音.md "wikilink")を、母音のうしろで[中舌母音](https://ja.wikipedia.org/wiki/中舌母音 "wikilink")を表す。
  - [ハイフンマイナス](../Page/ハイフンマイナス.md "wikilink") - は[非円唇母音](https://ja.wikipedia.org/wiki/非円唇母音 "wikilink")を表す。
  - [サーカムフレックス](../Page/サーカムフレックス.md "wikilink") ^ は[硬口蓋音](../Page/硬口蓋音.md "wikilink")を表す。
  - [グレイヴ・アクセント](../Page/グレイヴ・アクセント.md "wikilink") \` は無声子音のうしろで[放出音](../Page/放出音.md "wikilink")を、有声子音のうしろで[入破音](../Page/入破音.md "wikilink")を表す。
  - [感嘆符](../Page/感嘆符.md "wikilink") \! は[吸着音](../Page/吸着音.md "wikilink")を表す。

### その他の記号

算用数字は[声調](../Page/声調.md "wikilink")を表すために使う。各言語で伝統的に決まっている数字を使用する（たとえば[北京語](../Page/北京語.md "wikilink")では1から4までの数字を使う）。

弁別素性を表す記号は不等号で囲むが、字母的記号と紛れる心配のないものは不等号を省略できる。以下の表では省略した形で載せる。

| 記号    | 素性略号 | 意味                                                       | 備考                |
| ----- | ---- | -------------------------------------------------------- | ----------------- |
| '     |      | 第一強勢                                                     |                   |
| ,     |      | 第二強勢                                                     |                   |
| \#    |      | 音節または単語の境界                                               |                   |
| :     | lng  | [長](https://ja.wikipedia.org/wiki/長_\(発音記号\) "wikilink") | 半長・極短の記号は定義されていない |
| <o>   | vls  | [無声音](../Page/無声音.md "wikilink")                         | 有声音に後続したとき        |
| <h>   | asp  | [帯気音](https://ja.wikipedia.org/wiki/帯気音 "wikilink")      |                   |
| \-    | syl  | 音節主音                                                     | 子音に後続したとき         |
| <r>   | rzd  | [R音化](../Page/R音性母音.md "wikilink")                       |                   |
| \<?\> | mrm  | [息もれ声](../Page/息もれ声.md "wikilink")                       |                   |
| <w>   | lzd  | [唇音化](https://ja.wikipedia.org/wiki/唇音化 "wikilink")      |                   |
| ;     | pzd  | [口蓋化](../Page/口蓋化.md "wikilink")                         |                   |
| \~    | vzd  | [軟口蓋化](https://ja.wikipedia.org/wiki/軟口蓋化 "wikilink")    | 子音に後続したとき         |
| <H>   | fzd  | [咽頭化](https://ja.wikipedia.org/wiki/咽頭化 "wikilink")      |                   |
| \[    | dnt  | [歯音](../Page/歯音.md "wikilink")                           |                   |
| \~    | nzd  | [鼻音化](../Page/鼻音化.md "wikilink")                         | 母音に後続したとき         |
| <o>   | unx  | [開放なし](../Page/内破音.md "wikilink")                        | 無声音に後続したとき        |

キルシェンバウムには1989年に国際音声記号に加えられた多数の変更が反映されていない。[息もれ声](../Page/息もれ声.md "wikilink")の素性はあるが、[きしみ声](../Page/きしみ声.md "wikilink")の素性がないなど。
