**TV ASAHI NETWORK SPORTS**は、[テレビ朝日](../Page/テレビ朝日.md "wikilink")系列のスポーツ番組ブランドである。

[2002年](../Page/2002年.md "wikilink")に「TV ASAHI NETWORK SPORTS」ブランドを発表し、スポーツ選手の影をモチーフとしたマスコット及び統一イメージソングも設けられた。イメージソングは基本的に年毎に変更されている。テレビ朝日系列のスポーツ番組においてオープニングでマスコットとともに「TV ASAHI NETWORK SPORTS」のロゴが現れていた。

[2004年](../Page/2004年.md "wikilink")を最後に「TV ASAHI NETWORK SPORTS」の看板を外したが、統一イメージソングは[2005年](../Page/2005年.md "wikilink")以降も採用されている。

## イメージソング

イメージソングは2004年までは[B'z](../Page/B'z.md "wikilink")（2004年上半期は[TMG](../Page/TMG_\(バンド\).md "wikilink")）、2005年7月以降は[SMAP](../Page/SMAP.md "wikilink")が担当している。2005年上半期は設定せず。

  - 2002年「[熱き鼓動の果て](../Page/熱き鼓動の果て.md "wikilink")」
  - [2003年](../Page/2003年.md "wikilink")「[IT'S SHOWTIME\!\!](../Page/IT'S_SHOWTIME!!.md "wikilink")」
  - 2003年「[野性のENERGY](../Page/野性のENERGY.md "wikilink")」
  - 2004年上半期「[OH JAPAN 〜OUR TIME IS NOW〜](../Page/OH_JAPAN_〜OUR_TIME_IS_NOW〜.md "wikilink")」
  - 2004年下半期「[ARIGATO](../Page/ARIGATO_\(B'zの曲\).md "wikilink")」
  - 2005年「[BANG\! BANG\! バカンス\!](../Page/BANG!_BANG!_バカンス!.md "wikilink")」
  - 2005年11月「[Triangle](../Page/Triangle_\(SMAPの曲\).md "wikilink")」
  - [2006年](../Page/2006年.md "wikilink")「[buzzer beater](../Page/Dear_WOMAN.md "wikilink")」
  - [2007年](../Page/2007年.md "wikilink")11月「[弾丸ファイター](../Page/弾丸ファイター.md "wikilink")」
  - [2009年](../Page/2009年.md "wikilink")7月「[スーパースター★](https://ja.wikipedia.org/wiki/そっと_きゅっと/スーパースター★ "wikilink")」

### 例外

  - サッカー中継では[サラ・ブライトマン](https://ja.wikipedia.org/wiki/サラ・ブライトマン "wikilink")「[A Question Of Honour](../Page/クエスチョン・オブ・オナー.md "wikilink")」が使われている。ただしハーフタイムにおいて統一イメージソングも使われている。
  - ISUグランプリシリーズでは2006年以降も「Triangle」が使われている。
  - [世界水泳2007及び](../Page/2007年世界水泳選手権.md "wikilink")[世界競泳2007では](../Page/インターナショナル・スイム・ミート2007.md "wikilink")「[mermaid](https://ja.wikipedia.org/wiki/Super.modern.artistic.performance "wikilink")」が使用された。
  - [全国高等学校野球選手権大会](https://ja.wikipedia.org/wiki/全国高等学校野球選手権大会 "wikilink")など系列局制作では、イメージソングは使用されない場合が多い。
  - [世界水泳2011では](https://ja.wikipedia.org/wiki/2011年世界水泳選手権 "wikilink")「[ultra soul](https://ja.wikipedia.org/wiki/ultra_soul "wikilink")」をリアレンジした「[ultra soul 2011](https://ja.wikipedia.org/wiki/C'mon "wikilink")」が使用された。
