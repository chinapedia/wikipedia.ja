文法における**性**（せい）つまり**文法的性**（ぶんぽうてきせい）、**文法性**とは、関連する語のふるまいに文法的に反映する[名詞](../Page/名詞.md "wikilink")の分類体系のこと\[1\]\[2\]で、名詞の[文法範疇](https://ja.wikipedia.org/wiki/文法範疇 "wikilink")の一つ\[3\]である。[バントゥー語群](../Page/バントゥー語群.md "wikilink")や[コーカサス諸語](../Page/コーカサス諸語.md "wikilink")の記述では**名詞類**または**名詞クラス**とも呼ばれるが、実質的な差はほとんどない\[4\]。この名詞の分類が実際の[性と一致する場合もあるが](../Page/性_\(生物学\).md "wikilink")、そうでないことも多い\[5\]。[類別詞](../Page/類別詞.md "wikilink")とは区別すべきとされる\[6\]\[7\]。

名詞の性によって、その名詞と文法的に関連する語のかたちが変わる現象を「性の[一致](../Page/一致.md "wikilink")」という。例えば[ロシア語](https://ja.wikipedia.org/wiki/ロシア語 "wikilink")では主語の名詞の種類によって、[動詞](../Page/動詞.md "wikilink")の過去形が男性・女性・中性の3通りにかたちを変える。これは、動詞の過去形が主語の名詞と性の一致をしているのである。

| ロシア語の例\[8\] |
| ----------- |
| a.          |
|             |
|             |
| 「           |
| b.          |
|             |
|             |
| 「           |
| c.          |
|             |
|             |
| 「           |

動詞のほかに、[形容詞](../Page/形容詞.md "wikilink")、[限定詞](../Page/限定詞.md "wikilink")、[数詞](../Page/数詞.md "wikilink")、焦点標識などが性の一致をする言語がある\[9\]。

## 性の分け方

名詞をある性に分類することを、性を付与するという。どの性を付与するかの基準は言語ごとに異なる。大きく分けると、名詞の意味によって分類する言語と、名詞の語形に基づいて形式的に分類する言語がある。

### 意味による分類

性を持つ言語には、名詞の意味によって性を付与する言語がある。例えば[タミル語](../Page/タミル語.md "wikilink")では、男性と男神を指す名詞は男性名詞、女性と女神を意味する名詞は女性名詞、それ以外は中性名詞に分類される。

| 意味による性の付与の例（[タミル語](../Page/タミル語.md "wikilink")） |
| ----------------------------------------------- |
| 意味                                              |
| 男性と男神                                           |
| சிவன் (civaṉ)                                   |
| 女性と女神                                           |
| காளி (kāḷi)                                     |
| その他                                             |
| மரம் (maram)                                    |

タミル語ではほぼ例外なく、意味と性が対応している。

一方、[コンゴ民主共和国](../Page/コンゴ民主共和国.md "wikilink")、[スーダン](../Page/スーダン.md "wikilink")、[中央アフリカ共和国](../Page/中央アフリカ共和国.md "wikilink")などで話されている[ザンデ語](https://ja.wikipedia.org/wiki/ザンデ語 "wikilink")でも、名詞は意味によって性を付与されるが、例外が存在する。

| 意味による性の付与の例（[ザンデ語](https://ja.wikipedia.org/wiki/ザンデ語 "wikilink")） |
| ------------------------------------------------------------------ |
| 意味                                                                 |
| 男性                                                                 |
| 女性                                                                 |
| その他の[有生物](https://ja.wikipedia.org/wiki/有生物 "wikilink")            |
| その他                                                                |

男性の人間と女性の人間は、それぞれ男性名詞・女性名詞に分類されるが、動物名詞には例外があり、動物を意味しないおよそ80の名詞がこの性に分類される。

  - [天体](../Page/天体.md "wikilink")：*diwi*「[月](https://ja.wikipedia.org/wiki/月 "wikilink")」、*wangu*「[虹](../Page/虹.md "wikilink")」
  - [金属](../Page/金属.md "wikilink")の物体：*bande*「[ハンマー](../Page/槌.md "wikilink")」、*tongo*「[指輪](https://ja.wikipedia.org/wiki/指輪 "wikilink")」
  - [食用植物](../Page/野菜.md "wikilink")：*abangbe*「[さつまいも](../Page/サツマイモ.md "wikilink")」、*baundu*「[豆](../Page/豆.md "wikilink")」
  - 金属でない丸い物体：*mbasa*「[呼び子](../Page/呼子笛.md "wikilink")」、*badupo*「[ボール](../Page/ボール.md "wikilink")」
  - その他：*ze*「傷痕」

この例外にはザンデの神話によって説明できるものが少しだけあるが、ほとんどのものは例外としか言えない。しかし、ザンデ語の性の付与はほとんど意味に基づいているといえる。

### 形式による分類

形式的な特徴によって性を付与する言語も存在する。しかし、そのような言語でも性の分類の中核には意味的な基準があり、形式的な基準は意味の基準が適用できない場合に用いられる。形式的基準には、形態論的なものと、音韻論的なものがある。

#### 形態論による分類

意味的な基準が適用できない場合に、その名詞の屈折のしかたによって性を付与する言語がある。

例えばロシア語では、男性・女性・中性の三つの性があるが、男性と女性の付与には意味的な基準がある。男性や高等動物の雄を指す名詞は男性名詞に、女性や高等動物の雌を意味する名詞は女性名詞に分類される。

  - 男性や雄→男性名詞：** (otec)「父」、** (djadja)「おじ」、** (lev)「雄ライオン」
  - 女性や雌→女性名詞：** (mat')「母」、** (tjotja)「おば」、** (l'vica)「雌ライオン」

しかし、性別を持っていないものを表す名詞は、男性のこともあれば女性や中性のこともある。

| 性別を持たないものを指すロシア語の名詞の性      |
| -------------------------- |
| 男性                         |
| журнал (žurnal)「雑誌」        |
| дом (dom)「家」               |
| чай (čaj)「紅茶」              |
| автомобиль (avtomobil′)「車」 |
| вечер (večer)「夕方」          |
| флаг (flag)「旗」             |
| закон (zakon)「法律」          |

このように、性別を持たないものを指すロシア語の名詞は意味的には似ていても違う性が付与される。性別を持たないものを意味する名詞を分類する基準は[曲用](https://ja.wikipedia.org/wiki/曲用 "wikilink")のタイプという形態論的なものである。

| ロシア語の単数名詞の曲用                                        |
| --------------------------------------------------- |
|                                                     |
| [主格](../Page/主格.md "wikilink")                      |
| [対格](../Page/対格.md "wikilink")                      |
| [生格](https://ja.wikipedia.org/wiki/生格 "wikilink")   |
| [与格](../Page/与格.md "wikilink")                      |
| [造格](https://ja.wikipedia.org/wiki/造格 "wikilink")   |
| [前置格](https://ja.wikipedia.org/wiki/前置格 "wikilink") |
|                                                     |

ロシア語には大きく分けて四つの曲用のタイプがあり、タイプ I は男性名詞、タイプ IIとIIIは女性名詞、それ以外は中性名詞である。ただし、曲用のタイプよりも意味的な基準が優先するため、タイプIIにも男性名詞が存在する。このような、曲用のタイプと性との強い相関はインド・ヨーロッパ語族にはめずらしくない。

#### 音韻論による分類

意味的な基準が適用できない場合に、音韻的な基準によって性を付与する言語もある。

例えば[アファル語](https://ja.wikipedia.org/wiki/アファル語 "wikilink")では、人間の男性と動物の雄を表す名詞は男性名詞、人間の女性と動物の雌を表す名詞は女性名詞に分類される。

  - 男性や雄→男性名詞：*bàxa*「息子」、*toobokòyta*「兄弟」、*barisèyna*「男の先生」、*kùta*「雄犬」、*abbà*「父」
  - 女性や雌→女性名詞：*baxà*「娘」、*toobokoytà*「姉妹」、*bariseynà*「女の先生」、*kutà*「雌犬」、*gabbixeèra*「腰の細い女性」

それ以外の名詞は音韻的に性が決まる。強勢のある母音で終わる名詞は女性名詞であり、その他は男性名詞である。

  - 強勢のある母音で終わる→女性名詞：*catò*「助け」、*karmà*「秋」
  - そのほか→男性名詞：*cedèr*「夕食時」、*gilàl*「冬」、*tàmu*「味」、*baànta*「トランペット」

## 性の数

[グレヴィル・コーベット](https://ja.wikipedia.org/wiki/グレヴィル・コーベット "wikilink")は世界257の言語について、性がいくつあるか調査した。それによると、性のない言語が約半数で145、性が二つある言語が50、三つあるのが26言語、四つが12言語、五つ以上持つ言語は24あった。

地理的にみると、[アフリカ](../Page/アフリカ.md "wikilink")南部・中西部に広がる[ニジェール・コンゴ語族](../Page/ニジェール・コンゴ語族.md "wikilink")のほぼ全ての言語が性を持っているほか、[コイサン語族](https://ja.wikipedia.org/wiki/コイサン語族 "wikilink")や[アフロ・アジア語族](../Page/アフロ・アジア語族.md "wikilink")など、アフリカ・中東の言語には性があるものが多い。またヨーロッパの[印欧語族](https://ja.wikipedia.org/wiki/印欧語族 "wikilink")、コーカサスの[ナフ・ダゲスタン語族](https://ja.wikipedia.org/wiki/ナフ・ダゲスタン語族 "wikilink")、南アジアの印欧語族や[ドラヴィダ語族](../Page/ドラヴィダ語族.md "wikilink")も性のある言語の多い語族である。

一方、太平洋に分布する[オーストロネシア語族](../Page/オーストロネシア語族.md "wikilink")、アジアの[シナ・チベット語族](../Page/シナ・チベット語族.md "wikilink")、シベリアの[ウラル語族](../Page/ウラル語族.md "wikilink")・[アルタイ語族](https://ja.wikipedia.org/wiki/アルタイ語族 "wikilink")には性がほとんど見られない。

性がある言語では、二つあるものがもっとも多く、広い地域に分布している。アフロ・アジア語族は普通二つの性を持つ。印欧語族は三つあるものが多いが、二つになっているものも多い。四つある言語はナフ・ダゲスタン語族によく見られる。五つ以上の性を持つ言語はニジェール・コンゴ語族に多い。ナイジェリアの[フラ語](https://ja.wikipedia.org/wiki/フラ語 "wikilink")は例外的に多く、20の性を持っている。他には、パプア・ニューギニアのが13、北オーストラリアのが15の性を持つ。

## 印欧語族の性

\[\[<File:Gender> in European languages.png|thumb|欧州の言語の文法的性。 水色、文法的性なし。

黄色、通性•中性。

赤色、男性•女性。

暗青色、男性•女性•中性。

緑色、有生物•無生物。 \]\]

[印欧祖語](https://ja.wikipedia.org/wiki/印欧祖語 "wikilink")の名詞には元来は男性・女性・中性の3つの性があり、形容詞の変化もそれに[一致](../Page/一致.md "wikilink")していたとされる。現在の印欧語族においては、これら3つの性を全て残している言語もあれば、中性が消失して男性・女性のみになったもの、性をほぼ完全に失った言語もあり、性の様相は多彩である。

男性・女性・中性の区別を全て残している言語としては、[ラテン語](../Page/ラテン語.md "wikilink")、[ドイツ語](../Page/ドイツ語.md "wikilink")、[スラヴ語等がある](../Page/スラヴ語派.md "wikilink")。これらのうちスラヴ語は男性をさらに活動体と不活動体に分け、[ポーランド語](../Page/ポーランド語.md "wikilink")に至ってはこの区別に加えて複数形を男性人間と非男性人間のカテゴリーに分ける性質も有している。[ヘブライ文字](https://ja.wikipedia.org/wiki/ヘブライ文字 "wikilink")で表記されるもののドイツ語から強い影響を受けた[イディッシュ語](../Page/イディッシュ語.md "wikilink")には、男性、女性、中性という3つの性がある。これは後述する[セム語の](../Page/セム語派.md "wikilink")[ヘブライ語](../Page/ヘブライ語.md "wikilink")とは異なる区分である。

[英語](../Page/英語.md "wikilink")では性はほぼ失われており、[生物学](../Page/生物学.md "wikilink")的性に対応したもの及び[擬人法](../Page/擬人法.md "wikilink")を除けば、[船](https://ja.wikipedia.org/wiki/船 "wikilink")や[国名](../Page/国名.md "wikilink")など一部の名詞、3人称単数[代名詞](../Page/代名詞.md "wikilink")においてのみ現れる。また、[ペルシア語](https://ja.wikipedia.org/wiki/ペルシア語 "wikilink")においては、性は代名詞も含めてほぼ完全に失われている。

名詞の文法的性は、生物においては原則として生物学的性と一致するが、非生物においてはその対象の「男性性」や「女性性」とはほとんど無関係である。また、同じ対象を表す名詞について、言語によって性は異なる。ラテン語では太陽は男性名詞、月は女性名詞で、そこから派生したフランス語なども同じだが、[ゲルマン語派](../Page/ゲルマン語派.md "wikilink")では逆となっているし、[スラヴ語派](../Page/スラヴ語派.md "wikilink")では太陽が中性となる。文法的性が生物学的性と一致しない場合も稀にある。

また、語形と性が一致しない場合もわずかにあり、例えばポーランド語の [pianista](https://ja.wikipedia.org/wiki/wikt:pl:pianista "wikilink") 〈男性ピアニスト〉は代名詞や接続する形容詞が男性形となる歴とした男性名詞だが、女性語尾 -a を持ち単数形は女性名詞と全く同じ[曲用](https://ja.wikipedia.org/wiki/曲用 "wikilink")をする\[10\]。一方、フランス語などのように名詞の曲用を失った言語では、名詞だけでは性が判別できず、添えられる[冠詞](../Page/冠詞.md "wikilink")や形容詞で初めてわかることもある。ドイツ語も[格変化](../Page/格変化.md "wikilink")は残っているがこれに近い。

非常に変わった例として[ウェールズ語](../Page/ウェールズ語.md "wikilink")がある。全体としては性の指標は失われているが、ある場所で最初の子音が他の子音に変わるという特徴がある。たとえば merch という単語は女の子を意味するが、定冠詞を付けた形は y ferch である。これは女性名詞にのみ起こる現象で、男性名詞は定冠詞の後でも変化しない。性は名詞の後に続く形容詞にも同様に影響する。たとえば、「大きい女の子」は y ferch fawr だが、 「大きい息子」は y mab mawrである。

動詞の変化は通常は性によらないが、スラヴ語においては、[過去時制](https://ja.wikipedia.org/wiki/過去時制 "wikilink")のみ性によって変化する\[11\]。

## セム諸語

[ヘブライ語](../Page/ヘブライ語.md "wikilink")や[アラビア語](../Page/アラビア語.md "wikilink")などセム諸語は原則として男性と女性の2種類の性があり中性はない。以下は代表的なアラビア語を例にして説明する。名詞は語尾で男性か女性か判別できるものが多い。大抵の女性名詞は[ة](../Page/ة.md "wikilink")（ター・マルブータ）で終わり、男性名詞にةをつけて女性名詞にしたものも多い。双数や複数、動詞の人称変化でも男女の区別がある。形容詞は叙述用法、限定用法にかかわらず修飾する名詞によって形を変え、ほとんどの場合にはةをつけることで女性形となる。但し[定冠詞は性によらず常に](https://ja.wikipedia.org/wiki/冠詞#アラビア語 "wikilink") ال (al)を用いる。また名詞や形容詞の格変化も性によらない。ヘブライ語もアラビア語と同様に冠詞は性によらずהを用い、複数形や動詞の人称変化でも性の区別があり、形容詞は修飾する名詞によって語尾などが変化する。

アラビア語では体の対を成す部分や地名は女性名詞である。また、集合名詞を女性名詞化することで個別の名詞を作ることが出来る。例えばشجر（shajar: 樹木）に対するشجرة（shajara: 木）などである。

## コーカサス諸語

[コーカサス諸語](../Page/コーカサス諸語.md "wikilink")には、男性、女性、活動体やある種の物体、その他の４類を持つ言語がいちばん多いが、名詞クラスがない言語、2クラスだけある言語、8クラスをもつ[バツ語](https://ja.wikipedia.org/wiki/バツ語 "wikilink")もある。には虫のクラスというものがある。コーカサス諸語ではクラスは名詞そのものには明示されないが、動詞、形容詞、代名詞によって示される。

## アルゴンキン語族

北米の[アルゴンキン語族](https://ja.wikipedia.org/wiki/アルゴンキン語族 "wikilink")では活動体・不活動体の2クラスを区別するが、この区別はむしろ力のある・なしの区別だとする人もいる。すべての生物、また神聖なものや大地につながりのあるものは力のあるものと考えられ「活動体」に分類される。しかし分類は極めて恣意的で、たとえば「キイチゴ」が活動体、「イチゴ」が不活動体となる。

## バントゥー諸語

[バントゥー諸語](https://ja.wikipedia.org/wiki/バントゥー諸語 "wikilink")にはのべ22種の名詞クラスがある。1言語でそのすべてを持っているものはないが、少なくとも10種は持っている。たとえば[スワヒリ語](https://ja.wikipedia.org/wiki/スワヒリ語 "wikilink")には15種、[ソト語](https://ja.wikipedia.org/wiki/ソト語 "wikilink")には18種ある。人間に関して数種ある場合が多い。

## モンゴル語

[モンゴル語](https://ja.wikipedia.org/wiki/モンゴル語 "wikilink")の単語にも、男性語、女性語の分類があるが、これは[母音調和](../Page/母音調和.md "wikilink")に関する分類であり、文法的性とは異なり[一致](../Page/一致.md "wikilink")を生じない。モンゴル語では、母音を男性母音、女性母音、中性母音に分類し、男性母音の伴う語を男性語、女性母音の伴う語または中性母音のみ伴う語を女性語と呼ぶ。
