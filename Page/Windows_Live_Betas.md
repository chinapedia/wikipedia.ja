**Windows Live Betas**（ウィンドウズ ライブ ベータ、旧名：Windows Live Ideas）とは、[2005年](../Page/2005年.md "wikilink")11月に開設された[Windows Liveサービスの紹介ページである](../Page/Windows_Live.md "wikilink")。[Windows Live Essentialsのパブリックベータ版がリリースされたのに伴い](https://ja.wikipedia.org/wiki/Windows_Live_Essentials "wikilink")、[2008年](https://ja.wikipedia.org/wiki/2008年 "wikilink")[9月17日](../Page/9月17日.md "wikilink")にサービスを終了した。

## 機能

Windows Live ベータには、以下のような機能があった。

  - Windows Live 製品の製品紹介。
  - ベータサービスへの申し込みと参加（サインアップ）。
  - [公式ブログへの](../Page/ブログ.md "wikilink")[リンク](../Page/ハイパーリンク.md "wikilink")。
