**コンドルール**または**コンドリュール**（chondrule）とは、多くの[隕石](https://ja.wikipedia.org/wiki/隕石 "wikilink")に含まれている球状の粒子である。コンドリュールという名称は、[古代ギリシャ語](https://ja.wikipedia.org/wiki/古代ギリシャ語 "wikilink")のchondoros（『粒』）に由来する。

## 概説

[Chondrules_grassland_1.jpg](https://ja.wikipedia.org/wiki/File:Chondrules_grassland_1.jpg "fig:Chondrules_grassland_1.jpg") コンドリュールは、多くの隕石中に見られる。コンドリュールを含む隕石は[コンドライト](https://ja.wikipedia.org/wiki/コンドライト "wikilink")と呼ばれる。コンドリュールの大きさや、含有量でコンドライトは分類される。

コンドリュールは1500℃から1900℃に達する急な加熱の後、急速に冷却されたことによってできたと考えられている。隕石の[母天体](https://ja.wikipedia.org/wiki/母天体 "wikilink")である[小惑星](../Page/小惑星.md "wikilink")に成長する以前に、宇宙空間で形成されたと考えられており、その熱源がなんであったかについては色々な説がある。[太陽系](../Page/太陽系.md "wikilink")の形成時の情報を多く有していると考えられることから、コンドリュールについて様々な研究が行われている。

## コンドリュールの粒径と含有量によるコンドライトの分類

下表のCIからCHは[炭素質コンドライト](https://ja.wikipedia.org/wiki/炭素質コンドライト "wikilink")（Carbonaceous chondrites）に分類される。H、L、LLは[普通コンドライト](https://ja.wikipedia.org/wiki/普通コンドライト "wikilink")(Ordinary chondrites)と呼ばれる。EH、ELは[エンスタタイト・コンドライト](https://ja.wikipedia.org/wiki/エンスタタイト・コンドライト "wikilink")(Enstatite chondrites)と呼ばれる。

| コンドライトグループ | 含有量 (vol%) | 平均粒径 (mm)                         |
| ---------- | ---------- | --------------------------------- |
| CI         | 0          | –                                 |
| CM         | 20         | 0.3                               |
| CO         | 50         | 0.15                              |
| CV         | 45         | 1                                 |
| CK         | 45         | 1                                 |
| CR         | 50-60      | 0.7                               |
| CH         | 70         | 0.02                              |
| CB         | 20-40      | 10 (a subgroup), 0.2 (b subgroup) |
| H          | 60-80      | 0.3                               |
| L          | 60-80      | 0.7                               |
| LL         | 60-80      | 0.9                               |
| EH         | 60-80      | 0.2                               |
| EL         | 60-80      | 0.6                               |
| R          | \>40       | 0.4                               |
| K          | 30         | 0.6                               |

Table 1: コンドリュールの粒径と含有量による分類\[1\]

大部分のコンドリュールはガラス状か、結晶になった長石質の鉱物に囲まれた珪酸塩の鉱物からなる。
