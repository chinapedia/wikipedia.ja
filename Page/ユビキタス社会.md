とは「いつでも・どこでも・何でも・誰でも」が[コンピュータネットワーク](https://ja.wikipedia.org/wiki/コンピュータネットワーク "wikilink")、[インターネット](https://ja.wikipedia.org/wiki/インターネット "wikilink")を初めとした[ネットワーク](../Page/ネットワーク.md "wikilink")につながることにより、様々なサービスが提供され人々の生活をより豊かにする[社会](../Page/社会.md "wikilink")である。

「いつでも、どこでも」とは[パソコンによってネットワークにつながるだけでなく](../Page/パーソナルコンピュータ.md "wikilink")、[携帯情報端末](../Page/携帯情報端末.md "wikilink")をはじめ屋外や[電車](../Page/電車.md "wikilink")・[自動車](../Page/自動車.md "wikilink")等、あらゆる時間・場所でネットワークにつながる事であり、「何でも、誰でも」とはパソコン同士だけでなく[家電等のあらゆる物を含めて](../Page/家庭用電気機械器具.md "wikilink")、物と物、人と物、人と人がつながることである\[1\]。

## ユビキタス社会と関連

### 日本における国家戦略

日本国内においてユビキタス社会を推進するためのさまざまなプロジェクトが[2000年代](../Page/2000年代.md "wikilink")半ばまでに誕生している。ここでは、その概要を示す。

  - **[e-Japan](https://ja.wikipedia.org/wiki/e-Japan "wikilink")** (2000〜2005)
      - デバイス、[ブロードバンド環境を整備し](https://ja.wikipedia.org/wiki/ブロードバンドインターネット接続 "wikilink")、[IT化を促進した](../Page/情報技術.md "wikilink")。最終答申では、「**いつでも、どこでも**」という言葉が使われている。
  - **[u-Japan](https://ja.wikipedia.org/wiki/u-Japan "wikilink")** (2004〜2008)
      - u-Japanとは、「**ユビキタスネットジャパン**」と呼び、「**[ユビキタスネットワーク](https://ja.wikipedia.org/wiki/ユビキタスネットワーク "wikilink")が実現した社会**」を目指す戦略である。
      - u-Japanの"**u**"は、Ubiquitous / Universal / Unique / User-oriented をイメージしている。
      - [総務省](../Page/総務省.md "wikilink")は研究機関、企業に、さまざまな研究を委託し、ICT（情報通信技術）を促進している。

#### e-Japanの委託研究

  - **やおよろずプロジェクト** (2002 〜2005 )
      - 「横断的科学によるユビキタス情報社会の研究」の通称。
      - 株式会社[日立製作所](../Page/日立製作所.md "wikilink")システム開発研究所、[慶應義塾大学](https://ja.wikipedia.org/wiki/慶應義塾大学 "wikilink")、[東京工科大学](https://ja.wikipedia.org/wiki/東京工科大学 "wikilink")、[東京大学](https://ja.wikipedia.org/wiki/東京大学 "wikilink")、[文部科学省](../Page/文部科学省.md "wikilink")メディア教育開発センター (NIME) 、株式会社ユーディットが参加。

#### u-Japanの委託研究

  - **電子タグの高度利用活用技術に関する研究開発** (2004〜2007)
      - シームレス・タグ情報管理技術 / 相互変換ゲートウェイ技術 / セキュリティ適応制御技術を確立する。
      - [エヌ・ティ・ティ・コミュニケーションズ](https://ja.wikipedia.org/wiki/エヌ・ティ・ティ・コミュニケーションズ "wikilink")株式会社、[日本アイ・ビー・エム](../Page/日本アイ・ビー・エム.md "wikilink")株式会社、株式会社[NTTデータ](https://ja.wikipedia.org/wiki/NTTデータ "wikilink")、[NEC](../Page/日本電気.md "wikilink")（日本電気株式会社）、株式会社[東芝](https://ja.wikipedia.org/wiki/東芝 "wikilink")、[横河電機](https://ja.wikipedia.org/wiki/横河電機 "wikilink")株式会社が参加。
  - **ユビキタス[センサーネットワーク](https://ja.wikipedia.org/wiki/センサーネットワーク "wikilink")技術に関する研究開発** (2005〜2007)
      - ユビキタスセンサーノード技術 / センサーネットワーク制御管理技術 / リアルタイム大容量データ処理・管理技術を確立する。
      - ユビキタスプラットフォームとして、UCRモデル (ubiquitous code relation model) を提唱している。
      - [三菱電機](../Page/三菱電機.md "wikilink")株式会社情報技術総合研究所、[松下電器産業株式会社](https://ja.wikipedia.org/wiki/パナソニック "wikilink")、[パナソニックシステムソリューションズ](https://ja.wikipedia.org/wiki/パナソニックシステムソリューションズ "wikilink")社が参加。
  - **アジア・ユビキタスプラットフォーム技術に関する研究開発** (2005〜2007)
      - 状況情報処理を実現するユビキタスプラットフォームを策定し、[アジア](https://ja.wikipedia.org/wiki/アジア "wikilink")地区での[ユビキタスコンピューティング](../Page/ユビキタスコンピューティング.md "wikilink")の実証実験を行う。
      - ユビキタスプラットフォームとして、UCRモデル (ubiquitous code relation model) を提唱している。
      - [YRPユビキタス](https://ja.wikipedia.org/wiki/横須賀リサーチパーク "wikilink")・ネットワーキング研究所が参加。

### その他のプロジェクト

  - **uPlatea**
      - ユビキタスネットワーク社会を目指した、トータルなネットワークシステムの研究開発。「Ubilaプロジェクト」の実証実験拠点。
      - 株式会社[KDDI研究所](https://ja.wikipedia.org/wiki/KDDI総合研究所 "wikilink")、[九州工業大学](../Page/九州工業大学.md "wikilink")、[KDDI](../Page/KDDI.md "wikilink")株式会社、東京大学、NEC（日本電気株式会社）、[富士通](../Page/富士通.md "wikilink")株式会社、慶應義塾大学が参加、株式会社[内田洋行](https://ja.wikipedia.org/wiki/内田洋行 "wikilink")が技術協力。
  - **[自律移動支援プロジェクト](https://ja.wikipedia.org/wiki/自律移動支援プロジェクト "wikilink")**\[2\]
      - [国土交通省](https://ja.wikipedia.org/wiki/国土交通省 "wikilink")推進の下、産・学・官・民が連携、場所情報を提供し、障害者、高齢者をはじめ、すべての人の移動の支援を行う目的ではじまったプロジェクトである。
