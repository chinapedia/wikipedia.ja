**カルボニルジイミダゾール** (carbonyldiimidazole) は[ペプチド](https://ja.wikipedia.org/wiki/ペプチド "wikilink")合成における[アミノ酸](https://ja.wikipedia.org/wiki/アミノ酸 "wikilink")のカップリング剤など、有機合成に用いられる試薬である。**CDI**と略称される。[イミダゾール](../Page/イミダゾール.md "wikilink")環をImと略記し、Im<sub>2</sub>C=Oと表されることもある。

## 調製

[ホスゲン](../Page/ホスゲン.md "wikilink")と4当量のイミダゾールの反応で得られる。副生物の塩化イミダゾリウムと溶媒を除去すると、およそ 90% の収率で結晶性の生成物として得られる\[1\]。1-（トリメチルシリル）イミダゾールを用いても同様に調製できるが、原料の入手が煩雑になるだけで、特に利点は無い\[2\]。

  -
    4 ImH + Cl<sub>2</sub>C=O → Im<sub>2</sub>C=O + 2 ImH·HCl

容易に加水分解されてイミダゾールと[二酸化炭素](../Page/二酸化炭素.md "wikilink")を生じる。

  -
    Im<sub>2</sub>C=O + H<sub>2</sub>O → 2 ImH + CO<sub>2</sub>

この反応で発生する二酸化炭素は CDI と等[物質量](../Page/物質量.md "wikilink")なので、これを定量することによって CDI の純度を決定できる\[3\]。

## 有機合成での利用

イミダゾールの脱離能の高さを生かし、[アルコール](https://ja.wikipedia.org/wiki/アルコール "wikilink")や[アミン](../Page/アミン.md "wikilink")を[カルバメート](https://ja.wikipedia.org/wiki/カルバメート "wikilink")、[エステル](../Page/エステル.md "wikilink")、[尿素](../Page/尿素.md "wikilink")、[炭酸エステル](https://ja.wikipedia.org/wiki/炭酸エステル "wikilink")に変換するのに用いられる\[4\]。

  -
    Im<sub>2</sub>C=O + HX → ImC(=O)X + ImH (X = OR, NRR')

### カルボン酸誘導体

上記の反応の一般的な利用法は[カルボン酸](https://ja.wikipedia.org/wiki/カルボン酸 "wikilink")誘導体の[アシル基](../Page/アシル基.md "wikilink")移動である。CDI の反応性は[カルボン酸クロリド](https://ja.wikipedia.org/wiki/カルボン酸クロリド "wikilink")と近いが、取り扱いがより簡便で適用範囲も広い\[5\]。初期の適用例として、ペプチドやカルボン酸のイミダゾール誘導体合成がある。二酸化炭素の生成がこの反応の駆動力である。

  -
    Im<sub>2</sub>C=O + RC(=O)OH → RC(=O)Im + ImH + CO<sub>2</sub>
    RC(=O)Im + XH → RC(C=O)X + ImH

ペプチド合成の分野では、生成物 RC(=O)Im をアミノ酸やペプチドエステルで処理してカップリングを行う。副生物の二酸化炭素とイミダゾールは比較的無害である\[6\]。反応条件が穏やかなため、アミノ酸のラセミ化はほとんど起こらない。

エステル化に用いることもできるが、アルコールとの反応は加熱か[ナトリウムエトキシド](https://ja.wikipedia.org/wiki/ナトリウムエトキシド "wikilink")のような求核剤を必要とする\[7\]\[8\]。一般的に収率は高く、適用範囲も広い。[グリコシド](https://ja.wikipedia.org/wiki/グリコシド "wikilink")結合を形成するのにも使える\[9\]。ただし、比較的 α プロトンの酸性度が高いカルボン酸を三級アルコールでエステル化するのは困難で、[クライゼン縮合](../Page/クライゼン縮合.md "wikilink")が起こりやすい（この目的に CDI が利用されることもある）\[10\]。同様な反応は[チオール](../Page/チオール.md "wikilink")や[セレノール](https://ja.wikipedia.org/wiki/セレノール "wikilink")でも起こり、対応するエステルを与える\[11\]。

アルコールの代わりにカルボン酸を用いると[酸無水物](../Page/酸無水物.md "wikilink")が得られる。反応の平衡は、2当量の[トリフルオロ酢酸](../Page/トリフルオロ酢酸.md "wikilink")や[トリクロロ酢酸](../Page/トリクロロ酢酸.md "wikilink")を共存させて副生するイミダゾールを塩として沈殿させることにより、生成物側に大きく偏らせることができる。対称な酸無水物はトリフルオロアセチル基（またはトリクロロアセチル基）を望むカルボン酸で置換することで得られる。

また、[ギ酸](../Page/ギ酸.md "wikilink")と反応させればイミダゾール環の[ホルミル化](https://ja.wikipedia.org/wiki/ホルミル化 "wikilink")が起こる。この生成物はホルミル化試薬として用いられ、加熱によってイミダゾールが再生する。

トリフェニルアルケニルホスホラン（リン[イリド](../Page/イリド.md "wikilink")）とは以下の様な反応を起こす。

  -
    Ph<sub>3</sub>P=CHR + R'C(=O)Im → Ph<sub>3</sub>P<sup>+</sup>−CHR−C(=O)R' + Im<sup>−</sup>
    Ph<sub>3</sub>P<sup>+</sup>−CHR−C(=O)R + Ph<sub>3</sub>P=CHR → Ph<sub>3</sub>P=CR−C(=O)R' + Ph<sub>3</sub>P<sup>+</sup>−CH<sub>2</sub>R

これは[ウィッティヒ反応](../Page/ウィッティヒ反応.md "wikilink")で α,β-不飽和[ケトン](../Page/ケトン.md "wikilink")や[アルデヒド](../Page/アルデヒド.md "wikilink")を合成するのに使うことができる。

[ペルオキシド](../Page/ペルオキシド.md "wikilink")との反応により[過カルボン酸](../Page/過カルボン酸.md "wikilink")を与え、さらに反応を行えばジアシルペルオキシドが得られる。イミダゾール環は[水素化アルミニウムリチウム](../Page/水素化アルミニウムリチウム.md "wikilink")で還元でき、カルボン酸をアルデヒドに変換できる。[グリニャール試薬](https://ja.wikipedia.org/wiki/グリニャール試薬 "wikilink")と反応させればケトンが得られる\[12\]。

[マロン酸](../Page/マロン酸.md "wikilink")エステル類とは以下のような反応を起こす。これは[マクロライド系抗生物質](../Page/マクロライド系抗生物質.md "wikilink")の合成に有用である\[13\]。

  -
    ImC(=O)R + Mg(OC(=O)CHR'C(=O)R'')<sub>2</sub> → RC(=O)CHR'C(=O)R''

### その他の反応

*N*-フェニルイミノ誘導体が[アザ-ウィッティヒ反応によって得られる](../Page/トリフェニルホスフィン.md "wikilink")\[14\]。

  -
    Im<sub>2</sub>C=O + Ph<sub>3</sub>P=NPh → Im<sub>2</sub>C=NPh + Ph<sub>3</sub>P=O

塩基性条件下にカルボニル等価体として働き、ヒドロキシケトンやジケトンから[テトロン酸](https://ja.wikipedia.org/wiki/テトロン酸 "wikilink")骨格を合成できる\[15\]。

CDI の存在下、アルコールを3当量以上の活性ハライドと反応させると、対応するハライドに良好な収率で変換される。ブロミドやヨージドの場合に良い結果が得られ、アルコール上の立体化学を変化させない。同様に脱水に用いることもできる\[16\]。
