**ガウスの微分方程式**（ガウスのびぶんほうていしき）あるいは**超幾何微分方程式**（ちょうきかびぶんほうていしき）とは[ガウスにその名をちなむ](../Page/カール・フリードリヒ・ガウス.md "wikilink")、以下の形をした[常微分方程式](../Page/常微分方程式.md "wikilink")である\[1\]\[2\]\[3\]。

ここで α, β, γ は[複素定数である](../Page/複素数.md "wikilink")。

## 性質

### 特異点と厳密解

この微分方程式は \(\displaystyle  x=0,1,\infty\) において[確定特異点を持ち](https://ja.wikipedia.org/wiki/:en:regular_singular_point "wikilink")、 それ以外に[特異点を持たない](https://ja.wikipedia.org/wiki/特異点_\(数学\) "wikilink")\[4\]\[5\]\[6\]。 また各特異点での解はガウスの[超幾何関数](../Page/超幾何関数.md "wikilink") \(\displaystyle F(\alpha,\beta,\gamma ; x)\) を使って以下の様に表せる事が知られている\[7\]\[8\]\[9\]。

  - *x* = 0 での解
    \(\displaystyle  y_{1,0}(x) = F( \alpha, \beta, \gamma ; x)\)
    \(\displaystyle  y_{2,0}(x) = x^{1-\gamma}F(\alpha-\gamma+1,\beta-\gamma+1,2-\gamma ;x)\)
  - *x* = 1 での解
    \(\displaystyle  y_{1,1}(x) = F(\alpha,\beta,\alpha+\beta-\gamma+1 ; 1-x)\)
    \(\displaystyle  y_{2,1}(x) = (1-x)^{\gamma-\alpha-\beta}F(\gamma-\alpha,\gamma-\beta,\gamma-\alpha-\beta+1 ; 1-x)\)
  - *x* = ∞ での解
    \(\displaystyle  y_{1,\infty}(x) = x^{-\alpha}F(\alpha,\alpha+1-\gamma,\alpha-\beta+1 ; 1/x )\)
    \(\displaystyle  y_{2,\infty}(x) = x^{-\beta}F(\beta,\beta+1-\gamma,\beta-\alpha+1 ; 1/x )\)

### 変数変換でガウスの微分方程式に帰着する方程式

3点を[確定特異点にもつ](https://ja.wikipedia.org/wiki/:en:regular_singular_point "wikilink")[フックス型微分方程式](../Page/フックス型微分方程式.md "wikilink")は変数変換でガウスの微分方程式に帰着する\[10\]。
