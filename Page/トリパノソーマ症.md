**トリパノソーマ症**（トリパノソーマしょう、trypanosomiasis）とは[鞭毛虫類](https://ja.wikipedia.org/wiki/鞭毛虫類 "wikilink")[キネトプラスト門](https://ja.wikipedia.org/wiki/キネトプラスト門 "wikilink")[トリパノソーマ属](https://ja.wikipedia.org/wiki/トリパノソーマ属 "wikilink")[原虫](https://ja.wikipedia.org/wiki/原虫 "wikilink")の感染を原因とする疾病の総称。[クルーズトリパノソーマ](https://ja.wikipedia.org/wiki/クルーズトリパノソーマ "wikilink")（*Trypanosoma cruzi*）感染を原因とする[シャーガス病](../Page/シャーガス病.md "wikilink")、[ブルーストリパノソーマ](https://ja.wikipedia.org/wiki/ブルーストリパノソーマ "wikilink")の1[亜種](https://ja.wikipedia.org/wiki/亜種 "wikilink")である*Trypanosoma brucei gambiense*感染を原因とするガンビアトリパノソーマ症（慢性睡眠病）、*Trypanosoma brucei rhodesiense*感染を原因とするローデシアトリパノソーマ症（急性睡眠病）が代表的な疾患である。いずれも[吸血昆虫](https://ja.wikipedia.org/wiki/吸血昆虫 "wikilink")により媒介される。家畜に認められるトリパノソーマ症のうち，[媾疫トリパノソーマ](https://ja.wikipedia.org/wiki/媾疫トリパノソーマ "wikilink")（*Trypanosoma equiperdum*）による[媾疫](../Page/媾疫.md "wikilink")は[媒介者](https://ja.wikipedia.org/wiki/媒介者 "wikilink")なしで感染する唯一のトリパノソーマ症である。

|      | 原因                       | 感染経路     | 症状          |
| ---- | ------------------------ | -------- | ----------- |
| 媾疫   | *Trypanosoma equiperdum* | 交尾感染     | 生殖器の炎症、貧血   |
| スルラ病 | *Trypanosoma evansi*     | 吸血昆虫媒介   | 熱性浮腫、貧血     |
| ナガナ病 | *Trypanosoma brucei*     | ツェツェバエ媒介 | 腹部・陰囊の浮腫、貧血 |
