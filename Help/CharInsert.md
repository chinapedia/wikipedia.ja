## 入力できるマークアップや記号を追加する方法

入力できるマークアップや記号を追加したい場合、下記のコードを編集して、[利用者のcommon.jsに置いてください](https://ja.wikipedia.org/wiki/special:mypage/common.js "wikilink"):

``` javascript
// CharInsertで入力できるマークアップや記号を追加する
window.charinsertCustom = {
 "挿入": ' 言及: {\{u|+}}  {\{ping|+}}',
};
```

上記の例をそのまま追加した場合、「挿入」のメニューにとのマークアップが追加されます。

  - `"挿入":`はマークアップを追加するメニューの名称であり、先頭に引用符（`"`）を、後尾に引用符とコロン（`":`）を置く形で指定します。既定で存在しない名称を指定した場合はメニューを追加する設定になります。
  - `言及:`はメニュー内の見出しです（コロンで終わる文字列は見出しとみなされます）。見出しにスペースが含まれる場合、代わりにアンダースコア（`_`）を入力してください。
  - 追加するマークアップでスペースが含まれる場合、代わりに[終止符](../Page/終止符.md "wikilink")（`.`）を入力してください。また、ウィキマークアップとして扱われる可能性のあるものはで囲ってください。
  - `+`はマークアップを追加した後のカーソルの位置を表します。たとえば、上記の例において`{{u|}}`をクリックした場合、カーソルがパイプの後ろに移動します。また、編集画面で文字列をハイライトした状態で`{{u|}}`をクリックした場合、ハイライトした文字列が`+`のところに置かれます。
  - [バックスラッシュ](../Page/バックスラッシュ.md "wikilink")を入力したい場合はバックスラッシュ4つ（`\\\\`）を入力してください。CharInsertの表示ではバックスラッシュ2つになり、それをクリックした場合はバックスラッシュ1つが入力されます。例：`<math>\\\\scriptstyle{+}</math>`
  - 改行は`\n`と入力してください。

1つのメニューに複数のマークアップを追加したい場合、スペースで区切ってください。例：

``` javascript
// CharInsertで入力できるマークアップや記号を追加する
window.charinsertCustom = {
 "挿入": '<code>+</code> <code><nowiki>+</nowiki></code>',
};
```

複数のメニューにマークアップを追加したい場合：

``` javascript
// CharInsertで入力できるマークアップや記号を追加する
window.charinsertCustom = {
 "挿入": ' 言及: {\{u|+}}  {\{ping|+}}',
 "マークアップ": ' 言及: {\{u|+}}  {\{ping|+}}'
};
```

新しいメニューを作成する場合は既定で存在しない名称を指定してください：

``` javascript
// CharInsertで入力できるマークアップや記号を追加する
window.charinsertCustom = {
 'マークアップ2': ' – — ° ′ ″ ≈ ≠ ≤ ≥ ± − × ÷ ← → · §  <code>+</code> <code><nowiki>+</nowiki></code>'
};
```

## CharInsertが表示されない場合

CharInsertが表示されない場合、[特別:個人設定\#mw-prefsection-gadgets](https://ja.wikipedia.org/wiki/特別:個人設定#mw-prefsection-gadgets "wikilink")（個人設定の「ガジェット」タブ）でCharInsertが有効となっていることを確認してください。また、CharInsertは既定でJavaScriptを使用するが、カスタムJavaScriptに問題が発生した場合（個人設定の「表示」タブからカスタムJavaScript）はCharInsertが正しく表示されない可能性があります。

## JavaScriptが無効となっている場合

JavaScriptが有効となっている場合、CharInsertはガジェット（[MediaWiki:Gadget-charinsert.jsが](https://ja.wikipedia.org/wiki/MediaWiki:Gadget-charinsert.js "wikilink")[MediaWiki:Gadget-charinsert-core.jsを呼び出す形](../MediaWiki/Gadget-charinsert-core.js.md "wikilink")）として編集画面でロードされます。CharInsertガジェットは既定で有効となっていますが、個人設定の「ガジェット」タブで無効に設定できます。

JavaScriptが無効となっている場合、[MediaWiki](../Page/MediaWiki.md "wikilink")の[CharInsert拡張機能が編集画面でロードされ](https://ja.wikipedia.org/wiki/mw:Extension:CharInsert "wikilink")、[MediaWiki:Edittoolsにて定義された簡易版のCharInsertが表示されます](../MediaWiki/Edittools.md "wikilink")。
