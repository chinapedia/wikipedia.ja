**単調写像**（たんちょうしゃぞう、）または**単調関数**（たんちょうかんすう、）は、単調性、すなわち[順序集合](https://ja.wikipedia.org/wiki/順序集合 "wikilink")の間の写像が順序を保つような性質を持つ[写像](../Page/写像.md "wikilink")のことである。具体的な例としては以下の単調増加関数および単調減少関数がある。

**単調増加**（たんちょうぞうか、）とは、狭義には[実数](../Page/実数.md "wikilink")の値を持つ[関数](../Page/関数_\(数学\).md "wikilink")  が、 の増加につれて常に関数値  も増加することをいい、このような性質を持つ関数を**単調増加関数**（たんちょうぞうかかんすう、）と呼ぶ。同様に、引数  の増加につれて関数値  が常に減少することを**単調減少**（たんちょうげんしょう、）といい、そのような性質を持つ関数を**単調減少関数**（たんちょうげんしょうかんすう、）と呼ぶ。ある関数が単調増加または単調減少する性質をまとめて**単調性**（たんちょうせい、）と呼ぶ。単調性を満たす写像を単調写像と呼ぶ。

連続な単調増加関数  を縦軸、その引数  を横軸にとった[グラフ](https://ja.wikipedia.org/wiki/グラフ "wikilink")上の[曲線](../Page/曲線.md "wikilink")は常に右上りで、右下がりになっている部分がない。逆に単調減少関数の場合には、常に右下がりであり右上がりの部分がない。

## 単調性

### 広義と狭義

実数から実数への関数 \(f\) が

  -
    \(x \le y\) (より簡明に \(x < y\)) ならば \(f(x) \le f(y)\)

をみたすとき、\(f\) は**広義単調増加**（こうぎたんちょうぞうか）するという。広義単調増加のことを'''単調非減少 '''(たんちょうひげんしょう、)と呼ぶこともある。

また、

  -
    \(x < y\) ならば \(f(x) < f(y)\)

をみたすとき、\(f\) は**狭義単調増加** (きょうぎたんちょうぞうか、) するという。

\(f(x)\) と \(f(y)\) の間の不等号の向きを逆にすることで**広義単調減少**および**狭義単調減少**の定義が得られる。広義単調減少のことを'''単調非増加 '''(たんちょうひぞうか、)と呼ぶこともある。

文脈によって明らかなときは広義や狭義を省略することも多い。

### 順序集合

上記の単調性の定義は[定義域](../Page/定義域.md "wikilink")と[値域](../Page/値域.md "wikilink")が実数全体の集合でなくても(半)[順序集合](https://ja.wikipedia.org/wiki/順序集合 "wikilink")一般で意味を持つ。この場合、単調増加する写像は*'[順序を保つ写像](../Page/準同型.md "wikilink") **() であると言い替える事ができ、単調減少する写像は**順序を逆にする写像*' () であると言い替える事ができる。

### 有界

単調性は[有界](../Page/有界.md "wikilink")性と併せて使われることが多い。つまり、つねに[上限](https://ja.wikipedia.org/wiki/上限 "wikilink")を持つ順序集合への単調写像 \(f\) が上に有界であるとき、列 \(x_1 < x_2 < \cdots\) に対して \(\{f(x_i)\}_{i=1,2,\cdots}\) は上限を持つ。このことから上に有界な単調増加[実数列は常に収束し](https://ja.wikipedia.org/wiki/#実数列での単調性 "wikilink")、自然数上の再帰関数は必ず不動点を持つ([領域理論](../Page/領域理論.md "wikilink"))。

## 実関数での単調性

部分集合 \(I \subseteq \mathbb{R}\) で定義された関数 \(f(x)\) を考える。

| \(\forall x_1, \forall x_2 \in I\) に対し～が成り立つとき | \(f(x)\) は区間 *I* で～である |
| ---------------------------------------------- | ---------------------- |
| 語法1                                            | 語法2                    |
| \(x_1 < x_2 \Rightarrow f(x_1) < f(x_2)\,\)    | 単調増加                   |
| \(x_1 < x_2 \Rightarrow f(x_1) \le f(x_2)\,\)  | 広義単調増加                 |
| \(x_1 < x_2 \Rightarrow f(x_1) > f(x_2) \,\)   | 単調減少                   |
| \(x_1 < x_2 \Rightarrow f(x_1) \ge f(x_2)\,\)  | 広義単調減少                 |

等号の成り立つ場合の扱いは書籍によりさまざまで、統一が取れていない。

特に、定義域全体で単調増加/単調減少である関数を、単調増加関数/単調減少関数という。単調増加関数と単調減少関数をまとめて単調関数という。

関数\(f(x)\)が常に可[微分](../Page/微分.md "wikilink")な場合、単調性の概念は\(f(x)\)の[導関数](https://ja.wikipedia.org/wiki/導関数 "wikilink")\(f'(x)\)によって特徴づける事ができる。 \(f(x)\)が広義単調増加になるのは\(f'(x)\)が常に非負な事と同値であり、\(f(x)\)が広義単調減少になるのは\(f'(x)\)が常に非正な事と同値である。 更に\(f'(x)\)の零点が 存在しない場合、狭義の単調性が言える。

## 実数列での単調性

実数に値を取る[数列](../Page/数列.md "wikilink")は、自然数の集合([全順序集合である](https://ja.wikipedia.org/wiki/順序集合#全順序 "wikilink"))から実数の集合への写像であると解釈できる。 その写像が単調なとき、その数列は**単調数列**と呼ばれる。

実数列 \(\left\{ a_k \right\} _{k=1}^n\) を考える。（\(n\)は\(\infty\)でも構わない）

| \(\forall i, \forall j \in \left\{ 1,2, \cdots ,n \right\}\) に対し～が成り立つとき | \(\left\{ a_k \right\} _{k=1}^n\) は～である |
| ------------------------------------------------------------------------ | --------------------------------------- |
| 語法1                                                                      | 語法2                                     |
| \(i < j \Rightarrow a_i < a_j\,\)                                        | 単調増加                                    |
| \(i < j \Rightarrow a_i \le a_j\,\)                                      | 広義単調増加                                  |
| \(i < j \Rightarrow a_i > a_j\,\)                                        | 単調減少                                    |
| \(i < j \Rightarrow a_i \ge a_j\,\)                                      | 広義単調減少                                  |

関数の場合と同様、等号の成り立つ場合の扱いは書籍によりさまざまで、統一が取れていない。

特に、定義域全体で単調増加/単調減少である数列を、**単調増加数列**/**単調減少数列**という。単調増加数列と単調減少数列をまとめて単調数列という。
