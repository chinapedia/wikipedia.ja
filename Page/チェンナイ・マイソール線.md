**チェンナイ・マイソール線**は、[タミル・ナードゥ州](https://ja.wikipedia.org/wiki/タミル・ナードゥ州 "wikilink")[チェンナイ](https://ja.wikipedia.org/wiki/チェンナイ "wikilink")市の[チェンナイ中央駅](https://ja.wikipedia.org/wiki/チェンナイ中央駅 "wikilink")から[カルナータカ州](https://ja.wikipedia.org/wiki/カルナータカ州 "wikilink")の[マイソール駅](https://ja.wikipedia.org/wiki/マイソール駅 "wikilink")までを結ぶ、[インド南部鉄道](../Page/インド南部鉄道.md "wikilink")の[鉄道路線](../Page/鉄道路線.md "wikilink")である。

## 路線データ

  - 路線距離：500km
      - [チェンナイ](https://ja.wikipedia.org/wiki/チェンナイ "wikilink")・[バンガロール](https://ja.wikipedia.org/wiki/バンガロール "wikilink")間361km
      - [アラッコーナム](https://ja.wikipedia.org/wiki/アラッコーナム "wikilink")・[カトゥパティ](https://ja.wikipedia.org/wiki/カトゥパティ "wikilink")間61km
      - [ジョラールペーッタイ](https://ja.wikipedia.org/wiki/ジョラールペーッタイ "wikilink")・[バンガロール](https://ja.wikipedia.org/wiki/バンガロール "wikilink")間68km
  - [軌間](../Page/軌間.md "wikilink")：1676mm ([広軌](../Page/広軌.md "wikilink"))
  - 駅数：56駅（起終点駅含む）

## 列車種別

多くの種類の列車が運行している。種別、運行区間と列車番号は以下の通り。

  - 普通列車、チェンナイ・バンガロール間 : 101, 102
  - 普通列車、チェンナイ・アラッコーナム間 ([ティルパティ線](https://ja.wikipedia.org/wiki/ティルパティ線 "wikilink")直通) : 189, 190
  - 普通列車、アラッコーナム・カトゥパティ間 : 187, 188, 191, 192, 199
  - 普通列車、ジョラールペーッタイ・バンガラペートゥ間 : 533
  - 快速列車、アラッコーナム・マイソール間 (ティルパティ線直通) : 213, 214
  - 区間急行、バンガラペートゥ・バンガロール間 : 6522
  - 区間急行、ジョラールペーッタイ・バンガロール間 : 6519, 6520
  - 急行列車、チェンナイ・ティルパティ間 (ティルパティ線直通) : 6053, 6054, 6203, 6204
  - 急行列車、チェンナイ・バンガロール間 : 6523, 6524
  - 急行列車、チェンナイ・マイソール間 : 6221, 6222
  - 急行列車、エルナークラム・パトナ間 (カトゥパティ・ジョラールペーッタイ間通過、[インド中南部鉄道](https://ja.wikipedia.org/wiki/インド中南部鉄道 "wikilink")などおよび[コーヤンバットゥール・ジョラールペーッタイ線](https://ja.wikipedia.org/wiki/コーヤンバットゥール・ジョラールペーッタイ線 "wikilink")など直通) : 6359 (パトナ行、日曜走行), 6360 (エルナークラム行、金曜走行)
  - 急行列車、エルナークラム・ヴァーラーナシー間 (カトゥパティ・ジョラールペーッタイ間通過、インド中南部鉄道などおよびコーヤンバットゥール・ジョラールペーッタイ線など直通) : 7089 (ヴァーラーナシー行、日曜走行), 7090 (エルナークラム行、金曜走行)
  - 週一急行 : 6021 (バンガロール行、金曜), 6022 (チェンナイ行、日曜)
  - 郵便急行、チェンナイ・バンガロール間 : 2657, 2658
  - 郵便急行、ムンバイ・カンニヤークマリ間 (カトゥパティ・ジョラールペーッタイ間通過、インド中南部鉄道およびコーヤンバットゥール・ジョラールペーッタイ線など直通) : 1081, 1082
  - サプタギリ急行 (チェンナイ・アラッコーナム間通過、[ティルパティ線](https://ja.wikipedia.org/wiki/ティルパティ線 "wikilink")直通) : 6057, 6058
  - サバリ急行 (カトゥパティ・ジョラールペーッタイ間通過、インド中南部鉄道およびコーヤンバットゥール・ジョラールペーッタイ線など直通) : 7229, 7230, 7231, 7232
  - セーシャードリ急行 (カトゥパティ・バンガロール間乗入、インド中南部鉄道直通) : 7009, 7010
  - ブリンダーヴァン急行 : 2639, 2640
  - ラールバーグ急行 : 2608 (バンガロール発チェンナイ行)
  - シャタブディ急行 : 2007 (チェンナイ発マイソール行), 2008 (マイソール発チェンナイ行), 2028 (チェンナイ発バンガロール行)

この他、数々の急行列車が他線から部分的に乗り入れている。

## 駅名及び停車駅

  - エルナークラム・パトナ間急行列車 (カトゥパティ・ジョラールペーッタイ間のみ) は「郵急」に含む。
  - エルナークラム・ヴァーラーナシー間急行列車 (カトゥパティ・ジョラールペーッタイ間のみ) は「郵急」に含む。
  - サプタギリ急行 (チェンナイ・アラッコーナム間のみ) は「急行」に含む。
  - サバリ急行 (カトゥパティ・ジョラールペーッタイ間のみ) は「郵急」に含む。
