**ガボールフィルタ**（[英](../Page/英語.md "wikilink"): **Gabor filter**）とは、[線型](https://ja.wikipedia.org/wiki/線型性 "wikilink")[フィルタの一種で](../Page/フィルタ回路.md "wikilink")、その[インパルス応答](https://ja.wikipedia.org/wiki/インパルス応答 "wikilink")は[調和関数](../Page/調和関数.md "wikilink")と[ガウス関数](../Page/ガウス関数.md "wikilink")をかけたものになっている。[畳み込み](https://ja.wikipedia.org/wiki/畳み込み "wikilink")属性があるため、ガボールフィルタのインパルス応答の[フーリエ変換](../Page/フーリエ変換.md "wikilink")は、調和関数のフーリエ変換とガウス関数のフーリエ変換の[畳み込み](https://ja.wikipedia.org/wiki/畳み込み "wikilink")になる。

## 概説

ガボールフィルタはガボール[ウェーブレット](../Page/ウェーブレット.md "wikilink")と直接関係しており、ガボールフィルタに回転や膨張を加えることで設計される。しかし、拡大は非常に時間がかかるので、ガボール[ウェーブレット](../Page/ウェーブレット.md "wikilink")では適用されない。従って通常、ガボールフィルタで構成される[フィルタバンク](https://ja.wikipedia.org/wiki/フィルタバンク "wikilink")は、様々なスケールと回転のものが用意される。このフィルタを信号に適用することで、いわゆるガボール空間が得られる。ガボール空間は[画像処理](https://ja.wikipedia.org/wiki/画像処理 "wikilink")などでよく利用され、例えば[虹彩認識](https://ja.wikipedia.org/wiki/虹彩認識 "wikilink")や[指紋認証](https://ja.wikipedia.org/wiki/指紋認証 "wikilink")で使われている。画像内の物体の位置の認識にも役立つ。また、2次元ガボールフィルタによって、初期[視覚野](https://ja.wikipedia.org/wiki/視覚野 "wikilink")にある[単純型細胞](https://ja.wikipedia.org/wiki/単純型細胞 "wikilink")の活動をモデル化できることが示されている\[1\]。

2次元のガボールフィルタを画像に適用した結果は[Web-enabled image processing operators](http://matlabserver.cs.rug.nl)というオンラインアプリケーションのサイトで見ることができる。

## MATLABでの実装例

以下は、[MATLAB](https://ja.wikipedia.org/wiki/MATLAB "wikilink")での実装例である。

``` matlab
function gb=gabor_fn(sigma_x,sigma_y,theta,lambda,psi,gamma)

sz_x=fix(6*sigma_x);
if mod(sz_x,2)==0, sz_x=sz_x+1;end


sz_y=fix(6*sigma_y);
if mod(sz_y,2)==0, sz_y=sz_y+1;end

[x y]=meshgrid(-fix(sz_x/2):fix(sz_x/2),fix(-sz_y/2):fix(sz_y/2));

% Rotation
x_theta=x*cos(theta)+y*sin(theta);
y_theta=-x*sin(theta)+y*cos(theta);

gb=exp(-.5*(x_theta.^2/sigma_x^2+gamma^2*y_theta.^2/sigma_y^2)).*cos(2*pi/lambda*x_theta+psi);
```
