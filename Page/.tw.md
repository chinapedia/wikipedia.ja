**.tw**は[台湾](https://ja.wikipedia.org/wiki/台湾 "wikilink")の[国別コードトップレベルドメイン](../Page/国別コードトップレベルドメイン.md "wikilink")(ccTLD)である。このドメイン名は[ISO 3166-1alpha](../Page/ISO_3166-1.md "wikilink")-2の国名コードに基づいてつけられている。

[レジストリ](../Page/レジストリ.md "wikilink")は、[国家通訊伝播委員会](https://ja.wikipedia.org/wiki/国家通訊伝播委員会 "wikilink")(NCC)と[交通部によって指名された台湾の非営利団体である](../Page/交通部_\(中華民国\).md "wikilink")[TWNIC](https://ja.wikipedia.org/wiki/TWNIC "wikilink")が管理している。2001年3月1日以降は、TWNICに直接ドメイン名の登録申請をするのではなく、TWNICと契約した[レジストラ](https://ja.wikipedia.org/wiki/レジストラ "wikilink")を通して登録申請をするようになっている。現在のレジストラは[中華電信](../Page/中華電信.md "wikilink")・・・[台湾大哥大](https://ja.wikipedia.org/wiki/台湾大哥大 "wikilink")・亜太通信・PC Home・Net-Chinese・TISNet・IP Mirror・Webnic・である。

2010年6月25日、[ICANN](../Page/ICANN.md "wikilink")は以下の2つの[国際化ドメイン名](../Page/国際化ドメイン名.md "wikilink")(IDN)を台湾に割り当てた\[1\]。

  - 台灣（[正体字](https://ja.wikipedia.org/wiki/正体字 "wikilink")、Punycode: xn--kpry57d）
  - 台湾（[簡体字](https://ja.wikipedia.org/wiki/簡体字 "wikilink")、Punycode: xn--kprw13d）

2010年8月現在、これらのドメイン名を使用した登録はまだ処理されていない。

ドメインの登録はセカンドレベルか、以下のサードレベル以下で行われる。

  - : 教育と学術機関

  - : 中華民国の政府機関（台湾国内の施設）

  - : [中華民国国軍](../Page/中華民国国軍.md "wikilink")（台湾国内の施設）

  - : それぞれの国の法律により登録された国内外の企業

  - : ネットワークまたは通信事業者

  - : それぞれの国の法律により登録された国内外の非営利団体

  - : 個人（身元が電子メールにより確認できること）

  - : 無制限（ただし、登録者の身元が電子メールにより確認できること）

  - : 無制限（ただし、登録者の身元が電子メールにより確認できること）

  - : 無制限（ただし、登録者の身元が電子メールにより確認できること）

  - : 無制限

セカンドレベル以下には[漢字](../Page/漢字.md "wikilink")も使用できる。国内の登録に選ばれたセカンドレベルドメインの下のドメイン名の多くの登記はnet.tw=網路.tw、.org.tw=組織.tw、.com.tw=商業.twといった漢字と標準の二つのドメイン名を自動的に得る。
