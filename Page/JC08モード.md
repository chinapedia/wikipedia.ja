**JC08モード**（ジェイシー ゼロハチモード）とは、1[リットル](../Page/リットル.md "wikilink")の[燃料](../Page/燃料.md "wikilink")で何[キロメートル](../Page/キロメートル.md "wikilink")走行できるかを、いくつかの[自動車](../Page/自動車.md "wikilink")の走行パターンから測定する[燃費](../Page/燃費.md "wikilink")測定方法の一つである。

[国土交通省](../Page/国土交通省.md "wikilink")令「自動車のエネルギー消費効率の算定等に関する省令」\[1\]ならびに国土交通省告示「自動車のエネルギー消費効率の算定等に関する省令に規定する国土交通大臣が告示で定める方法」\[2\]によって規定され\[3\]、日本国内にて型式認定を受けた、総重量3.5トン以下の乗用・貨物自動車に対して適用されていた\[4\]。

[2018年](../Page/2018年.md "wikilink")（[平成](../Page/平成.md "wikilink")30年）[10月1日](../Page/10月1日.md "wikilink")から、日本の排出ガス/燃費試験の基準は、[国際連合](../Page/国際連合.md "wikilink")が定めた国際基準である[WLTP](https://ja.wikipedia.org/wiki/WLTP "wikilink")に全面移行された\[5\]。

[2020年](../Page/2020年.md "wikilink")（[令和](https://ja.wikipedia.org/wiki/令和 "wikilink")2年）[9月](../Page/9月.md "wikilink")には、2018年10月以降の新型車の燃費値におけるWLTCモードとの併記及び2018年9月以前に型式認定を受けた継続生産車においても、WLTCモードでの測定・表記の義務化により、既にJC08モードで測定済の燃費値の表記が完全に廃止される予定。

## 概要

従来、[日本](https://ja.wikipedia.org/wiki/日本 "wikilink")国内における[普通自動車](../Page/普通自動車.md "wikilink")の燃費は、[運輸省](../Page/運輸省.md "wikilink")が[1991年](../Page/1991年.md "wikilink")（[平成](../Page/平成.md "wikilink")3年）制定した「[10・15モード燃費](../Page/10・15モード燃費.md "wikilink")」により測定されていたが、加速にかける時間が極めて長かったり、測定するスピードが一般的な[公道](../Page/公道.md "wikilink")よりも低いなど、実際の使用条件とかけ離れており、[カタログ](../Page/カタログ.md "wikilink")燃費での数値と実燃費での数値の差が大き過ぎることが指摘されていた。これに対しJC08モードでは、より実際の走行パターンに近い測定法を実施。測定時間も倍近く長くなるほか、平均時速も高められ、最高速度も70[km/hから](https://ja.wikipedia.org/wiki/時速 "wikilink")80km/hに引き上げられる。重量区分に関しても10・15モードよりも細分化されたため、実際の重量により則した計測となる。

[2011年](../Page/2011年.md "wikilink")（[平成](../Page/平成.md "wikilink")23年）[4月1日](../Page/4月1日.md "wikilink")以降に型式認定を受ける自動車については、このJC08モード燃費値の表示が義務付けられている\[6\]。また、それ以前より販売されている自動車についても、[2013年](../Page/2013年.md "wikilink")（平成25年）[2月28日](../Page/2月28日.md "wikilink")までに、JC08モード燃費値を表示することが義務付けられた\[7\]。

10・15モードは[エンジン](../Page/エンジン.md "wikilink")が温まった状態（ホットスタート）による測定のみであったが、JC08では[排気ガス](../Page/排気ガス.md "wikilink")が濃く、[燃料](../Page/燃料.md "wikilink")も多く使用される暖気前のコールドスタート時の測定も、全体の25%程度加えられる分、以前の基準より厳しいものとなり、同時に測定される排ガスの測定でも厳しいものとなっている。このため、従来の車種をJC08モードに対応させるに当たって、相応の改修が必要なものも存在した。

これらの変更により、JC08モードの自動車カタログ上の燃費数値は、これまでの10・15モード燃費よりも一般的に1割程度低くなるとされる\[8\]。一部輸入車などには燃費が変わらないもの\[9\]、僅かながら燃費が向上するもの\[10\]があり、走行パターンの変化が燃費低下に繋がるとは限らない。表示される燃費の低下は測定法の変更によるものであり、自動車そのものの燃費性能が低下するわけではない。

### 批判

10・15モード燃費が「実燃費からかけ離れている」と、自動車利用者の批判を受け、より実際に近づけるために採用となったJC08モードの燃費測定であるが、目的を達したとは言えず\[11\]、「自動車の実燃費は、カタログ燃費の6割程度」と指摘されている\[12\]。

自動車製造メーカー団体の[日本自動車工業会](../Page/日本自動車工業会.md "wikilink")では『燃費表記に関する小冊子』を作成しており\[13\]、小冊子の中で以下のように説明している。

  - 実走行での燃費は、全車平均でJC08モード燃費より2割程度低下する。
  - カタログ燃費の良い車のほうが、電装品の影響が大きく出て、実燃費との差が大きくなる。なお、走行に関係しない電装品の消費エネルギーについては、燃費試験の際には考慮されない。

日本と[アメリカ合衆国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")で販売されている「[トヨタ自動車](https://ja.wikipedia.org/wiki/トヨタ自動車 "wikilink")の[プリウス](https://ja.wikipedia.org/wiki/トヨタ・プリウス "wikilink")」について、[アメリカ合衆国環境保護庁](../Page/アメリカ合衆国環境保護庁.md "wikilink")ではプリウスの燃費が「1[リットル](../Page/リットル.md "wikilink")21.98[km](../Page/キロメートル.md "wikilink")」と実燃費表記をしており、日本の国土交通省によるJC08モード燃費表記は「1リットル40.8km」と、54[パーセント](../Page/パーセント.md "wikilink")もの燃費表示の乖離が起きていることが批判されている\[14\]。

[三菱自動車工業](../Page/三菱自動車工業.md "wikilink")が行った型式承認について、燃費偽装を知っても承認を取り消さない二重の不正を行った[国土交通省](../Page/国土交通省.md "wikilink")の対応に対し、「もはや数値を担保するものがなくなった」と批判するユーザーも多い\[15\]。

自動的にいつどのように変速しても良い自動変速機車に対し、手動変速機車では変速タイミングや選定ギアが指定されているなど、手動変速機車に著しく不利になっており、自動変速機車燃費性能の[優良誤認](https://ja.wikipedia.org/wiki/優良誤認 "wikilink")を招く一因となった。

2018年（平成30年）[10月1日](../Page/10月1日.md "wikilink")から、日本は一度の試験で複数の[国家](../Page/国家.md "wikilink")や地域での型式認証に、必要なデータを取得することができるようになり、設計仕様の統一や部品の共通化を図って開発や認証に掛かる費用を低減するため、国際基準である[WLTP](https://ja.wikipedia.org/wiki/WLTP "wikilink")に移行した\[16\]。

## JC08モードの特徴

10・15モードの比較と併せると、以下のような特徴を持つ\[17\]\[18\]。

| 要素   | JC08モード  | 10・15モード |
| ---- | -------- | -------- |
| 平均速度 | 24.4km/h | 22.7km/h |
| 最高速度 | 81.6km/h | 70km/h   |
| 所要時間 | 1204秒    | 660秒     |
| 走行距離 | 8.172km  | 4.165km  |

試験はシャーシダイナモを用いて行うが、事前に走行抵抗を測定し、試験時に同等の負荷をかけることで実際の走行状態に近づけている。速度パターンの他、[MT車に適用されるギアポジションも定められており](../Page/マニュアルトランスミッション.md "wikilink")、これに従い変速しなければならない（[AT車は適用外で](../Page/オートマチックトランスミッション.md "wikilink")、Dレンジ固定であればどのギアに入っていてもよい）\[19\]。このため、JC08モード燃費に特化した制御の可能なAT車に対して、実際の路上におけるギアの選択とJC08モード規程における指定ギアとの乖離があるMT車にとって不利な計測モードとされるが、他方でディーゼル車においては現在までにJC08モード燃費が公表された全ての車種においてMT車の燃費値がAT車より優れている他、一部のガソリン車においてもMT車のほうが優れた燃費を記録する例があるため、一概にMTだから不利であるとは限らず、車両重量、エンジン特性、タイヤの外径を含むオーバーオールでの[歯車比](https://ja.wikipedia.org/wiki/歯車比 "wikilink")などの要素が総合的に関係していることに留意する必要がある\[20\]。
