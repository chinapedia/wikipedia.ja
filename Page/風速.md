[Wind_speed_climatology.jpg](https://ja.wikipedia.org/wiki/File:Wind_speed_climatology.jpg "fig:Wind_speed_climatology.jpg") **風速**（ふうそく）とは、[風](../Page/風.md "wikilink")として[空気](../Page/空気.md "wikilink")が移動する[速さ](../Page/速さ.md "wikilink")のことである。 [気象庁](../Page/気象庁.md "wikilink")などで通常使われる単位は[m/s](../Page/メートル毎秒.md "wikilink")（いわゆる秒速）、国際的には[ノット](../Page/ノット.md "wikilink") (kt) が用いられる。測定には[風速計](../Page/風速計.md "wikilink")が使用される。

## 概要

日本国内において単に「風速」という場合、地上気象観測では、地上約10[メートル](../Page/メートル.md "wikilink")の高さにおける10[分](../Page/分.md "wikilink")間の**平均風速**を表し、0.25[秒](../Page/秒.md "wikilink")ごとに更新される3秒（12サンプル）平均を**瞬間風速**という\[1\]。また、平均風速の最大値を**最大風速**、瞬間風速の最大値を**最大瞬間風速**という（「瞬間最大風速」は誤用\[2\]）。都市部では高層建築物が増えて、地上付近の正確な風の流れの測定が困難になっているため、地上数十メートル以上の高さに[風速計](../Page/風速計.md "wikilink")が設置されている場合も多い。

風速を計りたくても風速計がない場合は、[気象庁風力階級](https://ja.wikipedia.org/wiki/気象庁風力階級 "wikilink")または[ビューフォート風力階級](../Page/ビューフォート風力階級.md "wikilink")などを基に、地物または海面の状況から判断した風力により風速を推定する。

ある時間内における最大の瞬間風速を平均風速で割った値のことを**突風率**といい、一般には1.5から2.0くらいである。また、風圧は風速の2乗に[比例](https://ja.wikipedia.org/wiki/比例 "wikilink")して大きくなることが知られている。

一般的に、陸上より海上のほうが平均風速が速く、逆に突風率は小さい。風速は、地球的に見た大気の状態から建造物による構造的な大気の乱れまでと幅広いスケールの[気圧](../Page/気圧.md "wikilink")差に影響を受ける。

地球的に見た大気の状態に影響を受け、常に偏西風が吹くヨーロッパ西部は1年を通して一定の風速が保たれているといえる。より小さな例では高気圧や低気圧があり、[台風](../Page/台風.md "wikilink")などの熱帯低気圧が接近したときに観測される風速は比較的速い。より小さなものでは竜巻によるものが顕著な例で、100m/sを超える最大瞬間風速が観測されたこともある。また、[高層建築物](../Page/高層建築物.md "wikilink")が林立する地域では[ビル風](../Page/ビル風.md "wikilink")が吹く。

### スケール

大西洋や北東太平洋の[ハリケーン](../Page/ハリケーン.md "wikilink")の強度区分である[サファ・シンプソン・ハリケーン・スケール](https://ja.wikipedia.org/wiki/サファ・シンプソン・ハリケーン・スケール "wikilink")は、推定される最大風速によりハリケーンをカテゴリー1からカテゴリー5の5種類に区分している。竜巻の強度区分である[藤田スケール](../Page/藤田スケール.md "wikilink")（Fスケール）、[改良藤田スケール](https://ja.wikipedia.org/wiki/改良藤田スケール "wikilink")（EFスケール）、[TORROスケール](https://ja.wikipedia.org/wiki/TORROスケール "wikilink")も、推定される最大風速ごとに区分されている。

### 気象庁

また、気象庁・各気象台の発表や天気予報で用いる風の強さの表現として、以下のようなものがある。

| 表現      | 定義                               |
| ------- | -------------------------------- |
| |静穏     | 風速0.3m/s 未満 の風                   |
| |やや強い風  | 10m/s 以上 15m/s 未満 の風             |
| |強い風    | 15m/s 以上 20m/s 未満 の風             |
| |非常に強い風 | 20m/s 以上30m/s 未満 の風              |
| |猛烈な風   | 風速30m/s 以上 または 最大瞬間風速50m/s 以上 の風 |
