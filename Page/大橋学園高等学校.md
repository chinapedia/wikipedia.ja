**大橋学園高等学校**（おおはしがくえん こうとうがっこう）は、[三重県](https://ja.wikipedia.org/wiki/三重県 "wikilink")[四日市市](../Page/四日市市.md "wikilink")にある[私立の](../Page/私立学校.md "wikilink")[高等学校](https://ja.wikipedia.org/wiki/高等学校 "wikilink")。

## 最寄り駅

  - [近鉄](../Page/近畿日本鉄道.md "wikilink")[名古屋線](https://ja.wikipedia.org/wiki/近鉄名古屋線 "wikilink") [塩浜駅](../Page/塩浜駅.md "wikilink")より徒歩10分
  - [JR東海](../Page/東海旅客鉄道.md "wikilink")[関西本線](../Page/関西本線.md "wikilink") [南四日市駅](../Page/南四日市駅.md "wikilink")より徒歩30分、自転車10分

## 沿革

  - 昭和21年 - 塩浜文化服装学院 設立
  - 昭和27年 - 学校法人大橋学園 認可
  - 平成 7年 - 大橋学園高等学校 開校（四日市市南浜田）
  - 平成28年 - 大橋学園高等学校 移転（四日市市塩浜）
  - 平成29年 - 法人名を「大橋学園」から「みえ大橋学園」に変更
  - 令和 2年 - 明聡の森 完成

## 全日コースの特徴

普通科の学習に加え、様々な教養を学習できる時間割を設けている。

  - 2020年1月

|     | 授業スタイル    | 月                        | 火      | 水            | 木  | 金  |
| --- | --------- | ------------------------ | ------ | ------------ | -- | -- |
| 午前  | 高卒資格 取得学習 | 高卒資格（普通科）に必要な学習を行う       |        |              |    |    |

| 理美容 | 数学        | 調理                       | 福祉     | メイク          |    |    |
| 国語  | 特別校内研修    | 公民                       | 特別校外研修 | 人間形成         |    |    |
| 放課後 | チャレンジ タイム | 資格取得対策講座またはクラブ活動・校外研修・補習 |        |              |    |    |

  - 総合教養タイム
    それぞれの分野を総合的に学習していく。また選択することが可能な分野もある。
  - 人間形成
    人権学習を中心に、社会生活を豊かに営むために必要な知識・考え方・資質を高めてく。
  - 特別校外・校内研修
    その他の分野として、医療・ペット関係・美容などの分野を校外で学ぶ。
