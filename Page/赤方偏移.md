[Redshift.svg](https://ja.wikipedia.org/wiki/File:Redshift.svg "fig:Redshift.svg")の[スペクトル](https://ja.wikipedia.org/wiki/スペクトル "wikilink")（左）と比べ、遠方の[超銀河団](../Page/超銀河団.md "wikilink")のスペクトル（右）では、[フラウンホーファー線](https://ja.wikipedia.org/wiki/フラウンホーファー線 "wikilink")がより長波長側（赤い方）へシフトしている。\]\]  **赤方偏移**（せきほうへんい、）とは、主に[天文学](https://ja.wikipedia.org/wiki/天文学 "wikilink")において、観測対象からの光（[可視光](https://ja.wikipedia.org/wiki/可視光 "wikilink")だけでなく全ての[波長](https://ja.wikipedia.org/wiki/波長 "wikilink")の[電磁波](../Page/電磁波.md "wikilink")を含む）の[スペクトル](https://ja.wikipedia.org/wiki/スペクトル "wikilink")が長波長側（可視光で言うと赤に近い方）にずれる現象を指す。

波長λのスペクトルがΔλだけずれている場合、赤方偏移の量 *z* を

\[z = \frac{\Delta \lambda}{\lambda}\] と定義する。

## 赤方偏移が発生する原因

### 光のドップラー効果

遠ざかる音源からの音が[ドップラー効果](https://ja.wikipedia.org/wiki/ドップラー効果 "wikilink")により低くなるのと同様、遠ざかる光源から発せられた光には赤方偏移がおこる。例えば、地球に対して遠ざかるような運動をしている[恒星](../Page/恒星.md "wikilink")のスペクトルを測定すると、地球から見た視線方向の後退速度に対応する赤方偏移が観測される。

### 宇宙論的赤方偏移

[アメリカ合衆国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")の[天文学者](../Page/天文学者.md "wikilink")[エドウィン・ハッブル](https://ja.wikipedia.org/wiki/エドウィン・ハッブル "wikilink")は様々な[銀河](../Page/銀河.md "wikilink")までの距離とその銀河のスペクトルを調べ、ほとんど全ての銀河のスペクトルに赤方偏移が見られること、赤方偏移の量は遠方の銀河ほど大きいことを経験を生かして発見した（[ハッブルの法則](https://ja.wikipedia.org/wiki/ハッブルの法則 "wikilink")）。この事象は、銀河を出た光が地球に届くまでの間に空間自体が伸びて波長が引き伸ばされるためであると解釈でき、宇宙が膨張していることを示すと考えられている。2016年現在、観測されている最も z が大きい（すなわち最も遠方にあると考えられる）天体は *z* = 11.09 の銀河 GN-z11 である。

もう一つの代表的な例として、[宇宙背景放射](https://ja.wikipedia.org/wiki/宇宙背景放射 "wikilink")での現象が挙げられる。現在の宇宙では、[絶対温度](https://ja.wikipedia.org/wiki/絶対温度 "wikilink")約 3K の[黒体放射](https://ja.wikipedia.org/wiki/黒体放射 "wikilink")に相当する放射があらゆる方向からやってきており、宇宙背景放射と呼ばれている。これは、宇宙創成期に宇宙を満たしていた高温状態の[プラズマ](https://ja.wikipedia.org/wiki/プラズマ "wikilink")から発せられた[熱放射](https://ja.wikipedia.org/wiki/熱放射 "wikilink")が、[ビッグバン](../Page/ビッグバン.md "wikilink")後の急激な[宇宙](../Page/宇宙.md "wikilink")の膨張によって波長が引き伸ばされて極端な赤方偏移を受け、現在観測されるような[電磁波](../Page/電磁波.md "wikilink")(特に、[マイクロ波](https://ja.wikipedia.org/wiki/マイクロ波 "wikilink"))として観測されているものである。これは現在知られている最大の赤方偏移であり、 *z* = 1089 （距離換算で約138.12億光年\[1\]）である。

### 重力赤方偏移

重力赤方偏移（gravitational redshift）とは、重力場中の光の波長が長くなる現象である。

[一般相対性理論](https://ja.wikipedia.org/wiki/一般相対性理論 "wikilink")において、時間の流れがどれくらい変化するかは、計量という式によって表わされる。最も簡単な[シュヴァルツシルトの解](https://ja.wikipedia.org/wiki/シュヴァルツシルトの解 "wikilink")では、質量Mの天体から距離rだけ離れた地点における計量は以下の式になる。

\[\Delta\tau =  \sqrt {1- \frac{2GM}{c^2 r} } \Delta t\]

ここで、\(\Delta\tau\)は、距離rの地点における時間の流れを表わし、\(\Delta t\)はその天体から無限遠方の地点における時間の流れを表わす。この式により、距離rの地点における時間の流れが遅くなることが表わされる。また、\(\frac{2GM}{c^2}\)は[シュヴァルツシルト半径](https://ja.wikipedia.org/wiki/シュヴァルツシルト半径 "wikilink")を表しており、距離rがシュヴァルツシルト半径に等しいときは遠方からは時間が全く経過しないように見えることを示している。

光の波長λ、振動数νと光速度cとの間には、λν = c の関係がある。光の振動数は単位時間当たりの振動の回数であるから、時間の進み方が遅くなると遠方の観測者からは、振動数は小さく、波長は長く観測される。

これを使って、重力赤方偏移の大きさを計算すると以下のようになる。天体の中心から距離rの地点で波長\(\lambda_{em}\)の光が放射され、遠く離れた地点において、その光の波長が\(\lambda_{rec}\)と観測されたとすると、

\[\lambda_{em} = \sqrt{-g_{tt}} \lambda_{rec} = \sqrt{1- \frac{2GM}{c^2 r} } \lambda_{rec}\] この\(\lambda_{em}\)を、赤方偏移zの定義、

\[z = \frac { \lambda_{rec} - \lambda_{em} }{\lambda_{em}}\] に代入すると、zの式は\(\lambda_{rec}\)の式となる。すなわち、zの理論上の値が、観測値\(\lambda_{rec}\)から計算できる。

[1984年](../Page/1984年.md "wikilink")、[宇宙科学研究所](../Page/宇宙科学研究所.md "wikilink")（ISAS）のX線観測衛星 [てんま](https://ja.wikipedia.org/wiki/てんま "wikilink") が、[中性子星](https://ja.wikipedia.org/wiki/中性子星 "wikilink")の強い重力による重力赤方偏移を世界で初めて捉えたと報じた。
