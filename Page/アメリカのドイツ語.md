**アメリカドイツ語**（, ）とは、[アメリカ人が用いる](https://ja.wikipedia.org/wiki/アメリカ合衆国#国民 "wikilink")[アメリカ英語](../Page/アメリカ英語.md "wikilink")の影響を受けた[ドイツ語](../Page/ドイツ語.md "wikilink")。

## 概要

使用するグループは主に[ゲルマン](../Page/ゲルマン人.md "wikilink")[ドイツ系アメリカ人](../Page/ドイツ系アメリカ人.md "wikilink")および[ペンシルベニア・ダッチ](../Page/ペンシルベニア・ダッチ.md "wikilink")（[アーミッシュ](../Page/アーミッシュ.md "wikilink")など）を中心として、[高地ドイツ語に属する](../Page/ドイツ語.md "wikilink")[中部ドイツ語](https://ja.wikipedia.org/wiki/中部ドイツ語 "wikilink")や[上部ドイツ語](https://ja.wikipedia.org/wiki/上部ドイツ語 "wikilink")（[ペンシルベニアドイツ語](../Page/ペンシルベニアドイツ語.md "wikilink")など）および[低地ドイツ語](https://ja.wikipedia.org/wiki/低地ドイツ語 "wikilink")に属する[低ザクセン語](../Page/低ザクセン語.md "wikilink")、[低地フランク語](https://ja.wikipedia.org/wiki/低地フランク語 "wikilink")の系統など様々である（または、[ドイツ化した](https://ja.wikipedia.org/wiki/ゲルマン化 "wikilink")[西スラヴ系の](../Page/西スラヴ人.md "wikilink")[ソルブ人](../Page/ソルブ人.md "wikilink")の[ソルブ語](../Page/ソルブ語.md "wikilink")と[カシューブ人](https://ja.wikipedia.org/wiki/カシューブ人 "wikilink")の[カシューブ語](../Page/カシューブ語.md "wikilink")も含まれる）。

まれに、[ドイツ](https://ja.wikipedia.org/wiki/ドイツ "wikilink")各地に駐留している[アメリカ軍](../Page/アメリカ軍.md "wikilink")兵士や、ドイツ語初心のアメリカ人学習者にもみられる。

## 特徴

  - ドイツ語訛りが少ない
  - 英語の[音素](../Page/音素.md "wikilink")が現れる
  - 抑揚が強い
  - [枠構造](../Page/枠構造.md "wikilink")がなく文末に来る[動詞](../Page/動詞.md "wikilink")も前に来る
  - [口語](../Page/口語.md "wikilink")では[過去形](https://ja.wikipedia.org/wiki/過去形 "wikilink")と同じ意味の[完了形](https://ja.wikipedia.org/wiki/完了形 "wikilink")（haben＋過去分詞）を[現在形](https://ja.wikipedia.org/wiki/現在形 "wikilink")のように用いる
  - [ウムラウト](../Page/ウムラウト.md "wikilink")記号を無視する（パソコンのキーボードに字が無いため）
  - [名詞](../Page/名詞.md "wikilink")の頭の[大文字](../Page/大文字.md "wikilink")書きをしない
  - 英語の[慣用句](../Page/慣用句.md "wikilink")をそのままドイツ語の単語に置き換える

| 言語                          | 例                                               |
| --------------------------- | ----------------------------------------------- |
| **英語**                      | I am going to the store.                        |
|                             |                                                 |
| **進行形を使っている**               | Ich bin gehen zum laden.                        |
|                             |                                                 |
| **ドイツ語**                    | Ich gehe zum Laden.                             |
|                             |                                                 |
|                             |                                                 |
| **英語**                      | I bought the car.                               |
|                             |                                                 |
| **gekauft が habe の直後に来ている** | Ich kaufte das auto. Ich habe gekauft das auto. |
|                             |                                                 |
| **ドイツ語**                    | Ich habe das Auto gekauft.                      |
|                             |                                                 |
| **英語**                      | I have a cat named 'Fuzzy'.                     |
|                             |                                                 |
| **katze の k が小文字**          | Ich habe eine katze namens 'Fuzzy'.             |
|                             |                                                 |
| **ドイツ語**                    | Ich habe eine Katze namens 'Fuzzy'.             |
|                             |                                                 |
|                             |                                                 |
| **英語**                      | I am a citizen of the European Union.           |
|                             |                                                 |
| **2格を使わずに von を使う**         | Ich bin ein bürger von der Europaisch Bund.     |
|                             |                                                 |
| **ドイツ語**                    | Ich bin ein Bürger der Europäischen Union.      |
|                             |                                                 |
