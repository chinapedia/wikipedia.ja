**総肝動脈**（そうかんどうみゃく、）は、[肝臓](../Page/肝臓.md "wikilink")、胃の[幽門](https://ja.wikipedia.org/wiki/幽門 "wikilink")、[十二指腸](../Page/十二指腸.md "wikilink")、[膵臓](../Page/膵臓.md "wikilink")、[胆嚢](../Page/胆嚢.md "wikilink")に酸素を加えた[血液](../Page/血液.md "wikilink")を供給する短い[血管](../Page/血管.md "wikilink")である。

[腹腔動脈](../Page/腹腔動脈.md "wikilink")より生じ、次に示す分枝を持つ。

|                                                             |                                                                                                                                                                                                                                             |
| ----------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **分枝**                                                      | **詳細**                                                                                                                                                                                                                                      |
| [固有肝動脈](https://ja.wikipedia.org/wiki/固有肝動脈 "wikilink")     | [胆嚢動脈](https://ja.wikipedia.org/wiki/胆嚢動脈 "wikilink")を介して[胆嚢](../Page/胆嚢.md "wikilink")に[左肝動脈](https://ja.wikipedia.org/wiki/左肝動脈 "wikilink")と[右肝動脈](https://ja.wikipedia.org/wiki/右肝動脈 "wikilink")を介して[肝臓](../Page/肝臓.md "wikilink")に供給する。 |
|                                                             |                                                                                                                                                                                                                                             |
| [胃十二指腸動脈](https://ja.wikipedia.org/wiki/胃十二指腸動脈 "wikilink") | [右胃大網動脈](https://ja.wikipedia.org/wiki/右胃大網動脈 "wikilink")と[上膵十二指腸動脈](https://ja.wikipedia.org/wiki/上膵十二指腸動脈 "wikilink")に分岐する。                                                                                                               |
|                                                             |                                                                                                                                                                                                                                             |
| | [右胃動脈](https://ja.wikipedia.org/wiki/右胃動脈 "wikilink")     | 分岐し胃の小彎に下方に供給する                                                                                                                                                                                                                             |
