**那覇市消防局**（なはししょうぼうきょく）は、[沖縄県](../Page/沖縄県.md "wikilink")[那覇市](../Page/那覇市.md "wikilink")の消防部局（[消防本部](../Page/消防本部.md "wikilink")）。

## 沿革

  - [1954年](../Page/1954年.md "wikilink")8月10日　那覇市消防団本部を那覇市消防隊本部に改称する。
  - 1954年9月1日　[首里](../Page/首里.md "wikilink")市及び島尻郡[小禄](../Page/小禄.md "wikilink")村を編入合併する。首里市消防隊を首里出張所に、小禄村消防隊を小禄出張所に改称する。
  - 1954年11月25日　泊出張所を開設する。
  - [1957年](../Page/1957年.md "wikilink")8月　泊出張所を廃止する。
  - 1957年12月17日　[真和志](../Page/真和志.md "wikilink")市を編入合併する。真和志市消防隊を真和志出張所に改称する。
  - [1959年](../Page/1959年.md "wikilink")4月30日　消防庁舎を新築のうえ移転し、旧庁舎を松尾出張所に改称する。
  - [1962年](../Page/1962年.md "wikilink")12月15日　**那覇市消防本部**及び**那覇市消防署**を開設し、消防隊長を消防長に改称する。
  - [1969年](https://ja.wikipedia.org/wiki/1969年 "wikilink")2月17日　首里出張所、安謝出張所及び小禄出張所を分遣所に改称する。
  - [1971年](https://ja.wikipedia.org/wiki/1971年 "wikilink")5月10日　安謝分遣所が出張所に昇格する。
  - [1972年](../Page/1972年.md "wikilink")5月15日　真和志出張所が**中央消防署**に昇格し、2署体制となる。那覇市消防署を**西消防署**に改称する。
  - 1972年6月1日　警察の所管であった救急業務が移管され、業務を開始する。
  - [1975年](../Page/1975年.md "wikilink")7月24日　末吉出張所を開設する。
  - [1986年](https://ja.wikipedia.org/wiki/1986年 "wikilink")3月1日　小禄分遣所及び首里分遣所を出張所に改称する。
  - 1986年3月3日　国場出張所を開設する。
  - [1994年](../Page/1994年.md "wikilink")3月30日　[高規格救急車](https://ja.wikipedia.org/wiki/高規格救急車 "wikilink")を配備する。（同年11月16日運用開始）
  - [2003年](../Page/2003年.md "wikilink")11月4日　消防本部が現在地に移転する。末吉出張所を廃止する。
  - [2006年](../Page/2006年.md "wikilink")2月28日　西消防署が新庁舎へ移転し、中央消防署が消防本部庁舎へ移転する。旧中央消防署を真和志出張所に改称する。
  - [2011年](../Page/2011年.md "wikilink")3月19日　[東日本大震災](https://ja.wikipedia.org/wiki/東日本大震災 "wikilink")に[緊急消防援助隊](../Page/緊急消防援助隊.md "wikilink")を派遣した。
  - [2011年](../Page/2011年.md "wikilink")4月1日　 [高度救助隊](https://ja.wikipedia.org/wiki/高度救助隊 "wikilink")「スーパーレスキューナハ」が発隊し、西消防署に配置になる。
  - [2014年](../Page/2014年.md "wikilink")4月1日　市組織再編に伴う条例改正により、消防本部の名称を**那覇市消防本部**から**那覇市消防局**に改称する。\[1\]

## 組織

  - 本部－総務課、警防課、救急課、予防課、指令情報課
  - 消防署

## 消防・救急無線

| チャンネル        | 消防波       | 救急波       |
| ------------ | --------- | --------- |
| (1ch)        | 149.75MHz | 147.48MHz |
| (消防1chの中継)   | 151.27MHz |           |
| (2ch:県内共通波)  | 152.77MHz |           |
| (3ch:全国共通波1) | 150.73MHz |           |
| (4ch:全国共通波2) | 148.15MHz |           |
| (5ch:全国共通波3) | 154.15MHz |           |
| (AVM波)       | 151.69MHz | 153.35MHz |

　中継波は署所向けの為、送信出力は10W。警防波は25W。

  - 現場では460MHz帯アナログ簡易無線が使用されている。複数波実装されているためここでは明記していない。
  - 中央署の指揮隊車は25W送信。それ以外の車両は全て10W送信。
  - 救急隊は携帯無線機にて消防各波に開局する。

## 定時無線通信試験

  - 消防波:9時と17時の2回行われ、車載局が一方的にメリットを本部へ報告するスタイルを取っている。携帯局は行わない。
  - 救急波:9時と17時の消防波試験が始まる直前に行われる。

　なお、署外活動中の隊は通信試験に参加しない。
