[thumb](https://ja.wikipedia.org/wiki/ファイル:MotoLegende2009_93_ariel.jpg "wikilink") **アリエル**（Ariel）は、かつて存在した[イギリス](https://ja.wikipedia.org/wiki/イギリス "wikilink")の[オートバイ](../Page/オートバイ.md "wikilink")の[メーカーおよび](../Page/製造業.md "wikilink")[ブランド](../Page/ブランド.md "wikilink")である。[イングランド](../Page/イングランド.md "wikilink")の[バーミンガム](../Page/バーミンガム.md "wikilink")郊外の[ボーンブルック](https://ja.wikipedia.org/wiki/ボーンブルック "wikilink")に本拠を置き、一時期は4輪車も製造していた。

## 歴史

アリエルの名前は、1847年に[馬車](../Page/馬車.md "wikilink")に取り付けるために考案された、ごく初期のタイプの空気入りタイヤ付き車輪に遡る。これは事業として成功を得ることはできなかったが、1870年、[ジェームズ・スターレー](../Page/ジェームズ・スターレー.md "wikilink")と[ウィリアム・ヒルマンによって](../Page/ヒルマン_\(自動車\).md "wikilink")[ワイヤースポーク](../Page/スポーク.md "wikilink")[ホイールが発明されて軽量な自転車を造ることが可能になり](../Page/車輪.md "wikilink")、彼らはこれに「アリエル」の名前をつけた\[1\]。アリエルとは、[シェイクスピアの](../Page/ウィリアム・シェイクスピア.md "wikilink")[戯曲などに登場する空気の精霊の名前である](../Page/テンペスト_\(シェイクスピア\).md "wikilink")\[2\]。やがてアリエルは彼らが[ペニー・ファージング](https://ja.wikipedia.org/wiki/ペニー・ファージング "wikilink")型自転車と[ミシン](../Page/ミシン.md "wikilink")を製造するために設立した工場の名前となった。

1885年、ジェームズ・スターレーの甥である[ジョン・ケンプ・スターレー](../Page/ジョン・ケンプ・スターレー.md "wikilink")によって[ローバー安全型自転車](../Page/安全型自転車.md "wikilink")（前後ほぼ同じ大きさの車輪を持ち、チェーンによる[後輪駆動](../Page/後輪駆動.md "wikilink")の、今日のスタンダードとなっているデザインの自転車）が発明されると従来のペニー・ファージング型自転車は安全型自転車に取って代わられ、同時にアリエルの名前も忘れられていった。

しかし1896年に彼らの会社はウェストウッド・マニュファクチュアリング社 ( *Westwood Manufacturing* ) と合併し、車体の後部に2.25馬力のドディオン ( *[:en:De Dion-Bouton](https://ja.wikipedia.org/wiki/:en:De_Dion-Bouton "wikilink")* ) 製エンジンを積んだ[三輪車の製造を始めた](../Page/トライク.md "wikilink")。これがアリエルの最初のエンジン付きの乗り物である。この三輪車が多く生産された後、ヒルマンはアリエルを去ってプリメア・モーターサイクル ( *[:en:Premier Motorcycles](https://ja.wikipedia.org/wiki/:en:Premier_Motorcycles "wikilink")* ) を設立した。\[3\]。

[right](https://ja.wikipedia.org/wiki/ファイル:Ariel_3_pk_325_cc_1908.jpg "wikilink") 1902年、アリエルはチャールズ・サングスターがオーナーを務めるコンポーネンツ社 ( *Components Ltd.* ) によって買収され、オートバイと4輪車の生産を始めた。[第一次世界大戦](../Page/第一次世界大戦.md "wikilink")の勃発によって工場は一時閉鎖に追い込まれたが、戦争が終わった1918年、アリエルはチャールズの息子である[ジャック・サングスター](https://ja.wikipedia.org/wiki/ジャック・サングスター "wikilink") ( *[:en:Jack Sangster](https://ja.wikipedia.org/wiki/:en:Jack_Sangster "wikilink")* ) によってオートバイの生産を再開し、4馬力のホワイト＆ポップエンジン ( *[:en:White and Poppe](https://ja.wikipedia.org/wiki/:en:White_and_Poppe "wikilink")* ) によって成功を収める\[4\]。また、ジャックは元JAプレストウィッチ社 ( *[:en:JA Prestwich Industries](https://ja.wikipedia.org/wiki/:en:JA_Prestwich_Industries "wikilink")* ) のデザイナーである[ヴァル・ページ](https://ja.wikipedia.org/wiki/ヴァル・ページ "wikilink") ( *[:en:Val Page](https://ja.wikipedia.org/wiki/:en:Val_Page "wikilink")* ) を説得して会社に迎え入れ、オートバイのラインナップを広げた。1926年にはアリエルは4輪車の製造から手を引き、オートバイの生産に専念するようになる。

[right](https://ja.wikipedia.org/wiki/ファイル:Ariel_1000.jpg "wikilink") しかし、1911年頃から1930年代初頭にかけてコンポーネンツ社は財政的に苦しい状況が続いており、とうとう1932年に破産してしまう。ジャック・サングスターはコンポーネンツ社からアリエルの支配権を格安で買い取り、バーミンガムに新しい工場を建設して社名をアリエル・モータース ( *Ariel Motors* ) と変えた\[5\]。新工場では[エドワード・ターナー](https://ja.wikipedia.org/wiki/エドワード・ターナー "wikilink") ( *[:en:Edward Turner](https://ja.wikipedia.org/wiki/:en:Edward_Turner "wikilink")* ) の設計による**スクエアフォア** ( *[:en:Ariel Square Four](https://ja.wikipedia.org/wiki/:en:Ariel_Square_Four "wikilink")* ) や**レッドハンター** ( *[:en:Ariel Red Hunter](https://ja.wikipedia.org/wiki/:en:Ariel_Red_Hunter "wikilink")* ) といった後にアリエルを代表するモデルとなるオートバイを製造し、[第二次世界大戦](https://ja.wikipedia.org/wiki/第二次世界大戦 "wikilink")中は[軍用オートバイを生産した](../Page/軍用車両.md "wikilink")\[6\]。

ジャック・サングスターは1936年に[トライアンフの](https://ja.wikipedia.org/wiki/トライアンフ_\(オートバイ\) "wikilink")2輪車部門を買収していたが、1944年にアリエルを、1951年にはトライアンフを[バーミンガム・スモール・アームズ](https://ja.wikipedia.org/wiki/バーミンガム・スモール・アームズ "wikilink")（'''BSA '''）に売却し、2社はBSAグループの一員となった\[7\]。サングスターはBSAの経営陣に加わり、やがて周囲の信任を得て1956年にそれまでのバーナード・ドッカー卿に代わってBSAの会長職に就くと、すぐにエドワード・ターナーをグループの自動車部門（BSAモーターサイクル、アリエル、トライアンフ、[デイムラー](../Page/デイムラー.md "wikilink")、タクシー車両の製造で知られるカーボディーズなどが含まれる）のトップに据えた\[8\]。

1950年代後半からのオートバイ業界の不況に対してアリエルはラインナップを絞ることなどで対応したが、1967年にはオートバイ製造を中止した\[9\]。1970年にBSAによってアリエルの名を冠したモデルが生産されたが、短命に終わった。

1991年に設立された[スポーツカー](../Page/スポーツカー.md "wikilink")メーカーの *Solocrest Ltd.* が2001年に *Ariel Ltd.* と社名を変更したことにより[アリエルの名が復活したが](../Page/アリエル_\(自動車\).md "wikilink")、上記のアリエル社とは直接の繋がりはない。

## 2輪車

アリエルの最初のオートバイは1902年にはミネルバと呼ばれる自社製の211ccエンジンを搭載して発売された。以後はホワイト＆ポップエンジンの498ccモデルや249ccの小型モデルなどを生産していた\[10\]。

デザイナーのヴァル・ページが加わった1926年から、アリエルのオートバイは新しい時代を迎えた。3速[ギヤを持つ](../Page/トランスミッション.md "wikilink")[2ストロークエンジンの](../Page/2ストローク機関.md "wikilink")**アリエレッタ** ( *Arielette* ) を市場に送り出したページは、1927年には既存のエンジンを再設計して一群の新しい[単気筒エンジン](../Page/単気筒エンジン.md "wikilink")を生み出した。ページが設計したこれらのエンジンを搭載して1926年から1930年に発売された一連のオートバイは**ブラック・アリエル** ( *Black Ariels* ) と呼ばれた。またこの単気筒エンジンはその後も進化を続け、1950年代半ばのLHコルトを除いて1959年までにアリエルで製造された全ての単気筒エンジンのベースとなっている。

[left](https://ja.wikipedia.org/wiki/ファイル:Ariel_Square_Four_4G_1938.JPG "wikilink") エドワード・ターナーの設計による500ccの**スクエアフォア**は1931年に発売された。スクエアフォアはやがて600ccにスケールアップされ、1937年には**スクエアフォア4G**と呼ばれる995ccの[OHV](../Page/OHV.md "wikilink")版にモデルチェンジしたが、[エンジンレイアウトからくる後部](../Page/U型エンジン.md "wikilink")[シリンダー](../Page/シリンダー.md "wikilink")の冷却問題には終始悩まされ続けた\[11\]。1939年にはアンスティーリンクと呼ばれる[プランジャー](https://ja.wikipedia.org/wiki/プランジャー "wikilink")式の[リアサスペンションがオプション設定され](../Page/サスペンション.md "wikilink")、これは1946年に再生産された時にも採用された。この再生産時には、[フロントサスペンションはそれまでのガーダーフォークから](../Page/サスペンション.md "wikilink")[テレスコピック](https://ja.wikipedia.org/wiki/テレスコピック "wikilink")式に変更されている\[12\]。1949年の**スクエアフォア・マーク1** ( *Ariel Square Four Mark I* ) では[シリンダーブロック](../Page/シリンダーブロック.md "wikilink")と[シリンダーヘッド](https://ja.wikipedia.org/wiki/シリンダーヘッド "wikilink")がそれまでの[鋳鉄](../Page/鋳鉄.md "wikilink")製から[アルミ](../Page/アルミニウム.md "wikilink")[鋳造](../Page/鋳造.md "wikilink")に代わったことで軽量化され、最高速度90[mph](../Page/マイル毎時.md "wikilink")（約145km/h）を実現した。1953年には更に再設計されて**スクエアフォア・マーク2** ( *Ariel Square Four Mark II* ) となり、シリンダーヘッドの改良などによって最高速度100mphとなった。

[right](https://ja.wikipedia.org/wiki/ファイル:Ariel_350_cc_Red_Hunter_1938.jpg "wikilink") **アリエル・レッドハンター**は、ヴァル・ページ設計の[OHV](../Page/OHV.md "wikilink")単気筒500ccエンジンを搭載して1932年にデビューしたスポーツモデルである。当初は[トライアルなどの競技をするライダーにしか売れなかったが](../Page/トライアル_\(オートバイ\).md "wikilink")、後にエドワード・ターナーがデザインを一部変更したところ一般ライダーにも受け入れられ、1959年まで生産が続けられた\[13\]。1955年に[サミー・ミラー](https://ja.wikipedia.org/wiki/サミー・ミラー "wikilink") ( *[:en:Sammy Miller](https://ja.wikipedia.org/wiki/:en:Sammy_Miller "wikilink")* ) のライディングによって活躍したトライアル専用モデルのベースとなったのもレッドハンターである\[14\]。第二次世界大戦中にはレッドハンターの車高を上げて走破性を高めた軍用モデルの**W/NG 350** ( *[:en:Ariel W/NG 350](https://ja.wikipedia.org/wiki/:en:Ariel_W/NG_350 "wikilink")* ) が作られた\[15\]。

1944年にBSAに売却されると、アリエルは500ccの**KH** ( *Ariel KH* ) と、強力なBSA製A10エンジンを積んだ650ccの**ハントマスター** ( *Ariel Huntmaster* ) を送り出した。ハントマスターは高い信頼性と最高速度100mph（約160km/h）を誇り、特に[サイドカー](../Page/サイドカー.md "wikilink")マニアの間で人気のモデルとなった\[16\]。

[left](https://ja.wikipedia.org/wiki/ファイル:Ariel.leader.arp.600pix.jpg "wikilink") 1959年、アリエルは伝統のある[4ストロークエンジンのモデルの製造を中止し](../Page/4ストローク機関.md "wikilink")、ラインナップを2ストローク2気筒250ccの**リーダー** ( *[:en:Ariel Leader](https://ja.wikipedia.org/wiki/:en:Ariel_Leader "wikilink")* ) と**アロー** ( *[:en:Ariel Arrow](https://ja.wikipedia.org/wiki/:en:Ariel_Arrow "wikilink")* ) の2モデルに絞った。この決定は一、イギリスのモーターサイクル業界を襲った不況の中でアリエルが生き残るための決断だった。リーダーはヘッドライトからテールまでの車体全体を一体型ボディで包み込んだデザインの、オートバイと[スクーター](../Page/スクーター.md "wikilink")の長所を併せ持つことを意図したモデルであり、またリーダーをベースとしたスポーツモデルのアローは、ノーマルに手を加えただけのマシンがの[マン島TTレース](../Page/マン島TTレース.md "wikilink")で7位に入るという高性能モデルだった\[17\]。アローは一時期、200ccのモデルも作られた。

しかし1960年代には本格的にヨーロッパの市場に参入してきた日本製オートバイの脅威にアリエルは対抗することができず、1967年にオートバイの製造を中止した\[18\]。

1970年、BSAは2ストローク50ccの3輪[モペッド](../Page/モペッド.md "wikilink")の名称としてアリエルの名前を復活させた\[19\]。**アリエル3**と名付けられたこのモデルは、オートバイと同じように車体を傾けてカーブを曲がるという点で従来の3輪車とは異なっていた。車体の前部と後部が[蝶番](../Page/蝶番.md "wikilink")によって結合されており、三つの車輪を地面に接地させたまま車体前部を傾けることができるような構造だったのである。しかしアリエル3は商業的に成功を得ることはできず、間もなく生産中止となるとアリエルの名前を持つモデルは市場から姿を消すことになった。

### 主なモデル

[right](https://ja.wikipedia.org/wiki/ファイル:NG_350_1940.JPG "wikilink") [right](https://ja.wikipedia.org/wiki/ファイル:Ariel_Huntmaster_1955.jpg "wikilink")

  - **スクエア・フォア** - [スクエア4気筒](../Page/U型エンジン.md "wikilink")995cc。
  - **レッドハンター** - [OHV](../Page/OHV.md "wikilink")[単気筒](../Page/単気筒エンジン.md "wikilink")。250/350/500ccのモデルがあった。
  - **W/NG 350** - レッドハンターをベースにした軍用モデル。
  - **VB** - [SV単気筒](../Page/サイドバルブ.md "wikilink")598cc。
  - **リーダー** - [2ストローク](../Page/2ストローク機関.md "wikilink")250cc。
  - **アロー** - リーダーのスポーツモデル。
  - **フィールドマスター** - 500cc[並列2気筒](https://ja.wikipedia.org/wiki/並列2気筒 "wikilink")。
  - **ピクシー** - [BSA製](https://ja.wikipedia.org/wiki/バーミンガム・スモール・アームズ "wikilink")50ccエンジン搭載。
  - **ハントマスター** - 650cc[並列2気筒](https://ja.wikipedia.org/wiki/並列2気筒 "wikilink")、BSA製A10エンジン搭載。
  - **サイクロン** - [バディ・ホリー](../Page/バディ・ホリー.md "wikilink")が所有していたことで知られるモデル。
  - **HT** - [トライアルモデル](../Page/トライアル_\(オートバイ\).md "wikilink")。350ccと500ccのモデルがあった。
  - **HS** - [スクランブラー](../Page/モトクロス.md "wikilink")。

## 4輪車

アリエルが4輪車を生産していたのは、1900年から1915年にかけてと、1922年から1925年にかけての2度の期間である。

アリエルの最初の4輪車は、1902年に生産された10馬力の2気筒エンジンを積んだモデルだった。1903年には4気筒16馬力のモデルを開発。これらのモデルはいずれも[フライホイール](../Page/フライホイール.md "wikilink")別体式の革製[コーンクラッチを持っていた](https://ja.wikipedia.org/wiki/クラッチ#円錐クラッチ "wikilink")。1904年には早くも6気筒エンジンを[チューブラーフレームの](../Page/フレーム形式_\(自動車\).md "wikilink")[シャーシ](../Page/シャーシ.md "wikilink")に積んだモデルが造られた。

1905年末には、**エアロ・シンプレックス** ( *Aero-Simplex* ) と呼ばれる全く新しいシリーズが発表された。このシリーズは[メルセデスに範を取り](https://ja.wikipedia.org/wiki/メルセデス・ベンツ "wikilink")、15馬力から30馬力の4気筒のモデルと、35馬力と40馬力の6気筒モデルがラインナップされていた。また1907年から1908年にかけて、アリエルは排気量15900ccで50馬力以上という当時としては途方もなく巨大な6気筒エンジンを搭載したモデルを950[ポンドで販売していた](../Page/スターリング・ポンド.md "wikilink")。

1907年、アリエルはイギリス市場への参入を目論んでいたフランスの自動車メーカーであるローレイン・ディートリッヒ社 ( *[:en:Lorraine-Dietrich](https://ja.wikipedia.org/wiki/:en:Lorraine-Dietrich "wikilink")* ) にバーミンガムの工場を売却し、以後は造船会社のキャメル・レアード社 ( *[:en:Cammell Laird](https://ja.wikipedia.org/wiki/:en:Cammell_Laird "wikilink")* ) から分かれたコヴェントリー兵器工場 ( *[:en:Coventry Ordnance Works](https://ja.wikipedia.org/wiki/:en:Coventry_Ordnance_Works "wikilink")* ) で自動車の組み立てを行った。しかし1910年にローレイン・ディートリッヒ社との契約を解消。また、当時製造していた1300ccの軽量モデルは第一次世界大戦の勃発によって生産中止に追い込まれた。

戦争が終わった1918年、アリエルはジャック・サングスターが設計した**アリエル・ナイン** ( *Ariel Nine* ) で小型車市場への参入を試みた。これはサングスターが以前[ローバーで働いていた時に設計に携わった](../Page/ローバー_\(自動車\).md "wikilink")**ローバー・エイト** ( *Rover Eight* ) に酷似した[空冷](https://ja.wikipedia.org/wiki/空冷エンジン "wikilink")2気筒エンジンのモデルだったが、結局この時の挑戦は失敗に終わった。

アリエルが55馬力の[水冷](../Page/水冷エンジン.md "wikilink")[水平対向2気筒](https://ja.wikipedia.org/wiki/水平対向2気筒 "wikilink")996ccエンジンを搭載したモデルによって自動車市場に再度参入したのは1922年だった。このモデルは700台余りが生産され、同年に発表される4気筒1096ccの**アリエル・テン** ( *Ariel Ten* ) に繋がっていった。テンは[ギヤボックスをリアアクスルに直結した構造を持つモデルで](../Page/トランスアクスル.md "wikilink")、180ポンドで販売された。1926年にアリエルはオートバイの生産に集中するために4輪車市場から撤退するが、それまでに約250台のテンが生産された。
