**3 ドアーズ・ダウン**（3 Doors Down）は、[アメリカ合衆国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")の[ロック](../Page/ロック_\(音楽\).md "wikilink")[バンド](../Page/バンド_\(音楽\).md "wikilink")。1996年に[ミシシッピ州](https://ja.wikipedia.org/wiki/ミシシッピ州 "wikilink")のエスカタウパで結成された。

## メンバー

  - ブラッド・アーノルド (*Brad Arnold*、1996年–現在 - ) - ボーカル
  - チェット・ロバーツ (*Chet Roberts*、2012年-現在 - ) - ギター
  - ジャスティン・ビルトネン (*Justin Biltonen*、2014-現在 - ) - ベース
  - クリス・ヘンダーソン (*Chris Henderson*、1998年-現在 - ) - ギター
  - グレッグ・アップチャーチ (*Greg Upchurch*、2005年-現在 - ) - ドラム

　　　元[パドル・オブ・マッド](../Page/パドル・オブ・マッド.md "wikilink")のドラマー。

## 過去のメンバー

  - ダニエル・アデア (*Daniel Adair*、2002年–2005年 - )ドラム

　　　現在は[ニッケルバック](../Page/ニッケルバック.md "wikilink")のメンバーとして活動中。

  - リチャード・ライズ (*Richard Liles*、2000年–2001年 - )ドラム
  - マット・ロバーツ (*Matt Roberts*、1996年–2012年 - )ギター

　　　健康状態が悪くなり、治療に集中するため逸脱を発表。

　　　2016年8月20日 死没

  - トッド・ハレル (*Todd Harrell*、1996年-2014年 - )ベース

　　　2度目の不祥事のためビルトネンと交代。
