**ルゴシ・ベーラ**（**Lugosi Béla** ）、本名**ブラシュコー・ベーラ・フェレンツ・デジェー** ( )、米国では**ベラ・ルゴシ**（**Bela Lugosi** ) として知られる。**ルゴシ**および**ブラシュコー**が姓、[1882年](../Page/1882年.md "wikilink")[10月20日](../Page/10月20日.md "wikilink") - [1956年](../Page/1956年.md "wikilink")[8月16日](../Page/8月16日.md "wikilink")）は、日本では[米国映画](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")『[魔人ドラキュラ](../Page/魔人ドラキュラ.md "wikilink")』の主演俳優として知られている[ハンガリー人](https://ja.wikipedia.org/wiki/ハンガリー人 "wikilink")[俳優](../Page/俳優.md "wikilink")。[オーストリア=ハンガリー二重君主国の](../Page/オーストリア＝ハンガリー帝国.md "wikilink")[ハンガリー王国](../Page/ハンガリー王国.md "wikilink")の（[1920年](../Page/1920年.md "wikilink")の[トリアノン条約](../Page/トリアノン条約.md "wikilink")で[ルーマニア](../Page/ルーマニア.md "wikilink")に割譲された）[テメシュ城県](../Page/ティミシュ県.md "wikilink")[ルゴシュ市でブラシュコー](https://ja.wikipedia.org/wiki/ルゴジ "wikilink")・イシュトヴァーン ( ) とヴォニチチ・パウリナ ( ) の第4子として生まれる。身長185cm\[1\]。[1917](../Page/1917年.md "wikilink")–[1919年](../Page/1919年.md "wikilink")の間はハンガリーの映画界では**オルト・アリスティド** ( ) の芸名を使っていたが、[1920年](../Page/1920年.md "wikilink")に渡米後、[1923年](../Page/1923年.md "wikilink")以降ルゴシ・ベーラ\[2\]の芸名を使うようになった。

[ユニバーサル映画](https://ja.wikipedia.org/wiki/ユニバーサル映画 "wikilink")『[魔人ドラキュラ](../Page/魔人ドラキュラ.md "wikilink")』（[1931年](../Page/1931年.md "wikilink")）における[ドラキュラ](../Page/ドラキュラ.md "wikilink")役が特に名高く、戦前の[ホラー映画](../Page/ホラー映画.md "wikilink")界における、[フランケンシュタイン](../Page/フランケンシュタイン.md "wikilink")・モンスター役で知られる[ボリス・カーロフ](../Page/ボリス・カーロフ.md "wikilink")と並ぶ大スター。また、薬物中毒を公表した初めての俳優ともいわれる。

## 生涯

父親は厳しい商売人で息子にも後を継がせようとしていた。しかしベーラは12歳（または14歳）の時に家出をして、働き始める。当初は県都である[テメシュヴァール市で舞台に出演していたが](../Page/ティミショアラ.md "wikilink")、1910年からは[セゲド国民劇場と契約](https://ja.wikipedia.org/wiki/:hu:Szegedi_Nemzeti_Színház "wikilink")、1911年から[ハンガリー劇場に](https://ja.wikipedia.org/wiki/:hu:Pesti_Magyar_Színház "wikilink")、1912年からは[国民劇場に移籍](https://ja.wikipedia.org/wiki/:hu:Nemzeti_Színház "wikilink")、ハンガリーで最も有名な俳優として人気を博する。『マグダラのマリア』では[イエス・キリスト](https://ja.wikipedia.org/wiki/イエス・キリスト "wikilink")の役を演じる。同時にオルト・アリスティドの芸名で多くの映画にも出演。第一次世界大戦が始まると俳優は兵役免除だったにも関らず愛国心から入隊。陸軍大尉として従軍、ロシア戦線で戦う。戦闘で負傷して帰還。勲章を受ける。1919年の[ハンガリー評議会共和国](https://ja.wikipedia.org/wiki/ハンガリー評議会共和国 "wikilink")（[クン・ベーラ](../Page/クン・ベーラ.md "wikilink")を首班とする[社会主義](../Page/社会主義.md "wikilink")政権）の時には演劇人のために労働組合を組織するが、革命政権は戦勝国の[協商国の軍事介入によって崩壊](../Page/連合国_\(第一次世界大戦\).md "wikilink")、ブラシュコー・ベーラは亡命せざるを得なくなる。当初は[ウィーン](https://ja.wikipedia.org/wiki/ウィーン "wikilink")に向かい、その後[ベルリン](https://ja.wikipedia.org/wiki/ベルリン "wikilink")で俳優を続けるが、1921年に船員として米国に移住、やがて米国籍を獲得。英語は全くできなかったが、演劇に対する情熱は抑えきれず、1922年にニューヨークで在米ハンガリー人らとともにハンガリー語で上演する劇場を開設。[マダーチ・イムレの](https://ja.wikipedia.org/wiki/:hu:Madách_Imre_\(író\) "wikilink")[戯曲](../Page/戯曲.md "wikilink")『[人間の悲劇](https://ja.wikipedia.org/wiki/:hu:Az_ember_tragédiája_\(dráma\) "wikilink")』の[脚色](https://ja.wikipedia.org/wiki/脚色 "wikilink")作品が好評を博する。1926年にハンガリー政府より恩赦が与えられるが彼は帰国しなかった。1933年に “The Red Poppy” という作品で初めて英語で舞台に立つ。この時はまだ英語ができなかったので全ての台詞の発音をハンガリー語で書いて、それを暗記したと言う。独特の顔付きとクールな風貌により米国人の間でもすぐ人気俳優となる。彼の独特のハンガリー語訛りの英語と役者としての才能のために1927年から[ブロードウェイ](https://ja.wikipedia.org/wiki/ブロードウェイ "wikilink")の[フルトン劇場で上演された](https://ja.wikipedia.org/wiki/:en:Fulton_Theatre "wikilink")『[ドラキュラ](../Page/ドラキュラ.md "wikilink")』に主演することになり大成功を収める。舞台の成功を得て『ドラキュラ』の映画化の企画が持ち上がったが、最初はルゴシ・ベーラではなく、別の人気俳優で企画が進んでいった。しかし、[世界恐慌](../Page/世界恐慌.md "wikilink")のために製作費を抑えざるを得なくなり、ルゴシに話が行った。この時ルゴシはすでに舞台でドラキュラを500回以上演じていた\[3\]。

[1931年](../Page/1931年.md "wikilink")、[ユニバーサル映画](https://ja.wikipedia.org/wiki/ユニバーサル映画 "wikilink")が『ドラキュラ』を[トッド・ブラウニング](../Page/トッド・ブラウニング.md "wikilink")[監督より](../Page/映画監督.md "wikilink")[映画](../Page/映画.md "wikilink")化した『[魔人ドラキュラ](../Page/魔人ドラキュラ.md "wikilink")』（[ユニバーサル映画](https://ja.wikipedia.org/wiki/ユニバーサル映画 "wikilink")）に舞台と同様ドラキュラ役で主演した。当初この役にはサイレント映画の伝説的な怪優[ロン・チェイニー](../Page/ロン・チェイニー.md "wikilink")が予定されていたが、チェイニーの急死により舞台版に主演したルゴシに出演依頼が回ってきたものだった。この映画でルゴシは強烈なハンガリー訛りの英語と品格のある風貌、圧倒的な演技力により「黒いスーツに黒マント」という典型的ドラキュラ像と、[トランシルヴァニア](https://ja.wikipedia.org/wiki/トランシルヴァニア "wikilink")出身\[4\]の「ドラキュラ俳優」ルゴシ・ベーラという強いイメージを人々に強烈に植え付け、怪奇スターとして世界に知られることとなった。

続いてユニバーサルが制作した『[フランケンシュタイン](../Page/フランケンシュタイン_\(1931年の映画\).md "wikilink")』（1931年）のモンスター役の依頼を受けたが、台詞のない怪物役を嫌い拒否した。（代わりに起用された[ボリス・カーロフ](../Page/ボリス・カーロフ.md "wikilink")はこの映画の大ヒットによりルゴシを上回る名声を手にすることとなった）。[1932年](../Page/1932年.md "wikilink")、『[恐怖城](https://ja.wikipedia.org/wiki/恐怖城 "wikilink")』に主演。マイナーな映画会社による低予算作であったが、[ゾンビ](../Page/ゾンビ.md "wikilink")映画の始祖としてホラー史上に名を残し、ゾンビ達を操る呪術師を演じたルゴシにとっても『魔人ドラキュラ』に次ぐ代表作となった。[1934年](../Page/1934年.md "wikilink")のユニバーサル作品『[黒猫](../Page/黒猫_\(小説\).md "wikilink")』ではカーロフとの二大怪奇スター初共演が実現した。両主演ではあったが、序列はカーロフが上であった。その後も多くの怪奇映画への出演を続けるが、高齢とハンガリー訛りによる英語下手、生粋の舞台俳優特有のオーバーな演技が災いし、ユニバーサル作品では脇役に回る事が多くなりまたマイナー作への出演も増え、人気は衰えていった。

[1939年](../Page/1939年.md "wikilink")にフランケンシュタインシリーズの第3作となる『[フランケンシュタインの復活](../Page/フランケンシュタインの復活.md "wikilink")』に怪人イゴール役で出演、続く『フランケンシュタインの幽霊』（1942年）にも同じ役で出演した後、『フランケンシュタインと狼男』（1943年）で遂に12年前に拒否したモンスター役を演じた。既に61歳という老齢であり、迫力を出すことはできなかった。[1948年](../Page/1948年.md "wikilink")の『凸凹フランケンシュタインの巻』において映画では二度目で、最後となるドラキュラ役を演じた。

[Bela_Lugosi_star_on_HWF.JPG](https://ja.wikipedia.org/wiki/File:Bela_Lugosi_star_on_HWF.JPG "fig:Bela_Lugosi_star_on_HWF.JPG") 私生活では4度の離婚を経験。戦争の古傷を癒すために始めた薬物の依存症に陥り、転落して悲惨な末路を辿った映画スターの典型として伝説的存在となった。もっともこの点は誇張して伝えられた面もあり、5番目の妻と共に比較的穏やかな晩年を過ごした、ともいわれる。[1956年](../Page/1956年.md "wikilink")に[心臓発作](https://ja.wikipedia.org/wiki/心臓発作 "wikilink")で死去。葬儀の際はドラキュラ伯爵の衣装で埋葬された。

最晩年は「史上最低の[映画監督](../Page/映画監督.md "wikilink")」の異名で知られる[エド・ウッド](../Page/エド・ウッド.md "wikilink")監督の作品に出演した。『[グレンとグレンダ](../Page/グレンとグレンダ.md "wikilink")』（[1953年](https://ja.wikipedia.org/wiki/1953年 "wikilink")）で狂言回しを務めた後、『[怪物の花嫁](https://ja.wikipedia.org/wiki/怪物の花嫁 "wikilink")』（[1955年](../Page/1955年.md "wikilink")）に[マッドサイエンティスト](../Page/マッドサイエンティスト.md "wikilink")役で主演した。続いてウッド監督による新作の撮影に取り掛かるが、僅かなシーンのみを残して死去した。ウッドはこの生前撮影されたフィルムを利用し、『[プラン9・フロム・アウタースペース](../Page/プラン9・フロム・アウタースペース.md "wikilink")』（[1959年](../Page/1959年.md "wikilink")）を制作した。その為、この作品がルゴシにとって遺作となった。 [Bela_Lugosi_as_Dracula,_anonymous_photograph_from_1931,_Universal_Studios.jpg](https://ja.wikipedia.org/wiki/File:Bela_Lugosi_as_Dracula,_anonymous_photograph_from_1931,_Universal_Studios.jpg "fig:Bela_Lugosi_as_Dracula,_anonymous_photograph_from_1931,_Universal_Studios.jpg")』(1931年）にてドラキュラに扮するルゴシ・ベーラ。\]\]

## その他

  - [1994年](../Page/1994年.md "wikilink")の[ティム・バートン](../Page/ティム・バートン.md "wikilink")が監督したエド・ウッドの伝記映画『[エド・ウッド](../Page/エド・ウッド_\(映画\).md "wikilink")』では、ウッドが晩年のルゴシと過ごした数年間が描かれている。[マーティン・ランドー](../Page/マーティン・ランドー.md "wikilink")がルゴシ・ベーラ役を演じ、[アカデミー助演男優賞](../Page/アカデミー助演男優賞.md "wikilink")、[ゴールデングローブ賞 助演男優賞](https://ja.wikipedia.org/wiki/ゴールデングローブ賞_助演男優賞 "wikilink")、[全米批評家協会賞](../Page/全米批評家協会賞.md "wikilink")で助演男優賞を獲得している。
