[Flare_and_after-flare_prominence.jpg](https://ja.wikipedia.org/wiki/File:Flare_and_after-flare_prominence.jpg "fig:Flare_and_after-flare_prominence.jpg") [Solar_Blast.ogv](https://ja.wikipedia.org/wiki/File:Solar_Blast.ogv "fig:Solar_Blast.ogv")による撮影。 \]\] **太陽フレア**（たいようフレア、Solar flare）とは[太陽](../Page/太陽.md "wikilink")における爆発現象。別名・太陽面爆発\[1\]。

太陽系で最大の爆発現象で、小規模なものは1日3回ほど起きている\[2\]。多数の波長域の[電磁波](../Page/電磁波.md "wikilink")の増加によって観測される。特に大きな太陽フレアは白色光でも観測されることがあり、白色光フレアと呼ぶ。太陽の活動が活発なときに[太陽黒点](../Page/太陽黒点.md "wikilink")の付近で発生する事が多く、こうした領域を[太陽活動領域](https://ja.wikipedia.org/wiki/太陽活動領域 "wikilink")と呼ぶ。太陽フレアの初めての観測は、1859年にイギリスの[天文学者](../Page/天文学者.md "wikilink")、[リチャード・キャリントン](../Page/リチャード・キャリントン.md "wikilink")によって行われた（[1859年の太陽嵐](https://ja.wikipedia.org/wiki/1859年の太陽嵐 "wikilink")）。

「フレア」とは[火炎](../Page/火炎.md "wikilink")（燃え上がり）のことであるが、天文学領域では[恒星](../Page/恒星.md "wikilink")に発生する巨大な[爆発](../Page/爆発.md "wikilink")現象を指している。現在では太陽以外の様々な[天体](../Page/天体.md "wikilink")でも観測されている\[3\]。

[アメリカ航空宇宙局](../Page/アメリカ航空宇宙局.md "wikilink")(NASA)によると、2012年7月には巨大な太陽フレアが地球をかすめた\[4\]\[5\] 。次の10年間に同程度のフレアが実際に地球を襲う確率は12%であると推定される\[6\]。

## 概要

フレアの大きさは通常1～10万km程度であり、威力は[水素爆弾](../Page/水素爆弾.md "wikilink")10万から1億個と同等である\[7\]。100万度の[コロナ](https://ja.wikipedia.org/wiki/コロナ "wikilink")プラズマは数千万度にまで加熱され、多量の非熱的[粒子](../Page/粒子.md "wikilink")（10k[eV](../Page/電子ボルト.md "wikilink")-1MeVの[電子](https://ja.wikipedia.org/wiki/電子 "wikilink")や10MeV-1GeVの[陽子](../Page/陽子.md "wikilink")）が加速される。同時に[衝撃波](../Page/衝撃波.md "wikilink")や[プラズマ](https://ja.wikipedia.org/wiki/プラズマ "wikilink")噴出が発生し、時おりそれらは[地球](https://ja.wikipedia.org/wiki/地球 "wikilink")に接近して、突然の[磁気嵐](../Page/磁気嵐.md "wikilink")を起こす\[8\]。

フレアの発生機構については、太陽活動領域中に蓄えられた[磁気](https://ja.wikipedia.org/wiki/磁気 "wikilink")エネルギーが、[磁気再結合](https://ja.wikipedia.org/wiki/磁気再結合 "wikilink")によって[熱エネルギー](https://ja.wikipedia.org/wiki/熱エネルギー "wikilink")や[運動エネルギー](../Page/運動エネルギー.md "wikilink")に変換されるという説が有力である。このフレア発生の際には太陽表面に2種類の特殊な磁場構造が生じていることが[地球シミュレータ](../Page/地球シミュレータ.md "wikilink")による詳細な計算シミュレーションと太陽観測衛星[ひので (人工衛星)による観測データの精密解析で明らかとなった](../Page/ひので_\(人工衛星\).md "wikilink")\[9\]。また、全てのフレアを説明するモデルとして、[柴田一成の](https://ja.wikipedia.org/wiki/柴田一成_\(宇宙物理学者\) "wikilink")「フレアの統一モデル」\[10\]\[11\]がある。

フレアが発生すると、多くの[X線](../Page/X線.md "wikilink")、[ガンマ線](../Page/ガンマ線.md "wikilink")、[高エネルギー荷電粒子](https://ja.wikipedia.org/wiki/高エネルギー荷電粒子 "wikilink")が発生し、太陽表面では速度1000km/s程度で伝播距離50万kmにも及ぶ衝撃波が生じる事もある\[12\]。またフレアに伴い、太陽コロナ中の物質が惑星間空間に放出される（[コロナル・マス・エジェクション (CME)](https://ja.wikipedia.org/wiki/コロナ質量放出 "wikilink")）ことが多い。[高エネルギー荷電粒子](https://ja.wikipedia.org/wiki/高エネルギー荷電粒子 "wikilink")が[地球](https://ja.wikipedia.org/wiki/地球 "wikilink")に到達すると、[デリンジャー現象](../Page/デリンジャー現象.md "wikilink")、[磁気嵐](../Page/磁気嵐.md "wikilink")、[オーロラ](../Page/オーロラ.md "wikilink")発生の要因となる。2003年は、大規模なフレアが頻発し、デリンジャー現象により、地球上の[衛星](https://ja.wikipedia.org/wiki/衛星通信 "wikilink")、[無線通信](../Page/無線通信.md "wikilink")に多くの悪影響を与えた。また[地球磁気圏](https://ja.wikipedia.org/wiki/地球磁気圏 "wikilink")外では、フレア時のX線、ガンマ線による被曝により、人の[致死量](../Page/致死量.md "wikilink")を超えることもある。

フレアの活動は、太陽活動周期や[黒点](https://ja.wikipedia.org/wiki/黒点 "wikilink")の蝶形図（[コロナ](https://ja.wikipedia.org/wiki/コロナ "wikilink")の蝶形図）によって、関係付けを説明されることもしばしばある。

フレア時の高エネルギー荷電粒子の地球への到達、あるいは、フレアの発生そのものを観測・予報することは[宇宙天気予報](../Page/宇宙天気予報.md "wikilink")と呼ばれ、太陽研究者にとって重要課題となっている。

## 等級

### X線等級

X線強度による等級は、現在最も広範に普及している太陽フレアの規模の指標である。太陽全面から放射されるX線強度の最大値によって、低い方からA, B, C, M, Xの5つの等級に分類されており、Xが一番強い。10倍ごと（1桁上がるごと）に1つ上の等級となる。各等級はさらに1-10未満の数字で区分され、これらを組み合わせて「C3.2」というように表される。例えば、X2フレア (2 x 10 W/m)は、X1フレア (10 W/m) の2倍の強度、M5フレア (5 x 10 W/m<sup>2</sup>)の4倍の強度であることを示す。Xクラスの上はないため、Xクラスの数字は10を超えることがある\[20\]\[21\]。

この値は、アメリカの[GOES](https://ja.wikipedia.org/wiki/GOES "wikilink")衛星が常時観測している[大気圏外](https://ja.wikipedia.org/wiki/大気圏外 "wikilink")の[波長](../Page/波長.md "wikilink")100 - 800[ピコメートル](https://ja.wikipedia.org/wiki/ピコメートル "wikilink")のX線の[流束](https://ja.wikipedia.org/wiki/流束 "wikilink")（単位：[ワット](../Page/ワット.md "wikilink")毎[平方メートル](../Page/平方メートル.md "wikilink") = W/m<sup>2</sup>）に基づく\[22\]。

| 等級 | 100 - 800pmでの流束 \[W/m<sup>2</sup>\] 最大値 |
| -- | --------------------------------------- |
| A  | 10<sup>-8</sup> - 10<sup>-7</sup>       |
| B  | 10<sup>-7</sup> - 10<sup>-6</sup>       |
| C  | 10<sup>-6</sup> - 10<sup>-5</sup>       |
| M  | 10<sup>-5</sup> - 10<sup>-4</sup>       |
| X  | \> 10<sup>-4</sup>                      |

X線による太陽フレア等級\[23\]

### Hα等級

Hα等級は、GOES衛星の打ち上げ以前、太陽フレアの観測初期から用いられている太陽フレアの等級である。Hα線（Hアルファ線、[バルマー系列](../Page/バルマー系列.md "wikilink")のうち656[ナノメートル](../Page/ナノメートル.md "wikilink")の電磁波）の観測画像から得られる。Hα線の強度と放射面の広さの2要素からなる。強度は(**f**)aint（淡い）, (**n**)ormal（並）, (**b**)rilliant（鮮やか）の3つの等級で表され、放射面の広さは観測できる半球の太陽表面積6.2 x 10<sup>12</sup> km<sup>2</sup>に占める百万分率によりS, 1, 2, 3, 4の5つの等級で表される。例えば並の強度・広さSクラスであれば"Sn"（normal subflare, 並のサブフレア）と表される\[24\]。

| 等級     | 観測半球全体を100万とした時の割合 |
| ------ | ------------------ |
| S(sub) | \< 100             |
| 1      | 100 - 250          |
| 2      | 250 - 600          |
| 3      | 600 - 1200         |
| 4      | \> 1200            |

Hαによる太陽フレア等級\[25\]

## 被害

太陽嵐が起こると、8分程度で[電磁波](../Page/電磁波.md "wikilink")が到達して電波障害が生じ、数時間で放射線が到達。数日後には[コロナ](https://ja.wikipedia.org/wiki/コロナ "wikilink")からの質量放出が地球に届き、[誘導電流](https://ja.wikipedia.org/wiki/誘導電流 "wikilink")が送電線に混入し、電力系統がおかしくなる。ただ単に停電するのではなく、電機・電子系統に[瞬断](https://ja.wikipedia.org/wiki/瞬断 "wikilink")やEMP（[電磁パルス](../Page/電磁パルス.md "wikilink")）被害が出る。特に宇宙空間にある衛星（通信衛星、[GPS衛星](https://ja.wikipedia.org/wiki/GPS衛星 "wikilink")、気象衛星、偵察衛星など）や、巨大なアンテナとして働く送電線の被害が起こる。

被害の実例としては、カナダのケベック州で大停電を引き起こした[1989年3月の磁気嵐](https://ja.wikipedia.org/wiki/1989年3月の磁気嵐 "wikilink")や、[人工衛星「あすか」の機能停止](../Page/あすか_\(人工衛星\).md "wikilink")（2000年）などがある。

衛星観測が始まって以来のフレア等級で過去最大だったのは、2003年11月4日のフレアである。このときはGOES衛星でX28を記録したことが報じられたが\[31\]、後に電離層への影響から更に大きいX45相当であったとする研究も報告されている\[32\]。

太陽以外の恒星で度々観測される超巨大なフレアを「スーパーフレア」と呼び、太陽でも過去に起き、今後も発生する可能性があると警告する研究者もいる。[屋久杉](../Page/屋久杉.md "wikilink")の[年輪](../Page/年輪.md "wikilink")などに痕跡が残る「[775年の宇宙線飛来](https://ja.wikipedia.org/wiki/775年の宇宙線飛来 "wikilink")」発生源についての仮説の一つでもある\[33\]\[34\]。

2008年に[全米科学アカデミー](https://ja.wikipedia.org/wiki/全米科学アカデミー "wikilink")による「激しい宇宙気象――その社会的・経済的影響の把握」報告書\[35\]が出されている。それによると、強力な太陽フレアが地球の磁場を混乱させ、強力な電流によって高圧変圧器が故障（規模によっては溶融）し、電力網が停止する可能性について検討されている。通信ケーブル、放送、携帯電話は脆弱である。もしそうなれば、米国だけで最初の1年間で1兆〜2兆ドルの被害が出て、完全復旧には4年〜10年かかる。もし1859年規模のフレアが起これば、社会に大混乱が生じ文明は19世紀初頭に戻ると予想された\[36\]\[37\]。大型の変圧器は調達に年単位の時間がかかり、電力網が世界規模で破壊された場合に生産はほとんど出来ないとされる。また超高圧送電線の敷設にも時間がかかる\[38\]。
