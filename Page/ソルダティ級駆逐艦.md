**ソルダティ級駆逐艦** （ソルダティきゅうくちくかん''' *Soldati* Class Destroyer''' ) は、[イタリア王立海軍が建造](../Page/イタリア海軍.md "wikilink")・保有した最後の艦隊型[駆逐艦](../Page/駆逐艦.md "wikilink")。艦載武装・ボイラー罐の仕様の違いにより、2つの系統に分けられる。

## 概要

[1936年](../Page/1936年.md "wikilink")に第一系統グループ12隻の建造が発注され、[1938年](../Page/1938年.md "wikilink")から[1939年](../Page/1939年.md "wikilink")にかけて竣工した。第二系統グループ7隻は[第二次世界大戦](https://ja.wikipedia.org/wiki/第二次世界大戦 "wikilink")勃発後の[1940年](../Page/1940年.md "wikilink")に発注され、5隻が竣工したが、2隻は戦局の情勢変化により竣工前に建造中止になった。大戦中に10隻が戦没喪失し、3隻は[フランス海軍](../Page/フランス海軍.md "wikilink")へ、2隻は[ソビエト連邦海軍へそれぞれ戦時賠償艦として譲渡された](../Page/ロシア海軍.md "wikilink")。

「ソルダティ」は軍人の意味で、艦名には軍のさまざまな兵科の兵員を表す語が使われている。

## 同型艦

### 第一系統グループ

竣工12隻

  - アルピーノ ( Alpino )
  - ベルサリエーレ ( Bersagliere )
  - ジェニエーレ ( Geniere )
  - アルティリエーレ ( Artigliere ) - 1940年10月12日、[パッセロ岬沖海戦](https://ja.wikipedia.org/wiki/パッセロ岬沖海戦 "wikilink")において戦没。
  - カミチア・ネーラ ( Camicia Nera ) - 戦没した姉妹艦の名を引き継ぎ「アルティリエーレ」に改称。戦後ソ連に引渡され「[ローフキイ](https://ja.wikipedia.org/wiki/ローフキイ_\(駆逐艦・3代\) "wikilink")」(  )となる。
  - グラナティエーレ ( Granatiere )
  - アスカリ ( Ascari )
  - カラビニエーレ ( Carabiniere )
  - ランチエーレ ( Lanciere )
  - [アヴィエーレ](https://ja.wikipedia.org/wiki/アヴィエーレ_\(駆逐艦\) "wikilink") ( Aviere )
  - フチリエーレ ( Fuciliere ) - 戦後ソ連に引渡され「[リョーフキイ](https://ja.wikipedia.org/wiki/リョーフキイ_\(駆逐艦・3代\) "wikilink")」(  )となる。
  - コラッツィエーレ ( Corazziere )

### 第二系統グループ

竣工5隻

  - ボンバルディエーレ ( Bombardiere )
  - コルサロ ( Corsaro )
  - レジオナリオ ( Legionario ) - 戦後フランスに引渡され「Duchaffaultt」となる。
  - スクアドリスタ ( Squadrista )
  - ヴェリーテ ( Velite ) - 戦後フランスに引渡され「Duperre」となる。

未竣工2隻

  - カリスタ ( Carrista )
  - ミトラリエーレ ( Mitragliere ) - 戦後フランスに引渡され「Jurien de la Graviere」となる。
