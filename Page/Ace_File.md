**Ace File**（エースファイル）は、[ホリ・エージェンシー](https://ja.wikipedia.org/wiki/ホリ・エージェンシー "wikilink")所属の女性[アイドル](https://ja.wikipedia.org/wiki/アイドル "wikilink")4人組ユニット。2000年結成。2002年解散。

## 略歴

Ace Fileより前の[1999年](../Page/1999年.md "wikilink")に**ACE FILE'99**の名前で、全員ホリエージェンシーに所属する吉川茉絵、大沢安希、園田美幸、大山華林の4人で結成。翌2000年に吉川以外のメンバーを全て入れ替え、**Ace File**となる。Ace Fileになってからはイメージカラーを[白](../Page/白.md "wikilink")とし、毎回白い衣装でステージ、イベントに出演するようになる。因みに白のイメージには『世界を変える、と立ち上がった白い女の子の集まり』という意味があるという（[2000年](../Page/2000年.md "wikilink")7月発行のファンクラブ会報より）。ファンクラブイベントとしては、[那須高原](../Page/那須高原.md "wikilink")や[草津温泉](../Page/草津温泉.md "wikilink")へのツアーも行っていた。

[2001年](../Page/2001年.md "wikilink")9月より、工藤あさぎが大学受験を理由に一時活動を休止するようになり、約半年間の期間限定ユニットとして、吉川茉絵と久志麻理奈の2人で『**Ace File くしよし**』として活動を続行。ただ、この間工藤は全く出てこなかったわけではなく、2001年の[クリスマス](../Page/クリスマス.md "wikilink")イベントなどで奈良沙緒理との2人で『**くどなら**』を組んで時々くしよしのステージに飛び入り出演し、4人が再集結したということもあった。

工藤の復帰後の活動も期待されたが、それから間もなく、[2002年](../Page/2002年.md "wikilink")[4月7日](../Page/4月7日.md "wikilink")を以ってAce Fileは解散。メンバーは相次いで事務所を離れ、それぞれの道を歩んだ。

解散から6年経った[2008年](https://ja.wikipedia.org/wiki/2008年 "wikilink")4月7日に[新宿ロフトプラスワン](../Page/新宿ロフトプラスワン.md "wikilink")で行われた『復活なぐらちゃん祭り\!\!』において、メンバーのうち吉川、久志、工藤の3人が再集結し出演した。

## くしよし

ユニット内ユニット。縞模様や水玉模様などが入った赤い衣装がトレードマークだった。2001年9月から『ウッディーシアター中目黒』で月1回のライヴイベントを行っていた。因みに『くどなら』は同様の青い衣装。

なおここで、本格的に売れる前の[井上マー](../Page/井上マー.md "wikilink")、[デッカチャン](../Page/デッカチャン.md "wikilink")、[あべこうじ](../Page/あべこうじ.md "wikilink")、[カラテカ](../Page/カラテカ_\(お笑いコンビ\).md "wikilink")（カラテカは、当時[矢部太郎](../Page/矢部太郎.md "wikilink")が『[進ぬ\!電波少年](../Page/進ぬ!電波少年.md "wikilink")』の『[電波少年的○○人を笑わしに行こう](https://ja.wikipedia.org/wiki/進ぬ!電波少年#電波少年的○○人を笑わしに行こう "wikilink")』でロケに出ていたため、[入江慎也](../Page/入江慎也.md "wikilink")一人での出演が多かった）がMCを務めていた。
