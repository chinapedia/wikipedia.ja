**方位角**（ほういかく、）とは、[方位](../Page/方位.md "wikilink")（方向の水平成分）の数値表現であり、基準となる方位との間の[角度](../Page/角度.md "wikilink")のことを言う。

[航空](../Page/航空.md "wikilink")分野においては、機首の方向や進行方向を示すのに用いられる。

[水平座標系](https://ja.wikipedia.org/wiki/水平座標系 "wikilink")や[天測航法](https://ja.wikipedia.org/wiki/天測航法 "wikilink")、また衛星通信用地上アンテナ の[光軸](https://ja.wikipedia.org/wiki/光軸 "wikilink")方向を示すときには、水平方向の方位角と、垂直方向の[仰角という](https://ja.wikipedia.org/wiki/仰俯角 "wikilink")2つの角度を用いる。

角度の単位は測量や航法などの分野においては標準的には[度](../Page/度_\(角度\).md "wikilink")・[分](../Page/分_\(角度\).md "wikilink")・[秒が用いられる](../Page/秒_\(角度\).md "wikilink")。

## 基準方位と正の角度

[Cartesian_coordinate_system_handedness.svg](https://ja.wikipedia.org/wiki/File:Cartesian_coordinate_system_handedness.svg "fig:Cartesian_coordinate_system_handedness.svg")北基準（左図：[東](../Page/東.md "wikilink")は正のy方向）と[右手系](https://ja.wikipedia.org/wiki/右手系 "wikilink")東基準（右図：[東](../Page/東.md "wikilink")は正のx方向）。\]\]

基準方位は方位角0°の方位であり、座標軸のx軸方向である。

[右手系](https://ja.wikipedia.org/wiki/右手系 "wikilink")では[東](../Page/東.md "wikilink")を基準方位とし[反時計回り](https://ja.wikipedia.org/wiki/反時計回り "wikilink")を正の角度とする。

[左手系](https://ja.wikipedia.org/wiki/左手系 "wikilink")では[北](../Page/北.md "wikilink")を基準方位とし[時計回り](https://ja.wikipedia.org/wiki/時計回り "wikilink")を正の角度とする。 また[南](https://ja.wikipedia.org/wiki/南 "wikilink")を基準方位とする左手系の定義例も僅かに存在する。

この左手系の使用は[測量](../Page/測量.md "wikilink")や[航法](https://ja.wikipedia.org/wiki/航法 "wikilink")などの分野に限れば標準的である\[1\]。[経緯度](https://ja.wikipedia.org/wiki/経緯度 "wikilink")（[経度](../Page/経度.md "wikilink")・[緯度](../Page/緯度.md "wikilink")）の順序についても似た事情が存在する。

[東](../Page/東.md "wikilink")、[西](https://ja.wikipedia.org/wiki/西 "wikilink")、[南](https://ja.wikipedia.org/wiki/南 "wikilink")、[北](../Page/北.md "wikilink")の方向に相当する方位角の値を表に示す。

| [方位](../Page/方位.md "wikilink")/基準 | 左手系北基準  | 左手系南基準  | 右手系東基準  |
| --------------------------------- | ------- | ------- | ------- |
| 北                                 | 0°/360° | 180°    | 90°     |
| 東                                 | 90°     | 270°    | 0°/360° |
| 南                                 | 180°    | 0°/360° | 270°    |
| 西                                 | 270°    | 90°     | 180°    |

方位角の値と座標軸方向との対応を示す。

|       |       |       |        |        |
| ----- | ----- | ----- | ------ | ------ |
| 方位角   | 0°    | 90°   | 180°   | 270°   |
| 座標軸方向 | \(x\) | \(y\) | \(-x\) | \(-y\) |
