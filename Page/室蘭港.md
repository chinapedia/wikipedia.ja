**室蘭港**（むろらんこう）は、[北海道](https://ja.wikipedia.org/wiki/北海道 "wikilink")[室蘭市](../Page/室蘭市.md "wikilink")にある[港湾](../Page/港湾.md "wikilink")。港湾管理者は室蘭市、[港湾法](../Page/港湾法.md "wikilink")上の「[国際拠点港湾](../Page/国際拠点港湾.md "wikilink")」および[港則法](https://ja.wikipedia.org/wiki/港則法 "wikilink")上の「[特定港](../Page/特定港.md "wikilink")」に指定されている。

## 概要

室蘭港は[内浦湾](../Page/内浦湾.md "wikilink")（噴火湾）東端に位置しており、噴火湾に向かって突き出した[絵鞆半島](../Page/絵鞆半島.md "wikilink")によって[太平洋](../Page/太平洋.md "wikilink")から遮蔽し、天然の[入り江](../Page/入り江.md "wikilink")を形成する良港になっている。港湾区域は1,610 haあり、北海道内最大となる被覆内水域1,401 ha、北日本最大となる本航路の水深（16.5 m）を有している。[1872年](../Page/1872年.md "wikilink")（[明治](../Page/明治.md "wikilink")5年）の開港以来、石炭積出・製鉄・石油精製などを中心とした臨海工業港及び後背地の流通拠点港として[北海道工業地域](https://ja.wikipedia.org/wiki/北海道工業地域 "wikilink")の発展を支えた。近年は環境事業（[静脈物流](../Page/静脈物流.md "wikilink")など）や防災拠点港としての機能強化を推進している\[1\]\[2\]。港湾取扱貨物量は北海道内3位（平成28年）になっている\[3\]。

## 沿革

[Sorachi_Maru_Postcard.JPG](https://ja.wikipedia.org/wiki/File:Sorachi_Maru_Postcard.JPG "fig:Sorachi_Maru_Postcard.JPG")「空知丸」の絵葉書\]\] 「室蘭港の歴史」参照\[9\]

  - [1872年](../Page/1872年.md "wikilink")（[明治](../Page/明治.md "wikilink")5年） : トキカラモイに木製桟橋設置し、室蘭海関所設置（室蘭港開港）。室蘭—森間の官営噴火湾定期航路開設。
  - [1883年](../Page/1883年.md "wikilink")（明治16年） : 室蘭—森定期航路民間委託。
  - [1889年](../Page/1889年.md "wikilink")（明治22年） : 栗林吉次により新潟—室蘭間で貨物船による米運輸（1893年まで）。室蘭港における海運業開始。
  - [1890年](../Page/1890年.md "wikilink")（明治23年） : [海軍省](https://ja.wikipedia.org/wiki/海軍省 "wikilink")による第5の[鎮守府予定地](https://ja.wikipedia.org/wiki/鎮守府_\(日本海軍\) "wikilink")（現在の母恋付近）になる。
  - [1891年](../Page/1891年.md "wikilink")（明治24年） : [大黒島灯台点灯](../Page/大黒島_\(室蘭市\).md "wikilink")。
  - [1892年](../Page/1892年.md "wikilink")（明治25年） : 輪西に初代・[室蘭駅](../Page/室蘭駅.md "wikilink")開設。エトスケレップ桟橋より石炭船積み。
  - [1893年](../Page/1893年.md "wikilink")（明治26年） : [日本郵船](../Page/日本郵船.md "wikilink")の青森—函館間の定期航路が室蘭まで延長。室蘭—森間定期航路廃止。「軍港」指定。
  - [1894年](../Page/1894年.md "wikilink")（明治27年） : 税関所設置。外国貿易の「特別輸出港」指定。
  - [1897年](../Page/1897年.md "wikilink")（明治30年） : 室蘭駅が市街地近くの仏坂下に移転し、海岸町の埋立整備が進む。
  - [1899年](../Page/1899年.md "wikilink")（明治32年） : 「関税法」施行による開港。
  - [1903年](../Page/1903年.md "wikilink")（明治36年） : 軍港指定解除。海岸町の埋立整備が終了し、室蘭駅が再び移転。
  - [1906年](../Page/1906年.md "wikilink")（明治39年） : 日本郵船が3港連絡定期航路廃止し、[青森](../Page/青森港.md "wikilink")—室蘭間の青蘭定期航路に変更。
  - [1909年](../Page/1909年.md "wikilink")（明治42年） : [北海道炭礦汽船](../Page/北海道炭礦汽船.md "wikilink")輪西製鐵所（現在の[日本製鉄室蘭製鉄所](../Page/日本製鉄室蘭製鉄所.md "wikilink")）の50 t溶鉱炉火入れ。母恋の室蘭工場（現在の[日本製鋼所室蘭製作所](../Page/日本製鋼所室蘭製作所.md "wikilink")）第7工場運転開始。
  - [1910年](../Page/1910年.md "wikilink")（明治43年） : 日本製鋼所専用埠頭完成。
  - [1911年](../Page/1911年.md "wikilink")（明治44年） : 室蘭駅構内に石炭積出用海上高架桟橋完成。
  - [1915年](../Page/1915年.md "wikilink")（[大正](../Page/大正.md "wikilink")4年） : 日本郵船が青蘭定期航路から撤退し、北日本汽船に譲渡\[10\]。
  - [1918年](../Page/1918年.md "wikilink")（大正7年） : 港内防波堤築設起工（1927年完成）。
  - [1923年](../Page/1923年.md "wikilink")（大正12年） : [頭山満](../Page/頭山満.md "wikilink")、楢崎、三菱合資会社、開発局、鉄道院などによる西室蘭埋立整備終了。
  - [1943年](../Page/1943年.md "wikilink")（[昭和](../Page/昭和.md "wikilink")18年） : 海運統制に伴い、北日本汽船が[大阪商船](../Page/大阪商船.md "wikilink")に吸収合併\[11\]、青蘭定期航路が大阪商船の運用となる。
  - [1950年](../Page/1950年.md "wikilink")（昭和25年） : 大阪商船が青蘭航路を廃止し、新たに室蘭—東京（京浜）航路及び室蘭—阪神航路開設\[12\]。
  - [1951年](https://ja.wikipedia.org/wiki/1951年 "wikilink")（昭和26年） : 「[重要港湾](../Page/重要港湾.md "wikilink")」、「出入国港」、「検疫港」指定。
  - [1962年](../Page/1962年.md "wikilink")（昭和37年） : 「植物防疫輸入指定港」指定。
  - [1964年](../Page/1964年.md "wikilink")（昭和39年） : 北外防波堤の築設起工（1972年完成）。
  - [1965年](../Page/1965年.md "wikilink")（昭和40年） : 東北・北海道地方初となる「特定重要港湾」（現在の[国際拠点港湾](../Page/国際拠点港湾.md "wikilink")）指定。港内で[タンカー](../Page/タンカー.md "wikilink")「ヘイムバード号」桟橋衝突炎上事故発生\[13\]。5月23日、原油を積載したヘイムバードが操船ミスにより[日本石油精製室蘭製油所](../Page/新日本石油精製.md "wikilink")（当時）の桟橋に衝突炎上、6月18日に鎮火するまで27日間に渡って燃え続け、10名の犠牲者を出した。付近の住民にも一時避難命令が出された。
  - [1967年](../Page/1967年.md "wikilink")（昭和42年） : [東日本フェリー](../Page/東日本フェリー.md "wikilink")室蘭—青森間フェリー航路（[青蘭航路](https://ja.wikipedia.org/wiki/青蘭航路 "wikilink")）開設 （2008年12月廃止）。
  - [1969年](https://ja.wikipedia.org/wiki/1969年 "wikilink")（昭和44年） : 南外防波堤の築設起工（完成1978年）。
  - [1970年](../Page/1970年.md "wikilink")（昭和45年） : 室蘭—[大間間フェリー航路開設](https://ja.wikipedia.org/wiki/大間港 "wikilink")（1991年11月休止）。
  - [1974年](../Page/1974年.md "wikilink")（昭和49年） : 北外防波堤灯台新設に伴い、大黒島灯台廃止。
  - [1978年](https://ja.wikipedia.org/wiki/1978年 "wikilink")（昭和53年） : フェリー埠頭供用開始。
  - [1979年](../Page/1979年.md "wikilink")（昭和54年） : 室蘭—[八戸間フェリー航路開設](../Page/八戸港.md "wikilink")（2006年3月休止）。
  - [1981年](../Page/1981年.md "wikilink")（昭和56年） : 祝津コールセンター開設。
  - [1985年](https://ja.wikipedia.org/wiki/1985年 "wikilink")（昭和60年） : 室蘭駅—[西室蘭駅](https://ja.wikipedia.org/wiki/西室蘭駅 "wikilink")間の[室蘭本線](https://ja.wikipedia.org/wiki/室蘭本線 "wikilink")貨物支線[廃線](../Page/廃線.md "wikilink")。室蘭—[大洗間フェリー航路開設](../Page/大洗港.md "wikilink")（2002年6月休止）。
  - [1986年](https://ja.wikipedia.org/wiki/1986年 "wikilink")（昭和61年） : 東南アジアとコンテナ定期航路開設（1992年2月休止）。
  - [1988年](../Page/1988年.md "wikilink")（昭和63年） : 「動物等指定検疫物輸入港」指定。
  - [1990年](https://ja.wikipedia.org/wiki/1990年 "wikilink")（[平成](../Page/平成.md "wikilink")2年） : 室蘭—[直江津間フェリー航路開設](../Page/直江津港.md "wikilink")（1998年9月に[博多まで延航](../Page/博多港.md "wikilink")、2006年12月休止）。
  - [1991年](../Page/1991年.md "wikilink")（平成3年） : 室蘭—大畑間フェリー航路開設（1998年4月休止）。
  - [1992年](../Page/1992年.md "wikilink")（平成4年） : エンルムマリーナ開設\[14\]。
  - [1994年](../Page/1994年.md "wikilink")（平成6年） : 新フェリーターミナル（[室蘭港フェリーターミナル](https://ja.wikipedia.org/wiki/室蘭港フェリーターミナル "wikilink")）竣工。
  - [1995年](https://ja.wikipedia.org/wiki/1995年 "wikilink")（平成7年） : 「外国産食糧輸入港」指定。
  - [1997年](https://ja.wikipedia.org/wiki/1997年 "wikilink")（平成9年） : 崎守埠頭第6バース（水深14 m）供用開始。[横浜港](../Page/横浜港.md "wikilink")から[ガントリークレーン](../Page/ガントリークレーン.md "wikilink")譲受し、稼働開始（2017年更新）\[15\]。
  - [1998年](https://ja.wikipedia.org/wiki/1998年 "wikilink")（平成10年） : 室蘭港口に[白鳥大橋](../Page/白鳥大橋.md "wikilink")（[白鳥新道](../Page/白鳥新道.md "wikilink")一期区間）開通。
  - [1999年](../Page/1999年.md "wikilink")（平成11年） : 中央埠頭旅客船バース供用開始し、旅客船第一船として「[飛鳥](../Page/飛鳥_\(旅客船\).md "wikilink")」接岸。
  - [2000年](../Page/2000年.md "wikilink")（平成12年） : 大韓民国の[釜山港](https://ja.wikipedia.org/wiki/釜山港 "wikilink")・蔚山港との国際定期コンテナ航路開設（2006年休止、2008年釜山港との航路再開）。
  - [2001年](../Page/2001年.md "wikilink")（平成13年） : 中央埠頭旅客船バース完成。
  - [2002年](../Page/2002年.md "wikilink")（平成14年） : [静脈物流](../Page/静脈物流.md "wikilink")のリサイクルポート指定港湾「総合静脈物流拠点港（リサイクルポート）」に[苫小牧港](../Page/苫小牧港.md "wikilink")とともに一次指定\[16\]。絵鞆漁船だまり水中荷さばき地供用開始。
  - [2003年](../Page/2003年.md "wikilink")（平成15年） : 浮体式防災施設（広域防災フロート）供用開始（2010年に国へ売却）。
  - [2008年](https://ja.wikipedia.org/wiki/2008年 "wikilink")（平成20年） : エンルムマリーナが「[海の駅](../Page/海の駅.md "wikilink")」登録\[17\]\[18\]。入江地区耐震強化岸壁供用開始\[19\]。東日本フェリーのフェリー事業撤退に伴う室蘭—青森間の航路廃止により、室蘭港におけるすべてのフェリー航路が一旦なくなる。
  - [2010年](https://ja.wikipedia.org/wiki/2010年 "wikilink")（平成22年） : [国土交通省](../Page/国土交通省.md "wikilink")のパイロットモデル事業として日本国内初となる自動車専用運搬船の解体実証試験（[船舶解体](../Page/船舶解体.md "wikilink")）実施\[20\]。
  - [2011年](../Page/2011年.md "wikilink")（平成23年） : [東日本大震災](https://ja.wikipedia.org/wiki/東日本大震災 "wikilink")による被災地支援のため、浮体式防災施設（広域防災フロート）を[岩手県](../Page/岩手県.md "wikilink")[大船渡港](../Page/大船渡港.md "wikilink")及び[福島県](../Page/福島県.md "wikilink")[相馬港](../Page/相馬港.md "wikilink")へ派遣\[21\]\[22\]\[23\]。
  - [2012年](../Page/2012年.md "wikilink")（平成24年）：祝津絵鞆地区と入江地区が「[みなとオアシス](https://ja.wikipedia.org/wiki/みなとオアシス "wikilink")」登録\[24\]。
  - [2014年](../Page/2014年.md "wikilink")（平成26年）：JX日鉱日石エネルギー（現在の[JXTGエネルギー](https://ja.wikipedia.org/wiki/JXTGエネルギー "wikilink")）が室蘭製油所の原油処理を停止し、室蘭製造所として石化工場に転換（2019年に製造停止し、[JXTGエネルギー室蘭事業所](https://ja.wikipedia.org/wiki/JXTGエネルギー室蘭事業所 "wikilink")となる）\[25\]\[26\]\[27\]\[28\]。
  - [2015年](../Page/2015年.md "wikilink")（平成27年）：苫小牧沖で火災を起こした[商船三井フェリー](../Page/商船三井フェリー.md "wikilink")の「[さんふらわあ だいせつ](https://ja.wikipedia.org/wiki/さんふらわあ_だいせつ "wikilink")」が火災鎮火後の現場検証のため曳航\[29\]。
  - [2018年](../Page/2018年.md "wikilink")（平成30年）：フェリーターミナルビル・フェリー埠頭の改修工事完了し\[30\]、[川崎近海汽船](../Page/川崎近海汽船.md "wikilink")による[宮古](https://ja.wikipedia.org/wiki/宮古港 "wikilink")—室蘭間フェリー航路就航\[31\]\[32\]。
