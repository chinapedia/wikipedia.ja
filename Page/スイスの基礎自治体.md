**スイスの基礎自治体**（スイスのきそじちたい）では、[スイス](../Page/スイス.md "wikilink")の[基礎自治体](../Page/基礎自治体.md "wikilink")について説明する。

スイスでは基礎自治体は[ドイツ語](../Page/ドイツ語.md "wikilink")でゲマインデ（Gemeinde）、[フランス語](https://ja.wikipedia.org/wiki/フランス語 "wikilink")で[コミューン](../Page/コミューン.md "wikilink")（commune・コミューヌ）、[イタリア語](../Page/イタリア語.md "wikilink")で[コムーネ](../Page/コムーネ.md "wikilink")（comune）、[ロマンシュ語](../Page/ロマンシュ語.md "wikilink")では vischnanca と呼ばれる。[2017年](../Page/2017年.md "wikilink")現在、スイスには2255の基礎自治体がある\[1\]。そのほとんどは数百人程度の[人口](../Page/人口.md "wikilink")であるが、[チューリッヒ](../Page/チューリッヒ.md "wikilink")（Zürich）や[ジュネーヴ](../Page/ジュネーヴ.md "wikilink")（Genève）のような大都市も法的には基礎自治体である。基礎自治体の面積は、0.32平方キロメートル（[アールガウ州](../Page/アールガウ州.md "wikilink")[カイザーシュトゥールおよび](https://ja.wikipedia.org/wiki/:en:Kaiserstuhl,_Aargau "wikilink")[ヴォー州](../Page/ヴォー州.md "wikilink")[リヴァ](https://ja.wikipedia.org/wiki/:en:Rivaz "wikilink")）から439平方キロメートル（[グラウビュンデン州](../Page/グラウビュンデン州.md "wikilink")[シュクオル](https://ja.wikipedia.org/wiki/シュクオル "wikilink")）まで大きな幅がある。

各州（[カントン](../Page/スイスの地方行政区画.md "wikilink")）により、[教育](../Page/教育.md "wikilink")、[医療](../Page/医療.md "wikilink")、[社会福祉](../Page/福祉.md "wikilink")、[公共交通](https://ja.wikipedia.org/wiki/公共交通 "wikilink")、[徴税等の基礎自治体の権限は異なっている](../Page/租税.md "wikilink")。[中央集権](../Page/中央集権.md "wikilink")化の度合いもカントンにより異なっている。

各基礎自治体は、[行政](../Page/行政.md "wikilink")組織としての[市町村長](../Page/市町村長.md "wikilink")をトップとする委員会と[立法府](../Page/立法府.md "wikilink")としての[タウンミーティング](../Page/タウンミーティング.md "wikilink")によって統治されている。ほとんどのカントンでは、大きな基礎自治体には[議会](../Page/議会.md "wikilink")を設置する権限を与えている。カントンによっては、スイスに一定期間以上居住した[外国人](../Page/外国人.md "wikilink")にも[地方自治](../Page/地方自治.md "wikilink")に参加する権限を与えている。

スイスの[国籍](../Page/国籍.md "wikilink")は、各基礎自治体の市民権に基づいている。全てのスイス人はどれか1つまたは複数の基礎自治体の市民である。

基礎自治体の[財政](../Page/財政.md "wikilink")は、[所得税](../Page/所得税.md "wikilink")のような直接税によって賄われ、その税率は場所によって異なるもののカントンによって設定された範囲になっている。

多くの基礎自治体は、行政サービスを維持していくことが困難になってきており、歳出を減らす努力として多くの自治体が[合併](https://ja.wikipedia.org/wiki/合併 "wikilink")や特別行政区の設置によって行政サービスを統合している。各カントン政府はこの再編を支援しており、統合を選ぶ自治体は増えつつある。

**市**（villesまたはStädte）と呼ばれるのは、1万人以上の人口があるか、[中世](../Page/中世.md "wikilink")からの独自の都市権を持っている基礎自治体である。それより小さな基礎自治体には特別な区別はない。

|               |                 |
| ------------- | --------------- |
| 人口            | 基礎自治体数 (%)\[2\] |
| 20,000以上      | 30 (1.1%)       |
| 10,000–19,999 | 89 (3.2%)       |
| 5000–9999     | 180 (6.6%)      |
| 1000–4999     | 1025 (37.4%)    |
| 500–999       | 555 (20.3%)     |
| 500未満         | 861 (31.4%)     |
| 合計            | 2740 (100%)     |

基礎自治体には、スイス連邦統計事務所によって統計コードが割り振られている。また[郵便番号](../Page/郵便番号.md "wikilink")が各自治体に割り当てられているが、1つの基礎自治体に複数の番号がある場合や、いくつかの基礎自治体をまとめて1つの番号に割り当てている場合もある。
