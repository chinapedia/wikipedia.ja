**関係調**（かんけいちょう、）とは、[音楽理論](https://ja.wikipedia.org/wiki/音楽理論 "wikilink")用語のひとつで、ある調から「特に近い」と感じられる[調性感](https://ja.wikipedia.org/wiki/調性感 "wikilink")をもつ、幾つかの[調](../Page/調.md "wikilink")のこと\[1\]\[2\]。**近親調**とも呼ばれる\[3\]。

なお、関係調以外の調を「**遠隔調**」という\[4\]。 [古典派の後では](../Page/古典派音楽.md "wikilink")、遠隔調への転調も頻繁に行われるようになった。

## 概説

関係調（近親調）とは、ある調を基準にした時に、その調から「特に近い」と感じられる[調性感](https://ja.wikipedia.org/wiki/調性感 "wikilink")をもつ、幾つかの[調](../Page/調.md "wikilink")のことである。この基準となる調を**主調**（しゅちょう）または**基調**（きちょう）と呼ぶ\[5\]。

主調と関係調との間には音階の構成音や主要な[和音](../Page/和音.md "wikilink")に共通性がみられるので、[転調](../Page/転調.md "wikilink")等もスムーズに行われる\[6\]（転調をしても、聞き手の感覚としても比較的違和感が少ない、ということ）。

（同主調を除けば）関係調同士の相互関係は、[五度圏](../Page/五度圏.md "wikilink")上、主調を中心とした[扇形](../Page/扇形.md "wikilink")の関係図にまとめることができる\[7\]（[五度圏](../Page/五度圏.md "wikilink")を参照）。

## 分類

狭義でいう関係調は**[同主調](https://ja.wikipedia.org/wiki/同主調 "wikilink")**、**[平行調](https://ja.wikipedia.org/wiki/平行調 "wikilink")**、**[属調](https://ja.wikipedia.org/wiki/属調 "wikilink")**、**[下属調](https://ja.wikipedia.org/wiki/下属調 "wikilink")**の四つである。広義では**属調平行調**と**下属調平行調**が含まれる。さらに**属調の同主調**と**下属調の同主調**も含めることもある\[8\]が、あまり一般的ではない。関係調の範囲は厳密に決まったものではない。

### 同主調

**同主調**（どうしゅちょう、/パラレル・キー、）とは、同じ[主音](https://ja.wikipedia.org/wiki/主音 "wikilink")（第一音）を持つ長調と短調。「**同名調**（どうめいちょう）」ともいう\[9\]。調名は「長」、「短」を除けば同じである。長調の場合、短調の場合ともに主調と異なる音は[第三音のみである](https://ja.wikipedia.org/wiki/中音 "wikilink")（第六音も主調と異なるが、和声的長音階や旋律的短音階を考えると決定的ではない）。

  - ある長調に対して主音が同じ短調。例: ハ長調に対するハ短調。
  - ある短調に対して主音が同じ長調。例: ハ短調に対するハ長調｡

### 平行調

**平行調**（へいこうちょう、/レラティブ・キー）とは、同じ[調号](../Page/調号.md "wikilink")によって示される調\[10\]。並行調とも書く。主調と異なる音は主調が長調の場合、主調の[属音](https://ja.wikipedia.org/wiki/属音 "wikilink")（第五音）と平行調の[導音](https://ja.wikipedia.org/wiki/導音 "wikilink")（第七音）のみである。また主調が短調の場合、主調の導音と平行調の属音のみである。

  - ある長調に対して同じ調号の短調。例: ハ長調に対するイ短調。
  - ある短調に対して同じ調号の長調。例: イ短調に対するハ長調。

### 属調

**属調**（ぞくちょう、/ドミナント・キー）とは、主調の属音を主音とする同種の調（長調に対する長調、短調に対する短調）。調号は♯が一つ増えるか♭が一つ減る\[11\]。主調と異なる音は主調が長調の場合、主調の[下属音](https://ja.wikipedia.org/wiki/下属音 "wikilink")（第四音）と属調の導音のみである。また主調が短調の場合、主調の下属音と平行調の導音、及び主調の導音と属調の第三音のみである。

  - ある長調に対して調号の♯が一つ多い長調。例: ハ長調に対するト長調。
  - ある長調に対して調号の♭が一つ少ない長調。例: 変ロ長調に対するヘ長調。
  - ある短調に対して調号の♯が一つ多い短調。例: イ短調に対するホ短調。
  - ある短調に対して調号の♭が一つ少ない短調。例: ト短調に対するニ短調。

### 下属調

**下属調**（かぞくちょう、/サブドミナント・キー）とは主調の下属音を主音とする同種の調。調号は♭が一つ増えるか♯が一つ減る\[12\]。属調と逆の関係。主調と異なる音は主調が長調の場合、主調の導音と下属調の下属音のみである。また主調が短調の場合、主調の導音と下属調の属音、及び主調の第三音と下属調の導音のみである。

  - ある長調に対して調号の♯が一つ少ない長調。例: ト長調に対するハ長調。
  - ある長調に対して調号の♭が一つ多い長調。例: ヘ長調に対する変ロ長調。
  - ある短調に対して調号の♯が一つ少ない短調。例: ホ短調に対するイ短調。
  - ある短調に対して調号の♭が一つ多い短調。例: ニ短調に対するト短調。

### 属調平行調

**属調平行調**は、「属調の平行調、平行調の属調」ともいう。調号は♯が一つ増えるか♭が一つ減る\[13\]。主調と異なる音は主調が長調の場合、主調の[第二音と属調平行調の導音](https://ja.wikipedia.org/wiki/上主音 "wikilink")、及び主調の下属音と属調平行調の第二音のみである。また主調が短調の場合、主調の導音と属調平行調の主音のみである。

  - ある長調に対して調号の♯が一つ多い短調。例: ハ長調に対するホ短調。
  - ある長調に対して調号の♭が一つ少ない短調。例: 変ロ長調に対するニ短調。
  - ある短調に対して調号の♯が一つ多い長調。例: イ短調に対するト長調。
  - ある短調に対して調号の♭が一つ少ない長調。例: ト短調に対するヘ長調。

### 下属調平行調

**下属調平行調**は、「下属調の平行調、平行調の下属調」ともいう。調号は♯が一つ減るか♭が一つ増える\[14\]。主調と異なる音は主調が長調の場合、主調の主音と下属調平行調の導音のみである。主調が短調の場合、主調の導音と下属調平行調の第二音、及び主調の第二音と下属調平行調の[第六音のみである](https://ja.wikipedia.org/wiki/下中音 "wikilink")。

  - ある長調に対して調号の♯が一つ少ない短調。例: ハ長調に対するニ短調。
  - ある長調に対して調号の♭が一つ多い短調。例: 変ロ長調に対するハ短調。
  - ある短調に対して調号の♯が一つ少ない長調。例: ホ短調に対するハ長調。
  - ある短調に対して調号の♭が一つ多い長調。例: ト短調に対する変ホ長調。

ハ長調を例とした関係調を表にまとめると、以下のとおりである。

|         |            |            |
| ------- | ---------- | ---------- |
|         | 属調：ト長調     | 属調平行調：ホ短調  |
| 同主調：ハ短調 | **主調：ハ長調** | 平行調：イ短調    |
|         | 下属調：ヘ長調    | 下属調平行調：ニ短調 |

ハ長調の関係調

英語圏では、同主調を「parallel key（平行調の意）」とよび、平行調を「relative key（関係調の意）」と言い、日本語と一致していない。平行調、同主調などの名称は、日、英、独、仏 各国語がまちまちであり、混乱しやすい\[15\]。
