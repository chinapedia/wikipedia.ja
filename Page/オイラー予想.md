**オイラー予想**（オイラーよそう）とは、[スイス](../Page/スイス.md "wikilink")の数学者[レオンハルト・オイラー](../Page/レオンハルト・オイラー.md "wikilink")が提唱した、[フェルマーの最終定理](../Page/フェルマーの最終定理.md "wikilink")を発展させた数学的予想である。現在では、反例によってこの予想は正しくないことが証明されている。

## 予想の内容

オイラーは[フェルマーの最終定理](../Page/フェルマーの最終定理.md "wikilink")のn=3のとき、すなわち

  -
    *x*<sup>3</sup> + *y*<sup>3</sup> =*z*<sup>3</sup>

を満たす[自然数](../Page/自然数.md "wikilink")の解 (x, y, z) は存在しないことを証明した。 ここから、フェルマーの最終定理を拡張して、

  -
    *x*<sup>4</sup> + *y*<sup>4</sup> + *z*<sup>4</sup> = *w*<sup>4</sup>

を満たす[自然数](../Page/自然数.md "wikilink")の解 (x, y, z, w) は存在しない、と予想した。

同様に

  -
    *x*<sup>5</sup> + *y*<sup>5</sup> + *z*<sup>5</sup> + *w*<sup>5</sup> = *v*<sup>5</sup>
    *x*<sup>6</sup> + *y*<sup>6</sup> + *z*<sup>6</sup> + *w*<sup>6</sup> + *v*<sup>6</sup> = *u*<sup>6</sup>

を満たす自然数の解も存在しない、とした。

すなわち、*n* \> 3 とすると、*n* − 1 個の *n* 乗数の和を1個の *n* 乗数で表すことはできないということを示唆した。 これが、オイラー予想である。

## 歴史

オイラーの発表以降、比較的小さな自然数では反例を見つけることができず、長い間正しいと信じられてきた。

しかし1966年にレオン・J・ランダーとトーマス・R・パーキンによって *n* = 5 の場合の反例として解 (27, 84, 110, 133, 144) が発見され、27<sup>5</sup> + 84<sup>5</sup> + 110<sup>5</sup> + 133<sup>5</sup> = 144<sup>5</sup> が成り立つことが確認された。

この発見から *n* = 4 の場合も反例がある可能性があるとして研究が続けられ、1986年に[ハーバード大学](https://ja.wikipedia.org/wiki/ハーバード大学 "wikilink")のノーム・エルキーズ([Noam Elkies](https://ja.wikipedia.org/wiki/w:Noam_Elkies "wikilink"))が、[楕円曲線](../Page/楕円曲線.md "wikilink")論とコンピュータを用いて発見した。その反例は 2682440<sup>4</sup> + 15365639<sup>4</sup> + 18796760<sup>4</sup> = 20615673<sup>4</sup> という複雑なものだった。この発見と同時に解は無限に存在することも確認され、約200年間未解決となっていたオイラー予想は、否定的に証明された。

また、2004年にはジム・フライによって*n*=5の場合の反例85282<sup>5</sup> + 28969<sup>5</sup> + 3183<sup>5</sup> + 55<sup>5</sup> = 85359<sup>5</sup>が発見された。

## 反例

  - 箇条書されている式は*n* − 1 個の *n* 乗数の和を1個の *n* 乗数で表す例

そうではない式は*n* 個の *n* 乗数の和を1個の *n* 乗数で表す ===*k* = 4===

  - 95800<sup>4</sup> + 217519<sup>4</sup> + 414560<sup>4</sup> = 422481<sup>4</sup> (R. Frye, 1988)\[1\]

30<sup>4</sup> + 120<sup>4</sup> + 272<sup>4</sup> + 315<sup>4</sup> = 353<sup>4</sup> (R. Norrie, 1911)\[2\] （R. Norrie によって最小の解であることが示されている。）

\===*k* = 5===

  - 27<sup>5</sup> + 84<sup>5</sup> + 110<sup>5</sup> + 133<sup>5</sup> = 144<sup>5</sup> (Lander & Parkin, 1966)\[3\]
