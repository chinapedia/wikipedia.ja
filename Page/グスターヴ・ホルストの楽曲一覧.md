**グスターヴ・ホルストの楽曲一覧**では、[グスターヴ・ホルスト](../Page/グスターヴ・ホルスト.md "wikilink")の作品リストを示す。

H番号はホルストの生誕100周年にあたる1974年に娘の[イモージェン・ホルスト](../Page/イモージェン・ホルスト.md "wikilink")によって出版された主題別カタログ(*A Thematic Catalogue of Gustav Holst's Music*)の番号である。

## 概説

### 管弦楽曲

#### エグドン・ヒース

『エグドン・ヒース』 (*Egdon Heath* ) は、イギリスの作家[トーマス・ハーディ](../Page/トーマス・ハーディ.md "wikilink")の小説『帰郷』説（原題：の舞台を描いた管弦楽作品である。副題には「ハーディを賛えて」とある。

#### サマセット狂詩曲

サマセット狂詩曲 (*A Somerset Rhapsody*) は、イングランドの[サマセット](../Page/サマセット.md "wikilink")地方の旋律を素材に用いた[狂詩曲](../Page/狂詩曲.md "wikilink")であり、1906年に[民謡](../Page/民謡.md "wikilink")の収集家、[セシル・シャープ](https://ja.wikipedia.org/wiki/セシル・シャープ "wikilink")(Cecil Sharp) の提案で作曲された。セシル・シャープの採譜した中から、*Sheep Shearing Song*、*The True Love's Farewell*、「南部ドイツ人」(*High Germany*)、「カッコウ」(*The Cuckoo*)の4旋律が用いられている。

#### 日本組曲

『日本組曲』 (*Japanese Suite*) は、6楽章から成る[バレエ](../Page/バレエ.md "wikilink")音楽で、全編を通して日本民謡の旋律により構成されている。日本人舞踊家[伊藤道郎](../Page/伊藤道郎.md "wikilink")の依頼により、『[惑星](../Page/惑星_\(組曲\).md "wikilink")』の作業を中断して書かれた。旋律は伊藤道郎の口からホルストが採譜したとされ、「ねんねんころりよおころりよ」の歌詞で知られる子守唄(これはしばし「[江戸子守唄](https://ja.wikipedia.org/wiki/江戸子守唄 "wikilink")」などと呼ばれる)などが使われている。1916年にロンドンでこの曲の伊藤道郎による舞踊公演が行われた。

なお、武石みどり([東京音楽大学](../Page/東京音楽大学.md "wikilink"))の研究では、上記説の一部が否定されている。武石は“[ブリティッシュ・ライブラリーにある](../Page/大英図書館.md "wikilink")『日本組曲』のスコア（Add. Ms 57882）のタイトルページには、ホルストの自筆で「マリオネット・ダンス以外の部分の主題を提供してくれた日本のダンサー伊藤道郎のために作曲した」ことが明記されている。”とし、一部の曲は伊藤から採譜されたものではないことを示し、また“初めてパート譜が作成され作品が初演されたのは、伊藤がアメリカに渡った後、1919年3月22日”、“伊藤が1916年8月までロンドンに滞在していたにもかかわらず、伊藤の踊りで『日本組曲』が初演された記録はない。”、さらに“『日本組曲』は伊藤の依頼により特定の機会に初演するために作曲されたものではなかったということが推測される。”\[1\]として、伊藤の依頼とロンドンでの公演も否定している。

日本組曲の構成は以下の通りである。

1.  前奏曲 - 漁師の歌 (*Prelude - Song of the Fisherman* )
2.  儀式の踊り (*Ceremonial Dance* )
3.  操り人形の踊り (*Dance of the Marionette* )
4.  間奏曲 - 漁師の歌 (*Interlude - Song of the Fisherman* )
5.  桜の木の下での踊り (*Dance Under the Cherry Tree* )
6.  終曲 - 狼たちの踊り (*Finale - Dance of the Wolves* )

「操り人形の踊り」については“「惑星」の第3曲〈水星〉作曲の直前に作られたため、両者には性格と表現に共通性がある。”とされる。\[2\]。

#### ベニ・モラ

『ベニ・モラ』 (*Beni Mora* ) は、「東洋的」な曲調の組曲で、3つの楽章から成る。

1.  第一の踊り (*First Dance* )
2.  第二の踊り (*Second Dance* )
3.  終曲「ウレド・ナイルの街で」 (*Finale: In the Street of Ouled Nails* )
