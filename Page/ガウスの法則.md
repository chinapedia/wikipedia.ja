**ガウスの法則**（ガウスのほうそく、\[1\]）とは、[カール・フリードリヒ・ガウス](../Page/カール・フリードリヒ・ガウス.md "wikilink")が[1835年](../Page/1835年.md "wikilink")に発見し、[1867年](../Page/1867年.md "wikilink")に発表した[電荷](../Page/電荷.md "wikilink")と[電場](../Page/電場.md "wikilink")の関係をあらわす[方程式](../Page/方程式.md "wikilink")である。この式は[ジェームズ・クラーク・マクスウェル](../Page/ジェームズ・クラーク・マクスウェル.md "wikilink")により数学的に整備され、[マクスウェルの方程式](../Page/マクスウェルの方程式.md "wikilink")の1つとなった。[電気](../Page/電気.md "wikilink")における[アンペールの法則](../Page/アンペールの法則.md "wikilink")とみなすこともできる。

ちなみに、[単位](../Page/単位.md "wikilink")の[ガウスは](../Page/ガウス_\(単位\).md "wikilink")[磁束密度](../Page/磁束密度.md "wikilink")の単位であり、電場を扱うこの法則とは全く関係がない。

## 積分形式

一般に積分形式と呼ばれるガウスの法則は以下の形で表される。

ここで、

|        |                                                  |
| ------ | ------------------------------------------------ |
| **D**  | : [電束密度](../Page/電束密度.md "wikilink")             |
| *ρ*    | : [電荷密度](../Page/電荷密度.md "wikilink")             |
| *Q*    | : 積分領域 V の内部にある[電荷](../Page/電荷.md "wikilink")の総和 |
| d**S** | : [面素ベクトル](../Page/面積分.md "wikilink")            |
| *V*    | : 体積                                             |

である。

この式は、ある領域内に電荷が存在すると、その領域から電荷と等しい大きさの[電束](https://ja.wikipedia.org/wiki/電束 "wikilink")という物理量が出入りするということを示している。

[電場](../Page/電場.md "wikilink") **E** (**D** =ε**E**)を用いて  と表すこともできる。\(\varepsilon\)は[誘電率](../Page/誘電率.md "wikilink")であり、非線形素子においては行列となることもあるが、線形の場合はスカラー量である。

## 微分形式

### 発散

閉曲面Sにおいて、ガウスの法則（\(\oint_S \mathbf{D} \cdot d\mathbf{S} = Q\) ）において、体積Vの微小変化による電束（ガウスの法則、面積分）の変化率を[div](https://ja.wikipedia.org/wiki/発散_\(ベクトル解析\) "wikilink")***D*** で表す。  ここでΔSはΔVの表面である。

また

|   |        |
| - | ------ |
| ρ | : 電荷密度 |

となる。

ここで記号「div」はダイバージェンス (divergence) と読み、 [発散を表す](https://ja.wikipedia.org/wiki/発散_\(ベクトル解析\) "wikilink")。

|       |                                      |
| ----- | ------------------------------------ |
| **D** | : [電束密度](../Page/電束密度.md "wikilink") |
