[FWHM.svg](https://ja.wikipedia.org/wiki/File:FWHM.svg "fig:FWHM.svg") **半値幅**（はんちはば、half width）は、山形の[関数の広がりの程度を表す指標](https://ja.wikipedia.org/wiki/関数_\(数学\) "wikilink")。**半値全幅** (はんちぜんはば、full width at half maximum, **FWHM**) と、その半分の値の**半値半幅** (half width at half maximum, **HWHM**) とがある。単に半値幅と言うと半値全幅のことが多い。

## 定義

関数 *f*(*x*) が、ある箇所の前後で山形の局所的応答を示しているとする。尚、*f*(*x*) が[不連続](https://ja.wikipedia.org/wiki/不連続 "wikilink")な場合などは考えない。もし不連続なときは、近似的な[連続関数](https://ja.wikipedia.org/wiki/連続関数 "wikilink")を考える。

*f*(*x*) を、ベースライン関数 *b*(*x*) と局所的応答関数 *g*(*x*) の和

  -
    *f*(*x*) = *b*(*x*) + *g*(*x*)

で表す。山形の広がりの成分は *g*(*x*) に含まれ、十分大きい *x* と十分小さい *x* （あるいは、±∞ への極限）に対し *g*(*x*) = 0 となる。

なお、十分大きい *x* と十分小さい *x* に対し *f*(*x*) = 0 なら、*b*(*x*) = 0 とみなし、

  -
    *f*(*x*) = *g*(*x*)

とすることができる。実用上は、*f*(*x*) が上の条件を満たさなくてもこうすることがある。

*g*(*x*) の[最大値](https://ja.wikipedia.org/wiki/最大値 "wikilink")を *g*<sub>max</sub> = *g*(*x*<sub>max</sub>) とすると、*g*(*x*) = *g*<sub>max</sub>/2 を満たす *x* が2つ以上存在する（*g*(*x*) が[単峰性](https://ja.wikipedia.org/wiki/単峰性 "wikilink")なら *x*<sub>max</sub> の左右に1つずつ存在する）。*g*(*x*) = *g*<sub>max</sub>/2 を満たす最小の *x* を *x*<sub>1</sub>、最大の *x* を *x*<sub>2</sub> とすると、*x*<sub>2</sub> - *x*<sub>1</sub> が半値全幅、(*x*<sub>2</sub> - *x*<sub>1</sub>)/ 2 が半値半幅である。
