**デイヴィッド・マクミラン** (David W.C MacMillan、[1968年](../Page/1968年.md "wikilink") -)は[アメリカ合衆国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")の有機化学者。[プリンストン大学](https://ja.wikipedia.org/wiki/プリンストン大学 "wikilink")教授。 不斉有機触媒である[マクミラン触媒を用いた](https://ja.wikipedia.org/wiki/有機分子触媒#マクミラン触媒 "wikilink")[不斉合成](../Page/不斉合成.md "wikilink")や可視光レドックス触媒を用いた有機合成の研究で著名な化学者である。

## 経歴

[1968年](../Page/1968年.md "wikilink")、[スコットランド](../Page/スコットランド.md "wikilink")生まれ。[1998年](https://ja.wikipedia.org/wiki/1998年 "wikilink")に[博士研究員](../Page/博士研究員.md "wikilink")として[ハーバード大学](https://ja.wikipedia.org/wiki/ハーバード大学 "wikilink")にて[デイヴィッド・エヴァンス](https://ja.wikipedia.org/wiki/デイヴィッド・エヴァンス "wikilink")の下で研究を行なった。当時の研究は、[スズ](../Page/スズ.md "wikilink")-ビスオキサゾリン触媒を用いた不斉アルドール反応の開発であった。 同年、[カリフォルニア大学バークレイ校](https://ja.wikipedia.org/wiki/カリフォルニア大学バークレイ校 "wikilink")に着任、独立。[2000年](../Page/2000年.md "wikilink")、[カリフォルニア工科大学](../Page/カリフォルニア工科大学.md "wikilink")に異動。さらに[2006年](../Page/2006年.md "wikilink")、[プリンストン大学](https://ja.wikipedia.org/wiki/プリンストン大学 "wikilink")に着任、現在に至る。

## 研究

マクミランは、[2000年](../Page/2000年.md "wikilink")の[カリフォルニア工科大学](../Page/カリフォルニア工科大学.md "wikilink")時代にマクミラン触媒と呼ばれるイミダゾリジノン骨格をもつ不斉触媒を設計、開発を行なった <ref name = "MacMilan catalyst">*J. Am. Chem. Soc.*

**`2000`**`,`*`122`*`, 9874`</ref>`。それを用いた様々な`[`不斉合成`](../Page/不斉合成.md "wikilink")の開発も行なっている<ref name = "mukaiyama-michael">` `*`J.``   ``Am.``   ``Chem.``   ``Soc.`*
**`2003`**`,`*`125`*`, 1192`</ref>` `<ref name = "1-3fuka">` `*`J.``   ``Am.``   ``Chem.``   ``Soc.`*
**`2007`**`,`*`129`*`, 7004`</ref>`。マクミラン触媒は`[`天然物`](https://ja.wikipedia.org/wiki/天然物 "wikilink")の[`全合成`](../Page/全合成.md "wikilink")にも用いられており<ref name = "total synthesis">*`J.``   ``Am.``   ``Chem.``   ``Soc.`*` `
**`2016`**`,`*`138`*`, 10838`</ref>`、有用な`[`有機分子触媒`](../Page/有機分子触媒.md "wikilink")である`。`

また、[2007年](../Page/2007年.md "wikilink")に[SOMO](https://ja.wikipedia.org/wiki/SOMO "wikilink")-activationによる新たな合成方法を提唱<ref name = "somo-activation">*Science*

**`2007`**`,`*`316`*`, 582`</ref>`。その後、`[`2008年`](https://ja.wikipedia.org/wiki/2008年 "wikilink")には[`酸化剤`](../Page/酸化剤.md "wikilink")の代わりに可視光レドックス触媒を用いた不斉[`カップリング反応`](../Page/カップリング反応.md "wikilink")を報告した<ref name = "coupling">*`Science`*`  `
**`2008`**`,`*`322`*`, 77`</ref>`。`
