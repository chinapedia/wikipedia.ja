**細尿管**（さいにょうかん、renal tubule：**尿細管**、**腎細管**）は、[腎臓](../Page/腎臓.md "wikilink")における[糸球体より集合管にいたるまでの](../Page/ネフロン.md "wikilink")、[原尿](https://ja.wikipedia.org/wiki/原尿 "wikilink")が通り再吸収・分泌などを受ける組織のこと。細尿管には糸球体から伸びる毛細血管が取り巻いていて、腎小体で[ボーマン嚢](../Page/ボーマン嚢.md "wikilink")に排出された原尿の成分のうち、[ブドウ糖](https://ja.wikipedia.org/wiki/ブドウ糖 "wikilink")のすべてと、[水](../Page/水.md "wikilink")・[無機塩類](https://ja.wikipedia.org/wiki/無機塩類 "wikilink")のほとんど（99％）と[グリセリン](../Page/グリセリン.md "wikilink")などその他の物質を再吸収する。原尿からのグルコースの再吸収は、[ナトリウムイオン](https://ja.wikipedia.org/wiki/ナトリウムイオン "wikilink")の濃度勾配を用いたSGLT1/SGLT2による。再吸収能力を超えた濃度のグルコースは、細尿管で再吸収しきれず尿中に排出されるが、これが[糖尿病](../Page/糖尿病.md "wikilink")において尿中にグルコースが排出される理由である。

## 細尿管の分節

細尿管はその位置によって形状・機能が異なる。そのため、個々の部位ごとに名称が付けられている。ある部位と別の部位をどのように区切るか、すなわち分節するかには2つの方法がある。まず、腎臓の皮質、髄質のうちどの部分に位置するか、どのような外形をとっているか、すなわち走行による分類である。もう一つは、尿管の上皮細胞による分類である。細尿管の[上皮細胞](../Page/上皮細胞.md "wikilink")はいずれも単層上皮ではあるが、位置によって微絨毛（刷毛縁）の密度や長さ、細胞の形状と細胞内装置の分布や数、隣接上皮細胞とのふん合、基底膜からの陥入が異なるためである。以下の図では走行によるものと上皮細胞によるものを左右の端に配し、さらに細かい分類を中央にまとめた\[1\]。

| 近位曲部      | 近位曲細尿管  | S1文節      |
| S2文節      |         |           |
| ヘンレループ    | 近位細尿管   |           |
| S3文節      |         |           |
| 細い下行脚     | 短ループのもの | 中間細尿管     |

## 細尿管の経路

1.  （[糸球体](../Page/糸球体.md "wikilink")）
2.  [ボーマン嚢](../Page/ボーマン嚢.md "wikilink")
3.  [近位尿細管](../Page/近位尿細管.md "wikilink")
4.  [ヘンレのループ](https://ja.wikipedia.org/wiki/ヘンレのループ "wikilink")（[ヘンレ係蹄 (下行脚)](../Page/ヘンレ係蹄_\(下行脚\).md "wikilink")、[ヘンレ係蹄 (細い上行脚)](https://ja.wikipedia.org/wiki/ヘンレ係蹄_\(細い上行脚\) "wikilink")、[ヘンレ係蹄 (太い上行脚)](../Page/ヘンレ係蹄_\(太い上行脚\).md "wikilink")）
5.  [遠位尿細管](../Page/遠位尿細管.md "wikilink")
6.  [集合管](https://ja.wikipedia.org/wiki/集合管 "wikilink")
7.  （[尿管](https://ja.wikipedia.org/wiki/尿管 "wikilink")、輸尿管）
