[thumb](https://ja.wikipedia.org/wiki/ファイル:IBook_G4.jpg "wikilink")) を表示している[iBook G4](https://ja.wikipedia.org/wiki/IBook "wikilink")\]\] **Aqua**（**アクア**）とは、[アップルが](../Page/アップル_\(企業\).md "wikilink")[Mac OS X v10.0で採用した一連の](https://ja.wikipedia.org/wiki/Mac_OS_X_v10.0 "wikilink")[GUIのデザイン](https://ja.wikipedia.org/wiki/グラフィカルユーザインタフェース "wikilink")（ビジュアルテーマ）の名称である。

## 概要

水や滴（しずく）をイメージした、白と水色が基調の立体的なGUIが特徴である。流れるようなアニメーションが用いられ、動的な操作性を提供する。[スコット・フォーストール](https://ja.wikipedia.org/wiki/スコット・フォーストール "wikilink")にて開発が主導された\[1\]。

Aquaは[Quartz](../Page/Quartz.md "wikilink")と呼ばれるグラフィック[ライブラリ](../Page/ライブラリ.md "wikilink")によって、[PDFベースで実現される](../Page/Portable_Document_Format.md "wikilink")。

初期のMac OS XではGUI全体の動作速度について批判されていたが、バージョンアップのたびに、[Quartz Extremeや](https://ja.wikipedia.org/wiki/Quartz_Compositor "wikilink")[Core Imageにより](https://ja.wikipedia.org/wiki/Core_Image "wikilink")[GPUに処理を渡すことで改善された](../Page/Graphics_Processing_Unit.md "wikilink")。

## 変遷

OSのメジャーバージョンアップのたびに、デザインにも手を加えられ、初期の姿からは大きく変貌している。また、[Mac OS X v10.2以降](https://ja.wikipedia.org/wiki/Mac_OS_X_v10.2 "wikilink")、2x[AGP](https://ja.wikipedia.org/wiki/AGP "wikilink")以上のバスに接続されたGPUが搭載されたマシンでは、Quartz Extremeにより高速な描画が可能である。[Mac OS X v10.3から](https://ja.wikipedia.org/wiki/Mac_OS_X_v10.3 "wikilink")[Mac OS X v10.4までは金属表面のヘアライン処理を意識した](https://ja.wikipedia.org/wiki/Mac_OS_X_v10.4 "wikilink")"Brushed Metal"と呼ばれるユーザーインターフェースデザイン\[2\]も併用され、[Mac OS X v10.5以降は従来のAquaと統合された](https://ja.wikipedia.org/wiki/Mac_OS_X_v10.5 "wikilink")。

Mac OS X v10.3までのAquaウィンドウの[タイトルバー](https://ja.wikipedia.org/wiki/タイトルバー "wikilink")やプルダウンメニューは、半透明で淡い横縞模様が入った特徴的なデザインであったが、Mac OS X v10.4でAquaウィンドウのタイトルバーが灰色無地で不透明になり、メニューの透明度も下げられた。

Mac OS X v10.5では、"Brushed Metal"デザインがAquaデザインに加わり、タイトルバーがグレーになった。[プッシュボタンから水色の影がなくなった](https://ja.wikipedia.org/wiki/ボタン_\(GUI\) "wikilink")。プルダウンメニューの背景は不透明になり、[コンテキストメニュー](https://ja.wikipedia.org/wiki/コンテキストメニュー "wikilink")は低い透明度の白無地になった。

[Mac OS X Lionでは従来の水色の](https://ja.wikipedia.org/wiki/Mac_OS_X_Lion "wikilink")[スクロールバー](https://ja.wikipedia.org/wiki/スクロールバー "wikilink")が操作時にのみ表示されるグレーのオーバーレイ・スクロールバーに変えられた。プッシュボタンが角の丸い長方形となった。タブ、プルダウンメニューから水色が廃された。

[アプリケーションや](../Page/アプリケーションソフトウェア.md "wikilink")[ツールバー](https://ja.wikipedia.org/wiki/ツールバー "wikilink")のアイコンは、ガラスや[ステンレス](https://ja.wikipedia.org/wiki/ステンレス "wikilink")や[プラスチックを思わせる](../Page/合成樹脂.md "wikilink")、透明感や光沢を強調した写実的なものが用いられていたが、[Mac OS X v10.3あたりから](https://ja.wikipedia.org/wiki/Mac_OS_X_v10.3 "wikilink")[アルミや不透明なプラスチックを意識した](../Page/アルミニウム.md "wikilink")、ソリッドなアイコンも使われるようになった。

[iOS](https://ja.wikipedia.org/wiki/iOS "wikilink")のデザインにも多大な影響を与えていたが、[ジョナサン・アイブ](https://ja.wikipedia.org/wiki/ジョナサン・アイブ "wikilink")がソフトウェア含めたアップル製品全般のデザインを統括するようになり、一気にソフトウェアの方向性が変わった。それにより、iOS7を皮切りにアップル製品のOSが[フラットデザイン](https://ja.wikipedia.org/wiki/フラットデザイン "wikilink")に徐々に移行し、MacOSも[OS X Mavericksから](https://ja.wikipedia.org/wiki/OS_X_Mavericks "wikilink")[OS X Yosemiteにてフラットデザインに移行](https://ja.wikipedia.org/wiki/OS_X_Yosemite "wikilink")\[3\]となり、一部の古いソフト以外で従来のAqua/Metalデザインは使われなくなった。
