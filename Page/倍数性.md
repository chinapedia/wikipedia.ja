[Haploid_vs_diploid.svg](https://ja.wikipedia.org/wiki/File:Haploid_vs_diploid.svg "fig:Haploid_vs_diploid.svg") **倍数性**（ばいすうせい、 または ）とは、[生物](../Page/生物.md "wikilink")あるいはその[生活環](https://ja.wikipedia.org/wiki/生活環 "wikilink")の一時期において、生存に必要な最小限の[染色体](../Page/染色体.md "wikilink")の1組（[ゲノム](../Page/ゲノム.md "wikilink")）を何セット持つかを示す概念。

## 用語としての「ゲノム」と倍数性の関連

[ゲノム](../Page/ゲノム.md "wikilink")の最初の定義は「[配偶子](https://ja.wikipedia.org/wiki/配偶子 "wikilink")がもつ1組の[染色体](../Page/染色体.md "wikilink")」であったが、その後定義が変更され、使われる分野も広がった。倍数性の説明に用いる「ゲノム」は、現在の「生物をその生物たらしめるのに必須な遺伝情報」という定義から外れるものではないが、最初の定義に近く、「生存に必要な最小限の1組の染色体」を指す。生物のゲノム構成を記号で示すときは、二倍体であれば AA や BB のように、四倍体であれば AAAA や AABB のようにあらわす。

## 倍数性

**倍数性**とは、[ゲノム](../Page/ゲノム.md "wikilink")を何セット持つかを示す概念である。

ゲノム1セットに相当する染色体数を**基本数**といい、*x*（まれに *b*）で示す。倍数性は基本数の倍数で *x*, 2*x*, 3*x*, ... のように表され、それぞれ**一倍性**（**半数性**: monoploidy, haploidy）、**二倍性** (diploidy)、**三倍性** (triploidy)、... という。それらの倍数性の生物（**倍数体**）は、それぞれ**一倍体**（**半数体**）、**二倍体**、**三倍体**、... という。

例えば、基本数が 9 (*x* = 9) の四倍性 (4*x*) の倍数体生物がいるとすると、その生物の体細胞 (2*n*) における染色体数は、「2*n* = 4*x* = 36」「2*n* = 36 の四倍体」などと示される。動物の場合、特殊なものを除いてほぼ全てが二倍体 (2*x*) であり、二倍体であることを強調したい場合でない限り、「2*x* =」という表現はされない。

倍数性を、しばしば *n*, 2*n*, 3*n*, 4*n*, ... のように表現することがあり、生物学の教科書などでもこのような表現をしている場合があるが、これは**誤り**である。

*n* と 2*n* は、生物体またはその細胞の**核相**を示すものである。*n* と 2*n* は、それぞれ**単相**世代、**複相**世代であることを示す（[生活環](https://ja.wikipedia.org/wiki/生活環 "wikilink")を参照）ものであり、核相と倍数性の表現で直接的な関連はない（2*n* の **2** は、2倍体の **2** と関係ない）。染色体数の「*n* = ・・・」「2*n* = ・・・」という表現は、あくまで「**どの核相の細胞における染色体数であるか**」を示すものであり、たとえ半数体生物や3倍体生物でも、体細胞における染色体数は「2*n* = ・・・」、配偶子細胞における染色体数は「*n* = ・・・」で表す。

生物の核相は基本的に単相か複相のどちらかであり、3*n*, 4*n*, ... という核相は、特殊なもの（一般的な被子植物の二次胚乳細胞など）を除き、存在しない。

英語では二倍体以上の倍数性を示すpolyploidy（倍数性・多倍性）も同様に使われる用語である。多倍性の生物を**多倍体**という。

また、一倍体ゲノムのDNA量はC値と定義される。

## 倍数体

**倍数体**（ばいすうたい、）とは、倍数性に基づいて表現した生物の分け方。**一倍体**（**半数体**）、**二倍体**、**三倍体**など。

[有性生殖](https://ja.wikipedia.org/wiki/有性生殖 "wikilink")をする[動物](https://ja.wikipedia.org/wiki/動物 "wikilink")の多くは、両親から[配偶子](https://ja.wikipedia.org/wiki/配偶子 "wikilink")を通してそれぞれ1セットのゲノムを受け取り、計2セットのゲノムを持つ二倍体（ヒト, 2*n* = 46 など）である。

一方、[植物](../Page/植物.md "wikilink")には様々な倍数体が存在している。それらは、[農業](https://ja.wikipedia.org/wiki/農業 "wikilink")で役に立つ特性を持つことがあり、[作物](https://ja.wikipedia.org/wiki/作物 "wikilink")の[品種](../Page/品種.md "wikilink")・[種として成立している](../Page/種_\(分類学\).md "wikilink")。

植物の倍数体は、**同質倍数体**と**異質倍数体**とに分けられる。1種類のゲノムを複数持つ場合を同質倍数体といい、2種類以上のゲノムで構成されている場合を異質倍数体という。動物の場合は異質倍数体は少ないが、[生殖](../Page/生殖.md "wikilink")能力がない種間[雑種](../Page/雑種.md "wikilink")（[レオポン](https://ja.wikipedia.org/wiki/レオポン "wikilink")や[ラバ](../Page/ラバ.md "wikilink")）は異質二倍体と考えることもできる。アフリカツメガエル (*Xenopus laevis*) は異質4倍体と考えられており、*Xenopus*属にはさらに異質8倍体や異質12倍体のカエルもいる。

### 同質倍数体

**同質倍数体** (autopolyploid) とは、同じ種類のゲノムを複数持つ倍数体。しばしば、「同質」を省略して表記される。

三倍体や四倍体では[細胞](../Page/細胞.md "wikilink")・[器官](../Page/器官.md "wikilink")・植物体全体が大きくなる傾向があり、農作物の遺伝的改良（[育種](https://ja.wikipedia.org/wiki/育種 "wikilink")）に利用されている。倍数体は種子ができにくいこと（不稔・部分不稔）もあるが、逆に三倍体の不稔を利用して「種無し」の品種を作ることにも利用されている。人工的に倍数性を増加（**倍加**）させるには、[コルヒチン](https://ja.wikipedia.org/wiki/コルヒチン "wikilink")などの薬剤を使う。

また、倍加に伴って環境適応性が向上することがある。この現象は、高緯度地帯に倍数性が高い植物が多いことの一因と推定されている。

以下、同質倍数体の例を挙げるが、通常は同質倍数体に含めない一倍体も、便宜上、併せて取り扱う。

  - 一倍体（半数体）
    二倍体の[配偶子](https://ja.wikipedia.org/wiki/配偶子 "wikilink")の状態。植物では、[花粉](../Page/花粉.md "wikilink")[培養などで半数性の個体を作り](https://ja.wikipedia.org/wiki/組織培養 "wikilink")、遺伝的に[固定](https://ja.wikipedia.org/wiki/固定 "wikilink")する技術がある。動物の例としては、[蜂や](../Page/ハチ.md "wikilink")[蟻など](https://ja.wikipedia.org/wiki/アリ "wikilink")[膜翅目昆虫では](https://ja.wikipedia.org/wiki/ハチ目 "wikilink")、雄は半数体である。
  - 同質三倍体
    [栄養繁殖](https://ja.wikipedia.org/wiki/栄養繁殖 "wikilink")性の植物では、三倍体であることがそれほど珍しくない。[園芸植物などでも見られ](https://ja.wikipedia.org/wiki/園芸#生産園芸 "wikilink")、作物としては[バナナ](https://ja.wikipedia.org/wiki/バナナ "wikilink")や[フキ（蕗）の栽培品種などがある](https://ja.wikipedia.org/wiki/アキタブキ "wikilink")。人工的に作った三倍体の例は、四倍体と二倍体を掛け合わせて作った「[種なしスイカ](https://ja.wikipedia.org/wiki/種なしスイカ "wikilink")」の例がある。また、[魚類](../Page/魚類.md "wikilink")でも、三倍体を作成し、大型化した魚を[養殖](https://ja.wikipedia.org/wiki/養殖 "wikilink")することが試みられている。
  - 同質四倍体
    同質四倍体の代表例に[ジャガイモ](https://ja.wikipedia.org/wiki/ジャガイモ "wikilink")がある。

### 異質倍数体

**異質倍数体** (allopolyploid) とは、2種類以上のゲノムで構成されている倍数体。異なるゲノムを持つ植物間の雑種に由来すると考えられている。各ゲノムが2セットずつである場合、[減数分裂](../Page/減数分裂.md "wikilink")の際に各ゲノムの染色体が[娘細胞](https://ja.wikipedia.org/wiki/娘細胞 "wikilink")に均等に分配されるため、生殖能力は正常である。

染色体数を倍数性を明らかにして示すときは、例えば6セットのゲノムからなる場合、2*n* = 6*x* = 42 のように書く。

特殊な呼び方として、2種類のゲノムを持つ異質四倍体 (2*n* = 4*x*) を**複二倍体** (amphidiploid) という。人工的に複二倍体を作った例としては、ハクラン（ゲノム構成 AACC: [ハクサイ](../Page/ハクサイ.md "wikilink")+[キャベツ](https://ja.wikipedia.org/wiki/キャベツ "wikilink")）などがある。

  - 異質倍数体の例

:\* [普通コムギ](https://ja.wikipedia.org/wiki/パンコムギ "wikilink")（*Triticum aestivum*、2*n* = 6*x* = 42、ゲノム構成 AABBDD）

:\* [タカナ](https://ja.wikipedia.org/wiki/タカナ "wikilink")（*Brassica juncea*、2*n* = 4*x* = 36、ゲノム構成 AABB）、[セイヨウアブラナ](../Page/セイヨウアブラナ.md "wikilink")（*B. napus*、2*n* = 4*x* = 38、ゲノム構成 AACC）

  -

      -
        上記2つの植物は、Aゲノム種（*B. rapa*、2*n* = 20 : [ハクサイ](../Page/ハクサイ.md "wikilink")・日本在来[アブラナ](../Page/アブラナ.md "wikilink")など）、Bゲノム種（*B. nigra*、2*n* = 16 : セイヨウカラシナ）、Cゲノム種（*B. oleracea*、2*n* = 18 : [キャベツ](https://ja.wikipedia.org/wiki/キャベツ "wikilink")・[ケール](../Page/ケール.md "wikilink")など）のゲノムを持つ複二倍体である。

## 半倍数性

**半倍数性** (haplodiploidy) とは、[生活環](https://ja.wikipedia.org/wiki/生活環 "wikilink")の中に（生殖細胞としてではなく）独立の個体として振舞う**単相**（半数体）と**複相**（二倍体であることが多い）の世代が存在することである。
