**Agent Rois**（エージェント・ロイス）は総合的な[IT](https://ja.wikipedia.org/wiki/IT "wikilink")関連事業を展開する日本の企業\[1\]。 [ホームページ](../Page/ホームページ.md "wikilink")・[アプリ](https://ja.wikipedia.org/wiki/アプリ "wikilink")の制作やIT[コンサルティング](../Page/コンサルティング.md "wikilink")、個人の[起業](../Page/起業.md "wikilink")支援サービス、ビジネス記事を掲載するR-Newsなどを運営している。

## 概要

代表の[八木隆聖](https://ja.wikipedia.org/wiki/八木隆聖 "wikilink")\[2\]大学時代に始めた、[ネット通販](https://ja.wikipedia.org/wiki/ネット通販 "wikilink")サイトをきっかけに事業が始まる。2016年にホームページ制作会社に切り替え、事業が大幅に拡大する。 2018年IT関連の総合経営コンサルティングサービスを開始し、[個人](../Page/個人.md "wikilink")から[企業](../Page/企業.md "wikilink")に至るまでの独自経営戦略・集客システムの販売を行う。 起業支援サービスでは個人に対し、会社設立までの代行業務や初期段階でのシステム開発を行う。さらに自社商品の販売代行を、業務委託として提供している。

## 沿革

  - 2014年 - 八木隆聖が創業、インターネット上での通信販売を行う。
  - 2014年7月 - [東京都](../Page/東京都.md "wikilink")[世田谷区](../Page/世田谷区.md "wikilink")にCHARON\[3\]設立。
  - 2014年12月 - 並行輸入で海外ブランド商品の販売を開始。
  - 2015年6月 - 生活雑貨商品の販売を開始。
  - 2016年1月 - [Agent Rois Japan](https://www.agentrois.com) を設立。
  - 2016年2月 - ホームページ制作サービスを開始。
  - 2016年6月 - アプリ制作サービス開始。
  - 2016年12月 - Agent Rois SEOシステム「NEO-System」開発。
  - 2017年3月 - Agent Rois 「HP-Package」開発
  - 2017年6月 - [愛知県](https://ja.wikipedia.org/wiki/愛知県 "wikilink")[名古屋市](../Page/名古屋市.md "wikilink")[中区](../Page/中区_\(名古屋市\).md "wikilink")[栄](../Page/栄.md "wikilink")\[4\]に本社を移動。
  - 2017年11月 - Google認定パートナーに認定\[5\]
  - 2018年1月 - 企業むけにIT関連の総合コンサルティングサービスを開始する。
  - 2018年10月 - 「Agent Rois Members」を発行。
  - 2018年12月 - 「Gold Aesthetics」を開発、販売開始。
  - 2019年3月 - 個人向けサービスとして個人の企業支援サービスを開始。
  - 2019年8月 - 代表の出身地である[埼玉県](../Page/埼玉県.md "wikilink")にAgent Rois Japan 埼玉支社を設立。
