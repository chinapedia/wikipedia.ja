**フランス語の人称代名詞**では、現代標準[フランス語](https://ja.wikipedia.org/wiki/フランス語 "wikilink")における[人称代名詞](../Page/人称代名詞.md "wikilink")について記述する。強勢形を除く全てが[接語](../Page/接語.md "wikilink")である点が、フランス語の人称代名詞の特徴である。

## 人称代名詞の変化

強勢形を除き、全てが[動詞](../Page/動詞.md "wikilink")と緊密に結びつく接語である\[1\]。[主格](../Page/主格.md "wikilink")、[対格](../Page/対格.md "wikilink")、[与格](../Page/与格.md "wikilink")はそれぞれ[主語](../Page/主語.md "wikilink")、[直接目的補語](https://ja.wikipedia.org/wiki/直接目的補語 "wikilink")、[間接目的補語](https://ja.wikipedia.org/wiki/間接目的補語 "wikilink")の[格](../Page/格.md "wikilink")である。一部の語は、後続の[語](../Page/語.md "wikilink")が[母音](../Page/母音.md "wikilink")または無音の h で始まる時、[リエゾン](../Page/リエゾン.md "wikilink")または[エリジオン](../Page/エリジオン.md "wikilink")を起こす。

| 数  | 人称     | 性     | 主格    | 対格    | 与格    | 再帰  | 強勢 |
| -- | ------ | ----- | ----- | ----- | ----- | --- | -- |
| 単数 | 1      | \-    | je¹²  | me¹   | moi   |     |    |
| 2  | \-     | tu²   | te¹   | toi   |       |     |    |
| 3  | 男性     | il    | le¹   | lui   | se¹   | lui |    |
| 女性 | elle   | la¹   | elle  |       |       |     |    |
| \- | on³    | \-    | \-    | soi   |       |     |    |
| 複数 | 1      | \-    | nous³ | nous³ | nous⁴ |     |    |
| 2  | \-     | vous³ | vous³ | vous⁴ |       |     |    |
| 3  | 男性     | ils³  | les³  | leur  | se¹   | eux |    |
| 女性 | elles³ | elles |       |       |       |     |    |

1.  je, me, te, le, la, se は、後続の語が母音または無音の h で始まるとき、エリジオンを起こしてそれぞれ j' , m' , t' , l' , l' , s'  になる。
      - je: **J'**ai faim.  (**私は**空腹だ。)
      - me: Il **m'**a ecrit une lettre.  (彼は**私に**手紙を書きました。)
      - te: Je **t'**aime.  (私は**君を**愛しています。)
      - le: Tu **l'**as vu ?  (君は**彼を**見ましたか。)
      - la: Tu **l'**as vue ?  (君は**彼女を**見ましたか。)
      - se: Il **s'**est acheté une voiture.  (彼は**自分に**車を買いました。)
2.  [口語](../Page/口語.md "wikilink")では、 je は後続の語に関わらず常に、また tu は後続の語が母音または無音の h で始まるとき、エリジオンを起こしてそれぞれ j' , t'  になる。
      - je: **J'**vous dis tout.  (**私は**あなたに全てを話します。 - 口語)
      - tu: **T'**as un petit ami ?  (**君は**彼氏がいますか。 - 口語)
3.  on, nous, vous, ils, elles, les は、後続の語が母音または無音の h で始まるとき、リエゾンを起こして括弧内の[子音](../Page/子音.md "wikilink")が発音される。
4.  聞き手を含まない一人称複数を表すのに nous autres  を使うことがある\[2\]。これに対応する二人称複数は vous autres  であるが、vous は元々話し手を含まず、一人称ではなく三人称と対比される。

## 親称・敬称

二人称複数代名詞は、二人称単数の[敬称](https://ja.wikipedia.org/wiki/敬称 "wikilink")としても用いられる。したがって二人称単数は[親称](../Page/親称.md "wikilink") tu と敬称 vous の対立があるが、複数では区別無く vous を用いる。親称・敬称は親疎を表すものであり、年齢の上下に関係なく、普通は互いに tu を使うか、互いに vous を使う。大人と子供が話す時は非対称になり、大人が子供に tu を使い、子供が大人に vous を使う。また、上司と部下、教師と学生など、地位的・職権的な上下関係がある場合も非対称になる。二人称単数の敬称として vous を使う場合、動詞の[活用](../Page/活用.md "wikilink")は二人称複数に従うが、名詞や[形容詞](../Page/形容詞.md "wikilink")は単数に従う。

  - Vous êtes un grand homme.  (あなたは偉大な人です。)
  - Vous êtes de grands hommes.  (あなた方は偉大な人々です。)

また、敬称の vous に対しては、[所有限定詞に](https://ja.wikipedia.org/wiki/フランス語の限定詞#所有限定詞 "wikilink") votre  を、所有代名詞に le vôtre  を用いる。

## 性

フランス語では全ての[名詞](../Page/名詞.md "wikilink")に[性があり](../Page/性_\(文法\).md "wikilink")、男性と女性に分けられるので、中性の人称代名詞は無い。人間を指す場合は自動的に性が決まるが、物を指す場合はまず言語化しないと性が割り当てられず、人称代名詞を使うことができない\[3\]。言語化する前は、指示代名詞の ce および ça を使う。指示対象に男性と女性の両方が含まれている場合は男性複数として扱う。

一人称、二人称の代名詞は性と無関係だが、名詞や形容詞は指示対象の性に一致する。

  - Je suis heureux.  (私は幸せです。 - 話者は男性)
  - Je suis heureuse.  (私は幸せです。 - 話者は女性)

## 代名詞 on

三人称代名詞 on は、文脈に応じて異なる意味を持つ。動詞の活用は三人称単数に従うが、（主に口語では）名詞や形容詞は指示対象の数および性に一致する。

1.  不特定の人を指し、一般的な事柄を述べる。
      -
        **On** est tous égaux devant la loi.  (**人は**法の前ではみな平等である。)
        Au Japon, **on** mange du riz.  (日本では、**人々は**米を食べる。)
2.  不特定の人を指し、行為者を明示しない。
      -
        **On** me l'a donné.  (**誰かが**私にそれをくれた。 = それはもらった物だ。)
3.  [口語](../Page/口語.md "wikilink")で、nous の代わりに「私たち」を表す。この時、主格は on、再帰形は se を用いるが、対格、与格、強勢形は nous を用いる。また[所有限定詞も一人称複数を用いる](https://ja.wikipedia.org/wiki/フランス語の限定詞#所有限定詞 "wikilink")。
      -
        **On** a fait nos devoirs.  (**私たちは**宿題をやった。)
        Nous, **on** est japonais.  (**私たちは**日本人です。)

文語では、母音の後、特に si, que の後で l'on  になる。ただし次の語が  で始まるなら on のままにする。この l' は元々[定冠詞だったが](https://ja.wikipedia.org/wiki/フランス語の限定詞#定冠詞 "wikilink")、現在では発音の変化に過ぎない。

  - On n'est jamais aussi malheureux que **l'on** croit.  (人は決して自分が思うほど不幸ではない。)

## 代名詞 y

代名詞 y  は中性代名詞とも呼ばれ、基本的に à + 物を表す。三人称代名詞与格形の lui と leur は、à + 人を表す。

1.  間接目的語の à + 物を表す。
      -
        J'ai répondu à votre question.  (私はあなたの質問に答えました。)
          -
            → J'**y** ai répondu.  (私はそれに答えました。)
2.  形容詞の補語の à + 物を表す。
      -
        Elle est sensible au froid.  (彼女は寒さに弱いです。)
          -
            → Elle **y** est sensible.  (彼女はそれに弱いです。)
3.  場所の副詞の à + 場所あるいは類似の dans + 場所などを表す。これは代名詞ではなく[代副詞](https://ja.wikipedia.org/wiki/代副詞 "wikilink")（[副詞](../Page/副詞.md "wikilink")的代名詞とも）である。英語の there に当たる。
      -
        Il va à Paris.  (彼はパリに行きます。)
          -
            → Il **y** va.  (彼はそこに行きます。)
4.  成句で用いる。
      -
        Il **y** a une voiture là.  (そこに車があります。)

## 代名詞 en

代名詞 en  は中性代名詞とも呼ばれ、基本的に de + 人/物を表す。

1.  動詞が de + 人/物を取る時、それを表す。
      -
        Je parle du problème.  (私はその問題について話します。)
          -
            → J'**en** parle.  (私はそれについて話します。)
2.  場所の副詞の de + 場所を表す。これは代名詞ではなく代副詞である。英語の from there に当たる。
      -
        Il vient de Paris.  (彼はパリから来ました。)
          -
            → Il **en** vient.  (彼はそこから来ました。)
3.  不特定の名詞を表す。
      -
        J'ai bu du jus de pomme.  (私はリンゴジュースを多少飲みました。)
          -
            → J'**en** ai bu.  (私はそれを多少飲みました。)
        J'ai mangé une pomme.  (私はリンゴを一個食べました。)
          -
            → J'**en** ai mangé une.  (私はそれを一個食べました。)

## 与格と形容詞

[英語](../Page/英語.md "wikilink")と異なり、[コピュラ](../Page/コピュラ.md "wikilink") (être) + [形容詞](../Page/形容詞.md "wikilink")は与格代名詞を持てる。

  - Votre participation **nous** est indispensable.  (あなたの参加は**私たちに**不可欠です。)

## 強勢形

接語ではない人称代名詞を強勢形と呼ぶ。強勢形は基本的に人を表す\[4\]。接語の人称代名詞は必ず動詞と結びつき、また強勢を置くことができないので、以下の場合に強勢形が必要になる。

1.  前置詞の後:
      -
        Je l'ai fait pour **toi**.  (**君**のためにそれをしました。)
2.  [属詞](https://ja.wikipedia.org/wiki/属詞 "wikilink") (être の補語)、強調構文:
      -
        C'est **moi**.  (**私**です。)
        C'est **lui** qui y est allé.  (そこへ行ったのは**彼**です。)
3.  [話題](../Page/話題.md "wikilink"):
      -
        **Moi**, je suis japonais, mais **vous**, vous êtes française.  (**私は**日本の男性ですが、**あなたは**フランスの女性です。)
4.  三人称の話題化 (文語):
      -
        **Lui** le sait bien.  (**彼は**それを良く知っています。 - 文語)
        **Lui**, il le sait bien.  (〃 - 口語)
5.  主語を et でつなぐ時 (文語):
      -
        **Lui** et **moi** sommes frères.  (**彼**と**私**は兄弟です。 - 文語)
        **Lui** et **moi**, on est frères.  (〃 - 口語)
        <sup>\*</sup>Il et je sommes frères. (不可)
6.  接語の内容を示す時:
      -
        Je les aime beaucoup, **elle** et **lui**.  (私は**彼女**と**彼**のことが大好きです。)
        <sup>\*</sup>Je la et l'aime beaucoup. (不可)

## 接語の語順

接語の人称代名詞および[否定の ne](https://ja.wikipedia.org/wiki/フランス語の否定文#ne "wikilink") は、強勢を持たず、動詞または[助動詞と結び付いて発音される](https://ja.wikipedia.org/wiki/助動詞_\(言語学\) "wikilink")。その語順は下記のように厳密に定まっている\[5\]\[6\]。各枠からは最大 1 個の接語しか使えない。衝突する場合、間接目的語が接語でなくなり、 à + [強勢形で表される](https://ja.wikipedia.org/wiki/#強勢形 "wikilink")。なお以下で目的格とは[対格](../Page/対格.md "wikilink")と[与格](../Page/与格.md "wikilink")のいずれかとして働く人称代名詞を示す。

### 後接語の語順

後ろの動詞とつなげて発音される接語の語順を示す。枠 3 と枠 5 は共起できない。

| 数     | 人称   | 枠    |
| ----- | ---- | ---- |
| 1     | 2    | 3    |
| 主格    | 否定   | 目的格  |
| 単数    | 1    | je   |
| 2     | tu   | te   |
| 3     | il   | se   |
| elle  | la   |      |
| on    |      |      |
| 複数    | 1    | nous |
| 2     | vous | vous |
| 3     | ils  |      |
| elles |      |      |
|       |      | ne   |

  - Je vous la présente.  (私はあなたに彼女を紹介します。 - 枠 3 と 4)
  - Je la lui présente.  (私は彼女を彼に紹介します。 - 枠 4 と 5)
  - Je me présente à vous.  (私は自分をあなたに紹介します。 - 枠 3 からは 1 個のみ)
  - Je vous présente à elle.  (私はあなたを彼女に紹介します。 - 枠 3 と枠 5 は共起しない)
  - Je vous présente Yvonne.  (私はあなたにイヴォンヌを紹介します。 - 枠 3 は間接目的語)
  - Je vous présente à Yvonne.  (私はあなたをイヴォンヌに紹介します。 - 枠 3 は直接目的語)

### 前接語の語順

前の動詞とつなげて発音される接語の語順を示す。枠 5 と枠 6 は共存できない。この語順は、肯定命令文でのみ使われる。

| 数    | 人称     | 枠      |
| ---- | ------ | ------ |
| 4    | 5      | 6      |
| 対格   | 与格     | 目的格    |
| 単数   | 1      |        |
| 2    | \-toi¹ |        |
| 3    | \-le²  | \-lui  |
| \-la |        |        |
| 複数   | 1      |        |
| 2    | \-vous |        |
| 3    | \-les  | \-leur |

1.  \-moi  と -toi  は、後ろに -en  または -y  が続く時、それぞれ -m'  と -t'  になる。しかし口語では、-moi と -toi を変えずに -en と -y をそれぞれ -z-en  と -z-y  に変えたり、枠 6 を枠 8 の後に置いたりすることがある。
      -
        例: « Tu m'en donnes. »  に対応する命令文:
        Donne-m'en.  (文語、口語)
        Donne-moi-z-en.  (口語のみ)
        Donnes-en-moi.  (口語のみ)
2.  後接語 le の発音は  だが、前接語 -le の発音は  である。ただしほとんど違いは無い。
