**中継局**（ちゅうけいきょく）とは、[総務省](../Page/総務省.md "wikilink")[告示](../Page/告示.md "wikilink")[基幹放送用周波数使用計画](https://ja.wikipedia.org/wiki/基幹放送用周波数使用計画 "wikilink")第1 総則 第1項（3）に「[親局](../Page/親局.md "wikilink")以外の[基幹放送局](../Page/基幹放送局.md "wikilink")」と定義している。

## 概説

[地上基幹放送](https://ja.wikipedia.org/wiki/地上基幹放送 "wikilink")において、[地勢](https://ja.wikipedia.org/wiki/地勢 "wikilink")的な影響で[親局](../Page/親局.md "wikilink")[送信所](../Page/送信所.md "wikilink")だけでは[放送区域](https://ja.wikipedia.org/wiki/放送区域 "wikilink")をカバーできない時に、親局とは別の場所に設けられる補助的な[地上基幹放送局](https://ja.wikipedia.org/wiki/地上基幹放送局 "wikilink")である。[日本](https://ja.wikipedia.org/wiki/日本 "wikilink")では、サテライト局（[空中線電力](../Page/空中線電力.md "wikilink")0.1W以下の小規模な中継局は[ミニサテライト局](../Page/ミニサテライト局.md "wikilink")（ミニサテ局）と呼ばれることが多い）と、[英語](../Page/英語.md "wikilink")ではトランスポーザ（[transposer](https://ja.wikipedia.org/wiki/:en:transposer "wikilink")）という。中継方法としては、[演奏所](https://ja.wikipedia.org/wiki/演奏所 "wikilink")（スタジオ）・親局からの専用線（主に[中波放送](../Page/中波放送.md "wikilink")・[短波放送](../Page/短波放送.md "wikilink")）や、[無線](../Page/無線.md "wikilink")（[STL](https://ja.wikipedia.org/wiki/STL_\(放送局\) "wikilink")）で送信する方法、親局の電波を受信し周波数を変換して送信する放送波中継（主に[超短波放送](../Page/超短波放送.md "wikilink")（[FM放送](https://ja.wikipedia.org/wiki/FM放送 "wikilink")）・[テレビジョン放送](../Page/テレビジョン放送.md "wikilink")）がある。なお、1990年代以降に開設された、いわゆる[平成新局](../Page/平成新局.md "wikilink")は資金が乏しいことから、アナログテレビ放送では中継局が少なめであることが多かった\[1\]。

## 周波数

通常、親局とは別の[周波数が使われる](../Page/電波の周波数による分類.md "wikilink")。

同一周波数の使用に関しては、中波放送では[振幅変調](../Page/振幅変調.md "wikilink")の特性上同一周波数の中継が容易で[山陽放送](https://ja.wikipedia.org/wiki/山陽放送 "wikilink")、[山形放送](../Page/山形放送.md "wikilink")、[山梨放送](https://ja.wikipedia.org/wiki/山梨放送 "wikilink")、[四国放送](https://ja.wikipedia.org/wiki/四国放送 "wikilink")、[西日本放送](../Page/西日本放送.md "wikilink")、[南海放送](https://ja.wikipedia.org/wiki/南海放送 "wikilink")、[熊本放送](https://ja.wikipedia.org/wiki/熊本放送 "wikilink")、[宮崎放送](https://ja.wikipedia.org/wiki/宮崎放送 "wikilink")などで実施している。FM放送ではFMキャプチャ効果により出力差が大きいと小出力側が抑圧されるため、出力差の少ない[コミュニティ放送局](https://ja.wikipedia.org/wiki/コミュニティ放送局 "wikilink")で実施している。[地上デジタルテレビ放送では](https://ja.wikipedia.org/wiki/日本の地上デジタルテレビ放送 "wikilink")[OFDM方式の特性上](../Page/直交周波数分割多重方式.md "wikilink")、同一周波数での中継が比較的容易であるため、単一周波数ネットワーク（SFN - Single Frequency Network）による中継網を採用することもできる。

[周波数帯](https://ja.wikipedia.org/wiki/周波数帯 "wikilink")を異にするものとしては、中波放送では後述の**[\#補完中継局](https://ja.wikipedia.org/wiki/#補完中継局 "wikilink")**が設置されている。また、アナログテレビ放送においては、2012年（平成24年）3月31日の全廃までに[宮城県](https://ja.wikipedia.org/wiki/宮城県 "wikilink")、[東京都](../Page/東京都.md "wikilink")、[神奈川県](https://ja.wikipedia.org/wiki/神奈川県 "wikilink")、[兵庫県](../Page/兵庫県.md "wikilink")では[SHF中継局が](../Page/センチメートル波.md "wikilink")、[北海道](https://ja.wikipedia.org/wiki/北海道 "wikilink")、[長野県](../Page/長野県.md "wikilink")、[岐阜県](../Page/岐阜県.md "wikilink")、[三重県](https://ja.wikipedia.org/wiki/三重県 "wikilink")、[徳島県](https://ja.wikipedia.org/wiki/徳島県 "wikilink")、[香川県](../Page/香川県.md "wikilink")、[山口県](../Page/山口県.md "wikilink")、[佐賀県](../Page/佐賀県.md "wikilink")、[大分県](../Page/大分県.md "wikilink")、[熊本県](https://ja.wikipedia.org/wiki/熊本県 "wikilink")、[宮崎県](../Page/宮崎県.md "wikilink")\[2\]、[鹿児島県](../Page/鹿児島県.md "wikilink")では、[UHF親局に対し](../Page/極超短波.md "wikilink")[VHF](https://ja.wikipedia.org/wiki/VHF "wikilink")中継局が、それぞれ設置されていた。

## 操作

原則として[無線従事者](../Page/無線従事者.md "wikilink")の管理（常駐するという意味ではない。）を要する。例外は[総務](../Page/総務省.md "wikilink")[省令](../Page/省令.md "wikilink")[電波法施行規則](../Page/電波法施行規則.md "wikilink")第33条に基づく告示\[3\]にある、他の地上基幹放送局の番組を中継するのみでかつ[適合表示無線設備](https://ja.wikipedia.org/wiki/適合表示無線設備 "wikilink")のみを使用する、FM放送用で空中線電力0.25W以下又はテレビジョン放送用で空中線電力0.05W以下の地上基幹放送局の[無線設備](https://ja.wikipedia.org/wiki/無線設備 "wikilink")の外部の転換装置で電波の質に影響を及ぼさないものの技術操作のみ不要である。これは、[ギャップフィラー](https://ja.wikipedia.org/wiki/ギャップフィラー "wikilink")と呼ばれる[受信障害対策中継放送](https://ja.wikipedia.org/wiki/受信障害対策中継放送 "wikilink")用[送信機](../Page/送信機.md "wikilink")のことである。

## 損壊又は故障の対策

[放送法施行規則](https://ja.wikipedia.org/wiki/放送法施行規則 "wikilink")第103条には、「この款において使用する用語は、次の定義に従うもの」 としている。

  -
    2.「[プラン局](https://ja.wikipedia.org/wiki/プラン局 "wikilink")」とは、親局以外の基幹放送局のうち、基幹放送用周波数使用計画の表に掲げる中継局のことをいう。
    3.「その他の中継局」とは、親局及びプラン局以外の基幹放送局をいう。

また、親局の定義も引用されている。

この款とは、第4章第5節の第1款、すなわち基幹放送局の電気通信設備の損壊又は故障の対策について規定するもので、親局、プラン局、その他の中継局とに区分してとるべき対策を規定している。

## コールサイン

中継局には[コールサインが指定されているものがあり](https://ja.wikipedia.org/wiki/識別信号#呼出符号 "wikilink")、中波放送ではその局独自の編成を行うところがある。アナログテレビ放送では[HBCテレビ](../Page/北海道放送.md "wikilink")、[STVテレビ](../Page/札幌テレビ放送.md "wikilink")、[テレビ静岡](../Page/テレビ静岡.md "wikilink")などで、本局と内容を差し替えることがあった。この内、終了時まで独自編成を行っていたのはSTVテレビとテレビ静岡のみで、HBCテレビは2003年（平成15年）頃に取りやめた。終了時に全廃されたがそのコールサインはSTVテレビの札幌本局以外（[旭川局JOKY](https://ja.wikipedia.org/wiki/HBC・STV東旭川テレビ・ラジオ放送所#STV東旭川送信所 "wikilink")-TV、[帯広局JOWL](https://ja.wikipedia.org/wiki/帯広送信所#地上アナログテレビジョン放送 "wikilink")-TV \[4\]、[函館局JOMY](https://ja.wikipedia.org/wiki/函館山テレビ・FM放送所#地上アナログテレビジョン放送 "wikilink")-TV、[網走局JOVX](https://ja.wikipedia.org/wiki/網走送信所#地上アナログテレビジョン放送 "wikilink")-TV\[5\]、[釧路局JOSY](https://ja.wikipedia.org/wiki/釧路テレビ・ラジオ放送所#地上アナログテレビジョン放送送信設備 "wikilink")-TV、[室蘭局JOLY](https://ja.wikipedia.org/wiki/室蘭送信所#地上アナログテレビジョン放送 "wikilink")-TV）、テレビ静岡[浜松局](https://ja.wikipedia.org/wiki/浜松中継局#地上アナログテレビジョン放送送信設備 "wikilink")（JORH-TV）、[中国放送](../Page/中国放送.md "wikilink")[尾道局](https://ja.wikipedia.org/wiki/高見山_\(広島県\)#地上アナログテレビジョン放送送信設備 "wikilink")（JOEE-TV）、[山口放送](../Page/山口放送.md "wikilink")[関門局](https://ja.wikipedia.org/wiki/火の山_\(山口県\)#地上アナログテレビ放送 "wikilink")（JOPM-TV）、[テレビ西日本](../Page/テレビ西日本.md "wikilink")[北九州局](https://ja.wikipedia.org/wiki/八幡テレビ・FM放送所#地上アナログテレビジョン放送チャンネル・出力 "wikilink")（JOHX-TV）であった。

## 放送中継局

1950年（昭和25年）の電波法施行規則の全部改正\[6\]の際に第4条の「無線局の種別」に追加されたが、1958年（昭和33年）の改正\[7\]で削除され[固定局](https://ja.wikipedia.org/wiki/固定局 "wikilink")とされた。

## 補完中継局

基幹放送用周波数使用計画第1 総則 第5項に「中波放送の放送区域において災害対策などのため補完的に超短波放送用周波数を用いて放送を行う中継局」と規定\[8\]している。 混信による難聴取対策や災害対策として中波放送をFM放送の周波数で[サイマル放送](../Page/サイマル放送.md "wikilink")を行う中継局のことである。

詳細は、**[FM補完中継局](https://ja.wikipedia.org/wiki/FM補完中継局 "wikilink")**を参照。

## その他の中継局

### 陸上移動中継局

電波法施行規則第4条の無線局の種別の内、「中継局」と文言のある無線局である。 1982年（昭和57年）に[MCA無線業務が開始される際に制御局の為に制定](https://ja.wikipedia.org/wiki/第三者無線 "wikilink")\[9\]された種別で、1996年（平成8年）より[携帯電話の普及に伴い携帯電話用中継設備に対しても免許されることとなった](https://ja.wikipedia.org/wiki/日本における携帯電話 "wikilink")。

詳細は、**[陸上移動中継局](https://ja.wikipedia.org/wiki/陸上移動中継局 "wikilink")**を参照。

### 無線通信業務の中継局

[移動体通信](../Page/移動体通信.md "wikilink")では[基地局](https://ja.wikipedia.org/wiki/基地局 "wikilink")や[携帯基地局](https://ja.wikipedia.org/wiki/携帯基地局 "wikilink")を、[アマチュア無線](../Page/アマチュア無線.md "wikilink")では[アマチュア局](https://ja.wikipedia.org/wiki/アマチュア局 "wikilink")を見晴らしの良い山上や高層ビルの屋上、電波塔に設置して通信範囲を大きく広げる。**リピータ（レピータ）**（[repeater](https://ja.wikipedia.org/wiki/:en:repeater "wikilink")）と呼ばれることが多い。リピータは受信した電波を同時に送信する必要があるが、[周波数分割複信](https://ja.wikipedia.org/wiki/周波数分割複信 "wikilink")方式が一般的である。送受信周波数間隔は、システムにより様々で、周波数の割当てや[デュープレクサ](https://ja.wikipedia.org/wiki/デュープレクサ "wikilink")の実現難易度から総合的に決められる。

#### アマチュア無線

周波数帯域の広いVHF以上が用いられる。UHF（430MHz帯、1200MHz帯、2400MHz帯）が主であるが、28MHz帯（[短波](../Page/短波.md "wikilink")であるがVHFに準じる。）やSHF（5600MHz帯、10.1GHz帯）のものも一部にある。[デジタル変調](../Page/デジタル変調.md "wikilink")の普及に伴い[D-STAR](../Page/D-STAR.md "wikilink")方式も運用されはじめている。

日本では1982年より免許されることとなった。[日本アマチュア無線連盟](../Page/日本アマチュア無線連盟.md "wikilink")（JARL）のみが開設できることとされ、JARLが直轄して、又は所属する管理団体が設置している。但し、周波数は公開されておりJARL非会員でも利用できる。多数のアマチュア無線家が使用するため、短時間で要領よく通信を行うことがマナーとされる。

送受信周波数間隔は次のとおり。

| 周波数帯     | 間隔     | 備考          |
| -------- | ------ | ----------- |
| 28MHz帯   | 100kHz |             |
| 50MHz帯   | 1MHz   | 日本では免許されない。 |
| 144MHz帯  | 600kHz |             |
| 430MHz帯  | 5MHz   |             |
| 1200MHz帯 | 20MHz  |             |
| 2400MHz帯 |        |             |
| 5600MHz帯 | 40MHz  |             |
| 10.1GHz帯 | 70MHz  |             |
