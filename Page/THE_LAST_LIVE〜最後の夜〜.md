**THE LAST LIVE〜最後の夜〜**（ザ・ラスト・ライブ さいごのよる）は、日本のロックバンド[X JAPANが](../Page/X_JAPAN.md "wikilink")[1997年](https://ja.wikipedia.org/wiki/1997年 "wikilink")の[12月31日](../Page/12月31日.md "wikilink")に[東京ドーム](../Page/東京ドーム.md "wikilink")で行った解散コンサート。

## 解説

X JAPANは、1997年[9月22日](../Page/9月22日.md "wikilink")に解散を発表し、同年12月31日に本ライブを行った。（開催の発表は同年１１月） 例年、X JAPANは、東京ドームライブを12月30日31日の2DAYSで行なっていたが、本コンサートは、１日のみであった。 チケットは、S席11,000円、A席10,000円、B席9,000円であった。なお、収容人数を確保するため、ステージの後ろ側から見る席(A席)も用意されることになった。

解散するにあたりYOSHIKIとTOSHIに確執が生まれていた。

３曲目の、[SCARS](https://ja.wikipedia.org/wiki/SCARS "wikilink")演奏後に、TOSHIがMCに詰まる場面がある。側にいた、[HIDE](https://ja.wikipedia.org/wiki/HIDE "wikilink")が、「TOSHIくん、パーッと。パーッと。」と声をかけている。このわずかなやりとりが、TOSHIとHIDEの最後のコミュニケーションとなる。

4曲目の「[DAHLIA](https://ja.wikipedia.org/wiki/DAHLIA_\(X_JAPANの曲\) "wikilink")」演奏中、ドラムを叩くYOSHIKIの背後でTOSHIが歌う場面があるが、YOSHIKIは複雑な表情をしている。演奏終了後、YOSHIKIがドラムを破壊する。

１０曲目の [紅](https://ja.wikipedia.org/wiki/紅 "wikilink")でも、TOSHIがYOSHIKIに向かって歌う場面があるが、YOSHIKIはTOSHIと目を合わせていない。

アンコール２曲目の「[Forever Love](https://ja.wikipedia.org/wiki/Forever_Love_\(X_JAPANの曲\) "wikilink")」演奏時に、YOSHIKIとTOSHIが抱き合うシーンがある。 後日談で、ドラムソロを終えたYOSHIKIが花道を通り「Forever Love」を歌うTOSHIに対して、 「最初はぶんなぐってやろうと思っていました。でも、そこにいたのは、幼馴染のTOSHIだった。」 と語っている。

この他、ライブの代表曲「X」で「飛べ飛べ飛べ飛べ……」の掛け声はHIDEが行うのだが、本ライブではYOSHIKIが行っている。

ライブの最後、新曲、また最後の曲として「[THE LAST SONG](https://ja.wikipedia.org/wiki/THE_LAST_SONG "wikilink")」が披露された後、SEとして「[Tears](https://ja.wikipedia.org/wiki/Tears_\(X_JAPANの曲\) "wikilink")」が流れるメンバーが去った後のステージでは、これまでのXの歴史を振り返るフイルムが流された。またそのフィルムの最後では、これまで応援してくれたファンに対する、X JAPANメンバー一同のメッセージが表示された。「Tears」の最後にはドームに花火が上がり、正式にライブ終了となる。

このコンサートの後、X JAPANは『[第48回NHK紅白歌合戦](https://ja.wikipedia.org/wiki/第48回NHK紅白歌合戦 "wikilink")』のステージにおいて「Forever Love」を演奏した。前述の通り、YOSHIKIはこのコンサートも『紅白』も「最後ではない」とし、2000年に再結成をする計画を進めていたが（HIDEが死去した際に公表）、HIDEの急逝で計画は幻となった。X JAPANは解散ライブの後からの約10年間、コンサートを開催することはなく（HIDEの葬儀時を除く）、またYOSHIKIとTOSHIが面会することすらなかった。

しかし、2007年に事態は急転し、X JAPANは再結成を果たすこととなる（詳しくは「[X JAPAN](../Page/X_JAPAN.md "wikilink")」、「[攻撃再開 2008 I.V.〜破滅に向かって〜](https://ja.wikipedia.org/wiki/攻撃再開_2008_I.V.〜破滅に向かって〜 "wikilink")」の項を参照）。

現在は廃盤となっている『THE LAST LIVE VIDEO』（DVD、VHS）にはMCや20分以上に及ぶアンコールの映像まで収録されているが、「[ENDLESS RAIN](https://ja.wikipedia.org/wiki/ENDLESS_RAIN "wikilink")」以降の模様は収録されておらず、以降の模様は、TV放送物と[THE LAST SONGの](https://ja.wikipedia.org/wiki/THE_LAST_SONG "wikilink")[CD-EXTRA](https://ja.wikipedia.org/wiki/CD-EXTRA "wikilink")でしか視聴できない状態にあった（音声のみであれば、ライブCD「[The Last Live](https://ja.wikipedia.org/wiki/The_Last_Live_\(X_JAPANのアルバム\) "wikilink")」で聞くことができた）。

2011年6月11日、[YOSHIKI](../Page/YOSHIKI.md "wikilink")の公式[twitter](https://ja.wikipedia.org/wiki/twitter "wikilink")にて、13年越しの編集作業が終了したと書き込みがあった。これほどの時間がかかった理由は、YOSHIKI曰く「辛くて最後まで見られなかったから」。

2011年8月27日に[avexの夏フェス](https://ja.wikipedia.org/wiki/エイベックス・グループ "wikilink")「[a-nation](https://ja.wikipedia.org/wiki/a-nation "wikilink")'11」に出演した際に完全版DVDが10月26日に発売されることがアナウンスされ、10月26日に、通常版と当時のパンフレットの縮小復刻版などが付いたコレクターズBOXの２パッケージで「X JAPAN THE LAST LIVE 完全版」として発売された。

2013年9月25日に「X JAPAN THE LAST LIVE 完全版」の内容が[アップコンバート](https://ja.wikipedia.org/wiki/アップコンバート "wikilink")され、[Blu-ray Disc形式で発売された](../Page/Blu-ray_Disc.md "wikilink")。単体の通常版と、完全生産限定の1993年、1994年、1996年の東京ドームでのライブ(いずれも既にDVD化されていたのもの)とのBOX販売となる「X JAPAN Blu-ray BOX」の2パッケージで発売された。
