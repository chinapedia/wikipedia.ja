**メイリオ**（Meiryo）は、和文の[ゴシック体](../Page/ゴシック体.md "wikilink")[フォント](../Page/フォント.md "wikilink")の一つである。

## 解説

[Windows Vista](https://ja.wikipedia.org/wiki/Microsoft_Windows_Vista "wikilink") 日本語版のシステム用フォントとして開発された。 Vista 以降の[マイクロソフト](../Page/マイクロソフト.md "wikilink")製 [OS](../Page/オペレーティングシステム.md "wikilink") に標準で搭載され、 [Windows XP](../Page/Microsoft_Windows_XP.md "wikilink")、[Windows Server 2003](../Page/Microsoft_Windows_Server_2003.md "wikilink") までの [Microsoft Windows](https://ja.wikipedia.org/wiki/Microsoft_Windows "wikilink") で利用された [MS UI Gothic](https://ja.wikipedia.org/wiki/MS_ゴシック#種類 "wikilink") に代わる、日本語版の新しいシステム用フォントとして提供されている。

[TrueType アウトラインの](../Page/TrueType.md "wikilink") [OpenType フォントで](../Page/OpenType.md "wikilink")、[ビットマップを持たず](https://ja.wikipedia.org/wiki/フォント#ビットマップフォント "wikilink")[レンダリングの際に](../Page/レンダリング_\(コンピュータ\).md "wikilink") [ClearType](../Page/ClearType.md "wikilink") の使用を前提とするフォントである。欧文部は[プロポーショナルフォント](../Page/プロポーショナルフォント.md "wikilink")、和文部は[等幅フォント](../Page/等幅フォント.md "wikilink")で、全体として[プロポーショナルフォント](../Page/プロポーショナルフォント.md "wikilink")になっている。

[サンセリフ](../Page/サンセリフ.md "wikilink")（[ゴシック体](../Page/ゴシック体.md "wikilink")）系の書体で、直線部分と曲線部分がはっきりした若干膨らんだ印象の字形は横組みの[可読性](https://ja.wikipedia.org/wiki/可読性 "wikilink")を重視した。和文は[縦横比](../Page/アスペクト比.md "wikilink") 95:100 と若干横長である。デザインは欧文部の[ラテン文字](../Page/ラテン文字.md "wikilink")を[マシュー・カーター](../Page/マシュー・カーター.md "wikilink")が担当し、和文部の漢字と[仮名を](../Page/仮名_\(文字\).md "wikilink")[河野英一](https://ja.wikipedia.org/wiki/河野英一 "wikilink")とシーアンドジイ (C\&G) が担当した\[1\]。欧文部はカーターがデザインした [Verdana](https://ja.wikipedia.org/wiki/Verdana "wikilink") を基に再構築され、和文部は Verdana を基調にデザインされた\[2\]。和文はベースラインを少し高くするなど工夫して、和文と欧文が混在する文章で可読性を高めている。

[JIS X 0213:2004](../Page/JIS_X_0213.md "wikilink")（以下「JIS2004」）の文字に対応しており、[戸籍法](../Page/戸籍法.md "wikilink")施行規則に基づく全ての[人名用漢字](../Page/人名用漢字.md "wikilink")が支障なく使用できる。JIS2004 の文字に対応するその他のフォントは、[MS系統の](../Page/マイクロソフト.md "wikilink")[MS ゴシック、MS Pゴシック、MS UI Gothicと](../Page/MS_ゴシック.md "wikilink")[MS 明朝、MS P明朝があり](../Page/MS_明朝.md "wikilink")、これらは [Windows XP](../Page/Microsoft_Windows_XP.md "wikilink") などへも提供され\[3\]、JIS2004 の文字に対応していなかった Windows でも JIS2004 の文字との[互換性](../Page/互換性.md "wikilink")を持たせることができる。[Windows 7に搭載されたバージョンから](../Page/Microsoft_Windows_7.md "wikilink")[JIS X 0212の文字にも対応した](../Page/JIS_X_0212.md "wikilink")。

いくつかのOpenTypeのレイアウト機能に対応しており、対応アプリケーションではJIS78、JIS83、JIS90字形への切替\[4\]や、ひらがなやカタカナなどを半角等幅の字体へ切替\[5\]られる。[Windows 8](https://ja.wikipedia.org/wiki/Windows_8 "wikilink") 搭載のバージョンは JIS2004 で例示字形が変更される以前のJIS90字形の約120文字が、[IVS（異体字セレクタ）で利用できる](../Page/異体字セレクタ.md "wikilink")。

[Windows 7](https://ja.wikipedia.org/wiki/Windows_7 "wikilink") は後述する Meiryo UI が限定的に採用され、[Windows 8](https://ja.wikipedia.org/wiki/Windows_8 "wikilink") はメイリオに代わって Meiryo UI が全面採用された。[Windows 10](https://ja.wikipedia.org/wiki/Windows_10 "wikilink") はシステムフォントとして [Yu Gothic UI](https://ja.wikipedia.org/wiki/游書体#游ゴシック体 "wikilink") が新たに採用され、メイリオおよび Meiryo UI はシステムフォントではなくなった。

### フォント名

「メイリオ(Meiryo)」の語源は日本語の「明瞭」に基づき、画面上で見ても印刷しても極めて明瞭で読みやすいところから名付けられた。メイリョウでなく「メイリオ」であるのはアジア情緒な響きと「メイリョウ」より1文字少なくて済むためである\[6\]。フォントとしてはプロポーショナルフォントに属するが、フォント名に一部の TrueType フォントで採用されている等幅フォントではないことを示す「P」や「S」の文字が含まれていない。

### フォントファミリーの仕様

標準のレギュラー、太字のボールド、斜体の[イタリック](../Page/イタリック体.md "wikilink")、太字斜体のボールドイタリックと4種類のウェイトによるフォントファミリーで構成されている。ボールドはレギュラーを単純に太くせずに若干異なる造型である。 和文は[斜体](../Page/斜体.md "wikilink")表記を行う文化的背景がないとの意見から、イタリックはラテン文字部のみ提供されることになった。[\#注意点](https://ja.wikipedia.org/wiki/#注意点 "wikilink")に後述した。

### ClearType

メイリオは上述のように全てのサイズのフォント描画に [ClearType](../Page/ClearType.md "wikilink") を使用し、ビットマップを有していない\[7\]が、 Windows の日本語フォントとしては初めての試みである。[MS ゴシックなどの従来から提供されているフォントは](../Page/MS_ゴシック.md "wikilink")、画面上での[可読性](https://ja.wikipedia.org/wiki/可読性 "wikilink")を確保するために 17[pt](../Page/ポイント.md "wikilink") 未満の小さいサイズは内蔵のビットマップフォントが表示される\[8\]。メイリオでは小さなサイズもアウトラインで綺麗な表示を実現するために、膨大なヒンティングデータが盛り込まれている。このヒンティングにより、水平方向の線分が縦に並んでいる漢字も線分が間引かれて表示されるため、潰れることなく表示できる。

## Meiryo UI

[サムネイル](https://ja.wikipedia.org/wiki/ファイル:Meiryo_VS_Meiryo_UI.svg "wikilink") Windows Vista の後継 [Windows 7](../Page/Microsoft_Windows_7.md "wikilink") は新たに、[リボン UIに最適化して開発されたフォント](https://ja.wikipedia.org/wiki/リボン_\(GUI\) "wikilink")\[9\]の **Meiryo UI** を採用した

メイリオと比較して行間が狭く、ひらがなとカタカナの文字幅が MS UI Gothic に近似で、これまでの MS UI Gothic に近い形で[グラフィカルユーザインターフェースのデザインが可能となるが](https://ja.wikipedia.org/wiki/グラフィカルユーザインタフェース "wikilink")、[アルファベット](https://ja.wikipedia.org/wiki/アルファベット "wikilink")の文字幅はメイリオに等しく行間は MS UI Gothic に比して非常に広いため、 MS UI Gothic と完全な[互換性](../Page/互換性.md "wikilink")、すなわち単純にフォントを置換するだけでの置換可能性はない。[濁点](../Page/濁点.md "wikilink")や[句読点](../Page/句読点.md "wikilink")を再構成して ClearType を再調整しておりメイリオの文字幅を狭くしたものではないが、 MS UI Gothic 特有の非等幅性に起因する「不揃い」は残存する。

Windows 7 の[ユーザーインターフェイスの大部分は](../Page/ユーザインタフェース.md "wikilink")、 Vista に引き続きメイリオが用いられて全体として Meiryo UI は一部分であり、[コントロールパネル項目の一部](https://ja.wikipedia.org/wiki/コントロールパネル_\(Windows\) "wikilink")、[ペイント](../Page/Microsoft_Paint.md "wikilink")、[ワードパッド](../Page/ワードパッド.md "wikilink")など同梱[アプリケーションの一部などの](../Page/アプリケーションソフトウェア.md "wikilink") Vista で MS UI Gothic を用いた要素は Meiryo UI で、[ダイアログボックス](../Page/ダイアログボックス.md "wikilink")などは MS UI Gothic が残る。

[2009年](../Page/2009年.md "wikilink")11月17日にマイクロソフトが Windows Vista/Server 2008 向けに Meiryo UI を含んだメイリオ フォント コレクションのアップデートを公開\[10\]し、全ての Windows Vista ユーザーにインストールを推奨している。

[Windows 8](https://ja.wikipedia.org/wiki/Microsoft_Windows_8 "wikilink") 日本語版のユーザインタフェースは全面的に Meiryo UI が採用された。
