**ジャン・ピアジェ**（, [1896年](../Page/1896年.md "wikilink")[8月9日](../Page/8月9日.md "wikilink") - [1980年](https://ja.wikipedia.org/wiki/1980年 "wikilink")[9月16日](../Page/9月16日.md "wikilink")）は、[スイス](../Page/スイス.md "wikilink")の[心理学者](../Page/心理学者.md "wikilink")。[20世紀](../Page/20世紀.md "wikilink")において最も影響力の大きかった心理学者の一人。

知の[個体発生](https://ja.wikipedia.org/wiki/個体発生 "wikilink")としての[認知発達](https://ja.wikipedia.org/wiki/認知発達 "wikilink")と、知の[系統発生](https://ja.wikipedia.org/wiki/系統発生 "wikilink")としての[科学史](../Page/科学史.md "wikilink")を重ね合わせて考察する[発生的認識論](https://ja.wikipedia.org/wiki/発生的認識論 "wikilink")（genetic epistemology）を提唱した。[発達心理学](../Page/発達心理学.md "wikilink")者としては、「質問」と「診断」からの臨床的研究の手法を確立した。子どもの[言語](../Page/言語.md "wikilink")、[世界観](../Page/世界観.md "wikilink")、因果関係、数や量の概念などの研究を展開した。

## 来歴

[スイス](../Page/スイス.md "wikilink")の[フランス語圏](../Page/フランス語圏.md "wikilink")、[ヌーシャテル](../Page/ヌーシャテル.md "wikilink")に生まれる。父親のアルテュール・ピアジェは、[ヌーシャテル大学](https://ja.wikipedia.org/wiki/ヌーシャテル大学 "wikilink")の[中世文献学](https://ja.wikipedia.org/wiki/中世文献学 "wikilink")の[教授](../Page/教授.md "wikilink")で、ヌーシャテルの[地方史](https://ja.wikipedia.org/wiki/地方史 "wikilink")を専門としていた。母親は信仰に厚く、[プロテスタンティズムの雰囲気の中で彼を育てた](../Page/プロテスタント.md "wikilink")。彼は早熟で、[生物学](../Page/生物学.md "wikilink")に早くから興味を示し、特に[軟体動物](../Page/軟体動物.md "wikilink")に関心があった。

[1907年](../Page/1907年.md "wikilink")、10歳で[白](../Page/アルビノ.md "wikilink")[スズメ](../Page/スズメ.md "wikilink")（[仏語](https://ja.wikipedia.org/wiki/仏語 "wikilink"):moineau albinos）についての観察を[論文](../Page/論文.md "wikilink")にまとめ、「ヌーシャテル博物学雑誌」に発表。1頁だけのものであったが、[ヌーシャテル自然史博物館](https://ja.wikipedia.org/wiki/ヌーシャテル自然史博物館 "wikilink")の[館長](https://ja.wikipedia.org/wiki/館長 "wikilink")の[ポール・ゴデー](https://ja.wikipedia.org/wiki/ポール・ゴデー "wikilink")にそれを認められ、週2回は彼の元で学校の放課後非常勤の[助手](https://ja.wikipedia.org/wiki/助手 "wikilink")を勤める機会を与えられた。ゴデーは[1911年](../Page/1911年.md "wikilink")に死去するが、ピアジェは、スイス、フランス、プロイセンの[雑誌](../Page/雑誌.md "wikilink")に軟体動物についての論文を発表。次第に生物学が[認識論](../Page/認識論.md "wikilink")の諸問題を解決する糸口を持っているのではないかという考えを持つようになる。

19歳で、ヌーシャテル大学[動物学科](https://ja.wikipedia.org/wiki/動物学科 "wikilink")を卒業、「ヴァレの軟体動物学序説」（[1921年](../Page/1921年.md "wikilink")刊行）で理学博士号を取得。その後、生物学と認識論を結びつける接点としての[心理学](../Page/心理学.md "wikilink")に関心を移し、[ローザンヌ大学](../Page/ローザンヌ大学.md "wikilink")、[チューリッヒ大学](https://ja.wikipedia.org/wiki/チューリッヒ大学 "wikilink")、[パリ大学](../Page/パリ大学.md "wikilink")で心理学を学んだ。

[ジャン・ジャック・ルソー研究所](https://ja.wikipedia.org/wiki/ジャン・ジャック・ルソー研究所 "wikilink")の心理学研究主任を皮切りに、ヌーシャテル大学、[ジュネーヴ大学](../Page/ジュネーヴ大学.md "wikilink")、ローザンヌ大学で教鞭をとり、パリ大学では[モーリス・メルロー＝ポンティ](../Page/モーリス・メルロー＝ポンティ.md "wikilink")の後任として[児童心理学](https://ja.wikipedia.org/wiki/児童心理学 "wikilink")講座の教授を務めた。

[1955年](../Page/1955年.md "wikilink")、発生的認識論国際センターを[ジュネーヴ](../Page/ジュネーヴ.md "wikilink")に設立し、世界中のさまざまな分野の研究者たちとの共同研究を晩年まで精力的に行なった。共同研究者の中には[科学哲学](../Page/科学哲学.md "wikilink")のマリオ・ブンゲ、[トーマス・クーン](../Page/トーマス・クーン.md "wikilink")、[人工知能](../Page/人工知能.md "wikilink")研究の[シーモア・パパート](../Page/シーモア・パパート.md "wikilink")らがいる。

[1972年](../Page/1972年.md "wikilink")、[エラスムス賞](../Page/エラスムス賞.md "wikilink")を受賞した。

## 思考発達段階説

| \!段階 | 年齢      | 特徴    |
| ---- | ------- | ----- |
| 1    | 感覚運動段階  | 0～2   |
| 2    | 前操作段階   | 2～7   |
| 3    | 具体的操作段階 | 7～12  |
| 4    | 形式的操作段階 | 12歳以降 |
|      |         |       |

　この発達段階は突然変異するのではなく、徐々に変わるものである。 また「赤ちゃん返り」といった、逆戻りなんかもある。 加えて個人差も大きく、同じ子供でも分野別に早い・遅いがある。 ※そういった意味で、あくまで参考程度に留めた方が良い。

　また７歳は小学校入学、１２歳は中学校進学の影響がある。 つまり小中学校という環境であったり、親・世間からの期待（重圧）も大きくなる。 だからこそ、この年齢での変化が大きいだけと言われている。
