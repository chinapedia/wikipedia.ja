**熱的残存粒子** (ねつてきざんぞんりゅうし, thermal relic) とは、[現代宇宙論](https://ja.wikipedia.org/wiki/現代宇宙論 "wikilink")において、[初期宇宙](https://ja.wikipedia.org/wiki/初期宇宙 "wikilink")においては[熱平衡](https://ja.wikipedia.org/wiki/熱平衡 "wikilink")にあったものの、[宇宙の歴史のある時点において相互作用が実効的に無視できるようになり](../Page/宇宙の年表.md "wikilink")、それ以降自由粒子として振る舞うような粒子の総称である\[1\]\[2\]。「残存物」ともいう\[3\]。相互作用が無視できるようになった時点でその粒子が相対論的であるか非相対論的であるかに応じて「熱い」残存粒子または「冷たい」残存粒子と呼ばれる。例えば[宇宙ニュートリノ背景](https://ja.wikipedia.org/wiki/宇宙ニュートリノ背景 "wikilink")は熱い残存粒子であり、[暗黒物質](../Page/暗黒物質.md "wikilink")の候補である[WIMPは冷たい残存粒子モデルに基づくものである](https://ja.wikipedia.org/wiki/WIMP_\(暗黒物質\) "wikilink")。

## 概要

ある粒子 \(\chi\) が自身の反粒子 \(\bar{\chi}\) との[対消滅](https://ja.wikipedia.org/wiki/対消滅 "wikilink")チャンネルを持つ場合を考える。この対消滅反応の反応率は反粒子の数密度 \(n_\bar{\chi}\)、対消滅断面積 \(\sigma\)、粒子速度 \(v\) の積

\[P = n_\bar{\chi} \sigma v\] である\[4\]。高温高密度状態にある[初期宇宙](https://ja.wikipedia.org/wiki/初期宇宙 "wikilink")においてはこの反応が頻繁に起き、粒子 \(\chi\) はこの反応を通じて熱平衡にある。やがて宇宙が膨張し温度と数密度が下がると、反応率 \(P\) が減少し宇宙の膨張率（[ハッブルパラメータ](https://ja.wikipedia.org/wiki/ハッブル=ルメートルの法則#ハッブルパラメータの変化 "wikilink")）\(H\) を下回る。これにより対生成・対消滅反応が実効的に停止し（これを「脱結合 (decoupling)」と呼ぶ）、粒子 \(\chi\) の存在量が変化しなくなる\[5\]。その後この粒子が別の過程により崩壊しない限り、このようにして初期に熱平衡にあった粒子は現在の宇宙にも一定量が存在することになる。これが熱的残存粒子である。

例えば[ニュートリノ](../Page/ニュートリノ.md "wikilink")は[弱い相互作用](https://ja.wikipedia.org/wiki/弱い相互作用 "wikilink")により初期宇宙においては[レプトン](https://ja.wikipedia.org/wiki/レプトン "wikilink")（特に[電子](https://ja.wikipedia.org/wiki/電子 "wikilink")・[陽電子](../Page/陽電子.md "wikilink")）と相互作用し熱平衡にあるが、およそ温度 1.5 MeV で脱結合し、その後は自由粒子として振る舞う\[6\]。このように初期宇宙において熱的に生成されたニュートリノは現在の宇宙でも[宇宙ニュートリノ背景](https://ja.wikipedia.org/wiki/宇宙ニュートリノ背景 "wikilink")として存在し続けていると考えられており、間接的な証拠はあるものの、その数密度は一世代あたり112個/cm<sup>3</sup>と小さく\[7\]、またエネルギーが小さいため、2016年現在それを実験的に検出することはできていない\[8\]。

あるいは、熱的残存粒子は[暗黒物質](../Page/暗黒物質.md "wikilink")を生成するシナリオのひとつとして有力視されている。特に相互作用断面積や質量として[電弱相互作用](../Page/電弱相互作用.md "wikilink")から予想される値を仮定するとき、冷たい残存粒子シナリオはその粒子の現在の量として暗黒物質の存在量を再現するため、この種の粒子（[WIMPとして知られる](https://ja.wikipedia.org/wiki/WIMP_\(暗黒物質\) "wikilink")）は特に有力な暗黒物質の候補となっている\[9\]。

## 熱的残存粒子の存在量

本節では脱結合後の熱的残存粒子の存在量がどのように決定されるのかを述べる。なお通常の熱的残存粒子シナリオでは問題の粒子 \(\chi\) とその反粒子 \(\bar{\chi}\) の性質は同じであると仮定する。本記事もこれを仮定するが、[バリオン数生成](../Page/バリオン数生成.md "wikilink")の理論に動機づけられて粒子・反粒子間に非対称性を導入するモデルもあり、これは[Asymmetric dark matterとして知られている](https://ja.wikipedia.org/wiki/Asymmetric_dark_matter "wikilink")\[10\]。

### ボルツマン方程式

問題の粒子 \(\chi\) はその反粒子 \(\bar{\chi}\) との[対生成](../Page/対生成.md "wikilink")・[対消滅](https://ja.wikipedia.org/wiki/対消滅 "wikilink")チャンネルを持つものと仮定する\[11\]。

\[\chi + \bar{\chi} \rightleftharpoons X + \bar{X}\] ただしここではすべてのチャンネルで反応先の粒子 \(X\), \(\bar{X}\) は輻射温度 \(T\) に等しい熱平衡にあるものとする\[12\]。

この系は一般相対論的な[ボルツマン方程式](../Page/ボルツマン方程式.md "wikilink")によって記述されるが、それを積分することにより粒子数密度 \(n_\chi\) の時間変化を記述する次の方程式が導かれる\[13\]\[14\]\[15\]。

\[\frac{ d n_\chi }{ d t } + 3 H n_\chi = - \langle \sigma v \rangle \left[ n_\chi^2 - \left( n_\chi^\mathrm{EQ} \right)^2 \right]\] 左辺第2項は宇宙膨張による希釈を、右辺は対生成・対消滅による粒子数密度の変化を表す。なお右辺の \(\langle \sigma v \rangle\) は対消滅断面積と速度の積を熱平均したものであり\[16\]、\(n_\chi^\mathrm{EQ}\) は熱平衡時の数密度

\[n_\chi^\mathrm{EQ} = g_\chi \int \frac{ 1 }{ e^{\beta \varepsilon_\mathbf{p}} \mp 1} \frac{ d^3 p }{ ( 2 \pi \hbar )^3 } = g_\chi \frac{ \zeta ( 3 ) k_\mathrm{B}^3 }{ \pi^2 \hbar^3 c^3 } T^3 \times \frac{ 1 }{ 2 \zeta ( 3 ) } \int_0^\infty \frac{ y^2 }{ \exp \sqrt{ x^2 + y^2 } \mp 1 } dy\] である。後者は粒子 \(\chi\) が相対論的であるときには \(n_\chi = g'_\chi \frac{ \zeta (3) k_\mathrm{B}^3 }{ \pi^2 \hbar^3 c^3 } T^3\), 非相対論的であるときには \(n_\chi = g_\chi \left( \frac{ m_\chi k_\mathrm{B} T }{ 2 \pi \hbar^2 } \right)^\frac{3}{2} e^{- \beta m_\chi c^2}\) と書ける。ここに \(\zeta ( s )\) は[リーマンゼータ関数](../Page/リーマンゼータ関数.md "wikilink")（\(\zeta ( 3 )\) は[アペリーの定数](../Page/アペリーの定数.md "wikilink")）、\(g'_\chi\) は粒子 χ が[ボース粒子](../Page/ボース粒子.md "wikilink")であるときその内部自由度 \(g_\chi\)、[フェルミ粒子](../Page/フェルミ粒子.md "wikilink")であるとき \(\frac{ 3 }{ 4 } g_\chi\) である。

独立変数として時刻 \(t\) の代わりに温度 \(T\) を用いる。ただし温度は宇宙膨張とともに減少するため、粒子 χ の質量 \(m_\chi\) で無次元化した変数

\[x := \frac{ m_\chi c^2 }{ k_\mathrm{B} T }\] を導入する\[17\]\[18\]。このとき、\(x \lesssim 1\) のとき粒子 \(\chi\) は相対論的であり、\(x \gg 1\) のとき非相対論的である\[19\]\[20\]。さらに、宇宙膨張による希釈の効果には興味がないため、（あらゆる粒子の寄与を考慮した）エントロピー密度 \(s\) で規格化した、単位エントロピーあたりの粒子数

\[Y := \frac{ n_\chi }{ s }\] を用いる\[21\]\[22\]。このとき、上の数密度に関する方程式は次のように書き直せる\[23\]\[24\]。

\[\frac{ d Y }{ d x } = - \sqrt{ \frac{ \pi }{ 45 } } \frac{ g_{* S} }{ \sqrt{ g_* } } \frac{ \langle \sigma v \rangle }{ x^2 } \left[ Y^2 - Y_\mathrm{EQ}^2 \right]\] ここに \(g_*\), \(g_{* S}\) は相対論的な粒子の[有効自由度である](../Page/有効自由度_\(宇宙論\).md "wikilink")。

### 熱い残存粒子

時刻 \(x_f \lesssim 1\) にこの粒子が脱結合したとすると、それ以降 \(Y ( x) = n_\chi ( x ) / s ( x )\) は近似的に保存する（これを「凍結 (freeze out)」と表現する\[25\]）。そこで現在の宇宙での値と脱結合の瞬間での値を等置する。

\[\frac{ n_\chi ( x_f ) }{ s ( x_f ) } = \frac{ n_{\chi 0} }{ s_0 }\] 相対論的な粒子の数密度の表式 \(n_\chi = g'_\chi \frac{ \zeta (3) k_\mathrm{B}^3 }{ \pi^2 \hbar^3 c^3 } T^3\) およびエントロピー密度の表式 \(s ( x ) = g_{* S} ( x ) \frac{ 2 \pi^2 k_\mathrm{B}^4 }{ 45 \hbar^3 c^3 } T^3\) を代入すると、左辺の値は \(Y_f := \frac{ 45 \zeta ( 3 ) }{ 2 \pi^4 k_\mathrm{B} } \frac{ g'_\chi }{ g_{* S} ( x_f ) }\) と評価される\[26\]。

現在の宇宙では粒子 \(\chi\) は非相対論的であると仮定すると、その現在のエネルギー密度は

\[\rho_{\chi 0} c^2 = m_\chi c^2 n_{\chi 0} = m_\chi c^2 s_0 Y_f\] と書ける。従って、粒子 \(\chi\) の密度パラメータ \(\Omega_{\chi 0}\) は

\[\Omega_{\chi 0} = 2 \times \frac{ 8 \pi G \rho_{\chi 0} }{ 3 H_0^2 } = \frac{ 16 \zeta ( 3 ) }{ 3 \pi } \frac{ G m_\chi }{ \hbar^3 c^3 } g_{* S 0} \frac{ ( k_\mathrm{B} T_0 )^3 }{ H_0^2 } \frac{ g' }{ g_{* S} ( x_f ) }\] により与えられる\[27\]。なお係数 2 は反粒子 \(\bar{\chi}\) の寄与を考慮することを表す。[CMB温度](../Page/宇宙マイクロ波背景放射.md "wikilink") \(T_0\) および[ハッブル定数](https://ja.wikipedia.org/wiki/ハッブル定数 "wikilink") \(H_0\) の観測値を代入すると

\[\Omega_{\chi 0} = 0.15 h^{-2} \frac{ g' }{ g_{* S} ( x_f ) } \left( \frac{ m_\chi c^2 }{ 1\,\mathrm{eV} } \right)\] という結論を得る\[28\]。

### 冷たい残存粒子

[Cold_relic_abundance.svg](https://ja.wikipedia.org/wiki/File:Cold_relic_abundance.svg "fig:Cold_relic_abundance.svg") 粒子 \(\chi\) が温度 \(T\) の熱平衡にあるとき、それが非相対論的である (\(k_\mathrm{B} T \ll m_\chi c^2\)) ならば、その数密度 \(n_\chi\) は

\[n_\chi = g_\chi \left( \frac{ m_\chi k_\mathrm{B} T }{ 2 \pi \hbar^2 } \right)^\frac{3}{2} e^{- \beta m_\chi c^2}\] と書ける。従って熱平衡にあるときの \(Y\) の値は

\[Y_\mathrm{EQ} ( x ) = \frac{ 45 }{ 4 \sqrt{ 2 } \pi^{7/2} } \frac{ g_\chi }{ g_{* S} ( x ) } x^\frac{3}{2} e^{-x}\] と表示できる\[29\]。この場合の最終的な粒子 \(\chi\) の存在量は \(Y\) に関する方程式を数値的に解くことによって求められる。

しばしば \(\langle \sigma v \rangle\) が温度のべき乗の依存性を持つと仮定される\[30\]\[31\]: \(\langle \sigma v \rangle \propto T^n\)。この場合、定数 \(\lambda\) を

\[\lambda := \left. \frac{ s }{ H } \langle \sigma v \rangle \right|_{x = 1}\] により定義すると、\(Y\) に関する方程式は

\[\frac{ d Y }{ d x } = - \frac{ \lambda }{ x^{n+2} } \left[ Y^2 - Y_\mathrm{EQ}^2 \right]\] と書き直せる\[32\]。いくつかの \(\lambda\) に対する数値解を図に示す。ここからわかるように、初期には解 \(Y ( x )\) は熱平衡の場合の値 \(Y_\mathrm{EQ}\) に一致するが、ある時刻 \(x_f\) でそこから逸脱する。この時刻はガモフの基準により方程式

\[x_f^{n - \frac{1}{2}} e^{x_f} = \frac{ 45 }{ 4 \sqrt{ 2 } \pi^{7/2} } \frac{ g_\chi }{ g_{* S} } \lambda\] を満足する値 \(x_f\) として概算できる\[33\]。その後、対生成・対消滅反応が停止し、\(Y\) は最終的な値 \(Y ( \infty )\) に「凍結」する。その値は

\[Y ( \infty ) \sim Y_\mathrm{EQ} ( x_f ) = \frac{ x_f^{n+1} }{ \lambda }\] である\[34\]。

現在の宇宙における冷たい残存粒子の密度パラメータ \(\Omega_{\chi 0}\) は次のように求まる\[35\]。

\[\Omega_{\chi 0} = 2 \times \frac{ 8 \pi G }{ 3 H_0^2 } m_\chi s_0 Y ( \infty ) = \frac{ 32 \pi^{5/2} }{ 9 \sqrt{5} m_\mathrm{Pl}^3 } \frac{ g_{* S 0} T_0^3 }{ H_0^2 } \frac{ \sqrt{ g_* } }{ g_{* S} \langle \sigma v \rangle_1 } x_f^{n+1}\]

## 観測的制限

### 熱い残存粒子

熱い残存粒子が現在の宇宙に非相対論的な粒子として存在するならば、それは[ホットダークマター](https://ja.wikipedia.org/wiki/ホットダークマター "wikilink")として知られるタイプの[暗黒物質](../Page/暗黒物質.md "wikilink")として振る舞うが、その存在は宇宙論的観測によって棄却されている\[36\]。そのため、熱い残存粒子の密度パラメータは現在の暗黒物質の量 \(\Omega_{c 0} = 0.2582\) に比べて十分小さい必要があり、このことから熱い残存粒子の質量 \(m_\chi\) に上限が与えられる。例えば[ニュートリノ](../Page/ニュートリノ.md "wikilink")の場合、この考察から \(m_\nu \ll 10\,\mathrm{eV}\) という制限が得られる\[37\]。ただしこの種の制限は \(g_* ( x_f )\) の値を通じて断面積の大きさに依存し、断面積が小さく \(g_* ( x_f ) \geq 106.75\) で凍結する場合には熱い残存粒子の質量の上限は \(10\,\mathrm{eV}\) 程度まで緩和される\[38\]。

### 冷たい残存粒子

冷たい残存粒子は[コールドダークマター](https://ja.wikipedia.org/wiki/コールドダークマター "wikilink")と呼ばれるタイプの暗黒物質となる\[39\]。冷たい残存粒子の存在量は強く対消滅断面積に依存するため、質量に関する制限もまたその相互作用モデルに強く依存する\[40\]が、宇宙論的に意味のある量の残存粒子が生成されるためには断面積は極めて小さい必要がある\[41\]。一般論としては、理論の[ユニタリー性により相互作用断面積は粒子質量と](https://ja.wikipedia.org/wiki/ユニタリー性_\(物理学\) "wikilink")

\[\sigma \lesssim \frac{ 4 \pi }{ m_\chi^2 }\] という関係にあるため、\(\Omega_\chi \lesssim 0.2\) は \(m_\chi \lesssim 120\,\mathrm{TeV}\) を要求する\[42\]。

具体的に相互作用断面積およびその質量として[電弱相互作用](../Page/電弱相互作用.md "wikilink")から示唆される値を用いるとき、冷たい残存粒子シナリオが予測する暗黒物質量は現在の観測値を「奇跡的」に再現する\[43\]。この事実は「WIMPの奇跡\[44\] (WIMP miracle)」として知られており\[45\]、暗黒物質が weakly interacting massive particle ([WIMP](https://ja.wikipedia.org/wiki/WIMP_\(暗黒物質\) "wikilink")) と呼ばれる種類の素粒子であると考える根拠のひとつとなっている。
