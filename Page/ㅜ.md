呼称は**ウ**（<span lang="ko">우</span>）。21番目の字母（母音字母としては7番目）である。始めから終わりまで音色の変わらない単母音を表す。

## 音声

[音素](../Page/音素.md "wikilink")はで表される。唇をすぼめて舌を後ろに引いた[円唇後舌狭母音](https://ja.wikipedia.org/wiki/円唇後舌狭母音 "wikilink")で発音される。<span lang="ko>/[[ㅡ|ㅡ]]/</span>の唇が平らであるのに対して唇が十分な丸みを帯びる。

== 沿革 ==
『[[訓民正音|訓民正音]]』の中声字において地を象る「<span lang="ko>[ㅡ](../Page/ㅡ.md "wikilink")</span>」と天を象る「<span lang="ko">[ㆍ](../Page/ㆍ.md "wikilink")</span>」とによって構成され、天を表す「<span lang="ko">ㆍ</span>」が下側に置かれているので地から始まる陰母音の初出字に分類される。

その名称は『[訓蒙字会](https://ja.wikipedia.org/wiki/訓蒙字会 "wikilink")』（[1527年](../Page/1527年.md "wikilink")）でウ（午）とされた。

## 造字

  - 同じ陰母音の初出字と組み合わさって<span lang="ko">[ㅝ](https://ja.wikipedia.org/wiki/ㅝ "wikilink")</span> （または）が作られる（<span lang="ko">ㅜ</span> + <span lang="ko">[ㅓ](../Page/ㅓ.md "wikilink")</span>）。
  - 中性母音の字母と組み合わさって<span lang="ko">[ㅟ](https://ja.wikipedia.org/wiki/ㅟ "wikilink")</span> （元来は)が作られる（<span lang="ko">ㅜ</span> + <span lang="ko">[ㅣ](../Page/ㅣ.md "wikilink")</span>）。
  - 上記の2つの字母と組み合わさって<span lang="ko">[ㅞ](https://ja.wikipedia.org/wiki/ㅞ "wikilink")</span> が作られる（<span lang="ko">ㅜ</span> + <span lang="ko">ㅓ</span> + <span lang="ko">ㅣ</span>）。

## ラテン文字転写

[文化観光部2000年式](../Page/文化観光部2000年式.md "wikilink")、[マッキューン＝ライシャワー式](../Page/マッキューン＝ライシャワー式.md "wikilink")共に**u**と表記される。

企業名、人名、団体名などでは**oo**と表記されることもある。また、子音無しの우の場合はwを補って**woo**とすることが多い（例：[대우（大宇）](../Page/大宇財閥.md "wikilink")→Daewoo）。
