本記事では、[2022 FIFAワールドカップの](https://ja.wikipedia.org/wiki/2022_FIFAワールドカップ "wikilink")[アフリカ予選のうち](https://ja.wikipedia.org/wiki/2022_FIFAワールドカップ・アフリカ予選 "wikilink")**2次予選**（にじよせん）の詳細について示す。

[アフリカサッカー連盟](../Page/アフリカサッカー連盟.md "wikilink")（CAF）所属のナショナルチームにより競われる。本大会出場枠は5\[1\]。2020年10月5日から2021年10月4日にかけて実施予定である。

## フォーマット

1次予選免除の26チームおよび、 [1次予選の勝者](https://ja.wikipedia.org/wiki/2022_FIFAワールドカップ・アフリカ1次予選 "wikilink")14チームの合計40チームを4チームずつの10組に分け、ホーム・アンド・アウェーの2順の総当たり戦を行う。各組1位チームが3次予選に進出する。

## 競技日程

以下の競技日程は、FIFAが定めた国際Aマッチデー計画表に基づく\[2\]。[アフリカネイションズカップ2021](https://ja.wikipedia.org/wiki/アフリカネイションズカップ2021 "wikilink")本大会の日程が6・7月から1・2月開催に変更されたことを受け、ワールドカップ予選第1節および第2節の日程が変更された。\[3\]

| 節   | 日付                                  |
| --- | ----------------------------------- |
| 第1節 | 2020年10月5日-13日 (元々、 2020年3月23日-31日) |
| 第2節 | 2020年11月9日-17日 (元々、 2020年6月1日-9日)   |
| 第3節 | 2021年3月22日-30日                      |
| 第4節 |                                     |
| 第5節 | 2021年8月30日-9月7日                     |
| 第6節 | 2021年10月4日-12日                      |

## シード順

組み合わせ抽選会は、2020年1月21日19:00([UTC+2](../Page/UTC+2.md "wikilink"))にエジプト・[カイロ](../Page/カイロ.md "wikilink")のザ・リッツ・カールトン・ナイルで実施した\[4\]。

シード順は、2019年11月発表の[FIFAランキング](../Page/FIFAランキング.md "wikilink")に基づき、参加40チームを4つのポットに振り分けた

  - ポット1 は1位から10位。
  - ポット2 は11位から20位。
  - ポット3 は21位から30位。
  - ポット4 は31位から40位。
