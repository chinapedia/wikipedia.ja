**弦楽四重奏曲第1番 ト短調** [作品13は](../Page/作品番号.md "wikilink")、[カール・ニールセン](../Page/カール・ニールセン.md "wikilink")が作曲した[弦楽四重奏曲](../Page/弦楽四重奏曲.md "wikilink")。1889年12月18日に[コペンハーゲン](../Page/コペンハーゲン.md "wikilink")で私的に初演された。ニールセンが公表した4作品の弦楽四重奏曲のうち最初の作品である\[1\]。

## 概要

書き上げられたのは1889年であったが、その後1898年2月3日に公開初演を迎えるにあたり少々改訂が加えられた。特にニールセンは終楽章に「概要」（Résumé）と題される個所を追加し、第1、第3、第4楽章の主題をその部分にまとめている。演奏は[アントン・スヴェンスン](https://ja.wikipedia.org/wiki/アントン・スヴェンスン "wikilink")に率いられてコペンハーゲンにあるOdd Fellows Mansionの小ホールで行われた。ニールセンの作品のみを集めた演奏会は急な告知だったにもかかわらず大入りとなった。当日のプログラムはその他にピアノのための『ユモレスク・バガテル』、『交響的組曲』 作品8、ヴァイオリンソナタ第1番 イ長調 作品9であった\[2\] 。

曲は1900年に出版され、60歳の誕生日を迎えた[ヨハン・スヴェンセン](../Page/ヨハン・スヴェンセン.md "wikilink")へと献呈されている\[3\]\[4\]。

## 演奏時間

約26分\[5\]。

## 楽曲構成

### 第1楽章

  - [Allegretto](https://ja.wikipedia.org/wiki/wikt:allegretto "wikilink") [energico](https://ja.wikipedia.org/wiki/wikt:energico "wikilink") 4/4[拍子](../Page/拍子.md "wikilink") ト短調

冒頭から緊張感の高い第1主題が奏でられる（譜例1）。

譜例1 <score> \\relative c' {

`\key g \minor \time 4/4 \set Score.tempoHideNote = ##t \tempo "Allegretto energico." 4=132`
`r8 d\f ( g4~ g8) fis( d4~ d8 g) g( a) a( bes) bes( cis( ~ `
`cis) d( g4~ g8) fis( d4~ d8 g) g( a) a( bes) bes( cis) d4( a2)`

} </score>

間もなく16分音符の急速な動きによる経過句に入り、その小刻みな動きを残したまま[チェロ](../Page/チェロ.md "wikilink")に第2主題が出される（譜例2）。

譜例2 <score> \\relative c' \\new Staff \\with { \\remove "Time_signature_engraver" } {

`\key g \minor \time 4/4 \set Score.tempoHideNote = ##t \tempo "" 4=132 \clef bass`
`d4.\p( es8) f2~ f4 es8-- d-- c4( cis) d4.( es8) f2~ f4 es8-- d-- c4( cis)`

} </score>

譜例2が熱を帯びて頂点を迎えた後に譜例1を使った[コーダへと静まっていき](https://ja.wikipedia.org/wiki/コーダ_\(音楽\) "wikilink")、提示部を反復する。両主題を用いて充実した展開が行われ、譜例1が力強く奏されると再現部となる。提示部とは異なる経過をたどり、第2主題は第1ヴァイオリンにより再現される\[6\]。最後は[三連符が絶え間なく聞こえる中に静まっていく](https://ja.wikipedia.org/wiki/音符#連符（連音符） "wikilink")。

### 第2楽章

  - [Andante](https://ja.wikipedia.org/wiki/wikt:andante "wikilink") [amoroso](https://ja.wikipedia.org/wiki/wikt:amoroso "wikilink") 9/8拍子 [変ホ長調](../Page/変ホ長調.md "wikilink")

穏やかな譜例3の主題に開始する。

譜例3 <score> \\relative c'' {

`\key es \major \time 9/8 \set Score.tempoHideNote = ##t \tempo "Andante amoroso." 8=120`
`bes4.\p ( ~ bes8 aes g) c4( aes8) bes2. g4( f8) es4.( ~ es8 d c) f4( d8) bes2.~ bes8 c( d)`

} </score>

譜例3がおおらかに歌われてクライマックスを築いたのち、速度を上げて3/4拍子に転じ、[アジタートの中間部となる](https://ja.wikipedia.org/wiki/wikt:agitato "wikilink")（譜例4）。

譜例4 <score> \\relative c'' {

`\key g \minor \time 3/4 \set Score.tempoHideNote = ##t \tempo "Agitato." 4=130`
`g2\mf( a8 bes) c4.( b8 bes a) d2\< d4\! cis4.\fz g8\> ( bes g\! ) c!2\< c4\! b4.\fz f8\> ( aes f\! )  `

} </score>

やがて譜例3が16分音符の[トレモロ](../Page/トレモロ.md "wikilink")音型を伴って回帰し、盛り上がったのちに静まっていき[ピアニッシッシモ](../Page/強弱法.md "wikilink")（***ppp***）の終わりを迎える。

### 第3楽章

  - [Schrezo](../Page/スケルツォ.md "wikilink"). [Allegro](https://ja.wikipedia.org/wiki/wikt:allegro "wikilink") [molto](https://ja.wikipedia.org/wiki/wikt:molto "wikilink") 6/8拍子 [ハ短調](https://ja.wikipedia.org/wiki/ハ短調 "wikilink")

譜例5で勢いよく開始する。楽想が突如停止してチェロのアクセントが挿入される。

譜例5 <score> \\relative c'' {

`\key c \minor \time 6/8 \set Score.tempoHideNote = ##t \tempo "Allegro molto." 8=150`
`\appoggiatura { es16 f } g8.-.\ff [ f16-. es8-.] f-. es-. d-. es-. d-. c-. d-. b-. g-. c4.( bes!) aes( g)`

} </score>

中間楽節を挟んで譜例5を繰り返し、[ト長調](https://ja.wikipedia.org/wiki/ト長調 "wikilink")の[トリオとなる](https://ja.wikipedia.org/wiki/複合三部形式 "wikilink")。特徴的なチェロの導入に始まるトリオは2つのヴァイオリンが対位法的に先導していく（譜例6）。この楽章、とりわけこのトリオからはニールセンの幼少期の音楽体験が垣間見える\[7\]。

譜例6 <score> \\new StaffGroup \<\<

`\new Staff { \relative c'' { \key g \major \time 2/4 \tempo "Trio."`
` d8\p( [ \acciaccatura fis e d g] ) fis( [ d e c] d[ b) \appoggiatura { c16 d } c8( a] d4\open ) d( ~ d8[ e\open d e] ) d( [ e d e] )`
`} }`
`\new Staff { \relative c'' { \key g \major \time 2/4`
` R2 R R g8\p ( [ \acciaccatura b a g b] ) a( [ fis g e] fis[ d) \appoggiatura { e16 fis } e8 c]`
`  } }`

\>\> </score>

スケルツォ・[ダ・カーポとなり](../Page/ダ・カーポ_\(演奏記号\).md "wikilink")、譜例5が繰り返されるとコーダへと接続されて力強く締めくくられる。

### 第4楽章

  - Allegro (inquieto) 2/4拍子 ト短調

発想表記として掲げられるinquietoは「忙しなく」、「慌ただしく」といった意。譜例7により急き立てられるように幕が開ける。

譜例7 <score> \\relative c'' {

`\key g \minor \time 2/4 \set Score.tempoHideNote = ##t \tempo "Allegro. (inquieto.)" 4=120`
`g4\f fis8( g16 a) g4( d'-> ) es d8( es16 f) es4( bes'-> ) aes es'8( aes,16 g) fis4( c'-> ) g8( gis16 a) bes8( a16 g) d'-> d d d d,4`

} </score>

拍子が頻繁に入れ替わる経過を挟んでリラックスした主題が導入される（譜例8）。

譜例8 <score> \\relative c'' \\new Staff \\with { \\remove "Time_signature_engraver" } {

`\key g \minor \time 2/4 \set Score.tempoHideNote = ##t \tempo "" 4=120 \partial 4`
`bes8-.\p bes( c) [ c( d-.) d] ( f-.) [ f( g-.) g] ( d-.) [ d( f-.) f] ( c4--) d8-. d(`
`bes-.) [ bes( d-.) d] ( g,-.) [ g( bes-.) bes] ( ges-.) \> ges( bes4~ bes\! )`

} </score>

その後、提示部[コデッタで扱われた音型と譜例](https://ja.wikipedia.org/wiki/コーダ_\(音楽\) "wikilink")8を主に用いて大きく展開が行われていく。譜例7の再現に続いて置かれるのが「概要」（Résumé）であり、譜例1と譜例5が[対位法](../Page/対位法.md "wikilink")を用いて対置される\[8\]。曲はさらに譜例8を再現、ト長調で譜例1が奏されて堂々と全曲に幕を下ろす。

## 受容

カール・ニールセン協会によれば、本作はニールセン作品中でも屈指の人気を誇り、彼の弦楽四重奏の中では最も頻繁に演奏されているという\[9\]。
