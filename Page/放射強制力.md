[thumb](https://ja.wikipedia.org/wiki/ファイル:CO2-Temp.png "wikilink") **放射強制力**（ほうしゃきょうせいりょく）とは、[気候学](../Page/気候学.md "wikilink")における用語で、[地球](https://ja.wikipedia.org/wiki/地球 "wikilink")に出入りする[エネルギー](https://ja.wikipedia.org/wiki/エネルギー "wikilink")が地球の[気候](../Page/気候.md "wikilink")に対して持つ[放射](https://ja.wikipedia.org/wiki/放射 "wikilink")の大きさのこと。[英語](../Page/英語.md "wikilink")の"Radiative forcing"の訳語。正の放射強制力は[温暖化](https://ja.wikipedia.org/wiki/温暖化 "wikilink")、負の放射強制力は[寒冷化](https://ja.wikipedia.org/wiki/寒冷化 "wikilink")を起こす。

## 放射強制力とは

[thumbでは正の放射強制力](https://ja.wikipedia.org/wiki/ファイル:Ozone-3D-vdW.png "wikilink")、[成層圏](../Page/成層圏.md "wikilink")では負の放射強制力をもたらすとされる[オゾン](../Page/オゾン.md "wikilink")の分子模型\]\] [thumbによる](https://ja.wikipedia.org/wiki/ファイル:Radiative-forcings.svg "wikilink"))\]\] 地球の気候を左右する[気候因子](../Page/気候因子.md "wikilink")のうち、地球の気候システムによるものを除いた、[太陽放射](../Page/太陽放射.md "wikilink")や[温室効果](../Page/温室効果.md "wikilink")などを**外部因子**という。この外部因子にはそれぞれ、[光](../Page/光.md "wikilink")を吸収しやすい、光を反射しやすい、[熱](https://ja.wikipedia.org/wiki/熱 "wikilink")を吸収しやすい、熱を放出しやすいなどの性質があり、[大気](https://ja.wikipedia.org/wiki/大気 "wikilink")や[地面](https://ja.wikipedia.org/wiki/地面 "wikilink")・[海洋](https://ja.wikipedia.org/wiki/海洋 "wikilink")が蓄える熱量を左右し、[気温](../Page/気温.md "wikilink")や[海水温](../Page/海水温.md "wikilink")に影響を与える。

[気候変動に関する政府間パネル](../Page/気候変動に関する政府間パネル.md "wikilink")（IPCC）は[1994年](../Page/1994年.md "wikilink")の第1次評価報告書で、「**[対流圏](../Page/対流圏.md "wikilink")の上端（[圏界面](https://ja.wikipedia.org/wiki/圏界面 "wikilink")）における平均的な正味の放射の変化**」を放射強制力と定義し、現在もこの定義が使われている。つまり、気温が一定に保たれている状態（IPCCの予測では1750年）を基準として、地球から宇宙への放射によって地球が持つエネルギーを減らす（気温を下げる）外部因子を負の放射強制力、宇宙から地球への放射によって地球が持つエネルギーを増やす（気温を上げる）外部因子が正の放射強制力である。

[IPCC第4次評価報告書](../Page/IPCC第4次評価報告書.md "wikilink")によれば、放射強制力は、対流圏での循環バランスが取れた状態を初期状態とし、これに何らかの原因によってずれが生じたとき、成層圏の気温の変化を考慮したうえで、再び対流圏での循環バランスが取れるようになるまでに変わる放射の量として計算される。

例を挙げて言えば、バランスの取れた普通の気候が続いている中で、突然[一酸化二窒素](https://ja.wikipedia.org/wiki/一酸化二窒素 "wikilink")の濃度が2倍になったとする。すると、気温が上昇して大気の循環が乱れる（実際にはさらに複雑な変化が起きるが、ここでは単に「大気の循環が乱れる」とする）。この乱れは時間をかけてバランスの取れた気候へとなっていくが、バランスの取れた気候となったとき、対流圏の上端では以前に比べて放射の量が増えている。この増えた量が放射強制力となる。このとき、[成層圏](../Page/成層圏.md "wikilink")の気温の変化を考慮しなかった場合は**瞬間的放射強制力**と呼び区別される。ちなみに、気温が上昇している段階では地球に入るエネルギーが出るエネルギーを上回っているが、バランスの取れた気候となった時点で両者は均衡する。この過程については[温室効果](../Page/温室効果.md "wikilink")を参照のこと。

研究の進行や気候モデルの発達により、報告書ごとに少なくとも人為起源の放射強制力については、数値の信頼性は高まっているとされている。[IPCC第4次評価報告書](../Page/IPCC第4次評価報告書.md "wikilink")では、「1750 年以降の人間活動は、世界平均すると温暖化の効果を持ち、その放射強制力は+1.6\[+0.6～2.4\]Wm-2 であるとの結論の信頼性はかなり高い」とされている。 <small>注：Wm-2はW/m<sup>2</sup>と同義。</small>

## 問題点

ただ、このIPCCの定義やこの定義に基づいて推定される放射強制力の数値には、不確実性があるとされる。その根拠の1つが、大気中の[水](../Page/水.md "wikilink")の影響を取り除いているということである。IPCCの報告書では、大気中の水（水蒸気、雲、降水）は二次的な[フィードバック機構に過ぎないため対流圏中での熱の再分配を引き起こすだけであり](https://ja.wikipedia.org/wiki/気候変動#フィードバック機構 "wikilink")、放射強制力には含めないとしている。しかし、[産業革命](../Page/産業革命.md "wikilink")以前の「気温が一定に保たれている状態」においても現在においても、地球の気温を高く保っている温室効果の主因は、大気中の水とされており（約8割～9割）、大気中の水による熱の移動については現在もよく解明されていない部分が多い。しかも、人為的な温室効果ガスの排出が少ない産業革命以前にも気温は変動している（[過去の気温変化](../Page/過去の気温変化.md "wikilink")参照）。

他の根拠としては、科学的理解の水準が低いものが重要な放射強制力を持っている可能性がある、ということである。太陽放射、太陽以外の宇宙からの放射、地球内部の活動などについては分かっていない点が多く、太陽活動や地球内部の変化による地球への宇宙線量の増減なども放射強制力である可能性が指摘されている。

また、気候は常に変動するものであり、定常的な変化ではなく不規則に変化する[カオス](../Page/カオス理論.md "wikilink")（[非線形](../Page/線型性.md "wikilink")）である気候を、平均的な状態を基準として以後の変化を捉えようとする考え方自体が、科学的にあまり適切ではないという考え方もある。しかし、地球温暖化によって将来人類に起こり得る事象を予測するには、[線形の事象を対象とする](../Page/線型性.md "wikilink")[コンピュータ](../Page/コンピュータ.md "wikilink")を使用しなければならない部分があり、ある程度は仕方が無いことだと言える。

[例のような放射強制力の値は](https://ja.wikipedia.org/wiki/#さまざまな放射強制力 "wikilink")、あくまで1750年の状態と2005年の状態を比較した結果算出されたものである。それぞれの因子が持つ放射強制力は、気温や湿度などのさまざまな条件によって変わってしまう（正の放射強制力によって気温が上がることも、気温が上がることによって正の放射強制力がもたらされることもあるということ）ため、単純に「気温を1度上昇させるには、○○の放射強制力○○W/m<sup>2</sup>分が必要」などと表現することはできない。

### 主な要素と放射強制力の算出

いくつかの温室効果ガスについては、濃度変化と放射強制力変化の関係式が示されている。\(\Delta F\)が放射強制力(W/m<sup>2</sup>)にあたる。

Cは二酸化炭素の濃度(ppm)、Mはメタンの濃度(ppb)、Nは一酸化二窒素の濃度(ppb)、Xはそれぞれのフロンガスの濃度(ppb)。Cは変化後の濃度、C<sub>0</sub>は変化前の濃度であり、ほかの気体でも同様である。

  - 二酸化炭素(CO<sub>2</sub>)

IPCC (1990)およびMyhre et al. (1998)によれば

\(\Delta F = 5.35 \times \ln \left ( \frac{C}{C_0} \right )\)

Shi (1992)によれば

\(\Delta F = 4.841 \times \ln \left ( \frac{C}{C_0} \right ) + 0.0906 \times ( \sqrt{C} - \sqrt {C_0} )\)

WMO (1999)およびHansen et al. (1988)によれば

\(\Delta F = 3.35 \times ( g{C} - g{C_0} )\)

  -
    ただし、\(g ( C ) = \ln ( 1 + 1.2 C + 0.005 C^2 + 1.4 \times 10^{-6} C^3 )\)

<!-- end list -->

  - メタン(CH<sub>4</sub>)

\(\Delta F = 0.036 \times ( \sqrt{M} - \sqrt {M_0} ) - \left\{ f ( M, N_0 ) - f ( M_0, N_0 ) \right\}\)

  -
    ただし、メタンと一酸化二窒素については、\(f ( M, N ) = 0.47 \ln \left\{ 1 + 2.01 \times 10^{-5} \times ( M N )^{0.75} + 5.31 \times 10^{-15} \times M ( M N )^{1.52} \right\}\)とする。

<!-- end list -->

  - 一酸化二窒素(N<sub>2</sub>O)

\(\Delta F = 0.12 \times ( \sqrt{N} - \sqrt {N_0} ) - \left\{ f ( M_0, N ) - f ( M_0, N_0 ) \right\}\)

  - CFC-11a

\(\Delta F = 0.25 \times ( X - X_0 )\)

  - CFC-12

\(\Delta F = 0.32 \times ( X - X_0 )\)

例えば、地球大気中の二酸化炭素の平均濃度が300ppmから400ppmに上昇した場合、IPCC (1990)およびMyhre et al. (1998)の式から求めると、

\(\begin{align}
 \Delta F & = 5.35 \times \ln \left ( \frac{C}{C_0} \right ) \\
          & = 5.35 \times \ln \left ( \frac{400}{300} \right ) \\
          & = 1.539099\ldots \\
\end{align}\)

となり、約1.54W/m<sup>2</sup>の正の放射強制力が生まれる、と求められる。
