**千葉県道143号南総昭和線**（ちばけんどう143ごう なんそうしょうわせん）は、[千葉県](../Page/千葉県.md "wikilink")[市原市](../Page/市原市.md "wikilink")と[袖ケ浦市](https://ja.wikipedia.org/wiki/袖ケ浦市 "wikilink")を結ぶ[一般県道](../Page/都道府県道.md "wikilink")。

## 概要

[210px](https://ja.wikipedia.org/wiki/ファイル:Japanese_Route_297_and_Chiba_Route_143_at_Ichihara.JPG "wikilink")

  - 起点：市原市馬立（水神橋際・[国道297号](../Page/国道297号.md "wikilink")交点）
  - 終点：袖ケ浦市奈良輪（袖ケ浦市役所入口・[千葉県道87号袖ケ浦中島木更津線](../Page/千葉県道87号袖ケ浦中島木更津線.md "wikilink")交点）

## 路線状況

[平成22年](https://ja.wikipedia.org/wiki/平成22年 "wikilink")（[2010年](https://ja.wikipedia.org/wiki/2010年 "wikilink")）度時点での南総昭和線の交通量について、観測地点である袖ヶ浦市三ツ作での自動車類交通量はそれぞれ、

|                   |                 |
| ----------------- | --------------- |
| **昼間12時間自動車類交通量** | **24時間自動車類交通量** |
| style= |小型車       | style= |大型車     |
| 1844台             | 143台            |

であり、また道路部幅員10.90m、車道部幅員6.50m、車道幅員5.25m、歩道幅員は上下2.20mとなっている\[1\]。
