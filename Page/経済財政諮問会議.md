[thumb](https://ja.wikipedia.org/wiki/ファイル:Yasuo_Fukuda_20071004.jpg "wikilink")[10月4日](https://ja.wikipedia.org/wiki/10月4日 "wikilink")、[総理大臣官邸](../Page/総理大臣官邸.md "wikilink")にて）\]\] **経済財政諮問会議**（けいざいざいせいしもんかいぎ、）は、日本の[内閣府](../Page/内閣府.md "wikilink")に設置されている「[重要政策に関する会議](https://ja.wikipedia.org/wiki/重要政策に関する会議 "wikilink")」の一つである。設置根拠は[内閣府設置法](../Page/内閣府設置法.md "wikilink")第18条。[内閣総理大臣](https://ja.wikipedia.org/wiki/内閣総理大臣 "wikilink")の諮問を受けて、[経済財政政策](https://ja.wikipedia.org/wiki/経済財政政策 "wikilink")に関する重要事項について調査審議する。[橋本行革による](https://ja.wikipedia.org/wiki/行政改革会議 "wikilink")[2001年](../Page/2001年.md "wikilink")1月の[中央省庁再編](../Page/中央省庁再編.md "wikilink")によって設置された。

## 成員

会議は議長と10人以内の議員から成る。議長には内閣総理大臣が充てられ、議員としては[内閣官房長官](../Page/内閣官房長官.md "wikilink")、[内閣府特命担当大臣（経済財政政策担当）](../Page/内閣府特命担当大臣（経済財政政策担当）.md "wikilink")が法によって定められている。それ以外に、「各省大臣のうちから、内閣総理大臣が指定する者」として[財務大臣](../Page/財務大臣.md "wikilink")、[総務大臣](../Page/総務大臣.md "wikilink")、[経済産業大臣](../Page/経済産業大臣.md "wikilink")が、また「関係機関（国の行政機関を除く）の長のうちから、内閣総理大臣が任命する者」として[日本銀行](https://ja.wikipedia.org/wiki/日本銀行 "wikilink")総裁が議員となることが慣例化している。また、民間有識者数を議員の4割以上確保することが法により定められている。[民間議員](https://ja.wikipedia.org/wiki/民間議員 "wikilink")としては、これまでは財界から2名、学界から2名が選ばれている。民間議員の任期は2年間で、再任が可能。

これ以外に、会議には議案を絞って国務大臣を臨時議員として参加させることができる。また、必要に応じて審議会その他の関係行政機関の長や有識者に資料の提出、意見の開陳、説明などを求めることができる。[政府税制調査会](../Page/政府税制調査会.md "wikilink")会長や、[財政制度等審議会](https://ja.wikipedia.org/wiki/財政制度等審議会 "wikilink")会長、[規制改革・民間解放推進会議議長などの出席実績がある](https://ja.wikipedia.org/wiki/規制改革会議#規制改革・民間開放推進会議（2004年4月1日-2007年1月） "wikilink")。

### 歴代内閣の成員

内閣総理大臣を議長とする。内閣総理大臣が外遊その他で出席できないときは、内閣府特命担当大臣（経済財政政策担当）が議長代理を務める。ただし、内閣府特命担当大臣（経済財政政策担当）が置かれていないときは内閣官房長官が議長代理を務める。

## 歴史

### 橋本内閣

[橋本行革](https://ja.wikipedia.org/wiki/橋本行革 "wikilink")による[中央省庁再編](../Page/中央省庁再編.md "wikilink")の一環として[内閣府](../Page/内閣府.md "wikilink")が発足。[内閣府設置法](../Page/内閣府設置法.md "wikilink")を根拠として首相権限の強化が図られた。

### 小泉内閣

経済財政諮問会議は、従来の[大蔵省](../Page/大蔵省.md "wikilink")[主計局](../Page/主計局.md "wikilink")を主とした予算編成過程を、官邸主導型に転換する働きをしてきた。特に[小泉純一郎](https://ja.wikipedia.org/wiki/小泉純一郎 "wikilink")[首相は経済財政諮問諮問会議を最も重要な政策会議と位置づけ](https://ja.wikipedia.org/wiki/内閣総理大臣 "wikilink")、「[骨太の方針](../Page/骨太の方針.md "wikilink")」を打ち出すことによって与野党の“抵抗勢力”を退け、官邸主導の予算編成に活用した。

| 時期    | 事項                                                        | 形式               |
| ----- | --------------------------------------------------------- | ---------------- |
| 6月    | 予算編成の基本的考え方について                                           | 財政制度審議会財政制度分科会建議 |
| 6〜7月  | 経済財政運営と構造改革に関する基本方針（[骨太の方針](../Page/骨太の方針.md "wikilink")） | 閣議決定             |
| 7月    | 予算の全体像                                                    | 諮問会議とりまとめ        |
| 7月下旬  | 概算要求基準                                                    | 閣議了解             |
| 8月末   | 概算要求                                                      |                  |
| 11月下旬 | 予算編成の基本方針                                                 | 閣議決定             |
| 12月下旬 | 財務省原案                                                     | 閣議提出             |
| 予算政府案 | 閣議決定                                                      |                  |
| 1月下旬  | 政府案国会提出                                                   |                  |
| 1〜3月  | 国会審議                                                      |                  |

従来の予算編成は、8月に大蔵省が各省庁の概算要求を受けて査定し、年末に政府案として国会に提出された。政府案策定までの間に、各省と大蔵省、与党（自民党）幹部と大蔵省幹部などの折衝があり、実質的な調整が済んだ段階で政府案として提出された予算案は、大筋ではそのまま国会で可決され執行されることが多かった。この過程では、実質的な調整を担う大蔵省主計局と与党（自民党）政務調査会が、予算案策定に強い影響力を及ぼした。

小泉内閣以降の予算編成では、8月の概算要求に先立って経済財政諮問会議が経済成長率などのデータを検討したうえで「[骨太の方針](../Page/骨太の方針.md "wikilink")」を閣議決定し、予算の全体像をまず明らかにした。「骨太の方針」を踏まえて財務省が各府省の概算要求を査定する流れとなった。従来のような復活折衝は行われず、閣議決定が先行しているため与党幹部の影響力も限定的となり、予算編成への官邸の影響力が強まった。

### 鳩山内閣

2009年9月に誕生した[鳩山由紀夫内閣](https://ja.wikipedia.org/wiki/鳩山由紀夫内閣 "wikilink")は[国家戦略室を設置し](https://ja.wikipedia.org/wiki/国家戦略局#民主党による「国家戦略局」構想 "wikilink")、経済財政諮問会議は開催されなくなり、事実上活動を停止した\[1\]。

2010年2月、鳩山内閣は[政治主導確立法案](https://ja.wikipedia.org/wiki/政治主導確立法案 "wikilink")を国会に提出し、前述の国家戦略室を[国家戦略局](https://ja.wikipedia.org/wiki/国家戦略局 "wikilink")に改編した上で内閣府の経済財政諮問会議を廃止する方針を打ち出した。しかし、野党（特に自民党）の反発をうけ、2011年5月12日に[第177回国会](https://ja.wikipedia.org/wiki/第177回国会 "wikilink")において撤回が承諾され、廃案となった\[2\]。

### 第2次安倍内閣

経済財政諮問会議は2012年12月に誕生した[第2次安倍内閣](https://ja.wikipedia.org/wiki/第2次安倍内閣 "wikilink")で復活することとなった\[3\]\[4\]。

第一回は2013年1月9日に行われた\[5\]。

## 経済財政諮問会議が主導する具体的政策

  - [航空自由化](https://ja.wikipedia.org/wiki/航空自由化 "wikilink"):[国土交通省](../Page/国土交通省.md "wikilink")に対し、[アジア・ゲートウェイ構想](https://ja.wikipedia.org/wiki/アジア・ゲートウェイ構想 "wikilink")における航空自由化（アジア・オープンスカイ）の具体化を担保するための工程表（航空自由化工程表）の提出を求めており、2007年11月26日開催の平成19年第28回経済財政諮問会議に最初の工程表が提出された\[6\]。さらに、2008年5月20日開催の平成20年第12回経済財政諮問会議では、民間議員からの意見\[7\]に応じ、国土交通省から、[羽田空港の再拡張事業完了後](https://ja.wikipedia.org/wiki/東京国際空港 "wikilink")、同空港の早朝深夜帯で欧米も含めた国際線に3万回の枠を配分し、6時台・22時台の国際線発着も認めることが表明された\[8\]。

## 評価

経済財政諮問会議が設置された当時の[第2次森改造内閣では](../Page/第2次森内閣_\(改造_中央省庁再編後\).md "wikilink")、期間が短かったこともあって目立った成果をあげなかった。しかし、森内閣を引き継いで発足した[小泉内閣](https://ja.wikipedia.org/wiki/小泉内閣 "wikilink")では自民党内や官庁の反対派を**[抵抗勢力](../Page/抵抗勢力.md "wikilink")**として退け、官邸主導の政治を行う上で重要な役割を果たした。

小泉内閣の下での経済財政諮問会議の成果としては、予算編成過程の改革、金融システム改革、[郵政民営化](https://ja.wikipedia.org/wiki/郵政民営化 "wikilink")、三位一体の改革、政策金融改革、規制改革、税制改革、経済成長戦略、歳出・歳入一体改革などが挙げられる。

一方で民間メンバーの内2人は[日本経団連幹部であり](https://ja.wikipedia.org/wiki/日本経済団体連合会 "wikilink")、経団連の利害が強く反映されているのではないかという批判がある。

[第21回参議院議員通常選挙](../Page/第21回参議院議員通常選挙.md "wikilink")のマニフェストで同会議の見直しを掲げていた[国民新党](https://ja.wikipedia.org/wiki/国民新党 "wikilink")は2008年1月、“経済財政諮問会議は[格差](https://ja.wikipedia.org/wiki/格差 "wikilink")拡大を助長しており、[小泉内閣](https://ja.wikipedia.org/wiki/小泉内閣 "wikilink")の遺物である”として、民主党や社民党などと共に同会議の廃止法案を共同提出する事を決めた。
