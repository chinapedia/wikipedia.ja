[thumb](https://ja.wikipedia.org/wiki/ファイル:Granit_karkonoski.jpg "wikilink") **粒度**（りゅうど、\[1\]、\[2\]）は、複数種類の[鉱物](../Page/鉱物.md "wikilink")からなる[岩石](../Page/岩石.md "wikilink")について、主要鉱物[粒子](../Page/粒子.md "wikilink")の大きさを示す[工業量](https://ja.wikipedia.org/wiki/工業量 "wikilink")である。[堆積岩](../Page/堆積岩.md "wikilink")や[火成岩](../Page/火成岩.md "wikilink")、[変成岩](../Page/変成岩.md "wikilink")について粒度が定義できる。

なお、[土壌](../Page/土壌.md "wikilink")や破砕性[堆積物](../Page/堆積物.md "wikilink")がどのような粒度の構成粒子からなるかを示す[尺度](https://ja.wikipedia.org/wiki/尺度 "wikilink")を[粒度階区分](https://ja.wikipedia.org/wiki/粒度階区分 "wikilink")と呼ぶ。

## 定義

複数の定義が試みられている。

### クロスによる定義

ホイットマン・クロス\[3\]らは、1906年に発表した論文\[4\]において、直径ごとに下表のように定義した。

| 直径（\(\phi\)）                              | 名称 |
| ----------------------------------------- | -- |
| \(5\mathrm{mm} < \phi\)                   | 粗粒 |
| \(1\mathrm{mm} < \phi \leq 5\mathrm{mm}\) | 中粒 |
| \(0\mathrm{mm} < \phi \leq 1\mathrm{mm}\) | 細粒 |

### ハインリヒによる定義

ウィリアム・ハインリヒ\[5\]は、1956年に出版された書籍\[6\]において、直径ごとに下表のように定義した。

| 直径（\(\phi\)）                                | 名称     |
| ------------------------------------------- | ------ |
|                                             |        |
| \(30\mathrm{mm} < \phi\)                    | きわめて粗粒 |
| \(10\mathrm{mm} < \phi \leq 30\mathrm{mm}\) | 粗粒     |
| \(1\mathrm{mm} < \phi \leq 10\mathrm{mm}\)  | 中粒     |
| \(0\mathrm{mm} < \phi \leq 1\mathrm{mm}\)   | 細粒     |

## 測定

岩石を構成する鉱物は、[球形](https://ja.wikipedia.org/wiki/球形 "wikilink")とはほど遠い形状を採る場合も多く、粒度の正確な[測定](../Page/測定.md "wikilink")は簡単ではない。
