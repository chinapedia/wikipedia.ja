[thumb](https://ja.wikipedia.org/wiki/ファイル:Map_of_Korean_dialects.png "wikilink")）。薄いピンク色の領域が東南方言の使用地域である。\]\] **東南方言**（とうなんほうげん、）は、朝鮮半島の東南部で用いられている[朝鮮語の方言](../Page/朝鮮語の方言.md "wikilink")である。使用地域は[慶尚道](../Page/慶尚道.md "wikilink")地域（現在の[慶尚南道](../Page/慶尚南道.md "wikilink")・[慶尚北道](../Page/慶尚北道.md "wikilink")・[釜山広域市](../Page/釜山広域市.md "wikilink")・[大邱広域市](../Page/大邱広域市.md "wikilink")・[蔚山広域市](https://ja.wikipedia.org/wiki/蔚山広域市 "wikilink")）とほぼ一致し、**嶺南方言**（れいなんほうげん、）や**慶尚道方言**（けいしょうどうほうげん、）とも呼ばれる。

## 音韻

### 音韻体系の特徴

地域によって若干の差はあるが、概して単[母音](../Page/母音.md "wikilink")は /, , , , , / の6母音体系である。ソウル方言の // と // が // に合流し、// と // が // に合流している。// の音声はソウル方言とは異なり、中舌母音  である。// はソウル方言の若年層と同じく、 と  の中間音である。

一部の地域では鼻母音を持つ。主に ， といった[鼻音](../Page/鼻音.md "wikilink")が  と結合するときに[鼻母音](https://ja.wikipedia.org/wiki/鼻母音 "wikilink")となる。

  - ＜  （違う）

[半母音](https://ja.wikipedia.org/wiki/半母音 "wikilink")には /j/ と /w/ の2種があるが、出現環境に制約がある。とりわけ /w/ は語頭以外の位置や // 以外の母音の前ではほとんど現れない。

  - // ＜  （りんご）

[子音](../Page/子音.md "wikilink")は、濃音 // を持たない地域が多い。従って、ソウル方言の「」（買う）と「」（包む）は、ともに // と発音される。

### アクセント

東南方言は日本語に類する弁別的な高低[アクセント](../Page/アクセント.md "wikilink")を持ち、アクセントの違いにより単語の意味を区別する。

  - ― アクセントが「高低」ならば「馬が」、「低高」ならば「言葉が」の意

### 音韻変化

東南方言には以下のような特徴的な音韻変化がある。

母音の前舌化が広く見られる。母音 /, , , / が /i/ あるいは /j/ を含む音節の直前で、それぞれ /, , , / と発音される。なお、類似の前舌化現象はソウル方言でも見られる。

  - // ＜  （塞がる） cf. ソウル方言：//
  - // ＜  （食べさせる） cf. ソウル方言：//
  - // ＜  （移す） cf. ソウル方言：//
  - // ＜  （殺す） cf. ソウル方言：//

軟口蓋音の /, , / は 母音 /i/ および半母音 /j/ の前で[口蓋化](../Page/口蓋化.md "wikilink")し /, , / となる。

  - // ＜  （道）

同様に、喉頭音 // は 母音 /i/ および半母音 /j/ の前で[口蓋化](../Page/口蓋化.md "wikilink")し // となる。

  - // ＜  （力）

語頭における平音の濃音化が広く見られる。

  - // ＜  （菓子）

## 文法

以下に、東南方言に特徴的な文法的形式を中心に記述する。

### 体言

東南方言に特徴的な体言語尾には、以下のようなものがある。

  - 主格：-（ソウル方言：-）
      - ． （胸が痛い）
  - 対格：-（ソウル方言：-/-）
      - （水を飲んで）
  - 与格：-（ソウル方言：-）
      - ? （これ俺のところにあったやつじゃないの）
  - 共格：-（ソウル方言：-/-）
      - ． （俺といっしょに行こう）

### 用言

#### 活用

用言の--/--形（第Ⅲ語基形）において、陽母音語幹の場合でも -- が付くことがある\[1\]。

  - \- （受けて）

また、母音語幹の--/--形においては、ソウル方言とは異なる縮約形となるものがある。

  - \- （混ぜて） cf. ソウル方言：-

  - \- （くれ） cf. ソウル方言：-

ソウル方言の変格用言のうち、変格と変格は東南方言では正格用言として現れる。

  - \- （暑く） cf. 標準語：-

  - \- （治って） cf. 標準語：-

一方、南部の東南方言では、動詞「（食べる）」が以下のように変格用言として現れ、「変格」とでも言うべき様相を呈している。

#### 叙述形

叙述形には以下のような形式がある。

  - \-/- （上称）
  - \- （等称）
  - \-/- （下称）

ソウル方言の略待上称（体）に相当する形としては、- を付ける形がある。

  - . （ここにあります）

  - . （ここにありますが）

#### 疑問形

疑問形には以下のような形式がある。

  - \-/- （上称）
  - \-/-（） （上称）
  - \-/-（） （等称）
  - \- （下称）

東南方言の疑問形の最大の特徴は、判断疑問と疑問詞疑問とで形式が異なりうる点である。この区分があるのは等称と下称であり、判断疑問の場合には母音  で終わる形式が、疑問詞疑問の場合には母音  で終わる形式が用いられる。

|    | 判断疑問   | 疑問詞疑問  |
| -- | ------ | ------ |
| 等称 | \-/-（） | \-/-（） |
| 等称 | \-     | \-     |

  - ? （家に行くのか）

  - ? （どこに行くのか）

従って、疑問詞が未知の意味（「何、だれ」など）でなく不定の意味（「何か、だれか」など）で用いられている場合は、疑問詞疑問形でなく判断疑問形が用いられる。ソウル方言では区別のつかない形が東南方言では区別されることになる。

  - ? （そこにだれかがいるのか；疑問詞が不定、判断疑問文）

  - ? （そこにだれがいるのか；疑問詞が未知、疑問詞疑問文）

なお、体言が述語の場合の下称疑問形は、ソウル方言とは異なり指定詞を介さずして疑問形語尾 -, - が体言に付きうる。この形式は[中期朝鮮語](../Page/中期朝鮮語.md "wikilink")にも見られる古い特徴である。

  - ? （これはお前の本か；判断疑問文）

  - ? （これはだれの本か；疑問詞疑問文）

#### 命令形・勧誘形

[サムネイルの](https://ja.wikipedia.org/wiki/ファイル:Ikuno_Korea_Town04.jpg "wikilink")[生野コリアタウン](../Page/生野コリアタウン.md "wikilink")にて。\]\] 命令形には以下のような形がある。- はソウル方言ではほとんど用いられないが、東南方言では盛んに用いられる。

  - \-（）, -（） （上称）
  - \-（/） 〜 - （下称）

勧誘形には以下のような形がある。

  - \-（） （上称）
  - \- （下称）

#### 否定形

不可能を表すソウル方言の否定副詞  に相当するものとして、東南方言に  がある。

  - （できない）, （来られない）

#### 引用形

引用形を作る語尾は - （ソウル方言：-）である。- の直後に用言  が来るときは、融合して - という形で現れる。

  - ? （何ですって?） cf. ソウル方言：?

  - ? （何て言ったんだ?） cf. ソウル方言：?

#### 連体形

連体形のうち過去連体形として、過去接尾辞 --/--（ソウル方言の --/-- に相当）に非過去連体形 -（ソウル方言の - に相当）が付いた -/- という形がある。

  - （飯食べた人）
