[Boki_Kentei.jpg](https://ja.wikipedia.org/wiki/File:Boki_Kentei.jpg "fig:Boki_Kentei.jpg") **日商簿記検定**（にっしょうぼきけんてい）とは、商工会議所法（昭和28年法律第143号）第9条第9号の規定に基づき[日本商工会議所](https://ja.wikipedia.org/wiki/日本商工会議所 "wikilink")および各地商工会議所が実施する検定試験（商工会議所検定試験）のうち、[簿記](../Page/簿記.md "wikilink")に関する技能を検定するものを指す。正式名称は「日本商工会議所及び各地商工会議所主催簿記検定試験」である。

## 概要

[1954年](../Page/1954年.md "wikilink")[11月21日](../Page/11月21日.md "wikilink")に第1回が実施され、[1955年](../Page/1955年.md "wikilink")度からは年2回実施されていたが、[1997年](https://ja.wikipedia.org/wiki/1997年 "wikilink")度以降年3回実施されるようになった(1級は年2回)\[1\]。全国的に知名度の高い[資格](../Page/資格.md "wikilink")とあって、[経理](https://ja.wikipedia.org/wiki/経理 "wikilink")や[財務](../Page/財務.md "wikilink")、一般[事務職](https://ja.wikipedia.org/wiki/事務職 "wikilink")などへの[就職](https://ja.wikipedia.org/wiki/就職 "wikilink")・[転職](https://ja.wikipedia.org/wiki/転職 "wikilink")の際に有利な資格とされる。募集要項などに「簿記資格保有者優遇」と記載されているケースもある。

この検定を含む簿記・会計系の資格試験に共通する特色として、一定の制限はあるが、試験中に電卓の使用が認められていることがあげられる。また、[公認会計士試験](../Page/公認会計士試験.md "wikilink")（短答式）など一部を除いて[マークシート](https://ja.wikipedia.org/wiki/マークシート "wikilink")が採用されていないこともあげられる。

日商簿記検定のほかには、経理専門学校生（高等専修学校に通う生徒も含む）を対象（ただし、受験資格はこれに限定されない）とした[全経簿記能力検定や](https://ja.wikipedia.org/wiki/全国経理教育協会#簿記能力検定 "wikilink")、商業高校生を対象とした[全商簿記検定](https://ja.wikipedia.org/wiki/全商簿記検定 "wikilink")が実施されている。また、日商簿記検定に類似するものとして、[財務諸表](../Page/財務諸表.md "wikilink")の読み方や[財務分析](https://ja.wikipedia.org/wiki/財務分析 "wikilink")などについて出題される[ビジネス会計検定試験](https://ja.wikipedia.org/wiki/ビジネス会計検定試験 "wikilink")がある。

## 各級の基準

### 1級

[大学](../Page/大学.md "wikilink")程度の[商業簿記](https://ja.wikipedia.org/wiki/簿記#経済活動による分類 "wikilink")・[会計学](../Page/会計学.md "wikilink")・[工業簿記](https://ja.wikipedia.org/wiki/簿記#経済活動による分類 "wikilink")・[原価計算](https://ja.wikipedia.org/wiki/原価計算 "wikilink")を習得している。[大企業](https://ja.wikipedia.org/wiki/大企業 "wikilink")の簿記。[財務諸表](../Page/財務諸表.md "wikilink")規則・企業会計に関する法規を理解し、経営管理や経営分析ができる。

### 2級

商企業および工企業における経理担当者および経理事務員として必要な高校程度の商業簿記および工業簿記（初歩的な原価計算を含む）に関する知識を有している。財務諸表を読む力がつき、企業の経営状況を把握できる。相手の経営状況もわかるので、株式会社の経営管理に役立つ。[中小企業](../Page/中小企業.md "wikilink")の簿記に相当する。

### 3級

個人企業における経理担当者または経理補助者として必要な商業簿記に関する知識を有している。経理関連書類の読み取りができ、取引先企業の経営状況を数字から理解できるようになる。営業、管理部門に必要な知識として評価する企業が増えている。簿記初学者が学習をはじめるのは多くの場合、この級からである。

### 初級

2017年4月より、4級廃止に伴い新設。ネット試験による実施。個人企業や会社の経理担当者だけでなく広く社会人全般を対象に簿記の基本用語や複式簿記の仕組みを理解し、日常業務に利活用することができる（決算に関する部分は除かれる）。また、2018年4月より、原価計算初級が増設された。

## 試験の実施

試験は全国統一日程で年に3回（6月、11月、2月）実施される。ただし、2月は2級以下のみの実施である。申込みは各地商工会議所で受け付けるが、その日程や申込み方法は統一されていない。 なお、初級向けのネット試験は毎月実施される。

## 出題範囲

準拠する法令および会計基準は、毎年度4月1日現在施行のものとする（平成18年度の第113回（6月実施）以降）。第112回までは、毎年1月1日現在施行のものとされていた。

各級の出題範囲は、日本商工会議所の定める「商工会議所簿記検定試験出題区分表及び簿記検定試験許容勘定科目表」に従う。

### 出題区分の改訂

2019年度の試験（第152回試験～）から日商簿記3級の試験範囲（出題区分）が変更される。

３級の前提が「個人商店」から「小規模な株式会社」に変更されて、実務に合わせた論点の整理が行われる。

詳しくは日本商工会議所のHPを参照。[平成31年（2019年）度以降の簿記検定試験出題区分表の改定等について](https://www.kentei.ne.jp/22021)

## 試験科目および合否

| 級         | 試験科目           | 試験時間 | 合格基準  |
| --------- | -------------- | ---- | ----- |
| 1         | 商業簿記・会計学       | 90分  | 70点以上 |
| 工業簿記・原価計算 | 90分            |      |       |
| 2         | 商業簿記・工業簿記      | 120分 |       |
| 3         | 商業簿記           | 120分 |       |
| 初級        | ネット試験（簿記、原価計算） | 40分  |       |

1級については、総得点に加え商業簿記（25点）会計学（25点）工業簿記（25点）原価計算（25点）の科目ごとにつき、40％（10点）以上の得点を要する。1科目でも9点以下の科目がある場合は不合格となる（理論上84点でも不合格が生じる）。

2級は第1問で仕訳5問（20点）第2問で個別処理問題（20点）第3問で決算問題（20点）を中心とした商業簿記の問題が60点分出題され、第4問（20点）と第5問（20点）で工業簿記・原価計算の問題が40点分出題される。2級では1級のように科目ごとの足切りはない。ただし、合格点の70点に達するには商業簿記が満点であっても工業簿記が最低10点は必要であり、工業簿記が満点であっても商業簿記が最低30点は必要である。

1級、2級とも科目合格制度はない。

## 合格発表

主催している各[商工会議所によって若干の違いがある](https://ja.wikipedia.org/wiki/日本の商工会議所一覧 "wikilink")。合格発表で早いところと遅いところとでは、2週間程度の日付の違いがある。多くは、主催した[商工会議所にて発表されるが](https://ja.wikipedia.org/wiki/日本の商工会議所一覧 "wikilink")、最近では主催した[商工会議所の](https://ja.wikipedia.org/wiki/日本の商工会議所一覧 "wikilink")[サイトでも発表する場合がある](../Page/ウェブサイト.md "wikilink")。

[合格証書](https://ja.wikipedia.org/wiki/合格証書 "wikilink")は各[商工会議所によって様々である](https://ja.wikipedia.org/wiki/日本の商工会議所一覧 "wikilink")。受験票と引き換えに[合格証書](https://ja.wikipedia.org/wiki/合格証書 "wikilink")の交付や郵送（郵送がない都道府県がある）などがある。

## 合格者の特典

  - [商業高等学校](https://ja.wikipedia.org/wiki/商業高等学校 "wikilink")、[大学](../Page/大学.md "wikilink")（主に[経済学部](../Page/経済学部.md "wikilink")・[経営学部](../Page/経営学部.md "wikilink")・[商学部](../Page/商学部.md "wikilink")等）、[大学院](https://ja.wikipedia.org/wiki/大学院 "wikilink")、[短期大学](https://ja.wikipedia.org/wiki/短期大学 "wikilink")等では、取得級に応じて、[入学試験](https://ja.wikipedia.org/wiki/入学試験 "wikilink")（特に[推薦入学](https://ja.wikipedia.org/wiki/推薦入学 "wikilink")・[AO入試](../Page/AO入試.md "wikilink")）での優遇や、入学後の単位認定、[学費](https://ja.wikipedia.org/wiki/学費 "wikilink")無償の「[給費生](https://ja.wikipedia.org/wiki/給費生 "wikilink")（授業料が免除される）」の対象となることがある。
  - 1級合格者は[税理士試験](../Page/税理士試験.md "wikilink")の受験資格を得られる。
  - 1級合格者は[職業能力開発促進法](https://ja.wikipedia.org/wiki/職業能力開発促進法 "wikilink")の[職業訓練指導員_(事務科)](https://ja.wikipedia.org/wiki/職業訓練指導員_\(事務科\) "wikilink")の資格試験で一部の科目が免除される\[2\]。
