『**RUN\&RUN**』（**ラン アンド ラン**）は、[1980年](https://ja.wikipedia.org/wiki/1980年 "wikilink")[3月15日](../Page/3月15日.md "wikilink")に全国劇場公開された[矢沢永吉](../Page/矢沢永吉.md "wikilink")の[ドキュメンタリー映画](https://ja.wikipedia.org/wiki/ドキュメンタリー映画 "wikilink")\[1\]。製作・プルミエ・インターナショナル、[富士映画配給](../Page/松竹富士.md "wikilink")\[2\]。矢沢の[1979年](../Page/1979年.md "wikilink")[9月15日](../Page/9月15日.md "wikilink")の[ナゴヤ球場](../Page/ナゴヤ球場.md "wikilink")ライブと、普段見せない私生活を追う\[3\]。

## 概要

1980年のテーマを「青春―音楽―愛」をテーマにしていた富士映画が『[さらば青春の光](https://ja.wikipedia.org/wiki/さらば青春の光_\(映画\) "wikilink")』『 』に次いで配給を決め、1979年10月に発表した\[4\]。当時矢沢はテレビ、ラジオ出演を一切拒否していたため、その矢沢が映画に出るというのが大きな売りであった\[5\]。企画は矢沢が1980年代に何をやろうとするのか、矢沢の成り上がり哲学を基に描く、というものだった\[6\]。

ファンからの問い合わせが富士映画に殺到したが、当時の矢沢は徹底したマスコミ嫌いを標榜し\[7\]、富士映画が何度頭を下げても矢沢は表舞台に出て来てくれず\[8\]。このため製作記者会見も出来ず、完成披露試写会での出席も拒否され、映画の宣伝に一切協力しなかった\[9\]。ところが1980年2月10日に[NHK教育の](../Page/NHK教育テレビジョン.md "wikilink")『[若い広場](https://ja.wikipedia.org/wiki/若い広場 "wikilink")』に突如出演し\[10\]、富士映画の関係者をビックリさせた\[11\]。この放送が大きな反響を呼んだため、これ以上ない映画の宣伝になった\[12\]。

[なぎら健壱](../Page/なぎら健壱.md "wikilink")は、公開時に映画館に本作を観に行き、劇場内で矢沢タオルを宙に放るYAZAWA信者を目撃し、衝撃を受けたと話している\[13\]。
