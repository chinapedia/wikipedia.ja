**形声**（けいせい、形聲）とは、[漢字](../Page/漢字.md "wikilink")の造字法を表す[六書](../Page/六書.md "wikilink")（りくしょ）の一つである。**象声**、**諧声**ともいう。

## 構成

事物の類型を表す記号（意符）と発音を表す記号（音符）を組み合わせて新しい字を作る。形声によって作られた漢字を[形声文字](https://ja.wikipedia.org/wiki/形声文字 "wikilink")（けいせいもじ）という。漢字の 90% 以上が形声文字である。

例えば「江」（コウ）、「河」（カ）は、左側の「[氵](../Page/水部.md "wikilink")」（さんずい）が意符で、水を表し、右側の「工」や「可」が音符で、各字の音を表す。字典で[部首](../Page/部首.md "wikilink")とされているものが、意符となることが多い。

音符は音を表すものであり、例えば「銅」は「ドウ」という音の金属という意味であり、音符である「同」には意味は無い。

しかしながらまったくの[表音文字](https://ja.wikipedia.org/wiki/表音文字 "wikilink")とは異なり、ある程度は意味に関連がある場合もある。例えば「晴」は「青」を音符とし、「[日](https://ja.wikipedia.org/wiki/日部 "wikilink")」を意符とするが、「青」は晴れた空の青さを意味する。「清」は「氵」を意符とするが、「青」は清い海の青さを意味する。なお「[青](https://ja.wikipedia.org/wiki/wikt:青 "wikilink")」は[会意](../Page/会意.md "wikilink")。

「照」は「灬」を意符とするが、「昭」は明るいことを意味し、組み合わせることで「火を使って明るくする」という意味になる。ちなみに「[昭](https://ja.wikipedia.org/wiki/wikt:昭 "wikilink")」も「召」を音符とする形声であるが、「[召](https://ja.wikipedia.org/wiki/wikt:召 "wikilink")」は「まねく」の意を示しているとも解釈できる。

こういった面から、[宋代](https://ja.wikipedia.org/wiki/宋代 "wikilink")以降、音符のほうに意味を区別する語源があるのだとする[右文説](../Page/右文説.md "wikilink")が唱えられた。

## 類型

### 左が意符、右が音符

極めて多い。

|    | 音符 |
| -- | -- |
| 右  | 可  |
| 意符 | 亻  |
| 口  |    |
| 礻  | 祐  |

### 右が意符、左が音符

多い。

|    | 音符 |
| -- | -- |
| 工  | 交  |
| 意符 | 力  |
| 攵  | 攻  |
| 頁  | 項  |

### 上が意符、下が音符

多い。草 (艹 + 早)、雲 (雨 + 云)、宇 (宀 + 于) など。

### 下が意符、上が音符

多い。梨 (木 + 利)、斧 (斤 + 父)、想 (心 + 相) など。

### 外が意符、内が音符

[構](../Page/構.md "wikilink")が意符の字は多い。園 (囗 + 袁)、閥 (門 + 伐)、衷 (衣 + 中)、術 (行 + 朮) など。

### 内が意符、外が音符

少ない。聞 (耳 + 門)、齋 (示 + 齊)、辯 (言 + 辡)、疆 (土 + 彊)、衡 (角 + 大 + 行) など。

### 一隅が意符、三隅が音符

非常に少ない。修（彡 + 攸）、磨（石 + 麻）、雇（隹 + 戸）など。

### 三隅が意符、一隅が音符

[繞](../Page/繞.md "wikilink")や[垂](https://ja.wikipedia.org/wiki/垂 "wikilink")が意符の字は多い。遠 (辶 + 袁)、病 (疒 + 丙)、趣 (走 + 取) など。

## 造字法の変遷

歴史的に見ると、元の字に後から意符が付けられた例が少なくない。例えば、「然」はもともと「もえる」の意味であったが、「しかり」の意味が主になると、本義を表すために「[火](../Page/火部.md "wikilink")」を加えて「燃」とした。同様に「要」も「こし」から「かなめ」の意味が主になると、元の「こし」は「腰」とするようになった。

逆に後世では、新しい言葉がうまれると、形声によって新しい文字を作った。ここでの音符は意味はあまり考慮されず、[表音文字](https://ja.wikipedia.org/wiki/表音文字 "wikilink")的に使われている。

## 字書での扱い

字書では、意符を[部首](../Page/部首.md "wikilink")とする場合が多い。例えば、

  - 「銅」は意符「金」と音符「同」から成るので[金部](../Page/金部.md "wikilink")に属する
  - 「雲」は意符「雨」と音符「云」から成るので[雨部](../Page/雨部.md "wikilink")に属する

といった具合である。ただし意符が部首にないなどの場合はこの限りではない。例えば、

  - 意符「光」と音符「軍」から成る「輝」は、「光」が部首にないので便宜的に[車部](../Page/車部.md "wikilink")に入れられている。
