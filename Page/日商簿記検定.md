[Boki_Kentei.jpg](https://ja.wikipedia.org/wiki/File:Boki_Kentei.jpg "fig:Boki_Kentei.jpg") **日商簿記検定**（にっしょうぼきけんてい）とは、商工会議所法（昭和28年法律第143号）第9条第9号の規定に基づき[日本商工会議所](../Page/日本商工会議所.md "wikilink")および各地商工会議所が実施する検定試験（商工会議所検定試験）のうち、[簿記](../Page/簿記.md "wikilink")に関する技能を検定するものを指す。正式名称は「日本商工会議所及び各地商工会議所主催簿記検定試験」である。

## 概要

[1954年](../Page/1954年.md "wikilink")[11月21日](../Page/11月21日.md "wikilink")に第1回が実施され、[1955年](../Page/1955年.md "wikilink")度からは年2回実施されていたが、[1997年](https://ja.wikipedia.org/wiki/1997年 "wikilink")度以降年3回実施されるようになった(1級は年2回)\[1\]。全国的に知名度の高い[資格](../Page/資格.md "wikilink")とあって、[経理](https://ja.wikipedia.org/wiki/経理 "wikilink")や[財務](../Page/財務.md "wikilink")、一般[事務職](https://ja.wikipedia.org/wiki/事務職 "wikilink")などへの[就職](https://ja.wikipedia.org/wiki/就職 "wikilink")・[転職](../Page/転職.md "wikilink")の際に有利な資格とされる。募集要項などに「簿記資格保有者優遇」と記載されているケースもある。

この検定を含む簿記・会計系の資格試験に共通する特色として、一定の制限はあるが、試験中に電卓の使用が認められていることがあげられる。また、[公認会計士試験](../Page/公認会計士試験.md "wikilink")（短答式）など一部を除いて[マークシート](../Page/マークシート.md "wikilink")が採用されていないこともあげられる。

日商簿記検定のほかには、経理専門学校生（高等専修学校に通う生徒も含む）を対象（ただし、受験資格はこれに限定されない）とした[全経簿記能力検定](../Page/全経簿記能力検定.md "wikilink")や、商業高校生を対象とした[全商簿記検定](https://ja.wikipedia.org/wiki/全商簿記検定 "wikilink")が実施されている。また、日商簿記検定に類似するものとして、[財務諸表](../Page/財務諸表.md "wikilink")の読み方や[財務分析](../Page/財務分析.md "wikilink")などについて出題される[ビジネス会計検定試験](https://ja.wikipedia.org/wiki/ビジネス会計検定試験 "wikilink")がある。

## 各級の基準

### 1級

[大学](../Page/大学.md "wikilink")程度の[商業簿記](https://ja.wikipedia.org/wiki/簿記#経済活動による分類 "wikilink")・[会計学](../Page/会計学.md "wikilink")・[工業簿記](https://ja.wikipedia.org/wiki/簿記#経済活動による分類 "wikilink")・[原価計算](https://ja.wikipedia.org/wiki/原価計算 "wikilink")を習得している。[大企業](../Page/大企業.md "wikilink")の簿記。[財務諸表](../Page/財務諸表.md "wikilink")規則・企業会計に関する法規を理解し、経営管理や経営分析ができる。

### 2級

商企業および工企業における経理担当者および経理事務員として必要な高校程度の商業簿記および工業簿記（初歩的な原価計算を含む）に関する知識を有している。財務諸表を読む力がつき、企業の経営状況を把握できる。相手の経営状況もわかるので、株式会社の経営管理に役立つ。[中小企業](../Page/中小企業.md "wikilink")の簿記に相当する。

### 3級

個人企業における経理担当者または経理補助者として必要な商業簿記に関する知識を有している。経理関連書類の読み取りができ、取引先企業の経営状況を数字から理解できるようになる。営業、管理部門に必要な知識として評価する企業が増えている。簿記初学者が学習をはじめるのは多くの場合、この級からである。

かつては個人商店を対象とする内容であったが、2019年度から株式会社を対象としたものに変更されている。

### 初級

2017年4月より、4級廃止に伴い新設。ネット試験による実施。個人企業や会社の経理担当者だけでなく広く社会人全般を対象に簿記の基本用語や複式簿記の仕組みを理解し、日常業務に利活用することができる（決算に関する部分は除かれる）。また、2018年4月より、原価計算初級が増設された。

## 試験の難易度

### 1級

合格率は例年10％前後である。受験者の大部分は既に2級に合格できる実力を有している場合が多いため、かなりの難関資格であると言える。また、絶対評価である2級以下とは異なり、合格率の大きな変動がさほど発生していないことから、採点方法は[相対評価](../Page/相対評価.md "wikilink")であると目されている。

同じくらいの難易度と言われる他の資格試験としては、[社会保険労務士](../Page/社会保険労務士.md "wikilink")（社労士）試験や[中小企業診断士](../Page/中小企業診断士.md "wikilink")試験、[土地家屋調査士](../Page/土地家屋調査士.md "wikilink")試験、[マンション管理士](../Page/マンション管理士.md "wikilink")試験、[行政書士](https://ja.wikipedia.org/wiki/行政書士 "wikilink")試験、[海事代理士](../Page/海事代理士.md "wikilink")試験、[通関士](../Page/通関士.md "wikilink")試験、[応用情報技術者試験](../Page/応用情報技術者試験.md "wikilink")、[FP技能士](../Page/ファイナンシャル・プランニング技能士.md "wikilink")1級、[全経簿記能力検定](../Page/全経簿記能力検定.md "wikilink")上級などがあげられる。

### 2級

近年の合格率は平均して20〜30％程度であるが、採点方法が[絶対評価](../Page/絶対評価.md "wikilink")であるため、2015年（平成27年）11月実施の第141回試験（合格率11.8%）のように合格率が15％を下回った回から、逆に2013年（平成25年）2月実施の第133回試験（合格率47.6%）のように合格率が40％を上回った回まで、合格率の乱高下が激しい。2016年度以降の出題内容改定の影響を最も大きく受けた級でもあり、かつては1級の内容であった連結会計や税効果会計、外貨建取引などの内容が追加され、その出題内容をめぐって炎上騒ぎに発展したこともある。

商業簿記のみが問われる3級とは異なり、2級では工業簿記も問われ専門性が高くなる。そのため、簿記初学者がいきなり2級を受験して合格するのはかなり難しいと言われており、3級合格後のステップアップとして受験するのが良いとされる。

同じくらいの難易度と言われる他の資格試験としては、[宅地建物取引士](../Page/宅地建物取引士.md "wikilink")（宅建）試験や[管理業務主任者](../Page/管理業務主任者.md "wikilink")試験、[総合旅行業務取扱管理者試験](../Page/旅行業務取扱管理者.md "wikilink")、[基本情報技術者試験](../Page/基本情報技術者試験.md "wikilink")、FP技能士2級、[全経簿記能力検定](../Page/全経簿記能力検定.md "wikilink")1級などがあげられる。

なお、[大学入試センター試験](../Page/大学入試センター試験.md "wikilink")の[数学では](../Page/数学_\(教科\).md "wikilink")「簿記・会計」という科目を選択できる場合もある\[2\]が、日商簿記2級で出題される内容はこれよりも難易度が高いと言われることが多い。一般的にセンター試験の「簿記・会計」の問題は日商簿記の2級と3級の中間の水準で出題されることが多いとされる。ただし日商簿記2級の場合、予め設定された基準点（満点の70％）以上の得点を獲得できればほぼ確実に合格できるのに対し、センター試験の場合、[大学](../Page/大学.md "wikilink")によってはかなり高いボーダーラインを超えることが要求される場合があるため、一概に比較することは難しい。

### 3級

近年の合格率は平均して40〜50％程度である。2015年（平成27年）11月実施の第141回試験（合格率26.1%）のように合格率が30％を下回った回もある。

同じくらいの難易度と言われる他の資格試験としては、[ITパスポート試験](https://ja.wikipedia.org/wiki/ITパスポート試験 "wikilink")や国内旅行業務取扱管理者試験、FP技能士3級、[全経簿記能力検定](../Page/全経簿記能力検定.md "wikilink")2級などがあげられる。

## 試験の実施

試験は全国統一日程で年に3回（6月、11月、2月）実施される。ただし、2月は2級以下のみの実施である。申込みは各地商工会議所で受け付けるが、その日程や申込み方法は統一されていない。 なお、初級向けのネット試験は毎月実施される。

## 出題範囲

準拠する法令および会計基準は、毎年度4月1日現在施行のものとする（平成18年度の第113回（6月実施）以降）。第112回までは、毎年1月1日現在施行のものとされていた。

各級の出題範囲は、日本商工会議所の定める「商工会議所簿記検定試験出題区分表及び簿記検定試験許容勘定科目表」に従う。

## 試験科目および合否

| 級         | 試験科目           | 試験時間 | 合格基準  |
| --------- | -------------- | ---- | ----- |
| 1         | 商業簿記・会計学       | 90分  | 70点以上 |
| 工業簿記・原価計算 | 90分            |      |       |
| 2         | 商業簿記・工業簿記      | 120分 |       |
| 3         | 商業簿記           | 120分 |       |
| 初級        | ネット試験（簿記、原価計算） | 40分  |       |

1級については、総得点に加え商業簿記（25点）会計学（25点）工業簿記（25点）原価計算（25点）の科目ごとにつき、40％（10点）以上の得点を要する。1科目でも9点以下の科目がある場合は不合格となる（理論上84点でも不合格が生じる）。

2級は第1問で仕訳5問（20点）第2問で個別処理問題（20点）第3問で決算問題（20点）を中心とした商業簿記の問題が60点分出題され、第4問（20点）と第5問（20点）で工業簿記・原価計算の問題が40点分出題される。2級では1級のように科目ごとの足切りはない。ただし、合格点の70点に達するには商業簿記が満点であっても工業簿記が最低10点は必要であり、工業簿記が満点であっても商業簿記が最低30点は必要である。

1級、2級とも科目合格制度はない。

## 合格発表

主催している各[商工会議所によって若干の違いがある](https://ja.wikipedia.org/wiki/日本の商工会議所一覧 "wikilink")。合格発表で早いところと遅いところとでは、2週間程度の日付の違いがある。多くは、主催した[商工会議所にて発表されるが](https://ja.wikipedia.org/wiki/日本の商工会議所一覧 "wikilink")、最近では主催した[商工会議所の](https://ja.wikipedia.org/wiki/日本の商工会議所一覧 "wikilink")[サイトでも発表する場合がある](../Page/ウェブサイト.md "wikilink")。

[合格証書](https://ja.wikipedia.org/wiki/合格証書 "wikilink")は各[商工会議所によって様々である](https://ja.wikipedia.org/wiki/日本の商工会議所一覧 "wikilink")。受験票と引き換えに[合格証書](https://ja.wikipedia.org/wiki/合格証書 "wikilink")の交付や郵送（郵送がない都道府県がある）などがある。

## 合格者の特典

  - [民間企業](https://ja.wikipedia.org/wiki/民間企業 "wikilink")の[入社試験](../Page/入社試験.md "wikilink")・[履歴書](../Page/履歴書.md "wikilink")において、日商簿記検定の取得級に応じて採用判定で優先されることがある。また、企業における[昇進](https://ja.wikipedia.org/wiki/昇進 "wikilink")等の[人事評価](https://ja.wikipedia.org/wiki/人事評価 "wikilink")でも日商簿記検定の取得級が評価材料になることがある。一般的には[中小企業](../Page/中小企業.md "wikilink")では2級以上の、[大企業](../Page/大企業.md "wikilink")では1級の合格者は高く評価されると言われている。
  - [商業高等学校](../Page/商業高等学校.md "wikilink")、[大学](../Page/大学.md "wikilink")（主に[経済学部](../Page/経済学部.md "wikilink")・[経営学部](../Page/経営学部.md "wikilink")・[商学部](../Page/商学部.md "wikilink")等）、[大学院](https://ja.wikipedia.org/wiki/大学院 "wikilink")、[短期大学](../Page/短期大学.md "wikilink")等では、取得級に応じて、[入学試験](https://ja.wikipedia.org/wiki/入学試験 "wikilink")（特に[推薦入学](../Page/推薦入学.md "wikilink")・[AO入試](../Page/AO入試.md "wikilink")）での優遇や、入学後の単位認定、[学費](../Page/学費.md "wikilink")無償の「[給費生](https://ja.wikipedia.org/wiki/給費生 "wikilink")（授業料が免除される）」の対象となることがある。
  - 1級合格者は[税理士試験](../Page/税理士試験.md "wikilink")の受験資格を得られる。
  - 1級合格者は[職業能力開発促進法](../Page/職業能力開発促進法.md "wikilink")の[職業訓練指導員_(事務科)](https://ja.wikipedia.org/wiki/職業訓練指導員_\(事務科\) "wikilink")の資格試験で一部の科目が免除される\[3\]。
