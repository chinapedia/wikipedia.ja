**非言語コミュニケーション**（ひげんごコミュニケーション、[英語](../Page/英語.md "wikilink"): non-verbal communication）とは、[言葉](https://ja.wikipedia.org/wiki/言葉 "wikilink")以外の手段を用いた[コミュニケーション](../Page/コミュニケーション.md "wikilink")（メッセージのやり取り）のこと。略号で**NVC**とも。

## 概要

非言語コミュニケーションとは、言葉以外の手段によるコミュニケーションのことである。

人間は日常的に複数の非言語的手がかりを使い[メッセージ](../Page/メッセージ.md "wikilink")を伝達しあっている。これを「非言語的コミュニケーション」（nonverbal communication ノンバーバル・コミュニケーション、という\[1\]。この非言語的なコミュニケーションは、意識して用いていることもあれば、[無意識](https://ja.wikipedia.org/wiki/無意識 "wikilink")的に用いていることもある\[2\]。

人間はコミュニケーションを行う時、言葉を使い互いの感情や意思を伝えあってもいるが、「[目](../Page/目.md "wikilink")は口ほどにものをいう」といった諺にも示されているように、言葉よりも顔の表情・視線・身振りなどのほうが、より重要な役割を荷っていることがある\[3\]。

人間は非言語的コミュニケーションを、顔の[表情](https://ja.wikipedia.org/wiki/表情 "wikilink")、[顔色](https://ja.wikipedia.org/wiki/表情 "wikilink")、[視線](https://ja.wikipedia.org/wiki/視線 "wikilink")、[身振り](https://ja.wikipedia.org/wiki/身振り "wikilink")、手振り、体の[姿勢](https://ja.wikipedia.org/wiki/姿勢 "wikilink")、相手との物理的な[距離](../Page/距離.md "wikilink")の置き方などによって行っている\[4\]。

身振りなどの非言語コミュニケーションの多くは[文化](../Page/文化.md "wikilink")によって異なるが、人間の基礎的な[感情](../Page/感情.md "wikilink")である[怒り](https://ja.wikipedia.org/wiki/怒り "wikilink")、[失望](https://ja.wikipedia.org/wiki/失望 "wikilink")、[恐怖](https://ja.wikipedia.org/wiki/恐怖 "wikilink")、[喜び](https://ja.wikipedia.org/wiki/喜び "wikilink")、[感動](https://ja.wikipedia.org/wiki/感動 "wikilink")、[驚き](https://ja.wikipedia.org/wiki/驚き "wikilink")などに対する[表情](https://ja.wikipedia.org/wiki/表情 "wikilink")は普遍的なものとされる。非言語コミュニケーションを最初に研究対象としたのは[チャールズ・ダーウィン](../Page/チャールズ・ダーウィン.md "wikilink")の『人及び動物の表情について』（[1872](../Page/1872年.md "wikilink")）とされ、以来非常に多岐にわたる研究がなされている。

## 非言語と言語の区別

言語コミュニケーションとは｢言葉｣を使ったコミュニケーションの全てを対象とし、必ずしも音声を伴っている必要はない。例えば[手話](../Page/手話.md "wikilink")や、書記言語である[筆記](https://ja.wikipedia.org/wiki/筆記 "wikilink")を用いたコミュニケーションも言語コミュニケーションである。

反対に[手話](../Page/手話.md "wikilink")や[筆談](https://ja.wikipedia.org/wiki/筆談 "wikilink")の場合でも、例えば「手話がたどたどしいことから相手へ持った印象」や「字が汚い/丁寧なことからくるイメージ」などは、非言語のメッセージである。普通の言葉を使った会話であっても、それが発せられる[口調](https://ja.wikipedia.org/wiki/口調 "wikilink")や強さ、声の高さ、言い淀み、発するときの[表情](https://ja.wikipedia.org/wiki/表情 "wikilink")によって伝わる内容は異なる可能性があり、それらは[パラ言語](https://ja.wikipedia.org/wiki/パラ言語 "wikilink")と呼ばれる非言語コミュニケーションの一部である\[5\]。

| メッセージの種類     | 具体例                              |
| ------------ | -------------------------------- |
| 言語/音声メッセージ   | 口頭言語                             |
| 言語/非音声メッセージ  | 文面、ジェスチャーサイン                     |
| 非言語/音声メッセージ  | 声の張り、イントネーション、話す速度、言葉遣い          |
| 非言語/非音声メッセージ | 外見・身だしなみ、身振り手振り、姿勢、視線、対人距離、表情、呼吸 |
