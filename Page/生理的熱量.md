**生理的熱量**（せいりてきねつりょう、別名**生理的エネルギー値**、**生理的エネルギー量**、**代謝熱量**、**代謝エネルギー量**）とは、[生物](../Page/生物.md "wikilink")の活動に伴って吸収消費される[熱量](https://ja.wikipedia.org/wiki/熱量 "wikilink")（[エネルギー](https://ja.wikipedia.org/wiki/エネルギー "wikilink")）のことを言う。主に摂取する食物から得られる[栄養学](https://ja.wikipedia.org/wiki/栄養学 "wikilink")的熱量や、[運動](https://ja.wikipedia.org/wiki/運動 "wikilink")や[代謝](../Page/代謝.md "wikilink")によって消費されるエネルギーについて用いられる。

なお、日本では、生理的熱量に用いられる単位「**[カロリー](../Page/カロリー.md "wikilink")**」が、生理的熱量を表す名詞として一般的に通用している（例・低カロリー食、カロリー制限など）。

## 生理的熱量の発見

17世紀後半から、生物の呼気と吸気の組成の変化が、物体を燃焼させた前後の[空気](../Page/空気.md "wikilink")組成の変化に似ているという観察結果から、[呼吸](../Page/呼吸.md "wikilink")と[燃焼](../Page/燃焼.md "wikilink")の関係が論じられるようになった。ただしこの時代は空気の成分について現代的な理解とは異なっていたため、呼吸について、[メイヨー](https://ja.wikipedia.org/wiki/ジョン・メイヨー "wikilink")（1674）は[フック](../Page/ロバート・フック.md "wikilink")（1665）の「硝石の空気」が消費されると考え、[プリーストリー](../Page/ジョゼフ・プリーストリー.md "wikilink")（1775）は[フロギストン](https://ja.wikipedia.org/wiki/フロギストン "wikilink")の放出であると考えた。[ラヴォアジエ](../Page/アントワーヌ・ラヴォアジエ.md "wikilink")（1777）はこれを[酸素](../Page/酸素.md "wikilink")の消費と[二酸化炭素](../Page/二酸化炭素.md "wikilink")の排出であるとし、さらに運動強度に比例して酸素消費量が増えることなどから、生物による酸素消費は燃焼による酸素消費と同等であるとみなされるようになった。このことより、燃焼の量の指標である熱量が体内における「燃焼」（[代謝](../Page/代謝.md "wikilink")）にも当てはめられるという考えが生まれ、生理的熱量の概念が確立した。

体内で「燃焼」が起こっている妥当性は、人間の生命に体温の維持が必要であると認識されていたし、熱を運動に変換できることは[蒸気機関](../Page/蒸気機関.md "wikilink")からの類推で知られていた。後に[エネルギー保存の法則](../Page/エネルギー保存の法則.md "wikilink")が確立され、[解糖系](../Page/解糖系.md "wikilink")など生体内におけるエネルギー変換の[分子](../Page/分子.md "wikilink")的機序が明らかになって、生体が利用できるエネルギー、すなわち生理的熱量の考えは、現代では確固たる物として認められている。

また、糖や脂肪といった別々の[栄養素](../Page/栄養素.md "wikilink")はそれぞれ異なる[栄養](../Page/栄養.md "wikilink")を与えると考えられていたが、同じ熱量の糖や脂肪は熱量上等価で交換可能であり、生理的熱量は独立の栄養概念であると考えられている（**ルブネルのエネルギー等値の法則**）。また生理的熱量を与える栄養素を**熱量素**と呼ぶ。

## 生理学的熱量の定義と計測

  - [酸素](../Page/酸素.md "wikilink")消費からの定義。[有機物](https://ja.wikipedia.org/wiki/有機物 "wikilink")が体内で消費された場合と体外で燃焼させた場合で、酸素の消費が同量であれば、熱量の発生も同等だと推定する。厳密には気密容器の中で測定し、簡易的には呼気中の二酸化炭素濃度から求める（[ペッテンコーファー](../Page/マックス・フォン・ペッテンコーファー.md "wikilink"),1862など）。現在でも基礎代謝や運動によって消費される熱量はこの方法で求めることが多い。
  - 食物を燃やして得られる熱量による定義。食物を空気中で燃やして得られた熱量と、同量の食物を食べて出た[排泄](../Page/排泄.md "wikilink")物を燃やして得られた熱量の差から、食物から吸収した熱量を推定する（[ルブネル](https://ja.wikipedia.org/wiki/マックス・ルブネル "wikilink"),1883など）。食物の栄養学的熱量は主にこの方法で測定され、[消化](../Page/消化.md "wikilink")吸収率などを考慮して補正される。日本では代表的な食品材料について測定されており、[料理](../Page/料理.md "wikilink")などに表示される熱量は、一般的に食品材料の分量と重量あたり熱量から推定する。
  - 放出熱量からの定義。生物を断熱気密室に入れ、気温の上昇を直接測って放出した熱量を推定する（[ルブネル](https://ja.wikipedia.org/wiki/マックス・ルブネル "wikilink"),1894など）。酸素消費量からの測定法の補助として用いられることが多い。
  - 分子化学的機序からの定義。現代では[解糖系](../Page/解糖系.md "wikilink")など[栄養素](../Page/栄養素.md "wikilink")のエネルギー変換の[分子](../Page/分子.md "wikilink")的機序が明らかになっているため、その過程から得られる熱量を推定できる。

## 食品のエネルギー換算係数

食品のカロリー（熱量）計算には米国のRubnerとAtwaterが19世紀末から20世紀初頭にかけて行った実験結果から求めた[炭水化物](../Page/炭水化物.md "wikilink")、[脂質](../Page/脂質.md "wikilink")・[脂肪](../Page/脂肪.md "wikilink")、[タンパク質](../Page/タンパク質.md "wikilink")についての[アトウォーター係数](https://ja.wikipedia.org/wiki/アトウォーター係数 "wikilink")が広く用いられている。

アトウォーターの換算係数は各成分の物理的燃焼熱（kcal/g・kJ/g）から人体における消化吸収率（100%は吸収されず一部は排泄される）と排泄熱量（吸収されるが利用されず排泄される）を加味し求めたものである。タンパク質では吸収された一部は[尿素](../Page/尿素.md "wikilink")や[尿酸](https://ja.wikipedia.org/wiki/尿酸 "wikilink")などとして排泄される\[1\]。

| 成分    | 物理的燃焼熱    | 消化吸収率 | 排泄熱量       | 換算係数    |
| ----- | --------- | ----- | ---------- | ------- |
| 脂質    | 9.4kcal/g | 95%   |            | 9kcal/g |
| タンパク質 | 5.7kcal/g | 92%   | 1.25kcal/g | 4kcal/g |
| 炭水化物  | 4.1kcal/g | 97%   |            |         |

Atwaterの換算係数\[2\]

Atwater係数は脂質、タンパク質および炭水化物における平均値であり、便宜的な係数として多用されているが食品の成分により熱量は異なる。同じ炭水化物でも容易に利用可能なもの、[植物繊維](../Page/植物繊維.md "wikilink")、[糖アルコール](../Page/糖アルコール.md "wikilink")などの難消化性のものがあり、難消化性のものは[小腸](../Page/小腸.md "wikilink")まででは消化されず[大腸](../Page/大腸.md "wikilink")において菌などにより分解され吸収されるものがあるが分解・吸収率に差があり全く吸収されないものもある。日本の栄養表示基準では炭水化物を数段階に区分している\[3\]。

| 成分                                                      | 熱量換算係数      | 備考                                                                                             |
| ------------------------------------------------------- | ----------- | ---------------------------------------------------------------------------------------------- |
| 脂質・脂肪                                                   | 9kcal/g     |                                                                                                |
| タンパク質                                                   | 4kcal/g     |                                                                                                |
| 炭水化物                                                    | 消化性         | デンプン・砂糖など小腸までで吸収されるもの                                                                          |
| 難消化性糖質                                                  | 0 - 3kcal/g | [糖アルコール](../Page/糖アルコール.md "wikilink")、[オリゴ糖](https://ja.wikipedia.org/wiki/オリゴ糖 "wikilink")など |
| [食物繊維](../Page/食物繊維.md "wikilink")                      | 0 - 2kcal/g |                                                                                                |
| [エタノール](https://ja.wikipedia.org/wiki/エタノール "wikilink") | 7kcal/g     | 酒類                                                                                             |
| [有機酸](../Page/有機酸.md "wikilink")                        | 3kcal/g     |                                                                                                |

日本の栄養表示基準\[4\]

以上は食品の成分ごとの消化・吸収・排泄の数値に基づいた集計であるが、この他に食品の成分による展開はせず、食品のヒトによる消化吸収試験の結果からエネルギー換算係数を求める場合もある。この手法では全食品における人体実験が必要となるが、現実には食品をグループに分け代表的な食品（例えば米）で試験をし、その値をグループ内の食品に適用している\[5\]。

## その他

### 栄養学における意味

生理的熱量を与える熱量素は[栄養学](https://ja.wikipedia.org/wiki/栄養学 "wikilink")の中でも初期に発見され、かつ重要視されており、熱量素となる炭水化物、脂肪、タンパク質は**三大栄養素**と呼ばれている。

一方で、[消化](../Page/消化.md "wikilink")吸収されたものの消費されなかった熱量素は、主に[脂肪組織](../Page/脂肪組織.md "wikilink")に蓄積され、[肥満](../Page/肥満.md "wikilink")や[成人病](https://ja.wikipedia.org/wiki/成人病 "wikilink")を招く。このため、現代では熱量素の摂取を制限したり、[運動によって熱量の消費を増やすことで体脂肪率を一定に保つのがよい](../Page/スポーツ.md "wikilink")（[痩身](../Page/痩身.md "wikilink")）と考えられている。またこれは[美容](https://ja.wikipedia.org/wiki/美容 "wikilink")とも大きく関わるため、生理的熱量の摂取・消費は多くの国で国民的関心事となっている。なお、一日のエネルギー必要量（消費量）は、身体活動レベルに応じて[基礎代謝](../Page/基礎代謝.md "wikilink")量の1.5〜2倍程度となる。詳細は[栄養\#栄養学の観点から](https://ja.wikipedia.org/wiki/栄養#栄養学の観点から "wikilink")を参照のこと。

### 名称にまつわる問題

栄養学では生理的熱量は単に熱量と呼ばれることが多かったが、一般にはその単位である**カロリー**が生理的熱量をあらわす名詞として通用している。食品表示での熱量単位を[カロリー](../Page/カロリー.md "wikilink")から[ジュール](../Page/ジュール.md "wikilink")に置き換えることもあり、生理的熱量のほか、**生理的エネルギー値**、**生理的エネルギー量**、**代謝熱量**、**代謝エネルギー量**などの言葉で置き換えようとする動きはあるものの、成果はほとんど上がっておらず、[厚生労働省](../Page/厚生労働省.md "wikilink")や[農林水産省](../Page/農林水産省.md "wikilink")の広報でもカロリーという言葉が使われていることは珍しくない。また、伝統的に熱量という言葉を用いているものの、エネルギーの様態として熱を介さない代謝も多いことから、より一般的なエネルギーという言葉を用いたほうがいいという見方もある。
