**一致の定理**(いっちのていり、****)は、[複素解析](https://ja.wikipedia.org/wiki/複素解析 "wikilink")において、通常は[可算](https://ja.wikipedia.org/wiki/可算 "wikilink")[点列](https://ja.wikipedia.org/wiki/点列 "wikilink")上で局所的に一致する2つの[正則関数](https://ja.wikipedia.org/wiki/正則関数 "wikilink")が大域的に一致することを主張する[定理](../Page/定理.md "wikilink")である。重要な定理であり、[解析接続](https://ja.wikipedia.org/wiki/解析接続 "wikilink")の一意性の証明にはこの定理が必要となる。

この定理には名は冠されていないが、1844年頃、[リウヴィルが](https://ja.wikipedia.org/wiki/ジョゼフ・リウヴィル "wikilink")[楕円関数](https://ja.wikipedia.org/wiki/楕円関数 "wikilink")に特殊な形で適用したのが最初であり、直後に[コーシーが自分が開発した](../Page/オーギュスタン＝ルイ・コーシー.md "wikilink")[複素解析](https://ja.wikipedia.org/wiki/複素解析 "wikilink")の中に取り入れて一般化したものである\[1\]。

## 定理

次の2つの形式があり、どちらも**一致の定理**と呼ばれている (内容的にはほとんど言い換えに過ぎない)。

**(1)** 連結開領域 \(D \subset \mathbb{C}\) で正則な複素関数 \(f(z)\) の零点集合が \(D\) で[集積点](https://ja.wikipedia.org/wiki/集積点 "wikilink")を持てば、\(f(z)\) は \(D\) で恒等的に 0 である。

**(2)** 連結開領域 \(D \subset \mathbb{C}\) で正則な複素関数 \(f(z), g(z)\) が、\(D\) で集積点を持つ \(D\) の部分集合上で一致すれば領域 \(D\) 全体で一致する。

## 証明

**(1)** の形式について証明する。**(2)** の形式については、**(1)** の形式を \(f(z)- g(z)\) に対して適用すれば即時に出る。

証明を次の2段階に分ける。

**第1段階** \(z_0\) を \(f(z)\) の零点の集積点の1つとすれば、\(z_0\) を中心としたある正の半径 \(r\) の開円板上で \(f(z)\) は恒等的に 0 である。

**第2段階** 第1段階で証明した \(z_0\) の近傍における \(f(z)=0\) という結論を \(D\) 全体に拡張する。なお、通常は[解析接続](https://ja.wikipedia.org/wiki/解析接続 "wikilink")の一意性の証明には一致の定理を用いるので、本定理の証明の中で解析接続の一意性を援用することは好ましくない([循環論法](../Page/循環論法.md "wikilink")になる恐れがある)。

### 第1段階の証明

\(z_0\) を \(f(z)\) の零点の集積点の1つとする。 \(f(z)\) は \(D\) で正則であるから、\(z_0\) を中心として次のように[テイラー展開](../Page/テイラー展開.md "wikilink")が可能であり、その収束半径は 0 ではない。収束半径より小さな正数 \(r\) を適当に選んで、\(z_0\) を中心とした開円板 \(|z-z_0| < r\) が \(D\) に包含されるようにすることができる。この開円板を \(U\) と置く。

\[f(z)=\sum_{k=0}^{\infty}\frac{c_k}{k!}(z-z_0)^k\] もし、\(c_k\not=0\) が存在するなら、その中で最も添字の値が小さなものを\(c_n\) とし、

\[h(z)=\frac{c_n}{n!}+\sum_{k=1}^{\infty}\frac{c_{n+k}}{(n+k)!}(z-z_0)^k\] と置けば、

\[f(z)=(z-z_0)^n h(z)\] となる。上記の \(h(z)\) の \(z_0\) を中心とした テイラー展開の収束半径は \(f(z)\) と同じであり、 \(h(z)\) は \(U\) で正則で、\(h(z_0) \ne 0\) である。\(z \ne z_0\) であれば \((z-z_0)^n \ne 0\) であるから、 \(z_0\) 以外の \(f(z)\) の零点は \(h(z)\) の零点であり、\(z_0\) は \(h(z)\) の零点の集積点である。 \(h(z)\) は \(U\) で連続であるから、\(\delta\) を十分に小さな正数とすれば、\(|z-z_0| < \delta\) であれば \(h(z) \ne 0\) であるが、\(z_0\) は \(h(z)\) の零点の集積点であるから \(|z_1-z_0| < \delta\) を満たす \(h(z)\) の零点 \(z_1\) が存在するはずであるから矛盾である。

従って全ての整数 \(k\) について \(c_k=0\) であり、開円板 \(U\) 上では \(f(z)\) は恒等的に 0 である。

### 第2段階の証明

\(D\) に包含される \(f(z)\) の零点だけから成る開集合の全ての集合を \(Y\) と置く。定理の前提と第1段階の証明から明らかなように、 \(Y\) に属する元は少なくとも1つは存在する ( \(Y\) は空ではない）。

\(Y\) に属する全ての開集合の合併集合を \(D_1\) と置く。 \(D_1\) は[開集合族の公理から当然開集合である](https://ja.wikipedia.org/wiki/位相空間#定義 "wikilink")。 \(D_2 = D \cap {\overline{D_1 }}^c\) ( \({\overline{D_1}}^c\) は \(D_1\)の[閉包](https://ja.wikipedia.org/wiki/閉包 "wikilink")の[補集合](https://ja.wikipedia.org/wiki/補集合 "wikilink"))と置けば、\(D_2\) も開集合である。当然 \(D_1 \cap D_2 = \emptyset\) ([空集合](../Page/空集合.md "wikilink"))である。

\(\overline{D_1} = D_1 \cup \partial{\overline{D_1}}\) (\(\partial{\overline{D_1}}\) は\(D_1\) の[境界点](https://ja.wikipedia.org/wiki/境界点 "wikilink")の全集合) であるから\(\gamma = D \cap \partial{\overline{D_1}}\) と置けば、\(D = D_1 \cup D_2 \cup \gamma\) である。

\(D_2\) が空でないと仮定する。すると \(\gamma\) も空ではない。なぜなら、\(\gamma\) が空であれば、\(D = D_1 \cup D_2\) であるから、\(D\) は交点を持たない2つの開集合に分割されてしまって連結ではなくなるからである。

\(z_1\)、\(z_2\) をそれぞれ \(D_1\)、\(D_2\) の任意の点とすれば、 \(D\) は連結であるから \(z_1\)、\(z_2\) を結ぶ \(D\) 内に含まれる折線が存在する。これを \(C\) と置く。

\(C\) と \(\gamma\) の交点が存在する。なぜなら、これが存在しないとすれば、\(C_1 = C \cap D_1\)、\(C_2 = C \cap D_2\) と置けば、\(C = C_1 \cup C_2\) であり、\(C\) は交点を持たない開集合 \(D_1\)、\(D_2\) にそれぞれ含まれる折線 \(C_1\)、\(C_2\) に分割されることになり、\(C\) が連結でなくなるからである。

\(C\) と \(\gamma\) の交点の一つを \(z_3\) と置く。 \(z_3\) は \(D\) の[内点](https://ja.wikipedia.org/wiki/内点 "wikilink")であり、また \(D_1\) の境界点であるから \(f(z)\) の零点集合の集積点であり、証明の第1段階の結論からある正数 \(r\) が存在して、 \(D\) に含まれる開円板 \(U = \{ z |\ |z-z_3| < r \}\) 内では \(f(z)\) は恒等的に 0 である。従って、 \(D_1 \cup U\) は \(D\) に含まれる \(f(z)\) の零点集合であり \(Y\) に含まれる。

\(z_3\) は \(D_2\) の境界点でもあるから、\(U\) は \(D_2\) の点を含む。従って \(D_1 \cup U\) は \(D_1\) を真に包含し、 \(D_1\) が \(Y\) の全ての集合の合併集合であることに反する。従って \(D_2\) は空でなければならない。

従って \(D = D_1\) が成立し、\(D\) は \(f(z)\) の零点集合である。つまり \(D\) で \(f(z)\) は恒等的に 0 である。
