**レヴィ＝チヴィタ変換** (レヴィ＝チヴィタへんかん, Levi-Civita transformation\[1\]) とは、平面[ケプラー問題の](../Page/ケプラーの法則.md "wikilink")[運動方程式](https://ja.wikipedia.org/wiki/運動方程式 "wikilink")の特異性を除去し正則化する変換のこと\[2\]。[トゥーリオ・レヴィ＝チヴィタ](https://ja.wikipedia.org/wiki/トゥーリオ・レヴィ＝チヴィタ "wikilink")がその理論を発展させた\[3\]\[4\]\[5\]\[6\]。三次元ケプラー問題の正則化はこれを拡張した[クスターンヘイモ・シュティーフェル変換](https://ja.wikipedia.org/wiki/クスターンヘイモ・シュティーフェル変換 "wikilink")によって実現される\[7\]。

## 定義

2次元[ケプラー問題の](../Page/ケプラーの法則.md "wikilink")[ハミルトニアン](../Page/ハミルトニアン.md "wikilink")は、\(\mathbf{p} = ( p_1, p_2 )\) を運動量、\(\mathbf{q} = ( q_1, q_2 )\) を座標とするとき

\[H ( \mathbf{p}, \mathbf{q} ) = \frac{ 1 }{ 2 } ( p_1^2 + p_2^2 ) - \frac{ \mu }{ \sqrt{ q_1^2 + q_2^2 } }\] により与えられる。対応する運動方程式は

\[\frac{ d q_j }{ d t } = p_j , \ \ \frac{ d p_j }{ d t } = - \frac{ \mu q_j }{ ( q_1^2 + q_2^2 )^{3/2} } , \ \  ( j = 1, 2 )\] である。この方程式は重力源からの距離 \(r = \sqrt{ q_1^2 + q_2^2 }\) がゼロの極限で発散する特異性がある。Levi-Civita変換はこの特異性を除去するような変数変換である。

### ステップ1: 正準変換

この系に次の母関数 \(W ( \mathbf{p}, \mathbf{Q} )\) によって生成される正準変換 \(( \mathbf{p}, \mathbf{q} ) \mapsto ( \mathbf{P}, \mathbf{Q} )\) を施す\[8\]。

\[W ( \mathbf{p}, \mathbf{Q} ) = p_1 ( Q_1^2 + Q_2^2 ) + 2 p_2 Q_1 Q_2\] この正準変換は具体的に次のように表示できる。

\[\begin{pmatrix}q_1 \\ q_2\end{pmatrix} = \begin{pmatrix}Q_1 & - Q_2 \\ Q_2 & Q_1\end{pmatrix} \begin{pmatrix}Q_1 \\ Q_2\end{pmatrix} , \ \
\begin{pmatrix}p_1 \\ p_2\end{pmatrix} = \frac{ 1 }{ 4 ( Q_1^2 + Q_2^2 ) } \begin{pmatrix}2 Q_1 & - 2 Q_2 \\ 2 Q_2 & 2 Q_1\end{pmatrix} \begin{pmatrix}P_1 \\ P_2\end{pmatrix} , \ \\] 次節でこの変換の詳細な性質について見るが、ここでは \(\sqrt{ q_1^2 + q_2^2 } = Q_1^2 + Q_2^2\) が成立することを指摘しておく。さて、\(D := 4 ( Q_1^2 + Q_2^2 )\) とおくとき、変換後のハミルトニアン \(\tilde{H} ( \mathbf{P}, \mathbf{Q} ) = H ( \mathbf{p} ( \mathbf{P}, \mathbf{Q} ), \mathbf{q} ( \mathbf{P}, \mathbf{Q} ) )\) は

\[\tilde{H} ( \mathbf{P}, \mathbf{Q} ) = \frac{ 1 }{ 2 D } ( P_1^2 + P_2^2 ) - \frac{ \mu }{ Q_1^2 + Q_2^2 }\] であり、運動方程式は

\[\frac{ d Q_j }{ d t } = \frac{ P_j }{ D } , \ \ \frac{ d P_j }{ d t } = \frac{ 4 }{ D^2 } ( P_1^2 + P_2^2 - 8 \mu ) Q_j\] となる\[9\]。この段階ではまだ \(r \to 0\) での特異性が残っている。

### ステップ2: 時間変数の変換

Levi-Civita変換では物理的な時間 \(t\) の代わりにfictitious time \(s\) を独立変数として扱う。その定義は

\[dt = D ( Q_1, Q_2 ) ds\] である\[10\]。この変換を行うと、上の正準方程式は

\[\frac{ d Q_j }{ d s } = P_j , \ \ \frac{ d P_j }{ d s } = 8 \tilde{H} ( \mathbf{P}, \mathbf{Q} ) Q_j , \ \ \frac{ d t }{ d s } = D\] という方程式系へと変換される。これは極限 \(r = Q_1^2 + Q_2^2 \to 0\) での特異性を持たない。さらに \(\tilde{H}\) が保存量であることから、それが負の値を取る束縛軌道に関しては、これは角振動数 \(\omega = \sqrt{ - 8 \tilde{H} }\) の[調和振動子](../Page/調和振動子.md "wikilink")の方程式に等しい。

なお、この運動方程式は形式的に \(( \mathbf{P}, \mathbf{Q}; T, t )\) を正準変数とするハミルトニアン

\[\Gamma = \frac{ 1 }{ 2 } ( P_1^2 + P_2^2 ) + 4 T ( Q_1^2 + Q_2^2 ) - 4 \mu\] に対応する正準方程式に (\(T\) はもとのハミルトニアン \(\tilde{H}\) の符号を反転させたものと解釈するとき) 一致する\[11\]。

## 変換の性質

### 座標変数 \(Q_j\)

[Animation_of_Levi-Civita_transformation_for_Kepler_orbits.webm](https://ja.wikipedia.org/wiki/File:Animation_of_Levi-Civita_transformation_for_Kepler_orbits.webm "fig:Animation_of_Levi-Civita_transformation_for_Kepler_orbits.webm")と同じ[リサジュー図形](../Page/リサジュー図形.md "wikilink")となる。\]\] Levi-Civita変換による座標の変換 \(( q_1, q_2 ) \mapsto ( Q_1, Q_2 )\) について、その定義は

\[\begin{cases}
q_1 &= Q_1^2 - Q_2^2 \\
q_2 &= 2 Q_1 Q_2
\end{cases}\] であった。あるいは、行列表記ではこの変換は次のように書ける。

\[\begin{pmatrix}q_1 \\ q_2\end{pmatrix} = \begin{pmatrix}Q_1 & - Q_2 \\ Q_2 & Q_1\end{pmatrix} \begin{pmatrix}Q_1 \\ Q_2\end{pmatrix}\]

この変換は、[虚数単位](../Page/虚数単位.md "wikilink") \(i\) を導入すると、次の簡単な等式に書き直すことができる\[12\]。

\[q_1 + i q_2 = ( Q_1 + i Q_2 )^2\] このことは、もとの空間で座標原点 (\(r = 0\)) まわりに一周する (\(\theta = 0\) から \(\theta = 2 \pi\) まで変化する) とき、それはLevi-Civita変数の空間を半周すること (\(\psi = 0 \rightarrow \pi\)) を意味する。従ってひとつの \(( q_1, q_2 )\)-平面の点は \(( Q_1, Q_2 )\)-平面のふたつの点（原点に関して対称な二点）に対応することになる。

### fictitious time \(s\)

fictitious time \(s\) は、[離心近点角](../Page/離心近点角.md "wikilink") \(E\) と \(E = 2 \omega s\) という関係にある\[13\]。

## 応用

### ピタゴラス三体問題

[三体問題](https://ja.wikipedia.org/wiki/三体問題 "wikilink")の特別な初期条件のもとでの系の進化を問う[ピタゴラス三体問題](https://ja.wikipedia.org/wiki/ピタゴラス三体問題 "wikilink")は、系が最終状態に落ち着くまでに二体の近接散乱が繰り返される。SzebehelyとPetersが1967年にこの問題の数値シミュレーションを行った際には、計算精度が落ちるのを防ぐため、また計算時間を削減するために、近接散乱が発生する度にLevi-Civita変換を適用し、信頼できる解を得た\[14\]。
