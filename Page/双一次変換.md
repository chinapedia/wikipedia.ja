**双一次変換**（そういちじへんかん Bilinear transform 双一次Z変換 タスティン変換、台形差分法 Trapezoidal methodとも呼ばれる）は、[デジタル信号処理](../Page/デジタル信号処理.md "wikilink")において、[連続](https://ja.wikipedia.org/wiki/連続 "wikilink")[時間領域](https://ja.wikipedia.org/wiki/時間領域 "wikilink")における[線型](../Page/線型性.md "wikilink")[時不変](https://ja.wikipedia.org/wiki/時不変 "wikilink") ([LTI](https://ja.wikipedia.org/wiki/線形時不変システム "wikilink"))フィルタの[伝達関数](../Page/伝達関数法.md "wikilink")\(H_a(s) \\)([アナログフィルタ](https://ja.wikipedia.org/wiki/アナログフィルタ "wikilink")と呼ばれる)を[離散時間領域における線形シフト不変フィルタの伝達関数](https://ja.wikipedia.org/wiki/離散信号 "wikilink")\(H_d(z) \\)([スイッチトキャパシタ](../Page/スイッチトキャパシタ.md "wikilink")で構成されるアナログフィルタも離散時間フィルタだが、[デジタルフィルタ](https://ja.wikipedia.org/wiki/デジタルフィルタ "wikilink")と呼ばれる)に変換するのによく用いられる[等角写像](../Page/等角写像.md "wikilink")のひとつである。 この変換では、[s平面](https://ja.wikipedia.org/wiki/s平面 "wikilink")上の\(Re[s]=0 \\)、\(j \omega \\)を[z平面](https://ja.wikipedia.org/wiki/z平面 "wikilink")上の\(|z| = 1 \\)の[単位円](../Page/単位円.md "wikilink")に写像する。 双一次変換は元のフィルタの[安定性](https://ja.wikipedia.org/wiki/安定性 "wikilink")を保存し、連続時間フィルタ\(H_a(j \omega_a) \\)の[周波数応答](https://ja.wikipedia.org/wiki/周波数応答 "wikilink")のすべての点を離散時間フィルタ\(H_d(e^{j \omega_d T}) \\)の周波数応答の対応する点に1対1に写像する。ただし以下の周波数歪みの項で述べるようにもとの周波数とはすこし異なる周波数へ写像される。この歪みは低周波域ではほとんど目立たないが、[ナイキスト周波数](../Page/ナイキスト周波数.md "wikilink")に近づくほど顕著になる。

単位インパルスによってサンプリングされた離散時間信号に[ラプラス変換](https://ja.wikipedia.org/wiki/ラプラス変換 "wikilink")を行うと結果は正確に[Z変換](../Page/Z変換.md "wikilink")として表されるが、双一次変換は以下の式に示すように[z平面](https://ja.wikipedia.org/wiki/z平面 "wikilink")から[s平面](https://ja.wikipedia.org/wiki/s平面 "wikilink")への正確な写像を行う自然対数関数の一次近似である。

` &\approx \frac{1 + s T / 2}{1 - s T / 2}`

## 安定性と最小位相の保持

連続時間フィルタはその伝達関数の極が[s平面](https://ja.wikipedia.org/wiki/s平面 "wikilink")上の左半平面にあるとき、[安定である](https://ja.wikipedia.org/wiki/BIBO安定 "wikilink")。離散時間フィルタはその伝達関数の極が[z平面](https://ja.wikipedia.org/wiki/z平面 "wikilink")上の[単位円](../Page/単位円.md "wikilink")内にあるとき安定である。双一次変換はs平面の左半平面をz平面上の単位円内に写像する。このため連続時間領域で設計されたフィルタが安定ならそれを変換した離散時間領域のフィルタも安定である。

同様に、連続時間フィルタが[最小位相](https://ja.wikipedia.org/wiki/最小位相 "wikilink")ならその伝達関数の[零点](../Page/零点.md "wikilink")はs平面上の左半平面にあるため、離散時間フィルタの伝達関数の零点はz平面上の単位円内に位置し、最小位相となる。

## 周波数歪み

連続時間フィルタの周波数応答は、[伝達関数](https://ja.wikipedia.org/wiki/伝達関数 "wikilink")\(H_a(s) \\)で\(s = j \omega \\)とすれば求められる。同様に離散時間フィルタの周波数応答は伝達関数\(H_d(z) \\)でとすれば求められる。双一次変換で設計された離散時間フィルタに実際に\(\omega \\)が入力されたとき、その周波数応答が連続時間フィルタのどの周波数\(\omega_a \\)に対応するかというと、

| -------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| \(H_d(e^{ j \omega T}) \\) | \(= H_a \left( \frac{2}{T} \frac{e^{ j \omega T} - 1}{e^{ j \omega T} + 1}\right) \\)                                                                                                      |
|                            | \(= H_a \left( \frac{2}{T} \cdot \frac{e^{j \omega T/2} \left(e^{j \omega T/2} - e^{-j \omega T/2}\right)}{e^{j \omega T/2} \left(e^{j \omega T/2} + e^{-j \omega T/2 }\right)}\right) \\) |
|                            | \(= H_a \left( \frac{2}{T} \cdot \frac{\left(e^{j \omega T/2} - e^{-j \omega T/2}\right)}{\left(e^{j \omega T/2} + e^{-j \omega T/2 }\right)}\right) \\)                                   |
|                            | \(= H_a \left(j \frac{2}{T} \cdot \frac{ \left(e^{j \omega T/2} - e^{-j \omega T/2}\right) /(2j)}{\left(e^{j \omega T/2} + e^{-j \omega T/2 }\right) / 2}\right) \\)                       |
|                            | \(= H_a \left(j \frac{2}{T} \cdot \frac{ \sin(\omega T/2) }{ \cos(\omega T/2) }\right) \\)                                                                                                 |
|                            | \(= H_a \left(j \frac{2}{T} \cdot \tan \left( \omega \frac{T}{2} \right) \right) \\)                                                                                                       |
|                            | \(= H_a \left(j \omega_a \right) \\)                                                                                                                                                       |

離散時間フィルタは周波数\(\omega \\)において連続時間フィルタの周波数\((2/T) \tan(\omega T/2) \\)での振る舞いと同じ振る舞いをする。特にゲインと位相に関して、離散時間フィルタは周波数\(\omega \\)において連続時間フィルタの周波数\((2/T) \tan(\omega T/2) \\)でのゲイン・位相に等しくなる。その特性の現れる周波数はわずかに異なるが、連続時間フィルタの周波数応答のすべての特徴が離散時間フィルタに現れることを意味する。低周波域においては\(\omega \approx \omega_a \\)となる。

連続時間周波数の範囲  は周波数区間  に写像される。 連続時間フィルタの周波数\(\omega_a = 0 \\)なら対応する離散時間フィルタの周波数\(\omega = 0 \\)となり、連続時間フィルタの周波数\(\omega_a = \pm \infty \\)なら対応する離散時間フィルタの周波数\(\omega = \pm \pi / T\\)となる。

さらに\(\omega_a \\) と \(\omega \\)の関係は非線形であり、これは***周波数歪み***と呼ばれている。 連続時間フィルタの仕様として与えられている周波数（[遮断周波数](https://ja.wikipedia.org/wiki/遮断周波数 "wikilink")や[中心周波数](../Page/中心周波数.md "wikilink")）を、この \(\omega_a = \frac{2}{T} \tan \left( \omega \frac{T}{2} \right) \\) によってあらかじめ補正して設計することもでき、これは***プリワーピング***と呼ばれる。 この周波数歪みによる主な利点は、[インパルス不変法](https://ja.wikipedia.org/wiki/インパルス不変法 "wikilink")で見られるような周波数応答の[エイリアシング](https://ja.wikipedia.org/wiki/エイリアシング "wikilink")が発生しないことである。
