## 詳細

[言語](../Page/言語.md "wikilink")の内容である[思想](../Page/思想.md "wikilink")に重点を置いて体系を立てようとする、[心理主義](../Page/心理主義.md "wikilink")的な[内容主義](https://ja.wikipedia.org/wiki/内容主義 "wikilink")に基づく[論理主義](https://ja.wikipedia.org/wiki/論理主義 "wikilink")的な立場を貫く[理論主義](https://ja.wikipedia.org/wiki/理論主義 "wikilink")的な[本質主義](../Page/本質主義.md "wikilink")を基とする文法理論であり、その骨子は「[文](../Page/文.md "wikilink")の成立の契機とはどのようなものか」と言うことができ、そこから「[意味](https://ja.wikipedia.org/wiki/意味 "wikilink")と職能の上から判断して決定すべきである」という論が生まれる。要するに「意味に基く文法論」として、山田文法は成り立つことになる。そこでは「統覚作用」が重要な役割を果たす。この統覚作用については[ドイツ](https://ja.wikipedia.org/wiki/ドイツ "wikilink")の[ヴントの影響が大きい](../Page/ヴィルヘルム・ヴント.md "wikilink")。

### 語

山田は、独立して何らかの思想を代表し、言語を分解してそれ以上に分けられない究竟の思想の単位を「語」とした。そして、「職能」と「これに対応する意義」と「形式との変化」を基準とし、「職能」と「語の表す思想の内容」により分類する。

まず、大まかに分類すると、次の二種になる。

| 観念語 | 独立の観念を表す |
| --- | -------- |
| 関係語 | 関係を表す    |

これらの内、観念語を「単独で文を形成する骨子になるか、否か」に分類すると、次の二種になる。

| 自用語 | 単独で文を形成する骨子となる   |
| --- | ---------------- |
| 副用語 | 単独で文を形成する骨子とならない |

これらの内、自用語を「概念を表すか、陳述力を持つか」と「語形変化の有無」に分類すると、次の二種になる。

| 概念語 | 概念を表し、語形が変化しない |
| --- | -------------- |
| 陳述語 | 陳述力を持ち、語形が変化する |

また、山田の理論においては、用言と[助動詞の複合体は切り離されず](../Page/助動詞_\(国文法\).md "wikilink")、助動詞は複雑な語尾として「複語尾」と呼ばれた。これについては、「分類があまりにも大雑把過ぎた」という[金水敏](../Page/金水敏.md "wikilink")などの評言がある。

さらに、[接続詞](../Page/接続詞.md "wikilink")を「接続副詞」、[感動詞](../Page/感動詞.md "wikilink")を「感動副詞」と名付け、独立した品詞ではなく[副詞](../Page/副詞.md "wikilink")の一種とした。そして、前者を「語と語の中間に入り、これを結合するもの」と「文の最初にあり、前の文を受けて後ろの文を起こす働きをするもの」に分け、後者を「感情（「驚き」や「嘆き」など）を表すもの」と「意志の動き（「勧誘」や「呼び掛け」など）を表すもの」に分けた。

### 句

山田は、統覚作用が一回のものが単一の思想を表すものとして「句」と呼んだ。この句には次の二種がある。

| [喚体句](https://ja.wikipedia.org/wiki/喚体句 "wikilink") | 未分化        |
| --------------------------------------------------- | ---------- |
| 述体句                                                 | 分析的で、主述の構成 |

そして、単一の句からなる文を「[単文](https://ja.wikipedia.org/wiki/単文 "wikilink")」とし、複数の句からなる文を「[複文](../Page/複文.md "wikilink")」とした。後者は次の三つに分かれる。

| 合文  | 対等の価値を持つ二つ以上の句が合同し、成立した文 | （例）秋になり、葉っぱが色づく   |
| --- | ------------------------ | ----------------- |
| 重文  | 二つ以上の句が並列的に結合した文         | （例）太陽は燃え尽き、月は砕け散る |
| 有属文 | 付属句を持つ文                  | （例）象は鼻が長い         |

なお、付属句とは、主格・述格・修飾格のいずれかの中に、主述の関係を含む文が独立性を失い、他の文の中で一つの語のように働くものをいう。これについては[話題卓越性言語を参照されたい](../Page/主題優勢言語.md "wikilink")。

### 文

山田は、文の成立について、「陳述」という[用語](../Page/用語.md "wikilink")を用いた。この「陳述」というものの性質については後に様々な考察を喚起することになるが、[尾上圭介](https://ja.wikipedia.org/wiki/尾上圭介 "wikilink")による、「山田は『陳述』をテクニカル・タームとしては用いていなかった」という見方もある。「陳述」の内実は今後の[研究](../Page/研究.md "wikilink")をも待つ必要があるかもしれないが、山田が[用言](https://ja.wikipedia.org/wiki/用言 "wikilink")と陳述の関係をどのように見ていたか、簡単に触れる。山田は「文を終止させる用言は陳述を持つ」とした。この陳述は[係り結び](../Page/係り結び.md "wikilink")の現象と深い関係にあり、「結び」とはすなわち陳述であり、「係り」とは陳述との[呼応関係であるとした](https://ja.wikipedia.org/wiki/呼応_\(言語学\) "wikilink")。[連体](https://ja.wikipedia.org/wiki/連体 "wikilink")修飾の用言は陳述を担わないため、係りはそれを超えて文末に影響する。例えば  では「鳥が」は[係助詞を持たないため](https://ja.wikipedia.org/wiki/助詞#係助詞 "wikilink")、連体修飾句の中に影響がとどまるが、  では「鳥は」が係助詞「は」を持つため、連体修飾句を越えて文末の陳述に影響する。この観察は[三上章](../Page/三上章.md "wikilink")、[南不二男](https://ja.wikipedia.org/wiki/南不二男 "wikilink")によってより広範な現象のなかに位置づけられることになるが、そのような研究の端緒として重要な観察である。

また、文の成分について、[助詞](../Page/助詞.md "wikilink")以外の語が、文を組み立てる成分として用いられる時に起こる相互の関係を「格」と呼んだ。山田によると、「格」はすべて観念語の間にあるもので、関係語である助詞は、それ自身で一つの「格」に立つことが無いという。これには次の種類がある。

| 呼格  | 文や句の中にある他の語と、少しも形式上の関係なく立つ格。 |
| --- | ---------------------------- |
| 主格  | 賓位の観念に対立する格。                 |
| 修飾格 | 用言に対する格。                     |
| 述格  | 用言が陳述をなすに用いられる時の格。           |
| 賓格  | いわゆる述語格に当たる格。                |
| 捕格  | 用言だけで十二分に意義を表せない時、それを補充する格。  |
| 連体格 | 体言に対する格。                     |

山田文法については『日本文法論』、『日本文法学概論』で知ることができる。
