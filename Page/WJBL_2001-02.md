**WJBL 2001-02**（第3回Wリーグ）は、[2001年](../Page/2001年.md "wikilink")から[2002年](../Page/2002年.md "wikilink")にかけて開催された[WJBLである](https://ja.wikipedia.org/wiki/バスケットボール女子日本リーグ機構 "wikilink")。

### 入れ替え戦

| 戦 | 勝者               | スコア   | 敗者                    |
| - | ---------------- | ----- | --------------------- |
| 1 | 日立戸塚レパード（Wリーグ7位） |       | トヨタ自動車アンテロープ（W1リーグ2位） |
| 2 | トヨタ自動車アンテロープ     |       | 富士通レッドウェーブ（W1リーグ1位）   |
| 3 | 日立戸塚レパード         | 54-45 | 富士通レッドウェーブ            |

  - 三井生命が休部となったため、日立戸塚、富士通、トヨタ自動車の3チームによる総当たりの入れ替え戦が行われた。その結果、日立戸塚が2勝、トヨタ自動車が1勝1敗、富士通が2敗となり、日立戸塚のWリーグ残留、トヨタ自動車のWリーグ昇格が決まった。しかし、入れ替え戦後に日立戸塚の休部が決定したことにより、富士通がWリーグに昇格した。
