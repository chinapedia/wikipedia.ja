**ケルコゾア**（**アメーバ鞭毛虫**、Cercozoa、ギリシャ語*cercos* '尾' + *zoon* '動物'）は、糸状[仮足](../Page/仮足.md "wikilink")で捕食する[アメーバ](../Page/アメーバ.md "wikilink")様生物および[鞭毛虫](../Page/鞭毛虫.md "wikilink")などから成る[原生生物](../Page/原生生物.md "wikilink")の一群である。糸状仮足は細胞表面の一部分に限られていることもあるが、他の多くの原生動物に見られる真の[細胞口](https://ja.wikipedia.org/wiki/細胞口 "wikilink")は存在しない。ケルコゾアの生物は多様な形態をしており、その一体性は分子系統解析によって強く支持されているにもかかわらず、ケルコゾアという分類群を過不足なく定義できる形態形質は見つかっていない。

## ケルコゾアの主な生物

ケルコゾアで最もよく知られているのは、[ユーグリファ](https://ja.wikipedia.org/wiki/ユーグリファ "wikilink")（ウロコカムリ）類という珪質の鱗片（scale）から成る殻と糸状仮足をもつアメーバ様生物であり、これは土壌や富栄養の水や水生植物の上に普通に見出せる。汚水処理施設における[活性汚泥](../Page/活性汚泥.md "wikilink")微生物として注目されており、[汚水生物系列](https://ja.wikipedia.org/wiki/汚水生物系列 "wikilink")ではβ中腐水性種として扱われている。よく似たものに[tectofilosid](https://ja.wikipedia.org/wiki/tectofilosid "wikilink")類や*[Gromia](https://ja.wikipedia.org/wiki/Gromia "wikilink")*などのように有機的な殻を作るものがあり、以前はユーグリファ類とあわせて[有殻糸状根足虫](../Page/有殻糸状根足虫.md "wikilink")（Testaceafilosia）として分類していた。現在では有殻糸状根足虫を構成する分類群は[単系統](https://ja.wikipedia.org/wiki/単系統 "wikilink")群ではないと考えられているが、そのほとんどはケルコゾアかその近くに置かれている。

もう一つの重要なグループが[クロララクニオン藻](../Page/クロララクニオン藻.md "wikilink")で、これは網状の[群体](../Page/群体.md "wikilink")をつくる奇妙なアメーバ様生物である。この生物が際だっているのは摂食した[緑藻](../Page/緑藻.md "wikilink")に由来した二次共生[葉緑体](../Page/葉緑体.md "wikilink")が存在していることである。葉緑体は四重の膜で区切られており、[ヌクレオモルフ](../Page/ヌクレオモルフ.md "wikilink")と呼ばれる緑藻の[核の痕跡が残っている](https://ja.wikipedia.org/wiki/細胞核 "wikilink")。したがってクロララクニオン藻は細胞内共生起源の[細胞小器官](../Page/細胞小器官.md "wikilink")を調べている研究者たちにとって非常に興味深い。

さらに[ビンカムリ](https://ja.wikipedia.org/wiki/ビンカムリ "wikilink") *Paulinella chromatophora*という葉緑体を持つ生物がある。これはユーグリファ類であるが、[シアネレ](https://ja.wikipedia.org/wiki/シアネレ "wikilink")と呼ばれるソーセージに似た形の葉緑体を持っていて、シアノバクテリアが直接細胞内共生したものである。このシアネレは[アーケプラスチダ](../Page/アーケプラスチダ.md "wikilink")の葉緑体とは無関係であり、一次共生による[光合成](https://ja.wikipedia.org/wiki/光合成 "wikilink")能の獲得が複数回起きた例として注目されている。

その他の有名なものとして土壌によく見られる鞭毛虫の[ケルコモナス](https://ja.wikipedia.org/wiki/ケルコモナス "wikilink")類がある。また伝統的に[太陽虫](../Page/太陽虫.md "wikilink")としてまとめられていた生物のうち、有殻太陽虫類、ディモルファ類、ギムノスファエラ類がケルコゾアに属する。かつて[放散虫](../Page/放散虫.md "wikilink")に含まれていた[ファエオダリア](https://ja.wikipedia.org/wiki/ファエオダリア "wikilink")類（濃彩類、Phaeodarea）もケルコゾアに属する生物群である。[珪酸質](https://ja.wikipedia.org/wiki/珪酸質 "wikilink")の内骨格を持つアメーバ様生物であるエブリア類も、分類上の紆余曲折を経た末、ケルコゾアへの帰属が示された。

## 分類

ケルコゾアは門として扱われることが多い。複雑な殻を持ったアメーバ様生物である[有孔虫](../Page/有孔虫.md "wikilink")および[放散虫](../Page/放散虫.md "wikilink")と近縁で、これらと併せて[リザリア](https://ja.wikipedia.org/wiki/リザリア "wikilink")という上位群を成すが、リザリアの正確な構成生物と類縁性については未だ検討中の段階である。

ケルコゾアの主要な分類群を、[Cavalier-Smith](https://ja.wikipedia.org/wiki/Cavalier-Smith "wikilink") and Chao (2003)に準拠して以下に示す。

phylum Cercozoa ケルコゾア門

  - subphylum Filosa
      - class Chlorarachnea [クロララクニオン藻](../Page/クロララクニオン藻.md "wikilink")綱 （糸状仮足、葉緑体）
      - class Proteomyxidea （糸状仮足、有軸仮足、鞭毛）- 有殻太陽虫類など
      - class Sarcomonadea （糸状仮足、鞭毛）- ケルコモナス類など
      - class Thecofilosea （有機質殻、糸状仮足） - tectofilosid類など
      - class Imbricatea （珪質殻、糸状仮足） - ユーグリファ類など
      - class Phaeodarea ファエオダリア綱 （有軸仮足）

これ以外にケルコゾアへの所属が確定的でないグループがいくつかある。

  - subphylum Endomyxa
      - class [Phytomyxea](https://ja.wikipedia.org/wiki/Phytomyxea "wikilink") （植物寄生性）- [ネコブカビ類](https://ja.wikipedia.org/wiki/ネコブカビ類 "wikilink")など
      - class Ascetosporea [アセトスポラ](../Page/アセトスポラ.md "wikilink")綱（貝類寄生性）
      - class Gromiidea （有機質殻、糸状仮足、多核） - *Gromia*

以上は系統樹によっては有孔虫に近縁な位置を占める。

  - subphylum Filosa
      - class Spongomonadea （鞭毛虫）

これは誤同定した材料に基づいていて、[アメーボゾア](../Page/アメーボゾア.md "wikilink")に所属させる、という意見がある。
