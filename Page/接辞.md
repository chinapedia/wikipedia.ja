**接辞**（せつじ、affix）とは、文法的な機能を担い、それ自体では[語](../Page/語.md "wikilink")として自立しえない[形態素](../Page/形態素.md "wikilink")のことである。

## 基本となる概念

[屈折に関わる接辞を屈折接辞](https://ja.wikipedia.org/wiki/語形変化 "wikilink")、[派生](https://ja.wikipedia.org/wiki/派生 "wikilink")に関わる接辞を派生接辞という。接辞が付く相手を[語基](https://ja.wikipedia.org/wiki/語基 "wikilink")と呼ぶ。特に、屈折接辞が付く語基を[語幹](https://ja.wikipedia.org/wiki/語幹 "wikilink")という。

## 接辞の分類

接辞は語基に対する位置によって、[接頭辞](../Page/接頭辞.md "wikilink")、[接尾辞](../Page/接尾辞.md "wikilink")、[接中辞](https://ja.wikipedia.org/wiki/接中辞 "wikilink")に分類される。接頭辞と接尾辞が二つで一つの機能を持っている場合、それらを合わせて[接周辞](https://ja.wikipedia.org/wiki/接周辞 "wikilink")という。

|                                                     | 位置      | 例                       |                                                                 |
| --------------------------------------------------- | ------- | ----------------------- | --------------------------------------------------------------- |
| [接頭辞](../Page/接頭辞.md "wikilink")                    | 語基の前    | 花 → **お**-花             |                                                                 |
| [接尾辞](../Page/接尾辞.md "wikilink")                    | 語基の後ろ   | 食べ → 食べ-**る**           |                                                                 |
| [接中辞](https://ja.wikipedia.org/wiki/接中辞 "wikilink") | 語基の中    | sulat → s\<**um**\>ulat | [タガログ語](https://ja.wikipedia.org/wiki/タガログ語 "wikilink")《書く（過去）》 |
| [接周辞](https://ja.wikipedia.org/wiki/接周辞 "wikilink") | 語基の前と後ろ | seh → **ge**-seh-**en** | [ドイツ語](../Page/ドイツ語.md "wikilink")《見る（過去分詞）》                    |

屈折には接頭辞よりも接尾辞を主に用いる言語が多い。

語基の中および前後に不連続的に現れる接辞を[貫通接辞](../Page/貫通接辞.md "wikilink")という。例えば[アラビア語](../Page/アラビア語.md "wikilink")の ktb → k\<**a**\>t\<**a**\>b-**a** 《彼は書いた》に見られる <a><a>-a は貫通接辞である。

[複合語](../Page/複合語.md "wikilink")の語基と語基の間に現れる特に意味を持たない接辞を[中間接辞](https://ja.wikipedia.org/wiki/中間接辞 "wikilink")という。例えばドイツ語の Arbeit-**s**-zimmer《書斎》の -s- は中間接辞である。

その他、分節音として取り出すことのできない非連結的形態論を接辞として表現することがある。例えば[アクセント](../Page/アクセント.md "wikilink")や[声調](../Page/声調.md "wikilink")の交替を表す接辞を[超分節接辞](../Page/超分節接辞.md "wikilink")という。
