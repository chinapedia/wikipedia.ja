**三崎稲荷神社**（みさきいなりじんじゃ）は、[東京都](../Page/東京都.md "wikilink")[千代田区](https://ja.wikipedia.org/wiki/千代田区 "wikilink")[神田三崎町](../Page/神田三崎町.md "wikilink")にある神社。創建は、[鎌倉時代](../Page/鎌倉時代.md "wikilink")の建久年間より前と伝えられる。[室町時代](../Page/室町時代.md "wikilink")中期には[本郷にあったことが確認されているが](../Page/本郷_\(文京区\).md "wikilink")，その後，何度か移転を繰り返し1905年（[明治](../Page/明治.md "wikilink")38年）に現在の場所となった。三代将軍[徳川家光](../Page/徳川家光.md "wikilink")が崇敬し、登城する大名は必ず参拝し心身を清めたことから「清めの稲荷」とも称されている\[1\]\[2\]。

## 歴史

この神社の創建時期は不明とされるが、社伝などによれば、[建久](../Page/建久.md "wikilink")（1190年から1199年まで）より前の[仁安](../Page/仁安_\(日本\).md "wikilink")（1166年から1168年まで）頃または建久の頃と伝えられている\[3\]\[4\]。当初は[武蔵国](../Page/武蔵国.md "wikilink")[豊島郡三崎村の鎮守として](../Page/豊島郡_\(武蔵国\).md "wikilink")、神田山（現在の[駿河台](../Page/駿河台.md "wikilink")）の山麓部（[文京区](https://ja.wikipedia.org/wiki/文京区 "wikilink")本郷1丁目付近）に祀られたとされる\[5\]。

小田原の北条氏が江戸城にいた時期、[北条氏綱](../Page/北条氏綱.md "wikilink")（[北条早雲](../Page/北条早雲.md "wikilink")の子）がこの神社を尊宗したと伝わる\[6\]\[7\]\[8\]。1524年（[大永](../Page/大永.md "wikilink")4年）正月、氏綱は数千[歩](https://ja.wikipedia.org/wiki/歩 "wikilink")に及ぶ社地を寄進した\[9\]。その後、北条氏は1538年（天文7年）に社殿を造っている\[10\]。

1590年（天正18年）、[徳川家康](../Page/徳川家康.md "wikilink")が江戸に入城した\[11\]。家康は江戸の開発を計画し、神田山を崩して埋め立てを始めた\[12\]。当時海だった銀座や日比谷は、この時期に陸地となった\[13\]。この埋め立てによって、三崎稲荷神社は現在の文京区後楽に移転した\[14\]。二代将軍[徳川秀忠](../Page/徳川秀忠.md "wikilink")の時期には、神田川の流路付け替えによって社地内が掘割となったため、東南方に移転した\[15\]。

三代将軍徳川家光はこの神社を崇敬したため、参勤交代などで登城する大名は必ず参拝し心身を清めたことから「清めの稲荷」とも称されている\[16\]\[17\]。1659年（万治2年）、[神田川改修に携わった伊達家の家臣](../Page/神田川_\(東京都\).md "wikilink")、[原田甲斐が工事の成功を願って名刀を奉納したという社伝が残る](../Page/原田宗輔.md "wikilink")\[18\]\[19\]。

三崎町という町名は、1872年（明治5年）につけられている\[20\]。その由来は、この神社にあると推定される\[21\]。三崎という名は「岬」からきたものといわれる\[22\]。それは、かつてこの地域が海沿いだったため、岬の部分に神社を勧請したものという\[23\]。同年11月には、村社に列せられている\[24\]。

当初の社殿は神田川南岸の土手上に鎮座し、『[江戸名所図会](../Page/江戸名所図会.md "wikilink")』にも取り上げられていた\[25\]。現在の地に移ったのは1905年（明治38年）のことで、[甲武鉄道](https://ja.wikipedia.org/wiki/甲武鉄道 "wikilink")を[万世橋](../Page/万世橋.md "wikilink")まで延長したためであった\[26\]\[27\]。

その後[関東大震災](../Page/関東大震災.md "wikilink")、昭和期の[太平洋戦争](https://ja.wikipedia.org/wiki/太平洋戦争 "wikilink")によって、本殿を残して建造物はすべて焼失した\[28\]。現存の社殿は、1963年（昭和38年）に復興造営したものである\[29\]。

三崎稲荷神社は、家光や大名たちの故事によって旅行安全の利益があるとされる\[30\]\[31\]。かつて[白瀬矗](https://ja.wikipedia.org/wiki/白瀬矗 "wikilink")は南極探検に出発する際、この神社のお守りを携帯していた\[32\]。そして探検出発前に[大隈重信](../Page/大隈重信.md "wikilink")邸で開かれた決起集会のときには、重信の妻が隊員たちの無事を祈り、真綿のチョッキに三崎稲荷神社のお守りを縫い付けて全員に渡したと伝わる\[33\]。｢お砂守り｣は、現在でも旅の安全のために神社で授かる人々がいる\[34\]。

## 祭礼

三崎稲荷神社の行事は、宮司を中心に氏子総代会や氏子青年会が運営に当たっている。特に、節分祭や例祭、福餅つきなどには、氏子のみならず多くの参拝者が訪れるため、氏子組織が行事の事前準備、警備に大きくかかわることによって継承されてきた\[35\]\[36\]。

主な祭事、年中行事は以下のとおり\[37\]。

| 行事名     | 日付     |
| ------- | ------ |
| 歳旦祭     | 1月1日   |
| 新年祭、祈願祭 | 1月9日   |
| 節分祭     | 2月3日   |
| 例大祭     | 5月9日   |
| 夏越大祓式   | 9月9日   |
| 新嘗祭     | 11月25日 |
| 年越大祓式   | 12月31日 |

例大祭では2年に一度、5月に開催され、神輿渡御が行われる\[38\]。この例大祭は、[神田祭](../Page/神田祭.md "wikilink")の本祭がない年に開催され、氏子町会神輿の連合渡御や本社大神輿が氏子青年会に担がれて氏子9町会を巡り宮入を行う\[39\]\[40\]\[41\]。

## 文化財

参詣者が手や口を清めるために使った江戸時代後期の水盤1基が現存しており，2014年（平成26年）に千代田区指定文化財となった\[42\]。この水盤は，神田の商人2名によって奉納されたものである\[43\]。 [Suiban_misakiinari20200215.jpg](https://ja.wikipedia.org/wiki/File:Suiban_misakiinari20200215.jpg "fig:Suiban_misakiinari20200215.jpg")
