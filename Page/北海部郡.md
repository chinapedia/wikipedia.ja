[frame](https://ja.wikipedia.org/wiki/画像:Oita_KitaAmabe-gun.png "wikilink") **北海部郡**（きたあまべぐん）は、[大分県](../Page/大分県.md "wikilink")にあった[郡](../Page/郡.md "wikilink")。

## 郡域

[明治](../Page/明治.md "wikilink")11年（[1878年](../Page/1878年.md "wikilink")）に行政区画として発足した当時の郡域は、以下の区域にあたる。

  - [大分市](../Page/大分市.md "wikilink")の一部（志村、青崎、須賀、丹生、丹川、宮河内、広内以東）
  - [臼杵市](../Page/臼杵市.md "wikilink")の一部（[野津町](../Page/野津町.md "wikilink")大字各町以外）
  - [津久見市](../Page/津久見市.md "wikilink")の全域

## 歴史

### 江戸時代

[臼杵藩](../Page/臼杵藩.md "wikilink")が藩庁を置き、[佐伯藩](../Page/佐伯藩.md "wikilink")・[熊本藩](../Page/熊本藩.md "wikilink")が領地を有していた。

#### 臼杵藩領

幕府から公認された臼杵藩の御朱印村全279村のうち、海部郡分は97村である。

「旧高旧領取調帳」では、本村を分割したり、枝村のいくつかを村扱にして村高を割り付けているため、数字が1町114村に増えている。臼杵藩では、この御朱印村を数箇村ごとに一まとめにして「村組」という行政上の単位を設けて管理しており、[臼杵城](../Page/臼杵城.md "wikilink")下にあたる町屋鋪村を除いた278村は、いずれもこの村組に属していた。村組は各地域の事情によりたびたび組替えが実施されたため、組数・所属も時期によってまちまちだが、[文政](../Page/文政.md "wikilink")6年（[1823年](../Page/1823年.md "wikilink")）の時点ではほぼ最終的な村組が確定していたと見られている。

#### 佐伯藩領

幕府から公認された佐伯藩の御朱印村でのちに本郡域となる津久見村とのちの[南海部郡](../Page/南海部郡.md "wikilink")にまたがる2村であり、この御朱印村を基本単位として、その下に多数の枝郷が所属している。

| 御朱印村  | 枝郷                   |
| ----- | -------------------- |
| 大坂本村  | 八戸                   |
| 津久見村  | 津久見村組                |
| 津久見浦組 | 岩屋、宮本、井無田、日見浦、福良、網代浦 |
| 上浦村   | 鳩浦、落野浦、深浪津浦、久保泊浦、保戸島 |

枝郷は必要に応じて随時再編されており、その名称・範囲等についても時期によって変動が見られるが、[享和](../Page/享和.md "wikilink")3年（[1803年](../Page/1803年.md "wikilink")）に書かれた「郷村仮名附帳」により、概容を知ることができる。「旧高旧領取調帳」では、こうした枝郷やさらに下位の小字のいくつかを村扱にして村高を割り付けているため、のちの本郡域なる村数が22村と約7倍以上の数になっている。また佐伯藩では、領内の村を農山村である「**在方**」と漁村である「**浦方**」に分類し、それぞれの特質に応じた支配形態をとっていた。御朱印村のうち津久見村については在方と浦方が併在しており、それぞれ津久見村組・津久見浦組として別個に管理されていた。

明治11年（1878年）の海部郡分割に際しては、津久見村組・津久見浦組の全域（臼杵藩と相給の鬼丸村を含む）が本郡の管轄となり、上浦村組・大坂本村組はそれぞれ南海部郡・北海部郡に分割された。

#### 熊本藩領

[豊後国](https://ja.wikipedia.org/wiki/豊後国 "wikilink")内の熊本藩領全63村のうち、海部郡分は22村であり、全村が関[手永](https://ja.wikipedia.org/wiki/手永 "wikilink")に所属した。関手永の御茶屋および会所は関村内の佐賀関町に置かれた。なお「[旧高旧領取調帳](https://ja.wikipedia.org/wiki/旧高旧領取調帳 "wikilink")」では、本村を分割したり、枝村のいくつかを村扱にして村高を割り付けているため、数字が39村に増えている。

| 手永   | 村名                             |
| ---- | ------------------------------ |
| 関手永  | 佐賀郷                            |
| 大佐井郷 | 大西村、角子原村、北村、政所村、横田村、竹下村、浜村、城原村 |

      - 6月1日 - 東大在村・西大在村が合併して**[大在村](https://ja.wikipedia.org/wiki/大在村 "wikilink")**が発足。（2町25村）
      - 7月1日（2町19村）
          - 佐賀村・市村が合併して**[佐賀市村](https://ja.wikipedia.org/wiki/坂ノ市町 "wikilink")**が発足。
          - 神馬木村・大志生木村が合併して**[神崎村](https://ja.wikipedia.org/wiki/神崎村_\(大分県\) "wikilink")**が発足。
          - 上南津留村・中臼杵村が合併して**[南津留村](https://ja.wikipedia.org/wiki/南津留村 "wikilink")**が発足。
          - 臼杵町・下南津留村・市浜村・上浦村が合併し、改めて**[臼杵町](../Page/臼杵市.md "wikilink")**が発足。

  - 大正10年（[1921年](../Page/1921年.md "wikilink")）1月1日 - 津組村が町制施行・改称して**[津久見町](../Page/津久見市.md "wikilink")**（初代）となる。（4町17村）
  - 大正12年（[1923年](../Page/1923年.md "wikilink")）4月1日 - 郡会が廃止。郡役所は存続。
  - 大正15年（[1926年](../Page/1926年.md "wikilink")）7月1日 - 郡役所が廃止。以降は地理区分名称となる。
  - [昭和](../Page/昭和.md "wikilink")3年（[1928年](../Page/1928年.md "wikilink")）4月1日 - 青江村が町制施行して**[青江町](https://ja.wikipedia.org/wiki/青江町 "wikilink")**となる。（5町16村）
  - 昭和8年（[1933年](../Page/1933年.md "wikilink")）4月1日 - 津久見町・青江町・下浦村が合併し、改めて**津久見町**（2代目）が発足。（4町15村）
  - 昭和16年（[1941年](https://ja.wikipedia.org/wiki/1941年 "wikilink")）11月3日 - 坂ノ市町・小佐井村・丹生村が合併し、改めて**坂ノ市町**（2代目）が発足。（4町13村）
  - 昭和17年（[1942年](../Page/1942年.md "wikilink")）7月1日 - 「北海部[地方事務所](../Page/支庁.md "wikilink")」が臼杵町に設置され、本郡を管轄。
  - 昭和25年（[1950年](../Page/1950年.md "wikilink")）
      - 1月1日 - 川添村の所属郡が[大分郡](../Page/大分郡.md "wikilink")に変更。（4町12村）
      - 4月1日 - 臼杵町・海辺村が合併して**[臼杵市](../Page/臼杵市.md "wikilink")**が発足し、郡より離脱。（3町11村）
  - 昭和26年（[1951年](https://ja.wikipedia.org/wiki/1951年 "wikilink")）4月1日 - 津久見町・日代村・四浦村・保戸島村が合併して**[津久見市](../Page/津久見市.md "wikilink")**が発足し、郡より離脱。（2町8村）
  - 昭和29年（[1954年](../Page/1954年.md "wikilink")）3月31日 - 佐志生村・下ノ江村・上北津留村・下北津留村・南津留村が臼杵市に編入。（2町3村）

  - [平成](../Page/平成.md "wikilink")17年（[2005年](../Page/2005年.md "wikilink")）1月1日 - 佐賀関町が大分市に編入。同日北海部郡消滅。大分県内では1878年の郡の再編以来、初の郡消滅となった。
