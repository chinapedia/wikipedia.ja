## フランス語

フランス語では20以上の一部の数詞に[二十進法](../Page/二十進法.md "wikilink")が見られる。[フランス語の数詞](../Page/フランス語の数詞.md "wikilink")を参照。

| 数字 | 読み方    | 備考                     |
| -- | ------ | ---------------------- |
| 0  | zēro   |                        |
| 1  | un     | 「アン」と読む                |
| 2  | deux   | 「ドゥー」と読み、語末のxは発音しない    |
| 3  | trois  | 「トホァ」に近い発音で、語末のsを発音しない |
| 4  | quatre | 「キャットゥㇹ」に近い発音          |
| 5  | cinq   | 「サンク」と似た発音             |
| 6  | six    | 「シス」6からは語末の子音も読む       |
| 7  | sept   | 「セット」に近い発音             |
| 8  | huit   | hを発音せずに「ウィット」          |
| 9  | neuf   | 「ヌフ」に近い発音              |
| 10 | dix    | 「ディス」と読む               |
