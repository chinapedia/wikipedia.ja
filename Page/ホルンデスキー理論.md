**ホルンデスキー理論** (ホルンデスキーりろん、Horndeski theory) は、のうち[ラグランジアン](https://ja.wikipedia.org/wiki/ラグランジアン "wikilink")に高階微分を含むが[運動方程式](https://ja.wikipedia.org/wiki/運動方程式 "wikilink")が2階微分方程式となる最も一般的な[修正重力理論](https://ja.wikipedia.org/wiki/修正重力理論 "wikilink")である。[オストログラドスキー不安定性を自明に回避する健全な理論となっている](../Page/オストログラドスキーの定理.md "wikilink")。1974年に G. W. ホルンデスキーによって提案されたが忘れられ、2010年代になって再発見された。

## 概要

[スカラー場](https://ja.wikipedia.org/wiki/スカラー場 "wikilink") \(\phi\) と[計量](../Page/計量テンソル.md "wikilink") \(g_{\mu \nu}\) からなる[修正重力理論](https://ja.wikipedia.org/wiki/修正重力理論 "wikilink") ([スカラー・テンソル理論](https://ja.wikipedia.org/wiki/スカラー・テンソル理論 "wikilink")) について考える。G. W. ホルンデスキーは[1974年](../Page/1974年.md "wikilink")にラグランジアンに高階微分を含むが運動方程式が2階微分方程式になるようなスカラー・テンソル理論について考察し、その最も一般的なクラスを特定した\[1\]。このクラスの修正重力理論をホルンデスキー理論と呼ぶ。現在ではホルンデスキー理論の[作用は](https://ja.wikipedia.org/wiki/作用_\(物理学\) "wikilink")、自然単位系 \(c = \hbar = 1\) を採用するとき、次の形に書かれることが多い。

\[S = \int \left[ \sum_{i = 2}^5 \mathcal{L}_i [ g_{\mu \nu}, \phi ] \right] \sqrt{ - g } d^4 x\]

\[\mathcal{L}_2 = G_2 ( \phi, X )\]

\[\mathcal{L}_3 = G_3 ( \phi, X ) \Box \phi\]

\[\mathcal{L}_4 = G_4 ( \phi, X ) R + G_{4 X} (\phi, X) \left[ ( \Box \phi )^2 - \phi_{\mu \nu} \phi^{\mu \nu} \right]\]

\[\mathcal{L}_5 = G_5 ( \phi, X ) G_{\mu \nu} \phi^{\mu \nu} - \frac{ 1 }{ 6 } G_{5 X} ( \phi, X ) \left[ ( \Box \phi )^3 + 2 \phi_\mu {}^\nu \phi_\nu {}^\rho \phi_\rho {}^\mu - 3 \phi_{\mu \nu} \phi^{\mu \nu} \Box \phi \right]\] ここに \(X = - \frac{1}{2} \phi_\mu \phi^\mu\)はスカラー場の正準運動項、\(G_i ( \phi, X )\) は任意関数、\(R\) は[リッチテンソル](https://ja.wikipedia.org/wiki/リッチテンソル "wikilink")、\(G_{\mu \nu}\) は[アインシュタインテンソル](https://ja.wikipedia.org/wiki/アインシュタインテンソル "wikilink")、\(\Box \phi = \phi_\mu {}^\mu\) である。また場の量による添え字はその場による微分 (\(G_{i X} = \partial G_i / \partial X\))、スカラー場 \(\phi\) に添え字を付したものは[共変微分](https://ja.wikipedia.org/wiki/共変微分 "wikilink") (\(\phi_\mu = \nabla_\mu \phi\) 等) を表す。

ホルンデスキー理論は提案後長らく忘れ去られていた。[2011年](../Page/2011年.md "wikilink")に Deffayet, Gao, Steer & Zahariade は[ガリレオン理論](https://ja.wikipedia.org/wiki/ガリレオン理論 "wikilink")を[重力を含むように拡張し](../Page/重力場.md "wikilink")、その理論を generalized Galileon 理論と命名した\[2\]。その後すぐに Charmousis, Copeland, Padilla & Saffin がこの理論は Horndeski (1974) により既に与えられていることを指摘し\[3\]、両者の等価性が小林努、山口昌英と横山順一により証明された\[4\]。これによりホルンデスキー理論は最初の提案から35年以上が経過して初めて大きな注目を集めることとなった。

## 他の理論との関係

ホルンデスキー理論は運動方程式が2階微分方程式となる最も一般的なスカラー・テンソル理論であるから、良く知られている他の理論をその特殊な場合として含んでいる\[5\]。なお以下で \(M_\mathrm{Pl} = 1 / \sqrt{ 8 \pi G_\mathrm{N} }\) は[プランク質量](../Page/プランク質量.md "wikilink") (\(G_\mathrm{N}\) は[重力定数](../Page/万有引力定数.md "wikilink")) である.

  - [アインシュタイン重力](https://ja.wikipedia.org/wiki/一般相対性理論 "wikilink"): \(G_2 = G_3 = G_5 = 0\) かつ \(G_4 = M_\mathrm{Pl}^2/2\) のとき.

  - [クインテッセンス](https://ja.wikipedia.org/wiki/クインテッセンス_\(宇宙論\) "wikilink"): \(G_3 = G_5 = 0\) かつ \(G_2 = G_2 ( \phi, X )\), \(G_4 = \frac{1}{2} M_\mathrm{Pl}^2\) のとき.

  - : \(G_3 = G_5 = 0\) かつ \(G_2 = M_\mathrm{Pl}^2 \omega X / \phi - V ( \phi )\), \(G_4 = \frac{1}{2} M_\mathrm{Pl} \phi\) のとき.

  - [f(R)重力](https://ja.wikipedia.org/wiki/f\(R\)重力 "wikilink"): \(G_3 = G_5 = 0\), \(G_2 = -\frac{1}{2} ( R f_R - f )\) かつ \(G_4 = f_R\) のとき.

  - covariant Galileon: \(G_2 = \beta_1 X - m \phi^3\), \(G_3 = \beta_3 X\), \(G_5 = \beta_5 X^2\) かつ \(G_4 = \frac{1}{2} M_\mathrm{Pl}^2 + \beta_4 X^2\) のとき.

  - : \(\xi^{(n)} = \partial^n \xi / \partial \phi^n\) として

\[G_2 = X - V ( \phi ) + 8 \xi^{(4)} ( \phi ) X^2 ( 3 - \ln X), \ \ G_3 = - 4 \xi^{(3)} ( \phi ) X ( 7 - 3 \ln X)\]

\[G_4 = \frac{ 1 }{ 2 } M_\mathrm{Pl}^2 + 4 \xi^{(2)} ( \phi ) X ( 2 - \ln X ), \ \ G_5 = - 4 \xi^{(1)} ( \phi ) \ln X\]

また、これらの理論には[ダークエネルギー](../Page/ダークエネルギー.md "wikilink")の候補となり得るものが含まれるから、ホルンデスキー理論の範疇でダークエネルギーが説明される可能性がある\[6\]。

## beyond Horndeski 理論

当初ホルンデスキー理論は物理的に健全な最も一般的なスカラー・テンソル理論であると思われていたが、しばらくしてホルンデスキー理論の範疇に含まれないが依然として健全なスカラー・テンソル理論が存在することが認識されるようになった\[7\]。このクラスの理論は beyond Horndeski 理論と呼ばれる。

[2013年](../Page/2013年.md "wikilink")に Zumalacárregui & García-Bellido は[ヤコブ・ベッケンシュタイン](https://ja.wikipedia.org/wiki/ヤコブ・ベッケンシュタイン "wikilink")により提案された disformal 変換と呼ばれる計量の変換\[8\]

\[g_{\mu \nu} \mapsto \tilde{g}_{\mu \nu} = C ( \phi, X ) g_{\mu \nu} + D ( \phi, X ) \phi_\mu \phi_\nu\] をホルンデスキー理論に適用すると何が起こるのかを検討した\[9\]。その結果 disformal 変換後の理論は運動方程式に高階微分項が現れるにもかかわらず、系の物理的な自由度が増加しない (従って[オストログラドスキーゴーストが現れない](../Page/オストログラドスキーの定理.md "wikilink")) ことが明らかになった。このことはホルンデスキー理論より広い物理的に健全なスカラー・テンソル理論のクラスが存在することを意味する。実際に Langlois & Noui は[2015年](../Page/2015年.md "wikilink")に高階微分項を含む一般的なスカラー・テンソル理論に対してオストログラドスキー不安定性が現れないようにラグランジアンに縮退条件を課すことによってそのような理論を構成した\[10\]。この理論は degenerate higher-order scalar-tensor 理論 () として知られるものであり、それ以前に発見された[GLPV理論](https://ja.wikipedia.org/wiki/GLPV理論 "wikilink")\[11\]をも包含する。

## 重力波観測による制限

ホルンデスキー理論では[重力波の伝播速度](../Page/重力波_\(相対論\).md "wikilink") \(c_t\) がアインシュタイン重力 (この場合[光速](../Page/光速.md "wikilink")に等しい) から変更され

\[c_t = \frac{ G_4 - X ( \ddot{\phi} G_{5 X} + G_{5 \phi} ) }{ G_4 - 2 X G_{4 X} - X ( H \dot{\phi} G_{5 X} - G_{5 \phi} ) }\] となる\[12\]。重力波速度 \(c_t\) は[aLIGOと](https://ja.wikipedia.org/wiki/LIGO "wikilink")による[中性子星](../Page/中性子星.md "wikilink")連星合体の重力波イベント[GW170817](https://ja.wikipedia.org/wiki/GW170817 "wikilink")と[フェルミ衛星による](../Page/フェルミガンマ線宇宙望遠鏡.md "wikilink")[ガンマ線バースト](../Page/ガンマ線バースト.md "wikilink")観測の比較から

\[| c_t^2 - 1 | < 6 \times 10^{-15}\] という極めて強い制限が得られている\[13\]。この結果 \(G_{5}\) 項の存在および \(G_{4}\) の \(X\)-依存性が棄却され、ホルンデスキー理論の範疇では観測的に許される理論は

\[\mathcal{L} = G_4 ( \phi ) R + G_2 ( \phi, X ) - G_3 ( \phi, X ) \Box \phi\] という形に限られることになる\[14\]\[15\]。
