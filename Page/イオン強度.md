**イオン強度**（いおんきょうど）とは、電解質溶液の[活量係数](https://ja.wikipedia.org/wiki/活量係数 "wikilink")とイオン間の相互作用を関係づけるための概念で、溶液中のすべての[イオン](../Page/イオン.md "wikilink")種について、それぞれのイオンの[モル濃度](../Page/モル濃度.md "wikilink")\(m_i\)と[電荷](../Page/電荷.md "wikilink")\(z_i\)の2乗の積を加え合わせ、さらにそれを1/2にしたものである。例えば、2価の[陽イオン](https://ja.wikipedia.org/wiki/陽イオン "wikilink")と2価の[陰イオン](https://ja.wikipedia.org/wiki/陰イオン "wikilink")から成る[電解質](../Page/電解質.md "wikilink")なら[モル濃度](../Page/モル濃度.md "wikilink")の4倍の値となる。

  -
    \(I = \frac{1}{2} \sum_{i} m_iz_i^2\)

例

  -
    0.1 mol dm<sup>−3</sup> [NaCl](../Page/塩化ナトリウム.md "wikilink")(aq) ; *I* = 0.5×(0.1×1<sup>2</sup> + 0.1×(−1)<sup>2</sup>) = 0.1
    0.1 mol dm<sup>−3</sup> [MgCl<sub>2</sub>](../Page/塩化マグネシウム.md "wikilink")(aq) ; *I* = 0.5×(0.1×2<sup>2</sup> + 0.2×(−1)<sup>2</sup>) = 0.3
    0.1 mol dm<sup>−3</sup> [Na<sub>2</sub>SO<sub>4</sub>](../Page/硫酸ナトリウム.md "wikilink")(aq) ; *I* = 0.5×(0.2×1<sup>2</sup> + 0.1×(−2)<sup>2</sup>) = 0.3
    0.1 mol dm<sup>−3</sup> [MgSO<sub>4</sub>](../Page/硫酸マグネシウム.md "wikilink")(aq) ; *I* = 0.5×(0.1×2<sup>2</sup> + 0.1×(−2)<sup>2</sup>) = 0.4
    0.1 mol dm<sup>−3</sup> [Al<sub>2</sub>(SO<sub>4</sub>)<sub>3</sub>](../Page/硫酸アルミニウム.md "wikilink")(aq) ; *I* = 0.5×(0.2×3<sup>2</sup> + 0.3×(−2)<sup>2</sup>) = 1.5

[モル濃度](../Page/モル濃度.md "wikilink")（\(\rm mol{\cdot}dm^{-3}\)）でも成り立つが、質量モル濃度 （\(\rm mol{\cdot}kg^{-1}\)）でも成り立つ。[デバイ-ヒュッケルの式](../Page/デバイ-ヒュッケルの式.md "wikilink")にもとづく。

イオン強度は溶液中におけるイオンの[活量](../Page/活量.md "wikilink")に著しく影響を与えるものである。従って[酸解離定数](../Page/酸解離定数.md "wikilink")および[錯生成定数](https://ja.wikipedia.org/wiki/錯生成定数 "wikilink")などイオンの濃度に関わるものはイオン強度の影響を大きく受け、実験では[過塩素酸ナトリウム](https://ja.wikipedia.org/wiki/過塩素酸ナトリウム "wikilink")などの強電解質を用いてイオン強度を一定に保った上で条件定数が測定される。[熱力学](../Page/熱力学.md "wikilink")的な[平衡定数](../Page/平衡定数.md "wikilink")はイオン強度が0、すなわち無限希釈におけるものであり、異なるイオン強度における測定値をイオン強度=0に外挿して求められる。
