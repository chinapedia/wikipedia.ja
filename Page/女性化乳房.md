**女性化乳房**（じょせいかにゅうぼう、）は、男性の胸が女性の[乳房](../Page/乳房.md "wikilink")のように隆起する症状。それに伴い[乳首](../Page/乳首.md "wikilink")、[乳輪](../Page/乳輪.md "wikilink")も[女性化](https://ja.wikipedia.org/wiki/女性化 "wikilink")し、まれに[母乳](../Page/母乳.md "wikilink")のような[乳汁](https://ja.wikipedia.org/wiki/乳汁 "wikilink")分泌物があふれる場合もあるが個人差が出る。**女性化乳腺**（じょせいかにゅうせん）とも呼ばれる。最も多いのは、特に理由のない「特発性女性化乳房」であり、健康上の問題はなく、乳房が極端に大きくならないかぎり放置されることも多い。しかし[乳癌](../Page/乳癌.md "wikilink")のおそれや、後述の通り[精巣腫瘍](https://ja.wikipedia.org/wiki/精巣腫瘍 "wikilink")や肝機能低下の症状である可能性もある。

大学病院や一般病院では乳腺[外科や](https://ja.wikipedia.org/wiki/外科学 "wikilink")[形成外科が診療にあたり](../Page/形成外科学.md "wikilink")、健康保険が適用されれば[生命保険](https://ja.wikipedia.org/wiki/生命保険 "wikilink")の手術給付金の対象となり、入院すればそれも給付される（加入している生命保険の給付条件による）。健康保険が適用されると4日ほどの入院で約11万円前後と言われている。[自由診療](https://ja.wikipedia.org/wiki/自由診療 "wikilink")報酬での手術（美容整形外科）では自己負担が70万円以上となる場合がある。

## 症状

[thumb](https://ja.wikipedia.org/wiki/画像:Tanner_scale-female.svg "wikilink") 女性化乳房は女性の胸の成長と同じように[乳腺](../Page/乳腺.md "wikilink")の発達で起こる。乳首の中にしこりができたようになり痛みもあるため乳癌と間違うこともある。男性が「乳首にしこりができた」と病院で診察を受けると女性化乳房であることがある。

[思春期](../Page/思春期.md "wikilink")初期の男子にも見られ、女子の乳房の[タナー段階](https://ja.wikipedia.org/wiki/タナー段階 "wikilink")III並になることも珍しくない。これは思春期初期に[男性ホルモン](https://ja.wikipedia.org/wiki/男性ホルモン "wikilink")に比べて[女性ホルモン](https://ja.wikipedia.org/wiki/女性ホルモン "wikilink")の分泌が相対的に多くなるためで、後に男性ホルモンの分泌が増えてくると1-2[年](../Page/年.md "wikilink")で消失する\[1\]。

乳首・乳輪付近に何かが接触したり体を揺すったりすると[疼痛](../Page/疼痛.md "wikilink")や[痒み](../Page/痒み.md "wikilink")が生じたり、運動時などに[乳揺れ](../Page/乳揺れ.md "wikilink")が生じる為に[ブラジャー](../Page/ブラジャー.md "wikilink")などが必要になる例もある。思春期初期の男子を除いて女性のブラジャーを着用時、アンダーバストが足りない為、[メンズブラ](../Page/メンズブラ.md "wikilink")を着用する場合がある。その他にアンダーバストの制限が無い胸部分が2重になっている[タンクトップ](../Page/タンクトップ.md "wikilink")・[ハーフトップ](../Page/ハーフトップ.md "wikilink")・[キャミソール](../Page/キャミソール.md "wikilink")などを着る場合や[ニプレス](../Page/ニプレス.md "wikilink")などを貼り付ける場合もある。

## 原因

  - ホルモンバランスの乱れ

:; 肝機能の低下

:: 男性にも女性ホルモンがあるが、[肝臓](../Page/肝臓.md "wikilink")で分解・処理される。[肝機能](https://ja.wikipedia.org/wiki/肝機能 "wikilink")が低下すると、女性ホルモンが増える。

:; 女性ホルモンに似た働きをする薬物の摂取

:: 一部の薬には女性ホルモンに似た作用がある。例として[前立腺疾患](https://ja.wikipedia.org/wiki/前立腺疾患 "wikilink")治療薬、[男性型禿髪症治療薬などが挙げられる](../Page/男性型脱毛症.md "wikilink")。[リスペリドン](../Page/リスペリドン.md "wikilink")など[抗精神病薬](../Page/抗精神病薬.md "wikilink")が[高プロラクチン血症](https://ja.wikipedia.org/wiki/高プロラクチン血症 "wikilink")を起こすことも原因となる。

:; 男性ホルモンの過剰投与

:: いわゆる[ドーピング](../Page/ドーピング.md "wikilink")において男性ホルモンを過剰投与した場合、男性ホルモンが女性ホルモンに[代謝](../Page/代謝.md "wikilink")されること（[芳香化](https://ja.wikipedia.org/wiki/芳香化 "wikilink")）があるため、男性の乳房が女性化する。

:; 女性ホルモンの過剰投与

:: こちらもドーピングにおいて女性ホルモンを過剰投与した場合、乳腺が刺激されて胸が発達（[ホルモン胸](https://ja.wikipedia.org/wiki/ホルモン胸 "wikilink")）する。

:; 思春期や[更年期](https://ja.wikipedia.org/wiki/更年期 "wikilink")のホルモンバランスの乱れ

:: 思春期・更年期にはホルモンバランスが乱れることがある。

:; 脳腫瘍

:: [脳腫瘍](../Page/脳腫瘍.md "wikilink")のできる部位によっては、[脳下垂体](../Page/脳下垂体.md "wikilink")で生成されるホルモンの[プロラクチン](../Page/プロラクチン.md "wikilink")量が増え、乳房の発達は誘発される。

:; 前立腺肥大/前立腺癌

  -

      -
        [前立腺肥大](https://ja.wikipedia.org/wiki/前立腺肥大 "wikilink")（良性）と[前立腺癌](../Page/前立腺癌.md "wikilink")（悪性）の人は、そうでない人より黒線が多い\[2\]。

  - その他

:; 遺伝的要因

:: [クラインフェルター症候群](../Page/クラインフェルター症候群.md "wikilink")、[アロマターゼ過剰症](https://ja.wikipedia.org/wiki/アロマターゼ過剰症 "wikilink")など。

:; 原因不明（特発性女性化乳房）

:: 顕著なホルモンバランスの崩れがみられないにもかかわらず、男性の乳房が肥大することがある。成人の女性化乳房の症例では、この特発性が最も多い。

## 治療

消失しない場合の根治療法は手術による切除となる。手術は乳輪の半周を切開し、そこから乳腺と脂肪を除去する。[自由診療](https://ja.wikipedia.org/wiki/自由診療 "wikilink")の美容整形外科病院の場合は部分麻酔で日帰り手術が主体のようだが、大学病院や一般病院では大きさによっては全身麻酔で、4日〜7日間の入院が必要となる場合もある。術後数日間はドレン（術後創部に血が溜まるのを防ぐため体外に排出する管）を留置する場合が多いので、入院のほうが安心であると言える。

術後再び腫れることもあるが再発ではなく、時間が経つにつれ約1ヶ月程度で腫れは引き、乳輪が出た状態は消える。腫れを予防するには圧迫することが重要と言われている。

### 偽性女性化乳房

美容整形外科によっては、肥満で乳部が脂肪により肥大化することを「偽性女性化乳房」と呼び、本来の女性化乳房を「真性女性化乳房」と呼ぶ。また両方の発達による混合タイプもある。

  - 偽性女性化乳房 - 皮下脂肪が多い事が原因。治療は脂肪吸引を行う。
  - 真性女性化乳房 - 乳腺の発達が原因。通常の保険診療同様の乳腺切除法を行う。
  - 混合タイプ - 上記の2つを行う。

乳房の下垂が強い場合には乳房切断術および乳輪・乳頭の移植術を行うこともある\[3\] 。
