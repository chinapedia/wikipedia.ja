**平均赤血球容積**（**へいきんせっけっきゅうようせき**、）は、[血液検査](../Page/血液検査.md "wikilink")の項目のひとつ。[赤血球](../Page/赤血球.md "wikilink")の大きさを表し、単位はfl（[フェムト](https://ja.wikipedia.org/wiki/フェムト "wikilink")リットル）である。

## 意義

[大球性貧血](https://ja.wikipedia.org/wiki/大球性貧血 "wikilink")、[正球性貧血](https://ja.wikipedia.org/wiki/正球性貧血 "wikilink")、[小球性貧血](https://ja.wikipedia.org/wiki/小球性貧血 "wikilink")を見分ける際に用いられる。

## 計算法

その名の示すとおり赤血球の容積の平均値であり、[ヘマトクリット](https://ja.wikipedia.org/wiki/ヘマトクリット "wikilink")（血液に占める赤血球の体積の割合）および[赤血球数](https://ja.wikipedia.org/wiki/赤血球数 "wikilink")（単位体積あたりの血液に含まれる赤血球の個数）から計算することが出来る。

今、平均赤血球容積をMCV(fl)、ヘマトクリットをHt(%)、赤血球数をRBC×10<sup>6</sup>(/μl)とすると、平均赤血球容積は

|                |                                                                        |
| -------------- | ---------------------------------------------------------------------- |
| \(\mbox{MCV}\) | \(=\frac{\mbox{Ht} / 100}{\mbox{RBC} \times 10^6 \mathrm{(/ \mu l)}}\) |
|                | \(=\frac{\mbox{Ht} / 100}{\mbox{RBC} \times 10^{-3} \mathrm{(/ fl)}}\) |
|                | \(=\frac{\mbox{Ht}}{\mbox{RBC}}\times 10 \mathrm{(fl)}\)               |

と表せる。

例えば、[血液検査](../Page/血液検査.md "wikilink")でHtが40(%)、RBCが4.5×10<sup>6</sup>(/μ*l*)であった場合、 \(\mbox{MCV} = \frac{\mbox{Ht}}{\mbox{RBC}}\times 10 = \frac{40}{4.5}\times 10 \fallingdotseq 89 \mathrm{(fl)}\) と計算することが出来る。

## 正常値

平均赤血球容積の基準値はおおむね80～100 flである。

平均赤血球容積がこの範囲内に収まっている貧血を正球性貧血、80 flより小さい貧血を小球性貧血、100 flを超える貧血を大球性貧血と呼ぶ。平均赤血球容積が120 flを超すような大球性貧血は、[抗がん剤](https://ja.wikipedia.org/wiki/抗がん剤 "wikilink")などの薬物を使用していない限り、ほとんどが[ビタミンB12](https://ja.wikipedia.org/wiki/ビタミンB12 "wikilink")あるいは[葉酸](https://ja.wikipedia.org/wiki/葉酸 "wikilink")欠乏による\[1\]。[ヘリコバクター・ピロリ](https://ja.wikipedia.org/wiki/ヘリコバクター・ピロリ "wikilink")の感染による[萎縮性胃炎](https://ja.wikipedia.org/wiki/萎縮性胃炎 "wikilink")に伴う[内因子](https://ja.wikipedia.org/wiki/内因子 "wikilink")の分泌低下によりビタミンB12吸収障害が起こり[巨赤芽球性貧血](https://ja.wikipedia.org/wiki/巨赤芽球性貧血 "wikilink")がもたらされて平均赤血球容積は高値を呈する\[2\]。
