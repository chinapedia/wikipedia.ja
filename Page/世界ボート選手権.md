**世界ボート選手権**は[FISA](https://ja.wikipedia.org/wiki/FISA "wikilink")（Fédération Internationale des Sociétés d'Aviron；the International Rowing Federation）の主催する国際的な[ボート競技](../Page/ボート競技.md "wikilink")の大会（[レガッタ](../Page/レガッタ.md "wikilink")）である。ボート競技の中では[夏季オリンピックに次ぐ権威を持つ大会である](https://ja.wikipedia.org/wiki/オリンピックボート競技 "wikilink")。

## 歴史

第一回大会は[1962年](../Page/1962年.md "wikilink")に[スイス](../Page/スイス.md "wikilink")の[ルツェルン](../Page/ルツェルン.md "wikilink")で開催された\[1\]\[2\]。[1974年](../Page/1974年.md "wikilink")までは4年ごとに開催されていたが、その後は毎年開催されるようになった。また[1974年](../Page/1974年.md "wikilink")は、女子種目と男子軽量級種目が初めて開催された年でもある。女子軽量級種目は[1985年](https://ja.wikipedia.org/wiki/1985年 "wikilink")に加えられた。2002年には身体障害者を対象とした種目が加えられた。

## 種目

以下の23種目が開催される。但し、[夏季オリンピックが開催される年には](https://ja.wikipedia.org/wiki/オリンピックボート競技 "wikilink")[オリンピックにない種目のみが行われる](https://ja.wikipedia.org/wiki/オリンピックボート競技 "wikilink")。選手たちは[オリンピックに一番重きを置いていることが多いため](https://ja.wikipedia.org/wiki/オリンピックボート競技 "wikilink")、[オリンピックで開催されない種目は軽視されることが多い](https://ja.wikipedia.org/wiki/オリンピックボート競技 "wikilink")。

以下の表は、Oのつくものが[オリンピックと世界ボート選手権で開催される種目を](https://ja.wikipedia.org/wiki/オリンピックボート競技 "wikilink")、WCのつくものが世界ボート選手権のみで開催される種目をあらわしている。

| 種目 | 男子        | 軽量級男子 | 女子 | 軽量級女子 |
| -- | --------- | ----- | -- | ----- |
| 1x | シングルスカル   | O     | WC | O     |
| 2x | ダブルスカル    | O     | O  | O     |
| 2- | 舵手なしペア    | O     | WC | O     |
| 2+ | 舵手つきペア    | WC    |    |       |
| 4x | クォドルプルスカル | O     | WC | O     |
| 4- | 舵手なしフォア   | O     | O  | WC    |
| 4+ | 舵手つきフォア   | WC    |    |       |
| 8+ | エイト       | O     | WC | O     |
