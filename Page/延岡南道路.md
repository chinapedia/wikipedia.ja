**延岡南道路**（のべおかみなみどうろ）は、[宮崎県](../Page/宮崎県.md "wikilink")[延岡市](../Page/延岡市.md "wikilink")と[東臼杵郡](../Page/東臼杵郡.md "wikilink")[門川町](../Page/門川町.md "wikilink")を結ぶ延長6.1kmの[国道10号](../Page/国道10号.md "wikilink")[バイパスである](../Page/バイパス道路.md "wikilink")。延岡南IC - 門川IC間は[東九州自動車道](../Page/東九州自動車道.md "wikilink")に並行する[一般国道自動車専用道路であり](../Page/高速自動車国道に並行する一般国道自動車専用道路.md "wikilink")、この区間は[西日本高速道路](../Page/西日本高速道路.md "wikilink")が管理している。[全国路線網](https://ja.wikipedia.org/wiki/全国路線網 "wikilink")の1つ。

[延岡IC](../Page/延岡インターチェンジ.md "wikilink")/[JCT](../Page/延岡ジャンクション.md "wikilink") - [北川IC間が開通してからは](../Page/北川インターチェンジ.md "wikilink")、同路線は「東九州自動車道」（または「東九州道」）という[営業路線名で案内されている](https://ja.wikipedia.org/wiki/線名 "wikilink")。（ただし、[標識によっては法定路線名通りの](../Page/道路標識.md "wikilink")「延岡南道路」で案内されているものもある）

## 概要

  - 起点：宮崎県延岡市塩浜町一丁目
  - 終点：宮崎県東臼杵郡門川町大字加草
  - 全長：6.1km（うち有料区間3.7km）
  - 規格：第1種第3級
  - 道路幅員：
  - 車線数：[暫定2車線](../Page/暫定2車線.md "wikilink")
  - 車線幅員：
  - 設計速度：80km/h

## 沿革

  - [1990年](https://ja.wikipedia.org/wiki/1990年 "wikilink")[2月21日](../Page/2月21日.md "wikilink")：全線開通。
  - [2009年](../Page/2009年.md "wikilink")[1月21日](../Page/1月21日.md "wikilink")正午：[ETC](../Page/ETC.md "wikilink")運用開始
  - [2020年](../Page/2020年.md "wikilink")[3月30日](../Page/3月30日.md "wikilink")：延岡南IC料金所運用開始。料金値下げのほか、中型車以上を対象とする特例料金の運用を開始\[1\]。

## 料金

2018年3月時点での料金および収受方法は以下の通り\[2\]。料金は門川本線料金所で徴収を行っており、上り線（大分方向）では、門川TBまでの対距離料金と延岡南道路の料金が合算されて徴収、下り線（宮崎方向）では、現金車は延岡南道路の料金を払い、通行券を受け取る。

なお、2020年3月30日に、ETC車限定で料金体系を全国同等の金額に引き下げた。

| 料金体系/車種別     | 軽等   | 普通   | 中型   | 大型 | 特大 |
| ------------ | ---- | ---- | ---- | -- | -- |
| 改定前（改定後の現金車） | 260円 | 410円 | 930円 |    |    |
| 改定後のETC車     | 240円 | 280円 | 350円 |    |    |

2020年3月30日より、延岡南IC周辺の生活道路への大型車流入対策の一環として\[3\]、また、中型車以上で、延岡南道路を避けて延岡南ICの大分方向（[延岡道路](../Page/延岡道路.md "wikilink")）と、門川南SICもしくは日向ICを出入りした場合に「延岡南道路を使用した」とみなして通行料金を徴収する\[4\]\[5\]。これに伴い、延岡南ICに料金所が設置\[6\]\[7\]。（料金収受方法の詳細は[延岡南IC\#料金所を参照](https://ja.wikipedia.org/wiki/延岡南インターチェンジ#%E6%96%99%E9%87%91%E6%89%80 "wikilink")）
