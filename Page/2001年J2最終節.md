**2001年J2最終節**（2001ねん・J2・さいしゅうせつ）では、[2001年](../Page/2001年.md "wikilink")[11月18日](../Page/11月18日.md "wikilink")に行われた[日本プロサッカーリーグ](../Page/日本プロサッカーリーグ.md "wikilink")（Jリーグ）[ディビジョン2 (J2)最終節](https://ja.wikipedia.org/wiki/Jリーグ_ディビジョン2 "wikilink")（第44節）のことを記す。本項では特に、2位以内となり[Jリーグ ディビジョン1 (J1)昇格の可能性を残した](https://ja.wikipedia.org/wiki/Jリーグ_ディビジョン1 "wikilink")[モンテディオ山形](https://ja.wikipedia.org/wiki/モンテディオ山形 "wikilink")・[ベガルタ仙台](../Page/ベガルタ仙台.md "wikilink")・[大分トリニータ](https://ja.wikipedia.org/wiki/大分トリニータ "wikilink")の3チームの試合について記す。

なお、正式には「J2からJ1への昇格」はリーグ終了後のJリーグ理事会での協議で決定するが、2013年までに成績条件を満たしたクラブが理事会でJ1昇格が否決されたケースがないため、便宜上「J2で2位以内（成績条件）=J1昇格」として扱うものとする。

## 最終戦までの経緯

二部制導入3シーズン目となった[2001年のJ2は](https://ja.wikipedia.org/wiki/2001年のJリーグ_ディビジョン2 "wikilink")、[前シーズンのJ1年間成績下位](https://ja.wikipedia.org/wiki/2000年のJリーグ_ディビジョン1 "wikilink")2チームの[京都パープルサンガと](../Page/京都サンガF.C..md "wikilink")[川崎フロンターレ](https://ja.wikipedia.org/wiki/川崎フロンターレ "wikilink")、[前年の日本フットボールリーグ (JFL)](../Page/第2回日本フットボールリーグ.md "wikilink") 優勝により昇格した[横浜FC](../Page/横浜FC.md "wikilink")を加えた12チームによる4回戦総当たり（各チーム44試合・全44節）により行われた。

J1への昇格争いは、降格した京都、川崎、前シーズン及び前々シーズンと2年続けてを勝ち点差1により昇格を逃した大分の3チームが中心となると見られていた。

大分は元[ベルギー代表の](../Page/サッカーベルギー代表.md "wikilink")[ロレンツォ・スターレンス](../Page/ロレンツォ・スターレンス.md "wikilink")を獲得するなど戦力補強を行い、3年越しのJ1昇格を目指したものの開幕から思うように勝てず、第11節でモンテディオ山形に敗れると[石崎信弘](../Page/石崎信弘.md "wikilink")監督が解任された。後任には[小林伸二](../Page/小林伸二.md "wikilink")が昇格したが監督交代後も苦戦が続いた。 シーズン中盤までは元[コンサドーレ札幌の](https://ja.wikipedia.org/wiki/北海道コンサドーレ札幌 "wikilink")[フォワード](../Page/フォワード_\(サッカー\).md "wikilink")・[バルデスを補強した](../Page/ホルヘ・ルイス・デリー・バルデス.md "wikilink")[大宮アルディージャ](https://ja.wikipedia.org/wiki/大宮アルディージャ "wikilink")が一歩抜け出し、一時は2位の京都に勝点8差を付けた。しかし、バルデスは[パナマ代表の試合で故障](../Page/サッカーパナマ代表.md "wikilink")、さらにもう一人の得点源、[ジョルジーニョも怪我で戦線離脱](../Page/ジョルジェ・ルイス・デ・アモリン・シルヴァ.md "wikilink")、共にシーズン絶望の大怪我。得点源を失った大宮は一気に失速した。

代わって京都が首位に立ち40節までその座を保ち続けたが第41節で横浜FCに勝利した仙台が6試合負けなし（5勝1分）で首位に立ち、初のJ1昇格に向けて大きく前進したが、第42節の[ヴァンフォーレ甲府](https://ja.wikipedia.org/wiki/ヴァンフォーレ甲府 "wikilink")、第43節の[サガン鳥栖](https://ja.wikipedia.org/wiki/サガン鳥栖 "wikilink")と下位2チームに連敗。その間に京都は白星を重ね、第43節の[湘南ベルマーレ](https://ja.wikipedia.org/wiki/湘南ベルマーレ "wikilink")戦に勝利したことにより京都のJ2優勝およびJ1昇格が決まった。さらに連敗の仙台に代わって山形が仙台と同勝点ながら得失点差で2位となり、仙台は3位に後退。大分は終盤で3連勝したことで辛うじて昇格の可能性を残し、43節終了時点でJ1昇格の可能性があるチームは山形・仙台・大分の3チームに絞られた。

## 最終節

### 概要

2001年のJ2最終節。昇格の可能性を持つ3チームは、同時刻に以下の試合を行うことになった。

  - **モンテディオ山形** - 川崎フロンターレ（[山形県総合運動公園陸上競技場](../Page/山形県総合運動公園陸上競技場.md "wikilink")）
  - 京都パープルサンガ - **ベガルタ仙台**（[西京極総合運動公園陸上競技場兼球技場](https://ja.wikipedia.org/wiki/西京極総合運動公園陸上競技場兼球技場 "wikilink")）
  - サガン鳥栖 - **大分トリニータ**（[鳥栖スタジアム](../Page/鳥栖スタジアム.md "wikilink")）

山形はホームで、仙台と大分はアウェーでの試合となった。また、山形と仙台は前年降格チームとの対戦で、大分は[九州ダービー](https://ja.wikipedia.org/wiki/九州ダービー "wikilink")により昇格を争うこととなった。

各チームの昇格条件は、山形と仙台の勝敗に大きく委ねられることとなった。

条件が最も厳しかったのが大分で、「90分で勝利し、かつ山形と仙台が共に敗戦」という条件以外に昇格の可能性はなかった。

### 前半

鳥栖スタジアムでは鳥栖が前半12分[森田浩史](../Page/森田浩史.md "wikilink")のゴールで先制し、そのまま前半終了。山形と京都では得点の動きなし。

  -
    **山形 0 – 0 川崎**
    **京都 0 – 0 仙台**
    **鳥栖 1 – 0 大分**

### 後半

鳥栖スタジアムでは後半30分、[船越優蔵](../Page/船越優蔵.md "wikilink")のゴールが決まり、大分が同点に追い着く。

  -
    **山形 0 – 0 川崎**
    **京都 0 – 0 仙台**
    **鳥栖 1 – 1 大分**

この後は3会場とも得点の動きなく終盤へ。しかし西京極では後半終了直前、[岩本輝雄](../Page/岩本輝雄.md "wikilink")が左サイドから上げたクロスを[山田隆裕](../Page/山田隆裕.md "wikilink")が頭で折り返し、これを[財前宣之](../Page/財前宣之.md "wikilink")が右足ボレーシュート。これが京都ゴールに突き刺さり待望の先制点を挙げた。

  -
    **山形 0 – 0 川崎**
    **京都 0 – 1 仙台**
    **鳥栖 1 – 1 大分**

仙台はこの後のロスタイムを落ち着いて守り切り、1-0で試合終了。90分以内での勝利を収め、最終勝点を「83」とした。一方、山形-川崎戦は0-0のまま後半を終了し、鳥栖-大分戦も1-1のまま延長戦に突入。この結果、3チームのうち唯一90分で勝利した**仙台の2位が確定し、悲願のJ1初昇格が決定**。遠く仙台から駆けつけたサポーターと昇格の喜びを分かち合った。

  -
    **山形 0 – 0 川崎**
    **京都 0 – 1 仙台**（試合終了）
    **鳥栖 1 – 1 大分**

### 延長戦

山形は昇格が潰えたことで気落ちしたのか、延長前半開始直後に[我那覇和樹](../Page/我那覇和樹.md "wikilink")にVゴールを決められ、敗戦。最終勝点は「80」のままで3位に終わった。一方、鳥栖‐大分戦は互いに譲らず、1-1のまま引き分けに終わり、最終節でいずれも延長勝利した新潟、大宮に勝ち点で並ばれ、得失点差の関係で6位まで順位を下げた。

  -
    **山形 0 – 1v 川崎**（試合終了）
    **京都 0 – 1 仙台**（試合終了）
    **鳥栖 1 – 1 大分**（試合終了）

## その後

ちなみに、J2ではこの年を以って延長戦を廃止。翌[2002年からは](https://ja.wikipedia.org/wiki/2002年のJリーグ#J2リーグ "wikilink")90分で決着しない場合は引き分けとなり、両チームに勝点1ずつを与える方式に変わった（J1もその翌年から延長戦が廃止された）。
