**スーゴ・アッラ・ジェノヴェーゼ**（）は、[牛肉](../Page/牛肉.md "wikilink")と[タマネギ](../Page/タマネギ.md "wikilink")を基本とするの[ソース](../Page/ソース.md "wikilink")\[1\]\[2\]。

[ジェノヴァ](../Page/ジェノヴァ.md "wikilink")風を意味する「ジェノヴェーゼ」というその名称にもかかわらず、[ナポリ](https://ja.wikipedia.org/wiki/ナポリ "wikilink")における主要なパスタソースであり、15世紀から16世紀にかけて普及した\[3\]\[4\]。ジェノヴァとナポリがイタリア半島で最も重要な貿易港であった時代に、ジェノヴァからの商人あるいは移民により伝来した可能性が指摘されている\[5\]\[6\]。また、ジェノベーゼが[カンパニア地方ではよくある姓のため](../Page/カンパニア州.md "wikilink")、創作者に由来するとの説もある。

ナポリを離れると、ジェノヴァを含む他の地域では知名度の低いソースである\[7\]。[ペスト・ジェノヴェーゼ](../Page/ペスト・ジェノヴェーゼ.md "wikilink")、サルサ・ジェノヴェーゼ（赤ワインを用いた魚料理のソース）\[8\]、フランス料理のソース・ジュヌヴォワーズ（魚と香味野菜のソース、魚料理に用いる）\[9\]との混同には注意が必要である。

タマネギを用いる点ではフランス料理の影響が見られ、牛肉の煮込み料理であるを想起させる\[10\]。19世紀中頃には[パリ](https://ja.wikipedia.org/wiki/パリ "wikilink")の[パレ・ロワイヤル](../Page/パレ・ロワイヤル.md "wikilink")にあるレストランで「鮭の[オランデーズとジェノヴェーゼソース](https://ja.wikipedia.org/wiki/オランデーズソース "wikilink")」という一品が供されている\[11\]。

## 調理

|                     |       |
| ------------------- | ----- |
| 牛肉（子牛肉）             | 600g  |
| タマネギ                | 1kg   |
| ニンジン                | 60g   |
| セロリ                 | 60g   |
| オリーブオイルかラードもしくはその両方 | 必要なだけ |

4人前の材料

タマネギと牛肉を2時間から10時間かけて炒める\[12\]\[13\]\[14\]。タマネギは多くの場合と同様にニンジンやセロリと共に炒める\[15\]\[16\]\[17\]\[18\]\[19\]。じっくり炒めたタマネギはソースの風味の根幹をなし\[20\]、白ワインやストック（出汁）と合わせることでさらに引き立つ\[21\]\[22\]。

調理例：ソースと和えたパスタは肉（同じくソースをかけるか別添えで）と共にトマトを付け合わせとし、[ペコリーノ](../Page/ペコリーノ.md "wikilink")を上にかける\[23\]\[24\]。

[リガトーニ](https://ja.wikipedia.org/wiki/リガトーニ "wikilink")、、[カンデーレ](https://ja.wikipedia.org/wiki/カンデーレ "wikilink")といった大きい穴あきの[パスタ](../Page/パスタ.md "wikilink")を用いることでソースを中に含み、芳醇な味わいを楽しむことができるソースである\[25\]\[26\]。
