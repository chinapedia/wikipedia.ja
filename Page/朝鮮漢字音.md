**朝鮮漢字音**（ちょうせんかんじおん）は[朝鮮語における漢字の音のことである](https://ja.wikipedia.org/wiki/朝鮮漢字 "wikilink")。「朝鮮字音」ともいう。[中国語](https://ja.wikipedia.org/wiki/中国語 "wikilink")における漢字の音に由来するもので、日本語の[音読み](../Page/音読み.md "wikilink")に相当する。日本漢字音（音読み）の場合、漢字の読みは断続的に入ってきたため1つの漢字に対して持ち込まれた時代によって[呉音](../Page/呉音.md "wikilink")・[漢音](../Page/漢音.md "wikilink")・[唐音](../Page/唐音.md "wikilink")・その他の4種類以上があるが、朝鮮漢字音は原則的に1つの漢字に対して音は1つである\[1\]。

## 歴史と資料

朝鮮半島は中国と陸続きであるため、朝鮮には早い時期から間断なく漢字がもたらされてきたと推測しうる。しかしながら、[訓民正音](../Page/訓民正音.md "wikilink")創製（1443年）以前の[古代朝鮮語](../Page/古代朝鮮語.md "wikilink")については、表音文字による朝鮮語の表記がなかったため、漢字音が具体的にどのような様相であったのかを知るのが困難である。そのような中で古代朝鮮語の漢字音推測の手がかりとなるのが[郷札](../Page/郷札.md "wikilink")、[口訣](../Page/口訣.md "wikilink")、[吏読](../Page/吏読.md "wikilink")などの借字表記における音読字の読み方である\[2\]。

1443年に表音文字である訓民正音が作られると朝鮮語は明示的に示されるようになったが、15世紀の間は[東国正韻](https://ja.wikipedia.org/wiki/東国正韻 "wikilink")式漢字音よって漢字音が示された。東国正韻式漢字音は伝来漢字音（実際の漢字音）でなく人工的に考案された音なので、当時の現実の漢字音を知るための資料にはなりえない。

伝来漢字音が文献において初めて示されたのは、『六祖法宝壇経諺解』と『真言勤供三壇施食文諺解』（1496年）である。これ以降、伝来漢字音による漢字注音が一般化する。[崔世珍](https://ja.wikipedia.org/wiki/崔世珍 "wikilink")が作った漢字学習書『[訓蒙字会](https://ja.wikipedia.org/wiki/訓蒙字会 "wikilink")』（1527年）も伝来漢字音によって注音がなされている。

朝鮮漢字音の[初声](https://ja.wikipedia.org/wiki/初声 "wikilink")において、[阻害音](https://ja.wikipedia.org/wiki/阻害音 "wikilink")はそのほとんどが平音か激音である。初声に濃音を持つ漢字音は以下の3字の字音と極めて限られている。しかしながら、これらも[中期朝鮮語](https://ja.wikipedia.org/wiki/中期朝鮮語 "wikilink")では平音で現れ、現代語に至る過程で濃音化したものである。従って、朝鮮漢字音にはもともと初声に濃音を持つ漢字音がなかった。

  - ss（）：双 ssaq（），氏 ssi（）
  - gg（）：喫 ggig（）

「d，t（，）」は母音「i（；ie  などの半母音を含む）」と結合しない。また、「s，j，c（，，）」は半母音「i-（， など）」と結合しない。ただし、[中期朝鮮語](https://ja.wikipedia.org/wiki/中期朝鮮語 "wikilink")においてはそれらの結合がありえた。前者の場合、例えば「田」の字音は「dien（）」であったが、[近世朝鮮語](../Page/近世朝鮮語.md "wikilink")期に[口蓋音化が起こり](https://ja.wikipedia.org/wiki/口蓋化 "wikilink")「jien（）」となり、さらに現代朝鮮語に至る過程で「jen（）」となったものである。

### 中声

朝鮮語の中声（母音、半母音＋母音\[3\]、および二重母音）21種のうち、漢字音に現れないのは「iai（）」1種のみである。

| 単母音    | a  | e  | o  | u  | y  | i |   ai   | ei  | oi | ui |
| ------ | :-: | :-: | :-: | :-: | :-: | :-: | :----: | :-: | :-: | :-: |
| /y/＋母音 | ia | ie | io | iu |    |   | 《iai 》 | iei |    |    |
| /w/＋母音 | oa | ue |    |    |    |   |  oai   | uei |    |    |
| 二重母音   |    |    |    |    | yi |   |        |     |    |    |

中声の種類にはかなりのバリエーションがあるが、初声あるいは終声との結合に制約のあるものもある。例えば、単母音のうち「y（）」は唇音の初声「b，p，m（，，）」とは結合せず、後ろに必ず終声を伴うであるとか、/w/ を含む中声は「oa，ue（，）」を除いて終声と結合しないなどの制約がある。

「ai，ei（，）」は現代朝鮮語においてそれぞれ単母音 ， と発音されるが、これらの単母音は[中期朝鮮語](https://ja.wikipedia.org/wiki/中期朝鮮語 "wikilink")においてはその文字の構成の通りに[二重母音](https://ja.wikipedia.org/wiki/二重母音 "wikilink") ， と発音されたと見られる。すなわち、例えば「太」の漢字音「tai（）」は中期朝鮮語において であったと推測され、それは中国語音  の反映であり、日本漢字音の「タイ」とも対応関係にある。その後、[近世朝鮮語](../Page/近世朝鮮語.md "wikilink")期に二重母音が単母音化し、現代朝鮮語のような音に至るわけである。

### 終声

朝鮮語には7種の終声（音節末子音）があるが、このうち「d（）」を除いた6種が漢字音に現れる。

|     | 両唇音 | 歯茎音  | 軟口蓋音 |
| --- | --- | ---- | ---- |
| 阻害音 | b   | 《d 》 | g    |
| 鼻音  | m   | n    | q    |
| 流音  |     | r    |      |

中声との結合において制約のある場合がある。例えば、終声「b，m（，）」は中声「o，u（，）」と結合しないなど。

### アクセント・長音

[中期朝鮮語](https://ja.wikipedia.org/wiki/中期朝鮮語 "wikilink")には高調と低調からなる弁別的な高低[アクセント](../Page/アクセント.md "wikilink")の体系があり、低調（平声）・高調（去声）・低高調（上声）の3種類のパターンが存在した。漢字音においては低調・高調・低高調の全てがありえた。中期朝鮮語のアクセント体系は[近世朝鮮語](../Page/近世朝鮮語.md "wikilink")に至って崩壊・消滅する。この中で低調と高調の複合である低高調は低調あるいは高調に比べて音の長さが倍であるため、アクセント消滅後に[長母音](https://ja.wikipedia.org/wiki/長母音 "wikilink")としてその痕跡を留めることとなった。

  - 事 sa（）

## 南北分断と漢字音

[大韓民国](https://ja.wikipedia.org/wiki/大韓民国 "wikilink")（以下「南」）と[朝鮮民主主義人民共和国](../Page/朝鮮民主主義人民共和国.md "wikilink")（以下「北」）とで字音の異なる漢字が散発的に存在する。

| 漢字 | 南の漢字音  | 北の漢字音  |
| -- | ------ | ------ |
| 醵  | giag（） | gie（）  |
| 歪  | ’oai（） | ’oi（）  |
| 掲  | gei（）  | giei（） |

また、北において「讐」の漢字音「su（）」は「怨讐」という漢字語においてのみ「ssu（）」となる。これは「怨讐」が「元帥」と同音になるのを避け、「怨讐」を「’uenssu（）」に改めたためと見られる。

漢字音「miei，piei（，）」は北では「mei，pei（，）」となる。これは「miei，piei（，）」の実際の発音（//，//）に即してつづりを改めたものと推測される。

漢字語において、語頭に「r（）」あるいは「n（）」が来る場合は、南の標準発音では音が交替する（「頭音法則」と呼ばれる）。語頭の「r（）」は /i/ および半母音 /y/ の直前で「’（）」に、それ以外の場合は「n（）」に変わる。語頭の「n（）」は /i/ および /y/ の直前で「’（）」に変わる。北の標準発音では語頭の「」，「」は維持される。\[4\]

なお、南北間でのこれらの違いについては、「朝鮮語の南北間差異」中の[漢字語に関する表記も参照のこと](https://ja.wikipedia.org/wiki/朝鮮語の南北間差異#漢字語に関する表記 "wikilink")。

## 中古音との関係

河野六郎 (1979)、伊藤智ゆき (2007) をもとに、中国[中古音](https://ja.wikipedia.org/wiki/中古音 "wikilink")と[中期朝鮮語](https://ja.wikipedia.org/wiki/中期朝鮮語 "wikilink")における朝鮮漢字音との対応関係について概観する。ローマ字は[福井玲](https://ja.wikipedia.org/wiki/福井玲 "wikilink")式の[翻字](https://ja.wikipedia.org/wiki/翻字 "wikilink")\[5\]。

### 声母

朝鮮漢字音の声母は、おおよそ以下のように現れる。

| 牙音      | 見 g（）   | 渓 g（）      | 群 g（）         | 疑 ’（）   |
| 舌音      | 端・知 d（） | 透・徹 t（）    | 定・澄 d（）       | 泥・娘 n（） |
| 唇音      | 幫・非 b（） | 滂・敷 b，p（，） | 並・奉 b（）       | 明・微 m（） |
| 歯音      | 精・照 j（） | 清・穿 c（）    | 従・牀 s，j，c（，，） |         |
| 心・審 s（） |         | 邪・禅 s（）    |               |         |
| 喉音      | 影 ’（）   |            |               | 喩 ’（）   |
| 暁 h（）   |         | 匣 h（）      |               |         |
| 半舌音     |         |            |               | 来 r（）   |
| 半歯音     |         |            |               | 日 z（）   |

中古音の声母のうち[阻害音](https://ja.wikipedia.org/wiki/阻害音 "wikilink")は全清（無声無気音）・次清（無声有気音）・全濁（有声音）の3系列がある。中古音は[有気](../Page/有気音.md "wikilink")/[無気の対立とともに](https://ja.wikipedia.org/wiki/無気音 "wikilink")[有声](https://ja.wikipedia.org/wiki/有声音 "wikilink")/[無声](https://ja.wikipedia.org/wiki/無声音 "wikilink")（[清濁](https://ja.wikipedia.org/wiki/清濁 "wikilink")）の対立があった。これに対し朝鮮語の阻害音は有気/無気（激音/平音）の対立はあるが、有声/無声の対立は朝鮮語の音韻体系自体にない。よって、有気音である次清音は同じ有気音である激音に対応するが、全清音・全濁音はともに平音に対応することになる。だが、この対応は必ずしも整然とはしておらず、声母の種類や韻母との組合せによって異なりうる。主な特徴は以下の通りである。

  - 渓母が激音「k（）」で現れるのは夬韻「koai（）」のみである。
  - 見母・渓母において「h（）」で現れる例が若干見られる。
      - 革 hieg（）〔見母〕，恢 hoi（）〔渓母〕
  - 定母・澄母が激音「t（）」で現れる場合がある。
      - 弾 tan（）〔定母〕，治 ti（）〔澄母〕
  - 次清音は、重唇音の滂母は原則として「p（）」で現れ、軽唇音の敷母は原則として「b（）」で現れる。
  - 唇音および歯音の例外的な現れには、音節やアクセントの種類により偏りがある。例えば、韻母が「a（）」，「iei（）」，「io（）」，「ir（）」などの場合には、全清（幫母・非母）、全濁（並母・奉母）でも原則的に「p（）」で現れ、逆に韻母が「ag（）」などの場合には、次清（滂母・敷母）でも原則的に「b（）」で現れる。
      - 波 pa（）〔幫母〕，廃 piei（）〔非母〕
      - 弊 piei（）〔並母〕，乏 pib（）〔奉母〕
      - 拍 bag（）
  - 暁母・匣母は「」で現れるものが多いが、これらは類推によるものである。
      - 喝 gar（）〔暁母〕，暇 ga（）〔匣母〕
  - 影母では「区（區）」を諧声符とする字が徹底して「」で現れる。
      - 欧殴嘔謳 gu（）

以下、諸特徴について概観する。

#### 韻尾の子音

朝鮮漢字音においては、[中古音](https://ja.wikipedia.org/wiki/中古音 "wikilink")の6種類の子音韻尾 ，，，，， が完全な形で対応している。ただし、中古音の[入声](../Page/入声.md "wikilink")  は朝鮮漢字音においては「r（；音声は ）」で現れる。この現象は[古代朝鮮語](../Page/古代朝鮮語.md "wikilink")においてすでに存在したものと見られる。\[6\]

  - 監 gam（）
  - 間 gan（）
  - 江 gaq（）
  - 甲 gab（）
  - 渇 gar（）
  - 各 gag（）

#### 介音

母音の開合、すなわち[介音](https://ja.wikipedia.org/wiki/介音 "wikilink")  の反映の仕方は声母の種類によって異なる。牙音・歯音・喉音は原則として介音  が反映され、舌音は反映されない。

  - 果 goa（）〔牙音〕，坐 joa（）〔歯音〕，話 hoa（）〔喉音〕
  - 堕 ta（）〔舌音〕

母音の直拗、すなわち介音  は多くの場合反映される。

一部の漢字音に[重紐](https://ja.wikipedia.org/wiki/重紐 "wikilink")が反映されている。例えば、止摂の諸韻や仙韻などは、[介音](https://ja.wikipedia.org/wiki/介音 "wikilink")  を伴う音と介音  を伴う音とで朝鮮漢字音の現れ方が異なる。重紐の違いは牙音と喉音で反映され、唇音では反映されない。

  - 移 ’i（） ― 意 ’yi（）
  - 遣 gien（） ― 件 gen（）

この違いは中国漢字音（北京音）や日本漢字音では失われている。

### 声調

[中古音](https://ja.wikipedia.org/wiki/中古音 "wikilink")の[声調](../Page/声調.md "wikilink")と[中期朝鮮語](https://ja.wikipedia.org/wiki/中期朝鮮語 "wikilink")の漢字音における[アクセント](../Page/アクセント.md "wikilink")を比べてみると、中古音の平声は中期朝鮮語の低調に、入声は高調にそれぞれほぼ対応している。しかし、中古音の上声と去声は中期朝鮮語の高調あるいは低高調で現れ、その対応関係が明確でない\[7\]。

### 朝鮮漢字音の母胎音

朝鮮漢字音が中国のいつの時代の音を母胎としているのかについては、これまで複数の研究者が仮説を唱えてきた。伊藤智ゆき (2007) は、[上古音](../Page/上古音.md "wikilink")説（姜信沆など）、[切韻](../Page/切韻.md "wikilink")音説（朴炳采）、唐代長安説（[河野六郎](https://ja.wikipedia.org/wiki/河野六郎 "wikilink")）、宋代開封音説（有坂秀世）などを検討し、具体的に特定するのは難しいとしつつも、唐代長安音が朝鮮漢字音の元になっている可能性が高いとした。
