**EZアプリ (Java)**は[KDDI](../Page/KDDI.md "wikilink")および[沖縄セルラー電話](../Page/沖縄セルラー電話.md "wikilink")の各[auブランドが](../Page/Au_\(携帯電話\).md "wikilink")[2001年](../Page/2001年.md "wikilink")[7月](https://ja.wikipedia.org/wiki/7月 "wikilink")〜[2004年](../Page/2004年.md "wikilink")[6月](../Page/6月.md "wikilink")の間に発売されていた端末に提供していた、[EZweb](../Page/EZweb.md "wikilink")対応[携帯電話](../Page/携帯電話.md "wikilink")の一部で実行できる[Javaアプリケーション](https://ja.wikipedia.org/wiki/Javaアプリケーション "wikilink")およびサービスである。当初は**ezplus**と呼ばれていたが、2003年7月にEZアプリ (Java)に改称した。本項ではさらに[2011年](../Page/2011年.md "wikilink")[2月](https://ja.wikipedia.org/wiki/2月 "wikilink")から[フィーチャーフォン](https://ja.wikipedia.org/wiki/フィーチャーフォン "wikilink")向けに提供している**EZアプリ （J）**もまとめて記載する。2007年から提供している[オープンアプリプレーヤー](https://ja.wikipedia.org/wiki/オープンアプリプレーヤー "wikilink")はそちらの項目を参照。

## 概要

[2001年](../Page/2001年.md "wikilink")[7月](https://ja.wikipedia.org/wiki/7月 "wikilink")に発売されたC451H（[日立製作所](../Page/日立製作所.md "wikilink")製）の端末で初めて搭載された。EZアプリ (Java)の実体は[S\!アプリ](../Page/S!アプリ.md "wikilink")と同様に [Mobile Information Device Profile](https://ja.wikipedia.org/wiki/MIDP "wikilink")(MIDP)にしたがって作成されたJavaアプリケーションであり、[iアプリ](https://ja.wikipedia.org/wiki/iアプリ "wikilink")で使われている[DoJaプロファイル](../Page/DoJaプロファイル.md "wikilink")とは異なっている。

携帯電話の[電話帳](../Page/電話帳.md "wikilink")情報などの制御が可能だが、これらの機能は[セキュリティの関係上](../Page/コンピュータセキュリティ.md "wikilink")、一般には公開されていない。しかし、これらの機能を利用しないアプリであれば[勝手サイト](../Page/勝手サイト.md "wikilink")に公開されているアプリも[ダウンロード](https://ja.wikipedia.org/wiki/ダウンロード "wikilink")して利用できる。ただ、EZアプリ (Java)の実行速度はiアプリやS\!アプリなどに比べ非常に遅く、一部の利用者から批判があった。このため、[SH-mobileを搭載し](../Page/SuperH.md "wikilink")、実行速度を改善した機種もある。

対応端末が増えて行くにつれ、様々な改善等がなされたが、[iアプリDX](https://ja.wikipedia.org/wiki/iアプリDX "wikilink")のようにサービス名が加わることはなく、バージョンに相当する"Phase"が用いられた。

[2003年](../Page/2003年.md "wikilink")からJavaより実行速度が速く、表現力がある[EZアプリ (BREW)へ移行し始め](https://ja.wikipedia.org/wiki/EZアプリ_\(BREW\) "wikilink")、2004年6月に発売された[A5407CA](https://ja.wikipedia.org/wiki/A5407CA "wikilink")を最後にEZアプリ (Java)に対応した端末は発売されていない。

[2006年](../Page/2006年.md "wikilink")10月10日、[KDDI](../Page/KDDI.md "wikilink")は[2007年](../Page/2007年.md "wikilink")春以降発売の端末（W51シリーズ以降）にJava（MIDP2.0準拠）で作成されたアプリケーションの再生機能である「**[オープンアプリプレイヤー](../Page/オープンアプリプレイヤー.md "wikilink")**」を搭載すると発表した。「オープンアプリプレイヤー」はBREW上で動作するが、既存の端末では利用できない。また[KCP+](../Page/KCP+.md "wikilink")対応機種を含むW5×シリーズ以降でも、[簡単ケータイシリーズ](https://ja.wikipedia.org/wiki/簡単ケータイシリーズ "wikilink")全機種や[mamorino](https://ja.wikipedia.org/wiki/mamorino "wikilink")を含む[安心ジュニアケータイシリーズ](https://ja.wikipedia.org/wiki/安心ジュニアケータイシリーズ "wikilink")全機種、[W51P](../Page/W51P.md "wikilink")、[W52T](../Page/W52T.md "wikilink")、[MEDIA SKIN（W52K）](../Page/MEDIA_SKIN.md "wikilink")、[W52SH](../Page/W52SH.md "wikilink")、[W62K](../Page/W62K.md "wikilink")、[W63K](../Page/W63K.md "wikilink")、[Sportio water beat](https://ja.wikipedia.org/wiki/Sportio_water_beat "wikilink")、[NS01](https://ja.wikipedia.org/wiki/NS01 "wikilink")、[NS02](https://ja.wikipedia.org/wiki/NS02 "wikilink")、[misora](https://ja.wikipedia.org/wiki/misora "wikilink")、[PLY](https://ja.wikipedia.org/wiki/PLY "wikilink")、[SA001](https://ja.wikipedia.org/wiki/SA001 "wikilink")、[T003](https://ja.wikipedia.org/wiki/T003 "wikilink")、[PRISMOID](https://ja.wikipedia.org/wiki/PRISMOID "wikilink")など一部機種では「オープンアプリプレイヤー」に非対応となっている。また、オープンアプリはau独自拡張が搭載されておらず、EZアプリ (Java)のサブセットである。

[2011年](../Page/2011年.md "wikilink")春モデル[T006](https://ja.wikipedia.org/wiki/T006 "wikilink")より、かつての独自拡張を削り、オープンアプリプレイヤーをベースに新たに3Dグラフィックの描画機能やアプリデータをSDカードに書き込む機能を追加したほか、メモリ容量やアプリのサイズを増やした上で、**EZアプリ （J）**として復活した。

EZアプリ (Java)が、ezplusと呼称されていた時期にはゲームソフト向けコントローラである[ezplusコントローラ](https://ja.wikipedia.org/wiki/ezplusコントローラ "wikilink")も発売されていた。こちらはJava専用ではなく、携帯電話に配されたの各スイッチと同様の操作が行えるハードウェアであるため、携帯電話本体の操作をはじめ、非公式ながら後継のBREWアプリとの組み合わせでも使用する事ができる。

## 開発環境

Javaの実行環境には様々なエディションが存在するが、EZアプリ (Java)の場合は次の[コンポーネント](https://ja.wikipedia.org/wiki/コンポーネント "wikilink")を必要としている。

  - Java 2 Platform Standard Edition Version 1.3 及びそれ以降 （以下、J2SE）
  - J2ME Wireless Tool Kit （以下、J2ME）
  - KJX作成ツール

上二つはオラクルから、下はauのウェブサイトから無償で入手することができる。

EZアプリ（Java）はアプリックスのmicroJBlendを搭載しており、CLDC, MIDP 1.0, JBlend 独自拡張, KDDI 独自拡張の API があった。

EZアプリ（J）は以下の API に対応

  - CLDC 1.1
  - MIDP 2.0
  - FileConnection (JSR 75)
  - Location API for J2ME (JSR 179) (2011年秋冬の一部端末より利用可能)
  - com.mascotcapsule.micro3d.v3 - マスコットカプセル（独自拡張）
  - com.kddi.oap.opgl - [OpenGL ES](../Page/OpenGL_ES.md "wikilink") 1.1 バインディング（独自拡張）
  - com.kddi.oap.graphics - ImageConverter クラスのみ（独自拡張）
