[Belleville_washer.jpg](https://ja.wikipedia.org/wiki/File:Belleville_washer.jpg "fig:Belleville_washer.jpg") **皿ばね** （さらばね） は、中心に穴の開いた[円盤](https://ja.wikipedia.org/wiki/円盤 "wikilink")状の板を円錐状にし、底のない[お皿のような形状にした](../Page/皿.md "wikilink")[ばね](../Page/ばね.md "wikilink")である\[1\]。英語では、disc spring、coned disc spring、Belleville spring などと呼ぶ\[2\]。英語名の"Belleville"（ベリビル）は、1865年に皿ばねの原理を発明したジュリアン・フランソワ・ベルビル (Julian Francois Belleville) に由来する。

## 特徴

皿ばねの円錐上側部分と下側部分に荷重を加え、高さを低くする方向にたわませることでばね作用が得られる。主に以下のような特徴を持つ。

  - 小さな取付スペースで大きな荷重を受けることができる。一般的なばねの種類である[コイルばね](https://ja.wikipedia.org/wiki/コイルばね "wikilink")が入らないような場所にも使えることがある。
  - 荷重とたわみの関係は[非線形](https://ja.wikipedia.org/wiki/非線形 "wikilink")だが、形状の寸法比を変えることで累進的、逆進的、線形的などの様々なばね特性が得られる。一方、板厚などの製作誤差がばね特性に大きく影響しやすく、特性のばらつきが大きくなる。
  - 皿ばね同士を組み合せることにより、さらに様々なばね特性が得られ、全体としてのばね高さも変えることができる。ただし、組み合わせて使用する場合はばねのずれを生じさせないために内側か外側にガイドが必要となる。
  - 皿ばねがたわむと皿ばねは平らになる方向に変形する。そのため、たわみに伴って接触点が移動するため、摩擦によりばね特性に[ヒステリシス](../Page/ヒステリシス.md "wikilink")が生じる。これにより減衰特性を得ることもできる。一方、摩擦によりガイドやばねの摩耗が生じたり、ヒステリシスを嫌う場合は潤滑材や表面研磨などを要する。

## 使用例

[Torque_limiter_with_balls_Make_Ringspann.JPG](https://ja.wikipedia.org/wiki/File:Torque_limiter_with_balls_Make_Ringspann.JPG "fig:Torque_limiter_with_balls_Make_Ringspann.JPG")の例。ハの字状に積み重なった板が皿ばね。\]\] 皿ばねは、締結用の[座金](../Page/座金.md "wikilink")、[クラッチ](https://ja.wikipedia.org/wiki/クラッチ "wikilink")の加圧バネ、その他産業用機械など、広く使用されている。特に、自動車の[クラッチ](https://ja.wikipedia.org/wiki/クラッチ "wikilink")に使われるものは**ダイヤフラムスプリング**（diaphragm spring）とも呼ばれる。座金用として用いられる皿ばねは**皿ばね座金**と呼ばれる。

使用される皿ばねの大きさは、板厚は数十[μmから数十](../Page/マイクロメートル.md "wikilink")[mm程度まで](../Page/ミリメートル.md "wikilink")、外径は数mmから1[m程度までと多岐にわたる](../Page/メートル.md "wikilink")。皿ばねの材料には、用途に応じて様々な材料が使用される。[炭素鋼](../Page/炭素鋼.md "wikilink")、[合金鋼](../Page/合金鋼.md "wikilink")、[ステンレス鋼](../Page/ステンレス鋼.md "wikilink")、[銅合金](https://ja.wikipedia.org/wiki/銅合金 "wikilink")などが材料として利用されている。

## 計算式

### アルメン・ラスロの式

[Disc_spring_diagram.png](https://ja.wikipedia.org/wiki/File:Disc_spring_diagram.png "fig:Disc_spring_diagram.png") [Deformation_of_disc_spring_in_Almen_and_Laszlo_model.png](https://ja.wikipedia.org/wiki/File:Deformation_of_disc_spring_in_Almen_and_Laszlo_model.png "fig:Deformation_of_disc_spring_in_Almen_and_Laszlo_model.png") 皿ばねの[荷重](../Page/荷重.md "wikilink")と[たわみ](https://ja.wikipedia.org/wiki/たわみ "wikilink")（縮み）の関係式、および皿ばね四隅の[応力](../Page/応力.md "wikilink")の計算式としては、以下のアルメン・ラスロの近似式がある。皿ばねの外径を *d<sub>e</sub>*、内径を *d<sub>i</sub>*、板厚を *t*、自由高さを *h*<sub>0</sub> (≃ *H*<sub>0</sub> − *t*)とし、皿ばね材料の[ヤング率](../Page/ヤング率.md "wikilink")を *E*、[ポアソン比](../Page/ポアソン比.md "wikilink")を *ν* とする。係数を以下のように定義したとき、

\[\alpha = \frac{d_e}{d_i} \quad , \quad \beta = \frac{h_0}{t}\]

\[C_1 = \frac{1}{\pi}\frac{\left( \frac{\alpha-1}{\alpha} \right)^2}{\frac{\alpha+1}{\alpha-1}-\frac{2}{\ln\alpha}}\]

\[C_2 = \frac{1}{\pi}\frac{6}{\ln\alpha}\left( \frac{\alpha-1}{\ln\alpha}-1\right)\]

\[C_3 = \frac{3}{\pi}\frac{\alpha-1}{\ln\alpha}\]

荷重 *P* とたわみ *δ* の関係：

\[P = \frac{4E}{1-\nu^2}\frac{t^3}{C_1 d_e^2} \delta \left[ \left( \beta-\frac{\delta}{t} \right) \left( \beta-\frac{\delta}{2t} \right)+1 \right]\]

接線[ばね定数](https://ja.wikipedia.org/wiki/ばね定数 "wikilink") *k* ：

\[k = \frac{dP}{d\delta} = \frac{4E}{1-\nu^2}\frac{t^3}{C_1 d_e^2} \left[\beta^2 - 3\beta\frac{\delta}{t} +\frac{3}{2}\left(\frac{\delta}{t}\right)^2 +1 \right]\]

各四隅の応力：

\[\sigma_\mathrm{I} = \frac{4E}{1-\nu^2} \frac{t}{C_1 d_e^2} \delta \left[ -C_2 \left( \beta-\frac{\delta}{2t} \right) -C_3 \right]\]

\[\sigma_\mathrm{II} = \frac{4E}{1-\nu^2} \frac{t}{C_1 d_e^2} \delta \left[ -C_2 \left( \beta-\frac{\delta}{2t} \right) +C_3 \right]\]

\[\sigma_\mathrm{III} = \frac{4E}{1-\nu^2} \frac{t}{\alpha C_1 d_e^2} \delta \left[ (2C_3-C_2) \left( \beta-\frac{\delta}{2t} \right) +C_3 \right]\]

\[\sigma_\mathrm{IV} = \frac{4E}{1-\nu^2} \frac{t}{\alpha C_1 d_e^2} \delta \left[ (2C_3-C_2) \left( \beta-\frac{\delta}{2t} \right) -C_3 \right]\]

円周長が変化しない回転中心の外径 *d<sub>o</sub>* ：

\[d_o = \frac{d_e-d_i}{\ln\alpha}\] となる。ここで、ln は[自然対数](../Page/自然対数.md "wikilink")である。

上記の式は、1936年に[ゼネラルモーターズ](../Page/ゼネラルモーターズ.md "wikilink")研究員のアルメン (J. O. Almen) とラスロ (A. László) により発表された\[3\]。式導出の仮定として、

  - 断面形状は変形せず、円周長が変化しない中立点を中心にして回転するようにして皿ばねが変形する。
  - 回転角 *φ* は小さいとして、*φ* の高次の項は無視する。
  - 荷重は円周上に一様に負荷され、変形は軸対称とする。

以上を設定している。

### 荷重・たわみ特性曲線

[Disc_spring_load-deflection_characteristic_curve.svg](https://ja.wikipedia.org/wiki/File:Disc_spring_load-deflection_characteristic_curve.svg "fig:Disc_spring_load-deflection_characteristic_curve.svg") アルメン・ラスロの式で示されたように、荷重とたわみの関係は[3次曲線となっており](../Page/三次関数.md "wikilink")、[非線形](https://ja.wikipedia.org/wiki/非線形 "wikilink")な関係となっている。荷重・たわみ特性曲線の形状は、自由高さと板厚の比 *h*<sub>0</sub> / *t* によって決定される。*h*<sub>0</sub> / *t* の値を小さくすれば、荷重とたわみの関係はほとんど[線形](https://ja.wikipedia.org/wiki/線形 "wikilink")となる。*h*<sub>0</sub> / *t* = 1.4 になると、たわみ *β* が *h*<sub>0</sub> に等しい付近で *P* 一定状態となる。

さらに *h*<sub>0</sub> / *t* が大きくなると、曲線は極大値を持つようになり、極大値を過ぎたところでは負のばね定数を持つようになる。このような荷重・たわみ曲線の場合は、荷重を増やしていくと、たわみが一気に増加する飛び移りが発生する。この飛び移りの特性を積極的に利用する使用方法もある。一般的には、*h*<sub>0</sub> / *t* の値は0.4から1.3までの範囲内とすることが推奨されている。

実際の荷重・たわみ特性曲線は、アルメン・ラスロの式による曲線と差がある。*β* / *h*<sub>0</sub> が0.75を超えた辺りから、実際の曲線は計算式による曲線を大きく上回りだす。これは荷重支持点が変化する影響によるものである。このため、*β* / *h*<sub>0</sub> が 0.75あるいは0.8程度の範囲までのたわみで使用することが一般的には推奨されている。

### コーナーRの影響補正

アルメン・ラスロの式では皿ばね板の四隅にコーナーRはないものとして式が導出されている。実際にコーナーRが付くと荷重支点の移動が発生し、荷重とたわみの関係でアルメン・ラスロの式による計算値と実測値の差が大きくなる。日本ばね工業会などでは次のように補正した荷重-たわみの関係式を導入している。

\[P = \frac{d_e-d_i}{(d_e-d_i)-3R} \frac{4E}{1-\nu^2}\frac{h_0^3}{C_1 d_e^2} \delta \left[ \left( \beta-\frac{\delta}{h_0} \right) \left( \beta-\frac{\delta}{2h_0} \right)+1 \right]\] ここで、\(\frac{d_e-d_i}{(d_e-d_i)-3R}\)が補正項である。

## 重ね合わせによる利用

[Tellerfeder2.svg](https://ja.wikipedia.org/wiki/File:Tellerfeder2.svg "fig:Tellerfeder2.svg") [Tellerfeder.svg](https://ja.wikipedia.org/wiki/File:Tellerfeder.svg "fig:Tellerfeder.svg")

他のばねと同様、複数の皿ばねを使用することで直列あるいは並列のばね定数を得ることができる。[フックの法則\#ばねが複数の場合](https://ja.wikipedia.org/wiki/フックの法則#ばねが複数の場合 "wikilink")などを参照のこと。

皿ばねを同じ向きに重ね合わさると並列ばねの効果を発揮する。この組み合せ方を**並列重ね**と呼ぶ。皿ばねを互い違いに重ね合わさると直列ばねの効果を発揮する。この組み合せ方を**直列組合せ**と呼ぶ。重ね合わせをしたときはバネ間での摩擦が増えるので、より大きく[ヒステリシス](../Page/ヒステリシス.md "wikilink")が発生する。また、重ね合わせて使用する場合は皿ばねが崩れないようにガイドが必要となり、ピン状のものを皿ばねに通すか、筒状のものに皿ばねを入れるかをする必要がある。

皿ばねを *n* 枚並列重ねしたとき、総たわみ *δ<sub>T</sub>* は1枚のときと変わらない。一方で総荷重 *P<sub>T</sub>* は1枚のときの *n* 倍必要になる。*P* と *δ* を1枚のときの荷重とたわみ、*L*<sub>0</sub> を重ね合わした皿ばねにおける無負荷時の総高さとする。これらを式で表せば

\[P_{T}=nP\]

\[\delta_{T}=\delta\]

\[L_0=H_0 + (n-1)t\] である。

一方、皿ばねを *m* 枚直列組合せしたときは、総荷重 *P<sub>T</sub>* は1枚のときと変わらないが、総たわみ *δ<sub>T</sub>* は1枚のときの *m* 倍になる。式は

\[P_{T}=P\]

\[\delta_{T}=m \delta\]

\[L_0=m H_0\] である。

皿ばねを *n* 枚並列重ね・*m* 枚直列組合せしたときは、

\[P_{T}=nP\]

\[\delta_{T}=m \delta\]

\[L_0=\left [ H_0 + (n-1)t \right ] m\] となる。

## スリット付き

[Ressort_en_coupelle.svg](https://ja.wikipedia.org/wiki/File:Ressort_en_coupelle.svg "fig:Ressort_en_coupelle.svg") たわみを大きくしたいような場合、スリットを付けた皿ばねの種類がある。スリット付き皿ばねによって、応力あるいは荷重を低減させつつ、たわみを大きくすることができる。スリット付き皿ばねの荷重・たわみ特性は、スリット部の片持ち[はりとスリット無し部分の皿ばね本体の合成として考えることができる](https://ja.wikipedia.org/wiki/はり部材 "wikilink")。より正確な特性評価や応力評価には[有限要素法](../Page/有限要素法.md "wikilink")が必要とされる。
