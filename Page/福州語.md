**福州語**（ふくしゅうご、フッチュウア＝Hók-ciŭ-uâ、[英語](../Page/英語.md "wikilink")：Fuzhou dialect、Foochowese、[中国語](https://ja.wikipedia.org/wiki/中国語 "wikilink")：福州話 Fúzhōuhuà）は、[中国語](https://ja.wikipedia.org/wiki/中国語 "wikilink")の[方言](../Page/方言.md "wikilink")のひとつで、主に[中国](../Page/中華人民共和国.md "wikilink")[福建省](../Page/福建省.md "wikilink")東部の[福州](https://ja.wikipedia.org/wiki/福州 "wikilink")市周辺で話されている言葉である。また、[台湾](https://ja.wikipedia.org/wiki/台湾 "wikilink")では「閩北語」と呼ばれていた。

## 系統

福州語は、中国語十大方言、もしくは七大方言のうちのひとつである**[閩語](../Page/ビン語.md "wikilink")**（びんご）、別名**閩方言**（びんほうげん）に属し、上位分類である**[閩東語](../Page/ビン東語.md "wikilink")**（びんとう語）の代表とされる。[福州市](../Page/福州市.md "wikilink")に所属する、[福清](../Page/福清市.md "wikilink")、[閩侯](https://ja.wikipedia.org/wiki/ビン侯県 "wikilink")、[閩清](https://ja.wikipedia.org/wiki/ビン清県 "wikilink")、[永泰](https://ja.wikipedia.org/wiki/永泰県 "wikilink")、[羅源](https://ja.wikipedia.org/wiki/羅源県 "wikilink")、[連江](https://ja.wikipedia.org/wiki/連江県_\(中華人民共和国\) "wikilink")、[長楽](https://ja.wikipedia.org/wiki/長楽区 "wikilink")、[平潭の他](https://ja.wikipedia.org/wiki/平潭県 "wikilink")、[寧徳地区の一部](../Page/寧徳市.md "wikilink")、[古田などで話され](https://ja.wikipedia.org/wiki/古田県 "wikilink")、使用人口は約560万人。この他にも閩中地区、閩北地区には、福州語を第二言語などとして話せる人が少なくない。

海外では、[マレーシア](https://ja.wikipedia.org/wiki/マレーシア "wikilink")に比較的多くの福州系[華人](https://ja.wikipedia.org/wiki/華人 "wikilink")・[華僑](../Page/華僑.md "wikilink")がいる他、[アメリカ合衆国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")やアフリカ諸国にも福州系華人・華僑がおり、福州語を母語としている。

## 音声的特徴

福州語の音節は、他の中国語（漢語）方言と同じく、[声母](https://ja.wikipedia.org/wiki/声母 "wikilink")（語頭[子音](../Page/子音.md "wikilink")）と[韻母](../Page/韻母.md "wikilink")は48種、[声調](../Page/声調.md "wikilink")の組み合わせで成り立っている。

福州市内の福州語の場合、基本声母は15種、韻母は48種、声調は7種ある。福州語は2つ以上の音節がつながることによって、後の音節の声母及び前の音節の韻母と声調が変化するという、他の中国語の方言にみられない特徴を持っている。

### 声調

基本[声調](../Page/声調.md "wikilink")は下記の7種がある。うち陰平・上声・陽平・陽入に当たる音節の韻母は緊韻で発音し、陰去・陰入・陽去に当たる音節の韻母は鬆韻で発音する。

|                                                   |                                                   |    |
| ------------------------------------------------- | ------------------------------------------------- | -- |
| [調類](https://ja.wikipedia.org/wiki/調類 "wikilink") | [調値](https://ja.wikipedia.org/wiki/調値 "wikilink") | 例字 |
| 陰平                                                | 55                                                | 君  |
| 上声                                                | 33                                                | 滾  |
| 陰去                                                | 212                                               | 貢  |
| 陰入                                                | 24                                                | 谷  |
| 陽平                                                | 53                                                | 群  |
| 陽去                                                | 242                                               | 郡  |
| 陽入                                                | 5                                                 | 掘  |

**福州語の声調表**

[連続変調](https://ja.wikipedia.org/wiki/連続変調 "wikilink")により、上記の他に**半陰去**21と**半陽去**35の調値が現れる。

2音節の連続による前の音節の声調変化は下記の通り。

|                        |          |
| :--------------------: | :------: |
|          前の音節          |   後の音節   |
|          陰平55          | 陽平53 陽入5 |
| 陰平55 陰去212 陽去242 陰入乙24 |    55    |
|        陽平53 陽入5        |    55    |
|       上声33 陰入甲24       |    21    |

**福州語の声調変化表**

3音節以上の場合はさらに複雑である。

故に、福州話での複合語の発音は元の音節といくつの所で音韻変化することがある。例えば、「福」は本来、陰入調（甲類）でと発音する。「州」は陰平調でと発音する。組み合わせて「福州」の地名を構成する場合、「福」の声調は陰入調から半陰去調へ変化するため、その韻母も鬆韻のから緊韻のへ変化する。また、「福」は陰入調（甲類）に属するので、「州」の声母は変化しない。なので、「福州」の実際の発音はである。

また、「中」は本来、陰平調でと発音する。「国」は本来、陰入調でと発音する。「中国」という言葉を構成する場合、「中」の声調は陰平調から陽平調へ変化するため、韻母は緊韻のままで変化しないが、「国」の声母はの後ろでへ変化する。なので、「中国」の実際の発音はである。
