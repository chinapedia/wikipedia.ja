**昼飯大塚古墳**（ひるいおおつかこふん）は、[岐阜県](../Page/岐阜県.md "wikilink")[大垣市](../Page/大垣市.md "wikilink")昼飯町にある[古墳](https://ja.wikipedia.org/wiki/古墳 "wikilink")。形状は[前方後円墳](../Page/前方後円墳.md "wikilink")。不破古墳群を構成する古墳の1つ。国の[史跡](../Page/史跡.md "wikilink")に指定されている。

岐阜県では最大規模の古墳で\[1\]、[4世紀](../Page/4世紀.md "wikilink")末（[古墳時代](../Page/古墳時代.md "wikilink")中期初頭）頃の築造と推定される。

岐阜県西部、大垣市街地から北方の低位段丘（牧野台地）西縁に築造された大型前方後円墳である。一帯は本古墳含む盟主的首長墓クラスの大型古墳が集中する地域で、不破古墳群として認知される。発掘調査以前に墳丘の一部や周濠部は宅地造成等による削平を受けているほか、墳丘上は竹藪で覆われていた。これまでには、1879年-1880年（明治12-13年）頃に盗掘に遭ったのち、1979年度（昭和54年度）に第1次発掘調査、1994年-1999年度（平成6-11年度）に第2-7次発掘調査、2004年-2006年度（平成16-18年度）に第8-10次発掘調査が実施されている\[2\]。

墳形は前方後円形で、前方部を南西方に向ける。墳丘は3段築成\[3\]。 墳丘長は約150[メートル](../Page/メートル.md "wikilink")を測り、岐阜県で最大規模になる<ref group="注" name="規模">岐阜県における主な古墳は次の通り。

1.  **昼飯大塚古墳**（大垣市昼飯町） - 墳丘長150メートル。
2.  [坊の塚古墳](../Page/坊の塚古墳.md "wikilink")（各務原市鵜沼羽場町） - 墳丘長120メートル。
3.  [琴塚古墳](../Page/琴塚古墳.md "wikilink")（岐阜市琴塚） - 墳丘長115メートル。</ref>。墳丘表面では各段テラス部に[円筒埴輪](../Page/円筒埴輪.md "wikilink")列が、斜面に[葺石](https://ja.wikipedia.org/wiki/葺石 "wikilink")が、墳頂部などに[形象埴輪](../Page/形象埴輪.md "wikilink")（家形・靫形・盾形・蓋形・甲冑形埴輪）が認められている。墳丘周囲には鍵穴形（左右非対称の前方後円形）の周濠が巡らされており、周濠を含めた古墳全長は約180メートルにおよぶ。主体部の埋葬施設は、[竪穴式石室](../Page/竪穴式石室.md "wikilink")・[粘土槨](https://ja.wikipedia.org/wiki/粘土槨 "wikilink")・[木棺](https://ja.wikipedia.org/wiki/木棺 "wikilink")直葬の3基で、いずれも後円部において同一墓壙内に構築される\[4\]。大型古墳で同一墓壙内に異なる3基の埋葬施設が構築されるのは珍しい様相になる。竪穴式石室は盗掘に遭っているが他2基は未盗掘で、3基とも内部調査は実施されていないものの、周囲調査では多数の副葬品や供献儀礼品（玉類・土器など）が検出されている。

この昼飯大塚古墳は、[古墳時代](../Page/古墳時代.md "wikilink")中期初頭の[4世紀](../Page/4世紀.md "wikilink")末頃の築造と推定される\[5\]\[6\]。一帯は[揖斐川](../Page/揖斐川.md "wikilink")西岸の平野部であるとともに[関ヶ原](https://ja.wikipedia.org/wiki/関ヶ原 "wikilink")から[濃尾平野](../Page/濃尾平野.md "wikilink")に出る要地であり、本古墳はこの地域に営造された不破古墳群の盟主墳の1つに位置づけられるが、当地では各盟主墳は異なる造墓基盤勢力によって築造されている。それら盟主墳の中でも本古墳は最大級となり、その規模や内容は畿内の大王墓に準じるものであることから、当時の不破地域ひいては東海地方と[ヤマト王権](../Page/ヤマト王権.md "wikilink")との関わりを考察するうえで重要視される古墳になる\[7\]。

古墳域は2000年（平成12年）に国の[史跡](../Page/史跡.md "wikilink")に指定された\[8\]。現在は史跡整備工事の上で昼飯大塚古墳歴史公園として公開されている\[9\]。

## 構造

[Hirui_Otsuka_Kofun,_zenkei.jpg](https://ja.wikipedia.org/wiki/File:Hirui_Otsuka_Kofun,_zenkei.jpg "fig:Hirui_Otsuka_Kofun,_zenkei.jpg") 古墳の規模は次の通り。

|         | 全体    | 後円部 | 前方部  |
| ------- | ----- | --- | ---- |
| 長さ      | 直径    | 高さ  | 長さ   |
| 3段目（上段） |       | 55m | 6.8m |
| 2段目（中段） |       | 77m | 3.8m |
| 1段目（下段） | 約150m | 96m | 2.2m |
| 全体      | 約150m | 96m | 13m  |

墳丘企画は後円部に比して前方部が短い形状で、類例として[青塚古墳](../Page/青塚古墳_\(犬山市\).md "wikilink")（[愛知県](https://ja.wikipedia.org/wiki/愛知県 "wikilink")[犬山市](../Page/犬山市.md "wikilink")）・[正法寺古墳](https://ja.wikipedia.org/wiki/正法寺古墳_\(西尾市\) "wikilink")（愛知県[西尾市](../Page/西尾市.md "wikilink")）・[宝塚1号墳](../Page/宝塚古墳_\(松阪市\).md "wikilink")（[三重県](https://ja.wikipedia.org/wiki/三重県 "wikilink")[松阪市](../Page/松阪市.md "wikilink")）との関連が指摘される。

墳丘の各段のテラスには[円筒埴輪](../Page/円筒埴輪.md "wikilink")列が巡らされ、斜面には[葺石](https://ja.wikipedia.org/wiki/葺石 "wikilink")が施されている。後円部頂・スロープ部の埴輪列については詳細な調査が実施されており、後円部頂では直径20メートルの円周上に計約150本の埴輪が樹立したとされる。また葺石に用いられた川原石については、付近の[大谷川産と推測されている](https://ja.wikipedia.org/wiki/大谷川_\(岐阜県\) "wikilink")。

墳丘周囲に巡らされた周濠は深さ2メートルまで掘り下げた深いもので、一帯の古墳としては初となる本格的な周濠とされる。周濠の基底幅は、前方部北前端で11メートル、後円部で9-14メートル、くびれ部で23メートル。この周濠部では、地中探査により陸橋（渡り土手）や島状遺構の存在も推測されるが、いずれも発掘調査が実施されておらず詳らかでない。

現在では後円部の一部で築造当時の様子が再現されている。

## 埋葬施設

主体部の埋葬施設は、後円部墳頂において同一墓壙内に[竪穴式石室](../Page/竪穴式石室.md "wikilink")1基・[粘土槨](https://ja.wikipedia.org/wiki/粘土槨 "wikilink")1基・[木棺](https://ja.wikipedia.org/wiki/木棺 "wikilink")直葬1基の計3基の構築が認められている\[21\]。3基はそれぞれ次の通り。

:; [竪穴式石室](../Page/竪穴式石室.md "wikilink")（北棺）

:: 墓壙内北寄りに位置する。明治期に盗掘に遭ったことが知られるが、盗掘坑内の13世紀前半頃の土師器片の出土によって明治以前からの盗掘が認められており、昭和初期までは開口していたという。主軸は墳丘主軸と平行する北東-南西方向とする。観察によれば、内法は長さ約4.5メートル、東小口幅1.2メートル、西小口幅0.8メートルを測る。石室壁面には赤色顔料が塗布される。東小口側が広いことから頭位は東とされる。施設内部の詳細は未調査。

:; [粘土槨](https://ja.wikipedia.org/wiki/粘土槨 "wikilink")（南棺）

:: 墓壙内南東寄りに位置する。発掘調査の際に発見され、施設内は未調査のまま保存されている。北棺と同様に主軸は墳丘主軸と平行する北東-南西方向とする。長さは約8メートルを測る。竪穴式石室と同様に頭位は東と推測される。

:; 木棺直葬（西棺）

  -

      -
        墓壙内南西寄りに位置する。発掘調査の際に発見され、施設内は未調査のまま保存されている。北棺・南棺とは異なり主軸は墳丘主軸と直交する北西-南東方向とする。棺外の副葬品として刀剣類など多数の鉄製品群が出土しており、武人的性格が示唆される。

前述のように3基は同一墓壙内に位置するが、墓壙は計画的に配置されていることから、3基ともあらかじめ計画された上での埋葬とされる。3基のうち竪穴式石室・粘土槨はほぼ同時期の埋葬と推定される一方、これら2基を粘土で覆った上に第3の木棺が置かれるため、第3の木棺は若干下る時期と推定されるが、墓壙自体は3基が揃って初めて埋められている。大型前方後円墳に3人の被葬者が埋葬され、それぞれ異なる埋葬施設を採用するという例は珍しく、その背景は詳らかでない。

以上のほか、発掘調査では南側くびれ部付近で埴輪棺3基も検出されている。これらに用いられた埴輪は昼飯大塚古墳のものとされる。
