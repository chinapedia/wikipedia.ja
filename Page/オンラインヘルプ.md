**オンラインヘルプ**（）は、[ソフトウェア](../Page/ソフトウェア.md "wikilink")の一部として提供される手続き的情報や参考情報である。[ユーザアシスタンス](https://ja.wikipedia.org/wiki/ユーザアシスタンス "wikilink")の一形態である。一般に[アプリケーションソフトウェア](../Page/アプリケーションソフトウェア.md "wikilink")や[オペレーティングシステム](../Page/オペレーティングシステム.md "wikilink")を使用する際の補助となるよう設計されるが、それ以外の様々なものについての情報を提示する目的でも使用されることがある。オンラインヘルプとアプリケーションの状態（ユーザーが何をしようとしているか）を結びつけたものを[文脈依存ヘルプ](https://ja.wikipedia.org/wiki/文脈依存ヘルプ "wikilink")という。

オンラインヘルプは[ヘルプ作成ツール](https://ja.wikipedia.org/wiki/ヘルプ作成ツール "wikilink")を使って作成できる。フォーマットには、独自のものから以下のように[オープン標準](../Page/オープン標準.md "wikilink")を使ったものまで様々なものがある。

  - [HTML](../Page/HyperText_Markup_Language.md "wikilink") を使う場合: HTML Help、JavaHelp、Oracle Help など
  - [Portable Document Format](../Page/Portable_Document_Format.md "wikilink")（PDF）

## マイクロソフトのヘルプ機構

[マイクロソフト](../Page/マイクロソフト.md "wikilink")は、[Windows](https://ja.wikipedia.org/wiki/Microsoft_Windows "wikilink") 上のヘルプシステムのためのプラットフォームとして、以下のものを開発してきた。

  - Microsoft QuickHelp - 1987年に入社した Ralph Walden が MS-DOS と OS/2 向けに書いたオンラインヘルプシステム。Walden はその後の WinHelp と HTML Help 1.x にも関わった。
  - [Microsoft WinHelp](https://ja.wikipedia.org/wiki/Microsoft_WinHelp "wikilink")（.HLP） - [Rich Text Format](../Page/Rich_Text_Format.md "wikilink") に基づくもので、Windows 3.1 と Windows 95/NT での業界標準となった。WinHelp を実際に表示するヘルプエンジン（WinHlp32.exe）は Windows 3.0 から [Windows XPまでの全](../Page/Microsoft_Windows_XP.md "wikilink") Windows に含まれている。しかし、[Windows Vista](https://ja.wikipedia.org/wiki/Microsoft_Windows_Vista "wikilink") には無いので、必要ならダウンロード可能である（\[1\]）。
  - [Microsoft Compiled HTML Help](../Page/Microsoft_Compiled_HTML_Help.md "wikilink")（.CHM） - HTMLなどをベースとしている。2006年現在の最新版は HTML Help 1.4．最初の HTML Help 1.0 は[1997年](https://ja.wikipedia.org/wiki/1997年 "wikilink")にリリースされた。
  - [HTML Help 2.0](https://ja.wikipedia.org/wiki/Microsoft_Help_2 "wikilink") - [2001年](../Page/2001年.md "wikilink")にマイクロソフトが発表した HTML Help の第2版。[2002年](../Page/2002年.md "wikilink")に Visual Studio .NET、MSDN Library、Technet 製品などのヘルプ形式として実装された。しかし、Windows などへの搭載は行われていない。
  - [AP Help 1.0](https://ja.wikipedia.org/wiki/Microsoft_Assistance_Markup_Language "wikilink") - [Vistaに搭載された新しいオンラインヘルプシステム](https://ja.wikipedia.org/wiki/Microsoft_Windows_Vista "wikilink")。他のソフトウェアベンダー向けにヘルプ作成ツールが提供されておらず、マイクロソフトとOEM、一部のソフトウェア企業だけが使っている。

## その他

  - Apple Help (.HELP) - [アップル独自のヘルププラットフォーム](../Page/アップル_\(企業\).md "wikilink")。Mac OS 8.5 以降で使用。
  - [Sun JavaHelp](http://java.sun.com/products/javahelp/) (.js) - [サン・マイクロシステムズ](../Page/サン・マイクロシステムズ.md "wikilink")が[Java](https://ja.wikipedia.org/wiki/Java "wikilink")で書いたプラットフォーム独立なヘルプシステム。[Java Runtime Environment](../Page/Java_Runtime_Environment.md "wikilink")（JRE）の動作する任意のプラットフォームやブラウザで動作可能。
  - [Oracle Help](http://www.oracle.com/technology/tech/java/help/index.html) - [オラクルは](../Page/オラクル_\(企業\).md "wikilink")2種類のフォーマットを開発している。Oracle Help for Java（OHJ）と Oracle Help for Web（OHW）である。
  - DotNetHelp - HTML Help の代替となることを意図したフォーマット。
  - [Texinfo](../Page/Texinfo.md "wikilink") - [GNUプロジェクト](../Page/GNUプロジェクト.md "wikilink")における公式の[ドキュメンテーションシステム](../Page/ソフトウェアドキュメンテーション.md "wikilink")
  - [UNIXのmanページ](https://ja.wikipedia.org/wiki/manページ "wikilink") - [UNIX](../Page/UNIX.md "wikilink")コマンドの標準のヘルプ（ドキュメンテーション）システム
  - [Information Presentation Facility](https://ja.wikipedia.org/wiki/Information_Presentation_Facility "wikilink") (IPF) - IBMの[OS/2](https://ja.wikipedia.org/wiki/OS/2 "wikilink")システムで使われたヘルプシステム
