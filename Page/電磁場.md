**電磁場**（でんじば, , EMF）、あるいは**電磁界**（でんじかい）は、[電場](../Page/電場.md "wikilink")（電界）と[磁場](../Page/磁場.md "wikilink")（磁界）の総称\[1\]。

電場と磁場は時間的に変化しないような静的な場合を除いて必ず同時に存在し、[マクスウェル方程式](https://ja.wikipedia.org/wiki/マクスウェル方程式 "wikilink")で関連づけられる\[2\]。電場、磁場が時間的に一定で 0 でない場合、それぞれは分離され、[静電場](https://ja.wikipedia.org/wiki/静電場 "wikilink")、[静磁場](https://ja.wikipedia.org/wiki/静磁場 "wikilink")として別々に扱われる。

電磁場の変動が[波動](../Page/波動.md "wikilink")として[空間](../Page/空間.md "wikilink")中を伝播するとき、これを[電磁波](../Page/電磁波.md "wikilink")という。

## 概念

電磁場という用語を単なる[概念](../Page/概念.md "wikilink")として用いる場合と、[物理量](../Page/物理量.md "wikilink")として用いる場合がある。 概念として用いる場合は、[電場](../Page/電場.md "wikilink")の強度と[電束密度](../Page/電束密度.md "wikilink")、あるいは[磁場](../Page/磁場.md "wikilink")の強度と[磁束密度](../Page/磁束密度.md "wikilink")を明確に区別せずに用いるが、物理量として用いる場合は電場の強度と磁束密度の組であることが多い。 また、これらの物理量は[電磁ポテンシャル](../Page/電磁ポテンシャル.md "wikilink")によっても記述され、[ラグランジュ形式](https://ja.wikipedia.org/wiki/ラグランジュ形式 "wikilink")などで扱う場合は電磁ポテンシャルが基本的な物理量として扱われる。このような場合には電磁ポテンシャルを指して電磁場という事もある。[CGS単位系](../Page/CGS単位系.md "wikilink")では電場と磁場は同一の[物理次元](https://ja.wikipedia.org/wiki/物理次元 "wikilink")を持つが、[MKSA単位系](../Page/MKSA単位系.md "wikilink")では \([ E ] = c [ B ]\) となっている (\(c\) は[光速](../Page/光速.md "wikilink"))。

電磁場のふるまいは、**[マクスウェルの方程式](../Page/マクスウェルの方程式.md "wikilink")**、あるいは**[量子電磁力学](../Page/量子電磁力学.md "wikilink")**(QED)によって記述される。[マクスウェルの方程式](../Page/マクスウェルの方程式.md "wikilink")を解いて、電磁場のふるまいについて解析することを[電磁場解析](https://ja.wikipedia.org/wiki/電磁場解析 "wikilink")と言う。

## 電場と磁場の関係

電場と磁場は[ローレンツ変換](../Page/ローレンツ変換.md "wikilink")により互いに移り合う。座標系 O で電場 \(\mathbf{E}\), 磁場（磁束密度） \(\mathbf{B}\) が存在するとき、x軸方向に速度 v で運動する座標系 O' では次の電磁場 \(\mathbf{E}'\), \(\mathbf{B}'\) として観測される\[3\]。

\[E'_x = E_x, \ \ E'_y = \frac{ E_y - v B_z }{ \sqrt{ 1 - v^2/c^2 } } , \ \ E'_z = \frac{ E_z + v B_y }{ \sqrt{ 1 - v^2/c^2 } }\]

\[B'_x = B_x, \ \ B'_y = \frac{ B_y + v c^{-2} E_z }{ \sqrt{ 1 - v^2/c^2 } } , \ \ B'_z = \frac{ B_z - v c^{-2} E_y }{ \sqrt{ 1 - v^2/c^2 } }\] 特に \(v / c \ll 1\) のとき、これらの等式は次の公式に帰着される。

\[\mathbf{E}' = \mathbf{E} - \mathbf{B} \times \mathbf{v} , \ \ \mathbf{B}' = \mathbf{B} + \frac{ 1 }{ c^2 } \mathbf{E} \times \mathbf{v}\] また、\(\mathbf{E}^2 - c^2 \mathbf{B}^2\) と \(\mathbf{E} \cdot \mathbf{B}\) というふたつのスカラー量はローレンツ不変である。なお電場と磁場は[電磁テンソル](https://ja.wikipedia.org/wiki/電磁テンソル "wikilink")という単一の反対称テンソルとして統一的に扱うことができる。

## 電磁場のエネルギーと運動量

電磁場はそれ自体[エネルギー](https://ja.wikipedia.org/wiki/エネルギー "wikilink")と[運動量](../Page/運動量.md "wikilink")を担い、その密度 (エネルギー密度 \(u\) と運動量密度 \(\mathbf{p}\)) は次式で与えられる\[4\]\[5\]。

\[u = \frac{ \varepsilon_0 }{ 2 } \mathbf{E}^2 + \frac{ 1 }{ 2 \mu_0 } \mathbf{B}^2\]

\[\mathbf{p} = c^2 \mathbf{S} , \ \ \mathbf{S} = \frac{ 1 }{ \mu_0 } \mathbf{E} \times \mathbf{B}\] ここに \(\mathbf{S}\) は[ポインティング・ベクトル](../Page/ポインティング・ベクトル.md "wikilink")である。その保存則として次の連続の式が成り立つ。

\[\frac{ \partial u }{ \partial t } + \nabla \cdot \mathbf{S} = 0 , \ \ \frac{ \partial \mathbf{p} }{ \partial t } + \nabla \cdot \sigma = 0\] 従ってポインティングベクトルは電磁場の運動量密度を表すと同時に、電磁場のエネルギー流速密度をも表している。また \(\sigma\) は[マクスウェルの応力テンソル](../Page/マクスウェルの応力テンソル.md "wikilink")である\[6\]。

\[\sigma_{i j} = \varepsilon_0 \left( - E_i E_j + \frac{ 1 }{ 2 } \delta_{i j} \mathbf{E}^2 \right) + \frac{ 1 }{ \mu_0 } \left( - B_i B_j + \frac{ 1 }{ 2 } \delta_{i j} \mathbf{B}^2 \right)\]
