**スネルの法則**（スネルのほうそく、）とは[波動](../Page/波動.md "wikilink")一般の[屈折](../Page/屈折.md "wikilink")現象における二つの媒質中の[進行波](https://ja.wikipedia.org/wiki/進行波 "wikilink")の伝播速度と入射角・屈折角の関係を表した[法則](https://ja.wikipedia.org/wiki/法則 "wikilink")のことである。**屈折の法則**（くっせつのほうそく）とも呼ばれる。この法則は[ホイヘンスの原理](https://ja.wikipedia.org/wiki/ホイヘンスの原理 "wikilink")によって説明することができる。

## 定義

媒質Aにおける波の速度を\(v_{\mathrm{A}}\)、媒質Bにおける波の速度を\(v_{\mathrm{B}}\)、媒質Aから媒質Bへの入射角（またはBからAへの屈折角）を\(\theta_{\mathrm{A}}\)、媒質Bから媒質Aへの入射角（またはAからBへの屈折角）を\(\theta_{\mathrm{B}}\)とすると、以下の関係が成立する。

\[{ \sin \theta_{\mathrm{A}} \over {\sin \theta_{\mathrm{B}}} } = {v_{\mathrm{A}} \over {v_{\mathrm{B}}} }\]

ここで、\({v_{\mathrm{A}} \over {v_{\mathrm{B}}} }\)の値を**媒質Aに対する媒質Bの相対屈折率**と定義し、これを\(n_{\mathrm{AB}}\)（または\(n_{\mathrm{A\rightarrow B}}\)）で表す。以上のことをまとめると

\[{ \sin \theta_{\mathrm{A}} \over {\sin \theta_{\mathrm{B}}} } = {v_{\mathrm{A}} \over {v_{\mathrm{B}}} } = n_{{\mathrm{AB}}}\] となる。

## 歴史

[アレクサンドリア](../Page/アレクサンドリア.md "wikilink")のギリシャ人[プトレマイオス](../Page/クラウディオス・プトレマイオス.md "wikilink") \[1\] は光の入射角・屈折角の関係を見出したが、角度が大きいときには不正確だった。プトレマイオスは実験に基づいた正確な法則を見つけたと確信していたが、理論に合うようにデータをごまかしていた([確証バイアス](https://ja.wikipedia.org/wiki/確証バイアス "wikilink"))\[2\]。[イブン・アル・ハイサム](https://ja.wikipedia.org/wiki/イブン・アル・ハイサム "wikilink")は著書「光学の書」(1021)で屈折の法則の発見により近づいたが、発見には至らなかった\[3\]。

屈折の法則は、[バグダッド](https://ja.wikipedia.org/wiki/バグダッド "wikilink")のイブン・サフル([Ibn Sahl](https://ja.wikipedia.org/wiki/:en:Ibn_Sahl "wikilink"))の論文"Burning Mirrors and Lenses"(984)の中で初めて正確に記述された\[4\]\[5\]。サフルは幾何収差のないレンズの形状を算出するためにこの法則を利用した\[6\]。

屈折の法則は1602年に[トーマス・ハリオット](https://ja.wikipedia.org/wiki/トーマス・ハリオット "wikilink")によって再発見された\[7\]。ハリオットはこのテーマについて[ケプラーと文通していたにも関わらず](../Page/ヨハネス・ケプラー.md "wikilink")、この結果は出版されなかった。1621年に[ヴィレブロルト・スネル](https://ja.wikipedia.org/wiki/ヴィレブロルト・スネル "wikilink")も独立にこの法則を発見したが、生前には出版されなかった。これと独立して[ルネ・デカルト](../Page/ルネ・デカルト.md "wikilink")は1637年に発表した[方法序説](https://ja.wikipedia.org/wiki/方法序説 "wikilink")試論において、発見的な[運動量](../Page/運動量.md "wikilink")保存の議論を使って[正弦関数](https://ja.wikipedia.org/wiki/正弦関数 "wikilink")で表された屈折の法則を導き、光学の問題を解くために利用した。[ピエール・ド・フェルマー](https://ja.wikipedia.org/wiki/ピエール・ド・フェルマー "wikilink")はデカルトの導出を受け入れず、自身の[最小時間の原理に基づいて同じ結果を導いた](https://ja.wikipedia.org/wiki/フェルマーの原理 "wikilink")。

科学史家のディクステルホイスによれば\[8\]、「デカルトはスネルの論文を見て自分の証明を作り上げたと、イサーク・フォシウス([Issac Vossius](https://ja.wikipedia.org/wiki/:en:Isaac_Vossius "wikilink"))が"De natura lucis et proprietate"\[9\]の中で述べている。我々は今日この非難が不当なものであると知っているが、この話はこれまで何度も採用されてきた。」という。フェルマーとホイヘンスも、デカルトがスネルの論文を盗用したと非難している。

フランス語でスネルの法則は「デカルトの法則」「スネル-デカルトの法則」と呼ばれている。

[クリスティアーン・ホイヘンス](../Page/クリスティアーン・ホイヘンス.md "wikilink")は1678年に「光についての論考」の中で、今日[ホイヘンス＝フレネルの原理](https://ja.wikipedia.org/wiki/ホイヘンス＝フレネルの原理 "wikilink")と呼ばれる手法を使って、スネルの法則がどのように光の波動性から導かれるのかを明らかにした。

## 発展

[媒質](https://ja.wikipedia.org/wiki/媒質 "wikilink")が変化しても同一波の[周波数](../Page/周波数.md "wikilink")は変化しないので、上の[法則](https://ja.wikipedia.org/wiki/法則 "wikilink")をさらに発展させると、次のようになる。

\[{ \sin \theta_\mathrm{A} \over {\sin \theta_\mathrm{B}} } = {\lambda_\mathrm{A} \over {\lambda_\mathrm{B}} } = {v_\mathrm{A} \over {v_\mathrm{B}} } = n_\mathrm{AB}\]

\[\lambda_\mathrm{A}\]：媒質Aでの[波](../Page/波.md "wikilink")の[波長](../Page/波長.md "wikilink")

\[\lambda_\mathrm{B}\]：媒質Bでの波の波長

## 光波への発展

[光](../Page/光.md "wikilink")波は真空中も伝わる波なので、光波においては真空に対する物質固有の相対屈折率を**絶対屈折率**と定義する。ここで媒質Aの絶対屈折率を\(n_\mathrm{A}\)、媒質Bの絶対屈折率を\(n_\mathrm{B}\)、と表すと

\[{n_\mathrm{B} \over n_\mathrm{A}} = n_\mathrm{AB}\]

よって以上のことをまとめて

\[{ \sin \theta_\mathrm{A} \over {\sin \theta_\mathrm{B}} } = {\lambda_\mathrm{A} \over {\lambda_\mathrm{B}} } = {v_\mathrm{A} \over {v_\mathrm{B}} } = {n_\mathrm{B} \over n_\mathrm{A}} = n_\mathrm{AB}\]

という関係が成り立つ。

また、平行多重層における屈折については、媒質Xの絶対屈折率を\(n_\mathrm{X}\)と表すと、

\[{n_\mathrm{A} \sin \theta_\mathrm{A}} = {n_\mathrm{B} \sin \theta_\mathrm{B}} = {n_\mathrm{C} \sin \theta_\mathrm{C}} = \dots =\]一定

という関係が成り立ち、これは2媒質間に他媒質があった場合でもそれを無視してこの法則を用いることができることを示す。

ここで注意しておきたいのは、**絶対屈折率は光波についてのみの概念**であるということである。（電磁波以外の波は真空中には存在しない。） また、複屈折に於ける

\[{ \sin \theta_\mathrm{A} \over {\sin \theta_\mathrm{B}} }\]　は、常光線では角度によらない一定値であるが異常光線の方は角度に依存する。 狭義の定義ではスネルの法則とは屈折率 \(n\) は一定なのであるが、屈折率が角度の関数 \(n(\theta)\) である場合も（広義の）スネルの法則という。

## 全反射

以上の公式により、[臨界角](https://ja.wikipedia.org/wiki/臨界角 "wikilink")(屈折が起こる最大の[入射角](https://ja.wikipedia.org/wiki/入射角 "wikilink"))の大きさが屈折率によって定まることが分かる。 \(n_\mathrm{B} > n_\mathrm{A}\)で光が媒質Bから媒質Aに入射するとき、

\[\sin \theta_m = {\sin \theta_m\over {\sin 90^\circ}} = {1\over {n_\mathrm{AB}} }={n_\mathrm{A}\over{n_\mathrm{B}}}\]

\[\theta_m\]:臨界角(媒質Bから媒質Aへの入射角)

媒質BからAへの入射角を\(\theta_\mathrm{B}\)とすると、

\[\theta_\mathrm{B} > \theta_m\] が全反射の起こる条件である。
