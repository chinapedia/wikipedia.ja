**Intel 80186**（インテル 80186）は、[インテル](https://ja.wikipedia.org/wiki/インテル "wikilink")が[1982年](../Page/1982年.md "wikilink")6月に発表した、主として[組み込みシステム](../Page/組み込みシステム.md "wikilink")向けの[マイクロプロセッサ](../Page/マイクロプロセッサ.md "wikilink")、ないし[マイクロコントローラ](../Page/マイクロコントローラ.md "wikilink")である。

## 特徴

同年2月に発表されていた[80286や以後のチップとは異なり](../Page/Intel_80286.md "wikilink")、[8086に対して](../Page/Intel_8086.md "wikilink")11個の命令が追加されたが、命令セットアーキテクチャの大幅は拡張はなかった。コアの拡張ではなく周辺IC（クロックジェネレータ、割り込みコントローラ、タイマ、DMAC、チップセレクタ）をオンダイに組み込んだ性格的には[マイクロコントローラ](../Page/マイクロコントローラ.md "wikilink")に近いチップである。

一部の命令の実行クロック数の削減により、（プログラム内の命令に依るが）同一クロックで動作する8086と比較して1.3倍程度の性能がある。

[thumb](https://ja.wikipedia.org/wiki/ファイル:80186_Back.JPG "wikilink")

写真の独特なパッケージ形状は CLCC（セラミック・リードレス・チップ・キャリア）と称されるもので、専用のソケットを用いなければ基板に実装できない。のちに PLCC, QFP, PGA など他のパッケージも供給されるようになった。パッケージ形状はチップ名先頭で区別され、それぞれ R(CLCC), N(PLCC), S(QFP), A(PGA)となる。

もっとも、[IBM PC等で使用されているDMAC等とは互換性が無いため](../Page/IBM_PC.md "wikilink")、[パーソナルコンピュータ](../Page/パーソナルコンピュータ.md "wikilink")では、部品点数削減が重要なブック型パソコンで使用された程度である。

8086互換プロセッサを展開していた[NEC Vシリーズ中では](../Page/NEC_Vシリーズ.md "wikilink")、V35/V25が機能的に近い。

## 追加された命令

先行する80286に追加されていた新たな命令のうち、プロテクトモードや80287に関連しない汎用の命令11個\[1\]が80186にも採用された。これらはもともと286命令の一部であるが、数字的に186より大きい数字のプロセッサで使用できるようになったことから、結果的に「186命令」などとされることもある。

`ENTER … Make stack frame for procedure`
`LEAVE … High level procedure exit`
`PUSHA … Push all general registers`
`POPA … Pop all general registers`
`BOUND … Check array index against bounds`
`INS … Input string`
`INSB … Input string byte`
`INSW … Input string word`
`OUTS … Output string`
`OUTSB … Output string byte`
`OUTSW … Output string word`

このほか80186/80286では以下の10命令\[2\]\[3\]に対しても、既存の命令に対して[アドレッシングモード](../Page/アドレッシングモード.md "wikilink")が追加されている。いずれもオペランドにイミディエイト（即値）が使えるようになった\[4\]。

`IMUL   Integer multiplication`
`PUSH   Push word onto stack`
`RCL    Rotate left through carry`
`RCR    Rotate right through carry`
`ROL    Rotate left`
`ROR    Rotate right`
`SAL    Shift arithmetic left`
`SHL    Shift left logical`
`SAR    Shift arithmetic right`
`SHR    Shift right logical`

## その他の80186シリーズ

  - [80188](https://ja.wikipedia.org/wiki/Intel_80188 "wikilink")
    外部[データバス](https://ja.wikipedia.org/wiki/データバス "wikilink")幅が8ビットのCPU。[8086に対する](../Page/Intel_8086.md "wikilink")[8088にあたる](../Page/Intel_8088.md "wikilink")。
  - 18677
    16MBのメモリを扱えるように機能拡張したもの。富士通、インテルジャパン共同開発。
  - 80C186XL/80C188XL
    80186/80188を[CMOS](../Page/CMOS.md "wikilink")化し高クロック (16MHz) 低消費電力化したもの
  - 80C186Ex/80C188Ex
    シリアルインターフェイス等の周辺ICをさらに追加し、3V動作、パワーマネジメント機能を追加。
