**度数分布**（どすうぶんぷ、Frequency Distribution）は、[統計](../Page/統計.md "wikilink")において[標本として得たある変量の値のリストである](../Page/標本_\(統計学\).md "wikilink")。量の大小の順で並べ、各数値が現われた個数を表示する表（**度数分布表**）で示す。日本工業規格では、「特性値と，その度数または相対度数との関係を観測したもの」と定義している。

## 例

例えば、100人がある文章に同意するかを5段階の[リッカート尺度](../Page/リッカート尺度.md "wikilink")で回答したとする。このとき、1 は強く同意することを示し、5 は全く同意しないことを示す。その回答群を度数分布で表すと次のようになる:

| [階級](https://ja.wikipedia.org/wiki/階級 "wikilink") | 同意の度合      | 回答数 |
| ------------------------------------------------- | ---------- | --- |
| 1                                                 | 強く同意する     | 25  |
| 2                                                 | ある程度同意する   | 35  |
| 3                                                 | どちらとも言えない  | 20  |
| 4                                                 | ある程度同意できない | 15  |
| 5                                                 | 全く同意できない   | 5   |

この単純な表には2つの弱点がある。変量が連続的な値をとりうる場合や、変量が取りうる値の範囲が非常に広い場合、度数分布表の作成は難しくなり、分析に適さなくなる。そこで、これを若干修正した表の形態として、変量の値をある範囲で等間隔に区切る方法が取られる。例えば、生徒の身長の統計をとる場合、度数分布表は以下のようになる。

| 身長の範囲        | 生徒数 | 累積  |
| ------------ | --- | --- |
| 135 - 150 cm | 25  | 25  |
| 150 - 165 cm | 35  | 60  |
| 165 - 180 cm | 20  | 80  |
| 180 - 195 cm | 20  | 100 |
|              |     |     |

## 応用

度数分布化されたデータは元のデータよりも扱いが容易である。度数分布から中央値、平均値、標準偏差を求める単純なアルゴリズムが存在する。

複数の度数分布間の差異と類似性の評価から[仮説検定](../Page/仮説検定.md "wikilink")が考案された。この評価には[代表値](https://ja.wikipedia.org/wiki/代表値 "wikilink")や[平均](../Page/平均.md "wikilink")の測定や[統計的ばらつき](../Page/統計的ばらつき.md "wikilink")（[標準偏差](../Page/標準偏差.md "wikilink")や[分散](../Page/分散_\(確率論\).md "wikilink")）の測定が関係する。

平均と[中央値](../Page/中央値.md "wikilink")が異なる場合、度数分布に[歪みがあると称する](../Page/歪度.md "wikilink")。度数分布の[尖度](../Page/尖度.md "wikilink")とは、平均値への集中の度合であり、[ヒストグラム](../Page/ヒストグラム.md "wikilink")で表した場合のグラフの尖り具合である。[正規分布](https://ja.wikipedia.org/wiki/正規分布 "wikilink")以上に尖っている場合を「急尖的; leptokurtic」と称し、逆の場合を「緩尖的; platykurtic」と称する。

度数分布は暗号解読での[頻度解析にも使われる](../Page/頻度分析_\(暗号\).md "wikilink")。これは文字の種類ごとの出現頻度と言語における文字種ごとの出現頻度から暗号を解読する手法である。
