**オストログラドスキーの定理** (オストログラドスキーのていり、) とは、[力学変数](https://ja.wikipedia.org/wiki/力学変数 "wikilink")の高階微分を[運動方程式](https://ja.wikipedia.org/wiki/運動方程式 "wikilink")に含む物理系の[ハミルトニアン](../Page/ハミルトニアン.md "wikilink")が下に非有界となることを述べる定理である。1850年に[ミハイル・オストログラドスキーにより証明された](https://ja.wikipedia.org/wiki/ミハイル・オストログラツキー "wikilink")\[1\]。この定理は、運動方程式に高階微分項が含まれる系には一定の条件を満足する場合を除き物理的に不安定なモードが存在するため、そのような系は物理的ではないことを示している。2007年の Woodard の講義録\[2\]で紹介されたことにより[修正重力理論](https://ja.wikipedia.org/wiki/修正重力理論 "wikilink")および[宇宙論](../Page/宇宙論.md "wikilink")の文脈でこの定理への関心が高まった\[3\]。

## 概要

[一般化座標](https://ja.wikipedia.org/wiki/一般化座標系 "wikilink") \(q\) により記述される物理系について考える。通常の物理系ではその[ラグランジアン](https://ja.wikipedia.org/wiki/ラグランジアン "wikilink") \(L\) は座標 \(q\) と速度 \(\dot{q}\) の関数 \(L = L ( q, \dot{q} )\) である\[4\]。 このとき、系の運動方程式は[ラグランジュ方程式](https://ja.wikipedia.org/wiki/ラグランジュ方程式 "wikilink")

\[\frac{ \partial L }{ \partial q } - \frac{ d }{ d t } \frac{ \partial L }{ \partial \dot{q} } = 0\] であり、従ってこれは座標 \(q\) に関する2階[微分方程式である](../Page/常微分方程式.md "wikilink")、つまりこの系の運動方程式は座標 \(q\) の時間による2階微分 ([加速度](../Page/加速度.md "wikilink")) \(\ddot{q}\) を含む。例えばラグランジアン \(L = \frac{1}{2} m \dot{q}^2 - V ( q )\) に対応する[ニュートンの運動方程式](../Page/ニュートンの運動方程式.md "wikilink")

\[m \frac{ d^2 q }{ d t^2 } = F ( q ) , \ \ F ( q ) = - \frac{ d V }{ d q } ( q )\] はまさにそのようになっている他、[電磁場](../Page/電磁場.md "wikilink")の方程式である[マクスウェル方程式](https://ja.wikipedia.org/wiki/マクスウェル方程式 "wikilink")や[重力場](../Page/重力場.md "wikilink")の方程式である[アインシュタイン方程式](../Page/アインシュタイン方程式.md "wikilink")も同じように2階微分方程式である\[5\]。

ラグランジアン \(L\) が座標 \(q\), 速度 \(\dot{q}\), 加速度 \(\ddot{q}\) の関数 \(L = L ( q, \dot{q}, \ddot{q} )\) であるような物理系について考えると、そのラグランジュ方程式

\[\frac{ \partial L }{ \partial q } - \frac{ d }{ d t } \frac{ \partial L }{ \partial \dot{q} } + \frac{ d^2 }{ d t^2 } \frac{ \partial L }{ \partial \ddot{q} } = 0\] は座標 \(q\) に関する3階以上の微分方程式になり得る。このような系に関して、オストログラドスキーの定理は次のことを主張する\[6\]。

  -
    ラグランジアン \(L = L ( q, \dot{q}, \ddot{q} )\) により記述される物理系について、そのラグランジアンが加速度 \(\ddot{q}\) に関する非縮退条件 \(\frac{ \partial^2 L }{ \partial \ddot{q}^2 } \neq 0\) を満足するとき、この系のハミルトニアン \(H\) は上下ともに非有界となる。

一般にオストログラドスキーの定理が適用される系のハミルトニアンは、少なくともひとつの[正準運動量](https://ja.wikipedia.org/wiki/正準座標 "wikilink") \(P\) の線型な関数であり、上下ともに非有界となる。このこと自体は不安定性の存在を直ちに意味するわけではないが、そのような系が他の[自由度](https://ja.wikipedia.org/wiki/自由度 "wikilink")と相互作用を持つと、[エネルギー](https://ja.wikipedia.org/wiki/エネルギー "wikilink")最小状態 ([基底状態](https://ja.wikipedia.org/wiki/基底状態 "wikilink")) が存在しないため不安定な系となる\[7\]\[8\]。この意味でオストログラドスキーの定理により存在が保証される不安定性は線型不安定性またはオストログラドスキー不安定性と呼ばれ、また線型不安定性を持つ力学自由度のことをオストログラドスキーゴーストと呼ぶ\[9\]。オストログラドスキーの定理は、通常の物理系の運動方程式が (ニュートンの運動方程式のように) 2階微分方程式として定式化される理由を説明すると解釈される\[10\]。

より一般的な多自由度系におけるオストログラドスキー不安定性の解析は Motohashi et al. (2016) で与えられている\[11\]。

## 具体例

次のラグランジアンにより記述される系について考える\[12\]。

\[L = \frac{ 1 }{ 2 } \ddot{q}^2\] この系の運動方程式は \(\frac{ d^4 q }{ d t^4 } = 0\) という4階微分方程式である。この系は[ラグランジュ未定乗数](../Page/ラグランジュの未定乗数法.md "wikilink") \(\lambda\) を導入することにより、座標 \(( q, Q, \lambda )\) で記述される次のラグランジアンへと書き換えられる。

\[L = \frac{ 1 }{ 2 } \dot{Q}^2 - \lambda ( Q - \dot{q} )\] これを[正準形式へと書き換えることを考える](../Page/ハミルトン力学.md "wikilink")。\(q\), \(Q\), \(\lambda\) に対応する[正準運動量をそれぞれ](https://ja.wikipedia.org/wiki/正準座標 "wikilink") \(p\), \(P\), \(\Lambda\) とすると

\[p = \frac{ \partial L }{ \partial \dot{q} } = \lambda , \ \ P = \frac{ \partial L }{ \partial \dot{Q} } = \dot{Q} , \ \ \Lambda = \frac{ \partial L }{ \partial \dot{\lambda} } = 0\] となるが、このうち第1式および第3式は一般化速度 \(\dot{q}\), \(\dot{\lambda}\) について解くことができないため、これはとなっている。対応するハミルトニアン

\[H = p \dot{q} + P \dot{Q} + \Lambda \dot{\lambda} - L\] は、拘束条件を満足する超曲面上での等式 () として

\[H = \frac{ 1 }{ 2 } P^2 + p Q\] という形に求まる。これは運動量 \(p\) に線型に依存し、上下ともに非有界である。こうしてオストログラドスキーの定理が成立することが確認される\[13\]。なお一般に定理が成立することを証明するには、この議論を任意のラグランジアン \(L ( q, \dot{q}, \ddot{q} )\) に対して実行すればよい。

この系についてオストログラドスキーゴーストに対応する自由度を次のような書き換えを通じて陽に分離することも可能である。まず、上の系は

\[L = \ddot{q} Q - \frac{ 1 }{ 2 } Q^2\] というラグランジアンと等価である (つまり等価な運動方程式を導く)。そこで新しい変数 \(x\), \(y\) を

\[x = \frac{ q + Q }{ \sqrt{ 2 } } , \ \ y = \frac{ q - Q }{ \sqrt{ 2 } }\] により定義し、ラグランジアンを座標 \(x\), \(y\) を用いて書き直すと

\[L = - \frac{ 1 }{ 2 } \dot{x}^2 + \frac{ 1 }{ 2 } \dot{y}^2 - \frac{ 1 }{ 4 } ( x - y )^2\] となる。このとき、座標 \(x\) に関する運動項 \(- \frac{ 1 }{ 2 } \dot{x}^2\) は負符号を取り、このためにハミルトニアンが下に非有界となっていることが見て取れる。すなわち力学変数 \(x\) が表す自由度がオストログラドスキーゴーストである\[14\]。

## 修正重力理論への応用

[一般相対性理論](https://ja.wikipedia.org/wiki/一般相対性理論 "wikilink")では[重力場](../Page/重力場.md "wikilink")は[計量テンソル](../Page/計量テンソル.md "wikilink") \(g_{\mu \nu}\) により表現され、[アインシュタイン方程式](../Page/アインシュタイン方程式.md "wikilink")に従う。重力場に加えて[スカラー場](https://ja.wikipedia.org/wiki/スカラー場 "wikilink") \(\phi\) の自由度を持つ[修正重力理論](https://ja.wikipedia.org/wiki/修正重力理論 "wikilink") ([スカラー・テンソル理論](https://ja.wikipedia.org/wiki/スカラー・テンソル理論 "wikilink")) において、ラグランジアンに高階微分を含むが運動方程式が2階微分方程式となる最も一般的な理論のクラスとして[ホルンデスキー理論](../Page/ホルンデスキー理論.md "wikilink")が知られている。その拡張として運動方程式に高階微分を含む理論を考えようとすると、オストログラドスキー不安定性のために一般にはゴーストが現れるため、健全な理論を構成することは自明ではなくなる\[15\]。Langlois & Noui は2015年に実際にスカラー・テンソル理論に対してオストログラドスキーの定理を適用し、オストログラドスキーゴーストが存在しないが運動方程式が高階微分であるような理論を構成した\[16\]。これはオストログラドスキーの定理の仮定である非縮退条件を破る (縮退条件を課す) ことによって実現される\[17\]。
