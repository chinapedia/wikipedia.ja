**トレミーの定理**（トレミーのていり、）とは、[円に内接する](../Page/円_\(数学\).md "wikilink")[四角形](../Page/四角形.md "wikilink") *ABCD* において、[辺](https://ja.wikipedia.org/wiki/辺 "wikilink")の長さに関する[等式](../Page/等式.md "wikilink")：

\[AC\cdot BD = AD\cdot BC + AB\cdot DC\] が成り立つという[幾何学の](../Page/ユークリッド幾何学.md "wikilink")[定理](../Page/定理.md "wikilink")。トレミーは[古代ギリシア](../Page/古代ギリシア.md "wikilink")の[天文学者](../Page/天文学者.md "wikilink")[クラウディオス・プトレマイオス](../Page/クラウディオス・プトレマイオス.md "wikilink")の姓プトレマイオスの英語表記Ptolemyの音訳である。プトレマイオスの定理とも呼ばれる\[1\]。

トレミーの定理を一般化した**オイラーの定理**（オイラーのていり）とは、必ずしも円に内接しない四角形 *ABCD* において、辺の長さに関する**トレミーの不等式**（）：

\[AC\cdot BD \leqq AD\cdot BC + AB\cdot DC\] が成り立つという幾何学の定理のことである\[2\]。逆に、必ずしも同一平面上にない4点 *A*, *B*, *C*, *D* に関して、辺の長さに関する等式：

\[AC\cdot BD = AD\cdot BC + AB\cdot DC\] が成り立つならば、4点 *A*, *B*, *C*, *D* は同一直線上にあるか、または同一平面上にあり、かつ四角形 *ABCD* は同一の円に内接する\[3\]。

## 証明

計算の便宜をはかり、*a* = *AD*, *b* = *AB*, *c* = *BC*, *d* = *DC* とおくことにする。また、*A* = ∠*A* = ∠*DAB*, *B* = ∠*B* = ∠*ABC*, *C* = ∠*C* = ∠*BCD*, *D* = ∠*D* = ∠*CDA* のこととする。

[余弦定理](../Page/余弦定理.md "wikilink")および内接四角形の性質より、

\[BD^2 = a^2 + b^2 -2ab \cos A\]、

\[BD^2 = c^2 + d^2 -2cd \cos C = c^2 + d^2 +2cd \cos A\] が成り立つ。ここから cos *A* を消去して、

\[(ab + cd)BD^2 = (ad + bc)(ac + bd)\] を得る。また *AC* について同様にして

\[(ad + bc)AC^2 = (ab + cd)(ac + bd)\] となるから、2 式を掛けて

\[(ab + cd)(bc + ad)AC^2\cdot BD^2 = (ac + bd)^2(ad + bc)(ab + cd)\] を得る。これを整理すれば、

\[AC \cdot BD = ac + bd\] となる。すなわち、

\[AC\cdot BD = AD\cdot BC + AB\cdot DC\] が示された。

### 円に関する反転を用いた証明

[Ptolemy-crop.svg](https://ja.wikipedia.org/wiki/File:Ptolemy-crop.svg "fig:Ptolemy-crop.svg")

Dを中心とする適当な円 \(\Gamma\) に関する[反転](https://ja.wikipedia.org/wiki/反転幾何学#円に関する反転 "wikilink") によってABCDの内接円が直線に移されるようにする。 このとき \(A'B' + B'C' = A'C'\) が成り立つ。 このとき、一般性を失わずに \(\Gamma\) の半径を1と置くことができる。 このとき \(A'B', B'C', A'C'\)はそれぞれ以下のように表される。

\[\frac{AB}{DA \cdot DB}, \frac{BC}{DB \cdot DC}, \frac{AC}{DA \cdot DC}\] この式の両辺に \(DA \cdot DB \cdot DC\) をかけて最初の式に代入するとトレミーの定理が得られる。
