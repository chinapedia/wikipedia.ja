**[1975年](../Page/1975年.md "wikilink")の[パシフィック・リーグ](https://ja.wikipedia.org/wiki/パシフィック・リーグ "wikilink")[プレーオフ](../Page/プレーオフ制度_\(日本プロ野球\).md "wikilink")**は、前期優勝の[上田利治](../Page/上田利治.md "wikilink")監督率いる[阪急ブレーブスと後期優勝の](../Page/オリックス・バファローズ.md "wikilink")[西本幸雄](../Page/西本幸雄.md "wikilink")監督率いる[近鉄バファローズとの対決となった](../Page/大阪近鉄バファローズ.md "wikilink")。西本は2年前まで阪急の監督を務め、弱小チームだった阪急を5度の優勝を誇る強豪チームに育て上げた張本人であり、近鉄監督としてもBクラス常連のチームを就任2年でプレーオフ進出するまでに育て上げた。西本のチームを継承する形で阪急監督となった上田は、かつて阪急のコーチとして西本に師事しており、師弟対決としても話題を呼んだ。

序盤から点の取り合いになり、5回に[大熊忠義](../Page/大熊忠義.md "wikilink")のソロ本塁打で阪急が4-4と追いついたが、近鉄は次の6回に有田修のタイムリーで勝ち越し、なおも1死満塁で[伊勢孝夫](../Page/伊勢孝夫.md "wikilink")がグランドスラムを叩き込み、試合を決めた。阪急は5回途中から[足立光宏](../Page/足立光宏.md "wikilink")をリリーフした[山田久志](../Page/山田久志.md "wikilink")の乱調が誤算だった。

（近）●[神部](../Page/神部年男.md "wikilink")（1敗）、芝池－有田修

本塁打
（急）マルカーノ2号ソロ（6回神部）、河村1号ソロ（7回神部）、長池1号2ラン（8回神部）

1回、前日を勢いを残す近鉄は[山口高志](../Page/山口高志.md "wikilink")を攻め、[佐々木恭介](../Page/佐々木恭介.md "wikilink")、有田修のタイムリーで3点を先制した。ここで上田監督は投手の山口ではなく捕手の[中沢伸二](../Page/中沢伸二.md "wikilink")を交代する意外な選手起用策に出るが、これが功を奏し、山口を立ち直らせた。この選手起用は、[1958年の日本シリーズ](../Page/1958年の日本シリーズ.md "wikilink")第4戦（[西鉄](https://ja.wikipedia.org/wiki/埼玉西武ライオンズ "wikilink")－[巨人](https://ja.wikipedia.org/wiki/読売ジャイアンツ "wikilink")）で[稲尾和久](../Page/稲尾和久.md "wikilink")が序盤に打ち込まれた際、[三原脩](../Page/三原脩.md "wikilink")監督が投手ではなく捕手を代えて稲尾を立ち直らせることに成功したのを上田監督が思い出したことによるという（[近藤唯之](../Page/近藤唯之.md "wikilink")「ダッグアウトの指揮官たち」）。阪急はマルカーノ、中沢に代わった[河村健一郎](../Page/河村健一郎.md "wikilink")の本塁打で追い上げ、8回、[長池徳二が劇的な逆転](../Page/長池徳士.md "wikilink")2ランホームラン。山口は最後まで投げぬき、12安打を浴びながらも完投した。

雨で1日順延し、第1戦と同じ近鉄・鈴木、阪急・足立の顔合わせ。阪急は2回マルカーノのタイムリーで先制、3回長池の2ランが飛び出し、鈴木をKO。足立は2塁も踏ませないほぼ完璧な投球で3安打無四球の完封勝利。

### 第4戦

10月20日　藤井寺　観衆21000人

| 阪急 | 2 | 0 | 0 | 0 | 0 | 1 | 0 | 0 | 2 | 5 |
| -- | - | - | - | - | - | - | - | - | - | - |
| 近鉄 | 0 | 0 | 0 | 0 | 0 | 3 | 0 | 0 | 0 | 3 |

（急）○山口（2勝）－河村
（近）[井本](../Page/井本隆.md "wikilink")、●芝池（1勝1敗）－有田修
本塁打
（急）[福本](../Page/福本豊.md "wikilink")1号ソロ（9回芝池）、加藤秀1号ソロ（9回芝池）

0-3とリードされた近鉄は6回、無死満塁から佐々木の2点タイムリーで1点差に詰め寄り、さらに[西村俊二](https://ja.wikipedia.org/wiki/西村俊二 "wikilink")の[スクイズで同点に追いついた](../Page/スクイズプレイ.md "wikilink")。しかし9回、[福本豊](../Page/福本豊.md "wikilink")が芝池をとらえ、決勝本塁打。さらに[加藤秀司](../Page/加藤秀司.md "wikilink")がダメ押しの一発。阪急が3年ぶり、2シーズン制初の年間優勝を決めた。山口が第2戦に続く完投で胴上げ投手となった。阪急は過去2年間プレーオフで敗退しており、まさに三度目の正直だった。\[1\]
