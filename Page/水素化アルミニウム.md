| Section3 =  | Section8 =  }} **水素化アルミニウム**（すいそかアルミニウム、aluminum hydride）または**アラン**(alane)は、無色の固体で[化学式](../Page/化学式.md "wikilink")がAlH<sub>3</sub>で表される[無機化合物](https://ja.wikipedia.org/wiki/無機化合物 "wikilink")である。極めて酸化されやすく空気中で自然発火するため[危険物](../Page/危険物.md "wikilink")第3類に分類されている。そのため研究室内で[還元剤](../Page/還元剤.md "wikilink")として[有機合成](https://ja.wikipedia.org/wiki/有機合成 "wikilink")に使われる\[1\]以外ほとんど見られない。

## 構造

アランはポリマー構造をとり、その化学式はよく(AlH<sub>3</sub>)<sub>n</sub>と表される。アランは多数の[多形](../Page/多形.md "wikilink")体をとり、それぞれα-アラン、α’-アラン、β-アラン、δ-アラン、ε-アラン、θ-アラン、およびγ-アランと名付けられている。α-アランは[立方晶](https://ja.wikipedia.org/wiki/立方晶 "wikilink")または[菱面体晶](https://ja.wikipedia.org/wiki/菱面体晶 "wikilink")の形態ととるのに対し、α’-アランは針状結晶、γ-アランは結合した針の束状構造を作る。アランはテトラヒドロフランとエーテルに溶け、エーテルでの沈殿量はその調製法に依存する\[2\]。

α-アランは、アルミニウム原子が6個の水素原子に囲まれ、別のアルミニウム原子とはそれで架橋されている構造で決定された。Al-H間の距離はすべて同じ172 pmで、Al-H-Alの角度は141°である\[3\]。

α-アランは、最も熱的に安定な多形体である。β-アランとγ-アランは同時に合成され、熱するとα-アランに変わる。δ、ε、θ-アランはいろいろな結晶化条件で合成される。それらは安定ではないが、熱してもα-アランにはならない\[4\]。

### アランの分子構造

単量体のAlH<sub>3</sub>は、固体の希ガスマトリックス内の低温条件下で単離したところ平面形であることが示された\[5\]。二量体のAl<sub>2</sub>H<sub>6</sub>を固体水素中で単離したところ、[ジボラン](../Page/ジボラン.md "wikilink")(B<sub>2</sub>H<sub>6</sub>)と[ジガラン](https://ja.wikipedia.org/wiki/ジガラン "wikilink")(Ga<sub>2</sub>H<sub>6</sub>)と同形であった\[6\]\[7\]。

## 合成

水素化アルミニウムの不純物と関連化合物は古くから報告されている\[8\]。初めて合成法が公開されたのは1947年で、[アメリカ](https://ja.wikipedia.org/wiki/アメリカ "wikilink")では合成の特許が1999年にペトリらに割り当てられている\[9\]\[10\]。

主に[水素化アルミニウムリチウム](../Page/水素化アルミニウムリチウム.md "wikilink")の[エーテル](../Page/エーテル_\(化学\).md "wikilink")[溶液](../Page/溶液.md "wikilink")と[塩化アルミニウム](../Page/塩化アルミニウム.md "wikilink")から合成される\[11\]。この合成法は手順が複雑であり、反応後に[塩化リチウム](../Page/塩化リチウム.md "wikilink")を除去しなければならないことにも留意する必要がある。

  -
    <ce>3LiAlH4\\ + AlCl3 -\> 4AlH3\\ + 3LiCl</ce>

[塩化リチウム](../Page/塩化リチウム.md "wikilink")の沈殿の後、水素化アルミニウムの[エーテル](../Page/エーテル_\(化学\).md "wikilink")[溶液](../Page/溶液.md "wikilink")が得られる。

なお、[水素化アルミニウムリチウム](../Page/水素化アルミニウムリチウム.md "wikilink")からの生成は以下の方法でもできる。

  -
    <ce>2LiAlH4\\ + BeCl2 -\> 2AlH3\\ + LiBeH2Cl2</ce>
    <ce>2LiAlH4\\ + H2SO4 -\> 2AlH3\\ + Li2SO4\\ + 2H2</ce>
    <ce>2LiAlH4\\ + ZnCl2 -\> 2AlH3\\ + 2LiCl\\ + ZnH2</ce>

また、[水素化リチウム](../Page/水素化リチウム.md "wikilink")と[塩化アルミニウム](../Page/塩化アルミニウム.md "wikilink")の反応でも得られる。

  -
    <ce>LiH\\ + AlCl3 -\> AlH3\\ + 3LiCl</ce>

このほか、電気化学的な合成法もある\[12\]\[13\]。

## 反応

### ルイス塩基との付加体の形成

AlH<sub>3</sub>は容易に[強塩基](../Page/強塩基.md "wikilink")と付加体を作る。例えば、[トリメチルアミン](../Page/トリメチルアミン.md "wikilink")とは1:1、1:2の両方の錯体を作る。1:1錯体は気相では四面体構造であるが\[14\]、固相では水素中心が架橋した二量体構造となる(NMe<sub>3</sub>Al(μ-H))<sub>2</sub>\[15\]。一方、1:2錯体は[三方両錐形](https://ja.wikipedia.org/wiki/三方両錐形 "wikilink")の構造を採る\[16\]。いくつかの付加体（例えばジメチルエチルアミンアラン, NMe<sub>2</sub>Et.AlH<sub>3</sub>）は熱的に分解し金属アルミニウムが生成するため、[有機金属気相成長法](../Page/有機金属気相成長法.md "wikilink")に使われている\[17\]。

水素化リチウムと[エーテル](https://ja.wikipedia.org/wiki/エーテル "wikilink")中で反応させると[水素化アルミニウムリチウム](../Page/水素化アルミニウムリチウム.md "wikilink")が得られる。

  -
    <ce>AlH3\\ + LiH -\> LiAlH4</ce>

### 官能基との反応

[有機化学](../Page/有機化学.md "wikilink")では主に[官能基](https://ja.wikipedia.org/wiki/官能基 "wikilink")の[還元](../Page/還元.md "wikilink")に使われている\[18\]。反応は水素化アルミニウムリチウムとよく似ている。水素化アルミニウムは[アルデヒド](../Page/アルデヒド.md "wikilink")、[ケトン](../Page/ケトン.md "wikilink")、[カルボン酸](https://ja.wikipedia.org/wiki/カルボン酸 "wikilink")、[酸無水物](../Page/酸無水物.md "wikilink")、[酸塩化物](https://ja.wikipedia.org/wiki/酸塩化物 "wikilink")、[エステル](../Page/エステル.md "wikilink")および[ラクトン](../Page/ラクトン.md "wikilink")をそれぞれ対応する[アルコール](https://ja.wikipedia.org/wiki/アルコール "wikilink")へと[還元](../Page/還元.md "wikilink")する。[アミド](https://ja.wikipedia.org/wiki/アミド "wikilink")、[ニトリル](../Page/ニトリル.md "wikilink")、および[オキシム](../Page/オキシム.md "wikilink")はそれぞれ対応するアミンへと還元する。

官能基選択制に関しては、アランは他の水素化物試薬とは異なる。例えば、以下の[シクロヘキサノン](../Page/シクロヘキサノン.md "wikilink")の還元では、水素化アルミニウムリチウムは*trans* と *cis* の割合が1.9 : 1であるのに対し、水素化アルミニウムは7.3 : 1である\[19\]。 [Functional_Group_Reduction_1.svg](https://ja.wikipedia.org/wiki/File:Functional_Group_Reduction_1.svg "fig:Functional_Group_Reduction_1.svg")

数種のケトンのヒドロキシメチル化を可能にする\[20\]。ケトンは[エノラート](../Page/エノラート.md "wikilink")として保護されているためケトン自体は還元されない。 [Functional_Group_Reduction_2.svg](https://ja.wikipedia.org/wiki/File:Functional_Group_Reduction_2.svg "fig:Functional_Group_Reduction_2.svg")

[有機ハロゲン化合物](https://ja.wikipedia.org/wiki/有機ハロゲン化合物 "wikilink")はゆっくり還元されるか、まったく還元されない。そのため、ハロゲン部分が存在するカルボン酸などを還元させることができる\[21\]。 [Functional_Group_Reduction_3.svg](https://ja.wikipedia.org/wiki/File:Functional_Group_Reduction_3.svg "fig:Functional_Group_Reduction_3.svg")

ニトロ基は還元されない。同様にニトロ基のある[エステル](../Page/エステル.md "wikilink")を還元することができる\[22\]。 [Functional_Group_Reduction_4.svg](https://ja.wikipedia.org/wiki/File:Functional_Group_Reduction_4.svg "fig:Functional_Group_Reduction_4.svg")

アセタールを半保護されたアルコールへの還元にも使うことができる\[23\]。 [Functional_Group_Reduction_5.svg](https://ja.wikipedia.org/wiki/File:Functional_Group_Reduction_5.svg "fig:Functional_Group_Reduction_5.svg")

以下のようなエポキシド環の開環にも使える\[24\]。 [Epoxide_Ring_Opening.jpg](https://ja.wikipedia.org/wiki/File:Epoxide_Ring_Opening.jpg "fig:Epoxide_Ring_Opening.jpg")

アランが使われるアリル基転位は[S<sub>N</sub>2反応であり](../Page/求核置換反応.md "wikilink")、立体的な損傷がない\[25\]。 [Allylic_rearrangement_reaction.svg](https://ja.wikipedia.org/wiki/File:Allylic_rearrangement_reaction.svg "fig:Allylic_rearrangement_reaction.svg")

熱条件では[二酸化炭素](../Page/二酸化炭素.md "wikilink")を[メタン](../Page/メタン.md "wikilink")に還元できる。

  -
    <ce>4 AlH3\\ + 3 CO2 -\> 3 CH4\\ + 2 Al2O3</ce>
