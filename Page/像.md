[ミケランジェロのダビデ像](../Page/ダビデ像_\(ミケランジェロ\).md "wikilink")。\]\] **像**（ぞう）には、以下のとおり、複数の語義がある。

1.  物の[形](../Page/形.md "wikilink")。姿。→ [形](../Page/形.md "wikilink")を参照。
2.  [光](../Page/光.md "wikilink")が[光学](../Page/光学.md "wikilink")系を経て、再び集合してできた形象。[実像](../Page/実像.md "wikilink")と[虚像](https://ja.wikipedia.org/wiki/虚像 "wikilink")がある。
3.  [人](../Page/人.md "wikilink")や[物](../Page/物.md "wikilink")などの形を真似して作成された、[絵画](../Page/絵画.md "wikilink")や[彫刻](../Page/彫刻.md "wikilink")、[意匠など](../Page/デザイン.md "wikilink")。
4.  思い描いた姿、ありさま。→ [観念](../Page/観念.md "wikilink")、[イメージ](../Page/イメージ.md "wikilink")をも参照。
5.  [数学](../Page/数学.md "wikilink")において、[写像](../Page/写像.md "wikilink")によって写された先。[値域](../Page/値域.md "wikilink")の類語。→ 「[像 (数学)](../Page/像_\(数学\).md "wikilink")」「[像 (圏論)](https://ja.wikipedia.org/wiki/像_\(圏論\) "wikilink")」を参照。

本項では主に、上記の3について記載し、4についても記載する。

## 像の分類

様々な意味での区別・分類によって「像」は言い分けられ、使い分けられる。

### 心象的な像

#### 観念的な像の例

[観念](../Page/観念.md "wikilink")としての「人物像」 / [理想](https://ja.wikipedia.org/wiki/理想 "wikilink")像 / 人物の理想像（[ヒーロー](../Page/ヒーロー.md "wikilink")像、[アイドル](https://ja.wikipedia.org/wiki/アイドル "wikilink")像など） / [未来](../Page/未来.md "wikilink")像（将来像） / [政治](../Page/政治.md "wikilink")的[英雄](../Page/英雄.md "wikilink")像 / 性別・身分・職業・素性などに基づく像（男性像・女性像、[指導者](https://ja.wikipedia.org/wiki/指導者 "wikilink")像・[君主](../Page/君主.md "wikilink")像、[教師](https://ja.wikipedia.org/wiki/教師 "wikilink")像、[犯人](https://ja.wikipedia.org/wiki/犯人 "wikilink")像など） / [性的対象の像](../Page/性欲.md "wikilink")（[オナペット](../Page/オナペット.md "wikilink")など） / ほか

### 物象的な像

#### 物理的性質に着眼した分類

[平面](../Page/平面.md "wikilink")像（[二次元](../Page/2次元.md "wikilink")[画像](../Page/画像.md "wikilink")、二次元[動画](../Page/動画.md "wikilink")像） / [立体像](https://ja.wikipedia.org/wiki/立体像 "wikilink")（[stature](https://ja.wikipedia.org/wiki/:en:Statues "wikilink")、[三次元像](../Page/3次元.md "wikilink")） / 実像としての[映像](https://ja.wikipedia.org/wiki/映像 "wikilink")（三次元画像、三次元動画像） / 光による影の像（[影絵](../Page/影絵.md "wikilink")、[影](../Page/影.md "wikilink")、[ブロッケン現象](../Page/ブロッケン現象.md "wikilink")による像など） / [光の反射面に映る像](https://ja.wikipedia.org/wiki/反射#光の反射 "wikilink")（[鏡像](../Page/鏡像.md "wikilink")、水面に映る像など） / 心象としての像・映像（[イメージ](../Page/イメージ.md "wikilink")）

#### 素材・技法に着眼した分類

[偶像](../Page/偶像.md "wikilink") / [土](https://ja.wikipedia.org/wiki/土 "wikilink")の像（[粘土](../Page/粘土.md "wikilink")細工、ほか） / 砂の像（[サンドアートを含む](https://ja.wikipedia.org/wiki/砂遊び#サンドアート "wikilink")） / [石像](https://ja.wikipedia.org/wiki/石像 "wikilink")（[大理石](../Page/大理石.md "wikilink")像、玉彫［[玉製の彫刻](../Page/ヒスイ.md "wikilink")］、[水晶](https://ja.wikipedia.org/wiki/水晶 "wikilink")の像、ほか） / [金属](../Page/金属.md "wikilink")製の像（[銅像](../Page/銅像.md "wikilink")［[青銅](../Page/青銅.md "wikilink")の像］、[黄金像](../Page/金.md "wikilink")、ほか） / [陶磁器](../Page/陶磁器.md "wikilink")像 / [塑像](../Page/塑像.md "wikilink") / 木像 / [草](../Page/草.md "wikilink")・[藁](../Page/藁.md "wikilink")・[蔓](../Page/つる植物.md "wikilink")（[籐を含む](../Page/トウ.md "wikilink")）などの[植物](../Page/植物.md "wikilink")体を使って作った像（笹舟、藁で作られるタイプの昔ながらの[案山子](../Page/かかし.md "wikilink")、[藁人形](../Page/藁人形.md "wikilink")、[ユールゴート](https://ja.wikipedia.org/wiki/ユール#ユール･ゴート "wikilink")、ほか） / [紙](../Page/紙.md "wikilink")の像など[植物繊維](../Page/植物繊維.md "wikilink")の加工物を使って作った像（紙製の[仮面](../Page/仮面.md "wikilink")、[折り紙](../Page/折り紙.md "wikilink")、[人形［ひとがた］](https://ja.wikipedia.org/wiki/人形#祭礼用途の人形 "wikilink")、ほか） / [布](../Page/布.md "wikilink")製の像（[人形](../Page/人形.md "wikilink")、[ぬいぐるみ](../Page/ぬいぐるみ.md "wikilink")、[着ぐるみ](../Page/着ぐるみ.md "wikilink")、ほか） / [骨](../Page/骨.md "wikilink")の像 / [角](../Page/角.md "wikilink")の像、[牙](../Page/牙.md "wikilink")の像（[象牙](../Page/象牙.md "wikilink")の像、ほか） / ワックス彫像（[wax sculpture](https://ja.wikipedia.org/wiki/:en:Wax_sculpture "wikilink")。蝋人形が有名） / [雪像](../Page/雪像.md "wikilink") / [氷像](https://ja.wikipedia.org/wiki/氷像 "wikilink")（主に氷彫刻） / [合成樹脂](../Page/合成樹脂.md "wikilink")の像（合成樹脂製の仮面、[マネキン人形](../Page/マネキン人形.md "wikilink")、“[フィギュア](../Page/フィギュア.md "wikilink")”、[怪獣](../Page/怪獣.md "wikilink")の着ぐるみ、[風船](../Page/風船.md "wikilink")の像、ほか） / 状態が表す像（[あやとり](../Page/あやとり.md "wikilink")で作る像など） / 光による[影](../Page/影.md "wikilink")の像（[影絵](../Page/影絵.md "wikilink")） / ほか

彫像（[彫刻](../Page/彫刻.md "wikilink")像） / 鋳像（[鋳造](../Page/鋳造.md "wikilink")した像） / ほか

#### 主題・意図に着眼した分類

人物像（人物を表す像の全般）、[肖像](../Page/肖像.md "wikilink")（対象者の人となりを表すことに主眼が置かれた人物像で、基本的に動的表現はなされない） / [騎馬像](https://ja.wikipedia.org/wiki/騎馬像 "wikilink") / [神像](../Page/神像.md "wikilink")（[地母神](../Page/地母神.md "wikilink")像、[ヒンドゥー教](../Page/ヒンドゥー教.md "wikilink")の神像、[キリスト教](../Page/キリスト教.md "wikilink")の神像［キリスト像、[聖母子像](https://ja.wikipedia.org/wiki/聖母子像 "wikilink")、[マリア像](https://ja.wikipedia.org/wiki/マリア像 "wikilink")、[イコン](../Page/イコン.md "wikilink")など］、[道教](https://ja.wikipedia.org/wiki/道教 "wikilink")の神像、等々） / [仏像](../Page/仏像.md "wikilink")（広義では[仏教](../Page/仏教.md "wikilink")の神像・人物像なども含む） / [悪鬼](../Page/悪鬼.md "wikilink")像・[悪魔](../Page/悪魔.md "wikilink")像・[伝説上の生物の像](../Page/伝説の生物一覧.md "wikilink") / [神話](../Page/神話.md "wikilink")像・[寓話](../Page/寓話.md "wikilink")像など / [動物形象](../Page/動物形象.md "wikilink")の像 / 自然物・日常的事物の像 / [サブカルチャー](../Page/サブカルチャー.md "wikilink")の像、サブカルチャーの[キャラクター](../Page/キャラクター.md "wikilink")像 / 反意を表す像（好意によって作られる像の対極にあるもの） / ほか

#### 身体の部位に着眼した分類

全身像 / [首](https://ja.wikipedia.org/wiki/首 "wikilink")像 / [頭](../Page/頭.md "wikilink")像 / [顔](../Page/顔.md "wikilink")の像（[仮面](../Page/仮面.md "wikilink")。内側が空洞状になっている、顔にかぶせる形状の頭像） / [胸像](../Page/胸像.md "wikilink") / [トルソー](https://ja.wikipedia.org/wiki/トルソー "wikilink")（胴像） / [性器](../Page/性器.md "wikilink")の像 / ほか

#### 姿勢・状態に着眼した分類

立像 / 坐像（座像。*cf.* [座法](../Page/座法.md "wikilink")） / [騎馬像](https://ja.wikipedia.org/wiki/騎馬像 "wikilink") / 裸像と着衣像 / 巨像・小像 / ほか
