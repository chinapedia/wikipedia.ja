[線型代数学](../Page/線型代数学.md "wikilink")において、[ベクトル](../Page/ベクトル.md "wikilink")の[集合](../Page/集合.md "wikilink")が**線型独立** (せんけいどくりつ、) または**一次独立**であるとは、線型従属（一次従属）でないこと、つまり集合のベクトルの[線型結合](../Page/線型結合.md "wikilink")によるゼロベクトルの表示が自明なものに限ることをいう（[\#定義](https://ja.wikipedia.org/wiki/#定義 "wikilink")）。

## 定義

[Basis_graph_(no_label).svg](https://ja.wikipedia.org/wiki/File:Basis_graph_\(no_label\).svg "fig:Basis_graph_(no_label).svg") 任意の[ベクトル](../Page/ベクトル.md "wikilink") *v*<sub>1</sub>, *v*<sub>2</sub>, ..., *v<sub>n</sub>* に対して

\[0 \boldsymbol v_1 + 0 \boldsymbol v_2 + \dotsb + 0 \boldsymbol v_n = 0\] である。これを *v*<sub>1</sub>, *v*<sub>2</sub>, ..., *v<sub>n</sub>* の[自明な線型関係と呼ぶ](https://ja.wikipedia.org/wiki/自明な_\(数学\) "wikilink")。これは線型従属性と線型独立性の両方の非常に単純な定義を動機付ける。

### 線型従属

[ベクトル空間](https://ja.wikipedia.org/wiki/ベクトル空間 "wikilink") *V* の[部分集合](../Page/部分集合.md "wikilink") *S* が非自明な線型関係を満たすとき —— すなわち、ある*有限*個の*相異なる*ベクトル *v*<sub>1</sub>, *v*<sub>2</sub>, ..., *v<sub>n</sub>* ∈ *S* と[スカラー](https://ja.wikipedia.org/wiki/スカラー_\(数学\) "wikilink") *a*<sub>1</sub>, *a*<sub>2</sub>, ..., *a<sub>n</sub>* が存在して、(*a*<sub>1</sub>, *a*<sub>2</sub>, ..., *a<sub>n</sub>*) ≠ (0, 0, ..., 0) かつ

\[a_1 \boldsymbol v_1 + a_2 \boldsymbol v_2 + \dotsb + a_n \boldsymbol v_n = 0\] を満たすとき —— *S* は**線型従属**（**一次従属**）であるという。言い換えると、集合が線型従属であるとは、集合のベクトルの[線型結合](../Page/線型結合.md "wikilink")によるゼロベクトルの非自明な表示が存在することである。

### 線型独立

ベクトル空間 *V* の部分集合 *S* は線型従属でないとき *S* は**線型独立** （**一次独立**）であるという。明示的には、任意の*有限*個の*相異なる*ベクトル *v*<sub>1</sub>, *v*<sub>2</sub>, ..., *v<sub>n</sub>* ∈ *S* とスカラー *a*<sub>1</sub>, *a*<sub>2</sub>, ..., *a<sub>n</sub>* に対して

\[a_1 \boldsymbol v_1 + a_2 \boldsymbol v_2 + \dotsb + a_n \boldsymbol v_n = 0\] ならば (*a*<sub>1</sub>, *a*<sub>2</sub>, ..., *a<sub>n</sub>*) = (0, 0, ..., 0) となるとき *S* は線型独立であるという。言い換えると、集合が線型独立であるとは、集合のベクトルの線型結合によるゼロベクトルの表示が自明なものに限るということである\[1\]。

文脈から明らかなときには単に従属、独立などと言うこともある。また両方の定義において（厳密性に欠けるが部分集合 *S* ではなく）部分集合 *S* のベクトルが線型従属あるいは線型独立であるとも言うこともある。

## 基本的な性質

  - 線型独立な集合はゼロベクトルを含まない。
  - ゼロでないベクトル *v* に対して一元集合 {*v*} は線型独立である。
  - 線型独立な集合の部分集合は線型独立である。特に[空集合](../Page/空集合.md "wikilink")は線型独立である。

<!-- end list -->

  - 線型独立な集合は[基底](https://ja.wikipedia.org/wiki/基底 "wikilink")に拡張できる。
  - ベクトル空間全体を[生成する集合の線型独立な部分集合全体は](https://ja.wikipedia.org/wiki/線型包 "wikilink")[極大元](https://ja.wikipedia.org/wiki/極大元 "wikilink")（=基底）をもつ。

## 幾何学的な意味

地理的な例は線型独立性の概念を明確にする助けとなるだろう。ある場所の位置を記述している人は「それはここから3キロ北で4キロ東」と言うかもしれない。これは位置を記述するのに十分な情報である、なぜならば地理的な座標系は 2-次元ベクトル空間と考えることができるからである（高度と地球の表面の曲がりは無視して）。その人は「その場所はここから北東に5キロ」と付け加えるかもしれない。この主張は*正しい*が、必要でない。

この例において「3キロ北」ベクトルと「4キロ東」ベクトルは線型独立である。つまり、北ベクトルを東ベクトルの言葉では記述できないし、逆もまたしかり。三番目の「5キロ北東」ベクトルは他の 2 つのベクトルの[線型結合](../Page/線型結合.md "wikilink")であり、ベクトルの集合を「線型従属」にする、つまり、3つのベクトルのうち1つは不要である。

また次のことにも注意しよう。高度が無視されない場合、線型独立な集合に第三のベクトルを付け加えることが必要になる。一般に、*n* 個の線型独立なベクトルは *n*-次元空間の任意の位置を記述するために必要である。

この場合、ベクトルによって形成される行列は

\[A = \begin{bmatrix}1&-3\\1&2\end{bmatrix} . \,\!\] 列の線型結合を次のように書ける

\[A \Lambda = \begin{bmatrix}1&-3\\1&2\end{bmatrix} \begin{bmatrix}\lambda_1 \\ \lambda_2 \end{bmatrix} . \,\!\] ある 0 でないベクトル Λ に対して *A*Λ = **0** かどうかに興味がある。これは *A* の行列式に依存し、それは

\[\det A = 1\cdot2 - 1\cdot(-3) = 5 \ne 0 . \,\!\] 行列式が 0 でないから、ベクトル (1, 1) と (−3, 2) は線型独立である。

別のやり方で、*n* 座標の *m* ベクトルを持っていて *m* \< *n* とする。このとき *A* は *n*×*m* 行列であり Λ は *m* 成分を持つ列ベクトルで、再び *A*Λ = **0** に興味がある。前に見たように、これは *n* 方程式のリストに同値である。*A* の最初の *m* 列、最初の *m* 方程式を考えよう； 方程式の全リストの任意の解は減らされたリストでも解でなければならない。実は、〈*i*<sub>1</sub>,...,*i*<sub>*m*</sub>〉 が *m* 行の任意のリストであれば、方程式はそれらの行に対して正しくなければならない。

\[A_{{\lang i_1,\dots,i_m} \rang} \Lambda = \mathbf{0} . \,\!\] さらに、逆も正しい。つまり、*m* ベクトルが線型従属かどうかを *m* 行のすべての可能なリストに対して

\[\det A_{{\lang i_1,\dots,i_m} \rang} = 0 \,\!\] かどうかをテストすることによってテストできる。（*m* = *n* の場合、これは上のようにただ 1 つの行列式を要求する。*m* \> *n* ならばベクトルは線型従属でなければならないことは定理である。）この事実は理論に値する； 実用計算においてはより効率的な方法が利用可能である。

## 線型従属関係のなす射影空間

ベクトル  の間に成り立つ線型従属関係 (linuear dependence) の係数ベクトルとは、線型関係式

  -
    \(a_1 \boldsymbol{v}_1 + \cdots + a_n \boldsymbol{v}_n=0\)

を満たす  個のスカラーを成分に持つベクトル  で少なくとも一つの成分が  でないものをいう。そのような係数ベクトル  が存在するとき、 個のベクトル  は線型従属である。

個のベクトル  の間に二つの線型従属関係式が与えられたとき、一方の係数ベクトルが他方の非零定数倍となっているならば、これら二つは同じ線型関係を記述するものとなるから、これら二つを同一視することには意味がある。この同一視の下で、 の間の線型従属関係の全体は[射影空間](../Page/射影空間.md "wikilink")を成す。
