## 沿革

  - [1899年](../Page/1899年.md "wikilink")（[明治](../Page/明治.md "wikilink")32年） : [甲斐和里子](https://ja.wikipedia.org/wiki/甲斐和里子 "wikilink")によって顕道女学院が創立（学園の創始）
  - [1900年](../Page/1900年.md "wikilink")（明治33年） : [甲斐駒蔵](https://ja.wikipedia.org/wiki/甲斐駒蔵 "wikilink")・和里子夫妻によって文中園が創立（後に[文中女学校](https://ja.wikipedia.org/wiki/文中女学校 "wikilink")に改称）
  - [1910年](../Page/1910年.md "wikilink")（明治43年） : 文中女学校と[矢部善蔵](https://ja.wikipedia.org/wiki/矢部善蔵 "wikilink")経営の[京都高等女学校](https://ja.wikipedia.org/wiki/京都高等女学校 "wikilink")を統合する。[大谷籌子](https://ja.wikipedia.org/wiki/大谷籌子 "wikilink")・[九条武子](https://ja.wikipedia.org/wiki/九条武子 "wikilink")の尽力で[高等女学校令](https://ja.wikipedia.org/wiki/高等女学校令 "wikilink")による京都高等女学校となり、[西本願寺](https://ja.wikipedia.org/wiki/西本願寺 "wikilink")仏教婦人会連合本部が運営するようになる（学園の創立年）
  - [1920年](../Page/1920年.md "wikilink")（[大正](../Page/大正.md "wikilink")9年） : 京都女子高等専門学校（京都女子大学の前身）が開学
  - [1924年](../Page/1924年.md "wikilink")（大正13年） : [貞明皇后](../Page/貞明皇后.md "wikilink")（[大正天皇](https://ja.wikipedia.org/wiki/大正天皇 "wikilink")妃）が[12月5日](../Page/12月5日.md "wikilink")に行啓され、「あたたかに、そして香りゆかしき心の学校である。」とのお言葉を賜る
  - [1930年](../Page/1930年.md "wikilink")（[昭和](../Page/昭和.md "wikilink")5年） : 本派本願寺保姆養成所を付設
  - [1944年](../Page/1944年.md "wikilink")（昭和19年） : 財団名称を財団法人龍谷女子学園と改称。京都女子専門学校と改称。本派本願寺保姆養成所を京都保母養成所と改称
  - [1949年](../Page/1949年.md "wikilink")（昭和24年） : 新制**京都女子大学**となり、文学部（国文学科・英文学科・中国文史学科）・家政学部（食物学科・被服学科・児童学科）を設置。京都保母養成所を廃止
  - [1950年](../Page/1950年.md "wikilink")（昭和25年） : 文学部の中国文史学科を廃止し、東洋史学科を設置
  - [1951年](https://ja.wikipedia.org/wiki/1951年 "wikilink")（昭和26年） : 設置者を財団法人龍谷女子学園から学校法人京都女子学園に組織変更。京都女子専門学校を廃止
  - [1954年](../Page/1954年.md "wikilink")（昭和29年） : 文学部第二部（国文学科・英文学科）と家政学部第二部（食物学科・被服学科）を設置。京都女子大学附属幼稚園教員養成所を設置
  - [1956年](../Page/1956年.md "wikilink")（昭和31年） : 文学部に初等教育学科を増設
  - [1957年](../Page/1957年.md "wikilink")（昭和32年） : 京都女子大学附属幼稚園教員養成所を廃止
  - [1964年](../Page/1964年.md "wikilink")（昭和39年） : 文学部の初等教育学科を教育学科に改組し、初等教育学専攻と音楽教育学専攻を設置
  - [1966年](../Page/1966年.md "wikilink")（昭和41年） : 京都女子大学大学院文学研究科（国文学専攻・東洋史学専攻）、家政学研究科（児童学専攻）を設置
  - [1967年](../Page/1967年.md "wikilink")（昭和42年） : 大学文学部・家政学部の第二部を廃止。大学文学研究科に英文学専攻、家政学研究科に被服学専攻を増設
  - [1993年](../Page/1993年.md "wikilink")（[平成](../Page/平成.md "wikilink")5年） : 文学部の東洋史学科を史学科に、家政学部の被服学科を生活造形学科に改組。家政学部の食物学科を食物栄養学科に名称変更
  - [1995年](https://ja.wikipedia.org/wiki/1995年 "wikilink")（平成7年） : 大学院文学研究科に教育学専攻修士課程を増設
  - [1997年](https://ja.wikipedia.org/wiki/1997年 "wikilink")（平成9年） : 大学院文学研究科に教育学専攻博士後期課程を増設。大学院文学研究科の東洋史学専攻を史学専攻に改組
  - [1999年](../Page/1999年.md "wikilink")（平成11年） : 大学院文学研究科に史学専攻博士後期課程を増設
  - [2000年](../Page/2000年.md "wikilink")（平成12年） : 現代社会学部（現代社会学科）を設置
  - [2001年](../Page/2001年.md "wikilink")（平成13年） : 大学院文学研究科に国文学専攻博士後期課程、英文学専攻博士後期課程、表現文化専攻修士課程を増設。家政学研究科の被服学専攻を生活造形学専攻に改組。大学院こころの相談室を開設
  - [2004年](https://ja.wikipedia.org/wiki/2004年 "wikilink")（平成16年） : 大学院に現代社会研究科公共圏創生専攻修士課程を設置。大学院家政学研究科に生活環境学専攻博士後期課程を増設。文学部教育学科・家政学部児童学科を改組し、発達教育学部（教育学科（教育学専攻・心理学専攻・音楽教育学専攻）・児童学科）を設置。家政学部に生活福祉学科を増設
  - [2006年](https://ja.wikipedia.org/wiki/2006年 "wikilink")（平成18年） : 大学院に発達教育学研究科（教育学専攻（博士前期・後期課程）・心理学専攻（博士前期課程）・表現文化専攻（修士課程）・児童学専攻（修士課程））を設置。大学院家政学研究科に生活福祉学専攻（博士前期課程）を増設。大学院現代社会研究科に公共圏創成専攻博士後期課程を増設。
  - [2008年](https://ja.wikipedia.org/wiki/2008年 "wikilink")（平成20年） : 京都女子大学栄養クリニックを開設。京都教育大学を主幹大学、本学を含む8大学が聯合大学として参加し、京都教育大学大学院連合教職実践研究科を設置
  - [2009年](../Page/2009年.md "wikilink")（平成21年） : 大学院家政学研究科の食物学専攻博士前期課程を食物栄養学専攻博士前期課程に名称変更
  - [2010年](https://ja.wikipedia.org/wiki/2010年 "wikilink")（平成22年） : 学園創立100周年
  - [2011年](../Page/2011年.md "wikilink")（平成23年） : 日本の女子大学で初となる法学部（法学科）を設置
  - [2015年](../Page/2015年.md "wikilink")（平成27年） : 大学院に法学研究科法学専攻修士課程を設置

#### マスコットキャラクター

  - ふじのちゃん

コンセプトは京女生のイメージは“伝統と現代、両方の長所を兼ね備えている人”である。伝統ある和服姿の美と、華やかな行動的な現代女性をあわせて、京女のシンボルである藤の蝶を髪飾りに付けている。
