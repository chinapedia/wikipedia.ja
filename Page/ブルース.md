**ブルース**\[1\]（Blues）は、[米国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")[深南部で](../Page/ディープサウス.md "wikilink")[アフリカ系アメリカ人](../Page/アフリカ系アメリカ人.md "wikilink")の間から発生した[音楽](../Page/音楽.md "wikilink")の一種およびその[楽式](https://ja.wikipedia.org/wiki/楽式 "wikilink")。[19世紀](../Page/19世紀.md "wikilink")後半頃に米国深南部で[黒人霊歌](https://ja.wikipedia.org/wiki/霊歌 "wikilink")、フィールドハラー（農作業の際の叫び声）や、ワーク・ソング（[労働歌](../Page/労働歌.md "wikilink")）などから発展したものといわれている。[ジャズ](../Page/ジャズ.md "wikilink")が楽器による演奏が主役なのに対して、[ギター](../Page/ギター.md "wikilink")を[伴奏](https://ja.wikipedia.org/wiki/伴奏 "wikilink")に用いた歌が主役である \[2\]。[アコースティック・ギター](../Page/アコースティック・ギター.md "wikilink")の弾き語りを基本とした[デルタ・ブルース](../Page/デルタ・ブルース.md "wikilink")\[3\]、[カントリー・ブルース](https://ja.wikipedia.org/wiki/カントリー・ブルース "wikilink")\[4\]、[エレクトリック・ギター](../Page/エレクトリック・ギター.md "wikilink")を使用した[バンド形式に発展した](../Page/バンド_\(音楽\).md "wikilink")[シカゴ・ブルース](../Page/シカゴ・ブルース.md "wikilink")\[5\]など多様に展開している。

## 概要：音楽の特徴

「ブルース」とは、孤独感や悲しみを表現する独唱歌であり、悲しみや孤独の感情は、[英語](../Page/英語.md "wikilink")ではしばしば「[ブルー](../Page/青.md "wikilink")（*blue*）」の色でたとえられることに由来している\[6\]。[20世紀](../Page/20世紀.md "wikilink")以降の[ポピュラー音楽](../Page/ポピュラー音楽.md "wikilink")に幅広く影響を与え、[ジャズ](../Page/ジャズ.md "wikilink")や[ロックンロール](../Page/ロックンロール.md "wikilink")のルーツのひとつとしても知られている。1900年代にも白人がブルースを楽譜化した例があり、また1912年のフィドル奏者ハート・ワンドによる「ダラス・ブルース」は、ブルースをコピーした楽譜が著作権保護された初期の例とされている\[7\]\[8\]。

ブルースの基本的な構成として、12[小節](https://ja.wikipedia.org/wiki/小節 "wikilink")形式（[ブルース形式](https://ja.wikipedia.org/wiki/ブルース形式 "wikilink")）で綴られる詩が多い。12小節形式の基本はA・A・Bの形式をとる。つまり、4小節の同じ歌詞を二度繰り返し、最後の4小節で締めの歌詞を歌う。これがワンコーラスとなる。

ブルース形式（12小節形式）のコード進行

|    |          |   |         |
| :-: | :------: | :-: | :-----: |
| I  | I または IV | I |    I    |
| IV |    IV    | I |    I    |
| V  | IV または V | I | I または V |

これらのローマ数字は、[コード度数を表している](../Page/和音.md "wikilink")。これが例えばキーがC (ハ長調)だとすると以下の通りとなる。

|   |         |   |         |
| :-: | :-----: | :-: | :-----: |
| C | C または F | C |    C    |
| F |    F    | C |    C    |
| G | F または G | C | C または G |

歌詞は、身近な出来事、感情を表現したものが多い。日常の幸せなことや憂鬱なこと(blues)を12小節に乗せて歌う。[アメリカ南部の黒人たちにとって身近な存在だったギターは](../Page/アメリカ合衆国南部.md "wikilink")、伴奏楽器として適していたこともあり、初期のブルースはギターの弾き語りによるものが多かった。

[thumb](https://ja.wikipedia.org/wiki/ファイル:A_minor_pentatonic_scale.svg "wikilink") [ペンタトニックスケール](../Page/五音音階.md "wikilink")\]\] 旋律に独特の節回しがあり、一般に[ブルー・ノート・スケール](https://ja.wikipedia.org/wiki/ブルー・ノート・スケール "wikilink")と呼ばれている5音階（ペンタトニック・スケール）で演奏される。ブルースのシャッフルまたはウォーキング・[ベース・ギター](https://ja.wikipedia.org/wiki/ベース・ギター "wikilink")は、リズムのコール・アンド・レスポンスを強化し、[グルーヴ](../Page/グルーヴ.md "wikilink")と呼ばれる反復的な効果を形成する。アフリカ系アメリカ人の起源以来のブルースの特徴であるシャッフルは、[スウィング・ジャズ](../Page/スウィング・ジャズ.md "wikilink")で重要な役割を果たした。1940年代に始まった[R\&B](https://ja.wikipedia.org/wiki/R&B "wikilink")の最も単純なシャッフル、リフ・ギターの低音が重視される。このリフをベースとドラムで演奏すると、グルーヴのフィーリングが出てくる。シャッフルリズムは、ダウ、ダダウ、ダダウ、ダ、またはdump、da dump、da dump、da と不均一な、または揺れる8分音符で構成される。ギターでは、これは単純な安定した低音として再生されることもある。クォーターはカントリーやジャズには基本的にみられない音である。また[1950年代](../Page/1950年代.md "wikilink")からはジャズの影響によって、（[ブルー・ノート・スケール](https://ja.wikipedia.org/wiki/ブルー・ノート・スケール "wikilink")5音だけではない）[テンション・ノート](https://ja.wikipedia.org/wiki/テンション・ノート "wikilink")や[テンション・コード](https://ja.wikipedia.org/wiki/テンション・コード "wikilink")が用いられてきた。[シャッフルと呼ばれる](https://ja.wikipedia.org/wiki/シャッフル_\(音楽\) "wikilink")、はねるリズム・パターンも見られる。

## 歴史

### アメリカのブルース史

[250px](https://ja.wikipedia.org/wiki/ファイル:W._C._Handy_-_The_"St._Louis_Blues"_-_First_page.jpg "wikilink")"の譜面(1914)\]\] [19世紀](../Page/19世紀.md "wikilink")後半頃に[米国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")[深南部で](../Page/ディープサウス.md "wikilink")[黒人霊歌](https://ja.wikipedia.org/wiki/霊歌 "wikilink")、フィールドハラー（労働歌）などから発展したものと言われている\[9\]。

[1903年](../Page/1903年.md "wikilink")、[ミシシッピ州](https://ja.wikipedia.org/wiki/ミシシッピ州 "wikilink")の[デルタ地帯](https://ja.wikipedia.org/wiki/デルタ地帯 "wikilink")を旅行中だったが、同州[タトワイラー](https://ja.wikipedia.org/wiki/タトワイラー "wikilink")で黒人によるブルースの生演奏に遭遇した。この後、彼は楽曲を楽譜にして発表し、ブルースは世間に知られることになった。だが、ハンディはブルースを楽譜におこしただけであり、ブルースの父とするのは過大評価である。この年をブルースの生誕の年とする見方もあり、[2003年](../Page/2003年.md "wikilink")はブルース生誕100年を記念して[アメリカ合衆国議会](https://ja.wikipedia.org/wiki/アメリカ合衆国議会 "wikilink")により、「ブルースの年」と宣言された。\[10\]

[1920年](../Page/1920年.md "wikilink")、[メイミー・スミス](https://ja.wikipedia.org/wiki/メイミー・スミス "wikilink")がオーケー・レーベルに初レコーディング。これがブルースのレコーディングとしては初と言われている。彼女の"Crazy Blues"は、初年度75,000枚を売り上げるヒットを記録した。\[11\]また現在、知名度の高い戦前ブルース・シンガーはロバート・ジョンソン\[12\]だが、当時はチャーリー・パットン\[13\]の方が、黒人の間での人気が高かった。

戦前のアメリカにおいて、ブルースは[米国](https://ja.wikipedia.org/wiki/アメリカ合衆国 "wikilink")[深南部から](../Page/ディープサウス.md "wikilink")[セントルイス](../Page/セントルイス.md "wikilink")、[シカゴ](../Page/シカゴ.md "wikilink")、[ニューヨーク](../Page/ニューヨーク.md "wikilink")などへ北上し、各地でスタイルを変えながら発展した。元々ギターの弾き語り中心であったが、都市部に展開するにつれ、ピアノとギターのデュオ形式、バンド形式など、より都会的な洗練された形式へと変わって行った。都市部で展開されたブルースのスタイルをシティ・ブルースという。代表的なミュージシャンは、[リロイ・カー](https://ja.wikipedia.org/wiki/リロイ・カー "wikilink")など。しかし都会にあこがれる反面、故郷への想いが強く詩に影響を与えている歌が多い。\[14\]

シカゴでは、[1950年](../Page/1950年.md "wikilink")頃からエレクトリックのバンドによるブルースが登場した。[デルタ・ブルース](../Page/デルタ・ブルース.md "wikilink")を基調とした泥臭いサウンドで、戦前のシティ・ブルースとは一線を画すものであった。このサウンドはシカゴ・ブルースと呼ばれるようになった。その代表格となるのが、[マディ・ウォーターズ](../Page/マディ・ウォーターズ.md "wikilink")である。ロックンロールの巨匠、[チャック・ベリー](../Page/チャック・ベリー.md "wikilink")もこの頃のブルースに大きく影響を受け、後のロックバンドにも受け継がれているといえる。\[15\]

[1960年代](../Page/1960年代.md "wikilink")初めになると、[イギリス](https://ja.wikipedia.org/wiki/イギリス "wikilink")でアメリカから多くのブルースのレコードが輸入され、同国でブルースのブームが起きた。その流れの中で、[ローリング・ストーンズ](../Page/ローリング・ストーンズ.md "wikilink")、[フリートウッド・マック](../Page/フリートウッド・マック.md "wikilink")、クリーム、ジェスロ・タル（初期）、アニマルズなど、ブルースに影響を受けたバンドが多く登場し、ブルース・ロックが隆盛となった。\[16\]

## 日本のブルース・シーン

日本では、1960年代後半から70年代前半にブルース・ブームが起こったとされる。[1971年](https://ja.wikipedia.org/wiki/1971年 "wikilink")、[B.B.キング](../Page/B.B.キング.md "wikilink")が初来日を果たす。[1973年](../Page/1973年.md "wikilink")に[スリーピー・ジョン・エスティス](https://ja.wikipedia.org/wiki/スリーピー・ジョン・エスティス "wikilink")の「スリーピー・ジョン・エスティスの伝説(The Legend of Sleepy John Estes)」がオリコン・チャートに食い込むヒットとなった。

[1974年](../Page/1974年.md "wikilink")、「第1回ブルース・フェスティバル」開催。同フェスティバルは第3回まで開催され、エスティスを始め[ロバート・ロックウッド・ジュニア](../Page/ロバート・ロックウッド・ジュニア.md "wikilink")&[エイセズ](https://ja.wikipedia.org/wiki/エイセズ "wikilink")、[オーティス・ラッシュ](../Page/オーティス・ラッシュ.md "wikilink")らの来日が実現した。京都、大阪を中心に[ウエスト・ロード・ブルース・バンド](https://ja.wikipedia.org/wiki/ウエスト・ロード・ブルース・バンド "wikilink")、[憂歌団](../Page/憂歌団.md "wikilink")、[ブレイクダウンなど](https://ja.wikipedia.org/wiki/ブレイクダウン_\(バンド\) "wikilink")、ブルース・バンドが登場。日本の独自のブルース・シーンが形成されて行く。 また、楽譜がブルース形式でなくとも、タイトルに「〜ブルース」と付く曲も多く存在する。日本の歌謡曲の中にも、「雨のブルース」、「昭和ブルース」\[19\]など”ブルース”とタイトルがつく曲も多いが、[メロディ](../Page/メロディ.md "wikilink")ーや[アレンジはアメリカの黒人由来のブルースとは異なる](../Page/編曲.md "wikilink")。
