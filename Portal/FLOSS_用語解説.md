用語解説

</h2>

**フリーソフトウェア（Free Software）**

”フリーソフトウェア”と今では呼ばれるようなソフトウェアはかつてより存在していたものの、それが概念となるまでには、1983年に[リチャード・ストールマン](../Page/リチャード・ストールマン.md "wikilink")により提案されるまで待たなければならなかった。フリーソフトウェアの概念はストールマンによる[フリーソフトウェア財団](../Page/フリーソフトウェア財団.md "wikilink")（Free Software Foundation）により用いられ、また彼らによる[フリーソフトウェアの定義](../Page/フリーソフトウェアの定義.md "wikilink")（Free Software Definition）により定義された。これに似た定義が後に公表されており、例えば[Debianフリーソフトウェアガイドライン](../Page/Debianフリーソフトウェアガイドライン.md "wikilink")がそれにあたる。他にも、非公式な諸定義は[BSDベースのオペレーティングシステムのコミュニティ内にあるが](https://ja.wikipedia.org/wiki/Berkeley_Software_Distribution "wikilink")、それらは主に[コピーレフト](../Page/コピーレフト.md "wikilink")の概念に反するという点においてフリーソフトウェアの定義と違いがある。

**オープンソースソフトウェア（Open Source Software）**

[オープンソース](../Page/オープンソース.md "wikilink")とは、History of the OSIによれば、1998年2月3日に、パロアルト (カリフォルニア州)において、Netscapeブラウザのソースコードをどのような形で公開していくかという戦略会議の中でつけられた新たな用語であると説明されている。またオープンソースに関する本 "Open Sources: Voices from the Open Source Revolution" にも、マーケット向けのプロモーション用語として使う新しい言葉“オープンソース”を作り出したとある。これを合言葉として[ブルース・ペレンズ](../Page/ブルース・ペレンズ.md "wikilink")と[エリック・レイモンド](../Page/エリック・レイモンド.md "wikilink")は[オープンソース・イニシアティブ](https://ja.wikipedia.org/wiki/オープンソース・イニシアティブ "wikilink")（Open Source Initiative）という団体を設立するに至った。

**Free/Libre and Open Source Software（FLOSS）**

FLOSS (フロス) は、Free/Libre and Open Source Software の[頭字語](../Page/頭字語.md "wikilink")であり、[フリーソフトウェア](../Page/フリーソフトウェア.md "wikilink")と[オープンソース](../Page/オープンソース.md "wikilink")ソフトウェアをまとめて表現する語である。Free/Open Source Software の頭字語を取って **FOSS** とも呼ばれる。

古くから[フリーソフトウェア](../Page/フリーソフトウェア.md "wikilink")という用語はあったが、自由なソフトウェアを扱う企業からみるといくつかの問題点があった。そのため、1998年には中立的な用語として[オープンソース](../Page/オープンソース.md "wikilink")という用語が作られ、多くの[ユーザーや開発者](../Page/エンドユーザー.md "wikilink")、[マスメディア](../Page/マスメディア.md "wikilink")に広まった。しかし、その一方で、[フリーソフトウェア運動](../Page/フリーソフトウェア運動.md "wikilink")の創始者である[リチャード・ストールマン](../Page/リチャード・ストールマン.md "wikilink")は、その用語が意図を厳密には表していないとして、これを非難した。

その結果、フリーソフトウェアやオープンソースという区分のソフトウェアの双方を包括する表現として FOSS (Free/Open Source Software) や FLOSS が使われるようになっていった。
