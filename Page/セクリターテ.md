**セクリターテ** ( ) は、[ルーマニア社会主義共和国](https://ja.wikipedia.org/wiki/ルーマニア社会主義共和国 "wikilink")の[秘密警察](../Page/秘密警察.md "wikilink")。組織上は内務省の傘下にあり、正式名称は**国家保安局**（）であった。

定冠詞を付けた形で**セクリターテア** ( ) とも呼ばれる。ルーマニア語以外では一般的に、定冠詞を付けない  の形で広まっている。

## 名称

[ルーマニア語](../Page/ルーマニア語.md "wikilink")ではセクリターテの隊員は、単数の男性名詞が「セクリスト」 、単数の女性名詞が「セクリスタ」 、複数形が「セクリシュティ」  となる。

ルーマニア在住の[ハンガリー人達](https://ja.wikipedia.org/wiki/マジャル人 "wikilink")（もっぱらルーマニアに割譲された[トランシルヴァニア](https://ja.wikipedia.org/wiki/トランシルヴァニア "wikilink")に居住する、ルーマニア国籍となっている少数民族。[セーケイ人](../Page/セーケイ人.md "wikilink")や[チャーンゴー人](https://ja.wikipedia.org/wiki/チャーンゴー人 "wikilink")も含む）は、組織のことは[ハンガリー語](https://ja.wikipedia.org/wiki/ハンガリー語 "wikilink")で「セクリターテー」( ) と、セクリターテの隊員のことを、[ハンガリー語](https://ja.wikipedia.org/wiki/ハンガリー語 "wikilink")で警察官 ( ) の俗語である「ヘクシュ」( ) とハンガリー語化された組織名の「セクリターテー」( ) を掛け合わせて「セクシュ」( ) と呼んでいた。「セクシュ」() は単数形で、複数形は「セクショク」( )。（ハンガリー語には文法的な性が存在しないので、男性名詞と女性名詞の区別は存在しない。また   のようなルーマニア語風の表現は存在しなかった。）

ルーマニア在住の[ドイツ人](https://ja.wikipedia.org/wiki/ドイツ人 "wikilink")達（もっぱらルーマニアに割譲された[トランシルヴァニア](https://ja.wikipedia.org/wiki/トランシルヴァニア "wikilink")に居住する、ルーマニア国籍となっている、通常は出自に関わりなく[ザクセン人と呼ばれている少数民族](https://ja.wikipedia.org/wiki/トランシルヴァニア・ザクセン人 "wikilink")）は、[ドイツ語](../Page/ドイツ語.md "wikilink")でセクリターテの隊員は、単数の男性名詞が「ゼクリスト」 、単数の女性名詞が「ゼクリスティン」 、複数形が「ゼクリステン」  と呼ばれていた。

## 特徴

セクリターテの特徴は、メンバーが主として国営の[孤児院](https://ja.wikipedia.org/wiki/孤児院 "wikilink")出身者から構成されていたことであり、一般住民との接触はかなり制限されていた。「優秀な子供」として選ばれた孤児たちは、洗脳と特殊訓練を受けてセクリターテへと育てられた。ルーマニア社会主義共和国は、この組織の下で徹底的な監視社会となり、「街中のカフェやレストランには必ず盗聴器が仕掛けられていた」と言われている程である。

セクリターテの主要任務としては、防諜、反体制派対策、[ニコラエ・チャウシェスク](../Page/ニコラエ・チャウシェスク.md "wikilink")大統領と側近の個人警護を担当していた。その外、ルーマニアには純粋な対外諜報機関である参謀本部情報局と対外情報部（、略称）が存在したにも拘らず、国外でも活動していた。国外でも秘密作戦（反体制派の除去など）のために、セクリターテには「コンドル」グループが存在した。

## 歴史

セクリターテは、[1945年](../Page/1945年.md "wikilink")以前の代表的な[秘密警察](../Page/秘密警察.md "wikilink")である「[特別高等警察](../Page/特別高等警察.md "wikilink")」「[ゲーペーウー](https://ja.wikipedia.org/wiki/ゲーペーウー "wikilink")」「[ゲシュタポ](../Page/ゲシュタポ.md "wikilink")」、「[アーヴェーハー](../Page/ハンガリー国家保衛庁.md "wikilink")」、「[チェーカー](https://ja.wikipedia.org/wiki/チェーカー "wikilink")（反革命・サボタージュ取締全ロシア非常委員会）」、「[エヌカーヴェーデー](https://ja.wikipedia.org/wiki/内務人民委員部 "wikilink")」のルーマニア版である。

  - 1945年～1965年

創立者は、[ソビエト連邦](https://ja.wikipedia.org/wiki/ソビエト連邦 "wikilink")の協力者であった[ジョルジェスク](https://ja.wikipedia.org/wiki/ジョルジェスク "wikilink")である。[ルーマニア人民共和国](https://ja.wikipedia.org/wiki/ルーマニア人民共和国 "wikilink")が立てられた1945年から、[ニコラエ・チャウシェスク](../Page/ニコラエ・チャウシェスク.md "wikilink")が[ルーマニア共産党](../Page/ルーマニア共産党.md "wikilink")書記長に就任する[1965年](../Page/1965年.md "wikilink")までは、セクリターテは「反革命分子だ」「政府の敵だ」と称して民主主義勢力を大規模に弾圧した。

[1948年](../Page/1948年.md "wikilink")[8月30日](../Page/8月30日.md "wikilink")に全国人民保安庁 ( - ) が設立された。設立時の職員の民族構成は、総勢3,973 人の内、ルーマニア人 3,334人 (83.92%)、ユダヤ人 338人 (8.51%)、ハンガリー人 247人 (6.22%)、ロシア人 24人 (0.60%)、ユーゴスラビア人 13人 (0.33%)、ドイツ人 5 人 (0.13%)、チェコ人 5人 (0.13%)、アルメニア人 3人 (0.08%)、ブルガリア人 3人 (0.08%)、イタリア人 1人 (0.03%) であった。

1960年代～1970年代、[スペイン](https://ja.wikipedia.org/wiki/スペイン "wikilink")で「グレナダ」という渾名（あだな）を得た[ワシレ・ゲオルゲ](https://ja.wikipedia.org/wiki/ワシレ・ゲオルゲ "wikilink")がジョルジェスクの後を継ぎ、ゲオルゲは反体制派の尋問（拷問）に個人的に参加すらしていた。その後、ディンク、ヴラダ、ボブ、ポステルニク、ドラギチがセクリターテの長官となった。

  - 1965年～1989年

[ニコラエ・チャウシェスク](../Page/ニコラエ・チャウシェスク.md "wikilink")が共産党書記長に就任し、国名を[ルーマニア社会主義共和国](https://ja.wikipedia.org/wiki/ルーマニア社会主義共和国 "wikilink")に改名した1965年以後は、セクリターテの猛威は一層強まった。

チャウシェスク政権は、ルーマニアの全土に[盗聴](https://ja.wikipedia.org/wiki/盗聴 "wikilink")器を設置して、民衆を恐怖に陥れた。又、孤児たちを洗脳してセクリターテのメンバーにする一方で、全国の[小学校](../Page/小学校.md "wikilink")にも授業時間にセクリターテのメンバーを間諜として送り込み、チャウシェスク政権への不平や不満を政府に報告していた。

### 崩壊後

[1989年](../Page/1989年.md "wikilink")12月の[ルーマニア革命の際にはセクリターテは政権側に立ち](https://ja.wikipedia.org/wiki/ルーマニア革命_\(1989年\) "wikilink")、[ブカレスト](https://ja.wikipedia.org/wiki/ブカレスト "wikilink")では反体制に寝返った[国軍兵士との間に銃撃戦が発生した](https://ja.wikipedia.org/wiki/ルーマニア軍 "wikilink")。

[ニコラエ・チャウシェスク](../Page/ニコラエ・チャウシェスク.md "wikilink")が銃殺されて[ルーマニア社会主義共和国](https://ja.wikipedia.org/wiki/ルーマニア社会主義共和国 "wikilink")が倒され、民主国家が立てられた直後のルーマニアでは、[民主化後の](https://ja.wikipedia.org/wiki/東欧革命 "wikilink")[東ヨーロッパ](../Page/東ヨーロッパ.md "wikilink")で最も苛烈な[脱共産化](https://ja.wikipedia.org/wiki/非共産化 "wikilink")（旧共産党国家の政府関係者への弾圧）が行われ、「[民主主義](../Page/民主主義.md "wikilink")の敵だ」として民衆によってリンチ、殺害された共産党関係者も多かった。元セクリターテ長官のディンク、ヴラダ、ポープ、セクリターテの職員だった[ニク・チャウシェスク](https://ja.wikipedia.org/wiki/ニク・チャウシェスク "wikilink")（ニコラエ・チャウシェスク大統領の次男）など、数百人の元セクリターテのメンバーが裁判にかけられた。最後の長官であるドラガッチャは、[ハンガリー](../Page/ハンガリー.md "wikilink")第三共和国に亡命した。

また、民主政体下（[1990年](https://ja.wikipedia.org/wiki/1990年 "wikilink")以後）で施行された法律により、元セクリターテのメンバーに対して公職禁止が布かれた。元セクリターテのメンバーは、[イスラエル](../Page/イスラエル.md "wikilink")の特務機関で働くか、極右民族主義政党「[大ルーマニア党](https://ja.wikipedia.org/wiki/大ルーマニア党 "wikilink")」に避難したり、または裏の犯罪組織に入ったと考えられている。元メンバーのドゥミトレスクは、[ドイツ](https://ja.wikipedia.org/wiki/ドイツ "wikilink")でルーマニア・ジプシー・マフィア「親衛隊」を率い、かつての同僚を引き込んでいた。また、セクリターテに育成途中だった子供たちの多くは、他の孤児同様に『チャウシェスクの子供たち』と呼ばれる[ストリートチルドレン](../Page/ストリートチルドレン.md "wikilink")になったと言われている。
